<p><b>Понтус Делагарди</b> (<a href="/wiki/%D0%A8%D0%B2%D0%B5%D0%B4%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Шведский язык">швед.</a> <span lang="sv" style="font-style:italic;">Pontus De la Gardie</span>, <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Французский язык">фр.</a> <span lang="fr" style="font-style:italic;">Ponce d'Escouperie</span>; <a class="mw-redirect" href="/wiki/1520" title="1520">1520</a> (?), <a href="/wiki/%D0%9A%D0%BE%D0%BD-%D0%9C%D0%B8%D0%BD%D0%B5%D1%80%D0%B2%D1%83%D0%B0" title="Кон-Минервуа">Кон-Минервуа</a>, <a href="/wiki/%D0%9B%D0%B0%D0%BD%D0%B3%D0%B5%D0%B4%D0%BE%D0%BA" title="Лангедок">Лангедок</a> — <span class="no-wikidata" data-wikidata-property-id="P570"><a href="/wiki/5_%D0%BD%D0%BE%D1%8F%D0%B1%D1%80%D1%8F" title="5 ноября">5 ноября</a> <a href="/wiki/1585_%D0%B3%D0%BE%D0%B4" title="1585 год">1585</a></span>, <a class="mw-redirect" href="/wiki/%D0%9D%D0%B0%D1%80%D0%B2%D0%B0_(%D0%B3%D0%BE%D1%80%D0%BE%D0%B4)" title="Нарва (город)">Нарва</a>) — шведский <a href="/wiki/%D0%9F%D0%BE%D0%BB%D0%BA%D0%BE%D0%B2%D0%BE%D0%B4%D0%B5%D1%86" title="Полководец">полководец</a> и <a href="/wiki/%D0%94%D0%B8%D0%BF%D0%BB%D0%BE%D0%BC%D0%B0%D1%82" title="Дипломат">дипломат</a> <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%8B" title="Французы">французского</a> происхождения. Барон <a class="new" href="/w/index.php?title=%D0%AD%D0%BA%D1%85%D0%BE%D0%BB%D1%8C%D0%BC_(%D0%B7%D0%B0%D0%BC%D0%BE%D0%BA)&amp;action=edit&amp;redlink=1" title="Экхольм (замок) (страница отсутствует)">Экхольм</a> (с 1571), член <a href="/wiki/%D0%A0%D0%B8%D0%BA%D1%81%D1%80%D0%BE%D0%B4" title="Риксрод">риксрода</a>, губернатор <a href="/wiki/%D0%AD%D1%81%D1%82%D0%BB%D1%8F%D0%BD%D0%B4%D0%B8%D1%8F" title="Эстляндия">Эстляндии</a> (1574—1575), губернатор <a class="mw-redirect" href="/wiki/%D0%9B%D0%B8%D1%84%D0%BB%D1%8F%D0%BD%D0%B4%D0%B8%D1%8F" title="Лифляндия">Лифляндии</a> и <a href="/wiki/%D0%98%D0%BD%D0%B3%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%BB%D0%B0%D0%BD%D0%B4%D0%B8%D1%8F" title="Ингерманландия">Ингерманландии</a> (1583—1585). Родоначальник шведского <a href="/wiki/%D0%94%D0%B2%D0%BE%D1%80%D1%8F%D0%BD%D1%81%D1%82%D0%B2%D0%BE" title="Дворянство">дворянского</a> рода <a href="/wiki/%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8" title="Делагарди">Делагарди</a>.
</p><p>Участвовал в кампаниях французской армии в <a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D1%8C%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B5_%D0%B2%D0%BE%D0%B9%D0%BD%D1%8B" title="Итальянские войны">Пьемонте</a> и <a href="/wiki/%D0%A8%D0%BE%D1%82%D0%BB%D0%B0%D0%BD%D0%B4%D0%B8%D1%8F" title="Шотландия">Шотландии</a>, в <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D1%81%D0%B5%D0%BC%D0%B8%D0%BB%D0%B5%D1%82%D0%BD%D1%8F%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Северная семилетняя война">Северной семилетней войне</a> (1563—1570), сначала на стороне <a href="/wiki/%D0%94%D0%B0%D0%BD%D0%B8%D1%8F" title="Дания">Дании</a>, а потом на стороне <a href="/wiki/%D0%A8%D0%B2%D0%B5%D1%86%D0%B8%D1%8F" title="Швеция">Швеции</a>. На службе у <a href="/wiki/%D0%A8%D0%B2%D0%B5%D1%86%D0%B8%D1%8F" title="Швеция">шведских</a> королей и снискал наибольшую славу. Участвовал в заговоре <a href="/wiki/%D0%93%D0%B5%D1%80%D1%86%D0%BE%D0%B3" title="Герцог">герцога</a> Юхана против его брата, короля <a href="/wiki/%D0%AD%D1%80%D0%B8%D0%BA_XIV" title="Эрик XIV">Эрика XIV</a>, благодаря чему особенно возвысился после свержения Эрика и провозглашения заговорщика королём <a href="/wiki/%D0%AE%D1%85%D0%B0%D0%BD_III" title="Юхан III">Юханом III</a>. Выполняя различные военные и <a href="/wiki/%D0%94%D0%B8%D0%BF%D0%BB%D0%BE%D0%BC%D0%B0%D1%82%D0%B8%D1%8F" title="Дипломатия">дипломатические</a> поручения нового государя, посещал дворы европейских монархов и воевал в <a href="/wiki/%D0%AD%D1%81%D1%82%D0%BB%D1%8F%D0%BD%D0%B4%D0%B8%D1%8F" title="Эстляндия">Эстляндии</a> с войсками <a href="/wiki/%D0%98%D0%B2%D0%B0%D0%BD_%D0%93%D1%80%D0%BE%D0%B7%D0%BD%D1%8B%D0%B9" title="Иван Грозный">Ивана Грозного</a>. Наибольшие военные победы одержал в начале <a class="mw-redirect" href="/wiki/1580-%D0%B5" title="1580-е">1580-х годов</a>, когда, воспользовавшись тем, что русские войска были связаны войной с <a href="/wiki/%D0%A0%D0%B5%D1%87%D1%8C_%D0%9F%D0%BE%D1%81%D0%BF%D0%BE%D0%BB%D0%B8%D1%82%D0%B0%D1%8F" title="Речь Посполитая">Речью Посполитой</a>, во главе шведского войска захватил <a href="/wiki/%D0%9A%D0%BE%D1%80%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D1%83%D0%B5%D0%B7%D0%B4" title="Корельский уезд">Корельский уезд</a>, Эстляндию и часть <a href="/wiki/%D0%98%D0%B6%D0%BE%D1%80%D1%81%D0%BA%D0%B0%D1%8F_%D0%B7%D0%B5%D0%BC%D0%BB%D1%8F" title="Ижорская земля">Ижорской земли</a>, в том числе отобрав у <a href="/wiki/%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%BE%D0%B5_%D1%86%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%BE" title="Русское царство">Русского царства</a> главный <a class="mw-redirect" href="/wiki/%D0%9C%D0%BE%D1%80%D1%81%D0%BA%D0%BE%D0%B9_%D0%BF%D0%BE%D1%80%D1%82" title="Морской порт">морской порт</a> на <a href="/wiki/%D0%91%D0%B0%D0%BB%D1%82%D0%B8%D0%B9%D1%81%D0%BA%D0%BE%D0%B5_%D0%BC%D0%BE%D1%80%D0%B5" title="Балтийское море">Балтике</a> — <a href="/wiki/%D0%9D%D0%B0%D1%80%D0%B2%D0%B0" title="Нарва">Нарву</a>. Его военные успехи принудили Россию к заключению невыгодного для неё <a href="/wiki/%D0%9F%D0%BB%D1%8E%D1%81%D1%81%D0%BA%D0%BE%D0%B5_%D0%BF%D0%B5%D1%80%D0%B5%D0%BC%D0%B8%D1%80%D0%B8%D0%B5" title="Плюсское перемирие">Плюсского перемирия</a>, закреплявшего земельные приобретения Делагарди за <a href="/wiki/%D0%A8%D0%B2%D0%B5%D1%86%D0%B8%D1%8F" title="Швеция">Швецией</a>.
</p>

<h2><span id=".D0.96.D0.B8.D0.B7.D0.BD.D0.B5.D0.BE.D0.BF.D0.B8.D1.81.D0.B0.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Жизнеописание">Жизнеописание</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Жизнеописание»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81&amp;action=edit&amp;section=1" title="Редактировать код раздела «Жизнеописание»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id=".D0.A0.D0.BE.D0.B6.D0.B4.D0.B5.D0.BD.D0.B8.D0.B5_.D0.B8_.D0.BC.D0.BE.D0.BB.D0.BE.D0.B4.D1.8B.D0.B5_.D0.B3.D0.BE.D0.B4.D1.8B"></span><span class="mw-headline" id="Рождение_и_молодые_годы">Рождение и молодые годы</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Рождение и молодые годы»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81&amp;action=edit&amp;section=2" title="Редактировать код раздела «Рождение и молодые годы»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Понс Декупри (<a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Французский язык">фр.</a> <span lang="fr" style="font-style:italic;">Pons Descouperie</span>) родился около 1520 года в <a href="/wiki/%D0%98%D0%BC%D0%B5%D0%BD%D0%B8%D0%B5" title="Имение">имении</a> Ла Гарди близ города <a class="mw-redirect" href="/wiki/%D0%9A%D0%B0%D1%80%D0%BA%D0%B0%D1%81%D1%81%D0%BE%D0%BD" title="Каркассон">Каркассона</a> во французском <a href="/wiki/%D0%9B%D0%B0%D0%BD%D0%B3%D0%B5%D0%B4%D0%BE%D0%BA" title="Лангедок">Лангедоке</a>.
</p><p>Отец Понтуса, Жак Декупри (д’Эскупери), принадлежал к старинному французскому роду. В 1511 году он сочетался браком с <a class="new" href="/w/index.php?title=%D0%A1%D0%B5%D0%BD%D1%82-%D0%9A%D0%BE%D0%BB%D0%BE%D0%BC%D0%B1,_%D0%9A%D0%B0%D1%82%D1%80%D0%B8%D0%BD%D0%B0_%D0%B4%D0%B5&amp;action=edit&amp;redlink=1" title="Сент-Коломб, Катрина де (страница отсутствует)">Катриной де Сент-Коломб</a>, в котором у них родилось 3 детей, младшим из которых и был будущий полководец.
</p><p>Родители желали, чтобы их сын пошёл по <a href="/wiki/%D0%94%D1%83%D1%85%D0%BE%D0%B2%D0%B5%D0%BD%D1%81%D1%82%D0%B2%D0%BE" title="Духовенство">духовной стезе</a>, и отдали юношу в <a href="/wiki/%D0%90%D0%B1%D0%B1%D0%B0%D1%82%D1%81%D1%82%D0%B2%D0%BE" title="Аббатство">аббатство</a> <a href="/wiki/%D0%9C%D0%BE%D0%BD%D1%82%D0%BE%D0%BB%D1%8C%D1%91" title="Монтольё">Монтольё</a> <a class="new" href="/w/index.php?title=%D0%9A%D0%B0%D1%80%D0%BA%D0%B0%D1%81%D1%81%D0%BE%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B5%D0%BF%D0%B0%D1%80%D1%85%D0%B8%D1%8F&amp;action=edit&amp;redlink=1" title="Каркассонская епархия (страница отсутствует)">Каркассонской епархии</a>, однако Понтус не смог смириться с подобной участью. Обладая слишком живым характером и тяготясь <a href="/wiki/%D0%9C%D0%BE%D0%BD%D0%B0%D1%85" title="Монах">монашеской</a> жизнью, при первой возможности он покинул аббатство и отправился на войну.
</p><p>Свой первый поход Делагарди совершил под командованием <a href="/wiki/%D0%9A%D0%BE%D1%81%D1%81%D0%B5-%D0%91%D1%80%D0%B8%D1%81%D1%81%D0%B0%D0%BA,_%D0%90%D1%80%D1%82%D1%8E%D1%81_%D0%B4%D0%B5" title="Коссе-Бриссак, Артюс де">маршала де Бриссака</a> в 1551 году в <a href="/wiki/%D0%9F%D1%8C%D0%B5%D0%BC%D0%BE%D0%BD%D1%82" title="Пьемонт">Пьемонт</a>, где в разгаре была одна из <a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D1%8C%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B5_%D0%B2%D0%BE%D0%B9%D0%BD%D1%8B" title="Итальянские войны">итальянских войн</a> короля <a href="/wiki/%D0%93%D0%B5%D0%BD%D1%80%D0%B8%D1%85_II_(%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D1%8C_%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D0%B8)" title="Генрих II (король Франции)">Генриха II</a> с папой <a href="/wiki/%D0%AE%D0%BB%D0%B8%D0%B9_III" title="Юлий III">Юлием III</a> и императором <a class="mw-redirect" href="/wiki/%D0%9A%D0%B0%D1%80%D0%BB_V_%D0%93%D0%B0%D0%B1%D1%81%D0%B1%D1%83%D1%80%D0%B3" title="Карл V Габсбург">Карлом V</a>. В Пьемонте Понтусу удалось отличиться, но с 1552 года боевые действия приняли вялотекущий характер, и в 1556 году он вернулся во Францию. Почти сразу же Делагарди привлекла следующая авантюра: он отправился в <a href="/wiki/%D0%A8%D0%BE%D1%82%D0%BB%D0%B0%D0%BD%D0%B4%D0%B8%D1%8F" title="Шотландия">Шотландию</a> вместе с войсками <a class="new" href="/w/index.php?title=%D0%94%E2%80%99%D0%A3%D0%B0%D0%B7%D0%B5%D0%BB%D1%8C,_%D0%90%D0%BD%D1%80%D0%B8_%D0%9A%D0%BB%D1%8E%D1%82%D0%B5%D0%BD&amp;action=edit&amp;redlink=1" title="Д’Уазель, Анри Клютен (страница отсутствует)">Анри Клютена д’Уазеля</a>, посланного <a href="/wiki/%D0%93%D0%B5%D0%BD%D1%80%D0%B8%D1%85_II_(%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D1%8C_%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D0%B8)" title="Генрих II (король Франции)">Генрихом II</a> для поддержки регентши малолетней <a href="/wiki/%D0%9C%D0%B0%D1%80%D0%B8%D1%8F_%D0%A1%D1%82%D1%8E%D0%B0%D1%80%D1%82" title="Мария Стюарт">Марии Стюарт</a> — <a href="/wiki/%D0%9C%D0%B0%D1%80%D0%B8%D1%8F_%D0%B4%D0%B5_%D0%93%D0%B8%D0%B7" title="Мария де Гиз">Марии де Гиз</a>. Там будущий полководец во главе <a href="/wiki/%D0%9F%D0%B5%D1%85%D0%BE%D1%82%D0%B0" title="Пехота">пехотного</a> полка, кроме прочего, участвовал в гражданской войне между <a href="/wiki/%D0%9A%D0%B0%D1%82%D0%BE%D0%BB%D0%B8%D1%86%D0%B8%D0%B7%D0%BC" title="Католицизм">католиками</a> и <a href="/wiki/%D0%9F%D1%80%D0%BE%D1%82%D0%B5%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%B7%D0%BC" title="Протестантизм">протестантами</a>. В 1559 году Франция завершила многолетнюю борьбу за <a href="/wiki/%D0%93%D0%B5%D0%B3%D0%B5%D0%BC%D0%BE%D0%BD%D0%B8%D1%8F" title="Гегемония">гегемонию</a> в Европе <a href="/wiki/%D0%9A%D0%B0%D1%82%D0%BE-%D0%9A%D0%B0%D0%BC%D0%B1%D1%80%D0%B5%D0%B7%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D0%BC%D0%B8%D1%80" title="Като-Камбрезийский мир">Като-Камбрезийским миром</a>, не давшим ей ни значительных территориальных приобретений, ни каких-либо других преимуществ. В стране нарастали религиозные противоречия, что делало маловероятным французскую военную экспансию в ближайшие годы. Не желавший прекращать военную карьеру Делагарди вместе с 20-ю своими товарищами отправляется в <a href="/wiki/%D0%94%D0%B0%D0%BD%D0%B8%D1%8F" title="Дания">Данию</a>, находившуюся тогда в состоянии войны со <a href="/wiki/%D0%A8%D0%B2%D0%B5%D1%86%D0%B8%D1%8F" title="Швеция">Швецией</a>, и предлагает свои услуги <a class="mw-redirect" href="/wiki/%D0%A4%D1%80%D0%B5%D0%B4%D0%B5%D1%80%D0%B8%D0%BA_II_%D0%94%D0%B0%D1%82%D1%81%D0%BA%D0%B8%D0%B9" title="Фредерик II Датский">Фредерику II Датскому</a>. Шанс показать себя появился, когда заключённое в 1562 году перемирие было нарушено уже через год, разразилась изнурительная и кровавая <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D1%81%D0%B5%D0%BC%D0%B8%D0%BB%D0%B5%D1%82%D0%BD%D1%8F%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Северная семилетняя война">Северная семилетняя война</a>. В начавшихся боевых действиях Понтус Делагарди достаточно проявил себя, чтобы вскоре ему доверили командование наёмниками-<a class="mw-redirect" href="/wiki/%D0%9B%D0%B0%D0%BD%D0%B4%D1%81%D0%BA%D0%BD%D0%B5%D1%85%D1%82%D1%8B" title="Ландскнехты">ландскнехтами</a>. В сентябре <a href="/wiki/1565_%D0%B3%D0%BE%D0%B4" title="1565 год">1565 года</a> его служба датскому королю внезапно завершилась: во время штурма <a href="/wiki/%D0%92%D0%B0%D1%80%D0%B1%D0%B5%D1%80%D0%B3_(%D0%A8%D0%B2%D0%B5%D1%86%D0%B8%D1%8F)" title="Варберг (Швеция)">Варберга</a> Понтус был ранен в руку и взят в плен шведскими войсками<sup class="reference" id="cite_ref-cark_1-0"><a href="#cite_note-cark-1">[1]</a></sup>.
</p>
<h3><span id=".D0.9D.D0.B0_.D1.88.D0.B2.D0.B5.D0.B4.D1.81.D0.BA.D0.BE.D0.B9_.D1.81.D0.BB.D1.83.D0.B6.D0.B1.D0.B5"></span><span class="mw-headline" id="На_шведской_службе">На шведской службе</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81&amp;veaction=edit&amp;section=3" title="Редактировать раздел «На шведской службе»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81&amp;action=edit&amp;section=3" title="Редактировать код раздела «На шведской службе»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure class="mw-default-size" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Sofia_Gyllenhielm.jpg"><img class="mw-file-element" data-file-height="239" data-file-width="181" decoding="async" height="239" src="//upload.wikimedia.org/wikipedia/commons/3/3c/Sofia_Gyllenhielm.jpg" width="181"/></a><figcaption>София Гюлленхельм (<a class="extiw" href="https://sv.wikipedia.org/wiki/Sofia_Johansdotter_(Gyllenhielm)" title="sv:Sofia Johansdotter (Gyllenhielm)">швед.</a>), жена Понтуса Делагарди</figcaption></figure>
<p>Находясь в плену, Понтус Делагарди встретил там своего соотечественника <a href="/wiki/%D0%91%D0%B0%D1%80%D0%BE%D0%BD" title="Барон">барона</a> <a href="/wiki/%D0%9C%D0%BE%D1%80%D0%BD%D0%B5,_%D0%A8%D0%B0%D1%80%D0%BB%D1%8C_%D0%B4%D0%B5" title="Морне, Шарль де">Шарля де Морне</a><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>, состоявшего на шведской службе. Барон представил его королю <a href="/wiki/%D0%AD%D1%80%D0%B8%D0%BA_XIV" title="Эрик XIV">Эрику XIV Вазе</a>, который предложил опытному французскому воину поступить к себе на службу. Делагарди согласился и в том же 1565 году добился от Фредерика II письма с освобождением его от обязательств перед датской короной. Вскоре, проявив талант царедворца, он сумел добиться расположения Эрика. Даже недоверие со стороны фаворита короля, <a href="/wiki/%D0%9A%D0%B0%D0%BD%D1%86%D0%BB%D0%B5%D1%80" title="Канцлер">канцлера</a> <a href="/wiki/%D0%9F%D0%B5%D1%80%D1%81%D1%81%D0%BE%D0%BD,_%D0%99%D1%91%D1%80%D0%B0%D0%BD_(%D0%BA%D0%B0%D0%BD%D1%86%D0%BB%D0%B5%D1%80)" title="Перссон, Йёран (канцлер)">Йёрана Перссона</a>, известного своей подозрительностью, не помешало ему получить в 1568 году должность <a href="/wiki/%D0%93%D0%BE%D1%84%D0%BC%D0%B0%D1%80%D1%88%D0%B0%D0%BB" title="Гофмаршал">гофмаршала</a>. Впрочем, настороженность Перссона оправдалось в полной мере. В 1567 году во время очередного обострения у Эрика психического расстройства <a href="/wiki/%D0%A0%D0%B8%D0%BA%D1%81%D1%80%D0%BE%D0%B4" title="Риксрод">риксрод</a> выпустил из заключения его брата <a href="/wiki/%D0%AE%D1%85%D0%B0%D0%BD_III" title="Юхан III">Юхана</a>, <a class="new" href="/w/index.php?title=%D0%93%D0%B5%D1%80%D1%86%D0%BE%D0%B3_%D0%A4%D0%B8%D0%BD%D0%BB%D1%8F%D0%BD%D0%B4%D1%81%D0%BA%D0%B8%D0%B9&amp;action=edit&amp;redlink=1" title="Герцог Финляндский (страница отсутствует)">герцога Финляндского</a>, который с 1563 года содержался в замке <a href="/wiki/%D0%93%D1%80%D0%B8%D0%BF%D1%81%D1%85%D0%BE%D0%BB%D1%8C%D0%BC" title="Грипсхольм">Грипсхольм</a>. Вместе со вторым братом короля, герцогом <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%BB_IX_(%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D1%8C_%D0%A8%D0%B2%D0%B5%D1%86%D0%B8%D0%B8)" title="Карл IX (король Швеции)">Карлом</a>, Юхан начал готовить переворот, в котором принял участие и Делагарди.
</p><p>В 1568 году благодаря Делагарди герцогам удалось захватить <a href="/wiki/%D0%92%D0%B0%D0%B4%D1%81%D1%82%D0%B5%D0%BD%D0%B0" title="Вадстена">Вадстену</a>, где хранилась государственная <a href="/wiki/%D0%9A%D0%B0%D0%B7%D0%BD%D0%B0" title="Казна">казна</a>. Кроме того, он набрал для них <a href="/wiki/%D0%9D%D0%B5%D0%BC%D1%86%D1%8B" title="Немцы">немецких</a> и <a href="/wiki/%D0%A8%D0%BE%D1%82%D0%BB%D0%B0%D0%BD%D0%B4%D1%86%D1%8B" title="Шотландцы">шотландских</a> наёмников. Потерпев поражение, король Эрик заперся в <a href="/wiki/%D0%A1%D1%82%D0%BE%D0%BA%D0%B3%D0%BE%D0%BB%D1%8C%D0%BC" title="Стокгольм">Стокгольме</a>. В середине сентября армия мятежных герцогов разбила лагерь возле шведской столицы. У Делагарди в городе сохранились связи с горожанами и гарнизоном, и стокгольмцы, недовольные правлением Эрика, вскоре открыли северные ворота и впустили войска мятежников в Стокгольм.
</p><p>Герцог Юхан, провозглашённый королём под именем Юхана III, по достоинству оценил вклад Делагарди в осуществление переворота, пожаловал ему <a class="mw-redirect" href="/wiki/%D0%9B%D0%B5%D0%BD_(%D0%B7%D0%B5%D0%BC%D0%B5%D0%BB%D1%8C%D0%BD%D0%BE%D0%B5_%D0%B2%D0%BB%D0%B0%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5)" title="Лен (земельное владение)">ленные</a> владения, а также отметил иными знаками внимания, в том числе назначив распорядителем своей <a href="/wiki/%D0%9A%D0%BE%D1%80%D0%BE%D0%BD%D0%B0%D1%86%D0%B8%D1%8F" title="Коронация">коронации</a>, состоявшейся 10 июля 1569 года. Однако война с Данией продолжалась, и в том же 1569 году Понтус попал в плен к датчанам, в котором оставался вплоть до весны 1571 года<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>. В награду за долгий плен 17 июля 1571 года король пожаловал ему титул <a href="/wiki/%D0%91%D0%B0%D1%80%D0%BE%D0%BD" title="Барон">барона</a> <a class="mw-disambig" href="/wiki/%D0%AD%D0%BA%D1%85%D0%BE%D0%BB%D1%8C%D0%BC" title="Экхольм">Экхольма</a> и позволение пользоваться в Швеции своим фамильным <a href="/wiki/%D0%93%D0%B5%D1%80%D0%B1" title="Герб">гербом</a>. В 1580 году, после службы в Эстляндии и успешного выполнения многих дипломатических поручений, Делагарди удостаивается чести стать родственником короля, и сочетается браком с его внебрачной дочерью <a href="/wiki/%D0%AE%D0%BB%D0%BB%D0%B5%D0%BD%D1%8A%D0%B5%D0%BB%D1%8C%D0%BC,_%D0%A1%D0%BE%D1%84%D0%B8%D1%8F" title="Юлленъельм, София">Софией Юлленъельм</a>.
</p>
<h3><span id=".D0.94.D0.B8.D0.BF.D0.BB.D0.BE.D0.BC.D0.B0.D1.82.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.B0.D1.8F_.D0.B4.D0.B5.D1.8F.D1.82.D0.B5.D0.BB.D1.8C.D0.BD.D0.BE.D1.81.D1.82.D1.8C"></span><span class="mw-headline" id="Дипломатическая_деятельность">Дипломатическая деятельность</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Дипломатическая деятельность»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81&amp;action=edit&amp;section=4" title="Редактировать код раздела «Дипломатическая деятельность»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Тонкий ум и гибкость Делагарди были замечены ещё Эриком XIV, который не преминул воспользоваться ими, задействовав Понтуса на <a href="/wiki/%D0%94%D0%B8%D0%BF%D0%BB%D0%BE%D0%BC%D0%B0%D1%82%D0%B8%D1%8F" title="Дипломатия">дипломатическом поприще</a>. В 1566 году, в самый разгар Северной семилетней войны, король отправил Делагарди во <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%8F" title="Франция">Францию</a> с поручением воспрепятствовать вербовке наёмников, которую осуществляли датчане на французской территории. Понтус великолепно справился с поставленной задачей, добившись от французского короля права нанять на шведскую службу определённое количество солдат.
</p><p>После завершения Северной семилетней войны Юхан III обратил свои взоры на <a href="/wiki/%D0%9B%D0%B8%D0%B2%D0%BE%D0%BD%D0%B8%D1%8F" title="Ливония">Ливонию</a>, где в августе 1570 года русские войска осадили Ревель. <a href="/wiki/%D0%98%D0%B2%D0%B0%D0%BD_%D0%93%D1%80%D0%BE%D0%B7%D0%BD%D1%8B%D0%B9" title="Иван Грозный">Иван Грозный</a>, стремившийся оставить Швецию в изоляции, старался создать антишведскую коалицию, и Юхан III в 1571 году направил Делагарди в ряд европейских стран, чтобы нейтрализовать усилия русской дипломатии и подорвать торговлю своего восточного соседа. Уже в ноябре он вместе с К. Бельке отправился в <a href="/wiki/%D0%9B%D1%8E%D0%B1%D0%B5%D0%BA" title="Любек">Любек</a>, чтобы добиться от города прекращения торговли с Россией через <a href="/wiki/%D0%9D%D0%B0%D1%80%D0%B2%D0%B0" title="Нарва">Нарву</a>. Ему это, однако, не удалось, после чего он покинул Любек, отправившись сначала в <a href="/wiki/%D0%93%D0%B0%D0%BC%D0%B1%D1%83%D1%80%D0%B3" title="Гамбург">Гамбург</a>, а затем в <a href="/wiki/%D0%A8%D1%82%D1%80%D0%B0%D0%BB%D1%8C%D0%B7%D1%83%D0%BD%D0%B4" title="Штральзунд">Штральзунд</a>, <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%82%D0%BE%D0%BA" title="Росток">Росток</a>, <a href="/wiki/%D0%92%D0%B8%D1%81%D0%BC%D0%B0%D1%80" title="Висмар">Висмар</a> и <a href="/wiki/%D0%9C%D1%8E%D0%BD%D1%81%D1%82%D0%B5%D1%80" title="Мюнстер">Мюнстер</a>.
</p><p>В <a href="/wiki/%D0%9D%D0%B8%D0%B4%D0%B5%D1%80%D0%BB%D0%B0%D0%BD%D0%B4%D1%8B" title="Нидерланды">Нидерландах</a> Делагарди встретился с <a class="mw-redirect" href="/wiki/%D0%93%D0%B5%D1%80%D1%86%D0%BE%D0%B3_%D0%90%D0%BB%D1%8C%D0%B1%D0%B0" title="Герцог Альба">герцогом Альбой</a><sup class="reference" id="cite_ref-cark_1-1"><a href="#cite_note-cark-1">[1]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>, однако ничего, кроме обещания списаться с <a href="/wiki/%D0%A4%D0%B8%D0%BB%D0%B8%D0%BF%D0%BF_II_(%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D1%8C_%D0%98%D1%81%D0%BF%D0%B0%D0%BD%D0%B8%D0%B8)" title="Филипп II (король Испании)">королём</a>, от него не добился. Весной 1572 года шведский посол прибыл во Францию, где смог получить от <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%BB_IX_(%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D1%8C_%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D0%B8)" title="Карл IX (король Франции)">Карла IX</a> обещание отправить своему представителю в <a href="/wiki/%D0%9A%D0%BE%D0%BF%D0%B5%D0%BD%D0%B3%D0%B0%D0%B3%D0%B5%D0%BD" title="Копенгаген">Копенгагене</a> приказ о возобновлении старых франко-шведских договоров. По вопросу нарвской торговли французский король дал ответ, что не может полностью запретить данную торговлю, но Франция намерена придерживаться тех соглашений, которые будут заключены со шведской стороной. Находясь в <a href="/wiki/%D0%91%D0%BB%D1%83%D0%B0" title="Блуа">Блуа</a>, Делагарди получил от испанского короля письмо, в котором тот обещал поставлять в Швецию <a class="mw-redirect" href="/wiki/%D0%A1%D0%BE%D0%BB%D1%8C" title="Соль">соль</a>, <a href="/wiki/%D0%A5%D0%BC%D0%B5%D0%BB%D1%8C" title="Хмель">хмель</a> и прочие товары, ранее ввозившееся в шведское государство Любеком, с которым шведы находились в довольно прохладных отношениях.
</p><p>В сентябре 1573 года Делагарди вернулся в Швецию<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>.
</p><p>В 1576—1578 годах Понтус вновь возглавлял посольство, на этот раз к <a class="mw-redirect" href="/wiki/%D0%98%D0%BC%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%BE%D1%80%D1%8B_%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D0%BE%D0%B9_%D0%A0%D0%B8%D0%BC%D1%81%D0%BA%D0%BE%D0%B9_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D0%B8" title="Императоры Священной Римской империи">германскому императору</a>. Помимо переговоров с <a href="/wiki/%D0%A0%D1%83%D0%B4%D0%BE%D0%BB%D1%8C%D1%84_II" title="Рудольф II">Рудольфом II</a>, посол должен был посетить <a href="/wiki/%D0%A0%D0%B8%D0%BC" title="Рим">Рим</a> и обсудить с папой <a href="/wiki/%D0%93%D1%80%D0%B8%D0%B3%D0%BE%D1%80%D0%B8%D0%B9_XIII" title="Григорий XIII">Григорием XIII</a> вынашиваемые Юханом III планы по восстановлению в Швеции <a class="mw-redirect" href="/wiki/%D0%9A%D0%B0%D1%82%D0%BE%D0%BB%D0%B8%D1%87%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Католичество">католичества</a>.
</p>
<h3><span id=".D0.92.D0.BE.D0.B9.D0.BD.D0.B0_.D1.81_.D0.A0.D0.BE.D1.81.D1.81.D0.B8.D0.B5.D0.B9"></span><span class="mw-headline" id="Война_с_Россией">Война с Россией</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Война с Россией»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81&amp;action=edit&amp;section=5" title="Редактировать код раздела «Война с Россией»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Karta_Delagardi.gif"><img class="mw-file-element" data-file-height="545" data-file-width="817" decoding="async" height="180" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b1/Karta_Delagardi.gif/270px-Karta_Delagardi.gif" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/b1/Karta_Delagardi.gif/405px-Karta_Delagardi.gif 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/b1/Karta_Delagardi.gif/540px-Karta_Delagardi.gif 2x" width="270"/></a><figcaption>Карта части <a href="/wiki/%D0%9A%D0%BE%D1%80%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D1%83%D0%B5%D0%B7%D0%B4" title="Корельский уезд">Корельского</a> и <a href="/wiki/%D0%9E%D1%80%D0%B5%D1%85%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D1%83%D0%B5%D0%B7%D0%B4" title="Ореховский уезд">Ореховского уездов</a>, предположительно составленная по приказу Понтуса Делагарди в 1580—1582 годах.</figcaption></figure>
<p>По возвращении в Швецию из первой дипломатической поездки в Европу Делагарди был назначен одним из командующих шведскими войсками в Прибалтике, а в дополнение 4 июня 1574 года получил пост губернатора <a href="/wiki/%D0%AD%D1%81%D1%82%D0%BB%D1%8F%D0%BD%D0%B4%D0%B8%D1%8F" title="Эстляндия">Эстляндии</a><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>.
</p><p>Прибыв к войскам, Делагарди сумел восстановить пошатнувшуюся дисциплину, однако военные действия в 1573—1575 годах велись вяло, и дело по большей части ограничивалось осадой городов в Северной Эстляндии<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>. Участвовал в провальной <a href="/wiki/%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%92%D0%B5%D0%B7%D0%B5%D0%BD%D0%B1%D0%B5%D1%80%D0%B3%D0%B0_(1574)" title="Осада Везенберга (1574)">осаде Везенберга</a> 1574 года, однако вина за неудачу была возложена на <a href="/wiki/%D0%A2%D0%BE%D1%82%D1%82,_%D0%9A%D0%BB%D0%B0%D1%81_%D0%9E%D0%BA%D0%B5%D1%81%D1%81%D0%BE%D0%BD" title="Тотт, Клас Окессон">Класа Тотта</a>, и Делагарди был назначен единоличным командующим шведскими войсками в Прибалтике. Летом же 1575 года между Россией и Швецией было заключено двухлетнее перемирие, и Делагарди был отозван из Эстляндии, в декабре 1575 года сложив полномочия губернатора ради нового дипломатического поручения. Только в 1580 году ему вновь было доверено командование шведскими войсками в возобновившейся в 1577 году войне с русскими.
</p>
<h4><span id=".D0.9F.D0.B5.D1.80.D0.B2.D1.8B.D0.B5_.D1.83.D1.81.D0.BF.D0.B5.D1.85.D0.B8"></span><span class="mw-headline" id="Первые_успехи">Первые успехи</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Первые успехи»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81&amp;action=edit&amp;section=6" title="Редактировать код раздела «Первые успехи»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h4>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Gerb_keksgolm.jpg"><img class="mw-file-element" data-file-height="835" data-file-width="485" decoding="async" height="232" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/9f/Gerb_keksgolm.jpg/135px-Gerb_keksgolm.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/9f/Gerb_keksgolm.jpg/203px-Gerb_keksgolm.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/9/9f/Gerb_keksgolm.jpg/270px-Gerb_keksgolm.jpg 2x" width="135"/></a><figcaption>Герб Кексгольмского герцогства (1580)</figcaption></figure>
<p>В конце октября 1580 года Понтус Делагарди вышел с войском из <a href="/wiki/%D0%92%D1%8B%D0%B1%D0%BE%D1%80%D0%B3" title="Выборг">Выборга</a> и 26 числа осадил <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D1%80%D0%B5%D0%BB%D0%B0_(%D0%B3%D0%BE%D1%80%D0%BE%D0%B4)" title="Корела (город)">Корелу</a> — главный опорный пункт русских на <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B5%D1%80%D0%B5%D1%88%D0%B5%D0%B5%D0%BA" title="Карельский перешеек">Карельском перешейке</a> и в <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D0%BE%D0%B5_%D0%9F%D1%80%D0%B8%D0%BB%D0%B0%D0%B4%D0%BE%D0%B6%D1%8C%D0%B5" title="Северное Приладожье">Северном Приладожье</a>. 4 ноября подтянулись пушки, позволившие начать бомбардировку, и почти сразу же шведской <a href="/wiki/%D0%90%D1%80%D1%82%D0%B8%D0%BB%D0%BB%D0%B5%D1%80%D0%B8%D1%8F" title="Артиллерия">артиллерии</a> удалось поджечь деревянные стены и постройки внутри <a href="/wiki/%D0%9A%D0%BE%D1%80%D0%B5%D0%BB%D0%B0_(%D0%BA%D1%80%D0%B5%D0%BF%D0%BE%D1%81%D1%82%D1%8C)" title="Корела (крепость)">крепости</a>. Судьба города оказалась предрешена, 5 ноября гарнизон сдался, а весь <a href="/wiki/%D0%9A%D0%BE%D1%80%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D1%83%D0%B5%D0%B7%D0%B4" title="Корельский уезд">Корельский уезд</a> оказался под властью Швеции<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup>. Это событие послужило сюжетом для первого герба захваченной шведами провинции, дарованного в том же 1580 году созданному <a class="new" href="/w/index.php?title=%D0%9A%D0%B5%D0%BA%D1%81%D0%B3%D0%BE%D0%BB%D1%8C%D0%BC%D1%81%D0%BA%D0%BE%D0%B5_%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3%D1%81%D1%82%D0%B2%D0%BE&amp;action=edit&amp;redlink=1" title="Кексгольмское герцогство (страница отсутствует)">Кексгольмскому герцогству</a>. На нём была изображена горящая крепость, золотые языки пламени с красными отсветами и два пылающих <a href="/wiki/%D0%9F%D1%83%D1%88%D0%B5%D1%87%D0%BD%D0%BE%D0%B5_%D1%8F%D0%B4%D1%80%D0%BE" title="Пушечное ядро">пушечных ядра</a><sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>.
</p><p>После взятия Корелы Делагарди спешно двинулся к другой важнейшей русской крепости — <a href="/wiki/%D0%9E%D1%80%D0%B5%D1%88%D0%B5%D0%BA_(%D0%BA%D1%80%D0%B5%D0%BF%D0%BE%D1%81%D1%82%D1%8C)" title="Орешек (крепость)">Орешку</a>, запиравшей выход из <a href="/wiki/%D0%9D%D0%B5%D0%B2%D0%B0" title="Нева">Невы</a> в <a href="/wiki/%D0%9B%D0%B0%D0%B4%D0%BE%D0%B6%D1%81%D0%BA%D0%BE%D0%B5_%D0%BE%D0%B7%D0%B5%D1%80%D0%BE" title="Ладожское озеро">Ладожское озеро</a>, и, организовав её <a href="/wiki/%D0%92%D0%BE%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B1%D0%BB%D0%BE%D0%BA%D0%B0%D0%B4%D0%B0" title="Военная блокада">блокаду</a>, по льду <a href="/wiki/%D0%A4%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%B7%D0%B0%D0%BB%D0%B8%D0%B2" title="Финский залив">Финского залива</a> перевёл войско в <a href="/wiki/%D0%AD%D1%81%D1%82%D0%BB%D1%8F%D0%BD%D0%B4%D0%B8%D1%8F" title="Эстляндия">Эстляндию</a>. В Эстляндии за первую половину 1581 года полководец овладел <a href="/wiki/%D0%9B%D0%B8%D1%85%D1%83%D0%BB%D0%B0" title="Лихула">Леалем</a>, <a href="/wiki/%D0%A0%D0%B0%D0%BA%D0%B2%D0%B5%D1%80%D0%B5" title="Раквере">Везенбергом</a>, <a class="mw-redirect" href="/wiki/%D0%A2%D0%BE%D0%BB%D1%8C%D1%81%D0%B1%D1%83%D1%80%D0%B3" title="Тольсбург">Тольсбургом</a>, <a class="mw-redirect" href="/wiki/%D0%93%D0%B0%D0%BF%D1%81%D0%B0%D0%BB%D1%8C" title="Гапсаль">Гапсалем</a> и другими крепостями, практически полностью очистив эту область от русских <a href="/wiki/%D0%93%D0%B0%D1%80%D0%BD%D0%B8%D0%B7%D0%BE%D0%BD" title="Гарнизон">гарнизонов</a>. Последней, но самой крупной и защищённой русской крепостью в регионе оставалась <a href="/wiki/%D0%9D%D0%B0%D1%80%D0%B2%D0%B0" title="Нарва">Нарва</a>, являвшаяся в то время крупнейшим центром русской торговли с Европой.
</p>
<h4><span id=".D0.92.D0.B7.D1.8F.D1.82.D0.B8.D0.B5_.D0.9D.D0.B0.D1.80.D0.B2.D1.8B_.D0.B8_.D0.B7.D0.B0.D0.BF.D0.B0.D0.B4.D0.BD.D0.BE.D0.B9_.D0.98.D0.BD.D0.B3.D1.80.D0.B8.D0.B8"></span><span class="mw-headline" id="Взятие_Нарвы_и_западной_Ингрии"><a href="/wiki/%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%9D%D0%B0%D1%80%D0%B2%D1%8B_(1581)" title="Осада Нарвы (1581)">Взятие Нарвы</a> и западной Ингрии</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Взятие Нарвы и западной Ингрии»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81&amp;action=edit&amp;section=7" title="Редактировать код раздела «Взятие Нарвы и западной Ингрии»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h4>
<p>В августе 1581 году Делагарди вместе с <a href="/wiki/%D0%90%D0%B4%D0%BC%D0%B8%D1%80%D0%B0%D0%BB" title="Адмирал">адмиралом</a> <a href="/wiki/%D0%A4%D0%BB%D0%B5%D0%BC%D0%B8%D0%BD%D0%B3,_%D0%9A%D0%BB%D0%B0%D1%81_%D0%AD%D1%80%D0%B8%D0%BA%D1%81%D1%81%D0%BE%D0%BD" title="Флеминг, Клас Эрикссон">Флемингом</a> по <a href="/wiki/%D0%A4%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%B7%D0%B0%D0%BB%D0%B8%D0%B2" title="Финский залив">Финскому заливу</a> и реке <a href="/wiki/%D0%9D%D0%B0%D1%80%D0%B2%D0%B0_(%D1%80%D0%B5%D0%BA%D0%B0)" title="Нарва (река)">Нарове</a> на кораблях прибыл к крепости <a class="mw-redirect" href="/wiki/%D0%9D%D0%B0%D1%80%D0%B2%D0%B0_(%D0%B3%D0%BE%D1%80%D0%BE%D0%B4)" title="Нарва (город)">Нарва</a> и, обложив её с трёх сторон, начал готовиться к штурму. Перед его началом Делагарди объявил своим войскам, что на приступ пойдут лишь те, кто заранее запишется у него, все же остальные будут лишены права участвовать в разграблении города, на которое им отводилось 24 часа. 6 сентября Нарва была взята, и за следующие сутки шведами было убито около 7 тыс. человек мирного населения, не считая русских <a href="/wiki/%D0%A1%D1%82%D1%80%D0%B5%D0%BB%D1%8C%D1%86%D1%8B" title="Стрельцы">стрельцов</a> из гарнизона крепости. Вскоре Делагарди, пользуясь тем, что русские гарнизоны крепостей <a href="/wiki/%D0%98%D0%B6%D0%BE%D1%80%D1%81%D0%BA%D0%B0%D1%8F_%D0%B7%D0%B5%D0%BC%D0%BB%D1%8F" title="Ижорская земля">Ижорской земли</a> были частично отвлечены на <a class="mw-redirect" href="/wiki/%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%9F%D1%81%D0%BA%D0%BE%D0%B2%D0%B0_(1581%E2%80%941582)" title="Осада Пскова (1581—1582)">оборону Пскова</a> от войск <a href="/wiki/%D0%A1%D1%82%D0%B5%D1%84%D0%B0%D0%BD_%D0%91%D0%B0%D1%82%D0%BE%D1%80%D0%B8%D0%B9" title="Стефан Баторий">Стефана Батория</a>, победным маршем прошёл всю западную <a href="/wiki/%D0%98%D0%BD%D0%B3%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%BB%D0%B0%D0%BD%D0%B4%D0%B8%D1%8F" title="Ингерманландия">Ингерманландию</a> и, уже не встречая серьёзного сопротивления, занял крепости <a href="/wiki/%D0%98%D0%B2%D0%B0%D0%BD%D0%B3%D0%BE%D1%80%D0%BE%D0%B4" title="Ивангород">Ивангород</a>, <a href="/wiki/%D0%AF%D0%BC" title="Ям">Ям</a> и <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%BF%D0%BE%D1%80%D1%81%D0%BA%D0%B0%D1%8F_%D0%BA%D1%80%D0%B5%D0%BF%D0%BE%D1%81%D1%82%D1%8C" title="Копорская крепость">Копорье</a><sup class="reference" id="cite_ref-narva_13-0"><a href="#cite_note-narva-13">[13]</a></sup>. Однако продолжавшаяся безуспешная <a href="/wiki/%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%9E%D1%80%D0%B5%D1%88%D0%BA%D0%B0_(1582)" title="Осада Орешка (1582)">осада Орешка</a> не давала довершить завоевание <a class="mw-redirect" href="/wiki/%D0%9E%D1%80%D0%B5%D1%85%D0%BE%D0%B2%D0%B5%D1%86%D0%BA%D0%B8%D0%B9_%D1%83%D0%B5%D0%B7%D0%B4" title="Ореховецкий уезд">Ореховецкого уезда</a>, штурм же, предпринятый Понтусом осенью 1582 года, сопровождался большими потерями среди осаждавших и окончился неудачей. Последнюю попытку расширить приобретения данной кампании полководец предпринял в ноябре, послав на <a class="mw-redirect" href="/wiki/%D0%9D%D0%BE%D0%B2%D0%B3%D0%BE%D1%80%D0%BE%D0%B4" title="Новгород">Новгород</a> шведскую <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%BD%D0%BD%D0%B8%D1%86%D0%B0" title="Конница">конницу</a>. Однако посланный отряд не только не добрался до города из-за осенней <a href="/wiki/%D0%A0%D0%B0%D1%81%D0%BF%D1%83%D1%82%D0%B8%D1%86%D0%B0" title="Распутица">распутицы</a>, но и вернулся, потеряв на обратном пути большую часть коней и обозов.
</p>
<h4><span id=".D0.97.D0.B0.D0.B2.D0.B5.D1.80.D1.88.D0.B5.D0.BD.D0.B8.D0.B5_.D0.B1.D0.BE.D0.B5.D0.B2.D1.8B.D1.85_.D0.B4.D0.B5.D0.B9.D1.81.D1.82.D0.B2.D0.B8.D0.B9"></span><span class="mw-headline" id="Завершение_боевых_действий">Завершение боевых действий</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81&amp;veaction=edit&amp;section=8" title="Редактировать раздел «Завершение боевых действий»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81&amp;action=edit&amp;section=8" title="Редактировать код раздела «Завершение боевых действий»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h4>
<p>В 1582 году король назначил Делагарди главнокомандующим всеми шведскими войсками. Осознавая изменчивость военного счастья, тот посоветовал <a href="/wiki/%D0%AE%D1%85%D0%B0%D0%BD_III" title="Юхан III">Юхану III</a> заключить мир на предложенных русскими выгодных условиях, но король всё же продолжил войну. В феврале того же года из <a class="mw-redirect" href="/wiki/%D0%9D%D0%BE%D0%B2%D0%B3%D0%BE%D1%80%D0%BE%D0%B4" title="Новгород">Новгорода</a> выступило русское ополчение, один отряд — в направлении <a class="mw-redirect" href="/wiki/%D0%9D%D0%B0%D1%80%D0%B2%D0%B0_(%D0%B3%D0%BE%D1%80%D0%BE%D0%B4)" title="Нарва (город)">Нарвы</a>, другой, под командованием князя <a href="/wiki/%D0%A8%D1%83%D0%B9%D1%81%D0%BA%D0%B8%D0%B9,_%D0%90%D0%BD%D0%B4%D1%80%D0%B5%D0%B9_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87" title="Шуйский, Андрей Иванович">Андрея Шуйского</a> — к <a href="/wiki/%D0%9E%D1%80%D0%B5%D1%88%D0%B5%D0%BA_(%D0%BA%D1%80%D0%B5%D0%BF%D0%BE%D1%81%D1%82%D1%8C)" title="Орешек (крепость)">Орешку</a>. Оба отряда достигли определённых успехов — первый под командованием <a href="/wiki/%D0%A5%D0%B2%D0%BE%D1%80%D0%BE%D1%81%D1%82%D0%B8%D0%BD%D0%B8%D0%BD,_%D0%94%D0%BC%D0%B8%D1%82%D1%80%D0%B8%D0%B9_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87" title="Хворостинин, Дмитрий Иванович">Дмитрия Хворостинина</a> разбил шведов в <a href="/wiki/%D0%91%D0%B8%D1%82%D0%B2%D0%B0_%D0%BF%D0%BE%D0%B4_%D0%9B%D1%8F%D0%BB%D0%B8%D1%86%D0%B0%D0%BC%D0%B8" title="Битва под Лялицами">битве под Лялицами</a>, второй же заставил Делагарди снять осаду Ореховской крепости и отступить в <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B5%D0%BB%D0%B8%D1%8F" title="Карелия">Карелию</a>. Только ухудшившаяся внутренняя и внешняя политическая обстановка заставили царя <a class="mw-redirect" href="/wiki/%D0%98%D0%B2%D0%B0%D0%BD_IV" title="Иван IV">Ивана IV</a> остановить боевые действия и в 1583 году пойти на невыгодное <a href="/wiki/%D0%9F%D0%BB%D1%8E%D1%81%D1%81%D0%BA%D0%BE%D0%B5_%D0%BF%D0%B5%D1%80%D0%B5%D0%BC%D0%B8%D1%80%D0%B8%D0%B5" title="Плюсское перемирие">Плюсское перемирие</a>, в подписании которого участвовал и Понтус Делагарди<sup class="reference" id="cite_ref-gipping_14-0"><a href="#cite_note-gipping-14">[14]</a></sup>.
</p><p>За свои заслуги ещё до заключения перемирия Делагарди был назначен наместником <a class="mw-redirect" href="/wiki/%D0%9B%D0%B8%D1%84%D0%BB%D1%8F%D0%BD%D0%B4%D0%B8%D1%8F" title="Лифляндия">Лифляндии</a> и <a href="/wiki/%D0%98%D0%BD%D0%B3%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%BB%D0%B0%D0%BD%D0%B4%D0%B8%D1%8F" title="Ингерманландия">Ингерманландии</a>, а через два года, в 1585 году, получил должность члена <a href="/wiki/%D0%A0%D0%B8%D0%BA%D1%81%D1%80%D0%BE%D0%B4" title="Риксрод">риксрода</a>. В этом же году он принял участие в переговорах о продлении перемирия.
</p>
<h3><span id=".D0.93.D0.B8.D0.B1.D0.B5.D0.BB.D1.8C"></span><span class="mw-headline" id="Гибель">Гибель</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81&amp;veaction=edit&amp;section=9" title="Редактировать раздел «Гибель»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81&amp;action=edit&amp;section=9" title="Редактировать код раздела «Гибель»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/5_%D0%BD%D0%BE%D1%8F%D0%B1%D1%80%D1%8F" title="5 ноября">5 ноября</a> <a href="/wiki/1585_%D0%B3%D0%BE%D0%B4" title="1585 год">1585 года</a> лодка, в которой Делагарди возвращался из очередного похода, перевернулась, и знаменитый полководец утонул в реке <a href="/wiki/%D0%9D%D0%B0%D1%80%D0%B2%D0%B0_(%D1%80%D0%B5%D0%BA%D0%B0)" title="Нарва (река)">Нарова</a> в виду Нарвского замка вместе с восемнадцатью своими спутниками. Есть несколько версий случившегося: по одной версий лодка Делагарди развалилась от выстрела, произведенного с неё из пушки в знак победоносного возвращения, по другой, гребцы, заглядевшись на устроенную им встречу, налетели на препятствие под водой и пробили дно — «…принесло судно ветром на пень да вынесло доску, а Немцы почали метатца в воду, и судно потонуло, и на завтра в субботу выволокли из воды Пунцу». Есть также версия, что лодку Делагарди случайно поразили сами встречающие — ядром салюта. Обряженный в тяжёлые рыцарские доспехи Делагарди утонул<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup>.
</p><p>
Когда русские <a href="/wiki/%D0%9F%D0%BE%D1%81%D0%BE%D0%BB" title="Посол">послы</a> донесли о его гибели в Москву, ими был получен ответ: </p><div class="ts-Цитата"><blockquote><p>Писали вы нам, что Пунтус Делагарди утонул; сделалось это Божиим милосердием и великого <a href="/wiki/%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B9_%D0%A7%D1%83%D0%B4%D0%BE%D1%82%D0%B2%D0%BE%D1%80%D0%B5%D1%86" title="Николай Чудотворец">чудотворца Николы</a> милостию<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup>.</p></blockquote></div>
<p>В Москве, узнав о гибели Делагарди, служили благодарственный молебен<sup class="ts-fix-template noprint">[<i><a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%A1%D1%81%D1%8B%D0%BB%D0%BA%D0%B8_%D0%BD%D0%B0_%D0%B8%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8" title="Википедия:Ссылки на источники"><span title="нет источника (6 августа 2020)">источник не указан 1301 день</span></a></i>]</sup>.
</p><p>Полководец был похоронен в <a href="/wiki/%D0%94%D0%BE%D0%BC%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D0%BE%D0%B1%D0%BE%D1%80_(%D0%A2%D0%B0%D0%BB%D0%BB%D0%B8%D0%BD)" title="Домский собор (Таллин)">Домском соборе</a> <a class="mw-redirect" href="/wiki/%D0%A0%D0%B5%D0%B2%D0%B5%D0%BB%D1%8C" title="Ревель">Ревеля</a> (современный <a href="/wiki/%D0%A2%D0%B0%D0%BB%D0%BB%D0%B8%D0%BD" title="Таллин">Таллин</a>), рядом со своей женой, умершей при родах в <a href="/wiki/1583_%D0%B3%D0%BE%D0%B4" title="1583 год">1583 году</a>. Работу над <a href="/wiki/%D0%9D%D0%B0%D0%B4%D0%B3%D1%80%D0%BE%D0%B1%D0%B8%D0%B5" title="Надгробие">надгробием</a> четы Делагарди <a href="/wiki/%D0%A4%D0%BB%D0%B0%D0%BD%D0%B4%D1%80%D0%B8%D1%8F_(%D0%B8%D1%81%D1%82%D0%BE%D1%80%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C)" title="Фландрия (историческая область)">фламандский</a> <a class="mw-redirect" href="/wiki/%D0%A1%D0%BA%D1%83%D0%BB%D1%8C%D0%BF%D1%82%D0%BE%D1%80" title="Скульптор">скульптор</a> <a href="/wiki/%D0%9F%D0%B0%D1%81%D1%81%D0%B5%D1%80,_%D0%90%D1%80%D0%B5%D0%BD%D1%82" title="Пассер, Арент">Арент Пассер</a> начал только в <a href="/wiki/1589_%D0%B3%D0%BE%D0%B4" title="1589 год">1589 году</a>, а закончил в <a href="/wiki/1595_%D0%B3%D0%BE%D0%B4" title="1595 год">1595 году</a>. На крышке <a href="/wiki/%D0%A1%D0%B0%D1%80%D0%BA%D0%BE%D1%84%D0%B0%D0%B3" title="Саркофаг">саркофага</a> он поместил фигуры супругов, вырубленные из камня: слева — Понтуса, в военном облачении, справа — Софии, в наряде по <a href="/wiki/%D0%98%D1%81%D0%BF%D0%B0%D0%BD%D0%B8%D1%8F" title="Испания">испанской</a> моде того времени. По бокам были расположены <a href="/wiki/%D0%91%D0%B0%D1%80%D0%B5%D0%BB%D1%8C%D0%B5%D1%84" title="Барельеф">барельефы</a>, один из которых изображал осаду Нарвы. Над саркофагом была помещена резная <a href="/wiki/%D0%AD%D0%BF%D0%B8%D1%82%D0%B0%D1%84%D0%B8%D1%8F" title="Эпитафия">эпитафия</a>.
</p><p>Надгробие Делагарди является выдающимся памятником <a class="mw-redirect" href="/wiki/%D0%A0%D0%B5%D0%BD%D0%B5%D1%81%D1%81%D0%B0%D0%BD%D1%81" title="Ренессанс">ренессансного</a> искусства<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup><sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup>.
</p>
<h2><span id=".D0.94.D0.B2.D0.BE.D1.80.D1.8F.D0.BD.D1.81.D0.BA.D0.B8.D0.B9_.D1.80.D0.BE.D0.B4_.D0.94.D0.B5.D0.BB.D0.B0.D0.B3.D0.B0.D1.80.D0.B4.D0.B8"></span><span class="mw-headline" id="Дворянский_род_Делагарди">Дворянский род Делагарди</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81&amp;veaction=edit&amp;section=10" title="Редактировать раздел «Дворянский род Делагарди»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81&amp;action=edit&amp;section=10" title="Редактировать код раздела «Дворянский род Делагарди»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Delagardie_palace_stockholm.PNG"><img class="mw-file-element" data-file-height="1056" data-file-width="1442" decoding="async" height="183" src="//upload.wikimedia.org/wikipedia/commons/thumb/7/78/Delagardie_palace_stockholm.PNG/250px-Delagardie_palace_stockholm.PNG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/7/78/Delagardie_palace_stockholm.PNG/375px-Delagardie_palace_stockholm.PNG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/7/78/Delagardie_palace_stockholm.PNG/500px-Delagardie_palace_stockholm.PNG 2x" width="250"/></a><figcaption>Дворец семьи Делагарди, построенный Якобом Понтуссоном рядом с <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D1%80%D0%BE%D0%BB%D0%B5%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%B4%D0%B2%D0%BE%D1%80%D0%B5%D1%86_(%D0%A1%D1%82%D0%BE%D0%BA%D0%B3%D0%BE%D0%BB%D1%8C%D0%BC)" title="Королевский дворец (Стокгольм)">королевским дворцом</a> в <a href="/wiki/%D0%A1%D1%82%D0%BE%D0%BA%D0%B3%D0%BE%D0%BB%D1%8C%D0%BC" title="Стокгольм">Стокгольме</a></figcaption></figure>
<div class="hatnote navigation-not-searchable">Основная статья: <b><a href="/wiki/%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8" title="Делагарди">Делагарди</a></b></div>
<p>Полководец и дипломат Понтус Делагарди стал основателем знаменитого шведского <a href="/wiki/%D0%94%D0%B2%D0%BE%D1%80%D1%8F%D0%BD%D1%81%D1%82%D0%B2%D0%BE" title="Дворянство">дворянского</a> <a class="mw-redirect" href="/wiki/%D0%A0%D0%BE%D0%B4_(%D0%B3%D0%B5%D0%BD%D0%B5%D0%B0%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Род (генеалогия)">рода</a>, из которого вышли многие известные шведские государственные деятели. Женой полководца была внебрачная дочь Юхана III София (Юхансдоттер) Юлленъельм (Гюлленхельм, Гюлленьельм) (ок. 1559—1583). Через год после свадьбы, в <a href="/wiki/1581_%D0%B3%D0%BE%D0%B4" title="1581 год">1581 году</a>, в <a class="mw-redirect" href="/wiki/%D0%A0%D0%B5%D0%B2%D0%B5%D0%BB%D1%8C" title="Ревель">Ревеле</a> София родила Понтусу первого ребёнка — дочь Бриту (<a class="mw-redirect" href="/wiki/1581" title="1581">1581</a>—<a class="mw-redirect" href="/wiki/1645" title="1645">1645</a>), через год — сына Юхана (<a class="mw-redirect" href="/wiki/1582" title="1582">1582</a>—<a class="mw-redirect" href="/wiki/1642" title="1642">1642</a>), а ещё через год, в июне <a href="/wiki/1583_%D0%B3%D0%BE%D0%B4" title="1583 год">1583 года</a> — <a href="/wiki/%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%AF%D0%BA%D0%BE%D0%B1_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81%D1%81%D0%BE%D0%BD" title="Делагарди, Якоб Понтуссон">Якоба</a> (<a class="mw-redirect" href="/wiki/1583" title="1583">1583</a>—<a class="mw-redirect" href="/wiki/1652" title="1652">1652</a>). Последние тяжёлые роды София не пережила. Она похоронена вместе со своим мужем в <a href="/wiki/%D0%94%D0%BE%D0%BC%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D0%BE%D0%B1%D0%BE%D1%80_(%D0%A2%D0%B0%D0%BB%D0%BB%D0%B8%D0%BD)" title="Домский собор (Таллин)">Домском соборе Ревеля</a>. Дети же Понтуса, почти не знавшие своего отца, были воспитаны в <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BD%D1%8F%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%BE_%D0%A4%D0%B8%D0%BD%D0%BB%D1%8F%D0%BD%D0%B4%D1%81%D0%BA%D0%BE%D0%B5" title="Великое княжество Финляндское">Финляндии</a> своей бабкой, Карин Хансдоттер (<a href="/wiki/%D0%A8%D0%B2%D0%B5%D0%B4%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Шведский язык">швед.</a> <span lang="sv" style="font-style:italic;">Karin Hansdotter</span>)<sup class="reference" id="cite_ref-cark_1-2"><a href="#cite_note-cark-1">[1]</a></sup>. Младший сын Понтуса, <a href="/wiki/%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%AF%D0%BA%D0%BE%D0%B1_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81%D1%81%D0%BE%D0%BD" title="Делагарди, Якоб Понтуссон">Якоб Делагарди</a>, превзошёл славу своего отца, участвуя сначала в отражении Россией <a href="/wiki/%D0%9F%D0%BE%D0%BB%D1%8F%D0%BA%D0%B8" title="Поляки">польской</a> интервенции в <a href="/wiki/%D0%A1%D0%BC%D1%83%D1%82%D0%BD%D0%BE%D0%B5_%D0%B2%D1%80%D0%B5%D0%BC%D1%8F" title="Смутное время">Смутное время</a> (<a href="/wiki/%D0%9F%D0%BE%D1%85%D0%BE%D0%B4_%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8" title="Поход Делагарди">Поход Делагарди</a>), а потом в <a class="mw-redirect" href="/wiki/%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%BE-%D1%88%D0%B2%D0%B5%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0_1614%E2%80%941617" title="Русско-шведская война 1614—1617">войне с Россией</a>, называемой в иностранных источниках «Ингерманландской». Командуя шведскими войсками в ходе войны, Якоб довершил дело своего отца и всё-таки взял <a class="mw-redirect" href="/wiki/%D0%9D%D0%BE%D0%B2%D0%B3%D0%BE%D1%80%D0%BE%D0%B4" title="Новгород">Новгород</a>, а также многие другие города <a href="/wiki/%D0%9D%D0%BE%D0%B2%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D0%B7%D0%B5%D0%BC%D0%BB%D1%8F" title="Новгородская земля">Новгородской земли</a>. Результатом был тяжелейший для России <a href="/wiki/%D0%A1%D1%82%D0%BE%D0%BB%D0%B1%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%BC%D0%B8%D1%80" title="Столбовский мир">Столбовский мир</a>, почти на столетие лишивший её выхода к <a href="/wiki/%D0%91%D0%B0%D0%BB%D1%82%D0%B8%D0%B9%D1%81%D0%BA%D0%BE%D0%B5_%D0%BC%D0%BE%D1%80%D0%B5" title="Балтийское море">Балтийскому морю</a>. За свои заслуги Якоб Понтуссон получил от шведского монарха <a href="/wiki/%D0%93%D1%80%D0%B0%D1%84_(%D1%82%D0%B8%D1%82%D1%83%D0%BB)" title="Граф (титул)">графский</a> титул и обширные ленные владения. Прославился на государственном поприще и внук Понтуса, <a href="/wiki/%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9C%D0%B0%D0%B3%D0%BD%D1%83%D1%81_%D0%93%D0%B0%D0%B1%D1%80%D0%B8%D1%8D%D0%BB%D1%8C" title="Делагарди, Магнус Габриэль">Магнус Габриэль Делагарди</a>, являвшийся членом <a href="/wiki/%D0%A0%D0%B8%D0%BA%D1%81%D1%80%D0%BE%D0%B4" title="Риксрод">риксрода</a>, <a href="/wiki/%D0%A0%D0%B8%D0%BA%D1%81%D0%BC%D0%B0%D1%80%D1%88%D0%B0%D0%BB" title="Риксмаршал">риксмаршалом</a>, а с <a class="mw-redirect" href="/wiki/1660" title="1660">1660</a> по <a href="/wiki/1680_%D0%B3%D0%BE%D0%B4" title="1680 год">1680 год</a> — <a class="new" href="/w/index.php?title=%D0%A0%D0%B8%D0%BA%D1%81%D0%BA%D0%B0%D0%BD%D1%86%D0%BB%D0%B5%D1%80&amp;action=edit&amp;redlink=1" title="Риксканцлер (страница отсутствует)">риксканцлером</a> Швеции. Потомки первого обосновавшегося в Швеции Делагарди занимают различные государственные посты и в настоящее время. Одна из графских ветвей рода Делагарди оказалась и в России, но в середине <a href="/wiki/XIX_%D0%B2%D0%B5%D0%BA" title="XIX век">XIX века</a> она пресеклась, и право наследовать <a href="/wiki/%D0%93%D1%80%D0%B0%D1%84_(%D1%82%D0%B8%D1%82%D1%83%D0%BB)" title="Граф (титул)">графский</a> титул и фамилию Делагарди получил <a href="/wiki/%D0%91%D1%80%D0%B5%D0%B2%D0%B5%D1%80%D0%BD_%D0%B4%D0%B5_%D0%9B%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87" title="Бреверн де Лагарди, Александр Иванович">Александр Бреверн</a>, он и его потомки носили впоследствии двойную фамилию — <a class="new" href="/w/index.php?title=%D0%91%D1%80%D0%B5%D0%B2%D0%B5%D1%80%D0%BD-%D0%B4%D0%B5-%D0%9B%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8&amp;action=edit&amp;redlink=1" title="Бреверн-де-Лагарди (страница отсутствует)">Бреверн-де-Лагарди</a>.
</p>
<h2><span id=".D0.9F.D0.BE.D0.BD.D1.82.D1.83.D1.81_.D0.94.D0.B5.D0.BB.D0.B0.D0.B3.D0.B0.D1.80.D0.B4.D0.B8_.D0.B2_.D0.BD.D0.B0.D1.80.D0.BE.D0.B4.D0.BD.D0.BE.D0.B9_.D0.BF.D0.B0.D0.BC.D1.8F.D1.82.D0.B8"></span><span class="mw-headline" id="Понтус_Делагарди_в_народной_памяти">Понтус Делагарди в народной памяти</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81&amp;veaction=edit&amp;section=11" title="Редактировать раздел «Понтус Делагарди в народной памяти»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81&amp;action=edit&amp;section=11" title="Редактировать код раздела «Понтус Делагарди в народной памяти»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Командуя солдатами на поле боя, и с лёгкостью располагая к себе людей в мирной обстановке, Понтус Делагарди, очевидно, не был лишён определённых <a href="/wiki/%D0%A5%D0%B0%D1%80%D0%B8%D0%B7%D0%BC%D0%B0" title="Харизма">харизматических</a> качеств. Этот фактор, а также то, что земли, по которым он прошёл с войсками в <a class="mw-redirect" href="/wiki/1580" title="1580">1580</a>-<a href="/wiki/1582_%D0%B3%D0%BE%D0%B4" title="1582 год">1582 годах</a>, давно не знали военных потрясений, сделали его настоящим <a href="/wiki/%D0%A4%D0%BE%D0%BB%D1%8C%D0%BA%D0%BB%D0%BE%D1%80" title="Фольклор">фольклорным</a> героем среди местного населения — <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B5%D0%BB%D1%8B" title="Карелы">карел</a>, <a href="/wiki/%D0%98%D0%B6%D0%BE%D1%80%D0%B0" title="Ижора">ижор</a> и <a href="/wiki/%D0%92%D0%BE%D0%B4%D1%8C" title="Водь">води</a>. В первую очередь это запечатлилось в финско-карельско-ижорской <a href="/wiki/%D0%A2%D0%BE%D0%BF%D0%BE%D0%BD%D0%B8%D0%BC%D0%B8%D0%BA%D0%B0" title="Топонимика">топонимике</a>, во многих местах <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B5%D1%80%D0%B5%D1%88%D0%B5%D0%B5%D0%BA" title="Карельский перешеек">Карельского перешейка</a> сохранились названия, связанные с именем Делагарди. Активная деятельность полководца по строительству дорог и мостов, необходимых для передвижения войск по <a href="/wiki/%D0%91%D0%BE%D0%BB%D0%BE%D1%82%D0%BE" title="Болото">болотистым</a>, <a href="/wiki/%D0%9B%D0%B5%D1%81" title="Лес">лесистым</a> местностям, оставила на карте «Мосты Понтуса» (<a href="/wiki/%D0%A4%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Финский язык">фин.</a> <span lang="fi" style="font-style:italic;">Pontuxen sillat</span>), «<a href="/wiki/%D0%93%D0%B0%D1%82%D1%8C" title="Гать">Гати</a> Понтуса», «Рвы Понтуса» и пр. Севернее нынешнего посёлка <a href="/wiki/%D0%A0%D0%B5%D0%BF%D0%B8%D0%BD%D0%BE_(%D0%A1%D0%B0%D0%BD%D0%BA%D1%82-%D0%9F%D0%B5%D1%82%D0%B5%D1%80%D0%B1%D1%83%D1%80%D0%B3)" title="Репино (Санкт-Петербург)">Репино</a> существовало «Болото Понтуса», скорее всего носящее это название, так как по приказу Делагарди через него была проложена дорога, необходимая для передвижения его армии<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup>. Одна из гор недалеко от посёлка <a href="/wiki/%D0%A2%D0%BE%D0%BA%D1%81%D0%BE%D0%B2%D0%BE" title="Токсово">Токсово</a>, на которой по преданию в <a href="/wiki/1581_%D0%B3%D0%BE%D0%B4" title="1581 год">1581 году</a> находился лагерь Делагарди, среди <a href="/wiki/%D0%98%D0%BD%D0%B3%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%BB%D0%B0%D0%BD%D0%B4%D1%86%D1%8B" title="Ингерманландцы">финского</a> и <a href="/wiki/%D0%98%D0%B6%D0%BE%D1%80%D0%B0" title="Ижора">ижорского</a> населения долгое время сохраняла название «Гора Понтуса» (<a href="/wiki/%D0%A4%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Финский язык">фин.</a> <span lang="fi" style="font-style:italic;">Pontuxen mäki</span>). Развалины крепости <a class="mw-redirect" href="/wiki/%D0%9A%D0%B8%D0%B2%D0%B8%D0%BD%D0%B5%D0%B1%D0%B1" title="Кивинебб">Кивинебб</a> возле современного посёлка <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B2%D0%BE%D0%BC%D0%B0%D0%B9%D1%81%D0%BA%D0%BE%D0%B5_(%D0%9B%D0%B5%D0%BD%D0%B8%D0%BD%D0%B3%D1%80%D0%B0%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C)" title="Первомайское (Ленинградская область)">Первомайское</a>, уничтоженной русскими войсками в середине <a href="/wiki/XVI_%D0%B2%D0%B5%D0%BA" title="XVI век">XVI века</a>, также именовались у местного населения «Понтусова Крепость» (<a href="/wiki/%D0%A4%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Финский язык">фин.</a> <span lang="fi" style="font-style:italic;">Pontuxen linna</span>), что, видимо, указывает на попытку полководца восстановить её.
</p><p>В устных сказаниях жителей Карельского перешейка и <a href="/wiki/%D0%98%D0%BD%D0%B3%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%BB%D0%B0%D0%BD%D0%B4%D0%B8%D1%8F" title="Ингерманландия">Ингерманландии</a> Понтус представал чародеем, <a class="mw-redirect" href="/wiki/%D0%92%D0%BE%D0%BB%D1%88%D0%B5%D0%B1%D1%81%D1%82%D0%B2%D0%BE" title="Волшебство">волшебником</a>, которому всегда и во всём помогает <a href="/wiki/%D0%9D%D0%B5%D1%87%D0%B8%D1%81%D1%82%D0%B0%D1%8F_%D1%81%D0%B8%D0%BB%D0%B0" title="Нечистая сила">нечистая сила</a>. В преданиях полководец мог одновременно сражаться в нескольких местах, а недостаток в войске, восполнять превращая в воинов обычные перья<sup class="reference" id="cite_ref-gipping_14-1"><a href="#cite_note-gipping-14">[14]</a></sup>. Передававшиеся из уст в уста легенды о Понтусе записывали у ижор ещё в <a class="mw-redirect" href="/wiki/XIX" title="XIX">XIX</a> и <a href="/wiki/XX_%D0%B2%D0%B5%D0%BA" title="XX век">XX веках</a>. Например, хорошо известна историческая ижорская песня «Военный поход Понтуса». Впрочем, здесь образ Делагарди слился с образом его сына — <a href="/wiki/%D0%94%D0%B5%D0%BB%D0%B0%D0%B3%D0%B0%D1%80%D0%B4%D0%B8,_%D0%AF%D0%BA%D0%BE%D0%B1_%D0%9F%D0%BE%D0%BD%D1%82%D1%83%D1%81%D1%81%D0%BE%D0%BD" title="Делагарди, Якоб Понтуссон">Якоба Понтуссона</a>, также воевавшего в местах проживания ижор<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup>. Жестокость Понтуса запечатлена в <a href="/wiki/%D0%A2%D0%B0%D0%BB%D0%BB%D0%B8%D0%BD" title="Таллин">таллинской</a> <a href="/wiki/%D0%93%D0%BE%D1%80%D0%BE%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D0%BB%D0%B5%D0%B3%D0%B5%D0%BD%D0%B4%D0%B0" title="Городская легенда">городской легенде</a>, которую <a href="/wiki/%D0%9A%D1%80%D0%B5%D0%B9%D1%86%D0%B2%D0%B0%D0%BB%D1%8C%D0%B4,_%D0%A4%D1%80%D0%B8%D0%B4%D1%80%D0%B8%D1%85" title="Крейцвальд, Фридрих">Ф. Р. Крейцвальд</a> поместил в свой сборник «Старинные эстонские народные сказки». В ней полководец предстаёт настолько алчным, что якобы во время боевых действий сдирал кожу с убитых солдат и отдавал её <a href="/wiki/%D0%94%D1%83%D0%B1%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5" title="Дубление">дубильщику</a>. В наказание за эти злодеяния смерть не принесла покой Делагарди, его дух обречён бродить по улицам <a href="/wiki/%D0%A2%D0%B0%D0%BB%D0%BB%D0%B8%D0%BD" title="Таллин">Таллина</a>, пытаясь продать прохожим дублёные кожи, от которых покупатели шарахались, не в силах терпеть исходящий от них дурной запах<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup>.
</p><p>Некоторые легенды, живущие до сих пор, явно недостоверны, но продолжают бытовать даже в <a href="/wiki/%D0%9A%D1%80%D0%B0%D0%B5%D0%B2%D0%B5%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5" title="Краеведение">краеведческой</a> литературе. Например, вестник <a href="/wiki/%D0%92%D1%81%D0%B5%D0%B2%D0%BE%D0%BB%D0%BE%D0%B6%D1%81%D0%BA" title="Всеволожск">Всеволожского</a> краеведческого музея передаёт легенду о связи имени Понтуса Делагарди с историей <a href="/wiki/%D0%9A%D1%80%D0%B0%D1%81%D0%BD%D1%8B%D0%B9_%D0%B7%D0%B0%D0%BC%D0%BE%D0%BA" title="Красный замок">Красного замка</a> — строения начала XIX века в <a class="mw-redirect" href="/wiki/%D0%9C%D1%8B%D0%B7%D0%B0_%D0%A0%D1%8F%D0%B1%D0%BE%D0%B2%D0%BE" title="Мыза Рябово">Румболовском парке</a>. Полководец вместе с войском якобы молился в этом <a href="/wiki/%D0%97%D0%B0%D0%BC%D0%BE%D0%BA_(%D1%81%D1%82%D1%80%D0%BE%D0%B5%D0%BD%D0%B8%D0%B5)" title="Замок (строение)">замке</a>, бывшем тогда придорожной <a href="/wiki/%D0%9A%D0%B8%D1%80%D1%85%D0%B0" title="Кирха">кирхой</a>, перед походом к городу <a href="/wiki/%D0%9E%D1%80%D0%B5%D1%88%D0%B5%D0%BA_(%D0%BA%D1%80%D0%B5%D0%BF%D0%BE%D1%81%D1%82%D1%8C)" title="Орешек (крепость)">Орехову</a> в <a href="/wiki/1556_%D0%B3%D0%BE%D0%B4" title="1556 год">1556 году</a>. На самом деле в это время Делагарди ещё не было на шведской службе, и даже перенос действия легенды на <a href="/wiki/1580-%D0%B5_%D0%B3%D0%BE%D0%B4%D1%8B" title="1580-е годы">1580-е годы</a> вызывает сомнения, ведь тогда местность принадлежала России, и если существовал тогда Красный замок, то придорожной кирхой быть не мог<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup>.
</p>
