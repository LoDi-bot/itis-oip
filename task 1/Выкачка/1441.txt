<div class="mw-body-content" id="mw-content-text"><div class="mw-content-ltr mw-parser-output" dir="ltr" lang="ru"><p><b>Трилогия Нова</b> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">The Nova Trilogy</span>, также известная как <b>The Cut-up Trilogy</b><sup class="reference" id="cite_ref-Wald_1-0"><a href="#cite_note-Wald-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>) — название, используемое критиками для обозначения цикла из трёх самых известных экспериментальных <a href="/wiki/%D0%A0%D0%BE%D0%BC%D0%B0%D0%BD" title="Роман">романов</a> <a href="/wiki/%D0%91%D0%B5%D1%80%D1%80%D0%BE%D1%83%D0%B7,_%D0%A3%D0%B8%D0%BB%D1%8C%D1%8F%D0%BC_%D0%A1%D1%8C%D1%8E%D0%B0%D1%80%D0%B4" title="Берроуз, Уильям Сьюард">Уильяма Берроуза</a>: «<a href="/wiki/%D0%9C%D1%8F%D0%B3%D0%BA%D0%B0%D1%8F_%D0%BC%D0%B0%D1%88%D0%B8%D0%BD%D0%B0" title="Мягкая машина">Мягкая машина</a>», «<a href="/wiki/%D0%91%D0%B8%D0%BB%D0%B5%D1%82,_%D0%BA%D0%BE%D1%82%D0%BE%D1%80%D1%8B%D0%B9_%D0%BB%D0%BE%D0%BF%D0%BD%D1%83%D0%BB" title="Билет, который лопнул">Билет, который лопнул</a>» и «<a href="/wiki/%D0%9D%D0%BE%D0%B2%D0%B0_%D0%AD%D0%BA%D1%81%D0%BF%D1%80%D0%B5%D1%81%D1%81" title="Нова Экспресс">Нова Экспресс</a>»<sup class="reference" id="cite_ref-Wising_4-0"><a href="#cite_note-Wising-4">[4]</a></sup>.
</p><p>В некоторых случаях рассматриваемый в качестве <a href="/wiki/%D0%A2%D0%B5%D1%82%D1%80%D0%B0%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F" title="Тетралогия">тетралогии</a><sup class="reference" id="cite_ref-DeKoven_5-0"><a href="#cite_note-DeKoven-5">[5]</a></sup><sup class="reference" id="cite_ref-Stringer_6-0"><a href="#cite_note-Stringer-6">[6]</a></sup><sup class="reference" id="cite_ref-Zott_7-0"><a href="#cite_note-Zott-7">[7]</a></sup> с главным для <a href="/wiki/%D0%91%D0%B8%D1%82-%D0%BF%D0%BE%D0%BA%D0%BE%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5" title="Бит-поколение">бит-поколения</a> романом «<a href="/wiki/%D0%93%D0%BE%D0%BB%D1%8B%D0%B9_%D0%B7%D0%B0%D0%B2%D1%82%D1%80%D0%B0%D0%BA" title="Голый завтрак">Голый завтрак</a>» (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">The Naked Lunch</span>, <a href="/wiki/1959_%D0%B3%D0%BE%D0%B4" title="1959 год">1959</a>, рус. перевод <a href="/wiki/1971_%D0%B3%D0%BE%D0%B4" title="1971 год">1971</a>) в качестве <a href="/wiki/%D0%9F%D1%80%D0%B8%D0%BA%D0%B2%D0%B5%D0%BB" title="Приквел">приквела</a><sup class="reference" id="cite_ref-Stringer_6-1"><a href="#cite_note-Stringer-6">[6]</a></sup><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>, цикл романов возник из материала, написанного писателем в <a href="/wiki/%D0%A2%D0%B0%D0%BD%D0%B6%D0%B5%D1%80" title="Танжер">Танжере</a>, <a href="/wiki/%D0%9F%D0%B0%D1%80%D0%B8%D0%B6" title="Париж">Париже</a>, <a href="/wiki/%D0%9D%D1%8C%D1%8E-%D0%99%D0%BE%D1%80%D0%BA" title="Нью-Йорк">Нью-Йорке</a> и <a href="/wiki/%D0%9B%D0%BE%D0%BD%D0%B4%D0%BE%D0%BD" title="Лондон">Лондоне</a> в период с <a href="/wiki/1953_%D0%B3%D0%BE%D0%B4" title="1953 год">1953</a> по <a href="/wiki/1958_%D0%B3%D0%BE%D0%B4" title="1958 год">1958 год</a> и доработанного в <a href="/wiki/1959_%D0%B3%D0%BE%D0%B4" title="1959 год">1959 году</a><sup class="reference" id="cite_ref-Skerl_9-0"><a href="#cite_note-Skerl-9">[9]</a></sup>.
</p><p>Все романы созданы в технике «<i>fold-in</i>», заключающейся в составлении произведения путём случайной компоновки предварительно разрезанного на мелкие части текста. Методика была разработана <a href="/wiki/%D0%91%D0%B5%D1%80%D1%80%D0%BE%D1%83%D0%B7,_%D0%A3%D0%B8%D0%BB%D1%8C%D1%8F%D0%BC_%D0%A1%D1%8C%D1%8E%D0%B0%D1%80%D0%B4" title="Берроуз, Уильям Сьюард">Уильямом Берроузом</a> и <a href="/wiki/%D0%93%D0%B0%D0%B9%D1%81%D0%B8%D0%BD,_%D0%91%D1%80%D0%B0%D0%B9%D0%BE%D0%BD" title="Гайсин, Брайон">Брайоном Гайсином</a> на основе <a href="/wiki/%D0%9C%D0%B5%D1%82%D0%BE%D0%B4_%D0%BD%D0%B0%D1%80%D0%B5%D0%B7%D0%BE%D0%BA" title="Метод нарезок">«метода нарезок»</a> <a href="/wiki/%D0%9F%D0%BE%D1%8D%D1%82" title="Поэт">поэта</a>-<a href="/wiki/%D0%94%D0%B0%D0%B4%D0%B0%D0%B8%D0%B7%D0%BC" title="Дадаизм">дадаиста</a> <a href="/wiki/%D0%A2%D1%80%D0%B8%D1%81%D1%82%D0%B0%D0%BD_%D0%A2%D1%86%D0%B0%D1%80%D0%B0" title="Тристан Тцара">Тристана Тцара</a> — «Трилогия Нова» считается одной из наиболее известных работ, созданных по данной методике<sup class="reference" id="cite_ref-Linstead_10-0"><a href="#cite_note-Linstead-10">[10]</a></sup>.
</p>

<h2><span id=".D0.98.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.8F_.D1.81.D0.BE.D0.B7.D0.B4.D0.B0.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="История_создания">История создания</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D1%80%D0%B8%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F_%D0%9D%D0%BE%D0%B2%D0%B0&amp;veaction=edit&amp;section=1" title="Редактировать раздел «История создания»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D1%80%D0%B8%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F_%D0%9D%D0%BE%D0%B2%D0%B0&amp;action=edit&amp;section=1" title="Редактировать код раздела «История создания»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>К <a href="/wiki/%D0%91%D0%B5%D1%80%D1%80%D0%BE%D1%83%D0%B7,_%D0%A3%D0%B8%D0%BB%D1%8C%D1%8F%D0%BC_%D0%A1%D1%8C%D1%8E%D0%B0%D1%80%D0%B4" title="Берроуз, Уильям Сьюард">Берроузу</a>, в <a href="/wiki/1958_%D0%B3%D0%BE%D0%B4" title="1958 год">1958 году</a> проживающему и работающему в <span data-interwiki-article="Beat Hotel" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=Beat_Hotel&amp;action=edit&amp;redlink=1" title="Beat Hotel (страница отсутствует)">Beat Hotel</a></span> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;"><a class="extiw" href="https://en.wikipedia.org/wiki/Beat_Hotel" title="en:Beat Hotel"><span title="Beat Hotel — версия статьи «Beat Hotel» на английском языке">Beat Hotel</span></a></span>) в <a href="/wiki/%D0%9F%D0%B0%D1%80%D0%B8%D0%B6" title="Париж">Париже</a>, приезжает его близкий друг — художник Брайон Гайсин, некоторое время позже знакомящий писателя с математиком Иэном Соммервилем (<i>Ian Sommerville</i>) и представивший его как человека, который сможет присматривать за Берроузом в период детоксикации — на тот момент писатель переживал последствия <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BE%D0%B8%D0%BD#Устранение_физиологической_зависимости" title="Героин">лечения апоморфином</a> от <a href="/wiki/%D0%9D%D0%B0%D1%80%D0%BA%D0%BE%D0%BC%D0%B0%D0%BD%D0%B8%D1%8F" title="Наркомания">наркозависимости</a>, которое проходил в <a href="/wiki/%D0%9F%D0%B0%D1%80%D0%B8%D0%B6" title="Париж">Париже</a><sup class="reference" id="cite_ref-Hibbard_11-0"><a href="#cite_note-Hibbard-11">[11]</a></sup>.
</p><p>Примерно в это же время Берроуз знакомится с режиссёром <span data-interwiki-article="Antony Balch" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D1%87,_%D0%AD%D0%BD%D1%82%D0%BE%D0%BD%D0%B8&amp;action=edit&amp;redlink=1" title="Белч, Энтони (страница отсутствует)">Энтони Белчем</a></span> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;"><a class="extiw" href="https://en.wikipedia.org/wiki/Antony_Balch" title="en:Antony Balch"><span title="Antony Balch — версия статьи «Белч, Энтони» на английском языке">Antony Balch</span></a></span>), входящим в круг общения Гайсина, втроём они начинают совместную работу над рядом кинопроектов по мотивам творчества Берроуза<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup><sup class="reference" id="cite_ref-Sargeant_13-0"><a href="#cite_note-Sargeant-13">[13]</a></sup>.
</p><p>Путешествуя по <a href="/wiki/%D0%9F%D0%B0%D1%80%D0%B8%D0%B6" title="Париж">Парижу</a>, <a href="/wiki/%D0%A2%D0%B0%D0%BD%D0%B6%D0%B5%D1%80" title="Танжер">Танжеру</a>, <a href="/wiki/%D0%9B%D0%BE%D0%BD%D0%B4%D0%BE%D0%BD" title="Лондон">Лондону</a> и <a href="/wiki/%D0%9D%D1%8C%D1%8E-%D0%99%D0%BE%D1%80%D0%BA" title="Нью-Йорк">Нью-Йорку</a>, Берроуз начинает усердно работать над частично написанными книгами трилогии, перерабатывая их в соответствии с <i>fold-in</i> методом<sup class="reference" id="cite_ref-Skerl_9-1"><a href="#cite_note-Skerl-9">[9]</a></sup>.
</p><p>В результате тесного взаимодействия с Соммервилем, Гайсиным и Белчем, трилогия впитала в себя многое из того, что писатель почерпнул от общения со своими друзьями: экспериментальные магнитофонные записи Соммервиля и его же <a href="/wiki/%D0%A1%D1%82%D1%80%D0%BE%D0%B1%D0%BE%D1%81%D0%BA%D0%BE%D0%BF" title="Стробоскоп">стробоскопические</a> изображения, опыты с <a href="/wiki/%D0%9A%D0%BE%D0%BB%D0%BB%D0%B0%D0%B6" title="Коллаж">коллажами</a> из видеоряда Белча и, разработанный совместно с Гайсиным, <i>fold-in</i> метод.<sup class="reference" id="cite_ref-Sargeant_13-1"><a href="#cite_note-Sargeant-13">[13]</a></sup><sup class="reference" id="cite_ref-Geiger_14-0"><a href="#cite_note-Geiger-14">[14]</a></sup><sup class="reference" id="cite_ref-Geiger_14-1"><a href="#cite_note-Geiger-14">[14]</a></sup><sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup>.
</p>
<h2><span id=".D0.A1.D1.8E.D0.B6.D0.B5.D1.82"></span><span class="mw-headline" id="Сюжет">Сюжет</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D1%80%D0%B8%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F_%D0%9D%D0%BE%D0%B2%D0%B0&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Сюжет»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D1%80%D0%B8%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F_%D0%9D%D0%BE%D0%B2%D0%B0&amp;action=edit&amp;section=2" title="Редактировать код раздела «Сюжет»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>«Трилогия Нова» — это космическая одиссея о группе террористов, называющих себя «Банда Нова» (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Nova Mob</span>), которые хотят разгорячить планету <a href="/wiki/%D0%97%D0%B5%D0%BC%D0%BB%D1%8F" title="Земля">Земля</a> до состояния взрыва — «<i>Супер Нова</i>» (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Super Nova</span>), путём провокаций и создания на земле неразрешимых человечеством конфликтов<sup class="reference" id="cite_ref-TTI_16-0"><a href="#cite_note-TTI-16">[16]</a></sup>.
</p><p>Следующим Берроуз объясняет их действия:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>Основной принцип «Нова» прост: постоянно разжигать как можно больше неразрешимых конфликтов и постоянно раздувать существующие конфликты… Это делается с помощью сталкивания жизненных видов с несовместимыми способами существования на одной планете…&lt;…&gt; Их образы жизни в основе своей несовместимы в форме настоящего времени, а задача Банды Нова как раз и состоит в том, чтобы удерживать их в форме настоящего времени с целью разжигания и раздувания конфликтов, которые ведут к взрыву планеты, то есть к Нова…<sup class="reference" id="cite_ref-Nova_17-0"><a href="#cite_note-Nova-17">[17]</a></sup>
</p>
</blockquote>
<p>Несмотря на тот факт, что в некоторых случаях трилогия называется «бессюжетной» или имеющий сюжет номинально (<i>как сказано в краткой аннотации от российского издателя</i>), в повествовании выделяют доминирующую тему, которой является язык в качестве вируса — представляющий собой комплекс механизмов контроля за человечеством, поработивший людей.
</p><p>Данный вирус является результатом деятельности Банды Нова, осуществлению плана которых мешает <i>Полиция</i> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Nova Police</span>) во главе с Инспектором Ли, центральным персонажем истории<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup>.
</p>
<h2><span id=".D0.9F.D0.B5.D1.80.D1.81.D0.BE.D0.BD.D0.B0.D0.B6.D0.B8"></span><span class="mw-headline" id="Персонажи">Персонажи</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D1%80%D0%B8%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F_%D0%9D%D0%BE%D0%B2%D0%B0&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Персонажи»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D1%80%D0%B8%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F_%D0%9D%D0%BE%D0%B2%D0%B0&amp;action=edit&amp;section=3" title="Редактировать код раздела «Персонажи»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Единственными переходящими персонажами трилогии являются представители противоборствующих сторон — их последовательное перечисление с рядом разъяснений о характере деятельности «Банды Нова» появляется только в романе «<a href="/wiki/%D0%91%D0%B8%D0%BB%D0%B5%D1%82,_%D0%BA%D0%BE%D1%82%D0%BE%D1%80%D1%8B%D0%B9_%D0%BB%D0%BE%D0%BF%D0%BD%D1%83%D0%BB" title="Билет, который лопнул">Билет, который лопнул</a>»<sup class="reference" id="cite_ref-Ticket_19-0"><a href="#cite_note-Ticket-19">[19]</a></sup>.
</p><p>Со стороны «Полиции Нова» выступает главный герой трилогии, «Инспектор Ли» (полное имя <i>Уильям Ли</i> или <i>Билл Ли</i>). Примечательно, что данный персонаж так же является центральным для романа «<a href="/wiki/%D0%93%D0%BE%D0%BB%D1%8B%D0%B9_%D0%B7%D0%B0%D0%B2%D1%82%D1%80%D0%B0%D0%BA" title="Голый завтрак">Голый завтрак</a>» и одновременно предстаёт в качестве <a href="/wiki/Alter_ego" title="Alter ego">альтер эго</a> писателя<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup><sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup>. Помимо этого имя «Уильям (Билл) Ли» является псевдонимом, под которым Берроузу удалось опубликовать роман «<a href="/wiki/%D0%94%D0%B6%D0%B0%D0%BD%D0%BA%D0%B8" title="Джанки">Джанки</a>» (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Junkie</span>, <a href="/wiki/1953_%D0%B3%D0%BE%D0%B4" title="1953 год">1953</a>, рус. перевод <a href="/wiki/2003_%D0%B3%D0%BE%D0%B4" title="2003 год">2003</a>)<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup><sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup>.
</p><p>Со стороны «Банды Нова» выступают двенадцать человек, в разной степени задействованные в сюжете: Мистер и миссис С (он же «Мистер Брэдли мистер Мартин» и «Злой Дух» — главарь Банды), Сэмми Мясник, Зелёный Тони, Дешёвка, Джеки Минор, Джон Известняк, Иззи Громила, Мэри Гамбургер, Пэдди Жало, Подсознательный Малыш, Синий Динозавр, Уилли Крыса. В ходе повествования подробностей о них не раскрывается, в равной как и их настоящие имена.
</p>
<h2><span id=".D0.A5.D1.83.D0.B4.D0.BE.D0.B6.D0.B5.D1.81.D1.82.D0.B2.D0.B5.D0.BD.D0.BD.D1.8B.D0.B5_.D0.BE.D1.81.D0.BE.D0.B1.D0.B5.D0.BD.D0.BD.D0.BE.D1.81.D1.82.D0.B8"></span><span class="mw-headline" id="Художественные_особенности">Художественные особенности</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D1%80%D0%B8%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F_%D0%9D%D0%BE%D0%B2%D0%B0&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Художественные особенности»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D1%80%D0%B8%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F_%D0%9D%D0%BE%D0%B2%D0%B0&amp;action=edit&amp;section=4" title="Редактировать код раздела «Художественные особенности»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>«Трилогия Нова» Берроуза считается одной из наиболее известных работ, написанных в <a class="mw-redirect" href="/wiki/Cut-up" title="Cut-up">«технике cut-up»</a><sup class="reference" id="cite_ref-Linstead_10-1"><a href="#cite_note-Linstead-10">[10]</a></sup>. Описывая стиль Берроуза, его близкий друг, художник и писатель Брайон Гайсин, говорил: «Уильям экспериментировал со своим чрезвычайно изменчивым материалом, своими собственными неповторимыми текстами, которых он подверг жестоким и беспощадным „разрезкам“, он всегда был самым упорным. Ничто никогда не обескураживало его…»<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup>. В одном из интервью, посвящённых трилогии, сам писатель сказал:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
Я пытаюсь создать новую мифологию космической эпохи<sup class="reference" id="cite_ref-Hibbard_11-1"><a href="#cite_note-Hibbard-11">[11]</a></sup>.
</p>
<div class="mw-collapsible mw-collapsed has-collapsible-header ts-Скрытый_блок ts-Скрытый_блок-gray ts-Скрытый_блок-noneHideLink ts-oq has-collapsible-header" style=""><div class="ts-Скрытый_блок-title collapsible-header ts-Скрытый_блок-title-leftTitle ts-oq-header collapsible-header" style="text-align:left;">Оригинальный текст <small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></div><div class="mw-collapsible-content ts-oq-content" style="">
<div dir="auto" lang="en">I am attempting to create a new mythology for the space age.</div>
</div></div>
</blockquote>
<p>С «<a class="mw-redirect" href="/wiki/Cut-up" title="Cut-up">методом нарезок</a>» писателя познакомил Брайон Гайсин — с которым Берроуз впервые встретился в <a href="/wiki/%D0%A2%D0%B0%D0%BD%D0%B6%D0%B5%D1%80" title="Танжер">Танжере</a>. Сам Гайсин отмечал, что своим появлением данная писательская техника обязана <a href="/wiki/%D0%A0%D1%83%D0%BC%D1%8B%D0%BD%D0%B8%D1%8F" title="Румыния">румынскому</a> <a href="/wiki/%D0%9F%D0%BE%D1%8D%D1%82" title="Поэт">поэту</a>-<a href="/wiki/%D0%94%D0%B0%D0%B4%D0%B0%D0%B8%D0%B7%D0%BC" title="Дадаизм">дадаисту</a> <a href="/wiki/%D0%A2%D1%80%D0%B8%D1%81%D1%82%D0%B0%D0%BD_%D0%A2%D1%86%D0%B0%D1%80%D0%B0" title="Тристан Тцара">Тристрану Тцара</a>, который составлял <a href="/wiki/%D0%A1%D1%82%D0%B8%D1%85" title="Стих">стихи</a>, доставая предварительно разрезанный на мелкие кусочки текст из шляпы<sup class="reference" id="cite_ref-Wising_4-1"><a href="#cite_note-Wising-4">[4]</a></sup>. Переработка оригинального метода Гайсиным заключалась в том, что <i>несколько</i><sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup> готовых связных текстов разрезались, перемешивалась и компоновались в произвольном порядке, составляя новое произведение — техника базируется на принципе <a href="/wiki/%D0%9A%D0%BE%D0%BB%D0%BB%D0%B0%D0%B6" title="Коллаж">коллажа</a> и случайных действий<sup class="reference" id="cite_ref-Lawlor_26-0"><a href="#cite_note-Lawlor-26">[26]</a></sup>. Для <a href="/wiki/%D0%91%D0%B5%D1%80%D1%80%D0%BE%D1%83%D0%B7,_%D0%A3%D0%B8%D0%BB%D1%8C%D1%8F%D0%BC_%D0%A1%D1%8C%D1%8E%D0%B0%D1%80%D0%B4" title="Берроуз, Уильям Сьюард">Берроуза</a> же основополагающими были три полновесные книги, сотни мелких текстов, дюжины газетных коллажей, фотомонтажи, эксперименты со звукозаписью и несколько фильмов (снятых, разумеется, по <a class="mw-redirect" href="/wiki/Cut-up" title="Cut-up">методу нарезок</a>), созданных совместно с <span data-interwiki-article="Antony Balch" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D1%87,_%D0%AD%D0%BD%D1%82%D0%BE%D0%BD%D0%B8&amp;action=edit&amp;redlink=1" title="Белч, Энтони (страница отсутствует)">Энтони Белчом</a></span> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;"><a class="extiw" href="https://en.wikipedia.org/wiki/Antony_Balch" title="en:Antony Balch"><span title="Antony Balch — версия статьи «Белч, Энтони» на английском языке">Antony Balch</span></a></span>)<sup class="reference" id="cite_ref-Lawlor_26-1"><a href="#cite_note-Lawlor-26">[26]</a></sup>.<br/><br/>
Свою интерпретацию <a class="mw-redirect" href="/wiki/Cut-up" title="Cut-up">метода нарезок</a> Берроуз описал в интервью, взятым Конрадом Кникербокером (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Conrad Knickerbocker</span>)<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup>.
</p>
<div class="ts-Цитата"><blockquote><p>...совершая  путешествие, я всё записываю  в  трёх колонках  блокнота, который всегда  при  мне.  Первая  колонка  содержит  элементарный  отчёт  о путешествии, о том, что происходит: как я приехал в  аэропорт,  что говорили там  служащие,  что  я  случайно  услышал  в  самолёте,  в  какой  гостинице остановился. В  следующей  колонке отражены  мои  впечатления: о чём я  в то время думал, какие  воспоминания навеяны неожиданными встречами. И в третьей колонке, которую я называю "читательской", содержатся цитаты из любой книги, которую я беру с  собой.  Так из нескольких  поездок в  Гибралтар у меня уже получился целый роман...  К примеру, я читаю "Чудесную страну" Тома Ли,  где герой едет в Мексику и как раз пересекает границу. И именно в  этот момент я подъезжаю к испанской границе, что и отмечаю на полях.  Или, допустим, я еду в поезде или на пароходе и  читаю "Тихого американца".  Я озираюсь,  надеясь увидеть тихого  американца среди попутчиков, и  какой-нибудь тихоня из  юных американцев непременно находится -  с причёской "ёжиком" и бутылкой пива. Всё это просто  удивительно,  надо  лишь смотреть  в  оба.  Как-то раз  я  читал Раймонда Чандлера, и у  него там был  один персонаж- террорист-альбинос.  И, Боже мой, мне тут же попался на глаза альбинос. Правда, не террорист.</p></blockquote></div>
<p>Следующим образом Берроуз объяснял суть метода:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
Человек читает газету, и его взгляд скользит по колонке в разумной аристотелевской манере — мысль за мыслью, фраза за фразой. Но подсознательно он читает и колонки, расположенные по бокам, а также осознаёт присутствие сидящего рядом попутчика. Вот вам и нарезка<sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup>.
</p>
</blockquote>
<p>Подавляющим большинством обозревателей и критиков трилогия причисляется к жанру «<a href="/wiki/%D0%A0%D0%BE%D0%BC%D0%B0%D0%BD" title="Роман">романа</a>», однако некоторые авторы отмечают отсутствие традиционных элементов, характерных для романного повествования, говоря о романе «<a href="/wiki/%D0%9D%D0%BE%D0%B2%D0%B0_%D0%AD%D0%BA%D1%81%D0%BF%D1%80%D0%B5%D1%81%D1%81" title="Нова Экспресс">Нова Экспресс</a>» (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Nova Express</span>, <a href="/wiki/1964_%D0%B3%D0%BE%D0%B4" title="1964 год">1964</a>, рус. перевод <a href="/wiki/1999_%D0%B3%D0%BE%D0%B4" title="1999 год">1999</a>) — что в полной мере применимо ко всей трилогии, представляющей собой смешение стилей, места, времени и субъектов повествования<sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[29]</a></sup><sup class="reference" id="cite_ref-30"><a href="#cite_note-30">[30]</a></sup>. Всего критики Берроуза в «Трилогии Нова» выделяют соединение трёх жанров: <a href="/wiki/%D0%94%D0%B5%D1%82%D0%B5%D0%BA%D1%82%D0%B8%D0%B2" title="Детектив">детектива</a>, <a href="/wiki/%D0%9F%D1%80%D0%B8%D0%BA%D0%BB%D1%8E%D1%87%D0%B5%D0%BD%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D0%BB%D0%B8%D1%82%D0%B5%D1%80%D0%B0%D1%82%D1%83%D1%80%D0%B0" title="Приключенческая литература">приключений</a> и <a href="/wiki/%D0%9D%D0%B0%D1%83%D1%87%D0%BD%D0%B0%D1%8F_%D1%84%D0%B0%D0%BD%D1%82%D0%B0%D1%81%D1%82%D0%B8%D0%BA%D0%B0" title="Научная фантастика">научно-фантастического</a> <a href="/wiki/%D0%A0%D0%B0%D1%81%D1%81%D0%BA%D0%B0%D0%B7" title="Рассказ">рассказа</a><sup class="reference" id="cite_ref-Lydenberg_31-0"><a href="#cite_note-Lydenberg-31">[31]</a></sup>. В некоторых случаях трилогию называют «<a href="/wiki/%D0%9F%D0%BE%D0%BB%D0%B8%D1%82%D0%B8%D0%BA%D0%B0" title="Политика">политической</a> <a href="/wiki/%D0%A1%D0%B0%D1%82%D0%B8%D1%80%D0%B0" title="Сатира">сатирой</a>» на <a href="/wiki/%D0%A5%D0%BE%D0%BB%D0%BE%D0%B4%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Холодная война">Холодную войну</a> и сатирой на послевоенное общество<sup class="reference" id="cite_ref-32"><a href="#cite_note-32">[32]</a></sup><sup class="reference" id="cite_ref-33"><a href="#cite_note-33">[33]</a></sup>.
</p>
<div class="ts-Цитата"><blockquote><p>В Cut-up трилогии проза Берроуза находится в состоянии постоянного изменения. Его язык живой, волнующий и интуитивный. Творчество писателя и его литературные эксперименты странны, исповедальны и <a href="/wiki/%D0%AD%D0%B7%D0%BE%D1%82%D0%B5%D1%80%D0%B8%D0%B7%D0%BC" title="Эзотеризм">эзотеричны</a>. Его <a href="/wiki/%D0%9C%D0%B8%D1%84" title="Миф">мифическая</a> <a href="/wiki/%D0%9A%D0%BE%D1%81%D0%BC%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F" title="Космология">космология</a> — один из наиболее остро значимых артефактов <a href="/wiki/XX_%D0%B2%D0%B5%D0%BA" title="XX век">XX века</a>.<cite>Выдержка из рецензии <span data-interwiki-article="Mac Tonnies" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%A2%D0%BE%D0%BD%D0%BD%D0%B8%D1%81,_%D0%9C%D0%B0%D0%BA&amp;action=edit&amp;redlink=1" title="Тоннис, Мак (страница отсутствует)">Мака Тонниса</a></span> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;"><a class="extiw" href="https://en.wikipedia.org/wiki/Mac_Tonnies" title="en:Mac Tonnies"><span title="Mac Tonnies — версия статьи «Тоннис, Мак» на английском языке">Mac Tonnies</span></a></span>) на трилогию<sup class="reference" id="cite_ref-34"><a href="#cite_note-34">[34]</a></sup>.</cite></p></blockquote></div>
<h2><span id=".D0.90.D0.BB.D0.BB.D1.8E.D0.B7.D0.B8.D0.B8"></span><span class="mw-headline" id="Аллюзии">Аллюзии</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D1%80%D0%B8%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F_%D0%9D%D0%BE%D0%B2%D0%B0&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Аллюзии»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D1%80%D0%B8%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F_%D0%9D%D0%BE%D0%B2%D0%B0&amp;action=edit&amp;section=5" title="Редактировать код раздела «Аллюзии»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>В трилогии Берроуза содержится множество <a href="/wiki/%D0%90%D0%BB%D0%BB%D1%8E%D0%B7%D0%B8%D1%8F" title="Аллюзия">аллюзий</a>, отсылающих к другим писателям и произведениям.
</p><p>В ходе повествования писатель неоднократно ссылается на работы <a href="/wiki/%D0%A4%D1%80%D0%B5%D0%B9%D0%B4,_%D0%97%D0%B8%D0%B3%D0%BC%D1%83%D0%BD%D0%B4" title="Фрейд, Зигмунд">Зигмунда Фрейда</a>, часто упоминаются учёные <a href="/wiki/%D0%A0%D0%B0%D0%B9%D1%85,_%D0%92%D0%B8%D0%BB%D1%8C%D0%B3%D0%B5%D0%BB%D1%8C%D0%BC" title="Райх, Вильгельм">Вильгельм Райх</a>, <a href="/wiki/%D0%9F%D0%B0%D0%B2%D0%BB%D0%BE%D0%B2,_%D0%98%D0%B2%D0%B0%D0%BD_%D0%9F%D0%B5%D1%82%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Павлов, Иван Петрович">И. П. Павлов</a> и их научное наследие. Так же в каждой из трёх книг цикла неоднократно упоминается <a href="/wiki/%D0%93%D0%B0%D0%B9%D1%81%D0%B8%D0%BD,_%D0%91%D1%80%D0%B0%D0%B9%D0%BE%D0%BD" title="Гайсин, Брайон">Брайон Гайсин</a>.
</p><p>Некоторые главы романа «Нова Экспресс» содержанием и цитатами отсылают к стихам <a href="/wiki/%D0%AD%D0%BB%D0%B8%D0%BE%D1%82,_%D0%A2%D0%BE%D0%BC%D0%B0%D1%81_%D0%A1%D1%82%D0%B5%D1%80%D0%BD%D0%B7" title="Элиот, Томас Стернз">Т. С. Элиота</a> и «<a href="/wiki/%D0%91%D1%83%D1%80%D1%8F_(%D0%BF%D1%8C%D0%B5%D1%81%D0%B0)" title="Буря (пьеса)">Буре</a>» <a href="/wiki/%D0%A8%D0%B5%D0%BA%D1%81%D0%BF%D0%B8%D1%80,_%D0%A3%D0%B8%D0%BB%D1%8C%D1%8F%D0%BC" title="Шекспир, Уильям">Уильяма Шекспира</a>. Глава «Краткое изложение дела для первого слушания» из книги «Нова Экспресс» содержит обширную цитату из романа <a href="/wiki/%D0%9A%D0%B0%D1%84%D0%BA%D0%B0,_%D0%A4%D1%80%D0%B0%D0%BD%D1%86" title="Кафка, Франц">Франца Кафки</a> «<a href="/wiki/%D0%9F%D1%80%D0%BE%D1%86%D0%B5%D1%81%D1%81_(%D1%80%D0%BE%D0%BC%D0%B0%D0%BD)" title="Процесс (роман)">Процесс</a>». Описывая свои эксперименты с <a class="mw-redirect" href="/wiki/Cut-up" title="Cut-up">методом нарезок</a> в романе «Билет, который лопнул» автор вспоминает писательницу <a href="/wiki/%D0%A1%D1%82%D0%B0%D0%B9%D0%BD,_%D0%93%D0%B5%D1%80%D1%82%D1%80%D1%83%D0%B4%D0%B0" title="Стайн, Гертруда">Гертруду Стайн</a><sup class="reference" id="cite_ref-Wald_1-1"><a href="#cite_note-Wald-1">[1]</a></sup>. Несколько глав романа написаны в соавторстве с Майклом Портманом (<i>Michael Portman</i>). В работе над киноэкспериментами, описанными в «Билете», принимал участие <span data-interwiki-article="Antony Balch" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D1%87,_%D0%AD%D0%BD%D1%82%D0%BE%D0%BD%D0%B8&amp;action=edit&amp;redlink=1" title="Белч, Энтони (страница отсутствует)">Энтони Белч</a></span> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;"><a class="extiw" href="https://en.wikipedia.org/wiki/Antony_Balch" title="en:Antony Balch"><span title="Antony Balch — версия статьи «Белч, Энтони» на английском языке">Antony Balch</span></a></span>). Одна из глав «Нова Экспресса» написана в соавторстве с математиком Иэном Соммервилем (<i>Ian Sommerville</i>)<sup class="reference" id="cite_ref-Nova_17-1"><a href="#cite_note-Nova-17">[17]</a></sup>. Так же он познакомил <a href="/wiki/%D0%91%D0%B5%D1%80%D1%80%D0%BE%D1%83%D0%B7,_%D0%A3%D0%B8%D0%BB%D1%8C%D1%8F%D0%BC_%D0%A1%D1%8C%D1%8E%D0%B0%D1%80%D0%B4" title="Берроуз, Уильям Сьюард">Берроуза</a> с различными разновидностями экспериментов со звукозаписью, описанных в книге «Билет, который лопнул»<sup class="reference" id="cite_ref-Ticket_19-1"><a href="#cite_note-Ticket-19">[19]</a></sup>.
</p><p>Помимо этого в «Билете» несколько раз упоминается теория социальной революции, описанная Берроузом в <a href="/wiki/%D0%AD%D1%81%D1%81%D0%B5" title="Эссе">эссе</a> <span data-interwiki-article="The Electronic Revolution" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BD%D0%BD%D0%B0%D1%8F_%D1%80%D0%B5%D0%B2%D0%BE%D0%BB%D1%8E%D1%86%D0%B8%D1%8F&amp;action=edit&amp;redlink=1" title="Электронная революция (страница отсутствует)">Электронная революция</a></span> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;"><a class="extiw" href="https://en.wikipedia.org/wiki/The_Electronic_Revolution" title="en:The Electronic Revolution"><span title="The Electronic Revolution — версия статьи «Электронная революция» на английском языке">The Electronic Revolution</span></a></span>) (<a href="/wiki/1971_%D0%B3%D0%BE%D0%B4" title="1971 год">1971</a>)
</p>
<h2><span id=".D0.92.D0.BB.D0.B8.D1.8F.D0.BD.D0.B8.D0.B5_.D0.BD.D0.B0_.D0.BF.D0.BE.D0.BF.D1.83.D0.BB.D1.8F.D1.80.D0.BD.D1.83.D1.8E_.D0.BA.D1.83.D0.BB.D1.8C.D1.82.D1.83.D1.80.D1.83"></span><span class="mw-headline" id="Влияние_на_популярную_культуру">Влияние на популярную культуру</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D1%80%D0%B8%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F_%D0%9D%D0%BE%D0%B2%D0%B0&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Влияние на популярную культуру»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D1%80%D0%B8%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F_%D0%9D%D0%BE%D0%B2%D0%B0&amp;action=edit&amp;section=6" title="Редактировать код раздела «Влияние на популярную культуру»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>В определённой степени «Трилогия Нова» повлияла не только на становление <a class="mw-redirect" href="/wiki/Cut-up" title="Cut-up">метода нарезок</a> в литературе, но и в значительной мере способствовала развитию жанра <a href="/wiki/%D0%9A%D0%B8%D0%B1%D0%B5%D1%80%D0%BF%D0%B0%D0%BD%D0%BA" title="Киберпанк">киберпанк</a><sup class="reference" id="cite_ref-TTI_16-1"><a href="#cite_note-TTI-16">[16]</a></sup><sup class="reference" id="cite_ref-35"><a href="#cite_note-35">[35]</a></sup><sup class="reference" id="cite_ref-36"><a href="#cite_note-36">[36]</a></sup>. Самому писателю некоторые критики даже отводят роль «крёстного отца» жанра<sup class="reference" id="cite_ref-37"><a href="#cite_note-37">[37]</a></sup>.

В дополнение к этому ряд музыкальных коллективов заимствовали свои названия из работы Берроуза, к примеру — <a href="/wiki/Soft_Machine" title="Soft Machine">Soft Machine</a> (названа в честь одноимённого романа<sup class="reference" id="cite_ref-41"><a href="#cite_note-41">[41]</a></sup>), <a href="/wiki/%D0%9D%D0%B8%D0%B4%D0%B5%D1%80%D0%BB%D0%B0%D0%BD%D0%B4%D1%8B" title="Нидерланды">нидерландский</a> акустический дуэт <span data-interwiki-article="The Ticket That Exploded (band)" (страница отсутствует)">The Ticket That Exploded</a></span> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;"><a class="extiw" href="https://en.wikipedia.org/wiki/The_Ticket_That_Exploded_(band)" title="en:The Ticket That Exploded (band)"><span «That Subliminal Kid» — в честь одного из членов Банды, «Подсознательного Малыша»<sup class="reference" id="cite_ref-42"><a href="#cite_note-42">[42]</a></sup>. Цитаты из трилогии так же использовались в песнях — к примеру в композиции «<a href="/wiki/Lust_for_Life_(%D0%BF%D0%B5%D1%81%D0%BD%D1%8F_%D0%98%D0%B3%D0%B3%D0%B8_%D0%9F%D0%BE%D0%BF%D0%B0)" title="Lust for Life (песня Игги Попа)">Lust for Life</a>», американский <a href="/wiki/%D0%A0%D0%BE%D0%BA-%D0%BC%D1%83%D0%B7%D1%8B%D0%BA%D0%B0" title="Рок-музыка">рок-исполнитель</a> <a href="/wiki/%D0%98%D0%B3%D0%B3%D0%B8_%D0%9F%D0%BE%D0%BF" title="Игги Поп">Игги Поп</a> использовал строчку из романа «Билет, который лопнул» — «<i>That’s like hypnotizing chickens</i>»<sup class="reference" id="cite_ref-43"><a href="#cite_note-43">[43]</a></sup>.
</p>
<h2><span id=".D0.9E.D1.82.D0.B7.D1.8B.D0.B2.D1.8B_.D0.B8_.D0.BA.D1.80.D0.B8.D1.82.D0.B8.D0.BA.D0.B0"></span><span class="mw-headline" id="Отзывы_и_критика">Отзывы и критика</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D1%80%D0%B8%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F_%D0%9D%D0%BE%D0%B2%D0%B0&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Отзывы и критика»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D1%80%D0%B8%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F_%D0%9D%D0%BE%D0%B2%D0%B0&amp;action=edit&amp;section=7" title="Редактировать код раздела «Отзывы и критика»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/The_Guardian" title="The Guardian">Guardian</a> назвала <a href="/wiki/%D0%91%D0%B5%D1%80%D1%80%D0%BE%D1%83%D0%B7,_%D0%A3%D0%B8%D0%BB%D1%8C%D1%8F%D0%BC_%D0%A1%D1%8C%D1%8E%D0%B0%D1%80%D0%B4" title="Берроуз, Уильям Сьюард">Берроуза</a> «творящим отвратительную, захватывающую <a href="/wiki/%D0%9F%D0%BE%D1%8D%D0%B7%D0%B8%D1%8F" title="Поэзия">поэзию</a> из грязи, экскрементов и спаривания.»<sup class="reference" id="cite_ref-44"><a href="#cite_note-44">[44]</a></sup>. Журналы <a href="/wiki/The_New_York_Times" title="The New York Times">The New York Times</a> и <a href="/wiki/San_Francisco_Chronicle" title="San Francisco Chronicle">San Francisco Chronicle</a> так же не обошли стороной выход <a href="/wiki/%D0%9C%D1%8F%D0%B3%D0%BA%D0%B0%D1%8F_%D0%BC%D0%B0%D1%88%D0%B8%D0%BD%D0%B0" title="Мягкая машина">первого романа</a> трилогии — назвав его «Сюрреалистичной одиссеей по миру тёмных фантазий», а самого писателя — бесстрашным исследователем болезни нашего мира<sup class="reference" id="cite_ref-Soft_45-0"><a href="#cite_note-Soft-45">[45]</a></sup>.
</p><p>Однако иные авторы утверждают, что трилогия не является отображением современного общества ни в <a href="/wiki/%D0%9F%D0%BE%D0%BB%D0%B8%D1%82%D0%B8%D0%BA%D0%B0" title="Политика">политическом</a>, ни в <a href="/wiki/%D0%A4%D0%B8%D0%BB%D0%BE%D1%81%D0%BE%D1%84%D0%B8%D1%8F" title="Философия">философском</a> плане — в цикле романов Берроуз почти не обращается к политике, вместо этого он предлагает рассказы на манер кинофильмов, которые читатель вынужден самостоятельно связывать с реальностью<sup class="reference" id="cite_ref-Snedeker_46-0"><a href="#cite_note-Snedeker-46">[46]</a></sup>.
</p><p>Журнал <a href="/wiki/Playboy" title="Playboy">Playboy</a> назвал «<a href="/wiki/%D0%91%D0%B8%D0%BB%D0%B5%D1%82,_%D0%BA%D0%BE%D1%82%D0%BE%D1%80%D1%8B%D0%B9_%D0%BB%D0%BE%D0%BF%D0%BD%D1%83%D0%BB" title="Билет, который лопнул">Билет, который лопнул</a>» романом, в котором
</p>
<div class="ts-Цитата"><blockquote><p>Многослойная, многоуровневая техника и едкая социальная сатира комбинируется с колоссальной и жестокой творческой энергетикой Берроуза — <a href="/wiki/%D0%A1%D0%B2%D0%B8%D1%84%D1%82,_%D0%94%D0%B6%D0%BE%D0%BD%D0%B0%D1%82%D0%B0%D0%BD" title="Свифт, Джонатан">Джонатана Свифта</a> эпохи <a href="/wiki/%D0%A0%D0%BE%D0%BA-%D0%BD-%D1%80%D0%BE%D0%BB%D0%BB" title="Рок-н-ролл">рок-н-ролла</a></p></blockquote></div><p> — к данному мнению присоединилось британское издание <a class="mw-redirect" href="/wiki/The_Spectator_(1828)" title="The Spectator (1828)">The Spectator</a>, назвавшее «свифтианскими» взгляды писателя на <a href="/wiki/%D0%9F%D0%BE%D0%BB%D1%83%D1%84%D0%B0%D0%B1%D1%80%D0%B8%D0%BA%D0%B0%D1%82" title="Полуфабрикат">полуфабрикатную</a> жизнь<sup class="reference" id="cite_ref-Ticket_19-2"><a href="#cite_note-Ticket-19">[19]</a></sup><sup class="reference" id="cite_ref-47"><a href="#cite_note-47">[47]</a></sup>.
</p><p><a href="/wiki/%D0%9D%D0%BE%D0%B2%D0%B0_%D0%AD%D0%BA%D1%81%D0%BF%D1%80%D0%B5%D1%81%D1%81" title="Нова Экспресс">Нова Экспресс</a> журналом <a href="/wiki/Playboy" title="Playboy">Playboy</a> был назван «дьявольским по силе произведением»<sup class="reference" id="cite_ref-Nova_17-2"><a href="#cite_note-Nova-17">[17]</a></sup>. Критики из <a href="/wiki/San_Francisco_Chronicle" title="San Francisco Chronicle">San Francisco Chronicle</a> пошли дальше, высказав восхищение кошмарам писателя и отметив его воображение<sup class="reference" id="cite_ref-Nova_17-3"><a href="#cite_note-Nova-17">[17]</a></sup>.
</p><p>Критике так же подвергся <a href="/wiki/%D0%A0%D0%B0%D0%B4%D0%B8%D0%BA%D0%B0%D0%BB%D0%B8%D0%B7%D0%BC" title="Радикализм">радикализм</a> писателя — некоторые исследователи отмечают, что он предстаёт радикализмом вымышленным — по их мнению Берроуз пытался оживить возможность представлять альтернативные миры, не будучи втянутым в нашу крайне <a href="/wiki/%D0%9A%D0%BE%D1%80%D1%80%D1%83%D0%BF%D1%86%D0%B8%D1%8F" title="Коррупция">коррумпированную</a> вселенную, однако под сомнение ставится само понимание Берроуза борьбы за свободу в будничном и глобальном смыслах<sup class="reference" id="cite_ref-Snedeker_46-1"><a href="#cite_note-Snedeker-46">[46]</a></sup>.
</p>
<h2><span id=".D0.A1.D0.BE.D1.81.D1.82.D0.B0.D0.B2.D0.BB.D1.8F.D1.8E.D1.89.D0.B8.D0.B5_.D1.82.D1.80.D0.B8.D0.BB.D0.BE.D0.B3.D0.B8.D0.B8_.D0.B8_.D0.B8.D0.B7.D0.B4.D0.B0.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="Составляющие_трилогии_и_издания">Составляющие трилогии и издания</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D1%80%D0%B8%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F_%D0%9D%D0%BE%D0%B2%D0%B0&amp;veaction=edit&amp;section=8" title="Редактировать раздел «Составляющие трилогии и издания»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D1%80%D0%B8%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F_%D0%9D%D0%BE%D0%B2%D0%B0&amp;action=edit&amp;section=8" title="Редактировать код раздела «Составляющие трилогии и издания»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Традиционно используемое критиками и литературоведами обозначение раннего<sup class="reference" id="cite_ref-48"><a href="#cite_note-48">[48]</a></sup> цикла романов Берроуза — <a href="/wiki/%D0%A2%D1%80%D0%B8%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F" title="Трилогия">трилогия</a><sup class="reference" id="cite_ref-Wising_4-2"><a href="#cite_note-Wising-4">[4]</a></sup><sup class="reference" id="cite_ref-49"><a href="#cite_note-49">[49]</a></sup><sup class="reference" id="cite_ref-50"><a href="#cite_note-50">[50]</a></sup>. Её составили:
</p>

<p>Впрочем, некоторыми авторами утверждается, что трилогия может быть прочитана в произвольном порядке<sup class="reference" id="cite_ref-51"><a href="#cite_note-51">[51]</a></sup>.
</p><p>На русском языке в полном размере трилогия была выпущена в <a href="/wiki/1999_%D0%B3%D0%BE%D0%B4" title="1999 год">1999 году</a> в рамках совместно проекта издательств <a class="mw-redirect" href="/wiki/%D0%90%D0%B7%D0%B1%D1%83%D0%BA%D0%B0_(%D0%B8%D0%B7%D0%B4%D0%B0%D1%82%D0%B5%D0%BB%D1%8C%D1%81%D1%82%D0%B2%D0%BE)" title="Азбука (издательство)">«Азбука»</a> и <a href="/wiki/%D0%90%D0%A1%D0%A2_(%D0%B8%D0%B7%D0%B4%D0%B0%D1%82%D0%B5%D0%BB%D1%8C%D1%81%D1%82%D0%B2%D0%BE)" title="АСТ (издательство)">«АСТ»</a>. Примечательно, что годом ранее издательством «<a class="new" href="/w/index.php?title=%D0%9D%D0%B8%D0%BA%D0%B0-%D0%A6%D0%B5%D0%BD%D1%82%D1%80_(%D0%B8%D0%B7%D0%B4%D0%B0%D1%82%D0%B5%D0%BB%D1%8C%D1%81%D1%82%D0%B2%D0%BE)&amp;action=edit&amp;redlink=1" title="Ника-Центр (издательство) (страница отсутствует)">Ника-Центр</a>» была выпущена неполная версия цикла — в аннотации от издателя названного <a href="/wiki/%D0%A2%D0%B5%D1%82%D1%80%D0%B0%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F" title="Тетралогия">тетралогией</a>, а не трилогией, что соответствует мнению ряда исследователей творчества Берроуза<sup class="reference" id="cite_ref-DeKoven_5-1"><a href="#cite_note-DeKoven-5">[5]</a></sup><sup class="reference" id="cite_ref-Stringer_6-2"><a href="#cite_note-Stringer-6">[6]</a></sup><sup class="reference" id="cite_ref-Zott_7-1"><a href="#cite_note-Zott-7">[7]</a></sup>.
</p><p>Роман «<a href="/wiki/%D0%93%D0%BE%D0%BB%D1%8B%D0%B9_%D0%B7%D0%B0%D0%B2%D1%82%D1%80%D0%B0%D0%BA" title="Голый завтрак">Голый завтрак</a>»<sup class="reference" id="cite_ref-52"><a href="#cite_note-52">[52]</a></sup> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">The Naked Lunch</span>, <a href="/wiki/1959_%D0%B3%D0%BE%D0%B4" title="1959 год">1959</a>, рус. перевод <a href="/wiki/1971_%D0%B3%D0%BE%D0%B4" title="1971 год">1971</a>), согласно данному мнению, является частью (описывающей события, предшествующие основному циклу произведений) тетралогии — что, впрочем, противоречит мнению самого <a href="/wiki/%D0%91%D0%B5%D1%80%D1%80%D0%BE%D1%83%D0%B7,_%D0%A3%D0%B8%D0%BB%D1%8C%D1%8F%D0%BC_%D0%A1%D1%8C%D1%8E%D0%B0%D1%80%D0%B4" title="Берроуз, Уильям Сьюард">Берроуза</a>, обозначившего <i>трилогию</i> <a href="/wiki/%D0%A1%D0%B8%D0%BA%D0%B2%D0%B5%D0%BB" title="Сиквел">сиквелом</a> и «математическим расширением» для тем и техник, созданных в рамках «Голого завтрака»<sup class="reference" id="cite_ref-Wising_4-3"><a href="#cite_note-Wising-4">[4]</a></sup>.
</p>
