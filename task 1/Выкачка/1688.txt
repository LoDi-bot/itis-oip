<p><b>SuperCalc</b> — <a href="/wiki/%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BD%D0%BD%D0%B0%D1%8F_%D1%82%D0%B0%D0%B1%D0%BB%D0%B8%D1%86%D0%B0" title="Электронная таблица">электронная таблица</a>, выпущенная <a class="extiw" href="https://en.wikipedia.org/wiki/Sorcim" title="en:Sorcim"><span title="Sorcim — версия статьи «Sorcim» на английском языке">Sorcim</span></a><span class="noprint" style="font-style:normal; font-weight:normal;"> <span style="display:inline-block; font-size:80%; position:relative;"><span style="font-size:117.6%; margin-left:-0.43em; position:relative; top:0.28em;"> <small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></span><span class="link-ru metadata" style="margin-left:-0.43em; position:absolute; left:0; z-index:1; margin-top:-0.35em;"><span style="visibility:hidden; margin-right:0.099em;"> (</span><span data-interwiki-article="Sorcim" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=Sorcim&amp;action=edit&amp;redlink=1" title="Sorcim (страница отсутствует)">рус.</a></span></span></span></span> в <a href="/wiki/1981_%D0%B3%D0%BE%D0%B4" title="1981 год">1981 году</a> и изначально поставлявшаяся вместе с <a class="extiw" href="https://en.wikipedia.org/wiki/WordStar" title="en:WordStar"><span title="WordStar — версия статьи «WordStar» на английском языке">WordStar</span></a><span class="noprint" style="font-style:normal; font-weight:normal;"> <span style="display:inline-block; font-size:80%; position:relative;"><span style="font-size:117.6%; margin-left:-0.43em; position:relative; top:0.28em;"> <small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></span><span class="link-ru metadata" style="margin-left:-0.43em; position:absolute; left:0; z-index:1; margin-top:-0.35em;"><span style="visibility:hidden; margin-right:0.099em;"> (</span><span data-interwiki-article="WordStar" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=WordStar&amp;action=edit&amp;redlink=1" title="WordStar (страница отсутствует)">рус.</a></span></span></span></span> в составе набора программ <a href="/wiki/CP/M" title="CP/M">CP/M</a> для <a href="/wiki/%D0%9F%D0%BE%D1%80%D1%82%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D1%8B%D0%B9_%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80" title="Портативный компьютер">портативного компьютера</a> <a href="/wiki/Osborne_1" title="Osborne 1">Osborne 1</a>. В <a href="/wiki/1984_%D0%B3%D0%BE%D0%B4" title="1984 год">1984 году</a> Sorcim была куплена <a class="mw-redirect" href="/wiki/Computer_Associates" title="Computer Associates">Computer Associates</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>, из-за чего последние версии SuperCalc носили название CA-SuperCalc.
</p><p>В отличие от <a href="/wiki/VisiCalc" title="VisiCalc">VisiCalc</a>, SuperCalc является одной из первых электронных таблиц, способных <a class="mw-disambig" href="/wiki/%D0%98%D1%82%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F" title="Итерация">итеративно</a> разрешать <a class="extiw" href="https://en.wikipedia.org/wiki/Circular_reference" title="en:Circular reference"><span title="Circular reference — версия статьи «Кольцевые ссылки» на английском языке">кольцевые ссылки</span></a><span class="noprint" style="font-style:normal; font-weight:normal;"> <span style="display:inline-block; font-size:80%; position:relative;"><span style="font-size:117.6%; margin-left:-0.43em; position:relative; top:0.28em;"> <small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></span><span class="link-ru metadata" style="margin-left:-0.43em; position:absolute; left:0; z-index:1; margin-top:-0.35em;"><span style="visibility:hidden; margin-right:0.099em;"> (</span><span data-interwiki-article="Circular reference" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%9A%D0%BE%D0%BB%D1%8C%D1%86%D0%B5%D0%B2%D1%8B%D0%B5_%D1%81%D1%81%D1%8B%D0%BB%D0%BA%D0%B8&amp;action=edit&amp;redlink=1" title="Кольцевые ссылки (страница отсутствует)">рус.</a></span></span></span></span> (ячейки, значения которых зависят друг от друга)<sup class="reference" id="cite_ref-fei_2-0"><a href="#cite_note-fei-2">[2]</a></sup>.
</p><p>Версии SuperCalc были выпущены для компьютеров <a href="/wiki/Apple_II" title="Apple II">Apple II</a>, <a href="/wiki/IBM_PC" title="IBM PC">IBM PC</a> с операционной системой <a href="/wiki/MS-DOS" title="MS-DOS">DOS</a>, а также для <a class="mw-redirect" href="/wiki/Microsoft_Windows" title="Microsoft Windows">MS Windows</a>, <a href="/wiki/%D0%9C%D0%B5%D0%B9%D0%BD%D1%84%D1%80%D0%B5%D0%B9%D0%BC" title="Мейнфрейм">мейнфреймов</a> IBM (<a href="/wiki/IBM_System/360" title="IBM System/360">S/360</a>) и <a href="/wiki/VAX" title="VAX">VAX</a>/<a class="mw-redirect" href="/wiki/VMS" title="VMS">VMS</a>.
</p>

<h2><span id=".D0.92.D0.B5.D1.80.D1.81.D0.B8.D0.B8"></span><span class="mw-headline" id="Версии">Версии</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=SuperCalc&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Версии»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=SuperCalc&amp;action=edit&amp;section=1" title="Редактировать код раздела «Версии»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="SuperCalc_1">SuperCalc 1</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=SuperCalc&amp;veaction=edit&amp;section=2" title="Редактировать раздел «SuperCalc 1»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=SuperCalc&amp;action=edit&amp;section=2" title="Редактировать код раздела «SuperCalc 1»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Supercalc_1.png"><img class="mw-file-element" data-file-height="400" data-file-width="640" decoding="async" height="200" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/69/Supercalc_1.png/320px-Supercalc_1.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/69/Supercalc_1.png/480px-Supercalc_1.png 1.5x, //upload.wikimedia.org/wikipedia/commons/6/69/Supercalc_1.png 2x" width="320"/></a><figcaption>Основной экран SuperCalc 1</figcaption></figure>
<p>В конце 1980 года <a href="/wiki/%D0%9E%D1%81%D0%B1%D0%BE%D1%80%D0%BD,_%D0%90%D0%B4%D0%B0%D0%BC" title="Осборн, Адам">Адам Осборн</a> искал поставщика электронных таблиц для своего компьютера <a href="/wiki/Osborne_1" title="Osborne 1">Osborne 1</a>. Поскольку ему не удалось приобрести права на <a href="/wiki/VisiCalc" title="VisiCalc">VisiCalc</a>, он обратился в Sorcim с просьбой разработать электронную таблицу, которая могла бы соперничать с VisiCalc и была бы выпущена к апрелю 1981 года, чтобы её можно было показать на <a class="extiw" href="https://en.wikipedia.org/wiki/West_Coast_Computer_Faire" title="en:West Coast Computer Faire"><span title="West Coast Computer Faire — версия статьи «West Coast Computer Faire» на английском языке">West Coast Computer Faire</span></a><span class="noprint" style="font-style:normal; font-weight:normal;"> <span style="display:inline-block; font-size:80%; position:relative;"><span style="font-size:117.6%; margin-left:-0.43em; position:relative; top:0.28em;"> <small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></span><span class="link-ru metadata" style="margin-left:-0.43em; position:absolute; left:0; z-index:1; margin-top:-0.35em;"><span style="visibility:hidden; margin-right:0.099em;"> (</span><span data-interwiki-article="West Coast Computer Faire" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=West_Coast_Computer_Faire&amp;action=edit&amp;redlink=1" title="West Coast Computer Faire (страница отсутствует)">рус.</a></span></span></span></span>. Основным разработчиком программы стал Гари Балейсен (Gary Balleisen)<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>. В итоге презентация была встречена с достаточно большим энтузиазмом. SuperCalc был написан на <a href="/wiki/%D0%90%D1%81%D1%81%D0%B5%D0%BC%D0%B1%D0%BB%D0%B5%D1%80" title="Ассемблер">ассемблере</a>, разработка стоила $20 000<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>. Наряду с WordStar, SuperCalc вошёл в комплект поставки компьютера <a href="/wiki/Osborne_1" title="Osborne 1">Osborne 1</a>. Также он продавался отдельно по цене $295<sup class="reference" id="cite_ref-barry_5-0"><a href="#cite_note-barry-5">[5]</a></sup>.
</p><p>Программе требовалась операционная система <a href="/wiki/CP/M" title="CP/M">CP/M</a> версии 2.2, 48 килобайт оперативной памяти и один дисковод. Поскольку большинство машин с <a href="/wiki/CP/M" title="CP/M">CP/M</a> использовали вывод на <a href="/wiki/%D0%9A%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D1%8B%D0%B9_%D1%82%D0%B5%D1%80%D0%BC%D0%B8%D0%BD%D0%B0%D0%BB" title="Компьютерный терминал">последовательный терминал</a>, большинство других электронных таблиц для этой операционной системы не выводили таблицу в реальном времени; SuperCalc же содержал настройки для различных типов терминалов, позволявшие ему достаточно быстро перерисовывать экран в реальном времени<sup class="reference" id="cite_ref-barry_5-1"><a href="#cite_note-barry-5">[5]</a></sup>. Максимальный размер таблицы составлял 254 строки и 63 столбца. Максимальная длина текста в ячейке составляла 117 символов. Поддерживалось изменение ширины столбцов, а также вертикальное и горизонтальное разделение экрана. Была возможна защита ячеек от изменения<sup class="reference" id="cite_ref-barry_5-2"><a href="#cite_note-barry-5">[5]</a></sup>.
</p><p>Для математических вычислений использовался <a href="/wiki/%D0%94%D0%B2%D0%BE%D0%B8%D1%87%D0%BD%D0%BE-%D0%B4%D0%B5%D1%81%D1%8F%D1%82%D0%B8%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%BE%D0%B4" title="Двоично-десятичный код">двоично-десятичный код</a>, что позволяло увеличить точность вычислений<sup class="reference" id="cite_ref-fei_2-1"><a href="#cite_note-fei-2">[2]</a></sup>. Как целые числа, так и числа с <a class="mw-redirect" href="/wiki/%D0%A7%D0%B8%D1%81%D0%BB%D0%B0_%D1%81_%D0%BF%D0%BB%D0%B0%D0%B2%D0%B0%D1%8E%D1%89%D0%B5%D0%B9_%D0%B7%D0%B0%D0%BF%D1%8F%D1%82%D0%BE%D0%B9" title="Числа с плавающей запятой">плавающей запятой</a> обрабатывались в диапазоне <a href="/wiki/%D0%AD%D0%BA%D1%81%D0%BF%D0%BE%D0%BD%D0%B5%D0%BD%D1%82%D0%B0" title="Экспонента">экспонент</a> от 10<sup>−16</sup> до 10<sup>16</sup>. Среди поддерживаемых математических функций были вычисление <a href="/wiki/%D0%90%D0%B1%D1%81%D0%BE%D0%BB%D1%8E%D1%82%D0%BD%D0%B0%D1%8F_%D0%B2%D0%B5%D0%BB%D0%B8%D1%87%D0%B8%D0%BD%D0%B0" title="Абсолютная величина">абсолютной величины</a>, <a href="/wiki/%D0%A2%D1%80%D0%B8%D0%B3%D0%BE%D0%BD%D0%BE%D0%BC%D0%B5%D1%82%D1%80%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B5_%D1%84%D1%83%D0%BD%D0%BA%D1%86%D0%B8%D0%B8" title="Тригонометрические функции">тригонометрические функции</a>, <a href="/wiki/%D0%AD%D0%BA%D1%81%D0%BF%D0%BE%D0%BD%D0%B5%D0%BD%D1%82%D0%B0" title="Экспонента">экспонента</a> и <a href="/wiki/%D0%9B%D0%BE%D0%B3%D0%B0%D1%80%D0%B8%D1%84%D0%BC" title="Логарифм">логарифмы</a>. Имелись также <a class="mw-disambig" href="/wiki/%D0%9B%D0%BE%D0%B3%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%84%D1%83%D0%BD%D0%BA%D1%86%D0%B8%D1%8F" title="Логическая функция">логические функции</a> и функции поиска минимума и максимума.
</p><p>С программой поставлялось 108-страничное руководство пользователя. Также присутствовала встроенная в программу справка<sup class="reference" id="cite_ref-barry_5-3"><a href="#cite_note-barry-5">[5]</a></sup>.
</p><p>Обозревателем журнала <a href="/wiki/InfoWorld" title="InfoWorld">InfoWorld</a> SuperCalc был оценён достаточно высоко: он предположил, что у программы есть потенциал для того, чтобы стать «классикой в своём жанре»<sup class="reference" id="cite_ref-barry_5-4"><a href="#cite_note-barry-5">[5]</a></sup>.
</p>
<h3><span class="mw-headline" id="SuperCalc2">SuperCalc<sup>2</sup></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=SuperCalc&amp;veaction=edit&amp;section=3" title="Редактировать раздел «SuperCalc2»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=SuperCalc&amp;action=edit&amp;section=3" title="Редактировать код раздела «SuperCalc2»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Supercalc_2.png"><img class="mw-file-element" data-file-height="480" data-file-width="544" decoding="async" height="240" src="//upload.wikimedia.org/wikipedia/ru/thumb/1/1c/Supercalc_2.png/272px-Supercalc_2.png" srcset="//upload.wikimedia.org/wikipedia/ru/thumb/1/1c/Supercalc_2.png/408px-Supercalc_2.png 1.5x, //upload.wikimedia.org/wikipedia/ru/1/1c/Supercalc_2.png 2x" width="272"/></a><figcaption>Основной экран SuperCalc<sup>2</sup></figcaption></figure>
<p>SuperCalc<sup>2</sup> был выпущен в апреле <a href="/wiki/1983_%D0%B3%D0%BE%D0%B4" title="1983 год">1983 года</a> для <a href="/wiki/DOS" title="DOS">DOS</a> 1.1 и старше, а также <a href="/wiki/CP/M" title="CP/M">CP/M</a>-80 и 86 и продавался по цене $295. Под управлением DOS ему требовалось 64 килобайта памяти, CP/M — 48. Максимальный размер таблицы составлял 254 строки и 63 столбца<sup class="reference" id="cite_ref-jadsc2_6-0"><a href="#cite_note-jadsc2-6">[6]</a></sup>. На конец 1983 года пользовательская база программы составляла примерно 250 000 установок<sup class="reference" id="cite_ref-jad_7-0"><a href="#cite_note-jad-7">[7]</a></sup>.
</p><p>Были добавлены функции арифметики дат и работы с календарём. Было возможно слияние данных из нескольких таблиц. Данные можно было сортировать по строкам или столбцам, возможно было также скрытие определённого диапазона данных. Имелась поддержка записываемых в дисковый файл макрокоманд. Добавилась возможность возвращать из формулы строку текста (например, функция вычисления истинности значения могла выдавать текст «Хорошо» для истинного значения и «Плохо» для ложного). Среди математических функций добавились округление и <a href="/wiki/%D0%94%D0%B5%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5_%D1%81_%D0%BE%D1%81%D1%82%D0%B0%D1%82%D0%BA%D0%BE%D0%BC" title="Деление с остатком">деление с остатком</a><sup class="reference" id="cite_ref-jadsc2_6-1"><a href="#cite_note-jadsc2-6">[6]</a></sup>.
</p><p>С программой поставлялся модуль Super DataIntercharge, позволявший осуществлять экспорт и импорт файлов формата <a href="/wiki/CSV" title="CSV">CSV</a> (только данные) и SDF (SuperData Format, также формулы). Поддерживалась полная прямая совместимость с первой версией, обратная совместимость была ограничена файлами, не использующими новые функции<sup class="reference" id="cite_ref-jadsc2_6-2"><a href="#cite_note-jadsc2-6">[6]</a></sup>.
</p><p>Скорость работы была увеличена примерно в 2 раза. Были добавлены расширенные функции настройки печати (длина и ширина страницы, интервал, возможность задания специальных команд для конкретной модели принтера)<sup class="reference" id="cite_ref-jadsc2_6-3"><a href="#cite_note-jadsc2-6">[6]</a></sup>.
</p><p>В комплект входило руководство пользователя. По сравнению с первой версией, добавился короткий буклет «10 Minutes to SuperCalc<sup>2</sup>». Имелась также справочная карточка с основными командами<sup class="reference" id="cite_ref-jadsc2_6-4"><a href="#cite_note-jadsc2-6">[6]</a></sup>.
</p>
<h3><span class="mw-headline" id="SuperCalc_3">SuperCalc 3</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=SuperCalc&amp;veaction=edit&amp;section=4" title="Редактировать раздел «SuperCalc 3»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=SuperCalc&amp;action=edit&amp;section=4" title="Редактировать код раздела «SuperCalc 3»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Supercalc_3.png"><img class="mw-file-element" data-file-height="400" data-file-width="640" decoding="async" height="200" src="//upload.wikimedia.org/wikipedia/ru/thumb/b/b3/Supercalc_3.png/320px-Supercalc_3.png" srcset="//upload.wikimedia.org/wikipedia/ru/thumb/b/b3/Supercalc_3.png/480px-Supercalc_3.png 1.5x, //upload.wikimedia.org/wikipedia/ru/b/b3/Supercalc_3.png 2x" width="320"/></a><figcaption>Основной экран SuperCalc 3</figcaption></figure>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Supercalc_3_Graphs.png"><img class="mw-file-element" data-file-height="400" data-file-width="640" decoding="async" height="200" src="//upload.wikimedia.org/wikipedia/commons/thumb/f/fc/Supercalc_3_Graphs.png/320px-Supercalc_3_Graphs.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/f/fc/Supercalc_3_Graphs.png/480px-Supercalc_3_Graphs.png 1.5x, //upload.wikimedia.org/wikipedia/commons/f/fc/Supercalc_3_Graphs.png 2x" width="320"/></a><figcaption>Графики, построенные в SuperCalc 3</figcaption></figure>
<p>SuperCalc 3 был выпущен в конце <a href="/wiki/1983_%D0%B3%D0%BE%D0%B4" title="1983 год">1983 года</a> для <a href="/wiki/IBM_PC" title="IBM PC">IBM-совместимых</a> компьютеров и продавался по цене $395. Для его работы требовалась операционная система DOS версии 1.0 и выше, 96 килобайт оперативной памяти и один дисковод. Максимальный размер таблицы составлял 254 строки и 63 столбца<sup class="reference" id="cite_ref-jad_7-1"><a href="#cite_note-jad-7">[7]</a></sup>. SuperCalc 3 был написан на языке программирования <a class="mw-redirect" href="/wiki/Pascal" title="Pascal">Pascal</a><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>. Использовалась технология <a href="/wiki/%D0%A0%D0%B0%D0%B7%D1%80%D0%B5%D0%B6%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9_%D0%BC%D0%B0%D1%81%D1%81%D0%B8%D0%B2" title="Разреженный массив">разреженного</a> хранения данных: в памяти хранилась только информация в ячейках, содержащих данные, а также сведения о том, какие ячейки являются пустыми<sup class="reference" id="cite_ref-shen_9-0"><a href="#cite_note-shen-9">[9]</a></sup>.
</p><p>В программу было встроено более 50 функций. Были добавлены финансовые функции, которые включали вычисление <a href="/wiki/%D0%92%D0%BD%D1%83%D1%82%D1%80%D0%B5%D0%BD%D0%BD%D1%8F%D1%8F_%D0%BD%D0%BE%D1%80%D0%BC%D0%B0_%D0%B4%D0%BE%D1%85%D0%BE%D0%B4%D0%BD%D0%BE%D1%81%D1%82%D0%B8" title="Внутренняя норма доходности">внутренней нормы доходности</a>, <a href="/wiki/%D0%A7%D0%B8%D1%81%D1%82%D0%B0%D1%8F_%D0%BF%D1%80%D0%B8%D0%B2%D0%B5%D0%B4%D1%91%D0%BD%D0%BD%D0%B0%D1%8F_%D1%81%D1%82%D0%BE%D0%B8%D0%BC%D0%BE%D1%81%D1%82%D1%8C" title="Чистая приведённая стоимость">чистой приведённой стоимости</a> и выплат по кредитам<sup class="reference" id="cite_ref-jad_7-2"><a href="#cite_note-jad-7">[7]</a></sup>. Из математических функций имелись округление, вычисление суммы и среднего, генератор случайных чисел<sup class="reference" id="cite_ref-sc3r2_10-0"><a href="#cite_note-sc3r2-10">[10]</a></sup>. Для повышения производительности вместо <a href="/wiki/%D0%94%D0%B2%D0%BE%D0%B8%D1%87%D0%BD%D0%BE-%D0%B4%D0%B5%D1%81%D1%8F%D1%82%D0%B8%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%BE%D0%B4" title="Двоично-десятичный код">двоично-десятичного</a> кода использовалось <a href="/wiki/%D0%94%D0%B2%D0%BE%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D1%81%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0_%D1%81%D1%87%D0%B8%D1%81%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F" title="Двоичная система счисления">двоичное</a> представление чисел<sup class="reference" id="cite_ref-fei_2-2"><a href="#cite_note-fei-2">[2]</a></sup>. Поддерживалась работа с числами с <a class="mw-redirect" href="/wiki/%D0%A7%D0%B8%D1%81%D0%BB%D0%B0_%D1%81_%D0%BF%D0%BB%D0%B0%D0%B2%D0%B0%D1%8E%D1%89%D0%B5%D0%B9_%D0%B7%D0%B0%D0%BF%D1%8F%D1%82%D0%BE%D0%B9" title="Числа с плавающей запятой">плавающей запятой</a> от 1×10<sup>−63</sup> до 1×10<sup>62</sup><sup class="reference" id="cite_ref-wortman85_11-0"><a href="#cite_note-wortman85-11">[11]</a></sup>
</p><p>Были добавлены функции управления базами данных, сортировка по двум ключам, графические возможности. Имелось 7 типов графиков (линейный, круговой, гистограмма, гистограмма с накоплением, с областями, точечный, биржевой), было возможно изменение шрифта заголовков и меток (до 8 шрифтов), а также печать до 4 графиков на одной странице. Возможно было отображение графиков на экране, в том числе в цвете. Графики были двухмерными<sup class="reference" id="cite_ref-jad_7-3"><a href="#cite_note-jad-7">[7]</a></sup>.
</p><p>Программа поставлялась на двух дисках, включавших, помимо самой программы, примеры таблиц, дополнительные программы Sideways (печать таблиц в альбомной ориентации) и Superdata Intercharge (преобразование файлов Lotus 1-2-3 и Visicalc). В комплект также входили руководство пользователя для основной программы и Superdata Intercharge, короткий буклет «10 Minutes to SuperCalc 3» и две карточки, содержащие краткую информацию о командах и графических возможностях<sup class="reference" id="cite_ref-jad_7-4"><a href="#cite_note-jad-7">[7]</a></sup><sup class="reference" id="cite_ref-sc3r2_10-1"><a href="#cite_note-sc3r2-10">[10]</a></sup>.
</p><p>Защита от копирования отсутствовала<sup class="reference" id="cite_ref-jad_7-5"><a href="#cite_note-jad-7">[7]</a></sup>. Компанией <a class="new" href="/w/index.php?title=Sorcim&amp;action=edit&amp;redlink=1" title="Sorcim (страница отсутствует)">Sorcim</a> предоставлялась программа корпоративного лицензирования SuperCalc, понижавшая цену до $237 за копию при покупке лицензии на 25 пользователей, до $118 при покупке лицензии на 400 пользователей, а также лицензия с неограниченным числом пользователей за $65 000<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>.
</p>
<h4><span id=".D0.A0.D0.B5.D0.BB.D0.B8.D0.B7_2"></span><span class="mw-headline" id="Релиз_2">Релиз 2</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=SuperCalc&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Релиз 2»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=SuperCalc&amp;action=edit&amp;section=5" title="Редактировать код раздела «Релиз 2»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h4>
<p>SuperCalc 3 Release 2 был выпущен в <a href="/wiki/1984_%D0%B3%D0%BE%D0%B4" title="1984 год">1984 году</a> и продавался по цене $395. Для его работы требовалась операционная система DOS версии 2.0 и выше, 96 килобайт оперативной памяти и один дисковод. Поддерживались 33 модели принтеров и 18 моделей плоттеров<sup class="reference" id="cite_ref-sc3r2_10-2"><a href="#cite_note-sc3r2-10">[10]</a></sup>.
</p><p>По скорости вычислений Release 2 превосходил <a href="/wiki/Lotus_1-2-3" title="Lotus 1-2-3">Lotus 1-2-3</a>, причём использование математического сопроцессора давало прирост скорости до 10 раз. Максимальный размер таблицы составлял 9999 строк и 127 столбцов (при наличии менее чем 192 килобайт памяти — 254 строки и 63 столбца). Была возможность итеративного вычисления с кольцевыми ссылками без пересчёта остальной таблицы. Данная возможность была уникальна для электронных таблиц того периода.
</p><p>Среди улучшений по сравнению с первой версией SuperCalc 3 называлось улучшенная прокрутка, разделение экрана по горизонтали и вертикали<sup class="reference" id="cite_ref-sc3r2_10-3"><a href="#cite_note-sc3r2-10">[10]</a></sup>.
</p><p>Программа поставлялась на двух дисках, включавших, помимо самой программы, примеры таблиц, дополнительные программы Sideways (печать таблиц в альбомной ориентации) и Superdata Intercharge (преобразование файлов Lotus 1-2-3 и Visicalc). В комплект также входили руководство пользователя для основной программы и Superdata Intercharge, короткий буклет «10 Minutes to SuperCalc 3» и две карточки, содержащие краткую информацию о командах и графических возможностях<sup class="reference" id="cite_ref-sc3r2_10-4"><a href="#cite_note-sc3r2-10">[10]</a></sup>.
</p>
<h4><span class="mw-headline" id="SuperCalc_3a">SuperCalc 3a</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=SuperCalc&amp;veaction=edit&amp;section=6" title="Редактировать раздел «SuperCalc 3a»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=SuperCalc&amp;action=edit&amp;section=6" title="Редактировать код раздела «SuperCalc 3a»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h4>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Supercalc_3a.png"><img class="mw-file-element" data-file-height="471" data-file-width="635" decoding="async" height="237" src="//upload.wikimedia.org/wikipedia/ru/thumb/e/ed/Supercalc_3a.png/320px-Supercalc_3a.png" srcset="//upload.wikimedia.org/wikipedia/ru/thumb/e/ed/Supercalc_3a.png/480px-Supercalc_3a.png 1.5x, //upload.wikimedia.org/wikipedia/ru/e/ed/Supercalc_3a.png 2x" width="320"/></a><figcaption>Основной экран SuperCalc 3a</figcaption></figure>
<p>SuperCalc 3a был выпущен в 1985 году и предназначался для компьютеров <a class="mw-redirect" href="/wiki/Apple_IIc" title="Apple IIc">Apple IIc</a> и Enhanced <a class="mw-redirect" href="/wiki/Apple_IIe" title="Apple IIe">Apple IIe</a>. Из-за ограниченных возможностей этих компьютеров не поддерживалось итеративное вычисление кольцевых ссылок. Поддерживались таблицы размером до 63 столбцов и 254 строк (до 116 символов в ячейке). При отображении графиков на экране можно было использовать до 16 цветов, а при печати — до 90 цветов<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup>.
</p>
<h4><span id=".D0.A0.D0.B5.D0.BB.D0.B8.D0.B7_2.1"></span><span class="mw-headline" id="Релиз_2.1">Релиз 2.1</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=SuperCalc&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Релиз 2.1»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=SuperCalc&amp;action=edit&amp;section=7" title="Редактировать код раздела «Релиз 2.1»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h4>
<p>SuperCalc 3 Release 2.1 был выпущен в <a href="/wiki/1985_%D0%B3%D0%BE%D0%B4" title="1985 год">1985 году</a>. В программу была добавлена поддержка расширенной памяти стандарта <a class="mw-redirect" href="/wiki/Expanded_memory" title="Expanded memory">EMS</a> (до 8 мегабайт), а также графического адаптера <a href="/wiki/EGA" title="EGA">EGA</a>. Сама программа занимала в памяти 96 килобайт и поддерживала до 127 столбцов и 9999 строк таблицы. Была добавлена поддержка 24 новых принтеров и 8 <a href="/wiki/%D0%93%D1%80%D0%B0%D1%84%D0%BE%D0%BF%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B8%D1%82%D0%B5%D0%BB%D1%8C" title="Графопостроитель">плоттеров</a><sup class="reference" id="cite_ref-wortman85_11-1"><a href="#cite_note-wortman85-11">[11]</a></sup>.
</p><p>Была улучшена производительность (так, вывод графиков осуществлялся вдвое быстрее, чем в предыдущей версии). Была добавлена возможность настройки цветов элементов графика<sup class="reference" id="cite_ref-wortman85_11-2"><a href="#cite_note-wortman85-11">[11]</a></sup>.
</p><p>Имелась поддержка высококачественной графики, внешних макросов<sup class="reference" id="cite_ref-wortman85_11-3"><a href="#cite_note-wortman85-11">[11]</a></sup>.
</p>
<h3><span class="mw-headline" id="SuperCalc_4">SuperCalc 4</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=SuperCalc&amp;veaction=edit&amp;section=8" title="Редактировать раздел «SuperCalc 4»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=SuperCalc&amp;action=edit&amp;section=8" title="Редактировать код раздела «SuperCalc 4»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Supercalc_4.png"><img class="mw-file-element" data-file-height="400" data-file-width="640" decoding="async" height="200" src="//upload.wikimedia.org/wikipedia/ru/thumb/a/a2/Supercalc_4.png/320px-Supercalc_4.png" srcset="//upload.wikimedia.org/wikipedia/ru/thumb/a/a2/Supercalc_4.png/480px-Supercalc_4.png 1.5x, //upload.wikimedia.org/wikipedia/ru/a/a2/Supercalc_4.png 2x" width="320"/></a><figcaption>Основной экран SuperCalc 4</figcaption></figure>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Supercalc_4_Pie_Chart.png"><img class="mw-file-element" data-file-height="350" data-file-width="640" decoding="async" height="175" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/bb/Supercalc_4_Pie_Chart.png/320px-Supercalc_4_Pie_Chart.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/bb/Supercalc_4_Pie_Chart.png/480px-Supercalc_4_Pie_Chart.png 1.5x, //upload.wikimedia.org/wikipedia/commons/b/bb/Supercalc_4_Pie_Chart.png 2x" width="320"/></a><figcaption>Круговая диаграмма, построенная в SuperCalc 4</figcaption></figure>
<p>Версия SuperCalc 4 была выпущена в <a href="/wiki/1986_%D0%B3%D0%BE%D0%B4" title="1986 год">1986 году</a> и продавалась по цене $495. SuperCalc 4 позиционировался как основной конкурент <a href="/wiki/Lotus_1-2-3" title="Lotus 1-2-3">Lotus 1-2-3</a>. По состоянию на 1987 год исследователями рынка табличных процессоров SuperCalc ставился на второе место после Lotus 1-2-3, причём значительная часть установок приходилась на крупные корпорации, что было связано с политикой лицензирования, предполагавшей значительные скидки при покупке большого числа копий (так, при покупке 25 лицензий цена снижалась до $247), а также с отсутствием защиты от копирования<sup class="reference" id="cite_ref-pcs87_14-0"><a href="#cite_note-pcs87-14">[14]</a></sup>. В числе корпораций, использовавших SuperCalc, были <a href="/wiki/Hughes_Aircraft" title="Hughes Aircraft">Hughes Aircraft</a>, <a href="/wiki/Lockheed" title="Lockheed">Lockheed Missile and Space Inc.</a>, <a class="mw-redirect" href="/wiki/Jet_Propulsion_Laboratory" title="Jet Propulsion Laboratory">Jet Propulsion Laboratory</a> и <a class="mw-redirect" href="/wiki/Wells_Fargo_Bank" title="Wells Fargo Bank">Wells Fargo Bank</a><sup class="reference" id="cite_ref-autogenerated1_15-0"><a href="#cite_note-autogenerated1-15">[15]</a></sup>.
</p><p>Программа работала под управлением операционной системы DOS версии 2.0 и старше на IBM-совместимых компьютерах начиная с <a href="/wiki/IBM_PC" title="IBM PC">IBM PC</a>. Для запуска требовалось 256 килобайт оперативной памяти. Имелась возможность использования <a href="/wiki/%D0%A0%D0%B0%D1%81%D1%88%D0%B8%D1%80%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%BF%D0%B0%D0%BC%D1%8F%D1%82%D1%8C" title="Расширенная память">расширенной памяти</a> для работы с большими таблицами (поддерживалось до 8 мегабайт EMS-памяти). Поддерживались видеоадаптеры <a href="/wiki/EGA" title="EGA">EGA</a> и <a href="/wiki/HGC" title="HGC">Hercules</a>, а также математические сопроцессоры <a class="mw-redirect" href="/wiki/8087" title="8087">8087</a> и <a class="mw-redirect" href="/wiki/80287" title="80287">80287</a><sup class="reference" id="cite_ref-kall_16-0"><a href="#cite_note-kall-16">[16]</a></sup>.
</p><p>В числе основных нововведений данной версии — поддержка <a class="mw-redirect" href="/wiki/%D0%9C%D0%B0%D0%BA%D1%80%D0%BE%D1%81" title="Макрос">макрокоманд</a>, хранимых вместе с электронной таблицей. Многие команды совпадают с использовавшимися в <a href="/wiki/Lotus_1-2-3" title="Lotus 1-2-3">Lotus 1-2-3</a>, однако из-за различий в структуре меню использовать существующие макросы для <a href="/wiki/Lotus_1-2-3" title="Lotus 1-2-3">Lotus 1-2-3</a> было нельзя без их модификации. Макросы могли выводить на экран сообщения и видоизменять внешний вид рабочего экрана (например, можно было убрать подписи строк и столбцов). Кроме того, появилась возможность автоматической записи макросов. Это было существенным конкурентным преимуществом<sup class="reference" id="cite_ref-kall_16-1"><a href="#cite_note-kall-16">[16]</a></sup>.
</p><p>Присутствовали также функции работы с базами данных: печать отчётов, заполнение данными и т. д.<sup class="reference" id="cite_ref-kall_16-2"><a href="#cite_note-kall-16">[16]</a></sup> В SuperCalc 4 отсутствовали функции по работе со <a href="/wiki/%D0%A1%D1%82%D1%80%D0%BE%D0%BA%D0%BE%D0%B2%D1%8B%D0%B9_%D1%82%D0%B8%D0%BF" title="Строковый тип">строками</a> и <a href="/wiki/%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%86%D0%B0_(%D0%BC%D0%B0%D1%82%D0%B5%D0%BC%D0%B0%D1%82%D0%B8%D0%BA%D0%B0)" title="Матрица (математика)">матрицами</a>, а также по осуществлению <a href="/wiki/%D0%A0%D0%B5%D0%B3%D1%80%D0%B5%D1%81%D1%81%D0%B8%D0%BE%D0%BD%D0%BD%D1%8B%D0%B9_%D0%B0%D0%BD%D0%B0%D0%BB%D0%B8%D0%B7" title="Регрессионный анализ">множественного регрессионного анализа</a>, присутствовавшие у конкурентов, а также функции поиска решения (которые были ограничены созданием кольцевых ссылок с контролируемым числом итераций)<sup class="reference" id="cite_ref-btn_17-0"><a href="#cite_note-btn-17">[17]</a></sup>. В то же время были доступны многие дополнительные финансовые функции (например, вычисление основной суммы и процентов по <a href="/wiki/%D0%90%D0%BD%D0%BD%D1%83%D0%B8%D1%82%D0%B5%D1%82" title="Аннуитет">аннуитетам</a>), отсутствовавшие в <a href="/wiki/Lotus_1-2-3" title="Lotus 1-2-3">Lotus 1-2-3</a>, а также решение <a href="/wiki/%D0%A1%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0_%D1%83%D1%80%D0%B0%D0%B2%D0%BD%D0%B5%D0%BD%D0%B8%D0%B9" title="Система уравнений">систем уравнений</a> с использованием неограниченного числа <a class="mw-disambig" href="/wiki/%D0%98%D1%82%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F" title="Итерация">итеративных</a> приближений<sup class="reference" id="cite_ref-kall_16-3"><a href="#cite_note-kall-16">[16]</a></sup>. Возможно было слияние данных из нескольких таблиц (путём сложения), причём как с использованием отдельных диапазонов данных, так и таблиц целиком<sup class="reference" id="cite_ref-miller_18-0"><a href="#cite_note-miller-18">[18]</a></sup>. Поддерживалась сортировка по строкам и столбцам с использованием до 2 ключей сортировки<sup class="reference" id="cite_ref-btn_17-1"><a href="#cite_note-btn-17">[17]</a></sup>.
</p><p>В данной версии появились улучшенные средства импорта и экспорта файлов <a href="/wiki/Lotus_1-2-3" title="Lotus 1-2-3">Lotus 1-2-3</a>: при данных операциях сохранялись функции и формулы, однако не поддерживалась конвертация макросов. Кроме того, поддерживалась прямая и обратная совместимость с SuperCalc 3, ограниченная лишь невозможностью работы со встроенными в таблицу макросами в старой версии. Имелась также возможность импорта и экспорта данных в форматах <a class="new" href="/w/index.php?title=Data_Interchange_Format&amp;action=edit&amp;redlink=1" title="Data Interchange Format (страница отсутствует)">DIF</a> и <a href="/wiki/CSV" title="CSV">CSV</a><sup class="reference" id="cite_ref-kall_16-4"><a href="#cite_note-kall-16">[16]</a></sup>. Также поддерживалась совместимость с <a class="mw-redirect" href="/wiki/Visicalc" title="Visicalc">Visicalc</a><sup class="reference" id="cite_ref-sc4fl_19-0"><a href="#cite_note-sc4fl-19">[19]</a></sup>. Возможности по импорту и экспорту файлов формата <a href="/wiki/DBF" title="DBF">DBF</a> (<a href="/wiki/DBase" title="DBase">DBase</a>) отсутствовали<sup class="reference" id="cite_ref-btn_17-2"><a href="#cite_note-btn-17">[17]</a></sup>.
</p><p>Присутствовали интерфейсные улучшения: в строке меню появились полные наименования команд (а не однобуквенные коды), также из Microsoft <a class="extiw" href="https://en.wikipedia.org/wiki/Multiplan" title="en:Multiplan"><span title="Multiplan — версия статьи «Multiplan» на английском языке">Multiplan</span></a><span class="noprint" style="font-style:normal; font-weight:normal;"> <span style="display:inline-block; font-size:80%; position:relative;"><span style="font-size:117.6%; margin-left:-0.43em; position:relative; top:0.28em;"> <small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></span><span class="link-ru metadata" style="margin-left:-0.43em; position:absolute; left:0; z-index:1; margin-top:-0.35em;"><span style="visibility:hidden; margin-right:0.099em;"> (</span><span data-interwiki-article="Multiplan" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=Multiplan&amp;action=edit&amp;redlink=1" title="Multiplan (страница отсутствует)">рус.</a></span></span></span></span> и <a href="/wiki/Lotus_1-2-3" title="Lotus 1-2-3">Lotus 1-2-3</a> была заимствована возможность создавать именованные диапазоны<sup class="reference" id="cite_ref-kall_16-5"><a href="#cite_note-kall-16">[16]</a></sup>.
</p><p>Графические средства SuperCalc, которые и ранее превосходили таковые у <a href="/wiki/Lotus_1-2-3" title="Lotus 1-2-3">Lotus 1-2-3</a>, также были улучшены в данной версии. Создание и печать графиков (а также настройка принтера) осуществлялись без выхода из основной программы. На графиках могло быть представлено до 10 переменных, предоставлялись широкие возможности расстановки меток на графиках и выбора цветов. Поддерживались 56 чёрно-белых принтеров, 17 цветных принтеров и 26 <a class="mw-redirect" href="/wiki/%D0%9F%D0%BB%D0%BE%D1%82%D1%82%D0%B5%D1%80" title="Плоттер">плоттеров</a><sup class="reference" id="cite_ref-kall_16-6"><a href="#cite_note-kall-16">[16]</a></sup>. Печать таблиц в альбомной ориентации была возможна с использованием отдельной утилиты Sideways<sup class="reference" id="cite_ref-kall_16-7"><a href="#cite_note-kall-16">[16]</a></sup>. Определение используемого аппаратного обеспечения осуществлялось автоматически без использования отдельной процедуры установки. В совокупности с отсутствием защиты от копирования это облегчало установку программы на новый компьютер<sup class="reference" id="cite_ref-kall_16-8"><a href="#cite_note-kall-16">[16]</a></sup>.
</p><p>Была возможна работа с таблицами трёх размеров: 254 строки и 63 столбца (для компьютеров с малым объёмом памяти); 2000 строк и 127 столбцов; 9999 строк и 255 столбцов (больше, чем у <a href="/wiki/Lotus_1-2-3" title="Lotus 1-2-3">Lotus 1-2-3</a>, в котором максимальная длина таблицы составляла 8192 строки)<sup class="reference" id="cite_ref-kall_16-9"><a href="#cite_note-kall-16">[16]</a></sup>. Увеличение функциональности потребовало увеличения занимаемого программой объёма <a href="/wiki/%D0%9E%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D0%B0%D1%8F_%D0%BF%D0%B0%D0%BC%D1%8F%D1%82%D1%8C" title="Оперативная память">оперативной памяти</a>: вместе с операционной системой она занимала до 190 килобайт ОЗУ. Для работы с большими таблицами, созданными в SuperCalc 3, использовался режим <a href="/wiki/Overlay_(%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5)" title="Overlay (программирование)">оверлея</a>, в котором отдельные части программы подгружались в память по мере необходимости; в таком режиме программа занимала 96 килобайт ОЗУ (как и SuperCalc 3), но работала медленнее<sup class="reference" id="cite_ref-kall_16-10"><a href="#cite_note-kall-16">[16]</a></sup>.
</p><p>Производительность SuperCalc 4 была примерно на одном уровне с <a href="/wiki/Lotus_1-2-3" title="Lotus 1-2-3">Lotus 1-2-3</a>: загрузка таблиц выполнялась медленнее, но скорость вычислений и отзывчивость интерфейса при прокрутке были выше<sup class="reference" id="cite_ref-kall_16-11"><a href="#cite_note-kall-16">[16]</a></sup>. При этом не поддерживался минимальный пересчёт и фоновой пересчёт<sup class="reference" id="cite_ref-btn_17-3"><a href="#cite_note-btn-17">[17]</a></sup>.
</p><p>С программой поставлялись объёмное руководство пользователя, краткое введение в основные функции программы («10 Minute Guide»), две карточки с подсказками и накладка на клавиатуру с подписями для функциональных клавиш<sup class="reference" id="cite_ref-kall_16-12"><a href="#cite_note-kall-16">[16]</a></sup>. В отличие от предыдущей версии, при переходе в командный режим клавишей <kbd style="border:1px solid #a2a9b1; border-radius:.2em; box-shadow:.1em .2em .2em #c8ccd1; background-color:#f8f9fa; font-family:inherit; font-size:0.85em; background-image:linear-gradient(top, #eaecf0, #f8f9fa, #eaecf0); padding:.1em .3em; white-space:nowrap">/</kbd> показывалась двухстрочная подсказка с доступными командами<sup class="reference" id="cite_ref-sc4fl_19-1"><a href="#cite_note-sc4fl-19">[19]</a></sup>. Обозревателями отмечались также высокая доступность и подробность сообщений об ошибках, что вкупе с наличием подробной документации облегчало работу с программой<sup class="reference" id="cite_ref-kall_16-13"><a href="#cite_note-kall-16">[16]</a></sup>.
</p><p>Данная версия SuperCalc была переведена на русский язык и распространялась в России.

<h3><span class="mw-headline" id="SuperCalc_5">SuperCalc 5</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=SuperCalc&amp;veaction=edit&amp;section=9" title="Редактировать раздел «SuperCalc 5»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=SuperCalc&amp;action=edit&amp;section=9" title="Редактировать код раздела «SuperCalc 5»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Supercalc_5_main_window.png"><img class="mw-file-element" data-file-height="400" data-file-width="640" decoding="async" height="200" src="//upload.wikimedia.org/wikipedia/ru/thumb/c/cf/Supercalc_5_main_window.png/320px-Supercalc_5_main_window.png" srcset="//upload.wikimedia.org/wikipedia/ru/thumb/c/cf/Supercalc_5_main_window.png/480px-Supercalc_5_main_window.png 1.5x, //upload.wikimedia.org/wikipedia/ru/c/cf/Supercalc_5_main_window.png 2x" width="320"/></a><figcaption>Основной экран SuperCalc 5 (версия 5.1А)</figcaption></figure>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Supercalc_5_Line_Chart.png"><img class="mw-file-element" data-file-height="480" data-file-width="640" decoding="async" height="240" src="//upload.wikimedia.org/wikipedia/commons/thumb/7/7d/Supercalc_5_Line_Chart.png/320px-Supercalc_5_Line_Chart.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/7/7d/Supercalc_5_Line_Chart.png/480px-Supercalc_5_Line_Chart.png 1.5x, //upload.wikimedia.org/wikipedia/commons/7/7d/Supercalc_5_Line_Chart.png 2x" width="320"/></a><figcaption>Линейная диаграмма, построенная в SuperCalc 5</figcaption></figure>
<p>Версия SuperCalc 5 была выпущена в <a href="/wiki/1989_%D0%B3%D0%BE%D0%B4" title="1989 год">1989 году</a> и продавалась по цене $495 ($100 для пользователей предыдущих версий; к марту 1991 года цена была снижена до $149<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup>). Она работала под управлением операционной системы DOS версии 3.0 и старше на IBM-совместимых компьютерах начиная с <a href="/wiki/IBM_PC/XT" title="IBM PC/XT">PC/XT</a>. Для запуска требовалось 512 килобайт оперативной памяти. Обозревателями отмечалась сравнительная нетребовательность к вычислительным ресурсам: фактически, SuperCalc 5 можно было пользоваться на любом компьютере<sup class="reference" id="cite_ref-walk_22-0"><a href="#cite_note-walk-22">[22]</a></sup>. Однако для работы с большими таблицами требовалась <a href="/wiki/%D0%A0%D0%B0%D1%81%D1%88%D0%B8%D1%80%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%BF%D0%B0%D0%BC%D1%8F%D1%82%D1%8C" title="Расширенная память">расширенная память</a> (поддерживалось до 32 мегабайт EMS-памяти), так как пустой лист таблицы занимал в памяти 60 килобайт, а свободными из 640 килобайт памяти после запуска оставалось не более 180. Использованием памяти были ограничены также функции по работе с несколькими таблицами и многостраничными таблицами; однако возможно было установление связей с другими таблицами, находящимися на диске<sup class="reference" id="cite_ref-proc_23-0"><a href="#cite_note-proc-23">[23]</a></sup>.
</p><p>В данной версии добавились возможности по одновременной работе с тремя таблицами, поддержка команд <a href="/wiki/Lotus_1-2-3" title="Lotus 1-2-3">Lotus 1-2-3</a>, возможность приостановления пересчёта таблицы и отмены операций, трёхмерные графики и новые функции (в основном ориентированные на работу со строковыми данными). Поддерживалась также интеграция с базами данных (<a href="/wiki/DBase" title="DBase">DBase</a>), а дополнение Silverado добавляло функции работы с реляционными базами данных (при этом встроенные функции могли работать только с базами, целиком помещавшимися в оперативную память)<sup class="reference" id="cite_ref-proc_23-1"><a href="#cite_note-proc-23">[23]</a></sup>. Появилась поддержка <a href="/wiki/%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%86%D0%B0_(%D0%BC%D0%B0%D1%82%D0%B5%D0%BC%D0%B0%D1%82%D0%B8%D0%BA%D0%B0)" title="Матрица (математика)">матричных</a> операций и <a href="/wiki/%D0%A0%D0%B5%D0%B3%D1%80%D0%B5%D1%81%D1%81%D0%B8%D0%BE%D0%BD%D0%BD%D1%8B%D0%B9_%D0%B0%D0%BD%D0%B0%D0%BB%D0%B8%D0%B7" title="Регрессионный анализ">множественного регрессионного анализа</a> (с поддержкой линейных, квадратичных и кубических моделей)<sup class="reference" id="cite_ref-walk_22-1"><a href="#cite_note-walk-22">[22]</a></sup>. Макросы могли использоваться для создания использующих собственные меню приложений, однако средства их <a href="/wiki/%D0%9E%D1%82%D0%BB%D0%B0%D0%B4%D0%BA%D0%B0_%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D1%8B" title="Отладка программы">отладки</a> практически отсутствовали<sup class="reference" id="cite_ref-proc_23-2"><a href="#cite_note-proc-23">[23]</a></sup>. Была возможна сетевая установка и сетевая печать, использовались средства блокирования файлов для предотвращения их порчи при одновременной работе нескольких пользователей<sup class="reference" id="cite_ref-proc_23-3"><a href="#cite_note-proc-23">[23]</a></sup>.
</p><p>В комплекте поставлялись утилиты Sideways (печать таблиц в альбомной ориентации) и Privacy Plus (шифрование файлов)<sup class="reference" id="cite_ref-walk_22-2"><a href="#cite_note-walk-22">[22]</a></sup>. Построенные в SuperCalc 5 графики можно было отослать по <a href="/wiki/%D0%9C%D0%BE%D0%B4%D0%B5%D0%BC" title="Модем">модему</a> в фирму <a class="mw-redirect" href="/wiki/Computer_Associates" title="Computer Associates">Computer Associates</a>, и получить на следующий рабочий день распечатанные слайды или прозрачки<sup class="reference" id="cite_ref-proc_23-4"><a href="#cite_note-proc-23">[23]</a></sup>. Документация включала в себя подробное руководство по использованию, буклет для быстрого начала работы и буклет с кратким описанием команд.
</p><p>По скорости работы SuperCalc 5 несколько уступал конкурентам. В то же время в целом данная версия была оценена достаточно высоко, несмотря на скорость работы и отсутствие графического интерфейса<sup class="reference" id="cite_ref-walk_22-3"><a href="#cite_note-walk-22">[22]</a></sup>. Недостатком являлось отсутствие предварительного просмотра печати, что приводило к тому, что для получения желаемого печатного результата (в частности, разрывов страниц в нужных местах) приходилось прибегать к методу проб и ошибок<sup class="reference" id="cite_ref-proc_23-5"><a href="#cite_note-proc-23">[23]</a></sup>. Также отмечалось, что многие команды (например, перемещение и копирование диапазона) полноценно не работали с многостраничными таблицами<sup class="reference" id="cite_ref-proc_23-6"><a href="#cite_note-proc-23">[23]</a></sup>.
</p><p>В начале 1990 года было выпущено обновление Revision C, которое на 15—25 килобайт уменьшило объём занимаемой памяти, улучшило производительность и совместимость с <a href="/wiki/Lotus_1-2-3" title="Lotus 1-2-3">Lotus 1-2-3</a><sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup>.
</p><p>Данная версия SuperCalc была переведена на русский язык и распространялась в России.

<p>CA-SuperCalc 5.1 был выпущен в 1991 году и продавался по цене $149<sup class="reference" id="cite_ref-lowl_26-0"><a href="#cite_note-lowl-26">[26]</a></sup>. Нововведения включали возможность применения команд из меню к многостраничным таблицам, увеличение скорости работы, дополнительные цветовые схемы и возможность работы со 132-колоночным дисплеем<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup><sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup>.
</p>
<h4><span class="mw-headline" id="CA-SuperCalc_5.5">CA-SuperCalc 5.5</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=SuperCalc&amp;veaction=edit&amp;section=11" title="Редактировать раздел «CA-SuperCalc 5.5»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=SuperCalc&amp;action=edit&amp;section=11" title="Редактировать код раздела «CA-SuperCalc 5.5»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h4>
<p>CA-SuperCalc 5.5 был выпущен в 1992 году и продавался по цене $149<sup class="reference" id="cite_ref-lowl_26-1"><a href="#cite_note-lowl-26">[26]</a></sup>. Максимальный размер электронной таблицы составлял 9999 строк, 255 столбцов и 255 страниц. Одновременно можно было открыть до 255 таблиц<sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[29]</a></sup>.
</p><p>Данная версия принесла с собой поддержку <a href="/wiki/%D0%9A%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%BC%D1%8B%D1%88%D1%8C" title="Компьютерная мышь">мыши</a>, а также настраиваемой <a href="/wiki/%D0%9F%D0%B0%D0%BD%D0%B5%D0%BB%D1%8C_%D0%B8%D0%BD%D1%81%D1%82%D1%80%D1%83%D0%BC%D0%B5%D0%BD%D1%82%D0%BE%D0%B2" title="Панель инструментов">панели инструментов</a>. Другие улучшения были связаны с функциями печати и оформления. В частности, был добавлен предварительный просмотр печати с возможностью увеличения фрагмента. В комплекте также поставлялись шрифты производства компании <a class="extiw" href="https://en.wikipedia.org/wiki/Bitstream_Inc." title="en:Bitstream Inc."><span title="Bitstream Inc. — версия статьи «Bitstream Inc.» на английском языке">Bitstream</span></a><span class="noprint" style="font-style:normal; font-weight:normal;"> <span style="display:inline-block; font-size:80%; position:relative;"><span style="font-size:117.6%; margin-left:-0.43em; position:relative; top:0.28em;"> <small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></span><span class="link-ru metadata" style="margin-left:-0.43em; position:absolute; left:0; z-index:1; margin-top:-0.35em;"><span style="visibility:hidden; margin-right:0.099em;"> (</span><span data-interwiki-article="Bitstream Inc." data-interwiki-lang="en"><a class="new" href="/w/index.php?title=Bitstream_Inc.&amp;action=edit&amp;redlink=1" title="Bitstream Inc. (страница отсутствует)">рус.</a></span></span></span></span><sup class="reference" id="cite_ref-lowl_26-2"><a href="#cite_note-lowl-26">[26]</a></sup>.
</p>
<h3><span id="CA-SuperCalc_.D0.B4.D0.BB.D1.8F_Windows"></span><span class="mw-headline" id="CA-SuperCalc_для_Windows">CA-SuperCalc для Windows</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=SuperCalc&amp;veaction=edit&amp;section=12" title="Редактировать раздел «CA-SuperCalc для Windows»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=SuperCalc&amp;action=edit&amp;section=12" title="Редактировать код раздела «CA-SuperCalc для Windows»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>

<p>CA-SuperCalc для Windows был выпущен в 1994 году. Среди основных новшеств данной версии (помимо использования графического интерфейса пользователя Windows) выступили заимствованные из другого продукта Computer Associates, <a class="new" href="/w/index.php?title=CA-Compete&amp;action=edit&amp;redlink=1" title="CA-Compete (страница отсутствует)">CA-Compete</a>, функции многомерного анализа данных с использованием <a href="/wiki/%D0%A1%D0%B2%D0%BE%D0%B4%D0%BD%D0%B0%D1%8F_%D1%82%D0%B0%D0%B1%D0%BB%D0%B8%D1%86%D0%B0" title="Сводная таблица">сводных таблиц</a><sup class="reference" id="cite_ref-30"><a href="#cite_note-30">[30]</a></sup>.
</p>
<h3><span id="CA-SuperCalc.2FMF"></span><span class="mw-headline" id="CA-SuperCalc/MF">CA-SuperCalc/MF</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=SuperCalc&amp;veaction=edit&amp;section=13" title="Редактировать раздел «CA-SuperCalc/MF»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=SuperCalc&amp;action=edit&amp;section=13" title="Редактировать код раздела «CA-SuperCalc/MF»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Под торговой маркой CA-SuperCalc/MF стал продаваться табличный процессор <a class="new" href="/w/index.php?title=MegaCalc&amp;action=edit&amp;redlink=1" title="MegaCalc (страница отсутствует)">MegaCalc</a>, разработанный <a class="new" href="/w/index.php?title=Mega_Group&amp;action=edit&amp;redlink=1" title="Mega Group (страница отсутствует)">Mega Group</a> и работавший на <a href="/wiki/%D0%9C%D0%B5%D0%B9%D0%BD%D1%84%D1%80%D0%B5%D0%B9%D0%BC" title="Мейнфрейм">мейнфреймах</a> <a href="/wiki/IBM" title="IBM">IBM</a>, после приобретения этой компании <a class="mw-redirect" href="/wiki/Computer_Associates" title="Computer Associates">Computer Associates</a>. Переименование произошло в 1987 году. Основным отличием от версий для микрокомпьютеров являлись функции прямого доступа к базам данным (в частности, <a href="/wiki/DB2" title="DB2">DB2</a>). CA-SuperCalc/MF продавался по цене от $9600 до $44 000 в зависимости от конфигурации<sup class="reference" id="cite_ref-31"><a href="#cite_note-31">[31]</a></sup>.
</p>
<h3><span id="CA-SuperCalc.2FVAX"></span><span class="mw-headline" id="CA-SuperCalc/VAX">CA-SuperCalc/VAX</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=SuperCalc&amp;veaction=edit&amp;section=14" title="Редактировать раздел «CA-SuperCalc/VAX»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=SuperCalc&amp;action=edit&amp;section=14" title="Редактировать код раздела «CA-SuperCalc/VAX»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>CA-SuperCalc/VAX предназначалась для системы <a href="/wiki/VAX" title="VAX">VAX</a>/<a class="mw-redirect" href="/wiki/VMS" title="VMS">VMS</a> 5.0 и выше и была портом системы CA-SuperCalc для мэйнфреймов IBM. Версия 5.2 была выпущена в 1991 году и стоила от $1500 для использования на компьютерах <a class="mw-redirect" href="/wiki/VAXstation" title="VAXstation">VAXstation</a> до $50 000 на <a href="/wiki/%D0%9C%D0%B5%D0%B9%D0%BD%D1%84%D1%80%D0%B5%D0%B9%D0%BC" title="Мейнфрейм">мейнфреймах</a> серии <a class="extiw" href="https://en.wikipedia.org/wiki/VAX_9000" title="en:VAX 9000"><span title="VAX 9000 — версия статьи «VAX 9000» на английском языке">VAX 9000</span></a><span class="noprint" style="font-style:normal; font-weight:normal;"> <span style="display:inline-block; font-size:80%; position:relative;"><span style="font-size:117.6%; margin-left:-0.43em; position:relative; top:0.28em;"> <small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></span><span class="link-ru metadata" style="margin-left:-0.43em; position:absolute; left:0; z-index:1; margin-top:-0.35em;"><span style="visibility:hidden; margin-right:0.099em;"> (</span><span data-interwiki-article="VAX 9000" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=VAX_9000&amp;action=edit&amp;redlink=1" title="VAX 9000 (страница отсутствует)">рус.</a></span></span></span></span>. Помимо возможностей версии для IBM-совместимых компьютеров, данная версия имела средства прямого доступа к базам данных VAX/VMS<sup class="reference" id="cite_ref-32"><a href="#cite_note-32">[32]</a></sup>.
</p>
<h3><span id=".D0.9A.D0.BB.D0.BE.D0.BD.D1.8B"></span><span class="mw-headline" id="Клоны">Клоны</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=SuperCalc&amp;veaction=edit&amp;section=15" title="Редактировать раздел «Клоны»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=SuperCalc&amp;action=edit&amp;section=15" title="Редактировать код раздела «Клоны»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Как и у многих других популярных программ, у SuperCalc существовала «адаптированная» для советских персональных компьютеров версия под названием «Абак»; фактически данная программа представляла собой переведённый на русский язык SuperCalc 2. Также значительное сходство с SuperCalc имел пакет «Спринт» для компьютеров <a href="/wiki/%D0%9A%D0%BE%D1%80%D0%B2%D0%B5%D1%82_(%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80)" title="Корвет (компьютер)">«Корвет»</a><sup class="reference" id="cite_ref-judy_33-0"><a href="#cite_note-judy-33">[33]</a></sup>.
</p>
<h2><span id=".D0.9F.D0.BE.D0.BF.D1.83.D0.BB.D1.8F.D1.80.D0.BD.D0.BE.D1.81.D1.82.D1.8C_.D0.B8_.D1.80.D1.8B.D0.BD.D0.BE.D1.87.D0.BD.D0.B0.D1.8F_.D0.B4.D0.BE.D0.BB.D1.8F"></span><span class="mw-headline" id="Популярность_и_рыночная_доля">Популярность и рыночная доля</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=SuperCalc&amp;veaction=edit&amp;section=16" title="Редактировать раздел «Популярность и рыночная доля»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=SuperCalc&amp;action=edit&amp;section=16" title="Редактировать код раздела «Популярность и рыночная доля»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>На момент выпуска SuperCalc доминирующее положение на рынке электронных таблиц занимал <a href="/wiki/VisiCalc" title="VisiCalc">VisiCalc</a>, работавший на большинстве микрокомпьютеров с процессорами <a class="mw-redirect" href="/wiki/6502" title="6502">6502</a> и <a class="mw-redirect" href="/wiki/Z80" title="Z80">Z80</a>. SuperCalc был разработан для новой операционной системы <a href="/wiki/CP/M" title="CP/M">CP/M</a>, работавшей на более производительных компьютерах с процессором <a href="/wiki/Intel_8080" title="Intel 8080">Intel 8080</a>. VisiCalc был портирован на эту операционную систему не сразу, что позволило SuperCalc занять значительную долю на рынке электронных таблиц для <a href="/wiki/CP/M" title="CP/M">CP/M</a>. Кроме того, на многих персональных компьютерах этого периода SuperCalc предустанавливался в рамках <a href="/wiki/OEM" title="OEM">OEM</a>-соглашений<sup class="reference" id="cite_ref-llid_34-0"><a href="#cite_note-llid-34">[34]</a></sup>.
</p><p>Новый виток конкуренции оказался связан с выпуском в 1983 году <a href="/wiki/Lotus_1-2-3" title="Lotus 1-2-3">Lotus 1-2-3</a>. Данный пакет, работавший под управлением операционной системы <a href="/wiki/DOS" title="DOS">DOS</a> на <a href="/wiki/IBM_PC" title="IBM PC">IBM PC</a>-совместимых компьютерах, обладал значительными конкурентными преимуществами: возможность построения диаграмм, пересчёта таблицы с учётом зависимостей между ячейками, а не просто по строкам или по столбцам, использование разреженного хранения данных (что позволяло использовать таблицы большего размера), форматирование и именование ячеек, а также рудиментарные функции работы с базами данных<sup class="reference" id="cite_ref-llid_34-1"><a href="#cite_note-llid-34">[34]</a></sup>.
</p><p>Разработчики SuperCalc сумели к концу 1983 года выпустить версию своей электронной таблицы, копировавшую большинство новых возможностей <a href="/wiki/Lotus_1-2-3" title="Lotus 1-2-3">Lotus 1-2-3</a>, однако к этому моменту большая часть рынка электронных таблиц была уже занята Lotus. SuperCalc, по оценкам экспертов, к середине 1980-х занимал вторую позицию на рынке<sup class="reference" id="cite_ref-pcs87_14-1"><a href="#cite_note-pcs87-14">[14]</a></sup>. Это стало возможным из-за политики корпоративного лицензирования SuperCalc, которая предполагала возможность использования неограниченного числа копий за фиксированную цену, благодаря чему SuperCalc стал корпоративным стандартом в таких крупных компаниях, как <a href="/wiki/General_Electric" title="General Electric">General Electric</a><sup class="reference" id="cite_ref-cwb_35-0"><a href="#cite_note-cwb-35">[35]</a></sup>, <a href="/wiki/Hughes_Aircraft" title="Hughes Aircraft">Hughes Aircraft</a>, <a href="/wiki/Lockheed" title="Lockheed">Lockheed Missile and Space Inc.</a>, <a class="mw-redirect" href="/wiki/Jet_Propulsion_Laboratory" title="Jet Propulsion Laboratory">Jet Propulsion Laboratory</a> и <a class="mw-redirect" href="/wiki/Wells_Fargo_Bank" title="Wells Fargo Bank">Wells Fargo Bank</a><sup class="reference" id="cite_ref-autogenerated1_15-1"><a href="#cite_note-autogenerated1-15">[15]</a></sup>.
</p><p>В дальнейшем гонка версий не привела к существенному изменению ситуации: хотя Supercalc 5 и превосходил по возможностям Lotus 1-2-3 версии 2.2, однако достаточно скоро Lotus была выпущена версия 3.0, не уступавшая конкуренту<sup class="reference" id="cite_ref-llid_34-2"><a href="#cite_note-llid-34">[34]</a></sup>. Экспертами отмечалось, что у пользователей отсутствуют причины для смены поставщика электронных таблиц и перехода на новые версии; возможность такого перехода связывалась с грядущим обновлением доминирующей программно-аппаратной платформы: покупкой нового оборудования с переходом на <a class="mw-redirect" href="/wiki/Microsoft_Windows" title="Microsoft Windows">Windows</a>, <a href="/wiki/OS/2" title="OS/2">OS/2</a>, <a href="/wiki/Unix" title="Unix">Unix</a> или <a href="/wiki/Macintosh" title="Macintosh">Macintosh</a><sup class="reference" id="cite_ref-alsp_36-0"><a href="#cite_note-alsp-36">[36]</a></sup>.
</p><p>Действительно, в начале 1990-х годов ситуация изменилась в связи с ростом популярности операционной среды <a class="mw-redirect" href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a>. Первой электронной таблицей для этой платформы стал <a href="/wiki/Microsoft_Excel" title="Microsoft Excel">Microsoft Excel</a> (1988). Lotus 1-2-3 для Windows был выпущен лишь в 1991 году и, в отличие от Excel, который изначально представлял собой полноценное графическое приложение, учитывающее все особенности новой среды, представлял собой лишь графическую оболочку для прежней версии системы. Техническое превосходство Excel и опережение конкурентов при переходе на графический интерфейс пользователя привели к тому, что его рыночная доля к 1994 году превысила 60 %<sup class="reference" id="cite_ref-37"><a href="#cite_note-37">[37]</a></sup>. В таких условиях SuperCalc быстро терял позицию на рынке, так как его версия для Windows была выпущена лишь в 1994 году, а табличные процессоры для DOS уже не отвечали требованиям времени. После выпуска первой версии для Windows разработка SuperCalc была прекращена.
</p>
<h2><span id=".D0.A1.D0.BE.D0.B2.D0.BC.D0.B5.D1.81.D1.82.D0.B8.D0.BC.D0.BE.D1.81.D1.82.D1.8C_.D1.84.D0.BE.D1.80.D0.BC.D0.B0.D1.82.D0.BE.D0.B2"></span><span class="mw-headline" id="Совместимость_форматов">Совместимость форматов</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=SuperCalc&amp;veaction=edit&amp;section=17" title="Редактировать раздел «Совместимость форматов»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=SuperCalc&amp;action=edit&amp;section=17" title="Редактировать код раздела «Совместимость форматов»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>В настоящее время ни один распространённый <a class="mw-redirect" href="/wiki/%D0%A2%D0%B0%D0%B1%D0%BB%D0%B8%D1%87%D0%BD%D1%8B%D0%B9_%D0%BF%D1%80%D0%BE%D1%86%D0%B5%D1%81%D1%81%D0%BE%D1%80" title="Табличный процессор">табличный процессор</a> не имеет средств импорта электронных таблиц SuperCalc. При наличии работающей копии SuperCalc можно сохранить таблицу в форматах WKS (<a href="/wiki/Lotus_1-2-3" title="Lotus 1-2-3">Lotus 1-2-3</a>) или <a href="/wiki/CSV" title="CSV">CSV</a>, которые поддаются преобразованию<sup class="reference" id="cite_ref-38"><a href="#cite_note-38">[38]</a></sup>.
</p><p>Также просматривать файлы, сохранённые в Supercalc версии 5 и выше, можно в программе <a class="new" href="/w/index.php?title=Quick_View_Plus&amp;action=edit&amp;redlink=1" title="Quick View Plus (страница отсутствует)">Quick View Plus</a><sup class="reference" id="cite_ref-39"><a href="#cite_note-39">[39]</a></sup>.
</p>
