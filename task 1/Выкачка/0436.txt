<h2><span id=".D0.98.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.8F"></span><span class="mw-headline" id="История">История</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B5%D1%80%D0%BF%D1%83%D1%85%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D1%80%D0%B5%D0%BC%D0%BB%D1%8C&amp;veaction=edit&amp;section=1" title="Редактировать раздел «История»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B5%D1%80%D0%BF%D1%83%D1%85%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D1%80%D0%B5%D0%BC%D0%BB%D1%8C&amp;action=edit&amp;section=1" title="Редактировать код раздела «История»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Дубовый кремль в <a href="/wiki/%D0%A1%D0%B5%D1%80%D0%BF%D1%83%D1%85%D0%BE%D0%B2" title="Серпухов">Серпухове</a> был выстроен в <a href="/wiki/1374_%D0%B3%D0%BE%D0%B4" title="1374 год">1374 году</a> после того, как князь <a href="/wiki/%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80_%D0%90%D0%BD%D0%B4%D1%80%D0%B5%D0%B5%D0%B2%D0%B8%D1%87_%D0%A5%D1%80%D0%B0%D0%B1%D1%80%D1%8B%D0%B9" title="Владимир Андреевич Храбрый">Владимир Андреевич</a> сделал Серпухов главным городом удельного <a href="/wiki/%D0%A1%D0%B5%D1%80%D0%BF%D1%83%D1%85%D0%BE%D0%B2%D1%81%D0%BA%D0%BE-%D0%91%D0%BE%D1%80%D0%BE%D0%B2%D1%81%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BD%D1%8F%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Серпуховско-Боровское княжество">княжества</a>. Он был расположен на высоком <a href="/wiki/%D0%9C%D1%8B%D1%81" title="Мыс">мысу</a> при слиянии <a href="/wiki/%D0%A1%D0%B5%D1%80%D0%BF%D0%B5%D0%B9%D0%BA%D0%B0_(%D0%BF%D1%80%D0%B8%D1%82%D0%BE%D0%BA_%D0%9D%D0%B0%D1%80%D1%8B)" title="Серпейка (приток Нары)">речки Серпейки</a> с <a href="/wiki/%D0%9D%D0%B0%D1%80%D0%B0_(%D1%80%D0%B5%D0%BA%D0%B0)" title="Нара (река)">Нарой</a>. При строительстве мыс «с приступа» был отделен от Ильинской горы «перекопом», а затем подрезан и подсыпан глиной и песком. Из «перекопа» в кремль был устроен единственный въезд в виде крутого <a href="/wiki/%D0%9F%D0%B0%D0%BD%D0%B4%D1%83%D1%81" title="Пандус">пандуса</a>, следующего очертанию вала (второй въезд, с юга, сделан много позднее, в начале <a href="/wiki/XIX_%D0%B2%D0%B5%D0%BA" title="XIX век">XIX века</a>).
</p><p>После <a href="/wiki/1507_%D0%B3%D0%BE%D0%B4" title="1507 год">1507 года</a>, когда <a href="/wiki/%D0%9A%D1%80%D1%8B%D0%BC%D1%81%D0%BA%D0%BE%D0%B5_%D1%85%D0%B0%D0%BD%D1%81%D1%82%D0%B2%D0%BE" title="Крымское ханство">Крымское ханство</a> становится постоянным и непримиримым врагом <a href="/wiki/%D0%A0%D1%83%D1%81%D1%8C" title="Русь">Руси</a>, на берегах <a href="/wiki/%D0%9E%D0%BA%D0%B0" title="Ока">Оки</a> <a href="/wiki/%D0%92%D0%B0%D1%81%D0%B8%D0%BB%D0%B8%D0%B9_III" title="Василий III">Василий III</a> создает мощную линию обороны, опирающуюся на окские города-крепости. Центром этой оборонительной системы был Серпухов, стоящий на кратчайшей дороге из <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%B2%D0%B0" title="Москва">Москвы</a> в <a href="/wiki/%D0%A2%D1%83%D0%BB%D0%B0" title="Тула">Тулу</a>. Кроме того, недалеко от Серпухова находился и знаменитый <a class="new" href="/w/index.php?title=%D0%A1%D0%B5%D0%BD%D0%BA%D0%B8%D0%BD_%D0%BF%D0%B5%D1%80%D0%B5%D0%B2%D0%BE%D0%B7&amp;action=edit&amp;redlink=1" title="Сенкин перевоз (страница отсутствует)">Сенкин перевоз</a>, через который <a href="/wiki/%D0%A2%D0%B0%D1%82%D0%B0%D1%80%D1%8B" title="Татары">татары</a> чаще всего «лезли Оку».
</p><p>Белокаменный кремль появился в <a href="/wiki/1556_%D0%B3%D0%BE%D0%B4" title="1556 год">1556 году</a>. Строители каменных стен и башен, постепенно заменяя деревянные укрепления, повторили конфигурацию дубового кремля. Новая <a href="/wiki/%D0%A6%D0%B8%D1%82%D0%B0%D0%B4%D0%B5%D0%BB%D1%8C" title="Цитадель">крепость-цитадель</a>, нисколько не нарушив <a href="/wiki/%D0%90%D1%80%D1%85%D0%B8%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D0%BD%D1%8B%D0%B9_%D0%B0%D0%BD%D1%81%D0%B0%D0%BC%D0%B1%D0%BB%D1%8C" title="Архитектурный ансамбль">ансамбль</a> древнего Серпухова, внесла в городской пейзаж две новые краски: ослепительную белизну стен и золотистую <a href="/wiki/%D0%9E%D1%85%D1%80%D0%B0" title="Охра">охру</a> <a href="/wiki/%D0%A2%D1%91%D1%81" title="Тёс">тёсовых</a> башенных <a href="/wiki/%D0%9A%D1%80%D0%BE%D0%B2%D0%BB%D1%8F" title="Кровля">кровель</a>.
</p><p>Возведением каменного Серпуховского кремля укрепление города не закончилось. Одновременно с ним строится деревянный <a class="mw-redirect" href="/wiki/%D0%9E%D1%81%D1%82%D1%80%D0%BE%D0%B3_(%D1%83%D0%BA%D1%80%D0%B5%D0%BF%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5)" title="Острог (укрепление)">острог</a> для защиты <a href="/wiki/%D0%9F%D0%BE%D1%81%D0%B0%D0%B4" title="Посад">посада</a> с городским торгом. Общая протяженность <a class="mw-redirect" href="/wiki/%D0%9E%D1%81%D1%82%D1%80%D0%BE%D0%B3_(%D1%83%D0%BA%D1%80%D0%B5%D0%BF%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5)" title="Острог (укрепление)">острога</a> составляла более тысячи метров.
</p><p>В <a href="/wiki/%D0%A1%D0%BC%D1%83%D1%82%D0%BD%D0%BE%D0%B5_%D0%B2%D1%80%D0%B5%D0%BC%D1%8F" title="Смутное время">Смутное время</a> охраняемый городом-крепостью стратегический рубеж сделался рубежом смуты и мятежа, отделившим бурлящие, охваченные восстанием южные районы России от примыкающих к столице центральных <a href="/wiki/%D0%A3%D0%B5%D0%B7%D0%B4%D1%8B_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B8" title="Уезды России">уездов</a> государства. Окруженный многочисленными сторонниками <a href="/wiki/%D0%A1%D0%B0%D0%BC%D0%BE%D0%B7%D0%B2%D0%B0%D0%BD%D0%B5%D1%86" title="Самозванец">самозванец</a> принимал в Серпухове виднейших московских <a class="mw-redirect" href="/wiki/%D0%91%D0%BE%D1%8F%D1%80%D0%B8%D0%BD" title="Боярин">бояр</a>, в открытые жителями города ворота неприступной белокаменной крепости вступали отряды <a href="/wiki/%D0%91%D0%BE%D0%BB%D0%BE%D1%82%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2,_%D0%98%D0%B2%D0%B0%D0%BD_%D0%98%D1%81%D0%B0%D0%B5%D0%B2%D0%B8%D1%87" title="Болотников, Иван Исаевич">Болотникова</a>, воеводы царя <a class="mw-redirect" href="/wiki/%D0%A8%D1%83%D0%B9%D1%81%D0%BA%D0%B8%D0%B9,_%D0%92%D0%B0%D1%81%D0%B8%D0%BB%D0%B8%D0%B9_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87" title="Шуйский, Василий Иванович">Василия Шуйского</a> проводили смотр собравшейся в «Серпуховском походе» военной силы… В <a href="/wiki/1609_%D0%B3%D0%BE%D0%B4" title="1609 год">1609 году</a> Серпухов штурмуют поляки под предводительством пана <a class="new" href="/w/index.php?title=%D0%9C%D0%BB%D0%BE%D1%86%D0%BA%D0%B8%D0%B9&amp;action=edit&amp;redlink=1" title="Млоцкий (страница отсутствует)">Млоцкого</a>; они сожгут <a href="/wiki/%D0%9F%D0%BE%D1%81%D0%B0%D0%B4" title="Посад">посад</a>, но кремль взять так и не смогут. Попытается захватить город в <a href="/wiki/1618_%D0%B3%D0%BE%D0%B4" title="1618 год">1618 году</a> и гетман <a href="/wiki/%D0%A1%D0%B0%D0%B3%D0%B0%D0%B9%D0%B4%D0%B0%D1%87%D0%BD%D1%8B%D0%B9,_%D0%9F%D1%91%D1%82%D1%80_%D0%9A%D0%BE%D0%BD%D0%BE%D0%BD%D0%BE%D0%B2%D0%B8%D1%87" title="Сагайдачный, Пётр Кононович">Сагайдачный</a>, но дальше <a href="/wiki/%D0%9F%D0%BE%D1%81%D0%B0%D0%B4" title="Посад">посада</a> прорваться ему тоже не удастся.
</p><p>В середине <a href="/wiki/XVII_%D0%B2%D0%B5%D0%BA" title="XVII век">XVII века</a> возвели последнее оборонное сооружение Серпухова — новоприимочный <a href="/wiki/%D0%9E%D1%81%D1%82%D1%80%D0%BE%D0%B3" title="Острог">острог</a>, защитивший Подол города. Думается, что главной целью этого строительства была защита двух высоких холмов — так называемых Воскресенской и Афанасьевской гор, с которых можно было вести прямой артиллерийский обстрел кремля.
</p><p>В <a href="/wiki/1633_%D0%B3%D0%BE%D0%B4" title="1633 год">1633 году</a>, когда крымский царевич Мубарак-Герай грабил левобережные <a href="/wiki/%D0%A3%D0%B5%D0%B7%D0%B4%D1%8B_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B8" title="Уезды России">уезды</a> по <a href="/wiki/%D0%9E%D0%BA%D0%B0" title="Ока">Оке</a>, жители Серпухова последний раз затворялись в каменной крепости. С тех пор уже не неприятельское оружие, а пожары были для города самым грозным и неотвратимым бедствием. Огонь мгновенно распространялся по сплошь деревянному городу, уничтожая и жилые дворы, и деревянные постройки серпуховских укреплений, и верхи кремлёвских башен и стен. После пожара <a href="/wiki/1645_%D0%B3%D0%BE%D0%B4" title="1645 год">1645 года</a> починку кремля производил «горододелец, думный дьяк» Афанасий Осипович Прончищев. Но тем не менее в <a href="/wiki/1650_%D0%B3%D0%BE%D0%B4" title="1650 год">1650 году</a> воевода Андрей Оленин, осматривавший кремль, нашёл его в плохом состоянии: «Башни не покрыты, тайники утеряны». Сильный пожар <a href="/wiki/1669_%D0%B3%D0%BE%D0%B4" title="1669 год">1669 года</a> был основной причиной начала разрушения памятника. Сметный список <a href="/wiki/1681_%D0%B3%D0%BE%D0%B4" title="1681 год">1681 года</a> уже говорит о частых обвалах башен.
</p>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Serpukhov_Kremlin_1.jpg"><img class="mw-file-element" data-file-height="460" data-file-width="737" decoding="async" height="156" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/6b/Serpukhov_Kremlin_1.jpg/250px-Serpukhov_Kremlin_1.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/6b/Serpukhov_Kremlin_1.jpg/375px-Serpukhov_Kremlin_1.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/6b/Serpukhov_Kremlin_1.jpg/500px-Serpukhov_Kremlin_1.jpg 2x" width="250"/></a><figcaption>Вид на Серпуховский кремль. Начало XX века</figcaption></figure>
<p>К концу <a href="/wiki/XVII_%D0%B2%D0%B5%D0%BA" title="XVII век">XVII века</a> Серпухов окончательно теряет своё военное значение. Кремль и <a href="/wiki/%D0%9E%D1%81%D1%82%D1%80%D0%BE%D0%B3" title="Острог">остроги</a> не ремонтируются. Лишь в <a href="/wiki/1708_%D0%B3%D0%BE%D0%B4" title="1708 год">1708 году</a>, опасаясь наступления <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%BB_XII" title="Карл XII">Карла XII</a> на <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%B2%D0%B0" title="Москва">Москву</a>, <a href="/wiki/%D0%9F%D1%91%D1%82%D1%80_I" title="Пётр I">Пётр I</a> вспоминает о Серпухове и приказывает по возможности «укрепить и палисадировать» город. Но обстановка, как известно, вскоре переменилась, и надобность в ремонте серпуховских укреплений отпала. Их состояние постоянно беспокоит городской <a href="/wiki/%D0%9C%D0%B0%D0%B3%D0%B8%D1%81%D1%82%D1%80%D0%B0%D1%82" title="Магистрат">магистрат</a>, так как подгнившие деревянные башни грозят обвалом, а в кремле «камни немалым числом валятся». Потому в <a href="/wiki/1767_%D0%B3%D0%BE%D0%B4" title="1767 год">1767 году</a> было принято решение «о разрытии городового вала» и разборке оставшихся деревянных стен. Вероятно, для этих работ и был составлен первый масштабный план всех оборонительных сооружений города, хранящийся в Военно-историческом архиве. Этот документ служит пока единственным источником сведений об устройстве белокаменных укреплений русского города-крепости.
</p><p>Руины Серпуховского кремля обратили на себя внимание составителей «Атласа к материалам для статистики Российской империи», вышедшего в <a class="mw-redirect" href="/wiki/%D0%9F%D0%B5%D1%82%D0%B5%D1%80%D0%B1%D1%83%D1%80%D0%B3" title="Петербург">Петербурге</a> в <a href="/wiki/1839_%D0%B3%D0%BE%D0%B4" title="1839 год">1839 году</a>. На помещенных в нём двух <a href="/wiki/%D0%9B%D0%B8%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D1%8F" title="Литография">литографиях</a> с изображением Серпухова отмечена лишь одна юго-западная полубашня, а стены показаны с большими утратами и разрушениями. Возможно, такая картина явилась помимо всего прочего ещё и следствием известного указа <a href="/wiki/%D0%9F%D0%B0%D0%B2%D0%B5%D0%BB_I" title="Павел I">Павла I</a>, предписавшего в <a href="/wiki/1797_%D0%B3%D0%BE%D0%B4" title="1797 год">1797 году</a> разобрать обветшавшие крепостные сооружения древних подмосковных городов. И тем не менее в <a class="mw-redirect" href="/wiki/1852" title="1852">1852</a>—<a href="/wiki/1853_%D0%B3%D0%BE%D0%B4" title="1853 год">1853 годах</a> Серпуховский кремль как признанный «памятник древности» был укреплен и фрагментарно реставрирован. Спустя некоторое время (<a class="mw-redirect" href="/wiki/1888" title="1888">1888</a>) Археологическое общество добилось ассигнования денежных средств на дальнейшую реставрацию наиболее полно сохранившихся фрагментов.
</p><p>Зимой <a href="/wiki/1934_%D0%B3%D0%BE%D0%B4" title="1934 год">1934 года</a> кремль был разобран на материал для строительства <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%BC%D0%B5%D1%82%D1%80%D0%BE%D0%BF%D0%BE%D0%BB%D0%B8%D1%82%D0%B5%D0%BD" title="Московский метрополитен">Московского метрополитена</a>. Это было сделано по распоряжению Лазаря Кагановича<sup class="ts-fix-template noprint">[<i><a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%A1%D1%81%D1%8B%D0%BB%D0%BA%D0%B8_%D0%BD%D0%B0_%D0%B8%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8" title="Википедия:Ссылки на источники"><span title="нет источника (19 мая 2021)">источник не указан 1015 дней</span></a></i>]</sup>, согласно которому для строительных работ требовался именно бутовый камень. Поскольку строительство Московского метро было политической акцией, делать все необходимо было очень быстро. Камень решили взять из самого доступного и ближнего места — из Серпухова. Для этого Серпуховский кремль был разобран примерно в течение года. После этого груз был отправлен в Москву. Однако в конечном итоге камень был забракован. И достался не московским стройкам, а другим городам<sup class="ts-fix-template noprint">[<i><a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%A1%D1%81%D1%8B%D0%BB%D0%BA%D0%B8_%D0%BD%D0%B0_%D0%B8%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8" title="Википедия:Ссылки на источники"><span title="нет источника (19 мая 2021)">источник не указан 1015 дней</span></a></i>]</sup>. То есть он просто бесследно разошелся по России. При этом ущерб культурному наследию Серпухова никто не возмещал, поскольку это был государственный объект, а государство само себе ущерб возмещать не посчитало необходимым<sup class="ts-fix-template noprint">[<i><a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%A1%D1%81%D1%8B%D0%BB%D0%BA%D0%B8_%D0%BD%D0%B0_%D0%B8%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8" title="Википедия:Ссылки на источники"><span title="нет источника (22 августа 2017)">источник не указан 2381 день</span></a></i>]</sup>. На кремлёвском холме остались лишь два крохотных фрагмента стен.
</p>

<h2><span id=".D0.90.D1.80.D1.85.D0.B8.D1.82.D0.B5.D0.BA.D1.82.D1.83.D1.80.D0.B0_.D0.BA.D1.80.D0.B5.D0.BC.D0.BB.D1.8F"></span><span class="mw-headline" id="Архитектура_кремля">Архитектура кремля</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B5%D1%80%D0%BF%D1%83%D1%85%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D1%80%D0%B5%D0%BC%D0%BB%D1%8C&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Архитектура кремля»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B5%D1%80%D0%BF%D1%83%D1%85%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D1%80%D0%B5%D0%BC%D0%BB%D1%8C&amp;action=edit&amp;section=2" title="Редактировать код раздела «Архитектура кремля»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id=".D0.A1.D1.82.D0.B5.D0.BD.D1.8B_.D0.B8_.D0.B1.D0.B0.D1.88.D0.BD.D0.B8_.D0.BA.D1.80.D0.B5.D0.BC.D0.BB.D1.8F"></span><span class="mw-headline" id="Стены_и_башни_кремля">Стены и башни кремля</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B5%D1%80%D0%BF%D1%83%D1%85%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D1%80%D0%B5%D0%BC%D0%BB%D1%8C&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Стены и башни кремля»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B5%D1%80%D0%BF%D1%83%D1%85%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D1%80%D0%B5%D0%BC%D0%BB%D1%8C&amp;action=edit&amp;section=3" title="Редактировать код раздела «Стены и башни кремля»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Serpukhov_VTower_dig.jpg"><img class="mw-file-element" data-file-height="615" data-file-width="750" decoding="async" height="205" src="//upload.wikimedia.org/wikipedia/commons/thumb/5/58/Serpukhov_VTower_dig.jpg/250px-Serpukhov_VTower_dig.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/5/58/Serpukhov_VTower_dig.jpg/375px-Serpukhov_VTower_dig.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/5/58/Serpukhov_VTower_dig.jpg/500px-Serpukhov_VTower_dig.jpg 2x" width="250"/></a><figcaption>Фрагмент Высоцкой башни Серпуховского кремля, найденный при раскопках в 2006 году</figcaption></figure>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Serpukhov_Kremlin_Fragment_of_Wall.jpg"><img class="mw-file-element" data-file-height="1944" data-file-width="2592" decoding="async" height="188" src="//upload.wikimedia.org/wikipedia/commons/thumb/2/2e/Serpukhov_Kremlin_Fragment_of_Wall.jpg/250px-Serpukhov_Kremlin_Fragment_of_Wall.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/2/2e/Serpukhov_Kremlin_Fragment_of_Wall.jpg/375px-Serpukhov_Kremlin_Fragment_of_Wall.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/2/2e/Serpukhov_Kremlin_Fragment_of_Wall.jpg/500px-Serpukhov_Kremlin_Fragment_of_Wall.jpg 2x" width="250"/></a><figcaption>Фрагмент стены Серпуховского кремля, <a href="/wiki/2009_%D0%B3%D0%BE%D0%B4" title="2009 год">2009 год</a></figcaption></figure>
<p>Серпуховский кремль, не имеющий прямых аналогий в практике строительства каменных укреплений на протяжении всего <a href="/wiki/XVI_%D0%B2%D0%B5%D0%BA" title="XVI век">XVI века</a>, — интересный памятник русского крепостного зодчества. Прежде всего уникален его строительный материал: кремль полностью выстроен из белого камня. В других крепостях того времени белый камень уже играл вспомогательную роль. Этот материал ещё во второй половине <a href="/wiki/XV_%D0%B2%D0%B5%D0%BA" title="XV век">XV века</a> почти повсеместно сменяется кирпичом, но в Серпухове, где белый камень находится буквально под рукой, строительство из него продолжается вплоть до середины <a href="/wiki/XVII_%D0%B2%D0%B5%D0%BA" title="XVII век">XVII столетия</a>. К тому же в городе были собственные кадры профессиональных каменщиков. «Камень тешот» — написано про жителей Ивановской <a href="/wiki/%D0%A1%D0%BB%D0%BE%D0%B1%D0%BE%D0%B4%D0%B0" title="Слобода">слободы</a> в сотной (переписной) книге <a href="/wiki/1552_%D0%B3%D0%BE%D0%B4" title="1552 год">1552 года</a>, которая составлялась, видимо, для того, чтобы выявить экономические возможности города накануне строительства каменной крепости.
</p><p>Боевые стены охватили территорию кремля, следуя очертаниям естественного холма, а не линиям архитектурного плана, как это было характерно для крепостного зодчества времени <a class="mw-redirect" href="/wiki/%D0%98%D0%B2%D0%B0%D0%BD_IV" title="Иван IV">Ивана Грозного</a>. План кремля представлял собой неправильный треугольник с периметром в 933 метра. В формах укреплений сильно сказалось начавшееся широкое применение <a href="/wiki/%D0%90%D1%80%D1%82%D0%B8%D0%BB%D0%BB%D0%B5%D1%80%D0%B8%D1%8F" title="Артиллерия">артиллерии</a>, и в первую очередь это выразилось в конструкции стен. При значительной толщине (3,5 метра) они не столь высоки (6,5-8,5 метра). Широкая боевая <a href="/wiki/%D0%93%D0%B0%D0%BB%D0%B5%D1%80%D0%B5%D1%8F" title="Галерея">галерея</a> рассчитана на удобное размещение пушек и <a href="/wiki/%D0%9F%D0%B8%D1%89%D0%B0%D0%BB%D1%8C" title="Пищаль">пищалей</a>. Башни располагались в местах наиболее значительных изломов стены, что исключало возможность существования необстреливаемых пространств. Полубашни (их было три) — ещё одна интересная особенность Серпуховского кремля. Они представляли собой П-образные в плане выступы стен и были выше обычных <a href="/wiki/%D0%9F%D1%80%D1%8F%D1%81%D0%BB%D0%BE_(%D0%B0%D1%80%D1%85%D0%B8%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D0%B0)" title="Прясло (архитектура)">прясел</a>. Полубашни сооружались вблизи основных башен, на изломах стены, в тех случаях, когда промежуточное прясло получалось сравнительно коротким.
</p><p>Особенно значительное влияние применение <a href="/wiki/%D0%90%D1%80%D1%82%D0%B8%D0%BB%D0%BB%D0%B5%D1%80%D0%B8%D1%8F" title="Артиллерия">артиллерии</a> оказало на решение крепостных въездов. Если крепости <a href="/wiki/XIV_%D0%B2%D0%B5%D0%BA" title="XIV век">XIV</a>-<a href="/wiki/XV_%D0%B2%D0%B5%D0%BA" title="XV век">XV веков</a>, незнакомые с «огненной дуэлью», имели сложные въезды, состоящие из отводных <a class="new" href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B5%D0%BB%D1%8C%D0%BD%D0%B8%D1%86%D0%B0&amp;action=edit&amp;redlink=1" title="Стрельница (страница отсутствует)">стрельниц</a>, <a href="/wiki/%D0%97%D0%B0%D1%85%D0%B0%D0%B1" title="Захаб">захабов</a> и т. п., то в укреплениях <a href="/wiki/XVI_%D0%B2%D0%B5%D0%BA" title="XVI век">XVI столетия</a> необходимость в них отпала — появляются простые проездные башни. Тем более неожиданной по мастерству и изобретательности решения предстает система проезда Серпуховского кремля. Этот важнейший оборонительный узел представлял собой видоизмененный <a class="new" href="/w/index.php?title=%D0%A0%D1%83%D0%BA%D0%B0%D0%B2%D1%87%D0%B0%D1%82%D1%8B%D0%B9_%D0%B7%D0%B0%D1%85%D0%B0%D0%B1&amp;action=edit&amp;redlink=1" title="Рукавчатый захаб (страница отсутствует)">рукавчатый захаб</a>. Существующий въезд в кремль подводил к Спасским воротам, устроенным в стене между восьмигранной Караульной башней и отводной полубашней. Ворота вели в узкий длинный карман, образованный двумя параллельными стенами, одна из которых была отводная. И, наконец, чтобы попасть на территорию самого кремля, надо было миновать Большую проезжую башню, имевшую вдобавок изгибающееся под прямым углом колено проезда. Отводная часть контролировала въезд и в то же время не мешала стрельбе, так как высота её была меньше высоты основных стен. В ней был предусмотрен даже <a href="/wiki/%D0%A2%D0%B0%D0%B9%D0%BD%D0%B8%D0%BA_(%D1%84%D0%BE%D1%80%D1%82%D0%B8%D1%84%D0%B8%D0%BA%D0%B0%D1%86%D0%B8%D1%8F)" title="Тайник (фортификация)">тайный ход</a>, через который осажденные могли неожиданно ударить по врагу.
</p><p>Появление такого проезда, характерного для крепостей более раннего времени, конечно, нельзя объяснить только тем, что новый каменный кремль повторял планировку старого, деревянного. Создатели Серпуховского кремля были хорошо знакомы с вооружением <a href="/wiki/%D0%9A%D1%80%D1%8B%D0%BC%D1%81%D0%BA%D0%B8%D0%B5_%D1%82%D0%B0%D1%82%D0%B0%D1%80%D1%8B" title="Крымские татары">крымских татар</a>, не имевших собственной осадной <a href="/wiki/%D0%90%D1%80%D1%82%D0%B8%D0%BB%D0%BB%D0%B5%D1%80%D0%B8%D1%8F" title="Артиллерия">артиллерии</a>. Правда, турецкий <a href="/wiki/%D0%A1%D1%83%D0%BB%D1%82%D0%B0%D0%BD" title="Султан">султан</a> посылал им <a href="/wiki/%D0%AF%D0%BD%D1%8B%D1%87%D0%B0%D1%80%D1%8B" title="Янычары">янычар</a> с пушками, но их было очень мало, и при осаде крепостей применялись в основном полевые или устаревшие орудия. Поэтому <a href="/wiki/%D0%9A%D1%80%D1%8B%D0%BC%D1%81%D0%BA%D0%B8%D0%B5_%D1%82%D0%B0%D1%82%D0%B0%D1%80%D1%8B" title="Крымские татары">крымские татары</a> не могли рассчитывать на прорыв обороны крепости в каком-либо другом месте, кроме ворот. Серпуховский кремль полностью исключал и эту возможность.
</p><p>Если подойти к самой кромке кремлёвского холма, то можно обнаружить узкую дорожку, огибающую всю его территорию по тому же месту, где проходила боевая стена. Небольшие площадки, встречающиеся по пути, напоминают о некогда возвышавшихся здесь башнях. Площадка, расположенная над самым въездом, отмечает место восьмигранной Караульной башни, в которой размещалась пороховая палата. За ней, напротив <a href="/wiki/%D0%9F%D0%BE%D1%81%D0%B0%D0%B4" title="Посад">посада</a>, стояла Северная полубашня, фиксировавшая значительный излом стены. Между Северной полубашней и следовавшей за ней круглой Воскресенской башней прясло стены было небольшое. От него сохранился фрагмент с узким проходом, а в нём — устройство для подъёма решетки-<a href="/wiki/%D0%93%D0%B5%D1%80%D1%81%D0%B0_(%D0%B0%D1%80%D1%85%D0%B8%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D0%B0)" title="Герса (архитектура)">герсы</a>. Стены с внешней и внутренней стороны были выложены из массивных блоков белого камня, а середина забучена мелким постелистым камнем на известковом растворе. Изнутри стена была расчленена большими арочными нишами с каморами. Снаружи стена имела машикули — специальные навесные бойницы для обстрела неприятеля, прорвавшегося непосредственно к её подножию.
</p><p>Следующий участок стены от Воскресенской башни шёл параллельно <a href="/wiki/%D0%A1%D0%B5%D1%80%D0%BF%D0%B5%D0%B9%D0%BA%D0%B0_(%D0%BF%D1%80%D0%B8%D1%82%D0%BE%D0%BA_%D0%9D%D0%B0%D1%80%D1%8B)" title="Серпейка (приток Нары)">Серпейке</a> до круглой небольшой Никольской башни, в которой был устроен деревянный тайник с колодцем для снабжения осажденных водой. От этого прясла также сохранился фрагмент стены. Он был ровно обрублен с юга во время устройства нового въезда, тогда же пристроили несколько поддерживающих стену <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%BD%D1%82%D1%80%D1%84%D0%BE%D1%80%D1%81_(%D0%B0%D1%80%D1%85%D0%B8%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D0%B0)" title="Контрфорс (архитектура)">контрфорсов</a>. С юга, в той части, которой кремль был обращен в сторону <a href="/wiki/%D0%9E%D0%BA%D0%B0" title="Ока">Оки</a>, к <a href="/wiki/%D0%9C%D0%BE%D0%BD%D0%B0%D1%81%D1%82%D1%8B%D1%80%D1%8C" title="Монастырь">монастырям</a>, боевая стена была усилена (и украшена!) восьмигранной Высоцкой башней. От неё стена шла параллельно <a href="/wiki/%D0%9D%D0%B0%D1%80%D0%B0_(%D1%80%D0%B5%D0%BA%D0%B0)" title="Нара (река)">Наре</a>, а затем через Южную полубашню и Большую проезжую башню подходила к въезду.
</p>
<h3><span id=".D0.A1.D1.82.D1.80.D0.BE.D0.B5.D0.BD.D0.B8.D1.8F_.D0.B2.D0.BD.D1.83.D1.82.D1.80.D0.B8_.D0.BA.D1.80.D0.B5.D0.BC.D0.BB.D1.8F"></span><span class="mw-headline" id="Строения_внутри_кремля">Строения внутри кремля</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B5%D1%80%D0%BF%D1%83%D1%85%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D1%80%D0%B5%D0%BC%D0%BB%D1%8C&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Строения внутри кремля»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B5%D1%80%D0%BF%D1%83%D1%85%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D1%80%D0%B5%D0%BC%D0%BB%D1%8C&amp;action=edit&amp;section=4" title="Редактировать код раздела «Строения внутри кремля»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Serpukhov_Sobornaya_Gora_Trinity_Church_in_2006.jpg"><img class="mw-file-element" data-file-height="554" data-file-width="750" decoding="async" height="185" src="//upload.wikimedia.org/wikipedia/commons/thumb/2/25/Serpukhov_Sobornaya_Gora_Trinity_Church_in_2006.jpg/250px-Serpukhov_Sobornaya_Gora_Trinity_Church_in_2006.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/2/25/Serpukhov_Sobornaya_Gora_Trinity_Church_in_2006.jpg/375px-Serpukhov_Sobornaya_Gora_Trinity_Church_in_2006.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/2/25/Serpukhov_Sobornaya_Gora_Trinity_Church_in_2006.jpg/500px-Serpukhov_Sobornaya_Gora_Trinity_Church_in_2006.jpg 2x" width="250"/></a><figcaption>Вид на Троицкий собор с места бывшей Караульной башни Серпуховского кремля</figcaption></figure>
<p>Из всех построек внутри кремля в настоящее время сохранился <a href="/wiki/%D0%A2%D1%80%D0%BE%D0%B8%D1%86%D0%BA%D0%B8%D0%B9_%D1%81%D0%BE%D0%B1%D0%BE%D1%80_(%D0%A1%D0%B5%D1%80%D0%BF%D1%83%D1%85%D0%BE%D0%B2)" title="Троицкий собор (Серпухов)">Троицкий собор</a> — художественный, ландшафтный и духовный ориентир всего города.
</p><p>На небольшой площади перед собором существовал административный центр города и округи. Отсюда Серпухов, плативший в казну дань медом, управлялся <a href="/wiki/%D0%A2%D0%B8%D1%83%D0%BD" title="Тиун">тиунами</a> и наместниками. В период реформ середины <a href="/wiki/XVI_%D0%B2%D0%B5%D0%BA" title="XVI век">XVI века</a> их сменили выборные <a href="/wiki/%D0%93%D1%83%D0%B1%D0%B0_(%D0%BE%D0%BA%D1%80%D1%83%D0%B3)" title="Губа (округ)">губные</a> старосты и городовые приказчики. К тому же практически постоянно в городе пребывали назначаемые Москвой воеводы, ведавшие военными силами Берега… Между тем не следует представлять себе в центре древнего Серпухова репрезентативные, обширные здания местного управления. В съезжей и <a href="/wiki/%D0%93%D1%83%D0%B1%D0%B0_(%D0%BE%D0%BA%D1%80%D1%83%D0%B3)" title="Губа (округ)">губной избе</a>, воеводском доме, словом, всего в нескольких рубленых домах, с виду мало отличавшихся от обычных городских жилищ, размещалась серпуховская власть, административная и военная.
</p><p>Немного в стороне находился второй кремлёвский храм — церковь <a class="mw-redirect" href="/wiki/%D0%9F%D1%80%D0%B5%D0%BF%D0%BE%D0%B4%D0%BE%D0%B1%D0%BD%D1%8B%D0%B5" title="Преподобные">преподобного</a> <a class="mw-redirect" href="/wiki/%D0%94%D0%BC%D0%B8%D1%82%D1%80%D0%B8%D0%B9_%D0%A1%D0%BE%D0%BB%D1%83%D0%BD%D1%81%D0%BA%D0%B8%D0%B9" title="Дмитрий Солунский">Дмитрия Солунского</a>, упоминаемая с конца <a href="/wiki/XV_%D0%B2%D0%B5%D0%BA" title="XV век">XV столетия</a>.
</p><p>Остальная же территория каменного города была заполнена осадными дворами окрестных монастырей, вотчинников, сел и деревень. Полупустые в мирное время осадные дворы содержались специально поселенными в крепости людьми — дворниками. Тогда как все, что относилось к обороне кремля, находилось в ведении городовых приказчиков, наблюдавших за исправностью «клетей» для <a href="/wiki/%D0%9F%D0%B8%D1%89%D0%B0%D0%BB%D1%8C" title="Пищаль">пищалей</a>, «зелейной казны» с помещениями для ядер и пуль, «котлов селитряных медных», «хлебных житниц».
</p><p>В настоящее время кроме Троицкого собора, фундамента и фрагментов стен кремля на <a class="new" href="/w/index.php?title=%D0%A1%D0%BE%D0%B1%D0%BE%D1%80%D0%BD%D0%B0%D1%8F_%D0%B3%D0%BE%D1%80%D0%B0&amp;action=edit&amp;redlink=1" title="Соборная гора (страница отсутствует)">Соборной горе</a> находятся мемориал воинам, павшим в Великой Отечественной войне, конюшня и несколько частных домов.
</p>
<h2><span id=".D0.A0.D0.B0.D1.81.D0.BA.D0.BE.D0.BF.D0.BA.D0.B8_.D0.BD.D0.B0_.D1.82.D0.B5.D1.80.D1.80.D0.B8.D1.82.D0.BE.D1.80.D0.B8.D0.B8_.D0.BA.D1.80.D0.B5.D0.BC.D0.BB.D1.8F"></span><span class="mw-headline" id="Раскопки_на_территории_кремля">Раскопки на территории кремля</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B5%D1%80%D0%BF%D1%83%D1%85%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D1%80%D0%B5%D0%BC%D0%BB%D1%8C&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Раскопки на территории кремля»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B5%D1%80%D0%BF%D1%83%D1%85%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D1%80%D0%B5%D0%BC%D0%BB%D1%8C&amp;action=edit&amp;section=5" title="Редактировать код раздела «Раскопки на территории кремля»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Serpkreml_raskop.jpg"><img class="mw-file-element" data-file-height="563" data-file-width="750" decoding="async" height="188" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/69/Serpkreml_raskop.jpg/250px-Serpkreml_raskop.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/69/Serpkreml_raskop.jpg/375px-Serpkreml_raskop.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/69/Serpkreml_raskop.jpg/500px-Serpkreml_raskop.jpg 2x" width="250"/></a><figcaption>Раскоп на территории Серпуховского кремля в 2006 году</figcaption></figure>
<p>Первые официальные археологические исследования Серпуховского кремля проводились археологом-краеведом Н. В. Воронковым в 1920-х годах. В 1980-х годах Н. В. Колышницын и А. А. Молчанов раскрывали фундаменты (но не все) древних кремлёвских стен. В 2003 году Институт архитектуры РАН организовал новые раскопки — было изучено около 20 квадратных метров. Участие в раскопках принимал профессор, доктор химических наук А. Л. Александровский. Летом 2006 года площадь раскопок увеличилась больше чем в два раза. Одним из главных инициаторов этих исследований стал Артём Павлихин — научный сотрудник <a class="new" href="/w/index.php?title=%D0%A1%D0%B5%D1%80%D0%BF%D1%83%D1%85%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%B8%D1%81%D1%82%D0%BE%D1%80%D0%B8%D0%BA%D0%BE-%D1%85%D1%83%D0%B4%D0%BE%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%BE%D0%B3%D0%BE_%D0%BC%D1%83%D0%B7%D0%B5%D0%B9&amp;action=edit&amp;redlink=1" title="Серпуховский историко-художественного музей (страница отсутствует)">Серпуховского историко-художественного музея</a>. Возглавили раскопки профессора археологии Г. Н. Белорыбкин и В. В. Ставицкий.
</p><p>Во внутренней части крепости (на участке размером 56 квадратных метров) у исследователей было много находок. Обнаружены остатки Высоцкой башни кремля (<a href="/wiki/1556_%D0%B3%D0%BE%D0%B4" title="1556 год">1556 год</a>). Выяснено, что архитектура Серпуховской крепости имела много общего с северо-западной русской и западноевропейской. Удалось выявить средневековую улицу, которая шла от центра крепости к Высоцкой башне. Вдоль кривой и узкой улочки — впритык друг к другу — дома. Для <a href="/wiki/XIV_%D0%B2%D0%B5%D0%BA" title="XIV век">XIV</a>—<a href="/wiki/XV_%D0%B2%D0%B5%D0%BA" title="XV век">XV веков</a> — это большие дома княжеского подворья. Для <a href="/wiki/XVI_%D0%B2%D0%B5%D0%BA" title="XVI век">XVI</a>—<a href="/wiki/XVII_%D0%B2%D0%B5%D0%BA" title="XVII век">XVII веков</a> — маленькие домики горожан.
</p><p>Среди находок, датированных <a href="/wiki/XVI_%D0%B2%D0%B5%D0%BA" title="XVI век">XVI</a>—<a href="/wiki/XVII_%D0%B2%D0%B5%D0%BA" title="XVII век">XVII веками</a>, имеются серьги, пряжки, перстни, нательные кресты. Интересна медная монета с изображением птицы, а также редкая коллекция предметов вооружения первой половины <a href="/wiki/XVII_%D0%B2%D0%B5%D0%BA" title="XVII век">XVII века</a>: фрагмент сабли, <a class="new" href="/w/index.php?title=%D0%9F%D0%B8%D1%89%D0%B0%D0%BB%D1%8C%D0%BD%D0%BE%D0%B5_%D1%8F%D0%B4%D1%80%D0%BE&amp;action=edit&amp;redlink=1" title="Пищальное ядро (страница отсутствует)">пищальные ядра</a>, пуля, винт от огнестрельного оружия, <a href="/wiki/%D0%91%D0%BE%D0%B5%D0%B2%D0%BE%D0%B9_%D0%BD%D0%BE%D0%B6" title="Боевой нож">боевые ножи</a>, наконечники стрел арбалета и лука. Уникальны фрагменты подошвы кожаного сапога или туфли начала <a href="/wiki/XV_%D0%B2%D0%B5%D0%BA" title="XV век">XV века</a>. Коллекция дополнена керамическими сосудами <a href="/wiki/XIV_%D0%B2%D0%B5%D0%BA" title="XIV век">XIV</a>—<a href="/wiki/XVII_%D0%B2%D0%B5%D0%BA" title="XVII век">XVII веков</a>. Особенно интересной находкой является уникальная печать <a href="/wiki/XVI_%D0%B2%D0%B5%D0%BA" title="XVI век">XVI века</a> с надписью «печать наимнова крепость» и изображением государя на троне и ещё двух людей — одного с занесённой над головой саблей, другого — без оружия, над головами висит полумесяц.
</p><p>На площадке <a href="/wiki/%D0%93%D0%BE%D1%80%D0%BE%D0%B4%D0%B8%D1%89%D0%B5" title="Городище">городища</a> выявлены орудия из кремня — следы пребывания там человека в <a href="/wiki/%D0%9A%D0%B0%D0%BC%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9_%D0%B2%D0%B5%D0%BA" title="Каменный век">каменном веке</a>. В 2006 году археологам стало известно погребение на Соборной горе, совершённое по <a class="new" href="/w/index.php?title=%D0%9E%D0%B1%D1%80%D1%8F%D0%B4_%D1%82%D1%80%D1%83%D0%BF%D0%BE-%D1%81%D0%BE%D0%B6%D0%B6%D0%B5%D0%BD%D0%B8%D1%8F_%D0%BD%D0%B0_%D1%81%D1%82%D0%BE%D1%80%D0%BE%D0%BD%D0%B5&amp;action=edit&amp;redlink=1" title="Обряд трупо-сожжения на стороне (страница отсутствует)">обряду трупо-сожжения на стороне</a>, которое пока что ни к одной известной археологической культуре не отнесено. Исследованиями выявлена застройка городища в <a href="/wiki/XII_%D0%B2%D0%B5%D0%BA" title="XII век">XII</a>—<a href="/wiki/XVII_%D0%B2%D0%B5%D0%BA" title="XVII век">XVII веках</a>, охарактеризованы основные обнаруженные конструкции и предметы.
