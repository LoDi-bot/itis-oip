<p><b>STS-121</b> — <a href="/wiki/%D0%9F%D0%B8%D0%BB%D0%BE%D1%82%D0%B8%D1%80%D1%83%D0%B5%D0%BC%D1%8B%D0%B9_%D0%BA%D0%BE%D1%81%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%BE%D0%BB%D1%91%D1%82" title="Пилотируемый космический полёт">космический полёт</a> MTKK <a href="/wiki/%D0%94%D0%B8%D1%81%D0%BA%D0%B0%D0%B2%D0%B5%D1%80%D0%B8_(%D1%88%D0%B0%D1%82%D1%82%D0%BB)" title="Дискавери (шаттл)">«Дискавери»</a> по <a href="/wiki/%D0%A1%D0%BF%D0%B5%D0%B9%D1%81_%D1%88%D0%B0%D1%82%D1%82%D0%BB" title="Спейс шаттл">программе «Спейс Шаттл»</a>, с 4 по 16 июля <a href="/wiki/2006_%D0%B3%D0%BE%D0%B4" title="2006 год">2006 года</a>, второй испытательный полёт по программе <a class="new" href="/w/index.php?title=%D0%92%D0%BE%D0%B7%D0%B2%D1%80%D0%B0%D1%89%D0%B5%D0%BD%D0%B8%D0%B5_%D0%BA_%D0%BF%D0%BE%D0%BB%D1%91%D1%82%D0%B0%D0%BC_(%D0%BA%D0%BE%D1%81%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B0)&amp;action=edit&amp;redlink=1" title="Возвращение к полётам (космическая программа) (страница отсутствует)">«Возвращение к полётам»</a> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Return To Flight, RTF</span>)<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>.
</p><p>После <a class="mw-redirect" href="/wiki/%D0%9A%D0%B0%D1%82%D0%B0%D1%81%D1%82%D1%80%D0%BE%D1%84%D0%B0_%D0%9A%D0%BE%D0%BB%D1%83%D0%BC%D0%B1%D0%B8%D0%B8" title="Катастрофа Колумбии">катастрофы «Колумбии»</a> в феврале 2003 года, для проверки проведённых усовершенствований и испытания новых процедур, призванных повысить безопасность полётов, специалисты <a href="/wiki/%D0%9D%D0%90%D0%A1%D0%90" title="НАСА">НАСА</a> решили провести два испытательных полёта по программе «Возвращение к полётам».
</p><p>Первоначально STS-121 должен был стать полётом по обслуживанию телескопа <a href="/wiki/%D0%A5%D0%B0%D0%B1%D0%B1%D0%BB_(%D1%82%D0%B5%D0%BB%D0%B5%D1%81%D0%BA%D0%BE%D0%BF)" title="Хаббл (телескоп)">«Хаббл»</a>, для выполнения которого предполагалось использовать шаттл <a href="/wiki/%D0%9A%D0%BE%D0%BB%D1%83%D0%BC%D0%B1%D0%B8%D1%8F_(%D1%88%D0%B0%D1%82%D1%82%D0%BB)" title="Колумбия (шаттл)">«Колумбия»</a>. Поскольку после катастрофы этот полёт был отменён, а номера с STS-115 до STS-120 в расписании НАСА были заняты, для второго старта по программе «Возвращение к полётам» был выбран ближайший свободный номер — STS-121.
</p><p>Первоначально предполагалось выполнить два испытательных полёта: первый — STS-114 с шаттлом <a class="mw-redirect" href="/wiki/%D0%90%D1%82%D0%BB%D0%B0%D0%BD%D1%82%D0%B8%D1%81_(%D1%88%D0%B0%D1%82%D1%82%D0%BB)" title="Атлантис (шаттл)">«Атлантис»</a>, второй — STS-121 c «Дискавери» (предварительно намеченный на сентябрь 2005 года). Однако, когда во время подготовки к STS-114 обнаружились проблемы с шасси «Атлантиса», последовательность полётов шаттлов изменили. 
Первым в рамках программы «Возвращение к полётам» полетел «Дискавери» (<a class="mw-redirect" href="/wiki/%D0%94%D0%B8%D1%81%D0%BA%D0%B0%D0%B2%D0%B5%D1%80%D0%B8_STS-114" title="Дискавери STS-114">Дискавери STS-114</a>, с 26 июля по 9 августа 2005). Затем была ещё одна перестановка: теперь «Атлантис» был назначен в <a class="mw-redirect" href="/wiki/%D0%90%D1%82%D0%BB%D0%B0%D0%BD%D1%82%D0%B8%D1%81_STS-115" title="Атлантис STS-115">STS-115</a>, а шаттл «Дискавери» — в STS-121, как это и планировалось вначале.
</p><p>Как уже говорилось, старт STS-121 был намечен на осень 2005 года. Однако во время первого полёта по программе «Возвращение к полётам» («Дискавери» STS-114) выяснилось, что проблемы с отслаивающейся <a href="/wiki/%D0%A2%D0%B5%D0%BF%D0%BB%D0%BE%D0%B8%D0%B7%D0%BE%D0%BB%D1%8F%D1%86%D0%B8%D1%8F" title="Теплоизоляция">термоизоляцией</a> <a href="/wiki/%D0%A2%D0%BE%D0%BF%D0%BB%D0%B8%D0%B2%D0%BD%D1%8B%D0%B9_%D0%B1%D0%B0%D0%BA" title="Топливный бак">топливного бака</a> полностью устранены не были. Для их решения специалистам НАСА потребовалось дополнительное время, поэтому новый полёт был перенесён на 2006 год. По предварительному решению, старт должен был состояться не ранее 1 июля 2006 года.
</p><p>В случае повреждения «Дискавери» при старте и невозможности его безопасного возвращения на Землю, <a class="mw-redirect" href="/wiki/%D0%90%D1%81%D1%82%D1%80%D0%BE%D0%BD%D0%B0%D0%B2%D1%82" title="Астронавт">астронавты</a> должны были остаться на <a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D0%B0%D1%8F_%D0%BA%D0%BE%D1%81%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%81%D1%82%D0%B0%D0%BD%D1%86%D0%B8%D1%8F" title="Международная космическая станция">МКС</a> и дожидаться спасательной экспедиции «Атлантиса» <a class="new" href="/w/index.php?title=%D0%90%D1%82%D0%BB%D0%B0%D0%BD%D1%82%D0%B8%D1%81_STS-300&amp;action=edit&amp;redlink=1" title="Атлантис STS-300 (страница отсутствует)">STS-300</a>, которая могла бы стартовать в августе 2006 года. Такая мера предосторожности была рекомендована комиссией, расследовавшей катастрофу «Колумбии».
</p>

<p>Командир экипажа Стивен Линдсей, пилот Марк Келли и специалисты полёта Карлос Норьега и Майкл Фоссум были включены в состав экипажа <a href="/wiki/2_%D0%B4%D0%B5%D0%BA%D0%B0%D0%B1%D1%80%D1%8F" title="2 декабря">2 декабря</a> <a href="/wiki/2003_%D0%B3%D0%BE%D0%B4" title="2003 год">2003 года</a>. Астронавт Пирс Селлерс заменил <a class="mw-redirect" href="/wiki/%D0%9D%D0%BE%D1%80%D1%8C%D0%B5%D0%B3%D0%B0,_%D0%9A%D0%B0%D1%80%D0%BB%D0%BE%D1%81_%D0%98%D1%81%D0%BC%D0%B0%D0%B5%D0%BB%D1%8C" title="Норьега, Карлос Исмаель">Карлоса Норьегу</a>, ранее включенного в состав экипажа, но отстраненного от полёта по медицинским показаниям <a href="/wiki/15_%D0%B8%D1%8E%D0%BB%D1%8F" title="15 июля">15 июля</a> <a href="/wiki/2004_%D0%B3%D0%BE%D0%B4" title="2004 год">2004 года</a>. Женщины-астронавты Лиза Новак и Стефани Уилсон были включены в состав экипажа <a href="/wiki/18_%D0%BD%D0%BE%D1%8F%D0%B1%D1%80%D1%8F" title="18 ноября">18 ноября</a> 2004 года.
</p><p>Стартовав с STS-121, Томас Райтер присоединился к экипажу долговременной экспедиции <a href="/wiki/%D0%9C%D0%9A%D0%A1-13" title="МКС-13">МКС-13</a> и на семь месяцев остался на борту <a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D0%B0%D1%8F_%D0%BA%D0%BE%D1%81%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%81%D1%82%D0%B0%D0%BD%D1%86%D0%B8%D1%8F" title="Международная космическая станция">МКС</a>. Он вернулся на <a href="/wiki/%D0%97%D0%B5%D0%BC%D0%BB%D1%8F" title="Земля">Землю</a> <a href="/wiki/22_%D0%B4%D0%B5%D0%BA%D0%B0%D0%B1%D1%80%D1%8F" title="22 декабря">22 декабря</a> <a href="/wiki/2006_%D0%B3%D0%BE%D0%B4" title="2006 год">2006 года</a> с астронавтами экипажа <a class="mw-redirect" href="/wiki/%D0%94%D0%B8%D1%81%D0%BA%D0%B0%D0%B2%D0%B5%D1%80%D0%B8_STS-116" title="Дискавери STS-116">«Дискавери» STS-116</a>. Таким образом, в долговременной экспедиции на МКС вновь после трёхлетнего перерыва вновь стали работать три космонавта. Последняя долговременная экспедиция (<a href="/wiki/%D0%9C%D0%9A%D0%A1-6" title="МКС-6">МКС-6</a>), состоящая из трёх человек, вернулась на Землю <a href="/wiki/4_%D0%BC%D0%B0%D1%8F" title="4 мая">4 мая</a> <a href="/wiki/2003_%D0%B3%D0%BE%D0%B4" title="2003 год">2003 года</a>. Без возобновления полётов шаттлов было бы невозможным обеспечить постоянное присутствие трёх космонавтов на борту МКС.
</p><p>В экипаже «Дискавери» — трое ранее не летавших астронавтов: Майкл Фоссум, Лиза Новак и Стефани Уилсон.

<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Pegasus_barge_being_moved_by_Freedom_Star_and_towboat_American_1.jpg"><img class="mw-file-element" data-file-height="1875" data-file-width="3000" decoding="async" height="94" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/c6/Pegasus_barge_being_moved_by_Freedom_Star_and_towboat_American_1.jpg/150px-Pegasus_barge_being_moved_by_Freedom_Star_and_towboat_American_1.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/c/c6/Pegasus_barge_being_moved_by_Freedom_Star_and_towboat_American_1.jpg/225px-Pegasus_barge_being_moved_by_Freedom_Star_and_towboat_American_1.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/c/c6/Pegasus_barge_being_moved_by_Freedom_Star_and_towboat_American_1.jpg/300px-Pegasus_barge_being_moved_by_Freedom_Star_and_towboat_American_1.jpg 2x" width="150"/></a><figcaption>Транспортировка внешнего <a href="/wiki/%D0%A2%D0%BE%D0%BF%D0%BB%D0%B8%D0%B2%D0%BE" title="Топливо">топливного</a> бака</figcaption></figure>
<p><a href="/wiki/1_%D0%BC%D0%B0%D1%80%D1%82%D0%B0" title="1 марта">1 марта</a> в <a href="/wiki/%D0%9A%D0%BE%D1%81%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%86%D0%B5%D0%BD%D1%82%D1%80_%D0%9A%D0%B5%D0%BD%D0%BD%D0%B5%D0%B4%D0%B8" title="Космический центр Кеннеди">Космический центр Кеннеди</a>, на <a class="mw-redirect" href="/wiki/%D0%9C%D1%8B%D1%81_%D0%9A%D0%B0%D0%BD%D0%B0%D0%B2%D0%B5%D1%80%D0%B0%D0%BB" title="Мыс Канаверал">мысе Канаверал</a>, доставлен внешний топливный бак (ET-119) для <a href="/wiki/%D0%94%D0%B8%D1%81%D0%BA%D0%B0%D0%B2%D0%B5%D1%80%D0%B8_(%D1%88%D0%B0%D1%82%D1%82%D0%BB)" title="Дискавери (шаттл)">«Дискавери»</a>. Топливный бак был модернизирован на предприятии «Michoud Assembly Facility», компании «Lockheed Martin», в <a href="/wiki/%D0%9D%D0%BE%D0%B2%D1%8B%D0%B9_%D0%9E%D1%80%D0%BB%D0%B5%D0%B0%D0%BD" title="Новый Орлеан">Новом Орлеане</a>. Доставка топливного бака осуществлялась морем с <a href="/wiki/25_%D1%84%D0%B5%D0%B2%D1%80%D0%B0%D0%BB%D1%8F" title="25 февраля">25 февраля</a> по <a href="/wiki/1_%D0%BC%D0%B0%D1%80%D1%82%D0%B0" title="1 марта">1 марта</a>. Прибывший топливный бак был перевезён в <a class="mw-redirect" href="/wiki/%D0%97%D0%B4%D0%B0%D0%BD%D0%B8%D0%B5_%D0%B2%D0%B5%D1%80%D1%82%D0%B8%D0%BA%D0%B0%D0%BB%D1%8C%D0%BD%D0%BE%D0%B9_%D1%81%D0%B1%D0%BE%D1%80%D0%BA%D0%B8_(%D0%9A%D0%B0%D0%BD%D0%B0%D0%B2%D0%B5%D1%80%D0%B0%D0%BB)" title="Здание вертикальной сборки (Канаверал)">здание вертикальной сборки</a> шаттла, где был испытан в аэродинамическом туннеле, и затем к нему были подвешены два твердотопливных ускорителя.
</p><p><a href="/wiki/14_%D0%BC%D0%B0%D1%80%D1%82%D0%B0" title="14 марта">14 марта</a> 2006 года НАСА объявило о переносе старта миссии «Дискавери» STS-121. Новое окно для старта: 1 — 19 июля 2006 года. Решение о переносе связано с проблемой в функционировании датчиков, которые контролируют уровень топлива во внешнем топливном боке.
</p><p><a href="/wiki/12_%D0%BC%D0%B0%D1%8F" title="12 мая">12 мая</a> <a class="mw-redirect" href="/wiki/%D0%A1%D0%BF%D0%B5%D0%B9%D1%81_%D0%A8%D0%B0%D1%82%D1%82%D0%BB" title="Спейс Шаттл">шаттл</a> <a href="/wiki/%D0%94%D0%B8%D1%81%D0%BA%D0%B0%D0%B2%D0%B5%D1%80%D0%B8_(%D1%88%D0%B0%D1%82%D1%82%D0%BB)" title="Дискавери (шаттл)">«Дискавери»</a> перевезён из ангара, где он хранился, в здание вертикальной сборки. Здесь он будет соединен с внешним топливным баком и двумя твердотопливными ускорителями. <a href="/wiki/16_%D0%BC%D0%B0%D1%8F" title="16 мая">16 мая</a> в грузовой отсек шаттла будет вложен транспортный модуль «Леонардо». На <a href="/wiki/19_%D0%BC%D0%B0%D1%8F" title="19 мая">19 мая</a> намечена перевозка шаттла на стартовую площадку.
</p>
<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:STS-121_at_the_pad.jpg"><img class="mw-file-element" data-file-height="3008" data-file-width="2000" decoding="async" height="226" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/3f/STS-121_at_the_pad.jpg/150px-STS-121_at_the_pad.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/3f/STS-121_at_the_pad.jpg/225px-STS-121_at_the_pad.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/3/3f/STS-121_at_the_pad.jpg/300px-STS-121_at_the_pad.jpg 2x" width="150"/></a><figcaption>«Дискавери» на стартовой позиции</figcaption></figure>
<p>19 мая шаттл «Дискавери», соединённый с внешним топливным баком и двумя твердотопливными ускорителями, был перевезён из <a class="mw-redirect" href="/wiki/%D0%97%D0%B4%D0%B0%D0%BD%D0%B8%D0%B5_%D0%B2%D0%B5%D1%80%D1%82%D0%B8%D0%BA%D0%B0%D0%BB%D1%8C%D0%BD%D0%BE%D0%B9_%D1%81%D0%B1%D0%BE%D1%80%D0%BA%D0%B8_(%D0%9A%D0%B0%D0%BD%D0%B0%D0%B2%D0%B5%D1%80%D0%B0%D0%BB)" title="Здание вертикальной сборки (Канаверал)">здания вертикальной сборки</a> на стартовую площадку. С этого момента началась непосредственная подготовка к миссии STS-121, благоприятное окно для старта которой начиналось с 1 июля и заканчивалось 19 июля.
Транспортировка шаттла началась в 16:45 (UTC). Расстояние от здания вертикальной сборки до стартовой площадки — 4,2 <a href="/wiki/%D0%9C%D0%B8%D0%BB%D1%8F" title="Миля">мили</a>, максимальная скорость транспортировки — 1 миля в час.
</p><p>Запуск «Дискавери» STS-121 был назначен на 1 июля 15:48 EDT (19:48 UTC, 23:48 московское время)
</p><p>Незадолго перед назначенным временем старта 1 июля (23 часа 48 минут, московского времени) старт «Дискавери» STS-121 был отменён. Причина отмены — неблагоприятные погодные условия. Следующая попытка была возможна в воскресенье, 2 июля в 23 часа 26 минут, однако старт вновь перенесён по той же причине — неблагоприятные погодные условия. Руководством НАСА было решено, что следующая попытка будет предпринята 4 июля в 18:38 UTC, в <a href="/wiki/%D0%94%D0%B5%D0%BD%D1%8C_%D0%BD%D0%B5%D0%B7%D0%B0%D0%B2%D0%B8%D1%81%D0%B8%D0%BC%D0%BE%D1%81%D1%82%D0%B8_%D0%A1%D0%A8%D0%90" title="День независимости США">День независимости США</a>.
</p>
<h2><span id=".D0.9E.D0.BF.D0.B8.D1.81.D0.B0.D0.BD.D0.B8.D0.B5_.D0.BF.D0.BE.D0.BB.D1.91.D1.82.D0.B0"></span><span class="mw-headline" id="Описание_полёта">Описание полёта</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=STS-121&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Описание полёта»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=STS-121&amp;action=edit&amp;section=4" title="Редактировать код раздела «Описание полёта»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>После двух отменённых попыток старта, 4 июля 2006 года, в 18:38 UTC шаттл «Дискавери» STS-121 стартовал с космодрома на мысе Канаверал. В 19:23 UTC «Дискавери» вышел на стабильную орбиту с апогеем — 229 км и перигеем — 156 км.
</p><p><b>5 июля</b>. В первые сутки на орбите астронавты готовились к стыковке с МКС, которая была назначена на 6 июля 14:51 UTC. 5 июля астронавты с помощью камер, установленных на манипуляторе шаттла, осмотрели доступные области внешней поверхности шаттла, на предмет поиска возможных повреждений в теплозащитном покрытии шаттла, и не обнаружили их.
</p>
<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:STS-121_Discovery_approaching_ISS.jpg"><img class="mw-file-element" data-file-height="2005" data-file-width="3032" decoding="async" height="99" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/66/STS-121_Discovery_approaching_ISS.jpg/150px-STS-121_Discovery_approaching_ISS.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/66/STS-121_Discovery_approaching_ISS.jpg/225px-STS-121_Discovery_approaching_ISS.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/66/STS-121_Discovery_approaching_ISS.jpg/300px-STS-121_Discovery_approaching_ISS.jpg 2x" width="150"/></a><figcaption>«Дискавери» приближается к МКС</figcaption></figure>
<p><b>6 июля</b>, при подлёте «Дискавери» к МКС, космонавты 13-й долговременной экспедиции МКС <a href="/wiki/%D0%92%D0%B8%D0%BD%D0%BE%D0%B3%D1%80%D0%B0%D0%B4%D0%BE%D0%B2,_%D0%9F%D0%B0%D0%B2%D0%B5%D0%BB_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Виноградов, Павел Владимирович">Павел Виноградов</a> и <a href="/wiki/%D0%A3%D0%B8%D0%BB%D1%8C%D1%8F%D0%BC%D1%81,_%D0%94%D0%B6%D0%B5%D1%84%D1%84%D1%80%D0%B8_%D0%9D%D0%B5%D0%BB%D1%81" title="Уильямс, Джеффри Нелс">Джеффри Уильямс</a>, через иллюминаторы МКС, фотографировали внешнюю поверхность шаттла, с целю обнаружения возможных повреждений. Фотографии сделанные экипажем МКС были переданы на Землю, для изучения специалистами. В 14:52 UTC шаттл «Дискавери» состыковался с Международной космической станцией. Переходный люк между «Дискавери» и МКС был открыт в 16:30 UTC. На орбите встретились экипаж МКС: командир 13-й экспедиции Павел Виноградов, бортинженер Джеффри Уильямс и экипаж «Дискавери»: Стивен Линдсей, Марк Келли, Майкл Фоссум, Лиза Новак, Стефании Уильсон, Пирс Селлерс и Томас Райтер.
</p><p><b>7 июля</b> астронавты с помощью робота-манипулятора МКС «Канадарм-2» достали многофункциональный грузовой модуль «Леонардо» из грузового отсека шаттла и пристыковали его к МКС. В модуле «Леонардо» находилось более 3,5 тонн полезного груза, необходимого для продолжения работы МКС и его экипажа. Астронавты готовились к намеченному на следующий день в 13:13 UTC, первому выходу в открытый космос, готовили шлюзовой модуль «Квест» и инструменты, которые будут использованы астронавтами Фоссумом и Селлерсом во время выхода в космос.
</p><p><b>8 июля</b> астронавты Пирс Селлерс и Майкл Фоссум совершили выход в открытый космос. Выход начался в 13:17 UTC и закончился в 20:49, продолжительность выхода составила 7 часов 32 минуты. Во время выхода астронавты ремонтировали мобильный транспортёр МКС и испытывали удлинитель робота-манипулятора. НАСА приняло решение продлить полёт «Дискавери» на одни сутки и выполнить ещё один выход в космос. Было решено, что астронавты Селлерс и Фоссум выйдут второй раз в космос в понедельник (10 июля) и третий раз в среду (12 июля), и что во время третьего выхода астронавты будут обследовать теплозащитное покрытие шаттла.
</p><p><b>9 июля</b> астронавты продолжили разгрузку грузового модуля «Леонардо» и подготовку ко второму выходу в открытый космос. Кроме этого экипаж провёл пресс-конференцию, которая транслировалась телевидением НАСА.
</p><p><b>10 июля</b> астронавты Пирс Селлерс и Майкл Фоссум выполнили второй выход в открытый космос. Выход начался в 12:14 UTC и закончился в 19:01, продолжительность выхода составила 6 часов 47 минут. Во время выхода астронавты установили насос системы терморегулирования на внешней поверхности шлюзового модуля «Квест» и проводили работы на мобильном транспортёре. Координацию выхода осуществлял пилот «Дискавери» Марк Келли. Манипулятором управляли Лиза Новак и Стефани Уилсон. В это же время экипаж МКС загружал в модуль «Леонардо» результаты проведённых экспериментов и отработанные материалы и оборудование, для возвращения их на Землю.
</p><p><b>11 июля</b> экипажи «Дискавери» и МКС продолжили разгрузку грузового модуля «Леонардо» и подготовку к третьему выходу в открытый космос, который состоится 12 июля.
</p><p><b>12 июля</b> астронавты Пирс Селлерс и Майкл Фоссум выполнили третий выход в открытый космос. Выход начался в 11:20 UTC и закончился в 18:31, продолжительность выхода составила 7 часов 11 минут. Во время выхода астронавты проводили испытания техники ремонта теплозащитного покрытия шаттла. Эти испытания проводились в грузовом отсеке шаттла, где были установлены образцы теплозащитного покрытия. Астронавты также проверяли инфракрасную камеру, которая фиксирует градиент температуры на теплозащитных плитках. Координацию выхода осуществлял пилот «Дискавери» Марк Келли. Роботом-манипулятором управляли Лиза Новак и Стефани Уилсон.
</p><p><b>13 июля</b>. После выполнения трёх выходов в космос, разгрузки и загрузки модуля «Леонардо», астронавты «Дискавери» имели день отдыха. В этот же день состоялась пресс-конференция экипажей «Дискавери» и МКС. Джеффри Уильямс проводил последние операции, для подготовки модуля «Леонардо» к отстыковке от МКС.
</p><p><b>14 июля</b> многофункциональный грузовой модуль «Леонардо» был отстыкован от МКС и помещён в грузовой отсек «Дискавери». Роботом-манипулятором МКС управляли Лиза Новак и Стефани Уилсон. Модуль «Леонардо» был отсоединён от МКС в 13:25 UTC и помещён в грузовой отсек шаттла в 15:00 UTC. В модуле было доставлено на МКС около 3,5 тонн (7000 <a class="extiw" href="https://en.wikipedia.org/wiki/%D0%A4%D1%83%D0%BD%D1%82_(%D0%B5%D0%B4%D0%B8%D0%BD%D0%B8%D1%86%D0%B0_%D0%B8%D0%B7%D0%BC%D0%B5%D1%80%D0%B5%D0%BD%D0%B8%D1%8F)" title="w:Фунт (единица измерения)">фунтов</a>) полезных грузов. На Землю отправлено около 1,8 тонн (4000 <a class="extiw" href="https://en.wikipedia.org/wiki/%D0%A4%D1%83%D0%BD%D1%82_(%D0%B5%D0%B4%D0%B8%D0%BD%D0%B8%D1%86%D0%B0_%D0%B8%D0%B7%D0%BC%D0%B5%D1%80%D0%B5%D0%BD%D0%B8%D1%8F)" title="w:Фунт (единица измерения)">фунтов</a>) ненужных на станции материалов и оборудования, а также результаты научных экспериментов. Командир шаттла Линдсей ещё раз осмотрел кромки крыльев «Дискавери» в поисках возможных повреждений.
Отсоединение «Дискавери» от МКС было назначено на 15 июля в 10:08 UTC.
</p><p><b>15 июля</b> в 10:08 UTC шаттл «Дискавери» отстыковался от МКС, это произошло над <a href="/wiki/%D0%A2%D0%B8%D1%85%D0%B8%D0%B9_%D0%BE%D0%BA%D0%B5%D0%B0%D0%BD" title="Тихий океан">Тихим океаном</a> севернее <a href="/wiki/%D0%9D%D0%BE%D0%B2%D0%B0%D1%8F_%D0%97%D0%B5%D0%BB%D0%B0%D0%BD%D0%B4%D0%B8%D1%8F" title="Новая Зеландия">Новой Зеландии</a>. Люк между «Дискавери» и орбитальным комплексом был закрыт в 08:00. Через 10 минут, после отхода «Дискавери» от станции, расстояние между ними было 40 метров. В 10:30 на расстоянии 120 метров от МКС были включены двигатели системы орбитального маневрирования (OMS) и шаттл совершил облёт МКС сверху. «Дискавери» должен приземлиться 17 июля на мысе Канаверал.
</p><p><b>16 июля</b> центр управления полётом, после изучения всех данных полученных с «Дискавери»,
объявил, что для посадки шаттла нет препятствий. Экипаж проводит подготовку к возвращению на Землю, проверяет двигатели и системы шаттла. Посадка назначена на 13:14 UTC 17 июля. Включение двигателей на торможение и схода с орбиты назначено на 12:07 UTC.
Вторая возможность посадки в понедельник (17 июля) — включение двигателей в 13:43 UTC и посадка в 14:50 UTC. Прогноз погоды — благоприятный для посадки. На понедельник не планируется возможность посадки в Калифорнии, на военно-воздушной <a href="/wiki/%D0%91%D0%B0%D0%B7%D0%B0_%D0%92%D0%B0%D0%BD%D0%B4%D0%B5%D0%BD%D0%B1%D0%B5%D1%80%D0%B3" title="База Ванденберг">базе Ванденберг</a>.
</p><p><b>17 июля</b> шаттл «Дискавери» с шестью астронавтами на борту совершил мягкую посадку на мысе <a class="mw-redirect" href="/wiki/%D0%9C%D1%8B%D1%81_%D0%9A%D0%B0%D0%BD%D0%B0%D0%B2%D0%B5%D1%80%D0%B0%D0%BB" title="Мыс Канаверал">Канаверал</a> (штат <a href="/wiki/%D0%A4%D0%BB%D0%BE%D1%80%D0%B8%D0%B4%D0%B0" title="Флорида">Флорида</a>). Успешно завершён второй полёт шаттла по программе «Возвращение к полётам». Этот успех позволяет возобновить регулярные полёты шаттлов и продолжить работы по строительству <a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D0%B0%D1%8F_%D0%BA%D0%BE%D1%81%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%81%D1%82%D0%B0%D0%BD%D1%86%D0%B8%D1%8F" title="Международная космическая станция">Международной космической станции</a>.
</p>
<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Landing_of_Space_Shuttle_Discovery.jpg"><img class="mw-file-element" data-file-height="2000" data-file-width="3000" decoding="async" height="100" src="//upload.wikimedia.org/wikipedia/commons/thumb/2/28/Landing_of_Space_Shuttle_Discovery.jpg/150px-Landing_of_Space_Shuttle_Discovery.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/2/28/Landing_of_Space_Shuttle_Discovery.jpg/225px-Landing_of_Space_Shuttle_Discovery.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/2/28/Landing_of_Space_Shuttle_Discovery.jpg/300px-Landing_of_Space_Shuttle_Discovery.jpg 2x" width="150"/></a><figcaption>Приземление «Дискавери» STS-121</figcaption></figure>
<p>«Дискавери» с экипажем на борту приземлился в 13:14 <a class="mw-redirect" href="/wiki/UTC" title="UTC">UTC</a>, время пребывания в космосе составило 12 суток 18 часов 36 минут. Шаттл прошёл расстояние в 8,85 миллионов километров (5,5 миллионов миль). «Дискавери» доставил на борт МКС <a class="mw-redirect" href="/wiki/%D0%A0%D0%B0%D0%B9%D1%82%D0%B5%D1%80,_%D0%A2%D0%BE%D0%BC%D0%B0%D1%81_%D0%90%D1%80%D1%82%D1%83%D1%80" title="Райтер, Томас Артур">Томаса Райтера</a>, третьего члена 13-й долговременной экспедиции. Впервые с мая 2003 года, на МКС вновь экипаж из трёх астронавтов. На МКС были доставлены материалы и приборы для поддержания её функционирования, общим весом более чем 12 тонн (28000 фунтов). Во время пребывания на орбите было совершено три выхода в открытый космос, во время одного из них проводились ремонтные работы на мобильном транспортёре МКС, во время двух выходов отрабатывалась техника обследования и ремонта на орбите теплозащитного покрытия шаттла. Во время выходов в космос астронавты Селлерс и Фоссум испытали удлинитель робота-манипулятора, заменили кабель, который служит для передачи электропитания, команд, данных и для передачи видео изображения к мобильному транспортёру МКС. Мобильный транспортёр служит для передвижения платформы, на которой закреплён робот-манипулятор станции.
</p><p>Двигатель торможения шаттла был включён в 12:07 UTC, когда шаттл находился на Индийским океаном. Двигатель проработал 3 минуты, шаттл сошёл с орбиты и устремился к Земле. При спуске шаттл пролетел над Гватемалой, Мексикой, Мексиканским заливом и над юго-западным побережьем Флориды.
</p>
<h2><span id=".D0.98.D1.82.D0.BE.D0.B3.D0.B8"></span><span class="mw-headline" id="Итоги">Итоги</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=STS-121&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Итоги»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=STS-121&amp;action=edit&amp;section=5" title="Редактировать код раздела «Итоги»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Сразу после <a class="mw-redirect" href="/wiki/%D0%9F%D1%80%D0%B8%D0%B7%D0%B5%D0%BC%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5" title="Приземление">приземления</a> <a class="mw-redirect" href="/wiki/%D0%90%D1%81%D1%82%D1%80%D0%BE%D0%BD%D0%B0%D0%B2%D1%82" title="Астронавт">астронавты</a> <a class="mw-redirect" href="/wiki/%D0%9B%D0%B8%D0%BD%D0%B4%D1%81%D0%B5%D0%B9,_%D0%A1%D1%82%D0%B8%D0%B2%D0%B5%D0%BD_%D0%A3%D1%8D%D0%B9%D0%BD" title="Линдсей, Стивен Уэйн">Стивен Линдсей</a>, <a href="/wiki/%D0%9A%D0%B5%D0%BB%D0%BB%D0%B8,_%D0%9C%D0%B0%D1%80%D0%BA_%D0%AD%D0%B4%D0%B2%D0%B0%D1%80%D0%B4" title="Келли, Марк Эдвард">Марк Келли</a>, <a href="/wiki/%D0%A4%D0%BE%D1%81%D1%81%D1%83%D0%BC,_%D0%9C%D0%B0%D0%B9%D0%BA%D0%BB_%D0%AD%D0%B4%D0%B2%D0%B0%D1%80%D0%B4" title="Фоссум, Майкл Эдвард">Майкл Фоссум</a>, <a class="mw-redirect" href="/wiki/%D0%A1%D0%B5%D0%BB%D0%BB%D0%B5%D1%80%D1%81,_%D0%9F%D0%B8%D1%80%D1%81_%D0%94%D0%B6%D0%BE%D0%BD" title="Селлерс, Пирс Джон">Пирс Селлерс</a>, <a href="/wiki/%D0%9D%D0%BE%D0%B2%D0%B0%D0%BA,_%D0%9B%D0%B8%D0%B7%D0%B0_%D0%9C%D0%B0%D1%80%D0%B8%D1%8F" title="Новак, Лиза Мария">Лиза Новак</a> и <a class="mw-redirect" href="/wiki/%D0%A3%D0%B8%D0%BB%D1%81%D0%BE%D0%BD,_%D0%A1%D1%82%D0%B5%D1%84%D0%B0%D0%BD%D0%B8_%D0%94%D0%B8%D0%B0%D0%BD%D0%B0" title="Уилсон, Стефани Диана">Стефани Уилсон</a> прошли короткое медицинское обследование. После обследования астронавты осмотрели свой <a href="/wiki/%D0%9F%D0%B8%D0%BB%D0%BE%D1%82%D0%B8%D1%80%D1%83%D0%B5%D0%BC%D1%8B%D0%B9_%D0%BA%D0%BE%D1%81%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%BE%D1%80%D0%B0%D0%B1%D0%BB%D1%8C" title="Пилотируемый космический корабль">космический корабль</a> — <a href="/wiki/%D0%94%D0%B8%D1%81%D0%BA%D0%B0%D0%B2%D0%B5%D1%80%D0%B8_(%D1%88%D0%B0%D1%82%D1%82%D0%BB)" title="Дискавери (шаттл)">«Дискавери»</a>.
Командир <a href="/wiki/%D0%AD%D0%BA%D0%B8%D0%BF%D0%B0%D0%B6" title="Экипаж">экипажа</a> Стив Линдсей сказал: «Это был мой четвёртый <a href="/wiki/%D0%9F%D0%B8%D0%BB%D0%BE%D1%82%D0%B8%D1%80%D1%83%D0%B5%D0%BC%D1%8B%D0%B9_%D0%BA%D0%BE%D1%81%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%BE%D0%BB%D1%91%D1%82" title="Пилотируемый космический полёт">космический полёт</a> и я четыре раза обошёл вокруг <a class="mw-redirect" href="/wiki/%D0%A1%D0%BF%D0%B5%D0%B9%D1%81_%D0%A8%D0%B0%D1%82%D1%82%D0%BB" title="Спейс Шаттл">шаттла</a>, корабль выглядит так чисто, как никакой до него, у нас было очень мало проблем». Линдсей поблагодарил свой экипаж: «Я хотел бы сказать спасибо экипажу — во время полёта, все были великолепны. Для меня было честью работать с таким экипажем».
</p><p>Полёт STS-121 был наиболее фотографируемым из всех ранее состоявшихся. Во время старта съёмку «Дискавери» выполняли более 100 цифровых и видео камер с высоким разрешением.
</p><p>По завершении полёта <a href="/wiki/%D0%9D%D0%90%D0%A1%D0%90" title="НАСА">американское космическое агентство</a> объявило, что следующим стартом шаттла должна стать миссия <a href="/wiki/%D0%90%D1%82%D0%BB%D0%B0%D0%BD%D1%82%D0%B8%D1%81" title="Атлантис">«Атлантиса»</a> <a class="mw-redirect" href="/wiki/%D0%90%D1%82%D0%BB%D0%B0%D0%BD%D1%82%D0%B8%D1%81_STS-115" title="Атлантис STS-115">STS-115</a>. Окно для запуска: — с <a href="/wiki/28_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="28 августа">28 августа</a> по <a href="/wiki/13_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="13 сентября">13 сентября</a>. Практически, «Атлантис» должен был стартовать до <a href="/wiki/7_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="7 сентября">7 сентября</a>, чтобы не встретится на <a href="/wiki/%D0%9E%D1%80%D0%B1%D0%B8%D1%82%D0%B0" title="Орбита">орбите</a> с российским космическим кораблём <a href="/wiki/%D0%A1%D0%BE%D1%8E%D0%B7_%D0%A2%D0%9C%D0%90-9" title="Союз ТМА-9">«Союз ТМА-9»</a>, старт которого был назначен на <a href="/wiki/14_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="14 сентября">14 сентября</a>. Основной задачей «Союза» должна была стать доставка на <a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D0%B0%D1%8F_%D0%BA%D0%BE%D1%81%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%81%D1%82%D0%B0%D0%BD%D1%86%D0%B8%D1%8F" title="Международная космическая станция">МКС</a> <a href="/wiki/%D0%9C%D0%9A%D0%A1-14" title="МКС-14">14-го долговременного экипажа</a>: <a class="mw-redirect" href="/wiki/%D0%9B%D0%BE%D0%BF%D0%B5%D1%81-%D0%90%D0%BB%D0%B5%D0%B3%D1%80%D0%B8%D0%B0,_%D0%9C%D0%B0%D0%B9%D0%BA%D0%BB_%D0%AD%D0%BB%D0%B0%D0%B4%D0%B8%D0%BE" title="Лопес-Алегриа, Майкл Эладио">Майкла Лопес-Алегрия</a> (<a href="/wiki/%D0%A1%D0%BE%D0%B5%D0%B4%D0%B8%D0%BD%D1%91%D0%BD%D0%BD%D1%8B%D0%B5_%D0%A8%D1%82%D0%B0%D1%82%D1%8B_%D0%90%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B8" title="Соединённые Штаты Америки">США</a>) и <a href="/wiki/%D0%A2%D1%8E%D1%80%D0%B8%D0%BD,_%D0%9C%D0%B8%D1%85%D0%B0%D0%B8%D0%BB_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D1%81%D0%BB%D0%B0%D0%B2%D0%BE%D0%B2%D0%B8%D1%87" title="Тюрин, Михаил Владиславович">Михаила Тюрина</a> (<a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D1%8F" title="Россия">Россия</a>).
</p>
