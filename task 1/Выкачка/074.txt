<p><b>Княжество Антиохия</b> (<a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Латинский язык">лат.</a> <span lang="la" style="font-style:italic;">Principatus Antiochiae</span>, <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Французский язык">фр.</a> <span lang="fr" style="font-style:italic;">Principauté d'Antioche</span>, <a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D1%8C%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Итальянский язык">итал.</a> <span lang="it" style="font-style:italic;">Principato d'Antiochia</span>, <a href="/wiki/%D0%90%D1%80%D0%BC%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Армянский язык">арм.</a> <span lang="hy">Անտիոքի դքսություն</span>) — второе из <a href="/wiki/%D0%93%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B0_%D0%BA%D1%80%D0%B5%D1%81%D1%82%D0%BE%D0%BD%D0%BE%D1%81%D1%86%D0%B5%D0%B2" title="Государства крестоносцев">христианских государств</a>, основанных <a class="mw-redirect" href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82%D0%BE%D0%BD%D0%BE%D1%81%D1%86%D1%8B" title="Крестоносцы">крестоносцами</a> во время <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B2%D1%8B%D0%B9_%D0%BA%D1%80%D0%B5%D1%81%D1%82%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D0%BE%D1%85%D0%BE%D0%B4" title="Первый крестовый поход">1-го крестового похода</a> на территории современных <a href="/wiki/%D0%A1%D0%B8%D1%80%D0%B8%D1%8F" title="Сирия">Сирии</a> и <a href="/wiki/%D0%A2%D1%83%D1%80%D1%86%D0%B8%D1%8F" title="Турция">Турции</a>. Государство образовано в <a href="/wiki/1098_%D0%B3%D0%BE%D0%B4" title="1098 год">1098 году</a> после завоевания Антиохии <a class="mw-redirect" href="/wiki/%D0%91%D0%BE%D1%8D%D0%BC%D1%83%D0%BD%D0%B4_I,_%D0%BA%D0%BD%D1%8F%D0%B7%D1%8C_%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B8" title="Боэмунд I, князь Антиохии">Боэмундом I Тарентским</a>.
</p><p>В начале <a href="/wiki/XII_%D0%B2%D0%B5%D0%BA" title="XII век">XII века</a> княжество находилось в вассальной зависимости от <a href="/wiki/%D0%92%D0%B8%D0%B7%D0%B0%D0%BD%D1%82%D0%B8%D1%8F" title="Византия">Византии</a>, а с 1119 года — от <a href="/wiki/%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D1%81%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D0%B5%D0%B2%D1%81%D1%82%D0%B2%D0%BE" title="Иерусалимское королевство">Иерусалимского королевства</a>. В состав княжества входила территория, располагавшаяся между средиземноморским побережьем современной Сирии и рекой <a href="/wiki/%D0%95%D0%B2%D1%84%D1%80%D0%B0%D1%82" title="Евфрат">Евфрат</a>, в первой половине <a href="/wiki/XII_%D0%B2%D0%B5%D0%BA" title="XII век">XII века</a> в его состав входила также Равнинная <a href="/wiki/%D0%9A%D0%B8%D0%BB%D0%B8%D0%BA%D0%B8%D1%8F" title="Киликия">Киликия</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>. Князь Антиохии был <a href="/wiki/%D0%A1%D1%8E%D0%B7%D0%B5%D1%80%D0%B5%D0%BD" title="Сюзерен">сюзереном</a> <a class="mw-redirect" href="/wiki/%D0%93%D1%80%D0%B0%D1%84%D1%81%D1%82%D0%B2%D0%BE_%D0%AD%D0%B4%D0%B5%D1%81%D1%81%D0%B0" title="Графство Эдесса">графства Эдесса</a> (до <a href="/wiki/1144_%D0%B3%D0%BE%D0%B4" title="1144 год">1144 года</a>), а с <a href="/wiki/1163_%D0%B3%D0%BE%D0%B4" title="1163 год">1163 года</a> — <a class="mw-redirect" href="/wiki/%D0%93%D1%80%D0%B0%D1%84%D1%81%D1%82%D0%B2%D0%BE_%D0%A2%D1%80%D0%B8%D0%BF%D0%BE%D0%BB%D0%B8" title="Графство Триполи">графства Триполи</a>, с которым в <a href="/wiki/1207_%D0%B3%D0%BE%D0%B4" title="1207 год">1207 году</a> Антиохия была объединена <a class="mw-redirect" href="/wiki/%D0%91%D0%BE%D1%8D%D0%BC%D1%83%D0%BD%D0%B4_IV,_%D0%BA%D0%BD%D1%8F%D0%B7%D1%8C_%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B8" title="Боэмунд IV, князь Антиохии">Боэмундом IV</a>.
</p><p>В <a href="/wiki/1268_%D0%B3%D0%BE%D0%B4" title="1268 год">1268 году</a> княжество было завоёвано <a href="/wiki/%D0%9C%D0%B0%D0%BC%D0%BB%D1%8E%D0%BA%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D1%83%D0%BB%D1%82%D0%B0%D0%BD%D0%B0%D1%82" title="Мамлюкский султанат">мамлюкским султаном</a> <a href="/wiki/%D0%91%D0%B5%D0%B9%D0%B1%D0%B0%D1%80%D1%81_I" title="Бейбарс I">Бейбарсом</a> и прекратило существование, однако титул «князь Антиохии» продолжали носить графы Триполи до его завоевания в <a href="/wiki/1287_%D0%B3%D0%BE%D0%B4" title="1287 год">1287 году</a>.
</p>

Миниатюра <a href="/wiki/%D0%9A%D0%BE%D0%BB%D0%BE%D0%BC%D0%B1,_%D0%96%D0%B0%D0%BD" title="Коломб, Жан">Жана Коломба</a> из книги <a href="/wiki/%D0%9C%D0%B0%D0%BC%D1%80%D0%BE,_%D0%A1%D0%B5%D0%B1%D0%B0%D1%81%D1%82%D1%8C%D0%B5%D0%BD" title="Мамро, Себастьен">Себастьена Мамро</a> «<a class="mw-redirect" href="/wiki/%D0%9F%D0%BE%D1%85%D0%BE%D0%B4%D1%8B_%D1%84%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D0%BE%D0%B2_%D0%B7%D0%B0_%D0%BC%D0%BE%D1%80%D0%B5_%D0%BF%D1%80%D0%BE%D1%82%D0%B8%D0%B2_%D1%82%D1%83%D1%80%D0%BE%D0%BA,_%D1%81%D0%B0%D1%80%D0%B0%D1%86%D0%B8%D0%BD_%D0%B8_%D0%BC%D0%B0%D0%B2%D1%80%D0%BE%D0%B2" title="Походы французов за море против турок, сарацин и мавров">Походы французов за море против турок, сарацин и мавров</a>» (1474).</figcaption></figure>
<p>Пока <a class="mw-redirect" href="/wiki/%D0%91%D0%B0%D0%BB%D0%B4%D1%83%D0%B8%D0%BD_I,_%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D1%8C_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D0%B0" title="Балдуин I, король Иерусалима">Балдуин Булонский</a> и <a href="/wiki/%D0%A2%D0%B0%D0%BD%D0%BA%D1%80%D0%B5%D0%B4_%D0%A2%D0%B0%D1%80%D0%B5%D0%BD%D1%82%D1%81%D0%BA%D0%B8%D0%B9" title="Танкред Тарентский">Танкред Тарентский</a> пребывали в <a href="/wiki/%D0%9A%D0%B8%D0%BB%D0%B8%D0%BA%D0%B8%D1%8F" title="Киликия">Киликии</a> и <a href="/wiki/%D0%9C%D0%B5%D1%81%D0%BE%D0%BF%D0%BE%D1%82%D0%B0%D0%BC%D0%B8%D1%8F" title="Месопотамия">Месопотамии</a>, где Балдуином было основано <a class="mw-redirect" href="/wiki/%D0%93%D1%80%D0%B0%D1%84%D1%81%D1%82%D0%B2%D0%BE_%D0%AD%D0%B4%D0%B5%D1%81%D1%81%D0%BA%D0%BE%D0%B5" title="Графство Эдесское">графство Эдесское</a>, остальные крестоносцы продолжали наступление на юг и <a href="/wiki/21_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="21 октября">21 октября</a> <a href="/wiki/1097_%D0%B3%D0%BE%D0%B4" title="1097 год">1097 года</a> под предводительством <a class="mw-redirect" href="/wiki/%D0%91%D0%BE%D1%8D%D0%BC%D1%83%D0%BD%D0%B4_I,_%D0%BA%D0%BD%D1%8F%D0%B7%D1%8C_%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B8" title="Боэмунд I, князь Антиохии">Боэмунда Тарентского</a> осадили <a href="/wiki/%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D1%8F" title="Антиохия">Антиохию</a>. Город, толстые стены которого были надёжно укреплены четырьмя сотнями башен, оказался поистине неприступным, и <a href="/wiki/%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B8" title="Осада Антиохии">осада</a> продлилась всю зиму <a href="/wiki/1098_%D0%B3%D0%BE%D0%B4" title="1098 год">1098 года</a>.
</p>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Gustave_dore_crusades_the_massacre_of_antioch.jpg"><img class="mw-file-element" data-file-height="2194" data-file-width="1564" decoding="async" height="351" src="//upload.wikimedia.org/wikipedia/commons/thumb/1/18/Gustave_dore_crusades_the_massacre_of_antioch.jpg/250px-Gustave_dore_crusades_the_massacre_of_antioch.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/1/18/Gustave_dore_crusades_the_massacre_of_antioch.jpg/375px-Gustave_dore_crusades_the_massacre_of_antioch.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/1/18/Gustave_dore_crusades_the_massacre_of_antioch.jpg/500px-Gustave_dore_crusades_the_massacre_of_antioch.jpg 2x" width="250"/></a><figcaption>Резня в Антиохии (рисунок <a href="/wiki/%D0%94%D0%BE%D1%80%D0%B5,_%D0%93%D1%8E%D1%81%D1%82%D0%B0%D0%B2" title="Доре, Гюстав">Доре</a>)</figcaption></figure>
<p>Воины, как писал в письме супруге Адели Стефен Блуаский, «в течение всей зимы страдали ради нашего Господа Христа от непомерного мороза и страшных ливней»<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>. Затем в лагеря крестоносцев пришли болезни и голод. Положение христиан было настолько отчаянное, что по свидетельству очевидцев-хронистов, они были вынуждены пустить на мясо собственных лошадей. Существуют записи о том, что иногда дело доходило до <a href="/wiki/%D0%9A%D0%B0%D0%BD%D0%BD%D0%B8%D0%B1%D0%B0%D0%BB%D0%B8%D0%B7%D0%BC" title="Каннибализм">каннибализма</a> и голодающие воины поедали тела своих умерших товарищей<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>.
</p><p>В конце концов Боэмунд нашёл выход. Он подговорил воина из гарнизона Антиохии, бывшего христианина по имени Фируз (историк Ибн ал-Асир приводит другое имя — Рузбих<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>) открыть городские ворота, и <a href="/wiki/3_%D0%B8%D1%8E%D0%BD%D1%8F" title="3 июня">3 июня</a> <a href="/wiki/1098_%D0%B3%D0%BE%D0%B4" title="1098 год">1098 года</a>, спустя 8 месяцев после начала осады, крестоносцы вошли в Антиохию. Несколько дней в городе продолжалась кровавая резня, а через четыре дня к стенам Антиохии прибыла мусульманская армия под предводительством <a href="/wiki/%D0%9C%D0%BE%D1%81%D1%83%D0%BB" title="Мосул">мосульского</a> <a href="/wiki/%D0%90%D1%82%D0%B0%D0%B1%D0%B5%D0%BA" title="Атабек">атабека</a> <a href="/wiki/%D0%9A%D0%B5%D1%80%D0%B1%D0%BE%D0%B3%D0%B0" title="Кербога">Кербоги</a> и в свою очередь осадила город. <a class="mw-redirect" href="/wiki/%D0%92%D0%B8%D0%B7%D0%B0%D0%BD%D1%82%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B5_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%BE%D1%80%D1%8B" title="Византийские императоры">Император Византии</a> <a href="/wiki/%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_I_%D0%9A%D0%BE%D0%BC%D0%BD%D0%B8%D0%BD" title="Алексей I Комнин">Алексей Комнин</a> собирался сначала оказать крестоносцам помощь, но услышав, будто бы мусульмане отбили город, развернул войска обратно<sup class="reference" id="cite_ref-Rishar_41_5-0"><a href="#cite_note-Rishar_41-5">[5]</a></sup>.
</p><p>Атаку мусульман удалось отбить только благодаря чуду. После того, как по предсказанию марсельского <a href="/wiki/%D0%9C%D0%BE%D0%BD%D0%B0%D1%85" title="Монах">монаха</a> <a class="new" href="/w/index.php?title=%D0%9F%D1%8C%D0%B5%D1%80_%D0%91%D0%B0%D1%80%D1%82%D0%B5%D0%BB%D0%B5%D0%BC%D0%B8&amp;action=edit&amp;redlink=1" title="Пьер Бартелеми (страница отсутствует)">Пьера Бартелеми</a> в церкви Св. Петра было обнаружено <a href="/wiki/%D0%9A%D0%BE%D0%BF%D1%8C%D1%91_%D0%9B%D0%BE%D0%BD%D0%B3%D0%B8%D0%BD%D0%B0" title="Копьё Лонгина">копьё</a>, которым, по легенде, было прободено ребро распятого <a class="mw-redirect" href="/wiki/%D0%98%D0%B8%D1%81%D1%83%D1%81" title="Иисус">Иисуса</a>, крестоносцев, истощённых за долгие месяцы безрезультатной осады, охватило такое рвение, что им удалось обратить Кербогу в бегство и перебить значительную часть его армии. Далее начались споры о том, кому достанется право управлять городом, и в конце концов князем Антиохии стал <a class="mw-redirect" href="/wiki/%D0%91%D0%BE%D1%8D%D0%BC%D1%83%D0%BD%D0%B4_%D0%A2%D0%B0%D1%80%D0%B5%D0%BD%D1%82%D1%81%D0%BA%D0%B8%D0%B9" title="Боэмунд Тарентский">Боэмунд Тарентский</a><sup class="reference" id="cite_ref-Rishar_41_5-1"><a href="#cite_note-Rishar_41-5">[5]</a></sup>.
</p>
<h2><span id=".D0.A0.D0.B0.D0.BD.D0.BD.D1.8F.D1.8F_.D0.B8.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.8F"></span><span class="mw-headline" id="Ранняя_история">Ранняя история</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B9%D1%81%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BD%D1%8F%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%BE&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Ранняя история»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B9%D1%81%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BD%D1%8F%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%BE&amp;action=edit&amp;section=2" title="Редактировать код раздела «Ранняя история»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Князь Боэмунд в <a href="/wiki/1101_%D0%B3%D0%BE%D0%B4" title="1101 год">1101 году</a> попал в плен к сельджукскому эмиру Сиваса, <a class="mw-redirect" href="/wiki/%D0%94%D0%B0%D0%BD%D0%B8%D1%88%D0%BC%D0%B5%D0%BD%D0%B4_%D0%93%D0%B0%D0%B7%D0%B8" title="Данишменд Гази">Данишменд Гази</a> и провёл в плену два года. В это время княжеством управлял его племянник Танкред. Он значительно раздвинул границы княжества, воспользовавшись разногласиями среди мусульман и захватив византийские города <a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B0%D0%BA%D0%B8%D1%8F" title="Латакия">Латакию</a> и <a href="/wiki/%D0%A2%D0%B0%D1%80%D1%81%D1%83%D1%81" title="Тарсус">Тарсус</a>.
</p><p>Боэмунд вернулся из плена в <a href="/wiki/1103_%D0%B3%D0%BE%D0%B4" title="1103 год">1103 году</a>, но спустя два года, после <a href="/wiki/%D0%91%D0%B8%D1%82%D0%B2%D0%B0_%D0%BF%D1%80%D0%B8_%D0%A5%D0%B0%D1%80%D1%80%D0%B0%D0%BD%D0%B5" title="Битва при Харране">поражения при Харране</a>, отбыл в <a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D0%B8%D1%8F" title="Италия">Италию</a> набирать новую армию, и вновь назначил Танкреда <a href="/wiki/%D0%A0%D0%B5%D0%B3%D0%B5%D0%BD%D1%82" title="Регент">регентом</a> Антиохии<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>. Вернувшись с новыми силами на Восток, Боэмунд в <a href="/wiki/1107_%D0%B3%D0%BE%D0%B4" title="1107 год">1107 году</a> напал на <a class="mw-redirect" href="/wiki/%D0%92%D0%B8%D0%B7%D0%B0%D0%BD%D1%82%D0%B8%D0%B9%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F" title="Византийская империя">Византию</a>, но потерпел поражение и был вынужден в следующем году заключить с <a href="/wiki/%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_I_%D0%9A%D0%BE%D0%BC%D0%BD%D0%B8%D0%BD" title="Алексей I Комнин">Алексеем Комнином</a> унизительный <a href="/wiki/%D0%94%D0%B5%D0%B2%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D0%B4%D0%BE%D0%B3%D0%BE%D0%B2%D0%BE%D1%80" title="Девольский договор">Девольский договор</a><sup class="reference" id="cite_ref-autogenerated1_7-0"><a href="#cite_note-autogenerated1-7">[7]</a></sup>, согласно которому он признавал себя вассалом Византии, а после его смерти княжество должно было отойти императору<sup class="reference" id="cite_ref-autogenerated2_8-0"><a href="#cite_note-autogenerated2-8">[8]</a></sup>. Вскоре Боэмунд вновь отправился в Италию, оставив во главе княжества Танкреда, где умер в <a href="/wiki/1111_%D0%B3%D0%BE%D0%B4" title="1111 год">1111 году</a><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>.
</p><p>После смерти Боэмунда Византийская империя потребовала выполнить условия договора, но Танкред Тарентский при поддержке графа Триполи и короля Иерусалима ответил отказом. Фактически Танкред был единственным из лидеров 1-го крестового похода, кто не давал клятву возвратить завоёванные земли Византии (хотя никто из поклявшихся все равно не сдержал слово)<sup class="reference" id="cite_ref-autogenerated1_7-1"><a href="#cite_note-autogenerated1-7">[7]</a></sup>. После его смерти в <a href="/wiki/1112_%D0%B3%D0%BE%D0%B4" title="1112 год">1112 году</a> княжество унаследовал <a class="mw-redirect" href="/wiki/%D0%91%D0%BE%D1%8D%D0%BC%D1%83%D0%BD%D0%B4_II,_%D0%BA%D0%BD%D1%8F%D0%B7%D1%8C_%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B8" title="Боэмунд II, князь Антиохии">Боэмунд II</a>, сын Боэмунда Тарентского и Констанции, дочери французского короля <a href="/wiki/%D0%A4%D0%B8%D0%BB%D0%B8%D0%BF%D0%BF_I_(%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D1%8C_%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D0%B8)" title="Филипп I (король Франции)">Филиппа I</a>. Так как юному князю было на тот момент всего три года, регентом Антиохии был назначен родственник Танкреда, <a class="mw-redirect" href="/wiki/%D0%A0%D0%BE%D0%B6%D0%B5%D1%80,_%D0%BA%D0%BD%D1%8F%D0%B7%D1%8C_%D0%A1%D0%B0%D0%BB%D0%B5%D1%80%D0%BD%D0%BE" title="Рожер, князь Салерно">Рожер Салернский</a>, в <a href="/wiki/1113_%D0%B3%D0%BE%D0%B4" title="1113 год">1113 году</a> успешно отразивший нападение сельджуков.
</p><p><a href="/wiki/28_%D0%B8%D1%8E%D0%BD%D1%8F" title="28 июня">28 июня</a> <a href="/wiki/1119_%D0%B3%D0%BE%D0%B4" title="1119 год">1119 года</a> Рожер пал в <a href="/wiki/%D0%91%D0%B8%D1%82%D0%B2%D0%B0_%D0%BD%D0%B0_%D0%9A%D1%80%D0%BE%D0%B2%D0%B0%D0%B2%D0%BE%D0%BC_%D0%BF%D0%BE%D0%BB%D0%B5" title="Битва на Кровавом поле">битве на Кровавом поле</a><sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> и Антиохия стала вассальным княжеством <a href="/wiki/%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D1%81%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D0%B5%D0%B2%D1%81%D1%82%D0%B2%D0%BE" title="Иерусалимское королевство">Иерусалимского королевства</a> под регентством <a class="mw-redirect" href="/wiki/%D0%91%D0%B0%D0%BB%D0%B4%D1%83%D0%B8%D0%BD_II_(%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D1%8C_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D0%B0)" title="Балдуин II (король Иерусалима)">Балдуина II</a>.
</p><p>Боэмунд II в <a href="/wiki/1126_%D0%B3%D0%BE%D0%B4" title="1126 год">1126 году</a> в возрасте восемнадцати лет женился на дочери иерусалимского короля, Алисе де Бурк. Он правил Антиохией на протяжении коротких четырёх лет, а после его смерти княжество унаследовала его четырёхлетняя дочь <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%86%D0%B8%D1%8F,_%D0%BA%D0%BD%D1%8F%D0%B3%D0%B8%D0%BD%D1%8F_%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B8" title="Констанция, княгиня Антиохии">Констанция</a>. Некоторое время регентом Антиохии снова был Балдуин II, но в <a href="/wiki/1131_%D0%B3%D0%BE%D0%B4" title="1131 год">1131 году</a> он скончался и власть перешла к его дочери Мелисанде и её мужу <a class="mw-redirect" href="/wiki/%D0%A4%D1%83%D0%BB%D1%8C%D0%BA,_%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D1%8C_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D0%B0" title="Фульк, король Иерусалима">Фульку Анжуйскому</a>. В <a href="/wiki/1136_%D0%B3%D0%BE%D0%B4" title="1136 год">1136 году</a> наследница Антиохии Констанция в возрасте десяти лет была выдана замуж за <a class="mw-redirect" href="/wiki/%D0%A0%D0%B0%D0%B9%D0%BC%D1%83%D0%BD%D0%B4,_%D0%BA%D0%BD%D1%8F%D0%B7%D1%8C_%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B8" title="Раймунд, князь Антиохии">Раймунда де Пуатье</a>, который был в три раза старше своей юной жены<sup class="reference" id="cite_ref-Richar63_11-0"><a href="#cite_note-Richar63-11">[11]</a></sup><sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>.
</p><p>Подобно своим предшественникам, Раймунд делал попытки продолжить экспансию на территории византийской провинции <a href="/wiki/%D0%9A%D0%B8%D0%BB%D0%B8%D0%BA%D0%B8%D1%8F" title="Киликия">Киликия</a>. Однако на сей раз ему <a href="/wiki/%D0%9A%D0%B8%D0%BB%D0%B8%D0%BA%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B5_%D0%B2%D0%BE%D0%B9%D0%BD%D1%8B" title="Киликийские войны">оказал серьёзное сопротивление</a> император <a href="/wiki/%D0%98%D0%BE%D0%B0%D0%BD%D0%BD_II_%D0%9A%D0%BE%D0%BC%D0%BD%D0%B8%D0%BD" title="Иоанн II Комнин">Иоанн II Комнин</a>, который в <a href="/wiki/1137_%D0%B3%D0%BE%D0%B4" title="1137 год">1137 году</a> принудил Раймунда принести ему клятву верности. Однако эта клятва осталась формальной. В <a href="/wiki/1142_%D0%B3%D0%BE%D0%B4" title="1142 год">1142 году</a> Иоанн умер, так и не осуществив своих планов по отвоеванию захваченных крестоносцами земель<sup class="reference" id="cite_ref-Richar63_11-1"><a href="#cite_note-Richar63-11">[11]</a></sup>.
</p>
<h2><span id=".D0.90.D0.BD.D1.82.D0.B8.D0.BE.D1.85.D0.B8.D1.8F_.D0.BF.D0.BE.D0.B4_.D1.81.D1.8E.D0.B7.D0.B5.D1.80.D0.B5.D0.BD.D0.B8.D1.82.D0.B5.D1.82.D0.BE.D0.BC_.D0.92.D0.B8.D0.B7.D0.B0.D0.BD.D1.82.D0.B8.D0.B8"></span><span class="mw-headline" id="Антиохия_под_сюзеренитетом_Византии">Антиохия под <a class="mw-redirect" href="/wiki/%D0%A1%D1%8E%D0%B7%D0%B5%D1%80%D0%B5%D0%BD%D0%B8%D1%82%D0%B5%D1%82" title="Сюзеренитет">сюзеренитетом</a> Византии</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B9%D1%81%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BD%D1%8F%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%BE&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Антиохия под сюзеренитетом Византии»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B9%D1%81%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BD%D1%8F%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%BE&amp;action=edit&amp;section=3" title="Редактировать код раздела «Антиохия под сюзеренитетом Византии»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Во время <a href="/wiki/%D0%92%D1%82%D0%BE%D1%80%D0%BE%D0%B9_%D0%BA%D1%80%D0%B5%D1%81%D1%82%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D0%BE%D1%85%D0%BE%D0%B4" title="Второй крестовый поход">2-го крестового похода</a>, вскоре после падения <a href="/wiki/%D0%AD%D0%B4%D0%B5%D1%81%D1%81%D0%B0" title="Эдесса">Эдессы</a> (<a class="mw-redirect" href="/wiki/1144" title="1144">1144</a>), Антиохия была атакована армией атабека <a href="/wiki/%D0%90%D0%BB%D0%B5%D0%BF%D0%BF%D0%BE" title="Алеппо">Алеппо</a> <a href="/wiki/%D0%9D%D1%83%D1%80_%D0%B0%D0%B4-%D0%94%D0%B8%D0%BD_%D0%9C%D0%B0%D1%85%D0%BC%D1%83%D0%B4" title="Нур ад-Дин Махмуд">Нур ад-Дина Махмуда</a>, в результате чего были потеряны восточные территории княжества. После гибели Раймунда де Пуатье в <a href="/wiki/1149_%D0%B3%D0%BE%D0%B4" title="1149 год">1149 году</a> в <a href="/wiki/%D0%91%D0%B8%D1%82%D0%B2%D0%B0_%D0%BF%D1%80%D0%B8_%D0%98%D0%BD%D0%B0%D0%B1%D0%B5" title="Битва при Инабе">сражении при Инабе</a> регентом княжества на время, пока овдовевшая Констанция не вышла в <a href="/wiki/1153_%D0%B3%D0%BE%D0%B4" title="1153 год">1153 году</a> замуж за <a href="/wiki/%D0%A0%D0%B5%D0%BD%D0%BE_%D0%B4%D0%B5_%D0%A8%D0%B0%D1%82%D0%B8%D0%BB%D1%8C%D0%BE%D0%BD" title="Рено де Шатильон">Рено де Шатильона</a>, был иерусалимский король <a href="/wiki/%D0%91%D0%B0%D0%BB%D0%B4%D1%83%D0%B8%D0%BD_III_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D1%81%D0%BA%D0%B8%D0%B9" title="Балдуин III Иерусалимский">Балдуин III</a><sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup>.
</p><p>Рено в <a href="/wiki/1158_%D0%B3%D0%BE%D0%B4" title="1158 год">1158 году</a> заключил с императором <a href="/wiki/%D0%9C%D0%B0%D0%BD%D1%83%D0%B8%D0%BB_I_%D0%9A%D0%BE%D0%BC%D0%BD%D0%B8%D0%BD" title="Мануил I Комнин">Мануилом I</a> мирное соглашение, согласно которому Антиохия становилась вассальной территорией Византии и обязалась поставлять воинов для службы в византийских войсках. И хотя этот договор ставил крестоносцев в не слишком выгодное подчинённое положение, предполагалось, что дружба с могущественной Византией защитит княжество от нападения <a href="/wiki/%D0%9D%D1%83%D1%80_%D0%B0%D0%B4-%D0%94%D0%B8%D0%BD_%D0%9C%D0%B0%D1%85%D0%BC%D1%83%D0%B4" title="Нур ад-Дин Махмуд">Нур ад-Дина</a><sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup>.
</p><p>В <a href="/wiki/1160_%D0%B3%D0%BE%D0%B4" title="1160 год">1160 году</a> Рено де Шатильон попал в плен к мусульманам (он оставался в их руках на протяжении шестнадцати лет и так никогда и не вернулся в Антиохию), и регентом княжества стал <a class="mw-redirect" href="/wiki/%D0%9F%D0%B0%D1%82%D1%80%D0%B8%D0%B0%D1%80%D1%85%D0%B8_%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B8" title="Патриархи Антиохии">патриарх Антиохии</a> Эмери Лиможский. <a href="/wiki/24_%D0%B4%D0%B5%D0%BA%D0%B0%D0%B1%D1%80%D1%8F" title="24 декабря">24 декабря</a> <a href="/wiki/1161_%D0%B3%D0%BE%D0%B4" title="1161 год">1161 года</a> византийский император Мануил I, желая после смерти супруги жениться на одной из дочерей христианских правителей, взял в жены дочь Констанции Марию.
</p><p>К тому времени сын Констанции достиг совершеннолетия и заявил матери о своих правах на престол. Констанция не захотела уступать власть и в <a href="/wiki/1163_%D0%B3%D0%BE%D0%B4" title="1163 год">1163 году</a> обратилась за поддержкой в Киликию, однако жители Антиохии взбунтовались, изгнали Констанцию, а <a class="mw-redirect" href="/wiki/%D0%91%D0%BE%D1%8D%D0%BC%D1%83%D0%BD%D0%B4_III,_%D0%BA%D0%BD%D1%8F%D0%B7%D1%8C_%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B8" title="Боэмунд III, князь Антиохии">Боэмунд III</a> стал князем. В <a href="/wiki/1164_%D0%B3%D0%BE%D0%B4" title="1164 год">1164 году</a> во время <a href="/wiki/%D0%91%D0%B8%D1%82%D0%B2%D0%B0_%D0%BF%D1%80%D0%B8_%D0%A5%D0%B0%D1%80%D0%B8%D0%BC%D0%B5" title="Битва при Хариме">битвы при Хариме</a> Боэмунд попал в плен к Нуреддину, после чего <a href="/wiki/%D0%93%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B3%D1%80%D0%B0%D0%BD%D0%B8%D1%86%D0%B0" title="Государственная граница">граница</a> между Антиохией и <a href="/wiki/%D0%90%D0%BB%D0%B5%D0%BF%D0%BF%D0%BE" title="Алеппо">Алеппо</a> переместилась и стала проходить по реке <a class="mw-redirect" href="/wiki/%D0%9E%D1%80%D0%BE%D0%BD%D1%82_(%D1%80%D0%B5%D0%BA%D0%B0)" title="Оронт (река)">Оронт</a><sup class="reference" id="cite_ref-autogenerated4_15-0"><a href="#cite_note-autogenerated4-15">[15]</a></sup>. После того, как в <a href="/wiki/1165_%D0%B3%D0%BE%D0%B4" title="1165 год">1165 году</a> за Боэмунда был уплачен огромный выкуп, он вернулся в Антиохию и вскоре женился на одной из племянниц Мануила I<sup class="reference" id="cite_ref-autogenerated4_15-1"><a href="#cite_note-autogenerated4-15">[15]</a></sup>.
</p><p>После смерти Мануила I в <a href="/wiki/1180_%D0%B3%D0%BE%D0%B4" title="1180 год">1180 году</a> союз Антиохии и Византии, на протяжении двадцати лет надёжно защищавший территории княжества от вторжения мусульман, распался. Тем не менее Антиохия при помощи итальянского флота благополучно пережила нападение <a class="mw-redirect" href="/wiki/%D0%A1%D0%B0%D0%BB%D0%B0%D0%B4%D0%B8%D0%BD" title="Саладин">Саладина</a> на Иерусалимское королевство (<a class="mw-redirect" href="/wiki/1187" title="1187">1187</a>). Княжество Антиохия — равно как и <a class="mw-redirect" href="/wiki/%D0%93%D1%80%D0%B0%D1%84%D1%81%D1%82%D0%B2%D0%BE_%D0%A2%D1%80%D0%B8%D0%BF%D0%BE%D0%BB%D0%B8" title="Графство Триполи">графство Триполи</a> — не принимало участия в <a href="/wiki/%D0%A2%D1%80%D0%B5%D1%82%D0%B8%D0%B9_%D0%BA%D1%80%D0%B5%D1%81%D1%82%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D0%BE%D1%85%D0%BE%D0%B4" title="Третий крестовый поход">3-м крестовом походе</a> и соприкоснулось с ним лишь косвенно, когда в <a href="/wiki/1190_%D0%B3%D0%BE%D0%B4" title="1190 год">1190 году</a> в город пришли остатки армии <a href="/wiki/%D0%A4%D1%80%D0%B8%D0%B4%D1%80%D0%B8%D1%85_I_%D0%91%D0%B0%D1%80%D0%B1%D0%B0%D1%80%D0%BE%D1%81%D1%81%D0%B0" title="Фридрих I Барбаросса">Фридриха Барбароссы</a>, чтобы похоронить тело великого короля.
</p><p>В <a href="/wiki/1201_%D0%B3%D0%BE%D0%B4" title="1201 год">1201 году</a> Боэмунд III умер. Право наследования престола стали оспаривать его сын <a class="mw-redirect" href="/wiki/%D0%91%D0%BE%D1%8D%D0%BC%D1%83%D0%BD%D0%B4_IV,_%D0%BA%D0%BD%D1%8F%D0%B7%D1%8C_%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B8" title="Боэмунд IV, князь Антиохии">граф Триполи</a>, которого тоже звали Боэмунд, и внук <a class="mw-redirect" href="/wiki/%D0%A0%D0%B0%D0%B9%D0%BC%D1%83%D0%BD%D0%B4_%D0%A0%D1%83%D0%BF%D0%B5%D0%BD,_%D0%BA%D0%BD%D1%8F%D0%B7%D1%8C_%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B8" title="Раймунд Рупен, князь Антиохии">Раймунд Рупен</a>, по матери, армянской принцессе Алисе, приходившийся внуком королю Киликии <a href="/wiki/%D0%9B%D0%B5%D0%B2%D0%BE%D0%BD_II" title="Левон II">Левону II</a>.
</p><p>В <a href="/wiki/1207_%D0%B3%D0%BE%D0%B4" title="1207 год">1207 году</a> Боэмунд наконец был провозглашён князем Антиохии под именем Боэмунда IV и правил княжеством до своей смерти в <a href="/wiki/1233_%D0%B3%D0%BE%D0%B4" title="1233 год">1233 году</a> (с трёхлетним перерывом с <a class="mw-redirect" href="/wiki/1216" title="1216">1216</a> по <a href="/wiki/1219_%D0%B3%D0%BE%D0%B4" title="1219 год">1219 года</a>, когда власть находилась в руках Раймунда Рупена). Боэмунду наследовал сын <a class="mw-redirect" href="/wiki/%D0%91%D0%BE%D1%8D%D0%BC%D1%83%D0%BD%D0%B4_V,_%D0%BA%D0%BD%D1%8F%D0%B7%D1%8C_%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B8" title="Боэмунд V, князь Антиохии">Боэмунд V</a>, один из предводителей <a href="/wiki/%D0%9F%D1%8F%D1%82%D1%8B%D0%B9_%D0%BA%D1%80%D0%B5%D1%81%D1%82%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D0%BE%D1%85%D0%BE%D0%B4" title="Пятый крестовый поход">5-го</a>, <a href="/wiki/%D0%A8%D0%B5%D1%81%D1%82%D0%BE%D0%B9_%D0%BA%D1%80%D0%B5%D1%81%D1%82%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D0%BE%D1%85%D0%BE%D0%B4" title="Шестой крестовый поход">6-го</a> и <a href="/wiki/%D0%A1%D0%B5%D0%B4%D1%8C%D0%BC%D0%BE%D0%B9_%D0%BA%D1%80%D0%B5%D1%81%D1%82%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D0%BE%D1%85%D0%BE%D0%B4" title="Седьмой крестовый поход">7-го</a> крестовых походов<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup><sup class="reference" id="cite_ref-autogenerated3_17-0"><a href="#cite_note-autogenerated3-17">[17]</a></sup>.
</p>
<h2><span id=".D0.9F.D0.B0.D0.B4.D0.B5.D0.BD.D0.B8.D0.B5_.D0.BA.D0.BD.D1.8F.D0.B6.D0.B5.D1.81.D1.82.D0.B2.D0.B0"></span><span class="mw-headline" id="Падение_княжества">Падение княжества</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B9%D1%81%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BD%D1%8F%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%BE&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Падение княжества»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B9%D1%81%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BD%D1%8F%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%BE&amp;action=edit&amp;section=4" title="Редактировать код раздела «Падение княжества»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>В <a href="/wiki/1254_%D0%B3%D0%BE%D0%B4" title="1254 год">1254 году</a> <a class="mw-redirect" href="/wiki/%D0%91%D0%BE%D1%8D%D0%BC%D1%83%D0%BD%D0%B4_VI,_%D0%BA%D0%BD%D1%8F%D0%B7%D1%8C_%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B8" title="Боэмунд VI, князь Антиохии">Боэмунд VI</a>, сын Боэмунда V, женился на армянской принцессе Сибилле, положив конец конфликтам между двумя государствами. Тем не менее Антиохия переживала последние дни своего существования. В разгоревшемся между <a href="/wiki/%D0%9C%D0%B0%D0%BC%D0%BB%D1%8E%D0%BA%D0%B8" title="Мамлюки">мамлюками</a> и <a href="/wiki/%D0%9C%D0%BE%D0%BD%D0%B3%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F" title="Монгольская империя">монголами</a> конфликте Антиохия и <a class="mw-redirect" href="/wiki/%D0%9A%D0%B8%D0%BB%D0%B8%D0%BA%D0%B8%D0%B9%D1%81%D0%BA%D0%B0%D1%8F_%D0%90%D1%80%D0%BC%D0%B5%D0%BD%D0%B8%D1%8F" title="Киликийская Армения">Киликийская Армения</a> были на стороне последних, а Антиохия даже <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D0%BA%D0%BE-%D0%BC%D0%BE%D0%BD%D0%B3%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B5_%D1%81%D0%BE%D1%8E%D0%B7%D1%8B" title="Франко-монгольские союзы">заключила с монголами вассальный союз</a>. Поэтому после поражения войск хана <a href="/wiki/%D0%A5%D1%83%D0%BB%D0%B0%D0%B3%D1%83" title="Хулагу">Хулагу</a> в <a href="/wiki/%D0%91%D0%B8%D1%82%D0%B2%D0%B0_%D0%BF%D1%80%D0%B8_%D0%90%D0%B9%D0%BD-%D0%94%D0%B6%D0%B0%D0%BB%D1%83%D1%82%D0%B5" title="Битва при Айн-Джалуте">битве при Айн-Джалуте</a> (<a href="/wiki/1260_%D0%B3%D0%BE%D0%B4" title="1260 год">1260 год</a>) Антиохия попала под угрозу нападения мамлюкского <a href="/wiki/%D0%A1%D1%83%D0%BB%D1%82%D0%B0%D0%BD" title="Султан">султана</a> <a href="/wiki/%D0%91%D0%B5%D0%B9%D0%B1%D0%B0%D1%80%D1%81_I" title="Бейбарс I">Бейбарса</a>.
</p><p>Эта угроза воплотилась в реальность в <a href="/wiki/1268_%D0%B3%D0%BE%D0%B4" title="1268 год">1268 году</a>, когда Бейбарс взял Антиохию и подчинил себе северные территории <a href="/wiki/%D0%A1%D0%B8%D1%80%D0%B8%D1%8F" title="Сирия">Сирии</a><sup class="reference" id="cite_ref-autogenerated2_8-1"><a href="#cite_note-autogenerated2-8">[8]</a></sup>. Спустя 23 года пала <a class="mw-redirect" href="/wiki/%D0%90%D0%BA%D0%BA%D0%BE_(%D0%B3%D0%BE%D1%80%D0%BE%D0%B4)" title="Акко (город)">Акра</a>, и государства крестоносцев на Востоке прекратили своё существование. Титул князя Антиохии более не был поддержан правами на территории и отошёл <a href="/wiki/%D0%9A%D0%B8%D0%BF%D1%80%D1%81%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D0%B5%D0%B2%D1%81%D1%82%D0%B2%D0%BE" title="Кипрское королевство">королям Кипра</a><sup class="reference" id="cite_ref-autogenerated3_17-1"><a href="#cite_note-autogenerated3-17">[17]</a></sup>. Периодически титул жаловали в награду младшим отпрыскам королевских семей.
</p>
<h2><span id=".D0.93.D0.B5.D0.BE.D0.B3.D1.80.D0.B0.D1.84.D0.B8.D1.8F_.D0.B8_.D0.BD.D0.B0.D1.81.D0.B5.D0.BB.D0.B5.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="География_и_население">География и население</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B9%D1%81%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BD%D1%8F%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%BE&amp;veaction=edit&amp;section=5" title="Редактировать раздел «География и население»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B9%D1%81%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BD%D1%8F%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%BE&amp;action=edit&amp;section=5" title="Редактировать код раздела «География и население»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Княжество Антиохия было третьим по величине по сравнению с другими <a href="/wiki/%D0%93%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B0_%D0%BA%D1%80%D0%B5%D1%81%D1%82%D0%BE%D0%BD%D0%BE%D1%81%D1%86%D0%B5%D0%B2" title="Государства крестоносцев">государствами крестоносцев</a> в <a href="/wiki/%D0%9B%D0%B5%D0%B2%D0%B0%D0%BD%D1%82" title="Левант">Леванте</a> (ему уступало только <a class="mw-redirect" href="/wiki/%D0%93%D1%80%D0%B0%D1%84%D1%81%D1%82%D0%B2%D0%BE_%D0%A2%D1%80%D0%B8%D0%BF%D0%BE%D0%BB%D0%B8" title="Графство Триполи">графство Триполи</a>). Княжество занимало северо-восточное побережье <a href="/wiki/%D0%A1%D1%80%D0%B5%D0%B4%D0%B8%D0%B7%D0%B5%D0%BC%D0%BD%D0%BE%D0%B5_%D0%BC%D0%BE%D1%80%D0%B5" title="Средиземное море">Средиземного моря</a>, гранича с Киликийским царством и графством Эдесским на севере и с графством Триполи на юге. В <a href="/wiki/XIII_%D0%B2%D0%B5%D0%BA" title="XIII век">XIII веке</a> его население составляло около 30 000 человек и состояло преимущественно из <a href="/wiki/%D0%9F%D1%80%D0%B0%D0%B2%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%B8%D0%B5" title="Православие">православных</a> <a href="/wiki/%D0%93%D1%80%D0%B5%D0%BA%D0%B8" title="Греки">греков</a> и <a href="/wiki/%D0%90%D1%80%D0%BC%D1%8F%D0%BD%D0%B5" title="Армяне">армян</a>. Кроме того, за пределами города существовало некоторое количество мусульманских общин.
Подавляющее большинство крестоносцев, поселившихся в Антиохии, были родом из <a href="/wiki/%D0%9D%D0%BE%D1%80%D0%BC%D0%B0%D0%BD%D0%B4%D0%B8%D1%8F" title="Нормандия">Нормандии</a> и южной <a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D0%B8%D1%8F" title="Италия">Италии</a><sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup><sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup>.
</p>
