<p><b>Базилика Сан-Витале</b> (св. Виталия, <a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D1%8C%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Итальянский язык">итал.</a> <span lang="it" style="font-style:italic;">Basilica di San Vitale</span>) — раннехристианская <a href="/wiki/%D0%91%D0%B0%D0%B7%D0%B8%D0%BB%D0%B8%D0%BA%D0%B0" title="Базилика">базилика</a><sup class="reference" id="cite_ref-базилика_2-0"><a href="#cite_note-базилика-2">[2]</a></sup> в <a href="/wiki/%D0%A0%D0%B0%D0%B2%D0%B5%D0%BD%D0%BD%D0%B0" title="Равенна">Равенне</a> (<a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D0%B8%D1%8F" title="Италия">Италия</a>), важнейший памятник <a href="/wiki/%D0%98%D1%81%D0%BA%D1%83%D1%81%D1%81%D1%82%D0%B2%D0%BE_%D0%92%D0%B8%D0%B7%D0%B0%D0%BD%D1%82%D0%B8%D0%B8" title="Искусство Византии">византийского искусства</a> в <a href="/wiki/%D0%97%D0%B0%D0%BF%D0%B0%D0%B4%D0%BD%D0%B0%D1%8F_%D0%95%D0%B2%D1%80%D0%BE%D0%BF%D0%B0" title="Западная Европа">Западной Европе</a>. В <a href="/wiki/1996_%D0%B3%D0%BE%D0%B4" title="1996 год">1996 году</a> в составе раннехристианских памятников Равенны включена в число объектов <a href="/wiki/%D0%92%D1%81%D0%B5%D0%BC%D0%B8%D1%80%D0%BD%D0%BE%D0%B5_%D0%BD%D0%B0%D1%81%D0%BB%D0%B5%D0%B4%D0%B8%D0%B5" title="Всемирное наследие">Всемирного наследия</a> <a href="/wiki/%D0%AE%D0%9D%D0%95%D0%A1%D0%9A%D0%9E" title="ЮНЕСКО">ЮНЕСКО</a>. Выделяется среди восьми раннехристианских памятников Равенны совершенством своих <a href="/wiki/%D0%92%D0%B8%D0%B7%D0%B0%D0%BD%D1%82%D0%B8%D0%B9%D1%81%D0%BA%D0%B0%D1%8F_%D0%BC%D0%BE%D0%B7%D0%B0%D0%B8%D0%BA%D0%B0" title="Византийская мозаика">византийских мозаик</a>, которые не имеют равных за пределами <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD%D0%BE%D0%BF%D0%BE%D0%BB%D1%8C" title="Константинополь">Константинополя</a>.
</p>

<h2><span id=".D0.98.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.8F"></span><span class="mw-headline" id="История">История</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B0%D0%BD-%D0%92%D0%B8%D1%82%D0%B0%D0%BB%D0%B5&amp;veaction=edit&amp;section=1" title="Редактировать раздел «История»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B0%D0%BD-%D0%92%D0%B8%D1%82%D0%B0%D0%BB%D0%B5&amp;action=edit&amp;section=1" title="Редактировать код раздела «История»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Базилика была заложена в <a href="/wiki/527_%D0%B3%D0%BE%D0%B4" title="527 год">527 году</a> равеннским епископом <a href="/wiki/%D0%AD%D0%BA%D0%BA%D0%BB%D0%B5%D0%B7%D0%B8%D0%B9_%D0%A0%D0%B0%D0%B2%D0%B5%D0%BD%D0%BD%D1%81%D0%BA%D0%B8%D0%B9" title="Экклезий Равеннский">Экклезием</a> после его возвращения из <a href="/wiki/%D0%92%D0%B8%D0%B7%D0%B0%D0%BD%D1%82%D0%B8%D1%8F" title="Византия">Византии</a>, где он вместе с <a href="/wiki/%D0%9F%D0%B0%D0%BF%D1%81%D1%82%D0%B2%D0%BE" title="Папство">папой</a> <a href="/wiki/%D0%98%D0%BE%D0%B0%D0%BD%D0%BD_I_(%D0%BF%D0%B0%D0%BF%D0%B0_%D1%80%D0%B8%D0%BC%D1%81%D0%BA%D0%B8%D0%B9)" title="Иоанн I (папа римский)">Иоанном I</a> выполнял дипломатическую миссию по поручению <a href="/wiki/%D0%A2%D0%B5%D0%BE%D0%B4%D0%BE%D1%80%D0%B8%D1%85_%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B8%D0%B9" title="Теодорих Великий">Теодориха Великого</a><sup class="reference" id="cite_ref-РАИ-25_3-0"><a href="#cite_note-РАИ-25-3">[3]</a></sup>. Храм был освящён в честь раннехристианского <a href="/wiki/%D0%9C%D1%83%D1%87%D0%B5%D0%BD%D0%B8%D0%BA" title="Мученик">мученика</a> святого <a href="/wiki/%D0%92%D0%B8%D1%82%D0%B0%D0%BB%D0%B8%D0%B9_%D0%9C%D0%B8%D0%BB%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9" title="Виталий Миланский">Виталия Миланского</a>, чьё изображение помещено в <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%85%D0%B0" title="Конха">конхе</a> <a href="/wiki/%D0%90%D0%BF%D1%81%D0%B8%D0%B4%D0%B0" title="Апсида">апсиды</a>.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> Строительство велось на средства греческого <a href="/wiki/%D0%90%D1%80%D0%B3%D0%B8%D1%80%D0%BE%D0%BF%D1%80%D0%B0%D1%82%D1%8B" title="Аргиропраты">ростовщика</a> <a href="/wiki/%D0%AE%D0%BB%D0%B8%D0%B0%D0%BD_%D0%90%D1%80%D0%B3%D0%B5%D0%BD%D1%82%D0%B0%D1%80%D0%B8%D0%B9" title="Юлиан Аргентарий">Юлиана Аргентария</a> (Серебряника), который финансировал также строительство базилики <a href="/wiki/%D0%A1%D0%B0%D0%BD%D1%82-%D0%90%D0%BF%D0%BE%D0%BB%D0%BB%D0%B8%D0%BD%D0%B0%D1%80%D0%B5-%D0%B8%D0%BD-%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B5" title="Сант-Аполлинаре-ин-Классе">Сант-Аполлинаре-ин-Классе</a>. Освящение совершил <a href="/wiki/19_%D0%B0%D0%BF%D1%80%D0%B5%D0%BB%D1%8F" title="19 апреля">19 апреля</a> <a href="/wiki/548_%D0%B3%D0%BE%D0%B4" title="548 год">548 года</a> епископ <a href="/wiki/%D0%9C%D0%B0%D0%BA%D1%81%D0%B8%D0%BC%D0%B8%D0%B0%D0%BD_(%D0%B5%D0%BF%D0%B8%D1%81%D0%BA%D0%BE%D0%BF_%D0%A0%D0%B0%D0%B2%D0%B5%D0%BD%D0%BD%D1%81%D0%BA%D0%B8%D0%B9)" title="Максимиан (епископ Равеннский)">Максимиан</a>.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> Всё внутреннее мозаичное убранство церкви было создано одновременно в 546—547 годах; различия в стиле академик <a href="/wiki/%D0%9B%D0%B0%D0%B7%D0%B0%D1%80%D0%B5%D0%B2,_%D0%92%D0%B8%D0%BA%D1%82%D0%BE%D1%80_%D0%9D%D0%B8%D0%BA%D0%B8%D1%82%D0%B8%D1%87" title="Лазарев, Виктор Никитич">В. Н. Лазарев</a> объясняет участием в работе разных мастеров.<sup class="reference" id="cite_ref-lazarev_6-0"><a href="#cite_note-lazarev-6">[6]</a></sup>
</p><p>В <a href="/wiki/XIII_%D0%B2%D0%B5%D0%BA" title="XIII век">XIII веке</a> к южной стене церкви пристроена <a href="/wiki/%D0%9A%D0%BE%D0%BB%D0%BE%D0%BA%D0%BE%D0%BB%D1%8C%D0%BD%D1%8F" title="Колокольня">колокольня</a> и реконструированы деревянные <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B5%D0%BA%D1%80%D1%8B%D1%82%D0%B8%D0%B5" title="Перекрытие">перекрытия</a> <a href="/wiki/%D0%90%D1%80%D0%BA%D0%B0%D0%B4%D0%B0_(%D0%B0%D1%80%D1%85%D0%B8%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D0%B0)" title="Аркада (архитектура)">аркад</a>. Масштабная реконструкция храма проводилась в <a href="/wiki/XVI_%D0%B2%D0%B5%D0%BA" title="XVI век">XVI веке</a>: ввиду подъёма <a class="mw-redirect" href="/wiki/%D0%93%D1%80%D1%83%D0%BD%D1%82%D0%BE%D0%B2%D1%8B%D0%B5_%D0%B2%D0%BE%D0%B4%D1%8B" title="Грунтовые воды">грунтовых вод</a> был повышен на 80 см уровень пола, обновлён <a href="/wiki/%D0%9F%D1%80%D0%B5%D1%81%D0%B2%D0%B8%D1%82%D0%B5%D1%80%D0%B8%D0%B9_(%D1%85%D1%80%D0%B0%D0%BC)" title="Пресвитерий (храм)">пресбитерий</a>, убраны деревянные <a href="/wiki/%D0%A5%D0%BE%D1%80%D1%8B" title="Хоры">хоры</a>, перестроены внутренний дворик (<a href="/wiki/1562_%D0%B3%D0%BE%D0%B4" title="1562 год">1562 год</a>) и южный <a href="/wiki/%D0%9F%D0%BE%D1%80%D1%82%D0%B0%D0%BB_(%D0%B0%D1%80%D1%85%D0%B8%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D0%B0)" title="Портал (архитектура)">портал</a> здания. В <a href="/wiki/1688_%D0%B3%D0%BE%D0%B4" title="1688 год">1688 году</a> <a href="/wiki/%D0%97%D0%B5%D0%BC%D0%BB%D0%B5%D1%82%D1%80%D1%8F%D1%81%D0%B5%D0%BD%D0%B8%D0%B5" title="Землетрясение">землетрясением</a> была разрушена <a href="/wiki/%D0%9A%D0%BE%D0%BB%D0%BE%D0%BA%D0%BE%D0%BB%D1%8C%D0%BD%D1%8F" title="Колокольня">колокольня</a> XIII века, восстановленная в 1696—1698 годах.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>
</p><p>В <a href="/wiki/1780_%D0%B3%D0%BE%D0%B4" title="1780 год">1780 году</a> <a href="/wiki/%D0%9A%D1%83%D0%BF%D0%BE%D0%BB" title="Купол">купол</a> <a href="/wiki/%D0%A0%D0%BE%D1%82%D0%BE%D0%BD%D0%B4%D0%B0" title="Ротонда">ротонды</a> и подкупольные ниши, не имевшие при строительстве церкви каких-либо украшений, были расписаны <a href="/wiki/%D0%A4%D1%80%D0%B5%D1%81%D0%BA%D0%B0" title="Фреска">фресками</a> работы <a href="/wiki/%D0%91%D0%BE%D0%BB%D0%BE%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D1%88%D0%BA%D0%BE%D0%BB%D0%B0_%D0%B6%D0%B8%D0%B2%D0%BE%D0%BF%D0%B8%D1%81%D0%B8" title="Болонская школа живописи">болонцев</a> Бароцци и <a href="/wiki/%D0%93%D0%B0%D0%BD%D0%B4%D0%BE%D0%BB%D1%8C%D1%84%D0%B8,_%D0%93%D0%B0%D1%8D%D1%82%D0%B0%D0%BD%D0%BE" title="Гандольфи, Гаэтано">Гандольфи</a> и <a href="/wiki/%D0%92%D0%B5%D0%BD%D0%B5%D1%86%D0%B8%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D1%88%D0%BA%D0%BE%D0%BB%D0%B0_%D0%B6%D0%B8%D0%B2%D0%BE%D0%BF%D0%B8%D1%81%D0%B8" title="Венецианская школа живописи">венецианца</a> <a href="/wiki/%D0%93%D1%83%D0%B0%D1%80%D0%B0%D0%BD%D0%B0,_%D0%AF%D0%BA%D0%BE%D0%BF%D0%BE" title="Гуарана, Якопо">Гуараны</a> (<a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D1%8C%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Итальянский язык">итал.</a> <span lang="it" style="font-style:italic;">Jacopo Guarana</span>).
</p>
<h2><span id=".D0.9E.D1.81.D0.BE.D0.B1.D0.B5.D0.BD.D0.BD.D0.BE.D1.81.D1.82.D0.B8_.D0.B0.D1.80.D1.85.D0.B8.D1.82.D0.B5.D0.BA.D1.82.D1.83.D1.80.D1.8B"></span><span class="mw-headline" id="Особенности_архитектуры">Особенности архитектуры</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B0%D0%BD-%D0%92%D0%B8%D1%82%D0%B0%D0%BB%D0%B5&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Особенности архитектуры»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B0%D0%BD-%D0%92%D0%B8%D1%82%D0%B0%D0%BB%D0%B5&amp;action=edit&amp;section=2" title="Редактировать код раздела «Особенности архитектуры»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>

<p>Базилика построена в форме восьмиугольного <a href="/wiki/%D0%9C%D0%B0%D1%80%D1%82%D0%B8%D1%80%D0%B8%D1%83%D0%BC" title="Мартириум">мартирия</a> византийского типа, близкого по архитектуре к <a href="/wiki/%D0%A6%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C_%D0%A1%D0%B2%D1%8F%D1%82%D1%8B%D1%85_%D0%A1%D0%B5%D1%80%D0%B3%D0%B8%D1%8F_%D0%B8_%D0%92%D0%B0%D0%BA%D1%85%D0%B0" title="Церковь Святых Сергия и Вакха">церкви Сергия и Вакха</a> в <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD%D0%BE%D0%BF%D0%BE%D0%BB%D1%8C" title="Константинополь">Константинополе</a>.<sup class="reference" id="cite_ref-lazarev_6-1"><a href="#cite_note-lazarev-6">[6]</a></sup> Наружные стены не имеют каких-либо декоративных элементов и расчленены вертикальными и горизонтальными <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%82%D1%80%D1%84%D0%BE%D1%80%D1%81" title="Контрфорс">контрфорсами</a>. Здание увенчано гранёным <a href="/wiki/%D0%91%D0%B0%D1%80%D0%B0%D0%B1%D0%B0%D0%BD_(%D0%B0%D1%80%D1%85%D0%B8%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D0%B0)" title="Барабан (архитектура)">барабаном</a> <a href="/wiki/%D0%9A%D1%83%D0%BF%D0%BE%D0%BB" title="Купол">купола</a>. Архитектура Сан-Витале сочетает элементы классического римского зодчества (купол, порталы, ступенчатые башни) с византийскими веяниями (трёхлопастная <a class="mw-redirect" href="/wiki/%D0%90%D0%BF%D1%81%D0%B8%D0%B4%D0%B0_(%D0%B0%D1%80%D1%85%D0%B8%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D0%B0)" title="Апсида (архитектура)">апсида</a>, узкие кирпичи, трапециевидные <a href="/wiki/%D0%9A%D0%B0%D0%BF%D0%B8%D1%82%D0%B5%D0%BB%D1%8C" title="Капитель">капители</a>, так называемая подушкообразная капитель — пульван, или пульвино<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>, и т. д.). Низ внутренних стен базилики облицован <a href="/wiki/%D0%9C%D1%80%D0%B0%D0%BC%D0%BE%D1%80" title="Мрамор">мрамором</a>, наборный пол украшен геометрическим орнаментом.
</p>
<figure class="mw-default-size mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:San_Vitale_in_Ravenna_01.jpg"><img class="mw-file-element" data-file-height="1296" data-file-width="972" decoding="async" height="293" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/36/San_Vitale_in_Ravenna_01.jpg/220px-San_Vitale_in_Ravenna_01.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/36/San_Vitale_in_Ravenna_01.jpg/330px-San_Vitale_in_Ravenna_01.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/3/36/San_Vitale_in_Ravenna_01.jpg/440px-San_Vitale_in_Ravenna_01.jpg 2x" width="220"/></a><figcaption><center>Аркада ротонды</center></figcaption></figure>
<figure class="mw-default-size" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:San_Vitale_00.jpg"><img class="mw-file-element" data-file-height="589" data-file-width="376" decoding="async" height="345" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/a0/San_Vitale_00.jpg/220px-San_Vitale_00.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/a0/San_Vitale_00.jpg/330px-San_Vitale_00.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/a/a0/San_Vitale_00.jpg 2x" width="220"/></a><figcaption><center>Мозаичное украшение <a href="/wiki/%D0%9A%D0%B0%D0%BF%D0%B8%D1%82%D0%B5%D0%BB%D1%8C" title="Капитель">капители</a></center></figcaption></figure>
<p>Конструкцию здания поддерживают восемь центральных опор, на которых держится <a href="/wiki/%D0%9A%D1%83%D0%BF%D0%BE%D0%BB" title="Купол">купол</a> <a href="/wiki/%D0%94%D0%B8%D0%B0%D0%BC%D0%B5%D1%82%D1%80" title="Диаметр">диаметром</a> 16 метров. Для уменьшения бокового давления куполу придана <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%83%D1%81" title="Конус">конусообразная</a> форма. Купол воздвигнут из лёгкого материала — вдетых друг в друга <a href="/wiki/%D0%93%D0%BB%D0%B8%D0%BD%D0%B0" title="Глина">глиняных</a> <a href="/wiki/%D0%A2%D1%80%D1%83%D0%B1%D0%B0_(%D0%B8%D0%B7%D0%B4%D0%B5%D0%BB%D0%B8%D0%B5)" title="Труба (изделие)">труб</a>, закреплённых одна над другой в становящихся всё более узкими горизонтальных <a class="mw-disambig" href="/wiki/%D0%9A%D0%BE%D0%BB%D1%8C%D1%86%D0%BE" title="Кольцо">кольцах</a><sup class="reference" id="cite_ref-РАИ-30_9-0"><a href="#cite_note-РАИ-30-9">[9]</a></sup>. Опорные столбы образуют в центре храма <a href="/wiki/%D0%A0%D0%BE%D1%82%D0%BE%D0%BD%D0%B4%D0%B0" title="Ротонда">ротонду</a>, на втором ярусе которой расположены <a href="/wiki/%D0%A5%D0%BE%D1%80%D1%8B" title="Хоры">хоры</a>. В промежутки между столбами ротонды помещены полукруглые двухэтажные <a href="/wiki/%D0%90%D1%80%D0%BA%D0%B0%D0%B4%D0%B0_(%D0%B0%D1%80%D1%85%D0%B8%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D0%B0)" title="Аркада (архитектура)">аркады</a>, расположенные по дугам, выгнутым к внешним стенам. Благодаря описанному устройству внутренняя часть церкви кажется залитой светом, а окружающие её <a href="/wiki/%D0%93%D0%B0%D0%BB%D0%B5%D1%80%D0%B5%D1%8F" title="Галерея">галереи</a> искусственно погружены в мистическую полутьму, что сразу же обращает внимание входящего на мозаики <a class="mw-redirect" href="/wiki/%D0%90%D0%BF%D1%81%D0%B8%D0%B4%D0%B0_(%D0%B0%D1%80%D1%85%D0%B8%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D0%B0)" title="Апсида (архитектура)">апсиды</a> и <a class="mw-redirect" href="/wiki/%D0%9F%D1%80%D0%B5%D1%81%D0%B1%D0%B8%D1%82%D0%B5%D1%80%D0%B8%D0%B9" title="Пресбитерий">пресбитерия</a><sup class="reference" id="cite_ref-РАИ-26-30_10-0"><a href="#cite_note-РАИ-26-30-10">[10]</a></sup>. Использование данного архитектурного решения привело к достижению следующего пространственного эффекта:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
Выступающие за кольцо подкупольных столбов высокие аркады открывают центральное пространство глазам посетителя сразу же, как только он входит в церковь. Ещё находясь во внешнем обходе, он чувствует, что высоко поднятый купол словно вбирает в себя всё внутреннее пространство храма. При взгляде же из центра храма аркады воспринимаются как ещё одно кольцо опор, находящееся между столбами и внешними стенами и зрительно увеличивающее интерьер. Наконец, пространственный эффект усиливается своеобразной формой столбов, объём которых не воспринимается зрителем.<sup class="reference" id="cite_ref-Ist_isk_11-0"><a href="#cite_note-Ist_isk-11">[11]</a></sup>
</p>
</blockquote>
<p>Перед апсидой, освещённой тремя высокими окнами, круговой обход ротонды прерывается пресбитерием, который окружён двухэтажными аркадами. <a href="/wiki/%D0%90%D0%BB%D1%82%D0%B0%D1%80%D1%8C" title="Алтарь">Алтарь</a> вынесен за пределы апсиды в пресбитерий, а в ней установлена стационарная мраморная <a href="/wiki/%D0%9A%D0%B0%D1%84%D0%B5%D0%B4%D1%80%D0%B0_(%D1%85%D1%80%D0%B8%D1%81%D1%82%D0%B8%D0%B0%D0%BD%D1%81%D1%82%D0%B2%D0%BE)" title="Кафедра (христианство)">кафедра</a>. <a href="/wiki/%D0%9A%D0%B0%D0%BF%D0%B8%D1%82%D0%B5%D0%BB%D1%8C" title="Капитель">Капители</a> аркады пресбитерия выполнены в форме ажурных корзинок и украшены изображением креста, помещённого между двумя агнцами. Такое оформление, однако, разрушает тектоническую природу ордера.<sup class="reference" id="cite_ref-Ist_isk_11-1"><a href="#cite_note-Ist_isk-11">[11]</a></sup>
</p><p>К числу особенностей Сан-Витале следует отнести и необычное устройство <a href="/wiki/%D0%9F%D1%80%D0%B8%D1%82%D0%B2%D0%BE%D1%80" title="Притвор">нартекса</a>, расположенного здесь под углом к основной <a class="mw-disambig" href="/wiki/%D0%9E%D1%81%D1%8C" title="Ось">оси</a> здания, проходящей через пресбитерий. Причины такого устройства нартекса не установлены. По мнению исследователей, архитектор мог таким образом сохранить память о ранее существовавших на месте храма <a href="/wiki/%D0%A7%D0%B0%D1%81%D0%BE%D0%B2%D0%BD%D1%8F" title="Часовня">часовнях</a>, теснее включить в основной объём здания две <a href="/wiki/%D0%9B%D0%B5%D1%81%D1%82%D0%BD%D0%B8%D1%86%D0%B0" title="Лестница">лестничных</a> <a href="/wiki/%D0%91%D0%B0%D1%88%D0%BD%D1%8F" title="Башня">башни</a> или просто создать, помимо основного входа по оси здания, ещё один в боковой части церкви<sup class="reference" id="cite_ref-РАИ-26-30_10-1"><a href="#cite_note-РАИ-26-30-10">[10]</a></sup>.
</p><p>Базилика в Равенне послужила образцом для архитектуры <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%BE%D0%BB%D0%B8%D0%BD%D0%B3%D1%81%D0%BA%D0%BE%D0%B5_%D0%B2%D0%BE%D0%B7%D1%80%D0%BE%D0%B6%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5" title="Каролингское возрождение">Каролингского возрождения</a> в целом и, в частности, для её центрального произведения — <a href="/wiki/%D0%9A%D0%B0%D0%BF%D0%B5%D0%BB%D0%BB%D0%B0_%D0%9A%D0%B0%D1%80%D0%BB%D0%B0_%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%BE%D0%B3%D0%BE" title="Капелла Карла Великого">дворцовой капеллы</a> в <a href="/wiki/%D0%90%D1%85%D0%B5%D0%BD" title="Ахен">Ахене</a>. <a href="/wiki/%D0%91%D1%80%D1%83%D0%BD%D0%B5%D0%BB%D0%BB%D0%B5%D1%81%D0%BA%D0%B8,_%D0%A4%D0%B8%D0%BB%D0%B8%D0%BF%D0%BF%D0%BE" title="Брунеллески, Филиппо">Филиппо Брунеллески</a> изучал конструкцию купола Сан-Витале при разработке проекта первого европейского купола Нового времени (<a href="/wiki/%D0%A4%D0%BB%D0%BE%D1%80%D0%B5%D0%BD%D1%86%D0%B8%D1%8F" title="Флоренция">флорентийский</a> собор <a href="/wiki/%D0%A1%D0%B0%D0%BD%D1%82%D0%B0-%D0%9C%D0%B0%D1%80%D0%B8%D1%8F-%D0%B4%D0%B5%D0%BB%D1%8C-%D0%A4%D1%8C%D0%BE%D1%80%D0%B5" title="Санта-Мария-дель-Фьоре">Санта-Мария-дель-Фьоре</a>). Особенно он отметил то, что для облегчения купола равеннской церкви вместо наполнителя использовались полые глиняные сосуды.
</p>

<p>Рабы сквозь римские ворота 
<br/>Уже не ввозят мозаик. 
<br/>И догорает позолота 
<br/>В стенах прохладных базилик.<i></i>
</p>

<p>Основное пространство базилики украшено мраморной инкрустацией, а вогнутые поверхности апсиды (аркады, <a href="/wiki/%D0%A1%D0%B2%D0%BE%D0%B4" title="Свод">своды</a>, <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%85%D0%B0" title="Конха">конха</a>) стены (<a href="/wiki/%D0%92%D0%B8%D0%BC%D0%B0" title="Вима">вимы</a>) пресбитерия покрыты <a href="/wiki/%D0%92%D0%B8%D0%B7%D0%B0%D0%BD%D1%82%D0%B8%D0%B9%D1%81%D0%BA%D0%B0%D1%8F_%D0%BC%D0%BE%D0%B7%D0%B0%D0%B8%D0%BA%D0%B0" title="Византийская мозаика">византийской мозаикой</a>. Мозаики Сан-Витале были призваны продемонстрировать западному миру могущество и безупречный вкус <a class="mw-redirect" href="/wiki/%D0%92%D0%B8%D0%B7%D0%B0%D0%BD%D1%82%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B5_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%BE%D1%80%D1%8B" title="Византийские императоры">византийского императора</a> <a href="/wiki/%D0%AE%D1%81%D1%82%D0%B8%D0%BD%D0%B8%D0%B0%D0%BD_I" title="Юстиниан I">Юстиниана</a> во время недолгого владычества византийцев в Италии.
</p><p>Мозаики Сан-Витале являются редким для Европы образцом раннехристианской монументальной живописи, созданной в технике византийской мозаики.<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> Особую значимость представляют прижизненные портреты императора Юстиниана и его супруги Феодоры.
</p><p>С помощью мозаики мастера смогли подчеркнуть архитектурные элементы базилики и акцентировать символическую связь элемента конструкции и нанесённого на неё изображения:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
<i>Мозаики, которые покрывают весь этот интерьер за исключением цоколя, прекрасно выявляют конструктивный смысл архитектуры. Люнеты, распалубки, стены, арки, ниши и своды эффектно выделены различными типами декора. Так, рёбра крестового свода усилены растительными гирляндами, в то время как фигуры ангелов, олицетворяющих мощь несущих конструкций, поддерживают центральный медальон</i>.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup>
</p>
<div class="ts-Конец_цитаты-source">— <cite><a href="/wiki/%D0%94%D0%B5%D0%BC%D1%83%D1%81,_%D0%9E%D1%82%D1%82%D0%BE" title="Демус, Отто">Отто Демус</a>. <i>Мозаики византийских храмов</i></cite></div></blockquote>
<p>В боковых галереях находятся несколько раннехристианских <a href="/wiki/%D0%A1%D0%B0%D1%80%D0%BA%D0%BE%D1%84%D0%B0%D0%B3" title="Саркофаг">саркофагов</a>. Наиболее примечателен <a href="/wiki/%D0%9C%D1%80%D0%B0%D0%BC%D0%BE%D1%80" title="Мрамор">мраморный</a> саркофаг <a href="/wiki/V_%D0%B2%D0%B5%D0%BA" title="V век">V века</a>, несколько переделанный в середине <a href="/wiki/VI_%D0%B2%D0%B5%D0%BA" title="VI век">VI века</a>, в котором, как гласят <a href="/wiki/%D0%93%D1%80%D0%B5%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Греческий язык">греческая</a> и <a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Латинский язык">латинская</a> надписи на крышке, был погребён <a href="/wiki/%D0%A0%D0%B0%D0%B2%D0%B5%D0%BD%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8D%D0%BA%D0%B7%D0%B0%D1%80%D1%85%D0%B0%D1%82" title="Равеннский экзархат">равеннский экзарх</a> <a href="/wiki/%D0%98%D1%81%D0%B0%D0%B0%D0%BA_(%D1%8D%D0%BA%D0%B7%D0%B0%D1%80%D1%85_%D0%A0%D0%B0%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B)" title="Исаак (экзарх Равенны)">Исаак</a>. На боковых сторонах саркофага можно увидеть <a href="/wiki/%D0%91%D0%B0%D1%80%D0%B5%D0%BB%D1%8C%D0%B5%D1%84" title="Барельеф">барельефы</a>, изображающие <a href="/wiki/%D0%9F%D0%BE%D0%BA%D0%BB%D0%BE%D0%BD%D0%B5%D0%BD%D0%B8%D0%B5_%D0%B2%D0%BE%D0%BB%D1%85%D0%B2%D0%BE%D0%B2" title="Поклонение волхвов">поклонение волхвов</a>, <a class="mw-redirect" href="/wiki/%D0%92%D0%BE%D1%81%D0%BA%D1%80%D0%B5%D1%88%D0%B5%D0%BD%D0%B8%D0%B5_%D0%9B%D0%B0%D0%B7%D0%B0%D1%80%D1%8F" title="Воскрешение Лазаря">воскрешение Лазаря</a>, <a href="/wiki/%D0%94%D0%B0%D0%BD%D0%B8%D0%B8%D0%BB_(%D0%BF%D1%80%D0%BE%D1%80%D0%BE%D0%BA)" title="Даниил (пророк)">Даниила</a> во львином рву и <a href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82" title="Крест">крест</a> с двумя <a href="/wiki/%D0%9F%D0%B0%D0%B2%D0%BB%D0%B8%D0%BD%D1%8B" title="Павлины">павлинами</a><sup class="reference" id="cite_ref-РАИ-48_14-0"><a href="#cite_note-РАИ-48-14">[14]</a></sup>.
</p>

<ul><li><b>Конха апсиды</b></li></ul>
<p><a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%85%D0%B0" title="Конха">Конха</a> украшена мозаикой, представляющей <a href="/wiki/%D0%98%D0%B8%D1%81%D1%83%D1%81_%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%BE%D1%81" title="Иисус Христос">Иисуса Христа</a> в образе юноши с крестчатым <a href="/wiki/%D0%9D%D0%B8%D0%BC%D0%B1" title="Нимб">нимбом</a>, который восседает на лазоревой небесной сфере в окружении двух <a href="/wiki/%D0%90%D0%BD%D0%B3%D0%B5%D0%BB" title="Ангел">ангелов</a>. В одной руке Христос держит свиток, опечатанный семью печатями (<a href="/wiki/%D0%9E%D1%82%D0%BA%D1%80%D0%BE%D0%B2%D0%B5%D0%BD%D0%B8%D0%B5_%D0%98%D0%BE%D0%B0%D0%BD%D0%BD%D0%B0_%D0%91%D0%BE%D0%B3%D0%BE%D1%81%D0%BB%D0%BE%D0%B2%D0%B0" title="Откровение Иоанна Богослова">Откр.</a> <a class="extiw" href="https://ru.wikisource.org/wiki/%D0%9E%D1%82%D0%BA%D1%80%D0%BE%D0%B2%D0%B5%D0%BD%D0%B8%D0%B5_%D1%81%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%98%D0%BE%D0%B0%D0%BD%D0%BD%D0%B0_%D0%91%D0%BE%D0%B3%D0%BE%D1%81%D0%BB%D0%BE%D0%B2%D0%B0#5:1" title="s:Откровение святого Иоанна Богослова">5:1</a>), а другой протягивает <a href="/wiki/%D0%9C%D1%83%D1%87%D0%B5%D0%BD%D0%B8%D0%BA" title="Мученик">мученический</a> венец славы <a href="/wiki/%D0%92%D0%B8%D1%82%D0%B0%D0%BB%D0%B8%D0%B9_%D0%9C%D0%B8%D0%BB%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9" title="Виталий Миланский">святому Виталию</a>, которого подводит к нему ангел. Второй ангел представляет Иисусу <a href="/wiki/%D0%AD%D0%BA%D0%BA%D0%BB%D0%B5%D0%B7%D0%B8%D0%B9_%D0%A0%D0%B0%D0%B2%D0%B5%D0%BD%D0%BD%D1%81%D0%BA%D0%B8%D0%B9" title="Экклезий Равеннский">равеннского епископа Екклезия</a>, подносящего в дар макет основанной им базилики св. Виталия.
</p><p>Из-под ног Иисуса вытекают и струятся по каменистой земле, поросшей <a href="/wiki/%D0%9B%D0%B8%D0%BB%D0%B8%D1%8F" title="Лилия">лилиями</a>, четыре реки <a href="/wiki/%D0%AD%D0%B4%D0%B5%D0%BC" title="Эдем">Эдема</a>: Фисон, Гихон, <a href="/wiki/%D0%A2%D0%B8%D0%B3%D1%80_(%D1%80%D0%B5%D0%BA%D0%B0)" title="Тигр (река)">Хиддекель</a> и <a href="/wiki/%D0%95%D0%B2%D1%84%D1%80%D0%B0%D1%82" title="Евфрат">Евфрат</a>. Эта деталь прославляет Иисуса как <i>источник воды живой</i> (<a href="/wiki/%D0%9E%D1%82%D0%BA%D1%80%D0%BE%D0%B2%D0%B5%D0%BD%D0%B8%D0%B5_%D0%98%D0%BE%D0%B0%D0%BD%D0%BD%D0%B0_%D0%91%D0%BE%D0%B3%D0%BE%D1%81%D0%BB%D0%BE%D0%B2%D0%B0" title="Откровение Иоанна Богослова">Откр.</a> <a class="extiw" href="https://ru.wikisource.org/wiki/%D0%9E%D1%82%D0%BA%D1%80%D0%BE%D0%B2%D0%B5%D0%BD%D0%B8%D0%B5_%D1%81%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%98%D0%BE%D0%B0%D0%BD%D0%BD%D0%B0_%D0%91%D0%BE%D0%B3%D0%BE%D1%81%D0%BB%D0%BE%D0%B2%D0%B0#21:6" title="s:Откровение святого Иоанна Богослова">21:6</a>) и роднит равеннское изображение с мозаикой <a href="/wiki/%D0%9C%D0%BE%D0%BD%D0%B0%D1%81%D1%82%D1%8B%D1%80%D1%8C_%D0%9B%D0%B0%D1%82%D0%BE%D0%BC%D1%83" title="Монастырь Латому">монастыря Латому</a> (<a href="/wiki/%D0%93%D1%80%D0%B5%D1%86%D0%B8%D1%8F" title="Греция">Греция</a>), созданной в этот же период.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup>
</p><p><a href="/wiki/%D0%9B%D0%B0%D0%B7%D0%B0%D1%80%D0%B5%D0%B2,_%D0%92%D0%B8%D0%BA%D1%82%D0%BE%D1%80_%D0%9D%D0%B8%D0%BA%D0%B8%D1%82%D0%B8%D1%87" title="Лазарев, Виктор Никитич">В. Н. Лазарев</a> отмечает, что мозаика конхи является одной из самых тонких по исполнению, отличается подчёркнуто симметричной композицией и торжественным характером. По его мнению, над её созданием работали мозаичисты, знавшие византийское искусство в его столичных вариантах<sup class="reference" id="cite_ref-lazarev_6-2"><a href="#cite_note-lazarev-6">[6]</a></sup>. Вместе с тем мозаики апсиды обнаруживают и типично византийскую неподвижность фигур: все персонажи изображены <a href="/wiki/%D0%90%D0%BD%D1%84%D0%B0%D1%81" title="Анфас">анфас</a>, стоя. Даже участники двух процессий будто остановились на мгновение, чтобы показать себя в стационарном положении и позволить зрителю полюбоваться их особами<sup class="reference" id="cite_ref-РАИ-31_16-0"><a href="#cite_note-РАИ-31-16">[16]</a></sup>.
</p>
<ul><li><b>Нижний уровень апсиды</b></li></ul>
<p>На боковых стенах апсиды по сторонам от окон расположены мозаичные портреты, изображающие императора <a href="/wiki/%D0%AE%D1%81%D1%82%D0%B8%D0%BD%D0%B8%D0%B0%D0%BD_I" title="Юстиниан I">Юстиниана</a> и его супругу <a href="/wiki/%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%80%D0%B0" title="Феодора">Феодору</a> в окружении вельмож, придворных дам и <a href="/wiki/%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D0%BE%D1%81%D0%BB%D1%83%D0%B6%D0%B8%D1%82%D0%B5%D0%BB%D1%8C" title="Священнослужитель">священнослужителей</a>. Это исторические портреты, созданные лучшими равеннскими мастерами на основе столичных образцов (В. Н. Лазарев считает, что это были «<i>царские портреты, рассылавшиеся в провинции Византийской империи для копирования</i>»<sup class="reference" id="cite_ref-lazarev_6-3"><a href="#cite_note-lazarev-6">[6]</a></sup>). Эти композиции — символ триумфа императора, вернувшего Равенну под византийский патронат<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup>.
</p>

<p>Император с супругой изображены как <a href="/wiki/%D0%94%D0%BE%D0%BD%D0%B0%D1%82%D0%BE%D1%80" title="Донатор">донаторы</a>, приносящие в дар церкви драгоценные <a href="/wiki/%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D1%8B%D0%B5_%D1%81%D0%BE%D1%81%D1%83%D0%B4%D1%8B" title="Священные сосуды">литургические сосуды</a>. Выполненные как <a href="/wiki/%D0%A4%D1%80%D0%B8%D0%B7_(%D0%B0%D1%80%D1%85%D0%B8%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D0%B0)" title="Фриз (архитектура)">фриз</a> изображения отличаются фронтальной композицией и однообразием поз и жестов. При этом мастера смогли изобразить императорскую семью с индивидуальными чертами лиц в образе идеальных правителей, а сама композиция передаёт движение двух процессий по направлению к <a href="/wiki/%D0%90%D0%BB%D1%82%D0%B0%D1%80%D1%8C" title="Алтарь">алтарю</a>.
</p>
<ul><li><b>Юстиниан I</b></li></ul>
<p>Император Юстиниан приносит в дар церкви <a href="/wiki/%D0%9F%D0%B0%D1%82%D0%B5%D0%BD%D0%B0" title="Патена">патену</a>; он изображён, как и все другие фигуры, во фронтальной позе. Его голова, увенчанная <a href="/wiki/%D0%94%D0%B8%D0%B0%D0%B4%D0%B5%D0%BC%D0%B0" title="Диадема">диадемой</a>, окружена <a href="/wiki/%D0%9D%D0%B8%D0%BC%D0%B1" title="Нимб">нимбом</a>, что отражает византийскую традицию отмечать таким образом царствующую особу.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup>
</p>

<p>По сторонам от Юстиниана стоят придворные и священнослужители. Среди них выделяются пожилой человек в одежде сенатора (единственный во втором ряду; по одной из версий, это ростовщик <a href="/wiki/%D0%AE%D0%BB%D0%B8%D0%B0%D0%BD_%D0%90%D1%80%D0%B3%D0%B5%D0%BD%D1%82%D0%B0%D1%80%D0%B8%D0%B9" title="Юлиан Аргентарий">Юлиан Аргентарий</a>, финансировавший строительство базилики, по другой — полководец <a class="mw-redirect" href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D1%81%D0%B0%D1%80%D0%B8%D0%B9" title="Велисарий">Велисарий</a>, по третьей — <i>praefectus praetorio</i> (префект претория), то есть должностное лицо, представлявшее особу императора в день освящения храма<sup class="reference" id="cite_ref-РАИ-44_19-0"><a href="#cite_note-РАИ-44-19">[19]</a></sup>), епископ Максимиан с крестом в руке и два <a href="/wiki/%D0%94%D0%B8%D0%B0%D0%BA%D0%BE%D0%BD" title="Диакон">диакона</a> (один держит <a href="/wiki/%D0%95%D0%B2%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D0%B8%D0%B5" title="Евангелие">Евангелие</a>, а другой — <a href="/wiki/%D0%9A%D0%B0%D0%B4%D0%B8%D0%BB%D0%BE" title="Кадило">кадило</a>). На этой мозаике Юстиниан и Максимиан изображены как авторитарные представители светской и церковной власти, поэтому их фигуры занимают доминирующее место, а над головой епископа даже помещена горделивая надпись: <i>Maximianus</i>. Если портрет Юстиниана является, скорее всего, копией с официальных изображений, рассылаемых по всей империи<sup class="reference" id="cite_ref-RCA-70_20-0"><a href="#cite_note-RCA-70-20">[20]</a></sup>, то портреты Максимиана и персонажа, стоящего во втором ряду, выделяются характерными чертами, позволяющими предположить знакомство мозаичиста с оригиналами<sup class="reference" id="cite_ref-РАИ-44_19-1"><a href="#cite_note-РАИ-44-19">[19]</a></sup>.
</p>

<p><small>Роскошные одеяния дали возможность мозаичистам развернуть перед зрителем всё ослепительное богатство их палитры — начиная от нежных белых и пурпуровых тонов и кончая ярко-зелёными и оранжево-красными. Особой тонкости исполнения они достигли в лицах четырёх центральных фигур, набранных из более мелких кубиков. Это позволило им создать четыре великолепных по остроте характеристики портрета, в которых, несмотря на ярко выраженные индивидуальные черты, есть и нечто общее: особая строгость выражения и печать глубокой убеждённости.<sup class="reference" id="cite_ref-lazarev_6-4"><a href="#cite_note-lazarev-6">[6]</a></sup></small>
</p>

<ul><li><b>Феодора</b></li></ul>
<p>Императрица изображена стоящей в <a href="/wiki/%D0%9F%D1%80%D0%B8%D1%82%D0%B2%D0%BE%D1%80" title="Притвор">нарфике</a>; перед ней два телохранителя, один из которых отодвигает завесу перед дверью. В руках Феодоры дар, поднесённый церкви — золотой <a href="/wiki/%D0%9F%D0%BE%D1%82%D0%B8%D1%80" title="Потир">потир</a>. Голова увенчана <a href="/wiki/%D0%94%D0%B8%D0%B0%D0%B4%D0%B5%D0%BC%D0%B0" title="Диадема">диадемой</a> и окружена <a href="/wiki/%D0%9D%D0%B8%D0%BC%D0%B1" title="Нимб">нимбом</a>, на плечах тяжёлое ожерелье. Подол плаща украшает сцена <a href="/wiki/%D0%9F%D0%BE%D0%BA%D0%BB%D0%BE%D0%BD%D0%B5%D0%BD%D0%B8%D0%B5_%D0%B2%D0%BE%D0%BB%D1%85%D0%B2%D0%BE%D0%B2" title="Поклонение волхвов">поклонения волхвов</a>, что является намёком на подношение самой Феодоры. Фигура царицы (единственная из всех остальных) обрамлена нишей с конхой, которую <a href="/wiki/%D0%90%D0%BB%D1%84%D1%91%D0%BB%D0%B4%D0%B8,_%D0%90%D0%BD%D0%B4%D1%80%D0%B0%D1%88" title="Алфёлди, Андраш">А. Альфёльди</a> рассматривает как «<i>нишу прославления</i>».<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup> Группу придворных дам, идущую за Феодорой, возглавляют две женщины, чьи изображения наделены портретными чертами (предположительно, Антония и Иоанна — жена и дочь полководца Велисария<sup class="reference" id="cite_ref-РАИ-44_19-2"><a href="#cite_note-РАИ-44-19">[19]</a></sup>). Лица остальных придворных дам <a href="/wiki/%D0%A1%D1%82%D0%B5%D1%80%D0%B5%D0%BE%D1%82%D0%B8%D0%BF" title="Стереотип">стереотипны</a>.
</p>

<p><small>И здесь роскошные византийские одеяния дали мозаичистам повод блеснуть изысканными колористическими решениями. Особенно красивы краски на трёх центральных женских фигурах. Их лица набраны из более мелких и более разнообразных по форме кубиков, что облегчило передачу портретного сходства. Лица остальных придворных дам, как и лица стражи в мозаике с Юстинианом, носят стереотипный характер и маловыразительны. В них высокое искусство уступает место ремеслу и рутине.<sup class="reference" id="cite_ref-lazarev_6-5"><a href="#cite_note-lazarev-6">[6]</a></sup></small>

<h3><span id=".D0.A2.D1.80.D0.B8.D1.83.D0.BC.D1.84.D0.B0.D0.BB.D1.8C.D0.BD.D0.B0.D1.8F_.D0.B8_.D0.B2.D1.85.D0.BE.D0.B4.D0.BD.D0.B0.D1.8F_.D0.B0.D1.80.D0.BA.D0.B8"></span><span class="mw-headline" id="Триумфальная_и_входная_арки">Триумфальная и входная арки</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B0%D0%BD-%D0%92%D0%B8%D1%82%D0%B0%D0%BB%D0%B5&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Триумфальная и входная арки»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B0%D0%BD-%D0%92%D0%B8%D1%82%D0%B0%D0%BB%D0%B5&amp;action=edit&amp;section=5" title="Редактировать код раздела «Триумфальная и входная арки»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:San_Vitale_in_Ravenna_03.jpg"><img class="mw-file-element" data-file-height="357" data-file-width="908" decoding="async" height="79" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/81/San_Vitale_in_Ravenna_03.jpg/200px-San_Vitale_in_Ravenna_03.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/81/San_Vitale_in_Ravenna_03.jpg/300px-San_Vitale_in_Ravenna_03.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/81/San_Vitale_in_Ravenna_03.jpg/400px-San_Vitale_in_Ravenna_03.jpg 2x" width="200"/></a><figcaption><center>«Триумфальная арка»</center></figcaption></figure>
<p>В раннехристианской базилике арка, обрамляющая конху апсиды, по причине богатого декора получила название <a href="/wiki/%D0%A2%D1%80%D0%B8%D1%83%D0%BC%D1%84%D0%B0%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D0%B0%D1%80%D0%BA%D0%B0" title="Триумфальная арка">триумфальной</a>. Триумфальная арка Сан-Витале украшена мозаикой с изображением семи пар <a href="/wiki/%D0%A0%D0%BE%D0%B3_%D0%B8%D0%B7%D0%BE%D0%B1%D0%B8%D0%BB%D0%B8%D1%8F" title="Рог изобилия">рогов изобилия</a> в окружении цветов и птиц. Около верхней пары рогов помещены изображения императорских орлов, а между ними — <a href="/wiki/%D0%A5%D1%80%D0%B8%D0%B7%D0%BC%D0%B0" title="Хризма">монограмма Иисуса Христа</a>. Наружная сторона арки, обращённая в пресбитерий, украшена изображением двух ангелов, возносящих медальон с <a href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82" title="Крест">крестом</a>. Они изображены между двумя центрами христианского <a href="/wiki/%D0%9F%D0%B0%D0%BB%D0%BE%D0%BC%D0%BD%D0%B8%D1%87%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Паломничество">паломничества</a> — <a href="/wiki/%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC" title="Иерусалим">Иерусалимом</a> и <a href="/wiki/%D0%92%D0%B8%D1%84%D0%BB%D0%B5%D0%B5%D0%BC" title="Вифлеем">Вифлеемом</a>.
</p>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:San_Vitale_in_Ravenna_02.jpg"><img class="mw-file-element" data-file-height="770" data-file-width="1293" decoding="async" height="119" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/9a/San_Vitale_in_Ravenna_02.jpg/200px-San_Vitale_in_Ravenna_02.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/9a/San_Vitale_in_Ravenna_02.jpg/300px-San_Vitale_in_Ravenna_02.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/9/9a/San_Vitale_in_Ravenna_02.jpg/400px-San_Vitale_in_Ravenna_02.jpg 2x" width="200"/></a><figcaption><center>Входная арка</center></figcaption></figure>
<p>На склонах входной арки пресбитерия расположены 14 <a href="/wiki/%D0%9C%D0%B5%D0%B4%D0%B0%D0%BB%D1%8C%D0%BE%D0%BD" title="Медальон">медальонов</a> (по 7 на каждой стороне) с полуфигурами <a class="mw-redirect" href="/wiki/%D0%90%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB" title="Апостол">апостолов</a> и <a href="/wiki/%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B9" title="Святой">святых</a>, а в замке арки — медальон с ликом Христа. Медальоны разделяются парами <a href="/wiki/%D0%94%D0%B5%D0%BB%D1%8C%D1%84%D0%B8%D0%BD%D1%8B" title="Дельфины">дельфинов</a>. По направлению слева направо, если смотреть из основного пространства храма, на медальонах представлены: мученик <a href="/wiki/%D0%93%D0%B5%D1%80%D0%B2%D0%B0%D1%81%D0%B8%D0%B9_%D0%B8_%D0%9F%D1%80%D0%BE%D1%82%D0%B0%D1%81%D0%B8%D0%B9" title="Гервасий и Протасий">Гервасий</a>; апостолы <a href="/wiki/%D0%90%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB_%D0%A4%D0%B0%D0%B4%D0%B4%D0%B5%D0%B9" title="Апостол Фаддей">Фаддей</a>, <a href="/wiki/%D0%9B%D0%B5%D0%B2%D0%B8%D0%B9_%D0%9C%D0%B0%D1%82%D1%84%D0%B5%D0%B9" title="Левий Матфей">Матфей</a>, <a href="/wiki/%D0%90%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB_%D0%92%D0%B0%D1%80%D1%84%D0%BE%D0%BB%D0%BE%D0%BC%D0%B5%D0%B9" title="Апостол Варфоломей">Варфоломей</a>, <a href="/wiki/%D0%98%D0%BE%D0%B0%D0%BD%D0%BD_%D0%91%D0%BE%D0%B3%D0%BE%D1%81%D0%BB%D0%BE%D0%B2" title="Иоанн Богослов">Иоанн Богослов</a>, <a href="/wiki/%D0%90%D0%BD%D0%B4%D1%80%D0%B5%D0%B9_%D0%9F%D0%B5%D1%80%D0%B2%D0%BE%D0%B7%D0%B2%D0%B0%D0%BD%D0%BD%D1%8B%D0%B9" title="Андрей Первозванный">Андрей Первозванный</a>, <a href="/wiki/%D0%90%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB_%D0%9F%D1%91%D1%82%D1%80" title="Апостол Пётр">Пётр</a>; Христос; апостолы <a href="/wiki/%D0%90%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB_%D0%9F%D0%B0%D0%B2%D0%B5%D0%BB" title="Апостол Павел">Павел</a>, <a href="/wiki/%D0%98%D0%B0%D0%BA%D0%BE%D0%B2_%D0%97%D0%B5%D0%B2%D0%B5%D0%B4%D0%B5%D0%B5%D0%B2" title="Иаков Зеведеев">Иаков</a>, <a href="/wiki/%D0%90%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB_%D0%A4%D0%B8%D0%BB%D0%B8%D0%BF%D0%BF" title="Апостол Филипп">Филипп</a>, <a href="/wiki/%D0%90%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB_%D0%A4%D0%BE%D0%BC%D0%B0" title="Апостол Фома">Фома</a>, <a href="/wiki/%D0%98%D0%B0%D0%BA%D0%BE%D0%B2_%D0%90%D0%BB%D1%84%D0%B5%D0%B5%D0%B2" title="Иаков Алфеев">Иаков Алфеев</a>, <a href="/wiki/%D0%A1%D0%B8%D0%BC%D0%BE%D0%BD_%D0%9A%D0%B0%D0%BD%D0%B0%D0%BD%D0%B8%D1%82" title="Симон Кананит">Симон Кананит</a>, мученик <a href="/wiki/%D0%93%D0%B5%D1%80%D0%B2%D0%B0%D1%81%D0%B8%D0%B9_%D0%B8_%D0%9F%D1%80%D0%BE%D1%82%D0%B0%D1%81%D0%B8%D0%B9" title="Гервасий и Протасий">Протасий</a>. Таким образом, кроме <a href="/wiki/%D0%94%D0%B2%D0%B5%D0%BD%D0%B0%D0%B4%D1%86%D0%B0%D1%82%D1%8C_%D0%B0%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB%D0%BE%D0%B2" title="Двенадцать апостолов">двенадцати апостолов</a>, на арке изображены двое святых — <a href="/wiki/%D0%93%D0%B5%D1%80%D0%B2%D0%B0%D1%81%D0%B8%D0%B9_%D0%B8_%D0%9F%D1%80%D0%BE%D1%82%D0%B0%D1%81%D0%B8%D0%B9" title="Гервасий и Протасий">Гервасий и Протасий</a>, которых традиция называет сыновьями <a href="/wiki/%D0%92%D0%B8%D1%82%D0%B0%D0%BB%D0%B8%D0%B9_%D0%9C%D0%B8%D0%BB%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9" title="Виталий Миланский">святого Виталия</a>. Помещение их в один ряд с апостолами показывает, с одной стороны, связь Равенны с <a href="/wiki/%D0%9C%D0%B8%D0%BB%D0%B0%D0%BD" title="Милан">Миланом</a>, где были <a href="/wiki/%D0%9E%D0%B1%D1%80%D0%B5%D1%82%D0%B5%D0%BD%D0%B8%D0%B5_%D0%BC%D0%BE%D1%89%D0%B5%D0%B9" title="Обретение мощей">обретены</a> их <a href="/wiki/%D0%9C%D0%BE%D1%89%D0%B8" title="Мощи">мощи</a><sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup>, и напоминает, с другой стороны, о мистическом участии этих святых в победе над <a href="/wiki/%D0%90%D1%80%D0%B8%D0%B0%D0%BD%D1%81%D1%82%D0%B2%D0%BE" title="Арианство">арианством</a>. Лицам апостолов приданы индивидуальные черты: так, апостол Андрей изображён со всклокоченными волосами и широко раскрытыми глазами.
</p>
<h3><span id=".D0.9F.D1.80.D0.B5.D1.81.D0.B2.D0.B8.D1.82.D0.B5.D1.80.D0.B8.D0.B9"></span><span class="mw-headline" id="Пресвитерий">Пресвитерий</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B0%D0%BD-%D0%92%D0%B8%D1%82%D0%B0%D0%BB%D0%B5&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Пресвитерий»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B0%D0%BD-%D0%92%D0%B8%D1%82%D0%B0%D0%BB%D0%B5&amp;action=edit&amp;section=6" title="Редактировать код раздела «Пресвитерий»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%D0%92%D0%B8%D0%BC%D0%B0" title="Вима">Вимы</a> пресвитерия разделены на три мозаичных регистра: стены, верхнюю галерею и свод. Художественное исполнение мозаик пресбитерия гораздо грубее по сравнению с мозаикой апсиды. Вероятно, они были созданы другими мозаичистами, работавшими в местной традиции<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup>. Вместе с тем в мозаиках пресвитерия нет статичности мозаик апсиды: персонажи стоят, сидят; к зрителям они обращены <a href="/wiki/%D0%90%D0%BD%D1%84%D0%B0%D1%81" title="Анфас">анфас</a>, в <a href="/wiki/%D0%9F%D1%80%D0%BE%D1%84%D0%B8%D0%BB%D1%8C_%D1%87%D0%B5%D0%BB%D0%BE%D0%B2%D0%B5%D0%BA%D0%B0" title="Профиль человека">профиль</a>, вполоборота; на смену золотому фону апсиды пришли пейзажи<sup class="reference" id="cite_ref-РАИ-31_16-1"><a href="#cite_note-РАИ-31-16">[16]</a></sup>. Тематика мозаик посвящена символическому содержанию идеи жертвы Христовой, которая раскрывается через <a href="/wiki/%D0%92%D0%B5%D1%82%D1%85%D0%B8%D0%B9_%D0%97%D0%B0%D0%B2%D0%B5%D1%82" title="Ветхий Завет">ветхозаветные</a> образы в соответствии с <a href="/wiki/%D0%91%D0%BE%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%BB%D0%B8%D1%82%D1%83%D1%80%D0%B3%D0%B8%D1%8F" title="Божественная литургия">литургической</a> трактовкой таинства <a href="/wiki/%D0%95%D0%B2%D1%85%D0%B0%D1%80%D0%B8%D1%81%D1%82%D0%B8%D1%8F" title="Евхаристия">Евхаристии</a>.
</p><p>Академик <a href="/wiki/%D0%9B%D0%B0%D0%B7%D0%B0%D1%80%D0%B5%D0%B2,_%D0%92%D0%B8%D0%BA%D1%82%D0%BE%D1%80_%D0%9D%D0%B8%D0%BA%D0%B8%D1%82%D0%B8%D1%87" title="Лазарев, Виктор Никитич">В. Н. Лазарев</a> отмечает, что мозаики пресвитерия частично пострадали от грубых реставраций. Они отличаются сложной символикой, характерной для убранства константинопольских церквей. Это позволяет ему сделать вывод, что «<i>иконографическая программа этих мозаик восходит к византийским источникам</i>». В мозаиках пресвитерия особо выделяется художественное исполнение пейзажей — скалистые уступы, похожие на обломки кристаллов, раскрашены яркими тонами (голубой, зелёный, лиловый, пурпурный) и местами украшены золотом. Всё это создаёт иллюзию яркого цветочного ковра.
</p>
<h4><span id=".D0.A1.D1.82.D0.B5.D0.BD.D1.8B"></span><span class="mw-headline" id="Стены">Стены</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B0%D0%BD-%D0%92%D0%B8%D1%82%D0%B0%D0%BB%D0%B5&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Стены»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B0%D0%BD-%D0%92%D0%B8%D1%82%D0%B0%D0%BB%D0%B5&amp;action=edit&amp;section=7" title="Редактировать код раздела «Стены»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h4>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:San_Vitale_in_Ravenna_05.jpg"><img class="mw-file-element" data-file-height="414" data-file-width="971" decoding="async" height="107" src="//upload.wikimedia.org/wikipedia/commons/thumb/0/04/San_Vitale_in_Ravenna_05.jpg/250px-San_Vitale_in_Ravenna_05.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/0/04/San_Vitale_in_Ravenna_05.jpg/375px-San_Vitale_in_Ravenna_05.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/0/04/San_Vitale_in_Ravenna_05.jpg/500px-San_Vitale_in_Ravenna_05.jpg 2x" width="250"/></a><figcaption><center>Сцены из жизни Авраама</center></figcaption></figure>
<ul><li><b>Левая стена</b></li></ul>
<p>Мозаики левой стены посвящены событиям из жизни патриарха <a href="/wiki/%D0%90%D0%B2%D1%80%D0%B0%D0%B0%D0%BC" title="Авраам">Авраама</a>: «<i>Гостеприимство Авраама</i>» (или ветхозаветная <a href="/wiki/%D0%A2%D1%80%D0%BE%D0%B8%D1%86%D0%B0" title="Троица">Троица</a>) и «<i>Жертвоприношение Авраама</i>». В первой композиции три гостя сидят за столом, на котором лежат три хлеба, отмеченные знаком креста; Авраам преподносит заколотого им телёнка, а его жена <a href="/wiki/%D0%A1%D0%B0%D1%80%D1%80%D0%B0" title="Сарра">Сарра</a> наблюдает за этой сценой из хижины. На второй композиции основной акцент перенесён на божественную длань, которая отводит в сторону нож Авраама, занесённый над его сыном <a href="/wiki/%D0%98%D1%81%D0%B0%D0%B0%D0%BA" title="Исаак">Исааком</a>. По сторонам от центральных композиций изображены пророк <a href="/wiki/%D0%98%D0%B5%D1%80%D0%B5%D0%BC%D0%B8%D1%8F" title="Иеремия">Иеремия</a> со свитком в руках и <a href="/wiki/%D0%9C%D0%BE%D0%B8%D1%81%D0%B5%D0%B9" title="Моисей">Моисей</a>, получающий на <a href="/wiki/%D0%A1%D0%B8%D0%BD%D0%B0%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%BE%D0%BB%D1%83%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2" title="Синайский полуостров">Синае</a> <a href="/wiki/%D0%A1%D0%BA%D1%80%D0%B8%D0%B6%D0%B0%D0%BB%D0%B8_%D0%97%D0%B0%D0%B2%D0%B5%D1%82%D0%B0" title="Скрижали Завета">Скрижали Завета</a> на глазах двенадцати старейшин Израилевых. Следует отметить высокое мастерство мозаичиста, избежавшего однообразия при изображении трёх сидящих ангелов: их головы слегка наклонены, различны положения их рук и ног<sup class="reference" id="cite_ref-РАИ-34_24-0"><a href="#cite_note-РАИ-34-24">[24]</a></sup>. Мозаика также детально показывает листья <a href="/wiki/%D0%9C%D0%B0%D0%BC%D0%B2%D1%80%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D0%B4%D1%83%D0%B1" title="Мамврийский дуб">дерева</a>, под которым сидят ангелы, что позволяет распознать <a href="/wiki/%D0%94%D1%83%D0%B1" title="Дуб">дуб</a><sup class="reference" id="cite_ref-RCA-72_25-0"><a href="#cite_note-RCA-72-25">[25]</a></sup>.
</p>
<ul><li><b>Правая стена</b></li></ul>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:San_Vitale_in_Ravenna_06.jpg"><img class="mw-file-element" data-file-height="369" data-file-width="832" decoding="async" height="111" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/c2/San_Vitale_in_Ravenna_06.jpg/250px-San_Vitale_in_Ravenna_06.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/c/c2/San_Vitale_in_Ravenna_06.jpg/375px-San_Vitale_in_Ravenna_06.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/c/c2/San_Vitale_in_Ravenna_06.jpg/500px-San_Vitale_in_Ravenna_06.jpg 2x" width="250"/></a><figcaption><center>Жертвоприношение Авеля и Мельхиседека</center></figcaption></figure>
<p>Центр правой стены занимает композиция с изображением жертвоприношений <a href="/wiki/%D0%90%D0%B2%D0%B5%D0%BB%D1%8C" title="Авель">Авеля</a> и <a class="mw-redirect" href="/wiki/%D0%9C%D0%B5%D0%BB%D1%8C%D1%85%D0%B8%D1%81%D0%B5%D0%B4%D0%B5%D0%BA" title="Мельхиседек">Мельхиседека</a> (символический намёк на <a class="mw-redirect" href="/wiki/%D0%A0%D0%B0%D1%81%D0%BF%D1%8F%D1%82%D0%B8%D0%B5_%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%BE%D0%B2%D0%BE" title="Распятие Христово">крестную смерть Христа</a>). Изображение трёх жертвоприношений (Авеля, Мельхиседека и Авраама) на стенах пресбитерия иллюстрирует молитву <a href="/wiki/%D0%90%D0%BD%D0%B0%D1%84%D0%BE%D1%80%D0%B0_(%D0%BB%D0%B8%D1%82%D1%83%D1%80%D0%B3%D0%B8%D1%8F)" title="Анафора (литургия)">евхаристического канона</a> <a href="/wiki/%D0%9C%D0%B5%D1%81%D1%81%D0%B0" title="Месса">римской литургии</a><sup class="reference" id="cite_ref-РИИ-38-40_26-0"><a href="#cite_note-РИИ-38-40-26">[26]</a></sup>:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
Supra quae propitio ac sereno vultu respicere digneris: et accepta habere, sicuti accepta habere dignatus es munera pueri tui iusti Abel, et sacrificium Patriarcha nostri Abrahae: et quod tibi obtulit summus sacerdos tuus Melchisedech, sanctum sacrificium, immaculatam hostiam.
</p>
<div class="mw-collapsible mw-collapsed has-collapsible-header ts-Скрытый_блок ts-Скрытый_блок-gray ts-Скрытый_блок-rightHideLink" style=""><div class="ts-Скрытый_блок-title collapsible-header" style="">Перевод</div><div class="mw-collapsible-content" style="">
<p>Благоволи воззреть на сие милостиво и благоприятно и приими, как сподобил принять дары отрока Твоего, праведного Авеля и жертву патриарха нашего Авраама, и то, что принёс Тебе первосвященник Твой Мельхиседек, святое жертвоприношение, непорочную жертву<sup class="reference" id="cite_ref-месса_27-0"><a href="#cite_note-месса-27">[27]</a></sup>
</p>
</div></div>
</blockquote>
<p>Слева от центральной мозаики помещены две сцены из жизни Моисея: видение им <a href="/wiki/%D0%9D%D0%B5%D0%BE%D0%BF%D0%B0%D0%BB%D0%B8%D0%BC%D0%B0%D1%8F_%D0%BA%D1%83%D0%BF%D0%B8%D0%BD%D0%B0" title="Неопалимая купина">Неопалимой купины</a> на горе <a href="/wiki/%D0%A1%D0%B8%D0%BD%D0%B0%D0%B9_(%D0%B3%D0%BE%D1%80%D0%B0)" title="Синай (гора)">Хорив</a> и Моисей среди стада его тестя <a href="/wiki/%D0%98%D0%BE%D1%84%D0%BE%D1%80" title="Иофор">Иофора</a>. В правой части изображён пророк <a href="/wiki/%D0%98%D1%81%D0%B0%D0%B8%D1%8F" title="Исаия">Исаия</a> (его фигура является парной по отношению к пророку Иеремии на противоположной стене). Эти два ветхозаветных пророка были выбраны потому, что они предсказали воплощение Сына Божьего, его <a href="/wiki/%D0%A1%D1%82%D1%80%D0%B0%D1%81%D1%82%D0%B8_%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%BE%D0%B2%D1%8B" title="Страсти Христовы">страдания</a> и крестную смерть. Необычно изображена сцена с Неопалимой купиной: огнём объят не только куст, но вся гора<sup class="reference" id="cite_ref-RCA-76_28-0"><a href="#cite_note-RCA-76-28">[28]</a></sup>.

</p>

<h4><span id=".D0.92.D0.B5.D1.80.D1.85.D0.BD.D1.8F.D1.8F_.D0.B3.D0.B0.D0.BB.D0.B5.D1.80.D0.B5.D1.8F_.D0.B8_.D1.81.D0.B2.D0.BE.D0.B4"></span><span class="mw-headline" id="Верхняя_галерея_и_свод">Верхняя галерея и свод</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B0%D0%BD-%D0%92%D0%B8%D1%82%D0%B0%D0%BB%D0%B5&amp;veaction=edit&amp;section=8" title="Редактировать раздел «Верхняя галерея и свод»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B0%D0%BD-%D0%92%D0%B8%D1%82%D0%B0%D0%BB%D0%B5&amp;action=edit&amp;section=8" title="Редактировать код раздела «Верхняя галерея и свод»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h4>
<figure class="mw-default-size mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Basilica_of_San_Vitale_-_Lamb_of_God_mosaic.jpg"><img class="mw-file-element" data-file-height="3443" data-file-width="4309" decoding="async" height="176" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/a5/Basilica_of_San_Vitale_-_Lamb_of_God_mosaic.jpg/220px-Basilica_of_San_Vitale_-_Lamb_of_God_mosaic.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/a5/Basilica_of_San_Vitale_-_Lamb_of_God_mosaic.jpg/330px-Basilica_of_San_Vitale_-_Lamb_of_God_mosaic.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/a5/Basilica_of_San_Vitale_-_Lamb_of_God_mosaic.jpg/440px-Basilica_of_San_Vitale_-_Lamb_of_God_mosaic.jpg 2x" width="220"/></a><figcaption>Мозаика свода.</figcaption></figure>
<p>Арочные проёмы верхней галереи украшены изображениями четырёх <a href="/wiki/%D0%95%D0%B2%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D0%B8%D1%81%D1%82%D1%8B" title="Евангелисты">евангелистов</a> и их <a class="mw-redirect" href="/wiki/%D0%A1%D0%B8%D0%BC%D0%B2%D0%BE%D0%BB%D1%8B_%D0%B5%D0%B2%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D0%B8%D1%81%D1%82%D0%BE%D0%B2" title="Символы евангелистов">символами</a>. Из-под ног евангелистов вытекают ручьи чистой воды, символизирующей благую весть <a href="/wiki/%D0%95%D0%B2%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D0%B8%D0%B5" title="Евангелие">Евангелия</a>; в этой воде купаются птицы и водные звери, в том числе <a href="/wiki/%D0%A6%D0%B0%D0%BF%D0%BB%D0%B8" title="Цапли">цапля</a> и <a href="/wiki/%D0%A7%D0%B5%D1%80%D0%B5%D0%BF%D0%B0%D1%85%D0%B8" title="Черепахи">черепаха</a><sup class="reference" id="cite_ref-РАИ-38_29-0"><a href="#cite_note-РАИ-38-29">[29]</a></sup>. В основание арок помещены вазы, из которых растут <a href="/wiki/%D0%92%D0%B8%D0%BD%D0%BE%D0%B3%D1%80%D0%B0%D0%B4_(%D1%80%D0%BE%D0%B4)" title="Виноград (род)">виноградные</a> лозы и, переплетаясь, сходятся в замке арки, у креста.
</p><p>Свод украшен <a href="/wiki/%D0%9C%D0%B5%D0%B4%D0%B0%D0%BB%D1%8C%D0%BE%D0%BD" title="Медальон">медальоном</a> с изображением <a class="mw-redirect" href="/wiki/%D0%90%D0%BF%D0%BE%D0%BA%D0%B0%D0%BB%D0%B8%D0%BF%D1%81%D0%B8%D1%81" title="Апокалипсис">апокалипсического</a> <a href="/wiki/%D0%90%D0%B3%D0%BD%D0%B5%D1%86_%D0%91%D0%BE%D0%B6%D0%B8%D0%B9" title="Агнец Божий">Агнца</a> (<a href="/wiki/%D0%9E%D1%82%D0%BA%D1%80%D0%BE%D0%B2%D0%B5%D0%BD%D0%B8%D0%B5_%D0%98%D0%BE%D0%B0%D0%BD%D0%BD%D0%B0_%D0%91%D0%BE%D0%B3%D0%BE%D1%81%D0%BB%D0%BE%D0%B2%D0%B0" title="Откровение Иоанна Богослова">Откр.</a> <a class="extiw" href="https://ru.wikisource.org/wiki/%D0%9E%D1%82%D0%BA%D1%80%D0%BE%D0%B2%D0%B5%D0%BD%D0%B8%D0%B5_%D1%81%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%98%D0%BE%D0%B0%D0%BD%D0%BD%D0%B0_%D0%91%D0%BE%D0%B3%D0%BE%D1%81%D0%BB%D0%BE%D0%B2%D0%B0#5:12" title="s:Откровение святого Иоанна Богослова">5:12</a>), которого поддерживают четыре ангела в позе <a href="/wiki/%D0%9E%D1%80%D0%B0%D0%BD%D1%82%D0%B0" title="Оранта">Оранта</a>, символизирующие <a href="/wiki/%D0%A1%D1%82%D0%BE%D1%80%D0%BE%D0%BD%D1%8B_%D1%81%D0%B2%D0%B5%D1%82%D0%B0" title="Стороны света">стороны света</a>. Белоснежный агнец, увенчанный <a href="/wiki/%D0%9D%D0%B8%D0%BC%D0%B1" title="Нимб">нимбом</a>, изображён на фоне звёздного неба; медальон с ним обрамлён венком. Композиция окружена <a href="/wiki/%D0%A0%D0%B0%D0%B9" title="Рай">райскими</a> деревьями, растениями, птицами и животными. Божественный Агнец, взявший на себя грехи мира («<i>Agnus Dei, qui tollis peccata mundi</i>» из <a href="/wiki/%D0%9C%D0%B5%D1%81%D1%81%D0%B0" title="Месса">римской литургии</a><sup class="reference" id="cite_ref-месса_27-1"><a href="#cite_note-месса-27">[27]</a></sup>), венчает цикл мозаик пресбитерия, посвящённый жертвам, приносимым людьми <a href="/wiki/%D0%91%D0%BE%D0%B3" title="Бог">Богу</a>.
</p><p>Рядом с базиликой Сан-Витале расположен <a href="/wiki/%D0%9D%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BC%D1%83%D0%B7%D0%B5%D0%B9_%D0%A0%D0%B0%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B" title="Национальный музей Равенны">Национальный музей Равенны</a>.
</p>
