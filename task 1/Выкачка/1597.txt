<h2><span id=".D0.9E.D0.BF.D0.B8.D1.81.D0.B0.D0.BD.D0.B8.D0.B5_.D1.8F.D0.B2.D0.BB.D0.B5.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="Описание_явления">Описание явления</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9F%D0%B5%D1%80%D0%B2%D0%B0%D1%8F_%D0%BF%D0%B0%D0%BB%D0%B0%D1%82%D0%B0%D0%BB%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Описание явления»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9F%D0%B5%D1%80%D0%B2%D0%B0%D1%8F_%D0%BF%D0%B0%D0%BB%D0%B0%D1%82%D0%B0%D0%BB%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F&amp;action=edit&amp;section=1" title="Редактировать код раздела «Описание явления»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Первая палатализация представляет собой такое <a href="/wiki/%D0%A2%D0%B8%D0%BF%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F_(%D0%BB%D0%B8%D0%BD%D0%B3%D0%B2%D0%B8%D1%81%D1%82%D0%B8%D0%BA%D0%B0)" title="Типология (лингвистика)">типологически</a> распространённое явление, как переход <a href="/wiki/%D0%97%D0%B0%D0%B4%D0%BD%D0%B5%D1%8F%D0%B7%D1%8B%D1%87%D0%BD%D1%8B%D0%B5_%D1%81%D0%BE%D0%B3%D0%BB%D0%B0%D1%81%D0%BD%D1%8B%D0%B5" title="Заднеязычные согласные">заднеязычных согласных</a> в положении перед <a href="/wiki/%D0%93%D0%BB%D0%B0%D1%81%D0%BD%D1%8B%D0%B5" title="Гласные">гласными</a> <a href="/wiki/%D0%93%D0%BB%D0%B0%D1%81%D0%BD%D1%8B%D0%B5_%D0%BF%D0%B5%D1%80%D0%B5%D0%B4%D0%BD%D0%B5%D0%B3%D0%BE_%D1%80%D1%8F%D0%B4%D0%B0" title="Гласные переднего ряда">переднего ряда</a> в <a href="/wiki/%D0%90%D1%84%D1%84%D1%80%D0%B8%D0%BA%D0%B0%D1%82%D1%8B" title="Аффрикаты">аффрикаты</a> и/или <a href="/wiki/%D0%A4%D1%80%D0%B8%D0%BA%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D1%8B%D0%B5_%D1%81%D0%BE%D0%B3%D0%BB%D0%B0%D1%81%D0%BD%D1%8B%D0%B5" title="Фрикативные согласные">спиранты</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-Чекман55_2-0"><a href="#cite_note-Чекман55-2">[2]</a></sup>. Часть авторов разделяет палатализацию перед гласными переднего ряда и перед <i>*j</i><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>, часть считает их единым процессом<sup class="reference" id="cite_ref-ReferenceB_4-0"><a href="#cite_note-ReferenceB-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>.
</p><p><a href="/wiki/%D0%A7%D0%B5%D1%80%D0%B5%D0%B4%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5" title="Чередование">Чередования</a>, вызванные первой палатализацией, по сей день сохраняются всеми <a href="/wiki/%D0%A1%D0%BB%D0%B0%D0%B2%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B5_%D1%8F%D0%B7%D1%8B%D0%BA%D0%B8" title="Славянские языки">славянскими языками</a>:
</p>
<ul><li><a href="/wiki/%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Русский язык">рус.</a> <span lang="ru" style="font-style:italic;">ру<b>к</b>а́</span> — <span lang="ru" style="font-style:italic;">ру́<b>ч</b>ка</span>, <span lang="ru" style="font-style:italic;">но<b>г</b>а́</span> — <span lang="ru" style="font-style:italic;">но́<b>ж</b>ка</span>, <span lang="ru" style="font-style:italic;">му́<b>х</b>а</span> — <span lang="ru" style="font-style:italic;">му́<b>ш</b>ка</span>, <span lang="ru" style="font-style:italic;">пе<b>к</b>у</span> — <span lang="ru" style="font-style:italic;">пе<b>ч</b>ёшь</span>, <span lang="ru" style="font-style:italic;">мо<b>г</b>у</span> — <span lang="ru" style="font-style:italic;">мо<b>ж</b>ешь</span>, <span lang="ru" style="font-style:italic;">пи<b>ск</b></span> — <span lang="ru" style="font-style:italic;">пи<b>щ</b>а́ть</span>, <span lang="ru" style="font-style:italic;">ви<b>зг</b></span> — <span lang="ru" style="font-style:italic;">ви<b>зж</b>а́ть</span>;</li>
<li><a href="/wiki/%D0%91%D0%BE%D0%BB%D0%B3%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Болгарский язык">болг.</a> <span lang="bg" style="font-style:italic;">ръ<b>к</b>а́</span> — <span lang="bg" style="font-style:italic;">ръ́<b>ч</b>ка</span>, <span lang="bg" style="font-style:italic;">бла́<b>г</b>о</span> — <span lang="bg" style="font-style:italic;">бла<b>ж</b>е́н</span>, <span lang="bg" style="font-style:italic;">му<b>х</b>а́</span> — <span lang="bg" style="font-style:italic;">му<b>ш</b>и́ца</span><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>;</li>
<li><a href="/wiki/%D0%A1%D0%B5%D1%80%D0%B1%D0%BE%D1%85%D0%BE%D1%80%D0%B2%D0%B0%D1%82%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Сербохорватский язык">сербохорв.</a> <span lang="sh" style="font-style:italic;">дÿ<b>г</b></span> — <span lang="sh" style="font-style:italic;">дÿ<b>ж</b>ū</span>, <span lang="sh" style="font-style:italic;">јâ<b>к</b></span> — <span lang="sh" style="font-style:italic;">јä<b>ч</b>ū</span>, <span lang="sh" style="font-style:italic;">сŷ<b>х</b></span> — <span lang="sh" style="font-style:italic;">сÿ<b>ш</b>ū</span><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>.</li></ul>
<p>Однако в некоторых русских говорах существует тенденция к устранению результатов первой палатализации в парадигмах глаголов: <span lang="ru" style="font-style:italic;">стри<b>г</b>ёт</span>, <span lang="ru" style="font-style:italic;">пе<b>к</b>ёт</span>. Форма <span lang="ru" style="font-style:italic;">т<b>к</b>ёт</span> (вместо <span lang="ru" style="font-style:italic;">т<b>ч</b>ёт</span>) даже попала в литературный язык.
</p><p>Болгарский академик <a class="mw-redirect" href="/wiki/%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2,_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2" title="Георгиев, Владимир Иванов">В. И. Георгиев</a> предложил называть первую палатализацию второй, а <a class="mw-redirect" href="/wiki/%D0%98%D0%B7%D0%BE%D0%B3%D0%BB%D0%BE%D1%81%D1%81%D0%B0_%D0%BA%D0%B5%D0%BD%D1%82%D1%83%D0%BC-%D1%81%D0%B0%D1%82%D0%B5%D0%BC" title="Изоглосса кентум-сатем">сатемную палатализацию</a> первой (как это принято в <a href="/wiki/%D0%90%D1%80%D0%BC%D0%B5%D0%BD%D0%B8%D1%81%D1%82%D0%B8%D0%BA%D0%B0" title="Арменистика">арменистике</a> и индоиранистике), однако его инициатива не встретила поддержки у других учёных.
</p>
<h3><span id=".D0.A0.D0.B5.D1.84.D0.BB.D0.B5.D0.BA.D1.81.D1.8B"></span><span class="mw-headline" id="Рефлексы">Рефлексы</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9F%D0%B5%D1%80%D0%B2%D0%B0%D1%8F_%D0%BF%D0%B0%D0%BB%D0%B0%D1%82%D0%B0%D0%BB%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Рефлексы»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9F%D0%B5%D1%80%D0%B2%D0%B0%D1%8F_%D0%BF%D0%B0%D0%BB%D0%B0%D1%82%D0%B0%D0%BB%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F&amp;action=edit&amp;section=2" title="Редактировать код раздела «Рефлексы»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Изначально рефлексы первой палатализации были мягкими и в части славянских языков таковыми остаются по сей день. В <a href="/wiki/%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Русский язык">русском</a> в XIV веке отвердели <i>ж</i> и <i>ш</i>, но <i>ч</i> сохранил мягкость; в <a href="/wiki/%D0%91%D0%B5%D0%BB%D0%BE%D1%80%D1%83%D1%81%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Белорусский язык">белорусском</a> и <a href="/wiki/%D0%A3%D0%BA%D1%80%D0%B0%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Украинский язык">украинском</a> отвердел также и <i>ч</i><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>. Древняя мягкость шипящих отражается в <a class="mw-redirect" href="/wiki/%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D1%80%D1%84%D0%BE%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D1%8F" title="Русская орфография">русской орфографии</a> в виде написаний <i>жи</i> и <i>ши</i> при фактическом произношении <i>жы</i> и <i>шы</i>. Кроме того, перед гласными переднего ряда мягкость <i>ж</i> и <i>ш</i> до сих пор сохраняется в кировских говорах<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>.
</p><p>Уже в XVI веке в <a href="/wiki/%D0%9F%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Польский язык">польском языке</a> отвердели все рефлексы<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup>.
</p><p>Результат палатализации <i>g</i> — <i>ž'</i>, а не <i>dž'</i>, объясняется тем, что звонкие <a href="/wiki/%D0%90%D1%84%D1%84%D1%80%D0%B8%D0%BA%D0%B0%D1%82%D1%8B" title="Аффрикаты">аффрикаты</a> на протяжении всей истории праславянского языка были неустойчивы. Поэтому вскоре после завершения процесса палатализации <i>dž</i> упростилось до <i>ž</i><sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>.
</p>
<h3><span id=".D0.9F.D1.80.D0.BE.D0.BC.D0.B5.D0.B6.D1.83.D1.82.D0.BE.D1.87.D0.BD.D1.8B.D0.B5_.D1.81.D1.82.D0.B0.D0.B4.D0.B8.D0.B8"></span><span class="mw-headline" id="Промежуточные_стадии">Промежуточные стадии</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9F%D0%B5%D1%80%D0%B2%D0%B0%D1%8F_%D0%BF%D0%B0%D0%BB%D0%B0%D1%82%D0%B0%D0%BB%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Промежуточные стадии»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9F%D0%B5%D1%80%D0%B2%D0%B0%D1%8F_%D0%BF%D0%B0%D0%BB%D0%B0%D1%82%D0%B0%D0%BB%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F&amp;action=edit&amp;section=3" title="Редактировать код раздела «Промежуточные стадии»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%D0%9B%D0%B5%D1%81%D0%BA%D0%B8%D0%BD,_%D0%90%D0%B2%D0%B3%D1%83%D1%81%D1%82" title="Лескин, Август">А. Лескин</a> полагал, что первая палатализация проходила по схеме [k] &gt; [k’] &gt; [k’x’] &gt; [t’x’] &gt; [t’š’] &gt; [č’].
</p><p><a class="mw-redirect" href="/wiki/%D0%A8%D0%B0%D1%85%D0%BC%D0%B0%D1%82%D0%BE%D0%B2,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80%D0%BE%D0%B2%D0%B8%D1%87_(%D0%B0%D0%BA%D0%B0%D0%B4%D0%B5%D0%BC%D0%B8%D0%BA)" title="Шахматов, Алексей Александрович (академик)">А. А. Шахматов</a> предложил более короткую схему: [k’] &gt; [k’ћ’] &gt; [č’]; [g’] &gt; [g’ђ’] &gt; [dž’]; [x’] &gt; [š’]. <a href="/wiki/%D0%91%D0%B5%D1%80%D0%BD%D1%88%D1%82%D0%B5%D0%B9%D0%BD,_%D0%A1%D0%B0%D0%BC%D1%83%D0%B8%D0%BB_%D0%91%D0%BE%D1%80%D0%B8%D1%81%D0%BE%D0%B2%D0%B8%D1%87" title="Бернштейн, Самуил Борисович">С. Б. Бернштейн</a> позднее дополнил её и представил в следующем виде: [k’ː] &gt; [k<sup>ћ</sup>'] &gt; [ћ’] &gt; [č’]; [g’ː] &gt; [g<sup>ђ</sup>’] &gt; [ђ’] &gt; [dž’]; [x’ː] &gt; [x<sup>ś’</sup>] &gt; [š’]<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup>.
</p><p>С. Стойков критикует эту схему, указывая, что в разложских диалектах болгарского языка, где процесс, аналогичный первой палатализации, прошёл уже в XX веке, была всего одна промежуточная стадия: [k] &gt; [k’] &gt; [č’]<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup>. Именно такую схему постулирует для праславянского В. Н. Чекман<sup class="reference" id="cite_ref-Чекман55_2-1"><a href="#cite_note-Чекман55-2">[2]</a></sup>.
</p>
<h3><span id=".D0.9F.D0.BE.D1.81.D0.BB.D0.B5.D0.B4.D1.81.D1.82.D0.B2.D0.B8.D1.8F"></span><span class="mw-headline" id="Последствия">Последствия</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9F%D0%B5%D1%80%D0%B2%D0%B0%D1%8F_%D0%BF%D0%B0%D0%BB%D0%B0%D1%82%D0%B0%D0%BB%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Последствия»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9F%D0%B5%D1%80%D0%B2%D0%B0%D1%8F_%D0%BF%D0%B0%D0%BB%D0%B0%D1%82%D0%B0%D0%BB%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F&amp;action=edit&amp;section=4" title="Редактировать код раздела «Последствия»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Первая палатализация обогатила фонетику <a href="/wiki/%D0%9F%D1%80%D0%B0%D1%81%D0%BB%D0%B0%D0%B2%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Праславянский язык">праславянского языка</a> четырьмя новыми <a href="/wiki/%D0%97%D0%B2%D1%83%D0%BA" title="Звук">звуками</a> (<span class="IPA"><a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Международный фонетический алфавит">tʃ<sup>j</sup></a></span>, <span class="IPA"><a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Международный фонетический алфавит">ʃ<sup>j</sup></a></span>, <span class="IPA"><a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Международный фонетический алфавит">ʒ<sup>j</sup></a></span>, <span class="IPA"><a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Международный фонетический алфавит">dʒ<sup>j</sup></a></span>), а фонологию тремя новыми <a href="/wiki/%D0%A4%D0%BE%D0%BD%D0%B5%D0%BC%D0%B0" title="Фонема">фонемами</a> (<i>č</i>, <i>š</i>, <i>ž</i>). По мнению ряда учёных, <a href="/wiki/%D0%A4%D0%BE%D0%BD%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F" title="Фонологизация">фонологизация</a> <i>č</i>, <i>š</i>, <i>ž</i> произошла после изменения <i>ē</i> в <i>ā</i> после мягких согласных<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup><sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup>.

<h3><span id=".D0.92_.D0.B3.D1.80.D0.B0.D1.84.D0.B8.D0.BA.D0.B5"></span><span class="mw-headline" id="В_графике">В графике</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9F%D0%B5%D1%80%D0%B2%D0%B0%D1%8F_%D0%BF%D0%B0%D0%BB%D0%B0%D1%82%D0%B0%D0%BB%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F&amp;veaction=edit&amp;section=5" title="Редактировать раздел «В графике»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9F%D0%B5%D1%80%D0%B2%D0%B0%D1%8F_%D0%BF%D0%B0%D0%BB%D0%B0%D1%82%D0%B0%D0%BB%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F&amp;action=edit&amp;section=5" title="Редактировать код раздела «В графике»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>В <a href="/wiki/%D0%93%D1%80%D0%B5%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Греческий алфавит">греческом</a> и <a class="mw-redirect" href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Латинский алфавит">латинском</a> <a href="/wiki/%D0%90%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Алфавит">алфавитах</a>, на основе которых была создана письменность славян, не было <a href="/wiki/%D0%91%D1%83%D0%BA%D0%B2%D0%B0" title="Буква">букв</a> для обозначения звуков, получившихся в результате первой палатализации. Поэтому в состав <a href="/wiki/%D0%9A%D0%B8%D1%80%D0%B8%D0%BB%D0%BB%D0%B8%D1%86%D0%B0" title="Кириллица">кириллицы</a> были введены из <a href="/wiki/%D0%93%D0%BB%D0%B0%D0%B3%D0%BE%D0%BB%D0%B8%D1%86%D0%B0" title="Глаголица">глаголицы</a> буквы <a href="/wiki/%D0%A7" title="Ч">ч</a>, <a href="/wiki/%D0%96" title="Ж">ж</a> и <a href="/wiki/%D0%A8" title="Ш">ш</a>. В <a href="/wiki/%D0%A7%D0%B5%D1%88%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Чешский алфавит">чешском алфавите</a> <a class="mw-redirect" href="/wiki/%D0%AF%D0%BD_%D0%93%D1%83%D1%81" title="Ян Гус">Ян Гус</a> ввёл обозначение этих звуков при помощи точек над буквами <i>s</i>, <i>z</i> и <i>c</i>, которые позднее были заменены <a href="/wiki/%D0%93%D0%B0%D1%87%D0%B5%D0%BA" title="Гачек">крючком</a> (<a href="/wiki/%D0%A7%D0%B5%D1%88%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Чешский язык">чеш.</a> <span lang="cs" style="font-style:italic;">háček</span>)<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup>. Современный <a href="/wiki/%D0%9F%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Польский язык">польский язык</a> пользуется <a href="/wiki/%D0%94%D0%B8%D0%B3%D1%80%D0%B0%D1%84" title="Диграф">диграфами</a> для обозначения <span class="IPA"><a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Международный фонетический алфавит">tʃ</a></span> и <span class="IPA"><a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Международный фонетический алфавит">ʃ</a></span>: <i>cz</i> и <i>sz</i>, и точкой над буквой <i>z</i> (<a href="/wiki/%C5%BB" title="Ż">ż</a>) для обозначения <span class="IPA"><a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Международный фонетический алфавит">ʒ</a></span>.

<li>В <a href="/wiki/%D0%9F%D1%80%D0%BE%D1%82%D0%BE%D0%B0%D1%80%D0%BC%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Протоармянский язык">праармянском</a> в положении перед гласными переднего ряда и <i>j</i> <a href="/wiki/%D0%A1%D0%BE%D0%B3%D0%BB%D0%B0%D1%81%D0%BD%D1%8B%D0%B5" title="Согласные">согласные</a> <i>k</i>, <i>g</i>, <i>g<sup>h</sup></i> дали <i>tʃ<sup>h</sup></i>, <i>tʃ</i> и <i>dʒ</i> соответственно, например, <a href="/wiki/%D0%9F%D1%80%D0%B0%D0%B8%D0%BD%D0%B4%D0%BE%D0%B5%D0%B2%D1%80%D0%BE%D0%BF%D0%B5%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Праиндоевропейский язык">пра-и.е.</a> <span class="Unicode"><span lang="ieur"><i>g<sup>wh</sup>eros</i></span></span> «тёплый» &gt; <a href="/wiki/%D0%90%D1%80%D0%BC%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Армянский язык">арм.</a> <span lang="hy">ǰer</span>, <a href="/wiki/%D0%9F%D1%80%D0%B0%D0%B8%D0%BD%D0%B4%D0%BE%D0%B5%D0%B2%D1%80%D0%BE%D0%BF%D0%B5%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Праиндоевропейский язык">пра-и.е.</a> <span class="Unicode"><span lang="ieur"><i>gem-</i></span></span> «жать» &gt; <a href="/wiki/%D0%90%D1%80%D0%BC%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Армянский язык">арм.</a> <span lang="hy">čim</span> «<a href="/wiki/%D0%A3%D0%B7%D0%B4%D0%B0" title="Узда">узда</a>», <a href="/wiki/%D0%9F%D1%80%D0%B0%D0%B8%D0%BD%D0%B4%D0%BE%D0%B5%D0%B2%D1%80%D0%BE%D0%BF%D0%B5%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Праиндоевропейский язык">пра-и.е.</a> <span class="Unicode"><span lang="ieur"><i>*k<sup>w</sup>etūres</i></span></span> «четыре» &gt; <a href="/wiki/%D0%90%D1%80%D0%BC%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Армянский язык">арм.</a> <span lang="hy">č<sup>c</sup>ork<sup>c</sup></span>. В <a href="/wiki/%D0%90%D1%80%D0%BC%D0%B5%D0%BD%D0%B8%D1%81%D1%82%D0%B8%D0%BA%D0%B0" title="Арменистика">арменистике</a> этот процесс также называется второй палатализацией<sup class="reference" id="cite_ref-42"><a href="#cite_note-42">[42]</a></sup>.</li></ul>
