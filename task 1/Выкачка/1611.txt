<p><b>Алекса́ндр Ива́нович Каза́рский</b> (<a class="mw-redirect" href="/wiki/1797" title="1797">1797</a>, <a href="/wiki/%D0%9E%D1%80%D1%88%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%83%D0%B5%D0%B7%D0%B4" title="Оршанский уезд">Оршанский уезд</a>, <a href="/wiki/%D0%91%D0%B5%D0%BB%D0%BE%D1%80%D1%83%D1%81%D1%81%D0%BA%D0%B0%D1%8F_%D0%B3%D1%83%D0%B1%D0%B5%D1%80%D0%BD%D0%B8%D1%8F" title="Белорусская губерния">Белорусская губерния</a> — <abbr title="по юлианскому календарю">16</abbr> <a href="/wiki/28_%D0%B8%D1%8E%D0%BD%D1%8F" title="28 июня">[28] июня</a> <a href="/wiki/1833_%D0%B3%D0%BE%D0%B4" title="1833 год">1833</a>, <a href="/wiki/%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B5%D0%B2" title="Николаев">Николаев</a>) — российский военный моряк, герой <a class="mw-redirect" href="/wiki/%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%BE-%D1%82%D1%83%D1%80%D0%B5%D1%86%D0%BA%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0_1828%E2%80%941829_%D0%B3%D0%BE%D0%B4%D0%BE%D0%B2" title="Русско-турецкая война 1828—1829 годов">русско-турецкой войны</a> <a class="mw-redirect" href="/wiki/1828" title="1828">1828</a>—<a href="/wiki/1829_%D0%B3%D0%BE%D0%B4" title="1829 год">1829 годов</a>, <a href="/wiki/%D0%9A%D0%B0%D0%BF%D0%B8%D1%82%D0%B0%D0%BD_1-%D0%B3%D0%BE_%D1%80%D0%B0%D0%BD%D0%B3%D0%B0" title="Капитан 1-го ранга">капитан 1-го ранга</a> (1831)<sup class="reference" id="cite_ref-bre_1-0"><a href="#cite_note-bre-1">[1]</a></sup>, кавалер <a href="/wiki/%D0%9E%D1%80%D0%B4%D0%B5%D0%BD_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D1%8F" title="Орден Святого Георгия">ордена Святого Георгия</a>. В звании <a href="/wiki/%D0%A4%D0%BB%D0%B8%D0%B3%D0%B5%D0%BB%D1%8C-%D0%B0%D0%B4%D1%8A%D1%8E%D1%82%D0%B0%D0%BD%D1%82" title="Флигель-адъютант">флигель-адъютанта</a> состоял в свите императора <a href="/wiki/%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B9_I" title="Николай I">Николая I</a>. Получил широкую известность после того, как 18-пушечный бриг «<a href="/wiki/%D0%9C%D0%B5%D1%80%D0%BA%D1%83%D1%80%D0%B8%D0%B9_(%D0%B1%D1%80%D0%B8%D0%B3,_1820)" title="Меркурий (бриг, 1820)">Меркурий</a>» под его командованием одержал победу в бою с двумя турецкими <a href="/wiki/%D0%9B%D0%B8%D0%BD%D0%B5%D0%B9%D0%BD%D1%8B%D0%B9_%D0%BA%D0%BE%D1%80%D0%B0%D0%B1%D0%BB%D1%8C_(%D0%BF%D0%B0%D1%80%D1%83%D1%81%D0%BD%D1%8B%D0%B9)" title="Линейный корабль (парусный)">линейными кораблями</a>.
</p>

<h2><span id=".D0.91.D0.B8.D0.BE.D0.B3.D1.80.D0.B0.D1.84.D0.B8.D1.8F"></span><span class="mw-headline" id="Биография">Биография</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B0%D0%B7%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Биография»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B0%D0%B7%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=1" title="Редактировать код раздела «Биография»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id=".D0.9F.D1.80.D0.BE.D0.B8.D1.81.D1.85.D0.BE.D0.B6.D0.B4.D0.B5.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Происхождение">Происхождение</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B0%D0%B7%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Происхождение»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B0%D0%B7%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=2" title="Редактировать код раздела «Происхождение»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Родился в <a class="mw-redirect" href="/wiki/%D0%94%D1%83%D0%B1%D1%80%D0%BE%D0%B2%D0%BD%D0%BE_(%D0%92%D0%B8%D1%82%D0%B5%D0%B1%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C)" title="Дубровно (Витебская область)">Дубровно</a><sup class="reference" id="cite_ref-dbr_2-0"><a href="#cite_note-dbr-2">[2]</a></sup> (<a href="/wiki/%D0%9E%D1%80%D1%88%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%83%D0%B5%D0%B7%D0%B4" title="Оршанский уезд">Оршанский уезд</a>, <a href="/wiki/%D0%91%D0%B5%D0%BB%D0%BE%D1%80%D1%83%D1%81%D1%81%D0%BA%D0%B0%D1%8F_%D0%B3%D1%83%D0%B1%D0%B5%D1%80%D0%BD%D0%B8%D1%8F" title="Белорусская губерния">Белорусская губерния</a>) <abbr title="по юлианскому или григорианскому календарю?">16 июня 1797 года</abbr> в семье отставного <a href="/wiki/%D0%93%D1%83%D0%B1%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D0%B5%D0%BA%D1%80%D0%B5%D1%82%D0%B0%D1%80%D1%8C" title="Губернский секретарь">губернского секретаря</a>, управляющего имением князя <a href="/wiki/%D0%9B%D1%8E%D0%B1%D0%BE%D0%BC%D0%B8%D1%80%D1%81%D0%BA%D0%B8%D0%B5" title="Любомирские">Любомирского</a>. Александр был четвёртым ребёнком в семье Ивана Кузьмича и Татьяны Гавриловны Казарских. У него был младший брат <a class="new" href="/w/index.php?title=%D0%9A%D0%B0%D0%B7%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9,_%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B9_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87&amp;action=edit&amp;redlink=1" title="Казарский, Николай Иванович (страница отсутствует)">Николай</a> и три старших сестры: Прасковья, Екатерина, Матрёна<sup class="reference" id="cite_ref-runivers_3-0"><a href="#cite_note-runivers-3">[3]</a></sup>.
</p>
<h3><span id=".D0.A0.D0.B0.D0.BD.D0.BD.D0.B8.D0.B5_.D0.B3.D0.BE.D0.B4.D1.8B"></span><span class="mw-headline" id="Ранние_годы">Ранние годы</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B0%D0%B7%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Ранние годы»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B0%D0%B7%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=3" title="Редактировать код раздела «Ранние годы»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>В детстве Александр учился в <a href="/wiki/%D0%A6%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D0%BD%D0%BE-%D0%BF%D1%80%D0%B8%D1%85%D0%BE%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D1%88%D0%BA%D0%BE%D0%BB%D0%B0" title="Церковно-приходская школа">церковно-приходской школе</a>, где священник Дубровненского православного прихода преподавал ему грамоту, а молодой <a href="/wiki/%D0%9A%D1%81%D1%91%D0%BD%D0%B4%D0%B7" title="Ксёндз">ксёндз</a> — математику, латынь и французский язык. В доме Казарских не поощрялось <a href="/wiki/%D0%A7%D1%82%D0%B5%D0%BD%D0%B8%D0%B5" title="Чтение">чтение</a> книг, зато отец привил Александру уверенность в непоколебимости устоев империи<sup class="reference" id="cite_ref-biogr_4-0"><a href="#cite_note-biogr-4">[4]</a></sup>, дал чёткое понятие о чести и верности Отечеству.
</p><p>В 1808 году к Казарским приехал крёстный Александра — Василий Семёнович, двоюродный брат Ивана Кузьмича. Незадолго до этого он получил должность в обер-интендантстве <a class="mw-redirect" href="/wiki/%D0%A7%D0%B5%D1%80%D0%BD%D0%BE%D0%BC%D0%BE%D1%80%D1%81%D0%BA%D0%B8%D0%B9_%D1%84%D0%BB%D0%BE%D1%82_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B9%D1%81%D0%BA%D0%BE%D0%B9_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D0%B8" title="Черноморский флот Российской империи">Черноморского флота</a> и предложил определить Александра в Черноморское штурманское училище в Николаеве. Отец согласился и, по свидетельству капитан-лейтенанта Ивана Николаевича Сущева, первого биографа Казарского, сказал на прощание: «Честное имя, Саша, — это единственное, что оставлю тебе в наследство»<sup class="reference" id="cite_ref-belko2_5-0"><a href="#cite_note-belko2-5">[5]</a></sup>.
</p>
<h3><span id=".D0.9A.D0.B0.D1.80.D1.8C.D0.B5.D1.80.D0.B0"></span><span class="mw-headline" id="Карьера">Карьера</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B0%D0%B7%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Карьера»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B0%D0%B7%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=4" title="Редактировать код раздела «Карьера»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Александр Казарский поступил волонтёром на флот в 1811 году, став <a href="/wiki/%D0%9A%D0%B0%D0%B4%D0%B5%D1%82%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%BE%D1%80%D0%BF%D1%83%D1%81" title="Кадетский корпус">кадетом</a> <a href="/wiki/%D0%A7%D0%B5%D1%80%D0%BD%D0%BE%D0%BC%D0%BE%D1%80%D1%81%D0%BA%D0%BE%D0%B5_%D1%88%D1%82%D1%83%D1%80%D0%BC%D0%B0%D0%BD%D1%81%D0%BA%D0%BE%D0%B5_%D1%83%D1%87%D0%B8%D0%BB%D0%B8%D1%89%D0%B5" title="Черноморское штурманское училище">Николаевского штурманского училища</a>. Занятия в училище сопровождались историями о боевой славе русского флота. Среди учителей Казарского был Лука Андреевич Латышев, плававший с <a href="/wiki/%D0%A3%D1%88%D0%B0%D0%BA%D0%BE%D0%B2,_%D0%A4%D1%91%D0%B4%D0%BE%D1%80_%D0%A4%D1%91%D0%B4%D0%BE%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Ушаков, Фёдор Фёдорович">Ушаковым</a> и принимавший участие во <a href="/wiki/%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%9A%D0%BE%D1%80%D1%84%D1%83_(1798%E2%80%941799)" title="Осада Корфу (1798—1799)">взятии Корфу</a>. Со времён обучения в училище кумиром Казарского на всю жизнь стал адмирал <a href="/wiki/%D0%A1%D0%B5%D0%BD%D1%8F%D0%B2%D0%B8%D0%BD,_%D0%94%D0%BC%D0%B8%D1%82%D1%80%D0%B8%D0%B9_%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B5%D0%B2%D0%B8%D1%87" title="Сенявин, Дмитрий Николаевич">Сенявин</a><sup class="reference" id="cite_ref-biogr_4-1"><a href="#cite_note-biogr-4">[4]</a></sup>. Будучи скромным и застенчивым, Александр тяжело сходился с людьми и настоящих друзей не имел. Тем не менее, он сошёлся с некоторыми сокурсниками по кадетскому корпусу, из которых можно выделить <a href="/wiki/%D0%A7%D0%B8%D0%B6%D0%BE%D0%B2,_%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B9_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B5%D0%B2%D0%B8%D1%87" title="Чижов, Николай Алексеевич">Николая Чижова</a>, сына военного советника Алексея Петровича Чижов. Казарский и Чижов стали приятелями. Николай был начитан и привил Александру любовь к литературе<sup class="reference" id="cite_ref-biogr_4-2"><a href="#cite_note-biogr-4">[4]</a></sup>.
</p><p>30 августа 1813 года Казарский был записан <a href="/wiki/%D0%93%D0%B0%D1%80%D0%B4%D0%B5%D0%BC%D0%B0%D1%80%D0%B8%D0%BD" title="Гардемарин">гардемарином</a> на Черноморский флот, а в 1814-м произведён в <a href="/wiki/%D0%9C%D0%B8%D1%87%D0%BC%D0%B0%D0%BD" title="Мичман">мичманы</a><sup class="reference" id="cite_ref-Викитека_РБС_6-0"><a href="#cite_note-Викитека_РБС-6">[6]</a></sup> . В начале флотской карьеры он ходил на бригантинах «Десна» и «Клеопатра»<sup class="reference" id="cite_ref-biogr_4-3"><a href="#cite_note-biogr-4">[4]</a></sup>, перевозивших грузы между черноморскими портами<sup class="reference" id="cite_ref-rpf2_238_7-0"><a href="#cite_note-rpf2_238-7">[7]</a></sup>, а позже по собственному рапорту был направлен на <a href="/wiki/%D0%94%D1%83%D0%BD%D0%B0%D0%B9%D1%81%D0%BA%D0%B0%D1%8F_%D1%84%D0%BB%D0%BE%D1%82%D0%B8%D0%BB%D0%B8%D1%8F_(%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B9%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F)" title="Дунайская флотилия (Российская империя)">Дунайскую флотилию</a>, где его назначили командиром отряда мелких гребных судов в <a href="/wiki/%D0%98%D0%B7%D0%BC%D0%B0%D0%B8%D0%BB" title="Измаил">Измаиле</a><sup class="reference" id="cite_ref-runivers_3-1"><a href="#cite_note-runivers-3">[3]</a></sup>. Перед отправкой в Измаил Александр посетил Дубровно и нашёл родной дом в запустении: отец умер, сёстры Прасковья и Екатерина вышли замуж, мать Татьяна Гавриловна уехала на свою родину в <a href="/wiki/%D0%9C%D0%B0%D0%BB%D0%B0%D1%8F_%D0%A0%D1%83%D1%81%D1%8C" title="Малая Русь">Малороссию</a>, а Матрёна погибла, бросившись в <a href="/wiki/%D0%94%D0%BD%D0%B5%D0%BF%D1%80" title="Днепр">Днепр</a>, когда спасалась от преследовавших её французских солдат, <a href="/wiki/%D0%9E%D1%82%D0%B5%D1%87%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0_1812_%D0%B3%D0%BE%D0%B4%D0%B0" title="Отечественная война 1812 года">занявших город в 1812 году</a><sup class="reference" id="cite_ref-biogr_4-4"><a href="#cite_note-biogr-4">[4]</a></sup>.
</p><p>В 1819 году Казарский был произведён в <a href="/wiki/%D0%9B%D0%B5%D0%B9%D1%82%D0%B5%D0%BD%D0%B0%D0%BD%D1%82" title="Лейтенант">лейтенанты</a> и назначен на фрегат «<a href="/wiki/%D0%95%D0%B2%D1%81%D1%82%D0%B0%D1%84%D0%B8%D0%B9_(%D1%84%D1%80%D0%B5%D0%B3%D0%B0%D1%82,_1817)" title="Евстафий (фрегат, 1817)">Евстафий</a>», который отправился в <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B0%D1%81%D1%82%D0%BE%D0%BF%D0%BE%D0%BB%D1%8C" title="Севастополь">Севастополь</a>. На Черноморском флоте Казарский служил под началом <a href="/wiki/%D0%A1%D0%BA%D0%B0%D0%BB%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9,_%D0%98%D0%B2%D0%B0%D0%BD_%D0%A1%D0%B5%D0%BC%D1%91%D0%BD%D0%BE%D0%B2%D0%B8%D1%87" title="Скаловский, Иван Семёнович">Ивана Семёновича Скаловского</a>, которого считал своим кумиром с юности<sup class="reference" id="cite_ref-runivers_3-2"><a href="#cite_note-runivers-3">[3]</a></sup>. Под началом Скаловского Казарский прошёл хорошую командирскую школу, усвоил основные принципы, которыми должен руководствоваться офицер: действовать самостоятельно и решительно, уметь установить взаимопонимание с экипажем, разгадывать замыслы и опережать действия противника<sup class="reference" id="cite_ref-biogr_4-5"><a href="#cite_note-biogr-4">[4]</a></sup>.
</p><p>После службы на «Евстафии» Казарский выходил в практические плавания<sup class="reference" id="cite_ref-rpf2_138_8-0"><a href="#cite_note-rpf2_138-8">[8]</a></sup> на <a href="/wiki/%D0%A8%D1%85%D1%83%D0%BD%D0%B0" title="Шхуна">шхуне</a> «Севастополь», служил на транспортах «Ингул» и «Соперник»<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>, на катере «Сокол»<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>, бриге «Меркурий», командиром которого стал через несколько лет, и на линейном корабле<sup class="reference" id="cite_ref-biogr_4-6"><a href="#cite_note-biogr-4">[4]</a></sup>.
</p><p>В 1828 году Казарский командовал транспортным судном «<a href="/wiki/%D0%A1%D0%BE%D0%BF%D0%B5%D1%80%D0%BD%D0%B8%D0%BA_(%D0%B1%D0%BE%D0%BC%D0%B1%D0%B0%D1%80%D0%B4%D0%B8%D1%80%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%BE%D1%80%D0%B0%D0%B1%D0%BB%D1%8C)" title="Соперник (бомбардирский корабль)">Соперник</a>»<sup class="reference" id="cite_ref-rpf2_29_11-0"><a href="#cite_note-rpf2_29-11">[11]</a></sup>. Судно участвовало в высадке войск третьей бригады и доставке вооружений. <a href="/wiki/%D0%93%D1%80%D0%B5%D0%B9%D0%B3,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%A1%D0%B0%D0%BC%D1%83%D0%B8%D0%BB%D0%BE%D0%B2%D0%B8%D1%87" title="Грейг, Алексей Самуилович">А. С. Грейг</a> распорядился оборудовать транспорт «<a href="/wiki/%D0%95%D0%B4%D0%B8%D0%BD%D0%BE%D1%80%D0%BE%D0%B3_(%D0%B3%D0%B0%D1%83%D0%B1%D0%B8%D1%86%D0%B0)" title="Единорог (гаубица)">единорогом</a>», что перевело транспортное судно в разряд <a href="/wiki/%D0%91%D0%BE%D0%BC%D0%B1%D0%B0%D1%80%D0%B4%D0%B8%D1%80%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%BE%D1%80%D0%B0%D0%B1%D0%BB%D1%8C" title="Бомбардирский корабль">бомбардирских кораблей</a>. В то время, как основной флот не мог подойти к <a href="/wiki/%D0%90%D0%BD%D0%B0%D0%BF%D1%81%D0%BA%D0%B0%D1%8F_%D0%BA%D1%80%D0%B5%D0%BF%D0%BE%D1%81%D1%82%D1%8C" title="Анапская крепость">Анапской крепости</a> по мелководью, «Соперник» под командованием Казарского в течение трёх недель, маневрируя, обстреливал её укрепления<sup class="reference" id="cite_ref-biogr_4-7"><a href="#cite_note-biogr-4">[4]</a></sup>. За время осады Анапы «Соперник» получил шесть пробоин корпуса и два повреждения <a href="/wiki/%D0%A0%D0%B0%D0%BD%D0%B3%D0%BE%D1%83%D1%82" title="Рангоут">рангоута</a><sup class="reference" id="cite_ref-rpf2_29_11-1"><a href="#cite_note-rpf2_29-11">[11]</a></sup>, но до последнего дня осады продолжал атаковать крепость. За участие во взятии Анапы Александр Казарский был произведён в <a href="/wiki/%D0%9A%D0%B0%D0%BF%D0%B8%D1%82%D0%B0%D0%BD-%D0%BB%D0%B5%D0%B9%D1%82%D0%B5%D0%BD%D0%B0%D0%BD%D1%82" title="Капитан-лейтенант">капитан-лейтенанты</a><sup class="reference" id="cite_ref-biogr_4-8"><a href="#cite_note-biogr-4">[4]</a></sup><sup class="reference" id="cite_ref-taras504_12-0"><a href="#cite_note-taras504-12">[12]</a></sup>.
</p><p>В сентябре того же года по схожему сценарию была взята <a href="/wiki/%D0%92%D0%B0%D1%80%D0%BD%D0%B0" title="Варна">Варна</a>, и за проявленную при этом храбрость Казарский был награждён <a href="/wiki/%D0%97%D0%BE%D0%BB%D0%BE%D1%82%D0%BE%D0%B5_%D0%BE%D1%80%D1%83%D0%B6%D0%B8%D0%B5_%C2%AB%D0%97%D0%B0_%D1%85%D1%80%D0%B0%D0%B1%D1%80%D0%BE%D1%81%D1%82%D1%8C%C2%BB" title="Золотое оружие «За храбрость»">золотой саблей</a><sup class="reference" id="cite_ref-runivers_3-3"><a href="#cite_note-runivers-3">[3]</a></sup>.
</p>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Tkachenko_BrigMerkury.jpg"><img class="mw-file-element" data-file-height="780" data-file-width="1134" decoding="async" height="206" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/a2/Tkachenko_BrigMerkury.jpg/300px-Tkachenko_BrigMerkury.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/a2/Tkachenko_BrigMerkury.jpg/450px-Tkachenko_BrigMerkury.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/a2/Tkachenko_BrigMerkury.jpg/600px-Tkachenko_BrigMerkury.jpg 2x" width="300"/></a><figcaption>«Бой брига "Меркурий" с турецкими кораблями 14 мая 1829 года»</figcaption></figure>
<p>В 1829 году Грейг назначил Казарского командиром 18-пушечного брига «<a href="/wiki/%D0%9C%D0%B5%D1%80%D0%BA%D1%83%D1%80%D0%B8%D0%B9_(%D0%B1%D1%80%D0%B8%D0%B3,_1820)" title="Меркурий (бриг, 1820)">Меркурий</a>». Александр Иванович служил на «Меркурии» несколькими годами раньше, поэтому корабль был ему хорошо знаком<sup class="reference" id="cite_ref-biogr_4-9"><a href="#cite_note-biogr-4">[4]</a></sup>.
</p><p>
Под командованием капитан-лейтенанта Казарского «<a href="/wiki/%D0%9C%D0%B5%D1%80%D0%BA%D1%83%D1%80%D0%B8%D0%B9_(%D0%B1%D1%80%D0%B8%D0%B3,_1820)" title="Меркурий (бриг, 1820)">Меркурий</a>» совершил один из самых выдающихся подвигов в истории морских сражений. 14 мая 1829 года 18-пушечный бриг был настигнут двумя турецкими кораблями «Селимие» и «Реал-беем», имеющими в сумме десятикратное превосходство в количестве орудий. Приняв неравный бой, экипаж брига под командованием Казарского одержал блестящую победу, нанеся противнику повреждения, принудившие его выйти из боя<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup>. Турецкий офицер с «Реал-бея» писал позже<sup class="reference" id="cite_ref-irf_14-0"><a href="#cite_note-irf-14">[14]</a></sup>:</p><blockquote>
<p>В продолжение сражения <a href="/wiki/%D0%A1%D1%82%D1%80%D0%BE%D0%B9%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2,_%D0%A1%D0%B5%D0%BC%D1%91%D0%BD_%D0%9C%D0%B8%D1%85%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2%D0%B8%D1%87" title="Стройников, Семён Михайлович">командир</a> русского <a href="/wiki/%D0%A0%D0%B0%D1%84%D0%B0%D0%B8%D0%BB_(%D1%84%D1%80%D0%B5%D0%B3%D0%B0%D1%82,_1828)" title="Рафаил (фрегат, 1828)">фрегата</a><sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> говорил мне, что капитан сего брига никогда не сдастся, и если он потеряет всю надежду, то тогда взорвёт бриг свой на воздух. Ежели в великих деяниях древних и наших времён находятся подвиги храбрости, то сей поступок должен все оные помрачить, и имя сего героя достойно быть начертано золотыми литерами на храме Славы: он называется капитан-лейтенант Казарский, а бриг — «Меркурий».
</p>
</blockquote>

<p>За свой подвиг Казарский был произведён в <a href="/wiki/%D0%9A%D0%B0%D0%BF%D0%B8%D1%82%D0%B0%D0%BD_2-%D0%B3%D0%BE_%D1%80%D0%B0%D0%BD%D0%B3%D0%B0" title="Капитан 2-го ранга">капитаны II ранга</a>, награждён <a href="/wiki/%D0%9E%D1%80%D0%B4%D0%B5%D0%BD_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D1%8F" title="Орден Святого Георгия">орденом Святого Георгия</a> IV класса<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup> и назначен флигель-адъютантом<sup class="reference" id="cite_ref-ivanov_17-0"><a href="#cite_note-ivanov-17">[17]</a></sup>. Также в герб Казарского, как символ готовности пожертвовать собой<sup class="reference" id="cite_ref-taras505_18-0"><a href="#cite_note-taras505-18">[18]</a></sup>, было внесено изображение <a href="/wiki/%D0%A2%D1%83%D0%BB%D0%B0" title="Тула">тульского</a> пистолета, который Александр Иванович перед боем положил на шпиль у входа в <a href="/wiki/%D0%9A%D1%80%D1%8E%D0%B9%D1%82-%D0%BA%D0%B0%D0%BC%D0%B5%D1%80%D0%B0" title="Крюйт-камера">крюйт-камеру</a> для того, чтобы последний из оставшихся в живых офицеров выстрелом взорвал порох<sup class="reference" id="cite_ref-raport_19-0"><a href="#cite_note-raport-19">[19]</a></sup>.

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>Щит разделен на да три части, из коих в первой в голубом поле изображен золотый пистолет и под оным серебряная луна, рогами вниз обращенная. Во второй в золотом поле между лавровою и масличною ветвиями чернаго цвета стропило. В нижней пространной части В серебряном поле военное судно с распущенными парусами. Посредине в малом щитке изображен употребляемый в роде Козарскаго герб, то есть: в красном поле извивающийся уж, увенчанный дворянскою короною, и держащий во рту яблоко. Щит увенчан дворянскими шлемом и короною с пятью на оной строусовыми перьями. Намет на щите голубый и золотый, подложенный серебром и красным.
</p>
<div class="ts-Конец_цитаты-source">— <cite>Герб Казарского внесён в Часть 10 <a href="/wiki/%D0%9E%D0%B1%D1%89%D0%B8%D0%B9_%D0%B3%D0%B5%D1%80%D0%B1%D0%BE%D0%B2%D0%BD%D0%B8%D0%BA_%D0%B4%D0%B2%D0%BE%D1%80%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D1%85_%D1%80%D0%BE%D0%B4%D0%BE%D0%B2_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B9%D1%81%D0%BA%D0%BE%D0%B9_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D0%B8" title="Общий гербовник дворянских родов Российской империи">Общего гербовника дворянских родов Всероссийской империи</a>, стр. 122<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup></cite></div></blockquote>
<p>В 1829 году, с 26 мая по 17 июля Казарский командовал 44-пушечным <a href="/wiki/%D0%A4%D1%80%D0%B5%D0%B3%D0%B0%D1%82" title="Фрегат">фрегатом</a> «<a href="/wiki/%D0%9F%D0%BE%D1%81%D0%BF%D0%B5%D1%88%D0%BD%D1%8B%D0%B9_(%D1%84%D1%80%D0%B5%D0%B3%D0%B0%D1%82,_1821)" title="Поспешный (фрегат, 1821)">Поспешный</a>»<sup class="reference" id="cite_ref-rpf1_241_21-0"><a href="#cite_note-rpf1_241-21">[21]</a></sup> и принял участие <a href="/wiki/%D0%92%D0%B7%D1%8F%D1%82%D0%B8%D0%B5_%D0%9C%D0%B5%D1%81%D0%B5%D0%BC%D0%B2%D1%80%D0%B8%D0%B8" title="Взятие Месемврии">во взятии Месемврии</a><sup class="reference" id="cite_ref-biogr_4-10"><a href="#cite_note-biogr-4">[4]</a></sup>. С 17 июля 1829 по 1830 год он был капитаном 60-пушечного фрегата «<a href="/wiki/%D0%A2%D0%B5%D0%BD%D0%B5%D0%B4%D0%BE%D1%81_(%D1%84%D1%80%D0%B5%D0%B3%D0%B0%D1%82)" title="Тенедос (фрегат)">Тенедос</a>»<sup class="reference" id="cite_ref-rpf1_251_22-0"><a href="#cite_note-rpf1_251-22">[22]</a></sup>. Этот фрегат относился к самым крупным фрегатам русского флота, которые иногда называли 60-пушечными <a href="/wiki/%D0%9B%D0%B8%D0%BD%D0%B5%D0%B9%D0%BD%D1%8B%D0%B9_%D0%BA%D0%BE%D1%80%D0%B0%D0%B1%D0%BB%D1%8C_(%D0%BF%D0%B0%D1%80%D1%83%D1%81%D0%BD%D1%8B%D0%B9)" title="Линейный корабль (парусный)">линейными кораблями</a>. До октября 1829 года «Тенедос» под командованием Казарского трижды выходил к <a href="/wiki/%D0%91%D0%BE%D1%81%D1%84%D0%BE%D1%80" title="Босфор">Босфору</a>.
</p><p>В 1830 году Казарский был отправлен в Англию с князем Трубецким для поздравления короля <a href="/wiki/%D0%92%D0%B8%D0%BB%D1%8C%D0%B3%D0%B5%D0%BB%D1%8C%D0%BC_IV_(%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D1%8C_%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%BE%D0%B1%D1%80%D0%B8%D1%82%D0%B0%D0%BD%D0%B8%D0%B8)" title="Вильгельм IV (король Великобритании)">Вильгельма IV</a><sup class="reference" id="cite_ref-mes_23-0"><a href="#cite_note-mes-23">[23]</a></sup>. В 1831 году за отличную службу был произведён в <a class="mw-redirect" href="/wiki/%D0%9A%D0%B0%D0%BF%D0%B8%D1%82%D0%B0%D0%BD_1_%D1%80%D0%B0%D0%BD%D0%B3%D0%B0" title="Капитан 1 ранга">капитаны 1-го ранга</a><sup class="reference" id="cite_ref-taras504_12-1"><a href="#cite_note-taras504-12">[12]</a></sup>, после чего был уволен от командования кораблём и поступил в <a href="/wiki/%D0%A1%D0%B2%D0%B8%D1%82%D0%B0_%D0%95%D0%B3%D0%BE_%D0%98%D0%BC%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%BE%D1%80%D1%81%D0%BA%D0%BE%D0%B3%D0%BE_%D0%92%D0%B5%D0%BB%D0%B8%D1%87%D0%B5%D1%81%D1%82%D0%B2%D0%B0" title="Свита Его Императорского Величества">свиту</a> <a href="/wiki/%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B9_I" title="Николай I">Николая I</a><sup class="reference" id="cite_ref-runivers_3-4"><a href="#cite_note-runivers-3">[3]</a></sup>. Состоя в свите, был командирован в <a href="/wiki/%D0%9A%D0%B0%D0%B7%D0%B0%D0%BD%D1%8C" title="Казань">Казань</a> для определения целесообразности дальнейшего существования <a href="/wiki/%D0%9A%D0%B0%D0%B7%D0%B0%D0%BD%D1%81%D0%BA%D0%BE%D0%B5_%D0%B0%D0%B4%D0%BC%D0%B8%D1%80%D0%B0%D0%BB%D1%82%D0%B5%D0%B9%D1%81%D1%82%D0%B2%D0%BE" title="Казанское адмиралтейство">Казанского адмиралтейства</a>. После командировки прошёл по рекам и озёрам из <a href="/wiki/%D0%91%D0%B5%D0%BB%D0%BE%D0%B5_%D0%BC%D0%BE%D1%80%D0%B5" title="Белое море">Белого моря</a> до <a href="/wiki/%D0%9E%D0%BD%D0%B5%D0%B6%D1%81%D0%BA%D0%BE%D0%B5_%D0%BE%D0%B7%D0%B5%D1%80%D0%BE" title="Онежское озеро">Онеги</a> в поисках нового водного пути<sup class="reference" id="cite_ref-mes_23-1"><a href="#cite_note-mes-23">[23]</a></sup>.
</p>
<h3><span id=".D0.A1.D0.BC.D0.B5.D1.80.D1.82.D1.8C"></span><span class="mw-headline" id="Смерть">Смерть</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B0%D0%B7%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Смерть»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B0%D0%B7%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=5" title="Редактировать код раздела «Смерть»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>В 1833 году Казарский был направлен для проведения ревизии и проверки тыловых контор и складов в черноморских портах, но через короткое время после прибытия в <a class="mw-redirect" href="/wiki/%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B5%D0%B2_(%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B5%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C)" title="Николаев (Николаевская область)">Николаев</a> внезапно скончался от отравления. Для отравления, предположительно, использовался <a href="/wiki/%D0%9A%D0%BE%D1%84%D0%B5" title="Кофе">кофе</a> с <a href="/wiki/%D0%9C%D1%8B%D1%88%D1%8C%D1%8F%D0%BA" title="Мышьяк">мышьяком</a>.
</p>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Kazarsky_grave.jpg"><img class="mw-file-element" data-file-height="3616" data-file-width="2694" decoding="async" height="242" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/ad/Kazarsky_grave.jpg/180px-Kazarsky_grave.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/ad/Kazarsky_grave.jpg/270px-Kazarsky_grave.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/ad/Kazarsky_grave.jpg/360px-Kazarsky_grave.jpg 2x" width="180"/></a><figcaption>Могила А. И. Казарского в <a class="mw-redirect" href="/wiki/%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B5%D0%B2_(%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B5%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C)" title="Николаев (Николаевская область)">Николаеве</a></figcaption></figure>
<p>Дядя Казарского Моцкепич (по другим данным, Алексей Ильич Мацкевич<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup>) оставил ему в наследство 70 тысяч рублей, шкатулка с которыми была разграблена с участием <a href="/wiki/%D0%90%D0%B2%D1%82%D0%BE%D0%BC%D0%BE%D0%BD%D0%BE%D0%B2,_%D0%93%D1%80%D0%B8%D0%B3%D0%BE%D1%80%D0%B8%D0%B9_%D0%93%D1%80%D0%B8%D0%B3%D0%BE%D1%80%D1%8C%D0%B5%D0%B2%D0%B8%D1%87" title="Автомонов, Григорий Григорьевич">Григория Автомонова</a>, николаевского <a href="/wiki/%D0%9F%D0%BE%D0%BB%D0%B8%D1%86%D0%BC%D0%B5%D0%B9%D1%81%D1%82%D0%B5%D1%80" title="Полицмейстер">полицмейстера</a>. По свидетельству графа <a href="/wiki/%D0%91%D0%B5%D0%BD%D0%BA%D0%B5%D0%BD%D0%B4%D0%BE%D1%80%D1%84,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%BE%D1%84%D0%BE%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Бенкендорф, Александр Христофорович">А. Х. Бенкендорфа</a><sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup>, Казарский собирался непременно отыскать виновного. Бенкендорф утверждал, что Автомонов имел отношения с супругой <a href="/wiki/%D0%9A%D0%B0%D0%BF%D0%B8%D1%82%D0%B0%D0%BD-%D0%BA%D0%BE%D0%BC%D0%B0%D0%BD%D0%B4%D0%BE%D1%80" title="Капитан-командор">капитан-командора</a> Михайловой, приятельница которой, Роза Ивановна, имела близкое знакомство с неким аптекарем<sup class="reference" id="cite_ref-benken_26-0"><a href="#cite_note-benken-26">[26]</a></sup>.
</p><p>Казарский, отобедав у Михайловой, выпил чашку кофе и почувствовал себя плохо<sup class="reference" id="cite_ref-benken_26-1"><a href="#cite_note-benken-26">[26]</a></sup>. Близкая знакомая Казарских, Елизавета Фаренникова, утверждала, что в последние дни Казарский, заходя к кому-либо, ничего не ел и не пил, так как был предупреждён о возможном покушении. Даже некую немку, у которой остановился в Николаеве, он просил попробовать каждое блюдо, прежде чем самому приступить к еде. Однако Казарский не смог отказать красавице-дочери хозяина дома, которая поднесла ему чашку с отравленным кофе. За разговором Александр Иванович выпил всю чашку<sup class="reference" id="cite_ref-faren_27-0"><a href="#cite_note-faren-27">[27]</a></sup>. По утверждению штабс-лекаря Петрушевского, к которому обратился Казарский, тот постоянно плевал, от чего на полу образовались чёрные пятна, которые не удавалось смыть<sup class="reference" id="cite_ref-benken_26-2"><a href="#cite_note-benken-26">[26]</a></sup>. Фаренникова утверждает, что и доктор был в сговоре против Казарского, так как вместо того, чтобы дать ему <a href="/wiki/%D0%9F%D1%80%D0%BE%D1%82%D0%B8%D0%B2%D0%BE%D1%8F%D0%B4%D0%B8%D0%B5" title="Противоядие">противоядие</a>, усадил его в горячую ванну несмотря на то, что сам Казарский говорил ему, что отравлен<sup class="reference" id="cite_ref-faren_27-1"><a href="#cite_note-faren-27">[27]</a></sup>. После смерти тело Казарского почернело, голова и грудь раздулись, лицо обвалилось, волосы выпали, глазные яблоки лопнули, а ноги по ступни отвалились в гробу<sup class="reference" id="cite_ref-benken_26-3"><a href="#cite_note-benken-26">[26]</a></sup>. Эти изменения произошли менее чем за двое суток, и некоторые авторы утверждают, что их причиной стало не отравление, а летняя жара, а причиной смерти стал обыкновенный <a href="/wiki/%D0%93%D1%80%D0%B8%D0%BF%D0%BF" title="Грипп">грипп</a><sup class="reference" id="cite_ref-gorb_28-0"><a href="#cite_note-gorb-28">[28]</a></sup>. В своей записке Бенкендорф говорит, что следствие <a href="/wiki/%D0%93%D1%80%D0%B5%D0%B9%D0%B3,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%A1%D0%B0%D0%BC%D1%83%D0%B8%D0%BB%D0%BE%D0%B2%D0%B8%D1%87" title="Грейг, Алексей Самуилович">Грейга</a> по делу о смерти Казарского ничего не открыло и другое следствие вряд ли будет успешным, поскольку Автомонов, участие которого в заговоре против Казарского подозревал граф, является близким родственником <a href="/wiki/%D0%93%D0%B5%D0%BD%D0%B5%D1%80%D0%B0%D0%BB-%D0%B0%D0%B4%D1%8A%D1%8E%D1%82%D0%B0%D0%BD%D1%82" title="Генерал-адъютант">генерал-адъютанта</a> <a href="/wiki/%D0%9B%D0%B0%D0%B7%D0%B0%D1%80%D0%B5%D0%B2,_%D0%9C%D0%B8%D1%85%D0%B0%D0%B8%D0%BB_%D0%9F%D0%B5%D1%82%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Лазарев, Михаил Петрович">Лазарева</a><sup class="reference" id="cite_ref-benken_26-4"><a href="#cite_note-benken-26">[26]</a></sup>.
</p><p>По мнению историка флота Владимира Шигина версия, представленная в записке Бенкендорфа об отравлении из-за наследства, сфабрикована. Фактической причиной отравления была деятельность Казарского как ревизора Черноморского флота и черноморских портов и вскрытие им фактов злоупотребления и коррупции высших флотских начальников под руководством адмирала <a href="/wiki/%D0%93%D1%80%D0%B5%D0%B9%D0%B3,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%A1%D0%B0%D0%BC%D1%83%D0%B8%D0%BB%D0%BE%D0%B2%D0%B8%D1%87" title="Грейг, Алексей Самуилович">А. С. Грейга</a>.<sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[29]</a></sup>
</p>
<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Pushkin_kazarsky.jpg"><img class="mw-file-element" data-file-height="800" data-file-width="600" decoding="async" height="267" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/c4/Pushkin_kazarsky.jpg/200px-Pushkin_kazarsky.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/c/c4/Pushkin_kazarsky.jpg/300px-Pushkin_kazarsky.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/c/c4/Pushkin_kazarsky.jpg/400px-Pushkin_kazarsky.jpg 2x" width="200"/></a><figcaption>Рисунок <a href="/wiki/%D0%9F%D1%83%D1%88%D0%BA%D0%B8%D0%BD,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%A1%D0%B5%D1%80%D0%B3%D0%B5%D0%B5%D0%B2%D0%B8%D1%87" title="Пушкин, Александр Сергеевич">А. С. Пушкина</a>. Казарский в верхнем ряду слева</figcaption></figure>
<p>Внешние изменения Казарского после смерти подтверждала и Елизавета Фаренникова, бывшая тому свидетельницей: «голова, лицо распухли до невозможности, почернели, как уголь; руки распухли, почернели аксельбанты, эполеты, всё почернело… когда стали класть в гроб, то волосы упали на подушку»<sup class="reference" id="cite_ref-ivanov_17-1"><a href="#cite_note-ivanov-17">[17]</a></sup><sup class="reference" id="cite_ref-gorb_28-1"><a href="#cite_note-gorb-28">[28]</a></sup>. Гибель Казарского Фаренникова связывает с его ревизорской деятельностью и беспорядками и злоупотреблениями, царившими в то время на флоте<sup class="reference" id="cite_ref-faren_27-2"><a href="#cite_note-faren-27">[27]</a></sup>.
</p><p>Во время похорон за гробом шло множество людей, среди которых были вдовы и сироты, которым Казарский много помогал. Рыдая, они кричали: «Убили, погубили нашего благодетеля! Отравили нашего отца!»<sup class="reference" id="cite_ref-faren_27-3"><a href="#cite_note-faren-27">[27]</a></sup>.
</p><p>Через шесть месяцев из Санкт-Петербурга прибыла следственная комиссия, которая эксгумировала труп и извлекла внутренние органы для отправки в столицу, однако, как вспоминает Фаренникова, этим дело и кончилось<sup class="reference" id="cite_ref-30"><a href="#cite_note-30">[30]</a></sup>.
</p><p>Казарский был лично знаком с <a href="/wiki/%D0%9F%D1%83%D1%88%D0%BA%D0%B8%D0%BD,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%A1%D0%B5%D1%80%D0%B3%D0%B5%D0%B5%D0%B2%D0%B8%D1%87" title="Пушкин, Александр Сергеевич">А. С. Пушкиным</a>, <a class="mw-redirect" href="/wiki/%D0%92%D1%8F%D0%B7%D0%B5%D0%BC%D1%81%D0%BA%D0%B8%D0%B9,_%D0%9F%D0%B5%D1%82%D1%80_%D0%90%D0%BD%D0%B4%D1%80%D0%B5%D0%B5%D0%B2%D0%B8%D1%87" title="Вяземский, Петр Андреевич">П. А. Вяземским</a> и <a href="/wiki/%D0%94%D0%B0%D0%BB%D1%8C,_%D0%9A%D0%B0%D1%80%D0%BB_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87" title="Даль, Карл Иванович">К. И. Далем</a>. Известен «пророческий» рисунок Пушкина, на котором тот изобразил портреты Казарского, Сильво, Фурнье, Даля и Зайцевского (над рисунком сделана подпись заглавных букв фамилий изображённых людей: Q, S, F, D, Z) и топор, касающийся Даля и Казарского, которые после были отравлены в Николаеве<sup class="reference" id="cite_ref-31"><a href="#cite_note-31">[31]</a></sup>.
</p>
<h2><span id=".D0.A3.D0.B2.D0.B5.D0.BA.D0.BE.D0.B2.D0.B5.D1.87.D0.B8.D0.B2.D0.B0.D0.BD.D0.B8.D0.B5_.D0.BF.D0.B0.D0.BC.D1.8F.D1.82.D0.B8"></span><span class="mw-headline" id="Увековечивание_памяти">Увековечивание памяти</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B0%D0%B7%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Увековечивание памяти»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B0%D0%B7%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=6" title="Редактировать код раздела «Увековечивание памяти»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>В хранящемся в <a href="/wiki/%D0%93%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9_%D0%BC%D1%83%D0%B7%D0%B5%D0%B9_%D0%90._%D0%A1._%D0%9F%D1%83%D1%88%D0%BA%D0%B8%D0%BD%D0%B0" title="Государственный музей А. С. Пушкина">Государственном музее А. С. Пушкина</a> альбоме, принадлежавшем некогда Екатерине Юрковской — дочери генерал-майора <a href="/wiki/%D0%AE%D1%80%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9,_%D0%90%D0%BD%D0%B0%D1%81%D1%82%D0%B0%D1%81%D0%B8%D0%B9_%D0%90%D0%BD%D1%82%D0%BE%D0%BD%D0%BE%D0%B2%D0%B8%D1%87" title="Юрковский, Анастасий Антонович">А. А. Юрковского</a>, жившей в 1828—1831 годах в Севастополе, сохранилось несколько строк, написанных Казарским к любительскому рисунку с изображением легендарного боя брига «Меркурий»<sup class="reference" id="cite_ref-32"><a href="#cite_note-32">[32]</a></sup>
</p>
<h3><span id=".D0.92_.D0.BF.D0.BE.D1.8D.D0.B7.D0.B8.D0.B8"></span><span class="mw-headline" id="В_поэзии">В поэзии</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B0%D0%B7%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=7" title="Редактировать раздел «В поэзии»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B0%D0%B7%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=7" title="Редактировать код раздела «В поэзии»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Поэт <a href="/wiki/%D0%94%D0%B0%D0%B2%D1%8B%D0%B4%D0%BE%D0%B2,_%D0%94%D0%B5%D0%BD%D0%B8%D1%81_%D0%92%D0%B0%D1%81%D0%B8%D0%BB%D1%8C%D0%B5%D0%B2%D0%B8%D1%87" title="Давыдов, Денис Васильевич">Д. В. Давыдов</a> в стихотворении «<a href="/wiki/%D0%97%D0%B0%D0%B9%D1%86%D0%B5%D0%B2%D1%81%D0%BA%D0%B8%D0%B9,_%D0%95%D1%84%D0%B8%D0%BC_%D0%9F%D0%B5%D1%82%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Зайцевский, Ефим Петрович">Зайцевскому</a>, поэту-моряку», посвящённому ещё одному участнику русско-турецкой войны 1828—1829 годов, герою <a class="mw-redirect" href="/wiki/%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%92%D0%B0%D1%80%D0%BD%D1%8B" title="Осада Варны">штурма Варны</a>, писал<sup class="reference" id="cite_ref-33"><a href="#cite_note-33">[33]</a></sup>: «<i>Мужайся! — Казарский, живой <a href="/wiki/%D0%9B%D0%B5%D0%BE%D0%BD%D0%B8%D0%B4_I" title="Леонид I">Леонид</a>, / Ждет друга на новый пир славы… /О, будьте вы оба отечества щит…</i>»
</p>

<p>Первым предложил увековечить подвиг брига командующий Черноморской эскадрой адмирал <a href="/wiki/%D0%9B%D0%B0%D0%B7%D0%B0%D1%80%D0%B5%D0%B2,_%D0%9C%D0%B8%D1%85%D0%B0%D0%B8%D0%BB_%D0%9F%D0%B5%D1%82%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Лазарев, Михаил Петрович">М. П. Лазарев</a>. По его же инициативе проводился сбор средств на сооружение памятника<sup class="reference" id="cite_ref-34"><a href="#cite_note-34">[34]</a></sup>, всего было собрано 12 тысяч рублей<sup class="reference" id="cite_ref-biogr_4-11"><a href="#cite_note-biogr-4">[4]</a></sup>. Памятник был заложен к пятилетию подвига брига «Меркурий» — в 1834-м и открыт в 1839 году на Малом Бульваре (позже <i>Мичманский</i>, сейчас <i>Матросский</i>) Севастополя. Он был выполнен по проекту академика архитектуры <a href="/wiki/%D0%91%D1%80%D1%8E%D0%BB%D0%BB%D0%BE%D0%B2,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%9F%D0%B0%D0%B2%D0%BB%D0%BE%D0%B2%D0%B8%D1%87" title="Брюллов, Александр Павлович">А. П. Брюллова</a>. Строительные работы выполнил мастер О. Г. Нюман. Строительство велось на средства, собранные моряками Черноморского и <a class="mw-redirect" href="/wiki/%D0%91%D0%B0%D0%BB%D1%82%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%84%D0%BB%D0%BE%D1%82_%D0%92%D0%9C%D0%A4_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B8" title="Балтийский флот ВМФ России">Балтийского</a> флотов.
</p><p>Памятник сооружен в стиле <a href="/wiki/%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B8%D1%86%D0%B8%D0%B7%D0%BC" title="Классицизм">классицизма</a>. На усеченной пирамиде из крымбальского известняка установлена античная <a href="/wiki/%D0%A2%D1%80%D0%B8%D1%80%D0%B5%D0%BC%D0%B0" title="Трирема">трирема</a>. На подиуме в небольших нишах помещены <a href="/wiki/%D0%A0%D0%B5%D0%BB%D1%8C%D0%B5%D1%84_(%D1%81%D0%BA%D1%83%D0%BB%D1%8C%D0%BF%D1%82%D1%83%D1%80%D0%B0)" title="Рельеф (скульптура)">горельефные</a> изображения А. И. Казарского и античных богов — <a href="/wiki/%D0%9D%D0%B8%D0%BA%D0%B0_(%D0%BC%D0%B8%D1%84%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Ника (мифология)">Ники</a> (богиня победы), <a href="/wiki/%D0%9D%D0%B5%D0%BF%D1%82%D1%83%D0%BD_(%D0%BC%D0%B8%D1%84%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Нептун (мифология)">Нептуна</a> и <a href="/wiki/%D0%9C%D0%B5%D1%80%D0%BA%D1%83%D1%80%D0%B8%D0%B9_(%D0%BC%D0%B8%D1%84%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Меркурий (мифология)">Меркурия</a>. На постаменте — два <a href="/wiki/%D0%9C%D0%B0%D1%81%D0%BA%D0%B0%D1%80%D0%BE%D0%BD" title="Маскарон">маскарона</a> и военные атрибуты, символизирующие славу и доблесть. В одном из документов <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D0%B3%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9_%D0%B0%D1%80%D1%85%D0%B8%D0%B2_%D0%B4%D1%80%D0%B5%D0%B2%D0%BD%D0%B8%D1%85_%D0%B0%D0%BA%D1%82%D0%BE%D0%B2" title="Российский государственный архив древних актов">Центрального государственного архива</a> в <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%B2%D0%B0" title="Москва">Москве</a> указано, что надпись «Казарскому. Потомству в пример» велел написать сам <a href="/wiki/%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B9_I" title="Николай I">Николай I</a><sup class="reference" id="cite_ref-35"><a href="#cite_note-35">[35]</a></sup>.
</p><p>Памятник Александру Казарскому стал первым <a href="/wiki/%D0%9F%D0%B0%D0%BC%D1%8F%D1%82%D0%BD%D0%B8%D0%BA%D0%B8_%D0%A1%D0%B5%D0%B2%D0%B0%D1%81%D1%82%D0%BE%D0%BF%D0%BE%D0%BB%D1%8F" title="Памятники Севастополя">памятником, воздвигнутым в Севастополе</a><sup class="reference" id="cite_ref-36"><a href="#cite_note-36">[36]</a></sup>.
</p>
<h3><span id=".D0.9C.D0.B5.D0.BC.D0.BE.D1.80.D0.B8.D0.B0.D0.BB_.D0.B2_.D0.94.D1.83.D0.B1.D1.80.D0.BE.D0.B2.D0.BD.D0.BE"></span><span class="mw-headline" id="Мемориал_в_Дубровно">Мемориал в Дубровно</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B0%D0%B7%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=9" title="Редактировать раздел «Мемориал в Дубровно»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B0%D0%B7%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=9" title="Редактировать код раздела «Мемориал в Дубровно»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>На родине А. С. Казарского в центральной части города <a href="/wiki/%D0%94%D1%83%D0%B1%D1%80%D0%BE%D0%B2%D0%BD%D0%BE" title="Дубровно">Дубровно</a> установлен камень с мемориальной доской с надписью на <a href="/wiki/%D0%91%D0%B5%D0%BB%D0%BE%D1%80%D1%83%D1%81%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Белорусский язык">белорусском языке</a> : «<i>У гонар славутага земляка, героя руска-турэцкай вайны 1828—1829 гадоў, капітана першага рангу камандзіра брыга «Меркурый» Казарскага Аляксандра Іванавіча</i>»<sup class="reference" id="cite_ref-37"><a href="#cite_note-37">[37]</a></sup>.
</p>
<h3><span id=".D0.9E.D0.B1.D1.8A.D0.B5.D0.BA.D1.82.D1.8B.2C_.D0.BD.D0.B0.D0.B7.D0.B2.D0.B0.D0.BD.D0.BD.D1.8B.D0.B5_.D0.B2_.D1.87.D0.B5.D1.81.D1.82.D1.8C_.D0.9A.D0.B0.D0.B7.D0.B0.D1.80.D1.81.D0.BA.D0.BE.D0.B3.D0.BE"></span><span class="mw-headline" id="Объекты,_названные_в_честь_Казарского">Объекты, названные в честь Казарского</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B0%D0%B7%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=10" title="Редактировать раздел «Объекты, названные в честь Казарского»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B0%D0%B7%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=10" title="Редактировать код раздела «Объекты, названные в честь Казарского»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Имя Казарского получили <a href="/wiki/%D0%9A%D0%B0%D0%B7%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9_(%D0%B1%D1%80%D0%B8%D0%B3)" title="Казарский (бриг)">бриг</a> <a class="mw-redirect" href="/wiki/%D0%91%D0%B0%D0%BB%D1%82%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%84%D0%BB%D0%BE%D1%82_%D0%92%D0%9C%D0%A4_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B8" title="Балтийский флот ВМФ России">Балтийского флота</a><sup class="reference" id="cite_ref-rpf2_95_38-0"><a href="#cite_note-rpf2_95-38">[38]</a></sup>, серия из шести <a href="/wiki/%D0%9C%D0%B8%D0%BD%D0%BD%D1%8B%D0%B9_%D0%BA%D1%80%D0%B5%D0%B9%D1%81%D0%B5%D1%80" title="Минный крейсер">минных крейсеров</a> Черноморского флота, а также <a href="/wiki/%D0%9A%D0%B0%D0%B7%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9_(%D0%BC%D0%B8%D0%BD%D0%BD%D1%8B%D0%B9_%D0%BA%D1%80%D0%B5%D0%B9%D1%81%D0%B5%D1%80)" title="Казарский (минный крейсер)">первый из кораблей этого типа</a><sup class="reference" id="cite_ref-kruiser_39-0"><a href="#cite_note-kruiser-39">[39]</a></sup>. В советское время имя «Казарский» носил морской <a href="/wiki/%D0%A2%D1%80%D0%B0%D0%BB%D1%8C%D1%89%D0%B8%D0%BA" title="Тральщик">тральщик</a><sup class="reference" id="cite_ref-biogr_4-12"><a href="#cite_note-biogr-4">[4]</a></sup>.
</p><p>8 июня 1954 года в честь Казарского были названы улица в <a href="/wiki/%D0%9D%D0%B0%D1%85%D0%B8%D0%BC%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D1%80%D0%B0%D0%B9%D0%BE%D0%BD" title="Нахимовский район">Нахимовском районе</a> Севастополя, ранее носившая имя Четвёртой Параллельной<sup class="reference" id="cite_ref-street_40-0"><a href="#cite_note-street-40">[40]</a></sup> и улица в родном городе Дубровно, а в <a class="mw-redirect" href="/wiki/%D0%9B%D0%B5%D0%BD%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%80%D0%B0%D0%B9%D0%BE%D0%BD_(%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B5%D0%B2)" title="Ленинский район (Николаев)">Ленинском районе Николаева</a> (микрорайон Водопой) именем Казарского назван переулок<sup class="reference" id="cite_ref-41"><a href="#cite_note-41">[41]</a></sup>
</p>
