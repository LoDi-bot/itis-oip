<p><b>Синдром поликистозных яичников</b> (<b>СПКЯ</b>, известный также как <b>синдром Штейна—Левенталя</b>) — <a href="/wiki/%D0%AD%D0%BD%D0%B4%D0%BE%D0%BA%D1%80%D0%B8%D0%BD%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F" title="Эндокринология">полиэндокринный</a> <a href="/wiki/%D0%A1%D0%B8%D0%BD%D0%B4%D1%80%D0%BE%D0%BC" title="Синдром">синдром</a>, сопровождающийся нарушениями функции <a href="/wiki/%D0%AF%D0%B8%D1%87%D0%BD%D0%B8%D0%BA%D0%B8" title="Яичники">яичников</a> (отсутствием или нерегулярностью <a href="/wiki/%D0%9E%D0%B2%D1%83%D0%BB%D1%8F%D1%86%D0%B8%D1%8F" title="Овуляция">овуляции</a>, повышенной секрецией <a href="/wiki/%D0%90%D0%BD%D0%B4%D1%80%D0%BE%D0%B3%D0%B5%D0%BD%D1%8B" title="Андрогены">андрогенов</a> и <a href="/wiki/%D0%AD%D1%81%D1%82%D1%80%D0%BE%D0%B3%D0%B5%D0%BD%D1%8B" title="Эстрогены">эстрогенов</a>), <a href="/wiki/%D0%9F%D0%BE%D0%B4%D0%B6%D0%B5%D0%BB%D1%83%D0%B4%D0%BE%D1%87%D0%BD%D0%B0%D1%8F_%D0%B6%D0%B5%D0%BB%D0%B5%D0%B7%D0%B0" title="Поджелудочная железа">поджелудочной железы</a> (гиперсекреция <a href="/wiki/%D0%98%D0%BD%D1%81%D1%83%D0%BB%D0%B8%D0%BD" title="Инсулин">инсулина</a>), <a href="/wiki/%D0%9D%D0%B0%D0%B4%D0%BF%D0%BE%D1%87%D0%B5%D1%87%D0%BD%D0%B8%D0%BA%D0%B8" title="Надпочечники">коры надпочечников</a> (гиперсекреция надпочечниковых андрогенов), <a href="/wiki/%D0%93%D0%B8%D0%BF%D0%BE%D1%82%D0%B0%D0%BB%D0%B0%D0%BC%D1%83%D1%81" title="Гипоталамус">гипоталамуса</a> и <a href="/wiki/%D0%93%D0%B8%D0%BF%D0%BE%D1%84%D0%B8%D0%B7" title="Гипофиз">гипофиза</a>.
</p>

<h2><span id=".D0.9D.D0.BE.D0.BC.D0.B5.D0.BD.D0.BA.D0.BB.D0.B0.D1.82.D1.83.D1.80.D0.B0"></span><span class="mw-headline" id="Номенклатура">Номенклатура</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B8%D0%BD%D0%B4%D1%80%D0%BE%D0%BC_%D0%BF%D0%BE%D0%BB%D0%B8%D0%BA%D0%B8%D1%81%D1%82%D0%BE%D0%B7%D0%BD%D1%8B%D1%85_%D1%8F%D0%B8%D1%87%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Номенклатура»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B8%D0%BD%D0%B4%D1%80%D0%BE%D0%BC_%D0%BF%D0%BE%D0%BB%D0%B8%D0%BA%D0%B8%D1%81%D1%82%D0%BE%D0%B7%D0%BD%D1%8B%D1%85_%D1%8F%D0%B8%D1%87%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2&amp;action=edit&amp;section=1" title="Редактировать код раздела «Номенклатура»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Другие названия для этого синдрома следующие:
</p>
<ul><li>болезнь поликистозных яичников (некорректно, поскольку это состояние характеризуется не как <a href="/wiki/%D0%91%D0%BE%D0%BB%D0%B5%D0%B7%D0%BD%D1%8C" title="Болезнь">болезнь</a>, отдельная нозологическая форма, а как клинический <a href="/wiki/%D0%A1%D0%B8%D0%BD%D0%B4%D1%80%D0%BE%D0%BC" title="Синдром">синдром</a>, причины которого могут быть различны);</li>
<li>функциональная яичниковая <a class="mw-redirect" href="/wiki/%D0%90%D0%BD%D0%B4%D1%80%D0%BE%D0%B3%D0%B5%D0%BD" title="Андроген">гиперандрогения</a> (или функциональный яичниковый гиперандрогенизм);</li>
<li>гиперандрогенная хроническая <a href="/wiki/%D0%9E%D0%B2%D1%83%D0%BB%D1%8F%D1%86%D0%B8%D1%8F" title="Овуляция">ановуляция</a>;</li>
<li>яичниковый дисметаболический синдром;</li>
<li>синдром поликистоза яичников;</li>
<li>поликистоз яичников.</li></ul>
<h2><span id=".D0.9E.D0.BF.D1.80.D0.B5.D0.B4.D0.B5.D0.BB.D0.B5.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="Определения">Определения</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B8%D0%BD%D0%B4%D1%80%D0%BE%D0%BC_%D0%BF%D0%BE%D0%BB%D0%B8%D0%BA%D0%B8%D1%81%D1%82%D0%BE%D0%B7%D0%BD%D1%8B%D1%85_%D1%8F%D0%B8%D1%87%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Определения»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B8%D0%BD%D0%B4%D1%80%D0%BE%D0%BC_%D0%BF%D0%BE%D0%BB%D0%B8%D0%BA%D0%B8%D1%81%D1%82%D0%BE%D0%B7%D0%BD%D1%8B%D1%85_%D1%8F%D0%B8%D1%87%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2&amp;action=edit&amp;section=2" title="Редактировать код раздела «Определения»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Существуют два наиболее употребительных в клинической практике определения синдрома поликистозных яичников.
</p><p>Первое определение выработано в <a href="/wiki/1990_%D0%B3%D0%BE%D0%B4" title="1990 год">1990 году</a> консенсусом экспертной комиссии, сформированной <a class="new" href="/w/index.php?title=%D0%90%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%9D%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%B8%D0%BD%D1%81%D1%82%D0%B8%D1%82%D1%83%D1%82_%D0%B7%D0%B4%D1%80%D0%B0%D0%B2%D0%BE%D0%BE%D1%85%D1%80%D0%B0%D0%BD%D0%B5%D0%BD%D0%B8%D1%8F&amp;action=edit&amp;redlink=1" title="Американский Национальный институт здравоохранения (страница отсутствует)">Американским Национальным институтом здравоохранения</a> (NIH). Согласно этому определению, пациентке должен быть поставлен диагноз поликистоза яичников, если у неё одновременно присутствуют:
</p>
<ol><li>Симптомы избыточной активности или избыточной секреции андрогенов (клинические и/или биохимические);</li>
<li><a class="mw-redirect" href="/wiki/%D0%9E%D0%BB%D0%B8%D0%B3%D0%BE%D0%BE%D0%B2%D1%83%D0%BB%D1%8F%D1%86%D0%B8%D1%8F" title="Олигоовуляция">Олигоовуляция</a> или <a class="mw-redirect" href="/wiki/%D0%90%D0%BD%D0%BE%D0%B2%D1%83%D0%BB%D1%8F%D1%86%D0%B8%D1%8F" title="Ановуляция">ановуляция</a></li></ol>
<p>и если при этом исключены другие причины, способные вызывать поликистоз яичников.
</p><p>Второе определение сформулировано в <a href="/wiki/2003_%D0%B3%D0%BE%D0%B4" title="2003 год">2003 году</a> консенсусом европейских экспертов, сформированным в <a href="/wiki/%D0%A0%D0%BE%D1%82%D1%82%D0%B5%D1%80%D0%B4%D0%B0%D0%BC" title="Роттердам">Роттердаме</a><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>. По этому определению, диагноз ставится, если у пациентки присутствуют одновременно любые два из трёх следующих признаков:
</p>
<ol><li>Симптомы избыточной активности или избыточной секреции андрогенов (клинические или биохимические);</li>
<li>Олигоовуляция или ановуляция;</li>
<li>Поликистозные яичники при ультразвуковом исследовании органов малого таза</li></ol>
<p>и если при этом исключены другие причины, способные вызывать поликистоз яичников.
</p><p>Роттердамское определение значительно шире и включает в группу страдающих этим синдромом значительно больше пациенток. В частности, оно включает в себя и пациенток без клинических или биохимических признаков избытка андрогенов (поскольку обязательными являются любые два из трёх признаков, а не все три), в то время как в американском определении избыточная секреция или избыточная активность андрогенов является обязательным условием для диагноза поликистоза яичников. Критики Роттердамского определения утверждают, что находки, полученные при изучении пациенток с избытком андрогенов, не обязательно могут быть экстраполированы на пациенток без симптомов избытка андрогенов<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>.
</p>
<h2><span id=".D0.A1.D0.B8.D0.BC.D0.BF.D1.82.D0.BE.D0.BC.D1.8B"></span><span class="mw-headline" id="Симптомы">Симптомы</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B8%D0%BD%D0%B4%D1%80%D0%BE%D0%BC_%D0%BF%D0%BE%D0%BB%D0%B8%D0%BA%D0%B8%D1%81%D1%82%D0%BE%D0%B7%D0%BD%D1%8B%D1%85_%D1%8F%D0%B8%D1%87%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Симптомы»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B8%D0%BD%D0%B4%D1%80%D0%BE%D0%BC_%D0%BF%D0%BE%D0%BB%D0%B8%D0%BA%D0%B8%D1%81%D1%82%D0%BE%D0%B7%D0%BD%D1%8B%D1%85_%D1%8F%D0%B8%D1%87%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2&amp;action=edit&amp;section=3" title="Редактировать код раздела «Симптомы»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Общие симптомы поликистоза яичников следующие:
</p>
<ul><li><a class="new" href="/w/index.php?title=%D0%9E%D0%BB%D0%B8%D0%B3%D0%BE%D0%BC%D0%B5%D0%BD%D0%BE%D1%80%D0%B5%D1%8F&amp;action=edit&amp;redlink=1" title="Олигоменорея (страница отсутствует)">Олигоменорея</a>, <a href="/wiki/%D0%90%D0%BC%D0%B5%D0%BD%D0%BE%D1%80%D0%B5%D1%8F" title="Аменорея">аменорея</a> — нерегулярные, редкие <a href="/wiki/%D0%9C%D0%B5%D0%BD%D1%81%D1%82%D1%80%D1%83%D0%B0%D1%86%D0%B8%D1%8F" title="Менструация">менструации</a> или полное отсутствие менструаций; те менструации, которые всё же происходят, могут быть патологически скудными или, напротив, чрезмерно обильными, а также болезненными;</li>
<li><a href="/wiki/%D0%91%D0%B5%D1%81%D0%BF%D0%BB%D0%BE%D0%B4%D0%B8%D0%B5" title="Бесплодие">Бесплодие</a>, обычно являющееся результатом хронической <a class="mw-redirect" href="/wiki/%D0%90%D0%BD%D0%BE%D0%B2%D1%83%D0%BB%D1%8F%D1%86%D0%B8%D1%8F" title="Ановуляция">ановуляции</a> или <a class="mw-redirect" href="/wiki/%D0%9E%D0%BB%D0%B8%D0%B3%D0%BE%D0%BE%D0%B2%D1%83%D0%BB%D1%8F%D1%86%D0%B8%D1%8F" title="Олигоовуляция">олигоовуляции</a> (полного отсутствия овуляций либо овуляция происходит не в каждом цикле);</li>
<li>Повышенные уровни в крови <a class="mw-redirect" href="/wiki/%D0%90%D0%BD%D0%B4%D1%80%D0%BE%D0%B3%D0%B5%D0%BD" title="Андроген">андрогенов</a> (мужских гормонов), в особенности свободных фракций <a href="/wiki/%D0%A2%D0%B5%D1%81%D1%82%D0%BE%D1%81%D1%82%D0%B5%D1%80%D0%BE%D0%BD" title="Тестостерон">тестостерона</a>, <a href="/wiki/%D0%90%D0%BD%D0%B4%D1%80%D0%BE%D1%81%D1%82%D0%B5%D0%BD%D0%B4%D0%B8%D0%BE%D0%BD" title="Андростендион">андростендиона</a> и <a href="/wiki/%D0%94%D0%B5%D0%B3%D0%B8%D0%B4%D1%80%D0%BE%D1%8D%D0%BF%D0%B8%D0%B0%D0%BD%D0%B4%D1%80%D0%BE%D1%81%D1%82%D0%B5%D1%80%D0%BE%D0%BD" title="Дегидроэпиандростерон">дегидроэпиандростерона сульфата</a>, что вызывает <a href="/wiki/%D0%93%D0%B8%D1%80%D1%81%D1%83%D1%82%D0%B8%D0%B7%D0%BC" title="Гирсутизм">гирсутизм</a> и иногда <a href="/wiki/%D0%9C%D0%B0%D1%81%D0%BA%D1%83%D0%BB%D0%B8%D0%BD%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F" title="Маскулинизация">маскулинизацию</a><sup class="reference" id="cite_ref-AMN_6-0"><a href="#cite_note-AMN-6">[6]</a></sup>;</li>
<li><a class="new" href="/w/index.php?title=%D0%A6%D0%B5%D0%BD%D1%82%D1%80%D0%B0%D0%BB%D1%8C%D0%BD%D0%BE%D0%B5_%D0%BE%D0%B6%D0%B8%D1%80%D0%B5%D0%BD%D0%B8%D0%B5&amp;action=edit&amp;redlink=1" title="Центральное ожирение (страница отсутствует)">Центральное ожирение</a> — «паукообразное» или «в форме яблока» <a href="/wiki/%D0%9E%D0%B6%D0%B8%D1%80%D0%B5%D0%BD%D0%B8%D0%B5" title="Ожирение">ожирение</a> по мужскому типу, при котором основная масса жировой ткани концентрируется внизу живота и в брюшной полости;</li>
<li>Андрогенная <a href="/wiki/%D0%90%D0%BB%D0%BE%D0%BF%D0%B5%D1%86%D0%B8%D1%8F" title="Алопеция">алопеция</a> (существенное облысение или выпадение волос по мужскому типу с <a href="/wiki/%D0%9C%D1%8B%D1%81_%D0%B2%D0%B4%D0%BE%D0%B2%D1%8B" title="Мыс вдовы">залысинами по бокам лба</a>, на макушке, происходящее из-за нарушения гормонального баланса);</li>
<li><a href="/wiki/%D0%90%D0%BA%D0%BD%D0%B5" title="Акне">Угревая сыпь</a>, жирная кожа, <a href="/wiki/%D0%A1%D0%B5%D0%B1%D0%BE%D1%80%D0%B5%D0%B9%D0%BD%D1%8B%D0%B9_%D0%B4%D0%B5%D1%80%D0%BC%D0%B0%D1%82%D0%B8%D1%82" title="Себорейный дерматит">себорея</a>;</li>
<li><a class="mw-redirect" href="/wiki/%D0%90%D0%BA%D0%B0%D0%BD%D1%82%D0%BE%D0%B7" title="Акантоз">Акантоз</a> (тёмные пигментные пятна на коже, от светло-бежевого до тёмно-коричневого или чёрного);</li>
<li><a class="new" href="/w/index.php?title=%D0%90%D0%BA%D1%80%D0%BE%D1%85%D0%BE%D1%80%D0%B4%D0%BE%D0%BD&amp;action=edit&amp;redlink=1" title="Акрохордон (страница отсутствует)">Акрохордоны</a> (кожные складки) — мелкие складки и морщинки кожи;</li>
<li>Появление <a href="/wiki/%D0%A1%D1%82%D1%80%D0%B8%D0%B8" title="Стрии">стрий</a> (полос растяжения) на коже живота, бёдер, ягодиц, в результате быстрой прибавки массы тела на фоне гормонального дисбаланса;</li>
<li>Длительные периоды симптомов, напоминающих симптомы <a href="/wiki/%D0%9F%D1%80%D0%B5%D0%B4%D0%BC%D0%B5%D0%BD%D1%81%D1%82%D1%80%D1%83%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D1%81%D0%B8%D0%BD%D0%B4%D1%80%D0%BE%D0%BC" title="Предменструальный синдром">предменструального синдрома</a> (<a href="/wiki/%D0%9E%D1%82%D1%91%D0%BA" title="Отёк">отёки</a>, колебания настроения, <a href="/wiki/%D0%91%D0%BE%D0%BB%D1%8C" title="Боль">боль</a> внизу живота, в пояснице, боли или набухание молочных желёз);</li>
<li><a class="mw-redirect" href="/wiki/%D0%9D%D0%BE%D1%87%D0%BD%D1%8B%D0%B5_%D0%B0%D0%BF%D0%BD%D0%BE%D1%8D" title="Ночные апноэ">Ночные апноэ</a> — остановки дыхания во сне, приводящие к частым ночным пробуждениям больной;</li>
<li><a class="mw-redirect" href="/wiki/%D0%91%D0%BE%D0%BB%D1%8C%D1%88%D0%B0%D1%8F_%D0%B4%D0%B5%D0%BF%D1%80%D0%B5%D1%81%D1%81%D0%B8%D1%8F" title="Большая депрессия">Депрессия</a>, дисфория (раздражительность, нервозность, <a href="/wiki/%D0%90%D0%B3%D1%80%D0%B5%D1%81%D1%81%D0%B8%D0%B2%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Агрессивность">агрессивность</a>), нередко сонливость, вялость, <a href="/wiki/%D0%90%D0%BF%D0%B0%D1%82%D0%B8%D1%8F" title="Апатия">апатия</a>, жалобы на «туман в голове».</li>
<li>Множественные кисты яичников. Сонографически они могут выглядеть как «жемчужное ожерелье», скопление белёсых пузырьков или «косточки фрукта», рассеянные по всей ткани яичников;</li>
<li>Увеличение размеров яичников в 1,5—3 раза за счёт возникновения множества мелких кист;</li>
<li>Утолщённая, гладкая, жемчужно-белая наружная поверхность (капсула) яичников;</li>
<li>Утолщённый, гиперплазированный <a href="/wiki/%D0%AD%D0%BD%D0%B4%D0%BE%D0%BC%D0%B5%D1%82%D1%80%D0%B8%D0%B9" title="Эндометрий">эндометрий</a> матки — результат длительного избытка эстрогенов, не сбалансированного адекватными <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%B3%D0%B5%D1%81%D1%82%D0%B5%D1%80%D0%BE%D0%BD" title="Прогестерон">прогестероновыми</a> влияниями;</li>
<li>Хроническая <a href="/wiki/%D0%91%D0%BE%D0%BB%D1%8C" title="Боль">боль</a> внизу живота или в пояснице, в тазовой области, вероятно вследствие сдавления органов <a class="mw-redirect" href="/wiki/%D0%9C%D0%B0%D0%BB%D1%8B%D0%B9_%D1%82%D0%B0%D0%B7" title="Малый таз">малого таза</a> увеличенными яичниками или вследствие гиперсекреции <a href="/wiki/%D0%9F%D1%80%D0%BE%D1%81%D1%82%D0%B0%D0%B3%D0%BB%D0%B0%D0%BD%D0%B4%D0%B8%D0%BD%D1%8B" title="Простагландины">простагландинов</a> в яичниках и эндометрии; точная причина хронической боли при поликистозе яичников неизвестна;</li>
<li>Повышенный уровень <a href="/wiki/%D0%9B%D1%8E%D1%82%D0%B5%D0%B8%D0%BD%D0%B8%D0%B7%D0%B8%D1%80%D1%83%D1%8E%D1%89%D0%B8%D0%B9_%D0%B3%D0%BE%D1%80%D0%BC%D0%BE%D0%BD" title="Лютеинизирующий гормон">ЛГ</a> или повышенное соотношение ЛГ/<a href="/wiki/%D0%A4%D0%BE%D0%BB%D0%BB%D0%B8%D0%BA%D1%83%D0%BB%D0%BE%D1%81%D1%82%D0%B8%D0%BC%D1%83%D0%BB%D0%B8%D1%80%D1%83%D1%8E%D1%89%D0%B8%D0%B9_%D0%B3%D0%BE%D1%80%D0%BC%D0%BE%D0%BD" title="Фолликулостимулирующий гормон">ФСГ</a>: при измерении на 3-й день менструального цикла соотношение ЛГ/ФСГ оказывается больше 1:1;</li>
<li>Пониженный уровень <a class="new" href="/w/index.php?title=%D0%A1%D0%B5%D0%BA%D1%81-%D1%81%D1%82%D0%B5%D1%80%D0%BE%D0%B8%D0%B4-%D1%81%D0%B2%D1%8F%D0%B7%D1%8B%D0%B2%D0%B0%D1%8E%D1%89%D0%B8%D0%B9_%D0%B3%D0%BB%D0%BE%D0%B1%D1%83%D0%BB%D0%B8%D0%BD&amp;action=edit&amp;redlink=1" title="Секс-стероид-связывающий глобулин (страница отсутствует)">глобулина, связывающего половые стероиды</a>;</li>
<li><a class="new" href="/w/index.php?title=%D0%93%D0%B8%D0%BF%D0%B5%D1%80%D0%B8%D0%BD%D1%81%D1%83%D0%BB%D0%B8%D0%BD%D0%B5%D0%BC%D0%B8%D1%8F&amp;action=edit&amp;redlink=1" title="Гиперинсулинемия (страница отсутствует)">Гиперинсулинемия</a> (повышенный уровень <a href="/wiki/%D0%98%D0%BD%D1%81%D1%83%D0%BB%D0%B8%D0%BD" title="Инсулин">инсулина</a> в крови), <a href="/wiki/%D0%9D%D0%B0%D1%80%D1%83%D1%88%D0%B5%D0%BD%D0%B8%D0%B5_%D1%82%D0%BE%D0%BB%D0%B5%D1%80%D0%B0%D0%BD%D1%82%D0%BD%D0%BE%D1%81%D1%82%D0%B8_%D0%BA_%D0%B3%D0%BB%D1%8E%D0%BA%D0%BE%D0%B7%D0%B5" title="Нарушение толерантности к глюкозе">нарушение толерантности к глюкозе</a>, признаки <a href="/wiki/%D0%98%D0%BD%D1%81%D1%83%D0%BB%D0%B8%D0%BD%D0%BE%D1%80%D0%B5%D0%B7%D0%B8%D1%81%D1%82%D0%B5%D0%BD%D1%82%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Инсулинорезистентность">инсулинорезистентности</a> тканей при тестировании по <a href="/wiki/%D0%93%D0%BB%D1%8E%D0%BA%D0%BE%D0%B7%D0%BE%D1%82%D0%BE%D0%BB%D0%B5%D1%80%D0%B0%D0%BD%D1%82%D0%BD%D1%8B%D0%B9_%D1%82%D0%B5%D1%81%D1%82" title="Глюкозотолерантный тест">методу сахарной кривой</a><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>.</li></ul>
<h2><span id=".D0.A0.D0.B8.D1.81.D0.BA.D0.B8_.D0.B4.D0.BB.D1.8F_.D0.B7.D0.B4.D0.BE.D1.80.D0.BE.D0.B2.D1.8C.D1.8F_.D0.B8_.D0.BE.D1.81.D0.BB.D0.BE.D0.B6.D0.BD.D0.B5.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="Риски_для_здоровья_и_осложнения">Риски для здоровья и осложнения</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B8%D0%BD%D0%B4%D1%80%D0%BE%D0%BC_%D0%BF%D0%BE%D0%BB%D0%B8%D0%BA%D0%B8%D1%81%D1%82%D0%BE%D0%B7%D0%BD%D1%8B%D1%85_%D1%8F%D0%B8%D1%87%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Риски для здоровья и осложнения»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B8%D0%BD%D0%B4%D1%80%D0%BE%D0%BC_%D0%BF%D0%BE%D0%BB%D0%B8%D0%BA%D0%B8%D1%81%D1%82%D0%BE%D0%B7%D0%BD%D1%8B%D1%85_%D1%8F%D0%B8%D1%87%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2&amp;action=edit&amp;section=4" title="Редактировать код раздела «Риски для здоровья и осложнения»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Женщины, страдающие поликистозом яичников, подвержены повышенному риску развития следующих осложнений:
</p>
<ul><li><a href="/wiki/%D0%93%D0%B8%D0%BF%D0%B5%D1%80%D0%BF%D0%BB%D0%B0%D0%B7%D0%B8%D1%8F" title="Гиперплазия">Гиперплазия</a> <a href="/wiki/%D0%AD%D0%BD%D0%B4%D0%BE%D0%BC%D0%B5%D1%82%D1%80%D0%B8%D0%B9" title="Эндометрий">эндометрия</a> и <a class="new" href="/w/index.php?title=%D0%A0%D0%B0%D0%BA_%D1%8D%D0%BD%D0%B4%D0%BE%D0%BC%D0%B5%D1%82%D1%80%D0%B8%D1%8F&amp;action=edit&amp;redlink=1" title="Рак эндометрия (страница отсутствует)">рак эндометрия</a> вследствие отсутствия или нерегулярности менструаций и «накопления» неслущивающегося эндометрия, а также вследствие отсутствия или недостаточности <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%B3%D0%B5%D1%81%D1%82%D0%B5%D1%80%D0%BE%D0%BD" title="Прогестерон">прогестероновых</a> влияний, приводящего к длительной несбалансированной прогестероном гиперстимуляции клеток эндометрия повышенным уровнем эстрогенов;</li>
<li><a href="/wiki/%D0%A0%D0%B0%D0%BA_%D0%BC%D0%BE%D0%BB%D0%BE%D1%87%D0%BD%D0%BE%D0%B9_%D0%B6%D0%B5%D0%BB%D0%B5%D0%B7%D1%8B" title="Рак молочной железы">Рак молочной железы</a>;</li>
<li><a href="/wiki/%D0%9E%D0%B6%D0%B8%D1%80%D0%B5%D0%BD%D0%B8%D0%B5" title="Ожирение">Ожирение</a>;</li>
<li><a href="/wiki/%D0%98%D0%BD%D1%81%D1%83%D0%BB%D0%B8%D0%BD%D0%BE%D1%80%D0%B5%D0%B7%D0%B8%D1%81%D1%82%D0%B5%D0%BD%D1%82%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Инсулинорезистентность">Инсулинорезистентность</a> и <a href="/wiki/%D0%A1%D0%B0%D1%85%D0%B0%D1%80%D0%BD%D1%8B%D0%B9_%D0%B4%D0%B8%D0%B0%D0%B1%D0%B5%D1%82_2-%D0%B3%D0%BE_%D1%82%D0%B8%D0%BF%D0%B0" title="Сахарный диабет 2-го типа">сахарный диабет 2-го типа</a>;</li>
<li><a href="/wiki/%D0%90%D1%80%D1%82%D0%B5%D1%80%D0%B8%D0%B0%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D0%B3%D0%B8%D0%BF%D0%B5%D1%80%D1%82%D0%B5%D0%BD%D0%B7%D0%B8%D1%8F" title="Артериальная гипертензия">повышенное артериальное давление</a>;</li>
<li><a href="/wiki/%D0%A2%D1%80%D0%BE%D0%BC%D0%B1%D0%BE%D0%B7" title="Тромбоз">Тромбозы</a>, тромбоэмболии, тромбофлебиты вследствие повышения свёртываемости крови;</li>
<li><a class="mw-redirect" href="/wiki/%D0%94%D0%B8%D1%81%D0%BB%D0%B8%D0%BF%D0%B8%D0%B4%D0%B5%D0%BC%D0%B8%D1%8F" title="Дислипидемия">Дислипидемия</a> (нарушения обмена <a href="/wiki/%D0%A5%D0%BE%D0%BB%D0%B5%D1%81%D1%82%D0%B5%D1%80%D0%B8%D0%BD" title="Холестерин">холестерина</a> и <a class="mw-redirect" href="/wiki/%D0%A2%D1%80%D0%B8%D0%B3%D0%BB%D0%B8%D1%86%D0%B5%D1%80%D0%B8%D0%B4%D1%8B" title="Триглицериды">триглицеридов</a> с возможным развитием <a href="/wiki/%D0%90%D1%82%D0%B5%D1%80%D0%BE%D1%81%D0%BA%D0%BB%D0%B5%D1%80%D0%BE%D0%B7" title="Атеросклероз">атеросклероза сосудов</a>);</li>
<li><a href="/wiki/%D0%A1%D0%B5%D1%80%D0%B4%D0%B5%D1%87%D0%BD%D0%BE-%D1%81%D0%BE%D1%81%D1%83%D0%B4%D0%B8%D1%81%D1%82%D1%8B%D0%B5_%D0%B7%D0%B0%D0%B1%D0%BE%D0%BB%D0%B5%D0%B2%D0%B0%D0%BD%D0%B8%D1%8F" title="Сердечно-сосудистые заболевания">Сердечно-сосудистые заболевания</a>, <a href="/wiki/%D0%98%D0%BD%D1%84%D0%B0%D1%80%D0%BA%D1%82_%D0%BC%D0%B8%D0%BE%D0%BA%D0%B0%D1%80%D0%B4%D0%B0" title="Инфаркт миокарда">инфаркт миокарда</a>, <a href="/wiki/%D0%98%D0%BD%D1%81%D1%83%D0%BB%D1%8C%D1%82" title="Инсульт">инсульт</a>.</li></ul>
<p>Данные ряда исследователей указывают на то, что женщины с поликистозом яичников имеют повышенный риск <a href="/wiki/%D0%92%D1%8B%D0%BA%D0%B8%D0%B4%D1%8B%D1%88" title="Выкидыш">выкидыша</a> или преждевременных <a class="mw-redirect" href="/wiki/%D0%A0%D0%BE%D0%B4%D1%8B" title="Роды">родов</a>, невынашивания <a href="/wiki/%D0%91%D0%B5%D1%80%D0%B5%D0%BC%D0%B5%D0%BD%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Беременность">беременности</a>. Кроме того, многие женщины с этим синдромом не могут зачать или имеют трудности с зачатием вследствие нерегулярности менструального цикла и отсутствия или редко происходящих овуляций. Однако при правильном лечении эти женщины могут нормально зачать, выносить и родить здорового ребёнка.
</p>
<h2><span id=".D0.AD.D0.BF.D0.B8.D0.B4.D0.B5.D0.BC.D0.B8.D0.BE.D0.BB.D0.BE.D0.B3.D0.B8.D1.8F"></span><span class="mw-headline" id="Эпидемиология">Эпидемиология</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B8%D0%BD%D0%B4%D1%80%D0%BE%D0%BC_%D0%BF%D0%BE%D0%BB%D0%B8%D0%BA%D0%B8%D1%81%D1%82%D0%BE%D0%B7%D0%BD%D1%8B%D1%85_%D1%8F%D0%B8%D1%87%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Эпидемиология»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B8%D0%BD%D0%B4%D1%80%D0%BE%D0%BC_%D0%BF%D0%BE%D0%BB%D0%B8%D0%BA%D0%B8%D1%81%D1%82%D0%BE%D0%B7%D0%BD%D1%8B%D1%85_%D1%8F%D0%B8%D1%87%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2&amp;action=edit&amp;section=5" title="Редактировать код раздела «Эпидемиология»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Хотя при ультразвуковом обследовании малого таза яичники, выглядящие как поликистозные, обнаруживаются у до 20 % женщин репродуктивного возраста (в том числе и не предъявляющих никаких жалоб), лишь у 5-10 % женщин репродуктивного возраста обнаруживаются клинические признаки, позволяющие ставить диагноз синдрома поликистозных яичников. Синдром поликистозных яичников одинаково распространён в разных <a href="/wiki/%D0%AD%D1%82%D0%BD%D0%BE%D1%81" title="Этнос">этнических</a> группах. Он является наиболее частым гормональным расстройством у женщин детородного возраста и одной из ведущих причин женского <a href="/wiki/%D0%91%D0%B5%D1%81%D0%BF%D0%BB%D0%BE%D0%B4%D0%B8%D0%B5" title="Бесплодие">бесплодия</a>.
</p>
<h2><span id=".D0.AD.D1.82.D0.B8.D0.BE.D0.BB.D0.BE.D0.B3.D0.B8.D1.8F_.D0.B8_.D0.BF.D0.B0.D1.82.D0.BE.D0.B3.D0.B5.D0.BD.D0.B5.D0.B7"></span><span class="mw-headline" id="Этиология_и_патогенез">Этиология и патогенез</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B8%D0%BD%D0%B4%D1%80%D0%BE%D0%BC_%D0%BF%D0%BE%D0%BB%D0%B8%D0%BA%D0%B8%D1%81%D1%82%D0%BE%D0%B7%D0%BD%D1%8B%D1%85_%D1%8F%D0%B8%D1%87%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Этиология и патогенез»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B8%D0%BD%D0%B4%D1%80%D0%BE%D0%BC_%D0%BF%D0%BE%D0%BB%D0%B8%D0%BA%D0%B8%D1%81%D1%82%D0%BE%D0%B7%D0%BD%D1%8B%D1%85_%D1%8F%D0%B8%D1%87%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2&amp;action=edit&amp;section=6" title="Редактировать код раздела «Этиология и патогенез»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Точные причины развития синдрома неизвестны, однако большое значение придается <a href="/wiki/%D0%9F%D0%B0%D1%82%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F" title="Патология">патологическому</a> снижению <a href="/wiki/%D0%98%D0%BD%D1%81%D1%83%D0%BB%D0%B8%D0%BD" title="Инсулин">инсулин</a>-чувствительности периферических тканей, прежде всего жировой и мышечной ткани (развитию их инсулинорезистентности) при сохранении инсулин-чувствительности ткани <a href="/wiki/%D0%AF%D0%B8%D1%87%D0%BD%D0%B8%D0%BA%D0%B8" title="Яичники">яичников</a>. Возможна также ситуация патологически повышенной инсулин-чувствительности ткани яичников при сохранении нормальной чувствительности к инсулину периферических тканей.
</p><p>В первом случае, как следствие инсулинорезистентности организма, возникает компенсаторная гиперсекреция инсулина, приводящая к развитию гиперинсулинемии. А патологически повышенный уровень инсулина в крови приводит к гиперстимуляции яичников и повышению секреции яичниками <a class="mw-redirect" href="/wiki/%D0%90%D0%BD%D0%B4%D1%80%D0%BE%D0%B3%D0%B5%D0%BD" title="Андроген">андрогенов</a> и <a class="mw-redirect" href="/wiki/%D0%AD%D1%81%D1%82%D1%80%D0%BE%D0%B3%D0%B5%D0%BD" title="Эстроген">эстрогенов</a> и нарушению овуляции, поскольку яичники сохраняют нормальную чувствительность к инсулину.
</p><p>Во втором случае уровень инсулина в крови нормален, однако реакция яичников на стимуляцию нормальным уровнем инсулина патологически повышена, что приводит к тому же самому результату — гиперсекреции яичниками андрогенов и эстрогенов и нарушению овуляции.
</p><p>Патологическая инсулинорезистентность тканей, гиперинсулинемия и гиперсекреция инсулина при поликистозе яичников часто (но не всегда) бывают следствием ожирения или избыточной массы тела. Вместе с тем эти явления и сами могут приводить к ожирению, так как эффектами инсулина являются повышение аппетита, повышение отложения жира и уменьшение его мобилизации.
</p><p>В патогенезе поликистоза яичников придают значение также нарушениям регулирующих гипоталамо-гипофизарных влияний: избыточной секреции <a href="/wiki/%D0%9B%D1%8E%D1%82%D0%B5%D0%B8%D0%BD%D0%B8%D0%B7%D0%B8%D1%80%D1%83%D1%8E%D1%89%D0%B8%D0%B9_%D0%B3%D0%BE%D1%80%D0%BC%D0%BE%D0%BD" title="Лютеинизирующий гормон">ЛГ</a>, аномально повышенному соотношению <a href="/wiki/%D0%9B%D1%8E%D1%82%D0%B5%D0%B8%D0%BD%D0%B8%D0%B7%D0%B8%D1%80%D1%83%D1%8E%D1%89%D0%B8%D0%B9_%D0%B3%D0%BE%D1%80%D0%BC%D0%BE%D0%BD" title="Лютеинизирующий гормон">ЛГ</a>/<a class="mw-redirect" href="/wiki/%D0%A4%D0%A1%D0%93" title="ФСГ">ФСГ</a>, повышенному «<a href="/wiki/%D0%9E%D0%BF%D0%B8%D0%BE%D0%B8%D0%B4%D1%8B" title="Опиоиды">опиоидергическому</a>» <sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> и пониженному <a href="/wiki/%D0%94%D0%BE%D1%84%D0%B0%D0%BC%D0%B8%D0%BD" title="Дофамин">дофаминергическому</a> <sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup><sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup><sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> тонусу в системе гипоталамус-гипофиз. Состояние может отягощаться и труднее поддаваться лечению при наличии сопутствующей <a href="/wiki/%D0%93%D0%B8%D0%BF%D0%B5%D1%80%D0%BF%D1%80%D0%BE%D0%BB%D0%B0%D0%BA%D1%82%D0%B8%D0%BD%D0%B5%D0%BC%D0%B8%D1%8F" title="Гиперпролактинемия">гиперпролактинемии</a>, субклинической или клинически выраженной недостаточности <a href="/wiki/%D0%A9%D0%B8%D1%82%D0%BE%D0%B2%D0%B8%D0%B4%D0%BD%D0%B0%D1%8F_%D0%B6%D0%B5%D0%BB%D0%B5%D0%B7%D0%B0" title="Щитовидная железа">щитовидной железы</a>. Такие сочетания встречаются у этих женщин значительно чаще, чем в общей популяции, что может говорить о полиэндокринной или полиэтиологической природе синдрома Штейна-Левенталя.
</p><p>Некоторые исследователи придают значение повышенному уровню <a href="/wiki/%D0%9F%D1%80%D0%BE%D1%81%D1%82%D0%B0%D0%B3%D0%BB%D0%B0%D0%BD%D0%B4%D0%B8%D0%BD%D1%8B" title="Простагландины">простагландинов</a> и других медиаторов воспаления в текальной ткани яичников и в фолликулярной жидкости у больных поликистозом яичников и считают, что в патогенезе синдрома поликистозных яичников может играть роль возникающее по невыясненным пока причинам «холодное», асептическое воспаление ткани яичников, перенесённые воспалительные заболевания женской половой сферы или аутоиммунные механизмы. Известно, что введение простагландина E1 в яичник или в питающий его сосуд вызывает у лабораторных крыс значительное повышение секреции андрогенов и эстрогенов тека-тканью яичника.
</p>
<h2><span id=".D0.9B.D0.B5.D1.87.D0.B5.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Лечение">Лечение</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B8%D0%BD%D0%B4%D1%80%D0%BE%D0%BC_%D0%BF%D0%BE%D0%BB%D0%B8%D0%BA%D0%B8%D1%81%D1%82%D0%BE%D0%B7%D0%BD%D1%8B%D1%85_%D1%8F%D0%B8%D1%87%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Лечение»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B8%D0%BD%D0%B4%D1%80%D0%BE%D0%BC_%D0%BF%D0%BE%D0%BB%D0%B8%D0%BA%D0%B8%D1%81%D1%82%D0%BE%D0%B7%D0%BD%D1%8B%D1%85_%D1%8F%D0%B8%D1%87%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2&amp;action=edit&amp;section=7" title="Редактировать код раздела «Лечение»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id=".D0.98.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.8F"></span><span class="mw-headline" id="История">История</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B8%D0%BD%D0%B4%D1%80%D0%BE%D0%BC_%D0%BF%D0%BE%D0%BB%D0%B8%D0%BA%D0%B8%D1%81%D1%82%D0%BE%D0%B7%D0%BD%D1%8B%D1%85_%D1%8F%D0%B8%D1%87%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2&amp;veaction=edit&amp;section=8" title="Редактировать раздел «История»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B8%D0%BD%D0%B4%D1%80%D0%BE%D0%BC_%D0%BF%D0%BE%D0%BB%D0%B8%D0%BA%D0%B8%D1%81%D1%82%D0%BE%D0%B7%D0%BD%D1%8B%D1%85_%D1%8F%D0%B8%D1%87%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2&amp;action=edit&amp;section=8" title="Редактировать код раздела «История»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Исторически самые первые попытки лечения синдрома поликистозных яичников состояли в <a href="/wiki/%D0%A5%D0%B8%D1%80%D1%83%D1%80%D0%B3%D0%B8%D1%8F" title="Хирургия">оперативном</a> вмешательстве — декапсуляции <a class="mw-redirect" href="/wiki/%D0%AF%D0%B8%D1%87%D0%BD%D0%B8%D0%BA" title="Яичник">яичников</a> или их частичной резекции с удалением наиболее пораженных кистозом участков ткани, или иссечении овариального ложа (<span lang="en" style="font-style:italic;">ovarian wedge resection</span>), или в аккуратном применении диатермии (нагревания) яичников. В ряде случаев такие операции приводили к успеху и позволяли восстановить <a href="/wiki/%D0%A4%D0%B5%D1%80%D1%82%D0%B8%D0%BB%D1%8C%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Фертильность">фертильность</a> женщины, а также добиться резкого снижения секреции <a class="mw-redirect" href="/wiki/%D0%90%D0%BD%D0%B4%D1%80%D0%BE%D0%B3%D0%B5%D0%BD" title="Андроген">андрогенов</a> яичниками, нормализации <a href="/wiki/%D0%9C%D0%B5%D0%BD%D1%81%D1%82%D1%80%D1%83%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D1%86%D0%B8%D0%BA%D0%BB" title="Менструальный цикл">менструального цикла</a> и др. Однако оперативное вмешательство не всегда возможно, и не всегда приводило к успеху. Кроме того, возможны осложнения, например, формирование спаек. Поэтому специалисты искали <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%BD%D1%81%D0%B5%D1%80%D0%B2%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D0%BE%D0%B5_%D0%BB%D0%B5%D1%87%D0%B5%D0%BD%D0%B8%D0%B5" title="Консервативное лечение">консервативные</a>, нехирургические методы лечения поликистоза яичников.
</p><p>Традиционное консервативное лечение состояло в назначении анти<a class="mw-redirect" href="/wiki/%D0%90%D0%BD%D0%B4%D1%80%D0%BE%D0%B3%D0%B5%D0%BD" title="Андроген">андрогенов</a>, <a class="mw-redirect" href="/wiki/%D0%AD%D1%81%D1%82%D1%80%D0%BE%D0%B3%D0%B5%D0%BD" title="Эстроген">эстрогенов</a>, <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%B3%D0%B5%D1%81%D1%82%D0%B5%D1%80%D0%BE%D0%BD" title="Прогестерон">прогестинов</a> с антиандрогенной активностью или их комбинации (например, в виде противозачаточных таблеток типа Диане-35). Такое <a class="mw-redirect" href="/wiki/%D0%9B%D0%B5%D1%87%D0%B5%D0%BD%D0%B8%D0%B5" title="Лечение">лечение</a> обычно позволяло нормализовать менструальный цикл, но обладало недостаточной эффективностью в отношении кожных проявлений (<a href="/wiki/%D0%90%D0%BA%D0%BD%D0%B5" title="Акне">угри</a>, сальность кожи, андроген-зависимая <a href="/wiki/%D0%90%D0%BB%D0%BE%D0%BF%D0%B5%D1%86%D0%B8%D1%8F" title="Алопеция">алопеция</a>), не позволяло восстановить <a href="/wiki/%D0%9E%D0%B2%D1%83%D0%BB%D1%8F%D1%86%D0%B8%D1%8F" title="Овуляция">овуляцию</a> и фертильность, и не устраняло сами причины поликистоза яичников (нарушения <a href="/wiki/%D0%98%D0%BD%D1%81%D1%83%D0%BB%D0%B8%D0%BD" title="Инсулин">инсулиновой</a> секреции и инсулин-чувствительности тканей, функции оси <a href="/wiki/%D0%93%D0%B8%D0%BF%D0%BE%D1%82%D0%B0%D0%BB%D0%B0%D0%BC%D1%83%D1%81" title="Гипоталамус">гипоталамус</a>-<a href="/wiki/%D0%93%D0%B8%D0%BF%D0%BE%D1%84%D0%B8%D0%B7" title="Гипофиз">гипофиз</a> и др.). Более того, лечение эстрогенами, прогестинами и антиандрогенами часто сопровождалось дальнейшим увеличением веса пациенток, усугублением имеющихся проблем с <a href="/wiki/%D0%A1%D0%B0%D1%85%D0%B0%D1%80%D0%BD%D1%8B%D0%B9_%D0%B4%D0%B8%D0%B0%D0%B1%D0%B5%D1%82" title="Сахарный диабет">углеводным обменом</a> и щитовидной железой, <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%BB%D0%B0%D0%BA%D1%82%D0%B8%D0%BD" title="Пролактин">гиперпролактинемией</a>, <a class="mw-redirect" href="/wiki/%D0%91%D0%BE%D0%BB%D1%8C%D1%88%D0%B0%D1%8F_%D0%B4%D0%B5%D0%BF%D1%80%D0%B5%D1%81%D1%81%D0%B8%D1%8F" title="Большая депрессия">депрессией</a>.
</p><p>Следующая попытка усовершенствовать методы лечения синдрома поликистозных яичников была предпринята с появлением в арсенале <a href="/wiki/%D0%92%D1%80%D0%B0%D1%87" title="Врач">врачей</a> антиэстрогенных препаратов — <a class="mw-redirect" href="/wiki/%D0%9A%D0%BB%D0%BE%D1%81%D1%82%D0%B8%D0%BB%D0%B1%D0%B5%D0%B3%D0%B8%D1%82" title="Клостилбегит">клостилбегита</a> (кломифена цитрата) и <a href="/wiki/%D0%A2%D0%B0%D0%BC%D0%BE%D0%BA%D1%81%D0%B8%D1%84%D0%B5%D0%BD" title="Тамоксифен">тамоксифена</a>. Применение кломифенцитрата или тамоксифена в середине цикла позволяло в примерно 30 % случаев успешно индуцировать овуляцию, восстановить фертильность женщин и добиться стабильного овуляторного менструального цикла без применения экзогенных <a class="mw-redirect" href="/wiki/%D0%93%D0%BE%D1%80%D0%BC%D0%BE%D0%BD" title="Гормон">гормонов</a> (эстрогенов, прогестинов и антиандрогенов). Однако эффективность клостилбегита и тамоксифена в отношении остальных симптомов поликистоза яичников, в частности, проявлений гиперандрогении, оказалась ограниченной. Эффективность комбинированной терапии (эстрогены и прогестины либо антиандрогены по циклу, клостилбегит или тамоксифен в середине цикла) оказалась выше, но тоже недостаточной.
</p><p>Попытки улучшить эффективность лечения женщин с синдромом поликистоза яичников путём коррекции достоверно имеющихся или предполагаемых сопутствующих эндокринных нарушений (коррекция сопутствующей <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%BB%D0%B0%D0%BA%D1%82%D0%B8%D0%BD" title="Пролактин">гиперпролактинемии</a> <a href="/wiki/%D0%91%D1%80%D0%BE%D0%BC%D0%BE%D0%BA%D1%80%D0%B8%D0%BF%D1%82%D0%B8%D0%BD" title="Бромокриптин">бромокриптином</a>, сопутствующей субклинической недостаточности <a href="/wiki/%D0%A9%D0%B8%D1%82%D0%BE%D0%B2%D0%B8%D0%B4%D0%BD%D0%B0%D1%8F_%D0%B6%D0%B5%D0%BB%D0%B5%D0%B7%D0%B0" title="Щитовидная железа">щитовидной железы</a> назначением <a href="/wiki/%D0%A2%D0%B8%D1%80%D0%BE%D0%BA%D1%81%D0%B8%D0%BD" title="Тироксин">тиреоидных</a> гормонов, подавление гиперсекреции андрогенов надпочечниками с помощью назначения небольших доз <a href="/wiki/%D0%94%D0%B5%D0%BA%D1%81%D0%B0%D0%BC%D0%B5%D1%82%D0%B0%D0%B7%D0%BE%D0%BD" title="Дексаметазон">дексаметазона</a>) оказались отчасти успешными, но успех был индивидуален и недостаточно постоянен и предсказуем.
</p><p>Реальные сдвиги в эффективности лечения поликистоза яичников произошли тогда, когда удалось глубже проникнуть в понимание <a href="/wiki/%D0%9F%D0%B0%D1%82%D0%BE%D0%B3%D0%B5%D0%BD%D0%B5%D0%B7" title="Патогенез">патогенеза</a> поликистоза яичников и когда стали придавать основное, первостепенное значение в развитии этого состояния гиперсекреции <a href="/wiki/%D0%98%D0%BD%D1%81%D1%83%D0%BB%D0%B8%D0%BD" title="Инсулин">инсулина</a> и патологической инсулинорезистентности тканей при сохранной инсулинчувствительности яичников. С этого времени для лечения поликистоза яичников стали широко применять в качестве препаратов первой линии препараты, нормализующие чувствительность тканей к инсулину и понижающие секрецию инсулина — <a href="/wiki/%D0%9C%D0%B5%D1%82%D1%84%D0%BE%D1%80%D0%BC%D0%B8%D0%BD" title="Метформин">метформин</a>, глитазоны (<a href="/wiki/%D0%9F%D0%B8%D0%BE%D0%B3%D0%BB%D0%B8%D1%82%D0%B0%D0%B7%D0%BE%D0%BD" title="Пиоглитазон">пиоглитазон</a>, <a class="new" href="/w/index.php?title=%D0%A0%D0%BE%D0%B7%D0%B8%D0%B3%D0%BB%D0%B8%D1%82%D0%B0%D0%B7%D0%BE%D0%BD&amp;action=edit&amp;redlink=1" title="Розиглитазон (страница отсутствует)">розиглитазон</a>). Такой подход оказался очень успешным — у 80 % женщин с поликистозом яичников на монотерапии метформином или одним из глитазонов спонтанно восстанавливалась овуляция, нормализовывался менструальный цикл, снижалась секреция андрогенов яичниками и исчезали либо уменьшались симптомы гиперандрогении, снижался вес тела, нормализовывался углеводный обмен, улучшалось психическое состояние. Большинство этих женщин затем смогли выносить и родить здоровых детей.
</p><p>Ещё более высокий процент успеха, превышающий 90 %, дала комбинированная терапия — сочетание метформина или глитазонов с уже ранее известными методами (эстрогенами, антиандрогенами и прогестинами, и/или с антиэстрогенами в середине цикла и/или, возможно, коррекцией сопутствующих нарушений секреции пролактина, гормонов щитовидной железы, надпочечниковых андрогенов). Введение в практику гинекологов-эндокринологов такого комбинированного подхода к лечению поликистоза яичников позволило почти полностью исключить, кроме редких полирезистентных случаев, необходимость в хирургическом вмешательстве по поводу поликистоза яичников, а также сделать значительно более редкой необходимость в индукции овуляции с помощью гонадотропинов и в искусственном оплодотворении женщин с поликистозом яичников.
</p>
<h3><span id=".D0.A1.D0.BE.D0.B2.D1.80.D0.B5.D0.BC.D0.B5.D0.BD.D0.BD.D0.BE.D0.B5_.D1.81.D0.BE.D1.81.D1.82.D0.BE.D1.8F.D0.BD.D0.B8.D0.B5_.D0.B2.D0.BE.D0.BF.D1.80.D0.BE.D1.81.D0.B0"></span><span class="mw-headline" id="Современное_состояние_вопроса">Современное состояние вопроса</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B8%D0%BD%D0%B4%D1%80%D0%BE%D0%BC_%D0%BF%D0%BE%D0%BB%D0%B8%D0%BA%D0%B8%D1%81%D1%82%D0%BE%D0%B7%D0%BD%D1%8B%D1%85_%D1%8F%D0%B8%D1%87%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2&amp;veaction=edit&amp;section=9" title="Редактировать раздел «Современное состояние вопроса»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B8%D0%BD%D0%B4%D1%80%D0%BE%D0%BC_%D0%BF%D0%BE%D0%BB%D0%B8%D0%BA%D0%B8%D1%81%D1%82%D0%BE%D0%B7%D0%BD%D1%8B%D1%85_%D1%8F%D0%B8%D1%87%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2&amp;action=edit&amp;section=9" title="Редактировать код раздела «Современное состояние вопроса»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>На сегодняшний день препаратами первой линии при лечении поликистоза яичников являются <a href="/wiki/%D0%9C%D0%B5%D1%82%D1%84%D0%BE%D1%80%D0%BC%D0%B8%D0%BD" title="Метформин">метформин</a> и глитазоны (<a href="/wiki/%D0%9F%D0%B8%D0%BE%D0%B3%D0%BB%D0%B8%D1%82%D0%B0%D0%B7%D0%BE%D0%BD" title="Пиоглитазон">пиоглитазон</a>, <a class="new" href="/w/index.php?title=%D0%A0%D0%BE%D0%B7%D0%B8%D0%B3%D0%BB%D0%B8%D1%82%D0%B0%D0%B7%D0%BE%D0%BD&amp;action=edit&amp;redlink=1" title="Розиглитазон (страница отсутствует)">розиглитазон</a>). К ним могут быть присоединены, при необходимости, <a href="/wiki/%D0%90%D0%BD%D1%82%D0%B8%D0%B0%D0%BD%D0%B4%D1%80%D0%BE%D0%B3%D0%B5%D0%BD%D1%8B" title="Антиандрогены">антиандрогенные</a> препараты (<a href="/wiki/%D0%A6%D0%B8%D0%BF%D1%80%D0%BE%D1%82%D0%B5%D1%80%D0%BE%D0%BD%D0%B0_%D0%B0%D1%86%D0%B5%D1%82%D0%B0%D1%82" title="Ципротерона ацетат">ципротерона ацетат</a>), <a class="mw-redirect" href="/wiki/%D0%AD%D1%81%D1%82%D1%80%D0%BE%D0%B3%D0%B5%D0%BD" title="Эстроген">эстрогены</a> (<a href="/wiki/%D0%AD%D1%82%D0%B8%D0%BD%D0%B8%D0%BB%D1%8D%D1%81%D1%82%D1%80%D0%B0%D0%B4%D0%B8%D0%BE%D0%BB" title="Этинилэстрадиол">этинилэстрадиол</a> в виде отдельного препарата или в противозачаточных таблетках), <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%B3%D0%B5%D1%81%D1%82%D0%B5%D1%80%D0%BE%D0%BD" title="Прогестерон">прогестины</a>, малые дозы <a href="/wiki/%D0%94%D0%B5%D0%BA%D1%81%D0%B0%D0%BC%D0%B5%D1%82%D0%B0%D0%B7%D0%BE%D0%BD" title="Дексаметазон">дексаметазона</a> (0.5-1 мг в вечернее время для подавления секреции надпочечниковых андрогенов).
</p><p>Необходимы меры по нормализации массы тела: диета, физическая активность.
</p><p>Перспективным направлением эффективной и безопасной терапии СПКЯ является использование нутрицевтиков (в частности, <a href="/wiki/%D0%9C%D0%B8%D0%BE-%D0%B8%D0%BD%D0%BE%D0%B7%D0%B8%D1%82%D0%BE%D0%BB" title="Мио-инозитол">мио-инозитола</a>). Миоинозитол и его производные необходимы для осуществления эффектов <a href="/wiki/%D0%93%D0%BE%D0%BD%D0%B0%D0%B4%D0%BE%D0%BB%D0%B8%D0%B1%D0%B5%D1%80%D0%B8%D0%BD" title="Гонадолиберин">ГНВГ</a>, <a href="/wiki/%D0%9B%D1%8E%D1%82%D0%B5%D0%B8%D0%BD%D0%B8%D0%B7%D0%B8%D1%80%D1%83%D1%8E%D1%89%D0%B8%D0%B9_%D0%B3%D0%BE%D1%80%D0%BC%D0%BE%D0%BD" title="Лютеинизирующий гормон">ЛГ</a>, <a href="/wiki/%D0%A4%D0%BE%D0%BB%D0%BB%D0%B8%D0%BA%D1%83%D0%BB%D0%BE%D1%81%D1%82%D0%B8%D0%BC%D1%83%D0%BB%D0%B8%D1%80%D1%83%D1%8E%D1%89%D0%B8%D0%B9_%D0%B3%D0%BE%D1%80%D0%BC%D0%BE%D0%BD" title="Фолликулостимулирующий гормон">ФСГ</a>. Эффекты <a href="/wiki/%D0%9C%D0%B8%D0%BE-%D0%B8%D0%BD%D0%BE%D0%B7%D0%B8%D1%82%D0%BE%D0%BB" title="Мио-инозитол">мио-инозитола</a> у женщин с СПКЯ были изучены в систематическом анализе рандомизированных контролируемых исследований. В целом, результаты анализа позволяют рекомендовать использование мио-инозитола для улучшения функции яичников, а также метаболических и гормональных показателей у пациенток с СПКЯ <sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup>. 
</p><p>При наличии сопутствующей гиперпролактинемии показана её коррекция с помощью назначения <a href="/wiki/%D0%91%D1%80%D0%BE%D0%BC%D0%BE%D0%BA%D1%80%D0%B8%D0%BF%D1%82%D0%B8%D0%BD" title="Бромокриптин">бромокриптина</a>. При выявлении субклинической, а тем более клинически выраженной недостаточности <a href="/wiki/%D0%A9%D0%B8%D1%82%D0%BE%D0%B2%D0%B8%D0%B4%D0%BD%D0%B0%D1%8F_%D0%B6%D0%B5%D0%BB%D0%B5%D0%B7%D0%B0" title="Щитовидная железа">щитовидной железы</a> она подлежит коррекции с помощью назначения экзогенного L-тироксина.
</p><p>При необходимости индуцировать овуляцию, если она не восстанавливается спонтанно на фоне терапии метформином или глитазонами, женщине могут быть назначены <a class="mw-redirect" href="/wiki/%D0%9A%D0%BB%D0%BE%D1%81%D1%82%D0%B8%D0%BB%D0%B1%D0%B5%D0%B3%D0%B8%D1%82" title="Клостилбегит">клостилбегит</a> или <a href="/wiki/%D0%A2%D0%B0%D0%BC%D0%BE%D0%BA%D1%81%D0%B8%D1%84%D0%B5%D0%BD" title="Тамоксифен">тамоксифен</a> в середине цикла.
</p><p>В случае резистентности ко всем применяемым методам лечения показана хирургическая операция (лазерная или диатермокоагуляция яичников либо их декапсуляция, частичная резекция).
</p>
