<p><b>Белз</b><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> (<a href="/wiki/%D0%A3%D0%BA%D1%80%D0%B0%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Украинский язык">укр.</a> <span lang="uk" style="font-style:italic;">Белз</span>, <a href="/wiki/%D0%98%D0%B4%D0%B8%D1%88" title="Идиш">идиш</a> <span dir="rtl" lang="yi">בעלז</span>‎, <a href="/wiki/%D0%9F%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Польский язык">польск.</a> <span lang="pl" style="font-style:italic;">Bełz</span>) — город в <a href="/wiki/%D0%A7%D0%B5%D1%80%D0%B2%D0%BE%D0%BD%D0%BE%D0%B3%D1%80%D0%B0%D0%B4%D1%81%D0%BA%D0%B8%D0%B9_%D1%80%D0%B0%D0%B9%D0%BE%D0%BD" title="Червоноградский район">Червоноградском районе</a> <a href="/wiki/%D0%9B%D1%8C%D0%B2%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C" title="Львовская область">Львовской области</a> <a href="/wiki/%D0%A3%D0%BA%D1%80%D0%B0%D0%B8%D0%BD%D0%B0" title="Украина">Украины</a>. Административный центр <a href="/wiki/%D0%91%D0%B5%D0%BB%D0%B7%D1%81%D0%BA%D0%B0%D1%8F_%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D1%89%D0%B8%D0%BD%D0%B0" title="Белзская городская община">Белзской городской общины</a>.
</p><p>Один из древнейших городов Украины, столица <a href="/wiki/%D0%91%D0%B5%D0%BB%D0%B7%D1%81%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BD%D1%8F%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Белзское княжество">удельного княжества</a>, затем административный центр <a href="/wiki/%D0%91%D0%B5%D0%BB%D0%B7%D1%81%D0%BA%D0%BE%D0%B5_%D0%B2%D0%BE%D0%B5%D0%B2%D0%BE%D0%B4%D1%81%D1%82%D0%B2%D0%BE" title="Белзское воеводство">воеводства</a> <a href="/wiki/%D0%A0%D0%B5%D1%87%D1%8C_%D0%9F%D0%BE%D1%81%D0%BF%D0%BE%D0%BB%D0%B8%D1%82%D0%B0%D1%8F" title="Речь Посполитая">Речи Посполитой</a> и родина белзского направления в <a href="/wiki/%D0%A5%D0%B0%D1%81%D0%B8%D0%B4%D0%B8%D0%B7%D0%BC" title="Хасидизм">хасидизме</a>. Белз в древности, вероятно, был одним из <a href="/wiki/%D0%A7%D0%B5%D1%80%D0%B2%D0%B5%D0%BD%D1%81%D0%BA%D0%B8%D0%B5_%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%B0" title="Червенские города">Червенских городов</a>, а затем стал частью <a href="/wiki/%D0%93%D0%B0%D0%BB%D0%B8%D1%86%D0%BA%D0%BE-%D0%92%D0%BE%D0%BB%D1%8B%D0%BD%D1%81%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BD%D1%8F%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Галицко-Волынское княжество">Галицко-Волынского княжества</a><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>. С 2001 года Белз и его окрестности по решению <a class="mw-redirect" href="/wiki/%D0%9A%D0%B0%D0%B1%D0%B8%D0%BD%D0%B5%D1%82_%D0%9C%D0%B8%D0%BD%D0%B8%D1%81%D1%82%D1%80%D0%BE%D0%B2_%D0%A3%D0%BA%D1%80%D0%B0%D0%B8%D0%BD%D1%8B" title="Кабинет Министров Украины">правительства Украины</a> стали Государственным историко-культурным заповедником «Княжий Белз».
</p>

<h2><span id=".D0.AD.D1.82.D0.B8.D0.BC.D0.BE.D0.BB.D0.BE.D0.B3.D0.B8.D1.8F"></span><span class="mw-headline" id="Этимология">Этимология</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Этимология»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;action=edit&amp;section=1" title="Редактировать код раздела «Этимология»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Есть несколько версий происхождения названия города. Согласно наиболее распространённой версии, древнеславянское слово «<i>белз</i>» или «<i>бевз</i>» означало болотистую, влажную местность; то же слово в <a href="/wiki/%D0%91%D0%BE%D0%B9%D0%BA%D0%B8" title="Бойки">бойковском</a> диалекте означает непроходимое, болотистое место. Есть также версия о <a href="/wiki/%D0%9A%D0%B5%D0%BB%D1%8C%D1%82%D1%8B" title="Кельты">кельтском</a> происхождении названия; в кельтских языках «<i>белз</i>» или «<i>пелз</i>» означало воду, поток. Эта версия появилась в начале <a href="/wiki/XX_%D0%B2%D0%B5%D0%BA" title="XX век">XX века</a> в связи с многочисленными находками следов <a href="/wiki/%D0%9A%D0%B5%D0%BB%D1%8C%D1%82%D1%8B" title="Кельты">кельтской</a> латенской археологической культуры на территории Прикарпатья. Ещё одна версия связывает название с <a href="/wiki/%D0%94%D1%80%D0%B5%D0%B2%D0%BD%D0%B5%D1%80%D1%83%D1%81%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Древнерусский язык">древнерусским</a> словом «<i>бълизь</i>» (белое место, поляна среди тёмного леса)<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>. Все эти версии достаточно правдоподобны и передают особенности природного расположения города.
</p>
<h2><span id=".D0.93.D0.B5.D0.BE.D0.B3.D1.80.D0.B0.D1.84.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.BE.D0.B5_.D0.BF.D0.BE.D0.BB.D0.BE.D0.B6.D0.B5.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Географическое_положение">Географическое положение</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Географическое положение»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;action=edit&amp;section=2" title="Редактировать код раздела «Географическое положение»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Белз расположен на левом берегу реки <a href="/wiki/%D0%A1%D0%BE%D0%BB%D0%BE%D0%BA%D0%B8%D1%8F_(%D1%80%D0%B5%D0%BA%D0%B0)" title="Солокия (река)">Солокия</a> — притоке <a class="mw-redirect" href="/wiki/%D0%97%D0%B0%D0%BF%D0%B0%D0%B4%D0%BD%D1%8B%D0%B9_%D0%91%D1%83%D0%B3_(%D1%80%D0%B5%D0%BA%D0%B0)" title="Западный Буг (река)">Западного Буга</a> — и правом берегу ручья Речица (Жечица), в 3 км от украинско-польской границы, в 12 км от <a href="/wiki/%D0%A7%D0%B5%D1%80%D0%B2%D0%BE%D0%BD%D0%BE%D0%B3%D1%80%D0%B0%D0%B4" title="Червоноград">Червонограда</a>, центра <a class="mw-redirect" href="/wiki/%D0%9B%D1%8C%D0%B2%D0%BE%D0%B2%D1%81%D0%BA%D0%BE-%D0%92%D0%BE%D0%BB%D1%8B%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%83%D0%B3%D0%BE%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%B1%D0%B0%D1%81%D1%81%D0%B5%D0%B9%D0%BD" title="Львовско-Волынский угольный бассейн">Львовско-Волынского угольного бассейна</a>, в 25 км от <a href="/wiki/%D0%A1%D0%BE%D0%BA%D0%B0%D0%BB%D1%8C" title="Сокаль">Сокаля</a>, районного центра. Район Белза принадлежит двум крупным и своеобразным физико-географическим областям: низинному <a href="/wiki/%D0%9C%D0%B0%D0%BB%D0%BE%D0%B5_%D0%9F%D0%BE%D0%BB%D0%B5%D1%81%D1%8C%D0%B5_(%D0%A3%D0%BA%D1%80%D0%B0%D0%B8%D0%BD%D0%B0)" title="Малое Полесье (Украина)">Малому Полесью</a> и <a href="/wiki/%D0%92%D0%BE%D0%BB%D1%8B%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B7%D0%B2%D1%8B%D1%88%D0%B5%D0%BD%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Волынская возвышенность">Волынской возвышенности</a>, которые разделены невысоким линейно вытянутым уступом. Возвышенная сторона района занята <a href="/wiki/%D0%A1%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%BE%D0%B5_%D1%85%D0%BE%D0%B7%D1%8F%D0%B9%D1%81%D1%82%D0%B2%D0%BE" title="Сельское хозяйство">сельскохозяйственными</a> угодьями, а низинная — <a href="/wiki/%D0%9B%D1%83%D0%B3%D0%B0" title="Луга">лугами</a> и <a href="/wiki/%D0%9B%D0%B5%D1%81" title="Лес">лесами</a>. По геологическому происхождению район принадлежит крупной <a class="mw-redirect" href="/wiki/%D0%9A%D0%B5%D0%BC%D0%B1%D1%80%D0%B8%D0%B9" title="Кембрий">докембрийской</a> <a href="/wiki/%D0%92%D0%BE%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%BE-%D0%95%D0%B2%D1%80%D0%BE%D0%BF%D0%B5%D0%B9%D1%81%D0%BA%D0%B0%D1%8F_%D0%BF%D0%BB%D0%B0%D1%82%D1%84%D0%BE%D1%80%D0%BC%D0%B0" title="Восточно-Европейская платформа">Восточно-Европейской платформе</a>, которая здесь имеет наклон и пересечена сетью разломов, тянущимися в трёх направлениях: на северо-запад, северо-восток и на восток. Почти вся территория района покрыта морскими отложениями <a href="/wiki/%D0%9C%D0%B5%D0%BB%D0%BE%D0%B2%D0%BE%D0%B9_%D0%BF%D0%B5%D1%80%D0%B8%D0%BE%D0%B4" title="Меловой период">верхнемелового периода</a>, представленными <a href="/wiki/%D0%9C%D0%B5%D1%80%D0%B3%D0%B5%D0%BB%D1%8C" title="Мергель">мергелями</a>, <a href="/wiki/%D0%98%D0%B7%D0%B2%D0%B5%D1%81%D1%82%D0%BD%D1%8F%D0%BA" title="Известняк">известняками</a> и <a href="/wiki/%D0%9C%D0%B5%D0%BB" title="Мел">мелом</a>. Долины современных рек Солокии, <a href="/wiki/%D0%A0%D0%B0%D1%82%D0%B0_(%D1%80%D0%B5%D0%BA%D0%B0)" title="Рата (река)">Раты</a>, Речицы, Болотни, <a href="/wiki/%D0%9B%D1%83%D0%B3%D0%B0_(%D1%80%D0%B5%D0%BA%D0%B0)" title="Луга (река)">Луги</a> представляют собой остатки древних флювиогляциальных потоков. Эти долины в послеледниковом периоде стали заболоченными и развили отложения <a href="/wiki/%D0%A2%D0%BE%D1%80%D1%84" title="Торф">торфа</a> глубиной от нескольких дециметров до нескольких метров, причём наибольшие отложения торфа сложились в долинах Солокии, Болотни и Раты. <a href="/wiki/%D0%9B%D1%91%D1%81%D1%81" title="Лёсс">Лёссы</a> и лёссообразные <a class="mw-redirect" href="/wiki/%D0%A1%D1%83%D0%B3%D0%BB%D0%B8%D0%BD%D0%BA%D0%B8" title="Суглинки">суглинки</a> стали субстратом образования серых лесных грунтов. Южнее Белза проходит полоса <a href="/wiki/%D0%94%D1%8E%D0%BD%D0%B0" title="Дюна">дюн</a> высотой 5-20 метров и длиной до сотен метров, что представляет редкое явление для Украины<sup class="reference" id="cite_ref-my_7-0"><a href="#cite_note-my-7">[7]</a></sup>.
</p>
<h3><span id=".D0.9A.D0.BB.D0.B8.D0.BC.D0.B0.D1.82.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.B8.D0.B5_.D1.83.D1.81.D0.BB.D0.BE.D0.B2.D0.B8.D1.8F"></span><span class="mw-headline" id="Климатические_условия">Климатические условия</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Климатические условия»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;action=edit&amp;section=3" title="Редактировать код раздела «Климатические условия»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%D0%9A%D0%BB%D0%B8%D0%BC%D0%B0%D1%82" title="Климат">Климат</a> Белза влажный, умеренно тёплый. Средняя <a href="/wiki/%D0%A2%D0%B5%D0%BC%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D1%83%D1%80%D0%B0" title="Температура">температура</a> января в Белзе, как и в Сокальском районе в целом, составляет −4,2 — 4,4 °C, июля +18,0 — +18,4 °C. Общая продолжительность периода с температурой выше 10 °C составляет в среднем 155—160 дней. Среднегодовая норма <a href="/wiki/%D0%90%D1%82%D0%BC%D0%BE%D1%81%D1%84%D0%B5%D1%80%D0%BD%D1%8B%D0%B5_%D0%BE%D1%81%D0%B0%D0%B4%D0%BA%D0%B8" title="Атмосферные осадки">осадков</a> составляет 560—640 мм в год. Осадки выпадают в основном в тёплый период года<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>.
</p>
<h2><span id=".D0.9D.D0.B0.D1.81.D0.B5.D0.BB.D0.B5.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Население">Население</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Население»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;action=edit&amp;section=4" title="Редактировать код раздела «Население»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>До середины <a href="/wiki/XIX_%D0%B2%D0%B5%D0%BA" title="XIX век">XIX века</a> в населении города преобладали <a href="/wiki/%D0%A0%D1%83%D1%81%D0%B8%D0%BD%D1%8B" title="Русины">русины</a>, а <a href="/wiki/%D0%95%D0%B2%D1%80%D0%B5%D0%B8" title="Евреи">евреи</a> и <a href="/wiki/%D0%9F%D0%BE%D0%BB%D1%8F%D0%BA%D0%B8" title="Поляки">поляки</a> составляли примерно по 20 % численности (по расчётам украинского <a class="mw-redirect" href="/wiki/%D0%93%D0%B5%D0%BE%D0%B3%D1%80%D0%B0%D1%84" title="Географ">географа</a> <a href="/wiki/%D0%9A%D1%83%D0%B1%D0%B8%D0%B9%D0%BE%D0%B2%D0%B8%D1%87,_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80_%D0%9C%D0%B8%D1%85%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2%D0%B8%D1%87" title="Кубийович, Владимир Михайлович">Владимира Кубийовича</a>). С 1859 года и по 1931 год более половины населения Белза были евреями. В <a class="mw-redirect" href="/wiki/1900" title="1900">1900</a> г. в городе проживали 2872<sup class="reference" id="cite_ref-vitk_9-0"><a href="#cite_note-vitk-9">[9]</a></sup>, в <a class="mw-redirect" href="/wiki/1910" title="1910">1910</a> г. — 3625 евреев (60 % населения)<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>, 1,600 украинцев и 900 поляков.
</p><p>В <a class="mw-redirect" href="/wiki/%D0%A1%D0%BE%D0%B1%D1%8B%D1%82%D0%B8%D1%8F_%D0%B2_%D0%9F%D0%BE%D0%BB%D1%8C%D1%88%D0%B5_%D0%B2_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D0%B5_1939_%D0%B3%D0%BE%D0%B4%D0%B0" title="События в Польше в сентябре 1939 года">сентябре 1939 года</a> часть еврейского населения Белза покинула город вместе с частями Красной армии в конце сентября 1939 года (<a href="/wiki/%D0%94%D0%BE%D0%B3%D0%BE%D0%B2%D0%BE%D1%80_%D0%BE_%D0%B4%D1%80%D1%83%D0%B6%D0%B1%D0%B5_%D0%B8_%D0%B3%D1%80%D0%B0%D0%BD%D0%B8%D1%86%D0%B5_%D0%BC%D0%B5%D0%B6%D0%B4%D1%83_%D0%A1%D0%A1%D0%A1%D0%A0_%D0%B8_%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%B8%D0%B5%D0%B9" title="Договор о дружбе и границе между СССР и Германией">Договор о дружбе и границе между СССР и Германией</a>). Белз вошёл в состав <a class="mw-redirect" href="/wiki/%D0%93%D0%B5%D0%BD%D0%B5%D1%80%D0%B0%D0%BB-%D0%B3%D1%83%D0%B1%D0%B5%D1%80%D0%BD%D0%B0%D1%82%D0%BE%D1%80%D1%81%D1%82%D0%B2%D0%BE_(%D0%A2%D1%80%D0%B5%D1%82%D0%B8%D0%B9_%D1%80%D0%B5%D0%B9%D1%85)" title="Генерал-губернаторство (Третий рейх)">Генерал-губернаторства</a> (1939—1944). В мае 1942 года еврейское население Белза составляло 1504 человек, но в июне 1942 г. депортация евреев была проведена в <a class="mw-redirect" href="/wiki/%D0%93%D1%80%D1%83%D0%B1%D0%B5%D1%88%D0%BE%D0%B2" title="Грубешов">Грубешов</a> и лагерь смерти <a href="/wiki/%D0%A1%D0%BE%D0%B1%D0%B8%D0%B1%D0%BE%D1%80" title="Собибор">Собибор</a>.
</p><p>В конце июля 1944 года Белз вошёл в состав Польши и в 1947 году из города было выселено всё украинское население. В 1951 году в соответствии с советско-польским <a class="mw-redirect" href="/wiki/%D0%A1%D0%BE%D0%B2%D0%B5%D1%82%D1%81%D0%BA%D0%BE-%D0%BF%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D0%BE%D0%B1%D0%BC%D0%B5%D0%BD_%D1%83%D1%87%D0%B0%D1%81%D1%82%D0%BA%D0%B0%D0%BC%D0%B8_%D1%82%D0%B5%D1%80%D1%80%D0%B8%D1%82%D0%BE%D1%80%D0%B8%D0%B9_1951_%D0%B3%D0%BE%D0%B4%D0%B0" title="Советско-польский обмен участками территорий 1951 года">межправительственным соглашением</a> Белз вошёл в состав <a class="mw-redirect" href="/wiki/%D0%A3%D0%A1%D0%A1%D0%A0" title="УССР">УССР</a> и из города вывезли всех поляков. Город вновь был заселён в основном украинцами из других регионов УССР, и в настоящее время его население практически моноэтнично. В результате этих перемещений население Белза совершенно изменилось и не является прямым продолжением белзских горожан предшествующих эпох.
</p>
<h2><span id=".D0.98.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.8F"></span><span class="mw-headline" id="История">История</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;veaction=edit&amp;section=5" title="Редактировать раздел «История»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;action=edit&amp;section=5" title="Редактировать код раздела «История»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id=".D0.A1.D1.80.D0.B5.D0.B4.D0.BD.D0.B5.D0.B2.D0.B5.D0.BA.D0.BE.D0.B2.D1.8B.D0.B9_.D0.BF.D0.B5.D1.80.D0.B8.D0.BE.D0.B4"></span><span class="mw-headline" id="Средневековый_период">Средневековый период</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Средневековый период»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;action=edit&amp;section=6" title="Редактировать код раздела «Средневековый период»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:%D0%91%D0%B5%D0%BB%D0%B7_%D0%BF%D0%BB%D0%B0%D0%BD_%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%B0_4.png"><img class="mw-file-element" data-file-height="366" data-file-width="600" decoding="async" height="183" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/9d/%D0%91%D0%B5%D0%BB%D0%B7_%D0%BF%D0%BB%D0%B0%D0%BD_%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%B0_4.png/300px-%D0%91%D0%B5%D0%BB%D0%B7_%D0%BF%D0%BB%D0%B0%D0%BD_%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%B0_4.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/9d/%D0%91%D0%B5%D0%BB%D0%B7_%D0%BF%D0%BB%D0%B0%D0%BD_%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%B0_4.png/450px-%D0%91%D0%B5%D0%BB%D0%B7_%D0%BF%D0%BB%D0%B0%D0%BD_%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%B0_4.png 1.5x, //upload.wikimedia.org/wikipedia/commons/9/9d/%D0%91%D0%B5%D0%BB%D0%B7_%D0%BF%D0%BB%D0%B0%D0%BD_%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%B0_4.png 2x" width="300"/></a><figcaption></figcaption></figure>
<p>Первое упоминание о Белзе находится в «<a href="/wiki/%D0%9F%D0%BE%D0%B2%D0%B5%D1%81%D1%82%D1%8C_%D0%B2%D1%80%D0%B5%D0%BC%D0%B5%D0%BD%D0%BD%D1%8B%D1%85_%D0%BB%D0%B5%D1%82" title="Повесть временных лет">Повести временных лет</a>», где под 1030 годом говорится, что город был отвоёван <a href="/wiki/%D0%9A%D0%B8%D0%B5%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%A0%D1%83%D1%81%D1%8C" title="Киевская Русь">киевским</a> князем <a class="mw-redirect" href="/wiki/%D0%AF%D1%80%D0%BE%D1%81%D0%BB%D0%B0%D0%B2_%D0%9C%D1%83%D0%B4%D1%80%D1%8B%D0%B9" title="Ярослав Мудрый">Ярославом Мудрым</a> у <a href="/wiki/%D0%9F%D0%BE%D0%BB%D1%8C%D1%88%D0%B0" title="Польша">поляков</a>.
</p><p>Город основали на полуострове, образованном при слиянии рек Солокия и Речица, которые в ту пору были полноводными и богатыми рыбой. С северу к городу прилегала пологоволнистая Волынская возвышенность с плодородными и лёгкими для обработки почвами, на которых до сих пор сохранились дубово-липовые леса. Эти леса давали хороший строительный материал, кроме того в лесах развивалось <a class="mw-redirect" href="/wiki/%D0%91%D0%BE%D1%80%D1%82%D0%BD%D0%B8%D1%87%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Бортничество">бортничество</a>. На юге простирались сосновые леса Малого Полесья с многочисленными реками и ручьями. Торговые пути из города вели на запад к <a href="/wiki/%D0%9A%D1%80%D0%B0%D0%BA%D0%BE%D0%B2" title="Краков">Кракову</a>, на юг к <a href="/wiki/%D0%93%D0%B0%D0%BB%D0%B8%D1%87_(%D0%98%D0%B2%D0%B0%D0%BD%D0%BE-%D0%A4%D1%80%D0%B0%D0%BD%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C)" title="Галич (Ивано-Франковская область)">Галичу</a>, на восток к <a href="/wiki/%D0%9A%D0%B8%D0%B5%D0%B2" title="Киев">Киеву</a>, на север к <a href="/wiki/%D0%91%D0%B0%D0%BB%D1%82%D0%B8%D0%B9%D1%81%D0%BA%D0%BE%D0%B5_%D0%BC%D0%BE%D1%80%D0%B5" title="Балтийское море">Балтике</a>. Неудобным было приграничное положение Белза. Для его обороны были созданы укреплённые городища <a href="/wiki/%D0%A3%D0%B3%D0%BD%D0%B5%D0%B2" title="Угнев">Угнев</a> (на западе) и <a href="/wiki/%D0%92%D0%B0%D1%80%D1%8F%D0%B6_(%D0%9B%D1%8C%D0%B2%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C)" title="Варяж (Львовская область)">Варяж</a> (на севере). Самым северным форпостом Белзской земли стал <a class="new" href="/w/index.php?title=%D0%92%D1%81%D0%B5%D0%B2%D0%BE%D0%BB%D0%BE%D0%B6%D1%81%D0%BA_(%D0%92%D0%BE%D0%BB%D1%8B%D0%BD%D1%8C)&amp;action=edit&amp;redlink=1" title="Всеволожск (Волынь) (страница отсутствует)">Всеволожск</a>, самым южным — <a href="/wiki/%D0%91%D1%83%D1%81%D0%BA" title="Буск">Бужск</a><sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup>.
</p><p>Современные остатки древнего города расположены в урочище Замочек на низком левом берегу реки Солокии между её нынешним и старым руслами. Укреплённая часть поселения состоит из двух частей, <a href="/wiki/%D0%94%D0%B5%D1%82%D0%B8%D0%BD%D0%B5%D1%86" title="Детинец">детинца</a> (крепости) и окольного города, сохранились укрепления — рвы и валы. <a href="/wiki/%D0%91%D0%B5%D0%BB%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D0%B4%D0%B5%D1%82%D0%B8%D0%BD%D0%B5%D1%86" title="Белзский детинец">Белзский детинец</a> площадью около 4 га имеет прямоугольные очертания (220 на 190 м), он был обнесён по периметру валом, состоявшим из деревянных клетей, засыпанных землёй. Примыкавший к нему с юго-востока окольный город (площадь около 3 га) был укреплён валом с трёх сторон (кроме обращённой к детинцу). Город был окружён также неукреплёнными <a href="/wiki/%D0%9F%D0%BE%D1%81%D0%B0%D0%B4" title="Посад">посадами</a>, занимавшими соседние возвышенности среди болотистой низины. <a class="mw-redirect" href="/wiki/%D0%90%D1%80%D1%85%D0%B5%D0%BE%D0%BB%D0%BE%D0%B3" title="Археолог">Археологами</a> были обнаружены остатки домов с глинобитными полами, хозяйственные ямы, остатки гончарного горна. Среди находок есть также гончарная древнерусская посуда, орудия труда, ножи, боевой топор, наконечники стрел, шпоры, обломки стеклянных браслетов, шиферные пряслица, изделия из кости, свинцовая актовая печать. Абсолютное большинство находок датируется XI—XIV веками, преобладают материалы XII—XIII века.
</p><p>В XI—XVII веках Белз был одним из наибольших городов Галицкой Руси, а также крупным региональным центром с развитыми ремёслами и торговлей, и быстро стал столицей удельного <a href="/wiki/%D0%91%D0%B5%D0%BB%D0%B7%D1%81%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BD%D1%8F%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Белзское княжество">Белзского княжества</a> (1170—1462). Краткая история Белзского удела и список его князей выглядит так:
</p>
<ul><li>1170—1195 <a href="/wiki/%D0%92%D1%81%D0%B5%D0%B2%D0%BE%D0%BB%D0%BE%D0%B4_%D0%9C%D1%81%D1%82%D0%B8%D1%81%D0%BB%D0%B0%D0%B2%D0%B8%D1%87_(%D0%BA%D0%BD%D1%8F%D0%B7%D1%8C_%D0%B2%D0%BE%D0%BB%D1%8B%D0%BD%D1%81%D0%BA%D0%B8%D0%B9)" title="Всеволод Мстиславич (князь волынский)">Всеволод Мстиславич</a> (брат <a href="/wiki/%D0%A0%D0%BE%D0%BC%D0%B0%D0%BD_%D0%9C%D1%81%D1%82%D0%B8%D1%81%D0%BB%D0%B0%D0%B2%D0%B8%D1%87" title="Роман Мстиславич">Романа Мстиславича</a>, князя Галицко-Волынского)</li>
<li>1195—1207 и 1215—1233 <a href="/wiki/%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%92%D1%81%D0%B5%D0%B2%D0%BE%D0%BB%D0%BE%D0%B4%D0%BE%D0%B2%D0%B8%D1%87_(%D0%BA%D0%BD%D1%8F%D0%B7%D1%8C_%D0%B1%D0%B5%D0%BB%D0%B7%D1%81%D0%BA%D0%B8%D0%B9)" title="Александр Всеволодович (князь белзский)">Александр Всеволодович</a></li>
<li>1208—1211 <a href="/wiki/%D0%92%D0%B0%D1%81%D0%B8%D0%BB%D1%8C%D0%BA%D0%BE_%D0%A0%D0%BE%D0%BC%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87" title="Василько Романович">Василько Романович</a> (брат <a class="mw-redirect" href="/wiki/%D0%94%D0%B0%D0%BD%D0%B8%D0%BB%D0%BE_%D0%A0%D0%BE%D0%BC%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87" title="Данило Романович">Данила Романовича</a> Галицкого)</li>
<li>1233—1241 во владении Данила Романовича Галицкого, зимой 1240—1241 годов город разрушили <a class="mw-redirect" href="/wiki/%D0%A2%D0%B0%D1%82%D0%B0%D1%80%D0%BE-%D0%BC%D0%BE%D0%BD%D0%B3%D0%BE%D0%BB%D1%8B" title="Татаро-монголы">татаро-монголы</a>.</li>
<li>1241—1245 Всеволод Александрович</li>
<li>1245—1269 владение <a href="/wiki/%D0%9B%D0%B5%D0%B2_%D0%94%D0%B0%D0%BD%D0%B8%D0%BB%D0%BE%D0%B2%D0%B8%D1%87" title="Лев Данилович">Льва Даниловича</a> Галицкого</li>
<li>1269—1301 <a href="/wiki/%D0%AE%D1%80%D0%B8%D0%B9_I_%D0%9B%D1%8C%D0%B2%D0%BE%D0%B2%D0%B8%D1%87" title="Юрий I Львович">Юрий Львович</a>, после которого городом до 1340 года управляли воеводы и наместники <a href="/wiki/%D0%93%D0%B0%D0%BB%D0%B8%D1%86%D0%BA%D0%BE-%D0%92%D0%BE%D0%BB%D1%8B%D0%BD%D1%81%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BD%D1%8F%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Галицко-Волынское княжество">Галицко-Волынского княжества</a>.</li>
<li>1340—1377 в составе <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BD%D1%8F%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%BE_%D0%9B%D0%B8%D1%82%D0%BE%D0%B2%D1%81%D0%BA%D0%BE%D0%B5" title="Великое княжество Литовское">Литовского государства</a>, владение князя литовской династии <a href="/wiki/%D0%AE%D1%80%D0%B8%D0%B9_%D0%9D%D0%B0%D1%80%D0%B8%D0%BC%D1%83%D0%BD%D1%82%D0%BE%D0%B2%D0%B8%D1%87" title="Юрий Наримунтович">Юрия Наримунтовича</a>.</li>
<li>в 1352 город осадило сильное войско под предводительством польского и венгерского королей, однако гарнизон во главе с воеводой Дроздом (Дрозге) выстоял.</li>
<li>в 1366 поляки вновь осаждали Белз, княжество стало феодом Королевства Польского.</li>
<li>1377—1387 в составе <a href="/wiki/%D0%92%D0%B5%D0%BD%D0%B3%D1%80%D0%B8%D1%8F" title="Венгрия">Венгерского королевства</a>, управлял наместник, польский (силезский) князь и венгерский палатин <a href="/wiki/%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D1%81%D0%BB%D0%B0%D0%B2_%D0%9E%D0%BF%D0%BE%D0%BB%D1%8C%D1%87%D0%B8%D0%BA" title="Владислав Опольчик">Владислав Опольчик</a> в 1377 году даровал городу <a href="/wiki/%D0%9C%D0%B0%D0%B3%D0%B4%D0%B5%D0%B1%D1%83%D1%80%D0%B3%D1%81%D0%BA%D0%BE%D0%B5_%D0%BF%D1%80%D0%B0%D0%B2%D0%BE" title="Магдебургское право">магдебургское право</a>.</li>
<li>с 1387 года Белз является феодом Королевства Польского. В 1388 король <a href="/wiki/%D0%AF%D0%B3%D0%B0%D0%B9%D0%BB%D0%BE" title="Ягайло">Ягайло</a> передал удел <a href="/wiki/%D0%9C%D0%B0%D0%B7%D0%BE%D0%B2%D0%B8%D1%8F" title="Мазовия">мазовецкому</a> князю <a href="/wiki/%D0%97%D0%B5%D0%BC%D0%BE%D0%B2%D0%B8%D1%82_IV_%D0%9F%D0%BB%D0%BE%D1%86%D0%BA%D0%B8%D0%B9" title="Земовит IV Плоцкий">Земовиту IV Земовитовичу</a> (правил в 1388—1426). Однако князья литовской династии Юрий Наримунтович и его сын Иван Юрьевич продолжали называть себя князьями белзскими. В 1402 году в Белзе поселились монахи-<a href="/wiki/%D0%94%D0%BE%D0%BC%D0%B8%D0%BD%D0%B8%D0%BA%D0%B0%D0%BD%D1%86%D1%8B" title="Доминиканцы">доминиканцы</a>, которые стали крупными и богатыми землевладельцами, что позволило построить значительный комплекс церковных сооружений, сохранившийся до сих пор.</li>
<li>1426—1442 владение Казимира Земовитовича. В 1431 году он выступил против лидера русской партии Литвы князя <a href="/wiki/%D0%A1%D0%B2%D0%B8%D0%B4%D1%80%D0%B8%D0%B3%D0%B0%D0%B9%D0%BB%D0%BE" title="Свидригайло">Свидригайла</a>, осаждал восставших <a class="mw-redirect" href="/wiki/%D0%92%D0%BE%D1%81%D1%81%D1%82%D0%B0%D0%BD%D0%B8%D0%B5_%D0%A1%D0%B2%D0%B8%D0%B4%D1%80%D0%B8%D0%B3%D0%B0%D0%B9%D0%BB%D0%BE" title="Восстание Свидригайло">сторонников Свидригайла</a>, укрепившихся в <a class="mw-redirect" href="/wiki/%D0%9E%D0%BB%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B7%D0%B0%D0%BC%D0%BE%D0%BA" title="Олеский замок">Олеском замке</a>, однако в это же время восстали русские горожане Белза, организовали ополчение и захватили Бужск.</li>
<li>1442—1455 владение Владислава Земовитовича. В 1442, 1450, 1453 в землю вторгались войска волынских литовских князей и союзные им <a href="/wiki/%D0%97%D0%BE%D0%BB%D0%BE%D1%82%D0%B0%D1%8F_%D0%9E%D1%80%D0%B4%D0%B0" title="Золотая Орда">ордынские</a> отряды.</li>
<li>1455—1462 владение Земовита Владиславовича, последнего князя мазовецкой династии и отдельного белзского княжества.</li></ul>
<p>В 1462 году поляками было ликвидировано Белзское княжество и город стал центром Белзского воеводства. В 1499 и 1502 годах его сжигали татарско-турецкие войска. В 1590 году в Белзе происходил съезд русских <a href="/wiki/%D0%95%D0%BF%D0%B8%D1%81%D0%BA%D0%BE%D0%BF" title="Епископ">епископов</a> <a href="/wiki/%D0%A0%D0%B5%D1%87%D1%8C_%D0%9F%D0%BE%D1%81%D0%BF%D0%BE%D0%BB%D0%B8%D1%82%D0%B0%D1%8F" title="Речь Посполитая">Речи Посполитой</a> в деле подготовки <a href="/wiki/%D0%A3%D0%BD%D0%B8%D1%8F" title="Уния">унии</a> православной церкви с <a href="/wiki/%D0%9A%D0%B0%D1%82%D0%BE%D0%BB%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%86%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C" title="Католическая церковь">Римской церковью</a><sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>.
</p><p>Из-за конкуренции со стороны городов, основанных поляками (<a href="/wiki/%D0%A0%D0%B0%D0%B2%D0%B0-%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%B0%D1%8F" title="Рава-Русская">Рава-Русская</a>, <a href="/wiki/%D0%96%D0%BE%D0%BB%D0%BA%D0%B2%D0%B0" title="Жолква">Жолква</a>, <a class="mw-redirect" href="/wiki/%D0%9A%D1%80%D0%B8%D1%81%D1%82%D0%B8%D0%BD%D0%BE%D0%BF%D0%BE%D0%BB%D1%8C" title="Кристинополь">Кристинополь</a>) в XVIII веке город постепенно терял политико-административные функции и превращался в торгово-ремесленный городок.
</p>

<h3><span id=".D0.9D.D0.BE.D0.B2.D0.BE.D0.B5_.D0.B8_.D0.BD.D0.BE.D0.B2.D0.B5.D0.B9.D1.88.D0.B5.D0.B5_.D0.B2.D1.80.D0.B5.D0.BC.D1.8F"></span><span class="mw-headline" id="Новое_и_новейшее_время">Новое и новейшее время</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Новое и новейшее время»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;action=edit&amp;section=7" title="Редактировать код раздела «Новое и новейшее время»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure class="mw-default-size mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Belz_city_coa_1772.png"><img class="mw-file-element" data-file-height="253" data-file-width="200" decoding="async" height="253" src="//upload.wikimedia.org/wikipedia/commons/5/53/Belz_city_coa_1772.png" width="200"/></a><figcaption>Герб 1772 г.</figcaption></figure>
<p>В ходе войны под предводительством <a class="mw-redirect" href="/wiki/%D0%91%D0%BE%D0%B3%D0%B4%D0%B0%D0%BD_%D0%A5%D0%BC%D0%B5%D0%BB%D1%8C%D0%BD%D0%B8%D1%86%D0%BA%D0%B8%D0%B9" title="Богдан Хмельницкий">Богдана Хмельницкого</a> в 1648 во время наступления на <a href="/wiki/%D0%97%D0%B0%D0%BC%D0%BE%D1%81%D1%86%D1%8C" title="Замосць">Замостье</a> казацко-крестьянские отряды обошли город, ограничившись выкупом. В 1704 году Белз был разрушен <a href="/wiki/%D0%A8%D0%B2%D0%B5%D1%86%D0%B8%D1%8F" title="Швеция">шведскими</a> войсками. В 1772 году он вместе с <a href="/wiki/%D0%93%D0%B0%D0%BB%D0%B8%D1%86%D0%B8%D1%8F" title="Галиция">Галицией</a> вошёл в состав <a href="/wiki/%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%BE-%D0%92%D0%B5%D0%BD%D0%B3%D1%80%D0%B8%D1%8F" title="Австро-Венгрия">Габсбургской монархии</a>. В 1884 году была построена железнодорожная станция. В 1914−1915 году город был занят русскими войсками и входил в состав <a href="/wiki/%D0%93%D0%B0%D0%BB%D0%B8%D1%86%D0%B8%D0%B9%D1%81%D0%BA%D0%BE%D0%B5_%D0%B3%D0%B5%D0%BD%D0%B5%D1%80%D0%B0%D0%BB-%D0%B3%D1%83%D0%B1%D0%B5%D1%80%D0%BD%D0%B0%D1%82%D0%BE%D1%80%D1%81%D1%82%D0%B2%D0%BE" title="Галицийское генерал-губернаторство">Галицийского генерал-губернаторства</a>; в ходе военных действий он значительно пострадал. После распада <a href="/wiki/%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%BE-%D0%92%D0%B5%D0%BD%D0%B3%D1%80%D0%B8%D1%8F" title="Австро-Венгрия">Австро-Венгрии</a> в 1918 году Белз короткое время был уездным городом <a class="mw-redirect" href="/wiki/%D0%97%D0%A3%D0%9D%D0%A0" title="ЗУНР">ЗУНР</a>, а в 1919 году вошёл в состав <a href="/wiki/%D0%9F%D0%BE%D0%BB%D1%8C%D1%88%D0%B0" title="Польша">Польши</a>. В 1939 году Белз в ходе <a class="mw-redirect" href="/wiki/%D0%9F%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%BE%D1%85%D0%BE%D0%B4_%D0%A0%D0%9A%D0%9A%D0%90_(1939)" title="Польский поход РККА (1939)">Польского похода</a> был занят <a class="mw-redirect" href="/wiki/%D0%9A%D1%80%D0%B0%D1%81%D0%BD%D0%B0%D1%8F_%D0%B0%D1%80%D0%BC%D0%B8%D1%8F" title="Красная армия">Красной армией</a>, а затем по <a href="/wiki/%D0%94%D0%BE%D0%B3%D0%BE%D0%B2%D0%BE%D1%80_%D0%BE_%D0%BD%D0%B5%D0%BD%D0%B0%D0%BF%D0%B0%D0%B4%D0%B5%D0%BD%D0%B8%D0%B8_%D0%BC%D0%B5%D0%B6%D0%B4%D1%83_%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%B8%D0%B5%D0%B9_%D0%B8_%D0%A1%D0%BE%D0%B2%D0%B5%D1%82%D1%81%D0%BA%D0%B8%D0%BC_%D0%A1%D0%BE%D1%8E%D0%B7%D0%BE%D0%BC" title="Договор о ненападении между Германией и Советским Союзом">Договору о ненападении между Германией и Советским Союзом</a> вошёл в состав созданного <a class="mw-redirect" href="/wiki/%D0%9D%D0%B0%D1%86%D0%B8%D1%81%D1%82" title="Нацист">нацистами</a> на территории Польши <a class="mw-redirect" href="/wiki/%D0%93%D0%B5%D0%BD%D0%B5%D1%80%D0%B0%D0%BB-%D0%B3%D1%83%D0%B1%D0%B5%D1%80%D0%BD%D0%B0%D1%82%D0%BE%D1%80%D1%81%D1%82%D0%B2%D0%BE_(%D0%A2%D1%80%D0%B5%D1%82%D0%B8%D0%B9_%D1%80%D0%B5%D0%B9%D1%85)" title="Генерал-губернаторство (Третий рейх)">Генерального губернаторства</a>. Перед вхождением Белза в состав <a href="/wiki/%D0%9D%D0%B0%D1%86%D0%B8%D1%81%D1%82%D1%81%D0%BA%D0%B0%D1%8F_%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%B8%D1%8F" title="Нацистская Германия">нацистской Германии</a> город практически покинуло его основное население — <a href="/wiki/%D0%95%D0%B2%D1%80%D0%B5%D0%B8" title="Евреи">евреи</a>, однако, город практически не затронули <a href="/wiki/%D0%92%D0%BE%D0%BB%D1%8B%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D1%80%D0%B5%D0%B7%D0%BD%D1%8F" title="Волынская резня">столкновения между некоммунистическими украинскими и польскими партизанами</a>, в ходе которой исчезли соседние сёла Маджарки и Безуев<sup class="reference" id="cite_ref-ivah136_13-0"><a href="#cite_note-ivah136-13">[13]</a></sup>. Перед вступлением <a href="/wiki/%D0%A1%D0%BE%D0%B2%D0%B5%D1%82%D1%81%D0%BA%D0%B0%D1%8F_%D0%B0%D1%80%D0%BC%D0%B8%D1%8F" title="Советская армия">советской армии</a> ушло <a href="/wiki/%D0%9D%D0%B5%D0%BC%D1%86%D1%8B" title="Немцы">немецкое</a> население и прекратили существование немецкие сёла Иосиповка и Брукенталь<sup class="reference" id="cite_ref-ivah138_14-0"><a href="#cite_note-ivah138-14">[14]</a></sup>. В 1944 полуразрушенный Белз снова стал частью Польши. В ходе обмена населением между Польской народной республикой и Украинской ССР, а также после <a class="mw-redirect" href="/wiki/%D0%9E%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F_%D0%92%D0%B8%D1%81%D0%BB%D0%B0" title="Операция Висла">операции «Висла»</a>, из Белза и округи были выселены все украинцы. Этому сопротивлялись отряды <a class="mw-redirect" href="/wiki/%D0%A3%D0%9F%D0%90" title="УПА">УПА</a>, действовавшие в округе, один из них в 1949 обстрелял из миномётов железнодорожную станцию во время операции по депортации украинцев. В 1951 году находившуюся на территории Польши часть Львовско-Волынского угольного бассейна вместе с расположенным западнее угольных разработок Белзом, передали в состав <a class="mw-redirect" href="/wiki/%D0%A3%D0%A1%D0%A1%D0%A0" title="УССР">УССР</a> в <a class="mw-redirect" href="/wiki/%D0%A1%D0%BE%D0%B2%D0%B5%D1%82%D1%81%D0%BA%D0%BE-%D0%BF%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D0%BE%D0%B1%D0%BC%D0%B5%D0%BD_%D1%83%D1%87%D0%B0%D1%81%D1%82%D0%BA%D0%B0%D0%BC%D0%B8_%D1%82%D0%B5%D1%80%D1%80%D0%B8%D1%82%D0%BE%D1%80%D0%B8%D0%B9_1951_%D0%B3%D0%BE%D0%B4%D0%B0" title="Советско-польский обмен участками территорий 1951 года">обмен на горный нефтегазоносный Устрицкий район</a>. При этом польское население Белза (около 800 человек) вместе со всем движимым имуществом было вывезено, а на его место в пустой город поселены в основном украинцы из <a href="/wiki/%D0%9D%D0%B8%D0%B6%D0%BD%D0%B5-%D0%A3%D1%81%D1%82%D1%80%D0%B8%D1%86%D0%BA%D0%B8%D0%B9_%D1%80%D0%B0%D0%B9%D0%BE%D0%BD" title="Нижне-Устрицкий район">Нижне-Устрицкого района</a>, а также выходцы из других районов Украины и <a class="mw-redirect" href="/wiki/%D0%A1%D0%A1%D0%A1%D0%A0" title="СССР">СССР</a>.
</p><p>В 1951 году Белз стал центром Забужского района Львовщины, который в 1962 был упразднён и введён в состав Сокальского района. При советской власти были построены современные школа, профессионально-техническое училище, поликлиника, кинотеатр. Инфраструктуру дополнили клуб с библиотекой, детская музыкальная школа. В городе были разобраны разрушенные войной здания, отреставрирована часть памятников архитектуры. Промышленность включала филиал Львовского завода радиоэлектронной медицинской аппаратуры, маслобойное и асфальтное предприятия. Стремительное развитие соседнего <a href="/wiki/%D0%A7%D0%B5%D1%80%D0%B2%D0%BE%D0%BD%D0%BE%D0%B3%D1%80%D0%B0%D0%B4" title="Червоноград">Червонограда</a> как центра Львовско-Волынского угольного бассейна сузило зону влияния Белза и не дало ему возможность увеличить собственное население<sup class="reference" id="cite_ref-ivah136_13-1"><a href="#cite_note-ivah136-13">[13]</a></sup>.
</p>

<h2><span id=".D0.A1.D0.BE.D0.B2.D1.80.D0.B5.D0.BC.D0.B5.D0.BD.D0.BD.D0.BE.D1.81.D1.82.D1.8C"></span><span class="mw-headline" id="Современность">Современность</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;veaction=edit&amp;section=8" title="Редактировать раздел «Современность»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;action=edit&amp;section=8" title="Редактировать код раздела «Современность»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id=".D0.AD.D0.BA.D0.BE.D0.BD.D0.BE.D0.BC.D0.B8.D0.BA.D0.B0_.D0.B8_.D0.BA.D1.83.D0.BB.D1.8C.D1.82.D1.83.D1.80.D0.B0"></span><span class="mw-headline" id="Экономика_и_культура">Экономика и культура</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;veaction=edit&amp;section=9" title="Редактировать раздел «Экономика и культура»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;action=edit&amp;section=9" title="Редактировать код раздела «Экономика и культура»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Экономический кризис 1990-х оказал негативное влияние на экономику города, предприятия перепрофилировались и значительно снизили объёмы выпускаемой продукции, значительная часть трудоспособного населения оказалась без работы и вынуждена была уехать на заработки в <a href="/wiki/%D0%95%D0%B2%D1%80%D0%BE%D0%BF%D0%B5%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D0%BE%D1%8E%D0%B7" title="Европейский союз">Европейский союз</a> или <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D1%8F" title="Россия">Россию</a>. В 2001 году Белз стал историко-культурным заповедником с общей площадью охраняемой территории 310 га (директор Надежда Калыш). Стали предприниматься меры по реставрации памятников архитектуры, 13 из которых имеют всеукраинское значение, 50 — местное значение, есть также памятники истории, монументального искусства, градостроения, археологии<sup class="reference" id="cite_ref-loda_15-0"><a href="#cite_note-loda-15">[15]</a></sup>. Реставрационные работы и строительство заняли часть безработного населения. Планируется строительство таможенного перехода на проходящей мимо города украинско-польской границе, что должно оживить экономику города<sup class="reference" id="cite_ref-kmu_16-0"><a href="#cite_note-kmu-16">[16]</a></sup>.
</p><p>Культурно-образовательные учреждения города включают школу (в которой находится музей города), профессионально-техническое училище, детскую музыкальную школу, Народный дом с городской библиотекой, дом творчества учащихся. В результате кризиса 1990-х в городе перестал работать и сейчас перестраивается единственный кинотеатр.

<h3><span id=".D0.A0.D0.B5.D0.BB.D0.B8.D0.B3.D0.B8.D1.8F"></span><span class="mw-headline" id="Религия">Религия</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;veaction=edit&amp;section=10" title="Редактировать раздел «Религия»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;action=edit&amp;section=10" title="Редактировать код раздела «Религия»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Для <a href="/wiki/%D0%97%D0%B0%D0%BF%D0%B0%D0%B4%D0%BD%D0%B0%D1%8F_%D0%A3%D0%BA%D1%80%D0%B0%D0%B8%D0%BD%D0%B0" title="Западная Украина">западных регионов</a> в целом характерен самый высокий по Украине уровень <a href="/wiki/%D0%A0%D0%B5%D0%BB%D0%B8%D0%B3%D0%B8%D1%8F" title="Религия">религиозной</a> активности: здесь по данным 2005 года приходилось больше 9 зарегистрированных религиозных организации на каждые 10 тысяч жителей (в центральных областях — 2-5 организаций, в юго-восточных регионах — 1-2 организации)<sup class="reference" id="cite_ref-dnistr_341_17-0"><a href="#cite_note-dnistr_341-17">[17]</a></sup>. Не является исключением и Белз, в котором на две с половиной тысячи жителей приходится две <a class="mw-redirect" href="/wiki/%D0%A3%D0%93%D0%9A%D0%A6" title="УГКЦ">греко-католических</a> общины, одна православная (<a class="mw-redirect" href="/wiki/%D0%A3%D0%9F%D0%A6_(%D0%9A%D0%9F)" title="УПЦ (КП)">УПЦ (КП)</a>) и одна <a href="/wiki/%D0%9A%D0%B0%D1%82%D0%BE%D0%BB%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%86%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C" title="Католическая церковь">римско-католическая</a>, а также собрания <a class="mw-redirect" href="/wiki/%D0%9F%D1%80%D0%BE%D1%82%D0%B5%D1%81%D1%82%D0%B0%D0%BD%D1%82" title="Протестант">протестантов</a> (которые пока не обладают собственными помещениями). Белз принимает около 10 тысяч паломников-<a class="mw-redirect" href="/wiki/%D0%A5%D0%B0%D1%81%D0%B8%D0%B4" title="Хасид">хасидов</a> в год, для приёма которых построен дом с помещением для проведения богослужений<sup class="reference" id="cite_ref-zava_18-0"><a href="#cite_note-zava-18">[18]</a></sup>.
</p><p>Самая старинная в городе церковь — Пятницкая (Святой Параскевы), которая находится на христианском кладбище на Люблинском предместье. Эта двухсрубная, одноверхая церковь была построена только из дерева без использования железный гвоздей в XVII веке. К квадратному в плане центральному срубу с востока примыкает гранёный пятистенный сруб<sup class="reference" id="cite_ref-pamyat204_19-0"><a href="#cite_note-pamyat204-19">[19]</a></sup>. Центральный сруб перекрыт шлемовидной главой на световом восьмерике, восточный — накрыт пятискатной крышей<sup class="reference" id="cite_ref-pamyat204_19-1"><a href="#cite_note-pamyat204-19">[19]</a></sup>. В 1979 году были проведены реставрационные работы (архитекторы <a href="/wiki/%D0%9A%D0%B8%D0%BD%D0%B4%D0%B7%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9,_%D0%91%D0%BE%D0%B3%D0%B4%D0%B0%D0%BD_%D0%AF%D1%80%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%BE%D0%B2%D0%B8%D1%87" title="Киндзельский, Богдан Ярославович">Б. Я. Киндзельский</a>, Г. П. Крук, И. Р. Могытыч)<sup class="reference" id="cite_ref-pamyat204_19-2"><a href="#cite_note-pamyat204-19">[19]</a></sup>. По главной оси сооружения расположена колокольня, так же памятник архитектуры. Оба памятника создают скромный ансамбль народной архитектуры галицкой школы<sup class="reference" id="cite_ref-pamyat204_19-3"><a href="#cite_note-pamyat204-19">[19]</a></sup>. В советское время здесь размещался музей истории города, а во время <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B5%D1%81%D1%82%D1%80%D0%BE%D0%B9%D0%BA%D0%B0" title="Перестройка">перестройки</a> именно в этом храме были разрешены первые в городе собрания грекокатоликов.
</p>

<h2><span id=".D0.94.D0.BE.D1.81.D1.82.D0.BE.D0.BF.D1.80.D0.B8.D0.BC.D0.B5.D1.87.D0.B0.D1.82.D0.B5.D0.BB.D1.8C.D0.BD.D0.BE.D1.81.D1.82.D0.B8_.D0.B8_.D1.80.D0.B5.D0.BB.D0.B8.D0.B3.D0.B8.D0.BE.D0.B7.D0.BD.D1.8B.D0.B5_.D1.81.D0.B2.D1.8F.D1.82.D1.8B.D0.BD.D0.B8"></span><span class="mw-headline" id="Достопримечательности_и_религиозные_святыни">Достопримечательности и религиозные святыни</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;veaction=edit&amp;section=11" title="Редактировать раздел «Достопримечательности и религиозные святыни»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;action=edit&amp;section=11" title="Редактировать код раздела «Достопримечательности и религиозные святыни»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id=".D0.90.D0.BD.D1.81.D0.B0.D0.BC.D0.B1.D0.BB.D1.8C_.D0.B4.D0.BE.D0.BC.D0.B8.D0.BD.D0.B8.D0.BA.D0.B0.D0.BD.D1.81.D0.BA.D0.BE.D0.B3.D0.BE_.D0.BC.D0.BE.D0.BD.D0.B0.D1.81.D1.82.D1.8B.D1.80.D1.8F"></span><span class="mw-headline" id="Ансамбль_доминиканского_монастыря">Ансамбль доминиканского монастыря</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;veaction=edit&amp;section=12" title="Редактировать раздел «Ансамбль доминиканского монастыря»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;action=edit&amp;section=12" title="Редактировать код раздела «Ансамбль доминиканского монастыря»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Костёл Святого Николая был возведён из красного кирпича в середине XVI века, в 1867 году он реставрировался и реконструировался. Костёл был разрушен в 1944 году авиабомбой во время ожесточённых боёв между германскими и советскими войсками и до сих пор лежит в руинах. В междувоенный период кельи мужского монастыря были перестроены и стали зданием городской ратуши. В парке около руин монастыря в 1980 году был установлен памятник 950-летия Белза.
</p><p>Ансамбль женского доминиканского монастыря, расположенный по ул. Савенко, 3, характерен для архитектуры <a href="/wiki/%D0%91%D0%B0%D1%80%D0%BE%D0%BA%D0%BA%D0%BE#Архитектура_барокко" title="Барокко">барокко</a> XVII—XVIII веков состоял из костёла, к которому с юга и запада примыкают прямоугольные корпуса.
В 1743 году костёлу с южной и западной стороны были пристроены кельи мужского и женского монастырей. Оба корпуса кирпичные, прямоугольные в плане, двухэтажные, накрыты скатными крышами. Вместе с костёлом они образовывали замкнутый внутренний дворик, Фасады келий лишены архитектурного декора, прорезаны прямоугольными оконными проёмами, завершены профильными карнизами<sup class="reference" id="cite_ref-pamyat204_19-4"><a href="#cite_note-pamyat204-19">[19]</a></sup>. Костел женского монастыря в 1785 году передан грекокатоликам вместо сгоревшей деревянной церкви Св. Николая. До 1945 года церковь носила названия Св. Духа. С 1945 по 1951 год здесь был польский костел, после передачи города СССР храм был закрыт и лишь с 1990 года стал снова грекокатолической церковью Святого Николая. Кельи женского монастыря с 1960-х годов пребывают в руинах, большая часть комплекса не восстановлена до сих пор.
</p>

<h3><span id=".D0.98.D0.BA.D0.BE.D0.BD.D0.B0_.D0.91.D0.B5.D0.BB.D0.B7.D1.81.D0.BA.D0.BE.D0.B9_.28.D0.A7.D0.B5.D0.BD.D1.81.D1.82.D0.BE.D1.85.D0.BE.D0.B2.D1.81.D0.BA.D0.BE.D0.B9.29_.D0.91.D0.BE.D0.B3.D0.BE.D0.BC.D0.B0.D1.82.D0.B5.D1.80.D0.B8"></span><span class="mw-headline" id="Икона_Белзской_(Ченстоховской)_Богоматери">Икона Белзской (Ченстоховской) Богоматери</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;veaction=edit&amp;section=13" title="Редактировать раздел «Икона Белзской (Ченстоховской) Богоматери»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;action=edit&amp;section=13" title="Редактировать код раздела «Икона Белзской (Ченстоховской) Богоматери»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Наиболее почитаемая католиками Польши <a class="mw-redirect" href="/wiki/%D0%A7%D0%B5%D0%BD%D1%81%D1%82%D0%BE%D1%85%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%B0" title="Ченстоховская икона">Ченстоховская икона</a> <a class="mw-redirect" href="/wiki/%D0%91%D0%BE%D0%B3%D0%BE%D0%BC%D0%B0%D1%82%D0%B5%D1%80%D1%8C" title="Богоматерь">Богоматери</a>, находящаяся ныне в <a href="/wiki/%D0%AF%D1%81%D0%BD%D0%B0%D1%8F_%D0%93%D0%BE%D1%80%D0%B0" title="Ясная Гора">Ясногорском монастыре</a>, в пору существования удельного Белзского княжества находилась в церкви Белзского замка (теперь эта местность называется Замочек). Икона была вывезена в 1382 году из города польским (силезским) князем и венгерским палатином <a href="/wiki/%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D1%81%D0%BB%D0%B0%D0%B2_%D0%9E%D0%BF%D0%BE%D0%BB%D1%8C%D1%87%D0%B8%D0%BA" title="Владислав Опольчик">Владиславом Опольчиком</a>. Легенда приписывает написание иконы евангелисту <a href="/wiki/%D0%9B%D1%83%D0%BA%D0%B0_(%D0%B5%D0%B2%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D0%B8%D1%81%D1%82)" title="Лука (евангелист)">Луке</a>. По одной из версий, икону в <a href="/wiki/X_%D0%B2%D0%B5%D0%BA" title="X век">X веке</a> привезла в <a href="/wiki/%D0%9A%D0%B8%D0%B5%D0%B2" title="Киев">Киев</a> вместе со своим приданым Анна, сестра византийских императоров Константина и Василия, которую выдали замуж за <a class="mw-redirect" href="/wiki/%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80_%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B8%D0%B9" title="Владимир Великий">Владимира Великого</a>. Князь Владимир Великий приказал поместить икону в <a href="/wiki/%D0%94%D0%B5%D1%81%D1%8F%D1%82%D0%B8%D0%BD%D0%BD%D0%B0%D1%8F_%D1%86%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C" title="Десятинная церковь">Десятинной церкви</a> — первой каменной церкви Киева, где она находилась до 1240 года, а во время осады Киева ханом Батыем была тайно вывезена в небольшой княжеский город Белз. Согласно другой версии, икону привезли в XII веке два византийских императора-изгнанника — Андроник Комнин и Алексей Ател<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup>. Однако большинство современных польских исследователей не разделяет мнение об авторстве Святого Луки и считает, что икона была написана в Константинополе в XI—XII веках, хотя есть также гипотезы про её византийско-русское и итальянское происхождение<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup>. Порезы на иконе были оставлены, по одной из версий, осаждавшими Белз татарами, по другой — <a href="/wiki/%D0%93%D1%83%D1%81%D0%B8%D1%82%D1%8B" title="Гуситы">гуситами</a> во время осады ими <a href="/wiki/%D0%9A%D0%B0%D1%82%D0%BE%D0%BB%D0%B8%D1%86%D0%B8%D0%B7%D0%BC" title="Католицизм">католического</a> Ясногорского монастыря в 1415 году. В 2006 году копия иконы была передана администрации города и теперь находится в Белзе в часовне Святого Валентина (возле православной церкви Святого Николая, то есть в том месте, где она хранилась в старину).
</p>

<h3><span id=".D0.98.D1.83.D0.B4.D0.B0.D0.B8.D0.B7.D0.BC._.D0.A1.D0.B2.D1.8F.D1.82.D1.8B.D0.BD.D0.B8_.D0.B1.D0.B5.D0.BB.D0.B7.D1.81.D0.BA.D0.B8.D1.85_.D1.85.D0.B0.D1.81.D0.B8.D0.B4.D0.BE.D0.B2"></span><span class="mw-headline" id="Иудаизм._Святыни_белзских_хасидов">Иудаизм. Святыни белзских хасидов</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;veaction=edit&amp;section=14" title="Редактировать раздел «Иудаизм. Святыни белзских хасидов»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;action=edit&amp;section=14" title="Редактировать код раздела «Иудаизм. Святыни белзских хасидов»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Первые упоминания о белзских иудеях относятся к 1469 и 1494 годам. Евреи селились в Люблинском предместье, где и сейчас осталось их старинное кладбище. В 1570 году в городе уже было 20—25 еврейских семей, были построены деревянные <a href="/wiki/%D0%A1%D0%B8%D0%BD%D0%B0%D0%B3%D0%BE%D0%B3%D0%B0" title="Синагога">синагога</a>, <a class="mw-redirect" href="/wiki/%D0%9C%D0%B8%D0%BA%D0%B2%D0%B5" title="Микве">микве</a>, другие общинные здания. В 1665 белзская еврейская община добилась равноправия в правах и обязанностях и с этих пор активные в предпринимательстве евреи заселяют также площадь рынок, центральную часть города.<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup> В 1816 г. Белз стал одним из центров <a href="/wiki/%D0%A5%D0%B0%D1%81%D0%B8%D0%B4%D0%B8%D0%B7%D0%BC" title="Хасидизм">хасидизма</a> в Галиции и резиденцией собственной хасидской династий. Шолом Рокеах, называемый <i>Сар Шолом</i> (<i>«Князь мира»</i>), ученик Яакова Ицхака ха-Хозе («ясновидящего») из <a href="/wiki/%D0%9B%D1%8E%D0%B1%D0%BB%D0%B8%D0%BD" title="Люблин">Люблина</a>, был раввином в Белзе в 1817—1855 годах, после смерти своего учителя был признан цадиком. По инициативе Шолома Рокеаха в 1843 в Белзе была выстроена Большая синагога. Продолжили династию белзских ребе Егошуа Рокеах (1825—1894, второй белзский ребе), Иссохар Дов Рокеах (I) (1854—1926, третий белзский ребе), Арон Рокеах (1877—1957, четвёртый белзский ребе). С 1859 года и по 1931 год более половины населения Белза были евреями. В 1900 г. в городе проживало 2872 еврея<sup class="reference" id="cite_ref-vitk_9-1"><a href="#cite_note-vitk-9">[9]</a></sup>, в 1914 г. — 3600 евреев, 1600 украинцев, 900 поляков.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup>
В 1920—1930 гг. в Белзе активно действовала <a class="mw-redirect" href="/wiki/%D0%A1%D0%B8%D0%BE%D0%BD%D0%B8%D1%81%D1%82" title="Сионист">сионистская</a> организация, благодаря чему часть хасидов репатриировались в <a class="mw-redirect" href="/wiki/%D0%AD%D1%80%D0%B5%D1%86-%D0%98%D1%81%D1%80%D0%B0%D1%8D%D0%BB%D1%8C" title="Эрец-Исраэль">Палестину</a>. В сентябре — конце октября 1939 года Белз был занят <a class="mw-redirect" href="/wiki/%D0%9A%D1%80%D0%B0%D1%81%D0%BD%D0%B0%D1%8F_%D0%B0%D1%80%D0%BC%D0%B8%D1%8F" title="Красная армия">Красной Армией</a>, вслед за уходом которой покинули город почти все белзские евреи.<sup class="reference" id="cite_ref-vitk_9-2"><a href="#cite_note-vitk-9">[9]</a></sup> Еврейское население окрестных сёл было практически полностью уничтожено во время <a href="/wiki/%D0%A5%D0%BE%D0%BB%D0%BE%D0%BA%D0%BE%D1%81%D1%82" title="Холокост">Холокоста</a>.
</p><p>После войны в город вернулись всего две сотни евреев-белжан. После войны они эмигрировали из Советского Союза как граждане Польши, а затем в 1946 году большинство из них покинуло и Польшу<sup class="reference" id="cite_ref-vitk_9-3"><a href="#cite_note-vitk-9">[9]</a></sup>. Центр хасидизма белзского направления переместился в 1944 году в <a href="/wiki/%D0%A2%D0%B5%D0%BB%D1%8C-%D0%90%D0%B2%D0%B8%D0%B2" title="Тель-Авив">Тель-Авив</a> (Арон Рокеах)<sup class="reference" id="cite_ref-vitk_9-4"><a href="#cite_note-vitk-9">[9]</a></sup>, а потом в <a href="/wiki/%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC" title="Иерусалим">Иерусалим</a> (Иссохар Дов Рокеах (II) (с 1957, пятый белзский ребе).
</p><p>Паломники-хасиды посещают могилы белзских раввинов, почитаемых в качестве чудотворцев. Существует поверие, что приход мессии начнётся с белзской синагоги<sup class="reference" id="cite_ref-loda_15-1"><a href="#cite_note-loda-15">[15]</a></sup>.
</p><p>По одной из версий, известная еврейская мелодия «<a class="new" href="/w/index.php?title=%D0%9C%D0%B0%D0%B9%D0%BD_%D1%88%D1%82%D0%B5%D1%82%D0%B5%D0%BB%D0%B5_%D0%91%D1%8D%D0%BB%D1%86&amp;action=edit&amp;redlink=1" title="Майн штетеле Бэлц (страница отсутствует)">Майн штетеле Бэлц</a>» (Mein shtetele Belz) была придумана именно про данное местечко<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup><sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup>.
</p>

<h2><span id=".D0.A2.D1.80.D0.B0.D0.BD.D1.81.D0.BF.D0.BE.D1.80.D1.82.D0.BD.D0.BE.D0.B5_.D1.81.D0.BE.D0.BE.D0.B1.D1.89.D0.B5.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Транспортное_сообщение">Транспортное сообщение</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;veaction=edit&amp;section=16" title="Редактировать раздел «Транспортное сообщение»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;action=edit&amp;section=16" title="Редактировать код раздела «Транспортное сообщение»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Транспортное сообщение Белза с другими населёнными пунктами обслуживают железнодорожная станция Белз принадлежащая <a class="mw-redirect" href="/wiki/%D0%A3%D0%BA%D1%80%D0%B7%D0%B0%D0%BB%D0%B8%D0%B7%D0%BD%D1%8B%D1%86%D1%8F" title="Укрзализныця">Укрзализныце</a>, есть также автобусное сообщение. <a href="/wiki/%D0%94%D0%B8%D0%B7%D0%B5%D0%BB%D1%8C-%D0%BF%D0%BE%D0%B5%D0%B7%D0%B4" title="Дизель-поезд">Дизель-поездами</a> Белз связывается с <a href="/wiki/%D0%A0%D0%B0%D0%B2%D0%B0-%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%B0%D1%8F" title="Рава-Русская">Рава-Русской</a> и <a href="/wiki/%D0%A1%D0%BE%D0%BA%D0%B0%D0%BB%D1%8C" title="Сокаль">Сокалем</a>. Автобусы и маршрутные такси ходят во <a href="/wiki/%D0%9B%D1%8C%D0%B2%D0%BE%D0%B2" title="Львов">Львов</a>, <a href="/wiki/%D0%A7%D0%B5%D1%80%D0%B2%D0%BE%D0%BD%D0%BE%D0%B3%D1%80%D0%B0%D0%B4" title="Червоноград">Червоноград</a>, Сокаль, Варяж, Раву-Русскую, <a href="/wiki/%D0%A3%D0%B3%D0%BD%D0%B5%D0%B2" title="Угнев">Угнев</a>, сёла Каров, Хливчаны, Домашев, Диброва и другие.
</p><p>Добраться в Белз из Львова можно на <a class="mw-redirect" href="/wiki/%D0%9C%D0%B0%D1%80%D1%88%D1%80%D1%83%D1%82%D0%BA%D0%B0" title="Маршрутка">маршрутке</a> «Львов-Варяж», которая отправляется от автобусной станции № 2 (ул. Б. Хмельницкого, 225),<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup> а из Равы-Русской или Сокаля на дизель-поезде.<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup>
</p>
<h2><span id=".D0.94.D0.BE.D0.BF.D0.BE.D0.BB.D0.BD.D0.B8.D1.82.D0.B5.D0.BB.D1.8C.D0.BD.D1.8B.D0.B5_.D1.84.D0.B0.D0.BA.D1.82.D1.8B"></span><span class="mw-headline" id="Дополнительные_факты">Дополнительные факты</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;veaction=edit&amp;section=17" title="Редактировать раздел «Дополнительные факты»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%BB%D0%B7&amp;action=edit&amp;section=17" title="Редактировать код раздела «Дополнительные факты»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<ul><li>Из околиц Белза происходит и первый святой <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%A0%D1%83%D1%81%D1%8C" title="Московская Русь">Московской Руси</a>, первый <a href="/wiki/%D0%9C%D0%B8%D1%82%D1%80%D0%BE%D0%BF%D0%BE%D0%BB%D0%B8%D1%82" title="Митрополит">митрополит</a> Московский <a class="mw-redirect" href="/wiki/%D0%9F%D0%B5%D1%82%D1%80_(%D0%BC%D0%B8%D1%82%D1%80%D0%BE%D0%BF%D0%BE%D0%BB%D0%B8%D1%82_%D0%9A%D0%B8%D0%B5%D0%B2%D1%81%D0%BA%D0%B8%D0%B9)" title="Петр (митрополит Киевский)">Пётр Ратенский</a> (умер в 1353 году)<sup class="reference" id="cite_ref-loda_15-2"><a href="#cite_note-loda-15">[15]</a></sup>.</li>
<li>После <a class="mw-redirect" href="/wiki/%D0%A1%D0%BE%D0%B2%D0%B5%D1%82%D1%81%D0%BA%D0%BE-%D0%BF%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D0%BE%D0%B1%D0%BC%D0%B5%D0%BD_%D1%83%D1%87%D0%B0%D1%81%D1%82%D0%BA%D0%B0%D0%BC%D0%B8_%D1%82%D0%B5%D1%80%D1%80%D0%B8%D1%82%D0%BE%D1%80%D0%B8%D0%B9_1951_%D0%B3%D0%BE%D0%B4%D0%B0" title="Советско-польский обмен участками территорий 1951 года">передачи Белза из состава Польши в состав СССР</a> (<a class="mw-redirect" href="/wiki/%D0%A3%D0%A1%D0%A1%D0%A0" title="УССР">УССР</a>), в нём практически не было жителей. Последний коренной ныне живущий житель города, родившийся в довоенном Белзе, — Мария Мавко (1925 года рождения)<sup class="reference" id="cite_ref-kmu_16-1"><a href="#cite_note-kmu-16">[16]</a></sup>.</li>
<li>По утверждению <a href="/wiki/%D0%91%D1%8D%D0%BB%D0%B7%D0%B0,_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D1%81%D0%BB%D0%B0%D0%B2_%D0%98%D0%B3%D0%BE%D1%80%D0%B5%D0%B2%D0%B8%D1%87" title="Бэлза, Святослав Игоревич">С. И. Бэлза</a>, его фамилия происходит от названия городка<sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup>.</li></ul>
