<p><b>«Странная война», «Сидячая война»</b> (<a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Французский язык">фр.</a> <span lang="fr" style="font-style:italic;">Drôle de guerre</span>, <a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Phoney War</span>, <a href="/wiki/%D0%9D%D0%B5%D0%BC%D0%B5%D1%86%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Немецкий язык">нем.</a> <span lang="de" style="font-style:italic;">Sitzkrieg</span>) — период <a href="/wiki/%D0%92%D1%82%D0%BE%D1%80%D0%B0%D1%8F_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Вторая мировая война">Второй мировой войны</a> с <a href="/wiki/3_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="3 сентября">3 сентября</a> <a class="mw-redirect" href="/wiki/1939" title="1939">1939</a> по <a href="/wiki/10_%D0%BC%D0%B0%D1%8F" title="10 мая">10 мая</a> <a href="/wiki/1940_%D0%B3%D0%BE%D0%B4" title="1940 год">1940 года</a> на <a href="/wiki/%D0%97%D0%B0%D0%BF%D0%B0%D0%B4%D0%BD%D0%BE%D0%B5%D0%B2%D1%80%D0%BE%D0%BF%D0%B5%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%82%D0%B5%D0%B0%D1%82%D1%80_%D0%B2%D0%BE%D0%B5%D0%BD%D0%BD%D1%8B%D1%85_%D0%B4%D0%B5%D0%B9%D1%81%D1%82%D0%B2%D0%B8%D0%B9_%D0%92%D1%82%D0%BE%D1%80%D0%BE%D0%B9_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%BE%D0%B9_%D0%B2%D0%BE%D0%B9%D0%BD%D1%8B" title="Западноевропейский театр военных действий Второй мировой войны">Западном фронте</a>.
</p><p>Впервые название <i>Phoney War</i> (с <span style="font-style:normal"><a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a></span><span class="nowrap"> —</span> «фальшивая, ненастоящая война») было использовано американскими журналистами в <a href="/wiki/1939_%D0%B3%D0%BE%D0%B4" title="1939 год">1939 году</a><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>. Авторство французской версии <i>Drôle de guerre</i> (с <span style="font-style:normal"><a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Французский язык">фр.</a></span><span class="nowrap"> —</span> «странная война») принадлежит перу французского журналиста <a href="/wiki/%D0%94%D0%BE%D1%80%D0%B6%D0%B5%D0%BB%D0%B5%D1%81,_%D0%A0%D0%BE%D0%BB%D0%B0%D0%BD" title="Доржелес, Ролан">Ролана Доржелеса</a><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>. Таким образом был подчёркнут характер <a href="/wiki/%D0%91%D0%BE%D0%B5%D0%B2%D1%8B%D0%B5_%D0%B4%D0%B5%D0%B9%D1%81%D1%82%D0%B2%D0%B8%D1%8F" title="Боевые действия">боевых действий</a> между враждующими сторонами — почти полное их отсутствие, за исключением боевых действий на море. Враждующие стороны вели только бои локального значения на франко-немецкой границе, в основном находясь под защитой <a class="mw-redirect" href="/wiki/%D0%9E%D0%B1%D0%BE%D1%80%D0%BE%D0%BD%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D0%BB%D0%B8%D0%BD%D0%B8%D1%8F" title="Оборонительная линия">оборонительных линий</a> <a href="/wiki/%D0%9B%D0%B8%D0%BD%D0%B8%D1%8F_%D0%9C%D0%B0%D0%B6%D0%B8%D0%BD%D0%BE" title="Линия Мажино">Мажино</a> и <a href="/wiki/%D0%9B%D0%B8%D0%BD%D0%B8%D1%8F_%D0%97%D0%B8%D0%B3%D1%84%D1%80%D0%B8%D0%B4%D0%B0" title="Линия Зигфрида">Зигфрида</a>. Изредка западные союзники вели бомбардировки промышленных центров Германии. Период «Странной войны» в полной мере был использован немецким командованием в качестве стратегической паузы: Германии удалось успешно провести <a href="/wiki/%D0%9F%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B0%D1%8F_%D0%BA%D0%B0%D0%BC%D0%BF%D0%B0%D0%BD%D0%B8%D1%8F_%D0%B2%D0%B5%D1%80%D0%BC%D0%B0%D1%85%D1%82%D0%B0_(1939)" title="Польская кампания вермахта (1939)">Польскую кампанию</a>, <a class="mw-redirect" href="/wiki/%D0%94%D0%B0%D1%82%D1%81%D0%BA%D0%BE-%D0%9D%D0%BE%D1%80%D0%B2%D0%B5%D0%B6%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F" title="Датско-Норвежская операция">захватить Данию и Норвегию</a>, а также подготовиться к <a href="/wiki/%D0%9F%D0%BB%D0%B0%D0%BD_%C2%AB%D0%93%D0%B5%D0%BB%D1%8C%D0%B1%C2%BB" title="План «Гельб»">вторжению во Францию</a>.
</p>

<h2><span id=".D0.9F.D1.80.D0.B5.D0.B4.D0.BF.D0.BE.D1.81.D1.8B.D0.BB.D0.BA.D0.B8"></span><span class="mw-headline" id="Предпосылки">Предпосылки</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Предпосылки»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;action=edit&amp;section=1" title="Редактировать код раздела «Предпосылки»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>После <a class="mw-redirect" href="/wiki/%D0%9F%D1%80%D0%B8%D1%85%D0%BE%D0%B4_%D0%BA_%D0%B2%D0%BB%D0%B0%D1%81%D1%82%D0%B8_%D0%B2_%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%B8%D0%B8_%D0%BD%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%B0%D0%BB-%D1%81%D0%BE%D1%86%D0%B8%D0%B0%D0%BB%D0%B8%D1%81%D1%82%D0%BE%D0%B2" title="Приход к власти в Германии национал-социалистов">прихода к власти</a> Адольф Гитлер начал реализовывать идею по объединению всех земель с проживающими там немцами в единое государство. Опираясь на <a class="mw-redirect" href="/wiki/%D0%92%D0%BE%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%BC%D0%BE%D1%89%D1%8C" title="Военная мощь">военную мощь</a> и дипломатическое давление, в марте <a href="/wiki/1938_%D0%B3%D0%BE%D0%B4" title="1938 год">1938 года</a> Германия беспрепятственно провела <a class="mw-redirect" href="/wiki/%D0%90%D0%BD%D1%88%D0%BB%D1%8E%D1%81_%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B8%D0%B8" title="Аншлюс Австрии">аншлюс Австрии</a>.
В сентябре того же года заключение <a href="/wiki/%D0%9C%D1%8E%D0%BD%D1%85%D0%B5%D0%BD%D1%81%D0%BA%D0%BE%D0%B5_%D1%81%D0%BE%D0%B3%D0%BB%D0%B0%D1%88%D0%B5%D0%BD%D0%B8%D0%B5" title="Мюнхенское соглашение">Мюнхенского соглашения</a> привело к <a class="mw-redirect" href="/wiki/%D0%A0%D0%B0%D0%B7%D0%B4%D0%B5%D0%BB_%D0%A7%D0%B5%D1%85%D0%BE%D1%81%D0%BB%D0%BE%D0%B2%D0%B0%D0%BA%D0%B8%D0%B8" title="Раздел Чехословакии">разделу Чехословакии</a> между Германией, Польшей и Венгрией<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>.
</p><p><a href="/wiki/21_%D0%BC%D0%B0%D1%80%D1%82%D0%B0" title="21 марта">21 марта</a> 1939 года Германия потребовала присоединения города <a class="mw-redirect" href="/wiki/%D0%94%D0%B0%D0%BD%D1%86%D0%B8%D0%B3" title="Данциг">Данциг</a> (современный Гданьск), находившегося под управлением Лиги наций, и открытия «<a href="/wiki/%D0%9F%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%BE%D1%80%D0%B8%D0%B4%D0%BE%D1%80" title="Польский коридор">польского коридора</a>» (созданного после <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B2%D0%B0%D1%8F_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Первая мировая война">Первой мировой войны</a> для обеспечения выхода Польши к <a href="/wiki/%D0%91%D0%B0%D0%BB%D1%82%D0%B8%D0%B9%D1%81%D0%BA%D0%BE%D0%B5_%D0%BC%D0%BE%D1%80%D0%B5" title="Балтийское море">Балтийскому морю</a>). Польша отказалась выполнить требования Германии. В ответ <a href="/wiki/28_%D0%BC%D0%B0%D1%80%D1%82%D0%B0" title="28 марта">28 марта</a> 1939 года Гитлер объявил о недействительности <a class="mw-redirect" href="/wiki/%D0%94%D0%BE%D0%B3%D0%BE%D0%B2%D0%BE%D1%80_%D0%BE_%D0%BD%D0%B5%D0%BD%D0%B0%D0%BF%D0%B0%D0%B4%D0%B5%D0%BD%D0%B8%D0%B8_%D0%BC%D0%B5%D0%B6%D0%B4%D1%83_%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%B8%D0%B5%D0%B9_%D0%B8_%D0%9F%D0%BE%D0%BB%D1%8C%D1%88%D0%B5%D0%B9" title="Договор о ненападении между Германией и Польшей">Пакта о ненападении с Польшей</a> (подписан в январе 1934 года).
</p><p><a href="/wiki/31_%D0%BC%D0%B0%D1%80%D1%82%D0%B0" title="31 марта">31 марта</a> 1939 года <a href="/wiki/%D0%9F%D1%80%D0%B5%D0%BC%D1%8C%D0%B5%D1%80-%D0%BC%D0%B8%D0%BD%D0%B8%D1%81%D1%82%D1%80_%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%BE%D0%B1%D1%80%D0%B8%D1%82%D0%B0%D0%BD%D0%B8%D0%B8" title="Премьер-министр Великобритании">премьер-министр Великобритании</a> <a href="/wiki/%D0%A7%D0%B5%D0%BC%D0%B1%D0%B5%D1%80%D0%BB%D0%B5%D0%BD,_%D0%9D%D0%B5%D0%B2%D0%B8%D0%BB%D0%BB" title="Чемберлен, Невилл">Чемберлен</a> от имени английского и французского правительств заявил, что будет предоставлять всю возможную помощь Польше, если её безопасности будет что-то угрожать. Односторонняя английская гарантия Польше <a href="/wiki/6_%D0%B0%D0%BF%D1%80%D0%B5%D0%BB%D1%8F" title="6 апреля">6 апреля</a> была заменена двухсторонним соглашением о взаимопомощи между Англией и Польшей.
</p><p><a href="/wiki/15_%D0%BC%D0%B0%D1%8F" title="15 мая">15 мая</a> 1939 года был подписан польско-французский протокол, по которому французы обещали начать наступление в течение следующих двух недель после <a href="/wiki/%D0%9C%D0%BE%D0%B1%D0%B8%D0%BB%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F" title="Мобилизация">мобилизации</a>.
</p><p><a href="/wiki/25_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="25 августа">25 августа</a> 1939 года в Лондоне окончательно был оформлен и подписан англо-польский союз в виде <i>Соглашения о взаимопомощи и секретного договора</i>.
</p><p>Статья первая англо-польского Соглашения о взаимопомощи гласила:
</p>
<table align="center">
<tbody><tr>
<td><blockquote><p><i>«W razie gdyby jedna ze stron umawiających się znalazła się w działaniach wojennych w stosunku do jednego z mocarstw europejskich na skutek agresji tego ostatniego przeciwko tejże stronie umawiającej się, druga strona umawiająca się udzieli bezzwłocznie stronie umawiającej się znajdującej się w działaniach wojennych wszelkiej pomocy i poparcia będących w jej mocy.»</i></p></blockquote>
</td>
<td><blockquote><p><i>«В случае если одна из Сторон Договора будет втянута в боевые действия с европейским государством агрессией, устроенной последним против указанной Стороны Договора, другая Сторона Договора немедленно предоставит Стороне Договора, втянутой в боевые действия, всю поддержку и помощь, которую будет в состоянии оказать».</i><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></p></blockquote>
</td></tr></tbody></table>
<p>Под «европейским государством», как следовало из секретного договора, имелась в виду Германия.
</p><p><a href="/wiki/1_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="1 сентября">1 сентября</a> 1939 года <a href="/wiki/%D0%9F%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B0%D1%8F_%D0%BA%D0%B0%D0%BC%D0%BF%D0%B0%D0%BD%D0%B8%D1%8F_%D0%B2%D0%B5%D1%80%D0%BC%D0%B0%D1%85%D1%82%D0%B0_(1939)" title="Польская кампания вермахта (1939)">немецкие войска пересекли границу с Польшей</a>. В соответствии с договорённостями в тот же день во Франции была объявлена <a href="/wiki/%D0%9C%D0%BE%D0%B1%D0%B8%D0%BB%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F" title="Мобилизация">мобилизация</a>. Несколько позже, 17 сентября 1939 года, с Востока в пределы Польши вторглись войска <a class="mw-redirect" href="/wiki/%D0%A0%D0%9A%D0%9A%D0%90" title="РККА">РККА</a> <a class="mw-redirect" href="/wiki/%D0%A1%D0%A1%D0%A1%D0%A0" title="СССР">СССР</a>.
</p>

<h2><span id=".D0.A1.D0.B8.D0.BB.D1.8B_.D1.81.D1.82.D0.BE.D1.80.D0.BE.D0.BD"></span><span class="mw-headline" id="Силы_сторон">Силы сторон</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Силы сторон»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;action=edit&amp;section=2" title="Редактировать код раздела «Силы сторон»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>По своему потенциалу Великобритания и Франция значительно превосходили Германию. Население Германии (включая Австрию и Судеты) насчитывало 79,4 млн человек, а население Британской и Французской <a href="/wiki/%D0%9A%D0%BE%D0%BB%D0%BE%D0%BD%D0%B8%D0%B0%D0%BB%D0%B8%D0%B7%D0%BC" title="Колониализм">колониальных империй</a> — соответственно 560 млн и 110 млн человек (из них в метрополиях — 47,5 и 42 млн человек).
</p><p>В 1939 году в Германии было добыто 284 млн т угля, в Британской метрополии — 235 млн и во Французской — 49,8 млн т; выплавлено чугуна соответственно 19,8 млн, 8,1 млн и 7,4 млн т; произведено
стали 25,6 млн, 13,4 млн и 7,9 млн т, общее производство империй было ещё больше. Накануне войны Германия резко увеличила выпуск военной продукции, стоимость которой примерно в 3,4 раза превысила стоимость английской военной продукции, но это было преимущество чисто временного характера. Даже одна только Британская империя имела гораздо больше потенциальных ресурсов, чем Германия.
</p><p>Британская империя почти монопольно владела важнейшим стратегическим сырьём: оловом, каучуком, вольфрамом, молибденом, джутом — и имела доступ ко всем необходимым ей видам сырья. Германия же зависела от ввоза. Предпринятые ею попытки добиться самообеспечения не дали ожидаемых результатов.
</p>
<h3><span id=".D0.90.D0.B2.D0.B8.D0.B0.D1.86.D0.B8.D1.8F"></span><span class="mw-headline" id="Авиация">Авиация</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Авиация»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;action=edit&amp;section=3" title="Редактировать код раздела «Авиация»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>На момент объявления войны континентальная Франция располагала 34 дивизиями сухопутных войск, а также крупными военно-воздушными силами. ВВС Франции включали в себя около 3300 самолётов, 1275 из которых являлись новейшими боевыми машинами<sup class="ts-fix-template noprint">[<i><a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%A1%D1%81%D1%8B%D0%BB%D0%BA%D0%B8_%D0%BD%D0%B0_%D0%B8%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8" title="Википедия:Ссылки на источники"><span title="нет источника (7 января 2011)">источник не указан 4800 дней</span></a></i>]</sup>:
</p>
<ul><li>700 истребителей «<a href="/wiki/Morane-Saulnier_MS.406" title="Morane-Saulnier MS.406">Моран-Солнье MS-406</a>», «<a href="/wiki/Dewoitine_D.500" title="Dewoitine D.500">Девуатин D.510</a>» и «<a href="/wiki/Bloch_MB.150" title="Bloch MB.150">Bloch MB.152</a>»</li>
<li>175 бомбардировщиков «<a href="/wiki/Bloch_MB.131" title="Bloch MB.131">Bloch MB.131</a>»</li>
<li>400 разведывательных самолётов «<a href="/wiki/Potez_630" title="Potez 630">Potez 637</a>»</li></ul>
<p>В то же время люфтваффе на Западном фронте располагали 1193 самолётами. Из них 568 истребителей, 421 бомбардировщик и 152 разведчика<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>. Таким образом, воздушное превосходство только одной Франции над Германией было очевидным. А с прибытием во Францию британских авиачастей это превосходство стало бы подавляющим. Королевские ВВС выделили в помощь союзникам более 1500 самых современных самолётов: истребителей <a href="/wiki/Supermarine_Spitfire" title="Supermarine Spitfire">Спитфайр</a> и <a href="/wiki/Hawker_Hurricane" title="Hawker Hurricane">Харрикейн</a>, бомбардировщиков <a href="/wiki/Fairey_Battle" title="Fairey Battle">Фэйри «Бэттл»</a>, <a href="/wiki/Bristol_Blenheim" title="Bristol Blenheim">Бристоль Бленхейм</a> и <a href="/wiki/Armstrong_Whitworth_Whitley" title="Armstrong Whitworth Whitley">Уитли</a>. Однако все эти самолёты находились на британских аэродромах, и переброска их во Францию требовала значительного времени.
</p><p>В целом, на 1939 год Франция располагала третьей по количеству танков и самолётов сухопутной армией в мире после <a class="mw-redirect" href="/wiki/%D0%A0%D0%9A%D0%9A%D0%90" title="РККА">РККА</a> и вермахта, а также четвёртым в мире военно-морским флотом<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> после <a href="/wiki/%D0%9A%D0%BE%D1%80%D0%BE%D0%BB%D0%B5%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%B2%D0%BE%D0%B5%D0%BD%D0%BD%D0%BE-%D0%BC%D0%BE%D1%80%D1%81%D0%BA%D0%BE%D0%B9_%D1%84%D0%BB%D0%BE%D1%82_%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%BE%D0%B1%D1%80%D0%B8%D1%82%D0%B0%D0%BD%D0%B8%D0%B8" title="Королевский военно-морской флот Великобритании">британского</a>, <a href="/wiki/%D0%92%D0%BE%D0%B5%D0%BD%D0%BD%D0%BE-%D0%BC%D0%BE%D1%80%D1%81%D0%BA%D0%B8%D0%B5_%D1%81%D0%B8%D0%BB%D1%8B_%D0%A1%D0%A8%D0%90" title="Военно-морские силы США">американского</a> и <a href="/wiki/%D0%98%D0%BC%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%BE%D1%80%D1%81%D0%BA%D0%B8%D0%B9_%D1%84%D0%BB%D0%BE%D1%82_%D0%AF%D0%BF%D0%BE%D0%BD%D0%B8%D0%B8" title="Императорский флот Японии">японского</a> (за Францией следовала Италия).
</p>
<h3><span id=".D0.9D.D0.B0.D0.B7.D0.B5.D0.BC.D0.BD.D1.8B.D0.B5_.D0.B2.D0.BE.D0.B9.D1.81.D0.BA.D0.B0"></span><span class="mw-headline" id="Наземные_войска">Наземные войска</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Наземные войска»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;action=edit&amp;section=4" title="Редактировать код раздела «Наземные войска»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span id=".D0.93.D0.B5.D1.80.D0.BC.D0.B0.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="Германия">Германия</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Германия»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;action=edit&amp;section=5" title="Редактировать код раздела «Германия»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h4>
<p><b><a class="mw-redirect" href="/wiki/%D0%93%D1%80%D1%83%D0%BF%D0%BF%D0%B0_%D0%B0%D1%80%D0%BC%D0%B8%D0%B9_%C2%ABC%C2%BB" title="Группа армий «C»">Группа армий «C»</a></b>
</p><p>Западный фронт вермахта представляла группа армий «Ц» генерал-полковника <a class="mw-redirect" href="/wiki/%D0%9B%D0%B5%D0%B5%D0%B1,_%D0%92%D0%B8%D0%BB%D1%8C%D0%B3%D0%B5%D0%BB%D1%8C%D0%BC_%D0%A0%D0%B8%D1%82%D1%82%D0%B5%D1%80_%D1%84%D0%BE%D0%BD" title="Лееб, Вильгельм Риттер фон">Вильгельма фон Лееба</a> в составе 42 дивизий (в сентябре в срочном порядке к ней на усиление была переброшена 3-я горнострелковая дивизия), из которых лишь 12 могли быть названы полноценными<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>:
</p><p><b>Первый эшелон (1-й и 2-й этапы мобилизации)</b>
</p>
<ul><li>5, 6, 9, 15, 16, 22, 25, 26, 33, 34, 35, 36, 52, 58, 69, 71, 75, 76, 78, 79, 86, 87, 209 пд</li></ul>
<p><b>Второй эшелон (4-й этап мобилизации)</b>
</p>
<ul><li>253, 254, 262, 269, 260, 263, 267, 268 пд</li></ul>
<p><b>Резерв (3-й этап мобилизации)</b>
</p>
<ul><li>211, 212, 214, 215, 216, 223, 225, 227, 231, 246, 251 пд</li></ul>
<p>Германские войска занимали позиции вдоль голландской, бельгийской и французской границ. При этом они использовали созданную ранее <a href="/wiki/%D0%9B%D0%B8%D0%BD%D0%B8%D1%8F_%D0%97%D0%B8%D0%B3%D1%84%D1%80%D0%B8%D0%B4%D0%B0" title="Линия Зигфрида">Линию Зигфрида</a>.
</p>
<h4><span id=".D0.A4.D1.80.D0.B0.D0.BD.D1.86.D0.B8.D1.8F"></span><span class="mw-headline" id="Франция">Франция</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Франция»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;action=edit&amp;section=6" title="Редактировать код раздела «Франция»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h4>
<p><b>2-я группа армий</b>
</p>
<dl><dd>3, 4, 5, 8-я армии
<dl><dd>11, 13, 42, 43-я пехотные дивизии, 4-я пехотная колониальная дивизия</dd>
<dd>9-я и 25-я моторизованные дивизии</dd>
<dd>2-я кавалерийская дивизия</dd>
<dd>2-я и 4-я дивизии тяжёлой артиллерии</dd></dl></dd></dl>
<p>До <a href="/wiki/12_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="12 сентября">12 сентября</a> французские силы возросли до 78 дивизий (в том числе 4-х моторизованных) и 18 отдельных танковых батальонов<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup>. Немцы же не располагали на тот момент ни одной танковой дивизией или моторизованной дивизией — все были задействованы в Польше.
</p>
<h2><span id=".D0.9D.D0.B0.D1.87.D0.B0.D0.BB.D0.BE_.D0.B2.D0.BE.D0.B9.D0.BD.D1.8B"></span><span class="mw-headline" id="Начало_войны">Начало войны</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Начало войны»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;action=edit&amp;section=7" title="Редактировать код раздела «Начало войны»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/3_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="3 сентября">3 сентября</a> 1939 года <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%BE%D0%B1%D1%80%D0%B8%D1%82%D0%B0%D0%BD%D0%B8%D1%8F" title="Великобритания">Великобритания</a> (в 5:00) и <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%8F" title="Франция">Франция</a> (в 11:00) объявили войну Германии. Уже постфактум <a href="/wiki/4_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="4 сентября">4 сентября</a> было подписано франко-польское соглашение. После этого посол Польши во Франции стал настаивать на немедленном общем наступлении. В тот же день представители Великобритании, начальник имперского генерального штаба генерал <a class="mw-redirect" href="/wiki/%D0%90%D0%B9%D1%80%D0%BE%D0%BD%D1%81%D0%B0%D0%B9%D0%B4,_%D0%AD%D0%B4%D0%BC%D1%83%D0%BD%D0%B4_%D0%A3%D0%B8%D0%BB%D1%8C%D1%8F%D0%BC" title="Айронсайд, Эдмунд Уильям">Эдмунд Уильям Айронсайд</a> и главный маршал авиации <a href="/wiki/%D0%9D%D1%8C%D1%8E%D0%BE%D0%BB%D0%BB,_%D0%A1%D0%B8%D1%80%D0%B8%D0%BB" title="Ньюолл, Сирил">Сирил Ньюэлл</a> прибыли во Францию для проведения переговоров с французским генеральным штабом. Несмотря на многочисленные прошедшие встречи объединённого комитета штабов, которые начались с конца марта, к началу <a href="/wiki/%D0%A1%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8C" title="Сентябрь">сентября</a> ещё не было скоординированного плана действий по предоставлению помощи полякам.
</p>
<figure class="mw-default-size mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Westwall03.jpg"><img class="mw-file-element" data-file-height="679" data-file-width="992" decoding="async" height="151" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/aa/Westwall03.jpg/220px-Westwall03.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/aa/Westwall03.jpg/330px-Westwall03.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/aa/Westwall03.jpg/440px-Westwall03.jpg 2x" width="220"/></a><figcaption><a href="/wiki/%D0%9B%D0%B8%D0%BD%D0%B8%D1%8F_%D0%97%D0%B8%D0%B3%D1%84%D1%80%D0%B8%D0%B4%D0%B0" title="Линия Зигфрида">Линия Зигфрида</a> — <a class="mw-redirect" href="/wiki/%D0%9F%D1%80%D0%BE%D1%82%D0%B8%D0%B2%D0%BE%D1%82%D0%B0%D0%BD%D0%BA%D0%BE%D0%B2%D1%8B%D0%B5_%D0%BD%D0%B0%D0%B4%D0%BE%D0%BB%D0%B1%D1%8B" title="Противотанковые надолбы">противотанковые надолбы</a> на линии <a href="/wiki/%D0%90%D1%85%D0%B5%D0%BD" title="Ахен">Ахен</a>-<a href="/wiki/%D0%A1%D0%B0%D0%B0%D1%80%D0%B1%D1%80%D1%8E%D0%BA%D0%BA%D0%B5%D0%BD" title="Саарбрюккен">Саарбрюккен</a></figcaption></figure><figure class="mw-default-size mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:EntreeMunitionsKobenbusch.jpg"><img class="mw-file-element" data-file-height="404" data-file-width="600" decoding="async" height="148" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/e4/EntreeMunitionsKobenbusch.jpg/220px-EntreeMunitionsKobenbusch.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/e4/EntreeMunitionsKobenbusch.jpg/330px-EntreeMunitionsKobenbusch.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/e4/EntreeMunitionsKobenbusch.jpg/440px-EntreeMunitionsKobenbusch.jpg 2x" width="220"/></a><figcaption>Форт на линии Мажино</figcaption></figure>
<p>На следующий день Айронсайд и Ньюэлл доложили в кабинет министров, что после завершения мобилизации своих армий главнокомандующий французской армией <a href="/wiki/%D0%93%D0%B0%D0%BC%D0%B5%D0%BB%D0%B5%D0%BD,_%D0%9C%D0%BE%D1%80%D0%B8%D1%81_%D0%93%D1%8E%D1%81%D1%82%D0%B0%D0%B2" title="Гамелен, Морис Гюстав">Гамелен</a> собирался ориентировочно <a href="/wiki/17_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="17 сентября">17 сентября</a> <i>«нажать на линию Зигфрида»</i> и проверить надёжность её обороны.
</p><p>В результате подготовительных мероприятий с 18 августа и скрытой мобилизации с 25 августа немецкое командование развернуло на Западе группу армий «Ц» в составе 31 2/3 дивизий. Ещё до 1 сентября из резерва ОКХ в ГА «Ц» было передано 3 дивизии и ещё 9 после объявления союзниками войны Германии. Всего к 10 сентября на западных границах Германии имелось 43 2/3 дивизии<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup><sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup>. Воздушную поддержку им оказывали 2-й и 3-й воздушные флоты, имевшие 664 и 564 боевых самолётов соответственно<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup>. Французские мобилизационные мероприятия начались 21 августа и затронули в первую очередь дивизии мирного времени и крепостные и противовоздушные части. 1 сентября была объявлена всеобщая мобилизация (первый день 2 сентября с 0 часов) и началось формирование резервных дивизий серий «А» и «Б» (кроме двух, которые начали формироваться в конце августа)<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup><sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup>. После завершения мобилизации и развёртывания к началу 20-х чисел сентября в составе Северо-Восточного фронта, прикрывавшего границу с Бельгией и Германией, было сосредоточено 61 дивизия и 1 бригада, против Италии — 11 дивизий и 1 бригада, в Северной Африке (<a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D0%90%D0%BB%D0%B6%D0%B8%D1%80" title="Французский Алжир">Алжир</a>, <a class="mw-redirect" href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%BE%D0%B5_%D0%9C%D0%B0%D1%80%D0%BE%D0%BA%D0%BA%D0%BE" title="Французское Марокко">Марокко</a> и <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D1%80%D0%BE%D1%82%D0%B5%D0%BA%D1%82%D0%BE%D1%80%D0%B0%D1%82_%D0%A2%D1%83%D0%BD%D0%B8%D1%81" title="Французский протекторат Тунис">Тунис</a>) имелось 14 дивизий и 5 бригад<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup>. Четыре английские дивизии прибывали во Францию в течение всего сентября и к середине октября сосредоточились на бельгийской границе в районе Арраса между 1-й и 7-й французскими армиями<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup>. Длина северной границы Франции составляла 804,67 км, наступление французы могли вести только на небольшой территории шириной 144,84 км от <a href="/wiki/%D0%A0%D0%B5%D0%B9%D0%BD" title="Рейн">Рейна</a> до <a href="/wiki/%D0%9C%D0%BE%D0%B7%D0%B5%D0%BB%D1%8C" title="Мозель">Мозеля</a>. В другом случае Франция нарушила бы <a href="/wiki/%D0%9D%D0%B5%D0%B9%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D1%82%D0%B5%D1%82" title="Нейтралитет">нейтралитет</a> Бельгии и Люксембурга. Немцы смогли сосредоточить самые боеспособные дивизии именно на этой территории и прикрыли подходы к линии Зигфрида <a href="/wiki/%D0%9D%D0%B0%D0%B7%D0%B5%D0%BC%D0%BD%D0%B0%D1%8F_%D0%BC%D0%B8%D0%BD%D0%B0" title="Наземная мина">минными полями</a>. В такой ситуации наступательные действия французов значительно усложнялись.
</p><p>Однако важнее оказалось то, что французы не смогли начать наступательную операцию до 17 сентября. До этого времени франко-немецкое противостояние ограничивалось только боями локального значения. Неспособность Франции раньше ударить по немцам объяснялась устаревшей <a href="/wiki/%D0%9C%D0%BE%D0%B1%D0%B8%D0%BB%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F" title="Мобилизация">мобилизационной</a> системой: сформированные части не успевали пройти надлежащей подготовки. Другой причиной задержки было то, что французское командование придерживалось устаревших взглядов на ведение войны, считая, что перед любым наступлением, как и во времена Первой мировой войны, должна пройти мощная <a href="/wiki/%D0%90%D1%80%D1%82%D0%BF%D0%BE%D0%B4%D0%B3%D0%BE%D1%82%D0%BE%D0%B2%D0%BA%D0%B0" title="Артподготовка">артподготовка</a><sup class="ts-fix-template noprint">[<i><a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%A1%D1%81%D1%8B%D0%BB%D0%BA%D0%B8_%D0%BD%D0%B0_%D0%B8%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8" title="Википедия:Ссылки на источники"><span title="нет источника (14 октября 2012)">источник не указан 4154 дня</span></a></i>]</sup>. Однако большая часть тяжёлой артиллерии французов пребывала в консервации, и её нельзя было подготовить раньше, чем к пятнадцатому дню после объявления мобилизации.
</p><p>Относительно помощи Великобритании было ясно, что первые две дивизии <a href="/wiki/%D0%91%D1%80%D0%B8%D1%82%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B5_%D1%8D%D0%BA%D1%81%D0%BF%D0%B5%D0%B4%D0%B8%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D1%8B%D0%B5_%D1%81%D0%B8%D0%BB%D1%8B_(%D0%92%D1%82%D0%BE%D1%80%D0%B0%D1%8F_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0)" title="Британские экспедиционные силы (Вторая мировая война)">английского экспедиционного корпуса</a> могли прибыть на континент только в первых числах октября, ещё две — во второй половине октября. На другие английские дивизии рассчитывать не приходилось. Для французов это также служило поводом не начинать наступательных действий.
</p><p>Немецкая армия также не спешила начинать полномасштабную войну на Западном фронте. В «Распоряжении главнокомандующего военными силами <a href="/wiki/%D0%93%D0%B8%D1%82%D0%BB%D0%B5%D1%80,_%D0%90%D0%B4%D0%BE%D0%BB%D1%8C%D1%84" title="Гитлер, Адольф">Адольфа Гитлера</a> о нападении на Польшу (31.08.1939)» говорилось следующее:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
«3) На западе ответственность за начало войны стоит полностью возложить на англичан и французов. На незначительные нарушения границы нужно сначала ответить действиями чисто местного характера… <br/>
Немецкая сухопутная граница на западе не должна быть нарушена ни в одном пункте без моего разрешения. То же самое относится ко всем военно-морским операциям, а также к другим действиям на море, которые могут быть оценены как военные операции. <br/>
Действия военно-воздушных сил должны ограничиваться противовоздушной обороной государственных границ от налётов авиации противника…<br/>

Сухопутные силы удержат <a href="/wiki/%D0%9B%D0%B8%D0%BD%D0%B8%D1%8F_%D0%97%D0%B8%D0%B3%D1%84%D1%80%D0%B8%D0%B4%D0%B0" title="Линия Зигфрида">Западный вал</a> и готовятся к предотвращению его обхода с севера…»<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup>
</p>

<p>Для исполнения этого задания <a href="/wiki/%D0%93%D1%80%D1%83%D0%BF%D0%BF%D0%B0_%D0%B0%D1%80%D0%BC%D0%B8%D0%B9" title="Группа армий">группа армий</a> «Ц» под командованием генерал-полковника <a href="/wiki/%D0%9B%D0%B5%D0%B5%D0%B1,_%D0%92%D0%B8%D0%BB%D1%8C%D0%B3%D0%B5%D0%BB%D1%8C%D0%BC_%D1%84%D0%BE%D0%BD" title="Лееб, Вильгельм фон">Вильгельма фон Лееба</a> имела в своём распоряжении 11 2/3 кадровых и 32 резервных и ландверных дивизий. Последние нельзя было считать полностью боеспособными ни по технической оснащённости, ни по военной подготовке. <a href="/wiki/%D0%A2%D0%B0%D0%BD%D0%BA" title="Танк">Танковых</a> соединений группа армий «Запад» не имела. Западный вал (линия Зигфрида) значительно уступал в укреплённости <a href="/wiki/%D0%9B%D0%B8%D0%BD%D0%B8%D1%8F_%D0%9C%D0%B0%D0%B6%D0%B8%D0%BD%D0%BE" title="Линия Мажино">линии Мажино</a> и ещё строился. Немецкие войска были развёрнуты следующим образом: <a class="mw-redirect" href="/wiki/7-%D1%8F_%D0%B0%D1%80%D0%BC%D0%B8%D1%8F_(%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%B8%D1%8F)" title="7-я армия (Германия)">7-я армия</a> (командующий генерал артиллерии <a class="mw-redirect" href="/wiki/%D0%94%D0%BE%D0%BB%D0%BB%D1%8C%D0%BC%D0%B0%D0%BD,_%D0%A4%D1%80%D0%B8%D0%B4%D1%80%D0%B8%D1%85" title="Долльман, Фридрих">Долльман</a>) вдоль Рейна от <a href="/wiki/%D0%91%D0%B0%D0%B7%D0%B5%D0%BB%D1%8C" title="Базель">Базеля</a> до <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%BB%D1%81%D1%80%D1%83%D1%8D" title="Карлсруэ">Карлсруэ</a>, <a class="mw-redirect" href="/wiki/1-%D1%8F_%D0%B0%D1%80%D0%BC%D0%B8%D1%8F_(%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%B8%D1%8F)" title="1-я армия (Германия)">1-я армия</a> (командующий генерал-полковник <a href="/wiki/%D0%92%D0%B8%D1%86%D0%BB%D0%B5%D0%B1%D0%B5%D0%BD,_%D0%AD%D1%80%D0%B2%D0%B8%D0%BD_%D1%84%D0%BE%D0%BD" title="Вицлебен, Эрвин фон">Эрвин фон Вицлебен</a>) — от Рейна до границы с Люксембургом. Небольшая <a class="mw-disambig" href="/wiki/%D0%9E%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D0%B0%D1%8F_%D0%B3%D1%80%D1%83%D0%BF%D0%BF%D0%B0" title="Оперативная группа">оперативная группа</a> «А» под командованием генерал-полковника барона <a href="/wiki/%D0%A5%D0%B0%D0%BC%D0%BC%D0%B5%D1%80%D1%88%D1%82%D0%B5%D0%B9%D0%BD-%D0%AD%D0%BA%D0%B2%D0%BE%D1%80%D0%B4,_%D0%9A%D1%83%D1%80%D1%82_%D1%84%D0%BE%D0%BD" title="Хаммерштейн-Экворд, Курт фон">Курта фон Хаммерштейна</a> охраняла границу с нейтральными государствами до города <a href="/wiki/%D0%92%D0%B5%D0%B7%D0%B5%D0%BB%D1%8C" title="Везель">Везель</a><sup class="ts-fix-template noprint">[<i><a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%A1%D1%81%D1%8B%D0%BB%D0%BA%D0%B8_%D0%BD%D0%B0_%D0%B8%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8" title="Википедия:Ссылки на источники"><span title="нет источника (14 октября 2012)">источник не указан 4154 дня</span></a></i>]</sup>.
</p>
<h2><span id=".C2.AB.D0.90.D0.BA.D1.82.D0.B8.D0.B2.D0.BD.D1.8B.D0.B5_.D0.B4.D0.B5.D0.B9.D1.81.D1.82.D0.B2.D0.B8.D1.8F.C2.BB_.D0.BD.D0.B0_.D0.97.D0.B0.D0.BF.D0.B0.D0.B4.D0.BD.D0.BE.D0.BC_.D1.84.D1.80.D0.BE.D0.BD.D1.82.D0.B5"></span><span class="mw-headline" id="«Активные_действия»_на_Западном_фронте">«Активные действия» на Западном фронте</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;veaction=edit&amp;section=8" title="Редактировать раздел ««Активные действия» на Западном фронте»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;action=edit&amp;section=8" title="Редактировать код раздела ««Активные действия» на Западном фронте»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>

<p>С начала войны французы ограничились только несколькими атаками местного значения в районе Западного вала. При строительстве защитного барьера немцы не придерживались природной кривизны границ, поэтому линия в некоторых районах представляла собой прямую. Кроме того немецким войскам было приказано проводить только защиту линии Зигфрида и не вступать в затяжные боевые действия. <a href="/wiki/13_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="13 сентября">13 сентября</a> 1939 года французам удалось сравнительно легко занять два выступающих вперёд участка — участок «<a class="new" href="/w/index.php?title=%D0%92%D0%B0%D1%80%D0%BD%D0%B4%D1%82&amp;action=edit&amp;redlink=1" title="Варндт (страница отсутствует)">Варндт</a>» западнее <a href="/wiki/%D0%A1%D0%B0%D0%B0%D1%80%D0%B1%D1%80%D1%8E%D0%BA%D0%BA%D0%B5%D0%BD" title="Саарбрюккен">Саарбрюккена</a> и выступ границы между Саарбрюкеном и <a href="/wiki/%D0%9F%D1%84%D0%B0%D0%BB%D1%8C%D1%86%D1%81%D0%BA%D0%B8%D0%B9_%D0%9B%D0%B5%D1%81" title="Пфальцский Лес">Пфальцским лесом</a>.
</p><p>Когда после окончания войны с Польшей стала заметной передислокация немецких соединений с Восточного фронта на Западный, французы, начиная с 3 октября, освободили большую часть захваченной ими пограничной зоны и отошли к государственной границе, а местами и за неё. По свидетельству немецких военных, они были удивлены плохо подготовленными в инженерном отношении <a href="/wiki/%D0%A4%D0%BE%D1%80%D1%82%D0%B8%D1%84%D0%B8%D0%BA%D0%B0%D1%86%D0%B8%D1%8F#Полевая_фортификация" title="Фортификация">полевыми позициями</a>, которые покинули французы.
</p>
<h3><span id=".D0.A1.D0.B0.D0.B0.D1.80.D1.81.D0.BA.D0.BE.D0.B5_.D0.BD.D0.B0.D1.81.D1.82.D1.83.D0.BF.D0.BB.D0.B5.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Саарское_наступление">Саарское наступление</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;veaction=edit&amp;section=9" title="Редактировать раздел «Саарское наступление»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;action=edit&amp;section=9" title="Редактировать код раздела «Саарское наступление»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote navigation-not-searchable">Основная статья: <b><a href="/wiki/%D0%A1%D0%B0%D0%B0%D1%80%D1%81%D0%BA%D0%B0%D1%8F_%D0%BD%D0%B0%D1%81%D1%82%D1%83%D0%BF%D0%B0%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F" title="Саарская наступательная операция">Саарская наступательная операция</a></b></div>
<p>Согласно франко-польскому военному договору, обязательство французской армии заключалось в том, чтобы начать подготовку к крупному <a href="/wiki/%D0%9D%D0%B0%D1%81%D1%82%D1%83%D0%BF%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5" title="Наступление">наступлению</a> через 3 дня после начала <a href="/wiki/%D0%9C%D0%BE%D0%B1%D0%B8%D0%BB%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F" title="Мобилизация">мобилизации</a>. Французские войска должны были захватить район между французской границей и немецкой линией обороны и произвести разведку боем. На 15 день мобилизации (то есть по <a href="/wiki/16_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="16 сентября">16 сентября</a>) цель французской армии заключалась в том, чтобы начать полномасштабное наступление на Германию. Предварительная частичная мобилизация была начата во Франции уже <a href="/wiki/26_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="26 августа">26 августа</a>, а <a href="/wiki/1_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="1 сентября">1 сентября</a> была объявлена полномасштабная мобилизация.
</p><p><a href="/wiki/%D0%A1%D0%B0%D0%B0%D1%80%D1%81%D0%BA%D0%B0%D1%8F_%D0%BD%D0%B0%D1%81%D1%82%D1%83%D0%BF%D0%B0%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F" title="Саарская наступательная операция">Французское наступление в районе долины реки Рейн</a> началось <a href="/wiki/7_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="7 сентября">7 сентября</a>, через 4 дня после объявления Францией войны Германии. В этот момент силы вермахта были заняты в наступательной операции в Польше, и французы имели подавляющее численное превосходство вдоль границы с Германией. Однако действия французской армии не принесли облегчения полякам, да и сами французы оказались в опасном положении, не добившись крупных успехов. Так, близ Саарбрюккена сразу одиннадцать дивизий штурмовали позиции немцев, прорвавшись на 32 километра вперёд. Всего французам удалось взять за неделю 12 населённых пунктов. Однако немцы, без потерь сдав города, тем самым ввели французов в заблуждение, скапливая силы. Постепенно немцы стали контратаковать: <a href="/wiki/10_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="10 сентября">10 сентября</a> французы отбили первую атаку близ Апаха. Тем не менее, наступление продолжилось вплоть до захвата Варндтского леса. В этой операции пехота понесла тяжёлые потери от противопехотных мин, и французское наступление выдохлось. Французская армия даже не добралась до <a href="/wiki/%D0%9B%D0%B8%D0%BD%D0%B8%D1%8F_%D0%97%D0%B8%D0%B3%D1%84%D1%80%D0%B8%D0%B4%D0%B0" title="Линия Зигфрида">Западного вала</a>. <a href="/wiki/12_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="12 сентября">12 сентября</a> англо-французский верховный военный совет собрался в первый раз в <a href="/wiki/%D0%90%D0%B1%D0%B2%D0%B8%D0%BB%D1%8C" title="Абвиль">Аббевилле</a> во Франции. Было решено, что все наступательные действия должны были быть прекращены немедленно.
</p>
<figure class="mw-default-size" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Faury_Louis.jpg"><img class="mw-file-element" data-file-height="2560" data-file-width="1836" decoding="async" height="307" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/bf/Faury_Louis.jpg/220px-Faury_Louis.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/bf/Faury_Louis.jpg/330px-Faury_Louis.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/bf/Faury_Louis.jpg/440px-Faury_Louis.jpg 2x" width="220"/></a><figcaption><span data-interwiki-article="Louis Faury" data-interwiki-lang="fr"><a class="new" href="/w/index.php?title=%D0%A4%D0%BE%D1%80%D0%B8,_%D0%9B%D1%83%D0%B8&amp;action=edit&amp;redlink=1" title="Фори, Луи (страница отсутствует)">Луи Фори</a></span><span class="noprint" style="font-style:normal; font-weight:normal;"><sup><a class="extiw" href="https://fr.wikipedia.org/wiki/Louis_Faury" title="fr:Louis Faury"><span title="Louis Faury — версия статьи «Фори, Луи» на французском языке">[fr]</span></a></sup></span>, глава французской военной миссии в Польше</figcaption></figure>
<p>Операция не привела к передислокации немецких войск из Польши. Польша не была уведомлена о решении приостановить наступление. Вместо этого Гамелен сообщил маршалу <a href="/wiki/%D0%A0%D1%8B%D0%B4%D0%B7-%D0%A1%D0%BC%D0%B8%D0%B3%D0%BB%D1%8B,_%D0%AD%D0%B4%D0%B2%D0%B0%D1%80%D0%B4" title="Рыдз-Смиглы, Эдвард">Эдварду Рыдз-Смиглы</a>, что половина его дивизий вступили в бой с противником, и о том, что французские успехи вынудили вермахт вывести по крайней мере 6 дивизий из Польши. На следующий день командир французской военной миссии в Польше <span data-interwiki-article="Louis Faury" data-interwiki-lang="fr"><a class="new" href="/w/index.php?title=%D0%A4%D0%BE%D1%80%D0%B8,_%D0%9B%D1%83%D0%B8&amp;action=edit&amp;redlink=1" title="Фори, Луи (страница отсутствует)">Луи Фори</a></span><span class="noprint" style="font-style:normal; font-weight:normal;"><sup><a class="extiw" href="https://fr.wikipedia.org/wiki/Louis_Faury" title="fr:Louis Faury"><span title="Louis Faury — версия статьи «Фори, Луи» на французском языке">[fr]</span></a></sup></span> сообщил польскому начальнику штаба генералу <a href="/wiki/%D0%A1%D1%82%D0%B0%D1%85%D0%B5%D0%B2%D0%B8%D1%87,_%D0%92%D0%B0%D1%86%D0%BB%D0%B0%D0%B2" title="Стахевич, Вацлав">Вацлаву Стахевичу</a>, что планируемое полномасштабное наступление на западном фронте пришлось перенести с <a href="/wiki/17_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="17 сентября">17 сентября</a> на <a href="/wiki/20_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="20 сентября">20 сентября</a>. Запланированное полномасштабное наступление на Германию должно было осуществляться 40 дивизиями, в том числе одной бронетанковой дивизией, тремя механизированными дивизиями, 78 артиллерийскими <a href="/wiki/%D0%9F%D0%BE%D0%BB%D0%BA" title="Полк">полками</a> и 40 танковыми батальонами, однако в связи с безнадёжным положением Польши на 17 сентября оно было отменено.
</p><p>Контрнаступление немцев <a href="/wiki/16_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="16 октября">16</a> и <a href="/wiki/17_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="17 октября">17 октября</a> позволило Германии вернуть территории, потерянные во время Саарской операции. Французские войска вернулись за линию Мажино. Так началась Странная война.
</p>

</p>

<h3><span id=".D0.92.D0.B5.D0.BB.D0.B8.D0.BA.D0.BE.D0.B1.D1.80.D0.B8.D1.82.D0.B0.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="Великобритания">Великобритания</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;veaction=edit&amp;section=11" title="Редактировать раздел «Великобритания»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;action=edit&amp;section=11" title="Редактировать код раздела «Великобритания»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Англичане до середины октября четырьмя дивизиями (два <a href="/wiki/%D0%90%D1%80%D0%BC%D0%B5%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%BE%D1%80%D0%BF%D1%83%D1%81" title="Армейский корпус">армейских корпуса</a>) заняли позиции на бельгийско-французской границе между городами <span data-interwiki-article="Maulde" data-interwiki-lang="fr"><a class="new" href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D1%8C%D0%B4_(%D0%B3%D0%BE%D1%80%D0%BE%D0%B4)&amp;action=edit&amp;redlink=1" title="Мольд (город) (страница отсутствует)">Мольд</a></span><span class="noprint" style="font-style:normal; font-weight:normal;"><sup><a class="extiw" href="https://fr.wikipedia.org/wiki/Maulde" title="fr:Maulde"><span title="Maulde — версия статьи «Мольд (город)» на французском языке">[fr]</span></a></sup></span> и <a href="/wiki/%D0%91%D0%B0%D0%B9%D1%91%D0%BB%D1%8C" title="Байёль">Байёль</a>, достаточно далеко от линии фронта. В этом районе проходил почти сплошной <a href="/wiki/%D0%A0%D0%BE%D0%B2#Противотанковый_ров" title="Ров">противотанковый ров</a>, который прикрывался огнём <a href="/wiki/%D0%94%D0%BE%D0%BB%D0%B3%D0%BE%D0%B2%D1%80%D0%B5%D0%BC%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%BE%D0%B3%D0%BD%D0%B5%D0%B2%D0%B0%D1%8F_%D1%82%D0%BE%D1%87%D0%BA%D0%B0" title="Долговременная огневая точка">дотов</a>. Эта система укреплений строилась как продолжение линии Мажино на случай прорыва немецких войск через Бельгию.
</p><p><a href="/wiki/28_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="28 октября">28 октября</a> военный кабинет утвердил стратегическую концепцию Великобритании. Начальник британского генерального штаба генерал Эдмунд Айронсайд дал этой концепции такую характеристику: «пассивное ожидание со всеми волнениями и тревогами, которые из этого вытекают»<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup>.
</p><p>После этого на Западном фронте установилось полное затишье. Французский корреспондент <a class="mw-redirect" href="/wiki/%D0%A0%D0%BE%D0%BB%D0%B0%D0%BD_%D0%94%D0%BE%D1%80%D0%B6%D0%B5%D0%BB%D0%B5%D1%81" title="Ролан Доржелес">Ролан Доржелес</a>, который пребывал на линии фронта, писал:

</p>
<blockquote class="ts-Начало_цитаты-quote"><p>
…я был удивлён спокойствием, которое там царило. Артиллеристы, которые расположились на Рейне, спокойно смотрели на немецкие поезда с боеприпасами, которые курсируют на противоположном берегу, наши лётчики пролетали над дымящимися трубами завода Саара, не сбрасывая бомб. Очевидно, главная забота высшего командования заключалась в том, чтобы не беспокоить противника.</p><div class="ts-Конец_цитаты-source">— <cite><sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup></cite></div></blockquote>
<p>30 октября 1939 года британским истребителем впервые на Западном фронте был сбит немецкий самолёт-разведчик Do 17<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup>. 9 декабря 1939 года во время ночного патрулирования британский патруль зашёл на минное поле, и капрал Т. Прайдей стал первым британским пехотинцем, погибшим в боевых действиях во Второй мировой войне<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup> (но далеко не первой потерей Великобритании — в частности, более 800 моряков погибли при потоплении «<a href="/wiki/HMS_Royal_Oak_(1914)" title="HMS Royal Oak (1914)">Ройал Оук</a>»).
</p><p>В декабре 1939 года во Франции была сформирована пятая дивизия англичан, а в первые месяцы следующего года из Англии прибыло ещё пять дивизий<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup>. В тылу английских войск было создано почти 50 <a href="/wiki/%D0%90%D1%8D%D1%80%D0%BE%D0%B4%D1%80%D0%BE%D0%BC" title="Аэродром">аэродромов</a> с цементными взлётно-посадочными полосами, но вместо бомбардировки немецких позиций английские самолёты разбрасывали над линией фронта <a class="new" href="/w/index.php?title=%D0%90%D0%B3%D0%B8%D1%82%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D1%8B%D0%B5_%D0%BC%D0%B0%D1%82%D0%B5%D1%80%D0%B8%D0%B0%D0%BB%D1%8B&amp;action=edit&amp;redlink=1" title="Агитационные материалы (страница отсутствует)">агитационные листовки</a>.
</p>
<h3><span id=".D0.9F.D0.BE.D0.B7.D0.B8.D1.86.D0.B8.D1.8F_.D1.84.D1.80.D0.B0.D0.BD.D1.86.D1.83.D0.B7.D1.81.D0.BA.D0.B8.D1.85_.D0.BA.D0.BE.D0.BC.D0.BC.D1.83.D0.BD.D0.B8.D1.81.D1.82.D0.BE.D0.B2"></span><span class="mw-headline" id="Позиция_французских_коммунистов">Позиция французских коммунистов</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;veaction=edit&amp;section=12" title="Редактировать раздел «Позиция французских коммунистов»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;action=edit&amp;section=12" title="Редактировать код раздела «Позиция французских коммунистов»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>

<p>В сентябре 1939 <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B0%D1%8F_%D0%BA%D0%BE%D0%BC%D0%BC%D1%83%D0%BD%D0%B8%D1%81%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D0%BF%D0%B0%D1%80%D1%82%D0%B8%D1%8F" title="Французская коммунистическая партия">ФКП</a> начала антивоенную кампанию, призывая солдат <a href="/wiki/%D0%94%D0%B5%D0%B7%D0%B5%D1%80%D1%82%D0%B8%D1%80%D1%81%D1%82%D0%B2%D0%BE" title="Дезертирство">дезертировать</a> из армии. 2 сентября её депутаты проголосовали против военных кредитов. Генеральный секретарь партии <a href="/wiki/%D0%A2%D0%BE%D1%80%D0%B5%D0%B7,_%D0%9C%D0%BE%D1%80%D0%B8%D1%81" title="Торез, Морис">Морис Торез</a>, призванный в армию, дезертировал и бежал в СССР, за что был заочно приговорён военным судом к смертной казни<sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[прим. 1]</a></sup>.
</p>
<h2><span id=".D0.9F.D0.BB.D0.B0.D0.BD_.D0.B2.D1.82.D0.BE.D1.80.D0.B6.D0.B5.D0.BD.D0.B8.D1.8F_.D0.B2.D0.BE_.D0.A4.D1.80.D0.B0.D0.BD.D1.86.D0.B8.D1.8E"></span><span class="mw-headline" id="План_вторжения_во_Францию">План вторжения во Францию</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;veaction=edit&amp;section=13" title="Редактировать раздел «План вторжения во Францию»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;action=edit&amp;section=13" title="Редактировать код раздела «План вторжения во Францию»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="hatnote navigation-not-searchable">Основная статья: <b><a href="/wiki/%D0%9F%D0%BB%D0%B0%D0%BD_%C2%AB%D0%93%D0%B5%D0%BB%D1%8C%D0%B1%C2%BB" title="План «Гельб»">План «Гельб»</a></b></div>
<p><a href="/wiki/27_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="27 сентября">27 сентября</a> 1939 года на совете главнокомандующих видами вооружённых сил и начальников их штабов Гитлер приказал немедленно готовить наступление на западе: «Цель войны — поставить Великобританию на колени, разгромить Францию»<sup class="ts-fix-template noprint">[<i><a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%A1%D1%81%D1%8B%D0%BB%D0%BA%D0%B8_%D0%BD%D0%B0_%D0%B8%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8" title="Википедия:Ссылки на источники"><span title="нет источника (14 октября 2012)">источник не указан 4154 дня</span></a></i>]</sup>. Против выступили главнокомандующий сухопутными войсками <a href="/wiki/%D0%91%D1%80%D0%B0%D1%83%D1%85%D0%B8%D1%87,_%D0%92%D0%B0%D0%BB%D1%8C%D1%82%D0%B5%D1%80_%D1%84%D0%BE%D0%BD" title="Браухич, Вальтер фон">Вальтер фон Браухич</a> и начальник генерального штаба <a href="/wiki/%D0%93%D0%B0%D0%BB%D1%8C%D0%B4%D0%B5%D1%80,_%D0%A4%D1%80%D0%B0%D0%BD%D1%86" title="Гальдер, Франц">Франц Гальдер</a>. (Они даже подготовили план отстранения Гитлера от власти, но, не получив поддержки от командующего армией резерва генерала <a href="/wiki/%D0%A4%D1%80%D0%BE%D0%BC%D0%BC,_%D0%A4%D1%80%D0%B8%D0%B4%D1%80%D0%B8%D1%85" title="Фромм, Фридрих">Фромма</a>, оставили его).
</p><p>Уже <a href="/wiki/6_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="6 октября">6 октября</a> 1939 года немецкие войска окончательно завершили оккупацию Польши, а <a href="/wiki/9_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="9 октября">9 октября</a> командующим видами вооружённых сил Браухичу, <a class="mw-redirect" href="/wiki/%D0%93%D0%B5%D1%80%D0%B8%D0%BD%D0%B3,_%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD_%D0%92%D0%B8%D0%BB%D1%8C%D0%B3%D0%B5%D0%BB%D1%8C%D0%BC" title="Геринг, Герман Вильгельм">Герингу</a> и <a class="mw-redirect" href="/wiki/%D0%A0%D0%B5%D0%B4%D0%B5%D1%80,_%D0%AD%D1%80%D0%B8%D1%85_%D0%99%D0%BE%D1%85%D0%B0%D0%BD%D0%BD_%D0%90%D0%BB%D1%8C%D0%B1%D0%B5%D1%80%D1%82" title="Редер, Эрих Йоханн Альберт">Редеру</a> была прислана <i>«Памятная записка и главные указания насчёт ведения войны на Западе»</i><sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[28]</a></sup>. В этом документе на основе концепции «блицкрига» были обозначены стратегические цели будущей кампании. Тут же говорилось, что немецкие войска будут наступать на западе, игнорируя нейтралитет Бельгии, Нидерландов и Люксембурга. Несмотря на страхи провального завершения операции, Браухич дал поручение генеральному штабу разработать «Директиву Гельб о стратегическом развёртывании», которую подписал <a href="/wiki/29_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="29 октября">29 октября</a> 1939 года.
</p><p><a href="/wiki/%D0%9F%D0%BB%D0%B0%D0%BD_%C2%AB%D0%93%D0%B5%D0%BB%D1%8C%D0%B1%C2%BB" title="План «Гельб»">План «Гельб»</a> («Жёлтый») в своём первом варианте (<i>план ОКХ</i>) (который так и не был выполнен) предусматривал, что направление главного удара немецких войск будет проходить по обе стороны <a href="/wiki/%D0%9B%D1%8C%D0%B5%D0%B6" title="Льеж">Льежа</a>. Директива заканчивалась распоряжением группам армий «А» и группам армий «В» сосредоточить свои войска таким образом, чтобы они за шесть ночных переходов смогли занять исходные позиции для наступления. Начало наступления было назначено на <a href="/wiki/12_%D0%BD%D0%BE%D1%8F%D0%B1%D1%80%D1%8F" title="12 ноября">12 ноября</a>. <a href="/wiki/5_%D0%BD%D0%BE%D1%8F%D0%B1%D1%80%D1%8F" title="5 ноября">5 ноября</a> Браухич снова попробовал отговорить Гитлера от вторжения во Францию. Гитлер в свою очередь ещё раз подтвердил, что наступление необходимо проводить не позднее 12 ноября. Однако <a href="/wiki/7_%D0%BD%D0%BE%D1%8F%D0%B1%D1%80%D1%8F" title="7 ноября">7 ноября</a> приказ был отменён из-за неблагоприятных <a href="/wiki/%D0%9C%D0%B5%D1%82%D0%B5%D0%BE%D1%80%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F" title="Метеорология">метеорологических условий</a>. Позднее начало операции откладывали ещё 29 раз.
</p><p><a href="/wiki/10_%D1%8F%D0%BD%D0%B2%D0%B0%D1%80%D1%8F" title="10 января">10 января</a> 1940 года Гитлер назначил окончательную дату наступления — <a href="/wiki/17_%D1%8F%D0%BD%D0%B2%D0%B0%D1%80%D1%8F" title="17 января">17 января</a>. Но в тот же день, когда Гитлер принял это решение, произошёл весьма загадочный <i>«случай»</i>: самолёт с немецким офицером, который перевозил секретные документы, ошибочно приземлился на территории Бельгии и план «Гельб» попал в руки <a class="mw-redirect" href="/wiki/%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D0%B9%D1%86%D1%8B" title="Бельгийцы">бельгийцев</a> («Мехеленский инцидент»). Немцы вынуждены были изменить план операции. Новую редакцию предоставил начальник штаба группы армий «А» под командованием <a class="mw-redirect" href="/wiki/%D0%A0%D1%83%D0%BD%D0%B4%D1%88%D1%82%D0%B5%D0%B4%D1%82,_%D0%9A%D0%B0%D1%80%D0%BB_%D0%A0%D1%83%D0%B4%D0%BE%D0%BB%D1%8C%D1%84_%D0%93%D0%B5%D1%80%D0%B4_%D1%84%D0%BE%D0%BD" title="Рундштедт, Карл Рудольф Герд фон">Рундштедта</a> и <a class="mw-redirect" href="/wiki/%D0%AD%D1%80%D0%B8%D1%85_%D1%84%D0%BE%D0%BD_%D0%9C%D0%B0%D0%BD%D1%88%D1%82%D0%B5%D0%B9%D0%BD" title="Эрих фон Манштейн">Манштейна</a>. Манштейн пришёл к выводу, что лучше нанести главный удар через <a href="/wiki/%D0%90%D1%80%D0%B4%D0%B5%D0%BD%D0%BD%D1%8B_(%D0%B3%D0%BE%D1%80%D1%8B)" title="Арденны (горы)">Арденны</a> в направлении <a href="/wiki/%D0%A1%D0%B5%D0%B4%D0%B0%D0%BD_(%D0%B3%D0%BE%D1%80%D0%BE%D0%B4)" title="Седан (город)">Седана</a>, чего союзники никак не ожидали. Основной идеей <i>плана Манштейна</i> было <i>«заманивание»</i>. Манштейн не сомневался, что союзники обязательно отреагируют на вторжение в Бельгию. Но, развёртывая там свои войска, они потеряют свободный резерв (по крайней мере на несколько дней), загрузят до отказа дороги, и, главное, ослабят <i>«скольжением на север»</i> оперативный участок <a href="/wiki/%D0%94%D0%B8%D0%BD%D0%B0%D0%BD_(%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D1%8F)" title="Динан (Бельгия)">Динан</a> — Седан<sup class="ts-fix-template noprint">[<i><a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%A1%D1%81%D1%8B%D0%BB%D0%BA%D0%B8_%D0%BD%D0%B0_%D0%B8%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8" title="Википедия:Ссылки на источники"><span title="нет источника (14 октября 2012)">источник не указан 4154 дня</span></a></i>]</sup>.
</p>
<h2><span id=".D0.9E.D0.BA.D0.BA.D1.83.D0.BF.D0.B0.D1.86.D0.B8.D1.8F_.D0.94.D0.B0.D0.BD.D0.B8.D0.B8_.D0.B8_.D0.9D.D0.BE.D1.80.D0.B2.D0.B5.D0.B3.D0.B8.D0.B8"></span><span class="mw-headline" id="Оккупация_Дании_и_Норвегии">Оккупация Дании и Норвегии</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;veaction=edit&amp;section=14" title="Редактировать раздел «Оккупация Дании и Норвегии»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D1%82%D1%80%D0%B0%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0&amp;action=edit&amp;section=14" title="Редактировать код раздела «Оккупация Дании и Норвегии»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="hatnote navigation-not-searchable">Основная статья: <b><a class="mw-redirect" href="/wiki/%D0%94%D0%B0%D1%82%D1%81%D0%BA%D0%BE-%D0%9D%D0%BE%D1%80%D0%B2%D0%B5%D0%B6%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F" title="Датско-Норвежская операция">Датско-Норвежская операция</a></b></div>
<p>Планируя вторжение во Францию, немецкий генеральный штаб беспокоился, что в этом случае англо-французские войска могут занять <a href="/wiki/%D0%94%D0%B0%D0%BD%D0%B8%D1%8F" title="Дания">Данию</a> и <a href="/wiki/%D0%9D%D0%BE%D1%80%D0%B2%D0%B5%D0%B3%D0%B8%D1%8F" title="Норвегия">Норвегию</a>. <a href="/wiki/10_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="10 октября">10 октября</a> 1939 года главнокомандующий военно-морскими силами гроссадмирал Редер впервые указал Гитлеру на значение Норвегии в войне на море. <a href="/wiki/%D0%A1%D0%BA%D0%B0%D0%BD%D0%B4%D0%B8%D0%BD%D0%B0%D0%B2%D0%B8%D1%8F" title="Скандинавия">Скандинавия</a> была хорошим плацдармом для атаки на Германию. Оккупация Норвегии Великобританией и Францией для Германии означала бы фактическую блокаду её военно-морского флота.
</p>
<figure class="mw-default-size mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:HMS_Warspite,_Norway_1940.jpg"><img class="mw-file-element" data-file-height="578" data-file-width="800" decoding="async" height="159" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/af/HMS_Warspite%2C_Norway_1940.jpg/220px-HMS_Warspite%2C_Norway_1940.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/af/HMS_Warspite%2C_Norway_1940.jpg/330px-HMS_Warspite%2C_Norway_1940.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/af/HMS_Warspite%2C_Norway_1940.jpg/440px-HMS_Warspite%2C_Norway_1940.jpg 2x" width="220"/></a><figcaption>Морской бой у берегов Норвегии. 10 апреля 1940</figcaption></figure>
<p><a href="/wiki/14_%D0%B4%D0%B5%D0%BA%D0%B0%D0%B1%D1%80%D1%8F" title="14 декабря">14 декабря</a> 1939 года Гитлер отдал приказ о подготовке операции в Норвегии. <a href="/wiki/1_%D0%BC%D0%B0%D1%80%D1%82%D0%B0" title="1 марта">1 марта</a> 1940 года вышла специальная директива. В пункте 1-м директивы говорилось:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p><i>Развитие событий в Скандинавии требует совершить все приготовления для того, чтобы частью вооружённых сил оккупировать Данию и Норвегию. Это должно помешать англичанам закрепиться в Скандинавии и на Балтийском море, обеспечить нашу базу руды в Швеции и расширить для военно-морского флота и авиации исходные позиции против Англии.</i></p><div class="ts-Конец_цитаты-source">— <cite>[militera.lib.ru/h/patyanin_sv/13.html Оперативная директива на операцию «Везерюбунг»]</cite></div></blockquote>
<p><a href="/wiki/7_%D0%BC%D0%B0%D1%80%D1%82%D0%B0" title="7 марта">7 марта</a> 1940 года Гитлер утвердил окончательный план операции «Везерюбунг».
</p><p>Утром <a href="/wiki/9_%D0%B0%D0%BF%D1%80%D0%B5%D0%BB%D1%8F" title="9 апреля">9 апреля</a> немецкие послы в <a href="/wiki/%D0%9E%D1%81%D0%BB%D0%BE" title="Осло">Осло</a> и <a href="/wiki/%D0%9A%D0%BE%D0%BF%D0%B5%D0%BD%D0%B3%D0%B0%D0%B3%D0%B5%D0%BD" title="Копенгаген">Копенгагене</a> вручили властям Норвегии и Дании одинаковые по содержанию <a class="mw-redirect" href="/wiki/%D0%94%D0%B8%D0%BF%D0%BB%D0%BE%D0%BC%D0%B0%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D0%BD%D0%BE%D1%82%D0%B0" title="Дипломатическая нота">ноты</a>, в которых вооружённое выступление Германии было оправдано потребностью защитить обе нейтральные страны от якобы возможного в ближайшее время нападения со стороны Англии и Франции. Целью немецкого правительства, сообщалось в ноте, была мирная оккупация обеих стран.
</p><p>Дания подчинилась требованиям Германии почти без сопротивления.
</p><p>Другая ситуация <a class="mw-redirect" href="/wiki/%D0%9D%D0%BE%D1%80%D0%B2%D0%B5%D0%B6%D1%81%D0%BA%D0%B0%D1%8F_%D0%BA%D0%B0%D0%BC%D0%BF%D0%B0%D0%BD%D0%B8%D1%8F_1940" title="Норвежская кампания 1940">сложилась в Норвегии</a>. Там немцы 9-10 апреля захватили главные норвежские порты: Осло, <a href="/wiki/%D0%A2%D1%80%D0%BE%D0%BD%D1%85%D0%B5%D0%B9%D0%BC" title="Тронхейм">Тронхейм</a>, <a href="/wiki/%D0%91%D0%B5%D1%80%D0%B3%D0%B5%D0%BD" title="Берген">Берген</a>, <a href="/wiki/%D0%9D%D0%B0%D1%80%D0%B2%D0%B8%D0%BA" title="Нарвик">Нарвик</a>. 14 апреля англо-французский морской десант высадился под Нарвиком, 16 апреля — в <a href="/wiki/%D0%9D%D0%B0%D0%BC%D1%81%D1%83%D1%81" title="Намсус">Намсусе</a>, 17 апреля — в <a class="mw-redirect" href="/wiki/%D0%9E%D0%BD%D0%B4%D0%B0%D0%BB%D1%8C%D1%81%D0%BD%D0%B5%D1%81" title="Ондальснес">Ондальснесе</a>. 19 апреля англо-французские войска развернули наступление на Тронхейм, но потерпели поражение и в начале мая вынуждены были вывести свои части из центральной Норвегии. После боёв за Нарвик англо-французские силы в начале июня эвакуировались из северной части Норвегии. Позднее, <a href="/wiki/10_%D0%B8%D1%8E%D0%BD%D1%8F" title="10 июня">10 июня</a>, капитулировали последние части норвежской армии. Норвегия оказалась под управлением немецкой оккупационной администрации<sup class="ts-fix-template noprint">[<i><a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%A1%D1%81%D1%8B%D0%BB%D0%BA%D0%B8_%D0%BD%D0%B0_%D0%B8%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8" title="Википедия:Ссылки на источники"><span title="нет источника (14 октября 2012)">источник не указан 4154 дня</span></a></i>]</sup>.

<p>Период «странной войны» завершился <a href="/wiki/10_%D0%BC%D0%B0%D1%8F" title="10 мая">10 мая</a> 1940 года. В этот день немецкие войска, согласно плану «<a href="/wiki/%D0%9F%D0%BB%D0%B0%D0%BD_%C2%AB%D0%93%D0%B5%D0%BB%D1%8C%D0%B1%C2%BB" title="План «Гельб»">Гельб</a>», начали широкомасштабные наступательные действия на территории нейтральных <a href="/wiki/%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D1%8F" title="Бельгия">Бельгии</a>, <a href="/wiki/%D0%9D%D0%B8%D0%B4%D0%B5%D1%80%D0%BB%D0%B0%D0%BD%D0%B4%D1%8B" title="Нидерланды">Голландии</a> и <a href="/wiki/%D0%9B%D1%8E%D0%BA%D1%81%D0%B5%D0%BC%D0%B1%D1%83%D1%80%D0%B3" title="Люксембург">Люксембурга</a>. Затем через территорию Бельгии, обойдя <a href="/wiki/%D0%9B%D0%B8%D0%BD%D0%B8%D1%8F_%D0%9C%D0%B0%D0%B6%D0%B8%D0%BD%D0%BE" title="Линия Мажино">линию Мажино</a> с севера, немецкие войска вторглись в глубь <a class="mw-redirect" href="/wiki/%D0%A2%D1%80%D0%B5%D1%82%D1%8C%D1%8F_%D1%84%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B0%D1%8F_%D1%80%D0%B5%D1%81%D0%BF%D1%83%D0%B1%D0%BB%D0%B8%D0%BA%D0%B0" title="Третья французская республика">Франции</a>. Остатки боеспособных обороняющихся англо-французских сил были вытеснены в район <a href="/wiki/%D0%94%D1%8E%D0%BD%D0%BA%D0%B5%D1%80%D0%BA" title="Дюнкерк">Дюнкерка</a>, откуда они <a href="/wiki/%D0%94%D1%8E%D0%BD%D0%BA%D0%B5%D1%80%D0%BA%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F" title="Дюнкеркская операция">эвакуировались</a> в Великобританию.
</p>
