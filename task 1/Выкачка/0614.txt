<p><b>Домо́вый воробе́й</b><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> (<a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Латинский язык">лат.</a> <span lang="la" style="font-style:italic;">Passer domesticus</span>) — наиболее распространённый <a href="/wiki/%D0%91%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B2%D0%B8%D0%B4" title="Биологический вид">вид</a> из <a href="/wiki/%D0%A0%D0%BE%D0%B4_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Род (биология)">рода</a> <a href="/wiki/%D0%9D%D0%B0%D1%81%D1%82%D0%BE%D1%8F%D1%89%D0%B8%D0%B5_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D1%8C%D0%B8" title="Настоящие воробьи">настоящих воробьёв</a> (<i>Passer</i>) <a href="/wiki/%D0%A1%D0%B5%D0%BC%D0%B5%D0%B9%D1%81%D1%82%D0%B2%D0%BE" title="Семейство">семейства</a> <a href="/wiki/%D0%92%D0%BE%D1%80%D0%BE%D0%B1%D1%8C%D0%B8%D0%BD%D1%8B%D0%B5" title="Воробьиные">воробьиных</a> (Passeridae). Одна из самых известных <a href="/wiki/%D0%9F%D1%82%D0%B8%D1%86%D1%8B" title="Птицы">птиц</a><sup class="reference" id="cite_ref-ESBE_3-0"><a href="#cite_note-ESBE-3">[3]</a></sup>, обитающая по соседству с жилищем человека (отсюда её видовое название «домовый») и хорошо узнаваемая как по внешнему виду, так и по характерному чириканью<sup class="reference" id="cite_ref-Sev_4-0"><a href="#cite_note-Sev-4">[4]</a></sup>.
</p>

<p>Длина тела составляет 14—18 см<sup class="reference" id="cite_ref-ESBE_3-1"><a href="#cite_note-ESBE-3">[3]</a></sup><sup class="reference" id="cite_ref-Sev_4-1"><a href="#cite_note-Sev-4">[4]</a></sup>, масса — 21—37 г<sup class="reference" id="cite_ref-Sev_4-2"><a href="#cite_note-Sev-4">[4]</a></sup>. Общая окраска <a href="/wiki/%D0%9E%D0%BF%D0%B5%D1%80%D0%B5%D0%BD%D0%B8%D0%B5_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Оперение (биология)">оперения</a> — сверху коричневато-бурая, ржавчинного цвета с чёрными пятнами, снизу беловатая или серая. Щёки белые, ушная область бледно-серая. <a class="mw-redirect" href="/wiki/%D0%9A%D1%80%D1%8B%D0%BB%D0%BE_%D0%BF%D1%82%D0%B8%D1%86" title="Крыло птиц">Крылья</a> с желтовато-белой поперечной полосой<sup class="reference" id="cite_ref-ESBE_3-2"><a href="#cite_note-ESBE-3">[3]</a></sup>. <a href="/wiki/%D0%A1%D0%B0%D0%BC%D0%B5%D1%86" title="Самец">Самец</a> отличается от <a href="/wiki/%D0%A1%D0%B0%D0%BC%D0%BA%D0%B0" title="Самка">самки</a> наличием большого чёрного пятна («галстука»)<sup class="reference" id="cite_ref-ESBE_3-3"><a href="#cite_note-ESBE-3">[3]</a></sup><sup class="reference" id="cite_ref-Sev_4-3"><a href="#cite_note-Sev-4">[4]</a></sup><sup class="reference" id="cite_ref-BSE_5-0"><a href="#cite_note-BSE-5">[5]</a></sup>, охватывающего подбородок, горло, зоб и верхнюю часть груди, а также тёмно-серым (а не тёмно-бурым) верхом головы. У самки голова и горло серые, а над глазом имеется бледная серо-жёлтая полоса<sup class="reference" id="cite_ref-ESBE_3-4"><a href="#cite_note-ESBE-3">[3]</a></sup><sup class="reference" id="cite_ref-Sev_4-4"><a href="#cite_note-Sev-4">[4]</a></sup>.
</p>
<h3><span id=".D0.A0.D0.B0.D1.81.D0.BF.D1.80.D0.BE.D1.81.D1.82.D1.80.D0.B0.D0.BD.D0.B5.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Распространение">Распространение</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%BE%D0%B2%D1%8B%D0%B9_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D0%B5%D0%B9&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Распространение»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%BE%D0%B2%D1%8B%D0%B9_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D0%B5%D0%B9&amp;action=edit&amp;section=3" title="Редактировать код раздела «Распространение»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Прежде область обитания домового воробья ограничивалась <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%95%D0%B2%D1%80%D0%BE%D0%BF%D0%B0" title="Северная Европа">Северной Европой</a>. Впоследствии широко распространился в <a href="/wiki/%D0%95%D0%B2%D1%80%D0%BE%D0%BF%D0%B0" title="Европа">Европе</a> и <a href="/wiki/%D0%90%D0%B7%D0%B8%D1%8F" title="Азия">Азии</a><sup class="reference" id="cite_ref-ESBE_3-5"><a href="#cite_note-ESBE-3">[3]</a></sup><sup class="reference" id="cite_ref-Sev_4-5"><a href="#cite_note-Sev-4">[4]</a></sup><sup class="reference" id="cite_ref-BSE_5-1"><a href="#cite_note-BSE-5">[5]</a></sup> (за исключением <a href="/wiki/%D0%90%D1%80%D0%BA%D1%82%D0%B8%D0%BA%D0%B0" title="Арктика">Арктики</a>, северо-восточных, юго-восточных и центральных районов Азии)<sup class="reference" id="cite_ref-BSE_5-2"><a href="#cite_note-BSE-5">[5]</a></sup>, а также в <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%90%D1%84%D1%80%D0%B8%D0%BA%D0%B0" title="Северная Африка">Северной</a><sup class="reference" id="cite_ref-Sev_4-6"><a href="#cite_note-Sev-4">[4]</a></sup><sup class="reference" id="cite_ref-BSE_5-3"><a href="#cite_note-BSE-5">[5]</a></sup> и <a href="/wiki/%D0%92%D0%BE%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B0%D1%8F_%D0%90%D1%84%D1%80%D0%B8%D0%BA%D0%B0" title="Восточная Африка">Восточной Африке</a><sup class="reference" id="cite_ref-BSE_5-4"><a href="#cite_note-BSE-5">[5]</a></sup>, <a href="/wiki/%D0%A1%D0%B5%D0%BD%D0%B5%D0%B3%D0%B0%D0%BB" title="Сенегал">Сенегале</a><sup class="reference" id="cite_ref-ESBE_3-6"><a href="#cite_note-ESBE-3">[3]</a></sup>, <a href="/wiki/%D0%9C%D0%B0%D0%BB%D0%B0%D1%8F_%D0%90%D0%B7%D0%B8%D1%8F" title="Малая Азия">Малой Азии</a>, на <a href="/wiki/%D0%90%D1%80%D0%B0%D0%B2%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%BE%D0%BB%D1%83%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2" title="Аравийский полуостров">Аравийском полуострове</a> и <a class="mw-redirect" href="/wiki/%D0%AF%D0%B2%D0%B0_(%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2)" title="Ява (остров)">острове Ява</a><sup class="reference" id="cite_ref-ESBE_3-7"><a href="#cite_note-ESBE-3">[3]</a></sup>. В <a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D0%B8%D1%8F" title="Италия">Италии</a> обитает близкий вид — <a class="new" href="/w/index.php?title=%D0%98%D1%82%D0%B0%D0%BB%D1%8C%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D0%B5%D0%B9&amp;action=edit&amp;redlink=1" title="Итальянский воробей (страница отсутствует)">итальянский воробей</a> (<i><a href="/wiki/Passer_italiae" title="Passer italiae">Passer italiae</a>)</i><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>.
</p><p>Начиная с XX века был завезён в разные страны, широко там расселился и в настоящее время, кроме указанных выше мест, обитает также в <a href="/wiki/%D0%AE%D0%B6%D0%BD%D0%B0%D1%8F_%D0%90%D1%84%D1%80%D0%B8%D0%BA%D0%B0" title="Южная Африка">Южной Африке</a>, <a href="/wiki/%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D1%8F" title="Австралия">Австралии</a>, <a href="/wiki/%D0%9D%D0%BE%D0%B2%D0%B0%D1%8F_%D0%97%D0%B5%D0%BB%D0%B0%D0%BD%D0%B4%D0%B8%D1%8F" title="Новая Зеландия">Новой Зеландии</a>, <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%90%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B0" title="Северная Америка">Северной</a> и <a href="/wiki/%D0%AE%D0%B6%D0%BD%D0%B0%D1%8F_%D0%90%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B0" title="Южная Америка">Южной Америке</a> и на многих островах<sup class="reference" id="cite_ref-ESBE_3-8"><a href="#cite_note-ESBE-3">[3]</a></sup><sup class="reference" id="cite_ref-Sev_4-7"><a href="#cite_note-Sev-4">[4]</a></sup><sup class="reference" id="cite_ref-BSE_5-5"><a href="#cite_note-BSE-5">[5]</a></sup>.
</p><p>Почти повсеместно является оседлой птицей<sup class="reference" id="cite_ref-Sev_4-8"><a href="#cite_note-Sev-4">[4]</a></sup><sup class="reference" id="cite_ref-BSE_5-6"><a href="#cite_note-BSE-5">[5]</a></sup>, дальше 2—3 км от своего поселения обычно не улетает, лишь из некоторых частей <a href="/wiki/%D0%90%D1%80%D0%B5%D0%B0%D0%BB" title="Ареал">ареала</a> (<a href="/wiki/%D0%A1%D1%80%D0%B5%D0%B4%D0%BD%D1%8F%D1%8F_%D0%90%D0%B7%D0%B8%D1%8F" title="Средняя Азия">Средняя Азия</a>) на зиму улетает<sup class="reference" id="cite_ref-BSE_5-7"><a href="#cite_note-BSE-5">[5]</a></sup> в <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B5%D0%B4%D0%BD%D1%8F%D1%8F_%D0%90%D0%B7%D0%B8%D1%8F" title="Передняя Азия">Переднюю Азию</a> и <a href="/wiki/%D0%98%D0%BD%D0%B4%D0%B8%D1%8F" title="Индия">Индию</a>. При неблагоприятных условиях, из крайних северных районов может откочевывать на юг или в другие регионы<sup class="reference" id="cite_ref-bio.1september.ru_8-0"><a href="#cite_note-bio.1september.ru-8">[8]</a></sup>.
</p><p>Следуя за жильём человека, проник далеко на север в несвойственную для него зону <a href="/wiki/%D0%9B%D0%B5%D1%81%D0%BE%D1%82%D1%83%D0%BD%D0%B4%D1%80%D0%B0" title="Лесотундра">лесотундры</a> и даже <a href="/wiki/%D0%A2%D1%83%D0%BD%D0%B4%D1%80%D0%B0" title="Тундра">тундры</a> — до <a href="/wiki/%D0%9C%D1%83%D1%80%D0%BC%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C" title="Мурманская область">Мурманской области</a>, <a href="/wiki/%D0%A3%D1%81%D1%82%D1%8C%D0%B5" title="Устье">устья</a> <a class="mw-redirect" href="/wiki/%D0%9F%D0%B5%D1%87%D0%BE%D1%80%D0%B0_(%D1%80%D0%B5%D0%BA%D0%B0)" title="Печора (река)">Печоры</a>, севера <a href="/wiki/%D0%AF%D0%BA%D1%83%D1%82%D0%B8%D1%8F" title="Якутия">Якутии</a>. В северных поселениях (Салехард, <a href="/wiki/%D0%91%D0%B5%D1%80%D1%91%D0%B7%D0%BE%D0%B2%D0%BE_(%D0%A5%D0%B0%D0%BD%D1%82%D1%8B-%D0%9C%D0%B0%D0%BD%D1%81%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%B2%D1%82%D0%BE%D0%BD%D0%BE%D0%BC%D0%BD%D1%8B%D0%B9_%D0%BE%D0%BA%D1%80%D1%83%D0%B3)" title="Берёзово (Ханты-Мансийский автономный округ)">Берёзово</a>) домовые воробьи неоднократно полностью вымирали в холодные зимы, но затем снова появлялись. В Воркуте, по сообщениям биологов, в 1950-х годах жили всего около 50 пар воробьёв, зимой они удерживались благодаря тому, что жили в котельной теплоцентрали, а вылетали покормиться на короткое время к ближайшей столовой<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>.
</p><p>Согласно оценкам, опубликованным в 2021 году, домовый воробей — самый распространённый вид птиц, который насчитывает 1,6 млрд особей<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>.
</p>
<h3><span id=".D0.9E.D0.B1.D1.80.D0.B0.D0.B7_.D0.B6.D0.B8.D0.B7.D0.BD.D0.B8"></span><span class="mw-headline" id="Образ_жизни">Образ жизни</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%BE%D0%B2%D1%8B%D0%B9_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D0%B5%D0%B9&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Образ жизни»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%BE%D0%B2%D1%8B%D0%B9_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D0%B5%D0%B9&amp;action=edit&amp;section=4" title="Редактировать код раздела «Образ жизни»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Будучи <a href="/wiki/%D0%A1%D0%B8%D0%BD%D0%B0%D0%BD%D1%82%D1%80%D0%BE%D0%BF%D0%BD%D1%8B%D0%B5_%D0%BE%D1%80%D0%B3%D0%B0%D0%BD%D0%B8%D0%B7%D0%BC%D1%8B" title="Синантропные организмы">синантропным видом</a><sup class="reference" id="cite_ref-Sev_4-9"><a href="#cite_note-Sev-4">[4]</a></sup><sup class="reference" id="cite_ref-BSE_5-8"><a href="#cite_note-BSE-5">[5]</a></sup> — постоянным сожителем человека, домовый воробей хорошо приспособлен к жизни в обстановке, меняющейся под воздействием хозяйственной деятельности человека.
</p><p>Воробья равно можно встретить в сельской и городской местности, в поселениях <a href="/wiki/%D0%9A%D1%80%D0%B0%D0%B9%D0%BD%D0%B8%D0%B9_%D0%A1%D0%B5%D0%B2%D0%B5%D1%80" title="Крайний Север">Крайнего Севера</a> и Средней Азии. Впрочем, в Средней Азии и на юге Казахстана, помимо оседлого домового воробья, обитает перелётный индийский воробей (<i>Passer domesticus indicus,</i> подвид домового), который держится обычно подальше от человеческого жилья, часто гнездится в смешанных колониях с <a href="/wiki/%D0%A7%D0%B5%D1%80%D0%BD%D0%BE%D0%B3%D1%80%D1%83%D0%B4%D1%8B%D0%B9_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D0%B5%D0%B9" title="Черногрудый воробей">черногрудым воробьём</a><sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup>.
</p><p>Широкому распространению воробья на север и на юг вслед за расширением хлебопашества способствует его высокая плодовитость.
</p><p>Домовый воробей ведёт оседлый образ жизни<sup class="reference" id="cite_ref-Sev_4-10"><a href="#cite_note-Sev-4">[4]</a></sup><sup class="reference" id="cite_ref-BSE_5-9"><a href="#cite_note-BSE-5">[5]</a></sup>, чему благоприятствуют условия для гнездования и обильный корм в поселениях человека.
</p>
<h3><span id=".D0.9F.D0.B8.D1.82.D0.B0.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Питание">Питание</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%BE%D0%B2%D1%8B%D0%B9_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D0%B5%D0%B9&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Питание»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%BE%D0%B2%D1%8B%D0%B9_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D0%B5%D0%B9&amp;action=edit&amp;section=5" title="Редактировать код раздела «Питание»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Питается в основном растительной пищей, весной и летом частично <a href="/wiki/%D0%9D%D0%B0%D1%81%D0%B5%D0%BA%D0%BE%D0%BC%D1%8B%D0%B5" title="Насекомые">насекомыми</a>, которыми также вскармливает птенцов<sup class="reference" id="cite_ref-Sev_4-11"><a href="#cite_note-Sev-4">[4]</a></sup><sup class="reference" id="cite_ref-BSE_5-10"><a href="#cite_note-BSE-5">[5]</a></sup>.
</p><p>В рацион воробья входят семена <a href="/wiki/%D0%97%D0%B5%D1%80%D0%BD%D0%BE%D0%B2%D1%8B%D0%B5_%D0%BA%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D1%8B" title="Зерновые культуры">сельскохозяйственных культур</a><sup class="reference" id="cite_ref-ESBE_3-9"><a href="#cite_note-ESBE-3">[3]</a></sup><sup class="reference" id="cite_ref-Sev_4-12"><a href="#cite_note-Sev-4">[4]</a></sup><sup class="reference" id="cite_ref-BSE_5-11"><a href="#cite_note-BSE-5">[5]</a></sup>, отбросы различных продуктов, которые он подбирает в человеческих поселениях<sup class="reference" id="cite_ref-Sev_4-13"><a href="#cite_note-Sev-4">[4]</a></sup>, <a href="/wiki/%D0%A5%D0%BB%D0%B5%D0%B1%D0%BD%D1%8B%D0%B5_%D0%B7%D0%B5%D1%80%D0%BD%D0%BE%D0%B2%D1%8B%D0%B5_%D0%BA%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D1%8B" title="Хлебные зерновые культуры">хлебные</a> <a href="/wiki/%D0%97%D0%BB%D0%B0%D0%BA%D0%B8" title="Злаки">злаки</a> в полях<sup class="reference" id="cite_ref-ESBE_3-10"><a href="#cite_note-ESBE-3">[3]</a></sup><sup class="reference" id="cite_ref-BSE_5-12"><a href="#cite_note-BSE-5">[5]</a></sup>, <a class="mw-redirect" href="/wiki/%D0%AF%D0%B3%D0%BE%D0%B4%D0%B0_(%D0%BF%D1%80%D0%BE%D0%B4%D1%83%D0%BA%D1%82_%D0%BF%D0%B8%D1%82%D0%B0%D0%BD%D0%B8%D1%8F)" title="Ягода (продукт питания)">ягоды</a> <a href="/wiki/%D0%92%D0%B8%D1%88%D0%BD%D1%8F" title="Вишня">вишни</a><sup class="reference" id="cite_ref-ESBE_3-11"><a href="#cite_note-ESBE-3">[3]</a></sup>, <a href="/wiki/%D0%A1%D0%BC%D0%BE%D1%80%D0%BE%D0%B4%D0%B8%D0%BD%D0%B0" title="Смородина">смородины</a>, <a href="/wiki/%D0%92%D0%B8%D0%BD%D0%BE%D0%B3%D1%80%D0%B0%D0%B4_%D0%BA%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D0%BD%D1%8B%D0%B9" title="Виноград культурный">винограда</a> в садах<sup class="reference" id="cite_ref-ESBE_3-12"><a href="#cite_note-ESBE-3">[3]</a></sup>, весной <a href="/wiki/%D0%A6%D0%B2%D0%B5%D1%82%D0%BE%D0%BA" title="Цветок">цветочные</a> <a href="/wiki/%D0%9F%D0%BE%D1%87%D0%BA%D0%B0_(%D0%B1%D0%BE%D1%82%D0%B0%D0%BD%D0%B8%D0%BA%D0%B0)" title="Почка (ботаника)">почки</a>, осенью семена сорняков<sup class="reference" id="cite_ref-bio.1september.ru_8-1"><a href="#cite_note-bio.1september.ru-8">[8]</a></sup>. При отсутствии поблизости полей, вылетает кормиться на <a href="/wiki/%D0%9B%D1%83%D0%B3" title="Луг">луга</a>, опушки <a href="/wiki/%D0%9B%D0%B5%D1%81" title="Лес">лесов</a> и в <a href="/wiki/%D0%A1%D1%82%D0%B5%D0%BF%D1%8C" title="Степь">степи</a>, где собирает семена дикорастущих <a class="mw-redirect" href="/wiki/%D0%A2%D1%80%D0%B0%D0%B2%D0%B0" title="Трава">трав</a> и иногда ловит насекомых<sup class="reference" id="cite_ref-BSE_5-13"><a href="#cite_note-BSE-5">[5]</a></sup>. На полях часто кормится в смешанных стаях с <a href="/wiki/%D0%9F%D0%BE%D0%BB%D0%B5%D0%B2%D0%BE%D0%B9_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D0%B5%D0%B9" title="Полевой воробей">полевым воробьём</a>, в Средней Азии также с черногрудым<sup class="reference" id="cite_ref-bio.1september.ru_8-2"><a href="#cite_note-bio.1september.ru-8">[8]</a></sup>.
</p>
<h3><span id=".D0.A0.D0.B0.D0.B7.D0.BC.D0.BD.D0.BE.D0.B6.D0.B5.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Размножение">Размножение</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%BE%D0%B2%D1%8B%D0%B9_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D0%B5%D0%B9&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Размножение»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%BE%D0%B2%D1%8B%D0%B9_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D0%B5%D0%B9&amp;action=edit&amp;section=6" title="Редактировать код раздела «Размножение»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Гнездится воробей отдельными парами, но иногда и <a href="/wiki/%D0%9A%D0%BE%D0%BB%D0%BE%D0%BD%D0%B8%D1%8F_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Колония (биология)">колониями</a>, поселяясь непосредственно у жилья человека или вблизи его поселений. На юге <a href="/wiki/%D0%90%D1%80%D0%B5%D0%B0%D0%BB" title="Ареал">ареала</a> нередко строит <a href="/wiki/%D0%93%D0%BD%D0%B5%D0%B7%D0%B4%D0%BE" title="Гнездо">гнёзда</a> в стороне от них — в древесных и кустарниковых насаждениях, в оврагах, по крутым глинистым обрывам по соседству с полями.
</p><p>Гнёзда устраивает в разнообразных местах — в щелях строений, в норах<sup class="reference" id="cite_ref-Sev_4-14"><a href="#cite_note-Sev-4">[4]</a></sup><sup class="reference" id="cite_ref-BSE_5-14"><a href="#cite_note-BSE-5">[5]</a></sup> глинистых и меловых оврагов, в стенках гнёзд крупных птиц (<a class="mw-redirect" href="/wiki/%D0%9D%D0%B0%D1%81%D1%82%D0%BE%D1%8F%D1%89%D0%B8%D0%B5_%D1%86%D0%B0%D0%BF%D0%BB%D0%B8" title="Настоящие цапли">цапель</a>, <a class="mw-redirect" href="/wiki/%D0%9D%D0%B0%D1%81%D1%82%D0%BE%D1%8F%D1%89%D0%B8%D0%B5_%D0%B0%D0%B8%D1%81%D1%82%D1%8B" title="Настоящие аисты">аистов</a>, <a href="/wiki/%D0%9E%D1%80%D0%BB%D1%8B" title="Орлы">орлов</a>), в дуплах деревьев<sup class="reference" id="cite_ref-BSE_5-15"><a href="#cite_note-BSE-5">[5]</a></sup>, может занимать <a href="/wiki/%D0%A1%D0%BA%D0%B2%D0%BE%D1%80%D0%B5%D1%87%D0%BD%D0%B8%D0%BA" title="Скворечник">скворечники</a> и норы <a href="/wiki/%D0%91%D0%B5%D1%80%D0%B5%D0%B3%D0%BE%D0%B2%D1%83%D1%88%D0%BA%D0%B0" title="Береговушка">береговых ласточек</a><sup class="reference" id="cite_ref-Sev_4-15"><a href="#cite_note-Sev-4">[4]</a></sup>. Формально относится к категории птиц-дуплогнёздников<sup class="reference" id="cite_ref-:1_12-0"><a href="#cite_note-:1-12">[12]</a></sup>. На юге нередко сооружает гнёзда открыто, на ветвях различных деревьев<sup class="reference" id="cite_ref-BSE_5-16"><a href="#cite_note-BSE-5">[5]</a></sup>.
</p><p>По ряду наблюдений, воробьи образуют пары фактически на всю жизнь. Продолжительность жизни у них не очень длинная: хотя отмечали и 9-, и 11-летних воробьёв, большинство их не доживает и до 4 лет. Много молодых птиц гибнет ещё в первую зиму, так что средняя продолжительность жизни воробьёв составляет 9—21 месяц<sup class="reference" id="cite_ref-bio.1september.ru_8-3"><a href="#cite_note-bio.1september.ru-8">[8]</a></sup>.
</p><p>Гнездо, в постройке которого участвуют оба пола, представляет собой грубое сооружение из соломинок, мочалок, сухих травинок, пёрышек, с небольшим углублением посредине<sup class="reference" id="cite_ref-Sev_4-16"><a href="#cite_note-Sev-4">[4]</a></sup>. Гнёзда, устроенные на ветвях, крупные, в виде неправильного шара с толстыми стенками из травянистых растений и боковым входом.
</p><p>Домовые воробьи приступают к <a href="/wiki/%D0%A0%D0%B0%D0%B7%D0%BC%D0%BD%D0%BE%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5" title="Размножение">размножению</a> рано. В средней полосе <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D1%8F" title="Россия">России</a> предбрачное оживление начинается в марте, иногда в феврале, и сопровождается криками и драками. Во второй половине марта птицы разбиваются на пары и приступают к постройке гнёзд; откладывание <a href="/wiki/%D0%AF%D0%B9%D1%86%D0%BE" title="Яйцо">яиц</a> происходит в апреле<sup class="reference" id="cite_ref-Sev_4-17"><a href="#cite_note-Sev-4">[4]</a></sup>.
</p><p><a href="/wiki/%D0%9A%D0%BB%D0%B0%D0%B4%D0%BA%D0%B0_%D1%8F%D0%B8%D1%86" title="Кладка яиц">Кладка</a> состоит из 4—10, чаще 5—7, белых яиц с буроватыми крапинками и пятнами, <a href="/wiki/%D0%98%D0%BD%D1%81%D1%82%D0%B8%D0%BD%D0%BA%D1%82_%D0%BD%D0%B0%D1%81%D0%B8%D0%B6%D0%B8%D0%B2%D0%B0%D0%BD%D0%B8%D1%8F" title="Инстинкт насиживания">насиживание</a> которых занимает 11—13 дней.
</p><p>Птенцов выкармливают самец и самка, преимущественно насекомыми<sup class="reference" id="cite_ref-Sev_4-18"><a href="#cite_note-Sev-4">[4]</a></sup><sup class="reference" id="cite_ref-BSE_5-17"><a href="#cite_note-BSE-5">[5]</a></sup>. Через 10 дней после вылупления (в средней полосе в конце мая — начале июня) птенцы вылетают из гнезда.
</p>

<p>Воробьи весьма плодовиты и в течение лета выводят два (на севере) или три (на юге) выводка<sup class="reference" id="cite_ref-Sev_4-19"><a href="#cite_note-Sev-4">[4]</a></sup>. Второе откладывание яиц происходит во второй половине июня, вылет птенцов — в июле. В августе воробьи покидают гнезда, чтобы дать им очиститься от паразитов и ночуют в кронах деревьев<sup class="reference" id="cite_ref-:1_12-1"><a href="#cite_note-:1-12">[12]</a></sup>.
</p><p>Выводки обычно сбиваются в стаи, образуя иногда скопления до нескольких тысяч особей, и летают кормиться на поля<sup class="reference" id="cite_ref-Sev_4-20"><a href="#cite_note-Sev-4">[4]</a></sup>. В этот период происходят расселение и кочевки воробьев<sup class="reference" id="cite_ref-bio.1september.ru_8-4"><a href="#cite_note-bio.1september.ru-8">[8]</a></sup>.
</p><p>Поздней осенью наблюдается абортивный половой цикл, когда птицы вновь оживлённо чирикают, самцы ухаживают за самками и таскают в гнездо строительный материал. Подновлённые таким образом гнёзда служат зимой убежищем от ночных холодов. По итогам учётов птиц на Русской равнине выяснилось, что самцов домового воробья обычно значительно больше (примерно в 1,5 раза), чем самок. Поэтому осенью холостые самцы активно ищут себе свободную подругу на следующий сезон<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup>.
</p><p>В природе иногда образует гибриды с близкими видами: <a href="/wiki/%D0%A7%D0%B5%D1%80%D0%BD%D0%BE%D0%B3%D1%80%D1%83%D0%B4%D1%8B%D0%B9_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D0%B5%D0%B9" title="Черногрудый воробей">черногрудым</a> (испанским) и, очень редко, с <a href="/wiki/%D0%9F%D0%BE%D0%BB%D0%B5%D0%B2%D0%BE%D0%B9_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D0%B5%D0%B9" title="Полевой воробей">полевым</a> воробьём. Гибридизации с полевым воробьём, возможно, препятствует то, что окраска самок полевого воробья похожа на окраску самцов домового воробья<sup class="reference" id="cite_ref-:1_12-2"><a href="#cite_note-:1-12">[12]</a></sup>.
</p>

<p>Первоначальное название вида, <i>Fringilla domestica</i>, дано <a href="/wiki/%D0%9B%D0%B8%D0%BD%D0%BD%D0%B5%D0%B9,_%D0%9A%D0%B0%D1%80%D0%BB" title="Линней, Карл">Линнеем</a>, который относил домового воробья к роду <a href="/wiki/%D0%97%D1%8F%D0%B1%D0%BB%D0%B8%D0%BA" title="Зяблик">зябликов</a> (<span data-wikidata-property-id="P225" lang="la" style="font-style: italic;">Fringilla</span>).
</p>

<p><b><a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B8%D0%BE%D1%82%D0%B8%D0%BF" title="Кариотип">Кариотип</a></b>: 76—78 <a href="/wiki/%D0%A5%D1%80%D0%BE%D0%BC%D0%BE%D1%81%D0%BE%D0%BC%D0%B0" title="Хромосома">хромосом</a> (<a href="/wiki/%D0%94%D0%B8%D0%BF%D0%BB%D0%BE%D0%B8%D0%B4%D0%BD%D1%8B%D0%B5_%D0%BA%D0%BB%D0%B5%D1%82%D0%BA%D0%B8" title="Диплоидные клетки">2<i>n</i></a>)<sup class="reference" id="cite_ref-genomesize_15-0"><a href="#cite_note-genomesize-15">[15]</a></sup>.
</p>
<dl><dt><a href="/wiki/%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D1%8F%D1%80%D0%BD%D0%B0%D1%8F_%D0%B3%D0%B5%D0%BD%D0%B5%D1%82%D0%B8%D0%BA%D0%B0" title="Молекулярная генетика">Молекулярная генетика</a></dt></dl>
<ul><li><span class="dabhide"><a class="mw-disambig" href="/wiki/%D0%94%D0%B5%D0%BF%D0%BE%D0%BD%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5" title="Депонирование">Депонированные</a></span> <a class="mw-redirect" href="/wiki/%D0%9D%D1%83%D0%BA%D0%BB%D0%B5%D0%BE%D1%82%D0%B8%D0%B4" title="Нуклеотид">нуклеотидные</a> <a href="/wiki/%D0%A1%D0%B5%D0%BA%D0%B2%D0%B5%D0%BD%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5" title="Секвенирование">последовательности</a> в <a href="/wiki/%D0%91%D0%B0%D0%B7%D0%B0_%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85" title="База данных">базе данных</a> <i>EntrezNucleotide</i>, <a href="/wiki/GenBank" title="GenBank">GenBank</a>, <a class="mw-redirect" href="/wiki/NCBI" title="NCBI">NCBI</a>, <a href="/wiki/%D0%A1%D0%BE%D0%B5%D0%B4%D0%B8%D0%BD%D1%91%D0%BD%D0%BD%D1%8B%D0%B5_%D0%A8%D1%82%D0%B0%D1%82%D1%8B_%D0%90%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B8" title="Соединённые Штаты Америки">США</a>: <a class="external text" href="https://www.ncbi.nlm.nih.gov/nuccore/?term=txid48849%5BOrganism:exp%5D" rel="nofollow">2323</a> (по состоянию на 29 марта 2021).</li>
<li>Депонированные последовательности <a href="/wiki/%D0%91%D0%B5%D0%BB%D0%BA%D0%B8" title="Белки">белков</a> в базе данных <i>EntrezProtein</i>, GenBank, NCBI, США: <a class="external text" href="https://www.ncbi.nlm.nih.gov/protein/?term=txid48849%5BOrganism:exp%5D" rel="nofollow">983</a> (по состоянию на 29 марта 2021).</li></ul>
<p>Домовому воробью — как генетически наиболее изученному представителю всего семейства воробьиных — принадлежит бо́льшая часть депонированных последовательностей в роде <i>Passer</i> и в семействе в целом.
</p><p><b><a href="/wiki/%D0%93%D0%B5%D0%BD%D0%BE%D0%BC" title="Геном">Геном</a></b>: 1,26—1,57 пг (<a href="/wiki/C-%D0%BF%D0%B0%D1%80%D0%B0%D0%B4%D0%BE%D0%BA%D1%81_(%D0%B8%D0%B7%D0%B1%D1%8B%D1%82%D0%BE%D1%87%D0%BD%D0%BE%D1%81%D1%82%D1%8C_%D0%B3%D0%B5%D0%BD%D0%BE%D0%BC%D0%B0)" title="C-парадокс (избыточность генома)">C-value</a>)<sup class="reference" id="cite_ref-genomesize_15-1"><a href="#cite_note-genomesize-15">[15]</a></sup>.
</p><p>Домовый воробей 11 тысяч лет назад стал распространяться по Европе и Азии с Ближнего Востока, в это же время отделился подвид <i><a class="new" href="/w/index.php?title=Passer_domesticus_bactrianus&amp;action=edit&amp;redlink=1" title="Passer domesticus bactrianus (страница отсутствует)">P. domesticus bactrianus</a></i>. У домовых воробьев присутствует вариант гена <i><span data-interwiki-article="AMY2A" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=AMY2A&amp;action=edit&amp;redlink=1" title="AMY2A (страница отсутствует)">AMY2A</a></span><span class="noprint" style="font-style:normal; font-weight:normal;"><sup><a class="extiw" href="https://en.wikipedia.org/wiki/AMY2A" title="en:AMY2A"><span title="AMY2A — версия статьи «AMY2A» на английском языке">[en]</span></a></sup></span></i>, который обусловливает синтез более эффективной формы <a href="/wiki/%D0%90%D0%BB%D1%8C%D1%84%D0%B0-%D0%B0%D0%BC%D0%B8%D0%BB%D0%B0%D0%B7%D0%B0" title="Альфа-амилаза">альфа-амилазы</a> <a href="/wiki/%D0%9F%D0%BE%D0%B4%D0%B6%D0%B5%D0%BB%D1%83%D0%B4%D0%BE%D1%87%D0%BD%D0%B0%D1%8F_%D0%B6%D0%B5%D0%BB%D0%B5%D0%B7%D0%B0" title="Поджелудочная железа">поджелудочной железы</a>, что позволяет разлагать длинные молекулы <a href="/wiki/%D0%9A%D1%80%D0%B0%D1%85%D0%BC%D0%B0%D0%BB" title="Крахмал">крахмала</a> на более короткие<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup>.
</p>
<h2><span id=".D0.9B.D1.8E.D0.B4.D0.B8_.D0.B8_.D0.B2.D0.BE.D1.80.D0.BE.D0.B1.D1.8C.D0.B8"></span><span class="mw-headline" id="Люди_и_воробьи">Люди и воробьи</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%BE%D0%B2%D1%8B%D0%B9_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D0%B5%D0%B9&amp;veaction=edit&amp;section=10" title="Редактировать раздел «Люди и воробьи»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%BE%D0%B2%D1%8B%D0%B9_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D0%B5%D0%B9&amp;action=edit&amp;section=10" title="Редактировать код раздела «Люди и воробьи»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Faroe_stamp_344_house_sparrow_(passer_domesticus).jpg"><img class="mw-file-element" data-file-height="363" data-file-width="295" decoding="async" height="203" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/34/Faroe_stamp_344_house_sparrow_%28passer_domesticus%29.jpg/165px-Faroe_stamp_344_house_sparrow_%28passer_domesticus%29.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/34/Faroe_stamp_344_house_sparrow_%28passer_domesticus%29.jpg/248px-Faroe_stamp_344_house_sparrow_%28passer_domesticus%29.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/3/34/Faroe_stamp_344_house_sparrow_%28passer_domesticus%29.jpg 2x" width="165"/></a><figcaption>Пара домовых воробьёв на <a href="/wiki/%D0%98%D1%81%D1%82%D0%BE%D1%80%D0%B8%D1%8F_%D0%BF%D0%BE%D1%87%D1%82%D1%8B_%D0%B8_%D0%BF%D0%BE%D1%87%D1%82%D0%BE%D0%B2%D1%8B%D1%85_%D0%BC%D0%B0%D1%80%D0%BE%D0%BA_%D0%A4%D0%B0%D1%80%D0%B5%D1%80%D1%81%D0%BA%D0%B8%D1%85_%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2%D0%BE%D0%B2" title="История почты и почтовых марок Фарерских островов">почтовой марке Фарерских островов</a></figcaption></figure>
<p>Отношение у человека к воробьям, способным приносить как пользу, так и вред, варьирует соответственно от весьма позитивного до негативного.
</p><p>Иногда воробьи могут являться переносчиками различных вредителей и возбудителей некоторых <a href="/wiki/%D0%98%D0%BD%D1%84%D0%B5%D0%BA%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D1%8B%D0%B5_%D0%B7%D0%B0%D0%B1%D0%BE%D0%BB%D0%B5%D0%B2%D0%B0%D0%BD%D0%B8%D1%8F" title="Инфекционные заболевания">заболеваний</a>. Перелетая с одного элеватора на другой, они могут переносить на своём оперении опасных вредителей зерна — амбарных <a class="mw-redirect" href="/wiki/%D0%9A%D0%BB%D0%B5%D1%89%D0%B8_(%D1%87%D0%BB%D0%B5%D0%BD%D0%B8%D1%81%D1%82%D0%BE%D0%BD%D0%BE%D0%B3%D0%B8%D0%B5)" title="Клещи (членистоногие)">клещей</a>. Кроме того, воробьи могут распространять болезни домашних птиц: <a class="new" href="/w/index.php?title=%D0%9F%D1%82%D0%B8%D1%87%D1%8C%D1%8F_%D0%BE%D1%81%D0%BF%D0%B0&amp;action=edit&amp;redlink=1" title="Птичья оспа (страница отсутствует)">птичью оспу</a>, <a class="new" href="/w/index.php?title=%D0%9F%D1%82%D0%B8%D1%87%D1%8C%D1%8F_%D0%B4%D0%B8%D1%84%D1%82%D0%B5%D1%80%D0%B8%D1%8F&amp;action=edit&amp;redlink=1" title="Птичья дифтерия (страница отсутствует)">птичью дифтерию</a> и некоторые другие<sup class="reference" id="cite_ref-bio.1september.ru_8-5"><a href="#cite_note-bio.1september.ru-8">[8]</a></sup>. В Китае в 2004 году, когда <a href="/wiki/%D0%92%D0%B8%D1%80%D1%83%D1%81%D1%8B" title="Вирусы">вирус</a> <a href="/wiki/%D0%9F%D1%82%D0%B8%D1%87%D0%B8%D0%B9_%D0%B3%D1%80%D0%B8%D0%BF%D0%BF" title="Птичий грипп">птичьего гриппа</a> <a href="/wiki/H5N1" title="H5N1">H5N1</a> был впервые обнаружен в стране, были протестированы 38 воробьёв, и в некоторых образцах был найден опасный вирус<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup>.
</p><p>В летнее время местами, особенно на юге, где воробьи многочисленны, они могут приносить ощутимый вред созревающим <a href="/wiki/%D0%97%D0%B5%D1%80%D0%BD%D0%BE%D0%B2%D1%8B%D0%B5_%D0%BA%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D1%8B" title="Зерновые культуры">зерновым культурам</a> (<a href="/wiki/%D0%A5%D0%BB%D0%B5%D0%B1%D0%BD%D1%8B%D0%B5_%D0%B7%D0%B5%D1%80%D0%BD%D0%BE%D0%B2%D1%8B%D0%B5_%D0%BA%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D1%8B" title="Хлебные зерновые культуры">хлеба</a>, <a href="/wiki/%D0%A0%D0%B8%D1%81" title="Рис">рис</a>, <a href="/wiki/%D0%93%D0%BE%D1%80%D0%BE%D1%85" title="Горох">горох</a>)<sup class="reference" id="cite_ref-ESBE_3-13"><a href="#cite_note-ESBE-3">[3]</a></sup><sup class="reference" id="cite_ref-BSE_5-19"><a href="#cite_note-BSE-5">[5]</a></sup>, а также ягодам (<a href="/wiki/%D0%92%D0%B8%D1%88%D0%BD%D1%8F" title="Вишня">вишни</a>, <a href="/wiki/%D0%92%D0%B8%D0%BD%D0%BE%D0%B3%D1%80%D0%B0%D0%B4_%D0%BA%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D0%BD%D1%8B%D0%B9" title="Виноград культурный">виноград</a>)<sup class="reference" id="cite_ref-ESBE_3-14"><a href="#cite_note-ESBE-3">[3]</a></sup>, <a href="/wiki/%D0%9F%D0%BE%D0%B4%D1%81%D0%BE%D0%BB%D0%BD%D0%B5%D1%87%D0%BD%D0%B8%D0%BA" title="Подсолнечник">подсолнечнику</a> и <a href="/wiki/%D0%9A%D0%BE%D0%BD%D0%BE%D0%BF%D0%BB%D1%8F" title="Конопля">конопле</a>.
</p><p>В советское время в Средней Азии злостным вредителем зерновых считался индийский воробей, подвид домового, и с ним велась борьба отравленными приманками<sup class="reference" id="cite_ref-BSE_5-20"><a href="#cite_note-BSE-5">[5]</a></sup>. На юге Средней Азии численность <a class="mw-redirect" href="/wiki/%D0%9D%D0%BE%D0%BC%D0%B8%D0%BD%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D1%8B%D0%B9_%D0%BF%D0%BE%D0%B4%D0%B2%D0%B8%D0%B4" title="Номинативный подвид">номинативного</a> (оседлого) домового воробья — небольшая и вредителем он не считается<sup class="reference" id="cite_ref-bio.1september.ru_8-6"><a href="#cite_note-bio.1september.ru-8">[8]</a></sup>.
</p><p>В остальные сезоны вред от них незначителен. Во время выведения птенцов воробьи приносят несомненную пользу, истребляя множество вредных насекомых, особенно там, где мало других насекомоядных птиц. Истребление воробья в какой-либо местности всегда сопровождается чрезвычайно сильным размножением вредных насекомых<sup class="reference" id="cite_ref-ESBE_3-15"><a href="#cite_note-ESBE-3">[3]</a></sup>. В питании воробья осенью большую долю занимают семена сорняков<sup class="reference" id="cite_ref-bio.1september.ru_8-7"><a href="#cite_note-bio.1september.ru-8">[8]</a></sup>. Сами по себе воробьи являются кормом для полезных хищных птиц<sup class="reference" id="cite_ref-:1_12-3"><a href="#cite_note-:1-12">[12]</a></sup>.
</p><p>Широко известна кампания по <a href="/wiki/%D0%A3%D0%BD%D0%B8%D1%87%D1%82%D0%BE%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D1%8C%D1%91%D0%B2" title="Уничтожение воробьёв">уничтожению воробьёв</a> как вредителей <a href="/wiki/%D0%A1%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%BE%D0%B5_%D1%85%D0%BE%D0%B7%D1%8F%D0%B9%D1%81%D1%82%D0%B2%D0%BE" title="Сельское хозяйство">сельского хозяйства</a>, проведённая в <a href="/wiki/%D0%9A%D0%B8%D1%82%D0%B0%D0%B9" title="Китай">Китае</a> в 1950-е годы. В Китае обитает близкий родственник домового воробья — <a href="/wiki/%D0%9F%D0%BE%D0%BB%D0%B5%D0%B2%D0%BE%D0%B9_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D0%B5%D0%B9" title="Полевой воробей">полевой воробей</a>, который живёт и в городах<sup class="reference" id="cite_ref-:0_18-0"><a href="#cite_note-:0-18">[18]</a></sup>. В ходе развёрнутой в марте-апреле 1958 года кампании, только за три дня в <a href="/wiki/%D0%9F%D0%B5%D0%BA%D0%B8%D0%BD" title="Пекин">Пекине</a> и <a href="/wiki/%D0%A8%D0%B0%D0%BD%D1%85%D0%B0%D0%B9" title="Шанхай">Шанхае</a> было уничтожено 900 тыс. птиц, а к первой декаде ноября того же года в Китае, по неполной статистике, было истреблено 1,96 млрд полевых воробьёв и других мелких птиц. Однако эта борьба привела к массовому распространению насекомых-вредителей (саранчи и прочих) весной и летом 1959 года в городах и, особенно, в сельской местности. 18 марта 1960 года <a href="/wiki/%D0%9C%D0%B0%D0%BE_%D0%A6%D0%B7%D1%8D%D0%B4%D1%83%D0%BD" title="Мао Цзэдун">Мао Цзэдуном</a> было принято личное решение о приостановке борьбы с воробьями<sup class="reference" id="cite_ref-:0_18-1"><a href="#cite_note-:0-18">[18]</a></sup><sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup><sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup><sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup>. Воробьи в Шанхае, по свидетельству очевидца, встречаются сейчас очень редко<sup class="reference" id="cite_ref-:0_18-2"><a href="#cite_note-:0-18">[18]</a></sup>.
</p><p>Известны памятники воробью, сооружённые человеком в знак благодарности за помощь со стороны этой птицы в борьбе против насекомых-вредителей. Первый такой памятник был воздвигнут в середине XIX века в <a href="/wiki/%D0%91%D0%BE%D1%81%D1%82%D0%BE%D0%BD" title="Бостон">Бостоне</a>. В 2003 году аналогичный монумент был установлен республиканской общественной организацией «Охрана птиц Беларуси» в городе <a href="/wiki/%D0%91%D0%B0%D1%80%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87%D0%B8" title="Барановичи">Барановичи</a> в рамках национальной кампании «Домовый воробей — птица 2003 года»<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup>.
</p><p>Мясо воробья твёрдо и безвкусно, потому редко употребляется в пищу<sup class="reference" id="cite_ref-ESBE_3-16"><a href="#cite_note-ESBE-3">[3]</a></sup>.
</p><p>Воробьёв иногда содержат в клетках любители певчих птиц, при выращивании из птенцов они становятся ручными. Сложность в том, что птенцов требуется кормить насекомыми. При совместном содержании с другими певчими птицами обучаются их пению<sup class="reference" id="cite_ref-:1_12-4"><a href="#cite_note-:1-12">[12]</a></sup>. Писатель-натуралист <a href="/wiki/%D0%A1%D0%B5%D1%82%D0%BE%D0%BD-%D0%A2%D0%BE%D0%BC%D0%BF%D1%81%D0%BE%D0%BD,_%D0%AD%D1%80%D0%BD%D0%B5%D1%81%D1%82" title="Сетон-Томпсон, Эрнест">Э. Сетон-Томпсон</a> описывает историю самца воробья, которого высидели и вырастили <a href="/wiki/%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D1%8F%D1%8F_%D0%BA%D0%B0%D0%BD%D0%B0%D1%80%D0%B5%D0%B9%D0%BA%D0%B0" title="Домашняя канарейка">домашние канарейки</a> из воробьиного яйца, подложенного в их гнездо. Самец воробья смог обучиться пению самцов канареек. Если ему не удавалось превзойти кенара в певческом таланте, то он в драке старался заставить конкурента умолкнуть<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup>.
</p>
<h3><span id=".D0.98.D1.81.D1.87.D0.B5.D0.B7.D0.BD.D0.BE.D0.B2.D0.B5.D0.BD.D0.B8.D0.B5_.D0.B2.D0.BE.D1.80.D0.BE.D0.B1.D1.8C.D1.91.D0.B2"></span><span class="mw-headline" id="Исчезновение_воробьёв">Исчезновение воробьёв</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%BE%D0%B2%D1%8B%D0%B9_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D0%B5%D0%B9&amp;veaction=edit&amp;section=11" title="Редактировать раздел «Исчезновение воробьёв»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%BE%D0%B2%D1%8B%D0%B9_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D0%B5%D0%B9&amp;action=edit&amp;section=11" title="Редактировать код раздела «Исчезновение воробьёв»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Существует мнение, что домовый воробей как один из самых ближайших пернатых соседей человека может выступать в роли «биологического индикатора», и по состоянию <a href="/wiki/%D0%9F%D0%BE%D0%BF%D1%83%D0%BB%D1%8F%D1%86%D0%B8%D1%8F" title="Популяция">популяции</a> этого вида можно судить об изменениях в окружающей среде. К началу XXI века общеевропейская популяция домового воробья значительно сократилась<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup>, а в ряде стран этот вид оказался на грани исчезновения<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup>. Так, с середины 1980-х годов популяция воробьёв в Чехии уменьшилась на 75 %<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup>, в Великобритании — на 68 %<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup>, причём в крупных городах Британии они практически исчезли<sup class="reference" id="cite_ref-svobodanewsu_28-0"><a href="#cite_note-svobodanewsu-28">[28]</a></sup>. Основная причина — благоустройство городов, особенно частый покос газонов, уничтожающий среду обитания насекомых — основной пищи, которой воробьи кормят своих птенцов<sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[29]</a></sup>. Вредят также болезни, вызванные высоким содержанием пестицидов в злаках, исчезновение мест гнездования в городах из-за их реконструкции, уменьшение зелёных городских зон и уничтожение деревьев и кустарников, что опять-таки ведёт к гибели насекомых, необходимых для выкармливания птенцов<sup class="reference" id="cite_ref-svobodanewsu_28-1"><a href="#cite_note-svobodanewsu-28">[28]</a></sup><sup class="reference" id="cite_ref-30"><a href="#cite_note-30">[30]</a></sup><sup class="reference" id="cite_ref-31"><a href="#cite_note-31">[31]</a></sup>.
</p><p>В то же время в <a href="/wiki/%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D1%8F" title="Австралия">Австралии</a> численность завезённых из <a href="/wiki/%D0%A1%D1%82%D0%B0%D1%80%D1%8B%D0%B9_%D0%A1%D0%B2%D0%B5%D1%82" title="Старый Свет">Старого Света</a> воробьёв постоянно растёт, что грозит серьёзно нарушить <a href="/wiki/%D0%AD%D0%BA%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F" title="Экология">экологический</a> баланс на <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%82%D0%B8%D0%BD%D0%B5%D0%BD%D1%82" title="Континент">континенте</a>. В связи с этим в этой стране практикуется отстрел диких воробьёв, но особого успеха он не приносит<sup class="reference" id="cite_ref-32"><a href="#cite_note-32">[32]</a></sup>.
</p>
<h2><span id=".D0.92_.D0.BA.D1.83.D0.BB.D1.8C.D1.82.D1.83.D1.80.D0.B5_.D0.B8_.D1.82.D0.B5.D1.85.D0.BD.D0.B8.D0.BA.D0.B5"></span><span class="mw-headline" id="В_культуре_и_технике">В культуре и технике</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%BE%D0%B2%D1%8B%D0%B9_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D0%B5%D0%B9&amp;veaction=edit&amp;section=12" title="Редактировать раздел «В культуре и технике»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%BE%D0%B2%D1%8B%D0%B9_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D0%B5%D0%B9&amp;action=edit&amp;section=12" title="Редактировать код раздела «В культуре и технике»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>

<p>Невозможно перечислить все примеры встречаемости воробья в культуре; ниже даны лишь некоторые из них:
</p>
<ul><li>В своем стихотворении «Гимн Афродите» древнегреческая поэтесса <a href="/wiki/%D0%A1%D0%B0%D0%BF%D1%84%D0%BE" title="Сапфо">Сапфо</a> описывает колесницу <a href="/wiki/%D0%90%D1%84%D1%80%D0%BE%D0%B4%D0%B8%D1%82%D0%B0" title="Афродита">богини любви</a>, как запряжённую воробьями.</li>
<li>Наиболее популярные стихотворения, посвящённые воробьям и вызвавшие множество подражаний (<a href="/wiki/%D0%9E%D0%B2%D0%B8%D0%B4%D0%B8%D0%B9" title="Овидий">Овидий</a>, <a class="mw-redirect" href="/wiki/%D0%94%D0%B6%D0%BE%D0%BD_%D0%A1%D0%BA%D0%B5%D0%BB%D1%82%D0%BE%D0%BD" title="Джон Скелтон">Джон Скелтон</a>, <a href="/wiki/%D0%94%D0%B5%D0%BB%D1%8C%D0%B2%D0%B8%D0%B3,_%D0%90%D0%BD%D1%82%D0%BE%D0%BD_%D0%90%D0%BD%D1%82%D0%BE%D0%BD%D0%BE%D0%B2%D0%B8%D1%87" title="Дельвиг, Антон Антонович">Дельвиг</a>, <a href="/wiki/%D0%92%D0%BE%D1%81%D1%82%D0%BE%D0%BA%D0%BE%D0%B2,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%BE%D1%84%D0%BE%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Востоков, Александр Христофорович">А. Востоков</a> и проч.)<sup class="reference" id="cite_ref-гас_33-0"><a href="#cite_note-гас-33">[33]</a></sup> и живописных произведений (многочисленные картины «Лесбия с воробьём»), написаны древнеримским поэтом <a class="mw-redirect" href="/wiki/%D0%9A%D0%B0%D1%82%D1%83%D0%BB%D0%BB" title="Катулл">Катуллом</a>, воспевавшим («<a href="/wiki/%D0%9A_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D1%8C%D1%8E_%D0%9B%D0%B5%D1%81%D0%B1%D0%B8%D0%B8" title="К воробью Лесбии">К воробью Лесбии</a>» и «<a href="/wiki/%D0%9D%D0%B0_%D1%81%D0%BC%D0%B5%D1%80%D1%82%D1%8C_%D0%B2%D0%BE%D1%80%D0%BE%D0%B1%D1%8C%D1%8F_%D0%9B%D0%B5%D1%81%D0%B1%D0%B8%D0%B8" title="На смерть воробья Лесбии">На смерть воробья Лесбии</a>») домашнюю птичку своей возлюбленной <a href="/wiki/%D0%9B%D0%B5%D1%81%D0%B1%D0%B8%D1%8F" title="Лесбия">Лесбии</a>. Любопытно, что в честь воробья Лесбии получил своё имя небольшой род <i>Lesbia</i> в семействе <a href="/wiki/%D0%9A%D0%BE%D0%BB%D0%B8%D0%B1%D1%80%D0%B8" title="Колибри">колибри</a>.</li>
<li>В Новом Завете Иисус говорит, что даже воробей не может упасть с неба без воли господней (Лука 12:6; Матфей 10:29).</li>
<li>В книге <a href="/wiki/%D0%9B%D0%BE_%D0%93%D1%83%D0%B0%D0%BD%D1%8C%D1%87%D0%B6%D1%83%D0%BD" title="Ло Гуаньчжун">Ло Гуаньчжуна</a> «<a href="/wiki/%D0%A2%D1%80%D0%BE%D0%B5%D1%86%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B8%D0%B5_(%D1%80%D0%BE%D0%BC%D0%B0%D0%BD)" title="Троецарствие (роман)">Троецарствие</a>» рассказывается, как императрица увидела во сне яшмового воробья и родила Шуня, а также рассказывается о находке статуи бронзового воробья<sup class="reference" id="cite_ref-34"><a href="#cite_note-34">[34]</a></sup>.</li>
<li>В рассказах летописей о том, как <a class="mw-redirect" href="/wiki/%D0%9A%D0%BD%D1%8F%D0%B3%D0%B8%D0%BD%D1%8F_%D0%9E%D0%BB%D1%8C%D0%B3%D0%B0" title="Княгиня Ольга">княгиня Ольга</a> отомстила за смерть мужа, воробьи (наряду с голубями) выступают одним из инструментов — правительница попросила у своих врагов дани: по несколько птиц от каждого дома, затем привязала к каждому из них горящую паклю и выпустила их на свободу. Птицы вернулись под родные кровли и таким образом сожгли город дотла.</li>
<li>В рассказе <a href="/wiki/%D0%9A%D1%83%D0%BF%D1%80%D0%B8%D0%BD,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87" title="Куприн, Александр Иванович">Куприна</a> «Воробей» эта птица прилетает на свежую могилу и выступает символом молодой жизни, вызывая у героя противоречивые чувства.</li>
<li><a href="/wiki/%D0%9C%D0%B0%D0%BC%D0%B8%D0%BD-%D0%A1%D0%B8%D0%B1%D0%B8%D1%80%D1%8F%D0%BA,_%D0%94%D0%BC%D0%B8%D1%82%D1%80%D0%B8%D0%B9_%D0%9D%D0%B0%D1%80%D0%BA%D0%B8%D1%81%D0%BE%D0%B2%D0%B8%D1%87" title="Мамин-Сибиряк, Дмитрий Наркисович">Мамин-Сибиряк</a> написал «Сказку про Воробья Воробеича, Ерша Ершовича и весёлого трубочиста Яшу».</li>
<li>Тоскующий воробей фигурирует в стихах <a href="/wiki/%D0%98%D0%B1%D1%81%D0%B5%D0%BD,_%D0%93%D0%B5%D0%BD%D1%80%D0%B8%D0%BA" title="Ибсен, Генрик">Ибсена</a> «Жизненные осложнения»<sup class="reference" id="cite_ref-35"><a href="#cite_note-35">[35]</a></sup>.</li>
<li>Воробей — персонаж басни <a href="/wiki/%D0%9C%D0%B8%D1%85%D0%B0%D0%BB%D0%BA%D0%BE%D0%B2,_%D0%A1%D0%B5%D1%80%D0%B3%D0%B5%D0%B9_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Михалков, Сергей Владимирович">Сергея Михалкова</a> «Непьющий воробей».</li>
<li><a href="/wiki/%D0%9F%D0%B0%D1%83%D1%81%D1%82%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9,_%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87" title="Паустовский, Константин Георгиевич">Константин Паустовский</a> написал сказку «Растрёпанный воробей».</li>
<li>«Воробей» — детский журнал, издававшийся <a href="/wiki/%D0%9C%D0%B0%D1%80%D1%88%D0%B0%D0%BA,_%D0%A1%D0%B0%D0%BC%D1%83%D0%B8%D0%BB_%D0%AF%D0%BA%D0%BE%D0%B2%D0%BB%D0%B5%D0%B2%D0%B8%D1%87" title="Маршак, Самуил Яковлевич">Маршаком</a>, у него же есть стихотворение «Воробей в зоопарке»</li>
<li>Существует крылатое выражение «стреляный воробей», обозначающее опытного, бывалого человека.</li>
<li>Киноперсонаж пират капитан Джек Спэрроу (<a href="/wiki/%D0%94%D0%B6%D0%B5%D0%BA_%D0%92%D0%BE%D1%80%D0%BE%D0%B1%D0%B5%D0%B9" title="Джек Воробей">Джек Воробей</a>) заимствует своё прозвище у птицы; у <a href="/wiki/%D0%91%D1%83%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%B8,_%D0%A7%D0%B0%D1%80%D0%BB%D1%8C%D0%B7" title="Буковски, Чарльз">Чарльза Буковски</a> в «Макулатуре» есть персонаж Красный Воробей; персонаж по прозвищу Воробей есть в «<a class="mw-redirect" href="/wiki/%D0%A0%D0%B5%D1%81%D0%BF%D1%83%D0%B1%D0%BB%D0%B8%D0%BA%D0%B0_%D0%A8%D0%BA%D0%B8%D0%B4_(%D0%BF%D0%BE%D0%B2%D0%B5%D1%81%D1%82%D1%8C)" title="Республика Шкид (повесть)">Республике Шкид</a>»; у <a href="/wiki/%D0%A2%D0%BE%D0%BB%D1%81%D1%82%D0%BE%D0%B9,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B5%D0%B2%D0%B8%D1%87" title="Толстой, Алексей Николаевич">А. Н. Толстого</a> в «Мираже»; у <a href="/wiki/%D0%9A%D0%B0%D0%BB%D0%B5%D0%B4%D0%B8%D0%BD,_%D0%A1%D0%B5%D1%80%D0%B3%D0%B5%D0%B9_%D0%95%D0%B2%D0%B3%D0%B5%D0%BD%D1%8C%D0%B5%D0%B2%D0%B8%D1%87" title="Каледин, Сергей Евгеньевич">Сергея Каледина</a> в «Смиренном кладбище»; также называет одного из своих персонажей <a class="mw-redirect" href="/wiki/%D0%A1%D0%B8%D0%BD%D0%BA%D0%B5%D0%BD_%D0%A5%D0%BE%D0%BF%D0%BF" title="Синкен Хопп">Синкен Хопп</a> («Волшебный мелок»); <a class="mw-redirect" href="/wiki/%D0%AD%D0%BB%D0%B5%D0%BE%D0%BD%D0%BE%D1%80%D0%B0_%D0%A0%D0%B0%D1%82%D0%BA%D0%B5%D0%B2%D0%B8%D1%87" title="Элеонора Раткевич">Элеонора Раткевич</a> («Рукоять меча»). Также Воробей — прозвище Воробьёва в к/ф «<a href="/wiki/9_%D1%80%D0%BE%D1%82%D0%B0_(%D1%84%D0%B8%D0%BB%D1%8C%D0%BC)" title="9 рота (фильм)">Девятая рота</a>».</li>
<li>Псевдоним-фамилия великой <a class="mw-redirect" href="/wiki/%D0%AD%D0%B4%D0%B8%D1%82_%D0%9F%D0%B8%D0%B0%D1%84" title="Эдит Пиаф">Эдит Пиаф</a> означает на парижском арго «воробей».</li>
<li><a href="/wiki/AIM-7_Sparrow" title="AIM-7 Sparrow">AIM-7 Sparrow</a> (с <span style="font-style:normal"><a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a></span><span class="nowrap"> —</span> «Воробей») — <a href="/wiki/%D0%A1%D0%BE%D0%B5%D0%B4%D0%B8%D0%BD%D1%91%D0%BD%D0%BD%D1%8B%D0%B5_%D0%A8%D1%82%D0%B0%D1%82%D1%8B_%D0%90%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B8" title="Соединённые Штаты Америки">американская</a> <a class="mw-redirect" href="/wiki/%D0%A0%D0%B0%D0%BA%D0%B5%D1%82%D0%B0_%C2%AB%D0%B2%D0%BE%D0%B7%D0%B4%D1%83%D1%85-%D0%B2%D0%BE%D0%B7%D0%B4%D1%83%D1%85%C2%BB" title="Ракета «воздух-воздух»">ракета класса «воздух-воздух»</a>.</li></ul>
