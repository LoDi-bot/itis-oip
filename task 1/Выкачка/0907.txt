<p><b>Э́лия Евдоки́я Авгу́ста</b> (<a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Латинский язык">лат.</a> <span lang="la" style="font-style:italic;">Aelia Eudocia Augusta</span>, <a href="/wiki/%D0%94%D1%80%D0%B5%D0%B2%D0%BD%D0%B5%D0%B3%D1%80%D0%B5%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Древнегреческий язык">др.-греч.</a> <span lang="grc"><span style="font-family: palatino linotype, new athena unicode, athena, gentium, code2000, serif; font-size: 105%;">Αἰλία Εὐδοκία Αὐγούστα</span></span>; до <a href="/wiki/%D0%9A%D1%80%D0%B5%D1%89%D0%B5%D0%BD%D0%B8%D0%B5" title="Крещение">крещения</a> <i>Афинаида<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></i>, <a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Латинский язык">лат.</a> <span lang="la" style="font-style:italic;">Athenais</span>, <a href="/wiki/%D0%94%D1%80%D0%B5%D0%B2%D0%BD%D0%B5%D0%B3%D1%80%D0%B5%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Древнегреческий язык">др.-греч.</a> <span lang="grc"><span style="font-family: palatino linotype, new athena unicode, athena, gentium, code2000, serif; font-size: 105%;">Ἀθηναΐς</span></span>; ок. <a class="mw-redirect" href="/wiki/401" title="401">401</a> — <a href="/wiki/20_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="20 октября">20 октября</a> <a href="/wiki/460_%D0%B3%D0%BE%D0%B4" title="460 год">460 года</a>) — супруга византийского императора <a href="/wiki/%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D0%B9_II" title="Феодосий II">Феодосия II Малого</a>. Известна как талантливая поэтесса, некоторое время покровительствовала <a href="/wiki/%D0%9C%D0%BE%D0%BD%D0%BE%D1%84%D0%B8%D0%B7%D0%B8%D1%82%D1%81%D1%82%D0%B2%D0%BE" title="Монофизитство">монофизитам</a>. Провела последние годы своей жизни в <a href="/wiki/%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC" title="Иерусалим">Иерусалиме</a>, занимаясь строительством церквей и благотворительностью. Канонизирована <a href="/wiki/%D0%9F%D1%80%D0%B0%D0%B2%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%BD%D0%B0%D1%8F_%D1%86%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C" title="Православная церковь">Православной церковью</a> в <a href="/wiki/%D0%9B%D0%B8%D0%BA_%D1%81%D0%B2%D1%8F%D1%82%D0%BE%D1%81%D1%82%D0%B8" title="Лик святости">лике</a> <a href="/wiki/%D0%91%D0%BB%D0%B0%D0%B3%D0%BE%D0%B2%D0%B5%D1%80%D0%BD%D1%8B%D0%B9" title="Благоверный">благоверных</a>, память 13 августа (по <a href="/wiki/%D0%AE%D0%BB%D0%B8%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C" title="Юлианский календарь">юлианскому календарю</a>).
</p>

<h2><span id=".D0.9F.D1.80.D0.BE.D0.B8.D1.81.D1.85.D0.BE.D0.B6.D0.B4.D0.B5.D0.BD.D0.B8.D0.B5_.D0.B8_.D0.B1.D1.80.D0.B0.D0.BA_.D1.81_.D0.A4.D0.B5.D0.BE.D0.B4.D0.BE.D1.81.D0.B8.D0.B5.D0.BC"></span><span class="mw-headline" id="Происхождение_и_брак_с_Феодосием">Происхождение и брак с Феодосием</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%95%D0%B2%D0%B4%D0%BE%D0%BA%D0%B8%D1%8F_(%D1%81%D1%83%D0%BF%D1%80%D1%83%D0%B3%D0%B0_%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D1%8F_II)&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Происхождение и брак с Феодосием»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%95%D0%B2%D0%B4%D0%BE%D0%BA%D0%B8%D1%8F_(%D1%81%D1%83%D0%BF%D1%80%D1%83%D0%B3%D0%B0_%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D1%8F_II)&amp;action=edit&amp;section=1" title="Редактировать код раздела «Происхождение и брак с Феодосием»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Евдокия родилась в <a href="/wiki/%D0%90%D1%84%D0%B8%D0%BD%D1%8B" title="Афины">Афинах</a> около <a href="/wiki/401_%D0%B3%D0%BE%D0%B4" title="401 год">401 года</a>,<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> была дочерью <a href="/wiki/%D0%A4%D0%B8%D0%BB%D0%BE%D1%81%D0%BE%D1%84" title="Философ">философа</a> Леонтия (кроме дочери, у него было ещё два сына, <a href="/wiki/%D0%A4%D0%BB%D0%B0%D0%B2%D0%B8%D0%B9_%D0%92%D0%B0%D0%BB%D0%B5%D1%80%D0%B8%D0%B9" title="Флавий Валерий">Валерий</a> и <a class="new" href="/w/index.php?title=%D0%93%D0%B5%D1%81%D1%81%D0%B8%D0%B9_(%D0%BF%D1%80%D0%B5%D1%84%D0%B5%D0%BA%D1%82_%D0%BF%D1%80%D0%B5%D1%82%D0%BE%D1%80%D0%B8%D1%8F_%D0%98%D0%BB%D0%BB%D0%B8%D1%80%D0%B8%D0%B8)&amp;action=edit&amp;redlink=1" title="Гессий (префект претория Иллирии) (страница отсутствует)">Гессий</a>). Её отец был <a class="mw-disambig" href="/wiki/%D0%A0%D0%B8%D1%82%D0%BE%D1%80" title="Ритор">ритором</a> в Афинской академии и, возможно<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>, именно о нём сообщает в своём сочинении историк <a href="/wiki/%D0%9E%D0%BB%D0%B8%D0%BC%D0%BF%D0%B8%D0%BE%D0%B4%D0%BE%D1%80_(%D0%B8%D1%81%D1%82%D0%BE%D1%80%D0%B8%D0%BA)" title="Олимпиодор (историк)">Олимпиодор Фивянин</a>.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> По рождению Евдокия была <a href="/wiki/%D0%AF%D0%B7%D1%8B%D1%87%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Язычество">язычницей</a> и носила имя Афинаида (в честь богини <a href="/wiki/%D0%90%D1%84%D0%B8%D0%BD%D0%B0" title="Афина">Афины</a>). Когда её отец умер, то оставил следующее завещание:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
<i>…отрокам даровал имущество своё — все одежды, и всё золото, и весь скот, и все сосуды, и всех рабов — был он богат и немалым владел имуществом. Дочери же завещал, любя её, только сто золотых монет и предрёк ей счастливую судьбу, &lt;ибо есть у неё&gt; всё превосходящая женская прелесть</i>.<sup class="reference" id="cite_ref-manas_5-0"><a href="#cite_note-manas-5">[5]</a></sup>
</p>
</blockquote>
<p>Согласно византийскому хронисту <a href="/wiki/%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B9_%D0%90%D0%BC%D0%B0%D1%80%D1%82%D0%BE%D0%BB" title="Георгий Амартол">Георгию Амартолу</a>, Леонтий в своём завещании пророчески записал, объясняя малую долю наследства для Афинаиды, «<i>ведь ей достаточно участи её</i>».<sup class="reference" id="cite_ref-amartol_6-0"><a href="#cite_note-amartol-6">[6]</a></sup>
</p>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Pulcheria.jpg"><img class="mw-file-element" data-file-height="233" data-file-width="243" decoding="async" height="163" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/47/Pulcheria.jpg/170px-Pulcheria.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/4/47/Pulcheria.jpg 1.5x" width="170"/></a><figcaption><center>Пульхерия</center></figcaption></figure>
<p>Ранняя биография Евдокии изложена у <a href="/wiki/%D0%98%D0%BE%D0%B0%D0%BD%D0%BD_%D0%9D%D0%B8%D0%BA%D0%B8%D1%83%D1%81%D1%81%D0%BA%D0%B8%D0%B9" title="Иоанн Никиусский">Иоанна Никиусского</a><sup class="reference" id="cite_ref-johnnik_7-0"><a href="#cite_note-johnnik-7">[7]</a></sup> (конец <a href="/wiki/VII_%D0%B2%D0%B5%D0%BA" title="VII век">VII в.</a>), <a href="/wiki/%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B9_%D0%90%D0%BC%D0%B0%D1%80%D1%82%D0%BE%D0%BB" title="Георгий Амартол">Георгия Амартола</a> и в стихотворной хронике <a href="/wiki/XII_%D0%B2%D0%B5%D0%BA" title="XII век">XII века</a> <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD_%D0%9C%D0%B0%D0%BD%D0%B0%D1%81%D1%81%D0%B8%D1%8F" title="Константин Манассия">Константина Манассии</a>. Афинаида просила братьев выдать ей согласно завещанию долю из <a class="mw-redirect" href="/wiki/%D0%9D%D0%B0%D1%81%D0%BB%D0%B5%D0%B4%D1%81%D1%82%D0%B2%D0%BE" title="Наследство">наследства</a>, но получила отказ и ушла жить к тётке по матери, которая увезла её в <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD%D0%BE%D0%BF%D0%BE%D0%BB%D1%8C" title="Константинополь">Константинополь</a> к тётке по отцу. Обе тётки посоветовали Афинаиде обратиться за помощью в разрешении наследственного спора к <a href="/wiki/%D0%9F%D1%83%D0%BB%D1%8C%D1%85%D0%B5%D1%80%D0%B8%D1%8F" title="Пульхерия">Пульхерии</a>, старшей сестре императора <a href="/wiki/%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D0%B9_II" title="Феодосий II">Феодосия II</a><sup class="reference" id="cite_ref-_a8c09c099c460413_8-0"><a href="#cite_note-_a8c09c099c460413-8">[8]</a></sup>.
</p><p>Необходимо отметить, что Пульхерия в это время была занята по просьбе брата поисками ему невесты и удостоила Афинаиду встречей. Юная Афинаида была хороша собой, <a href="/wiki/%D0%98%D0%BE%D0%B0%D0%BD%D0%BD_%D0%9C%D0%B0%D0%BB%D0%B0%D0%BB%D0%B0" title="Иоанн Малала">Иоанн Малала</a> оставил такой отзыв о ней: «<i>Юная женщина с замечательной внешностью, утончённая в манерах, с хорошей фигурой, обладающая даром красноречия, эллинка, девственница и дочь философа</i>».<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> Историк начала XX века <a class="mw-redirect" href="/wiki/%D0%A8%D0%B0%D1%80%D0%BB%D1%8C_%D0%94%D0%B8%D0%BB%D1%8C" title="Шарль Диль">Шарль Диль</a> так представил себе встречу Афинаиды с всемогущей сестрой императора:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
<i>Афинаиде было двадцать лет. Поразительной красотой она обладала, и удивительным сложением, и довольно высоким ростом. Белокурые вьющиеся волосы золотым ореолом обрамляли лицо, ещё больше оттеняя свежесть и нежность её кожи; взгляд у неё был прекрасный, глаза умные и живые, в то же время скромно опускавшиеся; безукоризненной формы греческий нос и грациозная, благородная поступь дополняли обаяние молодой девушки. При этом она ещё умела хорошо говорить; свою просьбу она изложила в совершенстве. Пульхерия пришла в восторг, была сразу побеждена</i><sup class="reference" id="cite_ref-_a1dc27571291bb54_10-0"><a href="#cite_note-_a1dc27571291bb54-10">[10]</a></sup>.
</p>
</blockquote>
<p>Набожная Пульхерия спросила Афинаиду, сохранила ли она девственность, и услышав положительный ответ, представила её императору. Афинаида произвела впечатление на императора, и он согласился с выбором сестры:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
<i>В сем году <a class="mw-redirect" href="/wiki/%D0%90%D1%82%D1%82%D0%B8%D0%BA_(%D0%9F%D0%B0%D1%82%D1%80%D0%B8%D0%B0%D1%80%D1%85_%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD%D0%BE%D0%BF%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9)" title="Аттик (Патриарх Константинопольский)">Аттик</a> крестил Афинаиду, дочь философа Леонтия, и назвал её Евдокией. Согласно выбору Пульхерии, она вышла замуж за Феодосия…</i>
</p>
<div class="ts-Конец_цитаты-source">— <cite><i>Хронография <a href="/wiki/%D0%A4%D0%B5%D0%BE%D1%84%D0%B0%D0%BD_%D0%98%D1%81%D0%BF%D0%BE%D0%B2%D0%B5%D0%B4%D0%BD%D0%B8%D0%BA" title="Феофан Исповедник">Феофана</a></i>, год 5911 / 411 (421)</cite></div></blockquote>
<p>Свадьба состоялась 7 июня <a href="/wiki/421_%D0%B3%D0%BE%D0%B4" title="421 год">421 года</a>. Узнав об этом, братья Евдокии пришли в ужас. Однако она вызвала их в столицу и назначила на высокие посты в имперской администрации, успокоив словами: «<i>Если бы вы не поступили со мной плохо, я бы не оказалась в столице империи и не стала бы императрицей</i>».<sup class="reference" id="cite_ref-johnnik_7-1"><a href="#cite_note-johnnik-7">[7]</a></sup>
</p>
<h2><span id=".D0.96.D0.B8.D0.B7.D0.BD.D1.8C_.D0.B2_.D0.9A.D0.BE.D0.BD.D1.81.D1.82.D0.B0.D0.BD.D1.82.D0.B8.D0.BD.D0.BE.D0.BF.D0.BE.D0.BB.D0.B5"></span><span class="mw-headline" id="Жизнь_в_Константинополе">Жизнь в Константинополе</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%95%D0%B2%D0%B4%D0%BE%D0%BA%D0%B8%D1%8F_(%D1%81%D1%83%D0%BF%D1%80%D1%83%D0%B3%D0%B0_%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D1%8F_II)&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Жизнь в Константинополе»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%95%D0%B2%D0%B4%D0%BE%D0%BA%D0%B8%D1%8F_(%D1%81%D1%83%D0%BF%D1%80%D1%83%D0%B3%D0%B0_%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D1%8F_II)&amp;action=edit&amp;section=2" title="Редактировать код раздела «Жизнь в Константинополе»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Tremissis-Aelia_Eudocia-RIC_0253.jpg"><img class="mw-file-element" data-file-height="227" data-file-width="500" decoding="async" height="136" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/a0/Tremissis-Aelia_Eudocia-RIC_0253.jpg/300px-Tremissis-Aelia_Eudocia-RIC_0253.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/a0/Tremissis-Aelia_Eudocia-RIC_0253.jpg/450px-Tremissis-Aelia_Eudocia-RIC_0253.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/a/a0/Tremissis-Aelia_Eudocia-RIC_0253.jpg 2x" width="300"/></a><figcaption><center>Золотой <a href="/wiki/%D0%A1%D0%BE%D0%BB%D0%B8%D0%B4" title="Солид">тремисс</a> с изображением Евдокии.<br/> Вокруг портрета надпись <br/>AEL EVDO-CIA AVG (Элия Евдокия Августа).</center></figcaption></figure>
<p>К тому моменту, как Евдокия стала женой Феодосия, делами империи уже семь лет управляла его старшая сестра <a href="/wiki/%D0%9F%D1%83%D0%BB%D1%8C%D1%85%D0%B5%D1%80%D0%B8%D1%8F" title="Пульхерия">Пульхерия</a>. Она в 16 лет дала <a href="/wiki/%D0%9E%D0%B1%D0%B5%D1%82" title="Обет">обет</a> безбрачия и за время своего «правления» превратила императорский дворец в подобие <a href="/wiki/%D0%9C%D0%BE%D0%BD%D0%B0%D1%81%D1%82%D1%8B%D1%80%D1%8C" title="Монастырь">монастыря</a>. В этом духе ею был воспитан <a href="/wiki/%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D0%B9_II" title="Феодосий II">Феодосий</a>, и в эту аскетичную среду попала Евдокия, получившая в Афинах эллинское (языческое) воспитание и образование<sup class="reference" id="cite_ref-_928aff6f6a59d94f_11-0"><a href="#cite_note-_928aff6f6a59d94f-11">[11]</a></sup>.
</p><p>В конце <a href="/wiki/422_%D0%B3%D0%BE%D0%B4" title="422 год">422 года</a> Евдокия родила Феодосию дочь <a href="/wiki/%D0%9B%D0%B8%D1%86%D0%B8%D0%BD%D0%B8%D1%8F_%D0%95%D0%B2%D0%B4%D0%BE%D0%BA%D1%81%D0%B8%D1%8F" title="Лициния Евдоксия">Евдоксию</a>, ставшую позже женой императора <a href="/wiki/%D0%92%D0%B0%D0%BB%D0%B5%D0%BD%D1%82%D0%B8%D0%BD%D0%B8%D0%B0%D0%BD_III" title="Валентиниан III">Валентиниана III</a>. Вскоре после рождения дочери <a href="/wiki/2_%D1%8F%D0%BD%D0%B2%D0%B0%D1%80%D1%8F" title="2 января">2 января</a> <a href="/wiki/423_%D0%B3%D0%BE%D0%B4" title="423 год">423 года</a> Феодосий объявил Евдокию <a class="mw-redirect" href="/wiki/%D0%90%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0_(%D1%82%D0%B8%D1%82%D1%83%D0%BB)" title="Августа (титул)">августой</a><sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>. Этим он уравнял её в достоинстве со своей сестрой <a href="/wiki/%D0%9F%D1%83%D0%BB%D1%8C%D1%85%D0%B5%D1%80%D0%B8%D1%8F" title="Пульхерия">Пульхерией</a> и, как отмечают историки, с этого времени влияние Евдокии на Феодосия стало возрастать<sup class="reference" id="cite_ref-_a8c09c099c460417_13-0"><a href="#cite_note-_a8c09c099c460417-13">[13]</a></sup>. Вскоре Евдокия родила вторую дочь, Флаккиллу, умершую в <a href="/wiki/431_%D0%B3%D0%BE%D0%B4" title="431 год">431 году</a><sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup>. Из одной римской надписи делались предположения о рождении у Евдокии сына Аркадия, однако современные историки пришли к выводу, что речь в надписи идёт не о сыне Евдокии<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup>.
</p>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Theodosius_ii.jpg"><img class="mw-file-element" data-file-height="600" data-file-width="448" decoding="async" height="221" src="//upload.wikimedia.org/wikipedia/commons/thumb/5/50/Theodosius_ii.jpg/165px-Theodosius_ii.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/5/50/Theodosius_ii.jpg/248px-Theodosius_ii.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/5/50/Theodosius_ii.jpg/330px-Theodosius_ii.jpg 2x" width="165"/></a><figcaption><center><a href="/wiki/%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D0%B9_II" title="Феодосий II">Феодосий II</a> (бюст, <a href="/wiki/%D0%9B%D1%83%D0%B2%D1%80" title="Лувр">Лувр</a>)</center></figcaption></figure>
<p>Современники высоко отзываются о просвещённой императрице. Живший в одно время с ней церковный историк <a href="/wiki/%D0%A1%D0%BE%D0%BA%D1%80%D0%B0%D1%82_%D0%A1%D1%85%D0%BE%D0%BB%D0%B0%D1%81%D1%82%D0%B8%D0%BA" title="Сократ Схоластик">Сократ Схоластик</a> заметил по поводу празднования византийской победы в <a href="/wiki/422_%D0%B3%D0%BE%D0%B4" title="422 год">422 году</a> над персами: «Да и сама супруга царя написала стихи героическим размером, потому что почиталась женщиной умной»<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup>.
Не без участия Евдокии в <a href="/wiki/425_%D0%B3%D0%BE%D0%B4" title="425 год">425 году</a> в Константинополе был открыт <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD%D0%BE%D0%BF%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D1%83%D0%BD%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D1%82%D0%B5%D1%82" title="Константинопольский университет">университет</a>, в котором первенствующее место было отведено изучению греческой культуры и философии. В <a href="/wiki/428_%D0%B3%D0%BE%D0%B4" title="428 год">428 году</a>, после появления <a href="/wiki/%D0%9D%D0%B5%D1%81%D1%82%D0%BE%D1%80%D0%B8%D0%B0%D0%BD%D1%81%D1%82%D0%B2%D0%BE" title="Несторианство">несторианской</a> ереси, Евдокия встала на сторону мужа, который симпатизировал <a href="/wiki/%D0%9D%D0%B5%D1%81%D1%82%D0%BE%D1%80%D0%B8%D0%B9" title="Несторий">Несторию</a> и созвал <a href="/wiki/%D0%AD%D1%84%D0%B5%D1%81%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D0%BE%D0%B1%D0%BE%D1%80" title="Эфесский собор">Эфесский собор</a>, не имея цели осудить его<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup> (Пульхерия же не разделяла взгляды брата и поддерживала <a href="/wiki/%D0%9A%D0%B8%D1%80%D0%B8%D0%BB%D0%BB_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9" title="Кирилл Александрийский">Кирилла Александрийского</a>). Несмотря на это, Собор осудил учение <a href="/wiki/%D0%9D%D0%B5%D1%81%D1%82%D0%BE%D1%80%D0%B8%D0%B9" title="Несторий">Нестория</a>, с чем впоследствии согласился и Феодосий. Это был триумф Пульхерии, начинавшей было утрачивать свою роль в делах управления империей<sup class="reference" id="cite_ref-_a8c09c099c460415_18-0"><a href="#cite_note-_a8c09c099c460415-18">[18]</a></sup>.
</p><p>В <a href="/wiki/438_%D0%B3%D0%BE%D0%B4" title="438 год">438 году</a> Евдокия исполнила обет, что в случае супружества дочери она, подражая святой императрице <a href="/wiki/%D0%95%D0%BB%D0%B5%D0%BD%D0%B0_%D0%A0%D0%B0%D0%B2%D0%BD%D0%BE%D0%B0%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F" title="Елена Равноапостольная">Елене</a>, совершит <a href="/wiki/%D0%9F%D0%B0%D0%BB%D0%BE%D0%BC%D0%BD%D0%B8%D1%87%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Паломничество">паломничество</a> в <a href="/wiki/%D0%A1%D0%B2%D1%8F%D1%82%D0%B0%D1%8F_%D0%B7%D0%B5%D0%BC%D0%BB%D1%8F" title="Святая земля">Святую землю</a>.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup> В путешествии в <a href="/wiki/%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC" title="Иерусалим">Иерусалим</a> она провела год:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
После торжественного шествия по всей Азии, где она в <a href="/wiki/%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D1%8F" title="Антиохия">Антиохии</a>, с высоты золотого престола, произнесла собственного сочинения речь в честь города, она прибыла в Иерусалим и была встречена <a class="mw-redirect" href="/wiki/%D0%9C%D0%B5%D0%BB%D0%B0%D0%BD%D0%B8%D1%8F_%D0%A0%D0%B8%D0%BC%D0%BB%D1%8F%D0%BD%D1%8B%D0%BD%D1%8F" title="Мелания Римляныня">Меланией Младшей</a>, пользовавшейся незадолго перед тем её гостеприимством в Константинополе. В Иерусалиме она раздала обильные дары, пожертвовала для <a href="/wiki/%D0%93%D0%BE%D0%BB%D0%B3%D0%BE%D1%84%D0%B0" title="Голгофа">Голгофского</a> придела великолепный золотой крест, осыпанный драгоценными камнями, присутствовала при освящении церкви, выстроенной Меланией и, пробыв несколько месяцев в Святом Граде, возвратилась в Константинополь.<sup class="reference" id="cite_ref-evf_20-0"><a href="#cite_note-evf-20">[20]</a></sup>
</p>
</blockquote>
<p>В Антиохии Евдокия закончила свою речь к горожанам словами: «<i>Горжусь и тем, что мой род — от вашего рода и крови</i>»<sup class="reference" id="cite_ref-evag_21-0"><a href="#cite_note-evag-21">[21]</a></sup> (подразумевая своё родство с греческими колонистами, основавшими город). По сообщению <a href="/wiki/%D0%95%D0%B2%D0%B0%D0%B3%D1%80%D0%B8%D0%B9_%D0%A1%D1%85%D0%BE%D0%BB%D0%B0%D1%81%D1%82%D0%B8%D0%BA" title="Евагрий Схоластик">Евагрия Схоластика</a>, горожане почтили её «<i>искусно выработанной одной статуей</i>», а император Феодосий по её просьбе «<i>к этому городу присоединил значительную часть местности, проведя его стену до самых ворот дафнийского предместья</i>».<sup class="reference" id="cite_ref-evag_21-1"><a href="#cite_note-evag-21">[21]</a></sup>
</p><p>Среди <a href="/wiki/%D0%A0%D0%B5%D0%BB%D0%B8%D0%BA%D0%B2%D0%B8%D1%8F" title="Реликвия">реликвий</a>, привезённых Евдокией из Иерусалима в Константинополь, были часть <a href="/wiki/%D0%9C%D0%BE%D1%89%D0%B8" title="Мощи">мощей</a> <a href="/wiki/%D0%A1%D1%82%D0%B5%D1%84%D0%B0%D0%BD_%D0%9F%D0%B5%D1%80%D0%B2%D0%BE%D0%BC%D1%83%D1%87%D0%B5%D0%BD%D0%B8%D0%BA" title="Стефан Первомученик">первомученика Стефана</a>, <a href="/wiki/%D0%92%D0%B5%D1%80%D0%B8%D0%B3%D0%B8_%D1%81%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%9F%D0%B5%D1%82%D1%80%D0%B0" title="Вериги святого Петра">вериги апостола Петра</a> (одну из цепей она отправила дочери в Рим) и <a class="mw-redirect" href="/wiki/%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%B0_%D0%91%D0%BE%D0%B6%D0%B8%D0%B5%D0%B9_%D0%9C%D0%B0%D1%82%D0%B5%D1%80%D0%B8" title="Влахернская икона Божией Матери">Влахернская икона Божией Матери</a><sup class="reference" id="cite_ref-_a8c09c099c460414_22-0"><a href="#cite_note-_a8c09c099c460414-22">[22]</a></sup><sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup>.
</p><p>После возвращения в столицу влияние Евдокии заметно усилилось: префектом претория Востока стал её фаворит Кир из Панополя, учёный и поэт, близкий царице по эллинской культуре<sup class="reference" id="cite_ref-_a8c09c099c460414_22-1"><a href="#cite_note-_a8c09c099c460414-22">[22]</a></sup>. Она заручилась поддержкой <a href="/wiki/%D0%95%D0%B2%D0%BD%D1%83%D1%85" title="Евнух">евнуха</a> Хрисанфия, имевшего большое влияние на Феодосия, и вместе с ним приступила к отстранению Пульхерии от управления империей<sup class="reference" id="cite_ref-_a8c09c099c46041b_24-0"><a href="#cite_note-_a8c09c099c46041b-24">[24]</a></sup>. <a href="/wiki/%D0%A4%D0%B5%D0%BE%D1%84%D0%B0%D0%BD_%D0%98%D1%81%D0%BF%D0%BE%D0%B2%D0%B5%D0%B4%D0%BD%D0%B8%D0%BA" title="Феофан Исповедник">Феофан Исповедник</a> пишет, что сам Хрисанфий вошёл в доверие к Евдокии, чтобы использовать её для собственной цели по смене константинопольского патриарха, чему противилась <a href="/wiki/%D0%9F%D1%83%D0%BB%D1%8C%D1%85%D0%B5%D1%80%D0%B8%D1%8F" title="Пульхерия">Пульхерия</a><sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup>.
</p><p>По наущению евнуха Евдокия попросила Феодосия, чтобы тот повелел патриарху <a class="mw-redirect" href="/wiki/%D0%A4%D0%BB%D0%B0%D0%B2%D0%B8%D0%B0%D0%BD_(%D0%9F%D0%B0%D1%82%D1%80%D0%B8%D0%B0%D1%80%D1%85_%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD%D0%BE%D0%BF%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9)" title="Флавиан (Патриарх Константинопольский)">Флавиану</a> посвятить Пульхерию, как давшую обет девственности, в <a class="mw-redirect" href="/wiki/%D0%94%D0%B8%D0%B0%D0%BA%D0%BE%D0%BD%D0%B8%D1%81%D1%81%D0%B0" title="Диаконисса">диакониссы</a>.<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup> Флавиан не стал противиться воле императора, но попросил Пульхерию «<i>не допускать его к себе, чтобы не быть ему принуждену сделать ей какое-либо огорчение</i>». <a href="/wiki/%D0%9F%D1%83%D0%BB%D1%8C%D1%85%D0%B5%D1%80%D0%B8%D1%8F" title="Пульхерия">Пульхерия</a>, узнав о заговоре, не стала бороться и «<i>удалилась в Евдомон на покой</i>», а Феодосий и Евдокия «<i>вознегодовали на Флавиана за то, будто он открывает тайны дома их</i>». Такие действия Евдокии вызвали негодование у клерикальной партии, привыкшей управлять делами при Пульхерии, что привело к опале императрицы<sup class="reference" id="cite_ref-_a8c09c099c46041b_24-1"><a href="#cite_note-_a8c09c099c46041b-24">[24]</a></sup>.
</p>
<h2><span id=".D0.9E.D0.BF.D0.B0.D0.BB.D0.B0"></span><span class="mw-headline" id="Опала">Опала</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%95%D0%B2%D0%B4%D0%BE%D0%BA%D0%B8%D1%8F_(%D1%81%D1%83%D0%BF%D1%80%D1%83%D0%B3%D0%B0_%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D1%8F_II)&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Опала»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%95%D0%B2%D0%B4%D0%BE%D0%BA%D0%B8%D1%8F_(%D1%81%D1%83%D0%BF%D1%80%D1%83%D0%B3%D0%B0_%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D1%8F_II)&amp;action=edit&amp;section=3" title="Редактировать код раздела «Опала»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Около <a href="/wiki/441_%D0%B3%D0%BE%D0%B4" title="441 год">441 года</a> Евдокия была заподозрена мужем в любовной связи с Павлином, другом детства Феодосия II. <a href="/wiki/%D0%98%D0%BE%D0%B0%D0%BD%D0%BD_%D0%9C%D0%B0%D0%BB%D0%B0%D0%BB%D0%B0" title="Иоанн Малала">Иоанн Малала</a>, автор «Хронографии» (<a href="/wiki/VI_%D0%B2%D0%B5%D0%BA" title="VI век">VI век)</a>, первым передал эту историю в романтическом виде:
</p>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:34-manasses-chronicle.jpg"><img class="mw-file-element" data-file-height="1382" data-file-width="2013" decoding="async" height="213" src="//upload.wikimedia.org/wikipedia/commons/thumb/0/08/34-manasses-chronicle.jpg/310px-34-manasses-chronicle.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/0/08/34-manasses-chronicle.jpg/465px-34-manasses-chronicle.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/0/08/34-manasses-chronicle.jpg/620px-34-manasses-chronicle.jpg 2x" width="310"/></a><figcaption><center>История опалы Евдокии <br/>(миниатюра из хроники <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD_%D0%9C%D0%B0%D0%BD%D0%B0%D1%81%D1%81%D0%B8%D1%8F" title="Константин Манассия">Константина Манассии</a>)</center></figcaption></figure>
<p>
</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
Некий бедняк принёс Феодосию фригийское яблоко необыкновенно большого размера. Император удивился ему, так же как и весь его двор. Немедленно император выдал 150 <a href="/wiki/%D0%A1%D0%BE%D0%BB%D0%B8%D0%B4" title="Солид">номисм</a> человеку, принёсшему яблоко, и отослал яблоко августе Евдокии. Августа переслала его к магистру Павлину как другу императора. [Магистр Павлин из-за больной ноги оставался дома] Но Павлин, не зная, что император послал яблоко императрице, принял его и отослал императору Феодосию, как только тот вернулся во дворец. Когда император получил яблоко, он признал его и спрятал. Вызвав августу, он спросил её: «<i>Где яблоко, что я послал тебе?</i>» — Она ответила: «<i>Я его съела</i>». Тогда он, заклиная своим спасением, призвал её под клятвой признаться, съела ли она яблоко, или кому-нибудь послала. И она поклялась: «<i>Я никому его не посылала, но съела.</i>» Тогда император приказал принести яблоко и показал ей. Он был разгневан на неё, подозревая её в том, что она послала яблоко по причине её влюблённости в Павлина и потому отрицала это. По этому поводу Феодосий казнил Павлина. Императрица Евдокия переживала и чувствовала себя оскорблённой, потому что везде знали, что Павлина казнили из-за неё, так как был он очень красивый юноша.<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup>
</p>
<div class="ts-Конец_цитаты-source">— <cite><i><a href="/wiki/%D0%98%D0%BE%D0%B0%D0%BD%D0%BD_%D0%9C%D0%B0%D0%BB%D0%B0%D0%BB%D0%B0" title="Иоанн Малала">Иоанн Малала</a></i>, кн. XIV</cite></div></blockquote>
<p>Насколько верна эта история, изложенная в деталях лишь поздними историками, установить не представляется возможным<sup class="reference" id="cite_ref-_a8c09c099c46041b_24-2"><a href="#cite_note-_a8c09c099c46041b-24">[24]</a></sup>, но современник событий <a href="/wiki/%D0%9D%D0%B5%D1%81%D1%82%D0%BE%D1%80%D0%B8%D0%B9" title="Несторий">Несторий</a> пишет, что «демон супружеской измены погрузил императрицу в стыд и унижение»<sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup>.
</p><p>После этих событий Евдокия упросила царя отпустить её в <a href="/wiki/%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC" title="Иерусалим">Иерусалим</a> для молитвы. Феодосий легко согласился на это, так как, по мнению Шарля Диля, уже не чувствовал к Евдокии ничего, кроме ненависти, подозрительности и злобы, разжигаемыми его свитой<sup class="reference" id="cite_ref-_a8c09c099c46041b_24-3"><a href="#cite_note-_a8c09c099c46041b-24">[24]</a></sup>. Житийная литература умалчивает об опале Евдокии, сообщая, что в Иерусалим на поклонение святыням она отправилась только после смерти своего мужа.<sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[29]</a></sup>
Казнь Павлина произошла в <a href="/wiki/444_%D0%B3%D0%BE%D0%B4" title="444 год">444 году</a> в <a href="/wiki/%D0%9A%D0%B0%D0%BF%D0%BF%D0%B0%D0%B4%D0%BE%D0%BA%D0%B8%D1%8F" title="Каппадокия">Каппадокии</a>, куда император выслал его.<sup class="reference" id="cite_ref-30"><a href="#cite_note-30">[30]</a></sup>
</p><p>После того, как Евдокия удалилась в Иерусалим, <a href="/wiki/%D0%9F%D1%83%D0%BB%D1%8C%D1%85%D0%B5%D1%80%D0%B8%D1%8F" title="Пульхерия">Пульхерия</a> попыталась захватить всю её собственность. Согласно хронисту <a href="/wiki/VII_%D0%B2%D0%B5%D0%BA" title="VII век">VII века</a> Иоанну из Никиу<sup class="reference" id="cite_ref-31"><a href="#cite_note-31">[31]</a></sup>, стороннику <a class="mw-redirect" href="/wiki/%D0%9C%D0%BE%D0%BD%D0%BE%D1%84%D0%B8%D0%B7%D0%B8%D1%82%D1%8B" title="Монофизиты">монофизитов</a>, Пульхерия подготовила указ о переходе дворца и прочего имущества императрицы в её собственное владение. Когда Феодосий собрался подписать документ, он заметил дополнение: «<i>Императрица Евдокия становится моей [Пульхерии] рабыней</i>». Разгневанный император отослал сестру и перестал следовать её желаниям.<sup class="reference" id="cite_ref-"><a href="#cite_note-">[32]</a></sup> Впрочем более поздний хронист <a href="/wiki/%D0%A4%D0%B5%D0%BE%D1%84%D0%B0%D0%BD_%D0%98%D1%81%D0%BF%D0%BE%D0%B2%D0%B5%D0%B4%D0%BD%D0%B8%D0%BA" title="Феофан Исповедник">Феофан</a>, прославляющий деяния Пульхерии, обращает историю о несостоявшемся рабстве императрицы в шутку: «<i>Между прочим премудрая Пульхерия раз предложила ему [Феодосию] бумагу об отдаче ей в рабство супруги его, Евдокии, которую он не читавши, подписал, за что потом она укоряла его</i>»<sup class="reference" id="cite_ref-33"><a href="#cite_note-33">[33]</a></sup>.
</p>
<h2><span id=".D0.96.D0.B8.D0.B7.D0.BD.D1.8C_.D0.B2_.D0.98.D0.B5.D1.80.D1.83.D1.81.D0.B0.D0.BB.D0.B8.D0.BC.D0.B5"></span><span class="mw-headline" id="Жизнь_в_Иерусалиме">Жизнь в Иерусалиме</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%95%D0%B2%D0%B4%D0%BE%D0%BA%D0%B8%D1%8F_(%D1%81%D1%83%D0%BF%D1%80%D1%83%D0%B3%D0%B0_%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D1%8F_II)&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Жизнь в Иерусалиме»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%95%D0%B2%D0%B4%D0%BE%D0%BA%D0%B8%D1%8F_(%D1%81%D1%83%D0%BF%D1%80%D1%83%D0%B3%D0%B0_%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D1%8F_II)&amp;action=edit&amp;section=4" title="Редактировать код раздела «Жизнь в Иерусалиме»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Aelia_Eudocia_mosaic.jpg"><img class="mw-file-element" data-file-height="890" data-file-width="418" decoding="async" height="426" src="//upload.wikimedia.org/wikipedia/commons/thumb/1/1d/Aelia_Eudocia_mosaic.jpg/200px-Aelia_Eudocia_mosaic.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/1/1d/Aelia_Eudocia_mosaic.jpg/300px-Aelia_Eudocia_mosaic.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/1/1d/Aelia_Eudocia_mosaic.jpg/400px-Aelia_Eudocia_mosaic.jpg 2x" width="200"/></a><figcaption><center><i>Императрица Евдокия</i>. <br/> <a href="/wiki/%D0%98%D0%BA%D0%BE%D0%BD%D0%B0" title="Икона">Икона</a> из <a href="/wiki/%D0%9C%D1%80%D0%B0%D0%BC%D0%BE%D1%80" title="Мрамор">мрамора</a>, инкрустированная <a href="/wiki/%D0%94%D1%80%D0%B0%D0%B3%D0%BE%D1%86%D0%B5%D0%BD%D0%BD%D1%8B%D0%B5_%D0%BA%D0%B0%D0%BC%D0%BD%D0%B8" title="Драгоценные камни">драгоценными камнями</a>, <a href="/wiki/X_%D0%B2%D0%B5%D0%BA" title="X век">X век</a> — чуть ли не единственный уцелевший предмет в подобной технике. (<a href="/wiki/%D0%A1%D1%82%D0%B0%D0%BC%D0%B1%D1%83%D0%BB" title="Стамбул">Стамбул</a>, Археологический музей, ранее — церковь <a href="/wiki/%D0%9C%D0%BE%D0%BD%D0%B0%D1%81%D1%82%D1%8B%D1%80%D1%8C_%D0%9B%D0%B8%D0%BF%D1%81%D0%B0" title="Монастырь Липса">монастыря Липса</a>) </center></figcaption></figure>
<p>В <a href="/wiki/443_%D0%B3%D0%BE%D0%B4" title="443 год">443 году</a> Евдокия переехала в <a href="/wiki/%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC" title="Иерусалим">Иерусалим</a>. Первое время она жила в нём как императрица, окружённая свитой, где выделялись <a href="/wiki/%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D0%B8%D0%BA" title="Священник">священник</a> Север и <a href="/wiki/%D0%94%D0%B8%D0%B0%D0%BA%D0%BE%D0%BD" title="Диакон">диакон</a> Иоанн, которых она привезла вместе с собой из Константинополя. Правитель Иерусалима Сатурнин донёс об этом Феодосию и тот, всё ещё подозревая в гневе Евдокию в супружеской неверности, «дал повеление отрубить им головы»<sup class="reference" id="cite_ref-34"><a href="#cite_note-34">[34]</a></sup>. Евдокия отомстила доносчику за эту обиду, Сатурнин пал от рук убийц, возможно подкупленных или просто направляемых опальной императрицей<sup class="reference" id="cite_ref-_12a2dc4e90868940_35-0"><a href="#cite_note-_12a2dc4e90868940-35">[35]</a></sup>.
</p><p>После этого в жизни Евдокии начался новый период: Феодосий отозвал из Иерусалима всех составляющих её свиту императорских чиновников и, таким образом, лишил её преимуществ императорского звания.<sup class="reference" id="cite_ref-evf_20-1"><a href="#cite_note-evf-20">[20]</a></sup> Она начала жить как частное лицо, но благодаря оставшимся у неё богатствам прославилась как щедрая благотворительница и устроительница церквей, монастырей и больниц. Рядом с <a href="/wiki/%D0%A5%D1%80%D0%B0%D0%BC_%D0%93%D1%80%D0%BE%D0%B1%D0%B0_%D0%93%D0%BE%D1%81%D0%BF%D0%BE%D0%B4%D0%BD%D1%8F" title="Храм Гроба Господня">базиликой Гроба Господня</a> был построен великолепный дворец для <a class="mw-disambig" href="/wiki/%D0%9F%D0%B0%D1%82%D1%80%D0%B8%D0%B0%D1%80%D1%85_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D1%81%D0%BA%D0%B8%D0%B9" title="Патриарх Иерусалимский">иерусалимских епископов</a>, она восстановила разрушенные более столетия назад стены Иерусалима, считая, что именно к ней относятся строки <a href="/wiki/%D0%9F%D1%81%D0%B0%D0%BB%D0%BE%D0%BC_50" title="Псалом 50">50-го псалма</a>: «<i>Ублажи, Господи, благоволением (по гречески — Евдокия) Твоим, Сион, и пусть воздвигнутся стены Иерусалимские</i>».<sup class="reference" id="cite_ref-amartol_6-1"><a href="#cite_note-amartol-6">[6]</a></sup> Бурная строительная деятельность Евдокии была схожа с деятельностью императрицы <a href="/wiki/%D0%95%D0%BB%D0%B5%D0%BD%D0%B0_%D0%A0%D0%B0%D0%B2%D0%BD%D0%BE%D0%B0%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F" title="Елена Равноапостольная">Елены</a>, в результате чего в легендах произошло их отождествление. Так <a href="/wiki/%D0%9A%D0%BE%D0%BF%D1%82%D1%8B" title="Копты">коптское</a> предание об обретении <a href="/wiki/%D0%96%D0%B8%D0%B2%D0%BE%D1%82%D0%B2%D0%BE%D1%80%D1%8F%D1%89%D0%B8%D0%B9_%D0%9A%D1%80%D0%B5%D1%81%D1%82" title="Животворящий Крест">Животворящего Креста</a> приписывает его находку Евдокии.<sup class="reference" id="cite_ref-36"><a href="#cite_note-36">[36]</a></sup>
</p>
<h3><span id=".D0.9F.D0.BE.D0.BA.D1.80.D0.BE.D0.B2.D0.B8.D1.82.D0.B5.D0.BB.D1.8C.D1.81.D1.82.D0.B2.D0.BE_.D0.BC.D0.BE.D0.BD.D0.BE.D1.84.D0.B8.D0.B7.D0.B8.D1.82.D0.B0.D0.BC"></span><span class="mw-headline" id="Покровительство_монофизитам">Покровительство монофизитам</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%95%D0%B2%D0%B4%D0%BE%D0%BA%D0%B8%D1%8F_(%D1%81%D1%83%D0%BF%D1%80%D1%83%D0%B3%D0%B0_%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D1%8F_II)&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Покровительство монофизитам»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%95%D0%B2%D0%B4%D0%BE%D0%BA%D0%B8%D1%8F_(%D1%81%D1%83%D0%BF%D1%80%D1%83%D0%B3%D0%B0_%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D1%8F_II)&amp;action=edit&amp;section=5" title="Редактировать код раздела «Покровительство монофизитам»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="ts-Врезка tright" role="complementary" style="width:230px;">
<div class="ts-Врезка-содержание">
<p>«<i>…в истории Церкви бывали очень большие размолвки, все участники которых потом оказывались святыми</i>»<sup class="reference" id="cite_ref-37"><a href="#cite_note-37">[37]</a></sup>
</p>
<div class="ts-Врезка-подпись" style="">Слово в день памяти святой императрицы Евдокии</div>
</div>
</div>
<p>Христианство, став государственной религией Римской империи, в <a href="/wiki/V_%D0%B2%D0%B5%D0%BA" title="V век">V веке</a> ещё только формировало свою идеологию, разделяясь периодически на разные течения в попытках трактовать сущность Бога. Общепризнанные теологические взгляды, часто при поддержке власти, канонизировались на соборах, а отклоняющиеся от господствующей линии объявлялись ересями. Хотя ереси преследовались (но без истребления десятков тысяч отступников, как в более поздние века), в целом императорская власть относилась терпимо к инакомыслию, и особенно авторы этой эпохи хвалили за это императора <a href="/wiki/%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D0%B9_II" title="Феодосий II">Феодосия II</a>, отмечая, несмотря на его мягкость и слабый характер, твёрдый тон его религиозной политики<sup class="reference" id="cite_ref-38"><a href="#cite_note-38">[38]</a></sup>.
</p><p>Евдокия окружила себя в Иерусалиме аскетами и монахами, начала интересоваться вопросами христианского богословия, симпатизировать <a class="mw-redirect" href="/wiki/%D0%9C%D0%BE%D0%BD%D0%BE%D1%84%D0%B8%D0%B7%D0%B8%D1%82%D1%8B" title="Монофизиты">монофизитам</a> и их лидеру <a href="/wiki/%D0%94%D0%B8%D0%BE%D1%81%D0%BA%D0%BE%D1%80_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9" title="Диоскор Александрийский">Диоскору Александрийскому</a>, одержавшему на <a href="/wiki/%D0%92%D1%82%D0%BE%D1%80%D0%BE%D0%B9_%D0%AD%D1%84%D0%B5%D1%81%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D0%BE%D0%B1%D0%BE%D1%80" title="Второй Эфесский собор">Эфесском разбойничем соборе</a> в <a href="/wiki/449_%D0%B3%D0%BE%D0%B4" title="449 год">449 году</a> победу над православной партией. В <a href="/wiki/451_%D0%B3%D0%BE%D0%B4" title="451 год">451 году</a> учение монофизитов было осуждено <a href="/wiki/%D0%A5%D0%B0%D0%BB%D0%BA%D0%B8%D0%B4%D0%BE%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D0%BE%D0%B1%D0%BE%D1%80" title="Халкидонский собор">Халкидонским собором</a>, но Евдокия всё равно оставалась ему верна. Согласно Хронографии Феофана «<i>монах Феодосий, муж погибельный, тотчас после Халкидонского собора отправился в Иерусалим, и узнавши, что царица Евдокия расположена к Диоскору, низложенному собором, начал громко кричать против собора, обвиняя его в ниспровержении православной веры, чем привлёк на свою сторону царицу и монахов</i>».<sup class="reference" id="cite_ref-39"><a href="#cite_note-39">[39]</a></sup> То, что Евдокия так легко приняла проповедь Феодосия против Халкидонского собора, исследователи объясняют тем, что она считала этот собор — собором Пульхерии и поэтому была против него.<sup class="reference" id="cite_ref-40"><a href="#cite_note-40">[40]</a></sup> В результате к моменту возвращения иерусалимского патриарха <a class="mw-redirect" href="/wiki/%D0%AE%D0%B2%D0%B5%D0%BD%D0%B0%D0%BB%D0%B8%D0%B9_(%D0%9F%D0%B0%D1%82%D1%80%D0%B8%D0%B0%D1%80%D1%85_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D1%81%D0%BA%D0%B8%D0%B9)" title="Ювеналий (Патриарх Иерусалимский)">Ювеналия</a> (на Эфесском соборе он поддержал Диоскора, а на Халкидонском отрёкся от монофизитов, был оправдан и возвращён на иерусалимскую кафедру) в городе был подготовлен бунт:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
<i>Не теряя времени, Евдокия наняла войско, вооружила монахов, поставила стражу на возведённые ею Иерусалимские стены и, когда Ювеналий, гордясь достигнутым званием патриарха, вступил в Иерусалим, его окружила толпа, настойчиво требовавшая, чтобы он проклял собор и отказался от своей подписи. Патриарх не согласился, и тогда мятежная толпа рассеялась по городу, отворила двери тюрьмы, предала избиению приверженцев Ювеналия и наконец ринулась в базилику гробницы Господней, где провозгласила Феодосия Иерусалимским патриархом</i>.<sup class="reference" id="cite_ref-evf_20-2"><a href="#cite_note-evf-20">[20]</a></sup>
</p>
</blockquote>
<p>Феодосий стал рукополагать новых <a href="/wiki/%D0%95%D0%BF%D0%B8%D1%81%D0%BA%D0%BE%D0%BF" title="Епископ">епископов</a>, пользуясь поддержкой палестинского <a href="/wiki/%D0%9C%D0%BE%D0%BD%D0%B0%D1%88%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Монашество">монашества</a>. Византийский император <a href="/wiki/%D0%9C%D0%B0%D1%80%D0%BA%D0%B8%D0%B0%D0%BD" title="Маркиан">Маркиан</a> направил в Палестину войска, которые 16 января 452 года взяли в осаду <a href="/wiki/%D0%93%D1%80%D0%BE%D0%B1%D0%BD%D0%B8%D1%86%D0%B0_%D0%91%D0%BE%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%B8%D1%86%D1%8B" title="Гробница Богородицы">церковь Гефсимании</a>, куда на праздник собрались отступники. В <a href="/wiki/453_%D0%B3%D0%BE%D0%B4" title="453 год">453 году</a> после 20 месяцев отсутствия патриарх Ювеналий вернулся на свою кафедру.<sup class="reference" id="cite_ref-41"><a href="#cite_note-41">[41]</a></sup> Феодосий бежал в <a class="mw-redirect" href="/wiki/%D0%A1%D0%B8%D0%BD%D0%B0%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D0%BC%D0%BE%D0%BD%D0%B0%D1%81%D1%82%D1%8B%D1%80%D1%8C" title="Синайский монастырь">Синайский монастырь</a>, а Евдокия, невзирая на увещевания дочери Евдоксии, зятя Валентиниана и папы римского <a href="/wiki/%D0%9B%D0%B5%D0%B2_I_(%D0%BF%D0%B0%D0%BF%D0%B0_%D1%80%D0%B8%D0%BC%D1%81%D0%BA%D0%B8%D0%B9)" title="Лев I (папа римский)">Льва Великого</a>, оставалась сторонницей монофизитов<sup class="reference" id="cite_ref-evf_20-3"><a href="#cite_note-evf-20">[20]</a></sup>.
</p><p>История возвращения Евдокии незадолго до смерти в лоно православия весьма подробно описана у историков, а также в агиографической литературе, так как её раскаяние было важно для факта её <a href="/wiki/%D0%9A%D0%B0%D0%BD%D0%BE%D0%BD%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F" title="Канонизация">канонизации</a>. Евдокия раскаялась в своей поддержке учения монофизитов после жизненных неприятностей, которые она посчитала Божьей карой: её зять римский император Валентиниан был убит заговорщиками, а её дочь насильно взял в жены организатор заговора. <a class="mw-redirect" href="/wiki/%D0%92%D0%B0%D0%BD%D0%B4%D0%B0%D0%BB%D1%8B_(%D0%BD%D0%B0%D1%80%D0%BE%D0%B4)" title="Вандалы (народ)">Вандалы</a> в 455 году <a class="mw-redirect" href="/wiki/%D0%97%D0%B0%D1%85%D0%B2%D0%B0%D1%82_%D0%A0%D0%B8%D0%BC%D0%B0_%D0%B2%D0%B0%D0%BD%D0%B4%D0%B0%D0%BB%D0%B0%D0%BC%D0%B8_(455_%D0%B3%D0%BE%D0%B4)" title="Захват Рима вандалами (455 год)">разграбили Рим</a> и увели дочь Евдоксию и внучек Евдокии в плен.<sup class="reference" id="cite_ref-42"><a href="#cite_note-42">[42]</a></sup>
</p><p>Евдокия отправила посланника в <a href="/wiki/%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D1%8F" title="Антиохия">Антиохию</a> к <a href="/wiki/%D0%A1%D0%B8%D0%BC%D0%B5%D0%BE%D0%BD_%D0%A1%D1%82%D0%BE%D0%BB%D0%BF%D0%BD%D0%B8%D0%BA" title="Симеон Столпник">Симеону Столпнику</a> спросить, чем ей умилостивить этот «небесный гнев». Симеон ответил: «<i>Я же весьма удивляюсь тому, что, имея источник вблизи, пренебрегаешь им, и тщилась почерпнуть той же воды издалёка: ты имеешь там богоносного Евфимия, следуй его учению, — и спасёшься</i>».<sup class="reference" id="cite_ref-43"><a href="#cite_note-43">[43]</a></sup> Евдокия обратилась к проживавшему в Палестинской пустыне <a href="/wiki/%D0%95%D0%B2%D1%84%D0%B8%D0%BC%D0%B8%D0%B9_%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B8%D0%B9" title="Евфимий Великий">Евфимию Великому</a>, известному подвижнику своего времени. Он «<i>утешил Евдокию, убедил её в её заблуждениях и возвратил к православию</i>».<sup class="reference" id="cite_ref-evf_20-4"><a href="#cite_note-evf-20">[20]</a></sup> По примеру Евдокии многие из палестинских сторонников монофизитов примирились с патриархом Ювеналием<sup class="reference" id="cite_ref-evf_20-5"><a href="#cite_note-evf-20">[20]</a></sup>.
</p>
<h3><span id=".D0.A1.D0.BC.D0.B5.D1.80.D1.82.D1.8C"></span><span class="mw-headline" id="Смерть">Смерть</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%95%D0%B2%D0%B4%D0%BE%D0%BA%D0%B8%D1%8F_(%D1%81%D1%83%D0%BF%D1%80%D1%83%D0%B3%D0%B0_%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D1%8F_II)&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Смерть»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%95%D0%B2%D0%B4%D0%BE%D0%BA%D0%B8%D1%8F_(%D1%81%D1%83%D0%BF%D1%80%D1%83%D0%B3%D0%B0_%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D1%8F_II)&amp;action=edit&amp;section=6" title="Редактировать код раздела «Смерть»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Во искупление вины за длительную поддержку ереси Евдокия построила на месте смерти первомученика <a href="/wiki/%D0%A1%D1%82%D0%B5%D1%84%D0%B0%D0%BD_%D0%9F%D0%B5%D1%80%D0%B2%D0%BE%D0%BC%D1%83%D1%87%D0%B5%D0%BD%D0%B8%D0%BA" title="Стефан Первомученик">Стефана</a> <a href="/wiki/%D0%91%D0%B0%D0%B7%D0%B8%D0%BB%D0%B8%D0%BA%D0%B0_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%A1%D1%82%D0%B5%D1%84%D0%B0%D0%BD%D0%B0_%D0%B2_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D0%B5" title="Базилика Святого Стефана в Иерусалиме">базилику в его честь</a> — более обширную и богатую, чем храм Гроба Господня.<sup class="reference" id="cite_ref-evf_20-6"><a href="#cite_note-evf-20">[20]</a></sup> Рядом с ней она подготовила себе гробницу.
</p><p>Скончалась Евдокия <a href="/wiki/20_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="20 октября">20 октября</a> <a href="/wiki/460_%D0%B3%D0%BE%D0%B4" title="460 год">460 года</a>, пережив своего мужа <a href="/wiki/%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D0%B9_II" title="Феодосий II">Феодосия</a> († <a class="mw-redirect" href="/wiki/450" title="450">450</a> г.) и соперницу в борьбе за влияние на императора <a href="/wiki/%D0%9F%D1%83%D0%BB%D1%8C%D1%85%D0%B5%D1%80%D0%B8%D1%8F" title="Пульхерия">Пульхерию</a> († <a class="mw-redirect" href="/wiki/453" title="453">453</a> г.). По свидетельству хронистов, «<i>умирая же, она исповедовалась, что не сознается о наговорах &lt;против себя&gt; о Павлине</i>».<sup class="reference" id="cite_ref-amartol_6-2"><a href="#cite_note-amartol-6">[6]</a></sup> Готовясь к смерти, она торопилась с <a href="/wiki/%D0%9E%D1%81%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%B8%D0%B5_%D1%85%D1%80%D0%B0%D0%BC%D0%B0" title="Освящение храма">освящением</a> базилики Святого Стефана и вызвала из лавры святого Ефимия монахов для молитв на своей могилой<sup class="reference" id="cite_ref-evf_20-7"><a href="#cite_note-evf-20">[20]</a></sup>. Евдокия скончалась 20 октября 460 года, её погребение совершил иерусалимский патриарх Анастасий<sup class="reference" id="cite_ref-ПЭ_44-0"><a href="#cite_note-ПЭ-44">[44]</a></sup>.
</p><p>Позднее рядом с императрицей Евдокией была погребена её внучка, бежавшая от <a class="mw-redirect" href="/wiki/%D0%92%D0%B0%D0%BD%D0%B4%D0%B0%D0%BB%D1%8B_(%D0%BD%D0%B0%D1%80%D0%BE%D0%B4)" title="Вандалы (народ)">вандалов</a> и носившая тоже имя <a href="/wiki/%D0%95%D0%B2%D0%B4%D0%BE%D0%BA%D0%B8%D1%8F_(%D0%B4%D0%BE%D1%87%D1%8C_%D0%92%D0%B0%D0%BB%D0%B5%D0%BD%D1%82%D0%B8%D0%BD%D0%B8%D0%B0%D0%BD%D0%B0_III)" title="Евдокия (дочь Валентиниана III)">Евдокия</a>. Их гробницы были разрушены вместе с базиликой святого Стефана при нашествии персов в <a href="/wiki/614_%D0%B3%D0%BE%D0%B4" title="614 год">614 году</a><sup class="reference" id="cite_ref-evf_20-8"><a href="#cite_note-evf-20">[20]</a></sup>.
</p>
<h2><span id=".D0.9B.D0.B8.D1.82.D0.B5.D1.80.D0.B0.D1.82.D1.83.D1.80.D0.BD.D0.B0.D1.8F_.D0.B4.D0.B5.D1.8F.D1.82.D0.B5.D0.BB.D1.8C.D0.BD.D0.BE.D1.81.D1.82.D1.8C"></span><span class="mw-headline" id="Литературная_деятельность">Литературная деятельность</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%95%D0%B2%D0%B4%D0%BE%D0%BA%D0%B8%D1%8F_(%D1%81%D1%83%D0%BF%D1%80%D1%83%D0%B3%D0%B0_%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D1%8F_II)&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Литературная деятельность»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%95%D0%B2%D0%B4%D0%BE%D0%BA%D0%B8%D1%8F_(%D1%81%D1%83%D0%BF%D1%80%D1%83%D0%B3%D0%B0_%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D1%8F_II)&amp;action=edit&amp;section=7" title="Редактировать код раздела «Литературная деятельность»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Евдокия известна как талантливая поэтесса, сочинившая ряд произведений эпическим классическим гекзаметром как на исторические, так и на религиозные темы.<sup class="reference" id="cite_ref-45"><a href="#cite_note-45">[45]</a></sup> Она является единственной поэтессой, включённой константинопольским патриархом <a class="mw-redirect" href="/wiki/%D0%A4%D0%BE%D1%82%D0%B8%D0%B9_I_(%D0%9F%D0%B0%D1%82%D1%80%D0%B8%D0%B0%D1%80%D1%85_%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD%D0%BE%D0%BF%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9)" title="Фотий I (Патриарх Константинопольский)">Фотием</a> в его «<a href="/wiki/%D0%9C%D0%B8%D1%80%D0%B8%D0%BE%D0%B1%D0%B8%D0%B1%D0%BB%D0%B8%D0%BE%D0%BD" title="Мириобиблион">Библиотеку</a>»<sup class="reference" id="cite_ref-алекс_46-0"><a href="#cite_note-алекс-46">[46]</a></sup>.
</p><p>Написанные Евдокий произведения:
</p>
<ul><li>героические стихи об одержанных над персами победах в <a href="/wiki/422_%D0%B3%D0%BE%D0%B4" title="422 год">422 году</a> (утрачены);</li>
<li>стихотворное переложение избранных мест из первых восьми книг <a href="/wiki/%D0%92%D0%B5%D1%82%D1%85%D0%B8%D0%B9_%D0%97%D0%B0%D0%B2%D0%B5%D1%82" title="Ветхий Завет">Ветхого Завета</a> (этими стихами восхищался патриарх Фотий);</li>
<li>стихотворное переложение из пророческих книг <a href="/wiki/%D0%97%D0%B0%D1%85%D0%B0%D1%80%D0%B8%D1%8F_(%D0%BF%D1%80%D0%BE%D1%80%D0%BE%D0%BA)" title="Захария (пророк)">Захарии</a> и <a href="/wiki/%D0%94%D0%B0%D0%BD%D0%B8%D0%B8%D0%BB_(%D0%BF%D1%80%D0%BE%D1%80%D0%BE%D0%BA)" title="Даниил (пророк)">Даниила</a>;</li>
<li>«<i><span lang="el"><span style="font-family: palatino linotype, new athena unicode, athena, gentium, code2000, serif; font-size: 105%;">Όμηρόχεντρα</span></span></i>» (Гомероцентоны) — написанное <a href="/wiki/%D0%93%D0%B5%D0%BA%D0%B7%D0%B0%D0%BC%D0%B5%D1%82%D1%80#Героический_гекзаметр" title="Гекзаметр">героическим гекзаметром</a> стихотворное изложение <a href="/wiki/%D0%91%D0%B8%D0%B1%D0%BB%D0%B8%D1%8F" title="Библия">библейской</a> истории, преимущественно эпизоды из <a href="/wiki/%D0%9A%D0%BD%D0%B8%D0%B3%D0%B0_%D0%91%D1%8B%D1%82%D0%B8%D1%8F" title="Книга Бытия">книги Бытия</a> и земной жизни <a href="/wiki/%D0%98%D0%B8%D1%81%D1%83%D1%81_%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%BE%D1%81" title="Иисус Христос">Иисуса Христа,</a> составленное исключительно из гомеровских строк. Центоны состоят из 2344 строк. Они дают четкое представление о том, кем была Евдокия, и во что она верила. Поэма сочетает её классический афинский уровень образования и христианскую веру. Марк Ашер проанализировал это стихотворение, чтобы понять, почему Евдокия решила использовать гомерические темы как средство, чтобы выразить свои библейские интерпретации. По словам Ашера, Евдокии было необходимо передать человеческий опыт, относящийся к Библии. Она использовала темы из Илиады и Одиссеи, потому что «<i>они содержали всё, что было нужно Евдокии, чтобы рассказать историю Евангелия</i>»<sup class="reference" id="cite_ref-47"><a href="#cite_note-47">[47]</a></sup>. Всякий раз, когда и где Евдокии было необходимо выразить величие, боль, правдивость, лживость, красоту, страдание, траур, признание, понимание, страх или удивление, наиболее подходящими были гомеровские строки, автоматически возникавшие в её памяти, в силу её образования. Гомеровская поэзия Евдокии имеет важное значение для понимания её как образованной женщины-христианки в Восточной Римской империи, и понимание её роли в качестве императрицы. Её классический образовательный фон отчетливо виден в её поэзии, которая захватывает литературным талантом. Она в данном сочинении соединила свою любовь к изучению классической греческой литературы со своими христианскими убеждениями<sup class="reference" id="cite_ref-_a8c0a1099c460c90_48-0"><a href="#cite_note-_a8c0a1099c460c90-48">[48]</a></sup>;</li>
<li>стихотворение в 16 строк, написанное в банях Гадары (Хамат-Гадер, юго-восточнее Тивериадского озера), точно подписанное именем Евдокии и маркированное крестами, восхваляет целебный источник, богиню здоровья <a class="mw-redirect" href="/wiki/%D0%93%D0%B8%D0%B3%D0%B5%D1%8F" title="Гигея">Гигиею</a> и <a class="mw-redirect" href="/wiki/%D0%98%D0%BB%D1%8C%D1%8F_(%D0%BF%D1%80%D0%BE%D1%80%D0%BE%D0%BA)" title="Илья (пророк)">св. Илию</a>;</li></ul>

<ul><li>поэма «<i>Киприан и Юстина</i>» в трех книгах, посвящённая <a href="/wiki/%D0%9A%D0%B8%D0%BF%D1%80%D0%B8%D0%B0%D0%BD_%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9" title="Киприан Антиохийский">Киприану Антиохийскому</a>, была опубликована Евдокией под своим языческим именем Афинаида (сохранился отрывок в 900 строк, найденный во <a href="/wiki/%D0%A4%D0%BB%D0%BE%D1%80%D0%B5%D0%BD%D1%86%D0%B8%D1%8F" title="Флоренция">Флоренции</a> <a href="/wiki/%D0%91%D0%B0%D0%BD%D0%B4%D0%B8%D0%BD%D0%B8,_%D0%90%D0%BD%D0%B4%D0%B6%D0%B5%D0%BB%D0%BE-%D0%9C%D0%B0%D1%80%D0%B8%D1%8F" title="Бандини, Анджело-Мария">Анджело-Мария Бандини</a><sup class="reference" id="cite_ref-49"><a href="#cite_note-49">[49]</a></sup> — это I книга без начала и II книга без конца). В 1860 году её текст был опубликован в <a href="/wiki/Patrologia_Graeca" title="Patrologia Graeca">Греческой патрологии</a> <a href="/wiki/%D0%9C%D0%B8%D0%BD%D1%8C,_%D0%96%D0%B0%D0%BA_%D0%9F%D0%BE%D0%BB%D1%8C" title="Минь, Жак Поль">Миня</a><sup class="reference" id="cite_ref-алекс_46-1"><a href="#cite_note-алекс-46">[46]</a></sup>. В поэме выражена идея борьбы христианства и язычества на примере житийной истории Киприана, бывшего сначала известным <a href="/wiki/%D0%9A%D0%BE%D0%BB%D0%B4%D1%83%D0%BD" title="Колдун">магом</a>, вызывателем <a href="/wiki/%D0%94%D0%B5%D0%BC%D0%BE%D0%BD" title="Демон">демонов</a>, а затем после обращения в христианство <a class="mw-redirect" href="/wiki/%D0%A5%D0%B8%D1%80%D0%BE%D1%82%D0%BE%D0%BD%D0%B8%D1%8F" title="Хиротония">рукоположённым</a> в <a href="/wiki/%D0%95%D0%BF%D0%B8%D1%81%D0%BA%D0%BE%D0%BF" title="Епископ">епископы</a>. Киприан и его возлюбленная Юстина, пройдя путь от взаимной человеческой страсти до любви к Богу, приняли мученическую смерть.<sup class="reference" id="cite_ref-50"><a href="#cite_note-50">[50]</a></sup> По оценке критиков некоторые образы, созданные в этом сочинении Евдокией, «<i>дают предчувствовать прекрасные образы <a class="mw-redirect" href="/wiki/%D0%9C%D0%B8%D0%BB%D1%8C%D1%82%D0%BE%D0%BD" title="Мильтон">Мильтона</a> в его поэме „<a class="mw-redirect" href="/wiki/%D0%9F%D0%BE%D1%82%D0%B5%D1%80%D1%8F%D0%BD%D0%BD%D1%8B%D0%B9_%D0%A0%D0%B0%D0%B9" title="Потерянный Рай">Потерянный Рай</a>“ и даже некоторые черты <a class="mw-redirect" href="/wiki/%D0%A4%D0%B0%D1%83%D1%81%D1%82" title="Фауст">Фауста</a></i>».<sup class="reference" id="cite_ref-51"><a href="#cite_note-51">[51]</a></sup></li></ul>
<h2><span id=".D0.9A.D0.B0.D0.BD.D0.BE.D0.BD.D0.B8.D0.B7.D0.B0.D1.86.D0.B8.D1.8F"></span><span class="mw-headline" id="Канонизация">Канонизация</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%95%D0%B2%D0%B4%D0%BE%D0%BA%D0%B8%D1%8F_(%D1%81%D1%83%D0%BF%D1%80%D1%83%D0%B3%D0%B0_%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D1%8F_II)&amp;veaction=edit&amp;section=8" title="Редактировать раздел «Канонизация»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%95%D0%B2%D0%B4%D0%BE%D0%BA%D0%B8%D1%8F_(%D1%81%D1%83%D0%BF%D1%80%D1%83%D0%B3%D0%B0_%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D1%8F_II)&amp;action=edit&amp;section=8" title="Редактировать код раздела «Канонизация»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Для палестинских монахов и жителей Иерусалима императрица Евдокия стала «новой <a href="/wiki/%D0%95%D0%BB%D0%B5%D0%BD%D0%B0_%D0%A0%D0%B0%D0%B2%D0%BD%D0%BE%D0%B0%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F" title="Елена Равноапостольная">Еленой</a>»<sup class="reference" id="cite_ref-52"><a href="#cite_note-52">[52]</a></sup>. В ближневосточной христианской историографической традиции Евдокия предстаёт как праведная императрица<sup class="reference" id="cite_ref-ПЭ_44-1"><a href="#cite_note-ПЭ-44">[44]</a></sup>. Икона <a href="/wiki/X_%D0%B2%D0%B5%D0%BA" title="X век">X века</a> с изображением императрицы свидетельствует о том, что к этому времени её уже почитали святой. По мнению <a href="/wiki/%D0%90%D0%B3%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F" title="Агиология">агиолога</a> <a href="/wiki/%D0%A1%D0%B5%D1%80%D0%B3%D0%B8%D0%B9_(%D0%A1%D0%BF%D0%B0%D1%81%D1%81%D0%BA%D0%B8%D0%B9)" title="Сергий (Спасский)">Сергия (Спасского)</a>, Евдокия является <a class="mw-redirect" href="/wiki/%D0%9C%D0%B5%D1%81%D1%82%D0%BD%D0%BE%D1%87%D1%82%D0%B8%D0%BC%D1%8B%D0%B9_%D1%81%D0%B2%D1%8F%D1%82%D0%BE%D0%B9" title="Местночтимый святой">местночтимой святой</a> Константинопольской церкви и относится к числу святых, чьи имена взяты из <a href="/wiki/%D0%A1%D0%B8%D0%BD%D0%B0%D0%BA%D1%81%D0%B0%D1%80%D1%8C" title="Синаксарь">синаксаря</a> греческих богослужебных <a href="/wiki/%D0%9C%D0%B8%D0%BD%D0%B5%D1%8F" title="Минея">миней</a><sup class="reference" id="cite_ref-53"><a href="#cite_note-53">[53]</a></sup>.
</p>

<ul><li>Итальянский композитор <a href="/wiki/%D0%92%D0%B8%D0%B2%D0%B0%D0%BB%D1%8C%D0%B4%D0%B8,_%D0%90%D0%BD%D1%82%D0%BE%D0%BD%D0%B8%D0%BE" title="Вивальди, Антонио">Вивальди</a> написал в <a href="/wiki/1728_%D0%B3%D0%BE%D0%B4" title="1728 год">1728 году</a> оперу «Афинаида» («L’Atenaide») по мотивам жизни императрицы Евдокии.</li>
<li>Опера <a href="/wiki/%D0%90%D1%80%D0%B0%D0%B9%D1%8F,_%D0%A4%D1%80%D0%B0%D0%BD%D1%87%D0%B5%D1%81%D0%BA%D0%BE" title="Арайя, Франческо">Франческо Арайи</a> по либретто <a href="/wiki/%D0%91%D0%BE%D0%BD%D0%B5%D0%BA%D0%BA%D0%B8,_%D0%94%D0%B6%D1%83%D0%B7%D0%B5%D0%BF%D0%BF%D0%B5" title="Бонекки, Джузеппе">Джузеппе Бонекки</a> «Венчанная Евдокия, или Феодосий II» («Eudossa incoronata, o sia Teodosio II») поставлена в 1751 году в <a href="/wiki/%D0%A1%D0%B0%D0%BD%D0%BA%D1%82-%D0%9F%D0%B5%D1%82%D0%B5%D1%80%D0%B1%D1%83%D1%80%D0%B3" title="Санкт-Петербург">Санкт-Петербурге</a><sup class="reference" id="cite_ref-54"><a href="#cite_note-54">[54]</a></sup></li>

<li><b>Ефимов, Игорь</b>. <i>Невеста Императора</i> (другое название <i>Пелагий Британец</i>; журнальный вариант: <i>Не мир, но меч. Хроника времён заката</i>).</li>
