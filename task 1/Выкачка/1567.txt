<h1 class="firstHeading mw-first-heading" id="firstHeading"><span class="mw-page-title-main">Радужный флаг</span></h1>
<div class="vector-body" id="bodyContent">

<div class="mw-body-content" id="mw-content-text"><div class="mw-content-ltr mw-parser-output" dir="ltr" lang="ru"><div class="hatnote navigation-not-searchable dabhide">Эта статья — о флагах с цветами радуги. О символе ЛГБТ-движения см. <a href="/wiki/%D0%A0%D0%B0%D0%B4%D1%83%D0%B6%D0%BD%D1%8B%D0%B9_%D1%84%D0%BB%D0%B0%D0%B3_(%D0%9B%D0%93%D0%91%D0%A2)" title="Радужный флаг (ЛГБТ)">Радужный флаг (ЛГБТ)</a>.</div>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Rainbow_flag_breeze.jpg"><img class="mw-file-element" data-file-height="3503" data-file-width="5254" decoding="async" height="200" src="//upload.wikimedia.org/wikipedia/commons/thumb/f/fb/Rainbow_flag_breeze.jpg/300px-Rainbow_flag_breeze.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/f/fb/Rainbow_flag_breeze.jpg/450px-Rainbow_flag_breeze.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/f/fb/Rainbow_flag_breeze.jpg/600px-Rainbow_flag_breeze.jpg 2x" width="300"/></a><figcaption>Символика ЛГБТ.  Один из вариантов радужного флага с шестью полосами (без голубого цвета)</figcaption></figure>
<p><b>Радужный флаг</b> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Rainbow flag</span>) — разноцветный <a href="/wiki/%D0%A4%D0%BB%D0%B0%D0%B3" title="Флаг">флаг</a>, состоящий из палитры <a href="/wiki/%D0%A0%D0%B0%D0%B4%D1%83%D0%B3%D0%B0" title="Радуга">радуги</a>. Встречаются различные варианты флага, но все они основаны на классической последовательности цветов: <a href="/wiki/%D0%9A%D1%80%D0%B0%D1%81%D0%BD%D1%8B%D0%B9_%D1%86%D0%B2%D0%B5%D1%82" title="Красный цвет">красный</a>, <a href="/wiki/%D0%9E%D1%80%D0%B0%D0%BD%D0%B6%D0%B5%D0%B2%D1%8B%D0%B9_%D1%86%D0%B2%D0%B5%D1%82" title="Оранжевый цвет">оранжевый</a>, <a href="/wiki/%D0%96%D1%91%D0%BB%D1%82%D1%8B%D0%B9_%D1%86%D0%B2%D0%B5%D1%82" title="Жёлтый цвет">жёлтый</a>, <a href="/wiki/%D0%97%D0%B5%D0%BB%D1%91%D0%BD%D1%8B%D0%B9_%D1%86%D0%B2%D0%B5%D1%82" title="Зелёный цвет">зелёный</a>, <a href="/wiki/%D0%93%D0%BE%D0%BB%D1%83%D0%B1%D0%BE%D0%B9_%D1%86%D0%B2%D0%B5%D1%82" title="Голубой цвет">голубой</a> и (или) <a href="/wiki/%D0%A1%D0%B8%D0%BD%D0%B8%D0%B9_%D1%86%D0%B2%D0%B5%D1%82" title="Синий цвет">синий</a>, <a href="/wiki/%D0%A4%D0%B8%D0%BE%D0%BB%D0%B5%D1%82%D0%BE%D0%B2%D1%8B%D0%B9_%D1%86%D0%B2%D0%B5%D1%82" title="Фиолетовый цвет">фиолетовый</a>, хотя такое применение сочетаний не следует традиционным европейским канонам <a href="/wiki/%D0%93%D0%B5%D1%80%D0%B0%D0%BB%D1%8C%D0%B4%D0%B8%D0%BA%D0%B0" title="Геральдика">геральдики</a><sup class="reference" id="cite_ref-Символ_1-0"><a href="#cite_note-Символ-1">[1]</a></sup>.
</p><p>Радужные флаги известны в различных культурах и течениях. На сегодняшний день одними из наиболее известных и узнаваемых вариантов являются <i>«Прайд-флаг»</i> (символ <a href="/wiki/%D0%9B%D0%93%D0%91%D0%A2-%D0%B4%D0%B2%D0%B8%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5" title="ЛГБТ-движение">движения ЛГБТ</a>), <i>«Флаг мира»</i> (символ <a href="/wiki/%D0%9F%D0%B0%D1%86%D0%B8%D1%84%D0%B8%D0%B7%D0%BC" title="Пацифизм">движения за мир</a>), флаг движения коренных народов Америки. Схожесть их дизайна нередко приводит к путанице<sup class="reference" id="cite_ref-Инки_2-0"><a href="#cite_note-Инки-2">[2]</a></sup><sup class="reference" id="cite_ref-Альянс_3-0"><a href="#cite_note-Альянс-3">[3]</a></sup><sup class="reference" id="cite_ref-Ватикан_4-0"><a href="#cite_note-Ватикан-4">[4]</a></sup><sup class="reference" id="cite_ref-Боровой_5-0"><a href="#cite_note-Боровой-5">[5]</a></sup><sup class="reference" id="cite_ref-Рел_6-0"><a href="#cite_note-Рел-6">[6]</a></sup>.
</p>

<h2><span id=".D0.98.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.B8.D0.B5_.D0.BF.D1.80.D0.B5.D0.B4.D0.BF.D0.BE.D1.81.D1.8B.D0.BB.D0.BA.D0.B8"></span><span class="mw-headline" id="Исторические_предпосылки">Исторические предпосылки</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A0%D0%B0%D0%B4%D1%83%D0%B6%D0%BD%D1%8B%D0%B9_%D1%84%D0%BB%D0%B0%D0%B3&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Исторические предпосылки»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A0%D0%B0%D0%B4%D1%83%D0%B6%D0%BD%D1%8B%D0%B9_%D1%84%D0%BB%D0%B0%D0%B3&amp;action=edit&amp;section=1" title="Редактировать код раздела «Исторические предпосылки»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:2007-04_Stolberg_(Harz)_58.jpg"><img class="mw-file-element" data-file-height="1975" data-file-width="1704" decoding="async" height="232" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/8a/2007-04_Stolberg_%28Harz%29_58.jpg/200px-2007-04_Stolberg_%28Harz%29_58.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/8a/2007-04_Stolberg_%28Harz%29_58.jpg/300px-2007-04_Stolberg_%28Harz%29_58.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/8a/2007-04_Stolberg_%28Harz%29_58.jpg/400px-2007-04_Stolberg_%28Harz%29_58.jpg 2x" width="200"/></a><figcaption>Памятник <a href="/wiki/%D0%9C%D1%8E%D0%BD%D1%86%D0%B5%D1%80,_%D0%A2%D0%BE%D0%BC%D0%B0%D1%81" title="Мюнцер, Томас">Томасу Мюнцеру</a> с радужным флагом в <a href="/wiki/%D0%A8%D1%82%D0%BE%D0%BB%D1%8C%D0%B1%D0%B5%D1%80%D0%B3_(%D0%93%D0%B0%D1%80%D1%86)" title="Штольберг (Гарц)">Штольберге</a></figcaption></figure>
<p>Наряду с другими небесными явлениями радуга является одним из древних символов человечества. Так, в <a class="mw-redirect" href="/wiki/%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%BE-%D1%81%D0%BA%D0%B0%D0%BD%D0%B4%D0%B8%D0%BD%D0%B0%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%BC%D0%B8%D1%84%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F" title="Германо-скандинавская мифология">скандинавской мифологии</a> радуга считается <a href="/wiki/%D0%91%D0%B8%D0%B2%D1%80%D1%91%D1%81%D1%82" title="Биврёст">мостом, соединяющим мир людей с миром богов</a><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>. В <a href="/wiki/%D0%92%D0%B5%D0%B4%D0%B8%D0%B9%D1%81%D0%BA%D0%B0%D1%8F_%D0%BC%D0%B8%D1%84%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F" title="Ведийская мифология">ведийской мифологии</a> она является луком бога-громовержца <a href="/wiki/%D0%98%D0%BD%D0%B4%D1%80%D0%B0" title="Индра">Индры</a><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>. В <a href="/wiki/%D0%94%D1%80%D0%B5%D0%B2%D0%BD%D0%B5%D0%B3%D1%80%D0%B5%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D0%BC%D0%B8%D1%84%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F" title="Древнегреческая мифология">древнегреческой мифологии</a> радуга символизирует вестницу богов <a href="/wiki/%D0%98%D1%80%D0%B8%D0%B4%D0%B0" title="Ирида">Ириду</a><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>. В <a href="/wiki/%D0%A0%D0%B5%D0%BB%D0%B8%D0%B3%D0%B8%D1%8F_%D0%B0%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D1%85_%D0%B0%D0%B1%D0%BE%D1%80%D0%B8%D0%B3%D0%B5%D0%BD%D0%BE%D0%B2" title="Религия австралийских аборигенов">мифологии аборигенов Австралии</a> змей-радуга является покровителем неба, воды, плодородия и шаманов<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>. В <a href="/wiki/%D0%9A%D0%B8%D1%82%D0%B0%D0%B9" title="Китай">Китае</a> в радуге различают пять цветов, соединение которых означает единство <a href="/wiki/%D0%98%D0%BD%D1%8C_%D0%B8_%D1%8F%D0%BD" title="Инь и ян">инь и ян</a><sup class="reference" id="cite_ref-Символ_1-1"><a href="#cite_note-Символ-1">[1]</a></sup>.
</p><p>В <a href="/wiki/%D0%92%D0%B5%D1%82%D1%85%D0%B8%D0%B9_%D0%97%D0%B0%D0%B2%D0%B5%D1%82" title="Ветхий Завет">Ветхом Завете</a> радуга, появившаяся перед <a href="/wiki/%D0%9D%D0%BE%D0%B9" title="Ной">Ноем</a> на исходе <a href="/wiki/%D0%92%D1%81%D0%B5%D0%BC%D0%B8%D1%80%D0%BD%D1%8B%D0%B9_%D0%BF%D0%BE%D1%82%D0%BE%D0%BF" title="Всемирный потоп">Всемирного потопа</a>, является символом заключения <a href="/wiki/%D0%A1%D0%B5%D0%BC%D1%8C_%D0%B7%D0%B0%D0%BA%D0%BE%D0%BD%D0%BE%D0%B2_%D0%BF%D0%BE%D1%82%D0%BE%D0%BC%D0%BA%D0%BE%D0%B2_%D0%9D%D0%BE%D1%8F" title="Семь законов потомков Ноя">союза между Богом и человечеством</a><sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup>. Эта легенда нашла своё отражение в толковании построения флагов некоторых государств, например <a href="/wiki/%D0%A4%D0%BB%D0%B0%D0%B3_%D0%90%D1%80%D0%BC%D0%B5%D0%BD%D0%B8%D0%B8" title="Флаг Армении">Армении</a><sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup> и <a href="/wiki/%D0%A4%D0%BB%D0%B0%D0%B3_%D0%AD%D1%84%D0%B8%D0%BE%D0%BF%D0%B8%D0%B8" title="Флаг Эфиопии">Эфиопии</a><sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup>. Во время <a class="mw-redirect" href="/wiki/%D0%9D%D0%B5%D0%BC%D0%B5%D1%86%D0%BA%D0%B0%D1%8F_%D0%9A%D1%80%D0%B5%D1%81%D1%82%D1%8C%D1%8F%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Немецкая Крестьянская война">немецкой Крестьянской войны</a> лидер восстания <a href="/wiki/%D0%A0%D0%B5%D1%84%D0%BE%D1%80%D0%BC%D0%B0%D1%86%D0%B8%D1%8F" title="Реформация">реформатор</a> <a class="mw-redirect" href="/wiki/%D0%A2%D0%BE%D0%BC%D0%B0%D1%81_%D0%9C%D1%8E%D0%BD%D1%86%D0%B5%D1%80" title="Томас Мюнцер">Томас Мюнцер</a> выбрал радугу как знак вечного божественного союза, как символ нового времени, надежды и изменений. В апреле <a href="/wiki/1525_%D0%B3%D0%BE%D0%B4" title="1525 год">1525 года</a> был изготовлен белый флаг длиной в 30 локтей, на котором была размещена радуга, а также надпись: «Слово Бога вечно» (<a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Латинский язык">лат.</a> <span lang="la" style="font-style:italic;">Verbum Domini Manet in Aeternum</span>). С тех пор на некоторых знаменах <a href="/wiki/%D0%A2%D1%8E%D1%80%D0%B8%D0%BD%D0%B3%D0%B8%D1%8F" title="Тюрингия">Тюрингии</a> также имеются радужные полоски<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup>.
</p><p>В <a href="/wiki/1924_%D0%B3%D0%BE%D0%B4" title="1924 год">1924 году</a> известный французский экономист <a class="mw-redirect" href="/wiki/%D0%A8%D0%B0%D1%80%D0%BB%D1%8C_%D0%96%D0%B8%D0%B4" title="Шарль Жид">Шарль Жид</a> изготовил радужный флаг для празднования <a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D0%B4%D0%B5%D0%BD%D1%8C_%D0%BA%D0%BE%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%B8%D0%B2%D0%BE%D0%B2" title="Международный день кооперативов">Международного дня кооперативов</a>. Он подчёркивал, что радуга символизирует единство в разнообразии и мощность света, просвещения и прогресса. Флаг стал официальным символом <a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D0%BA%D0%BE%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D1%8B%D0%B9_%D0%B0%D0%BB%D1%8C%D1%8F%D0%BD%D1%81" title="Международный кооперативный альянс">Международного кооперативного альянса</a> в 1925 году. В послевоенном периоде Альянс начинает активно сотрудничать с <a href="/wiki/%D0%A0%D0%B0%D0%B1%D0%BE%D1%87%D0%B5%D0%B5_%D0%B4%D0%B2%D0%B8%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5" title="Рабочее движение">рабочим движением</a> во многих странах мира<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup>. В <a href="/wiki/2001_%D0%B3%D0%BE%D0%B4" title="2001 год">2001 году</a> дизайн флага был изменён из-за постоянно возникающей путаницы с другими движениями.<sup class="reference" id="cite_ref-Альянс_3-1"><a href="#cite_note-Альянс-3">[3]</a></sup><sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup>
</p><p>Во второй половине XX века получило распространение так называемое «<a href="/wiki/Rainbow_Gathering" title="Rainbow Gathering">Движение радуги</a>», декларировавшее приверженность идеалам мира, любви, гармонии, свободы и сотрудничества. Его идеология и символика были якобы основаны на древнем пророчестве североамериканских индейцев (впоследствии оно оказалось мистификацией). В конечном итоге символ радуги стал использоваться в субкультуре <a href="/wiki/%D0%A5%D0%B8%D0%BF%D0%BF%D0%B8" title="Хиппи">хиппи</a>, <a href="/wiki/%D0%9D%D1%8C%D1%8E-%D1%8D%D0%B9%D0%B4%D0%B6" title="Нью-эйдж">нью-эйдж</a>, <a href="/wiki/%D0%AD%D0%BA%D1%83%D0%BC%D0%B5%D0%BD%D0%B8%D0%B7%D0%BC" title="Экуменизм">экуменических</a>, <a href="/wiki/%D0%9C%D0%B8%D1%81%D1%82%D0%B8%D0%BA%D0%B0" title="Мистика">мистических</a>, <a href="/wiki/Rainbow_Warrior" title="Rainbow Warrior">экологическом</a> движениях<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup><sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup>.
</p>
<h2><span id=".D0.A4.D0.BB.D0.B0.D0.B3_.D0.9B.D0.93.D0.91.D0.A2-.D1.81.D0.BE.D0.BE.D0.B1.D1.89.D0.B5.D1.81.D1.82.D0.B2.D0.B0"></span><span class="mw-headline" id="Флаг_ЛГБТ-сообщества">Флаг ЛГБТ-сообщества</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A0%D0%B0%D0%B4%D1%83%D0%B6%D0%BD%D1%8B%D0%B9_%D1%84%D0%BB%D0%B0%D0%B3&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Флаг ЛГБТ-сообщества»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A0%D0%B0%D0%B4%D1%83%D0%B6%D0%BD%D1%8B%D0%B9_%D1%84%D0%BB%D0%B0%D0%B3&amp;action=edit&amp;section=2" title="Редактировать код раздела «Флаг ЛГБТ-сообщества»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Castrosanfranciscoflag.jpg"><img class="mw-file-element" data-file-height="713" data-file-width="849" decoding="async" height="160" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/89/Castrosanfranciscoflag.jpg/190px-Castrosanfranciscoflag.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/89/Castrosanfranciscoflag.jpg/285px-Castrosanfranciscoflag.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/89/Castrosanfranciscoflag.jpg/380px-Castrosanfranciscoflag.jpg 2x" width="190"/></a><figcaption>Площадь <a class="mw-redirect" href="/wiki/%D0%A5%D0%B0%D1%80%D0%B2%D0%B8_%D0%9C%D0%B8%D0%BB%D0%BA" title="Харви Милк">Харви Милка</a>, <a href="/wiki/%D0%9A%D0%B0%D1%81%D1%82%D1%80%D0%BE_(%D0%A1%D0%B0%D0%BD-%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%81%D0%BA%D0%BE)" title="Кастро (Сан-Франциско)">Кастро</a>, <a href="/wiki/%D0%A1%D0%B0%D0%BD-%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%81%D0%BA%D0%BE" title="Сан-Франциско">Сан-Франциско</a></figcaption></figure>
<div class="hatnote navigation-not-searchable">Основная статья: <b><a href="/wiki/%D0%A0%D0%B0%D0%B4%D1%83%D0%B6%D0%BD%D1%8B%D0%B9_%D1%84%D0%BB%D0%B0%D0%B3_(%D0%9B%D0%93%D0%91%D0%A2)" title="Радужный флаг (ЛГБТ)">Радужный флаг (ЛГБТ)</a></b></div>
<p><b>Радужный флаг</b> (также известный как <b><a class="mw-redirect" href="/wiki/%D0%93%D0%B5%D0%B9-%D0%BF%D1%80%D0%B0%D0%B9%D0%B4_(%D1%82%D0%B5%D1%80%D0%BC%D0%B8%D0%BD)" title="Гей-прайд (термин)">Прайд</a>-флаг</b> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Pride flag</span>)<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup><sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup>, <b>Флаг свободы</b> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Freedom flag</span>)<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup>) является <a class="mw-redirect" href="/wiki/%D0%9B%D0%93%D0%91%D0%A2-%D1%81%D0%B8%D0%BC%D0%B2%D0%BE%D0%BB%D1%8B" title="ЛГБТ-символы">интернациональным символом</a> сообщества <a class="mw-redirect" href="/wiki/%D0%9B%D0%B5%D1%81%D0%B1%D0%B8" title="Лесби">лесбиянок</a>, <a href="/wiki/%D0%93%D0%B5%D0%B9" title="Гей">геев</a>, <a class="mw-redirect" href="/wiki/%D0%91%D0%B8%D1%81%D0%B5%D0%BA%D1%81%D1%83%D0%B0%D0%BB" title="Бисексуал">бисексуалов</a> и <a class="mw-redirect" href="/wiki/%D0%A2%D1%80%D0%B0%D0%BD%D1%81%D0%B3%D0%B5%D0%BD%D0%B4%D0%B5%D1%80" title="Трансгендер">трансгендерных людей</a> (<i>сокр. <a href="/wiki/%D0%9B%D0%93%D0%91%D0%A2-%D1%81%D0%BE%D0%BE%D0%B1%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="ЛГБТ-сообщество">ЛГБТ-сообщество</a></i>), а также <a href="/wiki/%D0%9B%D0%93%D0%91%D0%A2-%D0%B4%D0%B2%D0%B8%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5" title="ЛГБТ-движение">движения в поддержку прав человека в их отношении</a><sup class="reference" id="cite_ref-Флаг_22-0"><a href="#cite_note-Флаг-22">[22]</a></sup>. Традиционно флаг состоит из шести продольных полос, цвета которых идут в соответствии с природным порядком радуги сверху вниз: красный, оранжевый, жёлтый, зелёный, синий и фиолетовый<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup>. Флаг используется в шествиях <a class="mw-redirect" href="/wiki/%D0%93%D0%B5%D0%B9-%D0%BF%D1%80%D0%B0%D0%B9%D0%B4_(%D1%82%D0%B5%D1%80%D0%BC%D0%B8%D0%BD)" title="Гей-прайд (термин)">гей-прайдов</a> и других общественных акциях, вывешивается в <a href="/wiki/%D0%93%D0%B5%D0%B9-%D0%BA%D0%B2%D0%B0%D1%80%D1%82%D0%B0%D0%BB" title="Гей-квартал">гей-кварталах</a>, на зданиях ЛГБТ-организаций, «<a href="/wiki/Gay-friendly" title="Gay-friendly">Gay-friendly</a>» заведений и так далее.
</p><p>Радужный флаг был разработан <a class="mw-redirect" href="/wiki/%D0%93%D0%B8%D0%BB%D0%B1%D0%B5%D1%80%D1%82_%D0%91%D0%B5%D0%B9%D0%BA%D0%B5%D1%80" title="Гилберт Бейкер">Гилбертом Бейкером</a> специально для <a class="mw-redirect" href="/wiki/%D0%93%D0%B5%D0%B9-%D0%BF%D1%80%D0%B0%D0%B9%D0%B4_(%D1%82%D0%B5%D1%80%D0%BC%D0%B8%D0%BD)" title="Гей-прайд (термин)">гей-прайда</a> в <a href="/wiki/%D0%A1%D0%B0%D0%BD-%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%81%D0%BA%D0%BE" title="Сан-Франциско">Сан-Франциско</a> 1978 года (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">San Francisco Gay Freedom Day</span>). Художник отмечал: «Когда я задумался о создании флага для гей-движения, не было никакого другого международного символа для нас, кроме <a href="/wiki/%D0%A0%D0%BE%D0%B7%D0%BE%D0%B2%D1%8B%D0%B9_%D1%82%D1%80%D0%B5%D1%83%D0%B3%D0%BE%D0%BB%D1%8C%D0%BD%D0%B8%D0%BA" title="Розовый треугольник">розового треугольника</a>, которым нацисты идентифицировали геев в концентрационных лагерях. Хотя розовый треугольник по-прежнему был мощным символом, но он все же был нам навязан»<sup class="reference" id="cite_ref-ТР_24-0"><a href="#cite_note-ТР-24">[24]</a></sup><sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup>.
</p>
<h2><span id=".D0.A4.D0.BB.D0.B0.D0.B3_.D0.B4.D0.B2.D0.B8.D0.B6.D0.B5.D0.BD.D0.B8.D1.8F_.D0.B7.D0.B0_.D0.BC.D0.B8.D1.80"></span><span class="mw-headline" id="Флаг_движения_за_мир">Флаг движения за мир</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A0%D0%B0%D0%B4%D1%83%D0%B6%D0%BD%D1%8B%D0%B9_%D1%84%D0%BB%D0%B0%D0%B3&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Флаг движения за мир»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A0%D0%B0%D0%B4%D1%83%D0%B6%D0%BD%D1%8B%D0%B9_%D1%84%D0%BB%D0%B0%D0%B3&amp;action=edit&amp;section=3" title="Редактировать код раздела «Флаг движения за мир»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Bandiere_della_pace_a_Milano_2003.jpg"><img class="mw-file-element" data-file-height="1360" data-file-width="2048" decoding="async" height="139" src="//upload.wikimedia.org/wikipedia/commons/thumb/f/fc/Bandiere_della_pace_a_Milano_2003.jpg/210px-Bandiere_della_pace_a_Milano_2003.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/f/fc/Bandiere_della_pace_a_Milano_2003.jpg/315px-Bandiere_della_pace_a_Milano_2003.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/f/fc/Bandiere_della_pace_a_Milano_2003.jpg/420px-Bandiere_della_pace_a_Milano_2003.jpg 2x" width="210"/></a><figcaption>Протест против войны в <a href="/wiki/%D0%98%D1%80%D0%B0%D0%BA" title="Ирак">Ираке</a>, <a href="/wiki/%D0%9C%D0%B8%D0%BB%D0%B0%D0%BD" title="Милан">Милан</a>, 18 марта 2003 года</figcaption></figure>
<p>«<b>Флаг мира</b>» (<a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D1%8C%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Итальянский язык">итал.</a> <span lang="it" style="font-style:italic;">Bandiera della Pace</span>) является одним из символов международного <a href="/wiki/%D0%9F%D0%B0%D1%86%D0%B8%D1%84%D0%B8%D0%B7%D0%BC" title="Пацифизм">движения за мир</a>. Его автором считается итальянский философ, теолог и гуманист <a class="mw-redirect" href="/wiki/%D0%90%D0%BB%D1%8C%D0%B4%D0%BE_%D0%9A%D0%B0%D0%BF%D0%B8%D1%82%D0%B8%D0%BD%D0%B8" title="Альдо Капитини">Альдо Капитини</a>, продемонстрировавший радужное знамя на первом марше за мир из <a href="/wiki/%D0%9F%D0%B5%D1%80%D1%83%D0%B4%D0%B6%D0%B0" title="Перуджа">Перуджи</a> в <a href="/wiki/%D0%90%D1%81%D1%81%D0%B8%D0%B7%D0%B8" title="Ассизи">Ассизи</a> 24 сентября <a href="/wiki/1961_%D0%B3%D0%BE%D0%B4" title="1961 год">1961 года</a>. Радуга была выбрана как символ празднования разнообразия и библейский знак примирения. Указывается, что Капитини увидел прототип своего флага в 1958 году на демонстрации, организованной британским писателем и общественным деятелем <a href="/wiki/%D0%A0%D0%B0%D1%81%D1%81%D0%B5%D0%BB,_%D0%91%D0%B5%D1%80%D1%82%D1%80%D0%B0%D0%BD" title="Рассел, Бертран">Бертраном Расселом</a><sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup>. Есть свидетельства того, что радужное знамя в качестве знака мира было предложено уже в 1949 году и возможно было заимствовано у кооперативного движения<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup><sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup>.
</p>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Peace_Flags.jpg"><img class="mw-file-element" data-file-height="480" data-file-width="640" decoding="async" height="135" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/36/Peace_Flags.jpg/180px-Peace_Flags.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/36/Peace_Flags.jpg/270px-Peace_Flags.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/3/36/Peace_Flags.jpg/360px-Peace_Flags.jpg 2x" width="180"/></a><figcaption>Марш за мир, <a href="/wiki/%D0%9C%D1%8E%D0%BD%D1%85%D0%B5%D0%BD" title="Мюнхен">Мюнхен</a>, 15 апреля 2006 года</figcaption></figure>
<p>Современный наиболее популярный вариант флага мира, предложенный итальянскими коммунистами в 1980-х годах, имеет семь цветов, расположенных в обратном природному порядке сверху вниз: фиолетовый, синий, голубой, зелёный, жёлтый, оранжевый, красный. Иногда фиолетовый и синий меняются местами. Посередине флага помещается надпись «<b>PACE</b>» (<a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D1%8C%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Итальянский язык">итал</a>. <i>«мир»</i>). В зависимости от языка надпись может иногда варьироваться, например: «PEACE» (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a>), «SHALOM» (<a href="/wiki/%D0%98%D0%B2%D1%80%D0%B8%D1%82" title="Иврит">иврит</a>), «EIPHNH» (<a href="/wiki/%D0%93%D1%80%D0%B5%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Греческий язык">греч.</a>) и так далее.
</p><p>Во время <a href="/wiki/%D0%98%D1%80%D0%B0%D0%BA%D1%81%D0%BA%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Иракская война">войны в Ираке</a> в 2003 году многие итальянцы последовали призыву «Мир со всех балконов» (<a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D1%8C%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Итальянский язык">итал.</a> <span lang="it" style="font-style:italic;">Pace da tutti i balconi</span>) и в знак протеста вывесили флаги на балконы, окна и стены домов. Эта кампания имела широкий общественный резонанс, радужное знамя приобрело большую популярность в Италии, а затем и по всей Европе. Одним из основателей нового движения выступил католический священник Алекс Дзанотелли (<a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D1%8C%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Итальянский язык">итал.</a> <span lang="it" style="font-style:italic;">Alex Zanotelli</span>). Активное участие в акции религиозных деятелей и даже размещение радужных флагов в церквях вызвало осуждение со стороны официального <a href="/wiki/%D0%92%D0%B0%D1%82%D0%B8%D0%BA%D0%B0%D0%BD" title="Ватикан">Ватикана</a>, который, в частности, был озабочен невольно возникающими ассоциациями с <a href="/wiki/%D0%9B%D0%93%D0%91%D0%A2-%D0%B4%D0%B2%D0%B8%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5" title="ЛГБТ-движение">гей-движением</a> и <a href="/wiki/%D0%9D%D1%8C%D1%8E-%D1%8D%D0%B9%D0%B4%D0%B6" title="Нью-эйдж">нью-эйджем</a><sup class="reference" id="cite_ref-Ватикан_4-1"><a href="#cite_note-Ватикан-4">[4]</a></sup><sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[29]</a></sup><sup class="reference" id="cite_ref-30"><a href="#cite_note-30">[30]</a></sup><sup class="reference" id="cite_ref-31"><a href="#cite_note-31">[31]</a></sup>.
</p>
<h2><span id=".D0.A4.D0.BB.D0.B0.D0.B3_.D0.B4.D0.B2.D0.B8.D0.B6.D0.B5.D0.BD.D0.B8.D1.8F_.D0.BA.D0.BE.D1.80.D0.B5.D0.BD.D0.BD.D1.8B.D1.85_.D0.BD.D0.B0.D1.80.D0.BE.D0.B4.D0.BE.D0.B2_.D0.AE.D0.B6.D0.BD.D0.BE.D0.B9_.D0.90.D0.BC.D0.B5.D1.80.D0.B8.D0.BA.D0.B8"></span><span class="mw-headline" id="Флаг_движения_коренных_народов_Южной_Америки">Флаг движения коренных народов Южной Америки</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A0%D0%B0%D0%B4%D1%83%D0%B6%D0%BD%D1%8B%D0%B9_%D1%84%D0%BB%D0%B0%D0%B3&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Флаг движения коренных народов Южной Америки»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A0%D0%B0%D0%B4%D1%83%D0%B6%D0%BD%D1%8B%D0%B9_%D1%84%D0%BB%D0%B0%D0%B3&amp;action=edit&amp;section=4" title="Редактировать код раздела «Флаг движения коренных народов Южной Америки»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Inca_rainbow_flags.jpg"><img class="mw-file-element" data-file-height="2000" data-file-width="3008" decoding="async" height="113" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/e3/Inca_rainbow_flags.jpg/170px-Inca_rainbow_flags.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/e3/Inca_rainbow_flags.jpg/255px-Inca_rainbow_flags.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/e3/Inca_rainbow_flags.jpg/340px-Inca_rainbow_flags.jpg 2x" width="170"/></a><figcaption>Флаг <a href="/wiki/%D0%92%D0%B8%D0%BF%D0%B0%D0%BB%D0%B0" title="Випала">Випала</a> на марше индейцев</figcaption></figure>
<p>По некоторым источникам, в <a class="mw-redirect" href="/wiki/%D0%98%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F_%D0%98%D0%BD%D0%BA%D0%BE%D0%B2" title="Империя Инков">Империи Инков</a> радуга являлась одним из употребляемых символов, однако есть разногласия относительно существования в древнюю эпоху радужного флага как такового<sup class="reference" id="cite_ref-32"><a href="#cite_note-32">[32]</a></sup><sup class="reference" id="cite_ref-33"><a href="#cite_note-33">[33]</a></sup>. В 1970-х годах радужный флаг (в частности, мозаичный вариант <i>Wiphala</i>) стал символом движения <a href="/wiki/%D0%98%D0%BD%D0%B4%D0%B5%D0%B9%D1%86%D1%8B" title="Индейцы">коренных народов Южной Америки</a> в <a href="/wiki/%D0%91%D0%BE%D0%BB%D0%B8%D0%B2%D0%B8%D1%8F" title="Боливия">Боливии</a>, <a href="/wiki/%D0%9F%D0%B5%D1%80%D1%83" title="Перу">Перу</a>, <a href="/wiki/%D0%A7%D0%B8%D0%BB%D0%B8" title="Чили">Чили</a> и <a href="/wiki/%D0%AD%D0%BA%D0%B2%D0%B0%D0%B4%D0%BE%D1%80" title="Эквадор">Эквадоре</a>.
</p><p>В 1978 г. <a href="/wiki/%D0%9F%D0%B5%D1%80%D1%83" title="Перу">перуанский</a> город <a href="/wiki/%D0%9A%D1%83%D1%81%D0%BA%D0%BE" title="Куско">Куско</a> принял семицветный радужный флаг в качестве своего <a href="/wiki/%D0%A4%D0%BB%D0%B0%D0%B3_%D0%9A%D1%83%D1%81%D0%BA%D0%BE" title="Флаг Куско">официального флага</a><sup class="reference" id="cite_ref-34"><a href="#cite_note-34">[34]</a></sup>. Однако в последнее время ведутся дискуссии об изменении главного атрибута города из-за частой путаницы его с «гей-флагом»<sup class="reference" id="cite_ref-Инки_2-1"><a href="#cite_note-Инки-2">[2]</a></sup>.
</p><p>Согласно внесённым в конце 2009 года поправкам в Конституцию Боливии <i>Wiphala</i> стал вторым <a href="/wiki/%D0%A4%D0%BB%D0%B0%D0%B3_%D0%91%D0%BE%D0%BB%D0%B8%D0%B2%D0%B8%D0%B8" title="Флаг Боливии">национальным флагом</a>, для того чтобы подчеркнуть исторические корни республики<sup class="reference" id="cite_ref-35"><a href="#cite_note-35">[35]</a></sup><sup class="reference" id="cite_ref-36"><a href="#cite_note-36">[36]</a></sup>.
</p>
<h2><span id=".D0.92_.D0.A0.D0.BE.D1.81.D1.81.D0.B8.D0.B8"></span><span class="mw-headline" id="В_России">В России</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A0%D0%B0%D0%B4%D1%83%D0%B6%D0%BD%D1%8B%D0%B9_%D1%84%D0%BB%D0%B0%D0%B3&amp;veaction=edit&amp;section=5" title="Редактировать раздел «В России»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A0%D0%B0%D0%B4%D1%83%D0%B6%D0%BD%D1%8B%D0%B9_%D1%84%D0%BB%D0%B0%D0%B3&amp;action=edit&amp;section=5" title="Редактировать код раздела «В России»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>В <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D1%8F" title="Россия">России</a> радуга присутствует на флагах городов Радужный во <a href="/wiki/%D0%A0%D0%B0%D0%B4%D1%83%D0%B6%D0%BD%D1%8B%D0%B9_(%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C)" title="Радужный (Владимирская область)">Владимирской</a>, <a href="/wiki/%D0%A0%D0%B0%D0%B4%D1%83%D0%B6%D0%BD%D1%8B%D0%B9_(%D0%9C%D0%BE%D1%81%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C)" title="Радужный (Московская область)">Московской</a> областях и <a href="/wiki/%D0%A0%D0%B0%D0%B4%D1%83%D0%B6%D0%BD%D1%8B%D0%B9_(%D0%A5%D0%B0%D0%BD%D1%82%D1%8B-%D0%9C%D0%B0%D0%BD%D1%81%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%B2%D1%82%D0%BE%D0%BD%D0%BE%D0%BC%D0%BD%D1%8B%D0%B9_%D0%BE%D0%BA%D1%80%D1%83%D0%B3)" title="Радужный (Ханты-Мансийский автономный округ)">Ханты-Мансийском округе</a>, на флаге партии «<a href="/wiki/%D0%9F%D0%B0%D1%82%D1%80%D0%B8%D0%BE%D1%82%D1%8B_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B8" title="Патриоты России">Патриоты России</a>». <a href="/wiki/%D0%A4%D0%BB%D0%B0%D0%B3_%D0%95%D0%B2%D1%80%D0%B5%D0%B9%D1%81%D0%BA%D0%BE%D0%B9_%D0%B0%D0%B2%D1%82%D0%BE%D0%BD%D0%BE%D0%BC%D0%BD%D0%BE%D0%B9_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D0%B8" title="Флаг Еврейской автономной области">Флаг Еврейской автономной области</a> представляет собой белое прямоугольное полотнище, по горизонтальной оси которого расположена цветная полоса радуги, состоящая из семи узких горизонтальных полосок: красной, оранжевой, жёлтой, зелёной, голубой, синей и фиолетовой<sup class="reference" id="cite_ref-37"><a href="#cite_note-37">[37]</a></sup>. Флаг автономной области даже проверяли на предмет «<a class="mw-redirect" href="/wiki/%D0%97%D0%B0%D0%BA%D0%BE%D0%BD%D0%BE%D0%B4%D0%B0%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B5_%D0%B7%D0%B0%D0%BF%D1%80%D0%B5%D1%82%D1%8B_%D0%BF%D1%80%D0%BE%D0%BF%D0%B0%D0%B3%D0%B0%D0%BD%D0%B4%D1%8B_%D0%B3%D0%BE%D0%BC%D0%BE%D1%81%D0%B5%D0%BA%D1%81%D1%83%D0%B0%D0%BB%D0%B8%D0%B7%D0%BC%D0%B0_%D0%B2_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B8" title="Законодательные запреты пропаганды гомосексуализма в России">пропаганды гомосексуализма</a>»<sup class="reference" id="cite_ref-38"><a href="#cite_note-38">[38]</a></sup>. Радужный флаг был выбран флагом основанной <a href="/wiki/%D0%91%D0%BE%D1%80%D0%BE%D0%B2%D0%BE%D0%B9,_%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD_%D0%9D%D0%B0%D1%82%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87" title="Боровой, Константин Натанович">К. Н. Боровым</a> «<a href="/wiki/%D0%9F%D0%B0%D1%80%D1%82%D0%B8%D1%8F_%D1%8D%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%BE%D0%B9_%D1%81%D0%B2%D0%BE%D0%B1%D0%BE%D0%B4%D1%8B" title="Партия экономической свободы">Партии экономической свободы</a>», что послужило причиной последующих конфузов (его путали с «гей-флагом»)<sup class="reference" id="cite_ref-Боровой_5-1"><a href="#cite_note-Боровой-5">[5]</a></sup>.
</p><p>Мотив радуги присутствует на логотипах ЛГБТ-организаций России, например, <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B9%D1%81%D0%BA%D0%B0%D1%8F_%D0%9B%D0%93%D0%91%D0%A2-%D1%81%D0%B5%D1%82%D1%8C" title="Российская ЛГБТ-сеть">Российской ЛГБТ-сети</a>, <a href="/wiki/%D0%92%D1%8B%D1%85%D0%BE%D0%B4_(%D0%BE%D1%80%D0%B3%D0%B0%D0%BD%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F)" title="Выход (организация)">Выхода</a>, <a href="/wiki/%D0%A0%D0%B0%D0%B2%D0%BD%D0%BE%D0%BF%D1%80%D0%B0%D0%B2%D0%B8%D0%B5_(%D0%9B%D0%93%D0%91%D0%A2-%D0%BE%D1%80%D0%B3%D0%B0%D0%BD%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F)" title="Равноправие (ЛГБТ-организация)">Равноправия</a>, <a class="mw-redirect" href="/wiki/%D0%9F%D1%80%D0%BE%D0%B5%D0%BA%D1%82_%C2%ABLaSky%C2%BB" title="Проект «LaSky»">LaSky</a>, <a href="/wiki/%D0%91%D0%BE%D0%BA_%D0%BE_%D0%B1%D0%BE%D0%BA_(%D0%BA%D0%B8%D0%BD%D0%BE%D1%84%D0%B5%D1%81%D1%82%D0%B8%D0%B2%D0%B0%D0%BB%D1%8C)" title="Бок о бок (кинофестиваль)">кинофестиваля «Бок о Бок»</a>, <a href="/wiki/%D0%93%D0%B5%D0%B9-%D0%BF%D0%B0%D1%80%D0%B0%D0%B4%D1%8B_%D0%B2_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B8" title="Гей-парады в России">Московского гей-прайда</a>.
</p>
