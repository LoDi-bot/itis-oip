<p><b>Ю́лиус Штре́йхер</b> (Штрайхер; <a href="/wiki/%D0%9D%D0%B5%D0%BC%D0%B5%D1%86%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Немецкий язык">нем.</a> <span lang="de" style="font-style:italic;">Julius Streicher</span>; <a href="/wiki/12_%D1%84%D0%B5%D0%B2%D1%80%D0%B0%D0%BB%D1%8F" title="12 февраля">12 февраля</a> <a class="mw-redirect" href="/wiki/1885" title="1885">1885</a>, <a class="new" href="/w/index.php?title=%D0%A4%D0%BB%D0%B5%D0%B9%D0%BD%D1%85%D0%B0%D1%83%D0%B7%D0%B5%D0%BD&amp;action=edit&amp;redlink=1" title="Флейнхаузен (страница отсутствует)">Флейнхаузен</a> близ <a href="/wiki/%D0%90%D1%83%D0%B3%D1%81%D0%B1%D1%83%D1%80%D0%B3" title="Аугсбург">Аугсбурга</a>, <a href="/wiki/%D0%9A%D0%BE%D1%80%D0%BE%D0%BB%D0%B5%D0%B2%D1%81%D1%82%D0%B2%D0%BE_%D0%91%D0%B0%D0%B2%D0%B0%D1%80%D0%B8%D1%8F" title="Королевство Бавария">Бавария</a> — <a href="/wiki/16_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="16 октября">16 октября</a> <a class="mw-redirect" href="/wiki/1946" title="1946">1946</a>, <a href="/wiki/%D0%9D%D1%8E%D1%80%D0%BD%D0%B1%D0%B5%D1%80%D0%B3" title="Нюрнберг">Нюрнберг</a>) — <a href="/wiki/%D0%93%D0%B0%D1%83%D0%BB%D1%8F%D0%B9%D1%82%D0%B5%D1%80" title="Гауляйтер">гауляйтер</a> <a class="new" href="/w/index.php?title=%D0%A4%D1%80%D0%B0%D0%BD%D0%BA%D0%BE%D0%BD%D0%B8%D1%8F_(%D0%B3%D0%B0%D1%83)&amp;action=edit&amp;redlink=1" title="Франкония (гау) (страница отсутствует)">Франконии</a>, главный редактор <a href="/wiki/%D0%90%D0%BD%D1%82%D0%B8%D1%81%D0%B5%D0%BC%D0%B8%D1%82%D0%B8%D0%B7%D0%BC" title="Антисемитизм">антисемитской</a> и антикоммунистической газеты «<a class="mw-redirect" href="/wiki/%D0%A8%D1%82%D1%83%D1%80%D0%BC%D0%BE%D0%B2%D0%B8%D0%BA_(%D0%B5%D0%B6%D0%B5%D0%BD%D0%B5%D0%B4%D0%B5%D0%BB%D1%8C%D0%BD%D0%B8%D0%BA)" title="Штурмовик (еженедельник)">Штурмовик</a>» (<a href="/wiki/%D0%9D%D0%B5%D0%BC%D0%B5%D1%86%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Немецкий язык">нем.</a> <span lang="de" style="font-style:italic;">Der Stürmer</span> — <i>Дер Штюрмер</i>), идеолог <a href="/wiki/%D0%A0%D0%B0%D1%81%D0%B8%D0%B7%D0%BC" title="Расизм">расизма</a>. Казнён по приговору <a href="/wiki/%D0%9D%D1%8E%D1%80%D0%BD%D0%B1%D0%B5%D1%80%D0%B3%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D1%80%D0%BE%D1%86%D0%B5%D1%81%D1%81" title="Нюрнбергский процесс">Нюрнбергского трибунала</a> за антисемитскую пропаганду и призывы к <a href="/wiki/%D0%93%D0%B5%D0%BD%D0%BE%D1%86%D0%B8%D0%B4" title="Геноцид">геноциду</a>.
</p>

<h2><span id=".D0.94.D0.BE_.D0.BD.D0.B0.D1.87.D0.B0.D0.BB.D0.B0_.D0.BF.D0.BE.D0.BB.D0.B8.D1.82.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.BE.D0.B9_.D0.BA.D0.B0.D1.80.D1.8C.D0.B5.D1.80.D1.8B"></span><span class="mw-headline" id="До_начала_политической_карьеры">До начала политической карьеры</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A8%D1%82%D1%80%D0%B5%D0%B9%D1%85%D0%B5%D1%80,_%D0%AE%D0%BB%D0%B8%D1%83%D1%81&amp;veaction=edit&amp;section=1" title="Редактировать раздел «До начала политической карьеры»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A8%D1%82%D1%80%D0%B5%D0%B9%D1%85%D0%B5%D1%80,_%D0%AE%D0%BB%D0%B8%D1%83%D1%81&amp;action=edit&amp;section=1" title="Редактировать код раздела «До начала политической карьеры»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Юлиус Штрейхер родился 12 февраля <a href="/wiki/1885_%D0%B3%D0%BE%D0%B4" title="1885 год">1885 года</a> в <span data-interwiki-article="Fleinhausen" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%A4%D0%BB%D0%B5%D0%B9%D0%BD%D1%85%D0%B0%D1%83%D0%B7%D0%B5%D0%BD&amp;action=edit&amp;redlink=1" title="Флейнхаузен (страница отсутствует)">Флейнхаузене</a></span><span class="noprint" style="font-style:normal; font-weight:normal;"><sup><a class="extiw" href="https://en.wikipedia.org/wiki/Fleinhausen" title="en:Fleinhausen"><span title="Fleinhausen — версия статьи «Флейнхаузен» на английском языке">[en]</span></a></sup></span> близ <a href="/wiki/%D0%90%D1%83%D0%B3%D1%81%D0%B1%D1%83%D1%80%D0%B3" title="Аугсбург">Аугсбурга</a>, <a href="/wiki/%D0%91%D0%B0%D0%B2%D0%B0%D1%80%D0%B8%D1%8F" title="Бавария">Бавария</a>. Он был девятым ребёнком в семье Фридриха Штрейхера, учителя римско-католической начальной школы. Перед войной поступил добровольцем в Германскую армию, проработав перед этим некоторое время учителем в начальной школе. Однако после года службы он был уволен из армии за недисциплинированность с запрещением в дальнейшем служить в вооружённых силах.
</p><p>В ходе <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B2%D0%B0%D1%8F_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Первая мировая война">Первой мировой войны</a> Штрейхер показал себя храбрым солдатом и заслужил <a href="/wiki/%D0%96%D0%B5%D0%BB%D0%B5%D0%B7%D0%BD%D1%8B%D0%B9_%D0%BA%D1%80%D0%B5%D1%81%D1%82" title="Железный крест">Железный крест</a> I и II степени, а также получил звание <a href="/wiki/%D0%9B%D0%B5%D0%B9%D1%82%D0%B5%D0%BD%D0%B0%D0%BD%D1%82" title="Лейтенант">лейтенанта</a>.
</p>
<h2><span id=".D0.9F.D0.BE.D1.81.D0.BB.D0.B5_.D0.9F.D0.B5.D1.80.D0.B2.D0.BE.D0.B9_.D0.BC.D0.B8.D1.80.D0.BE.D0.B2.D0.BE.D0.B9_.D0.B2.D0.BE.D0.B9.D0.BD.D1.8B"></span><span class="mw-headline" id="После_Первой_мировой_войны">После Первой мировой войны</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A8%D1%82%D1%80%D0%B5%D0%B9%D1%85%D0%B5%D1%80,_%D0%AE%D0%BB%D0%B8%D1%83%D1%81&amp;veaction=edit&amp;section=2" title="Редактировать раздел «После Первой мировой войны»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A8%D1%82%D1%80%D0%B5%D0%B9%D1%85%D0%B5%D1%80,_%D0%AE%D0%BB%D0%B8%D1%83%D1%81&amp;action=edit&amp;section=2" title="Редактировать код раздела «После Первой мировой войны»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>После войны Штрейхер продолжил преподавать в школе, однако вскоре начал участвовать в политической жизни страны на стороне крайне правых сил.
</p><p>В 1919 году Штрейхер создал в Нюрнберге отделение <a href="/wiki/%D0%90%D0%BD%D1%82%D0%B8%D1%81%D0%B5%D0%BC%D0%B8%D1%82%D0%B8%D0%B7%D0%BC" title="Антисемитизм">антисемитской</a> организации, Немецко-социалистической партии (НСП) (<a href="/wiki/%D0%9D%D0%B5%D0%BC%D0%B5%D1%86%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Немецкий язык">нем.</a> <span lang="de" style="font-style:italic;">Deutschsozialistische Partei</span>). В 1921 году, когда Гитлер уехал из <a href="/wiki/%D0%9C%D1%8E%D0%BD%D1%85%D0%B5%D0%BD" title="Мюнхен">Мюнхена</a> в <a href="/wiki/%D0%91%D0%B5%D1%80%D0%BB%D0%B8%D0%BD" title="Берлин">Берлин</a> налаживать контакты с лидерами нацистских организаций севера Германии, ряд членов <a class="mw-redirect" href="/wiki/%D0%9D%D0%A1%D0%94%D0%90%D0%9F" title="НСДАП">НСДАП</a>, включая её основателя <a href="/wiki/%D0%94%D1%80%D0%B5%D0%BA%D1%81%D0%BB%D0%B5%D1%80,_%D0%90%D0%BD%D1%82%D0%BE%D0%BD" title="Дрекслер, Антон">Антона Дрекслера</a>, обвинили Гитлера в <a href="/wiki/%D0%94%D0%B8%D0%BA%D1%82%D0%B0%D1%82%D0%BE%D1%80" title="Диктатор">диктаторстве</a> и пытались установить контакты с НСП. Гитлер срочно вернулся в <a href="/wiki/%D0%91%D0%B0%D0%B2%D0%B0%D1%80%D0%B8%D1%8F" title="Бавария">Баварию</a> и потребовал прекратить заигрывания с группой Штрейхера. Это вызвало серьёзный кризис в партии, однако Гитлер смог утвердить себя в качестве лидера. Вскоре Штрейхер смог найти с лидером национал-социалистов общий язык (Гитлер решил, что ему все равно, какие люди его окружают, лишь бы они помогали его делу<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>) и 8 октября 1922 года влился вместе с членами НСП в ряды НСДАП.
</p><p>Когда он услышал о <a href="/wiki/%D0%9F%D0%B8%D0%B2%D0%BD%D0%BE%D0%B9_%D0%BF%D1%83%D1%82%D1%87" title="Пивной путч">Пивном путче</a>, то, не желая оставаться в стороне, присоединился к Гитлеру и шёл рядом с ним в первых рядах. Некоторые свидетели утверждают, что когда толпе марширующих преградили дорогу полицейские, именно Штрейхер первым произвёл выстрел. Сам он впоследствии говорил, что именно благодаря участию в путче <a href="/wiki/%D0%A4%D1%8E%D1%80%D0%B5%D1%80" title="Фюрер">фюрер</a> всегда его поддерживал в дальнейшем. «Возможно, найдутся один или два человека, которым не нравится форма носа Штрейхера. Но в тот день, когда он лежал рядом со мной на мостовой <a href="/wiki/%D0%A4%D0%B5%D0%BB%D1%8C%D0%B4%D1%85%D0%B5%D1%80%D0%BD%D1%85%D0%B0%D0%BB%D0%BB%D0%B5" title="Фельдхернхалле">Фельдхернхалле</a>, я поклялся, что не брошу его, пока он не бросит меня», — говорил про него впоследствии Гитлер.
</p><p>С 16 апреля 1923 года он начал выпускать собственную газету «<a class="mw-redirect" href="/wiki/%D0%A8%D1%82%D1%83%D1%80%D0%BC%D0%BE%D0%B2%D0%B8%D0%BA_(%D0%B5%D0%B6%D0%B5%D0%BD%D0%B5%D0%B4%D0%B5%D0%BB%D1%8C%D0%BD%D0%B8%D0%BA)" title="Штурмовик (еженедельник)">Дер Штюрмер</a>», которая вскоре получила репутацию самого радикального антисемитского издания Германии. Он печатал рассказы о <a href="/wiki/%D0%A0%D0%B8%D1%82%D1%83%D0%B0%D0%BB%D1%8C%D0%BD%D0%BE%D0%B5_%D1%83%D0%B1%D0%B8%D0%B9%D1%81%D1%82%D0%B2%D0%BE" title="Ритуальное убийство">ритуальных убийствах</a> евреями «<a href="/wiki/%D0%90%D1%80%D0%B8%D0%B9%D1%86%D1%8B" title="Арийцы">арийских</a>» детей, а также обвинял евреев в таких вещах, как взрыв <a href="/wiki/%D0%93%D0%B8%D0%BD%D0%B4%D0%B5%D0%BD%D0%B1%D1%83%D1%80%D0%B3_(%D0%B4%D0%B8%D1%80%D0%B8%D0%B6%D0%B0%D0%B1%D0%BB%D1%8C)" title="Гинденбург (дирижабль)">дирижабля «Гинденбург»</a> в 1937 году. В газете было очень много <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B8%D0%BA%D0%B0%D1%82%D1%83%D1%80%D0%B0" title="Карикатура">карикатур</a>, порой откровенно порнографических, а также жалоб на евреев, мешающих жить «арийцам»: пациент психиатрической больницы жаловался, что его туда незаслуженно упекли евреи, покупатель писал, что ему не принесли рубашку из магазина, принадлежащего еврею, и так далее в каждом номере. В своих статьях Штрейхер использовал самые откровенные образы. Он придавал антисемитизму <a href="/wiki/%D0%A0%D0%B0%D1%81%D0%BE%D0%B2%D0%B0%D1%8F_%D1%82%D0%B5%D0%BE%D1%80%D0%B8%D1%8F" title="Расовая теория">научный оттенок</a>, доказывая, что «арийка», чья кровь испорчена евреем, уже <a class="mw-redirect" href="/wiki/%D0%A2%D0%B5%D0%BB%D0%B5%D0%B3%D0%BE%D0%BD%D0%B8%D1%8F_(%D1%82%D0%B5%D0%BE%D1%80%D0%B8%D1%8F)" title="Телегония (теория)">никогда не сможет</a> родить полноценных «арийских» детей. Юлиус был талантливым оратором и активно выступал по всей Германии с антисемитскими речами.
</p><p>Когда в 1925 году Штрейхер был назначен <a href="/wiki/%D0%93%D0%B0%D1%83%D0%BB%D1%8F%D0%B9%D1%82%D0%B5%D1%80" title="Гауляйтер">гауляйтером</a> <a href="/wiki/%D0%9D%D1%8E%D1%80%D0%BD%D0%B1%D0%B5%D1%80%D0%B3" title="Нюрнберг">Нюрнберга</a>, он все ещё преподавал в школе, и его ученики обязаны были приветствовать его возгласом «<a class="mw-redirect" href="/wiki/%D0%A5%D0%B0%D0%B9%D0%BB%D1%8C_%D0%93%D0%B8%D1%82%D0%BB%D0%B5%D1%80!" title="Хайль Гитлер!">Хайль Гитлер!</a>». В 1928 году его уволили из школы за пропаганду антисемитских идей, но это не помешало его дальнейшей нацистской карьере: в 1929 году он стал гауляйтером объединённой <a class="new" href="/w/index.php?title=%D0%A4%D1%80%D0%B0%D0%BD%D0%BA%D0%BE%D0%BD%D0%B8%D1%8F_(%D0%B3%D0%B0%D1%83)&amp;action=edit&amp;redlink=1" title="Франкония (гау) (страница отсутствует)">Франконии</a> и был избран в <a href="/wiki/%D0%91%D0%B0%D0%B2%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9_%D0%BB%D0%B0%D0%BD%D0%B4%D1%82%D0%B0%D0%B3" title="Баварский ландтаг">баварский парламент</a> от нацистов, а в <a href="/wiki/1933_%D0%B3%D0%BE%D0%B4" title="1933 год">1933 году</a> и в <a href="/wiki/%D0%A0%D0%B5%D0%B9%D1%85%D1%81%D1%82%D0%B0%D0%B3_(%D0%92%D0%B5%D0%B9%D0%BC%D0%B0%D1%80%D1%81%D0%BA%D0%B0%D1%8F_%D1%80%D0%B5%D1%81%D0%BF%D1%83%D0%B1%D0%BB%D0%B8%D0%BA%D0%B0)" title="Рейхстаг (Веймарская республика)">Рейхстаг</a>. В 1934 году он получил звание <a href="/wiki/%D0%93%D1%80%D1%83%D0%BF%D0%BF%D0%B5%D0%BD%D1%84%D1%8E%D1%80%D0%B5%D1%80_(%D0%A1%D0%A1_%D0%B8_%D0%A1%D0%90)" title="Группенфюрер (СС и СА)">группенфюрера</a> <a href="/wiki/%D0%A1%D0%A1" title="СС">СС</a>.
</p>
<h2><span id=".D0.92_.D0.A2.D1.80.D0.B5.D1.82.D1.8C.D0.B5.D0.BC_.D1.80.D0.B5.D0.B9.D1.85.D0.B5"></span><span class="mw-headline" id="В_Третьем_рейхе">В Третьем рейхе</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A8%D1%82%D1%80%D0%B5%D0%B9%D1%85%D0%B5%D1%80,_%D0%AE%D0%BB%D0%B8%D1%83%D1%81&amp;veaction=edit&amp;section=3" title="Редактировать раздел «В Третьем рейхе»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A8%D1%82%D1%80%D0%B5%D0%B9%D1%85%D0%B5%D1%80,_%D0%AE%D0%BB%D0%B8%D1%83%D1%81&amp;action=edit&amp;section=3" title="Редактировать код раздела «В Третьем рейхе»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>На посту <a href="/wiki/%D0%93%D0%B0%D1%83%D0%BB%D1%8F%D0%B9%D1%82%D0%B5%D1%80" title="Гауляйтер">гауляйтера</a> Штрейхер имел дурную репутацию, по утверждению <a href="/wiki/%D0%A8%D0%B8%D1%80%D0%B5%D1%80,_%D0%A3%D0%B8%D0%BB%D1%8C%D1%8F%D0%BC" title="Ширер, Уильям">Уильяма Ширера</a>, любил разгуливать по <a href="/wiki/%D0%9D%D1%8E%D1%80%D0%BD%D0%B1%D0%B5%D1%80%D0%B3" title="Нюрнберг">Нюрнбергу</a> с <a href="/wiki/%D0%A1%D1%82%D0%B5%D0%BA_(%D1%82%D1%80%D0%BE%D1%81%D1%82%D1%8C)" title="Стек (трость)">хлыстом</a>, мог делать в пределах своего <a href="/wiki/%D0%93%D0%B0%D1%83" title="Гау">гау</a> практически всё, что хотел<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>.
Однажды он лично избил заключённых Нюрнбергской тюрьмы, сказав после этого сотрудникам своего аппарата:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
Мне это было просто необходимо, теперь мне значительно полегчало.
</p>
<div class="ts-Конец_цитаты-source">— <cite><i><a class="mw-redirect" href="/wiki/%D0%95%D0%BB%D0%B5%D0%BD%D0%B0_%D0%A1%D1%8A%D1%8F%D0%BD%D0%BE%D0%B2%D0%B0" title="Елена Съянова">Елена Съянова</a></i>. «Десятка из колоды Гитлера». — <abbr title="Москва">М.</abbr>: Время, 2005. — С. 53.</cite></div></blockquote>
<p>Штрейхер имел огромное количество любовниц и постоянно шантажировал их мужей, любил с гордостью рассказывать о своих любовных похождениях и был известен своим увлечением <a href="/wiki/%D0%9F%D0%BE%D1%80%D0%BD%D0%BE%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D1%8F" title="Порнография">порнографией</a>.
</p><p>Отношение к Штрейхеру в партии было неоднозначным: <a class="mw-redirect" href="/wiki/%D0%93%D1%91%D1%80%D0%B8%D0%BD%D0%B3,_%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD" title="Гёринг, Герман">Герман Геринг</a>, <a href="/wiki/%D0%93%D0%B5%D1%81%D1%81,_%D0%A0%D1%83%D0%B4%D0%BE%D0%BB%D1%8C%D1%84" title="Гесс, Рудольф">Рудольф Гесс</a>, <a href="/wiki/%D0%9B%D0%B5%D0%B9,_%D0%A0%D0%BE%D0%B1%D0%B5%D1%80%D1%82" title="Лей, Роберт">Роберт Лей</a> и <a href="/wiki/%D0%A8%D0%B0%D1%85%D1%82,_%D0%AF%D0%BB%D0%BC%D0%B0%D1%80" title="Шахт, Ялмар">Ялмар Шахт</a> открыто заявляли, что он своими непристойными статьями и моральным обликом (Штрейхер активно занимался скупкой конфискованного еврейского имущества) наносит движению гораздо больше вреда, чем пользы. О его корыстолюбии в партии ходили легенды. В 1938 году <a href="/wiki/%D0%93%D0%B5%D0%B1%D0%B1%D0%B5%D0%BB%D1%8C%D1%81,_%D0%99%D0%BE%D0%B7%D0%B5%D1%84" title="Геббельс, Йозеф">Йозеф Геббельс</a> неоднократно запрещал его публичные выступления. С другой стороны, <a class="mw-redirect" href="/wiki/%D0%93%D0%B8%D1%82%D0%BB%D0%B5%D1%80" title="Гитлер">Гитлер</a> практически всегда его поддерживал, говоря: «Я не считаю, что задача политического руководителя состоит в том, чтобы попытаться улучшить человеческий материал, лежащий готовым в его руках».
</p><p>В 1937 году Штрейхер вступил в серьёзную конфронтацию с Герингом, назвав в одном из номеров «<a class="mw-redirect" href="/wiki/%D0%94%D0%B5%D1%80_%D0%A8%D1%82%D1%8E%D1%80%D0%BC%D0%B5%D1%80" title="Дер Штюрмер">Дер Штюрмер</a>» его единственную дочь «плодом искусственного оплодотворения». Этот выпад в адрес Геринга был связан с тем, что он хотел видеть <a href="/wiki/%D0%9C%D0%B8%D0%BB%D1%8C%D1%85,_%D0%AD%D1%80%D1%85%D0%B0%D1%80%D0%B4" title="Мильх, Эрхард">Эрхарда Мильха</a>, которого Штрейхер считал евреем, своим заместителем и скорректировал ему биографию<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>. В ответ Геринг добился от Гитлера финансовой проверки деятельности Штрейхера, результатом которой стало удаление его со всех партийных постов в 1940 году. После этого он прямо политикой не занимался, сосредоточившись на редактировании «Дер Штюрмер». Газета продолжала издаваться до февраля 1945 года. В последние недели войны де-факто стал вновь исполнять обязанности гауляйтера Франконии. 23 мая 1945 года он был арестован американцами.
</p>
<h2><span id=".D0.9D.D1.8E.D1.80.D0.BD.D0.B1.D0.B5.D1.80.D0.B3.D1.81.D0.BA.D0.B8.D0.B9_.D0.BF.D1.80.D0.BE.D1.86.D0.B5.D1.81.D1.81"></span><span class="mw-headline" id="Нюрнбергский_процесс">Нюрнбергский процесс</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A8%D1%82%D1%80%D0%B5%D0%B9%D1%85%D0%B5%D1%80,_%D0%AE%D0%BB%D0%B8%D1%83%D1%81&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Нюрнбергский процесс»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A8%D1%82%D1%80%D0%B5%D0%B9%D1%85%D0%B5%D1%80,_%D0%AE%D0%BB%D0%B8%D1%83%D1%81&amp;action=edit&amp;section=4" title="Редактировать код раздела «Нюрнбергский процесс»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Julius_Streicher_72-920.jpg"><img class="mw-file-element" data-file-height="499" data-file-width="390" decoding="async" height="256" src="//upload.wikimedia.org/wikipedia/commons/thumb/0/04/Julius_Streicher_72-920.jpg/200px-Julius_Streicher_72-920.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/0/04/Julius_Streicher_72-920.jpg/300px-Julius_Streicher_72-920.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/0/04/Julius_Streicher_72-920.jpg 2x" width="200"/></a><figcaption>Штрейхер на Нюрнбергском процессе</figcaption></figure>
<p>На <a href="/wiki/%D0%9D%D1%8E%D1%80%D0%BD%D0%B1%D0%B5%D1%80%D0%B3%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D1%80%D0%BE%D1%86%D0%B5%D1%81%D1%81" title="Нюрнбергский процесс">Нюрнбергском процессе</a>, проходившем в главном городе Франконии, гауляйтером которой был Штрейхер в 1925—1940 годах<sup class="reference" id="cite_ref-_7c8d9aac0c2550e2_9-0"><a href="#cite_note-_7c8d9aac0c2550e2-9">[9]</a></sup>, ему было предъявлено обвинение в подстрекательстве к убийствам евреев, что подпадало под Обвинение 4 процесса — преступления против человечества. Во время освидетельствования Штрейхер был признан вменяемым и способным отвечать за свои действия, хотя и одержимым навязчивой идеей. По словам <a class="mw-redirect" href="/wiki/%D0%9F%D1%81%D0%B8%D1%85%D0%B8%D0%B0%D1%82%D1%80" title="Психиатр">психиатров</a>, проводивших обследование, его фанатичный антисемитизм — продукт больной психики, однако в целом он производил впечатление адекватного человека.
</p><p><a href="/wiki/%D0%A4%D1%80%D0%B8%D0%BA,_%D0%92%D0%B8%D0%BB%D1%8C%D0%B3%D0%B5%D0%BB%D1%8C%D0%BC" title="Фрик, Вильгельм">Вильгельм Фрик</a>, за которым на судебных заседаниях сидел Штрейхер, по ходу процесса постоянно называл того «выродком», «злобствующим юдофобом» и «преступным фанатиком»<sup class="reference" id="cite_ref-_c9eeb258a3686c96_10-0"><a href="#cite_note-_c9eeb258a3686c96-10">[10]</a></sup>. Когда столовую для обвиняемых разделили на отсеки для приёма пищи, Штрейхер попал в компанию самых убеждённых нацистов — к <a href="/wiki/%D0%93%D0%B5%D1%81%D1%81,_%D0%A0%D1%83%D0%B4%D0%BE%D0%BB%D1%8C%D1%84" title="Гесс, Рудольф">Гессу</a>, <a href="/wiki/%D0%A0%D0%B5%D0%B4%D0%B5%D1%80,_%D0%AD%D1%80%D0%B8%D1%85" title="Редер, Эрих">Редеру</a> и <a href="/wiki/%D0%A0%D0%B8%D0%B1%D0%B1%D0%B5%D0%BD%D1%82%D1%80%D0%BE%D0%BF,_%D0%98%D0%BE%D0%B0%D1%85%D0%B8%D0%BC_%D1%84%D0%BE%D0%BD" title="Риббентроп, Иоахим фон">фон Риббентропу</a>. По свидетельству психолога <a href="/wiki/%D0%93%D0%B8%D0%BB%D0%B1%D0%B5%D1%80%D1%82,_%D0%93%D1%83%D1%81%D1%82%D0%B0%D0%B2_%D0%9C%D0%B0%D1%80%D0%BA" title="Гилберт, Густав Марк">Густава Гилберта</a>, работавшего с заключёнными, «последние двое сидели с оскорблённым видом — им явно было не по душе принимать пищу в обществе такой одиозной фигуры, как он».
</p><p>Его адвокат доктор Ганс Маркс пытался построить защиту своего клиента на том, что его «<a class="mw-redirect" href="/wiki/%D0%94%D0%B5%D1%80_%D0%A8%D1%82%D1%8E%D1%80%D0%BC%D0%B5%D1%80" title="Дер Штюрмер">Дер Штюрмер</a>» никто не воспринимал всерьёз. Однако Штрейхер выступал резко против этого. Он доказывал, что отдавал приказы о разрушении <a href="/wiki/%D0%A1%D0%B8%D0%BD%D0%B0%D0%B3%D0%BE%D0%B3%D0%B0" title="Синагога">синагог</a> «из чисто архитектурных соображений». По его словам, он читал об уничтожении евреев в зарубежных газетах, но не верил в это. Он утверждал, что призывал искоренять еврейский народ, но не в буквальном смысле.
</p><p>В последнем слове Штрейхер заявил, что хотел положить конец проникновению евреев во все сферы германской жизни, а также даже сослался на Талмуд в качестве аргумента за то, чтобы христиане принимали меры для защиты от евреев. Он назвал процесс «триумфом мирового еврейства», заявив, что трое из судей являются евреями<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup>. Когда его приговорили к смерти, он сказал: «Разумеется, смертная казнь! Чего ещё можно было ожидать! Да и они с самого начала это знали».
</p><p>Адвокат Штрейхера подавал апелляцию без его же согласия, однако она была отклонена<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>.
</p><p>Во время казни, встав под петлёй, он громко крикнул: «Purimfest!» (еврейский праздник <a href="/wiki/%D0%9F%D1%83%D1%80%D0%B8%D0%BC" title="Пурим">Пурим</a> — торжество над врагами иудеев). «Я отправляюсь к Богу. Однажды большевики повесят вас!» Затем он три раза выкрикнул: «<a class="mw-redirect" href="/wiki/%D0%A5%D0%B0%D0%B9%D0%BB%D1%8C_%D0%93%D0%B8%D1%82%D0%BB%D0%B5%D1%80!" title="Хайль Гитлер!">Хайль Гитлер!</a>» По свидетельству палача <a href="/wiki/%D0%92%D1%83%D0%B4%D0%B7,_%D0%94%D0%B6%D0%BE%D0%BD" title="Вудз, Джон">Джона Вудза</a>, последнее «Хайль Гитлер!» донеслось уже из мешка. После приведения приговора в исполнение тело Штрейхера вместе с телами других казнённых было <a href="/wiki/%D0%9A%D1%80%D0%B5%D0%BC%D0%B0%D1%86%D0%B8%D1%8F" title="Кремация">кремировано</a>, а прах тайно вывезен и <a href="/wiki/%D0%A0%D0%B0%D0%B7%D0%B2%D0%B5%D0%B8%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5_%D0%BF%D1%80%D0%B0%D1%85%D0%B0" title="Развеивание праха">развеян</a>.
</p>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Dead_Julius_Streicher.jpg"><img class="mw-file-element" data-file-height="328" data-file-width="421" decoding="async" height="234" src="//upload.wikimedia.org/wikipedia/commons/thumb/7/79/Dead_Julius_Streicher.jpg/300px-Dead_Julius_Streicher.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/7/79/Dead_Julius_Streicher.jpg 1.5x" width="300"/></a><figcaption>Тело Штрейхера после повешения, 16 октября 1946 года</figcaption></figure>
<p>Юлиус Штрейхер был:
</p>
<ul><li>единственным из обвиняемых, кто бывал уже в Нюрнбергском зале правосудия до процесса в качестве обвиняемого. Его обвиняли в <a href="/wiki/%D0%A0%D0%B0%D1%81%D1%82%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5" title="Растление">растлении</a> малолетней, но он был оправдан<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup>.</li>
<li>единственным из подсудимых в Нюрнберге, которого признали виновным исключительно за печатную пропаганду, а не за реальное участие в принятии решений о войне и геноциде.</li>
<li>единственным из казнённых в Нюрнберге, кто упомянул <a class="mw-redirect" href="/wiki/%D0%93%D0%B8%D1%82%D0%BB%D0%B5%D1%80" title="Гитлер">Гитлера</a> в предсмертной речи.</li></ul>
<h2><span id=".D0.A1.D0.B5.D0.BC.D1.8C.D1.8F_.D0.A8.D1.82.D1.80.D0.B5.D0.B9.D1.85.D0.B5.D1.80.D0.B0"></span><span class="mw-headline" id="Семья_Штрейхера">Семья Штрейхера</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A8%D1%82%D1%80%D0%B5%D0%B9%D1%85%D0%B5%D1%80,_%D0%AE%D0%BB%D0%B8%D1%83%D1%81&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Семья Штрейхера»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A8%D1%82%D1%80%D0%B5%D0%B9%D1%85%D0%B5%D1%80,_%D0%AE%D0%BB%D0%B8%D1%83%D1%81&amp;action=edit&amp;section=5" title="Редактировать код раздела «Семья Штрейхера»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>В 1913 году в <a href="/wiki/%D0%9D%D1%8E%D1%80%D0%BD%D0%B1%D0%B5%D1%80%D0%B3" title="Нюрнберг">Нюрнберге</a> он женился на дочери булочника Кунигунде Рот (<a href="/wiki/%D0%9D%D0%B5%D0%BC%D0%B5%D1%86%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Немецкий язык">нем.</a> <span lang="de" style="font-style:italic;">Kunigunde Roth</span>). У них родилось двое сыновей: Лотар (1915) и Элмар (1918). Его жена умерла в 1943 году после 30 лет совместной жизни. В мае 1945 года он женился на своей бывшей секретарше Адель Таппе. На Нюрнбергском процессе она выступила в его защиту, утверждая, что он был порядочным семьянином и вообще хорошим человеком<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup>. По словам <a href="/wiki/%D0%99%D0%BE%D0%B4%D0%BB%D1%8C,_%D0%90%D0%BB%D1%8C%D1%84%D1%80%D0%B5%D0%B4" title="Йодль, Альфред">Альфреда Йодля</a>, «она слишком мила для такого законченного подонка, каким является её супруг».
</p>
<h2><span id=".D0.9F.D0.B0.D0.BC.D1.8F.D1.82.D1.8C"></span><span class="mw-headline" id="Память">Память</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A8%D1%82%D1%80%D0%B5%D0%B9%D1%85%D0%B5%D1%80,_%D0%AE%D0%BB%D0%B8%D1%83%D1%81&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Память»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A8%D1%82%D1%80%D0%B5%D0%B9%D1%85%D0%B5%D1%80,_%D0%AE%D0%BB%D0%B8%D1%83%D1%81&amp;action=edit&amp;section=6" title="Редактировать код раздела «Память»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>26 ноября 2017 года в <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%B2%D0%B0" title="Москва">Москве</a> <a class="new" href="/w/index.php?title=%D0%92%D1%81%D0%B5%D1%80%D0%BE%D1%81%D1%81%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D1%8A%D0%B5%D0%B7%D0%B4_%D0%B2_%D0%B7%D0%B0%D1%89%D0%B8%D1%82%D1%83_%D0%BF%D1%80%D0%B0%D0%B2_%D1%87%D0%B5%D0%BB%D0%BE%D0%B2%D0%B5%D0%BA%D0%B0&amp;action=edit&amp;redlink=1" title="Всероссийский съезд в защиту прав человека (страница отсутствует)">Всероссийский съезд в защиту прав человека</a> принял решение об учреждении <a href="/wiki/%D0%90%D0%BD%D1%82%D0%B8%D0%BF%D1%80%D0%B5%D0%BC%D0%B8%D1%8F" title="Антипремия">антипремии</a> имени Юлиуса Штрейхера. Эту антипремию было решено вручать «представителю <a class="mw-redirect" href="/wiki/%D0%A1%D0%9C%D0%98_%D0%B2_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B8" title="СМИ в России">российских СМИ</a>, внёсшему наибольший вклад в атмосферу ненависти и лжи»<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup>.
</p>
