<p><b>Сант-Аполлина́ре-ин-Кла́ссе</b> (<a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D1%8C%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Итальянский язык">итал.</a> <span lang="it" style="font-style:italic;">basilica di Sant'Apollinare in Classe</span>) — <a href="/wiki/%D0%91%D0%B0%D0%B7%D0%B8%D0%BB%D0%B8%D0%BA%D0%B0" title="Базилика">базилика</a> в районе Классе (бывший порт Классис) <a href="/wiki/%D0%A0%D0%B0%D0%B2%D0%B5%D0%BD%D0%BD%D0%B0" title="Равенна">Равенны</a> (<a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D0%B8%D1%8F" title="Италия">Италия</a>), памятник <a href="/wiki/%D0%98%D1%81%D0%BA%D1%83%D1%81%D1%81%D1%82%D0%B2%D0%BE_%D0%92%D0%B8%D0%B7%D0%B0%D0%BD%D1%82%D0%B8%D0%B8" title="Искусство Византии">ранневизантийского искусства</a>. Построена во второй четверти <a href="/wiki/VI_%D0%B2%D0%B5%D0%BA" title="VI век">VI века</a> над могилой первого равеннского <a href="/wiki/%D0%95%D0%BF%D0%B8%D1%81%D0%BA%D0%BE%D0%BF" title="Епископ">епископа</a> <a class="mw-redirect" href="/wiki/%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B9_%D0%90%D0%BF%D0%BE%D0%BB%D0%BB%D0%B8%D0%BD%D0%B0%D1%80%D0%B8%D0%B9" title="Святой Аполлинарий">святого Аполлинария</a> (отсюда название). Храм украшен <a href="/wiki/%D0%9C%D0%BE%D0%B7%D0%B0%D0%B8%D0%BA%D0%B0" title="Мозаика">мозаиками</a> <a href="/wiki/%D0%AE%D1%81%D1%82%D0%B8%D0%BD%D0%B8%D0%B0%D0%BD_I" title="Юстиниан I">юстиниановского</a> периода (то есть самыми поздними из равеннских мозаик), сохранившимися в <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%85%D0%B0" title="Конха">конхе</a> <a class="mw-redirect" href="/wiki/%D0%90%D0%BF%D1%81%D0%B8%D0%B4%D0%B0_(%D0%B0%D1%80%D1%85%D0%B8%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D0%B0)" title="Апсида (архитектура)">апсиды</a>. В <a href="/wiki/1996_%D0%B3%D0%BE%D0%B4" title="1996 год">1996 году</a> базилика в составе раннехристианских памятников Равенны была включена в число объектов <a href="/wiki/%D0%92%D1%81%D0%B5%D0%BC%D0%B8%D1%80%D0%BD%D0%BE%D0%B5_%D0%BD%D0%B0%D1%81%D0%BB%D0%B5%D0%B4%D0%B8%D0%B5" title="Всемирное наследие">Всемирного наследия</a>.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>
</p>

<h2><span id=".D0.98.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.8F_.D0.B1.D0.B0.D0.B7.D0.B8.D0.BB.D0.B8.D0.BA.D0.B8"></span><span class="mw-headline" id="История_базилики">История базилики</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B0%D0%BD%D1%82-%D0%90%D0%BF%D0%BE%D0%BB%D0%BB%D0%B8%D0%BD%D0%B0%D1%80%D0%B5-%D0%B8%D0%BD-%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B5&amp;veaction=edit&amp;section=1" title="Редактировать раздел «История базилики»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B0%D0%BD%D1%82-%D0%90%D0%BF%D0%BE%D0%BB%D0%BB%D0%B8%D0%BD%D0%B0%D1%80%D0%B5-%D0%B8%D0%BD-%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B5&amp;action=edit&amp;section=1" title="Редактировать код раздела «История базилики»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Базилика была заложена епископом Урсицином во второй четверти VI века на месте захоронения св. Аполлинария. Строительство велось на средства греческого ростовщика <a href="/wiki/%D0%AE%D0%BB%D0%B8%D0%B0%D0%BD_%D0%90%D1%80%D0%B3%D0%B5%D0%BD%D1%82%D0%B0%D1%80%D0%B8%D0%B9" title="Юлиан Аргентарий">Юлиана Аргентария</a> (который финансировал также строительство других храмов Равенны — базилики <a href="/wiki/%D0%A1%D0%B0%D0%BD-%D0%92%D0%B8%D1%82%D0%B0%D0%BB%D0%B5" title="Сан-Витале">Сан-Витале</a> и <span data-interwiki-article="Chiesa di San Michele in Africisco" data-interwiki-lang="it"><a class="new" href="/w/index.php?title=%D0%A6%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C_%D1%81%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%9C%D0%B8%D1%85%D0%B0%D0%B8%D0%BB%D0%B0_(%D0%A0%D0%B0%D0%B2%D0%B5%D0%BD%D0%BD%D0%B0)&amp;action=edit&amp;redlink=1" title="Церковь святого Михаила (Равенна) (страница отсутствует)">церкви святого Михаила</a></span><span class="noprint" style="font-style:normal; font-weight:normal;"><sup><a class="extiw" href="https://it.wikipedia.org/wiki/Chiesa_di_San_Michele_in_Africisco" title="it:Chiesa di San Michele in Africisco"><span title="Chiesa di San Michele in Africisco — версия статьи «Церковь святого Михаила (Равенна)» на итальянском языке">[it]</span></a></sup></span>). Церковь была <a href="/wiki/%D0%9E%D1%81%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%B8%D0%B5_%D1%85%D1%80%D0%B0%D0%BC%D0%B0" title="Освящение храма">освящена</a> <a href="/wiki/9_%D0%BC%D0%B0%D1%8F" title="9 мая">9 мая</a> <a href="/wiki/549_%D0%B3%D0%BE%D0%B4" title="549 год">549 года</a> епископом <a href="/wiki/%D0%9C%D0%B0%D0%BA%D1%81%D0%B8%D0%BC%D0%B8%D0%B0%D0%BD_(%D0%B5%D0%BF%D0%B8%D1%81%D0%BA%D0%BE%D0%BF_%D0%A0%D0%B0%D0%B2%D0%B5%D0%BD%D0%BD%D1%81%D0%BA%D0%B8%D0%B9)" title="Максимиан (епископ Равеннский)">Максимианом</a>.<sup class="reference" id="cite_ref-laz_2-0"><a href="#cite_note-laz-2">[2]</a></sup> Главной реликвией церкви являются <a href="/wiki/%D0%9C%D0%BE%D1%89%D0%B8" title="Мощи">мощи</a> святого Аполлинария, обнаруженные в ходе строительства. В <a href="/wiki/856_%D0%B3%D0%BE%D0%B4" title="856 год">856 году</a> из-за угроз вражеских набегов их перенесли внутрь городских стен в базилику <a href="/wiki/%D0%A1%D0%B0%D0%BD%D1%82-%D0%90%D0%BF%D0%BE%D0%BB%D0%BB%D0%B8%D0%BD%D0%B0%D1%80%D0%B5-%D0%9D%D1%83%D0%BE%D0%B2%D0%BE" title="Сант-Аполлинаре-Нуово">Сант-Аполлинаре-Нуово</a>.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> В <a href="/wiki/1748_%D0%B3%D0%BE%D0%B4" title="1748 год">1748 году</a> мощи вернулись в базилику, их поместили в главном <a href="/wiki/%D0%90%D0%BB%D1%82%D0%B0%D1%80%D1%8C" title="Алтарь">алтаре</a> храма, а не в <a href="/wiki/%D0%9A%D1%80%D0%B8%D0%BF%D1%82%D0%B0" title="Крипта">крипте</a> где они изначально находились.
</p><p>В течение VI—IX веков было создано мозаичное убранство базилики. В <a href="/wiki/X_%D0%B2%D0%B5%D0%BA" title="X век">X веке</a> к ней были пристроены <a href="/wiki/%D0%9F%D1%80%D0%B8%D0%B4%D0%B5%D0%BB" title="Придел">придел</a>, <a class="mw-redirect" href="/wiki/%D0%9D%D0%B0%D1%80%D1%82%D0%B5%D0%BA%D1%81" title="Нартекс">нартекс</a> и <a href="/wiki/%D0%A0%D0%BE%D1%82%D0%BE%D0%BD%D0%B4%D0%B0" title="Ротонда">ротонда</a>-<a href="/wiki/%D0%9A%D0%BE%D0%BB%D0%BE%D0%BA%D0%BE%D0%BB%D1%8C%D0%BD%D1%8F" title="Колокольня">колокольня</a>. После разграбления Равенны венецианцами в <a href="/wiki/1449_%D0%B3%D0%BE%D0%B4" title="1449 год">1449 году</a> первоначальные <a href="/wiki/%D0%9C%D0%BE%D0%B7%D0%B0%D0%B8%D0%BA%D0%B0" title="Мозаика">мозаики</a> сохранились только в <a href="/wiki/%D0%90%D0%BF%D1%81%D0%B8%D0%B4%D0%B0" title="Апсида">апсиде</a>.
</p><p>В марте <a href="/wiki/1001_%D0%B3%D0%BE%D0%B4" title="1001 год">1001 года</a> в монастыре, находившемся при базилике, провёл <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B8%D0%B9_%D0%BF%D0%BE%D1%81%D1%82" title="Великий пост">Великий пост</a> император <a class="mw-redirect" href="/wiki/%D0%9E%D1%82%D1%82%D0%BE%D0%BD_III_(%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%BE%D1%80_%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D0%BE%D0%B9_%D0%A0%D0%B8%D0%BC%D1%81%D0%BA%D0%BE%D0%B9_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D0%B8)" title="Оттон III (император Священной Римской империи)">Оттон III</a>. Для императора, изгнанного из <a href="/wiki/%D0%A0%D0%B8%D0%BC" title="Рим">Рима</a> и глубоко переживавшего крушение своего замысла о восстановлении прежней Римской империи, Великий пост, проведённый в Сант-Аполлинаре-ин-Классе, стал временем духовного перелома. Наставником императора стал отшельник <a class="mw-redirect" href="/wiki/%D0%A0%D0%BE%D0%BC%D1%83%D0%B0%D0%BB%D1%8C%D0%B4_%D0%A0%D0%B0%D0%B2%D0%B5%D0%BD%D0%BD%D1%81%D0%BA%D0%B8%D0%B9" title="Ромуальд Равеннский">Ромуальд Равеннский</a>, убеждавший императора уйти из мира в <a href="/wiki/%D0%9C%D0%BE%D0%BD%D0%B0%D1%81%D1%82%D1%8B%D1%80%D1%8C" title="Монастырь">монастырь</a> (сохранились его пророческие слова: «<i>Если пойдёшь на Рим, то не увидишь более Равенны</i>»<sup class="reference" id="cite_ref-Рыжов_4-0"><a href="#cite_note-Рыжов-4">[4]</a></sup>), и Оттон III всерьёз стал раздумывать об этом. Затем, под влиянием своих советников, Оттон III вернулся к политической жизни и стал готовить поход на Рим, но умер уже в следующем <a href="/wiki/1002_%D0%B3%D0%BE%D0%B4" title="1002 год">1002 году</a><sup class="reference" id="cite_ref-Оттон_5-0"><a href="#cite_note-Оттон-5">[5]</a></sup>. В левом нефе базилики можно видеть мемориальную доску, напоминающую об этих событиях.<sup class="reference" id="cite_ref-RCA-26_6-0"><a href="#cite_note-RCA-26-6">[6]</a></sup>
</p>
<h2><span id=".D0.90.D1.80.D1.85.D0.B8.D1.82.D0.B5.D0.BA.D1.82.D1.83.D1.80.D0.BD.D1.8B.D0.B5_.D0.BE.D1.81.D0.BE.D0.B1.D0.B5.D0.BD.D0.BD.D0.BE.D1.81.D1.82.D0.B8"></span><span class="mw-headline" id="Архитектурные_особенности">Архитектурные особенности</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B0%D0%BD%D1%82-%D0%90%D0%BF%D0%BE%D0%BB%D0%BB%D0%B8%D0%BD%D0%B0%D1%80%D0%B5-%D0%B8%D0%BD-%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B5&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Архитектурные особенности»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B0%D0%BD%D1%82-%D0%90%D0%BF%D0%BE%D0%BB%D0%BB%D0%B8%D0%BD%D0%B0%D1%80%D0%B5-%D0%B8%D0%BD-%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B5&amp;action=edit&amp;section=2" title="Редактировать код раздела «Архитектурные особенности»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Church_of_S._Apollinaire_in_Classe,_Ravenna_(plan).jpg"><img class="mw-file-element" data-file-height="619" data-file-width="275" decoding="async" height="270" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/d9/Church_of_S._Apollinaire_in_Classe%2C_Ravenna_%28plan%29.jpg/120px-Church_of_S._Apollinaire_in_Classe%2C_Ravenna_%28plan%29.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/d/d9/Church_of_S._Apollinaire_in_Classe%2C_Ravenna_%28plan%29.jpg/180px-Church_of_S._Apollinaire_in_Classe%2C_Ravenna_%28plan%29.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/d/d9/Church_of_S._Apollinaire_in_Classe%2C_Ravenna_%28plan%29.jpg/240px-Church_of_S._Apollinaire_in_Classe%2C_Ravenna_%28plan%29.jpg 2x" width="120"/></a><figcaption><center>План базилики</center></figcaption></figure>
<p>Построена из тонкого обожжённого <a href="/wiki/%D0%9A%D0%B8%D1%80%D0%BF%D0%B8%D1%87" title="Кирпич">кирпича</a> (48 х 4 см.), скреплённого раствором, белые полосы которого достигают в толщину 4-х см. Фасад декорирован <a href="/wiki/%D0%9B%D0%B0%D0%BD%D0%B3%D0%BE%D0%B1%D0%B0%D1%80%D0%B4%D1%8B" title="Лангобарды">лангобардской</a> аркатурой: между плоскими <a class="mw-redirect" href="/wiki/%D0%9F%D0%B8%D0%BB%D1%8F%D1%81%D1%82%D1%80%D1%8B" title="Пилястры">пилястрами</a> расположены небольшие двойные арки. Свет в здание проникает через высокие полукруглые окна фасада и многочисленные окна центрального и боковых нефов. Центральный неф завершает пятигранная апсида с пятью окнами, снаружи к ней примыкают прямоугольные <a href="/wiki/%D0%9F%D0%B0%D1%81%D1%82%D0%BE%D1%84%D0%BE%D1%80%D0%B8%D0%B9" title="Пастофорий">пастофории</a> (вспомогательные богослужебные помещения). Апсида приподнята над уровнем пола центрального нефа, к ней ведёт широкая лестница, построенная в <a href="/wiki/1723_%D0%B3%D0%BE%D0%B4" title="1723 год">1723 году</a> (отреставрирована в <a href="/wiki/1909_%D0%B3%D0%BE%D0%B4" title="1909 год">1909 году</a>).
</p><p>Размеры базилики составляют 55,58 на 30,3 метра. Внутренне пространство разделено на три <a href="/wiki/%D0%9D%D0%B5%D1%84" title="Неф">нефа</a>. Центральный неф обрамляет колоннада из 12 колонн в каждом ряду. Они установлены на квадратные базы и увенчаны композитными <a href="/wiki/%D0%9A%D0%B0%D0%BF%D0%B8%D1%82%D0%B5%D0%BB%D1%8C" title="Капитель">капителями</a> византийской работы с развевающимися ажурными листьями в форме бабочек. Серый <a href="/wiki/%D0%9C%D1%80%D0%B0%D0%BC%D0%BE%D1%80" title="Мрамор">мрамор</a> с белыми прожилками для колонн был привезён из каменоломен горного массива <a class="mw-redirect" href="/wiki/%D0%93%D0%B8%D0%BC%D0%B5%D1%82%D1%82" title="Гиметт">Гиметт</a> (<a href="/wiki/%D0%93%D1%80%D0%B5%D1%86%D0%B8%D1%8F" title="Греция">Греция</a>) или острова <a href="/wiki/%D0%9C%D0%B0%D1%80%D0%BC%D0%B0%D1%80%D0%B0_(%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2)" title="Мармара (остров)">Проконнессос</a> (<a href="/wiki/%D0%9C%D1%80%D0%B0%D0%BC%D0%BE%D1%80%D0%BD%D0%BE%D0%B5_%D0%BC%D0%BE%D1%80%D0%B5" title="Мраморное море">Мраморное море</a>)<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>. Все колонны сходны и по исходному материалу, и по стилю обработки, что позволяет предположить, что они были заказаны для украшения именно данной базилики, а не изъяты из других, более ранних строений. Колоннада поддерживает аркаду над которой в медальонах в хронологическом порядке помещены <a href="/wiki/%D0%A4%D1%80%D0%B5%D1%81%D0%BA%D0%B0" title="Фреска">фресковые</a> изображения равеннских епископов, сделанные в <a href="/wiki/XVIII_%D0%B2%D0%B5%D0%BA" title="XVIII век">XVIII веке</a>. Нижний уровень стен базилики был изначально облицован мрамором, который был снят в <a href="/wiki/XV_%D0%B2%D0%B5%D0%BA" title="XV век">XV веке</a> по указанию <a href="/wiki/%D0%9C%D0%B0%D0%BB%D0%B0%D1%82%D0%B5%D1%81%D1%82%D0%B0,_%D0%A1%D0%B8%D0%B4%D0%B6%D0%B8%D0%B7%D0%BC%D0%BE%D0%BD%D0%B4%D0%BE_%D0%9F%D0%B0%D0%BD%D0%B4%D0%BE%D0%BB%D1%8C%D1%84%D0%BE" title="Малатеста, Сиджизмондо Пандольфо">Сиджизмондо Малатеста</a> и использован при строительстве <a href="/wiki/%D0%A2%D0%B5%D0%BC%D0%BF%D0%B8%D0%BE_%D0%9C%D0%B0%D0%BB%D0%B0%D1%82%D0%B5%D1%81%D1%82%D0%B8%D0%B0%D0%BD%D0%BE" title="Темпио Малатестиано">кафедрального собора</a> в <a href="/wiki/%D0%A0%D0%B8%D0%BC%D0%B8%D0%BD%D0%B8" title="Римини">Римини</a>.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> На полу базилики, в её северо-восточном и юго-западном углах, сохранились фрагменты оригинального наборного мозаичного пола. Мозаика в юго-западном углу сохранила имена заказчиков данной работы: Гауденции и Феликса.
</p><p>К базилике пристроена цилиндрическая <a href="/wiki/%D0%9A%D0%BE%D0%BB%D0%BE%D0%BA%D0%BE%D0%BB%D1%8C%D0%BD%D1%8F" title="Колокольня">колокольня</a> высотой 37,5 метров и диаметром 6,17 метров. Она является типичной равеннской <a href="/wiki/%D0%9A%D0%B0%D0%BC%D0%BF%D0%B0%D0%BD%D0%B8%D0%BB%D0%B0" title="Кампанила">кампанилой</a>: оконные проёмы имеют прогрессию створок — от одной до трёх.

</p>

<h2><span id=".D0.92.D0.BD.D1.83.D1.82.D1.80.D0.B5.D0.BD.D0.BD.D0.B5.D0.B5_.D1.83.D0.B1.D1.80.D0.B0.D0.BD.D1.81.D1.82.D0.B2.D0.BE"></span><span class="mw-headline" id="Внутреннее_убранство">Внутреннее убранство</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B0%D0%BD%D1%82-%D0%90%D0%BF%D0%BE%D0%BB%D0%BB%D0%B8%D0%BD%D0%B0%D1%80%D0%B5-%D0%B8%D0%BD-%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B5&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Внутреннее убранство»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B0%D0%BD%D1%82-%D0%90%D0%BF%D0%BE%D0%BB%D0%BB%D0%B8%D0%BD%D0%B0%D1%80%D0%B5-%D0%B8%D0%BD-%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B5&amp;action=edit&amp;section=3" title="Редактировать код раздела «Внутреннее убранство»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Ravenna_BW_3.JPG"><img class="mw-file-element" data-file-height="2556" data-file-width="3736" decoding="async" height="178" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/41/Ravenna_BW_3.JPG/260px-Ravenna_BW_3.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/41/Ravenna_BW_3.JPG/390px-Ravenna_BW_3.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/41/Ravenna_BW_3.JPG/520px-Ravenna_BW_3.JPG 2x" width="260"/></a><figcaption><center>Центральный неф</center></figcaption></figure>
<p>Базилика украшена самыми поздними из равеннских мозаик юстиниановского периода (середина <a href="/wiki/VI_%D0%B2%D0%B5%D0%BA" title="VI век">VI века</a>), сохранившимися в её <a class="mw-redirect" href="/wiki/%D0%90%D0%BF%D1%81%D0%B8%D0%B4%D0%B0_(%D0%B0%D1%80%D1%85%D0%B8%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D0%B0)" title="Апсида (архитектура)">апсиде</a> (534—549). Прочие мозаики были созданы во второй половине <a href="/wiki/VII_%D0%B2%D0%B5%D0%BA" title="VII век">VII</a> и <a href="/wiki/IX_%D0%B2%D0%B5%D0%BA" title="IX век">IX веках</a>. Академик <a href="/wiki/%D0%9B%D0%B0%D0%B7%D0%B0%D1%80%D0%B5%D0%B2,_%D0%92%D0%B8%D0%BA%D1%82%D0%BE%D1%80_%D0%9D%D0%B8%D0%BA%D0%B8%D1%82%D0%B8%D1%87" title="Лазарев, Виктор Никитич">В. Н. Лазарев</a> отмечает:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
<i>Мастера, работавшие в Сант Аполлинаре ин Классе, близки тем мастерам, которые исполнили мозаики <a class="mw-redirect" href="/wiki/%D0%9F%D1%80%D0%B5%D1%81%D0%B1%D0%B8%D1%82%D0%B5%D1%80%D0%B8%D0%B9" title="Пресбитерий">пресбитерия</a> в Сан Витале. И у них мы наблюдаем тягу к упрощенным формам и к излишне ярким цветовым сочетаниям. Этому сопутствует снижение качества рисунка и вялая трактовка <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%BD%D0%B0%D1%86%D0%B8%D1%8F" title="Карнация">карнации</a>, в которой заметно уменьшается количество тональных оттенков.</i><sup class="reference" id="cite_ref-laz_2-1"><a href="#cite_note-laz-2">[2]</a></sup>
</p>
</blockquote>
<p>Как отмечают исследователи, мозаики Сант-Аполлинаре-ин-Классе отражают появившуюся в послеюстиниановскую эпоху тенденцию фронтального изображения фигур при которой происходит отказ от передачи каких-либо движений и поворотов.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup> Мозаики Сант-Аполлинаре-ин-Классе, как и базилики Сан-Витале не представляют события священной истории в их исторической последовательности, они имеют своей целью иллюстрацию догматического учения церкви, раскрываемого через символизм <a href="/wiki/%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D0%BE%D0%B5_%D0%9F%D0%B8%D1%81%D0%B0%D0%BD%D0%B8%D0%B5" title="Священное Писание">Священного Писания</a>.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> Несмотря на использование в мозаиках цветного стекла, <a href="/wiki/%D0%A1%D0%BC%D0%B0%D0%BB%D1%8C%D1%82%D0%B0" title="Смальта">смальты</a>, золота и <a class="mw-redirect" href="/wiki/%D0%9F%D0%BE%D0%BB%D1%83%D0%B4%D1%80%D0%B0%D0%B3%D0%BE%D1%86%D0%B5%D0%BD%D0%BD%D1%8B%D0%B5_%D0%BA%D0%B0%D0%BC%D0%BD%D0%B8" title="Полудрагоценные камни">полудрагоценных камней</a>, для лиц и светлых одежд использован мрамор, что привело к появлению более плоских форм. Это особенно заметно в мозаиках с изображением равеннских епископов между окнами апсиды.
</p><p>Искусствовед <a href="/wiki/%D0%9C%D1%83%D1%80%D0%B0%D1%82%D0%BE%D0%B2,_%D0%9F%D0%B0%D0%B2%D0%B5%D0%BB_%D0%9F%D0%B0%D0%B2%D0%BB%D0%BE%D0%B2%D0%B8%D1%87" title="Муратов, Павел Павлович">Павел Муратов</a> в <a href="/wiki/1911_%D0%B3%D0%BE%D0%B4" title="1911 год">1911 году</a> писал о базилике:
</p>
<blockquote>
<p>Эта церковь во многом напоминает базилику Сан Аполлинаре Нуово. В ней также имеются мозаики, но только не на стенах главного нефа, а в алтарной апсиде. Эти мозаики исполнены позднее, чем все в Равенне, и здесь это искусство не стоит на такой высоте, как в Мавзолее Галлы Плацидии или даже в Сан Витале. Кроме того, здешних мозаик сильно коснулась реставрация, сделавшая сомнительными многие их части.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup>
</p>
</blockquote>
<p>В центральном нефе базилики на возвышении расположен большой <a href="/wiki/%D0%90%D0%BB%D1%82%D0%B0%D1%80%D1%8C" title="Алтарь">алтарь</a>, посвященный <a href="/wiki/%D0%91%D0%BE%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%B8%D1%86%D0%B0" title="Богородица">Деве Марии</a>, он был создан в <a href="/wiki/XI_%D0%B2%D0%B5%D0%BA" title="XI век">XI веке</a> с использованием материалов алтаря VI века. На стороне данного алтаря, обращённой к пресбитерию, указано, что на этом месте святой <a class="mw-redirect" href="/wiki/%D0%A0%D0%BE%D0%BC%D1%83%D0%B0%D0%BB%D1%8C%D0%B4_%D0%A0%D0%B0%D0%B2%D0%B5%D0%BD%D0%BD%D1%81%D0%BA%D0%B8%D0%B9" title="Ромуальд Равеннский">Ромуальд Равеннский</a> удостоился двух видений Богородицы (сведения об этом почерпнуты у <a href="/wiki/%D0%9F%D1%91%D1%82%D1%80_%D0%94%D0%B0%D0%BC%D0%B8%D0%B0%D0%BD%D0%B8" title="Пётр Дамиани">Петра Дамиани</a>). Перед этим же алтарём находится мемориальная плита <a href="/wiki/XVIII_%D0%B2%D0%B5%D0%BA" title="XVIII век">XVIII века</a>, сообщающая без каких-либо исторических подробностей, что мощи святого Аполлинария были обнаружены именно на этом месте<sup class="reference" id="cite_ref-RCA-12_12-0"><a href="#cite_note-RCA-12-12">[12]</a></sup>. На стене правого нефа имеется <a href="/wiki/%D0%9C%D0%B5%D0%BC%D0%BE%D1%80%D0%B8%D0%B0%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D0%B4%D0%BE%D1%81%D0%BA%D0%B0" title="Мемориальная доска">мемориальная доска</a> в память о посещении базилики папой <a href="/wiki/%D0%98%D0%BE%D0%B0%D0%BD%D0%BD_%D0%9F%D0%B0%D0%B2%D0%B5%D0%BB_II" title="Иоанн Павел II">Иоанном Павлом II</a> 11 мая <a href="/wiki/1986_%D0%B3%D0%BE%D0%B4" title="1986 год">1986 года</a>.<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup>
</p>
<h3><span id=".D0.9C.D0.BE.D0.B7.D0.B0.D0.B8.D0.BA.D0.B8_.D0.B0.D0.BF.D1.81.D0.B8.D0.B4.D1.8B"></span><span class="mw-headline" id="Мозаики_апсиды">Мозаики апсиды</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B0%D0%BD%D1%82-%D0%90%D0%BF%D0%BE%D0%BB%D0%BB%D0%B8%D0%BD%D0%B0%D1%80%D0%B5-%D0%B8%D0%BD-%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B5&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Мозаики апсиды»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B0%D0%BD%D1%82-%D0%90%D0%BF%D0%BE%D0%BB%D0%BB%D0%B8%D0%BD%D0%B0%D1%80%D0%B5-%D0%B8%D0%BD-%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B5&amp;action=edit&amp;section=4" title="Редактировать код раздела «Мозаики апсиды»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Ravenna_BW_4.JPG"><img class="mw-file-element" data-file-height="2592" data-file-width="3872" decoding="async" height="167" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/42/Ravenna_BW_4.JPG/250px-Ravenna_BW_4.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/42/Ravenna_BW_4.JPG/375px-Ravenna_BW_4.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/42/Ravenna_BW_4.JPG/500px-Ravenna_BW_4.JPG 2x" width="250"/></a><figcaption><center>Мозаика конхи апсиды</center></figcaption></figure>
<h4><span id=".D0.9A.D0.BE.D0.BD.D1.85.D0.B0"></span><span class="mw-headline" id="Конха">Конха</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B0%D0%BD%D1%82-%D0%90%D0%BF%D0%BE%D0%BB%D0%BB%D0%B8%D0%BD%D0%B0%D1%80%D0%B5-%D0%B8%D0%BD-%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B5&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Конха»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B0%D0%BD%D1%82-%D0%90%D0%BF%D0%BE%D0%BB%D0%BB%D0%B8%D0%BD%D0%B0%D1%80%D0%B5-%D0%B8%D0%BD-%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B5&amp;action=edit&amp;section=5" title="Редактировать код раздела «Конха»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h4>
<p>В <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%85%D0%B0" title="Конха">конхе</a> апсиды помещена мозаика с необычной для византийского искусства сценой <a href="/wiki/%D0%9F%D1%80%D0%B5%D0%BE%D0%B1%D1%80%D0%B0%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5_%D0%93%D0%BE%D1%81%D0%BF%D0%BE%D0%B4%D0%BD%D0%B5" title="Преображение Господне">Преображения Господня</a>. В ней <a href="/wiki/%D0%98%D0%B8%D1%81%D1%83%D1%81_%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%BE%D1%81" title="Иисус Христос">Иисус Христос</a> аллегорически олицетворён <a href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82" title="Крест">крестом</a>, украшенным <a href="/wiki/%D0%94%D1%80%D0%B0%D0%B3%D0%BE%D1%86%D0%B5%D0%BD%D0%BD%D1%8B%D0%B5_%D0%BA%D0%B0%D0%BC%D0%BD%D0%B8" title="Драгоценные камни">драгоценными камнями</a>, и заключённым в тёмно-синюю <a href="/wiki/%D0%9C%D0%B0%D0%BD%D0%B4%D0%BE%D1%80%D0%BB%D0%B0" title="Мандорла">мандорлу</a>, усыпанную золотыми шестиконечными звёздами. Лишь в пересечении ветвей креста помещён небольшой медальон с образом Христа. Над крестом помещён <a href="/wiki/%D0%90%D0%BA%D1%80%D0%BE%D0%BD%D0%B8%D0%BC" title="Акроним">акроним</a> <a href="/wiki/%D0%98%D1%85%D1%82%D0%B8%D1%81" title="Ихтис">ΙΧΘΥC</a>, по сторонам поперечной ветви креста греческие буквы Α и Ω, а у подножья креста латинская надпись SALUS MUNDI — «<i>Спаситель мира</i>».
</p><p>Присутствовавшие, согласно евангельскому описанию, при этом событии <a href="/wiki/%D0%90%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB%D1%8B" title="Апостолы">апостолы</a> <a href="/wiki/%D0%90%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB_%D0%9F%D1%91%D1%82%D1%80" title="Апостол Пётр">Пётр</a>, <a href="/wiki/%D0%98%D0%BE%D0%B0%D0%BD%D0%BD_%D0%91%D0%BE%D0%B3%D0%BE%D1%81%D0%BB%D0%BE%D0%B2" title="Иоанн Богослов">Иоанн</a> и <a class="mw-redirect" href="/wiki/%D0%90%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB_%D0%98%D0%B0%D0%BA%D0%BE%D0%B2" title="Апостол Иаков">Иаков</a> изображены в образе агнцев, предстоящих кресту (Пётр символически представлен одной овцой слева от креста (если смотреть от входа), братья Зеведеевы — двумя овцами справа от креста<sup class="reference" id="cite_ref-RCA-16_14-0"><a href="#cite_note-RCA-16-14">[14]</a></sup>). Выше в облаках помещена благословляющая десница <a href="/wiki/%D0%91%D0%BE%D0%B3_%D0%9E%D1%82%D0%B5%D1%86" title="Бог Отец">Бога Отца</a> и полуфигуры пророков <a href="/wiki/%D0%9C%D0%BE%D0%B8%D1%81%D0%B5%D0%B9" title="Моисей">Моисея</a> и <a class="mw-redirect" href="/wiki/%D0%98%D0%BB%D0%B8%D1%8F_(%D0%BF%D1%80%D0%BE%D1%80%D0%BE%D0%BA)" title="Илия (пророк)">Илии</a>, протягивающих свои руки к кресту.
</p><p>Сцена Преображения помещена в центре райского сада, полного деревьев и цветов. В его центре помещена фигура святого Аполлинария в позе <a href="/wiki/%D0%9E%D1%80%D0%B0%D0%BD%D1%82%D0%B0" title="Оранта">оранта</a> с воздетыми к небу руками. По сторонам от фигуры святого находятся 12 белоснежных овец (по шесть с каждой стороны от фигуры святого), отделённых друг от друга кустами белых <a href="/wiki/%D0%9B%D0%B8%D0%BB%D0%B8%D1%8F" title="Лилия">лилий</a>. Ряд исследователей считают овец символическим изображением <a href="/wiki/%D0%94%D0%B2%D0%B5%D0%BD%D0%B0%D0%B4%D1%86%D0%B0%D1%82%D1%8C_%D0%B0%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB%D0%BE%D0%B2" title="Двенадцать апостолов">двенадцати апостолов</a>, а другие — образом паствы Аполлинария, бывшего епископом Равенны.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup>
</p><p>Академик В. Н. Лазарев отмечает, что данная композиция символически намекает на вечную жизнь <a href="/wiki/%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D0%BE%D0%BC%D1%83%D1%87%D0%B5%D0%BD%D0%B8%D0%BA" title="Священномученик">священномученика</a> Аполлинарий, чей образ под крестом должен показать, что триумф Аполлинария перекликается с триумфом самого Христа.<sup class="reference" id="cite_ref-laz_2-2"><a href="#cite_note-laz-2">[2]</a></sup> Источником подобной композиции он считает палестинские <a href="/wiki/%D0%9C%D0%B0%D1%80%D1%82%D0%B8%D1%80%D0%B8%D1%83%D0%BC" title="Мартириум">мартирии</a>.
</p>

<h4><span id=".D0.A1.D1.82.D0.B5.D0.BD.D1.8B"></span><span class="mw-headline" id="Стены">Стены</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B0%D0%BD%D1%82-%D0%90%D0%BF%D0%BE%D0%BB%D0%BB%D0%B8%D0%BD%D0%B0%D1%80%D0%B5-%D0%B8%D0%BD-%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B5&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Стены»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B0%D0%BD%D1%82-%D0%90%D0%BF%D0%BE%D0%BB%D0%BB%D0%B8%D0%BD%D0%B0%D1%80%D0%B5-%D0%B8%D0%BD-%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B5&amp;action=edit&amp;section=6" title="Редактировать код раздела «Стены»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h4>
<figure class="mw-default-size mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Sacrifices_of_Abel,_Melchisedec_and_Abraham_-_Sant%27Apollinare_in_Classe_-_Ravenna_2016_(2).jpg"><img class="mw-file-element" data-file-height="3186" data-file-width="3480" decoding="async" height="201" src="//upload.wikimedia.org/wikipedia/commons/thumb/7/73/Sacrifices_of_Abel%2C_Melchisedec_and_Abraham_-_Sant%27Apollinare_in_Classe_-_Ravenna_2016_%282%29.jpg/220px-Sacrifices_of_Abel%2C_Melchisedec_and_Abraham_-_Sant%27Apollinare_in_Classe_-_Ravenna_2016_%282%29.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/7/73/Sacrifices_of_Abel%2C_Melchisedec_and_Abraham_-_Sant%27Apollinare_in_Classe_-_Ravenna_2016_%282%29.jpg/330px-Sacrifices_of_Abel%2C_Melchisedec_and_Abraham_-_Sant%27Apollinare_in_Classe_-_Ravenna_2016_%282%29.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/7/73/Sacrifices_of_Abel%2C_Melchisedec_and_Abraham_-_Sant%27Apollinare_in_Classe_-_Ravenna_2016_%282%29.jpg/440px-Sacrifices_of_Abel%2C_Melchisedec_and_Abraham_-_Sant%27Apollinare_in_Classe_-_Ravenna_2016_%282%29.jpg 2x" width="220"/></a><figcaption><center>Ветхозаветные жертвоприношения</center></figcaption></figure>
<p>Между окнами апсиды помещены мозаичные портреты епископов Равенны, бывших преемниками святого Аполлинария: <a class="mw-redirect" href="/wiki/%D0%AD%D0%BA%D0%BA%D0%BB%D0%B5%D1%81%D0%B8%D0%B9_%D0%A0%D0%B0%D0%B2%D0%B5%D0%BD%D0%BD%D1%81%D0%BA%D0%B8%D0%B9" title="Экклесий Равеннский">Экклесия</a>, <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80_%D0%A0%D0%B0%D0%B2%D0%B5%D0%BD%D0%BD%D1%81%D0%BA%D0%B8%D0%B9" title="Север Равеннский">Севера</a>, <a class="new" href="/w/index.php?title=%D0%A3%D1%80%D1%81%D1%83%D1%81_%D0%A0%D0%B0%D0%B2%D0%B5%D0%BD%D0%BD%D1%81%D0%BA%D0%B8%D0%B9&amp;action=edit&amp;redlink=1" title="Урсус Равеннский (страница отсутствует)">Урсуса</a> и <a href="/wiki/%D0%A3%D1%80%D1%81%D0%B8%D1%86%D0%B8%D0%BD_%D0%A0%D0%B0%D0%B2%D0%B5%D0%BD%D0%BD%D1%81%D0%BA%D0%B8%D0%B9" title="Урсицин Равеннский">Урсицина</a>. Они изображены в одинаковых одеждах, держащими в руках <a href="/wiki/%D0%95%D0%B2%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D0%B8%D0%B5" title="Евангелие">Евангелие</a>. Каждая фигура помещена в небольшую арку с белоснежными занавесями. На боковых стенах помещены две крупные мозаики:
</p>
<ul><li><i>Ветхозаветные жертвоприношения</i></li></ul>
<p>Изображены три сюжета из <a href="/wiki/%D0%91%D0%B8%D0%B1%D0%BB%D0%B8%D1%8F" title="Библия">библейской</a> истории: «<i><a class="mw-redirect" href="/wiki/%D0%96%D0%B5%D1%80%D1%82%D0%B2%D0%BE%D0%BF%D1%80%D0%B8%D0%BD%D0%BE%D1%88%D0%B5%D0%BD%D0%B8%D0%B5_%D0%90%D0%B2%D1%80%D0%B0%D0%B0%D0%BC%D0%B0" title="Жертвоприношение Авраама">Жертвоприношение Авраама</a></i>», «<i>Жертвоприношение <a href="/wiki/%D0%90%D0%B2%D0%B5%D0%BB%D1%8C" title="Авель">Авеля</a></i>» и «<i>Жертвоприношение <a class="mw-redirect" href="/wiki/%D0%9C%D0%B5%D0%BB%D1%8C%D1%85%D0%B8%D1%81%D0%B5%D0%B4%D0%B5%D0%BA" title="Мельхиседек">Мельхиседека</a></i>», которые являются символическим намёком на <a class="mw-redirect" href="/wiki/%D0%A0%D0%B0%D1%81%D0%BF%D1%8F%D1%82%D0%B8%D0%B5_%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%BE%D0%B2%D0%BE" title="Распятие Христово">крестную смерть Христа</a>. Все персонажи помещены вокруг одного <a href="/wiki/%D0%90%D0%BB%D1%82%D0%B0%D1%80%D1%8C" title="Алтарь">алтаря</a> к которому они принесли свои жертвы, из облаков видна благословляющая десница <a href="/wiki/%D0%91%D0%BE%D0%B3" title="Бог">Бога</a>. Объединение трёх жертвоприношений в одной сцене иллюстрирует молитву <a href="/wiki/%D0%90%D0%BD%D0%B0%D1%84%D0%BE%D1%80%D0%B0_(%D0%BB%D0%B8%D1%82%D1%83%D1%80%D0%B3%D0%B8%D1%8F)" title="Анафора (литургия)">евхаристического канона</a> <a href="/wiki/%D0%9C%D0%B5%D1%81%D1%81%D0%B0" title="Месса">римской литургии</a><sup class="reference" id="cite_ref-РИИ-38-40_16-0"><a href="#cite_note-РИИ-38-40-16">[16]</a></sup>:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
Supra quae propitio ac sereno vultu respicere digneris: et accepta habere, sicuti accepta habere dignatus es munera pueri tui iusti Abel, et sacrificium Patriarcha nostri Abrahae: et quod tibi obtulit summus sacerdos tuus Melchisedech, sanctum sacrificium, immaculatam hostiam.
</p>

</p>
</div></div>
</blockquote>
<p>Вся композиция окружена триумфальной аркой, увенчанной двумя орлами, на заднем плане помещена полураскрытая <a href="/wiki/%D0%9F%D0%BE%D1%80%D1%84%D0%B8%D1%80" title="Порфир">порфировая</a> занавесь.
</p>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Privil_classe.jpg"><img class="mw-file-element" data-file-height="328" data-file-width="485" decoding="async" height="203" src="//upload.wikimedia.org/wikipedia/commons/thumb/f/f5/Privil_classe.jpg/300px-Privil_classe.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/f/f5/Privil_classe.jpg/450px-Privil_classe.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/f/f5/Privil_classe.jpg 2x" width="300"/></a><figcaption><center>Дарование привилегий равеннской церкви</center></figcaption></figure>
<ul><li><i>Дарование привилегий равеннской церкви</i></li></ul>
<p><a class="mw-redirect" href="/wiki/%D0%92%D0%B8%D0%B7%D0%B0%D0%BD%D1%82%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B5_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%BE%D1%80%D1%8B" title="Византийские императоры">Император</a> <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD_IV" title="Константин IV">Константин IV</a> в окружении вельмож и <a href="/wiki/%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D0%BE%D1%81%D0%BB%D1%83%D0%B6%D0%B8%D1%82%D0%B5%D0%BB%D1%8C" title="Священнослужитель">священнослужителей</a> вручает <a href="/wiki/%D0%94%D0%B8%D0%B0%D0%BA%D0%BE%D0%BD" title="Диакон">диакону</a> равеннской церкви Репарату, представляемому ему <a href="/wiki/%D0%90%D1%80%D1%85%D0%B8%D0%B5%D0%BF%D0%B8%D1%81%D0%BA%D0%BE%D0%BF" title="Архиепископ">архиепископом</a> Мавром, свиток с латинской надписью <i>PRIVILEGIA</i>. Фигуры одеты в богатые византийские одежды, украшенные <a href="/wiki/%D0%A2%D0%B0%D0%B1%D0%BB%D0%B8%D0%BE%D0%BD" title="Таблион">таблионами</a>, слева от центра расположены представители светской власти во главе с императором, а справа священнослужители в <a class="mw-redirect" href="/wiki/%D0%91%D0%BE%D0%B3%D0%BE%D1%81%D0%BB%D1%83%D0%B6%D0%B5%D0%B1%D0%BD%D0%BE%D0%B5_%D0%BE%D0%B1%D0%BB%D0%B0%D1%87%D0%B5%D0%BD%D0%B8%D0%B5" title="Богослужебное облачение">литургических облачениях</a>. Головы императора и его братьев-соправителей Тиверия и Ираклия<sup class="reference" id="cite_ref-РИИ-112_18-0"><a href="#cite_note-РИИ-112-18">[18]</a></sup> окружены нимбами, что отражает византийскую традицию отмечать таким способом царствующих особ.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup> Мозаика выполнена по аналогии со знаменитым портретом императора <a href="/wiki/%D0%AE%D1%81%D1%82%D0%B8%D0%BD%D0%B8%D0%B0%D0%BD_I" title="Юстиниан I">Юстининана</a> в базилике <a href="/wiki/%D0%A1%D0%B0%D0%BD-%D0%92%D0%B8%D1%82%D0%B0%D0%BB%D0%B5" title="Сан-Витале">Сан-Витале</a>. Изображение также выполнено как <a href="/wiki/%D0%A4%D1%80%D0%B8%D0%B7_(%D0%B0%D1%80%D1%85%D0%B8%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D0%B0)" title="Фриз (архитектура)">фриз</a> и отличается фронтальной композицией.
</p>
<h3><span id=".D0.9C.D0.BE.D0.B7.D0.B0.D0.B8.D0.BA.D0.B8_.D1.84.D1.80.D0.BE.D0.BD.D1.82.D0.BE.D0.BD.D0.B0"></span><span class="mw-headline" id="Мозаики_фронтона">Мозаики фронтона</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B0%D0%BD%D1%82-%D0%90%D0%BF%D0%BE%D0%BB%D0%BB%D0%B8%D0%BD%D0%B0%D1%80%D0%B5-%D0%B8%D0%BD-%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B5&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Мозаики фронтона»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B0%D0%BD%D1%82-%D0%90%D0%BF%D0%BE%D0%BB%D0%BB%D0%B8%D0%BD%D0%B0%D1%80%D0%B5-%D0%B8%D0%BD-%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B5&amp;action=edit&amp;section=7" title="Редактировать код раздела «Мозаики фронтона»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Sheeps_classe.jpg"><img class="mw-file-element" data-file-height="357" data-file-width="620" decoding="async" height="144" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/3a/Sheeps_classe.jpg/250px-Sheeps_classe.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/3a/Sheeps_classe.jpg/375px-Sheeps_classe.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/3/3a/Sheeps_classe.jpg/500px-Sheeps_classe.jpg 2x" width="250"/></a><figcaption><center>Символическое изображение апостолов <br/>(мозаика «триумфальной арки»)</center></figcaption></figure>
<p>Фронтон «триумфальной арки», отделяющей апсиду от центрального нефа, украшен поздней мозаикой VII века (ряд искусствоведов датируют их IX веком). В центре композиции в медальоне помещено изображение <a href="/wiki/%D0%A1%D0%BF%D0%B0%D1%81_%D0%92%D1%81%D0%B5%D0%B4%D0%B5%D1%80%D0%B6%D0%B8%D1%82%D0%B5%D0%BB%D1%8C" title="Спас Вседержитель">Христа Пантократора</a>, держащего в руках <a href="/wiki/%D0%95%D0%B2%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D0%B8%D0%B5" title="Евангелие">Евангелие</a>, его глаза широко раскрыты, лоб нахмурен, голова окружена крестчатым <a href="/wiki/%D0%9D%D0%B8%D0%BC%D0%B1" title="Нимб">нимбом</a>. К нему на фоне тёмно-синего неба движутся <a class="mw-redirect" href="/wiki/%D0%A1%D0%B8%D0%BC%D0%B2%D0%BE%D0%BB%D1%8B_%D0%B5%D0%B2%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D0%B8%D1%81%D1%82%D0%BE%D0%B2" title="Символы евангелистов">символы евангелистов</a>, держащие в руках написанные ими евангельские книги. Ниже помещены 12 овец, символизирующих <a href="/wiki/%D0%94%D0%B2%D0%B5%D0%BD%D0%B0%D0%B4%D1%86%D0%B0%D1%82%D1%8C_%D0%B0%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB%D0%BE%D0%B2" title="Двенадцать апостолов">апостолов</a>. Они выходят на встречу <a href="/wiki/%D0%98%D0%B8%D1%81%D1%83%D1%81_%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%BE%D1%81" title="Иисус Христос">Иисусу</a> из ворот двух городов: <a href="/wiki/%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC" title="Иерусалим">Иерусалима</a> и <a href="/wiki/%D0%92%D0%B8%D1%84%D0%BB%D0%B5%D0%B5%D0%BC" title="Вифлеем">Вифлеема</a>.
</p><p>Паруса арки украшены изображениями плодоносящих пальм на фоне тёмно-синего неба. Эти мозаики являются самыми старыми из украшений фронтона (середина VI век).<sup class="reference" id="cite_ref-laz_2-3"><a href="#cite_note-laz-2">[2]</a></sup> К этому же периоду относятся помещённые на столбах арки образы <a href="/wiki/%D0%90%D1%80%D1%85%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB" title="Архангел">архангелов</a> <a href="/wiki/%D0%90%D1%80%D1%85%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB_%D0%9C%D0%B8%D1%85%D0%B0%D0%B8%D0%BB" title="Архангел Михаил">Михаила</a> и <a href="/wiki/%D0%90%D1%80%D1%85%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB_%D0%93%D0%B0%D0%B2%D1%80%D0%B8%D0%B8%D0%BB" title="Архангел Гавриил">Гавриила</a> в одеждах византийских сановников. Они держат в руках <a class="mw-redirect" href="/wiki/%D0%A5%D0%BE%D1%80%D1%83%D0%B3%D0%B2%D1%8C_(%D0%BF%D1%80%D0%B0%D0%B2%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%BD%D0%B0%D1%8F)" title="Хоругвь (православная)">хоругви</a> с литургической формулой божественного славословия: «<i>свят, свят, свят</i>» (<a href="/wiki/%D0%93%D1%80%D0%B5%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Греческий язык">греч.</a> <span lang="el"><span style="font-family: palatino linotype, new athena unicode, athena, gentium, code2000, serif; font-size: 105%;">Ἅγιος, Ἅγιος, Ἅγιος</span></span>). Под изображениями архангелов помещены фигуры <a href="/wiki/%D0%95%D0%B2%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D0%B8%D1%81%D1%82%D1%8B" title="Евангелисты">евангелистов</a> <a class="mw-redirect" href="/wiki/%D0%95%D0%B2%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D0%B8%D1%81%D1%82_%D0%9C%D0%B0%D1%82%D1%84%D0%B5%D0%B9" title="Евангелист Матфей">Матфея</a> (слева) и <a class="mw-redirect" href="/wiki/%D0%90%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB_%D0%9B%D1%83%D0%BA%D0%B0" title="Апостол Лука">Луки</a> (справа), добавленные уже в <a href="/wiki/XII_%D0%B2%D0%B5%D0%BA" title="XII век">XII веке</a><sup class="reference" id="cite_ref-РИИ-99_20-0"><a href="#cite_note-РИИ-99-20">[20]</a></sup>
</p>
<h3><span id=".D0.9A.D1.80.D0.B8.D0.BF.D1.82.D0.B0"></span><span class="mw-headline" id="Крипта">Крипта</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B0%D0%BD%D1%82-%D0%90%D0%BF%D0%BE%D0%BB%D0%BB%D0%B8%D0%BD%D0%B0%D1%80%D0%B5-%D0%B8%D0%BD-%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B5&amp;veaction=edit&amp;section=8" title="Редактировать раздел «Крипта»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B0%D0%BD%D1%82-%D0%90%D0%BF%D0%BE%D0%BB%D0%BB%D0%B8%D0%BD%D0%B0%D1%80%D0%B5-%D0%B8%D0%BD-%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B5&amp;action=edit&amp;section=8" title="Редактировать код раздела «Крипта»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Расположена под главным алтарём базилики. Она ограничена стенами апсиды и двумя боковыми перегородками, в которых находятся лестницы для спуска вниз. <a href="/wiki/%D0%9A%D1%80%D0%B8%D0%BF%D1%82%D0%B0" title="Крипта">Крипта</a>, служившая хранилищем мощей святого Аполлинария была заброшена после того как в <a href="/wiki/856_%D0%B3%D0%BE%D0%B4" title="856 год">856 году</a> они были перенесены в базилику <a href="/wiki/%D0%A1%D0%B0%D0%BD%D1%82-%D0%90%D0%BF%D0%BE%D0%BB%D0%BB%D0%B8%D0%BD%D0%B0%D1%80%D0%B5-%D0%9D%D1%83%D0%BE%D0%B2%D0%BE" title="Сант-Аполлинаре-Нуово">Сант-Аполлинаре-Нуово</a>. В крипте осталась пустая <a href="/wiki/%D0%A0%D0%B0%D0%BA%D0%B0" title="Рака">рака</a> для мощей, которую обновили в <a href="/wiki/1511_%D0%B3%D0%BE%D0%B4" title="1511 год">1511 году</a>. Однако после возвращения мощей в базилику не стали использовать и поместили их в алтаре.<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup> В примыкающих к крипте пластофориях устроены <a class="mw-redirect" href="/wiki/%D0%9A%D0%B0%D0%BF%D0%B5%D0%BB%D0%BB%D0%B0_(%D1%87%D0%B0%D1%81%D0%BE%D0%B2%D0%BD%D1%8F)" title="Капелла (часовня)">капеллы</a>. В одной из них, посвящённой <a class="mw-redirect" href="/wiki/%D0%9C%D0%B5%D1%81%D1%82%D0%BD%D0%BE%D1%87%D1%82%D0%B8%D0%BC%D1%8B%D0%B9_%D1%81%D0%B2%D1%8F%D1%82%D0%BE%D0%B9" title="Местночтимый святой">местночтимому святому</a> Еливкадию сохранилась мраморная сень над престолом, созданная в начале <a href="/wiki/IX_%D0%B2%D0%B5%D0%BA" title="IX век">IX века</a>.
</p>

<p>В боковых нефах базилики находятся 10 средневековых <a href="/wiki/%D0%A1%D0%B0%D1%80%D0%BA%D0%BE%D1%84%D0%B0%D0%B3" title="Саркофаг">саркофагов</a> очень хорошей сохранности<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup>, часть из которых принадлежит равеннским епископам.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup> Из них особо выделяются:
</p>
<ul><li>гробница епископа Феодора (ум. в <a href="/wiki/688_%D0%B3%D0%BE%D0%B4" title="688 год">688 году</a>), богато декорированная виноградными лозами, павлинами, голубками и <a class="mw-redirect" href="/wiki/%D0%9C%D0%BE%D0%BD%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B0_%D0%98%D0%B8%D1%81%D1%83%D1%81%D0%B0_%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%B0" title="Монограмма Иисуса Христа">монограммами Христа</a>;</li>
<li>«<i>саркофаг <a href="/wiki/%D0%94%D0%B2%D0%B5%D0%BD%D0%B0%D0%B4%D1%86%D0%B0%D1%82%D1%8C_%D0%B0%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB%D0%BE%D0%B2" title="Двенадцать апостолов">двенадцати апостолов</a></i>» — украшен композицией с изображением на лицевой панели Иисуса Христа сидящего на троне посреди шести <a class="mw-redirect" href="/wiki/%D0%90%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB" title="Апостол">апостолов</a>. <a href="/wiki/%D0%90%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB_%D0%9F%D0%B0%D0%B2%D0%B5%D0%BB" title="Апостол Павел">Апостолу Павлу</a> Иисус вручает свиток, <a href="/wiki/%D0%90%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB_%D0%9F%D1%91%D1%82%D1%80" title="Апостол Пётр">апостол Пётр</a> держит в руках ключи, а на плече крест, у остальных апостолов в руках мученические венцы. На торцах саркофага помещены фигуры остальных апостолов;</li>
<li>саркофаг архиепископа Грациозия (ум. в <a href="/wiki/788_%D0%B3%D0%BE%D0%B4" title="788 год">788 году</a>), рядом с которым на стене сохранилась надпись середины VI века, рассказывающая о истории постройки базилики и мощах святого Аполлинария:</li></ul>
<p>
</p>
<blockquote class="ts-Начало_цитаты-quote">
<p class="mw-empty-elt">
</p>
<div style="background-color:#eee9d9;">
<p>В этом месте находилась <a href="/wiki/%D0%A0%D0%B0%D0%BA%D0%B0" title="Рака">рака</a> блаженного Аполлинария священника и исповедника со времени [как он] почил, [которая по воле] блаженного Максимиана епископа перенесена и водворена в базилике, которую Юлиан Аргентарий от основания построил и [которая] посвящена этим же мужем праведнейшим в 7-й день майских ид 8-года [после] консульства Василия младшего.
</p>

<div dir="auto" lang="la">In hoc loco stetit arca beati Apollinaris sacerdotis et confessoris a tempore transitus sui usque dia e qua per virum beatum Maximianum episcopum translata est et introducta in basilicam quam Iulianus Argentarius a fundamentis aedificavit et dedicata ab eodem viro beatissimo die VII Idus Maiarum indicione XII octies post consilium Basili iunioris.<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup></div>
</div></div>

<p>Благодаря этим сведениям была установлена точная дата освящения храма — 9 мая <a href="/wiki/549_%D0%B3%D0%BE%D0%B4" title="549 год">549 года</a>;
</p>
<ul><li>маленькая мраморная урна <a href="/wiki/IV_%D0%B2%D0%B5%D0%BA" title="IV век">IV века</a>, не имеющая украшений, но содержащая трогательную надпись: «<i>Лицинии Валерии Фаустине Италике, почивающей в мире, прожившей один год, шесть месяцев, шесть дней, возлюбленной дочери от скорбящих родителей</i>». Урна была найдена под полом базилики в <a href="/wiki/1909_%D0%B3%D0%BE%D0%B4" title="1909 год">1909 году</a> и служит ещё одним подтверждением того, что церковь воздвигнута на месте античного кладбища города <a class="new" href="/w/index.php?title=%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B5&amp;action=edit&amp;redlink=1" title="Классе (страница отсутствует)">Классе</a>;</li>
<li>безымянный саркофаг с изображением креста, водружённого на арочную конструкцию, напоминающую изображение палат <a href="/wiki/%D0%A2%D0%B5%D0%BE%D0%B4%D0%BE%D1%80%D0%B8%D1%85_%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B8%D0%B9" title="Теодорих Великий">Теодориха</a> в базилике <a href="/wiki/%D0%A1%D0%B0%D0%BD%D1%82-%D0%90%D0%BF%D0%BE%D0%BB%D0%BB%D0%B8%D0%BD%D0%B0%D1%80%D0%B5-%D0%9D%D1%83%D0%BE%D0%B2%D0%BE" title="Сант-Аполлинаре-Нуово">Сант-Аполлинаре-Нуово</a>;</li>
<li>саркофаг, на передней стороне которого можно видеть латинский <a href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82" title="Крест">крест</a> со свисающими с его перекладины греческими буквами α и ω (в соответствии с <a href="/wiki/%D0%9E%D1%82%D0%BA%D1%80%D0%BE%D0%B2%D0%B5%D0%BD%D0%B8%D0%B5_%D0%98%D0%BE%D0%B0%D0%BD%D0%BD%D0%B0_%D0%91%D0%BE%D0%B3%D0%BE%D1%81%D0%BB%D0%BE%D0%B2%D0%B0" title="Откровение Иоанна Богослова">Отк.</a> <a class="extiw" href="https://ru.wikisource.org/wiki/%D0%9E%D1%82%D0%BA%D1%80%D0%BE%D0%B2%D0%B5%D0%BD%D0%B8%D0%B5_%D1%81%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%98%D0%BE%D0%B0%D0%BD%D0%BD%D0%B0_%D0%91%D0%BE%D0%B3%D0%BE%D1%81%D0%BB%D0%BE%D0%B2%D0%B0#1" title="s:Откровение святого Иоанна Богослова">1:8</a> эти буквы указывают на Христа), а на крышке — крест на горе, с которой стекают четыре реки (четыре реки, бравшие начало в <a href="/wiki/%D0%AD%D0%B4%D0%B5%D0%BC" title="Эдем">Эдеме</a> согласно <a href="/wiki/%D0%9A%D0%BD%D0%B8%D0%B3%D0%B0_%D0%91%D1%8B%D1%82%D0%B8%D1%8F" title="Книга Бытия">Быт.</a> <a class="extiw" href="https://ru.wikisource.org/wiki/%D0%91%D1%8B%D1%82%D0%B8%D0%B5#2" title="s:Бытие">2:10</a>)</li>
<li>саркофаг епископа Феликса (ум. в <a href="/wiki/723_%D0%B3%D0%BE%D0%B4" title="723 год">723 году</a>), переделанный из языческой гробницы, его скудное украшение свидетельствует об упадке в этот период равеннского искусства;</li>
<li>ещё три безымянных саркофага с простейшими резными украшениями.</li></ul>
