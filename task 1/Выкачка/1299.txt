<p><b>Генрих I</b> Смелый<sup class="reference" id="cite_ref-vaderlandsche_1-0"><a href="#cite_note-vaderlandsche-1">[1]</a></sup> (возможно <a href="/wiki/%D0%9B%D1%91%D0%B2%D0%B5%D0%BD" title="Лёвен">Лёвен</a>, <a href="/wiki/1165_%D0%B3%D0%BE%D0%B4" title="1165 год">1165 год</a> — <a href="/wiki/%D0%9A%D1%91%D0%BB%D1%8C%D0%BD" title="Кёльн">Кёльн</a>, <a href="/wiki/5_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="5 сентября">5 сентября</a> <a href="/wiki/1235_%D0%B3%D0%BE%D0%B4" title="1235 год">1235 года</a>) — граф <a href="/wiki/%D0%91%D1%80%D1%8E%D1%81%D1%81%D0%B5%D0%BB%D1%8C" title="Брюссель">Брюссельский</a> (<a href="/wiki/1179_%D0%B3%D0%BE%D0%B4" title="1179 год">1179</a>), первый <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3%D0%BE%D0%B2_%D0%91%D1%80%D0%B0%D0%B1%D0%B0%D0%BD%D1%82%D0%B0" title="Список герцогов Брабанта">герцог Брабанта</a> (с <a href="/wiki/1183_%D0%B3%D0%BE%D0%B4" title="1183 год">1183 года</a>) и <a class="mw-redirect" href="/wiki/%D0%93%D0%B5%D1%80%D1%86%D0%BE%D0%B3_%D0%9D%D0%B8%D0%B6%D0%BD%D0%B5%D0%B9_%D0%9B%D0%BE%D1%82%D0%B0%D1%80%D0%B8%D0%BD%D0%B3%D0%B8%D0%B8" title="Герцог Нижней Лотарингии">герцог Нижней Лотарингии</a> (формально, с <a href="/wiki/1190_%D0%B3%D0%BE%D0%B4" title="1190 год">1190 года</a>). Граф <a href="/wiki/%D0%9B%D1%91%D0%B2%D0%B5%D0%BD" title="Лёвен">Лувена</a>, <a class="mw-redirect" href="/wiki/%D0%9C%D0%B0%D1%80%D0%BA%D0%B3%D1%80%D0%B0%D1%84_%D0%90%D0%BD%D1%82%D0%B2%D0%B5%D1%80%D0%BF%D0%B5%D0%BD%D0%B0" title="Маркграф Антверпена">маркграф Антверпена</a>. Участник <a href="/wiki/%D0%A2%D1%80%D0%B5%D1%82%D0%B8%D0%B9_%D0%BA%D1%80%D0%B5%D1%81%D1%82%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D0%BE%D1%85%D0%BE%D0%B4" title="Третий крестовый поход">Третьего крестового похода</a>. На протяжении большей части правления вёл политику балансирования между соперничающими монархическими династиями <a href="/wiki/%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%A0%D0%B8%D0%BC%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F" title="Священная Римская империя">Священной Римской империи</a>: <a href="/wiki/%D0%92%D0%B5%D0%BB%D1%8C%D1%84%D1%8B" title="Вельфы">Вельфами</a> и <a href="/wiki/%D0%93%D0%BE%D0%B3%D0%B5%D0%BD%D1%88%D1%82%D0%B0%D1%83%D1%84%D0%B5%D0%BD%D1%8B" title="Гогенштауфены">Гогенштауфенами</a>. Неоднократный участник военных конфликтов, в том числе и битв при Степпе и <a href="/wiki/%D0%91%D0%B8%D1%82%D0%B2%D0%B0_%D0%BF%D1%80%D0%B8_%D0%91%D1%83%D0%B2%D0%B8%D0%BD%D0%B5" title="Битва при Бувине">Бувине</a>.
</p>

<h2><span id=".D0.91.D0.B8.D0.BE.D0.B3.D1.80.D0.B0.D1.84.D0.B8.D1.8F"></span><span class="mw-headline" id="Биография">Биография</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%93%D0%B5%D0%BD%D1%80%D0%B8%D1%85_I_(%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3_%D0%91%D1%80%D0%B0%D0%B1%D0%B0%D0%BD%D1%82%D0%B0)&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Биография»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%93%D0%B5%D0%BD%D1%80%D0%B8%D1%85_I_(%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3_%D0%91%D1%80%D0%B0%D0%B1%D0%B0%D0%BD%D1%82%D0%B0)&amp;action=edit&amp;section=1" title="Редактировать код раздела «Биография»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Отец Генриха <a class="mw-redirect" href="/wiki/%D0%93%D0%BE%D1%82%D1%84%D1%80%D0%B8%D0%B4_III_(%D0%B3%D1%80%D0%B0%D1%84_%D0%9B%D1%83%D0%B2%D0%B5%D0%BD%D1%81%D0%BA%D0%B8%D0%B9)" title="Готфрид III (граф Лувенский)">Готфрид III</a> рано стал привлекать сына к политике: уже в <a href="/wiki/1179_%D0%B3%D0%BE%D0%B4" title="1179 год">1179 году</a> Генрих стал известен как <i>рыцарь, но ещё не граф</i> (<a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Латинский язык">лат.</a> <span lang="la" style="font-style:italic;">miles nec non comes</span>).
</p><p>В том же году он женился на <a href="/wiki/%D0%9C%D0%B0%D1%82%D0%B8%D0%BB%D1%8C%D0%B4%D0%B0_%D0%91%D1%83%D0%BB%D0%BE%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_(%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3%D0%B8%D0%BD%D1%8F_%D0%91%D1%80%D0%B0%D0%B1%D0%B0%D0%BD%D1%82%D0%B0)" title="Матильда Булонская (герцогиня Брабанта)">Матильде Булонской</a>, племяннице <a class="mw-redirect" href="/wiki/%D0%A4%D0%B8%D0%BB%D0%B8%D0%BF%D0%BF_I_%D0%AD%D0%BB%D1%8C%D0%B7%D0%B0%D1%81%D1%81%D0%BA%D0%B8%D0%B9" title="Филипп I Эльзасский">Филиппа Эльзасского</a>, дочери его брата <a href="/wiki/%D0%9C%D0%B0%D1%82%D1%8C%D0%B5_%D0%AD%D0%BB%D1%8C%D0%B7%D0%B0%D1%81%D1%81%D0%BA%D0%B8%D0%B9" title="Матье Эльзасский">Матье</a>, графа <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D0%B3%D1%80%D0%B0%D1%84%D0%BE%D0%B2_%D0%91%D1%83%D0%BB%D0%BE%D0%BD%D1%81%D0%BA%D0%B8%D1%85" title="Список графов Булонских">Булонского</a> и <a href="/wiki/%D0%9C%D0%B0%D1%80%D0%B8%D1%8F_%D0%91%D1%83%D0%BB%D0%BE%D0%BD%D1%81%D0%BA%D0%B0%D1%8F" title="Мария Булонская">Марии</a>, дочери английского короля <a href="/wiki/%D0%A1%D1%82%D0%B5%D1%84%D0%B0%D0%BD_(%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D1%8C_%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B8)" title="Стефан (король Англии)">Стефана</a>. Согласно брачному контракту, Генрих фактически отказывался от управления большей частью Брабанта в пользу Филиппа Эльзасского<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>.
</p><p>Между <a href="/wiki/1182_%D0%B3%D0%BE%D0%B4" title="1182 год">1182</a> и <a href="/wiki/1184_%D0%B3%D0%BE%D0%B4" title="1184 год">1184 годами</a> Генрих участвовал в защите <a href="/wiki/%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC" title="Иерусалим">Иерусалима</a><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>, за что <a href="/wiki/%D0%A4%D1%80%D0%B8%D0%B4%D1%80%D0%B8%D1%85_I_%D0%91%D0%B0%D1%80%D0%B1%D0%B0%D1%80%D0%BE%D1%81%D1%81%D0%B0" title="Фридрих I Барбаросса">император</a> возвёл <a href="/wiki/%D0%91%D1%80%D0%B0%D0%B1%D0%B0%D0%BD%D1%82_(%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3%D1%81%D1%82%D0%B2%D0%BE)" title="Брабант (герцогство)">Брабант</a> в статус герцогства и сделал Генриха первым герцогом Брабанта.
</p>
<h3><span id=".D0.9A.D0.BE.D0.BD.D1.84.D0.BB.D0.B8.D0.BA.D1.82_.D1.81_.D0.B3.D1.80.D0.B0.D1.84.D0.BE.D0.BC_.D0.9B.D0.BE.D0.BE.D0.BD.D0.B0_.D0.B8_.D0.B3.D0.B5.D1.80.D1.86.D0.BE.D0.B3.D0.BE.D0.BC_.D0.9B.D0.B8.D0.BC.D0.B1.D1.83.D1.80.D0.B3.D0.B0_.281189.E2.80.941191.29"></span><span class="mw-headline" id="Конфликт_с_графом_Лоона_и_герцогом_Лимбурга_(1189—1191)">Конфликт с графом Лоона и герцогом Лимбурга (1189—1191)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%93%D0%B5%D0%BD%D1%80%D0%B8%D1%85_I_(%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3_%D0%91%D1%80%D0%B0%D0%B1%D0%B0%D0%BD%D1%82%D0%B0)&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Конфликт с графом Лоона и герцогом Лимбурга (1189—1191)»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%93%D0%B5%D0%BD%D1%80%D0%B8%D1%85_I_(%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3_%D0%91%D1%80%D0%B0%D0%B1%D0%B0%D0%BD%D1%82%D0%B0)&amp;action=edit&amp;section=2" title="Редактировать код раздела «Конфликт с графом Лоона и герцогом Лимбурга (1189—1191)»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>В <a href="/wiki/1189_%D0%B3%D0%BE%D0%B4" title="1189 год">1189 году</a> Генрих начал военные действия против собственного дяди, <a class="mw-redirect" href="/wiki/%D0%93%D0%B5%D0%BD%D1%80%D0%B8%D1%85_III_%D0%9B%D0%B8%D0%BC%D0%B1%D1%83%D1%80%D0%B3%D1%81%D0%BA%D0%B8%D0%B9" title="Генрих III Лимбургский">герцога Лимбурга</a><sup class="reference" id="cite_ref-vaderlandsche_1-1"><a href="#cite_note-vaderlandsche-1">[1]</a></sup>. Поводом для конфликта послужила продажа графу <a href="/wiki/%D0%9B%D0%BE%D0%BE%D0%BD" title="Лоон">Лоона</a> попечительства над <a href="/wiki/%D0%A1%D0%B8%D0%BD%D1%82-%D0%A2%D1%80%D1%91%D0%B9%D0%B4%D0%B5%D0%BD" title="Синт-Трёйден">Синт-Трёйденом</a>, ранее входившим в приданое матери Генриха Маргариты Лимбургской. Во главе войска из 700 всадников и 60 тысяч пехотинцев Генрих вторгся в Лоон и осадил Синт-Трёйден.
</p><p>В ответ по просьбе графа Лоона <a href="/wiki/%D0%91%D0%BE%D0%B4%D1%83%D1%8D%D0%BD_V_(%D0%B3%D1%80%D0%B0%D1%84_%D0%AD%D0%BD%D0%BE)" title="Бодуэн V (граф Эно)">граф Эно</a> ввёл войска в Брабант и Генриху пришлось срочно снять осаду. Тем не менее, с точки зрения политической кампания Генриха была весьма успешна: по заключённому при посредничестве <a class="mw-redirect" href="/wiki/%D0%A4%D0%B8%D0%BB%D0%B8%D0%BF%D0%BF_I_%D0%AD%D0%BB%D1%8C%D0%B7%D0%B0%D1%81%D1%81%D0%BA%D0%B8%D0%B9" title="Филипп I Эльзасский">графа Фландрии</a> мирному договору граф Лоона обязался выплатить 800 марок серебром и покуда эта сумма не уплачена полностью, признать совместное с Генрихом владение Синт-Трёйденом<sup class="reference" id="cite_ref-vaderlandsche_1-2"><a href="#cite_note-vaderlandsche-1">[1]</a></sup>. К <a href="/wiki/1191_%D0%B3%D0%BE%D0%B4" title="1191 год">1191 году</a> Генрих также заключил мир с герцогом Лимбурга, по которому к Брабанту отходили <a href="/wiki/%D0%90%D0%BB%D0%BB%D0%BE%D0%B4" title="Аллод">аллоды</a> в <a href="/wiki/%D0%90%D1%80%D0%BB%D0%BE%D0%BD" title="Арлон">Арлоне</a> и <a href="/wiki/%D0%A5%D0%B5%D1%80%D1%86%D0%BE%D0%B3%D0%B5%D0%BD%D1%80%D0%B0%D1%82" title="Херцогенрат">Херцогенрате</a><sup class="reference" id="cite_ref-vaderlandsche_1-3"><a href="#cite_note-vaderlandsche-1">[1]</a></sup>.
</p>
<h3><span id=".D0.92.D1.8B.D0.B1.D0.BE.D1.80.D1.8B_.D0.B5.D0.BF.D0.B8.D1.81.D0.BA.D0.BE.D0.BF.D0.B0_.D0.9B.D1.8C.D0.B5.D0.B6.D1.81.D0.BA.D0.BE.D0.B3.D0.BE_.281191.E2.80.941193.29"></span><span class="mw-headline" id="Выборы_епископа_Льежского_(1191—1193)">Выборы епископа Льежского (1191—1193)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%93%D0%B5%D0%BD%D1%80%D0%B8%D1%85_I_(%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3_%D0%91%D1%80%D0%B0%D0%B1%D0%B0%D0%BD%D1%82%D0%B0)&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Выборы епископа Льежского (1191—1193)»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%93%D0%B5%D0%BD%D1%80%D0%B8%D1%85_I_(%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3_%D0%91%D1%80%D0%B0%D0%B1%D0%B0%D0%BD%D1%82%D0%B0)&amp;action=edit&amp;section=3" title="Редактировать код раздела «Выборы епископа Льежского (1191—1193)»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>В борьбе <a href="/wiki/%D0%92%D0%B5%D0%BB%D1%8C%D1%84%D1%8B" title="Вельфы">Вельфов</a> с <a href="/wiki/%D0%93%D0%BE%D0%B3%D0%B5%D0%BD%D1%88%D1%82%D0%B0%D1%83%D1%84%D0%B5%D0%BD%D1%8B" title="Гогенштауфены">Гогенштауфенами</a> Генрих периодически менял свои политические пристрастия. Первый существенный конфликт с Гогенштауфенами возник в связи с выборами <a href="/wiki/%D0%9B%D1%8C%D0%B5%D0%B6%D1%81%D0%BA%D0%BE%D0%B5_%D0%B5%D0%BF%D0%B8%D1%81%D0%BA%D0%BE%D0%BF%D1%81%D1%82%D0%B2%D0%BE" title="Льежское епископство">епископа Льежского</a> после смерти епископа <a href="/wiki/5_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="5 августа">5 августа</a> <a href="/wiki/1191_%D0%B3%D0%BE%D0%B4" title="1191 год">1191 года</a>.
</p><p><a href="/wiki/8_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="8 сентября">8 сентября</a> льежский капитул должен был избрать епископа. Избрание должно было быть утверждено как <a href="/wiki/%D0%A6%D0%B5%D0%BB%D0%B5%D1%81%D1%82%D0%B8%D0%BD_III" title="Целестин III">папой</a>, так и императором, так как льежский епископ являлся главой княжества, входившего в состав империи. Одним кандидатом, поддержанным <a href="/wiki/%D0%91%D0%BE%D0%B4%D1%83%D1%8D%D0%BD_V_(%D0%B3%D1%80%D0%B0%D1%84_%D0%AD%D0%BD%D0%BE)" title="Бодуэн V (граф Эно)">графом Эно</a>, был двоюродный брат графа Альберт Ретельский, сын <a class="new" href="/w/index.php?title=%D0%91%D0%B5%D0%B0%D1%82%D1%80%D0%B8%D1%81%D0%B0_%D0%9D%D0%B0%D0%BC%D1%8E%D1%80%D1%81%D0%BA%D0%B0%D1%8F&amp;action=edit&amp;redlink=1" title="Беатриса Намюрская (страница отсутствует)">Беатрисы Намюрской</a>, сестры его матери Аделаиды.
</p><p>Шансы Альберта Ретельского казались особенно высокими<sup class="reference" id="cite_ref-vaderlandsche_1-4"><a href="#cite_note-vaderlandsche-1">[1]</a></sup>, так как он приходился дядей <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%86%D0%B8%D1%8F_(%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D0%B5%D0%B2%D0%B0_%D0%A1%D0%B8%D1%86%D0%B8%D0%BB%D0%B8%D0%B8)" title="Констанция (королева Сицилии)">Констанции</a>, супруге <a href="/wiki/%D0%93%D0%B5%D0%BD%D1%80%D0%B8%D1%85_VI_(%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%BE%D1%80_%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D0%BE%D0%B9_%D0%A0%D0%B8%D0%BC%D1%81%D0%BA%D0%BE%D0%B9_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D0%B8)" title="Генрих VI (император Священной Римской империи)">императора Генриха VI</a> (братом её матери <a href="/wiki/%D0%91%D0%B5%D0%B0%D1%82%D1%80%D0%B8%D1%81%D0%B0_%D0%A0%D0%B5%D1%82%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B0%D1%8F" title="Беатриса Ретельская">Беатрисы</a>). Другим кандидатом был брат Генриха <a class="mw-redirect" href="/wiki/%D0%90%D0%BB%D1%8C%D0%B1%D0%B5%D1%80%D1%82_I_%D0%9B%D1%83%D0%B2%D0%B5%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_(%D0%B5%D0%BF%D0%B8%D1%81%D0%BA%D0%BE%D0%BF_%D0%9B%D1%8C%D0%B5%D0%B6%D0%B0)" title="Альберт I Лувенский (епископ Льежа)">Альберт</a>. Большинством голосов предпочтение было отдано последнему, однако часть членов капитула предпочла и Альберта Ретельского<sup class="reference" id="cite_ref-vaderlandsche_1-5"><a href="#cite_note-vaderlandsche-1">[1]</a></sup><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>.
В начале <a href="/wiki/1192_%D0%B3%D0%BE%D0%B4" title="1192 год">1192 года</a> император Генрих VI объявил, что ввиду разногласий, имевших место в ходе выборов в сентябре, право назначения епископа перешло к нему по праву сюзерена, и что его выбор пал на Лотаря Хо(х)стадского, льежского канонника, как и оба Альберта. Решение императора противоречило условиям <a href="/wiki/%D0%92%D0%BE%D1%80%D0%BC%D1%81%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%BE%D0%BD%D0%BA%D0%BE%D1%80%D0%B4%D0%B0%D1%82" title="Вормсский конкордат">вормсского конкордата</a><sup class="reference" id="cite_ref-vaderlandsche_1-6"><a href="#cite_note-vaderlandsche-1">[1]</a></sup>. В ответ на решение императора Альберт Брабантский отправился в <a href="/wiki/%D0%A0%D0%B8%D0%BC" title="Рим">Рим</a>, где <a class="mw-redirect" href="/wiki/%D0%A0%D0%B8%D0%BC%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B0%D0%BF%D0%B0" title="Римский папа">папа</a> <a href="/wiki/%D0%A6%D0%B5%D0%BB%D0%B5%D1%81%D1%82%D0%B8%D0%BD_III" title="Целестин III">Целестин III</a> <a href="/wiki/%D0%A0%D1%83%D0%BA%D0%BE%D0%BF%D0%BE%D0%BB%D0%BE%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5" title="Рукоположение">рукоположил</a> его в <a href="/wiki/%D0%94%D0%B8%D0%B0%D0%BA%D0%BE%D0%BD" title="Диакон">диаконы</a> и послал его к <a href="/wiki/%D0%90%D1%80%D1%85%D0%B8%D0%B5%D0%BF%D0%B8%D1%81%D0%BA%D0%BE%D0%BF_%D0%A0%D0%B5%D0%B9%D0%BC%D1%81%D0%B0" title="Архиепископ Реймса">архиепископу Реймса</a>, который <a href="/wiki/19_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="19 сентября">19 сентября</a> <a href="/wiki/1192_%D0%B3%D0%BE%D0%B4" title="1192 год">1192 года</a> рукоположил Альберта в <a href="/wiki/%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D0%B8%D0%BA" title="Священник">священники</a>, а на следующий день в епископы. Когда <a href="/wiki/24_%D0%BD%D0%BE%D1%8F%D0%B1%D1%80%D1%8F" title="24 ноября">24 ноября</a> того же года Альберт отправился в <a href="/wiki/%D0%9B%D1%8C%D0%B5%D0%B6" title="Льеж">Льеж</a>, на него напали последователи Генриха VI, которые нанесли ему смертельную рану в сердце. В <a href="/wiki/1613_%D0%B3%D0%BE%D0%B4" title="1613 год">1613 году</a> епископ Льежа Альберт был причислен папой <a href="/wiki/%D0%9F%D0%B0%D0%B2%D0%B5%D0%BB_V" title="Павел V">Павлом V</a> к лику святых.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>
</p><p>В ответ на убийство брата в начале <a href="/wiki/1193_%D0%B3%D0%BE%D0%B4" title="1193 год">1193 года</a> герцог Генрих разрушил графство Хохстаде, принадлежавшее брату назначенного Генрихом VI епископа. Более того, герцогу удалось собрать целый ряд правителей низовий <a href="/wiki/%D0%A0%D0%B5%D0%B9%D0%BD" title="Рейн">Рейна</a> и тем самым вынудить императора пойти на перемирие (не позднее <a href="/wiki/13_%D0%BC%D0%B0%D1%8F" title="13 мая">13 мая</a> <a href="/wiki/1193_%D0%B3%D0%BE%D0%B4" title="1193 год">1193 года</a>)<sup class="reference" id="cite_ref-vaderlandsche_1-7"><a href="#cite_note-vaderlandsche-1">[1]</a></sup>.
</p>
<h3><span id=".D0.A3.D1.87.D0.B0.D1.81.D1.82.D0.B8.D0.B5_.D0.B2_.D0.BA.D1.80.D0.B5.D1.81.D1.82.D0.BE.D0.B2.D0.BE.D0.BC_.D0.BF.D0.BE.D1.85.D0.BE.D0.B4.D0.B5_.281197.29"></span><span class="mw-headline" id="Участие_в_крестовом_походе_(1197)">Участие в крестовом походе (1197)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%93%D0%B5%D0%BD%D1%80%D0%B8%D1%85_I_(%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3_%D0%91%D1%80%D0%B0%D0%B1%D0%B0%D0%BD%D1%82%D0%B0)&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Участие в крестовом походе (1197)»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%93%D0%B5%D0%BD%D1%80%D0%B8%D1%85_I_(%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3_%D0%91%D1%80%D0%B0%D0%B1%D0%B0%D0%BD%D1%82%D0%B0)&amp;action=edit&amp;section=4" title="Редактировать код раздела «Участие в крестовом походе (1197)»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure class="mw-default-size mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:The-Taking-of-Beirut.jpg"><img class="mw-file-element" data-file-height="451" data-file-width="900" decoding="async" height="110" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/8b/The-Taking-of-Beirut.jpg/220px-The-Taking-of-Beirut.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/8b/The-Taking-of-Beirut.jpg/330px-The-Taking-of-Beirut.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/8b/The-Taking-of-Beirut.jpg/440px-The-Taking-of-Beirut.jpg 2x" width="220"/></a><figcaption>Александр-Жан-Батист Эс (<a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Французский язык">фр.</a> <span lang="fr" style="font-style:italic;">Alexandre-Jean-Baptist Hesse</span>). «Взятие Бейрута войсками <a href="/wiki/%D0%90%D0%BC%D0%BE%D1%80%D0%B8_II_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D1%81%D0%BA%D0%B8%D0%B9" title="Амори II Иерусалимский">Амори II Иерусалимского</a> в <a href="/wiki/1197_%D0%B3%D0%BE%D0%B4" title="1197 год">1197 году</a>» (<a href="/wiki/1842_%D0%B3%D0%BE%D0%B4" title="1842 год">1842</a>).</figcaption></figure>
<p><a href="/wiki/1194_%D0%B3%D0%BE%D0%B4" title="1194 год">1194 год</a> ознаменовался рядом столкновений с <a href="/wiki/%D0%91%D0%B0%D0%BB%D0%B4%D1%83%D0%B8%D0%BD_I_%D0%A4%D0%BB%D0%B0%D0%BD%D0%B4%D1%80%D1%81%D0%BA%D0%B8%D0%B9" title="Балдуин I Фландрский">графом Фландрии</a>, начало которых восходит ещё к <a href="/wiki/1182_%D0%B3%D0%BE%D0%B4" title="1182 год">1182 году</a>, и закончившихся мирным соглашением в том же <a href="/wiki/1194_%D0%B3%D0%BE%D0%B4" title="1194 год">1194 году</a>. Гарантами соглашения выступили дворянство и города: <a href="/wiki/%D0%90%D0%BD%D1%82%D0%B2%D0%B5%D1%80%D0%BF%D0%B5%D0%BD" title="Антверпен">Антверпен</a>, <a href="/wiki/%D0%91%D1%80%D1%8E%D1%81%D1%81%D0%B5%D0%BB%D1%8C" title="Брюссель">Брюссель</a>, <a href="/wiki/%D0%96%D0%B0%D0%BC%D0%B1%D0%BB%D1%83" title="Жамблу">Жамблу</a>, <a href="/wiki/%D0%96%D0%BE%D0%B4%D0%BE%D0%BD%D1%8C" title="Жодонь">Жодонь</a>, <a class="new" href="/w/index.php?title=%D0%97%D0%B0%D1%83%D1%82%D0%BB%D0%B5%D1%83&amp;action=edit&amp;redlink=1" title="Заутлеу (страница отсутствует)">Заутлеу</a>, <a href="/wiki/%D0%9B%D1%91%D0%B2%D0%B5%D0%BD" title="Лёвен">Лёвен</a>, <a href="/wiki/%D0%9B%D0%B8%D1%80_(%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D1%8F)" title="Лир (Бельгия)">Лир</a>, <a href="/wiki/%D0%9D%D0%B8%D0%B2%D0%B5%D0%BB%D1%8C" title="Нивель">Нивель</a> и <a href="/wiki/%D0%A2%D0%B8%D0%BD%D0%B5%D0%BD" title="Тинен">Тинен</a><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>. Это соглашение позволило Генриху начать подготовку к участию в крестовом походе под предводительством Генриха VI, к каковому он и присоединился в <a href="/wiki/1197_%D0%B3%D0%BE%D0%B4" title="1197 год">1197 году</a><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>. Император в Святую землю так и не прибыл, скончавшись от лихорадки <a href="/wiki/28_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="28 сентября">28 сентября</a> <a href="/wiki/1197_%D0%B3%D0%BE%D0%B4" title="1197 год">1197 года</a>. Герцог Брабанта прибыл в Святую землю <a href="/wiki/22_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="22 сентября">22 сентября</a><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>, а уже в октябре того же года участвовал во взятии <a class="mw-redirect" href="/wiki/%D0%A1%D0%B8%D0%B4%D0%BE%D0%BD" title="Сидон">Сидона</a> и <a href="/wiki/%D0%91%D0%B5%D0%B9%D1%80%D1%83%D1%82" title="Бейрут">Бейрута</a><sup class="reference" id="cite_ref-hosten_9-0"><a href="#cite_note-hosten-9">[9]</a></sup> и вместе с <a class="mw-redirect" href="/wiki/%D0%90%D1%80%D1%85%D0%B8%D0%B5%D0%BF%D0%B8%D1%81%D0%BA%D0%BE%D0%BF_%D0%9C%D0%B0%D0%B9%D0%BD%D1%86%D0%B0" title="Архиепископ Майнца">архиепископом Майнца</a> <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%BD%D1%80%D0%B0%D0%B4_I_%D1%84%D0%BE%D0%BD_%D0%92%D0%B8%D1%82%D1%82%D0%B5%D0%BB%D1%8C%D1%81%D0%B1%D0%B0%D1%85_(%D0%B0%D1%80%D1%85%D0%B8%D0%B5%D0%BF%D0%B8%D1%81%D0%BA%D0%BE%D0%BF_%D0%9C%D0%B0%D0%B9%D0%BD%D1%86%D0%B0)" title="Конрад I фон Виттельсбах (архиепископ Майнца)">Конрадом Виттельсбахским</a> осадил стратегически важную крепость <a class="mw-redirect" href="/wiki/%D0%97%D0%B0%D0%BC%D0%BE%D0%BA_%D0%A2%D0%BE%D1%80%D0%BE%D0%BD" title="Замок Торон">Торон</a> (совр. Тибнин). Несмотря на то, что осада была успешной и Торон был готов капитулировать, осада была снята: войско достигли слухи о смерти императора и, учитывая, что <a href="/wiki/%D0%A4%D1%80%D0%B8%D0%B4%D1%80%D0%B8%D1%85_II_(%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%BE%D1%80_%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D0%BE%D0%B9_%D0%A0%D0%B8%D0%BC%D1%81%D0%BA%D0%BE%D0%B9_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D0%B8)" title="Фридрих II (император Священной Римской империи)">единственному сыну Генриха VI</a> было три года и предвидя борьбу за императорский престол Конрад Виттельсбахский и его бароны поспешили в Германию<sup class="reference" id="cite_ref-hosten_9-1"><a href="#cite_note-hosten-9">[9]</a></sup>.
</p>
<h3><span id=".D0.9C.D0.B5.D0.B6.D0.B4.D1.83_.D0.92.D0.B5.D0.BB.D1.8C.D1.84.D0.B0.D0.BC.D0.B8_.D0.B8_.D0.93.D0.BE.D0.B3.D0.B5.D0.BD.D1.88.D1.82.D0.B0.D1.83.D1.84.D0.B5.D0.BD.D0.B0.D0.BC.D0.B8_.281197.E2.80.941218.29"></span><span class="mw-headline" id="Между_Вельфами_и_Гогенштауфенами_(1197—1218)">Между Вельфами и Гогенштауфенами (1197—1218)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%93%D0%B5%D0%BD%D1%80%D0%B8%D1%85_I_(%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3_%D0%91%D1%80%D0%B0%D0%B1%D0%B0%D0%BD%D1%82%D0%B0)&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Между Вельфами и Гогенштауфенами (1197—1218)»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%93%D0%B5%D0%BD%D1%80%D0%B8%D1%85_I_(%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3_%D0%91%D1%80%D0%B0%D0%B1%D0%B0%D0%BD%D1%82%D0%B0)&amp;action=edit&amp;section=5" title="Редактировать код раздела «Между Вельфами и Гогенштауфенами (1197—1218)»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>

Генрих пожаловал Хертогенбосу статус города (не позднее 1196).</figcaption></figure>
<p>В своей политике Генрих стремился к расширению владений между <a href="/wiki/%D0%A8%D0%B5%D0%BB%D1%8C%D0%B4%D0%B0" title="Шельда">Шельдой</a> и <a href="/wiki/%D0%A0%D0%B5%D0%B9%D0%BD" title="Рейн">Рейном</a> и к контролю над торговым путём из <a href="/wiki/%D0%91%D1%80%D1%8E%D0%B3%D0%B3%D0%B5" title="Брюгге">Брюгге</a> в <a href="/wiki/%D0%9A%D1%91%D0%BB%D1%8C%D0%BD" title="Кёльн">Кёльн</a>, что заметно по значительному<sup class="reference" id="cite_ref-janssen_10-0"><a href="#cite_note-janssen-10">[10]</a></sup> числу населённых пунктов, получивших при Генрих статус города. Среди прочих, подобный статус был дан<sup class="reference" id="cite_ref-janssen_10-1"><a href="#cite_note-janssen-10">[10]</a></sup> <a href="/wiki/%D0%A5%D0%B5%D1%80%D1%82%D0%BE%D0%B3%D0%B5%D0%BD%D0%B1%D0%BE%D1%81" title="Хертогенбос">Хертогенбосу</a> (до <a href="/wiki/1196_%D0%B3%D0%BE%D0%B4" title="1196 год">1196 года</a>), <a class="mw-redirect" href="/wiki/%D0%9B%D0%B0%D0%BD%D0%B4%D0%B5%D0%BD" title="Ланден">Ландену</a> (около <a href="/wiki/1210_%D0%B3%D0%BE%D0%B4" title="1210 год">1210 года</a>), <a href="/wiki/%D0%96%D0%BE%D0%B4%D0%BE%D0%BD%D1%8C" title="Жодонь">Жодони</a> (<a href="/wiki/1211_%D0%B3%D0%BE%D0%B4" title="1211 год">1211</a>). Это приводило Генриха к регулярным противостояниям с его <a href="/wiki/%D0%A1%D1%8E%D0%B7%D0%B5%D1%80%D0%B5%D0%BD" title="Сюзерен">сюзереном</a><sup class="reference" id="cite_ref-pirenn_11-0"><a href="#cite_note-pirenn-11">[11]</a></sup>: так, после смерти Генриха VI он вместе с супругой Матильдой поддержал кандидата Вельфов <a href="/wiki/%D0%9E%D1%82%D1%82%D0%BE%D0%BD_IV_(%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%BE%D1%80_%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D0%BE%D0%B9_%D0%A0%D0%B8%D0%BC%D1%81%D0%BA%D0%BE%D0%B9_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D0%B8)" title="Оттон IV (император Священной Римской империи)">Оттона Брауншвейгского</a><sup class="reference" id="cite_ref-vaderlandsche_1-8"><a href="#cite_note-vaderlandsche-1">[1]</a></sup>. Более того, в <a href="/wiki/1202_%D0%B3%D0%BE%D0%B4" title="1202 год">1202 году</a> ему удалось склонить на сторону Вельфов своего противника, <a href="/wiki/%D0%9E%D1%82%D1%82%D0%BE%D0%BD_I_(%D0%B3%D1%80%D0%B0%D1%84_%D0%93%D0%B5%D0%BB%D0%B4%D0%B5%D1%80%D0%BD%D0%B0)" title="Оттон I (граф Гелдерна)">графа Гельдерна</a><sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>. Тем не менее, после победы <a href="/wiki/%D0%A4%D0%B8%D0%BB%D0%B8%D0%BF%D0%BF_II_%D0%90%D0%B2%D0%B3%D1%83%D1%81%D1%82" title="Филипп II Август">Филиппа-Августа</a> над <a href="/wiki/%D0%98%D0%BE%D0%B0%D0%BD%D0%BD_%D0%91%D0%B5%D0%B7%D0%B7%D0%B5%D0%BC%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9" title="Иоанн Безземельный">Иоанном Безземельным</a> и после победы в Германии кандидата Гогенштауфенов <a href="/wiki/%D0%A4%D0%B8%D0%BB%D0%B8%D0%BF%D0%BF_%D0%A8%D0%B2%D0%B0%D0%B1%D1%81%D0%BA%D0%B8%D0%B9" title="Филипп Швабский">Филиппа Швабского</a> над Оттоном Брауншвейгским, которого поддерживала Англия, Генрих изменил политические пристрастия и сблизился с французским королём и Гогенштауфенами.
</p>
<figure class="mw-default-size" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Henry_I_duc_of_Brabant_coin.JPG"><img class="mw-file-element" data-file-height="180" data-file-width="282" decoding="async" height="140" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/bc/Henry_I_duc_of_Brabant_coin.JPG/220px-Henry_I_duc_of_Brabant_coin.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/b/bc/Henry_I_duc_of_Brabant_coin.JPG 1.5x" width="220"/></a><figcaption>Серебряная монета из Маастрихта. На <a href="/wiki/%D0%90%D0%B2%D0%B5%D1%80%D1%81" title="Аверс">аверсе</a> изображение епископа в <a href="/wiki/%D0%9C%D0%B8%D1%82%D1%80%D0%B0_(%D0%B3%D0%BE%D0%BB%D0%BE%D0%B2%D0%BD%D0%BE%D0%B9_%D1%83%D0%B1%D0%BE%D1%80)" title="Митра (головной убор)">митре</a> и подпись IOH’S, на <a href="/wiki/%D0%A0%D0%B5%D0%B2%D0%B5%D1%80%D1%81_(%D1%81%D1%82%D0%BE%D1%80%D0%BE%D0%BD%D0%B0_%D0%BC%D0%BE%D0%BD%D0%B5%D1%82%D1%8B)" title="Реверс (сторона монеты)">реверсе</a> — всадник с обнажённым мечом, надпись hDUX.<sup class="reference" id="cite_ref-dewitte_13-0"><a href="#cite_note-dewitte-13">[13]</a></sup></figcaption></figure>
<p>В <a href="/wiki/1204_%D0%B3%D0%BE%D0%B4" title="1204 год">1204 году</a> благодарный Филипп Швабский признал наследуемость герцогского титула, в том числе и по женской линии, а также передал Генриху <a href="/wiki/%D0%9D%D0%B8%D0%B2%D0%B5%D0%BB%D1%8C" title="Нивель">Нивель</a>, <a class="mw-redirect" href="/wiki/%D0%9D%D0%B8%D0%BC%D0%B2%D0%B5%D0%B3%D0%B5%D0%BD" title="Нимвеген">Нимвеген</a><sup class="reference" id="cite_ref-pirenn_11-1"><a href="#cite_note-pirenn-11">[11]</a></sup> и <a href="/wiki/%D0%9C%D0%B0%D0%B0%D1%81%D1%82%D1%80%D0%B8%D1%85%D1%82" title="Маастрихт">Маастрихт</a><sup class="reference" id="cite_ref-blok_14-0"><a href="#cite_note-blok-14">[14]</a></sup>. За счёт этого принадлежавший до этого льежскому епископству Маастрихт стал <a href="/wiki/%D0%9A%D0%BE%D0%BD%D0%B4%D0%BE%D0%BC%D0%B8%D0%BD%D0%B8%D1%83%D0%BC_(%D0%BC%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D0%BE%D0%B5_%D0%BF%D1%80%D0%B0%D0%B2%D0%BE)" title="Кондоминиум (международное право)">кондоминиумом</a>, то есть, совладением епископа Льежа и герцог Брабанта. Девиз города гласил: «Один господин — нет господина, два господина — господин» (<a href="/wiki/%D0%9D%D0%B8%D0%B4%D0%B5%D1%80%D0%BB%D0%B0%D0%BD%D0%B4%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Нидерландский язык">нидерл.</a> <span lang="nl" style="font-style:italic;">Eén heer is geen heer, twee heren zijn één heer</span>). Принесённая им в следующем году присяга на верность Филиппу-Августу означала ежегодную ренту в 200 марок серебром<sup class="reference" id="cite_ref-pirenn_11-2"><a href="#cite_note-pirenn-11">[11]</a></sup>.
</p><p>Убийство Филиппа Швабского в <a href="/wiki/1208_%D0%B3%D0%BE%D0%B4" title="1208 год">1208 году</a> означало для Генриха очередную смену лагеря<sup class="reference" id="cite_ref-pirenn_11-3"><a href="#cite_note-pirenn-11">[11]</a></sup>: он снова поддержал Оттона IV. Отлучение Оттона IV от церкви в <a href="/wiki/1210_%D0%B3%D0%BE%D0%B4" title="1210 год">1210 году</a> послужило поводом для возобновления войны с льежским епископом, который, подчинившись решению папы, отошел от императора. Генрих добился, чтобы ему поручили привести епископа к повиновению. Другим поводом к военным действиям послужила смерть в 1212 году дальнего родственника Генриха, <a href="/wiki/%D0%90%D0%BB%D1%8C%D0%B1%D0%B5%D1%80%D1%82_II_%D1%84%D0%BE%D0%BD_%D0%94%D0%B0%D0%B3%D1%81%D0%B1%D1%83%D1%80%D0%B3" title="Альберт II фон Дагсбург">Альберта II</a>, графа Моха, не оставившего наследников мужского пола и отказавшего свои владения епископству. Причиной конфликта герцога с Льежским епископством, служило географическое положение последнего: посередине между Брабантом и Рейном, откуда в Брабант притекало богатство<sup class="reference" id="cite_ref-pirenn_11-4"><a href="#cite_note-pirenn-11">[11]</a></sup>. <a href="/wiki/3_%D0%BC%D0%B0%D1%8F" title="3 мая">3 мая</a> <a href="/wiki/1212_%D0%B3%D0%BE%D0%B4" title="1212 год">1212 года</a> Генрих захватил <a href="/wiki/%D0%9B%D1%8C%D0%B5%D0%B6" title="Льеж">Льеж</a> и отдал его на разграбление. Таким образом ему удалось завладеть переходом через <a href="/wiki/%D0%9C%D0%B0%D0%B0%D1%81" title="Маас">Маас</a> и получить контроль над торговым путём из Германии в Нидерланды.
</p>

<p>В апреле <a href="/wiki/1213_%D0%B3%D0%BE%D0%B4" title="1213 год">1213 года</a><sup class="reference" id="cite_ref-pirenn_11-5"><a href="#cite_note-pirenn-11">[11]</a></sup> Генрих женился на <a href="/wiki/%D0%9C%D0%B0%D1%80%D0%B8%D1%8F_%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B0%D1%8F_(%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3%D0%B8%D0%BD%D1%8F_%D0%91%D1%80%D0%B0%D0%B1%D0%B0%D0%BD%D1%82%D0%B0)" title="Мария Французская (герцогиня Брабанта)">Марии Французской</a> (около <a href="/wiki/1198_%D0%B3%D0%BE%D0%B4" title="1198 год">1198</a>-<a href="/wiki/1224_%D0%B3%D0%BE%D0%B4" title="1224 год">1224</a>), дочери короля Франции <a href="/wiki/%D0%A4%D0%B8%D0%BB%D0%B8%D0%BF%D0%BF_II_%D0%90%D0%B2%D0%B3%D1%83%D1%81%D1%82" title="Филипп II Август">Филиппа II Августа</a> и <a href="/wiki/%D0%90%D0%B3%D0%BD%D0%B5%D1%81%D1%81%D0%B0_%D0%9C%D0%B5%D1%80%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F" title="Агнесса Меранская">Агнессы Меранской</a>, и тем самым, перешёл на стороны французского короля. В октябре Генрих продолжил боевые действия против льежского епископства, однако на этот раз кампания была менее удачной: <a href="/wiki/13_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="13 октября">13 октября</a> <a href="/wiki/1213_%D0%B3%D0%BE%D0%B4" title="1213 год">1213 года</a> он потерпел сокрушительное поражение в <a href="/wiki/%D0%91%D0%B8%D1%82%D0%B2%D0%B0_%D0%BD%D0%B0_%D0%A1%D1%82%D0%B5%D0%BF%D0%B5" title="Битва на Степе">битве на Степе</a> от отрядов льежской городской милиции, подкрепленных частями, прибывшими из графства <a href="/wiki/%D0%9B%D0%BE%D0%BE%D0%BD" title="Лоон">Лоон</a>. Битва при Степпе была первой в истории Нидерландов битвой<sup class="reference" id="cite_ref-pirenn_11-6"><a href="#cite_note-pirenn-11">[11]</a></sup>, в которой отряды городской милиции успешно противостояли феодальной армии. Впечатление, произведённое этой победой на современников, заметно по «Беседам о чудесах» <a href="/wiki/%D0%A6%D0%B5%D0%B7%D0%B0%D1%80%D0%B8%D0%B9_%D0%93%D0%B5%D0%B9%D1%81%D1%82%D0%B5%D1%80%D0%B1%D0%B0%D1%85%D1%81%D0%BA%D0%B8%D0%B9" title="Цезарий Гейстербахский">Цезария Гейстербахского</a>: он пишет о величайшем ристалище демонов, захвативших на поле битвы большую добычу<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup>. В результате поражения при Степпе льежские войска вошли на территорию Брабанта и Генриху пришлось на коленях просить льежского епископа о снятии с него отлучения от церкви (<a href="/wiki/28_%D1%84%D0%B5%D0%B2%D1%80%D0%B0%D0%BB%D1%8F" title="28 февраля">28 февраля</a> <a href="/wiki/1214_%D0%B3%D0%BE%D0%B4" title="1214 год">1214 года</a>)<sup class="reference" id="cite_ref-pirenn_11-7"><a href="#cite_note-pirenn-11">[11]</a></sup>. Реакцией Генриха на подобное публичное унижение стало возвращение в стан Вельфов, ознаменовавшееся браком дочери Генриха <a href="/wiki/%D0%9C%D0%B0%D1%80%D0%B8%D1%8F_%D0%91%D1%80%D0%B0%D0%B1%D0%B0%D0%BD%D1%82%D1%81%D0%BA%D0%B0%D1%8F_(%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D1%80%D0%B8%D1%86%D0%B0_%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D0%BE%D0%B9_%D0%A0%D0%B8%D0%BC%D1%81%D0%BA%D0%BE%D0%B9_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D0%B8)" title="Мария Брабантская (императрица Священной Римской империи)">Марии</a> и Оттона IV <a href="/wiki/19_%D0%BC%D0%B0%D1%8F" title="19 мая">19 мая</a> <a href="/wiki/1214_%D0%B3%D0%BE%D0%B4" title="1214 год">1214 года</a>. Именно поэтому <a href="/wiki/27_%D0%B8%D1%8E%D0%BB%D1%8F" title="27 июля">27 июля</a> того же года <a href="/wiki/%D0%91%D0%B8%D1%82%D0%B2%D0%B0_%D0%BF%D1%80%D0%B8_%D0%91%D1%83%D0%B2%D0%B8%D0%BD%D0%B5" title="Битва при Бувине">при Бувине</a> Генрих сражался против французского короля <a href="/wiki/%D0%A4%D0%B8%D0%BB%D0%B8%D0%BF%D0%BF_II_%D0%90%D0%B2%D0%B3%D1%83%D1%81%D1%82" title="Филипп II Август">Филиппа II Августа</a>. Историки Филиппа Августа сообщают, что Генрих предал своих союзников<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup>. Победа Филиппа Августа при Бувине и разгром вельфов обозначала для Генриха окончание политики балансирования между Вельфами и Гогенштауфенами. Генрих примкнул к победителям, принёс присягу <a href="/wiki/%D0%A4%D1%80%D0%B8%D0%B4%D1%80%D0%B8%D1%85_II_(%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%BE%D1%80_%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D0%BE%D0%B9_%D0%A0%D0%B8%D0%BC%D1%81%D0%BA%D0%BE%D0%B9_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D0%B8)" title="Фридрих II (император Священной Римской империи)">Фридриху II</a> (август <a href="/wiki/1218_%D0%B3%D0%BE%D0%B4" title="1218 год">1218 года</a><sup class="reference" id="cite_ref-vaderlandsche_1-9"><a href="#cite_note-vaderlandsche-1">[1]</a></sup>) и получил от него признание своих прав на Маастрихт<sup class="reference" id="cite_ref-pirenn_11-8"><a href="#cite_note-pirenn-11">[11]</a></sup>.
</p>
<h3><span id=".D0.9F.D0.BE.D1.81.D0.BB.D0.B5.D0.B4.D0.BD.D0.B8.D0.B5_.D0.B3.D0.BE.D0.B4.D1.8B"></span><span class="mw-headline" id="Последние_годы">Последние годы</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%93%D0%B5%D0%BD%D1%80%D0%B8%D1%85_I_(%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3_%D0%91%D1%80%D0%B0%D0%B1%D0%B0%D0%BD%D1%82%D0%B0)&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Последние годы»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%93%D0%B5%D0%BD%D1%80%D0%B8%D1%85_I_(%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3_%D0%91%D1%80%D0%B0%D0%B1%D0%B0%D0%BD%D1%82%D0%B0)&amp;action=edit&amp;section=6" title="Редактировать код раздела «Последние годы»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Уже в <a href="/wiki/1192_%D0%B3%D0%BE%D0%B4" title="1192 год">1192 году</a> Генрих даровал права жителям <a href="/wiki/%D0%92%D0%B8%D0%BB%D0%B2%D0%BE%D1%80%D0%B4%D0%B5" title="Вилворде">Вилворде</a>, одного из старейших аллодов Брабантского дома. В то время как жители Вилворде не являлись крепостными, они так же не имели гражданских прав, сопоставимых с правами жителей других городов. Права предоставленные Генрихом ограничивали ситуации, в которых с ним мог взиматься налог; регламентировали вопросы судопроизводства; освобождали жителей Вилворде от воинской повинности вне границ герцогства. Генрих отменял барщину (за исключением обязанности косить сено на герцогских угодьях и просушивать его), и т. д.<sup class="reference" id="cite_ref-vaderlandsche_1-10"><a href="#cite_note-vaderlandsche-1">[1]</a></sup> В <a href="/wiki/1211_%D0%B3%D0%BE%D0%B4" title="1211 год">1211 году</a> Генрих издал для Лёвена законы, регулирующие вопросы торговли и уголовных наказаний, в <a href="/wiki/1213_%D0%B3%D0%BE%D0%B4" title="1213 год">1213 году</a> — учредил восьмидневную ярмарку в <a class="new" href="/w/index.php?title=%D0%97%D0%B0%D1%83%D1%82%D0%BB%D0%B5%D1%83&amp;action=edit&amp;redlink=1" title="Заутлеу (страница отсутствует)">Заутлеу</a>.
</p><p>Тем не менее, пик подобного рода законотворческой активности Генриха приходится на последний период его жизни<sup class="reference" id="cite_ref-vaderlandsche_1-11"><a href="#cite_note-vaderlandsche-1">[1]</a></sup>. В <a href="/wiki/1221_%D0%B3%D0%BE%D0%B4" title="1221 год">1221 году</a> статус города присваивается <a href="/wiki/%D0%A5%D0%B5%D1%80%D0%B5%D0%BD%D1%82%D0%B0%D0%BB%D1%81" title="Херенталс">Херенталсу</a>, через год жители <a href="/wiki/%D0%92%D0%B0%D0%B2%D1%80" title="Вавр">Вавра</a> получили права, сопоставимые с правами жителей Лёвена, в <a href="/wiki/1229_%D0%B3%D0%BE%D0%B4" title="1229 год">1229 году</a> свой «уголовный кодекс» получил <a href="/wiki/%D0%91%D1%80%D1%8E%D1%81%D1%81%D0%B5%D0%BB%D1%8C" title="Брюссель">Брюссель</a>, а <a href="/wiki/%D0%AD%D0%B9%D0%BD%D0%B4%D1%85%D0%BE%D0%B2%D0%B5%D0%BD" title="Эйндховен">Эйндховен</a> стал городом в <a href="/wiki/1232_%D0%B3%D0%BE%D0%B4" title="1232 год">1232</a>. В то же время Генрих основал несколько монастырей: Тер Банк у Лёвена (<a href="/wiki/1216_%D0%B3%D0%BE%D0%B4" title="1216 год">1216</a>), св. Бернарда на <a href="/wiki/%D0%A8%D0%B5%D0%BB%D1%8C%D0%B4%D0%B0" title="Шельда">Шельде</a> (<a href="/wiki/1233_%D0%B3%D0%BE%D0%B4" title="1233 год">1233</a>); более того, он стал ежегодно, начиная с <a href="/wiki/1221_%D0%B3%D0%BE%D0%B4" title="1221 год">1221 года</a>, выплачивать определённую сумму викарию алтаря Трёх <a class="mw-redirect" href="/wiki/%D0%92%D0%BE%D0%BB%D1%85%D0%B2" title="Волхв">волхвов</a> в <a href="/wiki/%D0%9A%D1%91%D0%BB%D1%8C%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D0%BE%D0%B1%D0%BE%D1%80" title="Кёльнский собор">кёльнском соборе</a><sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup>, а в <a href="/wiki/1224_%D0%B3%D0%BE%D0%B4" title="1224 год">1224 году</a> передал десятину нескольких деревень брюссельским каноникам<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup>.
</p>
<figure class="mw-default-size mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:20091015_leuven_stpierre06.JPG"><img class="mw-file-element" data-file-height="2592" data-file-width="3872" decoding="async" height="147" src="//upload.wikimedia.org/wikipedia/commons/thumb/2/2b/20091015_leuven_stpierre06.JPG/220px-20091015_leuven_stpierre06.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/2/2b/20091015_leuven_stpierre06.JPG/330px-20091015_leuven_stpierre06.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/2/2b/20091015_leuven_stpierre06.JPG/440px-20091015_leuven_stpierre06.JPG 2x" width="220"/></a><figcaption>Надгробие Генриха в ц. св. <p>B <a href="/wiki/1224_%D0%B3%D0%BE%D0%B4" title="1224 год">1224 году</a> он освободил от налогов жителей Бюнсвейка в обмен на обязательство использовать его герб в ходе военных операций<sup class="reference" id="cite_ref-boffa_19-0"><a href="#cite_note-boffa-19">[19]</a></sup>: подобная щедрость связана с неоднократными попытками захватить или убить человека с влиятельным гербом. Так, в вышеупомянутой битве при Степпе герб Генриха нёс заплативший за это своей жизнью Генрих Гульденбергский<sup class="reference" id="cite_ref-boffa_19-1"><a href="#cite_note-boffa-19">[19]</a></sup>, а сам Генрих планировал убийство графа <a href="/wiki/%D0%9B%D0%BE%D0%BE%D0%BD" title="Лоон">Лоона</a><sup class="reference" id="cite_ref-boffa_19-2"><a href="#cite_note-boffa-19">[19]</a></sup>.
</p><p>В <a href="/wiki/1235_%D0%B3%D0%BE%D0%B4" title="1235 год">1235 году</a> Генрих сопровождал в <a href="/wiki/%D0%9C%D0%B0%D0%B9%D0%BD%D1%86" title="Майнц">Майнц</a> невесту Фридриха II <a href="/wiki/%D0%98%D0%B7%D0%B0%D0%B1%D0%B5%D0%BB%D0%BB%D0%B0_%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B0%D1%8F" title="Изабелла Английская">Изабеллу Английскую</a>. <a href="/wiki/5_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="5 сентября">5 сентября</a><sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup>, на обратном пути, он заболел и умер.
</p><p>Надгробие Генриха, его первой супруги Матильды и их дочери <a href="/wiki/%D0%9C%D0%B0%D1%80%D0%B8%D1%8F_%D0%91%D1%80%D0%B0%D0%B1%D0%B0%D0%BD%D1%82%D1%81%D0%BA%D0%B0%D1%8F_(%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D1%80%D0%B8%D1%86%D0%B0_%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D0%BE%D0%B9_%D0%A0%D0%B8%D0%BC%D1%81%D0%BA%D0%BE%D0%B9_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D0%B8)" title="Мария Брабантская (императрица Священной Римской империи)">Марии</a> находятся в ц. св. Петра в <a href="/wiki/%D0%9B%D1%91%D0%B2%D0%B5%D0%BD" title="Лёвен">Лёвене</a>.

<h2><span id=".D0.98.D0.B7.D0.BE.D0.B1.D1.80.D0.B0.D0.B6.D0.B5.D0.BD.D0.B8.D1.8F_.D0.93.D0.B5.D0.BD.D1.80.D0.B8.D1.85.D0.B0_I_.D0.B2_.D0.B8.D1.81.D0.BA.D1.83.D1.81.D1.81.D1.82.D0.B2.D0.B5"></span><span class="mw-headline" id="Изображения_Генриха_I_в_искусстве">Изображения Генриха I в искусстве</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%93%D0%B5%D0%BD%D1%80%D0%B8%D1%85_I_(%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3_%D0%91%D1%80%D0%B0%D0%B1%D0%B0%D0%BD%D1%82%D0%B0)&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Изображения Генриха I в искусстве»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%93%D0%B5%D0%BD%D1%80%D0%B8%D1%85_I_(%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3_%D0%91%D1%80%D0%B0%D0%B1%D0%B0%D0%BD%D1%82%D0%B0)&amp;action=edit&amp;section=7" title="Редактировать код раздела «Изображения Генриха I в искусстве»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-default-size mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Jin1a.jpg"><img class="mw-file-element" data-file-height="190" data-file-width="323" decoding="async" height="129" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/30/Jin1a.jpg/220px-Jin1a.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/3/30/Jin1a.jpg 1.5x" width="220"/></a><figcaption>Серебряная монета Генриха I. На <a href="/wiki/%D0%90%D0%B2%D0%B5%D1%80%D1%81" title="Аверс">аверсе</a> портрет герцога в украшенном <a class="mw-redirect" href="/wiki/%D0%A4%D0%BB%D1%91%D1%80-%D0%B4%D0%B5-%D0%BB%D0%B8" title="Флёр-де-ли">лилией</a> шлеме, с щитом с изображением льва в левой руке и мечом в правой; подпись DUX. На <a href="/wiki/%D0%A0%D0%B5%D0%B2%D0%B5%D1%80%D1%81_(%D1%81%D1%82%D0%BE%D1%80%D0%BE%D0%BD%D0%B0_%D0%BC%D0%BE%D0%BD%D0%B5%D1%82%D1%8B)" title="Реверс (сторона монеты)">реверсе</a> — изображение льва и подпись LEO<sup class="reference" id="cite_ref-dewitte_13-1"><a href="#cite_note-dewitte-13">[13]</a></sup>.</figcaption></figure>
<p>До нас дошёл портрет Генриха, отчеканенный на его монетах. Генрих был первым властителем Лёвена или Брабанта, использовавшим изображение льва на монетах<sup class="reference" id="cite_ref-dewitte_13-2"><a href="#cite_note-dewitte-13">[13]</a></sup>.
</p><p>Позднее портреты Генриха I часто включались в «циклы правителей», в том числе в опубликованной около 1600 Адрианом ван Барландом и Яном Моретусом «Хронике герцогов Брабантских». В девятнадцатом веке с независимостью <a href="/wiki/%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D1%8F" title="Бельгия">Бельгии</a> подобные циклы были созданы <a href="/wiki/%D0%9B%D0%B5%D0%B9%D1%81,_%D0%A5%D0%B5%D0%BD%D0%B4%D1%80%D0%B8%D0%BA" title="Лейс, Хендрик">Хендриком Лейсом</a> и Альфонсом Петерсом для ратуши <a href="/wiki/%D0%90%D0%BD%D1%82%D0%B2%D0%B5%D1%80%D0%BF%D0%B5%D0%BD" title="Антверпен">Антверпена</a>, Полём Дюбуа для т. н. <a class="mw-redirect" href="/wiki/%D0%93%D1%80%D0%B0%D0%BD-%D0%BF%D0%BB%D0%B0%D1%81" title="Гран-плас">«дома короля»</a> в Брюсселе и Шарлем Фрекеном для приюта Пашеко (<a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Французский язык">фр.</a> <span lang="fr" style="font-style:italic;">Hospice Pachéco</span>) там же<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup>. Задуманный для антверпенской ратуши триптих Лейса с изображением <a href="/wiki/%D0%93%D0%BE%D1%82%D1%84%D1%80%D0%B8%D0%B4_%D0%91%D1%83%D0%BB%D1%8C%D0%BE%D0%BD%D1%81%D0%BA%D0%B8%D0%B9" title="Готфрид Бульонский">Готфрида Бульонского</a>, Генриха и <a href="/wiki/%D0%96%D0%B0%D0%BD_I_(%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3_%D0%91%D1%80%D0%B0%D0%B1%D0%B0%D0%BD%D1%82%D0%B0)" title="Жан I (герцог Брабанта)">Иоанна I</a> остался незавершённым из-за смерти художника в <a href="/wiki/1869_%D0%B3%D0%BE%D0%B4" title="1869 год">1869 году</a>. По решению города на месте ненаписанной панели с изображением Готфрида Бульонского находится надпись, объясняющая решение города не завершать триптих в память о Лейсе<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup>.
</p>
