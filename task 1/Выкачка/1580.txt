<p><b>Canon EOS 7D</b> — <a href="/wiki/%D0%A6%D0%B8%D1%84%D1%80%D0%BE%D0%B2%D0%BE%D0%B9_%D0%B7%D0%B5%D1%80%D0%BA%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D1%82%D0%BE%D0%B0%D0%BF%D0%BF%D0%B0%D1%80%D0%B0%D1%82" title="Цифровой зеркальный фотоаппарат">цифровой зеркальный фотоаппарат</a> семейства <a href="/wiki/Canon_EOS" title="Canon EOS">EOS</a> компании <a href="/wiki/Canon" title="Canon">«Кэнон»</a>, обладающий рядом характеристик, свойственных профессиональным моделям. Представлен 1 сентября 2009 года<sup class="reference" id="cite_ref-Canon_USA_press-release_2-0"><a href="#cite_note-Canon_USA_press-release-2">[2]</a></sup>, производился с осени 2009 года по 2014 год на предприятии «Оита Кэнон» в городе <a href="/wiki/%D0%9A%D1%83%D0%BD%D0%B8%D1%81%D0%B0%D0%BA%D0%B8" title="Кунисаки">Кунисаки</a>, префектура Оита, Япония<sup class="reference" id="cite_ref-oita_3-0"><a href="#cite_note-oita-3">[3]</a></sup>. Модель предназначена как для профессионалов, так и для опытных любителей<sup class="reference" id="cite_ref-Canon_USA_press-release_2-1"><a href="#cite_note-Canon_USA_press-release-2">[2]</a></sup>.
</p><p>Фотоаппарат имеет <a class="mw-redirect" href="/wiki/%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%86%D0%B0_(%D1%84%D0%BE%D1%82%D0%BE)" title="Матрица (фото)">матрицу</a> с <a href="/wiki/%D0%9A%D1%80%D0%BE%D0%BF-%D1%84%D0%B0%D0%BA%D1%82%D0%BE%D1%80" title="Кроп-фактор">кроп-фактором</a> 1,6 (формат <a href="/wiki/APS-C" title="APS-C">APS-C</a>) и позволяет использовать предназначенные для матриц такого размера объективы <a href="/wiki/Canon_EF-S" title="Canon EF-S">EF-S</a>. EOS 7D стал первым фотоаппаратом семейства EOS, оснащённым встроенным <a href="/wiki/%D0%9D%D0%B8%D0%B2%D0%B5%D0%BB%D0%B8%D1%80" title="Нивелир">нивелиром</a>, беспроводным передатчиком для управления внешними вспышками и рядом новых возможностей для управления видеосъёмкой<sup class="reference" id="cite_ref-behind_4-0"><a href="#cite_note-behind-4">[4]</a></sup>. Новая система автофокусировки, учитывающая положение фотокамеры, объявлена компанией-производителем наиболее передовой из всех когда-либо использовавшихся в фотоаппаратах семейства Canon EOS. Новая система экспозамера впервые для фотоаппаратов Canon учитывает информацию о цвете<sup class="reference" id="cite_ref-Canon_USA_press-release_2-2"><a href="#cite_note-Canon_USA_press-release-2">[2]</a></sup>.
</p><p>Функции видеосъёмки улучшены по сравнению с предыдущими моделями семейства, в частности, появилось полностью ручное управление <a href="/wiki/%D0%92%D1%8B%D0%B4%D0%B5%D1%80%D0%B6%D0%BA%D0%B0_(%D1%84%D0%BE%D1%82%D0%BE)" title="Выдержка (фото)">выдержкой</a> и <a href="/wiki/%D0%94%D0%B8%D0%B0%D1%84%D1%80%D0%B0%D0%B3%D0%BC%D0%B0_%D0%BE%D0%B1%D1%8A%D0%B5%D0%BA%D1%82%D0%B8%D0%B2%D0%B0" title="Диафрагма объектива">диафрагмой</a>. EOS 7D использовался в качестве <a href="/wiki/%D0%A6%D0%B8%D1%84%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%BA%D0%B8%D0%BD%D0%BE%D0%BA%D0%B0%D0%BC%D0%B5%D1%80%D0%B0" title="Цифровая кинокамера">цифровой кинокамеры</a> при съёмке ряда популярных <a href="/wiki/%D0%A4%D0%B8%D0%BB%D1%8C%D0%BC" title="Фильм">кинофильмов</a>, в том числе <a href="/wiki/%D0%A7%D1%91%D1%80%D0%BD%D1%8B%D0%B9_%D0%BB%D0%B5%D0%B1%D0%B5%D0%B4%D1%8C_(%D1%84%D0%B8%D0%BB%D1%8C%D0%BC,_2010)" title="Чёрный лебедь (фильм, 2010)">«Чёрный лебедь»</a><sup class="reference" id="cite_ref-Black_swan_5-0"><a href="#cite_note-Black_swan-5">[5]</a></sup>, <a href="/wiki/127_%D1%87%D0%B0%D1%81%D0%BE%D0%B2" title="127 часов">«127 часов»</a><sup class="reference" id="cite_ref-127_hours_6-0"><a href="#cite_note-127_hours-6">[6]</a></sup> и <a href="/wiki/%D0%9C%D1%81%D1%82%D0%B8%D1%82%D0%B5%D0%BB%D0%B8_(%D1%84%D0%B8%D0%BB%D1%8C%D0%BC,_2012)" title="Мстители (фильм, 2012)">«Мстители»</a><sup class="reference" id="cite_ref-avengers_7-0"><a href="#cite_note-avengers-7">[7]</a></sup>.
</p><p>EOS 7D стал одним из самых долгоживущих цифровых фотоаппаратов компании, оставаясь актуальной моделью на протяжении около пяти лет вплоть до появления наследника — <a href="/wiki/Canon_EOS_7D_Mark_II" title="Canon EOS 7D Mark II">EOS 7D Mark II</a>.
</p>

<h2><span id=".D0.9F.D0.BE.D0.B7.D0.B8.D1.86.D0.B8.D0.BE.D0.BD.D0.B8.D1.80.D0.BE.D0.B2.D0.B0.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Позиционирование">Позиционирование</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Canon_EOS_7D&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Позиционирование»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Canon_EOS_7D&amp;action=edit&amp;section=1" title="Редактировать код раздела «Позиционирование»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>На момент появления Canon EOS 7D семейство цифровых фотоаппаратов Canon EOS состояло из четырёх линеек:
</p>
<ol><li>профессиональные фотоаппараты <a href="/wiki/Canon_EOS-1D" title="Canon EOS-1D">EOS-1D</a>;</li>
<li>полнокадровые камеры для фотографов-энтузиастов и подготовленных любителей, актуальной на 2009 год моделью являлась <a href="/wiki/Canon_EOS_5D_Mark_II" title="Canon EOS 5D Mark II">Canon EOS 5D Mark II</a>;</li>
<li>фотоаппараты с матрицей формата <a href="/wiki/APS-C" title="APS-C">APS-C</a> (<a href="/wiki/%D0%9A%D1%80%D0%BE%D0%BF-%D1%84%D0%B0%D0%BA%D1%82%D0%BE%D1%80" title="Кроп-фактор">кроп-фактор</a> 1,6) и с двузначным числовым индексом модели, предназначенные для опытных фотолюбителей, последней моделью была <a href="/wiki/Canon_EOS_50D" title="Canon EOS 50D">Canon EOS 50D</a>;</li>
<li>любительские модели с матрицей формата APS-C, обозначаемые трёх- или четырёхзначным индексом<sup class="reference" id="cite_ref-comparison_8-0"><a href="#cite_note-comparison-8">[8]</a></sup>.</li></ol>
<p>EOS 7D стал наиболее технически оснащённым фотоаппаратом Canon с матрицей формата APS-C, сменив в этом качестве появившуюся годом ранее модель EOS 50D. Однако 7D не стал преемником 50D, а открыл собой новую линейку в семействе EOS<sup class="reference" id="cite_ref-comparison_8-1"><a href="#cite_note-comparison-8">[8]</a></sup><sup class="reference" id="cite_ref-dpreview_1_9-0"><a href="#cite_note-dpreview_1-9">[9]</a></sup>. Об этом также говорит и обозначение модели с индексом из одной цифры, и разница в стоимости: рекомендуемая цена 50D на официальном сайте Canon USA в 2008—2009 годах составляла 1199 <a href="/wiki/%D0%94%D0%BE%D0%BB%D0%BB%D0%B0%D1%80_%D0%A1%D0%A8%D0%90" title="Доллар США">долларов США</a> за вариант без объектива, а 7D — 1699 долларов.
</p><p>Компания «Кэнон» определяет 7D как бескомпромиссный фотоаппарат для серьёзных фотографов и как вторую камеру для профессионалов, работающих в нестудийных условиях<sup class="reference" id="cite_ref-Canon_USA_press-release_2-3"><a href="#cite_note-Canon_USA_press-release-2">[2]</a></sup>.
</p>
<h2><span id=".D0.98.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.8F_.D1.81.D0.BE.D0.B7.D0.B4.D0.B0.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="История_создания">История создания</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Canon_EOS_7D&amp;veaction=edit&amp;section=2" title="Редактировать раздел «История создания»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Canon_EOS_7D&amp;action=edit&amp;section=2" title="Редактировать код раздела «История создания»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>На этапе проектирования 7D компания «Кэнон» провела опрос более 5000 фотографов со всего мира с целью выяснить, каким они видят фотоаппарат своей мечты, какие идеи могут быть воплощены в новой модели. Были опрошены фотографы с разными возможностями и разным опытом; по словам одного из руководителей департамента разработки фотоаппаратов компании Кэнъити Синбори, это было важно для того, чтобы понять, что разные люди ищут для себя в фотокамере<sup class="reference" id="cite_ref-behind_4-1"><a href="#cite_note-behind-4">[4]</a></sup>.
</p><p>В качестве нововведений, воплощённых в фотоаппарате в результате этого исследования, названы в том числе большой видоискатель, защита корпуса от пыли и влаги, а также кнопки, позволяющие работать с камерой в перчатках<sup class="reference" id="cite_ref-behind_4-2"><a href="#cite_note-behind-4">[4]</a></sup>.
</p>
<h2><span id=".D0.9D.D0.BE.D0.B2.D1.88.D0.B5.D1.81.D1.82.D0.B2.D0.B0"></span><span class="mw-headline" id="Новшества">Новшества</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Canon_EOS_7D&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Новшества»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Canon_EOS_7D&amp;action=edit&amp;section=3" title="Редактировать код раздела «Новшества»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Canon EOS 7D получил целый ряд функций, новых для всего семейства Canon EOS, включая профессиональные модели. По словам Юити Исидзуки, старшего вице-президента «Кэнон США», новая модель стала наиболее оснащённой и инновационной из всех цифровых зеркальных фотоаппаратов компании<sup class="reference" id="cite_ref-Canon_USA_press-release_2-4"><a href="#cite_note-Canon_USA_press-release-2">[2]</a></sup>.
</p><p><b>Автофокусировка.</b>
Одно из главных нововведений — принципиально новая для фотоаппаратов марки 19-точечная система автофокусировки. Все 19 точек являются крестообразными, то есть дают возможность наводиться на объекты, контрастные в любой из двух плоскостей, горизонтальной или вертикальной. При этом центральная точка обладает повышенной точностью при использовании объективов со светосилой f/2,8 и больше благодаря дополнительным диагональным датчикам. В режиме <i>Spot AF</i> уменьшается используемый размер фокусировочного датчика, что позволяет точно наводиться на особо малые объекты и успешно функционировать в сложных условиях, например, при съёмке животных, находящихся в клетке. Возможен «захват» движущегося объекта любой из точек и слежение за ним с автоматическим переключением точек (предыдущие модули могли «захватить» объект только центральной точкой). Функция <i>AF Point Switching</i> позволяет настроить две разные точки для горизонтального и для вертикального положений фотоаппарата с автоматическим переключением между ними при изменении положения.
</p><p>Фотоаппарат автоматически распознаёт установленный макрообъектив и меняет алгоритм фокусировки в режиме <i>AI Servo</i>, подстраиваясь под колебания объекта и непроизвольные движения фотографа.
</p><p><b>Экспозамер.</b>
Новая 63-зонная система экспозамера iFCL впервые для фотоаппаратов EOS учитывает не только яркость света, но и его цвет, позволяя добиваться стабильных результатов при различных источниках освещения.
</p><p><b>Беспроводное управление внешними вспышками.</b>
Впервые для фотоаппаратов Canon реализована возможность беспроводного управления внешними вспышками <a class="mw-redirect" href="/wiki/Canon_EOS_flash_system" title="Canon EOS flash system">Speedlite</a>. Вспышка фотоаппарата имеет встроенный передатчик, что позволяет отказаться от использования отдельного внешнего передатчика. Возможно управление тремя группами вспышек, каждая из групп может состоять из четырёх вспышек.
</p>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Canon_EOS_7D_back_04.jpg"><img class="mw-file-element" data-file-height="1000" data-file-width="400" decoding="async" height="275" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ed/Canon_EOS_7D_back_04.jpg/110px-Canon_EOS_7D_back_04.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ed/Canon_EOS_7D_back_04.jpg/165px-Canon_EOS_7D_back_04.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ed/Canon_EOS_7D_back_04.jpg/220px-Canon_EOS_7D_back_04.jpg 2x" width="110"/></a><figcaption>Блок кнопок</figcaption></figure>
<p><b>Кнопки.</b>
Кнопки управления Canon EOS 7D спроектированы для удобной работы с фотоаппаратом в перчатках. Добавлены новые кнопки: <i>Quick Control</i> открывает доступ к меню быстрой настройки с помощью расположенного на задней панели ЖК-дисплея, <i>RAW/JPEG</i> позволяет оперативно включить запись снимка в максимальном качестве, отдельная кнопка управления видеосъёмкой. Возможности по переназначению функций кнопок у 7D шире, чем у предыдущих моделей.
</p><p><b>Видоискатель.</b>
Впервые в непрофессиональной модели семейства Canon EOS применён видоискатель со стопроцентным охватом кадра<sup class="reference" id="cite_ref-eos-comparison_10-0"><a href="#cite_note-eos-comparison-10">[10]</a></sup>. Встроенный в видоискатель жидкокристаллический дисплей отображает фокусировочные точки и зоны, линии сетки и круг зоны точечного экспозамера. Дисплей может подсвечиваться в условиях низкой освещённости. Отображение дополнительной информации может быть отключено.
</p><p><b>Электронный нивелир.</b>
Canon EOS 7D стал первым фотоаппаратом марки, в котором применён электронный <a href="/wiki/%D0%9D%D0%B8%D0%B2%D0%B5%D0%BB%D0%B8%D1%80" title="Нивелир">нивелир</a>. Двухосный нивелир фиксирует отклонения от горизонтали (поворот оптической оси по часовой и против часовой стрелки) и от вертикали (наклон вперёд и назад). Отклонение отображается на ЖК-дисплее в режиме <a class="mw-redirect" href="/wiki/%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%BE%D1%81%D0%BC%D0%BE%D1%82%D1%80_%D0%B2_%D1%80%D0%B5%D0%B0%D0%BB%D1%8C%D0%BD%D0%BE%D0%BC_%D0%B2%D1%80%D0%B5%D0%BC%D0%B5%D0%BD%D0%B8" title="Предпросмотр в реальном времени">предпросмотра в реальном времени</a> или с помощью подсветки точек фокусировки в видоискателе.
</p>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Canon_EOS_7D_button_01.jpg"><img class="mw-file-element" data-file-height="400" data-file-width="320" decoding="async" height="150" src="//upload.wikimedia.org/wikipedia/commons/thumb/7/76/Canon_EOS_7D_button_01.jpg/120px-Canon_EOS_7D_button_01.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/7/76/Canon_EOS_7D_button_01.jpg/180px-Canon_EOS_7D_button_01.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/7/76/Canon_EOS_7D_button_01.jpg/240px-Canon_EOS_7D_button_01.jpg 2x" width="120"/></a><figcaption>Кнопка управления видеосъёмкой</figcaption></figure>
<p><b>Видеосъёмка.</b>
Возможность видеосъёмки — и сразу в формате <a href="/wiki/Full_HD" title="Full HD">Full HD</a> — впервые для моделей EOS появилась у 5D Mark II, а затем была предложена в любительской модели <a href="/wiki/Canon_EOS_500D" title="Canon EOS 500D">500D</a>. Однако 7D, будучи третьей моделью с этой функцией в семействе, обладает рядом принципиально новых для Canon EOS возможностей видеосъёмки.
</p><p>Для более удобного и оперативного управления съёмкой добавлена отдельная кнопка перехода в режим предпросмотра в реальном времени и начала/прекращения видеозаписи. Впервые появилось полностью ручное управление выдержкой и диафрагмой в процессе съёмки видео. В отличие от предыдущих моделей, стало возможным выбрать частоту кадров, совместимую как с системой <a href="/wiki/NTSC" title="NTSC">НТСЦ</a>, так и с системой <a href="/wiki/PAL" title="PAL">ПАЛ</a><sup class="reference" id="cite_ref-behind_4-3"><a href="#cite_note-behind-4">[4]</a></sup>:
</p>
<ul><li>1920 × 1080 — 29,97; 25 или 23,976 кадров в секунду,</li>
<li>1280 × 720 — 59,94 или 50 кадров/с,</li>
<li>640 × 480 — 59,94 или 50 кадров/с.</li></ul>
<p>Тем не менее, возможность установки уровня записи звука появилась только с выходом в 2012 году микропрограммы версии 2<span class="ts-Переход noprint" title="#Версия 2.0"><span typeof="mw:File"><a href="#Версия_2.0" title=""><img alt="Перейти к разделу «#Версия 2.0»" class="mw-file-element" data-file-height="48" data-file-width="48" decoding="async" height="10" src="//upload.wikimedia.org/wikipedia/commons/thumb/5/5e/2rightarrow.png/10px-2rightarrow.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/5/5e/2rightarrow.png/15px-2rightarrow.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/5/5e/2rightarrow.png/20px-2rightarrow.png 2x" width="10"/></a></span></span> — и то лишь до начала съёмки, но не в процессе. Также у фотоаппарата отсутствует разъём для наушников (впервые для EOS появился в модели <a href="/wiki/Canon_EOS_5D_Mark_III" title="Canon EOS 5D Mark III">5D Mark III</a>), что не даёт возможности контролировать качество записи звука.
</p>
<h2><span id=".D0.9E.D1.81.D0.BE.D0.B1.D0.B5.D0.BD.D0.BD.D0.BE.D1.81.D1.82.D0.B8_.D0.BC.D0.BE.D0.B4.D0.B5.D0.BB.D0.B8"></span><span class="mw-headline" id="Особенности_модели">Особенности модели</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Canon_EOS_7D&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Особенности модели»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Canon_EOS_7D&amp;action=edit&amp;section=4" title="Редактировать код раздела «Особенности модели»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id=".D0.9A.D0.BE.D1.80.D0.BF.D1.83.D1.81_.D0.B8_.D0.BC.D0.B5.D1.85.D0.B0.D0.BD.D0.B8.D0.BA.D0.B0"></span><span class="mw-headline" id="Корпус_и_механика">Корпус и механика</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Canon_EOS_7D&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Корпус и механика»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Canon_EOS_7D&amp;action=edit&amp;section=5" title="Редактировать код раздела «Корпус и механика»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Canon_EOS_7D_front_05.jpg"><img class="mw-file-element" data-file-height="900" data-file-width="1200" decoding="async" height="180" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/85/Canon_EOS_7D_front_05.jpg/240px-Canon_EOS_7D_front_05.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/85/Canon_EOS_7D_front_05.jpg/360px-Canon_EOS_7D_front_05.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/85/Canon_EOS_7D_front_05.jpg/480px-Canon_EOS_7D_front_05.jpg 2x" width="240"/></a><figcaption>Canon EOS 7D без объектива</figcaption></figure>
<p>Фотоаппарат имеет корпус с защитой от влаги и пыли, что отличает его от младших моделей. Как заявляет производитель, уровень защиты соответствует таковому у профессиональной плёночной модели <a class="mw-redirect" href="/wiki/Canon_EOS-1N" title="Canon EOS-1N">Canon EOS-1N</a>, выпускавшейся в 1990-х годах<sup class="reference" id="cite_ref-behind_4-4"><a href="#cite_note-behind-4">[4]</a></sup>. Canon 5D Mark II тоже имеет защиту корпуса, но её уровень не уточняется.
</p><p>Расположение кнопок слева от дисплея под большим пальцем левой руки аналогично 5D Mark II и отличает 7D от 50D, у которой кнопки расположены под дисплеем<sup class="reference" id="cite_ref-comparison_8-2"><a href="#cite_note-comparison-8">[8]</a></sup>.
</p><p>Встроенная вспышка охватывает угол, эквивалентный углу обзора объективов с фокусным расстоянием 15 мм, не ограничивая использование нового объектива <a href="/wiki/Canon_EF-S_15-85_%D0%BC%D0%BC" title="Canon EF-S 15-85 мм">Canon EF-S 15-85mm</a>. Предыдущие модели с матрицами формата APS-C имеют угол охвата, соответствующий фокусному расстоянию 17 мм, а у 5D Mark II встроенная вспышка отсутствует.
</p>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Canon_EOS_7D_back_05.jpg"><img class="mw-file-element" data-file-height="900" data-file-width="1200" decoding="async" height="225" src="//upload.wikimedia.org/wikipedia/commons/thumb/5/5a/Canon_EOS_7D_back_05.jpg/300px-Canon_EOS_7D_back_05.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/5/5a/Canon_EOS_7D_back_05.jpg/450px-Canon_EOS_7D_back_05.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/5/5a/Canon_EOS_7D_back_05.jpg/600px-Canon_EOS_7D_back_05.jpg 2x" width="300"/></a><figcaption>Меню фотоаппарата. Выделен раздел с информацией об аккумуляторе</figcaption></figure>
<p>Как и в 5D Mark II, в новом фотоаппарате используется аккумулятор LP-E6, передающий дополнительную информацию в фотоаппарат. Фотоаппарат запоминает несколько последних использовавшихся аккумуляторных батарей и хранит информацию об остатке заряда каждой из них<sup class="reference" id="cite_ref-dpreview_p6_11-0"><a href="#cite_note-dpreview_p6-11">[11]</a></sup>.
</p>
<h3><span id=".D0.AD.D0.BB.D0.B5.D0.BA.D1.82.D1.80.D0.BE.D0.BD.D0.B8.D0.BA.D0.B0"></span><span class="mw-headline" id="Электроника">Электроника</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Canon_EOS_7D&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Электроника»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Canon_EOS_7D&amp;action=edit&amp;section=6" title="Редактировать код раздела «Электроника»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>В Canon EOS 7D установлена <a href="/wiki/%D0%9A%D0%9C%D0%9E%D0%9F-%D0%BC%D0%B0%D1%82%D1%80%D0%B8%D1%86%D0%B0" title="КМОП-матрица">КМОП-матрица</a> формата <a href="/wiki/APS-C" title="APS-C">APS-C</a>, разработанная для этого фотоаппарата и позднее использовавшаяся в целом ряде моделей: <a href="/wiki/Canon_EOS_550D" title="Canon EOS 550D">550D</a>, <a href="/wiki/Canon_EOS_60D" title="Canon EOS 60D">60D</a>, <a href="/wiki/Canon_EOS_600D" title="Canon EOS 600D">600D</a> и, в доработанном виде, в <a href="/wiki/Canon_EOS_650D" title="Canon EOS 650D">650D</a> и <a href="/wiki/Canon_EOS_M" title="Canon EOS M">EOS M</a>. Разрешение 17,9 <a href="/wiki/%D0%9C%D0%B5%D0%B3%D0%B0%D0%BF%D0%B8%D0%BA%D1%81%D0%B5%D0%BB%D1%8C" title="Мегапиксель">млн пикселей</a> стало рекордным для матриц этого формата на момент появления модели, и в течение двух лет столь же большим разрешением обладали  только другие модели семейства EOS<sup class="reference" id="cite_ref-camera-comparison_12-0"><a href="#cite_note-camera-comparison-12">[12]</a></sup>. Лишь в августе 2011 года были представлены фотоаппараты <a class="new" href="/w/index.php?title=Sony_NEX-7&amp;action=edit&amp;redlink=1" title="Sony NEX-7 (страница отсутствует)">Sony NEX-7</a> и <a href="/wiki/Sony_SLT-A77" title="Sony SLT-A77">Sony SLT-A77</a>, которые получили матрицу формата APS-C с разрешением 24 млн пикселей, а в сентябре был представлен <a class="mw-redirect" href="/wiki/%D0%91%D0%B5%D0%B7%D0%B7%D0%B5%D1%80%D0%BA%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D1%86%D0%B8%D1%84%D1%80%D0%BE%D0%B2%D0%BE%D0%B9_%D1%84%D0%BE%D1%82%D0%BE%D0%B0%D0%BF%D0%BF%D0%B0%D1%80%D0%B0%D1%82_%D1%81%D0%BE_%D1%81%D0%BC%D0%B5%D0%BD%D0%BD%D1%8B%D0%BC%D0%B8_%D0%BE%D0%B1%D1%8A%D0%B5%D0%BA%D1%82%D0%B8%D0%B2%D0%B0%D0%BC%D0%B8" title="Беззеркальный цифровой фотоаппарат со сменными объективами">беззеркальный фотоаппарат</a> <a class="new" href="/w/index.php?title=Samsung_NX200&amp;action=edit&amp;redlink=1" title="Samsung NX200 (страница отсутствует)">Samsung NX200</a> с матрицей в 20 мегапикселей.
</p><p>EOS 7D установил рекорд не только по разрешению, но и по количеству светочувствительных элементов для матриц формата APS-C. Этот рекорд был превзойдён фотоаппаратом <a href="/wiki/Sigma_SD1" title="Sigma SD1">Sigma SD1</a> (представлен в 2010, в продаже с июня 2011<sup class="reference" id="cite_ref-Sigma_SD1_13-0"><a href="#cite_note-Sigma_SD1-13">[13]</a></sup>), матрица которого имеет размер 24 × 16 мм и содержит три слоя светочувствительных элементов по 14,8 млн пикселей в каждом, то есть их общее число составляет примерно 44 млн.
</p><p>В фотоаппарате впервые для непрофессиональных моделей «Кэнон» применены сразу два процессора. Максимальная скорость съёмки 8 кадров/с являлась рекордной для непрофессиональных зеркальных фотоаппаратов фирмы до появления <a href="/wiki/Canon_EOS_7D_Mark_II" title="Canon EOS 7D Mark II">7D Mark II</a><sup class="reference" id="cite_ref-eos-comparison_10-1"><a href="#cite_note-eos-comparison-10">[10]</a></sup>.
</p><p>Благодаря использованию двух процессоров <a class="mw-redirect" href="/wiki/DIGIC_4" title="DIGIC 4">DIGIC 4</a> условная скорость обработки информации достигла рекордного для серии EOS значения 239 условных мегабит в секунду (5184 × 3456 пикселей × 14 бит/пиксель × 8 кадров/c). Для сравнения, выпускавшаяся в момент появления 7D профессиональная модель <a href="/wiki/Canon_EOS-1Ds_Mark_III" title="Canon EOS-1Ds Mark III">1Ds Mark III</a> обрабатывает данные со скоростью 175 Мбит/с, <a href="/wiki/Canon_EOS-1D_Mark_III" title="Canon EOS-1D Mark III">1D Mark III</a> — 168 Мбит/с. Более высокую скорость обработки информации обеспечила лишь следующая профессиональная модель, <a href="/wiki/Canon_EOS-1D_Mark_IV" title="Canon EOS-1D Mark IV">Canon EOS-1D Mark IV</a>, тоже оснащённая двумя процессорами DIGIC 4.
</p>
<h3><span id=".D0.98.D0.BD.D1.82.D0.B5.D1.80.D1.84.D0.B5.D0.B9.D1.81_.D0.B8_.D0.BD.D0.B0.D1.81.D1.82.D1.80.D0.BE.D0.B9.D0.BA.D0.B8"></span><span class="mw-headline" id="Интерфейс_и_настройки">Интерфейс и настройки</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Canon_EOS_7D&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Интерфейс и настройки»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Canon_EOS_7D&amp;action=edit&amp;section=7" title="Редактировать код раздела «Интерфейс и настройки»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Canon_EOS_7D_top_02.jpg"><img class="mw-file-element" data-file-height="800" data-file-width="800" decoding="async" height="140" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/62/Canon_EOS_7D_top_02.jpg/140px-Canon_EOS_7D_top_02.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/62/Canon_EOS_7D_top_02.jpg/210px-Canon_EOS_7D_top_02.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/62/Canon_EOS_7D_top_02.jpg/280px-Canon_EOS_7D_top_02.jpg 2x" width="140"/></a><figcaption>B — режим для съёмки с длительной выдержкой</figcaption></figure>
<p>Фотоаппарат позволяет редактировать информацию об авторе непосредственно в интерфейсе фотоаппарата. У более ранних моделей это возможно только при подключении к компьютеру<sup class="reference" id="cite_ref-ir-overview_14-0"><a href="#cite_note-ir-overview-14">[14]</a></sup>. Впервые для Canon EOS экспозиция может быть скорректирована в диапазоне ±5 ступеней<sup class="reference" id="cite_ref-compensation_15-0"><a href="#cite_note-compensation-15">[15]</a></sup>.
</p><p>Как и у 5D Mark II, у новой модели имеется отдельный режим B (bulb, «балб») для <a href="/wiki/%D0%92%D1%8B%D0%B4%D0%B5%D1%80%D0%B6%D0%BA%D0%B0_%D0%BE%D1%82_%D1%80%D1%83%D0%BA%D0%B8" title="Выдержка от руки">съёмки с длинной выдержкой</a>.
</p>
<h2><span id=".D0.9A.D0.BE.D0.BD.D0.BA.D1.83.D1.80.D0.B5.D0.BD.D1.82.D1.8B"></span><span class="mw-headline" id="Конкуренты">Конкуренты</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Canon_EOS_7D&amp;veaction=edit&amp;section=8" title="Редактировать раздел «Конкуренты»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Canon_EOS_7D&amp;action=edit&amp;section=8" title="Редактировать код раздела «Конкуренты»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Nikon_D300s_-_Front_Mk2.jpg"><img class="mw-file-element" data-file-height="1753" data-file-width="2341" decoding="async" height="180" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/a3/Nikon_D300s_-_Front_Mk2.jpg/240px-Nikon_D300s_-_Front_Mk2.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/a3/Nikon_D300s_-_Front_Mk2.jpg/360px-Nikon_D300s_-_Front_Mk2.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/a3/Nikon_D300s_-_Front_Mk2.jpg/480px-Nikon_D300s_-_Front_Mk2.jpg 2x" width="240"/></a><figcaption><a href="/wiki/Nikon_D300s" title="Nikon D300s">Nikon D300s</a> — основной конкурент Canon EOS 7D</figcaption></figure>
<p>Единственным прямым конкурентом Canon EOS 7D с его рекомендуемой розничной стоимостью в США 1700 долларов<sup class="reference" id="cite_ref-Canon_USA_press-release_2-5"><a href="#cite_note-Canon_USA_press-release-2">[2]</a></sup> являлся представленный месяцем ранее <a href="/wiki/Nikon_D300s" title="Nikon D300s">Nikon D300s</a> (рекомендуемая стоимость в США в начале сентября 2009 года — 1800 долларов<sup class="reference" id="cite_ref-dpreview_20_16-0"><a href="#cite_note-dpreview_20-16">[16]</a></sup><sup class="reference" id="cite_ref-Nikon_D300S_17-0"><a href="#cite_note-Nikon_D300S-17">[17]</a></sup>).
</p><p>Наиболее близкие по функциональности модели конкурентов 2009 года уступали в оснащённости и скорости съёмки, обладали матрицами с меньшим разрешением и были заметно дешевле: это <a href="/wiki/Olympus_E-3" title="Olympus E-3">Olympus E-3</a> (1300 долл.<sup class="reference" id="cite_ref-Olympus_E-3_18-0"><a href="#cite_note-Olympus_E-3-18">[18]</a></sup>) и впоследствии пришедший ему на смену <a href="/wiki/Olympus_E-5" title="Olympus E-5">Olympus E-5</a>, а также <a href="/wiki/Pentax_K-7" title="Pentax K-7">Pentax K-7</a> (1300 долл.<sup class="reference" id="cite_ref-Pentax_K-7_19-0"><a href="#cite_note-Pentax_K-7-19">[19]</a></sup>) и его преемники: <a href="/wiki/Pentax_K-5" title="Pentax K-5">Pentax K-5</a> и <a href="/wiki/Pentax_K-5_II" title="Pentax K-5 II">Pentax K-5 II</a>.
</p><p>Спустя почти два года после анонса 7D, в августе 2011 года, был представлен, а в октябре поступил в продажу <a href="/wiki/Sony_SLT-A77" title="Sony SLT-A77">Sony SLT-A77</a> — фотоаппарат с разрешением 24 млн пикселей и скоростью съёмки до 12 кадров в секунду. По этим своим характеристикам он выделялся в то время на фоне всех прочих непрофессиональных фотоаппаратов, в том числе и EOS 7D. Позднее эта модель была заменена на ещё более оснащённый <a class="mw-redirect" href="/wiki/Sony_SLT-A77_II" title="Sony SLT-A77 II">Sony SLT-A77 II</a>.
</p>
<h2><span id=".D0.94.D0.BE.D1.81.D1.82.D0.BE.D0.B8.D0.BD.D1.81.D1.82.D0.B2.D0.B0_.D0.B8_.D0.BD.D0.B5.D0.B4.D0.BE.D1.81.D1.82.D0.B0.D1.82.D0.BA.D0.B8"></span><span class="mw-headline" id="Достоинства_и_недостатки">Достоинства и недостатки</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Canon_EOS_7D&amp;veaction=edit&amp;section=9" title="Редактировать раздел «Достоинства и недостатки»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Canon_EOS_7D&amp;action=edit&amp;section=9" title="Редактировать код раздела «Достоинства и недостатки»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>По сравнению с наиболее оснащёнными фотоаппаратами других марок, имеющими матрицы формата <a href="/wiki/APS-C" title="APS-C">APS-C</a> и выпущенными до октября 2011, Canon EOS 7D обладает следующими достоинствами<sup class="reference" id="cite_ref-ir-overview_14-1"><a href="#cite_note-ir-overview-14">[14]</a></sup><sup class="reference" id="cite_ref-dpreview_30_20-0"><a href="#cite_note-dpreview_30-20">[20]</a></sup><sup class="reference" id="cite_ref-ir-comparison_21-0"><a href="#cite_note-ir-comparison-21">[21]</a></sup>:
</p>
<ul><li>18 млн пикселей против максимум 16 у других моделей.</li>
<li>Наилучшая детализация и разрешение при низких значениях ISO.</li>
<li>Скорость съёмки составляет 8 кадров/c против 7 кадров/c у Nikon D300s в стандартной комплектации (D300s обеспечивает скорость 8 кадров/c при использовании батарейной ручки MB-D10).</li>
<li>Гибкая система автофокусировки.</li>
<li>Беспроводное управление внешними вспышками.</li></ul>
<p>В то же время по ряду характеристик 7D проигрывает наиболее технически оснащённым непрофессиональным моделям других марок:
</p>
<ul><li>Отсутствие системы стабилизации матрицы, в отличие от моделей Pentax, Olympus, Sony.</li>
<li>Наличие только одного разъёма для карт памяти, в отличие от моделей Nikon, Olympus.</li>
<li>Не предусмотренная производителем возможность смены фокусировочного экрана, в отличие от моделей Pentax и Olympus, а также Canon EOS 50D и 60D.</li>
<li>Отсутствие лампы подсветки автофокуса.</li></ul>
<p>Также отмечается, что фотоаппарат имеет недостоверный автоматический баланс белого при искусственном освещении и обладает тенденцией к переэкспонированию в контрастных сюжетах<sup class="reference" id="cite_ref-ir-overview_14-2"><a href="#cite_note-ir-overview-14">[14]</a></sup><sup class="reference" id="cite_ref-dpreview_30_20-1"><a href="#cite_note-dpreview_30-20">[20]</a></sup><sup class="reference" id="cite_ref-ir-comparison_21-1"><a href="#cite_note-ir-comparison-21">[21]</a></sup>.
</p><p>Кэноновская матрица с <a href="/wiki/%D0%9A%D1%80%D0%BE%D0%BF-%D1%84%D0%B0%D0%BA%D1%82%D0%BE%D1%80" title="Кроп-фактор">кроп-фактором</a> 1,6 обеспечивает меньший угол охвата при использовании одних и тех же объективов по сравнению с матрицами с кроп-фактором 1,5, которыми обладают конкурирующие модели производства Nikon, Pentax и Sony. Это обстоятельство не позволяет в полной мере использовать возможности широкоугольных объективов сторонних производителей. Например, при использовании объектива Sigma 10mm F2.8 EX DC HSM Fisheye угол зрения по диагонали кадра на фотоаппаратах Nikon составляет 180°, а на фотоаппаратах Canon он равен 167°<sup class="reference" id="cite_ref-Sigma_chart_22-0"><a href="#cite_note-Sigma_chart-22">[22]</a></sup>.
</p>
<h2><span id=".D0.98.D0.B7.D0.B2.D0.B5.D1.81.D1.82.D0.BD.D1.8B.D0.B5_.D0.BF.D1.80.D0.BE.D0.B1.D0.BB.D0.B5.D0.BC.D1.8B_.D1.81_.D1.84.D0.BE.D1.82.D0.BE.D0.B0.D0.BF.D0.BF.D0.B0.D1.80.D0.B0.D1.82.D0.BE.D0.BC"></span><span class="mw-headline" id="Известные_проблемы_с_фотоаппаратом">Известные проблемы с фотоаппаратом</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Canon_EOS_7D&amp;veaction=edit&amp;section=10" title="Редактировать раздел «Известные проблемы с фотоаппаратом»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Canon_EOS_7D&amp;action=edit&amp;section=10" title="Редактировать код раздела «Известные проблемы с фотоаппаратом»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>За время производства и продажи Canon EOS 7D у этой модели были обнаружены различные недостатки, причиной которых, как правило, являлась некорректная работа <a href="/wiki/%D0%92%D1%81%D1%82%D1%80%D0%BE%D0%B5%D0%BD%D0%BD%D0%BE%D0%B5_%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%BD%D0%BE%D0%B5_%D0%BE%D0%B1%D0%B5%D1%81%D0%BF%D0%B5%D1%87%D0%B5%D0%BD%D0%B8%D0%B5" title="Встроенное программное обеспечение">микропрограммы</a>. Большинство из них было устранено в новых версиях прошивки.
</p><p><b>Отказы электроники и механики:</b>
</p><p>Error 40<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup> (вызывается либо накоплением конденсата на нижней плате DC-DC преобразователя и схемы зарядки конденсатора встроенной вспышки при резком перепаде температур или при закорачивании дорожек выкручивающимися от вибрации винтами крепления этой же платы, проблема не имеет системного характера, поскольку в сравнении с объемом выпуска камеры известные случаи относительно редки), перегрев матрицы при длительной работе LiveView или видеосъемке (впрочем, эта проблема типична для цифровых зеркальных фотокамер, матрицы которых не рассчитаны на длительную непрерывную работу для принудительного дистанцирования их от видеокамер)..
</p><p>C 2010г. производство камеры решили удешевить, и вместо металлического стержня колеса режимов начали ставить пластмассовый. Это приводило к эпизодическим поломкам при попытке переключения режимов съемки на холоде.
</p><p><b>Неестественные цвета видеороликов:</b>
</p><p>В ранних версиях микропрограммы имелся дефект, при некоторых условиях приводящий к неестественному цвету сделанных камерой видеороликов. Проблема была исправлена в версии 1.0.9<sup class="reference" id="cite_ref-dp109_24-0"><a href="#cite_note-dp109-24">[24]</a></sup><sup class="reference" id="cite_ref-firmware_25-0"><a href="#cite_note-firmware-25">[25]</a></sup>.
</p><p><b>Следы предыдущего изображения на кадре:</b>
</p><p>Вскоре после начала продаж 7D был обнаружен эффект проявления следов предыдущего изображения, возникающий в режиме непрерывной съёмки. Эффект не был заметен при нормальных экспозициях и проявлялся, например, в случае коррекции экспозиции при постобработке.
</p><p>Дефект был признан производителем<sup class="reference" id="cite_ref-dp_ghost_26-0"><a href="#cite_note-dp_ghost-26">[26]</a></sup> и устранён в микропрограмме начиная с версии 1.1.0<sup class="reference" id="cite_ref-firmware_25-1"><a href="#cite_note-firmware-25">[25]</a></sup><sup class="reference" id="cite_ref-dp110_27-0"><a href="#cite_note-dp110-27">[27]</a></sup>.
</p><p><b>Неполноценная работа датчиков фокусировки:</b>
</p><p>В феврале 2011 года компания сообщила о том, что с некоторыми старыми объективами не работают в полной мере периферийные датчики фокусировки: вместо того, чтобы работать как крестообразные — реагировать на контрастные объекты как в вертикальном, так и в горизонтальном направлении, — они работают как линейные<sup class="reference" id="cite_ref-cross_issue_28-0"><a href="#cite_note-cross_issue-28">[28]</a></sup>.
</p><p>Проблема затронула также некоторые современные объективы <a href="/wiki/Tamron" title="Tamron">Tamron</a>. Этот недостаток присущ не только 7D, но и моделям <a href="/wiki/Canon_EOS_40D" title="Canon EOS 40D">40D</a>, <a href="/wiki/Canon_EOS_50D" title="Canon EOS 50D">50D</a> и <a href="/wiki/Canon_EOS_60D" title="Canon EOS 60D">60D</a><sup class="reference" id="cite_ref-cr_tamron_29-0"><a href="#cite_note-cr_tamron-29">[29]</a></sup>.
</p>

<p>Canon EOS 7D предлагался в трёх основных вариантах комплектации:
</p>
<ul><li>без объектива,</li>
<li>с объективом EF-S 15-85mm f/3.5-5.6 IS USM,</li>
<li>с объективом EF-S 18-135mm f/3.5-5.6 IS<sup class="reference" id="cite_ref-dpreview_p6_11-1"><a href="#cite_note-dpreview_p6-11">[11]</a></sup><sup class="reference" id="cite_ref-de_pr_30-0"><a href="#cite_note-de_pr-30">[30]</a></sup>.</li></ul>
<p>В зависимости от страны были возможны иные варианты комплектации. Так, в США был предложен вариант с объективом EF 28-135mm f/3.5-5.6 IS USM<sup class="reference" id="cite_ref-Canon_USA_press-release_2-6"><a href="#cite_note-Canon_USA_press-release-2">[2]</a></sup>, а в Японии — с объективом EF-S 18-200mm f/3.5-5.6 IS<sup class="reference" id="cite_ref-jp_main_31-0"><a href="#cite_note-jp_main-31">[31]</a></sup>.

<h3><span class="mw-headline" id="Canon_EOS_7DSV">Canon EOS 7DSV</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Canon_EOS_7D&amp;veaction=edit&amp;section=18" title="Редактировать раздел «Canon EOS 7DSV»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Canon_EOS_7D&amp;action=edit&amp;section=18" title="Редактировать код раздела «Canon EOS 7DSV»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>17 августа 2010 года компания «Кэнон» представила Canon EOS 7DSV — студийную версию (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Studio Version</span>) фотоаппарата. Отличий от стандартной версии две: возможность блокировки различных функций в меню фотоаппарата и возможность работы со сканером <a class="mw-redirect" href="/wiki/%D0%A8%D1%82%D1%80%D0%B8%D1%85%D0%BA%D0%BE%D0%B4" title="Штрихкод">штрихкодов</a> с помощью решения Canon Barcode Solution.
</p><p>Функция блокировки позволяет администратору фотоаппарата отключать некоторые возможности и настройки в меню для «обеспечения профессиональным студийным фотографам комфортной работы с надёжными и воспроизводимыми результатами».
</p><p>Решение Canon Barcode Solution предназначено для организации данных в процессе съёмки и обработки. При съёмке считывается штрихкод, соответствующий конкретному клиенту, и соответствующая информация записывается в <a href="/wiki/EXIF" title="EXIF">EXIF</a>. Для использования Barcode Solution требуется беспроводной передатчик <a class="new" href="/w/index.php?title=WFT-E5&amp;action=edit&amp;redlink=1" title="WFT-E5 (страница отсутствует)">WFT-E5</a>, к которому подключается приобретаемое отдельно устройство для чтения штрихкодов. Для работы модуля WFT-E5 со штрихкодами требуется обновление его микропрограммы в сервис-центре, при этом теряется возможность записи данных <a href="/wiki/GPS" title="GPS">GPS</a>.
</p><p>Официальная цена Canon EOS 7D Studio Version составляет 1829 долларов США, что на 130 долларов больше стоимости обычного EOS 7D. Комплект с модулем WFT-E5, готовым к работе со сканером штрихкодов, предлагается за 2599 долларов<sup class="reference" id="cite_ref-7dsv_33-0"><a href="#cite_note-7dsv-33">[33]</a></sup>.
</p>

<p>В декабре 2010 «Кэнон» предложила пользователям Canon EOS 7D и <a href="/wiki/Canon_EOS_5D_Mark_II" title="Canon EOS 5D Mark II">Canon EOS 5D Mark II</a> платную услугу по замене колеса выбора режима съёмки. Новое колесо отличается расположенной в центре кнопкой, которую необходимо нажать и удерживать для смены режима, как это реализовано в представленной в августе 2010 года модели <a href="/wiki/Canon_EOS_60D" title="Canon EOS 60D">Canon EOS 60D</a><sup class="reference" id="cite_ref-dial_34-0"><a href="#cite_note-dial-34">[34]</a></sup> и позднее в модели <a href="/wiki/Canon_EOS_5D_Mark_III" title="Canon EOS 5D Mark III">Canon EOS 5D Mark III</a>.

<h2><span id=".D0.9C.D0.B8.D0.BA.D1.80.D0.BE.D0.BF.D1.80.D0.BE.D0.B3.D1.80.D0.B0.D0.BC.D0.BC.D0.B0"></span><span class="mw-headline" id="Микропрограмма">Микропрограмма</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Canon_EOS_7D&amp;veaction=edit&amp;section=20" title="Редактировать раздел «Микропрограмма»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Canon_EOS_7D&amp;action=edit&amp;section=20" title="Редактировать код раздела «Микропрограмма»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%D0%92%D1%81%D1%82%D1%80%D0%BE%D0%B5%D0%BD%D0%BD%D0%BE%D0%B5_%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%BD%D0%BE%D0%B5_%D0%BE%D0%B1%D0%B5%D1%81%D0%BF%D0%B5%D1%87%D0%B5%D0%BD%D0%B8%D0%B5" title="Встроенное программное обеспечение">Микропрограмма</a> может быть обновлена пользователем самостоятельно с использованием персональных компьютеров под управлением <a class="mw-redirect" href="/wiki/Mac_OS_X" title="Mac OS X">Mac OS X</a> и <a class="mw-redirect" href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a> (версии XP и выше). Первые Canon EOS 7D поставлялись с микропрограммой версии 1.0.7<sup class="reference" id="cite_ref-dslrfilm_35-0"><a href="#cite_note-dslrfilm-35">[35]</a></sup>. Впоследствии компания выпускала обновления, в апреле 2011 года вышла версия 1.2.5, которая оставалась актуальной более года.
</p>
<h3><span id=".D0.92.D0.B5.D1.80.D1.81.D0.B8.D1.8F_2.0"></span><span class="mw-headline" id="Версия_2.0">Версия 2.0</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Canon_EOS_7D&amp;veaction=edit&amp;section=21" title="Редактировать раздел «Версия 2.0»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Canon_EOS_7D&amp;action=edit&amp;section=21" title="Редактировать код раздела «Версия 2.0»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>28 июня 2012 года была анонсирована, а 7 августа опубликована микропрограмма версии 2.0.0, включающая в себя существенные изменения по сравнению с более ранними версиями.
</p><p>Удалось заметно увеличить количество снимков в серии при скоростной съёмке в формате RAW: с 15 до 25. Добавлена поддержка <a class="mw-redirect" href="/wiki/GPS-%D0%BF%D1%80%D0%B8%D1%91%D0%BC%D0%BD%D0%B8%D0%BA" title="GPS-приёмник">GPS-приёмника</a> GP-E2, который подключается к фотоаппарату через USB-разъём, в меню появилась поддержка часовых зон. Появилась возможность ручной настройки уровня звука при съёмке видео и выбора максимального значения ISO (в диапазоне от 400 до 6400 вместо единственного значения 3200 ранее) при съёмке с автоматической установкой чувствительности<sup class="reference" id="cite_ref-firmware_2A_36-0"><a href="#cite_note-firmware_2A-36">[36]</a></sup>.
</p><p>Также обновление добавило в меню возможности, впервые появившиеся у <a href="/wiki/Canon_EOS_60D" title="Canon EOS 60D">Canon EOS 60D</a>: преобразование RAW-файлов в JPEG (только для файлов, снятых в максимальном разрешении<sup class="reference" id="cite_ref-firmware_announce_37-0"><a href="#cite_note-firmware_announce-37">[37]</a></sup>), уменьшение разрешения JPEG-файлов. Стало возможным назначать каждой фотографии рейтинг от нуля до пяти, а затем использовать его для просмотра снимков в фотоаппарате и для функции <a href="/wiki/%D0%A1%D0%BB%D0%B0%D0%B9%D0%B4-%D1%88%D0%BE%D1%83" title="Слайд-шоу">слайд-шоу</a>. При просмотре сделанных снимков появился быстрый доступ к многочисленным функциям с помощью кнопки Q (<i>Quick Control</i> — быстрое управление).
</p><p>В новой версии пользователь может управлять именами создаваемых файлов: если ранее все названия начинались с префикса IMG_, за которыми следовал порядковый номер от 0000 до 9999, то теперь можно задать четыре других символа (цифры, латинские буквы, знак подчёркивания) либо задать первые три символа, а в качестве четвёртого будет добавляться буква, соответствующая разрешению фотоснимка: S, M или L.
</p><p>Ещё одним улучшением стала выросшая скорость прокрутки увеличенной части изображения<sup class="reference" id="cite_ref-firmware_2B_38-0"><a href="#cite_note-firmware_2B-38">[38]</a></sup>.
</p><p>При обновлении до версии 2.0.0 или выше восстановление прежней версии (1.2.5 или ниже) невозможно.
</p><p>С ноября 2016 года актуальной является версия 2.0.6 <a class="external autonumber" href="https://www.canon.ru/support/consumer_products/products/cameras/digital_slr/eos_7d.html?type=firmware" rel="nofollow">[1]</a> <a class="external text" href="https://web.archive.org/web/20210124150053/https://www.canon.ru/support/consumer_products/products/cameras/digital_slr/eos_7d.html?type=firmware" rel="nofollow">Архивная копия</a> от 24 января 2021 на <a href="/wiki/Wayback_Machine" title="Wayback Machine">Wayback Machine</a>.
</p>

<h2><span id=".D0.9F.D1.80.D0.BE.D0.B4.D0.B0.D0.B6.D0.B8"></span><span class="mw-headline" id="Продажи">Продажи</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Canon_EOS_7D&amp;veaction=edit&amp;section=23" title="Редактировать раздел «Продажи»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Canon_EOS_7D&amp;action=edit&amp;section=23" title="Редактировать код раздела «Продажи»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Компания «Кэнон» не раскрывает информацию об объёмах производства и продаж отдельных моделей. Согласно данным сайта bcnranking.jp, в 2009 году EOS 7D занял 17-е место (1,6 %) по количеству проданных в Японии фотоаппаратов со сменными объективами при том, что появился в продаже лишь осенью<sup class="reference" id="cite_ref-bcn2009_47-0"><a href="#cite_note-bcn2009-47">[47]</a></sup>. В первой половине 2010 года фотоаппарат занял 9-е место (3,5 %) по продажам в Японии среди фотоаппаратов со сменным объективами и 7-е место — среди зеркальных фотоаппаратов<sup class="reference" id="cite_ref-bcn2010-1_48-0"><a href="#cite_note-bcn2010-1-48">[48]</a></sup>.
</p><p>По итогам 2010 года — 10-е место (3,1 %) в Японии среди фотоаппаратов со сменными объективами и 7-е место среди зеркальных фотоаппаратов<sup class="reference" id="cite_ref-bcn2010_49-0"><a href="#cite_note-bcn2010-49">[49]</a></sup>. По итогам 2011 года — 16-е место (2,4 %) и 10-е место, соответственно<sup class="reference" id="cite_ref-bcn2011_50-0"><a href="#cite_note-bcn2011-50">[50]</a></sup>. В 2012 году 7D занял 8-е место с долей 4,3 % среди зеркальных камер<sup class="reference" id="cite_ref-bcn2012_51-0"><a href="#cite_note-bcn2012-51">[51]</a></sup>, в 2013 году — 13-е место с долей 2,8 %<sup class="reference" id="cite_ref-bcn2013_52-0"><a href="#cite_note-bcn2013-52">[52]</a></sup> (беззеркальные модели считались отдельно).
</p><p>И в 2010-м, и в 2011-м, и в 2012-м году EOS 7D опережал по продажам в Японии и <a href="/wiki/Nikon_D300s" title="Nikon D300s">Nikon D300s</a>, и <a href="/wiki/Canon_EOS_5D_Mark_II" title="Canon EOS 5D Mark II">Canon EOS 5D Mark II</a> (в 2012-м году — даже с учётом объёмов продаж новой модели <a href="/wiki/Canon_EOS_5D_Mark_III" title="Canon EOS 5D Mark III">5D Mark III</a>), и <a href="/wiki/Canon_EOS_50D" title="Canon EOS 50D">Canon EOS 50D</a><sup class="reference" id="cite_ref-bcn2010_49-1"><a href="#cite_note-bcn2010-49">[49]</a></sup><sup class="reference" id="cite_ref-bcn2011_50-1"><a href="#cite_note-bcn2011-50">[50]</a></sup><sup class="reference" id="cite_ref-bcn2012_51-1"><a href="#cite_note-bcn2012-51">[51]</a></sup>. В 2013 году 7D уступил близкой по цене полнокадровой модели <a href="/wiki/Canon_EOS_6D" title="Canon EOS 6D">EOS 6D</a>, но заметно опередил 5D Mark III<sup class="reference" id="cite_ref-bcn2013_52-1"><a href="#cite_note-bcn2013-52">[52]</a></sup>.
</p>

<p>Canon EOS 7D стал лауреатом премий <a class="new" href="/w/index.php?title=TIPA&amp;action=edit&amp;redlink=1" title="TIPA (страница отсутствует)">TIPA</a> (Technical Image Press Association) и <a href="/wiki/European_Imaging_and_Sound_Association" title="European Imaging and Sound Association">EISA</a> (European Imaging and Sound Association)<sup class="reference" id="cite_ref-jp_main_31-1"><a href="#cite_note-jp_main-31">[31]</a></sup>:
</p>
<ul><li>«лучший цифровой зеркальный фотоаппарат для специалистов» (TIPA Best DSLR Expert, 2010)<sup class="reference" id="cite_ref-tipa_53-0"><a href="#cite_note-tipa-53">[53]</a></sup>,</li>

</p>
<ul><li>PC Magazine Readers' Choice Award (2010),</li>
<li>Professional Photographer Magazine Hot One Award в категории цифровых зеркальных фотоаппаратов стоимостью от 1000 до 3000 долларов (2010).</li></ul>
