<p><b>Джон Бейзилон</b> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">John Basilone</span>; <a href="/wiki/4_%D0%BD%D0%BE%D1%8F%D0%B1%D1%80%D1%8F" title="4 ноября">4 ноября</a> <a class="mw-redirect" href="/wiki/1916" title="1916">1916</a>, <a href="/wiki/%D0%91%D1%83%D1%84%D1%84%D0%B0%D0%BB%D0%BE" title="Буффало">Буффало</a>, штат <a href="/wiki/%D0%9D%D1%8C%D1%8E-%D0%99%D0%BE%D1%80%D0%BA_(%D1%88%D1%82%D0%B0%D1%82)" title="Нью-Йорк (штат)">Нью-Йорк</a> — <a href="/wiki/19_%D1%84%D0%B5%D0%B2%D1%80%D0%B0%D0%BB%D1%8F" title="19 февраля">19 февраля</a> <a class="mw-redirect" href="/wiki/1945" title="1945">1945</a>, <a class="mw-redirect" href="/wiki/%D0%98%D0%B2%D0%BE%D0%B4%D0%B7%D0%B8%D0%BC%D0%B0" title="Иводзима">Иводзима</a>) — <a href="/wiki/%D0%9A%D0%BE%D0%BC%D0%B5%D0%BD%D0%B4%D0%BE%D1%80-%D1%81%D0%B5%D1%80%D0%B6%D0%B0%D0%BD%D1%82" title="Комендор-сержант">комендор-сержант</a> <a href="/wiki/%D0%9A%D0%BE%D1%80%D0%BF%D1%83%D1%81_%D0%BC%D0%BE%D1%80%D1%81%D0%BA%D0%BE%D0%B9_%D0%BF%D0%B5%D1%85%D0%BE%D1%82%D1%8B_%D0%A1%D0%A8%D0%90" title="Корпус морской пехоты США">морской пехоты США</a>, кавалер <a href="/wiki/%D0%9C%D0%B5%D0%B4%D0%B0%D0%BB%D1%8C_%D0%9F%D0%BE%D1%87%D1%91%D1%82%D0%B0_(%D0%A1%D0%A8%D0%90)" title="Медаль Почёта (США)">Медали Почёта</a>, полученной за действия в <a href="/wiki/%D0%91%D0%B8%D1%82%D0%B2%D0%B0_%D0%B7%D0%B0_%D0%93%D1%83%D0%B0%D0%B4%D0%B0%D0%BB%D0%BA%D0%B0%D0%BD%D0%B0%D0%BB" title="Битва за Гуадалканал">битве за Гуадалканал</a> во время <a href="/wiki/%D0%92%D1%82%D0%BE%D1%80%D0%B0%D1%8F_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Вторая мировая война">Второй мировой войны</a>. Единственный морской пехотинец в этой войне, получивший Медаль Почёта и <a class="mw-redirect" href="/wiki/%D0%92%D0%BE%D0%B5%D0%BD%D0%BD%D0%BE-%D0%BC%D0%BE%D1%80%D1%81%D0%BA%D0%BE%D0%B9_%D0%BA%D1%80%D0%B5%D1%81%D1%82_(%D0%A1%D0%A8%D0%90)" title="Военно-морской крест (США)">Военно-морской крест</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>.
</p><p>Джон служил три года в <a href="/wiki/%D0%90%D1%80%D0%BC%D0%B8%D1%8F_%D0%A1%D0%A8%D0%90" title="Армия США">армии Соединённых Штатов</a> на <a href="/wiki/%D0%A4%D0%B8%D0%BB%D0%B8%D0%BF%D0%BF%D0%B8%D0%BD%D1%8B" title="Филиппины">Филиппинах</a>, в <a href="/wiki/1940_%D0%B3%D0%BE%D0%B4" title="1940 год">1940 году</a> вступил в корпус морской пехоты. После прохождения курса подготовки часть Бейзилона была развёрнута на <a class="mw-redirect" href="/wiki/%D0%92%D0%BE%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B1%D0%B0%D0%B7%D0%B0_%D0%93%D1%83%D0%B0%D0%BD%D1%82%D0%B0%D0%BD%D0%B0%D0%BC%D0%BE" title="Военная база Гуантанамо">базе Гуантанамо</a> (<a href="/wiki/%D0%9A%D1%83%D0%B1%D0%B0_(%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2)" title="Куба (остров)">Куба</a>), затем переброшена на <a href="/wiki/%D0%A1%D0%BE%D0%BB%D0%BE%D0%BC%D0%BE%D0%BD%D0%BE%D0%B2%D1%8B_%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2%D0%B0_(%D0%B0%D1%80%D1%85%D0%B8%D0%BF%D0%B5%D0%BB%D0%B0%D0%B3)" title="Соломоновы острова (архипелаг)">Соломоновы острова</a> и, в конечном счёте, — на <a href="/wiki/%D0%93%D1%83%D0%B0%D0%B4%D0%B0%D0%BB%D0%BA%D0%B0%D0%BD%D0%B0%D0%BB" title="Гуадалканал">Гуадалканал</a><sup class="reference" id="cite_ref-paradewebsite_2-0"><a href="#cite_note-paradewebsite-2">[2]</a></sup>.
</p><p>Высшей воинской награды США он удостоился за то, что в ходе двухдневных боёв сдерживал атаку отряда японских войск численностью около 3000 солдат, даже после того, как из 15 человек его подразделения в строю остались только двое. Был убит в бою в первый день <a href="/wiki/%D0%91%D0%B8%D1%82%D0%B2%D0%B0_%D0%B7%D0%B0_%D0%98%D0%B2%D0%BE%D0%B4%D0%B7%D0%B8%D0%BC%D1%83" title="Битва за Иводзиму">битвы за Иводзиму</a>, после чего был посмертно удостоен Военно-морского креста. В честь Бейзилона были названы улицы, военные объекты и <a class="mw-redirect" href="/wiki/%D0%AD%D1%81%D0%BC%D0%B8%D0%BD%D0%B5%D1%86" title="Эсминец">эсминец</a> <a href="/wiki/%D0%92%D0%BE%D0%B5%D0%BD%D0%BD%D0%BE-%D0%BC%D0%BE%D1%80%D1%81%D0%BA%D0%B8%D0%B5_%D1%81%D0%B8%D0%BB%D1%8B_%D0%A1%D0%A8%D0%90" title="Военно-морские силы США">ВМС США</a><sup class="reference" id="cite_ref-paradewebsite_2-1"><a href="#cite_note-paradewebsite-2">[2]</a></sup>.
</p>

<h2><span id=".D0.A0.D0.B0.D0.BD.D0.BD.D0.B8.D0.B5_.D0.B3.D0.BE.D0.B4.D1.8B"></span><span class="mw-headline" id="Ранние_годы">Ранние годы</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%B9%D0%B7%D0%B8%D0%BB%D0%BE%D0%BD,_%D0%94%D0%B6%D0%BE%D0%BD&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Ранние годы»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%B9%D0%B7%D0%B8%D0%BB%D0%BE%D0%BD,_%D0%94%D0%B6%D0%BE%D0%BD&amp;action=edit&amp;section=1" title="Редактировать код раздела «Ранние годы»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Бейзилон родился <a href="/wiki/4_%D0%BD%D0%BE%D1%8F%D0%B1%D1%80%D1%8F" title="4 ноября">4 ноября</a> <a href="/wiki/1916_%D0%B3%D0%BE%D0%B4" title="1916 год">1916 года</a> в <a href="/wiki/%D0%91%D1%83%D1%84%D1%84%D0%B0%D0%BB%D0%BE" title="Буффало">Буффало</a> (штат Нью-Йорк) и был шестым из десяти детей. Его отец, Сальваторе Базилоне, эмигрировал из <a href="/wiki/%D0%9D%D0%B5%D0%B0%D0%BF%D0%BE%D0%BB%D1%8C" title="Неаполь">Неаполя</a> (<a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D0%B8%D1%8F" title="Италия">Италия</a>) в <a href="/wiki/1903_%D0%B3%D0%BE%D0%B4" title="1903 год">1903 году</a> и поселился в городе <a class="extiw" href="https://en.wikipedia.org/wiki/Raritan,_New_Jersey" title="en:Raritan, New Jersey"><span title="Raritan, New Jersey — версия статьи «Реритан» на английском языке">Реритан</span></a><span class="noprint" style="font-style:normal; font-weight:normal;"> <span style="display:inline-block; font-size:80%; position:relative;"><span style="font-size:117.6%; margin-left:-0.43em; position:relative; top:0.28em;"> <small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></span><span class="link-ru metadata" style="margin-left:-0.43em; position:absolute; left:0; z-index:1; margin-top:-0.35em;"><span style="visibility:hidden; margin-right:0.099em;"> (</span><span data-interwiki-article="Raritan, New Jersey" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%A0%D0%B5%D1%80%D0%B8%D1%82%D0%B0%D0%BD&amp;action=edit&amp;redlink=1" title="Реритан (страница отсутствует)">рус.</a></span></span></span></span> (<a href="/wiki/%D0%9D%D1%8C%D1%8E-%D0%94%D0%B6%D0%B5%D1%80%D1%81%D0%B8" title="Нью-Джерси">Нью-Джерси</a>). Его мать, Дора Бендживенга (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Dora Bengivenga</span>), родилась в <a href="/wiki/1889_%D0%B3%D0%BE%D0%B4" title="1889 год">1889 году</a> и выросла в <a class="extiw" href="https://en.wikipedia.org/wiki/Manville,_New_Jersey" title="en:Manville, New Jersey"><span title="Manville, New Jersey — версия статьи «Манвилл (Нью-Джерси)» на английском языке">Манвилле</span></a><span class="noprint" style="font-style:normal; font-weight:normal;"> <span style="display:inline-block; font-size:80%; position:relative;"><span style="font-size:117.6%; margin-left:-0.43em; position:relative; top:0.28em;"> <small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></span><span class="link-ru metadata" style="margin-left:-0.43em; position:absolute; left:0; z-index:1; margin-top:-0.35em;"><span style="visibility:hidden; margin-right:0.099em;"> (</span><span data-interwiki-article="Manville, New Jersey" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%9C%D0%B0%D0%BD%D0%B2%D0%B8%D0%BB%D0%BB_(%D0%9D%D1%8C%D1%8E-%D0%94%D0%B6%D0%B5%D1%80%D1%81%D0%B8)&amp;action=edit&amp;redlink=1" title="Манвилл (Нью-Джерси) (страница отсутствует)">рус.</a></span></span></span></span> (Нью-Джерси), но её родители, Карло и Катарина, также приехали в США из Неаполя. Родители Джона познакомились в церкви и поженились через три года. Бейзилон вырос недалеко от Реритана, где учился в местной приходской школе. В возрасте пятнадцати лет он бросил учёбу<sup class="reference" id="cite_ref-Brady79-80_3-0"><a href="#cite_note-Brady79-80-3">[3]</a></sup>.
</p>
<h2><span id=".D0.92.D0.BE.D0.B5.D0.BD.D0.BD.D0.B0.D1.8F_.D1.81.D0.BB.D1.83.D0.B6.D0.B1.D0.B0"></span><span class="mw-headline" id="Военная_служба">Военная служба</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%B9%D0%B7%D0%B8%D0%BB%D0%BE%D0%BD,_%D0%94%D0%B6%D0%BE%D0%BD&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Военная служба»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%B9%D0%B7%D0%B8%D0%BB%D0%BE%D0%BD,_%D0%94%D0%B6%D0%BE%D0%BD&amp;action=edit&amp;section=2" title="Редактировать код раздела «Военная служба»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>До призыва в вооружённые силы Бейзилон работал <a href="/wiki/%D0%9A%D0%B5%D0%B4%D0%B4%D0%B8" title="Кедди">кедди</a> в местном <a href="/wiki/%D0%93%D0%BE%D0%BB%D1%8C%D1%84" title="Гольф">гольф</a>-клубе. Был призван в армию США<sup class="reference" id="cite_ref-Brady80_4-0"><a href="#cite_note-Brady80-4">[4]</a></sup> и завершил свою трёхлетнюю службу на Филиппинах, где стал чемпионом по <a href="/wiki/%D0%91%D0%BE%D0%BA%D1%81" title="Бокс">боксу</a><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>. По возвращении Джон работал водителем грузовика в городе <a class="extiw" href="https://en.wikipedia.org/wiki/Reisterstown,_Maryland" title="en:Reisterstown, Maryland"><span title="Reisterstown, Maryland — версия статьи «Рейерстерстаун» на английском языке">Рейерстерстаун</span></a><span class="noprint" style="font-style:normal; font-weight:normal;"> <span style="display:inline-block; font-size:80%; position:relative;"><span style="font-size:117.6%; margin-left:-0.43em; position:relative; top:0.28em;"> <small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></span><span class="link-ru metadata" style="margin-left:-0.43em; position:absolute; left:0; z-index:1; margin-top:-0.35em;"><span style="visibility:hidden; margin-right:0.099em;"> (</span><span data-interwiki-article="Reisterstown, Maryland" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%A0%D0%B5%D0%B9%D0%B5%D1%80%D1%81%D1%82%D0%B5%D1%80%D1%81%D1%82%D0%B0%D1%83%D0%BD&amp;action=edit&amp;redlink=1" title="Рейерстерстаун (страница отсутствует)">рус.</a></span></span></span></span> (штат <a href="/wiki/%D0%9C%D1%8D%D1%80%D0%B8%D0%BB%D0%B5%D0%BD%D0%B4" title="Мэриленд">Мэриленд</a>)<sup class="reference" id="cite_ref-whoswho_6-0"><a href="#cite_note-whoswho-6">[6]</a></sup>. Поработав водителем грузовика в течение нескольких месяцев, Бейзилон хотел вернуться в <a href="/wiki/%D0%9C%D0%B0%D0%BD%D0%B8%D0%BB%D0%B0" title="Манила">Манилу</a> и считал, что сможет попасть туда быстрее, если запишется в морскую пехоту. Джон поступил в морскую пехоту в июле <a href="/wiki/1940_%D0%B3%D0%BE%D0%B4" title="1940 год">1940 года</a> в <a href="/wiki/%D0%91%D0%B0%D0%BB%D1%82%D0%B8%D0%BC%D0%BE%D1%80" title="Балтимор">Балтиморе</a> и прошёл обучение, как обычный новобранец. Последующая подготовка проходила на базах Корпуса морской пехоты в <a class="extiw" href="https://en.wikipedia.org/wiki/Marine_Corps_Base_Quantico" title="en:Marine Corps Base Quantico"><span title="Marine Corps Base Quantico — версия статьи «Квантико (база)» на английском языке">Квантико</span></a><span class="noprint" style="font-style:normal; font-weight:normal;"> <span style="display:inline-block; font-size:80%; position:relative;"><span style="font-size:117.6%; margin-left:-0.43em; position:relative; top:0.28em;"> <small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></span><span class="link-ru metadata" style="margin-left:-0.43em; position:absolute; left:0; z-index:1; margin-top:-0.35em;"><span style="visibility:hidden; margin-right:0.099em;"> (</span><span data-interwiki-article="Marine Corps Base Quantico" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%9A%D0%B2%D0%B0%D0%BD%D1%82%D0%B8%D0%BA%D0%BE_(%D0%B1%D0%B0%D0%B7%D0%B0)&amp;action=edit&amp;redlink=1" title="Квантико (база) (страница отсутствует)">рус.</a></span></span></span></span> и <a class="extiw" href="https://en.wikipedia.org/wiki/Marine_Corps_Air_Station_New_River" title="en:Marine Corps Air Station New River"><span title="Marine Corps Air Station New River — версия статьи «Нью-Ривер (база)» на английском языке">Нью-Ривер</span></a><span class="noprint" style="font-style:normal; position:absolute; left:0; z-index:1; margin-top:-0.35em;"><span style="visibility:hidden; margin-right:0.099em;"> (</span><span data-interwiki-article="Marine Corps Air Station New River" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%9D%D1%8C%D1%8E-%D0%A0%D0%B8%D0%B2%D0%B5%D1%80_(%D0%B1%D0%B0%D0%B7%D0%B0)&amp;action=edit&amp;redlink=1" title="Нью-Ривер (база) (страница отсутствует)">рус.</a></span></span></span></span>. Позже Джон был направлен на базу Гуантанамо на Кубе для последующего назначения на место службы, а затем — Гуадалканал на <a href="/wiki/%D0%A1%D0%BE%D0%BB%D0%BE%D0%BC%D0%BE%D0%BD%D0%BE%D0%B2%D1%8B_%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2%D0%B0_(%D0%B0%D1%80%D1%85%D0%B8%D0%BF%D0%B5%D0%BB%D0%B0%D0%B3)" title="Соломоновы острова (архипелаг)">Соломоновых островах</a> в составе 1-го батальона <a class="extiw" href="https://en.wikipedia.org/wiki/7th_Marine_Regiment_(United_States)" title="en:7th Marine Regiment (United States)"><span title="7th Marine Regiment (United States) — версия статьи «7-й полк морской пехоты (США)» на английском языке">7-го полка</span></a><span class="noprint" style="font-style:normal; font-weight:normal;"> <span style="display:inline-block; font-size:80%; position:relative;"><span style="font-size:117.6%; margin-left:-0.43em; position:relative; top:0.28em;"> <small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></span><span class="link-ru metadata" style="margin-left:-0.43em; position:absolute; left:0; z-index:1; margin-top:-0.35em;"><span style="visibility:hidden; margin-right:0.099em;"> (</span><span data-interwiki-article="7th Marine Regiment (United States)" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=7-%D0%B9_%D0%BF%D0%BE%D0%BB%D0%BA_%D0%BC%D0%BE%D1%80%D1%81%D0%BA%D0%BE%D0%B9_%D0%BF%D0%B5%D1%85%D0%BE%D1%82%D1%8B_(%D0%A1%D0%A8%D0%90)&amp;action=edit&amp;redlink=1" title="7-й полк морской пехоты (США) (страница отсутствует)">рус.</a></span></span></span></span> <a class="mw-redirect" href="/wiki/1-%D1%8F_%D0%B4%D0%B8%D0%B2%D0%B8%D0%B7%D0%B8%D1%8F_%D0%BA%D0%BE%D1%80%D0%BF%D1%83%D1%81%D0%B0_%D0%BC%D0%BE%D1%80%D1%81%D0%BA%D0%BE%D0%B9_%D0%BF%D0%B5%D1%85%D0%BE%D1%82%D1%8B_(%D0%A1%D0%A8%D0%90)" title="1-я дивизия корпуса морской пехоты (США)">1-й дивизии корпуса морской пехоты</a><sup class="reference" id="cite_ref-whoswho_6-1"><a href="#cite_note-whoswho-6">[6]</a></sup>.

<h3><span id=".D0.93.D1.83.D0.B0.D0.B4.D0.B0.D0.BB.D0.BA.D0.B0.D0.BD.D0.B0.D0.BB"></span><span class="mw-headline" id="Гуадалканал">Гуадалканал</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%B9%D0%B7%D0%B8%D0%BB%D0%BE%D0%BD,_%D0%94%D0%B6%D0%BE%D0%BD&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Гуадалканал»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%B9%D0%B7%D0%B8%D0%BB%D0%BE%D0%BD,_%D0%94%D0%B6%D0%BE%D0%BD&amp;action=edit&amp;section=3" title="Редактировать код раздела «Гуадалканал»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:DeadJapaneseGuadalcanal.jpg"><img class="mw-file-element" data-file-height="1114" data-file-width="1491" decoding="async" height="187" src="//upload.wikimedia.org/wikipedia/commons/thumb/0/02/DeadJapaneseGuadalcanal.jpg/250px-DeadJapaneseGuadalcanal.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/0/02/DeadJapaneseGuadalcanal.jpg/375px-DeadJapaneseGuadalcanal.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/0/02/DeadJapaneseGuadalcanal.jpg/500px-DeadJapaneseGuadalcanal.jpg 2x" width="250"/></a><figcaption>Тела японских солдат после боя на <a class="mw-redirect" href="/wiki/%D0%93%D1%83%D0%B0%D0%B4%D0%B0%D0%BB%D0%BA%D0%B0%D0%BD%D0%B0%D0%BB_(%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2)" title="Гуадалканал (остров)">Гуадалканале</a></figcaption></figure>
<p>На острове Гуадалканал морские пехотинцы дали Джону прозвище «Манильский Джон» (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Manila John</span>) из-за того, что он много рассказывал о своей службе на Филиппинах<sup class="reference" id="cite_ref-whoswho_6-2"><a href="#cite_note-whoswho-6">[6]</a></sup>. Во время <a href="/wiki/%D0%91%D0%B8%D1%82%D0%B2%D0%B0_%D0%B7%D0%B0_%D0%A5%D0%B5%D0%BD%D0%B4%D0%B5%D1%80%D1%81%D0%BE%D0%BD-%D0%A4%D0%B8%D0%BB%D0%B4" title="Битва за Хендерсон-Филд">боя за аэродром Хендерсон-филд</a> подразделение Бейзилона подверглось атаке примерно 3000 японцев из <a class="extiw" href="https://en.wikipedia.org/wiki/2nd_Division_(Imperial_Japanese_Army)" title="en:2nd Division (Imperial Japanese Army)"><span title="2nd Division (Imperial Japanese Army) — версия статьи «2-я дивизия Япония» на английском языке">Сэндайской дивизии</span></a><span class="noprint" style="font-style:normal; font-weight:normal;"> <span style="display:inline-block; font-size:80%; position:relative;"><span style="font-size:117.6%; margin-left:-0.43em; position:relative; top:0.28em;"> <small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></span><span class="link-ru metadata" style="margin-left:-0.43em; position:absolute; left:0; z-index:1; margin-top:-0.35em;"><span style="visibility:hidden; margin-right:0.099em;"> (</span><span data-interwiki-article="2nd Division (Imperial Japanese Army)" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=2-%D1%8F_%D0%B4%D0%B8%D0%B2%D0%B8%D0%B7%D0%B8%D1%8F_%D0%AF%D0%BF%D0%BE%D0%BD%D0%B8%D1%8F&amp;action=edit&amp;redlink=1" title="2-я дивизия Япония (страница отсутствует)">рус.</a></span></span></span></span>. <a href="/wiki/24_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="24 октября">24 октября</a> <a href="/wiki/1942_%D0%B3%D0%BE%D0%B4" title="1942 год">1942 года</a> японские войска начали фронтальные атаки с использованием <a href="/wiki/%D0%9F%D1%83%D0%BB%D0%B5%D0%BC%D1%91%D1%82" title="Пулемёт">пулемётов</a>, гранат и <a href="/wiki/%D0%9C%D0%B8%D0%BD%D0%BE%D0%BC%D1%91%D1%82" title="Миномёт">миномётов</a> против американских тяжёлых пулемётов. Бейзилон командовал двумя пулемётными расчётами, которые сражались в течение следующих двух дней, пока в строю не остались только Бейзилон и двое других морских пехотинцев<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>. Установив дополнительный пулемёт, он вёл непрерывный огонь по наступающим японским войскам. Затем Джон отремонтировал и укомплектовал другой пулемёт и удерживал оборонительную линию до прибытия подкреплений. Когда ещё шёл бой, боеприпасов стало критически мало. Так как линии снабжения были отрезаны, Бейзилон, прорываясь через вражескую территорию, пополнял запасы патронов своих расчётов. К концу боя японский полк был практически уничтожен. За действия во время этого боя Бейзилон получил высшую военную награду США — <a href="/wiki/%D0%9C%D0%B5%D0%B4%D0%B0%D0%BB%D1%8C_%D0%9F%D0%BE%D1%87%D1%91%D1%82%D0%B0_(%D0%A1%D0%A8%D0%90)" title="Медаль Почёта (США)">Медаль Почёта</a><sup class="reference" id="cite_ref-AMOHW_9-0"><a href="#cite_note-AMOHW-9">[9]</a></sup>.
</p><p>Рядовой первого класса Нэш У. Филлипс, из <a href="/wiki/%D0%A4%D0%B5%D0%B9%D0%B5%D1%82%D0%B2%D0%B8%D0%BB%D0%BB_(%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%9A%D0%B0%D1%80%D0%BE%D0%BB%D0%B8%D0%BD%D0%B0)" title="Фейетвилл (Северная Каролина)">Файетсвилла</a> (<a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%9A%D0%B0%D1%80%D0%BE%D0%BB%D0%B8%D0%BD%D0%B0" title="Северная Каролина">Северная Каролина</a>) позже так отзывался об этом бое:

<h3><span id=".D0.9F.D0.BE.D0.B5.D0.B7.D0.B4.D0.BA.D0.B8_.D0.BF.D0.BE_.D0.A1.D0.A8.D0.90_.D0.B8_.D0.B6.D0.B5.D0.BD.D0.B8.D1.82.D1.8C.D0.B1.D0.B0"></span><span class="mw-headline" id="Поездки_по_США_и_женитьба">Поездки по США и женитьба</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%B9%D0%B7%D0%B8%D0%BB%D0%BE%D0%BD,_%D0%94%D0%B6%D0%BE%D0%BD&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Поездки по США и женитьба»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%B9%D0%B7%D0%B8%D0%BB%D0%BE%D0%BD,_%D0%94%D0%B6%D0%BE%D0%BD&amp;action=edit&amp;section=4" title="Редактировать код раздела «Поездки по США и женитьба»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>После получения Медали Почёта Джон вернулся в Соединённые Штаты и принял участие в туре, призванном привлечь деньги для военного займа. Приезд Бейзилона получил широкую огласку и в его родном городе, когда он туда вернулся, состоялся парад в его честь. Парад прошёл в воскресенье, <a href="/wiki/19_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="19 сентября">19 сентября</a> <a href="/wiki/1943_%D0%B3%D0%BE%D0%B4" title="1943 год">1943 года</a>, и собрал много людей (в том числе политиков, знаменитостей и национальную прессу). О параде было упомянуто в журналах «<a href="/wiki/Life_(%D0%B6%D1%83%D1%80%D0%BD%D0%B0%D0%BB)" title="Life (журнал)">Life</a>» и «Fox Movietone News»<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>. После парада Бейзилон продолжил поездку по стране для сбора средств на военные нужды и, в конце концов, стал знаменитостью. Хотя Джон был благодарен за возможность стать знаменитым, он чувствовал себя не в своей тарелке и просил, чтобы его вернули на фронт. Корпус морской пехоты отказал в просьбе, мотивируя тем, что в Джоне больше нуждаются на «внутреннем фронте». Через год Бейзилону предложили увольнение со службы, но он отказался и стал инструктором новобранцев. Джон снова подал рапорт об отправке на фронт, и на этот раз его просьба была удовлетворена. Бейзилон уехал <a href="/wiki/27_%D0%B4%D0%B5%D0%BA%D0%B0%D0%B1%D1%80%D1%8F" title="27 декабря">27 декабря</a> <a href="/wiki/1943_%D0%B3%D0%BE%D0%B4" title="1943 год">1943 года</a> в <a class="extiw" href="https://en.wikipedia.org/wiki/Marine_Corps_Base_Camp_Pendleton" title="en:Marine Corps Base Camp Pendleton"><span title="Marine Corps Base Camp Pendleton — версия статьи «Кэмп-Пендлтон (база)» на английском языке">Кэмп-Пендлтон</span></a><span class="noprint" style="font-style:normal; font-weight:normal;"> <span style="display:inline-block; font-size:80%; position:relative;"><span style="font-size:117.6%; margin-left:-0.43em; position:relative; top:0.28em;"> <small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></span><span class="link-ru metadata" style="margin-left:-0.43em; position:absolute; left:0; z-index:1; margin-top:-0.35em;"><span style="visibility:hidden; margin-right:0.099em;"> (</span><span data-interwiki-article="Marine Corps Base Camp Pendleton" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%9A%D1%8D%D0%BC%D0%BF-%D0%9F%D0%B5%D0%BD%D0%B4%D0%BB%D1%82%D0%BE%D0%BD_(%D0%B1%D0%B0%D0%B7%D0%B0)&amp;action=edit&amp;redlink=1" title="Кэмп-Пендлтон (база) (страница отсутствует)">рус.</a></span></span></span></span> (штат <a href="/wiki/%D0%9A%D0%B0%D0%BB%D0%B8%D1%84%D0%BE%D1%80%D0%BD%D0%B8%D1%8F" title="Калифорния">Калифорния</a>) для подготовки. Во время своего пребывания в Кэмп-Пендлтоне он познакомился со своей будущей женой, Леной Мэй Ригги, которая была сержантом резерва морской пехоты. Джон и Лена поженились в церкви Звезды Святой Марии в городе <a href="/wiki/%D0%9E%D1%88%D0%B5%D0%BD%D1%81%D0%B0%D0%B9%D0%B4_(%D0%9A%D0%B0%D0%BB%D0%B8%D1%84%D0%BE%D1%80%D0%BD%D0%B8%D1%8F)" title="Ошенсайд (Калифорния)">Ошенсайд</a> <a href="/wiki/10_%D0%B8%D1%8E%D0%BB%D1%8F" title="10 июля">10 июля</a> <a href="/wiki/1944_%D0%B3%D0%BE%D0%B4" title="1944 год">1944 года</a>, банкет прошёл в отеле «Карлсбад». Молодожёны провели свой медовый месяц на луковой ферме родителей невесты в <a href="/wiki/%D0%9F%D0%BE%D1%80%D1%82%D0%BB%D0%B5%D0%BD%D0%B4_(%D0%9E%D1%80%D0%B5%D0%B3%D0%BE%D0%BD)" title="Портленд (Орегон)">Портленде</a> (штат <a href="/wiki/%D0%9E%D1%80%D0%B5%D0%B3%D0%BE%D0%BD" title="Орегон">Орегон</a>)<sup class="reference" id="cite_ref-paradewebsite_2-2"><a href="#cite_note-paradewebsite-2">[2]</a></sup>. Затем Джон вернулся на Тихий океан<sup class="reference" id="cite_ref-paradewebsite_2-3"><a href="#cite_note-paradewebsite-2">[2]</a></sup>.
</p>
<h3><span id=".D0.98.D0.B2.D0.BE.D0.B4.D0.B7.D0.B8.D0.BC.D0.B0"></span><span class="mw-headline" id="Иводзима">Иводзима</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%B9%D0%B7%D0%B8%D0%BB%D0%BE%D0%BD,_%D0%94%D0%B6%D0%BE%D0%BD&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Иводзима»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%B9%D0%B7%D0%B8%D0%BB%D0%BE%D0%BD,_%D0%94%D0%B6%D0%BE%D0%BD&amp;action=edit&amp;section=5" title="Редактировать код раздела «Иводзима»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Marines_on_the_beach_of_Iwo_Jima.jpg"><img class="mw-file-element" data-file-height="597" data-file-width="640" decoding="async" height="187" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ee/Marines_on_the_beach_of_Iwo_Jima.jpg/200px-Marines_on_the_beach_of_Iwo_Jima.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ee/Marines_on_the_beach_of_Iwo_Jima.jpg/300px-Marines_on_the_beach_of_Iwo_Jima.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ee/Marines_on_the_beach_of_Iwo_Jima.jpg/400px-Marines_on_the_beach_of_Iwo_Jima.jpg 2x" width="200"/></a><figcaption>Морские пехотинцы высаживаются на Иводзиме</figcaption></figure>
<p>После того как просьба Бейзилона о возвращении во флот была одобрена, он был назначен в роту «С» (Чарли) 1-го батальона 27-го полка 5-й дивизии морской пехоты во время <a href="/wiki/%D0%91%D0%B8%D1%82%D0%B2%D0%B0_%D0%B7%D0%B0_%D0%98%D0%B2%D0%BE%D0%B4%D0%B7%D0%B8%D0%BC%D1%83" title="Битва за Иводзиму">высадки на Иводзиму</a>. <a href="/wiki/19_%D1%84%D0%B5%D0%B2%D1%80%D0%B0%D0%BB%D1%8F" title="19 февраля">19 февраля</a> <a href="/wiki/1945_%D0%B3%D0%BE%D0%B4" title="1945 год">1945 года</a> Джон стал командиром пулемётного расчёта. Его подразделению было поручено высадиться на участке Red Beach II. Японцы встретили наступающих огнём сильно укреплённых огневых точек, построенных по всей территории острова. Так как его подразделение было сковано огнём противника, Бейзилон обошёл японские позиции, пока не оказался непосредственно на крыше японского <a class="mw-redirect" href="/wiki/%D0%94%D0%9E%D0%A2" title="ДОТ">ДОТа</a>. Джон забросал японцев гранатами и в одиночку уничтожил целый опорный пункт и его гарнизон. Затем он пробился к Аэродрому № 1 и помог американскому танку, попавшему на минном поле под сильный миномётный и артиллерийский огонь. Бейзилон провёл тяжёлое транспортное средство по опасной местности, несмотря на сильный огонь японцев. Когда Джон двигался по краю аэродрома, он был убит осколком японского снаряда . Действия Бейзилона помогли морским пехотинцам преодолеть оборону японцев на месте высадки, где морпехи были прижаты к земле плотным огнём противника и могли в любой момент быть сброшены обратно в море. За доблесть в битве за Иводзиму, Джон Бейзилон был посмертно награждён второй по значимости наградой морской пехоты — <a class="mw-redirect" href="/wiki/%D0%92%D0%BE%D0%B5%D0%BD%D0%BD%D0%BE-%D0%BC%D0%BE%D1%80%D1%81%D0%BA%D0%BE%D0%B9_%D0%BA%D1%80%D0%B5%D1%81%D1%82_(%D0%A1%D0%A8%D0%90)" title="Военно-морской крест (США)">Военно-морским крестом</a><sup class="reference" id="cite_ref-hallofvalor_11-0"><a href="#cite_note-hallofvalor-11">[11]</a></sup>.
</p><p>Джон Бейзилон был похоронен на <a href="/wiki/%D0%90%D1%80%D0%BB%D0%B8%D0%BD%D0%B3%D1%82%D0%BE%D0%BD%D1%81%D0%BA%D0%BE%D0%B5_%D0%BD%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%B0%D0%BB%D1%8C%D0%BD%D0%BE%D0%B5_%D0%BA%D0%BB%D0%B0%D0%B4%D0%B1%D0%B8%D1%89%D0%B5" title="Арлингтонское национальное кладбище">Арлингтонском национальном кладбище</a> в штате <a href="/wiki/%D0%92%D0%B8%D1%80%D0%B3%D0%B8%D0%BD%D0%B8%D1%8F" title="Виргиния">Виргиния</a>: его могила под номером 384 находится на 12-м участке сетки Y / Z 23.5<sup class="reference" id="cite_ref-ANC_12-0"><a href="#cite_note-ANC-12">[12]</a></sup>. Жена Джона Лена М. Бейзилон умерла <a href="/wiki/11_%D0%B8%D1%8E%D0%BD%D1%8F" title="11 июня">11 июня</a> <a href="/wiki/1999_%D0%B3%D0%BE%D0%B4" title="1999 год">1999 года</a> в возрасте 86 лет и была похоронена на Риверсайдском национальном кладбище<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup>. В некрологе было отмечено, что она после гибели мужа так и не вышла замуж<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup>.
</p>

<p>Бейзилон получил посмертно множество почестей, в числе которых — названный в честь него <a class="mw-redirect" href="/wiki/%D0%AD%D1%81%D0%BC%D0%B8%D0%BD%D0%B5%D1%86" title="Эсминец">эсминец</a> <a class="mw-redirect" href="/wiki/%D0%92%D0%9C%D0%A1_%D0%A1%D0%A8%D0%90" title="ВМС США">ВМС США</a>, почтовая марка, несколько мемориальных досок, памятников и географических достопримечательностей, включая бюст в Маленькой Италии в <a href="/wiki/%D0%A1%D0%B0%D0%BD-%D0%94%D0%B8%D0%B5%D0%B3%D0%BE" title="Сан-Диего">Сан-Диего</a> на <i>пьяцца</i>, названной в его честь.
</p>

<p><a class="extiw" href="https://en.wikipedia.org/wiki/USS_Basilone_(DD-824)" title="en:USS Basilone (DD-824)"><span title="USS Basilone (DD-824) — версия статьи «USS Basilone (DD-824)» на английском языке">«Бейзилон»</span></a><span class="noprint" style="font-style:normal; font-weight:normal;"> <span style="display:inline-block; font-size:80%; position:relative;"><span style="font-size:117.6%; margin-left:-0.43em; position:relative; top:0.28em;"> <small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></span><span class="link-ru metadata" style="margin-left:-0.43em; position:absolute; left:0; z-index:1; margin-top:-0.35em;"><span style="visibility:hidden; margin-right:0.099em;"> (</span><span data-interwiki-article="USS Basilone (DD-824)" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=USS_Basilone_(DD-824)&amp;action=edit&amp;redlink=1" title="USS Basilone (DD-824) (страница отсутствует)">рус.</a></span></span></span></span> — эсминец ВМС США <a href="/wiki/%D0%AD%D1%81%D0%BA%D0%B0%D0%B4%D1%80%D0%B5%D0%BD%D0%BD%D1%8B%D0%B5_%D0%BC%D0%B8%D0%BD%D0%BE%D0%BD%D0%BE%D1%81%D1%86%D1%8B_%D1%82%D0%B8%D0%BF%D0%B0_%C2%AB%D0%93%D0%B8%D1%80%D0%B8%D0%BD%D0%B3%C2%BB" title="Эскадренные миноносцы типа «Гиринг»">типа «Гиринг»</a> — был включен в состав флота <a href="/wiki/1949_%D0%B3%D0%BE%D0%B4" title="1949 год">1949 году</a>. Киль был заложен <a href="/wiki/7_%D0%B8%D1%8E%D0%BB%D1%8F" title="7 июля">7 июля</a> <a href="/wiki/1945_%D0%B3%D0%BE%D0%B4" title="1945 год">1945 года</a> в городе <a href="/wiki/%D0%9E%D1%80%D0%B8%D0%BD%D0%B4%D0%B6_(%D0%A2%D0%B5%D1%85%D0%B0%D1%81)" title="Ориндж (Техас)">Ориндж (Техас)</a>, корабль спущен на воду <a href="/wiki/21_%D0%B4%D0%B5%D0%BA%D0%B0%D0%B1%D1%80%D1%8F" title="21 декабря">21 декабря</a> <a href="/wiki/1945_%D0%B3%D0%BE%D0%B4" title="1945 год">1945 года</a>. Сержант Лена Mэй Бейзилон присвоила кораблю имя<sup class="reference" id="cite_ref-danfs_15-0"><a href="#cite_note-danfs-15">[15]</a></sup>.
</p>
<h3><span id=".D0.9F.D0.B0.D0.BC.D1.8F.D1.82.D1.8C_.D0.B7.D0.B0_.D0.BF.D1.80.D0.B5.D0.B4.D0.B5.D0.BB.D0.B0.D0.BC.D0.B8_.D0.BA.D0.BE.D1.80.D0.BF.D1.83.D1.81.D0.B0_.D0.BC.D0.BE.D1.80.D1.81.D0.BA.D0.BE.D0.B9_.D0.BF.D0.B5.D1.85.D0.BE.D1.82.D1.8B"></span><span class="mw-headline" id="Память_за_пределами_корпуса_морской_пехоты">Память за пределами корпуса морской пехоты</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%B9%D0%B7%D0%B8%D0%BB%D0%BE%D0%BD,_%D0%94%D0%B6%D0%BE%D0%BD&amp;veaction=edit&amp;section=8" title="Редактировать раздел «Память за пределами корпуса морской пехоты»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%B9%D0%B7%D0%B8%D0%BB%D0%BE%D0%BD,_%D0%94%D0%B6%D0%BE%D0%BD&amp;action=edit&amp;section=8" title="Редактировать код раздела «Память за пределами корпуса морской пехоты»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Джону Бейзилону посвящена памятная доска на флотском Мемориале Соединённых Штатов в городе <a class="mw-redirect" href="/wiki/%D0%92%D0%B0%D1%88%D0%B8%D0%BD%D0%B3%D1%82%D0%BE%D0%BD_(%D0%BE%D0%BA%D1%80%D1%83%D0%B3_%D0%9A%D0%BE%D0%BB%D1%83%D0%BC%D0%B1%D0%B8%D1%8F)" title="Вашингтон (округ Колумбия)">Вашингтоне</a><sup class="reference" id="cite_ref-MacGillis_16-0"><a href="#cite_note-MacGillis-16">[16]</a></sup>. В честь него установлен бюст в Маленькой Италии в городе <a href="/wiki/%D0%A1%D0%B0%D0%BD-%D0%94%D0%B8%D0%B5%D0%B3%D0%BE" title="Сан-Диего">Сан-Диего</a>. Военный мемориал, посвящённый жителям Маленькой Италии, которые участвовали во Второй мировой и <a href="/wiki/%D0%9A%D0%BE%D1%80%D0%B5%D0%B9%D1%81%D0%BA%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Корейская война">Корейской войнах</a> называется в честь Бейзилона<sup class="reference" id="cite_ref-piazza_17-0"><a href="#cite_note-piazza-17">[17]</a></sup>.
</p><p><a href="/wiki/10_%D0%BD%D0%BE%D1%8F%D0%B1%D1%80%D1%8F" title="10 ноября">10 ноября</a> <a href="/wiki/2005_%D0%B3%D0%BE%D0%B4" title="2005 год">2005 года</a> <a href="/wiki/%D0%9F%D0%BE%D1%87%D1%82%D0%BE%D0%B2%D0%B0%D1%8F_%D1%81%D0%BB%D1%83%D0%B6%D0%B1%D0%B0_%D0%A1%D0%A8%D0%90" title="Почтовая служба США">Почтовая служба США</a> опубликовала серию марок «Отличившиеся морские пехотинцы» в честь четырёх героев корпуса морской пехоты, в том числе марки с Бейзилоном<sup class="reference" id="cite_ref-stamp_18-0"><a href="#cite_note-stamp-18">[18]</a></sup><sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup>.
</p><p>В городке Реритан, где Джон вырос, чтят его память, его именем названы: футбольное поле в средней школе Бриджуотер-Реритан «Бейзилон-филд»<sup class="reference" id="cite_ref-KOCC13264_20-0"><a href="#cite_note-KOCC13264-20">[20]</a></sup>; путепровод на Сомервилл-Секл в <a class="extiw" href="https://en.wikipedia.org/wiki/Somerville,_New_Jersey" title="en:Somerville, New Jersey"><span title="Somerville, New Jersey — версия статьи «Сомервилл (Нью-Джерси)» на английском языке">Сомервилле</span></a><span class="noprint" style="font-style:normal; font-weight:normal;"> <span style="display:inline-block; font-size:80%; position:relative;"><span style="font-size:117.6%; margin-left:-0.43em; position:relative; top:0.28em;"> <small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></span><span class="link-ru metadata" style="margin-left:-0.43em; position:absolute; left:0; z-index:1; margin-top:-0.35em;"><span style="visibility:hidden; margin-right:0.099em;"> (</span><span data-interwiki-article="Somerville, New Jersey" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%A1%D0%BE%D0%BC%D0%B5%D1%80%D0%B2%D0%B8%D0%BB%D0%BB_(%D0%9D%D1%8C%D1%8E-%D0%94%D0%B6%D0%B5%D1%80%D1%81%D0%B8)&amp;action=edit&amp;redlink=1" title="Сомервилл (Нью-Джерси) (страница отсутствует)">рус.</a></span></span></span></span> (штат <a href="/wiki/%D0%9D%D1%8C%D1%8E-%D0%94%D0%B6%D0%B5%D1%80%D1%81%D0%B8" title="Нью-Джерси">Нью-Джерси</a>). Мост через реку <a class="extiw" href="https://en.wikipedia.org/wiki/Raritan_River" title="en:Raritan River"><span title="Raritan River — версия статьи «Реритан (река)» на английском языке">Реритан</span></a><span class="noprint" style="font-style:normal; font-weight:normal;"> <span style="display:inline-block; font-size:80%; position:relative;"><span style="font-size:117.6%; margin-left:-0.43em; position:relative; top:0.28em;"> <small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></span><span class="link-ru metadata" style="margin-left:-0.43em; position:absolute; left:0; z-index:1; margin-top:-0.35em;"><span style="visibility:hidden; margin-right:0.099em;"> (</span><span data-interwiki-article="Raritan River" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%A0%D0%B5%D1%80%D0%B8%D1%82%D0%B0%D0%BD_(%D1%80%D0%B5%D0%BA%D0%B0)&amp;action=edit&amp;redlink=1" title="Реритан (река) (страница отсутствует)">рус.</a></span></span></span></span> в штате Нью-Джерси называется «Бейзилон-бридж». Другой мост, который пересекает реку Реритан на Первой авеню и Канал-стрит города Реритан, также носит его имя. Также в память о Джоне была установлена мемориальная статуя, изображающая его держащим пулемёт «Браунинг». Она расположена на пересечении Олд-Йорк-Роуд и Канал-Стрит. В Реританской общественной библиотеке есть комната Бейзилона, где хранятся памятные вещи, связанные с Джоном<sup class="reference" id="cite_ref-Brady84_21-0"><a href="#cite_note-Brady84-21">[21]</a></sup>. Ежегодно в последние выходные сентября в городе отмечают парадом и концертом «День Бейзилона»<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup>.
</p>
<h3><span id=".D0.97.D0.B4.D0.B0.D0.BD.D0.B8.D1.8F_.D0.B8_.D0.BF.D0.B0.D0.BC.D1.8F.D1.82.D0.BD.D0.B8.D0.BA.D0.B8_.D0.B2_.D0.BA.D0.BE.D1.80.D0.BF.D1.83.D1.81.D0.B5_.D0.BC.D0.BE.D1.80.D1.81.D0.BA.D0.BE.D0.B9_.D0.BF.D0.B5.D1.85.D0.BE.D1.82.D1.8B"></span><span class="mw-headline" id="Здания_и_памятники_в_корпусе_морской_пехоты">Здания и памятники в корпусе морской пехоты</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%B9%D0%B7%D0%B8%D0%BB%D0%BE%D0%BD,_%D0%94%D0%B6%D0%BE%D0%BD&amp;veaction=edit&amp;section=9" title="Редактировать раздел «Здания и памятники в корпусе морской пехоты»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%B9%D0%B7%D0%B8%D0%BB%D0%BE%D0%BD,_%D0%94%D0%B6%D0%BE%D0%BD&amp;action=edit&amp;section=9" title="Редактировать код раздела «Здания и памятники в корпусе морской пехоты»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Корпус морской пехоты назвал в честь Джона некоторые объекты на базе Корпуса морской пехоты в Кэмп-Пендлтон, в том числе точка въезда на базу с 5-й федеральной трассы называется «Бейзилон-роуд»<sup class="reference" id="cite_ref-GlobalSecurity_23-0"><a href="#cite_note-GlobalSecurity-23">[23]</a></sup>, 5-я федеральная трасса проходит через базу, которая называется «Сержант Джон Бейзилон»<sup class="reference" id="cite_ref-memhigh_24-0"><a href="#cite_note-memhigh-24">[24]</a></sup>, а зона приземления с парашютом — «Точка сброса Бейзилона»<sup class="reference" id="cite_ref-BasiloneDZ_25-0"><a href="#cite_note-BasiloneDZ-25">[25]</a></sup>.
</p>
<h3><span id=".D0.A2.D0.B8.D1.85.D0.B8.D0.B9_.D0.BE.D0.BA.D0.B5.D0.B0.D0.BD_.28.D1.81.D0.B5.D1.80.D0.B8.D0.B0.D0.BB_2010_.D0.B3.D0.BE.D0.B4.D0.B0.29"></span><span class="mw-headline" id="Тихий_океан_(сериал_2010_года)">Тихий океан (сериал 2010 года)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%91%D0%B5%D0%B9%D0%B7%D0%B8%D0%BB%D0%BE%D0%BD,_%D0%94%D0%B6%D0%BE%D0%BD&amp;veaction=edit&amp;section=10" title="Редактировать раздел «Тихий океан (сериал 2010 года)»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%91%D0%B5%D0%B9%D0%B7%D0%B8%D0%BB%D0%BE%D0%BD,_%D0%94%D0%B6%D0%BE%D0%BD&amp;action=edit&amp;section=10" title="Редактировать код раздела «Тихий океан (сериал 2010 года)»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>История Джона Бейзилона вместе с историями двух других морских пехотинцев стала основой 10-серийного фильма <a href="/wiki/HBO" title="HBO">HBO</a> «<a href="/wiki/%D0%A2%D0%B8%D1%85%D0%B8%D0%B9_%D0%BE%D0%BA%D0%B5%D0%B0%D0%BD_(%D0%BC%D0%B8%D0%BD%D0%B8-%D1%81%D0%B5%D1%80%D0%B8%D0%B0%D0%BB)" title="Тихий океан (мини-сериал)">Тихий океан</a>», преемника сериала «<a href="/wiki/%D0%91%D1%80%D0%B0%D1%82%D1%8C%D1%8F_%D0%BF%D0%BE_%D0%BE%D1%80%D1%83%D0%B6%D0%B8%D1%8E_(%D1%82%D0%B5%D0%BB%D0%B5%D1%81%D0%B5%D1%80%D0%B8%D0%B0%D0%BB)" title="Братья по оружию (телесериал)">Братья по оружию</a>»<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup>. Актёр <a href="/wiki/%D0%A1%D0%B5%D0%B4%D0%B0,_%D0%94%D0%B6%D0%BE%D0%BD" title="Седа, Джон">Джон Седа</a> сыграл Джона Бейзилона.
</p><p>Сериал рассказывает о боях на Тихом Океане с точки зрения рядовых морских пехотинцев. Он был хорошо встречен как критиками, так и зрителями. <a href="/wiki/29_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="29 августа">29 августа</a> <a href="/wiki/2010_%D0%B3%D0%BE%D0%B4" title="2010 год">2010 года</a> сериал стал победителем премии <a href="/wiki/%D0%AD%D0%BC%D0%BC%D0%B8" title="Эмми">Эмми</a> в жанре мини-сериала<sup class="reference" id="cite_ref-about_27-0"><a href="#cite_note-about-27">[27]</a></sup>.
</p>
