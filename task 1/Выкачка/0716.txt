<p><b>AMD Athlon XP</b> («Атло́н экс-пи» в русской разговорной речи) — <a href="/wiki/X86" title="X86">x86</a>-совместимый <a class="mw-redirect" href="/wiki/%D0%9F%D1%80%D0%BE%D1%86%D0%B5%D1%81%D1%81%D0%BE%D1%80" title="Процессор">процессор</a> архитектуры K7, ставший результатом развития семейства процессоров <a href="/wiki/AMD" title="AMD">AMD</a> <a href="/wiki/Athlon" title="Athlon">Athlon</a>. Важным отличием Athlon XP от своих предшественников стала поддержка набора инструкций <a href="/wiki/SSE" title="SSE">SSE</a>, которая в комбинации с технологией <a href="/wiki/3DNow!" title="3DNow!">3DNow!</a> получила название 3DNow! Professional. Также был доработан механизм работы с <a href="/wiki/%D0%92%D0%B8%D1%80%D1%82%D1%83%D0%B0%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D0%BF%D0%B0%D0%BC%D1%8F%D1%82%D1%8C" title="Виртуальная память">виртуальной памятью</a> (<a class="mw-redirect" href="/wiki/Translation_lookaside_buffer" title="Translation lookaside buffer">TLB</a>) и блок аппаратной предвыборки данных из <a href="/wiki/%D0%9E%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D0%B0%D1%8F_%D0%BF%D0%B0%D0%BC%D1%8F%D1%82%D1%8C" title="Оперативная память">оперативной памяти</a>.
</p><p>Модернизированная архитектура K7 получила маркетинговое название — «QuantiSpeed Architecture».<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> Новое название было придумано с целью введения для Athlon XP системы PR-рейтингов, в соответствии с которой процессор получал числовое обозначение, соответствующее некой условной частоте процессора с ядром «Thunderbird», на которой могла быть получена аналогичная производительность. Такой подход позволял оценивать производительность не только пропорционально тактовой частоте процессора, а также учитывая количество инструкций, выполненное за один такт (IPC).
</p><p>Помимо самого Athlon XP, к серии XP относятся процессоры Athlon XP-M (Athlon 4), предназначенные для <a href="/wiki/%D0%9D%D0%BE%D1%83%D1%82%D0%B1%D1%83%D0%BA" title="Ноутбук">ноутбуков</a>, и Athlon MP, предназначенные для <a href="/wiki/%D0%A1%D0%B5%D1%80%D0%B2%D0%B5%D1%80_(%D0%B0%D0%BF%D0%BF%D0%B0%D1%80%D0%B0%D1%82%D0%BD%D0%BE%D0%B5_%D0%BE%D0%B1%D0%B5%D1%81%D0%BF%D0%B5%D1%87%D0%B5%D0%BD%D0%B8%D0%B5)" title="Сервер (аппаратное обеспечение)">серверов</a>. Кроме того, часть процессоров <a href="/wiki/Duron" title="Duron">Duron</a>, предназначенных для систем нижнего ценового уровня, представляет собой Athlon XP с частично отключённым кэшем второго уровня, а процессоры <a href="/wiki/Geode" title="Geode">Geode</a> NX, предназначенные для <a class="mw-redirect" href="/wiki/%D0%92%D1%81%D1%82%D1%80%D0%B0%D0%B8%D0%B2%D0%B0%D0%B5%D0%BC%D1%8B%D0%B5_%D1%81%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D1%8B" title="Встраиваемые системы">встраиваемых систем</a> — Athlon XP с пониженным энергопотреблением.
</p><p>Производство Athlon XP было начато в <a href="/wiki/2001_%D0%B3%D0%BE%D0%B4" title="2001 год">2001 году</a>. В конце <a href="/wiki/2003_%D0%B3%D0%BE%D0%B4" title="2003 год">2003 года</a> началось их постепенное вытеснение в нижнюю ценовую категорию процессорами архитектуры <a href="/wiki/Athlon_64" title="Athlon 64">K8</a>, а в <a href="/wiki/2004_%D0%B3%D0%BE%D0%B4" title="2004 год">2004 году</a> компанией AMD были представлены новые бюджетные процессоры <a href="/wiki/Sempron" title="Sempron">Sempron</a>, часть которых представляла собой переименованные Athlon XP.
</p>

<h2><span id=".D0.9E.D0.B1.D1.89.D0.B0.D1.8F_.D0.B8.D0.BD.D1.84.D0.BE.D1.80.D0.BC.D0.B0.D1.86.D0.B8.D1.8F"></span><span class="mw-headline" id="Общая_информация">Общая информация</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Athlon_XP&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Общая информация»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Athlon_XP&amp;action=edit&amp;section=1" title="Редактировать код раздела «Общая информация»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>

<p><a class="mw-redirect" href="/wiki/%D0%9F%D1%80%D0%BE%D1%86%D0%B5%D1%81%D1%81%D0%BE%D1%80" title="Процессор">Процессоры</a> семейства Athlon XP для <a href="/wiki/%D0%9D%D0%B0%D1%81%D1%82%D0%BE%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80" title="Настольный компьютер">настольных компьютеров</a> («настольные») и серверов («серверные»), а также большая часть процессоров для <a href="/wiki/%D0%9D%D0%BE%D1%83%D1%82%D0%B1%D1%83%D0%BA" title="Ноутбук">ноутбуков</a> («мобильные»), выполнены в корпусе типа <a class="mw-redirect" href="/wiki/FCPGA" title="FCPGA">FCPGA</a> (размеры корпуса — 49,5 × 49,5 мм) и предназначены для установки в системные платы с 462-контактным гнездовым разъёмом <a href="/wiki/Socket_A" title="Socket A">Socket A</a> (процессоры имеют 453 контакта, так как часть контактных отверстий разъёма закрыта).
</p><p>Корпус процессоров Athlon 4 и ранних Athlon MP на <a href="/wiki/%D0%AF%D0%B4%D1%80%D0%BE_%D0%BC%D0%B8%D0%BA%D1%80%D0%BE%D0%BF%D1%80%D0%BE%D1%86%D0%B5%D1%81%D1%81%D0%BE%D1%80%D0%B0" title="Ядро микропроцессора">ядре</a> Palomino (модели 1000 и 1200 МГц) имеет подложку из керамического материала, а корпус остальных процессоров Athlon XP и MP — из органического материала (зелёного или коричневого цвета). Переход на новые материалы был связан с тем, что применение органического вещества для изготовления подложки корпуса позволяет повысить стабильность работы процессоров на более высоких тактовых частотах.<sup class="reference" id="cite_ref-bulka_2-0"><a href="#cite_note-bulka-2">[2]</a></sup>
</p><p>Открытый кристалл процессора расположен на лицевой стороне подложки корпуса и соединён с ней при помощи специального вещества (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">underfill</span>), позволяющего компенсировать разницу в скорости теплового расширения кристалла и подложки.<sup class="reference" id="cite_ref-bulka_2-1"><a href="#cite_note-bulka-2">[2]</a></sup> На лицевой стороне подложки находятся также <a href="/wiki/%D0%9F%D0%BE%D0%B2%D0%B5%D1%80%D1%85%D0%BD%D0%BE%D1%81%D1%82%D0%BD%D1%8B%D0%B9_%D0%BC%D0%BE%D0%BD%D1%82%D0%B0%D0%B6" title="Поверхностный монтаж">SMD</a>-элементы (за исключением процессоров на ядре Palomino, начиная с модели 1500+) и перемычки (обычно называемые мостиками), задающие <a href="/wiki/%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%BE%D0%B5_%D0%BD%D0%B0%D0%BF%D1%80%D1%8F%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5" title="Электрическое напряжение">напряжение</a> питания, <a class="mw-redirect" href="/wiki/%D0%A2%D0%B0%D0%BA%D1%82%D0%BE%D0%B2%D0%B0%D1%8F_%D1%87%D0%B0%D1%81%D1%82%D0%BE%D1%82%D0%B0" title="Тактовая частота">частоту</a> и размер включённой <a class="mw-redirect" href="/wiki/L2_(%D0%BA%D0%B5%D1%88)" title="L2 (кеш)">кеш-памяти второго уровня</a>. Перемычки располагаются группами, которые имеют обозначения L1—L11 в процессорах на ядре Palomino и L1—L12 в процессорах на более новых ядрах (Thoroughbred, Barton, Thorton). С помощью графита или токопроводящего клея <a class="mw-redirect" href="/wiki/%D0%9E%D0%B2%D0%B5%D1%80%D0%BA%D0%BB%D0%BE%D0%BA%D0%B5%D1%80" title="Оверклокер">оверклокеры</a> пользовались перемычками для управления параметрами процессора. В поздних процессорах Athlon XP использовалась также «безмостиковая» упаковка, в которой конфигурационные перемычки скрыты под слоем лака.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> На обратной стороне подложки корпуса расположены контакты, а в процессорах на ядре Palomino (начиная с модели 1500+) между контактами установлены SMD-элементы.
</p><p>Часть мобильных процессоров на ядрах Thoroughbred и Barton выпускалась в 563-контактном корпусе типа <a class="mw-redirect" href="/wiki/MPGA" title="MPGA">mPGA</a> (33 × 33<span class="nowrap"> мм</span>). Такие процессоры устанавливались в разъём <a href="/wiki/Socket_563" title="Socket 563">Socket 563</a>, несовместимый с разъёмом Socket A. Корпус типа mPGA также предусматривает подложку из органического материала и открытый кристалл, однако его габариты меньше по сравнению с корпусом типа FCPGA. Несмотря на то что процессоры в таком корпусе предназначены для <a href="/wiki/%D0%9D%D0%BE%D1%83%D1%82%D0%B1%D1%83%D0%BA" title="Ноутбук">мобильных компьютеров</a>, существует системная плата с разъёмом Socket 563 для настольных компьютеров — <a class="new" href="/w/index.php?title=PC_Chips&amp;action=edit&amp;redlink=1" title="PC Chips (страница отсутствует)">PC Chips</a> M863G.
</p><p>Маркировка процессоров на ядре Palomino нанесена на кристалл процессора, а процессоров на более новых ядрах — на наклейку, расположенную возле кристалла. В связи с тем, что процессоры семейства Athlon XP имеют открытый кристалл, для предотвращения повреждений, вызванных перекосом радиатора, предусмотрена защита в форме четырёх круглых прокладок, расположенных в углах подложки корпуса. Однако, несмотря на наличие этих прокладок, при неаккуратной установке радиатора (особенно неопытными пользователями) кристалл получал трещины и сколы.
</p><p>Влияние таких повреждений на работоспособность процессора зависело от места скола. В некоторых случаях процессор, получивший существенные повреждения кристалла (сколы до 2—3 мм с угла), продолжал работать без сбоев или с редкими сбоями, в то же время, процессор с незначительными сколами мог полностью выйти из строя. В том случае, если скол приходился на область кэша второго уровня, существовала возможность отключить повреждённый кэш, изменяя конфигурацию мостиков, отвечающих за его размер. Таким образом процессор Athlon XP приобретал работоспособность, однако фактически работал как процессор <a href="/wiki/Duron" title="Duron">Duron</a> (в случае ядра Thoroughbred)<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>, либо как процессор на ядре Thorton с меньшим рейтингом (в случае ядра Barton).
</p>
<h2><span id=".D0.9E.D1.81.D0.BE.D0.B1.D0.B5.D0.BD.D0.BD.D0.BE.D1.81.D1.82.D0.B8_.D0.B0.D1.80.D1.85.D0.B8.D1.82.D0.B5.D0.BA.D1.82.D1.83.D1.80.D1.8B"></span><span class="mw-headline" id="Особенности_архитектуры">Особенности архитектуры</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Athlon_XP&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Особенности архитектуры»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Athlon_XP&amp;action=edit&amp;section=2" title="Редактировать код раздела «Особенности архитектуры»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="hatnote navigation-not-searchable">Основная статья: <b><a class="mw-redirect" href="/wiki/AMD_K7" title="AMD K7">AMD K7</a></b></div>
<p>Ключевыми особенностями процессоров архитектуры К7 являются:
</p>
<ul><li>Блок целочисленных вычислений (<a class="mw-redirect" href="/wiki/ALU" title="ALU">ALU</a>), содержащий три конвейера глубиной 10<span class="nowrap"> стадий</span>, позволяющий процессору выполнять до трёх инструкций за такт.</li>
<li>Блок вещественночисленных вычислений (<a class="mw-redirect" href="/wiki/FPU" title="FPU">FPU</a>), содержащий три конвейера глубиной 15<span class="nowrap"> стадий</span>.</li>
<li>Системная шина EV6, лицензированная у компании <a class="mw-redirect" href="/wiki/DEC" title="DEC">DEC</a>, обеспечивающая передачу данных по обоим фронтам тактового сигнала. Это позволило при физической частоте 100—200<span class="nowrap"> МГц</span> получить эффективную частоту 200—400<span class="nowrap"> МГц</span>, что соответствует пропускной способности 1,6—3,2<span class="nowrap"> Гбайт/с</span>. Кроме того, шина EV6 использует протокол точка-точка, что позволяет организовать эффективную многопроцессорную систему.</li>
<li><a class="mw-redirect" href="/wiki/%D0%9A%D0%B5%D1%88-%D0%BF%D0%B0%D0%BC%D1%8F%D1%82%D1%8C" title="Кеш-память">Кеш-память</a> первого уровня объёмом 128<span class="nowrap"> кбайт</span> (64<span class="nowrap"> кбайт</span> кеша инструкций и 64<span class="nowrap"> кбайт</span> кеша данных).</li>
<li>Расширенный набор инструкций Extended <a href="/wiki/3DNow!" title="3DNow!">3DNow!</a>, поддержка набора <a href="/wiki/SSE" title="SSE">SSE</a> поздними представителями архитектуры K7.</li></ul>
<h2><span id=".D0.9C.D0.BE.D0.B4.D0.B5.D0.BB.D0.B8"></span><span class="mw-headline" id="Модели">Модели</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Athlon_XP&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Модели»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Athlon_XP&amp;action=edit&amp;section=3" title="Редактировать код раздела «Модели»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Первыми процессорами семейства стали мобильные процессоры Athlon 4, анонсированные <a href="/wiki/14_%D0%BC%D0%B0%D1%8F" title="14 мая">14 мая</a> <a href="/wiki/2001_%D0%B3%D0%BE%D0%B4" title="2001 год">2001 года</a>. <a href="/wiki/5_%D0%B8%D1%8E%D0%BD%D1%8F" title="5 июня">5 июня</a> 2001 года были анонсированы процессоры Athlon MP, предназначенные для использования в <a href="/wiki/%D0%9C%D0%BD%D0%BE%D0%B3%D0%BE%D0%BF%D1%80%D0%BE%D1%86%D0%B5%D1%81%D1%81%D0%BE%D1%80%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Многопроцессорность">многопроцессорных</a> системах. Первый процессор Athlon XP для <a href="/wiki/%D0%9D%D0%B0%D1%81%D1%82%D0%BE%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80" title="Настольный компьютер">настольных компьютеров</a> был представлен <a href="/wiki/9_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="9 октября">9 октября</a> 2001 года.
</p><p>Впервые с <a href="/wiki/1996_%D0%B3%D0%BE%D0%B4" title="1996 год">1996 года</a> процессоры <a href="/wiki/AMD" title="AMD">AMD</a> получили рейтинговую систему обозначения моделей. Официально рейтинг процессора Athlon XP был приравнен к <a class="mw-redirect" href="/wiki/%D0%A2%D0%B0%D0%BA%D1%82%D0%BE%D0%B2%D0%B0%D1%8F_%D1%87%D0%B0%D1%81%D1%82%D0%BE%D1%82%D0%B0" title="Тактовая частота">тактовой частоте</a> процессора <a href="/wiki/Athlon" title="Athlon">Athlon</a>, имеющего равную или чуть меньшую производительность в наборе офисных, графических и <a href="/wiki/%D0%9C%D1%83%D0%BB%D1%8C%D1%82%D0%B8%D0%BC%D0%B5%D0%B4%D0%B8%D0%B0" title="Мультимедиа">мультимедийных программ</a>, <a href="/wiki/%D0%9A%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%B8%D0%B3%D1%80%D0%B0" title="Компьютерная игра">игр</a> и игровых <a class="mw-redirect" href="/wiki/%D0%91%D0%B5%D0%BD%D1%87%D0%BC%D0%B0%D1%80%D0%BA_(%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80)" title="Бенчмарк (компьютер)">бенчмарков</a>. Однако в действительности рейтинг показывал производительность процессоров Athlon XP относительно конкурирующего процессора <a href="/wiki/Intel" title="Intel">Intel</a> <a href="/wiki/Pentium_4" title="Pentium 4">Pentium 4</a>.<sup class="reference" id="cite_ref-ixbt_0_5-0"><a href="#cite_note-ixbt_0-5">[5]</a></sup>
</p><p>Первые процессоры Athlon XP на ядре Palomino производились по 180-нм технологии. Дальнейшим развитием линейки стали процессоры на ядре Thoroughbred (130 нм). Ядро Thoroughbred использовалось также в бюджетных процессорах <a href="/wiki/Sempron" title="Sempron">Sempron</a>. Последним ядром, использованным в процессорах Athlon XP, стало ядро Barton, отличавшееся от ядра Thoroughbred увеличенным размером кэш-памяти второго уровня. Также в процессорах Athlon XP использовалось ядро Thorton, представлявшее собой ядро Barton с частично отключённым кэшем второго уровня. По своим характеристикам процессоры на ядре Thorton практически не отличались от процессоров на ядре Thoroughbred, однако были дороже в производстве за счёт большей площади кристалла.
</p><p>Ниже представлены даты анонса различных моделей процессоров Athlon XP, Athlon MP, Athlon 4 и Mobile Athlon XP (Athlon XP-M), а также их стоимость на момент анонса.

<h3><span class="mw-headline" id="Athlon_XP">Athlon XP</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Athlon_XP&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Athlon XP»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Athlon_XP&amp;action=edit&amp;section=4" title="Редактировать код раздела «Athlon XP»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<h4><span id="Palomino_.28Model_6.29"></span><span class="mw-headline" id="Palomino_(Model_6)">Palomino (Model 6)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Athlon_XP&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Palomino (Model 6)»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Athlon_XP&amp;action=edit&amp;section=5" title="Редактировать код раздела «Palomino (Model 6)»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h4>
<figure class="mw-default-size" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Athlon_XP_1600_Palomino_Front.jpg"><img class="mw-file-element" data-file-height="649" data-file-width="648" decoding="async" height="220" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/d6/Athlon_XP_1600_Palomino_Front.jpg/220px-Athlon_XP_1600_Palomino_Front.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/d/d6/Athlon_XP_1600_Palomino_Front.jpg/330px-Athlon_XP_1600_Palomino_Front.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/d/d6/Athlon_XP_1600_Palomino_Front.jpg/440px-Athlon_XP_1600_Palomino_Front.jpg 2x" width="220"/></a><figcaption>AMD Athlon XP 1600+</figcaption></figure>
<p>Процессоры Athlon XP на ядре Palomino были представлены компанией <a href="/wiki/AMD" title="AMD">AMD</a> в октябре <a href="/wiki/2001_%D0%B3%D0%BE%D0%B4" title="2001 год">2001 года</a> и представляли собой модернизированное ядро Thunderbird, использовавшееся в процессорах <a href="/wiki/Athlon" title="Athlon">Athlon</a>. Также как и процессоры Athlon, Athlon XP имели разделённый кэш первого уровня, объём которого составлял 128 <a href="/wiki/%D0%9A%D0%B8%D0%BB%D0%BE%D0%B1%D0%B0%D0%B9%D1%82" title="Килобайт">Кбайт</a> (по 64 Кбайт на данные и инструкции), и встроенный объединённый кэш второго уровня объёмом 256 Кбайт, по-прежнему имевший 64-битную <a class="mw-redirect" href="/wiki/BSB" title="BSB">шину</a>.
</p><p>Основными нововведениями, представленными в ядре Palomino, являлись блок инструкций <a href="/wiki/SSE" title="SSE">SSE</a>, а также механизм аппаратной предвыборки данных (hardware prefetch). Благодаря этим нововведениям производительность процессоров Athlon XP на ядре Palomino на 2—5 процентов превышала производительность процессоров Athlon на ядре Thunderbird при одинаковой тактовой частоте.<sup class="reference" id="cite_ref-ixbt_0_5-1"><a href="#cite_note-ixbt_0-5">[5]</a></sup>
</p><p>Кроме того, процессоры Athlon XP имели встроенный датчик температуры (<a class="new" href="/w/index.php?title=%D0%A2%D0%B5%D1%80%D0%BC%D0%BE%D0%B4%D0%B8%D0%BE%D0%B4&amp;action=edit&amp;redlink=1" title="Термодиод (страница отсутствует)">термодиод</a>), что позволяло при использовании <a href="/wiki/%D0%9C%D0%B0%D1%82%D0%B5%D1%80%D0%B8%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BF%D0%BB%D0%B0%D1%82%D0%B0" title="Материнская плата">системной платы</a>, поддерживающей работу с этим датчиком, организовать более эффективную защиту от перегрева, чем при использовании внешнего термодатчика. Однако в связи с тем, что лишь немногие системные платы имели такую возможность, а в процессорах отсутствовал встроенный механизм аварийного отключения, эффективность термозащиты, как и у процессоров Athlon, оставалась низкой.<sup class="reference" id="cite_ref-thermal_7-0"><a href="#cite_note-thermal-7">[7]</a></sup>
</p><p>Процессоры Athlon XP на ядре Palomino выпускались по 180-нм технологии и содержали 37,5 млн транзисторов. Площадь кристалла при этом составляла 129 мм². Процессоры работали с эффективной частотой системной шины (<a class="mw-disambig" href="/wiki/FSB" title="FSB">FSB</a>) 266 МГц.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> Напряжение ядра процессоров на ядре Palomino составляло 1,75 В, максимальное тепловыделение — 72 Вт (на частоте 1733 МГц, модель 2100+).
</p>
<h4><span id="Thoroughbred_.28Model_8.29"></span><span class="mw-headline" id="Thoroughbred_(Model_8)">Thoroughbred (Model 8)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Athlon_XP&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Thoroughbred (Model 8)»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Athlon_XP&amp;action=edit&amp;section=6" title="Редактировать код раздела «Thoroughbred (Model 8)»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h4>

<p>Ядро Thoroughbred представляло собой ядро Palomino, произведённое по новой 130-нм технологии. Архитектура ядра при этом осталась неизменной. Планировалось, что процессоры на ядре Thoroughbred будут представлены в начале 2002 года, но из-за технологических проблем анонс неоднократно переносился на более поздний срок. Официально процессоры Athlon XP на ядре Thoroughbred были представлены 10 июня <a href="/wiki/2002_%D0%B3%D0%BE%D0%B4" title="2002 год">2002 года</a>, однако компания AMD не могла обеспечить их массовые поставки в течение нескольких месяцев после анонса.<sup class="reference" id="cite_ref-iamd_9-0"><a href="#cite_note-iamd-9">[9]</a></sup>
</p><p>Максимальная тактовая частота, которую смогли достичь процессоры на ядре Thoroughbred первой версии, составляла 1800 МГц (для сравнения, старшая модель Athlon XP на ядре Palomino работала на частоте 1733 МГц), поэтому осенью 2002 года компания AMD выпустила обновлённую версию ядра Thoroughbred (ревизия B0). Старшая модель Athlon XP на ядре Thoroughbred ревизии B0 работала на частоте 2200 МГц (данная модель была предназначена только для рынка <a href="/wiki/OEM" title="OEM">OEM</a> и не поступила в широкую продажу). Среди пользователей процессоры с ядром первой ревизии (A0) обычно называли Thoroughbred-A, а процессоры с ядром ревизии B0 — Thoroughbred-B.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>
</p><p>Благодаря высокому частотному потенциалу и низкой цене, младшие процессоры на ядре Thoroughbred-B пользовались популярностью среди <a class="mw-redirect" href="/wiki/%D0%9E%D0%B2%D0%B5%D1%80%D0%BA%D0%BB%D0%BE%D0%BA%D0%B5%D1%80" title="Оверклокер">оверклокеров</a>. Кроме того, ранние процессоры на этом ядре имели свободный множитель, что значительно упрощало их <a href="/wiki/%D0%A0%D0%B0%D0%B7%D0%B3%D0%BE%D0%BD_%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BE%D0%B2" title="Разгон компьютеров">разгон</a>.
</p><p>Первую и вторую ревизии различали по <a class="mw-redirect" href="/wiki/Cpuid" title="Cpuid">CPU Id</a> процессора (процессоры на ядре Thoroughbred-A имели CPU Id 0x680h, а на ядре Thoroughbred-B — 0x681h), а также по его маркировке (пятая буква во второй строке маркировки указывала на ревизию ядра, например, AIUG<b>A</b>0247UPMW и JIUH<b>B</b>0251XPMW).<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup>
</p><p>Процессоры Athlon XP на ядре Thoroughbred выпускались по 130-нм технологии и содержали 37,2 млн транзисторов. Площадь кристалла процессоров ревизии A0 при этом составляла 80,89 мм², а ревизии B0 — 84,66 мм². Процессоры работали с частотой системной шины 266 или 333 МГц<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>, напряжение ядра составляло 1,5—1,65 В в зависимости от модели, максимальное тепловыделение — 68,3 Вт (на частоте 2167 МГц, модель 2700+).
</p><p>В конце 2002 года появилась информация о выпуске компанией AMD третьей ревизии ядра Thoroughbred — C0 (CPU Id 0x682h), которая должна была производиться с использованием технологии <a class="mw-redirect" href="/wiki/SOI" title="SOI">SOI</a>, иметь бо́льшую площадь ядра (86,97 мм²) и работать на более высокой частоте<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup><sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup>, однако в официальной документации компании AMD упоминания об этой ревизии отсутствуют.
</p><p>Помимо процессоров Athlon XP, ядро Thoroughbred использовалось в младших моделях процессоров <a href="/wiki/Sempron" title="Sempron">Sempron</a>, в процессорах <a href="/wiki/Duron" title="Duron">Duron</a> и во встраиваемом процессоре <a href="/wiki/Geode" title="Geode">Geode</a> NX. Ядро, использовавшееся в процессорах Duron, имело собственное название «Applebred» и представляло собой Thoroughbred с частично отключённым кэшем второго уровня.
</p><p>Процессоры Geode NX по-прежнему выпускаются компанией AMD и работают на частотах 667—1400 МГц, имеют напряжение ядра 1—1,25 В и типичное тепловыделение 8—14,3 Вт (максимальное — 25 Вт на частоте 1400 МГц).<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup>
</p>
<h4><span id="Barton_.28Model_10.29"></span><span class="mw-headline" id="Barton_(Model_10)">Barton (Model 10)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Athlon_XP&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Barton (Model 10)»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Athlon_XP&amp;action=edit&amp;section=7" title="Редактировать код раздела «Barton (Model 10)»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h4>
<figure class="mw-default-size" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:AMD_Barton.jpg"><img class="mw-file-element" data-file-height="696" data-file-width="711" decoding="async" height="215" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b2/AMD_Barton.jpg/220px-AMD_Barton.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/b2/AMD_Barton.jpg/330px-AMD_Barton.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/b2/AMD_Barton.jpg/440px-AMD_Barton.jpg 2x" width="220"/></a><figcaption>AMD Athlon XP 2500+</figcaption></figure>
<p>Barton — последнее ядро, использовавшееся в процессорах Athlon XP. Оно было представлено в феврале <a href="/wiki/2003_%D0%B3%D0%BE%D0%B4" title="2003 год">2003 года</a> и представляло собой ядро Thoroughbred с увеличенной до 512 Кбайт кэш-памятью второго уровня. Процессоры на ядре Barton содержали 51,3 млн транзисторов, производились по 130-нм технологии и имели площадь кристалла 100,99 мм². Напряжение ядра составляло 1,65 В, максимальное тепловыделение — 79,2 Вт (на частоте 2333 МГц). Планировалось, что процессоры на ядре Barton будут производиться с применением технологии <a class="mw-redirect" href="/wiki/SOI" title="SOI">SOI</a> (кремний на изоляторе), позволяющей увеличить тактовые частоты и снизить тепловыделение процессоров, однако осенью <a href="/wiki/2002_%D0%B3%D0%BE%D0%B4" title="2002 год">2002 года</a> было объявлено, что технология SOI не будет применяться при производстве процессоров Athlon XP.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup>
</p><p>Процессоры Athlon XP на ядре Barton работали с частотой системной шины 333 и 400 МГц, однако существовали модели, не предназначенные для розничной продажи (модели для рынка <a href="/wiki/OEM" title="OEM">OEM</a>, поставляемые компаниям-сборщикам готовых систем), имеющие частоту системной шины 266 МГц. Максимальная тактовая частота ядра массовых процессоров составляла 2200 МГц (модель 3200+), однако, существовал процессор с частотой 2333 МГц (частота системной шины — 333 МГц, рейтинг — 3200+), выпущенный специально для компании <a href="/wiki/Hewlett-Packard" title="Hewlett-Packard">Hewlett-Packard</a>, использовавшей его в персональном компьютере бизнес-серии «d325».<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup> Специально для компании <a class="new" href="/w/index.php?title=SystemMax&amp;action=edit&amp;redlink=1" title="SystemMax (страница отсутствует)">SystemMax</a> была выпущена модель 2900+, имевшая тактовую частоту 2000 МГц.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup>
</p><p>Помимо процессоров Athlon XP, ядро Barton использовалось также в некоторых процессорах <a href="/wiki/AMD" title="AMD">AMD</a> <a href="/wiki/Sempron" title="Sempron">Sempron</a>. Ядро Barton с частично отключённым кэшем второго уровня имело собственное название — Thorton, однако физически от «полноценного» Barton не отличалось.
</p>
<h4><span id="Thorton_.28Model_10.29"></span><span class="mw-headline" id="Thorton_(Model_10)">Thorton (Model 10)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Athlon_XP&amp;veaction=edit&amp;section=8" title="Редактировать раздел «Thorton (Model 10)»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Athlon_XP&amp;action=edit&amp;section=8" title="Редактировать код раздела «Thorton (Model 10)»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h4>
<p>Процессоры Athlon XP на ядре Thorton появились на рынке осенью <a href="/wiki/2003_%D0%B3%D0%BE%D0%B4" title="2003 год">2003 года</a>. Они представляли собой процессоры на ядре Barton с частично отключённым кэшем второго уровня (256 Кбайт). В том случае, если отключённая часть кэш-памяти была работоспособной, а упаковка процессора позволяла манипуляции с конфигурационными мостиками, существовала возможность включения всех 512 Кбайт кэша второго уровня. Таким образом процессор на ядре Thorton превращался в процессор на ядре Barton.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup>
</p><p>Старшие модели Athlon XP на ядре Thorton (2400+, 2600+ и 3100+), как и процессоры на ядре Barton, имели напряжение ядра 1,65 В, младшие (2000+ и 2200+) — пониженное до 1,5 или 1,6 В в зависимости от партии. Все Athlon XP на ядре Thorton работали с частотой системной шины 266 МГц, за исключением модели 3000+ (400 МГц) и некоторых 2600+ (333 МГц). Максимальное тепловыделение было снижено по сравнению с процессорами на ядре Barton и составляло 60,3—68,3 Вт в зависимости от модели.
</p><p>Помимо процессоров Athlon XP, ядро Thorton некоторое время использовалось в бюджетных процессорах <a href="/wiki/Sempron" title="Sempron">Sempron</a>, однако из-за большей площади кристалла и более высокой себестоимости процессоров на этом ядре по сравнению с ядром Thoroughbred, имевшем аналогичные характеристики, AMD отказалась от использования ядра Thorton в пользу ядра Thoroughbred ревизии «B0».<sup class="reference" id="cite_ref-lowend_20-0"><a href="#cite_note-lowend-20">[20]</a></sup>
</p>
<h3><span class="mw-headline" id="Athlon_MP">Athlon MP</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Athlon_XP&amp;veaction=edit&amp;section=9" title="Редактировать раздел «Athlon MP»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Athlon_XP&amp;action=edit&amp;section=9" title="Редактировать код раздела «Athlon MP»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Процессоры Athlon MP (аббревиатура «MP» в названии процессора расшифровывается как MultiProcessor) предназначались для работы в двухпроцессорных системах и представляли собой процессоры Athlon XP с включённой поддержкой двухпроцессорной конфигурации (в частности, протокола <a href="/wiki/MOESI" title="MOESI">MOESI</a>, позволяющего двум процессорам организовать обмен данными через кэш-память второго уровня, а не через системную память). В основе этих процессоров лежали ядра Palomino, Thoroughbred и Barton.
</p><p>Все процессоры Athlon MP работали с частотой системной шины 266 МГц, напряжение ядра составляло 1,75 В для процессоров на ядре Palomino, 1,6—1,65 В для процессоров на ядре Thoroughbred и 1,6 В для процессоров на ядре Barton. Максимальное тепловыделение составляло 46,1—66 Вт для процессоров на ядре Palomino и 60 Вт для процессоров на ядрах Thoroughbred и Barton.
</p><p>На основе ядра Palomino выпускались модели Athlon MP 1000 МГц, 1200 МГц, 1500+ (1333 МГц), 1600+ (1400 МГц), 1800+ (1533 МГц), 1900+ (1600 МГц), 2000+ (1667 МГц) и 2100+ (1733 МГц). Ядро Thoroughbred лежало в основе моделей Athlon MP 2000+ (1667 МГц), 2200+ (1800 МГц), 2400+ (2000 МГц) и 2600+ (2133 МГц). Модели 2600+ (2000 МГц) и 2800+ (2133 МГц), имевшие 512 Кбайт кэш-памяти второго уровня, выпускались на базе ядра Barton.
</p><p>Производительность процессоров Athlon MP практически не отличалась от производительности Athlon XP с аналогичными характеристиками. Кроме того, многие процессоры Athlon XP (исключение составляли процессоры в «безмостиковой» упаковке) могли получить поддержку многопроцессорности путём изменения состояния конфигурационных мостиков. С учётом более высокой цены Athlon MP, использование Athlon XP в двухпроцессорной конфигурации было более выгодным для рядовых пользователей (однако, на переделанные таким образом процессоры не распространялись гарантийные обязательства компании AMD, что во многих случаях делало этот вариант неприемлемым).<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup>
</p>
<h3><span class="mw-headline" id="Mobile_Athlon_4">Mobile Athlon 4</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Athlon_XP&amp;veaction=edit&amp;section=10" title="Редактировать раздел «Mobile Athlon 4»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Athlon_XP&amp;action=edit&amp;section=10" title="Редактировать код раздела «Mobile Athlon 4»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Мобильный Athlon 4 стал первым выпущенным компанией AMD процессором с архитектурой QuantiSpeed. В его основе лежало ядро Corvette, аналогичное ядру Palomino, позже использовавшемуся в настольных процессорах Athlon XP. Основными отличиями процессоров Athlon 4 от Athlon XP было использование керамического корпуса, пониженное до 1,2—1,6 В напряжение питания, более низкое тепловыделение (25 Вт у стандартных процессоров и 35 Вт у процессоров для ноутбуков класса <a class="new" href="/w/index.php?title=DTR&amp;action=edit&amp;redlink=1" title="DTR (страница отсутствует)">DTR</a>), а также поддержка энергосберегающей технологии <a href="/wiki/PowerNow!" title="PowerNow!">PowerNow!</a>. Все процессоры Athlon 4 работали с частотой системной шины 200 МГц.
</p>
<h3><span id="Mobile_Athlon_XP_.28Athlon_XP-M.29"></span><span class="mw-headline" id="Mobile_Athlon_XP_(Athlon_XP-M)">Mobile Athlon XP (Athlon XP-M)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Athlon_XP&amp;veaction=edit&amp;section=11" title="Редактировать раздел «Mobile Athlon XP (Athlon XP-M)»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Athlon_XP&amp;action=edit&amp;section=11" title="Редактировать код раздела «Mobile Athlon XP (Athlon XP-M)»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Процессоры Athlon XP-M (первые модели на ядре Thoroughbred назывались Mobile Athlon XP) представляли собой процессоры на ядрах Thoroughbred и Barton, имеющие пониженное напряжение питания и тепловыделение, энергосберегающую технологию AMD <a href="/wiki/PowerNow!" title="PowerNow!">PowerNow!</a> и свободный множитель, необходимый для её работы. Максимально допустимая температура корпуса была повышена по сравнению с процессорами для настольных компьютеров и составляла 100 °C (у Athlon XP — 90 °C), что было связано с условиями работы в ноутбуке (небольшое воздушное пространство и размеры радиатора, более слабый воздушный поток). Мобильные процессоры Athlon XP работали с частотой системной шины 200 или 266 МГц в зависимости от модели.
</p><p>Существовало несколько модификаций мобильных процессоров Athlon XP в зависимости от максимального тепловыделения (TDP):
</p>
<ul><li>Mobile Athlon XP Balanced. Предназначались для использования в лёгких ноутбуках и отличались пониженным энергопотреблением. Максимальное тепловыделение — 25 Вт, напряжение ядра — от 1,25 до 1,3 В в зависимости от модели.</li>
<li>Mobile Athlon XP Standard. Предназначались для использования в ноутбуках различных классов. Максимальное тепловыделение — 35 Вт, напряжение ядра — от 1,4 до 1,45 В в зависимости от модели.</li>
<li>Athlon XP-M Low Voltage. Предназначались для использования в лёгких ноутбуках, имели напряжение ядра от 1,1 до 1,35 В зависимости от модели, максимальное тепловыделение составляло 35 Вт.</li>
<li>Athlon XP-M Mainstream. Предназначались для использования в ноутбуках различных классов. Процессоры на ядре Thoroughbred имели TDP 45 Вт и напряжение ядра 1,4—1,55 В в зависимости от модели. Процессоры на ядре Barton имели TDP 45 Вт и напряжение ядра 1,45 В. Кроме того, существовали модели с TDP 53 Вт и напряжением ядра 1,55 В.</li>
<li>Athlon XP-M DTR. Предназначались для использования в ноутбуках класса DTR. TDP таких процессоров — 72 Вт. Напряжение ядра составляло 1,6—1,65 В в зависимости от модели.</li></ul>
<p>Последние модели процессоров Athlon XP-M имели архитектуру <a href="/wiki/Athlon_64" title="Athlon 64">K8</a> (ядро Dublin). Впоследствии эти процессоры были переименованы в Mobile Sempron.
</p>
<h2><span id=".D0.9F.D0.BE.D0.BB.D0.BE.D0.B6.D0.B5.D0.BD.D0.B8.D0.B5_.D0.BD.D0.B0_.D1.80.D1.8B.D0.BD.D0.BA.D0.B5"></span><span class="mw-headline" id="Положение_на_рынке">Положение на рынке</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Athlon_XP&amp;veaction=edit&amp;section=12" title="Редактировать раздел «Положение на рынке»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Athlon_XP&amp;action=edit&amp;section=12" title="Редактировать код раздела «Положение на рынке»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Athlon XP являлся флагманским процессором компании <a href="/wiki/AMD" title="AMD">AMD</a> для настольных компьютеров с момента выхода в октябре <a href="/wiki/2001_%D0%B3%D0%BE%D0%B4" title="2001 год">2001 года</a> и до появления на рынке процессора <a href="/wiki/Athlon_64" title="Athlon 64">Athlon 64</a> в сентябре <a href="/wiki/2003_%D0%B3%D0%BE%D0%B4" title="2003 год">2003 года</a>. В момент своего выхода процессоры Athlon XP занимали верхнюю ценовую нишу, затем постепенно расширяли своё присутствие на рынке, вытесняя с него бюджетные процессоры семейства <a href="/wiki/Duron" title="Duron">Duron</a>. После выхода процессоров <a href="/wiki/Athlon_64" title="Athlon 64">Athlon 64</a> младшие модели Athlon XP заняли нижнюю ценовую нишу, а старшие — среднюю. В июле <a href="/wiki/2004_%D0%B3%D0%BE%D0%B4" title="2004 год">2004 года</a> на смену процессорам Athlon XP и Duron пришло новое семейство бюджетных процессоров — <a href="/wiki/Sempron" title="Sempron">Sempron</a>. При этом младшие модели Sempron представляли собой процессоры Athlon XP на ядрах Thoroughbred, Barton и Thorton с изменёнными рейтингами производительности: рейтинги процессоров Sempron соответствовали частотам конкурирующих с ними процессоров <a href="/wiki/Intel" title="Intel">Intel</a> <a href="/wiki/Celeron" title="Celeron">Celeron</a> (так, например, характеристики процессора Athlon XP 3200+ соответствовали характеристикам Sempron 3300+).
</p>
<h3><span id=".D0.A0.D1.8B.D0.BD.D0.BE.D1.87.D0.BD.D1.8B.D0.B5_.D0.BF.D0.BE.D0.B7.D0.B8.D1.86.D0.B8.D0.B8_.D0.BA.D0.BE.D0.BC.D0.BF.D0.B0.D0.BD.D0.B8.D0.B8_AMD_.282001.E2.80.942003_.D0.B3.D0.BE.D0.B4.D1.8B.29"></span><span class="mw-headline" id="Рыночные_позиции_компании_AMD_(2001—2003_годы)">Рыночные позиции компании AMD (2001—2003 годы)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Athlon_XP&amp;veaction=edit&amp;section=13" title="Редактировать раздел «Рыночные позиции компании AMD (2001—2003 годы)»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Athlon_XP&amp;action=edit&amp;section=13" title="Редактировать код раздела «Рыночные позиции компании AMD (2001—2003 годы)»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Благодаря модернизации процессоров семейства <a href="/wiki/Duron" title="Duron">Duron</a> и выходу первых процессоров Athlon XP, не уступавших конкурирующим процессорам компании Intel, а зачастую их опережавших, в 2001 году компании AMD удалось увеличить свою долю на рынке процессоров архитектуры <a href="/wiki/X86" title="X86">x86</a> с 18 % до 22 %, а объём продаж процессоров — с 2,34 до 2,42 млрд долларов.<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup><sup class="reference" id="cite_ref-report02_23-0"><a href="#cite_note-report02-23">[23]</a></sup>
</p><p>В 2002 году компания AMD столкнулась с рядом трудностей. Технологические проблемы, не позволявшие массово выпускать процессоры Athlon XP на новом ядре (Thoroughbred) до середины 2002 года (выход этих процессоров был запланирован на начало 2002 года), а также несколько более высокая производительность конкурирующих процессоров, привели к значительному сокращению доли рынка компании AMD и снижению объёма продаж процессоров. В середине 2002 года присутствие компании на рынке процессоров сократилось до 18 %, а к концу 2002 года доля AMD составляла уже около 14 %. Объём продаж процессоров за год сократился более чем на 30 % (до 1,75 млрд долларов).<sup class="reference" id="cite_ref-iamd_9-1"><a href="#cite_note-iamd-9">[9]</a></sup><sup class="reference" id="cite_ref-report02_23-1"><a href="#cite_note-report02-23">[23]</a></sup><sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup>
</p><p>Снижение объёма продаж процессоров в 2002 году, помимо технологических проблем, было вызвано также и тем, что для успешной конкуренции с процессорами <a href="/wiki/Intel" title="Intel">Intel</a> <a href="/wiki/Pentium_4" title="Pentium 4">Pentium 4</a>, компания AMD была вынуждена продавать процессоры Athlon XP по более низким ценам, чем равные по производительности процессоры основного конкурента.<sup class="reference" id="cite_ref-costvsp4_25-0"><a href="#cite_note-costvsp4-25">[25]</a></sup> Поэтому, несмотря на популярность процессоров Athlon XP среди пользователей (в частности, среди <a class="mw-redirect" href="/wiki/%D0%9E%D0%B2%D0%B5%D1%80%D0%BA%D0%BB%D0%BE%D0%BA%D0%B5%D1%80" title="Оверклокер">оверклокеров</a><sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup>), компания AMD несла значительные убытки.<sup class="reference" id="cite_ref-report02_23-2"><a href="#cite_note-report02-23">[23]</a></sup>
</p><p>К концу 2002 года компании AMD удалось наладить выпуск процессоров на ядре Thoroughbred, а в феврале 2003 года были анонсированы процессоры на ядре Barton. Объём продаж процессоров в 2003 году увеличился до 1,96 млрд долларов, а доля компании на рынке к осени 2003 года, когда были анонсированы первые процессоры архитектуры <a href="/wiki/Athlon_64" title="Athlon 64">K8</a>, составила около 16 %.<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup><sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup>
</p>
<h2><span id=".D0.A1.D1.80.D0.B0.D0.B2.D0.BD.D0.B5.D0.BD.D0.B8.D0.B5_.D1.81_.D0.BA.D0.BE.D0.BD.D0.BA.D1.83.D1.80.D0.B5.D0.BD.D1.82.D0.B0.D0.BC.D0.B8"></span><span class="mw-headline" id="Сравнение_с_конкурентами">Сравнение с конкурентами</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Athlon_XP&amp;veaction=edit&amp;section=14" title="Редактировать раздел «Сравнение с конкурентами»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Athlon_XP&amp;action=edit&amp;section=14" title="Редактировать код раздела «Сравнение с конкурентами»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Параллельно с Athlon XP существовали следующие x86-процессоры:
</p>
<ul><li>AMD <a href="/wiki/Duron" title="Duron">Duron</a> (Morgan и Applebred). Предназначался для рынка недорогих настольных компьютеров. Уступал процессорам Athlon XP за счёт меньшего объёма кэша второго уровня и за счёт менее быстрой системной шины.</li>
<li>AMD <a href="/wiki/Athlon_64" title="Athlon 64">Athlon 64</a> (ClawHammer, SledgeHammer и NewCastle). Пришёл на смену Athlon XP в качестве процессора для высокопроизводительных систем. Опережал Athlon XP за счёт более прогрессивной архитектуры, поддержки <a href="/wiki/SSE2" title="SSE2">SSE2</a> и встроенного контроллера памяти.</li>
<li>Intel <a href="/wiki/Pentium_III" title="Pentium III">Pentium III</a> (Tualatin). Конкурировал с процессорами Athlon и Athlon MP (на рынке двухпроцессорных систем). Опережал конкурентов при работе в оптимизированных приложениях (например, в <a href="/wiki/Adobe_Photoshop" title="Adobe Photoshop">Adobe Photoshop</a>), а также в приложениях трёхмерного моделирования за счёт более удачной реализации <a href="/wiki/AGP" title="AGP">AGP</a> в чипсетах Intel, уступая им в задачах, требующих высокой пропускной способности памяти (ПСП), и <a class="mw-redirect" href="/wiki/FPU" title="FPU">вычислениях с плавающей запятой</a>.<sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[29]</a></sup></li>
<li>Intel <a href="/wiki/Pentium_4" title="Pentium 4">Pentium 4</a> (Northwood). Серьёзно уступал Athlon XP на равных частотах, однако за счёт архитектуры NetBurst имел значительно более высокий частотный потенциал. В связи с этим компания AMD была вынуждена ввести рейтинговую систему обозначения моделей для процессоров Athlon XP (более высокая тактовая частота Pentium 4 давала ему серьёзное преимущество с точки зрения маркетинга: потребители «покупают мегагерцы»). Pentium 4 опережали «равнорейтинговые» Athlon XP в приложениях, оптимизированных под архитектуру NetBurst, требовавших наличие поддержки инструкций <a href="/wiki/SSE2" title="SSE2">SSE2</a> или высокой ПСП, однако значительно уступали в вычислениях с плавающей запятой и неоптимизированных приложениях. За счёт поддержки технологии <a class="mw-redirect" href="/wiki/Hyper-threading" title="Hyper-threading">HyperThreading</a>, старшие Pentium 4 опережали Athlon XP в приложениях, поддерживающих многопроцессорность.</li>
<li>Intel Pentium 4 (Prescott). Появился на рынке после выхода Athlon 64, был нацелен на рынок высокопроизводительных систем и поэтому с процессорами Athlon XP напрямую не конкурировал. Производительность старших моделей процессоров Pentium 4 на ядре Prescott была значительно выше, чем производительность Athlon XP, за счёт увеличения максимальной тактовой частоты по сравнению с процессорами Pentium 4 на ядре Northwood.</li>
<li>Intel <a href="/wiki/Pentium_M" title="Pentium M">Pentium M</a> и <a href="/wiki/Celeron#Celeron_M" title="Celeron">Celeron M</a>. Являлись дальнейшим развитием процессоров Pentium III. Предназначались для мобильных компьютеров, обладали низким энергопотреблением и тепловыделением. Pentium M опережал мобильные процессоры Athlon XP в большинстве задач.<sup class="reference" id="cite_ref-30"><a href="#cite_note-30">[30]</a></sup> При использовании в настольных компьютерах (с помощью специального переходника) Pentium M также опережал и настольные процессоры Athlon XP.<sup class="reference" id="cite_ref-31"><a href="#cite_note-31">[31]</a></sup> Процессор Celeron M имел близкую к Pentium M производительность, незначительно отставая от него.</li>
<li>Intel <a href="/wiki/Celeron" title="Celeron">Celeron</a> (Tualatin-256, Willamette-128 и Northwood-128). Предназначался для рынка недорогих настольных компьютеров. Конкурировал с процессорами AMD Duron. Уступал процессорам Athlon XP в подавляющем большинстве задач.</li>
<li>Intel <a href="/wiki/Celeron#Prescott-256" title="Celeron">Celeron D</a> (Prescott-256). Конкурировал в основном с процессорами AMD Sempron. В целом приблизительно соответствовал «равнорейтинговому» Sempron и уступал Athlon XP. Был быстрее конкурентов в кодировании видео и архивировании, уступал им в играх.<sup class="reference" id="cite_ref-lowend_20-1"><a href="#cite_note-lowend-20">[20]</a></sup></li>
<li><a href="/wiki/VIA_C3" title="VIA C3">VIA C3</a> (Nehemiah) и <a class="new" href="/w/index.php?title=VIA_Eden&amp;action=edit&amp;redlink=1" title="VIA Eden (страница отсутствует)">VIA Eden</a>. Предназначались для компьютеров с низким энергопотреблением и ноутбуков (C3 и Eden-N) и для интегрирования в системные платы (Eden), имели низкую производительность и уступали конкурирующим процессорам.</li>
<li><a href="/wiki/Transmeta" title="Transmeta">Transmeta</a> <a href="/wiki/Efficeon" title="Efficeon">Efficeon</a>. Предназначался для ноутбуков, имел низкое энергопотребление и тепловыделение. Уступал в большинстве задач мобильным процессорам AMD и Intel, опережая мобильные процессоры VIA.<sup class="reference" id="cite_ref-32"><a href="#cite_note-32">[32]</a></sup></li></ul>
<figure class="mw-default-size" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Pentium4_northwood.png"><img class="mw-file-element" data-file-height="268" data-file-width="300" decoding="async" height="197" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/9b/Pentium4_northwood.png/220px-Pentium4_northwood.png" srcset="//upload.wikimedia.org/wikipedia/commons/9/9b/Pentium4_northwood.png 1.5x" width="220"/></a><figcaption>Основной конкурент Athlon XP — Intel Pentium 4 на ядре Northwood</figcaption></figure>
<p>Благодаря более низкой по сравнению с конкурентами цене<sup class="reference" id="cite_ref-costvsp4_25-1"><a href="#cite_note-costvsp4-25">[25]</a></sup> и достаточно высокой производительности, процессоры Athlon XP пользовались популярностью среди опытных пользователей, многие из которых приобретали недорогие младшие модели с целью эксплуатации в <a class="mw-redirect" href="/wiki/%D0%9E%D0%B2%D0%B5%D1%80%D0%BA%D0%BB%D0%BE%D0%BA%D0%B8%D0%BD%D0%B3" title="Оверклокинг">нештатных режимах</a>, так как это позволяло достичь производительность старшей модели по значительно меньшей цене. Так, например, производительность популярного среди <a class="mw-redirect" href="/wiki/%D0%9E%D0%B2%D0%B5%D1%80%D0%BA%D0%BB%D0%BE%D0%BA%D0%B5%D1%80" title="Оверклокер">оверклокеров</a> процессора Athlon XP 2500+ при повышении частоты системной шины с 333 до 400 МГц оказывалась равной производительности процессора Athlon XP 3200+ при значительно более низкой стоимости.<sup class="reference" id="cite_ref-33"><a href="#cite_note-33">[33]</a></sup><sup class="reference" id="cite_ref-34"><a href="#cite_note-34">[34]</a></sup><sup class="reference" id="cite_ref-cost_6-4"><a href="#cite_note-cost-6">[6]</a></sup>
</p><p>Высокая производительность процессоров Athlon XP в задачах, использующих <a class="mw-redirect" href="/wiki/FPU" title="FPU">вычисления с плавающей запятой</a>, позволяла эффективно использовать их не только в персональных компьютерах, ноутбуках и серверах, но и в <a href="/wiki/%D0%A1%D1%83%D0%BF%D0%B5%D1%80%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80" title="Суперкомпьютер">суперкомпьютерах</a>. Так, например, <a href="/wiki/%D0%9A%D0%BB%D0%B0%D1%81%D1%82%D0%B5%D1%80_(%D0%B3%D1%80%D1%83%D0%BF%D0%BF%D0%B0_%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BE%D0%B2)" title="Кластер (группа компьютеров)">кластер</a> Presto III, построенный в Токийском институте технологий (GSIC Center, Tokyo Institute of Technology) в 2000 году, изначально содержал 78 процессоров <a href="/wiki/Athlon" title="Athlon">Athlon</a>. Позже он был модернизирован и с 480 процессорами Athlon MP, работавшими на 1600 МГц, занял 47 место в списке <a class="mw-redirect" href="/wiki/TOP500" title="TOP500">TOP500</a> за июнь <a href="/wiki/2002_%D0%B3%D0%BE%D0%B4" title="2002 год">2002 года</a>.<sup class="reference" id="cite_ref-35"><a href="#cite_note-35">[35]</a></sup>
</p><p>Однако, несмотря на свои достоинства, Athlon XP не был популярен среди большинства пользователей, особенно на корпоративном рынке, по ряду причин, в частности, из-за агрессивной <a href="/wiki/%D0%A0%D0%B5%D0%BA%D0%BB%D0%B0%D0%BC%D0%B0" title="Реклама">рекламной</a> и <a href="/wiki/%D0%9C%D0%B0%D1%80%D0%BA%D0%B5%D1%82%D0%B8%D0%BD%D0%B3" title="Маркетинг">маркетинговой политики</a> компании <a href="/wiki/Intel" title="Intel">Intel</a><sup class="reference" id="cite_ref-36"><a href="#cite_note-36">[36]</a></sup><sup class="reference" id="cite_ref-37"><a href="#cite_note-37">[37]</a></sup> в сочетании с неудачной маркетинговой политикой компании AMD, которая из-за высокой тактовой частоты процессоров конкурента была вынуждена ввести рейтинг производительности процессоров Athlon XP, часто вводивший неопытных пользователей в заблуждение<sup class="reference" id="cite_ref-38"><a href="#cite_note-38">[38]</a></sup>, а из-за финансовых проблем не могла эффективно рекламировать свои процессоры.
</p>

<p>Процессоры Athlon XP, в отличие от процессоров Athlon, имели встроенные средства измерения температуры ядра. Однако термозащита процессоров (отключение питания при перегреве) осуществлялась средствами материнской платы. Некоторые производители материнских плат, особенно в первое время после начала выпуска процессоров Athlon XP, нарушали рекомендации AMD по термозащите, что делало защиту неэффективной при включении без радиатора или разрушении его крепления. В некоторых случаях измерение температуры материнской платой осуществлялось не с помощью встроенного термодиода процессора, а с помощью термодатчика, расположенного под процессором («подсокетный датчик»), и отличалось низкой точностью. В ряде случаев датчик не контактировал с корпусом процессора, а измерял температуру воздуха возле процессора. Тем не менее эффективность термозащиты в процессорах Athlon XP была достаточной для защиты процессора в обычных условиях эксплуатации, защищая от таких ситуаций, как остановка кулера. В то же время установка процессора требовала некоторой квалификации: при неправильной установке кулера были возможны механические и тепловые повреждения (например, в том случае, если перекос радиатора не привёл к выходу процессора из строя вследствие скола, отсутствие контакта между кристаллом процессора и радиатором может привести к тепловым повреждениям процессора). Распространённое среди неопытных пользователей мнение о ненадёжности процессоров Athlon XP было связано со случаями неправильной установки процессора, с агрессивными действиями (так, например, в известном видеоролике Томаса Пабста<sup class="reference" id="cite_ref-39"><a href="#cite_note-39">[39]</a></sup> была представлена малореальная ситуация полного отказа системы охлаждения), а также недостатком доступных в продаже эффективных и удобных в установке кулеров в первое время после выхода процессоров K7. С появлением эффективных кулеров проблема охлаждения процессоров K7 перестала существовать.
</p><p>Несмотря на то что ситуация с полным отказом системы охлаждения (например, в случае разрушения крепления <a class="mw-disambig" href="/wiki/%D0%A0%D0%B0%D0%B4%D0%B8%D0%B0%D1%82%D0%BE%D1%80" title="Радиатор">радиатора</a>), смоделированная в экспериментах, маловероятна, а в случае возникновения приводит к более серьёзным последствиям (например, к разрушению плат расширения или системной платы в результате падения на них радиатора) вне зависимости от модели процессора<sup class="reference" id="cite_ref-thermal_7-1"><a href="#cite_note-thermal-7">[7]</a></sup>, результаты эксперимента Томаса Пабста отрицательно повлияли на популярность процессоров AMD, а мнение о их ненадёжности получило широкое распространение. Даже после выхода процессоров <a href="/wiki/Athlon_64" title="Athlon 64">Athlon 64</a>, имеющих более эффективную систему защиты от перегрева, а также теплораспределительную крышку, защищающую кристалл от сколов, многие пользователи по-прежнему использовали в качестве аргумента в пользу процессоров Pentium 4 ненадёжность процессоров компании AMD.<sup class="reference" id="cite_ref-40"><a href="#cite_note-40">[40]</a></sup>
</p><p>Тепловыделение Athlon XP (<a href="#Технические_характеристики">72—79 Вт</a>) значительно превышало тепловыделение процессоров Pentium III (<a href="/wiki/Pentium_III#Технические_характеристики" title="Pentium III">33 Вт</a>), поэтому многие пользователи ошибочно считали, что процессоры Pentium 4 также выделяют меньше тепла, чем процессоры Athlon XP. Однако в действительности тепловыделение Athlon XP было несколько ниже, чем у Pentium 4 (<a href="/wiki/Pentium_4#Технические_характеристики_различных_ядер" title="Pentium 4">75—89 Вт</a>).<sup class="reference" id="cite_ref-41"><a href="#cite_note-41">[41]</a></sup> Кроме того, задействование режима «Bus Disconnect» позволяло значительно снизить температуру процессора в моменты простоя или неполной загрузки за счёт отключения буферов системной шины. Для включения данного режима требовалась либо поддержка его системной платой, либо специальное программное обеспечение.<sup class="reference" id="cite_ref-42"><a href="#cite_note-42">[42]</a></sup><sup class="reference" id="cite_ref-43"><a href="#cite_note-43">[43]</a></sup>
</p>

<h2><span id=".D0.9D.D0.B0.D0.B8.D0.BC.D0.B5.D0.BD.D0.BE.D0.B2.D0.B0.D0.BD.D0.B8.D0.B5_.D0.BC.D0.BE.D0.B4.D0.B5.D0.BB.D0.B5.D0.B9"></span><span class="mw-headline" id="Наименование_моделей">Наименование моделей</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Athlon_XP&amp;veaction=edit&amp;section=16" title="Редактировать раздел «Наименование моделей»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Athlon_XP&amp;action=edit&amp;section=16" title="Редактировать код раздела «Наименование моделей»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Маркировка процессоров Athlon XP и Athlon MP состоит из трёх строк. Первая строка является наименованием модели (<i>Ordering Part Number, OPN</i>), вторая содержит информацию о ревизии ядра процессора (<i>степпинг-код</i>, первые пять символов строки) и дате его выпуска (год и неделя, два двухзначных числа после степпинг-кода), третья — информацию о партии процессоров. У процессоров, маркировка которых располагается на наклейке, третья строка расположена напротив первой во второй колонке.

<h2><span id=".D0.98.D0.B7.D0.BC.D0.B5.D0.BD.D0.B5.D0.BD.D0.B8.D0.B5_.D0.BF.D0.B0.D1.80.D0.B0.D0.BC.D0.B5.D1.82.D1.80.D0.BE.D0.B2_.D0.BF.D1.80.D0.BE.D1.86.D0.B5.D1.81.D1.81.D0.BE.D1.80.D0.B0"></span><span class="mw-headline" id="Изменение_параметров_процессора">Изменение параметров процессора</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Athlon_XP&amp;veaction=edit&amp;section=19" title="Редактировать раздел «Изменение параметров процессора»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Athlon_XP&amp;action=edit&amp;section=19" title="Редактировать код раздела «Изменение параметров процессора»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Такие параметры процессоров Athlon XP/MP, как <a class="mw-redirect" href="/wiki/%D0%A2%D0%B0%D0%BA%D1%82%D0%BE%D0%B2%D0%B0%D1%8F_%D1%87%D0%B0%D1%81%D1%82%D0%BE%D1%82%D0%B0" title="Тактовая частота">тактовая частота</a>, <a href="/wiki/%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%BE%D0%B5_%D0%BD%D0%B0%D0%BF%D1%80%D1%8F%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5" title="Электрическое напряжение">напряжение</a> питания, объём включённой кэш-памяти второго уровня, поддержка многопроцессорности, тип процессора (мобильный/настольный) и частота системной шины задаются с помощью нескольких групп <a href="/wiki/%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%BE%D0%BD%D1%82%D0%B0%D0%BA%D1%82" title="Электрический контакт">контактов</a>, расположенных на подложке процессора. Контакты могут быть либо замкнуты, либо пережжены <a href="/wiki/%D0%9B%D0%B0%D0%B7%D0%B5%D1%80" title="Лазер">лазером</a> в процессе производства процессора.
</p><p>Расположение контактов на подложке позволяет пользователю изменять параметры процессора без использования специального оборудования, соединяя разорванные контакты, либо перерезая замкнутые, в том случае, если упаковка процессора позволяет производить такие манипуляции.
</p><p>Ниже приведён список групп контактов и их функциональность для процессоров Athlon XP/MP на различных ядрах и процессора Athlon 4.
</p>
<dl><dt>Palomino, Corvette</dt></dl>
<ul><li>L1 — линии, ответственные за изменение коэффициента умножения.</li>
<li>L3, L4, L10 — коэффициент умножения настольных процессоров.</li>
<li>L5 — работа в многопроцессорных системах, выбор типа процессора (мобильный/настольный).</li>
<li>L6 — максимальный коэффициент умножения мобильных процессоров.</li>
<li>L11 — напряжение питания.</li></ul>
<dl><dt>Thoroughbred, Barton и Thorton</dt></dl>
<ul><li>L1 — линии, ответственные за изменение коэффициента умножения.</li>
<li>L2, L9 — объём включённой кэш-памяти второго уровня.</li>
<li>L3 — коэффициент умножения.</li>
<li>L5 — работа в многопроцессорных системах, выбор типа процессора (мобильный/настольный).</li>
<li>L6 — коэффициент умножения мобильных процессоров.</li>
<li>L11 — напряжение питания.</li>
<li>L12 — частота системной шины.</li></ul>
<p>Также возможно изменение коэффициента умножения путём замыкания контактов разъёма <a href="/wiki/Socket_A" title="Socket A">Socket A</a>. Данный способ работает в том случае, если изменение коэффициента умножения не заблокировано. Существует также специальное устройство, устанавливающееся между процессором и гнездовым разъёмом и позволяющее изменять коэффициент умножения процессоров Athlon XP со свободным множителем.<sup class="reference" id="cite_ref-45"><a href="#cite_note-45">[45]</a></sup>
</p><p>В поздних процессорах на ядрах Thoroughbred, Barton и Thorton, выпущенных после 39 недели <a href="/wiki/2003_%D0%B3%D0%BE%D0%B4" title="2003 год">2003 года</a> (а также в некоторых процессорах, выпущенных после 34 недели), коэффициент умножения жёстко зафиксирован и не может быть разблокирован обычным способом с помощью контактов группы L1. Однако существует возможность изменения типа процессора на «мобильный» с возможностью изменения коэффициента умножения.<sup class="reference" id="cite_ref-46"><a href="#cite_note-46">[46]</a></sup> Данный способ работает только на системных платах с <a href="/wiki/%D0%A7%D0%B8%D0%BF%D1%81%D0%B5%D1%82" title="Чипсет">чипсетом</a>, поддерживающим изменение множителя во время работы.<sup class="reference" id="cite_ref-47"><a href="#cite_note-47">[47]</a></sup>
</p><p>Дата выпуска процессора определяется по второй строке маркировки: четырёхзначное число после буквенного кода содержит информацию о годе и неделе выпуска. Так, например, процессор с маркировкой «MIRGA0337VPMW» выпущен на 37 неделе <a href="/wiki/2003_%D0%B3%D0%BE%D0%B4" title="2003 год">2003 года</a>.
</p>
<h2><span id=".D0.98.D1.81.D0.BF.D1.80.D0.B0.D0.B2.D0.BB.D0.B5.D0.BD.D0.BD.D1.8B.D0.B5_.D0.BE.D1.88.D0.B8.D0.B1.D0.BA.D0.B8"></span><span class="mw-headline" id="Исправленные_ошибки">Исправленные ошибки</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Athlon_XP&amp;veaction=edit&amp;section=20" title="Редактировать раздел «Исправленные ошибки»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Athlon_XP&amp;action=edit&amp;section=20" title="Редактировать код раздела «Исправленные ошибки»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Процессор представляет собой сложное микроэлектронное устройство, что не позволяет исключить возможность его некорректной работы. Ошибки появляются на этапе проектирования и могут быть исправлены обновлениями микрокода процессора (заменой <a href="/wiki/BIOS" title="BIOS">BIOS</a> системной платы на более новую версию), либо выпуском новой ревизии ядра процессора.
</p><p>В процессорах Athlon XP на ядре Palomino и Athlon 4 обнаружено 10 различных ошибок, 2 из которых исправлены в ревизии A5:
</p>
<ul><li>Ошибка при очистке страницы TLB (4 Мб) с помощью инструкции INVLPG.</li>
<li>Ошибка при изменении частоты в мобильных процессорах.</li></ul>
<p>Процессоры Athlon XP/MP на ядре Thoroughbred содержали 8 незначительных ошибок, которые либо не возникали в условиях реальной работы, либо не влияли на её стабильность, либо исправлялись программно, либо обходились чипсетом. При переходе на ядро Barton были исправлены 2 ошибки:
</p>
<ul><li>Ошибка, возникающая при чтении счётчиков производительности при вводе некорректного номера счётчика.</li>
<li>Ошибка при вызове <a href="/wiki/System_Management_Mode#.D0.90.D0.BA.D1.82.D0.B8.D0.B2.D0.B0.D1.86.D0.B8.D1.8F_SMM" title="System Management Mode">SMI</a> в режиме отладки (debug mode).</li></ul>
