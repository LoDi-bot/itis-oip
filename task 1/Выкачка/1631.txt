<p><b>Гипсилофодон</b><sup class="reference" id="cite_ref-пол22_1-0"><a href="#cite_note-пол22-1">[1]</a></sup> (<a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Латинский язык">лат.</a> <span lang="la" style="font-style:italic;">Hypsilóphodon</span>) — <a href="/wiki/%D0%A0%D0%BE%D0%B4_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Род (биология)">род</a> <a class="mw-redirect" href="/wiki/%D0%A0%D0%B0%D1%81%D1%82%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D0%BE%D1%8F%D0%B4%D0%BD%D1%8B%D0%B5_%D0%B4%D0%B8%D0%BD%D0%BE%D0%B7%D0%B0%D0%B2%D1%80%D1%8B" title="Растительноядные динозавры">растительноядных</a> <a href="/wiki/%D0%94%D0%B8%D0%BD%D0%BE%D0%B7%D0%B0%D0%B2%D1%80%D1%8B" title="Динозавры">динозавров</a>, живших во время <a href="/wiki/%D0%9D%D0%B8%D0%B6%D0%BD%D0%B8%D0%B9_%D0%BC%D0%B5%D0%BB" title="Нижний мел">раннемеловой эпохи</a> (<a href="/wiki/%D0%93%D0%BE%D1%82%D0%B5%D1%80%D0%B8%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D1%80%D1%83%D1%81" title="Готеривский ярус">готеривский</a> — <a href="/wiki/%D0%91%D0%B0%D1%80%D1%80%D0%B5%D0%BC%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D1%80%D1%83%D1%81" title="Барремский ярус">барремский</a> века, 130—125 миллионов лет назад<sup class="reference" id="cite_ref-:2_2-0"><a href="#cite_note-:2-2">[2]</a></sup>). Известен по <a href="/wiki/%D0%A4%D0%BE%D1%81%D1%81%D0%B8%D0%BB%D0%B8%D0%B8" title="Фоссилии">ископаемым остаткам</a>, обнаруженным на <a class="mw-redirect" href="/wiki/%D0%9E%D1%81%D1%82%D1%80%D0%BE%D0%B2_%D0%A3%D0%B0%D0%B9%D1%82" title="Остров Уайт">острове Уайт</a> в <a href="/wiki/%D0%AE%D0%B6%D0%BD%D0%B0%D1%8F_%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D1%8F" title="Южная Англия">Южной Англии</a>. В настоящее время выделяется только <a href="/wiki/%D0%9C%D0%BE%D0%BD%D0%BE%D1%82%D0%B8%D0%BF%D0%B8%D1%8F_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%81%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0%D1%82%D0%B8%D0%BA%D0%B0)" title="Монотипия (биологическая систематика)">один</a> <a href="/wiki/%D0%91%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B2%D0%B8%D0%B4" title="Биологический вид">вид</a> этого рода динозавров — <i><b>Hypsilophodon foxii</b></i>. Гипсилофодон достигал длины 2,3 метра и высоты 1,7 метра при массе до 70 кг. Впервые как новый род динозавров его описал <a class="mw-redirect" href="/wiki/%D0%A5%D0%B0%D0%BA%D1%81%D0%BB%D0%B8,_%D0%A2%D0%BE%D0%BC%D0%B0%D1%81_%D0%93%D0%B5%D0%BD%D1%80%D0%B8" title="Хаксли, Томас Генри">Томас Генри Хаксли</a> (или Гексли<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>) в <a href="/wiki/1869_%D0%B3%D0%BE%D0%B4_%D0%B2_%D0%BD%D0%B0%D1%83%D0%BA%D0%B5" title="1869 год в науке">1869 году</a><sup class="reference" id="cite_ref-NSWC04_4-0"><a href="#cite_note-NSWC04-4">[4]</a></sup>.
</p><p>В некоторых источниках роду приписываются ископаемые остатки спорной <a href="/wiki/%D0%91%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%81%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0%D1%82%D0%B8%D0%BA%D0%B0" title="Биологическая систематика">систематической</a> принадлежности, обнаруженные на территории <a href="/wiki/%D0%98%D1%81%D0%BF%D0%B0%D0%BD%D0%B8%D1%8F" title="Испания">Испании</a>, <a href="/wiki/%D0%A0%D1%83%D0%BC%D1%8B%D0%BD%D0%B8%D1%8F" title="Румыния">Румынии</a>, <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%8F" title="Франция">Франции</a> и <a href="/wiki/%D0%AF%D0%BF%D0%BE%D0%BD%D0%B8%D1%8F" title="Япония">Японии</a><sup class="reference" id="cite_ref-:PBDB_5-0"><a href="#cite_note-:PBDB-5">[5]</a></sup>.
</p>

<h2><span id=".D0.AD.D1.82.D0.B8.D0.BC.D0.BE.D0.BB.D0.BE.D0.B3.D0.B8.D1.8F"></span><span class="mw-headline" id="Этимология">Этимология</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%93%D0%B8%D0%BF%D1%81%D0%B8%D0%BB%D0%BE%D1%84%D0%BE%D0%B4%D0%BE%D0%BD&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Этимология»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%93%D0%B8%D0%BF%D1%81%D0%B8%D0%BB%D0%BE%D1%84%D0%BE%D0%B4%D0%BE%D0%BD&amp;action=edit&amp;section=1" title="Редактировать код раздела «Этимология»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Название <i>Hypsilophodon</i> составлено из <a href="/wiki/%D0%94%D1%80%D0%B5%D0%B2%D0%BD%D0%B5%D0%B3%D1%80%D0%B5%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Древнегреческий язык">др.-греч.</a> <span lang="grc"><span style="font-family: palatino linotype, new athena unicode, athena, gentium, code2000, serif; font-size: 105%;">ὕψι</span></span> — «вверх, кверху», <span lang="grc"><span style="font-family: palatino linotype, new athena unicode, athena, gentium, code2000, serif; font-size: 105%;">λόφος</span></span> — «гребень», <span lang="grc"><span style="font-family: palatino linotype, new athena unicode, athena, gentium, code2000, serif; font-size: 105%;">ὀδούς</span></span> — «зуб»<sup class="reference" id="cite_ref-isbn5-353-01605-X_6-0"><a href="#cite_note-isbn5-353-01605-X-6">[6]</a></sup>. Такое название Хаксли дал гипсилофодону за особое строение зубов. Они были вытянутыми, неровными по краям и прекрасно подходили для того, чтобы резать листву и некоторые тонкие ветки<sup class="reference" id="cite_ref-autogenerated1_7-0"><a href="#cite_note-autogenerated1-7">[7]</a></sup>.
</p>
<h2><span id=".D0.9E.D0.BF.D0.B8.D1.81.D0.B0.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Описание">Описание</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%93%D0%B8%D0%BF%D1%81%D0%B8%D0%BB%D0%BE%D1%84%D0%BE%D0%B4%D0%BE%D0%BD&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Описание»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%93%D0%B8%D0%BF%D1%81%D0%B8%D0%BB%D0%BE%D1%84%D0%BE%D0%B4%D0%BE%D0%BD&amp;action=edit&amp;section=2" title="Редактировать код раздела «Описание»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-default-size mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:HypsilophodonBrussels.jpg"><img class="mw-file-element" data-file-height="725" data-file-width="1353" decoding="async" height="118" src="//upload.wikimedia.org/wikipedia/commons/thumb/2/29/HypsilophodonBrussels.jpg/220px-HypsilophodonBrussels.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/2/29/HypsilophodonBrussels.jpg/330px-HypsilophodonBrussels.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/2/29/HypsilophodonBrussels.jpg/440px-HypsilophodonBrussels.jpg 2x" width="220"/></a><figcaption>Копия скелета гипсилофодона, смонтированная в соответствии с современными представлениями об анатомии динозавра</figcaption></figure>
<p>Гипсилофодон передвигался на двух длинных лёгких задних конечностях, перебирая ими очень быстро и делая достаточно большие шаги. Скорее всего, задние лапы гипсилофодона были похожи на лапы современной нелетающей птицы, например, на лапы <a href="/wiki/%D0%AD%D0%BC%D1%83" title="Эму">эму</a>. На концах этих задних лап находилось четыре вытянутых пальца, также способствовавших скорости и помогавших ему сохранять равновесие при беге<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>. Также он имел очень сильные мышцы таза и бёдер<sup class="reference" id="cite_ref-autogenerated1_7-1"><a href="#cite_note-autogenerated1-7">[7]</a></sup>. Гипсилофодон имел длинный хвост, служивший ему как рулём для управления телом, так и для обороны: если за ним гнался не слишком большой хищник, он мог сбить его с ног мощным ударом<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>. Кости его скелета лёгкие<sup class="reference" id="cite_ref-autogenerated1_7-2"><a href="#cite_note-autogenerated1-7">[7]</a></sup>. Всё это помогало гипсилофодону развивать высокую скорость — до 50 км/ч, хотя сам он был довольно маленьким животным<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>. Из-за этого его ещё иногда называют «газелью мира динозавров». В основном, высокую скорость он развивал для того, чтобы спастись от таких хищников, как <a href="/wiki/%D0%90%D0%BB%D1%8C%D1%82%D0%B8%D1%81%D0%BF%D0%B8%D0%BD%D0%B0%D0%BA%D1%81" title="Альтиспинакс">альтиспинакс</a><sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup><sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>.
</p>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Hypsilophodon_foxii.jpg"><img class="mw-file-element" data-file-height="1484" data-file-width="2039" decoding="async" height="146" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/3e/Hypsilophodon_foxii.jpg/200px-Hypsilophodon_foxii.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/3e/Hypsilophodon_foxii.jpg/300px-Hypsilophodon_foxii.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/3/3e/Hypsilophodon_foxii.jpg/400px-Hypsilophodon_foxii.jpg 2x" width="200"/></a><figcaption>Реконструкция скелета гипсилофодона, выполненная <a class="mw-redirect" href="/wiki/%D0%9C%D0%B0%D1%80%D1%88,_%D0%93%D0%BE%D1%84%D0%BE%D0%BD%D0%B8%D0%B8%D0%BB_%D0%A7%D0%B0%D1%80%D0%BB%D0%B7" title="Марш, Гофониил Чарлз">Чарлзом Маршем</a>, до 1923 года</figcaption></figure>
<p>Череп у него был глубокий и вытянутый, размером чуть больше ладони взрослого <a href="/wiki/%D0%A7%D0%B5%D0%BB%D0%BE%D0%B2%D0%B5%D0%BA" title="Человек">человека</a><sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup>. Ротовая полость гипсилофодона была характерна для <a href="/wiki/%D0%93%D0%B8%D0%BF%D1%81%D0%B8%D0%BB%D0%BE%D1%84%D0%BE%D0%B4%D0%BE%D0%BD%D1%82%D0%B8%D0%B4%D1%8B" title="Гипсилофодонтиды">гипсилофодонтид</a>: в самом начале располагался роговой <a href="/wiki/%D0%9A%D0%BB%D1%8E%D0%B2" title="Клюв">клюв</a>, позади него несколько передних <a href="/wiki/%D0%97%D1%83%D0%B1_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Зуб (биология)">зубов</a> на верхней стенке, а также жевательные зубы сзади. Общее количество зубов взрослой особи от 28 до 30 штук. Впадины в челюстях с обеих сторон позволяют предположить, что животное имело защёчные карманы, помогавшие удерживать пищу во рту при пережёвывании<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup>. <a class="mw-redirect" href="/wiki/%D0%97%D1%80%D0%B5%D0%BD%D0%B8%D0%B5" title="Зрение">Зрение</a> у гипсилофодона было отличным — об этом свидетельствуют два кольца, каждое из множества мелких косточек, располагавшихся вокруг глаз. Передние конечности гипсилофодона были короткими с пятью пальцами на каждой, на пальцах располагались маленькие коготки.
</p>
<h2><span id=".D0.9E.D0.B1.D1.80.D0.B0.D0.B7_.D0.B6.D0.B8.D0.B7.D0.BD.D0.B8"></span><span class="mw-headline" id="Образ_жизни">Образ жизни</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%93%D0%B8%D0%BF%D1%81%D0%B8%D0%BB%D0%BE%D1%84%D0%BE%D0%B4%D0%BE%D0%BD&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Образ жизни»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%93%D0%B8%D0%BF%D1%81%D0%B8%D0%BB%D0%BE%D1%84%D0%BE%D0%B4%D0%BE%D0%BD&amp;action=edit&amp;section=3" title="Редактировать код раздела «Образ жизни»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Из-за своего маленького размера гипсилофодоны, скорее всего, вели образ жизни, похожий на образ жизни современных африканских антилоп: жили стадами, употребляли в пищу только нежные листья и ветки низкорослых деревьев (иногда <a class="mw-redirect" href="/wiki/%D0%9C%D0%BE%D1%85" title="Мох">мох</a>), а от крупных хищников спасались быстрым бегом<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup><sup class="reference" id="cite_ref-isbn5-329-00819-0_16-0"><a href="#cite_note-isbn5-329-00819-0-16">[16]</a></sup>. Жили гипсилофодоны в зоне с умеренным климатом: летом было тепло и много растительности — это было время повышенной активности гипсилофодонов, они старались накопить как можно больше питательных веществ на зиму; зимой температура опускалась до −6 °С, очевидно, в это время года гипсилофодоны впадали в спячку<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup>. Скорее всего, манерой поведения они были похожи на <a class="mw-redirect" href="/wiki/%D0%9E%D1%82%D0%BD%D0%B8%D0%B5%D0%BB%D0%B8%D1%8F" title="Отниелия">отниелию</a>: находились ближе к более крупным травоядным животным, которые могли отпугнуть хищников<sup class="reference" id="cite_ref-isbn978-5-353-02642-6_18-0"><a href="#cite_note-isbn978-5-353-02642-6-18">[18]</a></sup>.
</p>
<h2><span id=".D0.98.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.8F_.D0.B8.D1.81.D1.81.D0.BB.D0.B5.D0.B4.D0.BE.D0.B2.D0.B0.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="История_исследования">История исследования</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%93%D0%B8%D0%BF%D1%81%D0%B8%D0%BB%D0%BE%D1%84%D0%BE%D0%B4%D0%BE%D0%BD&amp;veaction=edit&amp;section=4" title="Редактировать раздел «История исследования»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%93%D0%B8%D0%BF%D1%81%D0%B8%D0%BB%D0%BE%D1%84%D0%BE%D0%B4%D0%BE%D0%BD&amp;action=edit&amp;section=4" title="Редактировать код раздела «История исследования»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Hypsilophodon_model.jpg"><img class="mw-file-element" data-file-height="3456" data-file-width="2204" decoding="async" height="251" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/63/Hypsilophodon_model.jpg/160px-Hypsilophodon_model.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/63/Hypsilophodon_model.jpg/240px-Hypsilophodon_model.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/63/Hypsilophodon_model.jpg/320px-Hypsilophodon_model.jpg 2x" width="160"/></a><figcaption>Полноразмерная модель гипсилофодона, экспонируемая в <a href="/wiki/%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D0%BC%D1%83%D0%B7%D0%B5%D0%B9" title="Австралийский музей">Австралийском музее</a>, <a href="/wiki/%D0%A1%D0%B8%D0%B4%D0%BD%D0%B5%D0%B9" title="Сидней">Сидней</a>, 2009 год</figcaption></figure>
<p>Впервые остатки гипсилофодона были найдены в <a href="/wiki/1849_%D0%B3%D0%BE%D0%B4" title="1849 год">1849 году</a> на острове Уайт в Южной Англии<sup class="reference" id="cite_ref-isbn978-5-9603-0089-6_19-0"><a href="#cite_note-isbn978-5-9603-0089-6-19">[19]</a></sup>. После того, как кости гипсилофодона изучили в первый раз, учёные решили, что они принадлежат детёнышам <a href="/wiki/%D0%98%D0%B3%D1%83%D0%B0%D0%BD%D0%BE%D0%B4%D0%BE%D0%BD" title="Игуанодон">игуанодона</a>.
</p><p>В <a href="/wiki/1870_%D0%B3%D0%BE%D0%B4" title="1870 год">1870 году</a> палеонтолог Хаксли пересмотрел прежнее описание и понял, что перед ним не что иное, как новый вид динозавров<sup class="reference" id="cite_ref-isbn978-5-9603-0089-6_19-1"><a href="#cite_note-isbn978-5-9603-0089-6-19">[19]</a></sup>. Он сделал новое описание, основываясь на нескольких скелетах, найденных в <a href="/wiki/1868_%D0%B3%D0%BE%D0%B4" title="1868 год">1868 году</a> палеонтологом-любителем преподобным <span data-interwiki-article="William Fox (palaeontologist)" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%A4%D0%BE%D0%BA%D1%81,_%D0%A3%D0%B8%D0%BB%D1%8C%D1%8F%D0%BC_(%D0%BF%D0%B0%D0%BB%D0%B5%D0%BE%D0%BD%D1%82%D0%BE%D0%BB%D0%BE%D0%B3)&amp;action=edit&amp;redlink=1" title="Фокс, Уильям (палеонтолог) (страница отсутствует)">Уильямом Фоксом</a></span> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;"><a class="extiw" href="https://en.wikipedia.org/wiki/William_Fox_(palaeontologist)" title="en:William Fox (palaeontologist)"><span title="William Fox (palaeontologist) — версия статьи «Фокс, Уильям (палеонтолог)» на английском языке">William Fox, 1813—1881</span></a></span>), в честь которого и был назван вид <span data-wikidata-property-id="P225" lang="la" style="font-style: italic;">Hypsilophodon foxii</span>, и выделил гипсилофодона как новый род.
</p><p>В <a href="/wiki/1882_%D0%B3%D0%BE%D0%B4" title="1882 год">1882 году</a> учёный-палеонтолог из Великобритании <a class="new" href="/w/index.php?title=%D0%A5%D0%B0%D0%BB%D1%8C%D0%BA,_%D0%94%D0%B6%D0%B5%D0%B9%D0%BC%D1%81&amp;action=edit&amp;redlink=1" title="Хальк, Джеймс (страница отсутствует)">Джеймс Хальк</a><sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup> выдвинул идею о том, что, судя по строению длинных пальцев на передних лапах, гипсилофодон мог карабкаться по деревьям, как современный <a href="/wiki/%D0%94%D1%80%D0%B5%D0%B2%D0%B5%D1%81%D0%BD%D1%8B%D0%B5_%D0%BA%D0%B5%D0%BD%D0%B3%D1%83%D1%80%D1%83" title="Древесные кенгуру">древесный кенгуру</a> в <a href="/wiki/%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D1%8F" title="Австралия">Австралии</a>. Эту идею поддержали и другие палеонтологи того времени, так как, согласно реконструкции Халька, его загнутые когти затрудняли бы передвижение по земле, однако были очень удобны для лазанья по деревьям.
</p><p>Но в <a href="/wiki/1974_%D0%B3%D0%BE%D0%B4_%D0%B2_%D0%BD%D0%B0%D1%83%D0%BA%D0%B5" title="1974 год в науке">1974 году</a> британский палеонтолог <span data-interwiki-article="Peter Galton" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%9F%D0%B8%D1%82%D0%B5%D1%80_%D0%9C._%D0%93%D0%B0%D0%BB%D1%82%D0%BE%D0%BD&amp;action=edit&amp;redlink=1" title="Питер М. Галтон (страница отсутствует)">П. М. Галтон</a></span> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;"><a class="extiw" href="https://en.wikipedia.org/wiki/Peter_Galton" title="en:Peter Galton"><span title="Peter Galton — версия статьи «Питер М. Галтон» на английском языке">Peter M. Galton</span></a></span>) собрал скелет гипсилофодона в несколько другом порядке — он правильно повернул пальцы ног животного: теперь они были направлены в одну сторону, и, следовательно, были непригодны для жизни на деревьях. Эта реконструкция остаётся действительной в настоящее время<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup>.
</p>
<h2><span id=".D0.9F.D0.B0.D0.BB.D0.B5.D0.BE.D1.8D.D0.BA.D0.BE.D0.BB.D0.BE.D0.B3.D0.B8.D1.8F"></span><span class="mw-headline" id="Палеоэкология"><a href="/wiki/%D0%9F%D0%B0%D0%BB%D0%B5%D0%BE%D1%8D%D0%BA%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F" title="Палеоэкология">Палеоэкология</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%93%D0%B8%D0%BF%D1%81%D0%B8%D0%BB%D0%BE%D1%84%D0%BE%D0%B4%D0%BE%D0%BD&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Палеоэкология»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%93%D0%B8%D0%BF%D1%81%D0%B8%D0%BB%D0%BE%D1%84%D0%BE%D0%B4%D0%BE%D0%BD&amp;action=edit&amp;section=5" title="Редактировать код раздела «Палеоэкология»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Гипсилофодоны, как и все травоядные животные, являлись частью <a href="/wiki/%D0%9F%D0%B8%D1%89%D0%B5%D0%B2%D0%B0%D1%8F_%D1%86%D0%B5%D0%BF%D1%8C" title="Пищевая цепь">пищевой цепи</a>. Так же, как и сегодня, мы наблюдаем тысячи пищевых цепей по всему миру (например, <a class="mw-redirect" href="/wiki/%D0%A2%D1%80%D0%B0%D0%B2%D0%B0" title="Трава">трава</a> → <a class="mw-redirect" href="/wiki/%D0%97%D0%B0%D1%8F%D1%86" title="Заяц">заяц</a> → <a href="/wiki/%D0%9B%D0%B8%D1%81%D0%B8%D1%86%D0%B0" title="Лисица">лисица</a>), в раннем меловом периоде их было не меньше. Поэтому, зная особенности флоры и фауны, климата и рельефа того времени, мы можем сконструировать подобную пищевую цепь: низкорослые <a href="/wiki/%D0%9F%D0%B0%D0%BF%D0%BE%D1%80%D0%BE%D1%82%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2%D0%B8%D0%B4%D0%BD%D1%8B%D0%B5" title="Папоротниковидные">папоротниковидные</a> растения → гипсилофодон → <a href="/wiki/%D0%9D%D0%B5%D0%BE%D0%B2%D0%B5%D0%BD%D0%B0%D1%82%D0%BE%D1%80" title="Неовенатор">неовенатор</a> (<a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Латинский язык">лат.</a> <span lang="la" style="font-style:italic;">Neovenator</span>)<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup><sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup><sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup>. Возможно, цепочка была намного длиннее, так как в Англии и Испании обитало ещё огромное количество видов деревьев, средних по величине и огромных хищников.
</p>
<h2><span id=".D0.9A.D0.BB.D0.B0.D1.81.D1.81.D0.B8.D1.84.D0.B8.D0.BA.D0.B0.D1.86.D0.B8.D1.8F_.D0.B8_.D0.B2.D1.8B.D0.BC.D0.B8.D1.80.D0.B0.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Классификация_и_вымирание">Классификация и вымирание</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%93%D0%B8%D0%BF%D1%81%D0%B8%D0%BB%D0%BE%D1%84%D0%BE%D0%B4%D0%BE%D0%BD&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Классификация и вымирание»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%93%D0%B8%D0%BF%D1%81%D0%B8%D0%BB%D0%BE%D1%84%D0%BE%D0%B4%D0%BE%D0%BD&amp;action=edit&amp;section=6" title="Редактировать код раздела «Классификация и вымирание»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Гипсилофодоны были <a href="/wiki/%D0%9F%D1%82%D0%B8%D1%86%D0%B5%D1%82%D0%B0%D0%B7%D0%BE%D0%B2%D1%8B%D0%B5" title="Птицетазовые">птицетазовыми</a> динозаврами и принадлежали к <a class="mw-redirect" href="/wiki/%D0%98%D0%BD%D1%84%D1%80%D0%B0%D0%BE%D1%82%D1%80%D1%8F%D0%B4" title="Инфраотряд">инфраотряду</a> <a href="/wiki/%D0%9E%D1%80%D0%BD%D0%B8%D1%82%D0%BE%D0%BF%D0%BE%D0%B4%D1%8B" title="Орнитоподы">орнитопод</a>. Они относились к семейству <a href="/wiki/%D0%93%D0%B8%D0%BF%D1%81%D0%B8%D0%BB%D0%BE%D1%84%D0%BE%D0%B4%D0%BE%D0%BD%D1%82%D0%B8%D0%B4%D1%8B" title="Гипсилофодонтиды">гипсилофодонтид</a><sup class="reference" id="cite_ref-ButlerEtAl_25-0"><a href="#cite_note-ButlerEtAl-25">[25]</a></sup> до момента признания семейства <a class="mw-redirect" href="/wiki/%D0%9F%D0%B0%D1%80%D0%B0%D1%84%D0%B8%D0%BB%D0%B8%D1%8F_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%81%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0%D1%82%D0%B8%D0%BA%D0%B0)" title="Парафилия (биологическая систематика)">парафилетической группой</a><sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup>. В 2008 году род отнесли к <a href="/wiki/%D0%91%D0%B0%D0%B7%D0%B0%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D0%B3%D1%80%D1%83%D0%BF%D0%BF%D0%B0" title="Базальная группа">базальным</a> орнитоподам<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup>.
</p><p>Род гипсилофодон включал в себя лишь один вид <span data-wikidata-property-id="P225" lang="la" style="font-style: italic;">Hypsilophodon foxii</span>. Вымер этот род динозавров примерно 100,5 млн лет назад, просуществовав чуть меньше 36 млн лет. Почему он вымер, до сих пор не выяснено, возможно, гипсилофодоны не смогли приспособиться к изменениям климата<sup class="reference" id="cite_ref-BG08_28-0"><a href="#cite_note-BG08-28">[28]</a></sup>.
</p>

<p>Упоминается в романах <a href="/wiki/%D0%9A%D1%80%D0%B0%D0%B9%D1%82%D0%BE%D0%BD,_%D0%9C%D0%B0%D0%B9%D0%BA%D0%BB" title="Крайтон, Майкл">Майкла Крайтона</a> «<a class="mw-redirect" href="/wiki/%D0%9F%D0%B0%D1%80%D0%BA_%D0%AE%D1%80%D1%81%D0%BA%D0%BE%D0%B3%D0%BE_%D0%BF%D0%B5%D1%80%D0%B8%D0%BE%D0%B4%D0%B0_(%D1%80%D0%BE%D0%BC%D0%B0%D0%BD)" title="Парк Юрского периода (роман)">Парк Юрского периода</a>» и «<a href="/wiki/%D0%97%D0%B0%D1%82%D0%B5%D1%80%D1%8F%D0%BD%D0%BD%D1%8B%D0%B9_%D0%BC%D0%B8%D1%80_(%D1%80%D0%BE%D0%BC%D0%B0%D0%BD,_1995)" title="Затерянный мир (роман, 1995)">Затерянный мир</a>»:
</p><p>
</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
Неподалёку паслось стадо гипсилофодонов. Зелёные газелеподобные животные опасливо поднимали головы каждый раз, когда слышали металлический лязг.
</p>
</blockquote>
