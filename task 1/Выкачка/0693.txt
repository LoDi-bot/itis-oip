<p><b>Костёл Свято́го Ду́ха</b> и доминиканский монастырь в <a href="/wiki/%D0%92%D0%B8%D0%BB%D1%8C%D0%BD%D1%8E%D1%81" title="Вильнюс">Вильнюсе</a> (<i>Доминиканский костёл</i>; <a href="/wiki/%D0%9B%D0%B8%D1%82%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Литовский язык">лит.</a> <span lang="lt" style="font-style:italic;">Šventosios Dvasios bažnyčia</span>, <a href="/wiki/%D0%9F%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Польский язык">польск.</a> <span lang="pl" style="font-style:italic;">kościół Świętego Ducha</span>) — <a class="mw-redirect" href="/wiki/%D0%A0%D0%B8%D0%BC%D1%81%D0%BA%D0%BE-%D0%BA%D0%B0%D1%82%D0%BE%D0%BB%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%86%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C" title="Римско-католическая церковь">римско-католический</a> <a href="/wiki/%D0%9F%D1%80%D0%B8%D1%85%D0%BE%D0%B4" title="Приход">приходской</a> <a href="/wiki/%D0%9A%D0%BE%D1%81%D1%82%D1%91%D0%BB" title="Костёл">костёл</a> с бывшим <a class="mw-redirect" href="/wiki/%D0%9E%D1%80%D0%B4%D0%B5%D0%BD_%D0%B4%D0%BE%D0%BC%D0%B8%D0%BD%D0%B8%D0%BA%D0%B0%D0%BD%D1%86%D0%B5%D0%B2" title="Орден доминиканцев">доминиканским</a> монастырём, <a href="/wiki/%D0%9F%D0%B0%D0%BC%D1%8F%D1%82%D0%BD%D0%B8%D0%BA_%D0%B0%D1%80%D1%85%D0%B8%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D1%8B" title="Памятник архитектуры">памятник архитектуры</a> и искусства позднего <a href="/wiki/%D0%91%D0%B0%D1%80%D0%BE%D0%BA%D0%BA%D0%BE" title="Барокко">барокко</a>. Храм <a href="/wiki/%D0%9D%D0%B5%D1%84" title="Неф">трёхнефный</a>, в форме <a href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82#Латинский_крест" title="Крест">латинского креста</a> в плане, длиной 57 м, шириной 26 м, может вместить 1450 человек. Ансамбль зданий доминиканского монастыря и храма включён в Регистр культурных ценностей Литовской Республики и охраняется государством как объект национального значения; код всего ансамбля 21<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>, костёла — 27302<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>.
</p><p>Располагается в <a href="/wiki/%D0%A1%D1%82%D0%B0%D1%80%D1%8B%D0%B9_%D0%B3%D0%BE%D1%80%D0%BE%D0%B4_(%D0%92%D0%B8%D0%BB%D1%8C%D0%BD%D1%8E%D1%81)" title="Старый город (Вильнюс)">Старом городе</a>, протянувшись боковым фасадом вдоль <a class="mw-redirect" href="/wiki/%D0%A3%D0%BB%D0%B8%D1%86%D0%B0_%D0%94%D0%BE%D0%BC%D0%B8%D0%BD%D0%B8%D0%BA%D0%BE%D0%BD%D1%83" title="Улица Доминикону">улицы Доминикону</a> (в советское время улица Ю. Гарялё) и задним фасадом выходя на <a href="/wiki/%D0%A3%D0%BB%D0%B8%D1%86%D0%B0_%D0%A8%D0%B2%D1%8F%D0%BD%D1%82%D0%BE_%D0%98%D0%B3%D0%BD%D0%BE%D1%82%D0%BE" title="Улица Швянто Игното">улицу Игното</a> (<span lang="lt" style="font-style:italic;">Šv. Ignoto g.</span>, в советское время улица К. Гедрё). Нижняя часть храма опоясана с трёх сторон корпусами бывшего монастыря, благодаря чему выделяется его верхняя часть с двумя невысокими башнями (36,3 м), четырьмя <a href="/wiki/%D0%A4%D1%80%D0%BE%D0%BD%D1%82%D0%BE%D0%BD" title="Фронтон">фронтонами</a> и <a href="/wiki/%D0%9A%D1%83%D0%BF%D0%BE%D0%BB" title="Купол">куполом</a> с фонарём над центральным <a href="/wiki/%D0%9D%D0%B5%D1%84" title="Неф">нефом</a>.
</p><p>Относится к Вильнюсскому деканату <a href="/wiki/%D0%90%D1%80%D1%85%D0%B8%D0%B5%D0%BF%D0%B0%D1%80%D1%85%D0%B8%D1%8F_%D0%92%D0%B8%D0%BB%D1%8C%D0%BD%D1%8E%D1%81%D0%B0" title="Архиепархия Вильнюса">Вильнюсской архиепархии</a>. Официальный адрес: ул. Доминикону, 8 (<span lang="lt" style="font-style:italic;">Dominikonų g. 8</span>). Ежедневные службы на <a href="/wiki/%D0%9F%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Польский язык">польском языке</a>.
</p>

<h2><span id=".D0.98.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.8F"></span><span class="mw-headline" id="История">История</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%BE%D1%81%D1%82%D1%91%D0%BB_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%94%D1%83%D1%85%D0%B0_(%D0%92%D0%B8%D0%BB%D1%8C%D0%BD%D1%8E%D1%81)&amp;veaction=edit&amp;section=1" title="Редактировать раздел «История»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%BE%D1%81%D1%82%D1%91%D0%BB_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%94%D1%83%D1%85%D0%B0_(%D0%92%D0%B8%D0%BB%D1%8C%D0%BD%D1%8E%D1%81)&amp;action=edit&amp;section=1" title="Редактировать код раздела «История»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Dominikonai3.jpg"><img class="mw-file-element" data-file-height="3072" data-file-width="2304" decoding="async" height="213" src="//upload.wikimedia.org/wikipedia/commons/thumb/7/73/Dominikonai3.jpg/160px-Dominikonai3.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/7/73/Dominikonai3.jpg/240px-Dominikonai3.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/7/73/Dominikonai3.jpg/320px-Dominikonai3.jpg 2x" width="160"/></a><figcaption><i>Фасад (улица Доминикону)</i></figcaption></figure>
<p>Нынешнее здание — уже, по меньшей мере, пятое, возведённое на этом же месте. Первые здания были выстроены в <a href="/wiki/%D0%93%D0%BE%D1%82%D0%B8%D0%BA%D0%B0" title="Готика">готическом</a> стиле<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>. По некоторым предположениям, небольшая деревянная церковь в этом месте стояла уже во времена князя <a href="/wiki/%D0%93%D0%B5%D0%B4%D0%B8%D0%BC%D0%B8%D0%BD" title="Гедимин">Гедимина</a>. В <a href="/wiki/1408_%D0%B3%D0%BE%D0%B4" title="1408 год">1408 году</a> князь <a href="/wiki/%D0%92%D0%B8%D1%82%D0%BE%D0%B2%D1%82" title="Витовт">Витовт</a> поставил здесь костёл во имя Святого Духа. При <a class="mw-redirect" href="/wiki/%D0%9A%D0%B0%D0%B7%D0%B8%D0%BC%D0%B8%D1%80_%D0%AF%D0%B3%D0%B5%D0%BB%D0%BB%D0%BE%D0%BD" title="Казимир Ягеллон">Казимире Ягеллоне</a> в <a href="/wiki/1441_%D0%B3%D0%BE%D0%B4" title="1441 год">1441 году</a> храм был перестроен и расширен уже как каменный.
</p><p>До <a href="/wiki/XVI_%D0%B2%D0%B5%D0%BA" title="XVI век">XVI века</a> храм был приходским. В <a href="/wiki/1501_%D0%B3%D0%BE%D0%B4" title="1501 год">1501 году</a> в Вильну был приглашён <a class="mw-redirect" href="/wiki/%D0%94%D0%BE%D0%BC%D0%B8%D0%BD%D0%B8%D0%BA%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BE%D1%80%D0%B4%D0%B5%D0%BD" title="Доминиканский орден">Доминиканский орден</a>, и король <a class="mw-redirect" href="/wiki/%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%AF%D0%B3%D0%B5%D0%BB%D0%BB%D0%BE%D0%BD" title="Александр Ягеллон">Александр</a>, с согласия своего духовного отца — настоятеля костёла <a class="new" href="/w/index.php?title=%D0%9A%D0%BE%D1%80%D1%87%D0%B0%D0%BA,_%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B9&amp;action=edit&amp;redlink=1" title="Корчак, Николай (страница отсутствует)">Николая Корчака</a>, передал его монахам. Епископ <a class="new" href="/w/index.php?title=%D0%A2%D0%B0%D0%B1%D0%BE%D1%80,_%D0%90%D0%B4%D0%B0%D0%BB%D1%8C%D0%B1%D0%B5%D1%80%D1%82_%D0%92%D0%BE%D0%B9%D1%86%D0%B5%D1%85&amp;action=edit&amp;redlink=1" title="Табор, Адальберт Войцех (страница отсутствует)">Адальберт Войцех Табор</a> подарил ордену большой участок земли с постройками рядом с храмом. По инициативе короля Александра на этом участке был построен монастырь, а сам костёл перестроен. <a href="/wiki/%D0%9D%D0%B5%D1%84" title="Неф">Однонефное</a> здание в форме <a href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82#Латинский_крест" title="Крест">латинского креста</a> несколько раз горело и перестраивалось. Восстановленный после опустошительного пожара <a href="/wiki/1610_%D0%B3%D0%BE%D0%B4" title="1610 год">1610 года</a>, он был разграблен и сожжён во время взятия Вильны украинскими казаками и русским войсками в <a href="/wiki/1655_%D0%B3%D0%BE%D0%B4" title="1655 год">1655 году</a>. Небольшой храм, отстроенный после этого, усилиями настоятеля доминиканского монастыря <a class="new" href="/w/index.php?title=%D0%92%D0%BE%D0%B9%D0%BD%D0%B8%D0%BB%D0%BE%D0%B2%D0%B8%D1%87,_%D0%9C%D0%B8%D1%85%D0%B0%D0%B8%D0%BB&amp;action=edit&amp;redlink=1" title="Войнилович, Михаил (страница отсутствует)">Михаила Войниловича</a>, предпринимавшимися с <a href="/wiki/1679_%D0%B3%D0%BE%D0%B4" title="1679 год">1679 года</a>, сменило новое здание, возведённое на средства и из материалов доминиканского ордена. Новый костёл был освящён в <a href="/wiki/1688_%D0%B3%D0%BE%D0%B4" title="1688 год">1688 году</a> виленским епископом <a href="/wiki/%D0%91%D0%B6%D0%BE%D1%81%D1%82%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9,_%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD_%D0%9A%D0%B0%D0%B7%D0%B8%D0%BC%D0%B8%D1%80" title="Бжостовский, Константин Казимир">Константином Бжостовским</a>.
</p>
<figure class="mw-default-size" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Dominikonai8.jpg"><img class="mw-file-element" data-file-height="2304" data-file-width="3072" decoding="async" height="165" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/da/Dominikonai8.jpg/220px-Dominikonai8.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/d/da/Dominikonai8.jpg/330px-Dominikonai8.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/d/da/Dominikonai8.jpg/440px-Dominikonai8.jpg 2x" width="220"/></a><figcaption><i>Колокольни</i></figcaption></figure>
<p>Храм пострадал при нашествии шведов в <a href="/wiki/1702_%D0%B3%D0%BE%D0%B4" title="1702 год">1702 году</a>. Катастрофические разрушения приносили опустошительные пожары <a href="/wiki/XVIII_%D0%B2%D0%B5%D0%BA" title="XVIII век">XVIII века</a>. По свидетельству современника пожара <a href="/wiki/1748_%D0%B3%D0%BE%D0%B4" title="1748 год">1748 года</a>, выгорело всё, включая первый в Вильне <a href="/wiki/%D0%9E%D1%80%D0%B3%D0%B0%D0%BD_(%D0%BC%D1%83%D0%B7%D1%8B%D0%BA%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%B8%D0%BD%D1%81%D1%82%D1%80%D1%83%D0%BC%D0%B5%D0%BD%D1%82)" title="Орган (музыкальный инструмент)">орган</a> (стоивший монастырю 40 тысяч золотых) и гробы в подземельях костёла<sup class="reference" id="cite_ref-Jono,_Dominikonų_1998._P._143_4-0"><a href="#cite_note-Jono,_Dominikonų_1998._P._143-4">[4]</a></sup>. Остались лишь стены с обгоревшей и осыпавшейся <a href="/wiki/%D0%A8%D1%82%D1%83%D0%BA%D0%B0%D1%82%D1%83%D1%80%D0%BA%D0%B0" title="Штукатурка">штукатуркой</a>.
</p><p>После пожаров <a class="mw-redirect" href="/wiki/1748" title="1748">1748</a> и <a href="/wiki/1749_%D0%B3%D0%BE%D0%B4" title="1749 год">1749 годов</a> в сравнительно короткий срок стараниями доминиканцев, магната <a class="new" href="/w/index.php?title=%D0%9F%D0%B0%D1%86,_%D0%9B%D1%8E%D0%B4%D0%B2%D0%B8%D0%BA&amp;action=edit&amp;redlink=1" title="Пац, Людвик (страница отсутствует)">Людвика Паца</a>, Бригитты Сологуб из <a href="/wiki/%D0%A0%D0%B0%D0%B4%D0%B7%D0%B8%D0%B2%D0%B8%D0%BB%D0%BB%D1%8B" title="Радзивиллы">Радзивиллов</a> и других меценатов к <a href="/wiki/1770_%D0%B3%D0%BE%D0%B4" title="1770 год">1770 году</a> храм и монастырь были заново отстроены. После этого костёл приобрёл пышный декор в стиле <a href="/wiki/%D0%A0%D0%BE%D0%BA%D0%BE%D0%BA%D0%BE" title="Рококо">рококо</a>.
</p><p>В <a href="/wiki/1812_%D0%B3%D0%BE%D0%B4" title="1812 год">1812 году</a> храм пострадал от <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%8F" title="Франция">французских</a> солдат и ремонтировался в <a class="mw-redirect" href="/wiki/1813" title="1813">1813</a>—<a href="/wiki/1815_%D0%B3%D0%BE%D0%B4" title="1815 год">1815 годах</a>. В <a href="/wiki/1818_%D0%B3%D0%BE%D0%B4" title="1818 год">1818 году</a> бурей сорвало крышу и пришлось заново перекрывать кровлю. В <a href="/wiki/1844_%D0%B3%D0%BE%D0%B4" title="1844 год">1844 году</a> российские власти упразднили монастырь. В его помещениях по распоряжению генерал-губернатора <a href="/wiki/%D0%9C%D1%83%D1%80%D0%B0%D0%B2%D1%8C%D1%91%D0%B2-%D0%92%D0%B8%D0%BB%D0%B5%D0%BD%D1%81%D0%BA%D0%B8%D0%B9,_%D0%9C%D0%B8%D1%85%D0%B0%D0%B8%D0%BB_%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B5%D0%B2%D0%B8%D1%87" title="Муравьёв-Виленский, Михаил Николаевич">М. Н. Муравьёва</a> содержались в заключении участники <a class="mw-redirect" href="/wiki/%D0%92%D0%BE%D1%81%D1%81%D1%82%D0%B0%D0%BD%D0%B8%D0%B5_1863_%D0%B3%D0%BE%D0%B4%D0%B0" title="Восстание 1863 года">восстания 1863 года</a>.
</p>

<p>С упразднением монастыря костёл с <a href="/wiki/1844_%D0%B3%D0%BE%D0%B4" title="1844 год">1844 года</a> снова стал приходским и действовал на всём протяжении <a href="/wiki/XIX_%D0%B2%D0%B5%D0%BA" title="XIX век">XIX</a> и <a href="/wiki/XX_%D0%B2%D0%B5%D0%BA" title="XX век">XX веков</a>. О нём идёт речь в части седьмой цикла «Литовский дивертисмент» (<a class="mw-redirect" href="/wiki/1971" title="1971">1971</a>) <a href="/wiki/%D0%91%D1%80%D0%BE%D0%B4%D1%81%D0%BA%D0%B8%D0%B9,_%D0%98%D0%BE%D1%81%D0%B8%D1%84_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Бродский, Иосиф Александрович">Иосифа Бродского</a>, который ознакомился с костёлом и монастырём весной <a href="/wiki/1966_%D0%B3%D0%BE%D0%B4" title="1966 год">1966 года</a><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>.

</p>

Ближе двубашенный белый <a href="/wiki/%D0%9A%D0%BE%D1%81%D1%82%D1%91%D0%BB_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B9_%D0%95%D0%BA%D0%B0%D1%82%D0%B5%D1%80%D0%B8%D0%BD%D1%8B_(%D0%92%D0%B8%D0%BB%D1%8C%D0%BD%D1%8E%D1%81)" title="Костёл Святой Екатерины (Вильнюс)">костёл Св. Екатерины</a>; чуть далее круглый купол Доминиканцев, изнутри странной на вид и как бы неправильной формы — «ушная раковина Бога» из последнего стихотворения цикла.</i> (<a href="/wiki/%D0%92%D0%B5%D0%BD%D1%86%D0%BB%D0%BE%D0%B2%D0%B0,_%D0%A2%D0%BE%D0%BC%D0%B0%D1%81" title="Венцлова, Томас">Томас Венцлова</a>).<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> </p></blockquote>

<h2><span id=".D0.A4.D0.B0.D1.81.D0.B0.D0.B4"></span><span class="mw-headline" id="Фасад">Фасад</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%BE%D1%81%D1%82%D1%91%D0%BB_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%94%D1%83%D1%85%D0%B0_(%D0%92%D0%B8%D0%BB%D1%8C%D0%BD%D1%8E%D1%81)&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Фасад»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%BE%D1%81%D1%82%D1%91%D0%BB_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%94%D1%83%D1%85%D0%B0_(%D0%92%D0%B8%D0%BB%D1%8C%D0%BD%D1%8E%D1%81)&amp;action=edit&amp;section=2" title="Редактировать код раздела «Фасад»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Dominikonai1.jpg"><img class="mw-file-element" data-file-height="3072" data-file-width="2304" decoding="async" height="213" src="//upload.wikimedia.org/wikipedia/commons/thumb/1/18/Dominikonai1.jpg/160px-Dominikonai1.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/1/18/Dominikonai1.jpg/240px-Dominikonai1.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/1/18/Dominikonai1.jpg/320px-Dominikonai1.jpg 2x" width="160"/></a><figcaption><i>Портал</i></figcaption></figure>
<p>Купол с фонарём над центральным нефом костёла возвышается на 51 м.<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>. Он был реконструирован в <a href="/wiki/XVIII_%D0%B2%D0%B5%D0%BA" title="XVIII век">XVIII веке</a> архитекторами Иоганном Валентином Дидерштейном и Яном Незамовским. Две барочные башни, схожие с теми, которые сохранились у миссионерского <a href="/wiki/%D0%9A%D0%BE%D1%81%D1%82%D1%91%D0%BB_%D0%92%D0%BE%D0%B7%D0%BD%D0%B5%D1%81%D0%B5%D0%BD%D0%B8%D1%8F_%D0%93%D0%BE%D1%81%D0%BF%D0%BE%D0%B4%D0%BD%D1%8F_(%D0%92%D0%B8%D0%BB%D1%8C%D0%BD%D1%8E%D1%81)" title="Костёл Вознесения Господня (Вильнюс)">костёла Вознесения Господня</a>, были разрушены в начале <a href="/wiki/XIX_%D0%B2%D0%B5%D0%BA" title="XIX век">XIX века</a>.
</p><p>Барочный <a href="/wiki/%D0%9C%D0%B5%D0%B4%D1%8C" title="Медь">медный</a> колокол высотой 84 см и диаметром 112 см был отлит в <a href="/wiki/1779_%D0%B3%D0%BE%D0%B4" title="1779 год">1779 году</a><sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup>.
Костёл выделяется среди других храмов Вильнюса необычной ориентацией. Он располагается вдоль улицы Доминикону и не имеет главного фасада. Вход с этой стороны украшен <a href="/wiki/%D0%A4%D1%80%D0%BE%D0%BD%D1%82%D0%BE%D0%BD" title="Фронтон">фронтоном</a> с двумя парами <a href="/wiki/%D0%94%D0%BE%D1%80%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BE%D1%80%D0%B4%D0%B5%D1%80" title="Дорический ордер">дорических</a> <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%BB%D0%BE%D0%BD%D0%BD%D0%B0_(%D0%B0%D1%80%D1%85%D0%B8%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D0%B0)" title="Колонна (архитектура)">колонн</a>, повёрнутых по диагонали к плоскости фасада, и пышным <a href="/wiki/%D0%9A%D0%B0%D1%80%D1%82%D1%83%D1%88" title="Картуш">картушем</a> в стиле рококо с изображением Орла (герб Польши), Погони (герб Литвы) и гербом династии Вазов над круглой аркой. Нижняя часть обрамляющих окно <a href="/wiki/%D0%9F%D0%B8%D0%BB%D1%8F%D1%81%D1%82%D1%80%D0%B0" title="Пилястра">пилястров</a> на втором ярусе <a href="/wiki/%D0%9F%D0%BE%D1%80%D1%82%D0%B0%D0%BB_(%D0%B0%D1%80%D1%85%D0%B8%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D0%B0)" title="Портал (архитектура)">портала</a> изгибается и завершается <a href="/wiki/%D0%92%D0%BE%D0%BB%D1%8E%D1%82%D0%B0" title="Волюта">волютами</a>. Главный восточный портал (с улицы Доминикону) создал архитектор и скульптор <a class="new" href="/w/index.php?title=%D0%93%D0%BE%D1%84%D0%B5%D1%80,_%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%81%D0%BA_%D0%98%D0%B3%D0%BD%D0%B0%D1%82%D0%B8%D0%B9&amp;action=edit&amp;redlink=1" title="Гофер, Франциск Игнатий (страница отсутствует)">Франциск Игнатий Гофер</a>. По правой стороне длинного коридора, ведущего в помещения бывшего монастыря, расположен вход в костёл.
</p>
<h2><span id=".D0.98.D0.BD.D1.82.D0.B5.D1.80.D1.8C.D0.B5.D1.80"></span><span class="mw-headline" id="Интерьер">Интерьер</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%BE%D1%81%D1%82%D1%91%D0%BB_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%94%D1%83%D1%85%D0%B0_(%D0%92%D0%B8%D0%BB%D1%8C%D0%BD%D1%8E%D1%81)&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Интерьер»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%BE%D1%81%D1%82%D1%91%D0%BB_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%94%D1%83%D1%85%D0%B0_(%D0%92%D0%B8%D0%BB%D1%8C%D0%BD%D1%8E%D1%81)&amp;action=edit&amp;section=3" title="Редактировать код раздела «Интерьер»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Авторство пышного внутреннего убранства одни искусствоведы приписывают Франциску Игнатию Гоферу, другие — <a href="/wiki/%D0%93%D0%BB%D0%B0%D1%83%D0%B1%D0%B8%D1%86,_%D0%98%D0%BE%D0%B3%D0%B0%D0%BD%D0%BD_%D0%9A%D1%80%D0%B8%D1%81%D1%82%D0%BE%D1%84" title="Глаубиц, Иоганн Кристоф">Иоганну Кристофу Глаубицу</a><sup class="reference" id="cite_ref-Jono,_Dominikonų_1998._P._143_4-1"><a href="#cite_note-Jono,_Dominikonų_1998._P._143-4">[4]</a></sup><sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>. Во второй половине <a href="/wiki/XVIII_%D0%B2%D0%B5%D0%BA" title="XVIII век">XVIII века</a> в храме были сооружены 16 <a href="/wiki/%D0%90%D0%BB%D1%82%D0%B0%D1%80%D1%8C" title="Алтарь">алтарей</a> в стиле <a href="/wiki/%D0%A0%D0%BE%D0%BA%D0%BE%D0%BA%D0%BE" title="Рококо">рококо</a> из материалов, имитирующих <a href="/wiki/%D0%9C%D1%80%D0%B0%D0%BC%D0%BE%D1%80" title="Мрамор">мрамор</a> разных цветов: центральный алтарь Пресвятой Троицы, рядом с ним в южной части большого алтаря — алтари <a href="/wiki/%D0%98%D0%B8%D1%81%D1%83%D1%81_%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%BE%D1%81" title="Иисус Христос">Иисуса Христа</a> и <a class="mw-redirect" href="/wiki/%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B9_%D0%94%D0%BE%D0%BC%D0%B8%D0%BD%D0%B8%D0%BA" title="Святой Доминик">Святого Доминика</a> (в <a href="/wiki/%D0%A2%D1%80%D0%B0%D0%BD%D1%81%D0%B5%D0%BF%D1%82" title="Трансепт">трансепте</a>). В северной части — алтари Ченстоховской Божией Матери и <a href="/wiki/%D0%A4%D0%BE%D0%BC%D0%B0_%D0%90%D0%BA%D0%B2%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9" title="Фома Аквинский">Святого Фомы Аквинского</a> (в трансепте). Прежде в центральном нефе у пилонов располагалось пять алтарей, в южном и северном нефах — по три. Ныне в южной части центрального нефа у первого от большого алтаря <a href="/wiki/%D0%9F%D0%B8%D0%BB%D0%BE%D0%BD" title="Пилон">пилона</a> — наиболее пышно декорированный алтарь Господа Милосердного. Напротив него располагается исповедальня и <a href="/wiki/%D0%90%D0%BC%D0%B2%D0%BE%D0%BD" title="Амвон">амвон</a>, окружённые 12 гипсовыми скульптурами; по сторонам её — статуи короля <a class="mw-redirect" href="/wiki/%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D1%81%D0%BB%D0%B0%D0%B2_%D0%92%D0%B0%D0%B7%D0%B0" title="Владислав Ваза">Владислава Вазы</a> и <a class="mw-disambig" href="/wiki/%D0%AF%D0%B4%D0%B2%D0%B8%D0%B3%D0%B0" title="Ядвига">святой Ядвиги</a>.
</p><p>У пилонов и в боковых нефах справа — алтари <a href="/wiki/%D0%A4%D0%B0%D0%B4%D0%B4%D0%B5%D0%B9_(%D0%B0%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB_%D0%BE%D1%82_70)" title="Фаддей (апостол от 70)">Святого Фаддея</a>, <a href="/wiki/%D0%90%D0%BD%D0%B3%D0%B5%D0%BB-%D1%85%D1%80%D0%B0%D0%BD%D0%B8%D1%82%D0%B5%D0%BB%D1%8C" title="Ангел-хранитель">Ангела Хранителя</a>, <a href="/wiki/%D0%91%D0%B8%D1%87%D0%B5%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5_%D0%98%D0%B8%D1%81%D1%83%D1%81%D0%B0_%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%B0" title="Бичевание Иисуса Христа">Бичевания Христа</a>, <a href="/wiki/%D0%A1%D0%B2%D1%8F%D1%82%D0%B0%D1%8F_%D0%90%D0%BD%D0%BD%D0%B0" title="Святая Анна">Святой Анны</a>, <a class="mw-redirect mw-disambig" href="/wiki/%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B9_%D0%90%D0%BD%D1%82%D0%BE%D0%BD%D0%B8%D0%B9" title="Святой Антоний">Святого Антония</a>, <a href="/wiki/%D0%9F%D0%B8%D0%B9_V" title="Пий V">Святого Пия</a>, слева — <a href="/wiki/%D0%9E%D0%B4%D1%80%D0%BE%D0%B2%D0%BE%D0%BD%D0%B6,_%D0%AF%D1%86%D0%B5%D0%BA" title="Одровонж, Яцек">Святого Яцека</a>, <a class="mw-redirect" href="/wiki/%D0%9C%D0%B0%D1%80%D0%B8%D1%8F_%D0%9C%D0%B0%D0%B3%D0%B4%D0%B0%D0%BB%D0%B5%D0%BD%D0%B0" title="Мария Магдалена">Марии Магдалены</a>, <a class="mw-redirect" href="/wiki/%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B9_%D0%98%D0%BE%D1%81%D0%B8%D1%84" title="Святой Иосиф">Святого Иосифа</a>, <a class="mw-redirect" href="/wiki/%D0%A1%D0%B2%D1%8F%D1%82%D0%B0%D1%8F_%D0%92%D0%B0%D1%80%D0%B2%D0%B0%D1%80%D0%B0" title="Святая Варвара">Святой Варвары</a>, <a href="/wiki/%D0%95%D0%BA%D0%B0%D1%82%D0%B5%D1%80%D0%B8%D0%BD%D0%B0_%D0%A1%D0%B8%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F" title="Екатерина Сиенская">Святой Екатерины Сиенской</a>.
</p><p>Храм богато расписан барочными <a href="/wiki/%D0%A4%D1%80%D0%B5%D1%81%D0%BA%D0%B0" title="Фреска">фресками</a>. Роспись сводов выполнена разными художниками в <a class="mw-redirect" href="/wiki/1765" title="1765">1765</a>—<a href="/wiki/1770_%D0%B3%D0%BE%D0%B4" title="1770 год">1770 годах</a>. Настенная роспись в коридоре, ведущем от портала в храм, относится к середине <a href="/wiki/XVIII_%D0%B2%D0%B5%D0%BA" title="XVIII век">XVIII века</a>. В середине <a href="/wiki/XIX_%D0%B2%D0%B5%D0%BA" title="XIX век">XIX века</a> выполнена фреска в куполе. В конце <a href="/wiki/XIX_%D0%B2%D0%B5%D0%BA" title="XIX век">XIX века</a> фрески обновлялись. В <a class="mw-redirect" href="/wiki/1898" title="1898">1898</a>—<a href="/wiki/1899_%D0%B3%D0%BE%D0%B4" title="1899 год">1899 годах</a> художники из <a href="/wiki/%D0%A2%D0%B8%D1%80%D0%BE%D0%BB%D1%8C_(%D0%B8%D1%81%D1%82%D0%BE%D1%80%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C)" title="Тироль (историческая область)">Тироля</a> написали четыре композиции над входами в боковые нефы и фреску с изображением Святой Анны на своде южного нефа.
</p><p>В храме есть 45 ценных образов и <a href="/wiki/%D0%9F%D0%BE%D1%80%D1%82%D1%80%D0%B5%D1%82" title="Портрет">портретов</a> <a href="/wiki/XVI_%D0%B2%D0%B5%D0%BA" title="XVI век">XVI</a> — первой половины <a href="/wiki/XIX_%D0%B2%D0%B5%D0%BA" title="XIX век">XIX века</a>: <a href="/wiki/%D0%A4%D1%91%D0%BA%D0%BB%D0%B0_%D0%98%D0%BA%D0%BE%D0%BD%D0%B8%D0%B9%D1%81%D0%BA%D0%B0%D1%8F" title="Фёкла Иконийская">святой Фёклы</a>, <a href="/wiki/%D0%9E%D0%B4%D1%80%D0%BE%D0%B2%D0%BE%D0%BD%D0%B6,_%D0%AF%D1%86%D0%B5%D0%BA" title="Одровонж, Яцек">святого Гиацинта</a>, святого Франциска, святой Варвары, великого князя литовского Александра, Николая Корчака и другие.
</p><p>Орган <a href="/wiki/XVIII_%D0%B2%D0%B5%D0%BA" title="XVIII век">XVIII века</a>, созданный мастером <a href="/wiki/%D0%9A%D0%B0%D1%81%D0%BF%D0%B0%D1%80%D0%B8%D0%BD%D0%B8,_%D0%90%D0%B4%D0%B0%D0%BC_%D0%93%D0%BE%D1%82%D0%BB%D0%BE%D0%B1" title="Каспарини, Адам Готлоб">Адамом Готлибом Каспарини</a> из <a href="/wiki/%D0%9A%D1%91%D0%BD%D0%B8%D0%B3%D1%81%D0%B1%D0%B5%D1%80%D0%B3" title="Кёнигсберг">Кёнигсберга</a> в <a href="/wiki/1776_%D0%B3%D0%BE%D0%B4" title="1776 год">1776 году</a>, считается старейшим в Литве<sup class="reference" id="cite_ref-Jono,_Dominikonų_1998._P._148_13-0"><a href="#cite_note-Jono,_Dominikonų_1998._P._148-13">[13]</a></sup>.
</p><p>В стене коридора у входа в храм находится мемориальная плита, вмурованная <a href="/wiki/12_%D0%B8%D1%8E%D0%BB%D1%8F" title="12 июля">12 июля</a> <a href="/wiki/1786_%D0%B3%D0%BE%D0%B4" title="1786 год">1786 года</a> на месте захоронения Варвары Секлуцкой-Костровицкой её сыном.
</p>

<h2><span id=".D0.9F.D0.BE.D0.B4.D0.B7.D0.B5.D0.BC.D0.B5.D0.BB.D1.8C.D1.8F"></span><span class="mw-headline" id="Подземелья">Подземелья</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%BE%D1%81%D1%82%D1%91%D0%BB_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%94%D1%83%D1%85%D0%B0_(%D0%92%D0%B8%D0%BB%D1%8C%D0%BD%D1%8E%D1%81)&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Подземелья»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%BE%D1%81%D1%82%D1%91%D0%BB_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%94%D1%83%D1%85%D0%B0_(%D0%92%D0%B8%D0%BB%D1%8C%D0%BD%D1%8E%D1%81)&amp;action=edit&amp;section=4" title="Редактировать код раздела «Подземелья»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Под храмом располагаются подземелья — девять готических подвалов, образующих овеянный слухами и легендами лабиринт. Длина самого большого из них 33 м. Предполагается, что они двухуровневые(а возможно и больше). Издавна в подземельях осуществлялись захоронения, в <a href="/wiki/XVI_%D0%B2%D0%B5%D0%BA" title="XVI век">XVI</a>—<a href="/wiki/XVII_%D0%B2%D0%B5%D0%BA" title="XVII век">XVII веках</a> не только монахов, но и знатных жителей города. Среди захороненных здесь, по упоминаниям в различных документах и исследованиях, лидский староста <a class="mw-redirect" href="/wiki/%D0%A1%D0%B0%D0%BF%D0%B5%D0%B3%D0%B0,_%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%BE%D1%84%D0%BE%D1%80_%D0%A1%D1%82%D0%B5%D1%84%D0%B0%D0%BD" title="Сапега, Христофор Стефан">Христофор Стефан Сапега</a> (<a class="mw-redirect" href="/wiki/1627" title="1627">1627</a>), его жена Анна и её отец князь Александр (<a class="mw-redirect" href="/wiki/1643" title="1643">1643</a>), трокский воевода Александр Пац (<a class="mw-redirect" href="/wiki/1771" title="1771">1771</a>), Людвик Пац и его жена Виктория из Потоцких (<a class="mw-redirect" href="/wiki/1773" title="1773">1773</a>), княгиня Теодора Сапежина (<a class="mw-redirect" href="/wiki/1774" title="1774">1774</a>). В подземельях хоронились также многочисленные жертвы эпидемий <a href="/wiki/%D0%A5%D0%BE%D0%BB%D0%B5%D1%80%D0%B0" title="Холера">холеры</a> и <a href="/wiki/%D0%A7%D1%83%D0%BC%D0%B0" title="Чума">чумы</a>. Вероятно, среди похороненных здесь и те, кто умер во французском военном госпитале, размещённом в монастыре во время войны с Россией в <a href="/wiki/1812_%D0%B3%D0%BE%D0%B4" title="1812 год">1812 году</a>. Особенности микроклимата (постоянная температура и сухой воздух) способствовали тому, что часть трупов не разлагалась, а консервировались, подвергаясь своего рода мумификации.
</p><p>Подземелья в <a href="/wiki/XIX_%D0%B2%D0%B5%D0%BA" title="XIX век">XIX веке</a> исследовали и описывали <a href="/wiki/%D0%9A%D1%80%D0%B0%D1%88%D0%B5%D0%B2%D1%81%D0%BA%D0%B8%D0%B9,_%D0%AE%D0%B7%D0%B5%D1%84_%D0%98%D0%B3%D0%BD%D0%B0%D1%86%D1%8B" title="Крашевский, Юзеф Игнацы">Юзеф Игнацы Крашевский</a>, <a class="mw-redirect" href="/wiki/%D0%A2%D1%8B%D1%88%D0%BA%D0%B5%D0%B2%D0%B8%D1%87,_%D0%95%D0%B2%D1%81%D1%82%D0%B0%D1%85%D0%B8%D0%B9" title="Тышкевич, Евстахий">Евстахий Тышкевич</a>, <a class="mw-redirect" href="/wiki/%D0%9A%D0%B8%D1%80%D0%BA%D0%BE%D1%80,_%D0%90%D0%B4%D0%B0%D0%BC" title="Киркор, Адам">Адам Киркор</a>, в <a href="/wiki/1903_%D0%B3%D0%BE%D0%B4" title="1903 год">1903 году</a> — историк <a class="new" href="/w/index.php?title=%D0%97%D0%B0%D0%B3%D0%BE%D1%80%D1%81%D0%BA%D0%B8%D0%B9,_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D1%81%D0%BB%D0%B0%D0%B2&amp;action=edit&amp;redlink=1" title="Загорский, Владислав (страница отсутствует)">Владислав Загорский</a>, в середине <a class="mw-redirect" href="/wiki/1930-%D0%B5" title="1930-е">1930-х</a> годов — студенты <a href="/wiki/%D0%A3%D0%BD%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D1%82%D0%B5%D1%82_%D0%A1%D1%82%D0%B5%D1%84%D0%B0%D0%BD%D0%B0_%D0%91%D0%B0%D1%82%D0%BE%D1%80%D0%B8%D1%8F" title="Университет Стефана Батория">Университета Стефана Батория</a>, объединённые в клуб «Влучендзы» (<span lang="pl" style="font-style:italic;">Włóczędzy</span>; «Бродяги»); обширные исследования проводились в <a class="mw-redirect" href="/wiki/1960-%D0%B5" title="1960-е">1960-х</a> годах.
</p><p>Они выявили около четырёх тысяч захоронений, 153 гроба, мумифицировавшиеся тела 473 взрослых и 85 детей, 205 черепа. Для любителей острых ощущений были организованы экскурсии, однако вскоре их пришлось прекратить, поскольку из-за нарушений микроклимата мумии начали разлагаться, к тому же экскурсанты пытались вынести из подземелий кости и черепа.<sup class="reference" id="cite_ref-Jono,_Dominikonų_1998._P._148_13-1"><a href="#cite_note-Jono,_Dominikonų_1998._P._148-13">[13]</a></sup>.
</p>
<h2><span id=".D0.9C.D0.BE.D0.BD.D0.B0.D1.81.D1.82.D1.8B.D1.80.D1.8C"></span><span class="mw-headline" id="Монастырь">Монастырь</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%BE%D1%81%D1%82%D1%91%D0%BB_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%94%D1%83%D1%85%D0%B0_(%D0%92%D0%B8%D0%BB%D1%8C%D0%BD%D1%8E%D1%81)&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Монастырь»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%BE%D1%81%D1%82%D1%91%D0%BB_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%94%D1%83%D1%85%D0%B0_(%D0%92%D0%B8%D0%BB%D1%8C%D0%BD%D1%8E%D1%81)&amp;action=edit&amp;section=5" title="Редактировать код раздела «Монастырь»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Dominikonai10.jpg"><img class="mw-file-element" data-file-height="2304" data-file-width="3072" decoding="async" height="188" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/9f/Dominikonai10.jpg/250px-Dominikonai10.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/9f/Dominikonai10.jpg/375px-Dominikonai10.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/9/9f/Dominikonai10.jpg/500px-Dominikonai10.jpg 2x" width="250"/></a><figcaption><i>Бывший монастырь</i></figcaption></figure>
<p>Ансамбль храма и монастыря занимает обширный участок земли между улицами Доминикону (<span lang="lt" style="font-style:italic;">Dominikonų g.</span>), Швянто Игното (<span lang="lt" style="font-style:italic;">Švento Ignoto g.</span>) и <a href="/wiki/%D0%A3%D0%BB%D0%B8%D1%86%D0%B0_%D0%92%D0%B8%D0%BB%D1%8C%D0%BD%D1%8F%D1%83%D1%81" title="Улица Вильняус">Вильняус</a> (<span lang="lt" style="font-style:italic;">Vilniaus g.</span>) общей площадью 5,6 га. Это был центр доминиканской провинции, в нём действовал <a href="/wiki/%D0%9D%D0%BE%D0%B2%D0%B8%D1%86%D0%B8%D0%B0%D1%82" title="Новициат">новициат</a> и училище теологии, выпускникам которого присваивались те же степени, что и в академии. Число монахов составляло 140 человек. Монастырские помещения вместе с храмом страдали от пожаров, грабежей, разрушений и несколько раз восстанавливались и перестраивались.
</p><p>Трёхэтажные здания окружают квадратный (19 на 18 м) <a href="/wiki/%D0%9A%D0%BB%D1%83%D0%B0%D1%82%D1%80" title="Клуатр">клуатр</a> — внутренний двор обители с остатками галерей и невысокой барочной дорической колонны посередине в память жертв чумы. С костёлом монастырь соединяют коридоры вокруг этого двора, прежде бывшими открытыми галереями.
</p>
<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Image-Dominikonai7.jpg"><img class="mw-file-element" data-file-height="2304" data-file-width="3072" decoding="async" height="188" src="//upload.wikimedia.org/wikipedia/commons/thumb/0/04/Image-Dominikonai7.jpg/250px-Image-Dominikonai7.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/0/04/Image-Dominikonai7.jpg/375px-Image-Dominikonai7.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/0/04/Image-Dominikonai7.jpg/500px-Image-Dominikonai7.jpg 2x" width="250"/></a><figcaption><i>Бывшая доминиканская аптека</i></figcaption></figure>
<p>В отличие от других монастырей, в доминиканском монастыре не было отдельных келий. Поэтому его особенностью являются достаточно просторные помещения дормиториума (спальни), больницы, зала для диспутов, рефлекторума (столовая). Левую стену западного корпуса украшают восемь фресок с изображениями эпизодов из истории доминиканского ордена. На другой стене написаны портреты четырёх римских пап; на шести плафонах сводов изображены сцены из <a href="/wiki/%D0%91%D0%B8%D0%B1%D0%BB%D0%B8%D1%8F" title="Библия">Библии</a> и видение святого Доминика. Фрески на сводах северного корпуса изображают сцены жизни Христа, на сводах восточного корпуса в медальонах изображены религиозная символика и дева Мария со святыми, на сводах южного корпуса — пять фресок со сценами жизни Христа и девы Марии. В <a href="/wiki/1899_%D0%B3%D0%BE%D0%B4" title="1899 год">1899 году</a> роспись была подновлена. С <a href="/wiki/1978_%D0%B3%D0%BE%D0%B4" title="1978 год">1978 года</a> фрески изучались и консервировались.
</p><p>В <a href="/wiki/1807_%D0%B3%D0%BE%D0%B4" title="1807 год">1807 году</a> российская администрация устроила в части принадлежащих доминиканцам помещений военный госпиталь и тюрьму. В <a href="/wiki/1812_%D0%B3%D0%BE%D0%B4" title="1812 год">1812 году</a> здесь же был французский госпиталь. Монастырь значительно пострадал от французских солдат: мебель, окна, двери были сожжены, утварь и библиотека расхищена.
</p><p>После <a href="/wiki/%D0%9E%D1%82%D0%B5%D1%87%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0_1812_%D0%B3%D0%BE%D0%B4%D0%B0" title="Отечественная война 1812 года">Отечественной войны 1812 года</a> в обращённой в тюрьму части построек содержались в заключении <a href="/wiki/%D0%A4%D0%B8%D0%BB%D0%B0%D1%80%D0%B5%D1%82%D1%8B" title="Филареты">филареты</a> (<a class="mw-redirect" href="/wiki/1822" title="1822">1822</a>—<a class="mw-redirect" href="/wiki/1823" title="1823">1823</a>), позднее участники <a class="mw-redirect" href="/wiki/%D0%92%D0%BE%D1%81%D1%81%D1%82%D0%B0%D0%BD%D0%B8%D0%B5_1830_%D0%B3%D0%BE%D0%B4%D0%B0" title="Восстание 1830 года">восстания 1830—1831 годов</a>. Окончательно монастырь был упразднён в <a href="/wiki/1844_%D0%B3%D0%BE%D0%B4" title="1844 год">1844 году</a>. Часть зданий (за храмом вдоль улицы Игното) стала тюрьмой. Здания юго-западной части монастыря до улицы Вильняус были переданы магистрату.
</p><p>После череды смен владельцев и назначений, сопровождавшихся перестройками, часть этих зданий была разрушена при бомбардировке города советской авиацией в <a href="/wiki/1944_%D0%B3%D0%BE%D0%B4" title="1944 год">1944 году</a>, часть была снесена уже после <a href="/wiki/%D0%92%D1%82%D0%BE%D1%80%D0%B0%D1%8F_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Вторая мировая война">Второй мировой войны</a><sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup>. Сейчас там находится гимназия имени Саломеи Нерис и школьный стадион. В примыкавшем к храму корпусе былой доминиканской аптеки с <a href="/wiki/1922_%D0%B3%D0%BE%D0%B4" title="1922 год">1922 года</a> действовал книжный магазин Святого Войцеха. После Второй мировой войны и до <a href="/wiki/1985_%D0%B3%D0%BE%D0%B4" title="1985 год">1985 года</a> в этом здании, с фасадом украшенным <a href="/wiki/%D0%A1%D0%B3%D1%80%D0%B0%D1%84%D1%84%D0%B8%D1%82%D0%BE" title="Сграффито">сграффито</a> чёрного и красного цветов, действовал единственный в то время в Вильнюсе антиквариат, фактически букинистический магазин. Позднее здесь располагалось католическое издательство <span lang="lt" style="font-style:italic;">Katalikų pasaulis</span> и магазин религиозной литературы.
</p><p>В заключении в бывшем монастыре пребывали участники <a class="mw-redirect" href="/wiki/%D0%9F%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%BE%D0%B5_%D0%B2%D0%BE%D1%81%D1%81%D1%82%D0%B0%D0%BD%D0%B8%D0%B5_1863_%D0%B3%D0%BE%D0%B4%D0%B0" title="Польское восстание 1863 года">восстания 1863 года</a> и его руководители — <a class="mw-redirect" href="/wiki/%D0%93%D0%B5%D0%B9%D1%88%D1%82%D0%BE%D1%80,_%D0%AF%D0%BA%D1%83%D0%B1" title="Гейштор, Якуб">Якуб Гейштор</a>, <a href="/wiki/%D0%9A%D0%B0%D0%BB%D0%B8%D0%BD%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9,_%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD_%D0%A1%D0%B5%D0%BC%D1%91%D0%BD%D0%BE%D0%B2%D0%B8%D1%87" title="Калиновский, Константин Семёнович">Кастусь Калиновский</a>, его ближайший помощник <a href="/wiki/%D0%94%D0%B0%D0%BB%D0%B5%D0%B2%D1%81%D0%BA%D0%B8%D0%B9,_%D0%A2%D0%B8%D1%82%D1%83%D1%81" title="Далевский, Титус">Титус Далевский</a>, его брат, один из основателей <a href="/wiki/%D0%91%D1%80%D0%B0%D1%82%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D0%BE%D1%8E%D0%B7_%D0%BB%D0%B8%D1%82%D0%BE%D0%B2%D1%81%D0%BA%D0%BE%D0%B9_%D0%BC%D0%BE%D0%BB%D0%BE%D0%B4%D1%91%D0%B6%D0%B8" title="Братский союз литовской молодёжи">Братского союза литовской молодёжи</a> <a href="/wiki/%D0%94%D0%B0%D0%BB%D0%B5%D0%B2%D1%81%D0%BA%D0%B8%D0%B5" title="Далевские">Франциск Далевский</a> и другие. В <a href="/wiki/1993_%D0%B3%D0%BE%D0%B4" title="1993 год">1993 году</a> на здании по улице Игното (<span lang="lt" style="font-style:italic;">Švento Ignoto g. 11</span>) была открыта мемориальная плита Кастусю Калиновскому<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup>.
</p><p>В межвоенные годы предпринимались усилия по возвращению костёлу всех зданий монастыря. В <a class="mw-redirect" href="/wiki/1980-%D0%B5" title="1980-е">1980-х</a> годах шли подготовительные работы по устройству в монастыре конструкторского бюро Министерства связи с лабораториями и производственными помещениями. Со сменой государственного строя в порядке <a href="/wiki/%D0%A0%D0%B5%D1%81%D1%82%D0%B8%D1%82%D1%83%D1%86%D0%B8%D1%8F" title="Реституция">реституции</a> заброшенные постройки были переданы церкви. Часть их в настоящее время ремонтируется.
</p>
