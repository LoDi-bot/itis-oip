<div id="mw-fr-revisiondetails-wrapper" style="position:relative;"><div class="flaggedrevs_short_details" id="mw-fr-revisiondetails" style="display:none">Текущая версия страницы пока <a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%9F%D1%80%D0%BE%D0%B2%D0%B5%D1%80%D0%BA%D0%B0_%D1%81%D1%82%D0%B0%D1%82%D0%B5%D0%B9/%D0%9F%D0%BE%D1%8F%D1%81%D0%BD%D0%B5%D0%BD%D0%B8%D0%B5_%D0%B4%D0%BB%D1%8F_%D1%87%D0%B8%D1%82%D0%B0%D1%82%D0%B5%D0%BB%D0%B5%D0%B9" title="Википедия:Проверка статей/Пояснение для читателей">не проверялась</a> опытными участниками и может значительно отличаться от <a class="external text" href="https://ru.wikipedia.org/w/index.php?title=%D0%92%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C&amp;stable=1">версии</a>, проверенной 13 сентября 2022 года; проверки требует <a class="external text" href="https://ru.wikipedia.org/w/index.php?title=%D0%92%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C&amp;oldid=125439627&amp;diff=cur&amp;diffonly=0">1 правка</a>.</div>

<div class="mw-body-content" id="mw-content-text"><div class="mw-content-ltr mw-parser-output" dir="ltr" lang="ru"><p><b>Ве́чный календа́рь</b> — календарь на широкий диапазон лет, предназначенный для <a href="/wiki/%D0%9E%D0%BF%D1%80%D0%B5%D0%B4%D0%B5%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5_%D0%B4%D0%BD%D1%8F_%D0%BD%D0%B5%D0%B4%D0%B5%D0%BB%D0%B8" title="Определение дня недели">определения дня недели</a>. Многие из вечных календарей фактически создают сетку календаря на выбранный месяц выбранного года.
</p><p>Вечные календари могут также использоваться для определения <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B5%D1%85%D0%BE%D0%B4%D1%8F%D1%89%D0%B8%D0%B5_%D0%BF%D1%80%D0%B0%D0%B7%D0%B4%D0%BD%D0%B8%D0%BA%D0%B8" title="Переходящие праздники">церковных праздников без фиксированной даты</a>.
</p>

<h2><span id=".D0.98.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.8F"></span><span class="mw-headline" id="История">История</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%92%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C&amp;veaction=edit&amp;section=1" title="Редактировать раздел «История»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%92%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C&amp;action=edit&amp;section=1" title="Редактировать код раздела «История»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:P%C3%A5sktavla_ur_Liljegrens_Runl%C3%A4ra_(ur_Sverige_Runinskrifter).png"><img class="mw-file-element" data-file-height="729" data-file-width="478" decoding="async" height="458" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/e7/P%C3%A5sktavla_ur_Liljegrens_Runl%C3%A4ra_%28ur_Sverige_Runinskrifter%29.png/300px-P%C3%A5sktavla_ur_Liljegrens_Runl%C3%A4ra_%28ur_Sverige_Runinskrifter%29.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/e7/P%C3%A5sktavla_ur_Liljegrens_Runl%C3%A4ra_%28ur_Sverige_Runinskrifter%29.png/450px-P%C3%A5sktavla_ur_Liljegrens_Runl%C3%A4ra_%28ur_Sverige_Runinskrifter%29.png 1.5x, //upload.wikimedia.org/wikipedia/commons/e/e7/P%C3%A5sktavla_ur_Liljegrens_Runl%C3%A4ra_%28ur_Sverige_Runinskrifter%29.png 2x" width="300"/></a><figcaption>Вечный календарь из <a href="/wiki/%D0%A8%D0%B2%D0%B5%D1%86%D0%B8%D1%8F" title="Швеция">Швеции</a> для вычисления дня <a href="/wiki/%D0%9F%D0%B0%D1%81%D1%85%D0%B0" title="Пасха">Пасхи</a> в 1140—1671 по <a href="/wiki/%D0%AE%D0%BB%D0%B8%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C" title="Юлианский календарь">юлианскому календарю</a>. Каждая <a class="mw-redirect" href="/wiki/%D0%A0%D1%83%D0%BD%D0%B0" title="Руна">руна</a> соответствует определённому номеру недели, на которую будет приходиться праздник</figcaption></figure>
<p>Известные сегодня табличные календари приобрели современную форму только после того, как была <a href="/wiki/%D0%98%D1%81%D1%82%D0%BE%D1%80%D0%B8%D1%8F_%D0%BA%D0%BD%D0%B8%D0%B3%D0%BE%D0%BF%D0%B5%D1%87%D0%B0%D1%82%D0%B0%D0%BD%D0%B8%D1%8F_%D0%B2_%D0%95%D0%B2%D1%80%D0%BE%D0%BF%D0%B5" title="История книгопечатания в Европе">изобретена печать</a>. Вечные же календари придуманы гораздо раньше — время их создания восходит ещё к <a href="/wiki/%D0%9C%D0%BE%D0%BD%D0%B3%D0%BE%D0%BB%D0%BE-%D1%82%D0%B0%D1%82%D0%B0%D1%80%D1%81%D0%BA%D0%BE%D0%B5_%D0%B8%D0%B3%D0%BE" title="Монголо-татарское иго">домонгольскому</a> времени — вероятно, один из первых подобных календарей был найден на стене <a href="/wiki/%D0%A1%D0%BE%D1%84%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D0%BE%D0%B1%D0%BE%D1%80_(%D0%9A%D0%B8%D0%B5%D0%B2)" title="Софийский собор (Киев)">Киевской Софии</a><sup class="reference" id="cite_ref-malävin_1-0"><a href="#cite_note-malävin-1">[1]</a></sup>.
</p><p>В <a href="/wiki/1902_%D0%B3%D0%BE%D0%B4" title="1902 год">1902 году</a> на <a href="/wiki/%D0%92%D0%B8%D0%BB%D1%8C%D0%BD%D1%8E%D1%81" title="Вильнюс">Виленской</a> сельскохозяйственной выставке были представлены вечные календари, которые стали основой для многих конструкций.
</p><p>Вечные календари в больших объёмах выпускались в <a class="mw-redirect" href="/wiki/%D0%A1%D0%A1%D0%A1%D0%A0" title="СССР">СССР</a>. Массовое производство и столь же массовая реализация карманных календарей была начата в <a href="/wiki/1946_%D0%B3%D0%BE%D0%B4" title="1946 год">1946 году</a>. Первые экземпляры из картона выпускались в <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%B2%D0%B0" title="Москва">Москве</a> и <a class="mw-redirect" href="/wiki/%D0%9B%D0%B5%D0%BD%D0%B8%D0%BD%D0%B3%D1%80%D0%B0%D0%B4" title="Ленинград">Ленинграде</a>. Однако первые устройства были настолько недолговечны, что не могли прослужить даже десятую часть покрываемого ими срока.
</p><p>Второй раз активный выпуск календарей начался в <a class="mw-redirect" href="/wiki/1960-%D0%B5" title="1960-е">1960</a>—<a class="mw-redirect" href="/wiki/1970-%D0%B5" title="1970-е">1970-х годах</a>, когда материал был сменён на алюминий. На этот раз производились календари <a href="#Дисковый_вечный_календарь">дискового типа</a>.
</p><p>В 1970-е годы подобные календари стали массово делаться из картона. Однообразный дизайн и невысокое качество продукции способствовало отсутствию спроса на эти устройства у населения страны и у коллекционеров.
</p><p>Тогда же на прилавках и в киосках стали появляться <a href="#Вечный_календарь-таблица">табличные</a> варианты. Для экономии места и компактности многие из них не снабжались даже инструкцией.
</p><p>После <a href="/wiki/%D0%A0%D0%B0%D1%81%D0%BF%D0%B0%D0%B4_%D0%A1%D0%A1%D0%A1%D0%A0" title="Распад СССР">распада СССР</a> вечные календари прекратили выпускаться и использоваться<sup class="reference" id="cite_ref-malävin_1-1"><a href="#cite_note-malävin-1">[1]</a></sup>. На это повлияло и то, что почти все календари были рассчитаны на период до <a href="/wiki/2000_%D0%B3%D0%BE%D0%B4" title="2000 год">2000 года</a>, а также повсеместное распространение компьютеров.
</p>
<h2><span id=".D0.A3.D1.81.D1.82.D1.80.D0.BE.D0.B9.D1.81.D1.82.D0.B2.D0.BE"></span><span class="mw-headline" id="Устройство">Устройство</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%92%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Устройство»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%92%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C&amp;action=edit&amp;section=2" title="Редактировать код раздела «Устройство»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Вечный календарь может быть реализован самыми разнообразными способами, внешне абсолютно непохожими друг на друга. Но в принципе их действия лежит одно — вечные <a href="/wiki/%D0%AE%D0%BB%D0%B8%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C" title="Юлианский календарь">юлианские календари</a> образуют цикл с периодичностью в 28 лет, вечные <a href="/wiki/%D0%93%D1%80%D0%B8%D0%B3%D0%BE%D1%80%D0%B8%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C" title="Григорианский календарь">григорианские календари</a> — 400 лет. Более простые версии вечного <a href="/wiki/%D0%93%D1%80%D0%B8%D0%B3%D0%BE%D1%80%D0%B8%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C" title="Григорианский календарь">григорианского календаря</a>, не включающие в диапазон лет <a href="/wiki/%D0%92%D0%B8%D1%81%D0%BE%D0%BA%D0%BE%D1%81%D0%BD%D1%8B%D0%B9_%D0%B3%D0%BE%D0%B4" title="Високосный год">невисокосные годы</a>, заканчивающиеся на 00, используют цикл в 28 лет.
</p>
<h3><span id=".D0.92.D0.B5.D1.87.D0.BD.D1.8B.D0.B9_.D0.BA.D0.B0.D0.BB.D0.B5.D0.BD.D0.B4.D0.B0.D1.80.D1.8C-.D1.82.D0.B0.D0.B1.D0.BB.D0.B8.D1.86.D0.B0"></span><span class="mw-headline" id="Вечный_календарь-таблица">Вечный календарь-таблица</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%92%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Вечный календарь-таблица»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%92%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C&amp;action=edit&amp;section=3" title="Редактировать код раздела «Вечный календарь-таблица»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Придумано множество вечных календарей такого вида, однако почти все они сводятся к одному: имеется 14 календарных сеток (7 возможных дней недели, с которых начинается год × 2 варианта — <a href="/wiki/%D0%92%D0%B8%D1%81%D0%BE%D0%BA%D0%BE%D1%81%D0%BD%D1%8B%D0%B9_%D0%B3%D0%BE%D0%B4" title="Високосный год">високосный</a> и <a href="/wiki/%D0%9D%D0%B5%D0%B2%D0%B8%D1%81%D0%BE%D0%BA%D0%BE%D1%81%D0%BD%D1%8B%D0%B9_%D0%B3%D0%BE%D0%B4" title="Невисокосный год">невисокосный год</a>), и таблица, помогающая выбрать календарную сетку для нужного года либо определить сдвиг, помогающий правильно преобразовать одну сетку.
</p><p>Календари этого вида различаются по степени удобства и быстроте использования. Такие календари иногда публиковались в газетах и журналах для вырезания и использования читателями. Наиболее часто применялись два варианта.
</p>
<h4><span id=".D0.92.D0.B0.D1.80.D0.B8.D0.B0.D0.BD.D1.82.D1.8B_.D1.82.D0.B0.D0.B1.D0.BB.D0.B8.D1.87.D0.BD.D1.8B.D1.85_.D0.B2.D0.B5.D1.87.D0.BD.D1.8B.D1.85_.D0.BA.D0.B0.D0.BB.D0.B5.D0.BD.D0.B4.D0.B0.D1.80.D0.B5.D0.B9"></span><span class="mw-headline" id="Варианты_табличных_вечных_календарей">Варианты табличных вечных календарей</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%92%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Варианты табличных вечных календарей»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%92%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C&amp;action=edit&amp;section=4" title="Редактировать код раздела «Варианты табличных вечных календарей»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h4>
<h5><span id=".D0.92.D0.B0.D1.80.D0.B8.D0.B0.D0.BD.D1.82_1_.E2.80.94_.D1.81_1901_.D0.BF.D0.BE_2096_.D0.B3.D0.BE.D0.B4"></span><span class="mw-headline" id="Вариант_1_—_с_1901_по_2096_год">Вариант 1 — с 1901 по 2096 год</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%92%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Вариант 1 — с 1901 по 2096 год»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%92%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C&amp;action=edit&amp;section=5" title="Редактировать код раздела «Вариант 1 — с 1901 по 2096 год»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h5>
<ul><li>Для того, чтобы узнать день недели конкретного дня, требуется:
<ol><li>Найти в первой таблице цифру, соответствующую указанному году и месяцу;</li>
<li>Сложить эту цифру с числом дня;</li>
<li>Найти во второй таблице получившееся значение и посмотреть, какому дню недели оно соответствует.</li></ol></li></ul>
<ul><li>Пример: требуется определить, каким днём недели было <a href="/wiki/22_%D1%84%D0%B5%D0%B2%D1%80%D0%B0%D0%BB%D1%8F" title="22 февраля">22 февраля</a> <a href="/wiki/2007_%D0%B3%D0%BE%D0%B4" title="2007 год">2007 года</a>.
<ol><li>Цифра, соответствующая <a href="/wiki/%D0%A4%D0%B5%D0%B2%D1%80%D0%B0%D0%BB%D1%8C" title="Февраль">февралю</a> (<b>ф</b>) 2007 в таблице 1, равна <b>3</b>.</li>
<li>22+3=<b>25</b>.</li>
<li>Числу 25 в таблице 2 соответствует <b>четверг</b> — это и есть искомый день недели.</li></ol></li></ul>
<ul><li>Таблица. <i>Примечание:</i> серым цветом выделены високосные годы. Месяцы обозначены первыми буквами их названий.</li></ul>

<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Ewiger_Kalender_Russisch.png"><img class="mw-file-element" data-file-height="1506" data-file-width="2169" decoding="async" height="208" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/61/Ewiger_Kalender_Russisch.png/300px-Ewiger_Kalender_Russisch.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/61/Ewiger_Kalender_Russisch.png/450px-Ewiger_Kalender_Russisch.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/61/Ewiger_Kalender_Russisch.png/600px-Ewiger_Kalender_Russisch.png 2x" width="300"/></a><figcaption>Таблица для вечного григорианского календаря (с 15 октября 1582 года)</figcaption></figure>
<ul><li>Для того, чтобы узнать день недели, требуется:
<ol><li>В верхнем левом столбце найти первые две цифры года;</li>
<li>В найденной строке найти столбец с последними двумя цифрами номера года;</li>
<li>В месячных календарях (расположены слева и справа) найти строку, соответствующую числу месяца;</li>
<li>На пересечении строки числа месяца и столбца года найти день недели.</li></ol></li></ul>
<ul><li>Пример: требуется определить, каким днём недели было <a href="/wiki/4_%D1%84%D0%B5%D0%B2%D1%80%D0%B0%D0%BB%D1%8F" title="4 февраля">4 февраля</a> <a href="/wiki/2011_%D0%B3%D0%BE%D0%B4" title="2011 год">2011 года</a>.
<ol><li>В столбце «Столетия» найти строку с первыми двумя цифрами номера года «20»;</li>
<li>В найденной строке найти столбец с последними двумя цифрами номера года «11»;</li>
<li>В календаре «Февраль» найти строку, соответствующую числу месяца «4»;</li>
<li>На пересечении строки числа месяца и столбца года найти день недели «пятница».</li></ol></li></ul>
<ul><li>Примечание: <b>29 февраля</b> — только в високосные годы (столбцы выделены <b>жирным</b> шрифтом).</li></ul>
<h5><span id=".D0.92.D0.B0.D1.80.D0.B8.D0.B0.D0.BD.D1.82_3_.E2.80.94_.D0.BA.D0.B0.D0.BB.D0.B5.D0.BD.D0.B4.D0.B0.D1.80.D1.8C_.D0.B4.D0.BB.D1.8F_.D1.81.D1.82.D0.B0.D1.80.D0.BE.D0.B3.D0.BE_.D0.B8_.D0.BD.D0.BE.D0.B2.D0.BE.D0.B3.D0.BE_.D1.81.D1.82.D0.B8.D0.BB.D0.B5.D0.B9_.D0.BD.D0.B0_28_.D0.B2.D0.B5.D0.BA.D0.BE.D0.B2"></span><span class="mw-headline" id="Вариант_3_—_календарь_для_старого_и_нового_стилей_на_28_веков">Вариант 3 — календарь для старого и нового стилей на 28 веков</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%92%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Вариант 3 — календарь для старого и нового стилей на 28 веков»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%92%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C&amp;action=edit&amp;section=7" title="Редактировать код раздела «Вариант 3 — календарь для старого и нового стилей на 28 веков»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h5>

любой календарной даты старого и нового стилей
 
</caption>
<tbody><tr>
<td align="left" bgcolor="honeydew" colspan="7" rowspan="19" valign="top">
<center>Правила</center>
<p><font size="-3">
В левой части таблицы отыскать строку, содержащую число столетий нужного года по старому или новому стилю, а в верхней части таблицы — столбец с двумя последними цифрами номера года. Запомнить букву на пересечении строки и столбца, это — <a href="/wiki/%D0%92%D1%80%D1%83%D1%86%D0%B5%D0%BB%D0%B5%D1%82%D0%BE" title="Вруцелето">вруцелето</a> года.
</font></p><p><font size="-3">В таблице с правой стороны найти нужный месяц. В строке, в которой находится этот месяц, отыскать ту же букву.
</font></p><p><font size="-3">В левой нижней части, в разделе «числа месяца», найти нужное число. Пересечение строки, в которой находится это число, со столбиком дней недели, расположенным под найденной буквой, и даёт искомый день недели.
</font></p><p><font size="-3">При этом следует различать январь и февраль в простых и високосных годах: они занимают различные положения в графе месяцев (для простых годов они обозначены «Яп» и «Фп», для високосных — <b>«Яв»</b> и <b>«Фв»</b>). Високосные годы выделены полужирным шрифтом. Год «00» подчёркнут, в старом стиле он всегда високосный, а в новом — только, если число столетий делится на 4.
</font></p><p class="mw-empty-elt">
</p>
</td>
<td align="center" colspan="7">Последние цифры года
</td>
<td bgcolor="honeydew" colspan="6" rowspan="19" valign="top">
<center>Пример 1</center>
<p><font size="-3">
Определить, на какой день недели приходится 31 марта 2006 г.
Слева в колонке нового стиля находим число «20», сверху — число «06». На пересечении строки и столбика стоит буква «Ѕ». Находим эту же букву в строке, в которой стоит обозначение «Март». Внизу под ней расположен столбик с днями недели, соответствующий марту, числа которого слева. Итак, 31 марта 2006 г. приходится на пятницу.
</font>
</p>
<center>Пример 2</center>
<p><font size="-3">
Определить, на какой день недели приходится 1 января 45 г. до н. э.
Прежде всего находим число года астрономического счёта: «−44». Далее представляем его в виде «−100+56». На пересечении столбика «56» со строкой «−1» находим вруцелето «Е». Год 45 до н. э. был високосным. Поэтому в правой части таблицы «месяцы» находим «Яв» и напротив числа «1» в колонке, соответствующей вруцелето «Е», находим день — пятница.
</font>
</p>
<center>Пример 3</center>
<p><font size="-3">
1 января 1900 г. ст. ст.
Год високосный, вруцелето «Ѕ» («00» подчёркнуто). Месяц — «Яв». Суббота.
</font>
</p>
<center>Пример 4</center>
<p><font size="-3">
1 января 1900 г. н. ст.
Год простой, вруцелето «З» («00» подчёркнуто). Месяц — «Яп». Понедельник.
</font></p><p class="mw-empty-elt">

<h3><span id=".D0.94.D0.B8.D1.81.D0.BA.D0.BE.D0.B2.D1.8B.D0.B9_.D0.B2.D0.B5.D1.87.D0.BD.D1.8B.D0.B9_.D0.BA.D0.B0.D0.BB.D0.B5.D0.BD.D0.B4.D0.B0.D1.80.D1.8C"></span><span class="mw-headline" id="Дисковый_вечный_календарь">Дисковый вечный календарь</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%92%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C&amp;veaction=edit&amp;section=8" title="Редактировать раздел «Дисковый вечный календарь»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%92%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C&amp;action=edit&amp;section=8" title="Редактировать код раздела «Дисковый вечный календарь»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure class="mw-default-size mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:PerpetualCalendar.svg"><img class="mw-file-element" data-file-height="800" data-file-width="800" decoding="async" height="220" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/c7/PerpetualCalendar.svg/220px-PerpetualCalendar.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/c/c7/PerpetualCalendar.svg/330px-PerpetualCalendar.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/c/c7/PerpetualCalendar.svg/440px-PerpetualCalendar.svg.png 2x" width="220"/></a><figcaption>Дисковый вечный календарь на 28 лет (2001—2028). <a href="/wiki/%D0%92%D0%B8%D1%81%D0%BE%D0%BA%D0%BE%D1%81%D0%BD%D1%8B%D0%B9_%D0%B3%D0%BE%D0%B4" title="Високосный год">Високосные годы</a> выделены <b>жирным</b> шрифтом</figcaption></figure>
<figure class="mw-default-size mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Cal1969_cheljabinsk_bw.jpg"><img class="mw-file-element" data-file-height="2055" data-file-width="1450" decoding="async" height="312" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/cc/Cal1969_cheljabinsk_bw.jpg/220px-Cal1969_cheljabinsk_bw.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/c/cc/Cal1969_cheljabinsk_bw.jpg/330px-Cal1969_cheljabinsk_bw.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/c/cc/Cal1969_cheljabinsk_bw.jpg/440px-Cal1969_cheljabinsk_bw.jpg 2x" width="220"/></a><figcaption>Дисковый вечный календарь из алюминия на 1969—1991 годы, изготовленный в <a href="/wiki/%D0%A7%D0%B5%D0%BB%D1%8F%D0%B1%D0%B8%D0%BD%D1%81%D0%BA" title="Челябинск">Челябинске</a></figcaption></figure>
<figure class="mw-default-size mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Perpetual_calendar.jpg"><img class="mw-file-element" data-file-height="3426" data-file-width="3120" decoding="async" height="242" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/e8/Perpetual_calendar.jpg/220px-Perpetual_calendar.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/e8/Perpetual_calendar.jpg/330px-Perpetual_calendar.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/e8/Perpetual_calendar.jpg/440px-Perpetual_calendar.jpg 2x" width="220"/></a><figcaption>Популярный в СССР сувенирный дисковый вечный календарь «<a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%B2%D0%B0" title="Москва">Москва</a>». На фото показан календарь на 1973—2000 годы, у которого также есть термометр и подставки для перьевых ручек</figcaption></figure>
<p>Дисковый вечный календарь представляет собой конструкцию, в которую вставлен диск, на внешний радиус которого нанесены номера годов (при этом високосные годы выделяются особым шрифтом или цветом), а на внутренний — дни недели. При выборе месяца требуется повернуть диск так, чтобы номер текущего года и название месяца совпали. В результате дни недели расположатся таким образом, что будет получена сетка календаря на выбранный месяц.
</p><p>Как правило, диск таких календарей делится на 28 частей, а сами календари рассчитаны на 23—28 лет, однако возможны и другие модификации.
</p><p>Иногда на диск наносятся не дни недели, а сами числа месяца, что позволяет получить календарную сетку в более привычной форме — неделя всегда начинается с понедельника или воскресенья, в зависимости от страны, где используется календарь. Но такие календари производились редко, в основном из-за сложности нанесения цифр месяца по спирали.
</p><p>На календаре остаётся много свободного пространства, которое очень часто заполняется различными изображениями, инструкцией по применению, списками праздников и т. п.)
</p>
<h3><span id=".D0.92_.D0.BF.D1.80.D0.BE.D0.B3.D1.80.D0.B0.D0.BC.D0.BC.D0.B8.D1.80.D0.BE.D0.B2.D0.B0.D0.BD.D0.B8.D0.B8"></span><span class="mw-headline" id="В_программировании">В программировании</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%92%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C&amp;veaction=edit&amp;section=9" title="Редактировать раздел «В программировании»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%92%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C&amp;action=edit&amp;section=9" title="Редактировать код раздела «В программировании»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Существуют разнообразные методы вычисления дня недели. Некоторые из них основываются на заранее просчитанных таблицах сдвигов, а некоторые вычисляют все необходимые значения «с нуля» при каждом запуске. Иногда алгоритмы представляют собой автоматизированные версии табличных календарей.
</p><p>Как правило, каждый метод имеет две модификации — для юлианского и для григорианского календаря.
</p>
<h2><span id=".D0.92.D0.B5.D1.87.D0.BD.D1.8B.D0.B9_.D0.BA.D0.B0.D0.BB.D0.B5.D0.BD.D0.B4.D0.B0.D1.80.D1.8C_.E2.80.94_.D0.BD.D0.B0.D0.B1.D0.BE.D1.80_.D1.82.D0.B0.D0.B1.D0.BB.D0.B8.D1.86"></span><span class="mw-headline" id="Вечный_календарь_—_набор_таблиц">Вечный календарь — набор таблиц</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%92%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C&amp;veaction=edit&amp;section=10" title="Редактировать раздел «Вечный календарь — набор таблиц»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%92%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C&amp;action=edit&amp;section=10" title="Редактировать код раздела «Вечный календарь — набор таблиц»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>В григорианском календаре годы есть високосные и невисокосные; годы начинаются с понедельника по воскресенье. В совокупности это даёт (2×7=14) <a href="/wiki/%D0%A1%D1%82%D1%80%D1%83%D0%BA%D1%82%D1%83%D1%80%D0%B0_%D0%B3%D1%80%D0%B8%D0%B3%D0%BE%D1%80%D0%B8%D0%B0%D0%BD%D1%81%D0%BA%D0%BE%D0%B3%D0%BE_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8F" title="Структура григорианского календаря">14 вариантов календаря</a><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>.
</p>
<h2><span id=".D0.9F.D1.80.D0.B8.D0.BC.D0.B5.D0.BD.D0.B5.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Применение">Применение</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%92%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C&amp;veaction=edit&amp;section=11" title="Редактировать раздел «Применение»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%92%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C&amp;action=edit&amp;section=11" title="Редактировать код раздела «Применение»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<table class="mbox-Орисс plainlinks metadata ambox ambox-content" role="presentation"><tbody><tr><td class="mbox-image"><div style="width:52px"><span typeof="mw:File"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Emblem-important.svg"><img alt="" class="mw-file-element" data-file-height="48" data-file-width="48" decoding="async" height="40" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/4c/Emblem-important.svg/40px-Emblem-important.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/4c/Emblem-important.svg/60px-Emblem-important.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/4c/Emblem-important.svg/80px-Emblem-important.svg.png 2x" width="40"/></a></span></div></td><td class="mbox-text"><div class="mbox-text-div">Возможно, этот раздел содержит <b><a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%9D%D0%B5%D0%B4%D0%BE%D0%BF%D1%83%D1%81%D1%82%D0%B8%D0%BC%D0%BE%D1%81%D1%82%D1%8C_%D0%BE%D1%80%D0%B8%D0%B3%D0%B8%D0%BD%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D1%85_%D0%B8%D1%81%D1%81%D0%BB%D0%B5%D0%B4%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B9" title="Википедия:Недопустимость оригинальных исследований">оригинальное исследование</a></b>.</div><div class="mbox-textsmall-div hide-when-compact" style="font-size:85%"><span class="hide-when-compact"> Проверьте соответствие информации приведённым источникам и удалите или исправьте информацию, являющуюся оригинальным исследованием. В случае необходимости подтвердите информацию авторитетными <a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%A1%D1%81%D1%8B%D0%BB%D0%BA%D0%B8_%D0%BD%D0%B0_%D0%B8%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8" title="Википедия:Ссылки на источники">источниками</a>. В противном случае этот раздел может быть удалён.</span> <span class="mbox-date"><i>(<span class="date">25 мая 2011</span>)</i></span></div></td></tr></tbody></table>
<p>До начала <a href="/wiki/XX_%D0%B2%D0%B5%D0%BA" title="XX век">XX века</a> календари не применялись в целях планирования времени. Основное значение таких приспособлений было религиозным — они использовались для вычисления праздников и знаменательных дат.
</p><p>Во времена СССР вечные календари можно было найти почти во всех учреждениях и во многих квартирах — календарь не занимал много места и мог использоваться несколько десятков лет. Однако лишь немногие календари прослужили весь свой срок.
</p><p>Сейчас многие из календарей используются детьми в качестве игрушек. Иногда вечные календари до сих пор используют для получения дней недели (хотя экземпляры, сохраняющие работоспособность после <a href="/wiki/2000_%D0%B3%D0%BE%D0%B4" title="2000 год">2000 года</a>, очень редки). Иногда они являются предметом коллекционирования.
</p><p>Несмотря на то, что с появлением компьютеров необходимость в вечных календарях отпала, подобные устройства до сих пор встречаются в механических <a href="/wiki/%D0%9D%D0%B0%D1%80%D1%83%D1%87%D0%BD%D1%8B%D0%B5_%D1%87%D0%B0%D1%81%D1%8B" title="Наручные часы">наручных часах</a><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>.
</p><p>Существуют так же специальные онлайн сервисы для определения дня недели по дате<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>.
</p>
