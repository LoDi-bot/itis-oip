<div class="noprint" id="siteSub">Материал из Википедии — свободной энциклопедии</div>

<figure class="mw-default-size" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:RNA-codon.png"><img class="mw-file-element" data-file-height="382" data-file-width="225" decoding="async" height="374" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/cd/RNA-codon.png/220px-RNA-codon.png" srcset="//upload.wikimedia.org/wikipedia/commons/c/cd/RNA-codon.png 1.5x" width="220"/></a><figcaption>Последовательность <a href="/wiki/%D0%9A%D0%BE%D0%B4%D0%BE%D0%BD" title="Кодон">кодонов</a> в части молекулы <a class="mw-redirect" href="/wiki/%D0%9C%D0%A0%D0%9D%D0%9A" title="МРНК">мРНК</a>. Каждый <a href="/wiki/%D0%9A%D0%BE%D0%B4%D0%BE%D0%BD" title="Кодон">кодон</a> состоит из трёх <a class="mw-redirect" href="/wiki/%D0%9D%D1%83%D0%BA%D0%BB%D0%B5%D0%BE%D1%82%D0%B8%D0%B4" title="Нуклеотид">нуклеотидов</a>, обычно соответствующих единственной <a class="mw-redirect" href="/wiki/%D0%90%D0%BC%D0%B8%D0%BD%D0%BE%D0%BA%D0%B8%D1%81%D0%BB%D0%BE%D1%82%D0%B0" title="Аминокислота">аминокислоте</a>. Эта молекула <a class="mw-redirect" href="/wiki/%D0%9C%D0%A0%D0%9D%D0%9A" title="МРНК">мРНК</a> указывает <a href="/wiki/%D0%A0%D0%B8%D0%B1%D0%BE%D1%81%D0%BE%D0%BC%D0%B0" title="Рибосома">рибосоме</a> синтезировать <a class="mw-redirect" href="/wiki/%D0%91%D0%B5%D0%BB%D0%BE%D0%BA" title="Белок">белок</a> согласно данному <a href="/wiki/%D0%93%D0%B5%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%BE%D0%B4" title="Генетический код">генетическому коду</a>.</figcaption></figure>
<p><b>Ма́тричная рибонуклеи́новая кислота́</b> (<b>мРНК</b>, синоним — <b>информацио́нная РНК, иРНК</b>) — <a href="/wiki/%D0%A0%D0%B8%D0%B1%D0%BE%D0%BD%D1%83%D0%BA%D0%BB%D0%B5%D0%B8%D0%BD%D0%BE%D0%B2%D0%B0%D1%8F_%D0%BA%D0%B8%D1%81%D0%BB%D0%BE%D1%82%D0%B0" title="Рибонуклеиновая кислота">РНК</a>, содержащая информацию о <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B2%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D1%81%D1%82%D1%80%D1%83%D0%BA%D1%82%D1%83%D1%80%D0%B0" title="Первичная структура">первичной структуре</a> (аминокислотной последовательности) белков<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>. мРНК синтезируется на основе <a class="mw-redirect" href="/wiki/%D0%94%D0%9D%D0%9A" title="ДНК">ДНК</a> в ходе <a href="/wiki/%D0%A2%D1%80%D0%B0%D0%BD%D1%81%D0%BA%D1%80%D0%B8%D0%BF%D1%86%D0%B8%D1%8F_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Транскрипция (биология)">транскрипции</a>, после чего, в свою очередь, используется в ходе <a href="/wiki/%D0%A2%D1%80%D0%B0%D0%BD%D1%81%D0%BB%D1%8F%D1%86%D0%B8%D1%8F_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Трансляция (биология)">трансляции</a> как матрица для синтеза белков. Тем самым мРНК играет важную роль в «проявлении» (<a href="/wiki/%D0%AD%D0%BA%D1%81%D0%BF%D1%80%D0%B5%D1%81%D1%81%D0%B8%D1%8F_%D0%B3%D0%B5%D0%BD%D0%BE%D0%B2" title="Экспрессия генов">экспрессии</a>) <a href="/wiki/%D0%93%D0%B5%D0%BD" title="Ген">генов</a>.
</p><p>Длина типичной зрелой мРНК составляет от нескольких сотен до нескольких тысяч <a class="mw-redirect" href="/wiki/%D0%9D%D1%83%D0%BA%D0%BB%D0%B5%D0%BE%D1%82%D0%B8%D0%B4" title="Нуклеотид">нуклеотидов</a>. Самые длинные мРНК отмечены у (+)оц РНК-содержащих <a href="/wiki/%D0%92%D0%B8%D1%80%D1%83%D1%81%D1%8B" title="Вирусы">вирусов</a>, например <a href="/wiki/%D0%9F%D0%B8%D0%BA%D0%BE%D1%80%D0%BD%D0%B0%D0%B2%D0%B8%D1%80%D1%83%D1%81%D1%8B" title="Пикорнавирусы">пикорнавирусов</a> — однако следует помнить, что у этих вирусов мРНК образует весь их <a href="/wiki/%D0%93%D0%B5%D0%BD%D0%BE%D0%BC" title="Геном">геном</a>.
</p><p>ДНК нередко сравнивают с чертежами — и, одновременно, инструкциями — для изготовления белков. Развивая эту инженерно-производственную аналогию, можно сказать, что, если ДНК — «полный набор чертежей-инструкций для изготовления белков, находящийся на хранении в сейфе директора завода», то мРНК — «временная рабочая копия чертежа-инструкции для отдельной детали, выдаваемая в сборочный цех». ДНК содержат не детальный образ <i>взрослого</i> организма, а больше похожа на «рецепт» по его изготовлению, который применяется в зависимости от сложившихся текущих условий в ходе экспрессии генов — какие-то из полного набора инструкций используются, а какие-то — нет.
</p>

<h2><span id=".D0.98.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.8F_.D0.BE.D1.82.D0.BA.D1.80.D1.8B.D1.82.D0.B8.D1.8F"></span><span class="mw-headline" id="История_открытия">История открытия</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;veaction=edit&amp;section=1" title="Редактировать раздел «История открытия»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;action=edit&amp;section=1" title="Редактировать код раздела «История открытия»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>К середине XX века были накоплены научные данные, которые позволили заключить, что <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B2%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D1%81%D1%82%D1%80%D1%83%D0%BA%D1%82%D1%83%D1%80%D0%B0" title="Первичная структура">структура</a> <a href="/wiki/%D0%91%D0%B5%D0%BB%D0%BA%D0%B8" title="Белки">белков</a> кодируется участками ДНК — <a href="/wiki/%D0%93%D0%B5%D0%BD" title="Ген">генами</a><sup class="reference" id="cite_ref-Ичас_2-0"><a href="#cite_note-Ичас-2">[2]</a></sup>. Однако непосредственный механизм кодирования не был установлен.
</p><p>Работы <a href="/wiki/%D0%91%D1%80%D0%B0%D1%88%D0%B5,_%D0%96%D0%B0%D0%BD" title="Браше, Жан">Ж. Браше</a> (1944) и <a href="/wiki/%D0%9A%D0%B0%D1%81%D0%BF%D0%B5%D1%80%D1%81%D1%81%D0%BE%D0%BD,_%D0%A2%D0%BE%D1%80%D0%B1%D1%8C%D1%91%D1%80%D0%BD_%D0%9E%D1%81%D0%BA%D0%B0%D1%80" title="Касперссон, Торбьёрн Оскар">Т. Касперссона</a> (1947) показали, что клетки, активно синтезирующие белок, содержат большое количество РНК в <a href="/wiki/%D0%A6%D0%B8%D1%82%D0%BE%D0%BF%D0%BB%D0%B0%D0%B7%D0%BC%D0%B0" title="Цитоплазма">цитоплазме</a>. Впоследствии выяснилось, что это относится главным образом к <a class="mw-redirect" href="/wiki/%D0%A0%D0%A0%D0%9D%D0%9A" title="РРНК">рибосомальной РНК</a>, а не к мРНК, количество которой в клетке относительно невелико. Тем не менее, это наблюдение связывало между собой ДНК, РНК и белок и, вероятно, сыграло роль в предположении о возможной роли РНК как посредника, способного переносить информацию от ДНК в <a href="/wiki/%D0%9A%D0%BB%D0%B5%D1%82%D0%BE%D1%87%D0%BD%D0%BE%D0%B5_%D1%8F%D0%B4%D1%80%D0%BE" title="Клеточное ядро">ядре</a> к аппарату биосинтеза белка в цитоплазме<sup class="reference" id="cite_ref-Crick_3-0"><a href="#cite_note-Crick-3">[3]</a></sup>.
</p><p>В это же время были открыты <a href="/wiki/%D0%A0%D0%B8%D0%B1%D0%BE%D1%81%D0%BE%D0%BC%D0%B0" title="Рибосома">рибосомы</a> — <a href="/wiki/%D0%9D%D1%83%D0%BA%D0%BB%D0%B5%D0%BE%D0%BF%D1%80%D0%BE%D1%82%D0%B5%D0%B8%D0%B4%D1%8B" title="Нуклеопротеиды">рибонуклеопротеидные</a> частицы, синтезирующие белок. Было сделано предположение о том, что гены транскрибируются в РНК рибосом, которые и служат матрицами для синтеза белка<sup class="reference" id="cite_ref-Спирин_4-0"><a href="#cite_note-Спирин-4">[4]</a></sup>. Однако в 1956—1958 годах <a href="/wiki/%D0%91%D0%B5%D0%BB%D0%BE%D0%B7%D0%B5%D1%80%D1%81%D0%BA%D0%B8%D0%B9,_%D0%90%D0%BD%D0%B4%D1%80%D0%B5%D0%B9_%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B5%D0%B2%D0%B8%D1%87" title="Белозерский, Андрей Николаевич">А. Белозерский</a> и <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%80%D0%B8%D0%BD,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%A1%D0%B5%D1%80%D0%B3%D0%B5%D0%B5%D0%B2%D0%B8%D1%87" title="Спирин, Александр Сергеевич">А. Спирин</a>, проведя сравнительный анализ нуклеотидного состава ДНК и РНК ряда микроорганизмов, показали, что, при больших вариациях в составе ДНК, РНК разных видов были довольно похожи<sup class="reference" id="cite_ref-Belozersky_5-0"><a href="#cite_note-Belozersky-5">[5]</a></sup>. Это указывало на то, что основная масса клеточной РНК (рРНК) не отражает нуклеотидный состав ДНК данного организма и не может служить матрицей для синтеза белков. В то же время авторам удалось наблюдать слабую положительную <a href="/wiki/%D0%9A%D0%BE%D1%80%D1%80%D0%B5%D0%BB%D1%8F%D1%86%D0%B8%D1%8F" title="Корреляция">корреляцию</a> между составом ДНК и РНК при больших различиях между видами. Это позволило им предположить, что в клетке, помимо рРНК, существует ещё одна небольшая фракция РНК, которая может быть посредником при экспрессии генов.
</p><p>Эллиот Волкин и Лазарь Астрачан независимо друг от друга пришли к сходным выводам: они обнаружили, что при заражении <a href="/wiki/%D0%91%D0%B0%D0%BA%D1%82%D0%B5%D1%80%D0%B8%D0%B8" title="Бактерии">бактериальных</a> клеток <a href="/wiki/%D0%91%D0%B0%D0%BA%D1%82%D0%B5%D1%80%D0%B8%D0%BE%D1%84%D0%B0%D0%B3%D0%B8" title="Бактериофаги">бактериофагом</a> Т2 они полностью переключаются на синтез белков вируса. В то время как большая часть РНК клетки-хозяина остаётся неизменной, после заражения синтезируется небольшое количество короткоживущей РНК, сходной по нуклеотидному составу с ДНК фага<sup class="reference" id="cite_ref-Volkin1_6-0"><a href="#cite_note-Volkin1-6">[6]</a></sup><sup class="reference" id="cite_ref-Volkin2_7-0"><a href="#cite_note-Volkin2-7">[7]</a></sup>.
</p><p>В 1961 году несколькими группами исследователей было прямо доказано существование короткоживущего РНК-посредника, близкого по структуре к генам в ДНК, который служит матрицей для синтеза белка, связываясь с рибосомами<sup class="reference" id="cite_ref-Brenner_8-0"><a href="#cite_note-Brenner-8">[8]</a></sup><sup class="reference" id="cite_ref-Gros_9-0"><a href="#cite_note-Gros-9">[9]</a></sup>.
</p>
<h2><span id=".C2.AB.D0.96.D0.B8.D0.B7.D0.BD.D0.B5.D0.BD.D0.BD.D1.8B.D0.B9_.D1.86.D0.B8.D0.BA.D0.BB.C2.BB"></span><span class="mw-headline" id="«Жизненный_цикл»">«Жизненный цикл»</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;veaction=edit&amp;section=2" title="Редактировать раздел ««Жизненный цикл»»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;action=edit&amp;section=2" title="Редактировать код раздела ««Жизненный цикл»»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Жизненный цикл молекулы мРНК начинается её «считыванием» с матрицы ДНК (транскрипция) и завершается её деградацией до отдельных нуклеотидов. Молекула мРНК в течение своей жизни может подвергаться различным модификациям перед синтезом белка (трансляцией). <a href="/wiki/%D0%AD%D1%83%D0%BA%D0%B0%D1%80%D0%B8%D0%BE%D1%82%D1%8B" title="Эукариоты">Эукариотические</a> молекулы мРНК часто требуют сложной обработки и транспортировки из ядра — места синтеза мРНК, на рибосомы, где происходит трансляция, в то время как <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%BA%D0%B0%D1%80%D0%B8%D0%BE%D1%82%D1%8B" title="Прокариоты">прокариотические</a> молекулы мРНК этого не требуют и синтез РНК у них сопряжён с синтезом белка<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>.
</p>
<h3><span id=".D0.A2.D1.80.D0.B0.D0.BD.D1.81.D0.BA.D1.80.D0.B8.D0.BF.D1.86.D0.B8.D1.8F"></span><span class="mw-headline" id="Транскрипция">Транскрипция</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Транскрипция»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;action=edit&amp;section=3" title="Редактировать код раздела «Транскрипция»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote navigation-not-searchable">Основная статья: <b><a href="/wiki/%D0%A2%D1%80%D0%B0%D0%BD%D1%81%D0%BA%D1%80%D0%B8%D0%BF%D1%86%D0%B8%D1%8F_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Транскрипция (биология)">Транскрипция (биология)</a></b></div>
<p>Транскрипцией называют процесс копирования генетической информации с ДНК на РНК, в частности на мРНК. Транскрипция осуществляется ферментом <a href="/wiki/%D0%A0%D0%9D%D0%9A-%D0%BF%D0%BE%D0%BB%D0%B8%D0%BC%D0%B5%D1%80%D0%B0%D0%B7%D0%B0" title="РНК-полимераза">РНК-полимеразой</a>, строящей, согласно <a href="/wiki/%D0%9A%D0%BE%D0%BC%D0%BF%D0%BB%D0%B5%D0%BC%D0%B5%D0%BD%D1%82%D0%B0%D1%80%D0%BD%D0%BE%D1%81%D1%82%D1%8C_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Комплементарность (биология)">принципу комплементарности</a>, копию участка ДНК на основании одной из цепей двойной спирали. Этот процесс как у эукариот, так и у прокариот организован одинаково. Основное различие между про- и эукариотами состоит в том, что у эукариот РНК-полимераза во время транскрипции ассоциируется с мРНК-обрабатывающими ферментами, поэтому у них обработка мРНК и транскрипция могут проходить одновременно. Короткоживущие необработанные или частично обработанные продукты транскрипции называются <i><a class="new" href="/w/index.php?title=%D0%9F%D1%80%D0%B5-%D0%BC%D0%A0%D0%9D%D0%9A&amp;action=edit&amp;redlink=1" title="Пре-мРНК (страница отсутствует)">пре-мРНК</a></i>; после полной обработки — <i><a class="new" href="/w/index.php?title=%D0%97%D1%80%D0%B5%D0%BB%D0%B0%D1%8F_%D0%BC%D0%A0%D0%9D%D0%9A&amp;action=edit&amp;redlink=1" title="Зрелая мРНК (страница отсутствует)">зрелая мРНК</a></i>.
</p>
<h3><span id=".D0.A1.D0.BE.D0.B7.D1.80.D0.B5.D0.B2.D0.B0.D0.BD.D0.B8.D0.B5_.D1.8D.D1.83.D0.BA.D0.B0.D1.80.D0.B8.D0.BE.D1.82.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.BE.D0.B9_.D0.BC.D0.A0.D0.9D.D0.9A"></span><span class="mw-headline" id="Созревание_эукариотической_мРНК">Созревание эукариотической мРНК</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Созревание эукариотической мРНК»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;action=edit&amp;section=4" title="Редактировать код раздела «Созревание эукариотической мРНК»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote navigation-not-searchable">Основная статья: <b><a class="mw-redirect" href="/wiki/%D0%9F%D0%BE%D1%81%D1%82%D1%82%D1%80%D0%B0%D0%BD%D1%81%D0%BA%D1%80%D0%B8%D0%BF%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D1%8B%D0%B5_%D0%BC%D0%BE%D0%B4%D0%B8%D1%84%D0%B8%D0%BA%D0%B0%D1%86%D0%B8%D0%B8_%D0%A0%D0%9D%D0%9A" title="Посттранскрипционные модификации РНК">Посттранскрипционные модификации РНК</a></b></div>
<p>В то время как мРНК прокариот (<a href="/wiki/%D0%91%D0%B0%D0%BA%D1%82%D0%B5%D1%80%D0%B8%D0%B8" title="Бактерии">бактерий</a> и <a href="/wiki/%D0%90%D1%80%D1%85%D0%B5%D0%B8" title="Археи">архей</a>), за редкими исключениями, сразу готовы к трансляции и не требуют специальной обработки, эукариотические пре-мРНК подвергаются интенсивным модификациям. Так, одновременно с транскрипцией происходит добавление на 5'-конец молекулы РНК специального модифицированного нуклеотида (<a href="/wiki/%D0%9A%D1%8D%D0%BF" title="Кэп">кэпа</a>), удаление определённых участков РНК (<a href="/wiki/%D0%A1%D0%BF%D0%BB%D0%B0%D0%B9%D1%81%D0%B8%D0%BD%D0%B3" title="Сплайсинг">сплайсинг</a>), а также добавление на 3'-конец адениновых нуклеотидов (так называемый полиадениновый, или поли(А)-, хвост)<sup class="reference" id="cite_ref-proudfoot_11-0"><a href="#cite_note-proudfoot-11">[11]</a></sup>. Обычно эти посттранскрипционные изменения мРНК эукариот обозначают термином «процессинг мРНК».
</p><p>Кэпирование является первым этапом процессинга мРНК. Оно осуществляется, когда синтезируемый <a href="/wiki/%D0%A2%D1%80%D0%B0%D0%BD%D1%81%D0%BA%D1%80%D0%B8%D0%BF%D1%82_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Транскрипт (биология)">транскрипт</a> достигает длины 25—30 нуклеотидов<sup class="reference" id="cite_ref-Rasmussen_12-0"><a href="#cite_note-Rasmussen-12">[12]</a></sup>. Сразу после присоединения кэпа к 5'-концу транскрипта с ним связывается кэп-связывающий комплекс CBC (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">cap binding complex</span>), который остаётся связанным с мРНК до завершения процессинга и важен для всех последующих его этапов<sup class="reference" id="cite_ref-Topisirovic_13-0"><a href="#cite_note-Topisirovic-13">[13]</a></sup>. В процессе сплайсинга из пре-мРНК удаляются не кодирующие белок последовательности — <a href="/wiki/%D0%98%D0%BD%D1%82%D1%80%D0%BE%D0%BD" title="Интрон">интроны</a>. Полиаденилирование необходимо для транспорта большинства мРНК в цитоплазму и защищает молекулы мРНК от быстрой деградации (увеличивает время их полужизни). Лишённые поли(А)-участка молекулы мРНК (например, вирусные) быстро разрушаются в цитоплазме клеток эукариот <a href="/wiki/%D0%A0%D0%B8%D0%B1%D0%BE%D0%BD%D1%83%D0%BA%D0%BB%D0%B5%D0%B0%D0%B7%D1%8B" title="Рибонуклеазы">рибонуклеазами</a>.
</p><p>После завершения всех стадий процессинга мРНК проходит проверку на отсутствие преждевременных <a class="mw-redirect" href="/wiki/%D0%A2%D0%B5%D1%80%D0%BC%D0%B8%D0%BD%D0%B0%D1%82%D0%BE%D1%80%D0%BD%D1%8B%D0%B9_%D0%BA%D0%BE%D0%B4%D0%BE%D0%BD" title="Терминаторный кодон">стоп-кодонов</a>, после чего она становится полноценной матрицей для трансляции<sup class="reference" id="cite_ref-Maquat_14-0"><a href="#cite_note-Maquat-14">[14]</a></sup>. В цитоплазме кэп узнаётся <a href="/wiki/%D0%A4%D0%B0%D0%BA%D1%82%D0%BE%D1%80%D1%8B_%D0%B8%D0%BD%D0%B8%D1%86%D0%B8%D0%B0%D1%86%D0%B8%D0%B8_%D1%82%D1%80%D0%B0%D0%BD%D1%81%D0%BB%D1%8F%D1%86%D0%B8%D0%B8" title="Факторы инициации трансляции">факторами инициации</a>, белками, отвечающими за присоединение к мРНК рибосомы, полиадениновый хвост связывается со специальным поли(А)-связывающим белком PABP1.
</p>

<p>Сплайсинг — процесс, в котором из пре-мРНК удаляются участки, не кодирующие белок, называемые <a href="/wiki/%D0%98%D0%BD%D1%82%D1%80%D0%BE%D0%BD" title="Интрон">интронами</a>; последовательности, которые остаются, несут информацию о структуре белка и называются <a href="/wiki/%D0%AD%D0%BA%D0%B7%D0%BE%D0%BD" title="Экзон">экзонами</a>. Иногда продукты сплайсинга пре-мРНК могут быть соединены разными способами, позволяя одному гену кодировать несколько белков. Этот процесс называется <a href="/wiki/%D0%90%D0%BB%D1%8C%D1%82%D0%B5%D1%80%D0%BD%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D1%8B%D0%B9_%D1%81%D0%BF%D0%BB%D0%B0%D0%B9%D1%81%D0%B8%D0%BD%D0%B3" title="Альтернативный сплайсинг">альтернативным сплайсингом</a>. Сплайсинг обычно производится РНК-белковым комплексом, который называется <a href="/wiki/%D0%A1%D0%BF%D0%BB%D0%B0%D0%B9%D1%81%D0%BE%D1%81%D0%BE%D0%BC%D0%B0" title="Сплайсосома">сплайсосома</a>, но некоторые молекулы мРНК также могут катализировать сплайсинг без участия белков (см. <a class="mw-redirect" href="/wiki/%D0%A0%D0%B8%D0%B1%D0%BE%D0%B7%D0%B8%D0%BC" title="Рибозим">рибозимы</a>)<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup>.
</p>
<h4><span id=".D0.A2.D1.80.D0.B0.D0.BD.D1.81.D0.BF.D0.BE.D1.80.D1.82"></span><span class="mw-headline" id="Транспорт">Транспорт</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Транспорт»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;action=edit&amp;section=6" title="Редактировать код раздела «Транспорт»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h4>
<p>Другое различие между эукариотами и прокариотами — транспорт мРНК. Из-за того, что эукариотические транскрипция и трансляция пространственно разделены, эукариотические мРНК должны быть выведены из <a href="/wiki/%D0%9A%D0%BB%D0%B5%D1%82%D0%BE%D1%87%D0%BD%D0%BE%D0%B5_%D1%8F%D0%B4%D1%80%D0%BE" title="Клеточное ядро">ядра</a> в <a href="/wiki/%D0%A6%D0%B8%D1%82%D0%BE%D0%BF%D0%BB%D0%B0%D0%B7%D0%BC%D0%B0" title="Цитоплазма">цитоплазму</a><sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup>. Зрелые мРНК распознаются по наличию модификаций и покидают ядро через <a href="/wiki/%D0%AF%D0%B4%D0%B5%D1%80%D0%BD%D1%8B%D0%B5_%D0%BF%D0%BE%D1%80%D1%8B" title="Ядерные поры">ядерные поры</a>, в цитоплазме мРНК образует <a href="/wiki/%D0%9D%D1%83%D0%BA%D0%BB%D0%B5%D0%BE%D0%BF%D1%80%D0%BE%D1%82%D0%B5%D0%B8%D0%B4%D1%8B" title="Нуклеопротеиды">нуклеопротеидные комплексы</a> — информосомы, в составе которых транспортируется к <a class="mw-redirect" href="/wiki/%D0%A0%D0%B8%D0%B1%D0%BE%D1%81%D0%BE%D0%BC%D1%8B" title="Рибосомы">рибосомам</a>. Многие мРНК содержат сигналы, которые определяют их локализацию<sup class="reference" id="cite_ref-Ainger1997_17-0"><a href="#cite_note-Ainger1997-17">[17]</a></sup>. В <a href="/wiki/%D0%9D%D0%B5%D0%B9%D1%80%D0%BE%D0%BD" title="Нейрон">нейронах</a> мРНК должна транспортироваться из тела нейронов в <a href="/wiki/%D0%94%D0%B5%D0%BD%D0%B4%D1%80%D0%B8%D1%82" title="Дендрит">дендриты</a>, где трансляция происходит в ответ на внешние раздражители<sup class="reference" id="cite_ref-Job1912_18-0"><a href="#cite_note-Job1912-18">[18]</a></sup>.
</p><p>Экспорт мРНК осуществляется при участии комплекса транспортных факторов Mex67—Mtr2 (у дрожжей) или TAP—p15 (у многоклеточных)<sup class="reference" id="cite_ref-Kohler_19-0"><a href="#cite_note-Kohler-19">[19]</a></sup>. Однако этот комплекс связывает мРНК не напрямую, а через адаптерный белок Yra1 (у <a href="/wiki/%D0%94%D1%80%D0%BE%D0%B6%D0%B6%D0%B8" title="Дрожжи">дрожжей</a>) или ALY/REF (у многоклеточных), который является одной из субъединиц белкового комплекса TREX. В свою очередь, TREX привлекается в комплекс с мРНК за счёт прямого взаимодействия ALY/REF с CBC80 субъединицей <a href="/wiki/%D0%9A%D1%8D%D0%BF" title="Кэп">кэп</a>-связывающего комплекса<sup class="reference" id="cite_ref-Cheng_20-0"><a href="#cite_note-Cheng-20">[20]</a></sup>. Такой механизм обеспечивает присоединение транспортного комплекса близко к 5'-концу мРНК и соответствующую направленность её транспорта, 5'-концом в сторону цитоплазмы.
</p>
<h3><span id=".D0.9C.D0.B5.D1.82.D0.B8.D0.BB.D0.B8.D1.80.D0.BE.D0.B2.D0.B0.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Метилирование">Метилирование</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Метилирование»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;action=edit&amp;section=7" title="Редактировать код раздела «Метилирование»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>мРНК <a href="/wiki/%D0%AD%D1%83%D0%BA%D0%B0%D1%80%D0%B8%D0%BE%D1%82%D1%8B" title="Эукариоты">эукариот</a> подвергаются посттранскрипционному <a href="/wiki/%D0%9C%D0%B5%D1%82%D0%B8%D0%BB%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5" title="Метилирование">метилированию</a>. Наиболее распространённой модификацией является метилирование остатков <a href="/wiki/%D0%90%D0%B4%D0%B5%D0%BD%D0%BE%D0%B7%D0%B8%D0%BD" title="Аденозин">аденозина</a> по положению N<sup>6</sup> с образованием N<sup>6</sup>-метиладенозина (m<sup>6</sup>A). Этот процесс катализируют ферменты N<sup>6</sup>-аденозинметилтрансферазы, которые распознают остатки аденозина в консенсусных последовательностях GAC (70 % случаев) и AAC (30 % случаев). Соответствующие деметилазы катализируют обратный процесс деметилирования. Учитывая обратимость и динамичность процесса метилирования мРНК, а также повышенную концентрацию m<sup>6</sup>A в длинных экзонах и вокруг <a class="mw-redirect" href="/wiki/%D0%A2%D0%B5%D1%80%D0%BC%D0%B8%D0%BD%D0%B0%D1%82%D0%BE%D1%80%D0%BD%D1%8B%D0%B9_%D0%BA%D0%BE%D0%B4%D0%BE%D0%BD" title="Терминаторный кодон">стоп-кодонов</a>, предполагают, что метилирование мРНК выполняет регуляторную функцию<sup class="reference" id="cite_ref-Wang2014_21-0"><a href="#cite_note-Wang2014-21">[21]</a></sup>.
</p>
<h3><span id=".D0.A2.D1.80.D0.B0.D0.BD.D1.81.D0.BB.D1.8F.D1.86.D0.B8.D1.8F"></span><span class="mw-headline" id="Трансляция">Трансляция</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;veaction=edit&amp;section=8" title="Редактировать раздел «Трансляция»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;action=edit&amp;section=8" title="Редактировать код раздела «Трансляция»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote navigation-not-searchable">Основная статья: <b><a href="/wiki/%D0%A2%D1%80%D0%B0%D0%BD%D1%81%D0%BB%D1%8F%D1%86%D0%B8%D1%8F_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Трансляция (биология)">Трансляция (биология)</a></b></div>
<p>Поскольку прокариотическая мРНК не нуждается в обработке и транспортировке, <b>трансляция</b> <a href="/wiki/%D0%A0%D0%B8%D0%B1%D0%BE%D1%81%D0%BE%D0%BC%D0%B0" title="Рибосома">рибосомой</a> может начаться немедленно после транскрипции. Следовательно, можно сказать, что трансляция у прокариот <i>совмещена</i> с транскрипцией и происходит <i>ко-транскрипционно</i>.
</p><p>Эукариотическая мРНК должна быть обработана и доставлена из ядра в цитоплазму, и только тогда может быть транслирована рибосомой. Трансляция может происходить как на <a href="/wiki/%D0%A0%D0%B8%D0%B1%D0%BE%D1%81%D0%BE%D0%BC%D0%B0" title="Рибосома">рибосомах</a>, находящихся в цитоплазме в свободном виде, так и на рибосомах, ассоциированных со стенками <a href="/wiki/%D0%AD%D0%BD%D0%B4%D0%BE%D0%BF%D0%BB%D0%B0%D0%B7%D0%BC%D0%B0%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%80%D0%B5%D1%82%D0%B8%D0%BA%D1%83%D0%BB%D1%83%D0%BC" title="Эндоплазматический ретикулум">эндоплазматического ретикулума</a>. Таким образом, у эукариот трансляция <i>не</i> совмещена напрямую с транскрипцией.
</p>
<h3><span id=".D0.A0.D0.B5.D0.B3.D1.83.D0.BB.D1.8F.D1.86.D0.B8.D1.8F_.D1.82.D1.80.D0.B0.D0.BD.D1.81.D0.BB.D1.8F.D1.86.D0.B8.D0.B8"></span><span class="mw-headline" id="Регуляция_трансляции">Регуляция трансляции</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;veaction=edit&amp;section=9" title="Редактировать раздел «Регуляция трансляции»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;action=edit&amp;section=9" title="Редактировать код раздела «Регуляция трансляции»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Так как у прокариот транскрипция совмещена с трансляцией, прокариотическая клетка может быстро реагировать на изменения в окружающей среде путём синтеза новых белков, то есть регуляция происходит, в основном, на уровне <a href="/wiki/%D0%A2%D1%80%D0%B0%D0%BD%D1%81%D0%BA%D1%80%D0%B8%D0%BF%D1%86%D0%B8%D1%8F_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Транскрипция (биология)">транскрипции</a>. У эукариот из-за необходимости процессинга и транспорта мРНК ответ на внешние стимулы занимает больше времени. Поэтому их синтез белка интенсивно регулируется на посттранскрипционном уровне. Не всякая зрелая мРНК транслируется, поскольку в клетке существуют механизмы регуляции экспрессии белков на посттранскрипционном уровне, например, <a href="/wiki/%D0%A0%D0%9D%D0%9A-%D0%B8%D0%BD%D1%82%D0%B5%D1%80%D1%84%D0%B5%D1%80%D0%B5%D0%BD%D1%86%D0%B8%D1%8F" title="РНК-интерференция">РНК-интерференция</a>.
</p><p>Некоторые мРНК в действительности содержат два тандемных <a class="mw-redirect" href="/wiki/%D0%A2%D0%B5%D1%80%D0%BC%D0%B8%D0%BD%D0%B0%D1%82%D0%BE%D1%80%D0%BD%D1%8B%D0%B9_%D0%BA%D0%BE%D0%B4%D0%BE%D0%BD" title="Терминаторный кодон">терминаторных кодона</a> (стоп-кодона) — часто это кодоны различного типа на конце кодирующей последовательности<sup class="reference" id="cite_ref-Ial_22-0"><a href="#cite_note-Ial-22">[22]</a></sup>.
</p>

<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:MRNA_structure.svg"><img class="mw-file-element" data-file-height="150" data-file-width="1300" decoding="async" height="69" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/ba/MRNA_structure.svg/langru-600px-MRNA_structure.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/ba/MRNA_structure.svg/langru-900px-MRNA_structure.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/ba/MRNA_structure.svg/langru-1200px-MRNA_structure.svg.png 2x" width="600"/></a><figcaption>Схема строения зрелой эукариотической мРНК</figcaption></figure>
<p>Зрелая мРНК состоит из нескольких участков, различающихся по функциям: «5'-кэп», 5'-нетранслируемая область, кодирующая (транслируемая) область, 3'-нетранслируемая область и 3'-полиадениновый «хвост».
</p>
<h3><span id="5.27-.D0.9A.D1.8D.D0.BF"></span><span class="mw-headline" id="5'-Кэп">5'-Кэп</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;veaction=edit&amp;section=11" title="Редактировать раздел «5'-Кэп»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;action=edit&amp;section=11" title="Редактировать код раздела «5'-Кэп»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>

<p>5'-кэп (от <a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">cap</span> — шапочка) — модифицированный <a class="mw-redirect" href="/wiki/%D0%93%D1%83%D0%B0%D0%BD%D0%BE%D0%B7%D0%B8%D0%BD" title="Гуанозин">гуанозиновый</a> <a class="mw-redirect" href="/wiki/%D0%9D%D1%83%D0%BA%D0%BB%D0%B5%D0%BE%D1%82%D0%B8%D0%B4" title="Нуклеотид">нуклеотид</a>, который добавляется на 5'- (<i>передний</i>) конец незрелой мРНК. Эта модификация очень важна для узнавания мРНК при <a href="/wiki/%D0%A2%D1%80%D0%B0%D0%BD%D1%81%D0%BB%D1%8F%D1%86%D0%B8%D1%8F_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Трансляция (биология)">инициации трансляции</a>, а также для защиты от 5'-нуклеаз — ферментов, разрушающих цепи нуклеиновых кислот с незащищённым 5'-концом.
</p>
<h3><span id=".D0.9A.D0.BE.D0.B4.D0.B8.D1.80.D1.83.D1.8E.D1.89.D0.B8.D0.B5_.D0.BE.D0.B1.D0.BB.D0.B0.D1.81.D1.82.D0.B8"></span><span class="mw-headline" id="Кодирующие_области">Кодирующие области</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;veaction=edit&amp;section=12" title="Редактировать раздел «Кодирующие области»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;action=edit&amp;section=12" title="Редактировать код раздела «Кодирующие области»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Кодирующие области состоят из <a href="/wiki/%D0%9A%D0%BE%D0%B4%D0%BE%D0%BD" title="Кодон">кодонов</a> — следующих непосредственно друг за другом последовательностей из трёх нуклеотидов, каждая из которых соответствует в <a href="/wiki/%D0%93%D0%B5%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%BE%D0%B4" title="Генетический код">генетическом коде</a> определённой аминокислоте или началу и концу синтеза белка. Кодирующие области начинаются со <a href="/wiki/%D0%A1%D1%82%D0%B0%D1%80%D1%82-%D0%BA%D0%BE%D0%B4%D0%BE%D0%BD" title="Старт-кодон">старт-кодона</a> и заканчиваются одним из трёх стоп-кодонов. Считывание последовательности кодонов и сборка на её основе последовательности аминокислот синтезируемой молекулы белка осуществляется <a href="/wiki/%D0%A0%D0%B8%D0%B1%D0%BE%D1%81%D0%BE%D0%BC%D0%B0" title="Рибосома">рибосомами</a> при участии <a href="/wiki/%D0%A2%D0%A0%D0%9D%D0%9A" title="ТРНК">транспортных РНК</a> в процессе <a href="/wiki/%D0%A2%D1%80%D0%B0%D0%BD%D1%81%D0%BB%D1%8F%D1%86%D0%B8%D1%8F_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Трансляция (биология)">трансляции</a>. В дополнение к кодированию белков, части кодирующих областей могут служить управляющими последовательностями. Например, <a class="mw-redirect" href="/wiki/%D0%9C%D0%A0%D0%9D%D0%9A#Вторичная_структура" title="МРНК">вторичная структура РНК</a> в некоторых случаях определяет результат трансляции.
</p>
<h3><span id=".D0.9C.D0.BE.D0.BD.D0.BE.D1.86.D0.B8.D1.81.D1.82.D1.80.D0.BE.D0.BD.D0.BD.D0.B0.D1.8F_.D0.B8_.D0.BF.D0.BE.D0.BB.D0.B8.D1.86.D0.B8.D1.81.D1.82.D1.80.D0.BE.D0.BD.D0.BD.D0.B0.D1.8F_.D0.BC.D0.A0.D0.9D.D0.9A"></span><span class="mw-headline" id="Моноцистронная_и_полицистронная_мРНК">Моноцистронная и полицистронная мРНК</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;veaction=edit&amp;section=13" title="Редактировать раздел «Моноцистронная и полицистронная мРНК»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;action=edit&amp;section=13" title="Редактировать код раздела «Моноцистронная и полицистронная мРНК»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>мРНК называют моноцистронной, если она содержит информацию, необходимую для <a href="/wiki/%D0%A2%D1%80%D0%B0%D0%BD%D1%81%D0%BB%D1%8F%D1%86%D0%B8%D1%8F_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Трансляция (биология)">трансляции</a> только одного <a class="mw-redirect" href="/wiki/%D0%91%D0%B5%D0%BB%D0%BE%D0%BA" title="Белок">белка</a> (один <a href="/wiki/%D0%A6%D0%B8%D1%81%D1%82%D1%80%D0%BE%D0%BD" title="Цистрон">цистрон</a>). Полицистронная мРНК кодирует несколько белков. Гены (цистроны) в такой мРНК разделены интергенными, некодирующими последовательностями. Полицистронные мРНК характерны для <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%BA%D0%B0%D1%80%D0%B8%D0%BE%D1%82%D1%8B" title="Прокариоты">прокариот</a> и <a class="mw-redirect" href="/wiki/%D0%92%D0%B8%D1%80%D1%83%D1%81" title="Вирус">вирусов</a>, у <a href="/wiki/%D0%AD%D1%83%D0%BA%D0%B0%D1%80%D0%B8%D0%BE%D1%82%D1%8B" title="Эукариоты">эукариот</a> большая часть мРНК является моноцистронной<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup><sup class="reference" id="cite_ref-Kozak_1983_24-0"><a href="#cite_note-Kozak_1983-24">[24]</a></sup><sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup>.
</p>
<h3><span id=".D0.9D.D0.B5.D1.82.D1.80.D0.B0.D0.BD.D1.81.D0.BB.D0.B8.D1.80.D1.83.D0.B5.D0.BC.D1.8B.D0.B5_.D0.BE.D0.B1.D0.BB.D0.B0.D1.81.D1.82.D0.B8"></span><span class="mw-headline" id="Нетранслируемые_области">Нетранслируемые области</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;veaction=edit&amp;section=14" title="Редактировать раздел «Нетранслируемые области»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;action=edit&amp;section=14" title="Редактировать код раздела «Нетранслируемые области»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote navigation-not-searchable">Основная статья: <b><a href="/wiki/%D0%9D%D0%B5%D1%82%D1%80%D0%B0%D0%BD%D1%81%D0%BB%D0%B8%D1%80%D1%83%D0%B5%D0%BC%D1%8B%D0%B5_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D0%B8" title="Нетранслируемые области">Нетранслируемые области</a></b></div>
<p><a href="/wiki/%D0%9D%D0%B5%D1%82%D1%80%D0%B0%D0%BD%D1%81%D0%BB%D0%B8%D1%80%D1%83%D0%B5%D0%BC%D1%8B%D0%B5_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D0%B8" title="Нетранслируемые области">Нетранслируемые области</a> — участки РНК, расположенные до <a href="/wiki/%D0%A1%D1%82%D0%B0%D1%80%D1%82-%D0%BA%D0%BE%D0%B4%D0%BE%D0%BD" title="Старт-кодон">старт-кодона</a> и после стоп-кодона, которые не кодируют белок. Они называются 5'-нетранслируемая область и 3'-нетранслируемая область, соответственно. Эти области транскрибируются в составе того же самого транскрипта, что и кодирующий участок. Нетранслируемые области имеют несколько функций в жизненном цикле мРНК, включая регуляцию стабильности мРНК, локализации мРНК и эффективности трансляции. Стабильность мРНК может контролироваться 5'- и/или 3'-областью из-за различной чувствительности к <a class="mw-redirect" href="/wiki/%D0%A4%D0%B5%D1%80%D0%BC%D0%B5%D0%BD%D1%82" title="Фермент">ферментам</a>, которые отвечают за деградацию РНК — <a class="new" href="/w/index.php?title=%D0%A0%D0%9D%D0%9A%D0%B0%D0%B7%D0%B0&amp;action=edit&amp;redlink=1" title="РНКаза (страница отсутствует)">РНКазам</a> и регуляторным белкам, которые убыстряют или замедляют деградацию<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup>.
</p>
<h3><span id="3.27-.D0.BF.D0.BE.D0.BB.D0.B8.D0.B0.D0.B4.D0.B5.D0.BD.D0.B8.D0.BD.D0.BE.D0.B2.D1.8B.D0.B9_.D1.85.D0.B2.D0.BE.D1.81.D1.82"></span><span class="mw-headline" id="3'-полиадениновый_хвост">3'-полиадениновый хвост</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;veaction=edit&amp;section=15" title="Редактировать раздел «3'-полиадениновый хвост»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;action=edit&amp;section=15" title="Редактировать код раздела «3'-полиадениновый хвост»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Длинная (часто несколько сотен нуклеотидов) последовательность адениновых оснований, которая присутствует на 3'-«хвосте» мРНК <a class="mw-redirect" href="/wiki/%D0%AD%D1%83%D0%BA%D0%B0%D1%80%D0%B8%D0%BE%D1%82" title="Эукариот">эукариот</a>, синтезируется ферментом полиаденилатполимеразой. У высших эукариот поли(А)-хвост добавляется к транскрибированной РНК, которая содержит специфическую последовательность, AAUAAA. Важность этой последовательности можно увидеть на примере мутации в гене человеческого 2-<a class="mw-redirect" href="/wiki/%D0%93%D0%BB%D0%BE%D0%B1%D0%B8%D0%BD" title="Глобин">глобина</a>, которая изменяет AAUAAA на AAUAAG, что приводит к недостаточному количеству глобина в организме<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup>.
</p>
<h2><span id=".D0.92.D1.82.D0.BE.D1.80.D0.B8.D1.87.D0.BD.D0.B0.D1.8F_.D1.81.D1.82.D1.80.D1.83.D0.BA.D1.82.D1.83.D1.80.D0.B0"></span><span class="mw-headline" id="Вторичная_структура">Вторичная структура</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;veaction=edit&amp;section=16" title="Редактировать раздел «Вторичная структура»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;action=edit&amp;section=16" title="Редактировать код раздела «Вторичная структура»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-default-size mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Stem-loop.svg"><img class="mw-file-element" data-file-height="229" data-file-width="336" decoding="async" height="150" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/3f/Stem-loop.svg/220px-Stem-loop.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/3f/Stem-loop.svg/330px-Stem-loop.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/3/3f/Stem-loop.svg/440px-Stem-loop.svg.png 2x" width="220"/></a><figcaption>«Стебель-петля» — элемент вторичной структуры мРНК, схематично</figcaption></figure>
<figure class="mw-default-size mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Pseudoknot.svg"><img class="mw-file-element" data-file-height="262" data-file-width="499" decoding="async" height="116" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/9f/Pseudoknot.svg/220px-Pseudoknot.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/9f/Pseudoknot.svg/330px-Pseudoknot.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/9/9f/Pseudoknot.svg/440px-Pseudoknot.svg.png 2x" width="220"/></a><figcaption>«Псевдоузел» — элемент вторичной структуры мРНК, схематично</figcaption></figure>
<p>Кроме первичной структуры (последовательности нуклеотидов), мРНК обладает вторичной структурой. В отличие от ДНК — вторичная структура которой основана на межмолекулярных взаимодействиях (двойная спираль ДНК образована двумя линейными молекулами, соединенными друг с другом по всей длине водородными связями), — вторичная структура мРНК основана на внутримолекулярных взаимодействиях (линейная молекула «складывается», и водородные связи возникают между разными участками одной и той же молекулы).
</p><p>Примерами вторичной структуры могут служить стебель, петля и псевдоузел.<sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup>
</p><p>Вторичные структуры в мРНК служат для регуляции трансляции. Например, вставка в белки необычных <a href="/wiki/%D0%90%D0%BC%D0%B8%D0%BD%D0%BE%D0%BA%D0%B8%D1%81%D0%BB%D0%BE%D1%82%D1%8B" title="Аминокислоты">аминокислот</a>, <a href="/wiki/%D0%A1%D0%B5%D0%BB%D0%B5%D0%BD%D0%BE%D0%BC%D0%B5%D1%82%D0%B8%D0%BE%D0%BD%D0%B8%D0%BD" title="Селенометионин">селенометионина</a> и <a href="/wiki/%D0%9F%D0%B8%D1%80%D1%80%D0%BE%D0%BB%D0%B8%D0%B7%D0%B8%D0%BD" title="Пирролизин">пирролизина</a>, зависит от стебля-петли, расположенной в 3'-нетранслируемой области. Псевдоузлы служат для программированного изменения <a class="mw-redirect" href="/wiki/%D0%A0%D0%B0%D0%BC%D0%BA%D0%B0_%D1%81%D1%87%D0%B8%D1%82%D1%8B%D0%B2%D0%B0%D0%BD%D0%B8%D1%8F" title="Рамка считывания">рамки считывания</a> генов. Также вторичная структура служит для замедления деградации определённых мРНК<sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[29]</a></sup><sup class="reference" id="cite_ref-30"><a href="#cite_note-30">[30]</a></sup>
</p><p>В <a class="mw-redirect" href="/wiki/%D0%92%D0%B8%D1%80%D1%83%D1%81" title="Вирус">вирусных</a> мРНК сложные вторичные структуры (<a href="/wiki/IRES" title="IRES">IRES</a>) направляют трансляцию, не зависящую от узнавания кэпа и факторов инициации трансляции (см. «<a href="/wiki/%D0%A2%D1%80%D0%B0%D0%BD%D1%81%D0%BB%D1%8F%D1%86%D0%B8%D1%8F_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)#Инициация_трансляции" title="Трансляция (биология)">Инициация трансляции</a>»).
</p>
<h2><span id=".D0.A0.D0.B0.D0.B7.D1.80.D1.83.D1.88.D0.B5.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Разрушение">Разрушение</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;veaction=edit&amp;section=17" title="Редактировать раздел «Разрушение»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;action=edit&amp;section=17" title="Редактировать код раздела «Разрушение»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Различные мРНК имеют различную продолжительность жизни (стабильность). В клетках бактерий молекула мРНК может существовать от нескольких секунд до более чем часа, а в клетках млекопитающих — от нескольких минут до нескольких дней. Чем больше стабильность мРНК, тем больше белка может быть синтезировано с данной молекулы. Ограниченное время жизни мРНК клетки позволяет быстро изменять синтез белка в ответ на изменяющиеся потребности клетки. По прошествии некоторого времени, определяемого её нуклеотидной последовательностью, — в частности, длиной полиаденинового участка на 3'-конце молекулы, — мРНК разрушается на составляющие её нуклеотиды с участием <a href="/wiki/%D0%A0%D0%B8%D0%B1%D0%BE%D0%BD%D1%83%D0%BA%D0%BB%D0%B5%D0%B0%D0%B7%D1%8B" title="Рибонуклеазы">РНКаз</a>. К настоящему времени известно много механизмов деградации мРНК, некоторые из которых описаны ниже.
</p>
<h3><span id=".D0.94.D0.B5.D0.B3.D1.80.D0.B0.D0.B4.D0.B0.D1.86.D0.B8.D1.8F_.D0.BC.D0.A0.D0.9D.D0.9A_.D1.83_.D0.BF.D1.80.D0.BE.D0.BA.D0.B0.D1.80.D0.B8.D0.BE.D1.82"></span><span class="mw-headline" id="Деградация_мРНК_у_прокариот">Деградация мРНК у прокариот</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;veaction=edit&amp;section=18" title="Редактировать раздел «Деградация мРНК у прокариот»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;action=edit&amp;section=18" title="Редактировать код раздела «Деградация мРНК у прокариот»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>У прокариот стабильность мРНК намного меньше, чем у эукариот. Деградация мРНК в клетках прокариот происходит под действием комбинации рибонуклеаз, в том числе эндонуклеаз, 3'-экзонуклеаз и 5'-экзонуклеаз. В некоторых случаях малые молекулы РНК длиной от десятков до сотен нуклеотидов могут стимулировать деградацию мРНК, комплементарно спариваясь с соответствующими последовательностями в мРНК и содействуя рибонуклеазам<sup class="reference" id="cite_ref-pmid17574901_31-0"><a href="#cite_note-pmid17574901-31">[31]</a></sup><sup class="reference" id="cite_ref-pmid18981732_32-0"><a href="#cite_note-pmid18981732-32">[32]</a></sup> . В 2008 году было показано, что бактерии имеют нечто вроде кэпа — трифосфат на 5'-конце<sup class="reference" id="cite_ref-Deana2008_33-0"><a href="#cite_note-Deana2008-33">[33]</a></sup>. Удаление двух фосфатов оставляет монофосфат на 5'-конце, в результате чего мРНК расщепляется эндонуклеазой <a href="/wiki/%D0%A0%D0%B8%D0%B1%D0%BE%D0%BD%D1%83%D0%BA%D0%BB%D0%B5%D0%B0%D0%B7%D1%8B" title="Рибонуклеазы">РНКаза E</a>.
</p>
<h3><span id=".D0.A3_.D1.8D.D1.83.D0.BA.D0.B0.D1.80.D0.B8.D0.BE.D1.82"></span><span class="mw-headline" id="У_эукариот">У эукариот</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;veaction=edit&amp;section=19" title="Редактировать раздел «У эукариот»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;action=edit&amp;section=19" title="Редактировать код раздела «У эукариот»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Как правило, разрушение начинается с удаления кэпа на 5'-конце, полиаденинового хвоста на 3'-конце, и затем <a href="/wiki/%D0%9D%D1%83%D0%BA%D0%BB%D0%B5%D0%B0%D0%B7%D0%B0" title="Нуклеаза">нуклеазы</a> одновременно разрушают мРНК в направлениях 5' → 3' и 3' → 5'. мРНК, в которой сигнал завершения синтеза белка, стоп-кодон, в результате ошибки транскрипции находится в середине кодирующей последовательности, подвержена особой быстрой форме деградации, <a href="/wiki/%D0%9D%D0%BE%D0%BD%D1%81%D0%B5%D0%BD%D1%81-%D0%BE%D0%BF%D0%BE%D1%81%D1%80%D0%B5%D0%B4%D0%BE%D0%B2%D0%B0%D0%BD%D0%BD%D1%8B%D0%B9_%D1%80%D0%B0%D1%81%D0%BF%D0%B0%D0%B4" title="Нонсенс-опосредованный распад">НМД</a>.
</p>
<h2><span id=".D0.9C.D0.B5.D1.82.D0.BE.D0.B4.D1.8B_.D0.BE.D0.BF.D1.80.D0.B5.D0.B4.D0.B5.D0.BB.D0.B5.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="Методы_определения">Методы определения</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;veaction=edit&amp;section=20" title="Редактировать раздел «Методы определения»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%A0%D0%9D%D0%9A&amp;action=edit&amp;section=20" title="Редактировать код раздела «Методы определения»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>В последнее время разработаны очень чувствительные методы, позволяющие проанализировать <a href="/wiki/%D0%A2%D1%80%D0%B0%D0%BD%D1%81%D0%BA%D1%80%D0%B8%D0%BF%D1%82_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Транскрипт (биология)">транскриптом</a> из образцов размером в 50—100 клеток<sup class="reference" id="cite_ref-34"><a href="#cite_note-34">[34]</a></sup><sup class="reference" id="cite_ref-35"><a href="#cite_note-35">[35]</a></sup><sup class="reference" id="cite_ref-36"><a href="#cite_note-36">[36]</a></sup>.
</p>
