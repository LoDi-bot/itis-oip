<h1 class="firstHeading mw-first-heading" id="firstHeading"><span class="mw-page-title-main">Кинематограф</span></h1>
<div class="vector-body" id="bodyContent">
<div class="noprint" id="siteSub">Материал из Википедии — свободной энциклопедии</div>
<div id="contentSub"><div id="mw-content-subtitle"></div></div>

<div class="mw-body-content" id="mw-content-text"><div class="mw-content-ltr mw-parser-output" dir="ltr" lang="ru"><div class="hatnote navigation-not-searchable noprint dabhide">У этого термина существуют и другие значения, см. <a class="mw-disambig" href="/wiki/%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84_(%D0%B7%D0%BD%D0%B0%D1%87%D0%B5%D0%BD%D0%B8%D1%8F)" title="Кинематограф (значения)">Кинематограф (значения)</a>.</div>
<div class="hatnote dabhide">Запросы «Кино» и «Синематограф» перенаправляются сюда; см. также другие значения терминов <a class="mw-disambig" href="/wiki/%D0%9A%D0%B8%D0%BD%D0%BE_(%D0%B7%D0%BD%D0%B0%D1%87%D0%B5%D0%BD%D0%B8%D1%8F)" title="Кино (значения)">Кино</a> и <a class="mw-disambig" href="/wiki/%D0%A1%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84_(%D0%B7%D0%BD%D0%B0%D1%87%D0%B5%D0%BD%D0%B8%D1%8F)" title="Синематограф (значения)">Синематограф</a>.</div>
<p><b>Кинемато́граф</b> (от <a href="/wiki/%D0%93%D1%80%D0%B5%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Греческий язык">греч.</a> <span lang="el"><span style="font-family: palatino linotype, new athena unicode, athena, gentium, code2000, serif; font-size: 105%;">κινημα</span></span>, род. п. <span lang="el"><span style="font-family: palatino linotype, new athena unicode, athena, gentium, code2000, serif; font-size: 105%;">κινηματος</span></span> — движение и <a href="/wiki/%D0%93%D1%80%D0%B5%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Греческий язык">греч.</a> <span lang="el"><span style="font-family: palatino linotype, new athena unicode, athena, gentium, code2000, serif; font-size: 105%;">γραφω</span></span> — писать, рисовать; то есть «записывающий движение») — отрасль человеческой деятельности, заключающаяся в создании движущихся изображений. Иногда также упоминается как <b>синемато́граф</b> (от <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Французский язык">фр.</a> <span lang="fr" style="font-style:italic;">cinématographe</span>, устар.) и <b>кинематогра́фия</b>. Название заимствовано у <a href="/wiki/%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84_(%D0%B0%D0%BF%D0%BF%D0%B0%D1%80%D0%B0%D1%82)" title="Кинематограф (аппарат)">одноимённого аппарата</a>, изобретённого <a href="/wiki/%D0%91%D1%80%D0%B0%D1%82%D1%8C%D1%8F_%D0%9B%D1%8E%D0%BC%D1%8C%D0%B5%D1%80" title="Братья Люмьер">братьями Люмьер</a> и положившего начало коммерческому использованию технологии. Кинематограф был изобретён в конце <a href="/wiki/XIX_%D0%B2%D0%B5%D0%BA" title="XIX век">XIX века</a> и стал крайне популярен в <a href="/wiki/XX_%D0%B2%D0%B5%D0%BA" title="XX век">XX веке</a>.
</p><p>В понятие кинематографа входят:
</p>
<ul><li><a href="/wiki/%D0%9A%D0%B8%D0%BD%D0%BE%D0%B8%D1%81%D0%BA%D1%83%D1%81%D1%81%D1%82%D0%B2%D0%BE" title="Киноискусство">киноискусство</a> — вид изобразительного <a href="/wiki/%D0%98%D1%81%D0%BA%D1%83%D1%81%D1%81%D1%82%D0%B2%D0%BE" title="Искусство">искусства</a>, произведения которого создаются при помощи движущихся изображений;</li>
<li><a href="/wiki/%D0%9A%D0%B8%D0%BD%D0%BE%D0%B8%D0%BD%D0%B4%D1%83%D1%81%D1%82%D1%80%D0%B8%D1%8F" title="Киноиндустрия">киноиндустрия</a> (кинопромышленность) — <a href="/wiki/%D0%9E%D1%82%D1%80%D0%B0%D1%81%D0%BB%D1%8C_%D1%8D%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D0%BA%D0%B8" title="Отрасль экономики">отрасль экономики</a>, производящая <a class="mw-redirect" href="/wiki/%D0%A4%D0%B8%D0%BB%D1%8C%D0%BC_(%D0%BA%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D1%8F)" title="Фильм (кинематография)">кинофильмы</a>, спецэффекты для кинофильмов<a class="mw-redirect" href="/wiki/%D0%9C%D1%83%D0%BB%D1%8C%D1%82%D0%B8%D0%BF%D0%BB%D0%B8%D0%BA%D0%B0%D1%86%D0%B8%D1%8F" title="Мультипликация">, мультипликацию</a>, и демонстрирующая эти произведения для зрителей.</li></ul>
<p>Произведения киноискусства создаются при помощи <a href="/wiki/%D0%9A%D0%B8%D0%BD%D0%BE%D1%82%D0%B5%D1%85%D0%BD%D0%B8%D0%BA%D0%B0" title="Кинотехника">кинотехники</a>. Наука, занимающаяся изучением — <a href="/wiki/%D0%9A%D0%B8%D0%BD%D0%BE%D0%B2%D0%B5%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5" title="Киноведение">кинове́дение</a>. Сами же кинофильмы могут сниматься в различных <a href="/wiki/%D0%96%D0%B0%D0%BD%D1%80%D1%8B_%D0%B8%D0%B3%D1%80%D0%BE%D0%B2%D0%BE%D0%B3%D0%BE_%D0%BA%D0%B8%D0%BD%D0%BE" title="Жанры игрового кино">жанрах, как игрового</a>, так и <a href="/wiki/%D0%94%D0%BE%D0%BA%D1%83%D0%BC%D0%B5%D0%BD%D1%82%D0%B0%D0%BB%D1%8C%D0%BD%D0%BE%D0%B5_%D0%BA%D0%B8%D0%BD%D0%BE" title="Документальное кино">документального кино</a>.
</p><p>Кинематограф занимает значительную часть современной <a href="/wiki/%D0%9A%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D0%B0" title="Культура">культуры</a> многих стран. Во многих странах киноиндустрия является значимой отраслью экономики. Производство кинофильмов сосредоточено на <a href="/wiki/%D0%9A%D0%B8%D0%BD%D0%BE%D1%81%D1%82%D1%83%D0%B4%D0%B8%D1%8F" title="Киностудия">киностудиях</a>. Фильмы демонстрируются в <a href="/wiki/%D0%9A%D0%B8%D0%BD%D0%BE%D1%82%D0%B5%D0%B0%D1%82%D1%80" title="Кинотеатр">кинотеатрах</a>, по <a href="/wiki/%D0%A2%D0%B5%D0%BB%D0%B5%D0%B2%D0%B8%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5" title="Телевидение">телевидению</a>, распространяются «на <a href="/wiki/%D0%92%D0%B8%D0%B4%D0%B5%D0%BE" title="Видео">видео</a>» в форме <a href="/wiki/%D0%92%D0%B8%D0%B4%D0%B5%D0%BE%D0%BA%D0%B0%D1%81%D1%81%D0%B5%D1%82%D0%B0" title="Видеокассета">видеокассет</a> и <a href="/wiki/%D0%92%D0%B8%D0%B4%D0%B5%D0%BE%D0%B4%D0%B8%D1%81%D0%BA" title="Видеодиск">видеодисков</a>, а с появлением скоростного интернета стало доступным скачивание кинофильмов в форме видеофайлов на специализированных сайтах или посредством <a class="mw-redirect" href="/wiki/%D0%9F%D0%B8%D1%80%D0%B8%D0%BD%D0%B3%D0%BE%D0%B2%D0%B0%D1%8F_%D1%81%D0%B5%D1%82%D1%8C" title="Пиринговая сеть">пиринговых сетей</a>, а также <a href="/wiki/%D0%92%D0%B8%D0%B4%D0%B5%D0%BE%D1%85%D0%BE%D1%81%D1%82%D0%B8%D0%BD%D0%B3" title="Видеохостинг">просмотр онлайн</a> (что может <a href="/wiki/%D0%90%D0%B2%D1%82%D0%BE%D1%80%D1%81%D0%BA%D0%BE%D0%B5_%D0%BF%D1%80%D0%B0%D0%B2%D0%BE" title="Авторское право">нарушать права</a> правообладателей кинофильма).
</p>

<h2><span id=".D0.98.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.8F"></span><span class="mw-headline" id="История">История</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84&amp;veaction=edit&amp;section=1" title="Редактировать раздел «История»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84&amp;action=edit&amp;section=1" title="Редактировать код раздела «История»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="hatnote navigation-not-searchable">Основная статья: <b><a href="/wiki/%D0%98%D1%81%D1%82%D0%BE%D1%80%D0%B8%D1%8F_%D0%BA%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84%D0%B0" title="История кинематографа">История кинематографа</a></b></div>
<p>Кинематограф появился благодаря нескольким техническим изобретениям, практически совпавшим по времени: <a href="/wiki/%D0%96%D0%B5%D0%BB%D0%B0%D1%82%D0%B8%D0%BD%D0%BE%D1%81%D0%B5%D1%80%D0%B5%D0%B1%D1%80%D1%8F%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D1%82%D0%BE%D0%BF%D1%80%D0%BE%D1%86%D0%B5%D1%81%D1%81" title="Желатиносеребряный фотопроцесс">сухому броможелатиновому фотопроцессу</a> с высокой <a href="/wiki/%D0%A1%D0%B2%D0%B5%D1%82%D0%BE%D1%87%D1%83%D0%B2%D1%81%D1%82%D0%B2%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D0%BE%D1%81%D1%82%D1%8C_%D1%84%D0%BE%D1%82%D0%BE%D0%BC%D0%B0%D1%82%D0%B5%D1%80%D0%B8%D0%B0%D0%BB%D0%BE%D0%B2" title="Светочувствительность фотоматериалов">светочувствительностью</a> (1878 год), <a href="/wiki/%D0%9A%D0%B8%D0%BD%D0%BE%D0%BF%D0%BB%D1%91%D0%BD%D0%BA%D0%B0" title="Киноплёнка">киноплёнке</a> на гибкой и прочной основе (1889 год), скоростному аппарату <a href="/wiki/%D0%A5%D1%80%D0%BE%D0%BD%D0%BE%D1%84%D0%BE%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D1%8F" title="Хронофотография">хронофотографической</a> съёмки (1891 год) и проектору с таким же быстродействием (1895 год). Первые съёмочные аппараты были сконструированы в 80-х годах <a href="/wiki/XIX_%D0%B2%D0%B5%D0%BA" title="XIX век">XIX века</a>. К ним относятся: «<a href="/wiki/%D0%A4%D0%BE%D1%82%D0%BE%D1%80%D1%83%D0%B6%D1%8C%D1%91" title="Фоторужьё">фоторужьё</a>» французского физиолога <a href="/wiki/%D0%9C%D0%B0%D1%80%D0%B5,_%D0%AD%D1%82%D1%8C%D0%B5%D0%BD-%D0%96%D1%8E%D0%BB%D1%8C" title="Маре, Этьен-Жюль">Жюля Марэ</a> (<a class="mw-redirect" href="/wiki/1882" title="1882">1882</a>), аппарат французского изобретателя <a href="/wiki/%D0%9B%D0%B5%D0%BF%D1%80%D0%B5%D0%BD%D1%81,_%D0%9B%D1%83%D0%B8" title="Лепренс, Луи">Луи Лепренса</a> (<a class="mw-redirect" href="/wiki/1888" title="1888">1888</a>), аппарат английских изобретателей <a href="/wiki/%D0%A4%D1%80%D0%B8%D0%B7-%D0%93%D1%80%D0%B8%D0%BD,_%D0%A3%D0%B8%D0%BB%D1%8C%D1%8F%D0%BC" title="Фриз-Грин, Уильям">У. Фрис-Грина</a> и М. Эванса (1889), аппарат российского фотографа В. А. Дюбюка (1891), «Фоноскоп» французского физиолога Ж. Демени (1892). Однако большинство из этих устройств были рассчитаны на использование дисковых фотопластинок (фоторужьё Марэ) или светочувствительной фотобумаги, непригодных для массового использования. Гибкая целлулоидная основа для киноплёнки была создана Ганнибалом Гудвином в <a href="/wiki/1887_%D0%B3%D0%BE%D0%B4" title="1887 год">1887 году</a>, который первым применил противоскручивающий контрслой из желатина<sup class="reference" id="cite_ref-_04c10a7cc9058992_1-0"><a href="#cite_note-_04c10a7cc9058992-1">[1]</a></sup>. <a class="mw-redirect" href="/wiki/%D0%94%D0%B6%D0%BE%D1%80%D0%B4%D0%B6_%D0%98%D1%81%D1%82%D0%BC%D0%B5%D0%BD" title="Джордж Истмен">Джордж Истмен</a> в <a href="/wiki/1889_%D0%B3%D0%BE%D0%B4" title="1889 год">1889 году</a> первым наладил массовый выпуск фотоматериалов на гибкой прозрачной подложке из нитроцеллюлозы. Пионерами в создании аппаратов для проекции на экран быстро сменяющихся изображений были: немецкий и российский фотографы <a href="/wiki/%D0%90%D0%BD%D1%88%D1%8E%D1%82%D1%86,_%D0%9E%D1%82%D1%82%D0%BE%D0%BC%D0%B0%D1%80" title="Аншютц, Оттомар">О. Анщюц</a> и В. А. Дюбюк, создавшие соответственно в <a class="mw-redirect" href="/wiki/1891" title="1891">1891</a> и <a href="/wiki/1892_%D0%B3%D0%BE%D0%B4" title="1892 год">1892 годах</a> проекционные аппараты различной конструкции, но с одинаковым названием — «Тахископ», французский изобретатель <a href="/wiki/%D0%A0%D0%B5%D0%B9%D0%BD%D0%BE,_%D0%AD%D0%BC%D0%B8%D0%BB%D1%8C" title="Рейно, Эмиль">Э. Рейно</a>, создавший в <a href="/wiki/1892_%D0%B3%D0%BE%D0%B4" title="1892 год">1892 году</a> проектор под названием «Оптический театр», и российские изобретатели <a href="/wiki/%D0%A2%D0%B8%D0%BC%D1%87%D0%B5%D0%BD%D0%BA%D0%BE,_%D0%98%D0%BE%D1%81%D0%B8%D1%84_%D0%90%D0%BD%D0%B4%D1%80%D0%B5%D0%B5%D0%B2%D0%B8%D1%87" title="Тимченко, Иосиф Андреевич">И. А. Тимченко</a> и <a href="/wiki/%D0%A4%D1%80%D0%B5%D0%B9%D0%B4%D0%B5%D0%BD%D0%B1%D0%B5%D1%80%D0%B3,_%D0%9C%D0%B8%D1%85%D0%B0%D0%B8%D0%BB_%D0%A4%D0%B8%D0%BB%D0%B8%D0%BF%D0%BF%D0%BE%D0%B2%D0%B8%D1%87" title="Фрейденберг, Михаил Филиппович">М. Ф. Фрейденберг</a> (<a class="mw-redirect" href="/wiki/1893" title="1893">1893</a>). В <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%BE%D0%B1%D1%80%D0%B8%D1%82%D0%B0%D0%BD%D0%B8%D1%8F" title="Великобритания">Великобритании</a> свой патент на проектор и кинокамеру оформил <a href="/wiki/%D0%94%D0%BE%D0%BD%D0%B8%D1%81%D1%82%D0%BE%D1%80%D0%BF,_%D0%92%D0%BE%D1%80%D0%B4%D1%81%D0%B2%D0%BE%D1%80%D1%82" title="Донисторп, Вордсворт">Вордсворт Донисторп</a>.
</p>
<figure class="mw-default-size mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Fratelli_Lumiere.jpg"><img class="mw-file-element" data-file-height="480" data-file-width="356" decoding="async" height="297" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/93/Fratelli_Lumiere.jpg/220px-Fratelli_Lumiere.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/93/Fratelli_Lumiere.jpg/330px-Fratelli_Lumiere.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/9/93/Fratelli_Lumiere.jpg 2x" width="220"/></a><figcaption>Братья Люмьер (Огюст слева, Луи справа на фото)</figcaption></figure>
<p>Изобретениями, наиболее приблизившимися к кинематографу по своим техническим характеристикам, являются: «<a href="/wiki/%D0%9A%D0%B8%D0%BD%D0%B5%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84" title="Кинетограф">кинетограф</a>» <a href="/wiki/%D0%AD%D0%B4%D0%B8%D1%81%D0%BE%D0%BD,_%D0%A2%D0%BE%D0%BC%D0%B0%D1%81_%D0%90%D0%BB%D0%B2%D0%B0" title="Эдисон, Томас Алва">Эдисона</a> (1891), аппарат И. А. Тимченко (1893), «хронофотограф» Ж. Демени (1893), проектор американского изобретателя Ж. А. Ле Роя (<a class="mw-redirect" href="/wiki/1894" title="1894">1894</a>), проектор «паноптиком» американского изобретателя У. Латама (1895), «плеограф» польского изобретателя К. Прушинского (1894) и др. А уже в <a class="mw-redirect" href="/wiki/1895" title="1895">1895</a>—<a href="/wiki/1896_%D0%B3%D0%BE%D0%B4" title="1896 год">1896 годах</a> были изобретены аппараты, сочетающие в себе все основные элементы кинематографа: во Франции — «<a class="mw-redirect" href="/wiki/%D0%A1%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84_(%D0%B0%D0%BF%D0%BF%D0%B0%D1%80%D0%B0%D1%82)" title="Синематограф (аппарат)">синематограф</a>» братьев <a class="mw-redirect" href="/wiki/%D0%9B%D1%8E%D0%BC%D1%8C%D0%B5%D1%80,_%D0%9B%D1%83%D0%B8_%D0%96%D0%B0%D0%BD" title="Люмьер, Луи Жан">Л. Люмьер</a> и О. Люмьер (1895) и «хронофотограф» Ж. Демени (1895); в Германии — «биоскоп» <a href="/wiki/%D0%A1%D0%BA%D0%BB%D0%B0%D0%B4%D0%B0%D0%BD%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9,_%D0%9C%D0%B0%D0%BA%D1%81" title="Складановский, Макс">М. Складановского</a> (1895) и кинопроектор О. Местера (1896); в Англии — «аниматограф» Р. У. Пола (1890); в России — «хронофотограф» А. Самарского<sup class="ts-fix-template noprint">[<i><a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%A1%D1%81%D1%8B%D0%BB%D0%BA%D0%B8_%D0%BD%D0%B0_%D0%B8%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8" title="Википедия:Ссылки на источники"><span title="нет источника (14 января 2015)">источник не указан 3328 дней</span></a></i>]</sup> (1896) и «стробограф» И. Акимова (1896), в США — «<a class="mw-redirect" href="/wiki/%D0%92%D0%B8%D1%82%D0%B0%D1%81%D0%BA%D0%BE%D0%BF" title="Витаскоп">витаскоп</a>» Т. Армата (1896).
</p><p>Начало распространения кинематографа в его современном виде было положено съёмкой и публичной демонстрацией первых короткометражных фильмов. <a href="/wiki/22_%D0%BC%D0%B0%D1%80%D1%82%D0%B0" title="22 марта">22 марта</a> <a href="/wiki/1895_%D0%B3%D0%BE%D0%B4_%D0%B2_%D0%BA%D0%B8%D0%BD%D0%BE" title="1895 год в кино">1895 года</a> в Париже братьями Люмьер был впервые продемонстрирован их «синематограф», <a href="/wiki/1_%D0%BD%D0%BE%D1%8F%D0%B1%D1%80%D1%8F" title="1 ноября">1 ноября</a> в том же году в Берлине М. Складановский продемонстрировал свой «биоскоп», а <a href="/wiki/28_%D0%B4%D0%B5%D0%BA%D0%B0%D0%B1%D1%80%D1%8F" title="28 декабря">28 декабря</a> в <a href="/wiki/%D0%9F%D0%B0%D1%80%D0%B8%D0%B6" title="Париж">Париже</a> состоялся первый коммерческий сеанс в салоне «Гран-кафе» на <a href="/wiki/%D0%91%D1%83%D0%BB%D1%8C%D0%B2%D0%B0%D1%80_%D0%9A%D0%B0%D0%BF%D1%83%D1%86%D0%B8%D0%BD%D0%BE%D0%BA" title="Бульвар Капуцинок">бульваре Капуцинок</a><sup class="reference" id="cite_ref-_ec08918b965ed3b6_2-0"><a href="#cite_note-_ec08918b965ed3b6-2">[2]</a></sup>. В течение 1896—<a href="/wiki/1897_%D0%B3%D0%BE%D0%B4" title="1897 год">1897 годов</a> публичные демонстрации короткометражных фильмов были произведены во всех мировых столицах. В России первый показ был организован <abbr title="по юлианскому календарю">4</abbr> <a href="/wiki/16_%D0%BC%D0%B0%D1%8F" title="16 мая">(16) мая</a> <a href="/wiki/1896_%D0%B3%D0%BE%D0%B4" title="1896 год">1896 года</a> в Санкт-Петербурге (в саду «Аквариум»), затем в Москве и на <a href="/wiki/%D0%9D%D0%B8%D0%B6%D0%B5%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D1%8F%D1%80%D0%BC%D0%B0%D1%80%D0%BA%D0%B0" title="Нижегородская ярмарка">Всероссийской ярмарке в Нижнем Новгороде</a>.
Тогда же были сняты первые российские <a href="/wiki/%D0%9B%D1%8E%D0%B1%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%BE%D0%B5_%D0%BA%D0%B8%D0%BD%D0%BE" title="Любительское кино">любительские киносъёмки</a> (<a href="/wiki/%D0%A1%D0%B0%D1%88%D0%B8%D0%BD,_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Сашин, Владимир Александрович">В. Сашин</a>, А. Федецкий, С. Макаров и др.). 3 января <a href="/wiki/1897_%D0%B3%D0%BE%D0%B4_%D0%B2_%D0%BA%D0%B8%D0%BD%D0%BE" title="1897 год в кино">1897 года</a> прошёл первый пробный сеанс «живых картин» системы братьев Люмьер в Киевском дворянском собрании, на углу <a href="/wiki/%D0%9C%D0%B0%D0%B9%D0%B4%D0%B0%D0%BD_%D0%9D%D0%B5%D0%B7%D0%B0%D0%BB%D0%B5%D0%B6%D0%BD%D0%BE%D1%81%D1%82%D0%B8" title="Майдан Незалежности">Думской площади</a> и <a href="/wiki/%D0%9A%D1%80%D0%B5%D1%89%D0%B0%D1%82%D0%B8%D0%BA" title="Крещатик">Крещатика</a>. Через несколько месяцев известный театральный антрепренёр <a href="/wiki/%D0%A1%D0%BE%D0%BB%D0%BE%D0%B2%D1%86%D0%BE%D0%B2,_%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B9_%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B5%D0%B2%D0%B8%D1%87" title="Соловцов, Николай Николаевич">Николай Соловцов</a> приобрёл киноаппарат и начал с ним гастроли по <a href="/wiki/%D0%9A%D0%B8%D0%B5%D0%B2" title="Киев">Киеву</a>, волжским городам и далее по <a href="/wiki/%D0%A1%D0%B8%D0%B1%D0%B8%D1%80%D1%8C" title="Сибирь">Сибири</a>.
</p><p>Первая киносъёмка в Российской империи была сделана фотохудожником А. Федецким в <a href="/wiki/%D0%A5%D0%B0%D1%80%D1%8C%D0%BA%D0%BE%D0%B2" title="Харьков">Харькове</a> (1,5 минуты, «Перенесение иконы Озерянской Божьей матери»). Первым российским документальным фильмом стал «Вид харьковского вокзала в момент отхода поезда с находящимся на платформе начальством» (<a href="/wiki/1896_%D0%B3%D0%BE%D0%B4_%D0%B2_%D0%BA%D0%B8%D0%BD%D0%BE" title="1896 год в кино">1896</a>).
</p><p>

И вдруг что-то щёлкает, всё исчезает, и на экране является поезд железной дороги. Он мчится стрелой прямо на вас — берегитесь! Кажется, что вот-вот он ринется во тьму, в которой вы сидите, и превратит вас в рваный мешок кожи, полный измятого мяса и раздробленных костей, и разрушит, превратит в обломки и в пыль этот зал и это здание, где так много вина, женщин, музыки и порока.</p><div class="ts-Конец_цитаты-source">— <cite><a href="/wiki/%D0%9C%D0%B0%D0%BA%D1%81%D0%B8%D0%BC_%D0%93%D0%BE%D1%80%D1%8C%D0%BA%D0%B8%D0%B9" title="Максим Горький">Максим Горький</a><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></cite></div></blockquote><div class="hatnote navigation-not-searchable">Основная статья: <b><a href="/wiki/%D0%9D%D0%B5%D0%BC%D0%BE%D0%B9_%D0%BA%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84" title="Немой кинематограф">Немой кинематограф</a></b></div>

<p>Первые короткометражные фильмы длиной 50 <a href="/wiki/%D0%A4%D1%83%D1%82" title="Фут">футов</a> (примерно 15 <a href="/wiki/%D0%9C%D0%B5%D1%82%D1%80" title="Метр">метров</a> или 1,5 минуты демонстрации) были по большей части <a class="mw-redirect" href="/wiki/%D0%94%D0%BE%D0%BA%D1%83%D0%BC%D0%B5%D0%BD%D1%82%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D1%84%D0%B8%D0%BB%D1%8C%D0%BC" title="Документальный фильм">документальные</a>, однако уже в комедийной инсценировке братьев Люмьер <a href="/wiki/%D0%9F%D0%BE%D0%BB%D0%B8%D1%82%D1%8B%D0%B9_%D0%BF%D0%BE%D0%BB%D0%B8%D0%B2%D0%B0%D0%BB%D1%8C%D1%89%D0%B8%D0%BA" title="Политый поливальщик">Политый поливальщик</a> отражаются тенденции зарождения игрового кино<sup class="reference" id="cite_ref-_81721d228048b0c0_4-0"><a href="#cite_note-_81721d228048b0c0-4">[4]</a></sup>.
</p><p>Небольшая длина первых фильмов была обусловлена техническим несовершенством киноаппаратуры, тем не менее уже к 1900-м годам длина кинокартин увеличилась до 200—300 метров (15-20 минут демонстрации). Совершенствование съёмочной и проекционной техники способствовало дальнейшему увеличению длины фильмов, качественному и количественному увеличению художественных приёмов съёмки, актёрской игры и режиссуры. А широкое распространение и популярность кинематографа обеспечили его экономическую выгодность, что, однако, не могло не сказаться на художественной ценности снимаемых кинокартин.
В этот период с усложнением и удлинением сюжета фильмов начинают формироваться жанры кинематографа, оформляется их художественное своеобразие, создаётся специфический для каждого жанра набор изобразительных приёмов.
</p><p>Наивысшего своего расцвета <a href="/wiki/%D0%9D%D0%B5%D0%BC%D0%BE%D0%B9_%D0%BA%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84" title="Немой кинематограф">«немое» кино</a> достигает к 20-м годам XX века, когда оно уже вполне оформляется как самостоятельный род искусства, обладающий своими собственными художественными средствами.
</p>
<h3><span id=".D0.9F.D0.BE.D1.8F.D0.B2.D0.BB.D0.B5.D0.BD.D0.B8.D0.B5_.D0.B7.D0.B2.D1.83.D0.BA.D0.B0_.D0.B2_.D0.BA.D0.B8.D0.BD.D0.BE"></span><span class="mw-headline" id="Появление_звука_в_кино">Появление звука в кино</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Появление звука в кино»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84&amp;action=edit&amp;section=2" title="Редактировать код раздела «Появление звука в кино»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote navigation-not-searchable">Основная статья: <b><a href="/wiki/%D0%97%D0%B2%D1%83%D0%BA%D0%BE%D0%B2%D0%BE%D0%B9_%D0%BA%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84" title="Звуковой кинематограф">Звуковой кинематограф</a></b></div>
<p>Ещё до начала XX века <a class="mw-redirect" href="/wiki/%D0%A2%D0%BE%D0%BC%D0%B0%D1%81_%D0%AD%D0%B4%D0%B8%D1%81%D0%BE%D0%BD" title="Томас Эдисон">Томас Эдисон</a> пытался синхронизировать «<a href="/wiki/%D0%9A%D0%B8%D0%BD%D0%B5%D1%82%D0%BE%D1%81%D0%BA%D0%BE%D0%BF" title="Кинетоскоп">кинетоскоп</a>» c <a href="/wiki/%D0%A4%D0%BE%D0%BD%D0%BE%D0%B3%D1%80%D0%B0%D1%84" title="Фонограф">фонографом</a>, но потерпел неудачу. Однако впоследствии <a href="/wiki/%D0%94%D0%B8%D0%BA%D1%81%D0%BE%D0%BD,_%D0%A3%D0%B8%D0%BB%D1%8C%D1%8F%D0%BC_(%D0%B8%D0%B7%D0%BE%D0%B1%D1%80%D0%B5%D1%82%D0%B0%D1%82%D0%B5%D0%BB%D1%8C)" title="Диксон, Уильям (изобретатель)">Уильям Диксон</a>, соавтор Эдисона, утверждал, что ему уже в 1889 году удалось создать <a class="mw-redirect" href="/wiki/%D0%9A%D0%B8%D0%BD%D0%B5%D1%82%D0%BE%D1%84%D0%BE%D0%BD%D0%BE%D0%B3%D1%80%D0%B0%D1%84" title="Кинетофонограф">кинетофонограф</a> — прибор, воспроизводивший звук и изображение одновременно. Однако не существует никаких доказательств, подтверждающих его слова.
</p><p>В ранний период кинематографа звуковое кино пытались создать во множестве стран, но столкнулись с двумя основными проблемами: трудность в синхронизации изображения и звука и недостаточная громкость. Последняя проблема была решена после изобретения <a href="/wiki/%D0%A3%D1%81%D0%B8%D0%BB%D0%B8%D1%82%D0%B5%D0%BB%D1%8C_%D0%BD%D0%B8%D0%B7%D0%BA%D0%BE%D0%B9_%D1%87%D0%B0%D1%81%D1%82%D0%BE%D1%82%D1%8B" title="Усилитель низкой частоты">усилителя низкой частоты</a>, что произошло лишь в 1912 году, когда <a href="/wiki/%D0%9A%D0%B8%D0%BD%D0%BE%D1%8F%D0%B7%D1%8B%D0%BA" title="Киноязык">киноязык</a> развился настолько, что отсутствие звука уже не воспринималось как серьёзный недостаток. Трудности синхронизации удалось преодолеть использованием общего носителя для изображения и фонограммы, однако получить приемлемое качество фотографической звукозаписи стало возможно лишь к началу 1930-х годов.
</p><p>В результате <a href="/wiki/%D0%9F%D0%B0%D1%82%D0%B5%D0%BD%D1%82" title="Патент">патент</a> на ту систему звукового кинематографа, которая впоследствии совершила звуковую революцию, был получен в 1919 году, но кинокомпании не обратили никакого внимания на возможность кино заговорить, желая избежать удорожания стоимости производства и проката кинофильмов и потери иноязычных рынков.
</p>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Optical-film-soundtrack.svg"><img class="mw-file-element" data-file-height="520" data-file-width="420" decoding="async" height="176" src="//upload.wikimedia.org/wikipedia/commons/thumb/7/77/Optical-film-soundtrack.svg/142px-Optical-film-soundtrack.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/7/77/Optical-film-soundtrack.svg/213px-Optical-film-soundtrack.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/7/77/Optical-film-soundtrack.svg/284px-Optical-film-soundtrack.svg.png 2x" width="142"/></a><figcaption>Оптическая запись звука на пленке</figcaption></figure>
<p>Первый звуковой фильм снял <a class="new" href="/w/index.php?title=%D0%AE%D0%B4%D0%B6%D0%B8%D0%BD_%D0%90%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B8%D0%BD_%D0%9B%D0%B0%D1%81%D1%82&amp;action=edit&amp;redlink=1" title="Юджин Августин Ласт (страница отсутствует)">Юджин Августин Ласт</a>. Он запатентовал своё изобретение, прототип оптической системы записи звука на киноплёнку, которое с помощью <a href="/wiki/%D0%93%D0%B0%D0%BB%D1%8C%D0%B2%D0%B0%D0%BD%D0%BE%D0%BC%D0%B5%D1%82%D1%80" title="Гальванометр">гальванометра</a>, работало как функционирующая система, ещё в августе 1906 года<sup class="reference" id="cite_ref-автоссылка1_5-0"><a href="#cite_note-автоссылка1-5">[5]</a></sup>. Впервые он продемонстрировал данную технологию миру в 1910 году.
</p><p>Однако в 1919 году, благодаря изобретению технологии «<a href="/wiki/%D0%A2%D1%80%D0%B8%D1%8D%D1%80%D0%B3%D0%BE%D0%BD" title="Триэргон">Триэргон</a>», которая была применена <a href="/wiki/17_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="17 сентября">17 сентября</a> <a href="/wiki/1922_%D0%B3%D0%BE%D0%B4" title="1922 год">1922 года</a> в Берлине<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>, когда впервые в мире, публично был показан звуковой фильм «Der Brandstifter»<sup class="reference" id="cite_ref-Звук_7-0"><a href="#cite_note-Звук-7">[7]</a></sup> в кинотеатре «Аламбра»<sup class="reference" id="cite_ref-автоссылка1_5-1"><a href="#cite_note-автоссылка1-5">[5]</a></sup>.
</p><p>В 1925 году компания <a class="mw-redirect" href="/wiki/Warner_Brothers" title="Warner Brothers">Warner Brothers</a>, находившаяся в тот момент на грани банкротства, вложилась в рискованный звуковой проект. Уже в 1926 году Warner Brothers выпустила несколько звуковых фильмов, состоящих в основном из музыкальных номеров, но особого успеха у зрителей они не имели. Успех пришёл только с фильмом «<a class="mw-redirect" href="/wiki/%D0%9F%D0%B5%D0%B2%D0%B5%D1%86_%D0%B4%D0%B6%D0%B0%D0%B7%D0%B0_(%D1%84%D0%B8%D0%BB%D1%8C%D0%BC,_1927)" title="Певец джаза (фильм, 1927)">Певец джаза</a>», в котором кроме музыкальных номеров <a href="/wiki/%D0%AD%D0%BB_%D0%94%D0%B6%D0%BE%D0%BB%D1%81%D0%BE%D0%BD" title="Эл Джолсон">Эла Джолсона</a> присутствовали и его короткие реплики. <a href="/wiki/6_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="6 октября">6 октября</a> <a href="/wiki/1927_%D0%B3%D0%BE%D0%B4" title="1927 год">1927 года</a> — дату премьерного показа этой картины на большом экране, принято считать днём рождения звукового кино.
</p>
<h3><span id=".D0.9F.D0.BE.D1.8F.D0.B2.D0.BB.D0.B5.D0.BD.D0.B8.D0.B5_.D1.86.D0.B2.D0.B5.D1.82.D0.B0"></span><span class="mw-headline" id="Появление_цвета">Появление цвета</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Появление цвета»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84&amp;action=edit&amp;section=3" title="Редактировать код раздела «Появление цвета»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure class="mw-default-size" typeof="mw:File/Thumb"><span><video class="mw-file-element" controls="" data-durationhint="37" data-mwprovider="wikimediacommons" data-mwtitle="Annabelle_Serpentine.ogv" height="165" id="mwe_player_0" poster="//upload.wikimedia.org/wikipedia/commons/thumb/3/34/Annabelle_Serpentine.ogv/220px--Annabelle_Serpentine.ogv.jpg" preload="none" resource="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Annabelle_Serpentine.ogv" width="220"><source data-height="288" data-transcodekey="360p.webm" data-width="384" src="//upload.wikimedia.org/wikipedia/commons/transcoded/3/34/Annabelle_Serpentine.ogv/Annabelle_Serpentine.ogv.360p.webm" type='video/webm; codecs="vp8, vorbis"'/><source data-height="288" data-width="384" src="//upload.wikimedia.org/wikipedia/commons/3/34/Annabelle_Serpentine.ogv" type='video/ogg; codecs="theora"'/><source data-height="240" data-transcodekey="240p.vp9.webm" data-width="320" src="//upload.wikimedia.org/wikipedia/commons/transcoded/3/34/Annabelle_Serpentine.ogv/Annabelle_Serpentine.ogv.240p.vp9.webm" type='video/webm; codecs="vp9, opus"'/><source data-height="240" data-transcodekey="m3u8" data-width="320" src="//upload.wikimedia.org/wikipedia/commons/transcoded/3/34/Annabelle_Serpentine.ogv/Annabelle_Serpentine.ogv.m3u8" type="application/vnd.apple.mpegurl"/></video></span><figcaption>Кадры из фильма Annabelle Serpentine Dance (1894—1896)</figcaption></figure>
<div class="hatnote navigation-not-searchable">Основная статья: <b><a href="/wiki/%D0%A6%D0%B2%D0%B5%D1%82%D0%BD%D0%BE%D0%B9_%D0%BA%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84" title="Цветной кинематограф">Цветной кинематограф</a></b></div>
<p>Первым сохранившимся цветным фильмом стал короткометражный фильм «<a href="/wiki/%D0%A2%D0%B0%D0%BD%D0%B5%D1%86_%D0%9B%D0%BE%D0%B8_%D0%A4%D1%83%D0%BB%D0%BB%D0%B5%D1%80" title="Танец Лои Фуллер">Танец Лои Фуллер</a>» (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Annabelle Serpentine Dance</span>). Он был снят в чёрно-белом варианте в 1894 году, а в 1895 или в 1896 году был раскрашен вручную (кисточкой раскрашивался каждый <a href="/wiki/%D0%9A%D0%B0%D0%B4%D1%80%D0%B8%D0%BA" title="Кадрик">кадрик</a>). Часть <a href="/wiki/%D0%A4%D0%B8%D0%BB%D1%8C%D0%BC%D0%BE%D0%BA%D0%BE%D0%BF%D0%B8%D1%8F" title="Фильмокопия">фильмокопий</a> первого коммерчески успешного цветного фильма «<a href="/wiki/%D0%9F%D1%83%D1%82%D0%B5%D1%88%D0%B5%D1%81%D1%82%D0%B2%D0%B8%D0%B5_%D0%BD%D0%B0_%D0%9B%D1%83%D0%BD%D1%83_(%D1%84%D0%B8%D0%BB%D1%8C%D0%BC,_1902)" title="Путешествие на Луну (фильм, 1902)">Путешествие на Луну</a>», созданного Жоржем Мельесом в 1902 году, также была раскрашена вручную бригадой художниц.
</p><p>В 1899 году фотограф Эдвард Рэймонд Тёрнер запатентовал процесс <a href="/wiki/%D0%A1%D1%8A%D1%91%D0%BC%D0%BA%D0%B0_(%D0%BA%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84)" title="Съёмка (кинематограф)">съёмки</a> цветного кино. По технологии Тёрнера каждый кадр последовательно снимался через три специальных <a href="/wiki/%D0%A1%D0%B2%D0%B5%D1%82%D0%BE%D1%84%D0%B8%D0%BB%D1%8C%D1%82%D1%80" title="Светофильтр">светофильтра</a> красного, зелёного и синего цветов. В 2012 году сотрудники Национального музея СМИ и технологий в <a href="/wiki/%D0%91%D1%80%D0%B0%D0%B4%D1%84%D0%BE%D1%80%D0%B4" title="Брадфорд">Брадфорде</a> отыскали цветную кинозапись Эдварда Тёрнера, датированную 1902 годом<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup><sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>. Ранее самой старой считалась цветная плёнка 1909 года, созданная по технологии <a href="/wiki/Kinemacolor" title="Kinemacolor">Kinemacolor</a>.
</p><p>Британская технология «<a class="mw-redirect" href="/wiki/%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D0%BA%D0%BE%D0%BB%D0%BE%D1%80" title="Кинемаколор">Кинемаколор</a>» (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Kinemacolor</span>), изобретённая в 1906 году, была первой в мире системой цветного кинематографа, имевшей коммерческий успех. Однако, по сравнению с фильмами, раскрашенными вручную, она имела недостаток: все цвета создавались путём смешивания не трёх, а только двух основных цветов: красно-оранжевого и сине-зелёного<sup class="reference" id="cite_ref-ksa_11-0"><a href="#cite_note-ksa-11">[11]</a></sup>. В этой системе в 1908 году был снят «Визит к морю» (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">A Visit to the Seaside</span>) — первый цветной фильм, показанный в кинотеатрах, в 1910 году — первый цветной игровой фильм «Шахматист» (англ. Checkmated), в 1911 году — первый цветной полнометражный документальный фильм «Торжественный приём в Дели» (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">With Our King and Queen Through India</span>).
</p><p>Первыми <a href="/wiki/%D0%9F%D0%BE%D0%BB%D0%BD%D0%BE%D0%BC%D0%B5%D1%82%D1%80%D0%B0%D0%B6%D0%BD%D1%8B%D0%B9_%D1%84%D0%B8%D0%BB%D1%8C%D0%BC" title="Полнометражный фильм">полнометражными</a> игровыми цветными фильмами, показанными в кинотеатрах, были «The World, the Flesh and the Devil» (1914) и «Little Lord Fauntleroy» (1914), снятые по технологии «Кинемаколор», «The Gulf Between» (1917), изготовленный по технологии «<a href="/wiki/%D0%A2%D0%B5%D1%85%D0%BD%D0%B8%D0%BA%D0%BE%D0%BB%D0%BE%D1%80" title="Техниколор">Техниколор</a>» (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Technicolor</span>), и «Cupid Angling» (1918), созданный по технологии <i>Douglass Natural Color process</i>.
</p>
<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Anna_May_Wong_holds_child_in_The_Toll_of_the_Sea.jpg"><img class="mw-file-element" data-file-height="478" data-file-width="640" decoding="async" height="149" src="//upload.wikimedia.org/wikipedia/commons/thumb/0/08/Anna_May_Wong_holds_child_in_The_Toll_of_the_Sea.jpg/200px-Anna_May_Wong_holds_child_in_The_Toll_of_the_Sea.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/0/08/Anna_May_Wong_holds_child_in_The_Toll_of_the_Sea.jpg/300px-Anna_May_Wong_holds_child_in_The_Toll_of_the_Sea.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/0/08/Anna_May_Wong_holds_child_in_The_Toll_of_the_Sea.jpg/400px-Anna_May_Wong_holds_child_in_The_Toll_of_the_Sea.jpg 2x" width="200"/></a><figcaption>Кадр из первого голливудского цветного фильма «Жертвы моря» (1922)</figcaption></figure>
<p>Первый <a href="/wiki/%D0%93%D0%BE%D0%BB%D0%BB%D0%B8%D0%B2%D1%83%D0%B4" title="Голливуд">голливудский</a> двухцветный фильм, снятый по технологии «<a href="/wiki/%D0%91%D0%B8%D0%BF%D0%B0%D0%BA" title="Бипак">бипак</a>» вышел в <a href="/wiki/1922_%D0%B3%D0%BE%D0%B4" title="1922 год">1922 году</a>, он не впечатлил зрителей. Однако последовавшие за ним цветные голливудские фильмы, такие как «Странник пустоты» (1924), получили огромный кассовый успех.
</p><p>Вслед за бешеной популярностью цветных фильмов в Европе, а затем и в США, наступил период охлаждения интереса к цветным фильмам. Цветные фильмы были более дорогими. Изображение на них было менее чётким. Сочетаниями двух цветов нельзя было изобразить все цвета в природе (требовалось три цвета). Существовали и трёхцветные системы, но в них изображение было ещё хуже, так как в них использовались три объектива, и параллакс между ними приводил к образованию цветной каймы у предметов. Режиссёры «серьёзных» фильмов избегали цветного кинематографа, и все шедевры того времени были чёрно-белыми. Публика воспринимала цветные фильмы как аттракцион. Положение было исправлено после изобретения трёхцветного однокамерного варианта технологии «<a class="mw-redirect" href="/wiki/Technicolor" title="Technicolor">Technicolor</a>». Впервые эту технику применил Уолт Дисней в 1932 году в короткометражном мультфильме «Цветы и деревья». Первый «полноценно цветной» художественный короткометражный фильм системы «Техниколор» под названием «La Cucaracha» вышел в <a href="/wiki/1934_%D0%B3%D0%BE%D0%B4" title="1934 год">1934 году</a>.
</p><p>Первый же <a href="/wiki/%D0%9F%D0%BE%D0%BB%D0%BD%D0%BE%D0%BC%D0%B5%D1%82%D1%80%D0%B0%D0%B6%D0%BD%D1%8B%D0%B9_%D1%84%D0%B8%D0%BB%D1%8C%D0%BC" title="Полнометражный фильм">полнометражный</a> цветной фильм «<a class="mw-redirect" href="/wiki/%D0%91%D0%B5%D0%BA%D0%BA%D0%B8_%D0%A8%D0%B0%D1%80%D0%BF_(%D1%84%D0%B8%D0%BB%D1%8C%D0%BC)" title="Бекки Шарп (фильм)">Бекки Шарп</a>» американского режиссёра <a class="mw-redirect" href="/wiki/%D0%A0%D1%83%D0%B1%D0%B5%D0%BD_%D0%9C%D0%B0%D0%BC%D1%83%D0%BB%D1%8F%D0%BD" title="Рубен Мамулян">Рубена Мамуляна</a> вышел в <a class="mw-redirect" href="/wiki/1935" title="1935">1935</a>, этот год и принято считать годом появления цветного кино.
</p><p>Хотя система «Техниколор» (и её советский аналог ЦКС-1) была однокамерной, но показ в кинотеатрах производился с трёх кинопроекторов, что налагало большие требования к квалификации и аккуратности киномехаников. Неточная настройка приводила к двоению или троению изображения. Неодновременность включения проекторов приводила к рассинхронизации изображений разных цветов. Избавить от этих недостатков смогла технология многослойных киноплёнок. Первым таким фотоматериалом стала плёнка «Гаспарколор», выпущенная в 1933 году по патенту венгерского учёного Бела Гаспара. В <a href="/wiki/1935_%D0%B3%D0%BE%D0%B4" title="1935 год">1935 году</a> <a class="mw-redirect" href="/wiki/Kodak" title="Kodak">Kodak</a> представил первую многослойную киноплёнку, пригодную для фотосъёмки, но для киносъёмки она не годилась. Летом 1937 года немецкая <a href="/wiki/Agfa-Gevaert" title="Agfa-Gevaert">AGFA</a> запустила производство первых в мире <a href="/wiki/%D0%A5%D1%80%D0%BE%D0%BC%D0%BE%D0%B3%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D1%82%D0%BE%D0%BC%D0%B0%D1%82%D0%B5%D1%80%D0%B8%D0%B0%D0%BB" title="Хромогенный фотоматериал">хромогенных</a> негативных многослойных киноплёнок, на которые отснят короткометражный игровой фильм «Отзвучит песня» (<a href="/wiki/%D0%9D%D0%B5%D0%BC%D0%B5%D1%86%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Немецкий язык">нем.</a> <span lang="de" style="font-style:italic;">Ein lied verklingt</span>)<sup class="reference" id="cite_ref-trip_12-0"><a href="#cite_note-trip-12">[12]</a></sup>. До конца <a href="/wiki/%D0%92%D1%82%D0%BE%D1%80%D0%B0%D1%8F_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Вторая мировая война">Второй Мировой войны</a> немецкие цветные плёнки не <a href="/wiki/%D0%AD%D0%BA%D1%81%D0%BF%D0%BE%D1%80%D1%82" title="Экспорт">экспортировались</a> и использовались только немецкой государственной киностудией <a class="mw-redirect" href="/wiki/UFA" title="UFA">UFA</a> под контролем <a href="/wiki/%D0%98%D0%BC%D0%BF%D0%B5%D1%80%D1%81%D0%BA%D0%BE%D0%B5_%D0%BC%D0%B8%D0%BD%D0%B8%D1%81%D1%82%D0%B5%D1%80%D1%81%D1%82%D0%B2%D0%BE_%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D0%BE%D0%B3%D0%BE_%D0%BF%D1%80%D0%BE%D1%81%D0%B2%D0%B5%D1%89%D0%B5%D0%BD%D0%B8%D1%8F_%D0%B8_%D0%BF%D1%80%D0%BE%D0%BF%D0%B0%D0%B3%D0%B0%D0%BD%D0%B4%D1%8B" title="Имперское министерство народного просвещения и пропаганды">министерства пропаганды Германии</a><sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup>. После победы над Германией союзники получили доступ к технологиям Agfa. В Голливуде производство многослойных киноплёнок по немецкой технологии началось в 1949 году<sup class="reference" id="cite_ref-_fbf026c0416dc0b8_14-0"><a href="#cite_note-_fbf026c0416dc0b8-14">[14]</a></sup>. Многослойные киноплёнки вытеснили другие технологии и господствовали в цветном кино вплоть до расцвета цифровых технологий в 2000-х годах.
</p>
<h4><span id=".D0.9F.D0.BE.D1.8F.D0.B2.D0.BB.D0.B5.D0.BD.D0.B8.D0.B5_.D1.86.D0.B2.D0.B5.D1.82.D0.BD.D0.BE.D0.B3.D0.BE_.D0.BA.D0.B8.D0.BD.D0.BE_.D0.B2_.D0.A0.D0.BE.D1.81.D1.81.D0.B8.D0.B8"></span><span class="mw-headline" id="Появление_цветного_кино_в_России">Появление цветного кино в России</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Появление цветного кино в России»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84&amp;action=edit&amp;section=4" title="Редактировать код раздела «Появление цветного кино в России»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h4>
<p>Первой в России вручную раскрашенной в цвет чёрно-белой картиной была короткометражная лента «<a href="/wiki/%D0%A3%D1%85%D0%B0%D1%80%D1%8C-%D0%BA%D1%83%D0%BF%D0%B5%D1%86_(%D1%84%D0%B8%D0%BB%D1%8C%D0%BC)" title="Ухарь-купец (фильм)">Ухарь-купец</a>» (1909)<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup>. В 1925 году в полнометражном фильме «<a href="/wiki/%D0%91%D1%80%D0%BE%D0%BD%D0%B5%D0%BD%D0%BE%D1%81%D0%B5%D1%86_%C2%AB%D0%9F%D0%BE%D1%82%D1%91%D0%BC%D0%BA%D0%B8%D0%BD%C2%BB_(%D1%84%D0%B8%D0%BB%D1%8C%D0%BC)" title="Броненосец «Потёмкин» (фильм)">Броненосец „Потёмкин“</a>» был раскрашен флаг.
</p><p>Первый в России (и во всём <a class="mw-redirect" href="/wiki/%D0%A1%D0%A1%D0%A1%D0%A0" title="СССР">СССР</a>) цветной документальный короткометражный фильм «Праздник труда» был снят по технологии «Спектроколор», похожей на «Кинемаколор», в 1931 году<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup>.
</p><p>В <a class="mw-redirect" href="/wiki/%D0%A1%D0%A1%D0%A1%D0%A0" title="СССР">СССР</a> первый игровой цветной виражный фильм «<a class="mw-redirect" href="/wiki/%D0%93%D1%80%D1%83%D0%BD%D1%8F_%D0%9A%D0%BE%D1%80%D0%BD%D0%B0%D0%BA%D0%BE%D0%B2%D0%B0" title="Груня Корнакова">Груня Корнакова</a>» был снят по системе «бипак», сходной с «<a class="mw-redirect" href="/wiki/%D0%A1%D0%B8%D0%BD%D0%B5%D0%BA%D0%BE%D0%BB%D0%BE%D1%80" title="Синеколор">Синеколором</a>» в <a href="/wiki/1936_%D0%B3%D0%BE%D0%B4" title="1936 год">1936 году</a>. В конце 30-х годов в СССР были изготовлены первые кинокамеры трёхцветной однокамерной системы «ЦКС-1». Первый полнометражный фильм по этой технологии («<a href="/wiki/%D0%98%D0%B2%D0%B0%D0%BD_%D0%9D%D0%B8%D0%BA%D1%83%D0%BB%D0%B8%D0%BD_%E2%80%94_%D1%80%D1%83%D1%81%D1%81%D0%BA%D0%B8%D0%B9_%D0%BC%D0%B0%D1%82%D1%80%D0%BE%D1%81" title="Иван Никулин — русский матрос">Иван Никулин — русский матрос</a>») из-за войны вышел только в 1944 году. Первым же цветным фильмом, снятым на многослойную цветную киноплёнку <a href="/wiki/Agfa-Gevaert" title="Agfa-Gevaert">AGFA</a> стал фильм о <a href="/wiki/%D0%9F%D0%B0%D1%80%D0%B0%D0%B4_%D0%9F%D0%BE%D0%B1%D0%B5%D0%B4%D1%8B_(%D1%84%D0%B8%D0%BB%D1%8C%D0%BC)" title="Парад Победы (фильм)">Параде Победы</a> 1945 года.
</p><p>Своеобразное развитие в цвете в последнее время получили уже известные советские киноленты, снятые в чёрно-белых тонах («В бой идут одни старики», «17 мгновений весны» и т. д.) — с помощью компьютерных технологий они <a href="/wiki/%D0%9A%D0%BE%D0%BB%D0%BE%D1%80%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F" title="Колоризация">были раскрашены</a>. При этом публика неоднозначно встретила нововведение, так как варианты фильмов в цвете не имеют существенных преимуществ перед оригиналами.
Напротив, в современных фильмах о <a class="mw-redirect" href="/wiki/%D0%92%D0%9E%D0%92" title="ВОВ">Великой Отечественной войне</a> часто намеренно приглушают цветность.
</p>
<h3><span id=".D0.94.D0.B0.D0.BB.D1.8C.D0.BD.D0.B5.D0.B9.D1.88.D0.B8.D0.B9_.D1.82.D0.B5.D1.85.D0.BD.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.B8.D0.B9_.D0.BF.D1.80.D0.BE.D0.B3.D1.80.D0.B5.D1.81.D1.81_.D0.B2_.D0.BA.D0.B8.D0.BD.D0.BE"></span><span class="mw-headline" id="Дальнейший_технический_прогресс_в_кино">Дальнейший технический прогресс в кино</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Дальнейший технический прогресс в кино»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84&amp;action=edit&amp;section=5" title="Редактировать код раздела «Дальнейший технический прогресс в кино»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>В <a class="mw-redirect" href="/wiki/1950-%D0%B5" title="1950-е">1950-х</a> годах значительную долю рынка стало отбирать у кинематографа широко распространившееся телевещание, и стремление привлечь зрителя высоким качеством кинопоказа завело технический прогресс в кино ещё дальше. Разработка и внедрение <a href="/wiki/%D0%9C%D0%B0%D0%B3%D0%BD%D0%B8%D1%82%D0%BD%D0%B0%D1%8F_%D0%B7%D0%B2%D1%83%D0%BA%D0%BE%D0%B7%D0%B0%D0%BF%D0%B8%D1%81%D1%8C" title="Магнитная звукозапись">магнитной записи</a> и воспроизведения звука, а также создание и освоение новых видов кинематографа (<a class="mw-redirect" href="/wiki/%D0%9F%D0%B0%D0%BD%D0%BE%D1%80%D0%B0%D0%BC%D0%BD%D0%BE%D0%B5_%D0%BA%D0%B8%D0%BD%D0%BE" title="Панорамное кино">панорамного</a>, <a href="/wiki/%D0%92%D0%B8%D0%B4%D0%B5%D0%BE#Стереоскопическое_видео" title="Видео">стереоскопического</a>, <a href="/wiki/%D0%9F%D0%BE%D0%BB%D0%B8%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%BD%D0%BE%D0%B5_%D0%BA%D0%B8%D0%BD%D0%BE" title="Полиэкранное кино">полиэкранного</a> и др.) привели к значительному повышению качества показа фильмов, стали говорить об «эффекте присутствия» зрителя. Впечатление усиливалось <a href="/wiki/%D0%A1%D1%82%D0%B5%D1%80%D0%B5%D0%BE%D1%84%D0%BE%D0%BD%D0%B8%D1%8F" title="Стереофония">стереофоническим</a> воспроизведением звука, позволявшим создавать «пространственную звуковую перспективу» — звук как бы следует за изображением его источника, вызывая иллюзию реальности источника звука.
</p><p>Для создания стереоскопического изображения необходимо произвести киносъёмку из двух точек, имитирующих два глаза наблюдателя. На экране кинотеатра оба изображения проецируются вместе, и разделяются посредством очков, содержащих цветовые светофильтры или <a href="/wiki/%D0%9F%D0%BE%D0%BB%D1%8F%D1%80%D0%B8%D0%B7%D0%B0%D1%82%D0%BE%D1%80" title="Поляризатор">поляризаторы</a> в двух перпендикулярных плоскостях поляризации.
</p><p>В настоящее же время существуют очень изощрённые системы звукового сопровождения кино. Число отдельных звуковых каналов доходит до 7, а в перспективных системах даже до <a class="mw-redirect" href="/wiki/UHDTV" title="UHDTV">24</a>. Разумеется, всё это призвано усилить глубину погружения зрителя в атмосферу просматриваемого фильма.
</p>
<h2><span id=".D0.A2.D0.B5.D1.85.D0.BD.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.B8.D0.B5_.D0.BE.D1.81.D0.BE.D0.B1.D0.B5.D0.BD.D0.BD.D0.BE.D1.81.D1.82.D0.B8"></span><span class="mw-headline" id="Технические_особенности">Технические особенности</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Технические особенности»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84&amp;action=edit&amp;section=6" title="Редактировать код раздела «Технические особенности»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id=".D0.A1.D0.BE.D0.BE.D1.82.D0.BD.D0.BE.D1.88.D0.B5.D0.BD.D0.B8.D0.B5_.D1.81.D1.82.D0.BE.D1.80.D0.BE.D0.BD_.D1.8D.D0.BA.D1.80.D0.B0.D0.BD.D0.B0"></span><span class="mw-headline" id="Соотношение_сторон_экрана"><a href="/wiki/%D0%A1%D0%BE%D0%BE%D1%82%D0%BD%D0%BE%D1%88%D0%B5%D0%BD%D0%B8%D0%B5_%D1%81%D1%82%D0%BE%D1%80%D0%BE%D0%BD_%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0" title="Соотношение сторон экрана">Соотношение сторон экрана</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Соотношение сторон экрана»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84&amp;action=edit&amp;section=7" title="Редактировать код раздела «Соотношение сторон экрана»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Соотношение ширины и высоты кадра (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">aspect ratio</span>) — важнейшее понятие в кинематографе. В немом кинематографе отношение горизонтальной к вертикальной стороне кадра составляло примерно 4:3 (4 единицы в ширину к 3 единицам в высоту; иногда ещё записывается как 1,33:1 или просто 1,33) — сложившееся ещё во времена Эдисона и Люмьеров в силу достаточно случайных причин, хотя и близкое к самому распространённому формату полотна в живописи. Это же отношение было перенято и телевидением. С появлением звукового кино был узаконен так называемый <a class="mw-redirect" href="/wiki/%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%B8%D0%BD%D0%BE%D1%84%D0%BE%D1%80%D0%BC%D0%B0%D1%82" title="Классический киноформат">«академический» формат</a> с соотношением сторон кадра 1,375:1, чаще всего сокращаемое до 1,37:1<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup>. С развитием и распространением телевидения, кино стало активно обращаться к широкому экрану, в котором постепенно утвердились два основных формата: 2,35:1 (то есть примерно 7:3) и 2,2:1. Существуют экспериментальные фильмы с иным соотношением (например, <a href="/wiki/%D0%9A%D1%80%D1%83%D0%B3%D0%BE%D1%80%D0%B0%D0%BC%D0%BD%D0%BE%D0%B5_%D0%BA%D0%B8%D0%BD%D0%BE" title="Кругорамное кино">кругорамные</a> киносистемы с горизонтальным обзором 360°).
</p><p>Однако <a href="/wiki/%D0%A8%D0%B8%D1%80%D0%BE%D0%BA%D0%BE%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84" title="Широкоэкранный кинематограф">широкоэкранное</a> кино никак не могло претендовать на всеобщее применение, поскольку оно подходит для масштабных эпических композиций, и в редких случаях для камерного психологического кино (не только из общеэстетических соображений, но из элементарного обстоятельства, что на изолированном крупном плане человеческого лица при широкоэкранной съёмке примерно две трети кадра остаются незаполненными). В то же время, и классическое соотношение 1,37:1 не всегда является выигрышным, и как только возник сам по себе вопрос об изменении всей технологии кинопроцесса, кинематограф стал тяготеть к соотношению сторон близкому к <a href="/wiki/%D0%97%D0%BE%D0%BB%D0%BE%D1%82%D0%BE%D0%B5_%D1%81%D0%B5%D1%87%D0%B5%D0%BD%D0%B8%D0%B5" title="Золотое сечение">золотому сечению</a> (это примерно 1,62:1). В результате появился формат 5:3 (1,66:1), на который довольно быстро перешло западноевропейское кино; в США же стал доминировать формат, промежуточный между европейским и широким — 1,85:1<sup class="reference" id="cite_ref-_3f2d21d47dfca747_18-0"><a href="#cite_note-_3f2d21d47dfca747-18">[18]</a></sup>.
</p>
<h3><span id=".D0.9A.D0.B8.D0.BD.D0.B5.D0.BC.D0.B0.D1.82.D0.BE.D0.B3.D1.80.D0.B0.D1.84.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.B8.D0.B5_.D1.81.D0.B8.D1.81.D1.82.D0.B5.D0.BC.D1.8B"></span><span class="mw-headline" id="Кинематографические_системы">Кинематографические системы</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84&amp;veaction=edit&amp;section=8" title="Редактировать раздел «Кинематографические системы»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84&amp;action=edit&amp;section=8" title="Редактировать код раздела «Кинематографические системы»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Соотношение сторон изображения, получаемого на экране и другие технические характеристики кинофильма, зависят от формата, в котором он изготовлен. Существует огромное количество различных систем кинематографа, классифицируемых, прежде всего, по ширине используемой киноплёнки и соотношению сторон изображения.
</p>
<div class="hatnote navigation-not-searchable">Основная статья: <b><a href="/wiki/%D0%A1%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0_%D0%BA%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84%D0%B0" title="Система кинематографа">Система кинематографа</a></b></div>
<h3><span id=".C2.AB.D0.AD.D1.84.D1.84.D0.B5.D0.BA.D1.82_25-.D0.B3.D0.BE_.D0.BA.D0.B0.D0.B4.D1.80.D0.B0.C2.BB"></span><span class="mw-headline" id="«Эффект_25-го_кадра»">«Эффект 25-го кадра»</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84&amp;veaction=edit&amp;section=9" title="Редактировать раздел ««Эффект 25-го кадра»»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84&amp;action=edit&amp;section=9" title="Редактировать код раздела ««Эффект 25-го кадра»»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Кинокамера фиксирует фазы движения объекта на киноплёнке в виде ряда последовательных <a href="/wiki/%D0%A4%D0%BE%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D1%8F" title="Фотография">фотоснимков</a> (кадров киноизображения). Затем эти кадры проецируются на экране. Частота кадров старых (немых) чёрно-белых фильмов составляла 1000 кадров в минуту (16⅔ кадра в секунду), так как иллюзия движения объектов на экране возникает, когда время между кадрами становится меньше, чем время <a href="/wiki/%D0%98%D0%BD%D0%B5%D1%80%D1%86%D0%B8%D1%8F_%D0%B7%D1%80%D0%B5%D0%BD%D0%B8%D1%8F" title="Инерция зрения">инерции зрения</a>, которое составляет примерно 0,1 с<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup> (см. <a href="/wiki/%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D1%80%D0%B8%D0%BD%D1%86%D0%B8%D0%BF" title="Кинематографический принцип">Кинематографический принцип</a>). С возникновением звукового кинематографа количество кадров в секунду было увеличено до 24, что и стало стандартом для съёмки практически на весь XX век. В современных кинотеатрах минимальная частота проекции составляет 48 мельканий в секунду (это 24 кадра в секунду при двойном мелькании <a href="/wiki/%D0%9E%D0%B1%D1%82%D1%8E%D1%80%D0%B0%D1%82%D0%BE%D1%80_(%D0%BE%D0%BF%D1%82%D0%B8%D0%BA%D0%B0)" title="Обтюратор (оптика)">обтюратора</a>).
</p><p>В середине <a href="/wiki/XX_%D0%B2%D0%B5%D0%BA" title="XX век">XX века</a> был распространён миф о том, что сознание человека якобы может воспринять лишь 24 кадра в секунду — а <a href="/wiki/25-%D0%B9_%D0%BA%D0%B0%D0%B4%D1%80" title="25-й кадр">25-й кадр</a>, если его вставить в воспроизведение, якобы будет восприниматься <a href="/wiki/%D0%A7%D0%B5%D0%BB%D0%BE%D0%B2%D0%B5%D0%BA" title="Человек">человеком</a> на <a href="/wiki/%D0%9F%D0%BE%D0%B4%D1%81%D0%BE%D0%B7%D0%BD%D0%B0%D0%BD%D0%B8%D0%B5" title="Подсознание">подсознательном</a> уровне. Из этого заблуждения были сделаны выводы об эффективности «феномена 25-го кадра» в различных видах внушения и подсознательного воздействия (например, в целях политической <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%BF%D0%B0%D0%B3%D0%B0%D0%BD%D0%B4%D0%B0" title="Пропаганда">пропаганды</a>, коммерческой <a href="/wiki/%D0%A0%D0%B5%D0%BA%D0%BB%D0%B0%D0%BC%D0%B0" title="Реклама">рекламы</a>, при обучении иностранным <a class="mw-redirect" href="/wiki/%D0%AF%D0%B7%D1%8B%D0%BA_(%D1%81%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0_%D0%B7%D0%BD%D0%B0%D0%BA%D0%BE%D0%B2)" title="Язык (система знаков)">языкам</a>, лечении от <a class="mw-redirect" href="/wiki/%D0%9D%D0%B0%D1%80%D0%BA%D0%BE%D0%B7%D0%B0%D0%B2%D0%B8%D1%81%D0%B8%D0%BC%D0%BE%D1%81%D1%82%D1%8C" title="Наркозависимость">наркозависимости</a> и пр.). Все вымыслы о влиянии 25-го кадра на подсознание человека не имеют отношения к реальности.
</p>
<h3><span id=".D0.A6.D0.B8.D1.84.D1.80.D0.BE.D0.B2.D0.BE.D0.B9_.D0.BA.D0.B8.D0.BD.D0.B5.D0.BC.D0.B0.D1.82.D0.BE.D0.B3.D1.80.D0.B0.D1.84"></span><span class="mw-headline" id="Цифровой_кинематограф">Цифровой кинематограф</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84&amp;veaction=edit&amp;section=10" title="Редактировать раздел «Цифровой кинематограф»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84&amp;action=edit&amp;section=10" title="Редактировать код раздела «Цифровой кинематограф»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>В начале <a href="/wiki/XXI_%D0%B2%D0%B5%D0%BA" title="XXI век">XXI века</a>, с развитием цифровых технологий записи изображения, появилось понятие «<a href="/wiki/%D0%A6%D0%B8%D1%84%D1%80%D0%BE%D0%B2%D0%BE%D0%B5_%D0%BA%D0%B8%D0%BD%D0%BE" title="Цифровое кино">цифровой кинематограф</a>». Под этим термином понимают новую технологию фильмопроизводства, позволяющую обходиться без использования киноплёнки. В цифровом кино съёмка, обработка, монтаж и демонстрация фильма происходят при помощи цифрового оборудования. Исходный материал записывается при помощи <a href="/wiki/%D0%A6%D0%B8%D1%84%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%BA%D0%B8%D0%BD%D0%BE%D0%BA%D0%B0%D0%BC%D0%B5%D1%80%D0%B0" title="Цифровая кинокамера">цифровой кинокамеры</a> прямо на цифровой носитель данных. В этом случае обычный кинопроектор заменяется <a href="/wiki/%D0%A6%D0%B8%D1%84%D1%80%D0%BE%D0%B2%D0%BE%D0%B9_%D0%BA%D0%B8%D0%BD%D0%BE%D0%BF%D1%80%D0%BE%D0%B5%D0%BA%D1%82%D0%BE%D1%80" title="Цифровой кинопроектор">цифровым</a>. Часть копий фильма печатается на киноплёнке при помощи <a href="/wiki/%D0%A4%D0%B8%D0%BB%D1%8C%D0%BC-%D1%80%D0%B5%D0%BA%D0%BE%D1%80%D0%B4%D0%B5%D1%80" title="Фильм-рекордер">фильм-рекордера</a>. При этом изготовляется высококачественный <a class="mw-redirect" href="/wiki/%D0%94%D1%83%D0%B1%D0%BB%D1%8C%D0%BD%D0%B5%D0%B3%D0%B0%D1%82%D0%B8%D0%B2" title="Дубльнегатив">дубльнегатив</a> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">internegative</span>) для последующей <a href="/wiki/%D0%9A%D0%B8%D0%BD%D0%BE%D0%BA%D0%BE%D0%BF%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%B0%D0%BF%D0%BF%D0%B0%D1%80%D0%B0%D1%82" title="Кинокопировальный аппарат">печати</a> плёночных <a href="/wiki/%D0%A4%D0%B8%D0%BB%D1%8C%D0%BC%D0%BE%D0%BA%D0%BE%D0%BF%D0%B8%D1%8F" title="Фильмокопия">фильмокопий</a>. Современные цифровые камеры обеспечивают очень высокое разрешение изображения, хорошую цветопередачу и широчайший, недоступный до недавнего времени, спектр манипуляций с цветовой гаммой изображения. <a href="/wiki/%D0%A6%D0%B8%D1%84%D1%80%D0%BE%D0%B2%D1%8B%D0%B5_%D1%82%D0%B5%D1%85%D0%BD%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D0%B8" title="Цифровые технологии">Цифровые технологии</a> также предоставляют большие возможности для использования <a class="new" href="/w/index.php?title=%D0%92%D0%B8%D0%B4%D0%B5%D0%BE%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D0%BA%D0%B0&amp;action=edit&amp;redlink=1" title="Видеографика (страница отсутствует)">видеографики</a> и <a href="/wiki/%D0%A1%D0%BF%D0%B5%D1%86%D1%8D%D1%84%D1%84%D0%B5%D0%BA%D1%82" title="Спецэффект">спецэффектов</a> в кино. Однако до сих пор киноплёнка, особенно, <a class="mw-redirect" href="/wiki/%D0%A8%D0%B8%D1%80%D0%BE%D0%BA%D0%BE%D1%84%D0%BE%D1%80%D0%BC%D0%B0%D1%82%D0%BD%D0%BE%D0%B5_%D0%BA%D0%B8%D0%BD%D0%BE" title="Широкоформатное кино">больших форматов</a>, превосходит по разрешающей способности большинство цифровых кинокамер.
На несколько лет раньше полностью бесплёночных технологий получила распространение технология, предусматривающая <a href="/wiki/%D0%A1%D0%BA%D0%B0%D0%BD%D0%B5%D1%80_%D0%BA%D0%B8%D0%BD%D0%BE%D0%BF%D0%BB%D1%91%D0%BD%D0%BA%D0%B8" title="Сканер киноплёнки">сканирование</a> <a href="/wiki/%D0%9A%D0%B8%D0%BD%D0%BE%D0%BD%D0%B5%D0%B3%D0%B0%D1%82%D0%B8%D0%B2" title="Кинонегатив">негатива</a> изображения и последующая цифровая обработка полученных данных, <a class="mw-redirect" href="/wiki/Digital_Intermediate" title="Digital Intermediate">Digital Intermediate</a>. Такая технология обладает большей гибкостью, чем плёночная и позволяет обойтись без многих промежуточных стадий фильмопроизводства.
</p>

<h3><span class="mw-headline" id="IMDb"><a class="mw-redirect" href="/wiki/IMDb" title="IMDb">IMDb</a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84&amp;veaction=edit&amp;section=14" title="Редактировать раздел «IMDb»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84&amp;action=edit&amp;section=14" title="Редактировать код раздела «IMDb»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote navigation-not-searchable">Основная статья: <b><a class="mw-redirect" href="/wiki/IMDb" title="IMDb">IMDb</a></b></div>
<p>Огромный вклад в упорядочение информации о кинематографе внесла <a class="external text" href="https://www.imdb.com/" rel="nofollow">База данных фильмов в Интернете</a> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Internet Movie Database; IMDb</span>) — крупнейшая на планете <a href="/wiki/%D0%91%D0%B0%D0%B7%D0%B0_%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85" title="База данных">база данных</a> и <a href="/wiki/%D0%A1%D0%B0%D0%B9%D1%82" title="Сайт">веб-сайт</a> о кинематографе. Сейчас это один из редких примеров удачного сотрудничества крупного <a href="/wiki/%D0%91%D0%B8%D0%B7%D0%BD%D0%B5%D1%81" title="Бизнес">бизнеса</a> и <a class="new" href="/w/index.php?title=%D0%9A%D0%B8%D0%BD%D0%BE%D0%BC%D0%B0%D0%BD&amp;action=edit&amp;redlink=1" title="Киноман (страница отсутствует)">киноманов</a>-<a href="/wiki/%D0%90%D0%BB%D1%8C%D1%82%D1%80%D1%83%D0%B8%D0%B7%D0%BC" title="Альтруизм">альтруистов</a>. Некоторые разделы базы до сих пор в значительной степени наполняются добровольцами, это схоже с <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%86%D0%B5%D0%BF%D1%86%D0%B8%D1%8F" title="Концепция">концепцией</a> <a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8" title="Вики">вики</a>. IMDb выбрана как базовый источник информации для кинематографических ресурсов <a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F" title="Википедия">Википедии</a>.
</p>
