<p><b>Розовый треугольник</b> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Pink triangle</span>, <a href="/wiki/%D0%9D%D0%B5%D0%BC%D0%B5%D1%86%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Немецкий язык">нем.</a> <span lang="de" style="font-style:italic;">Rosa Winkel</span>) — старейший и, наравне с <a href="/wiki/%D0%A0%D0%B0%D0%B4%D1%83%D0%B6%D0%BD%D1%8B%D0%B9_%D1%84%D0%BB%D0%B0%D0%B3" title="Радужный флаг">радужным флагом</a>, один из наиболее узнаваемых <a href="/wiki/%D0%A1%D0%B8%D0%BC%D0%B2%D0%BE%D0%BB%D0%B8%D0%BA%D0%B0_%D0%9B%D0%93%D0%91%D0%A2" title="Символика ЛГБТ">символов сообщества</a> <a class="mw-redirect" href="/wiki/%D0%9B%D0%B5%D1%81%D0%B1%D0%B8" title="Лесби">лесбиянок</a>, <a href="/wiki/%D0%93%D0%B5%D0%B9" title="Гей">геев</a>, <a class="mw-redirect" href="/wiki/%D0%91%D0%B8%D1%81%D0%B5%D0%BA%D1%81%D1%83%D0%B0%D0%BB" title="Бисексуал">бисексуалов</a> и <a href="/wiki/%D0%A2%D1%80%D0%B0%D0%BD%D1%81%D0%B3%D0%B5%D0%BD%D0%B4%D0%B5%D1%80%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Трансгендерность">трансгендерных людей</a> (<i>сокр. <a href="/wiki/%D0%9B%D0%93%D0%91%D0%A2-%D1%81%D0%BE%D0%BE%D0%B1%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="ЛГБТ-сообщество">ЛГБТ-сообщество</a></i>), а также <a href="/wiki/%D0%9B%D0%93%D0%91%D0%A2-%D0%B4%D0%B2%D0%B8%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5" title="ЛГБТ-движение">движения в поддержку прав человека в их отношении</a>.
</p><p>Нашивка в виде розового <a href="/wiki/%D0%9F%D1%80%D0%B0%D0%B2%D0%B8%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D1%82%D1%80%D0%B5%D1%83%D0%B3%D0%BE%D0%BB%D1%8C%D0%BD%D0%B8%D0%BA" title="Правильный треугольник">равностороннего треугольника</a> с вершиной, обращённой вниз, во времена <a class="mw-redirect" href="/wiki/%D0%A2%D1%80%D0%B5%D1%82%D0%B8%D0%B9_%D1%80%D0%B5%D0%B9%D1%85" title="Третий рейх">нацистской Германии</a> использовалась для <a href="/wiki/%D0%9F%D0%BE%D0%BC%D0%B5%D1%82%D0%BA%D0%B0_%D0%B7%D0%B0%D0%BA%D0%BB%D1%8E%D1%87%D1%91%D0%BD%D0%BD%D1%8B%D1%85_%D0%B2_%D0%BA%D0%BE%D0%BD%D1%86%D0%B5%D0%BD%D1%82%D1%80%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D1%8B%D1%85_%D0%BB%D0%B0%D0%B3%D0%B5%D1%80%D1%8F%D1%85" title="Пометка заключённых в концентрационных лагерях">идентификации в концлагерях</a> мужчин <a href="/wiki/%D0%93%D0%BE%D0%BC%D0%BE%D1%81%D0%B5%D0%BA%D1%81%D1%83%D0%B0%D0%BB%D1%8C%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Гомосексуальность">гомосексуальной ориентации</a>. Секс между мужчинами считался преступлением и <a class="mw-redirect" href="/wiki/%D0%A1%D0%B5%D0%BA%D1%81%D1%83%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B5_%D0%BC%D0%B5%D0%BD%D1%8C%D1%88%D0%B8%D0%BD%D1%81%D1%82%D0%B2%D0%B0_%D0%B2_%D0%A2%D1%80%D0%B5%D1%82%D1%8C%D0%B5%D0%BC_%D1%80%D0%B5%D0%B9%D1%85%D0%B5" title="Сексуальные меньшинства в Третьем рейхе">уголовно преследовался</a> в Третьем рейхе. Через несколько десятилетий новое поколение геев и лесбиянок стало использовать этот символ для увековечивания памяти о трагическом прошлом, манифестации борьбы за права человека и выражения надежды на новую эру свободы, <a href="/wiki/%D0%9A%D0%B0%D0%BC%D0%B8%D0%BD%D0%B3-%D0%B0%D1%83%D1%82" title="Каминг-аут">открытости</a> и <a class="mw-redirect" href="/wiki/%D0%93%D0%B5%D0%B9-%D0%BF%D1%80%D0%B0%D0%B9%D0%B4_(%D1%82%D0%B5%D1%80%D0%BC%D0%B8%D0%BD)" title="Гей-прайд (термин)">гордости</a><sup class="reference" id="cite_ref-Эн_1-0"><a href="#cite_note-Эн-1">[1]</a></sup><sup class="reference" id="cite_ref-Энц_2-0"><a href="#cite_note-Энц-2">[2]</a></sup>.
</p>

<h2><span id=".D0.92_.D0.BD.D0.B0.D1.86.D0.B8.D1.81.D1.82.D1.81.D0.BA.D0.BE.D0.B9_.D0.93.D0.B5.D1.80.D0.BC.D0.B0.D0.BD.D0.B8.D0.B8"></span><span class="mw-headline" id="В_нацистской_Германии">В нацистской Германии</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A0%D0%BE%D0%B7%D0%BE%D0%B2%D1%8B%D0%B9_%D1%82%D1%80%D0%B5%D1%83%D0%B3%D0%BE%D0%BB%D1%8C%D0%BD%D0%B8%D0%BA&amp;veaction=edit&amp;section=1" title="Редактировать раздел «В нацистской Германии»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A0%D0%BE%D0%B7%D0%BE%D0%B2%D1%8B%D0%B9_%D1%82%D1%80%D0%B5%D1%83%D0%B3%D0%BE%D0%BB%D1%8C%D0%BD%D0%B8%D0%BA&amp;action=edit&amp;section=1" title="Редактировать код раздела «В нацистской Германии»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="hatnote navigation-not-searchable">Основная статья: <b><a class="mw-redirect" href="/wiki/%D0%93%D0%BE%D0%BC%D0%BE%D1%81%D0%B5%D0%BA%D1%81%D1%83%D0%B0%D0%BB%D1%8C%D0%BD%D0%BE%D1%81%D1%82%D1%8C_%D0%B2_%D0%A2%D1%80%D0%B5%D1%82%D1%8C%D0%B5%D0%BC_%D1%80%D0%B5%D0%B9%D1%85%D0%B5" title="Гомосексуальность в Третьем рейхе">Гомосексуальность в Третьем рейхе</a></b></div>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Prisoners_in_the_concentration_camp_at_Sachsenhausen,_Germany,_12-19-1938_-_NARA_-_540175_(cleanup).jpg"><img class="mw-file-element" data-file-height="1971" data-file-width="2929" decoding="async" height="168" src="//upload.wikimedia.org/wikipedia/commons/thumb/5/54/Prisoners_in_the_concentration_camp_at_Sachsenhausen%2C_Germany%2C_12-19-1938_-_NARA_-_540175_%28cleanup%29.jpg/250px-Prisoners_in_the_concentration_camp_at_Sachsenhausen%2C_Germany%2C_12-19-1938_-_NARA_-_540175_%28cleanup%29.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/5/54/Prisoners_in_the_concentration_camp_at_Sachsenhausen%2C_Germany%2C_12-19-1938_-_NARA_-_540175_%28cleanup%29.jpg/375px-Prisoners_in_the_concentration_camp_at_Sachsenhausen%2C_Germany%2C_12-19-1938_-_NARA_-_540175_%28cleanup%29.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/5/54/Prisoners_in_the_concentration_camp_at_Sachsenhausen%2C_Germany%2C_12-19-1938_-_NARA_-_540175_%28cleanup%29.jpg/500px-Prisoners_in_the_concentration_camp_at_Sachsenhausen%2C_Germany%2C_12-19-1938_-_NARA_-_540175_%28cleanup%29.jpg 2x" width="250"/></a><figcaption>Узники-гомосексуалы <a href="/wiki/%D0%97%D0%B0%D0%BA%D1%81%D0%B5%D0%BD%D1%85%D0%B0%D1%83%D0%B7%D0%B5%D0%BD_(%D0%BA%D0%BE%D0%BD%D1%86%D0%B5%D0%BD%D1%82%D1%80%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D1%8B%D0%B9_%D0%BB%D0%B0%D0%B3%D0%B5%D1%80%D1%8C)" title="Заксенхаузен (концентрационный лагерь)">концлагеря Заксенхаузен</a> с розовыми треугольниками на груди.</figcaption></figure>
<p>В конце 1930-х годов правительство нацистской Германии приступило к реализации программы «чистки» немецкого общества от «нежелательных элементов». Начались массовые аресты представителей различных групп и отправка их без суда и следствия в <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%BD%D1%86%D0%B5%D0%BD%D1%82%D1%80%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D1%8B%D0%B5_%D0%BB%D0%B0%D0%B3%D0%B5%D1%80%D1%8F_%D0%A2%D1%80%D0%B5%D1%82%D1%8C%D0%B5%D0%B3%D0%BE_%D1%80%D0%B5%D0%B9%D1%85%D0%B0" title="Концентрационные лагеря Третьего рейха">концлагеря</a> под так называемый <a href="/wiki/%D0%97%D0%B0%D1%89%D0%B8%D1%82%D0%BD%D1%8B%D0%B9_%D0%B0%D1%80%D0%B5%D1%81%D1%82" title="Защитный арест">защитный арест</a>, где была разработана <i><a href="/wiki/%D0%9F%D0%BE%D0%BC%D0%B5%D1%82%D0%BA%D0%B0_%D0%B7%D0%B0%D0%BA%D0%BB%D1%8E%D1%87%D1%91%D0%BD%D0%BD%D1%8B%D1%85_%D0%B2_%D0%BA%D0%BE%D0%BD%D1%86%D0%B5%D0%BD%D1%82%D1%80%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D1%8B%D1%85_%D0%BB%D0%B0%D0%B3%D0%B5%D1%80%D1%8F%D1%85" title="Пометка заключённых в концентрационных лагерях">специальная система меток для заключённых</a></i>. На форму узников делались нашивки в виде равностороннего треугольника («винкеля»), цвет которого служил для указания «категории». Красный треугольник носили <a href="/wiki/%D0%9F%D0%BE%D0%BB%D0%B8%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B7%D0%B0%D0%BA%D0%BB%D1%8E%D1%87%D1%91%D0%BD%D0%BD%D1%8B%D0%B9" title="Политический заключённый">политические заключённые</a>, жёлтый (или <a href="/wiki/%D0%96%D1%91%D0%BB%D1%82%D0%B0%D1%8F_%D0%B7%D0%B2%D0%B5%D0%B7%D0%B4%D0%B0" title="Жёлтая звезда">жёлтую звезду Давида</a>) — евреи, зелёный — уголовники, коричневый — цыгане, <a href="/wiki/%D0%A7%D1%91%D1%80%D0%BD%D1%8B%D0%B9_%D1%82%D1%80%D0%B5%D1%83%D0%B3%D0%BE%D0%BB%D1%8C%D0%BD%D0%B8%D0%BA" title="Чёрный треугольник">чёрный</a> — «асоциальные элементы»<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>, <a class="mw-redirect" href="/wiki/%D0%A1%D0%B2%D0%B8%D0%B4%D0%B5%D1%82%D0%B5%D0%BB%D0%B8_%D0%98%D0%B5%D0%B3%D0%BE%D0%B2%D1%8B_%D0%B2_%D0%A2%D1%80%D0%B5%D1%82%D1%8C%D0%B5%D0%BC_%D1%80%D0%B5%D0%B9%D1%85%D0%B5" title="Свидетели Иеговы в Третьем рейхе">лиловый</a> — религиозные меньшинства<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>, синий — лица, пытавшиеся незаконно эмигрировать из страны, а розовый — гомосексуальные мужчины<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>.
</p>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Kennzeichen_f%C3%BCr_Schutzh%C3%A4ftlinge_in_den_Konzentrationslagern.jpg"><img class="mw-file-element" data-file-height="526" data-file-width="351" decoding="async" height="255" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/92/Kennzeichen_f%C3%BCr_Schutzh%C3%A4ftlinge_in_den_Konzentrationslagern.jpg/170px-Kennzeichen_f%C3%BCr_Schutzh%C3%A4ftlinge_in_den_Konzentrationslagern.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/92/Kennzeichen_f%C3%BCr_Schutzh%C3%A4ftlinge_in_den_Konzentrationslagern.jpg/255px-Kennzeichen_f%C3%BCr_Schutzh%C3%A4ftlinge_in_den_Konzentrationslagern.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/9/92/Kennzeichen_f%C3%BCr_Schutzh%C3%A4ftlinge_in_den_Konzentrationslagern.jpg/340px-Kennzeichen_f%C3%BCr_Schutzh%C3%A4ftlinge_in_den_Konzentrationslagern.jpg 2x" width="170"/></a><figcaption>Памятка для СС. <a href="/wiki/%D0%9F%D0%BE%D0%BC%D0%B5%D1%82%D0%BA%D0%B0_%D0%B7%D0%B0%D0%BA%D0%BB%D1%8E%D1%87%D1%91%D0%BD%D0%BD%D1%8B%D1%85_%D0%B2_%D0%BA%D0%BE%D0%BD%D1%86%D0%B5%D0%BD%D1%82%D1%80%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D1%8B%D1%85_%D0%BB%D0%B0%D0%B3%D0%B5%D1%80%D1%8F%D1%85" title="Пометка заключённых в концентрационных лагерях">Пометка заключённых в концлагерях</a></figcaption></figure>
<p>Уголовное преследование гомосексуалов в Германии не было нацистским изобретением, однако если до 1935 года <a href="/wiki/%D0%9F%D0%B0%D1%80%D0%B0%D0%B3%D1%80%D0%B0%D1%84_175" title="Параграф 175">параграф 175</a> провозглашал ответственность преимущественно за <a href="/wiki/%D0%90%D0%BD%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D1%81%D0%B5%D0%BA%D1%81" title="Анальный секс">анальный секс</a>, то после реформы Уголовного кодекса заключению подлежали все мужчины, которые были заподозрены в гомосексуальных отношениях из-за поцелуев, объятий или даже каких-то намёков<sup class="reference" id="cite_ref-Эн_1-1"><a href="#cite_note-Эн-1">[1]</a></sup><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>.
</p><p>Между 1933 и 1944 годами по 175-му параграфу в Третьем рейхе было осуждено около 50 тысяч мужчин<sup class="reference" id="cite_ref-Хо_7-0"><a href="#cite_note-Хо-7">[7]</a></sup> (в том числе около четырёх тысяч подростков)<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>, многие из которых были отправлены в тюрьмы. Также от 5 до 15 тысяч мужчин были депортированы в концентрационные лагеря. Неизвестно точно, сколько гомосексуалов погибло, но по данным немецкого историка и социолога <a href="/wiki/%D0%9B%D0%B0%D1%83%D1%82%D0%BC%D0%B0%D0%BD,_%D0%A0%D1%8E%D0%B4%D0%B8%D0%B3%D0%B5%D1%80" title="Лаутман, Рюдигер">Рюдигера Лаутмана</a>, уровень смертности осужденных по 175-му параграфу в лагерях был самым высоким и доходил до шестидесяти процентов (для сравнения: среди политзаключенных — 41 %)<sup class="reference" id="cite_ref-Хо_7-1"><a href="#cite_note-Хо-7">[7]</a></sup>. На форму мужчин-гомосексуалов помещались розовые треугольники, эта маркировка выделяла их в одну из самых низких групп в <a href="/wiki/%D0%98%D0%B5%D1%80%D0%B0%D1%80%D1%85%D0%B8%D1%8F" title="Иерархия">иерархии</a> лагерной социальной системы: с ними жестоко обращались не только охранники и администрация, но и другие заключённые<sup class="reference" id="cite_ref-Эн_1-2"><a href="#cite_note-Эн-1">[1]</a></sup>.
</p><p>Лесбиянки не подвергались систематическому преследованию со стороны нацистов, поскольку, по мнению фашистов, не представляли серьёзной угрозы для общества. Женщины не могли осуждаться по 175 параграфу, однако известно, что некоторые из них отправлялись в концлагеря как «асоциальные» и носили чёрные треугольники<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>.
</p><p>Репрессии против геев и лесбиянок в нацистской Германии считаются одной из частей <a href="/wiki/%D0%A5%D0%BE%D0%BB%D0%BE%D0%BA%D0%BE%D1%81%D1%82" title="Холокост">Холокоста</a>. Однако закон, по которому осуждались геи, просуществовал ещё долгие годы после победы над <a class="mw-redirect" href="/wiki/%D0%93%D0%B8%D1%82%D0%BB%D0%B5%D1%80" title="Гитлер">Гитлером</a>. С незначительными изменениями он оставался действовать и после окончания <a href="/wiki/%D0%92%D1%82%D0%BE%D1%80%D0%B0%D1%8F_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Вторая мировая война">войны</a> ещё в течение 19 лет в <a class="mw-redirect" href="/wiki/%D0%93%D0%94%D0%A0" title="ГДР">ГДР</a> и в течение 20 лет в <a class="mw-redirect" href="/wiki/%D0%A4%D0%A0%D0%93" title="ФРГ">ФРГ</a> (см. <a href="/wiki/%D0%9F%D0%B0%D1%80%D0%B0%D0%B3%D1%80%D0%B0%D1%84_175" title="Параграф 175">Параграф 175</a>). <a class="mw-redirect" href="/wiki/%D0%91%D1%80%D0%B0%D0%B7%D0%B4%D0%B0,_%D0%A0%D1%83%D0%B4%D0%BE%D0%BB%D1%8C%D1%84" title="Бразда, Рудольф">Рудольф Бразда</a>, последний из известных переживших нацистские концентрационные лагеря геев, умер во сне 3 августа 2011 года в возрасте 98 лет<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>. Преследование гомосексуалов в Третьем рейхе отражено в постоянной экспозиции <a href="/wiki/%D0%91%D0%B5%D1%80%D0%BB%D0%B8%D0%BD" title="Берлин">берлинского</a> «<a href="/wiki/%D0%9C%D1%83%D0%B7%D0%B5%D0%B9_%D0%B3%D0%BE%D0%BC%D0%BE%D1%81%D0%B5%D0%BA%D1%81%D1%83%D0%B0%D0%BB%D1%8C%D0%BD%D0%BE%D1%81%D1%82%D0%B8" title="Музей гомосексуальности">Музея гомосексуальности</a>».
</p>

<p>В начале 1970-х годов <a class="mw-redirect" href="/wiki/%D0%94%D0%B2%D0%B8%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5_%D0%B7%D0%B0_%D0%BF%D1%80%D0%B0%D0%B2%D0%B0_%D1%81%D0%B5%D0%BA%D1%81%D1%83%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D1%85_%D0%B8_%D0%B3%D0%B5%D0%BD%D0%B4%D0%B5%D1%80%D0%BD%D1%8B%D1%85_%D0%BC%D0%B5%D0%BD%D1%8C%D1%88%D0%B8%D0%BD%D1%81%D1%82%D0%B2" title="Движение за права сексуальных и гендерных меньшинств">организации по защите прав человека в отношении ЛГБТ-людей</a> в США и Германии начали кампанию по популяризации розового треугольника как символа движения.
</p><p>Так, в 1973 году западногерманская <a class="mw-redirect" href="/wiki/%D0%93%D0%B5%D0%B9-%D0%BE%D1%81%D0%B2%D0%BE%D0%B1%D0%BE%D0%B4%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D0%BE%D0%B5_%D0%B4%D0%B2%D0%B8%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5" title="Гей-освободительное движение">гей-освободительная</a> организация <i><a href="/wiki/Homosexuelle_Aktion_Westberlin" title="Homosexuelle Aktion Westberlin">Homosexuelle Aktion Westberlin</a></i> (HAW) призвала геев носить розовый треугольник в память о жертвах прошлого и в знак протеста против продолжающейся <a href="/wiki/%D0%94%D0%B8%D1%81%D0%BA%D1%80%D0%B8%D0%BC%D0%B8%D0%BD%D0%B0%D1%86%D0%B8%D1%8F" title="Дискриминация">дискриминации</a>, предупреждая однако, что это может сделать их мишенью агрессии <a href="/wiki/%D0%93%D0%BE%D0%BC%D0%BE%D1%84%D0%BE%D0%B1%D0%B8%D1%8F" title="Гомофобия">гомофобов</a>. Двумя годами позже гей-журнал <i>HAW-Info</i> повторил этот манифест. В 1976 году вышел фильм «Розовый треугольник? Это был такой давно…» (<a href="/wiki/%D0%9D%D0%B5%D0%BC%D0%B5%D1%86%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Немецкий язык">нем.</a> <span lang="de" style="font-style:italic;">Rosa Winkel? Das ist doch schon lange vorbei</span>) режиссёров Петера Рехта, Детлефа Штоффеля и Кристианы Шмерль, который вновь поднял данный вопрос, а кроме того задокументировал преследование мужчин-гомосексуалов. Гей-журнал <i>Rosa Flieder</i> в 1980 году писал, что розовый треугольник — это не только память об истреблении геев в прошлом, но и напоминание о сохраняющемся угнетении и дискриминации в их отношении, поскольку параграф 175 ещё существовал в уголовном кодексе <a class="mw-redirect" href="/wiki/%D0%A4%D0%A0%D0%93" title="ФРГ">ФРГ</a>. В следующем номере журнал отметил, что полиция в некоторых регионах Германии по-прежнему составляет списки граждан гомосексуальной ориентации<sup class="reference" id="cite_ref-Эн_1-3"><a href="#cite_note-Эн-1">[1]</a></sup><sup class="reference" id="cite_ref-Энц_2-1"><a href="#cite_note-Энц-2">[2]</a></sup>.
</p><p>Американские гей-активисты 1970-х годов также использовали розовый треугольник как символ памяти. Журналы <i>Gay Sunshine</i> (в 1973 году) в Сан-Франциско и <i>The Body Politic</i> в Торонто (в 1974 году) опубликовали материалы про использование розового треугольника в <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%BD%D1%86%D0%B5%D0%BD%D1%82%D1%80%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D1%8B%D0%B5_%D0%BB%D0%B0%D0%B3%D0%B5%D1%80%D1%8F_%D0%A2%D1%80%D0%B5%D1%82%D1%8C%D0%B5%D0%B3%D0%BE_%D1%80%D0%B5%D0%B9%D1%85%D0%B0" title="Концентрационные лагеря Третьего рейха">нацистских концентрационных лагерях</a> и настоятельно призвали использовать этот символ в память о тех, кто подвергался преследованиям<sup class="reference" id="cite_ref-Эн_1-4"><a href="#cite_note-Эн-1">[1]</a></sup>. Розовый треугольник стал активно использоваться в <a class="mw-redirect" href="/wiki/%D0%93%D0%B5%D0%B9-%D0%BE%D1%81%D0%B2%D0%BE%D0%B1%D0%BE%D0%B4%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D0%BE%D0%B5_%D0%B4%D0%B2%D0%B8%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5" title="Гей-освободительное движение">гей-освободительном</a> движении. Благодаря этому когда какие-либо общественные деятели призывали к ограничению прав гомосексуалов, то они рисковали быть сравненными с нацистами. Например, розовый треугольник приобрёл особое политическое значение в 1974 году в <a href="/wiki/%D0%9D%D1%8C%D1%8E-%D0%99%D0%BE%D1%80%D0%BA" title="Нью-Йорк">Нью-Йорке</a> во время общественной дискуссии вокруг законопроекта о защите прав человека в отношении гомосексуалов. Ортодоксальные еврейские группы выступали против постановления. Гей-активисты устроили пикет, участники которого надели нарукавные повязки с розовым треугольником в знак того, что гомосексуальные мужчины наравне с евреями были жертвами нацистских концлагерей. В следующем году в передовой статье <a href="/wiki/The_New_York_Times" title="The New York Times">The New York Times</a> руководитель <a class="extiw" href="https://en.wikipedia.org/wiki/New_York_Civil_Liberties_Union" title="en:New York Civil Liberties Union"><span title="New York Civil Liberties Union — версия статьи «Нью-Йоркский Союз защиты гражданских свобод» на английском языке">Нью-Йоркского Союза защиты гражданских свобод</span></a><span class="noprint" style="font-style:normal; font-weight:normal;"> <span style="display:inline-block; font-size:80%; position:relative;"><span style="font-size:117.6%; margin-left:-0.43em; position:relative; top:0.28em;"> <small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></span><span class="link-ru metadata" style="margin-left:-0.43em; position:absolute; left:0; z-index:1; margin-top:-0.35em;"><span style="visibility:hidden; margin-right:0.099em;"> (</span><span data-interwiki-article="New York Civil Liberties Union" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%9D%D1%8C%D1%8E-%D0%99%D0%BE%D1%80%D0%BA%D1%81%D0%BA%D0%B8%D0%B9_%D0%A1%D0%BE%D1%8E%D0%B7_%D0%B7%D0%B0%D1%89%D0%B8%D1%82%D1%8B_%D0%B3%D1%80%D0%B0%D0%B6%D0%B4%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D1%85_%D1%81%D0%B2%D0%BE%D0%B1%D0%BE%D0%B4&amp;action=edit&amp;redlink=1" title="Нью-Йоркский Союз защиты гражданских свобод (страница отсутствует)">рус.</a></span></span></span></span> <a class="extiw" href="https://en.wikipedia.org/wiki/Ira_Glasser" title="en:Ira Glasser"><span title="Ira Glasser — версия статьи «Глассер, Айра» на английском языке">Айра Глассер</span></a><span class="noprint" style="font-style:normal; font-weight:normal;"> <span style="display:inline-block; font-size:80%; position:relative;"><span style="font-size:117.6%; margin-left:-0.43em; position:relative; top:0.28em;"> <small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></span><span class="link-ru metadata" style="margin-left:-0.43em; position:absolute; left:0; z-index:1; margin-top:-0.35em;"><span style="visibility:hidden; margin-right:0.099em;"> (</span><span data-interwiki-article="Ira Glasser" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%93%D0%BB%D0%B0%D1%81%D1%81%D0%B5%D1%80,_%D0%90%D0%B9%D1%80%D0%B0&amp;action=edit&amp;redlink=1" title="Глассер, Айра (страница отсутствует)">рус.</a></span></span></span></span> призвал всех читателей носить розовый треугольник, чтобы высказать поддержку готовящемуся законопроекту, который был призван прекратить дискриминацию гомосексуалов в сферах занятости, жилья и общественных заведений. Касаясь вопроса о преследовании геев в нацистской Германии, Глассер писал: «Многие знают о <a href="/wiki/%D0%96%D1%91%D0%BB%D1%82%D0%B0%D1%8F_%D0%B7%D0%B2%D0%B5%D0%B7%D0%B4%D0%B0" title="Жёлтая звезда">жёлтой звезде</a>, но розовый треугольник всё ещё погребён как некая историческая тайна»<sup class="reference" id="cite_ref-Энц_2-2"><a href="#cite_note-Энц-2">[2]</a></sup>.
</p><p>В начале 1980-х многие отдельные мужчины-геи и группы начали использовать символ розового треугольника в его современном восприятии — как знак принадлежности к ЛГБТ-сообществу. Так как этот символ не был повсеместно известен, то треугольник стал играть роль своего рода тайной отметки, по которой можно было распознавать других геев и лесбиянок, когда они размещали его на бампере автомобиля или в виде нагрудного значка. Треугольник служил напоминанием о прошлом и текущем угнетении, данью памяти тем, кто погиб в лагерях, и тем, кто умирал от новой болезни — <a class="mw-redirect" href="/wiki/%D0%A1%D0%9F%D0%98%D0%94" title="СПИД">СПИДа</a><sup class="reference" id="cite_ref-Эн_1-5"><a href="#cite_note-Эн-1">[1]</a></sup><sup class="reference" id="cite_ref-Энц_2-3"><a href="#cite_note-Энц-2">[2]</a></sup>.
</p><p>Поскольку 1980-е годы в значительной мере прошли для ЛГБТ-сообщества под знаком начавшейся эпидемии <a class="mw-redirect" href="/wiki/%D0%A1%D0%9F%D0%98%D0%94" title="СПИД">СПИДа</a>, поначалу особенно затронувшей геев, — многие гей-активисты направили свои действия на борьбу с распространением болезни, в том числе на законодательные и политические изменения. Зародившаяся в нью-йоркском социальном гей-центре организация <a href="/wiki/ACT_UP" title="ACT UP">ACT UP</a> начала использовать розовый треугольник в изменённом виде — с вершиной, обращённой вверх, — как призыв к «активному отпору, а не пассивной покорности судьбе», в качестве символа необходимости активной борьбы со СПИДом и продолжающимся угнетением ЛГБТ-людей. Розовый треугольник часто появлялся над девизом «Молчание = Смерть». В своем манифесте активисты проводили параллели между нацистским периодом и кризисом СПИДа, заявляя, что «молчание об угнетении и уничтожении геев тогда и сейчас должно быть разбито в целях нашего выживания». Лозунг таким образом протестовал против <a href="/wiki/%D0%A2%D0%B0%D0%B1%D1%83" title="Табу">табу</a> на обсуждение безопасного секса и нежелания некоторых противостоять социальной несправедливости и безразличию правительства <a href="/wiki/%D0%A0%D0%B5%D0%B9%D0%B3%D0%B0%D0%BD,_%D0%A0%D0%BE%D0%BD%D0%B0%D0%BB%D1%8C%D0%B4" title="Рейган, Рональд">Рейгана</a><sup class="reference" id="cite_ref-Энц_2-4"><a href="#cite_note-Энц-2">[2]</a></sup><sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup>.
</p><p>В России розовый треугольник в 1990-е гг. дал название центру «Треугольник» — одной из первых организаций геев, лесбиянок и бисексуалов, среди руководителей которой были <a href="/wiki/%D0%94%D0%B5%D0%B1%D1%80%D1%8F%D0%BD%D1%81%D0%BA%D0%B0%D1%8F,_%D0%95%D0%B2%D0%B3%D0%B5%D0%BD%D0%B8%D1%8F_%D0%95%D0%B2%D0%B3%D0%B5%D0%BD%D1%8C%D0%B5%D0%B2%D0%BD%D0%B0" title="Дебрянская, Евгения Евгеньевна">Евгения Дебрянская</a> и <a class="mw-redirect" href="/wiki/%D0%9C%D0%B0%D1%88%D0%B0_%D0%93%D0%B5%D1%81%D1%81%D0%B5%D0%BD" title="Маша Гессен">Маша Гессен</a><sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>.
</p>
<h3><span id=".D0.9F.D1.80.D0.BE.D1.82.D0.B8.D0.B2.D0.BE.D1.80.D0.B5.D1.87.D0.B8.D1.8F"></span><span class="mw-headline" id="Противоречия">Противоречия</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A0%D0%BE%D0%B7%D0%BE%D0%B2%D1%8B%D0%B9_%D1%82%D1%80%D0%B5%D1%83%D0%B3%D0%BE%D0%BB%D1%8C%D0%BD%D0%B8%D0%BA&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Противоречия»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A0%D0%BE%D0%B7%D0%BE%D0%B2%D1%8B%D0%B9_%D1%82%D1%80%D0%B5%D1%83%D0%B3%D0%BE%D0%BB%D1%8C%D0%BD%D0%B8%D0%BA&amp;action=edit&amp;section=4" title="Редактировать код раздела «Противоречия»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Flags_and_Trees_(7889846002).jpg"><img class="mw-file-element" data-file-height="1024" data-file-width="768" decoding="async" height="333" src="//upload.wikimedia.org/wikipedia/commons/thumb/2/20/Flags_and_Trees_%287889846002%29.jpg/250px-Flags_and_Trees_%287889846002%29.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/2/20/Flags_and_Trees_%287889846002%29.jpg/375px-Flags_and_Trees_%287889846002%29.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/2/20/Flags_and_Trees_%287889846002%29.jpg/500px-Flags_and_Trees_%287889846002%29.jpg 2x" width="250"/></a><figcaption>Праздничное украшение улицы</figcaption></figure>
<p>Розовый треугольник имеет яркий и праздничный вид, в результате чего он вскоре стал использоваться для маркировки принадлежности к ЛГБТ-сообществу в очень широком спектре: его изображения помещаются на запонки, серьги, значки, брелоки, плакаты. Сам <a href="/wiki/%D0%A0%D0%BE%D0%B7%D0%BE%D0%B2%D1%8B%D0%B9_%D1%86%D0%B2%D0%B5%D1%82" title="Розовый цвет">розовый цвет</a> начал ассоциироваться с гомосексуальностью. С точки зрения популярности как <a class="mw-redirect" href="/wiki/%D0%A1%D0%B8%D0%BC%D0%B2%D0%BE%D0%BB%D0%B8%D0%BA%D0%B0_%D0%9B%D0%93%D0%91%D0%A2-%D0%B4%D0%B2%D0%B8%D0%B6%D0%B5%D0%BD%D0%B8%D1%8F" title="Символика ЛГБТ-движения">символа гомосексуальности</a> он соперничает только с <a href="/wiki/%D0%A0%D0%B0%D0%B4%D1%83%D0%B6%D0%BD%D1%8B%D0%B9_%D1%84%D0%BB%D0%B0%D0%B3" title="Радужный флаг">радужным флагом</a>.
</p><p>Однако некоторые геи и лесбиянки, особенно еврейского происхождения, ставят под сомнение целесообразность столь широкого использования нацистского знака. Другие считают, что использование розового треугольника в качестве символа <a class="mw-redirect" href="/wiki/%D0%93%D0%B5%D0%B9-%D0%BF%D1%80%D0%B0%D0%B9%D0%B4_(%D1%82%D0%B5%D1%80%D0%BC%D0%B8%D0%BD)" title="Гей-прайд (термин)">гордости</a> девальвирует память о страданиях гомосексуалов во времена нацизма, в возражение чему оппоненты апеллируют к продолжающейся борьбе против ненависти к геям и лесбиянкам, против дискриминационных в их отношении законов, к равнодушию правительства <a class="mw-redirect" href="/wiki/%D0%A0%D0%B5%D0%B9%D0%B3%D0%B0%D0%BD" title="Рейган">Рейгана</a> во время эпидемии СПИДа<sup class="reference" id="cite_ref-Эн_1-6"><a href="#cite_note-Эн-1">[1]</a></sup><sup class="reference" id="cite_ref-Энц_2-5"><a href="#cite_note-Энц-2">[2]</a></sup>. Как отмечал историк Эрик Н. Дженсен, «розовый треугольник выполнял множество задач: он объединял разнообразное сообщество гомосексуальных мужчин и женщин, мобилизовывал его политическую активность и служил интерпретирующей основой для современных событий»<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup>. Некоторые исследователи отмечают политический характер символа, противопоставляя его невнятной, с их точки зрения, концепции радужного флага, и связывают с этим падение его популярности на всё более коммерциализирующихся гей-мероприятиях<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup>.
</p>
<h3><span id=".D0.92.D0.B0.D1.80.D0.B8.D0.B0.D0.BD.D1.82.D1.8B"></span><span class="mw-headline" id="Варианты">Варианты</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A0%D0%BE%D0%B7%D0%BE%D0%B2%D1%8B%D0%B9_%D1%82%D1%80%D0%B5%D1%83%D0%B3%D0%BE%D0%BB%D1%8C%D0%BD%D0%B8%D0%BA&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Варианты»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A0%D0%BE%D0%B7%D0%BE%D0%B2%D1%8B%D0%B9_%D1%82%D1%80%D0%B5%D1%83%D0%B3%D0%BE%D0%BB%D1%8C%D0%BD%D0%B8%D0%BA&amp;action=edit&amp;section=5" title="Редактировать код раздела «Варианты»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Многие <a href="/wiki/%D0%9B%D0%B5%D1%81%D0%B1%D0%B8%D1%8F%D0%BD%D1%81%D1%82%D0%B2%D0%BE" title="Лесбиянство">лесбиянки</a> воспринимали розовый треугольник как недостаточно органичный для себя символ, так как в нацистских лагерях он был зна́ком только для гомосексуальных мужчин; лесбиянство само по себе в Рейхе уголовно не преследовалось, однако лесбиянки нередко квалифицировались как «асоциальные элементы» и носили <a href="/wiki/%D0%A7%D1%91%D1%80%D0%BD%D1%8B%D0%B9_%D1%82%D1%80%D0%B5%D1%83%D0%B3%D0%BE%D0%BB%D1%8C%D0%BD%D0%B8%D0%BA" title="Чёрный треугольник">чёрный треугольник</a>. Поэтому некоторые лесби-активистки в 1980-х годах стали использовать именно чёрный треугольник вместо розового, желая подчеркнуть часто приуменьшаемую роль женщин в гей-движении. По образцу розового треугольника созданы также аналогичные символы <a class="mw-redirect" href="/wiki/%D0%91%D0%B8%D1%81%D0%B5%D0%BA%D1%81%D1%83%D0%B0%D0%BB" title="Бисексуал">бисексуалов</a> и трансгендерных людей: первый представляет собой неполное наложение розового и синего треугольников, второй — розовый треугольник со вписанным в него особым символом трансгендерности (комбинацией символов Марса и Венеры, означающих мужское и женское начала)<sup class="reference" id="cite_ref-Энц_2-6"><a href="#cite_note-Энц-2">[2]</a></sup>.
</p>
<h3><span id=".D0.92_.D0.BA.D1.83.D0.BB.D1.8C.D1.82.D1.83.D1.80.D0.B5"></span><span class="mw-headline" id="В_культуре">В культуре</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A0%D0%BE%D0%B7%D0%BE%D0%B2%D1%8B%D0%B9_%D1%82%D1%80%D0%B5%D1%83%D0%B3%D0%BE%D0%BB%D1%8C%D0%BD%D0%B8%D0%BA&amp;veaction=edit&amp;section=6" title="Редактировать раздел «В культуре»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A0%D0%BE%D0%B7%D0%BE%D0%B2%D1%8B%D0%B9_%D1%82%D1%80%D0%B5%D1%83%D0%B3%D0%BE%D0%BB%D1%8C%D0%BD%D0%B8%D0%BA&amp;action=edit&amp;section=6" title="Редактировать код раздела «В культуре»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure class="mw-halign-left" typeof="mw:File"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Logo_International_Day_Against_Homophobia.jpg"><img class="mw-file-element" data-file-height="135" data-file-width="125" decoding="async" height="108" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/d2/Logo_International_Day_Against_Homophobia.jpg/100px-Logo_International_Day_Against_Homophobia.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/d/d2/Logo_International_Day_Against_Homophobia.jpg 1.5x" width="100"/></a><figcaption></figcaption></figure>
<p>Розовый треугольник является основой символа <a class="mw-redirect" href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D0%B4%D0%B5%D0%BD%D1%8C_%D0%B1%D0%BE%D1%80%D1%8C%D0%B1%D1%8B_%D1%81_%D0%B3%D0%BE%D0%BC%D0%BE%D1%84%D0%BE%D0%B1%D0%B8%D0%B5%D0%B9" title="Международный день борьбы с гомофобией">Международного дня борьбы с гомофобией</a>, а также других различных логотипов ЛГБТ-организаций и мероприятий.
</p><p>Мемориальное значение розового треугольника отражено в различных памятниках ЛГБТ-сообщества. Этот знак изображен на <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D0%BF%D0%B0%D0%BC%D1%8F%D1%82%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2_%D1%80%D0%B5%D0%BF%D1%80%D0%B5%D1%81%D1%81%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%BD%D1%8B%D0%BC_%D0%B3%D0%B5%D1%8F%D0%BC_%D0%B8_%D0%BB%D0%B5%D1%81%D0%B1%D0%B8%D1%8F%D0%BD%D0%BA%D0%B0%D0%BC" title="Список памятников репрессированным геям и лесбиянкам">мемориальных досках в память о нацистском геноциде</a> геев и лесбиянок (<a href="/wiki/%D0%9C%D0%B0%D1%83%D1%82%D1%85%D0%B0%D1%83%D0%B7%D0%B5%D0%BD_(%D0%BA%D0%BE%D0%BD%D1%86%D0%B5%D0%BD%D1%82%D1%80%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D1%8B%D0%B9_%D0%BB%D0%B0%D0%B3%D0%B5%D1%80%D1%8C)" title="Маутхаузен (концентрационный лагерь)">Маутхаузен</a> (1984), <a class="mw-redirect" href="/wiki/%D0%9D%D0%BE%D0%B9%D0%B5%D0%BD%D0%B3%D0%B0%D0%BC%D0%BC%D0%B5" title="Нойенгамме">Нойенгамме</a> (1985), <a href="/wiki/%D0%94%D0%B0%D1%85%D0%B0%D1%83_(%D0%BA%D0%BE%D0%BD%D1%86%D0%B5%D0%BD%D1%82%D1%80%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D1%8B%D0%B9_%D0%BB%D0%B0%D0%B3%D0%B5%D1%80%D1%8C)" title="Дахау (концентрационный лагерь)">Дахау</a> (1985), на <a href="/wiki/%D0%9D%D0%BE%D0%BB%D0%BB%D0%B5%D0%BD%D0%B4%D0%BE%D1%80%D1%84%D0%BF%D0%BB%D0%B0%D1%86" title="Ноллендорфплац">Ноллендорфплац</a> в Берлине (1989), в парке Чезарини в <a href="/wiki/%D0%91%D0%BE%D0%BB%D0%BE%D0%BD%D1%8C%D1%8F" title="Болонья">Болонье</a> (1990), <a class="mw-redirect" href="/wiki/%D0%A0%D0%B8%D1%81%D1%8C%D0%B5%D1%80%D0%B0-%D0%B4%D0%B8-%D0%A1%D0%B0%D0%BD-%D0%A1%D0%B0%D0%B1%D0%B1%D0%B0" title="Рисьера-ди-Сан-Сабба">Рисьера-ди-Сан-Сабба</a> (2005), <a href="/wiki/%D0%91%D1%83%D1%85%D0%B5%D0%BD%D0%B2%D0%B0%D0%BB%D1%8C%D0%B4" title="Бухенвальд">Бухенвальд</a> (2006)). Также розовый треугольник является центральной частью скульптуры <a href="/wiki/%D0%93%D0%BE%D0%BC%D0%BE%D0%BC%D0%BE%D0%BD%D1%83%D0%BC%D0%B5%D0%BD%D1%82" title="Гомомонумент">Гомомонумента</a> в <a репрессированным ЛГБТ-людям в <a href="/wiki/%D0%91%D0%B0%D1%80%D1%81%D0%B5%D0%BB%D0%BE%D0%BD%D0%B0" title="Барселона">Барселоне</a> (2011), памятника репрессированным ЛГБТ-людям в <a
