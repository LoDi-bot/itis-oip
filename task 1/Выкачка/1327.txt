<p><b>Bleach: Blade Battlers</b> <span style="font-weight: normal;">(<a href="/wiki/%D0%AF%D0%BF%D0%BE%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Японский язык">яп.</a> <span lang="ja" style="font-size: 110%; font-style: normal;">BLEACH ～ブレイド・バトラーズ～</span> <i>бурити бурэйдо батора:дзу</i>, также <b>Bleach: Blade Battles</b><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>)</span> — серия <a href="/wiki/%D0%9A%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%B8%D0%B3%D1%80%D0%B0" title="Компьютерная игра">компьютерных игр</a> в жанре <a href="/wiki/%D0%A4%D0%B0%D0%B9%D1%82%D0%B8%D0%BD%D0%B3" title="Файтинг">файтинга</a> по мотивам <a href="/wiki/%D0%9C%D0%B0%D0%BD%D0%B3%D0%B0" title="Манга">манги</a> и <a href="/wiki/%D0%90%D0%BD%D0%B8%D0%BC%D0%B5" title="Аниме">аниме</a> «<a href="/wiki/%D0%91%D0%BB%D0%B8%D1%87" title="Блич">Блич</a>». Так как <i>Blade Battlers</i> основана на конкретном произведении, герои, <a class="mw-redirect" href="/wiki/%D0%A1%D0%B5%D1%82%D1%82%D0%B8%D0%BD%D0%B3_(%D0%B8%D0%B3%D1%80%D1%8B)" title="Сеттинг (игры)">мир</a> и элементы <a href="/wiki/%D0%93%D0%B5%D0%B9%D0%BC%D0%BF%D0%BB%D0%B5%D0%B9" title="Геймплей">игрового процесса</a> заимствованы из первоисточника — манги. Игроку предлагается выбрать персонажа из числа героев «Блич» и побороться на арене с противником. Серия состоит из двух игр для <a href="/wiki/%D0%98%D0%B3%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%BF%D1%80%D0%B8%D1%81%D1%82%D0%B0%D0%B2%D0%BA%D0%B0" title="Игровая приставка">платформы</a> <a href="/wiki/PlayStation_2" title="PlayStation 2">PlayStation 2</a>: <i>Bleach: Blade Battlers</i> и <i>Bleach: Blade Battlers 2nd</i> <span style="font-weight: normal;">(<a href="/wiki/%D0%AF%D0%BF%D0%BE%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Японский язык">яп.</a> <span lang="ja" style="font-size: 110%; font-style: normal;">BLEACH ～ブレイド・バトラーズ2nd～</span>)</span>, которые были разработаны компанией <a class="new" href="/w/index.php?title=Racjin&amp;action=edit&amp;redlink=1" title="Racjin (страница отсутствует)">Racjin</a> и изданы <a class="mw-redirect" href="/wiki/Sony_Computer_Entertainment" title="Sony Computer Entertainment">Sony Computer Entertainment</a> (SCEI).<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup><sup class="reference" id="cite_ref-ign_4-0"><a href="#cite_note-ign-4">[4]</a></sup> Первая часть появилась в продаже <a href="/wiki/12_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="12 октября">12 октября</a> <a href="/wiki/2006_%D0%B3%D0%BE%D0%B4_%D0%B2_%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D1%8B%D1%85_%D0%B8%D0%B3%D1%80%D0%B0%D1%85" title="2006 год в компьютерных играх">2006 года</a>, вторая — <a href="/wiki/27_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="27 сентября">27 сентября</a> <a href="/wiki/2007_%D0%B3%D0%BE%D0%B4_%D0%B2_%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D1%8B%D1%85_%D0%B8%D0%B3%D1%80%D0%B0%D1%85" title="2007 год в компьютерных играх">2007 года</a>. Игры выходили только в <a href="/wiki/%D0%AF%D0%BF%D0%BE%D0%BD%D0%B8%D1%8F" title="Япония">Японии</a>, где пользовались коммерческим успехом. На другие языки не переводились.
</p><p>В <i>Blade Battlers</i> предусмотрено несколько режимов, которые позволяют играть как против компьютера, так и против <a href="/wiki/PvP" title="PvP">другого человека</a>. Обе игры поддерживают вплоть до четырёх игроков одновременно: четыре <a href="/wiki/%D0%94%D0%B6%D0%BE%D0%B9%D1%81%D1%82%D0%B8%D0%BA" title="Джойстик">джойстика</a> подключаются к приставке PlayStation 2 через <a class="mw-redirect mw-disambig" href="/wiki/%D0%9F%D0%B5%D1%80%D0%B5%D1%85%D0%BE%D0%B4%D0%BD%D0%B8%D0%BA" title="Переходник">адаптер</a>.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup> Другой особенностью серии является тот факт, что разработчики попытались удовлетворить вкусы как любителей файтингов, так и <a class="mw-disambig" href="/wiki/%D0%9D%D0%BE%D0%B2%D0%B8%D1%87%D0%BE%D0%BA" title="Новичок">новичков</a> в данном жанре, поэтому упростили боевую систему, исключив «<a class="new" href="/w/index.php?title=%D0%9A%D0%BE%D0%BC%D0%B1%D0%BE_(%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D1%8B%D0%B5_%D0%B8%D0%B3%D1%80%D1%8B)&amp;action=edit&amp;redlink=1" title="Комбо (компьютерные игры) (страница отсутствует)">комбо</a>» — сложные комбинации ударов, выполняющиеся путём совершения некоей последовательности действий (например, последовательного нажатия определённых кнопок).<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup><sup class="reference" id="cite_ref-juegos_7-0"><a href="#cite_note-juegos-7">[7]</a></sup>
</p>

<p>Основные персонажи аниме и манги «Блич», ставшие <a href="/wiki/%D0%98%D0%B3%D1%80%D0%BE%D0%B2%D0%BE%D0%B9_%D0%BF%D0%B5%D1%80%D1%81%D0%BE%D0%BD%D0%B0%D0%B6" title="Игровой персонаж">главными героями игры</a>, живут в вымышленном японском городе Каракура. <a href="/wiki/%D0%98%D1%87%D0%B8%D0%B3%D0%BE_%D0%9A%D1%83%D1%80%D0%BE%D1%81%D0%B0%D0%BA%D0%B8" title="Ичиго Куросаки">Ичиго Куросаки</a>, <a href="/wiki/%D0%9E%D1%80%D0%B8%D1%85%D0%B8%D0%BC%D1%8D_%D0%98%D0%BD%D0%BE%D1%83%D1%8D" title="Орихимэ Иноуэ">Орихимэ Иноуэ</a>, <a href="/wiki/%D0%AF%D1%81%D1%83%D1%82%D0%BE%D1%80%D0%B0_%D0%A1%D0%B0%D0%B4%D0%BE" title="Ясутора Садо">Чад</a> являются людьми, обретшими способность видеть злых духов — <i>«<a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D0%BF%D1%83%D1%81%D1%82%D1%8B%D1%85_%D0%B2_%C2%AB%D0%91%D0%BB%D0%B8%D1%87%C2%BB" title="Список пустых в «Блич»">пустых</a>»</i> — и сражаться с ними. <a href="/wiki/%D0%A0%D1%83%D0%BA%D0%B8%D1%8F_%D0%9A%D1%83%D1%87%D0%B8%D0%BA%D0%B8" title="Рукия Кучики">Рукия Кучики</a> — «<a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D0%BF%D1%80%D0%BE%D0%B2%D0%BE%D0%B4%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2_%D0%B4%D1%83%D1%88_%D0%B2_%C2%AB%D0%91%D0%BB%D0%B8%D1%87%C2%BB" title="Список проводников душ в «Блич»">проводник душ</a>», невидимый для обычных людей. Проводники душ патрулируют мир людей, сражаются с <i>пустыми</i> и сопровождают души умерших в загробный мир (<a href="/wiki/%D0%91%D0%BB%D0%B8%D1%87#Мир" title="Блич">Сообщество душ</a>)<sup class="reference" id="cite_ref-Блич-1-1-25_8-0"><a href="#cite_note-Блич-1-1-25-8">[8]</a></sup>. По стечению обстоятельств Рукия потеряла свою силу и также находится в человеческом теле в городе Каракура. Наконец, <a href="/wiki/%D0%A3%D1%80%D1%8E_%D0%98%D1%81%D0%B8%D0%B4%D0%B0" title="Урю Исида">Урю Исида</a> — последний выживший представитель расы <i><a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D0%BF%D0%B5%D1%80%D1%81%D0%BE%D0%BD%D0%B0%D0%B6%D0%B5%D0%B9_%C2%AB%D0%91%D0%BB%D0%B8%D1%87%C2%BB#Квинси" title="Список персонажей «Блич»">квинси</a></i>, в битве полагающийся на <a href="/wiki/%D0%9B%D1%83%D0%BA_(%D0%BE%D1%80%D1%83%D0%B6%D0%B8%D0%B5)" title="Лук (оружие)">лук</a> и <a class="mw-redirect" href="/wiki/%D0%A1%D1%82%D1%80%D0%B5%D0%BB%D1%8B" title="Стрелы">стрелы</a>. В первой части <i>Blade Battlers</i> Ичиго, Орихимэ, Чад, Рукия и Урю имеются в распоряжении игрока с самого начала. Чтобы получить доступ к остальным, необходимо выполнять игровые миссии. В числе персонажей, присутствующих в первой части, также находятся некоторые проводники душ из оригинальной манги и плюшевый лев <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D0%BF%D0%B5%D1%80%D1%81%D0%BE%D0%BD%D0%B0%D0%B6%D0%B5%D0%B9_%C2%AB%D0%91%D0%BB%D0%B8%D1%87%C2%BB#Кон" title="Список персонажей «Блич»">Кон</a>, второстепенный комический персонаж «Блич». В <i>Blade Battlers 2nd</i> остались те же герои и были добавлены новые проводники душ, арранкары (более сильные <i>пустые</i>, которые смогли частично приобрести способности проводников душ) и <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D0%BF%D0%B5%D1%80%D1%81%D0%BE%D0%BD%D0%B0%D0%B6%D0%B5%D0%B9_%C2%AB%D0%91%D0%BB%D0%B8%D1%87%C2%BB#Вайзарды" title="Список персонажей «Блич»">вайзарды</a> — проводники душ-отступники, запретными путями получившие силы <i>пустых</i>. В связи с тем, что боевые особенности арранкаров и вайзардов на тот момент были либо совсем не показаны в манге и аниме, либо продемонстрированы частично, разработчикам пришлось их изобрести.<sup class="reference" id="cite_ref-pixfans_9-0"><a href="#cite_note-pixfans-9">[9]</a></sup>

<p>Игрок управляет одним из персонажей «Блич» и сражается с другими героями. Воины достаточно сильно ограничены в комбинациях приёмов: имеются три базовые атаки различной мощности, которые меняются при отработке их в воздухе, и несколько вариантов тех же атак, использующихся при блокировании.<sup class="reference" id="cite_ref-pixfans_9-1"><a href="#cite_note-pixfans-9">[9]</a></sup> Все выполняются одинаковыми кнопками и их <a href="/wiki/%D0%A1%D0%BE%D1%87%D0%B5%D1%82%D0%B0%D0%BD%D0%B8%D0%B5_%D0%BA%D0%BB%D0%B0%D0%B2%D0%B8%D1%88" title="Сочетание клавиш">сочетаниями</a>.
</p><p>Все персонажи способны прыгать, парировать удары, выполнять <a class="extiw" href="https://ru.wiktionary.org/wiki/%D0%9F%D1%80%D0%B8%D0%BB%D0%BE%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5:%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D1%84%D0%B0%D0%B9%D1%82%D0%B8%D0%BD%D0%B3-%D1%82%D0%B5%D1%80%D0%BC%D0%B8%D0%BD%D0%BE%D0%B2#Dash" title="wikt:Приложение:Список файтинг-терминов">рывки</a> и командную атаку, если герои дерутся <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D0%B0%D1%8F_%D0%B8%D0%B3%D1%80%D0%B0_(%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D1%8B%D0%B5_%D0%B8%D0%B3%D1%80%D1%8B)" title="Кооперативная игра (компьютерные игры)">партией</a>.
</p><p><span class="mw-default-size" typeof="mw:File"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Bleach_Blade_Battlers_controls.jpg"><img class="mw-file-element" data-file-height="236" data-file-width="373" decoding="async" height="236" src="//upload.wikimedia.org/wikipedia/commons/9/92/Bleach_Blade_Battlers_controls.jpg" width="373"/></a></span>
</p><p>Скудность технического выбора компенсируется личными особенностями в стиле боя, присущими отдельным персонажам — у каждого свой строго индивидуальный набор способностей.<sup class="reference" id="cite_ref-pixfans_9-2"><a href="#cite_note-pixfans-9">[9]</a></sup> У каждого героя своё оружие, соответствующее его роли в «Блич»: <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D0%BF%D1%80%D0%BE%D0%B2%D0%BE%D0%B4%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2_%D0%B4%D1%83%D1%88_%D0%B2_%C2%AB%D0%91%D0%BB%D0%B8%D1%87%C2%BB" title="Список проводников душ в «Блич»">проводники душ</a> сражаются на мечах, представитель расы <i>квинси</i> <a href="/wiki/%D0%A3%D1%80%D1%8E_%D0%98%D1%81%D0%B8%D0%B4%D0%B0" title="Урю Исида">Урю Исида</a> стреляет из лука, <a href="/wiki/%D0%AF%D1%81%D1%83%D1%82%D0%BE%D1%80%D0%B0_%D0%A1%D0%B0%D0%B4%D0%BE" title="Ясутора Садо">Чад</a> (человек) бьётся врукопашную, и так далее. Проводники душ также владеют «мерцающими шагами» <span style="font-weight: normal;">(<a href="/wiki/%D0%AF%D0%BF%D0%BE%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Японский язык">яп.</a> <span lang="ja" style="font-size: 110%; font-style: normal;">瞬歩</span> <i>сюмпо, shunpo</i>)</span>, что позволяет почти моментально перемещаться на большие расстояния, а остальные персонажи вместо этой техники выполняют перекаты или <a href="/wiki/%D0%A2%D0%B5%D0%BB%D0%B5%D0%BF%D0%BE%D1%80%D1%82%D0%B0%D1%86%D0%B8%D1%8F" title="Телепортация">телепортируются</a> из одного места в другое. Командные атаки зависят от совместимости конкретных персонажей. Если они хорошо ладят в оригинальной манге и аниме, то, скорее всего, смогут совершить и командную атаку.
</p>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Bleach-_Blade_Battlers_(1).jpg"><img class="mw-file-element" data-file-height="299" data-file-width="400" decoding="async" height="284" src="//upload.wikimedia.org/wikipedia/ru/thumb/4/47/Bleach-_Blade_Battlers_%281%29.jpg/380px-Bleach-_Blade_Battlers_%281%29.jpg" srcset="//upload.wikimedia.org/wikipedia/ru/4/47/Bleach-_Blade_Battlers_%281%29.jpg 1.5x" width="380"/></a><figcaption>Битва между четырьмя персонажами в первой части <i>Blade Battlers</i>. На переднем плане <a href="/wiki/%D0%A0%D1%83%D0%BA%D0%B8%D1%8F_%D0%9A%D1%83%D1%87%D0%B8%D0%BA%D0%B8" title="Рукия Кучики">Рукия Кучики</a> (справа) наносит <a href="/wiki/%D0%A3%D1%80%D1%8E_%D0%98%D1%81%D0%B8%D0%B4%D0%B0" title="Урю Исида">Урю Исиде</a> удар льдом. Одеяние и шкала <i>рэйацу</i> Рукии, ставшая красной, свидетельствуют о том, что она находится в режиме «ярости».</figcaption></figure>
<p>У каждого персонажа есть <a class="mw-redirect" href="/wiki/%D0%9E%D1%87%D0%BA%D0%B8_%D0%B6%D0%B8%D0%B7%D0%BD%D0%B8" title="Очки жизни">линейка жизненной силы</a> и <a class="extiw" href="https://ru.wiktionary.org/wiki/%D0%9F%D1%80%D0%B8%D0%BB%D0%BE%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5:%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D1%84%D0%B0%D0%B9%D1%82%D0%B8%D0%BD%D0%B3-%D1%82%D0%B5%D1%80%D0%BC%D0%B8%D0%BD%D0%BE%D0%B2#Power_Gauge" title="wikt:Приложение:Список файтинг-терминов">энергетическая шкала</a> <i>«рэйацу»</i> <span style="font-weight: normal;">(<a href="/wiki/%D0%AF%D0%BF%D0%BE%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Японский язык">яп.</a> <span lang="ja" style="font-size: 110%; font-style: normal;">霊圧</span>, досл. «давление духа»)</span> — паранормальной <a class="mw-redirect" href="/wiki/%D0%9C%D0%B0%D0%BD%D0%B0_(%D0%B8%D0%B3%D1%80%D1%8B)" title="Мана (игры)">духовной энергии</a>, благодаря которой воины могут выполнять специальные приёмы. Шкала <i>рэйацу</i> (энергии) заполняется, если персонаж наносит или получает урон. С частично заполненной шкалой <i>рэйацу</i> можно исполнить быструю и мощную атаку, хотя такой удар противник в состоянии парировать, а с полной шкалой — активировать так называемый «EX». От этого суперудара, сопровождающегося специальной анимацией, можно только увернуться.
</p><p>В первой части игры, при наличии менее 50 % жизненной силы и полной шкалы <i>рэйацу</i>, игрок может активировать особое усиление способностей — «<a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D0%BF%D1%80%D0%BE%D0%B2%D0%BE%D0%B4%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2_%D0%B4%D1%83%D1%88_%D0%B2_%C2%AB%D0%91%D0%BB%D0%B8%D1%87%C2%BB#Характеристика" title="Список проводников душ в «Блич»">банкай</a>» у проводников душ, различные бонусы для остальных героев: высокая скорость передвижения или усиление наносимого урона. Такое состояние длится до тех пор, пока не опустеет шкала <i>рэйацу</i>, но лишь во время него можно активировать наиболее сильный вариант «EX»-удара. По истечении короткого промежутка времени сила героя становится прежней, а шкала <i>рэйацу</i> пустеет. Во второй части <i>Blade Battlers</i> требование менее 50 % жизненной силы было отменено, а вместо этого была введена дополнительная шкала «ярости», при заполнении которой появляется возможность активировать усиление. «Ярость», как и <i>рэйацу</i>, медленно растёт по мере нанесения или получения урона.
</p><p>Под шкалой <i>рэйацу</i> находятся небольшие оранжевые сферы — дополнительные жизни, автоматически расходуемые в том случае, когда линейка жизненной силы подходит к концу: одна из сфер пропадает и линейка заполняется снова. Чем больше сфер (обычно две, если игрок вручную не увеличит или не уменьшит их количество), тем дольше длится бой, но когда и они подходят к концу, герой терпит окончательное поражение.
</p><p>В ходе сражения <a href="/wiki/%D0%A0%D0%B0%D0%B7%D1%80%D1%83%D1%88%D0%B0%D0%B5%D0%BC%D0%BE%D0%B5_%D0%BE%D0%BA%D1%80%D1%83%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5" title="Разрушаемое окружение">разрушаются</a> некоторые объекты, из которых падают бонусные предметы: аптечка пополняет шкалу жизненных сил; синие конфеты пополняют <i>рэйацу</i>; жёлтые дают неограниченную <i>рэйацу</i> на короткий срок; бутылочки делают игрока невидимым. Другие предметы наносят вред: металлический шар катится по полю и сбивает персонажей с ног, ловушка парализует героев, бомба, взорвавшись, наносит ущерб всем находящимся поблизости героям. Дополнительно отнимать жизненную силу в бою могут взрывающиеся от случайного удара автомобили, падение с обрыва на некоторых локациях и другие элементы окружения игры.<sup class="reference" id="cite_ref-progames_11-0"><a href="#cite_note-progames-11">[11]</a></sup>
</p>

<h3><span class="mw-headline" id="Bleach:_Blade_Battlers"><i>Bleach: Blade Battlers</i></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Bleach:_Blade_Battlers&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Bleach: Blade Battlers»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Bleach:_Blade_Battlers&amp;action=edit&amp;section=4" title="Редактировать код раздела «Bleach: Blade Battlers»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>

<p><i>Bleach: Blade Battlers</i> — первая игра одноимённой серии. В ней два уровня сложности: лёгкий (стоит по умолчанию) и более сложный, так называемый «капитанский». Присутствуют 23 персонажа «Блич», но с начала игры доступны лишь пятеро: Ичиго Куросаки, Орихимэ Иноуэ, Чад, Рукия Кучики и Урю Исида, — остальных можно разблокировать, зарабатывая очки в режиме «Миссий». Для каждого героя требуется разное количество очков: от 3 баллов за <a href="/wiki/%D0%A0%D1%8D%D0%BD%D0%B4%D0%B7%D0%B8_%D0%90%D0%B1%D0%B0%D1%80%D0%B0%D0%B8" title="Рэндзи Абараи">Рэндзи Абараи</a> до наиболее «дорогостоящего» <a class="mw-redirect" href="/wiki/%D0%A1%D0%BE%D1%81%D1%83%D0%BA%D1%8D_%D0%90%D0%B9%D0%B4%D0%B7%D1%8D%D0%BD" title="Сосукэ Айдзэн">Сосукэ Айдзэна</a> (70 баллов).
</p><p>Игровые локации заимствованы из первоисточника. Главный герой, Ичиго Куросаки, живёт в вымышленном городе Каракура, где <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D0%BF%D0%B5%D1%80%D1%81%D0%BE%D0%BD%D0%B0%D0%B6%D0%B5%D0%B9_%C2%AB%D0%91%D0%BB%D0%B8%D1%87%C2%BB#Семья_Куросаки" title="Список персонажей «Блич»">его отец</a> владеет собственной клиникой. Соответственно, клиника Куросаки и школа Ичиго становятся в <i>Blade Battlers</i> полем битвы. Большинство локаций расположены в Сообществе душ, куда души людей отправляются после смерти. Там находятся казармы проводников, Лес мёртвых душ, столица Сообщества Сэйрэйтэй и так называемая «Арена Сокёку», на которой, согласно манге, проводятся казни проводников душ.
</p>
<h4><span id=".D0.98.D0.B3.D1.80.D0.BE.D0.B2.D1.8B.D0.B5_.D1.80.D0.B5.D0.B6.D0.B8.D0.BC.D1.8B"></span><span class="mw-headline" id="Игровые_режимы">Игровые режимы</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Bleach:_Blade_Battlers&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Игровые режимы»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Bleach:_Blade_Battlers&amp;action=edit&amp;section=5" title="Редактировать код раздела «Игровые режимы»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h4>
<ul><li>Режим истории — режим, в котором раскрывается сюжет игры. Он состоит из последовательности заранее заданных боёв, перемежаемых диалогами.</li>
<li>Режим <a class="mw-redirect" href="/wiki/%D0%A3%D1%80%D0%BE%D0%B2%D0%B5%D0%BD%D1%8C_(%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D1%8B%D0%B5_%D0%B8%D0%B3%D1%80%D1%8B)" title="Уровень (компьютерные игры)">миссий</a> включает «Обычные миссии» и «Миссии „охоты“». Выполняющий обыкновенные миссии игрок должен победить соперника, соблюдая разные условия: выиграть за ограниченное время, добить врага «EX»-ударом и так далее, при этом у противника могут быть дополнительные преимущества, например, бесконечная <i>рэйацу</i> или увеличенное количество оранжевых жизненных сфер. После каждой битвы игрок получает ранг D (самый низкий), C, B, A или <a class="extiw" href="https://ru.wiktionary.org/wiki/%D1%81%D1%83%D0%BF%D0%B5%D1%80-" title="wikt:супер-">S</a>, который доступен лишь на максимальном уровне сложности («капитанском»). Чем выше ранг, тем большее количество очков даётся в награду. По мере набора очков и выполнения миссий, открываются новые герои и дальнейшие миссии, общее число которым — 50. Для прохождения «обычных» миссий игрок может выбирать персонажа поддержки, управляемого <a class="mw-redirect" href="/wiki/CPU" title="CPU">CPU</a> или другим человеком. Миссии «охоты» — это битвы один на один. Проходя такие миссии, игрок получает новые приёмы для своего и других персонажей. Некоторые затем можно купить в магазине.<sup class="reference" id="cite_ref-progames_11-1"><a href="#cite_note-progames-11">[11]</a></sup> Чтобы победить, битва обязательно должна быть окончена суперприёмом.</li>
<li>Свободная битва — сражение любым персонажем один на один или бой в команде. Условия для битвы задаёт игрок.</li>
<li>Тренировка — <a href="/wiki/%D0%A3%D1%87%D0%B5%D0%B1%D0%BD%D0%B0%D1%8F_%D0%BC%D0%B8%D1%81%D1%81%D0%B8%D1%8F" title="Учебная миссия">обучающий режим</a>.</li>
<li>Магазин <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D0%BF%D0%B5%D1%80%D1%81%D0%BE%D0%BD%D0%B0%D0%B6%D0%B5%D0%B9_%C2%AB%D0%91%D0%BB%D0%B8%D1%87%C2%BB#Кискэ_Урахара" title="Список персонажей «Блич»">Урахары</a> — место, где за заработанные баллы можно купить видеоролики сюжетных сцен, секретные приёмы, юмористические ролики с участием плюшевого льва Кона и разблокировать новых персонажей. Урахара является персонажем в манге «Блич», где владеет собственным магазином товаров для проводников душ.</li></ul>

<table class="wikitable mw-collapsible ts-Оценки_игры" style="float:right;clear:right;width:23em;font-size:85%"><caption>Оценки</caption><tbody><tr><th colspan="2" scope="colgroup" style="background:#d1dbdf;font-size:112%">Иноязычные издания</th></tr><tr><th scope="col" style="background:#e8f4f8;text-align:center;vertical-align:middle">Издание</th><th scope="col" style="background:#e8f4f8;vertical-align:middle">Оценка</th></tr><tr><td style="text-align:center;vertical-align:middle"><a href="/wiki/GameSpot" title="GameSpot">GameSpot</a></td><td style="text-align:center;font-size:105%">8.9/10<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup></td></tr><tr><td style="text-align:center;vertical-align:middle"><a href="/wiki/GameTrailers" title="GameTrailers">GameTrailers</a></td><td style="text-align:center;font-size:105%">8.7/10<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup></td></tr><tr><td style="text-align:center;vertical-align:middle"><a class="new" href="/w/index.php?title=%D0%90%D0%BD%D0%B8%D0%9C%D0%B0%D0%B3&amp;action=edit&amp;redlink=1" title="АниМаг (страница отсутствует)">АниМаг</a></td><td style="text-align:center;font-size:105%">4.5/5<sup class="reference" id="cite_ref-animag_17-0"><a href="#cite_note-animag-17">[17]</a></sup></td></tr></tbody></table>
<p><i>Bleach: Blade Battlers 2nd</i> является прямым продолжением первой части, претерпевшим небольшие изменения. Основы боевой системы остались нетронутыми.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup> Появились новые игровые персонажи (теперь их стало 36<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup>) и новый игровой режим — «Battlers», были добавлены удары, действующие по области (<a href="/wiki/%D0%AF%D0%BF%D0%BE%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Японский язык">яп.</a> <span lang="ja" style="font-size: 110%; font-style: normal;">フィールドエフェクト</span>, «field effect»).<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup> Линейка <i>рэйацу</i> заполняется быстрее, что увеличивает темп игры.<sup class="reference" id="cite_ref-pixfans_9-3"><a href="#cite_note-pixfans-9">[9]</a></sup>
</p><p>«Миссии» были заменены на новый режим «Battlers». Игрок выбирает героя из манги «Блич» и, если желает, — персонажа поддержки для <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D0%B0%D1%8F_%D0%B8%D0%B3%D1%80%D0%B0_(%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D1%8B%D0%B5_%D0%B8%D0%B3%D1%80%D1%8B)" title="Кооперативная игра (компьютерные игры)">кооперативной игры</a>, затем перемещается на поле, заполненное клетками красного и голубого цвета в произвольном порядке. Двигаясь фигуркой персонажа по клеткам, игроки выбирают сражение и вступают в бой. В битвах на красной клетке игрок сам выбирает персонажа, а на голубых клетках нужно победить заранее заданным членом партии. Как в первой <i>Blade Battlers</i>, в некоторых миссиях поставлены дополнительные условия и без их соблюдения победу или вообще не засчитают, или засчитают с меньшим количеством очков.<sup class="reference" id="cite_ref-metodologic_21-0"><a href="#cite_note-metodologic-21">[21]</a></sup>
</p><p>Как и прежде, изначально доступно лишь несколько героев, но по мере набора очков в режиме «Battlers» автоматически открываются остальные.<sup class="reference" id="cite_ref-metodologic_21-1"><a href="/wiki/%D0%A0%D1%8D%D0%BD%D0%B4%D0%B7%D0%B8_%D0%90%D0%B1%D0%B0%D1%80%D0%B0%D0%B8" title="Рэндзи Абараи">Рэндзи Абараи</a>, <a title="Список проводников душ в «Блич»">Рангику Мацумото</a>, <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D0%BF%D1%80%D0%BE%D0%B2%D0%BE%D0%B4%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2_%D0%B4%D1%83%D1%88_%D0%B2_%C2%AB%D0%91%D0%BB%D0%B8%D1%87%C2%BB#Одиннадцатый_отряд" title="Список проводников душ в «Блич»">Мадарамэ Иккаку</a> и <a Джагерджак</a>, вайзарды <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D0%BF%D0%B5%D1%80%D1%81%D0%BE%D0%BD%D0%B0%D0%B6%D0%B5%D0%B9_%C2%AB%D0%91%D0%BB%D0%B8%D1%87%C2%BB#Вайзарды" title="Список персонажей «Блич»">Синдзи Хирако</a> и <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D0%BF%D0%B5%D1%80%D1%81%D0%BE%D0%BD%D0%B0%D0%B6%D0%B5%D0%B9_%C2%AB%D0%91%D0%BB%D0%B8%D1%87%C2%BB#Вайзарды" </p><p>Во <i>Blade Battlers 2nd</i> были добавлены новые арены: как в земном мире (крыша госпиталя), так и в Сообществе душ. Разработчики включили локации, связанные с появлением новых героев — вайзардов и <i>пустых</i>. <i>Пустые</i> большую часть времени находятся в мире <a href="/wiki/%D0%91%D0%BB%D0%B8%D1%87#Мир" title="Блич">Уэко Мундо</a>, пустынной области между миром людей и Сообществом душ. Секретное жилище вайзардов, где по сюжету «Блич» проходит тренировку Ичиго, находится в земном мире. Обе локации — серая пустыня Уэко Мундо и дом вайзардов — появились в игре. Кроме того, дополнительную арену представляет подсознание Ичиго, где происходит борьба с его отрицательным внутренним «я» — <a href="/wiki/%D0%98%D1%87%D0%B8%D0%B3%D0%BE_%D0%9A%D1%83%D1%80%D0%BE%D1%81%D0%B0%D0%BA%D0%B8#Силы_пустого" title="Ичиго Куросаки"><i>пустым</i> Ичиго</a>.
</p>
<h3><span id=".D0.A1.D1.8E.D0.B6.D0.B5.D1.82"></span><span class="mw-headline" id="Сюжет">Сюжет</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Bleach:_Blade_Battlers&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Сюжет»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Bleach:_Blade_Battlers&amp;action=edit&amp;section=7" title="Редактировать код раздела «Сюжет»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>В обеих играх в «Режиме истории» присутствует условный сюжет. «Режим истории» объединяет шесть коротких сценариев, не связанных с мангой или аниме. Они представляют собой чередование диалогов и боевых миссий: тренировки в специальном зале под магазином Урахары, защиту города Каракуры от <i>пустых</i> (тогда, соответственно, герои сражаются с <i>пустыми</i>), турнир на звание сильнейшего проводника душ (бои проводников душ друг с другом), отношения Ичиго с его отрицательным внутренним «я», интриги Женской ассоциации проводников душ и т. п. «Режим миссий» (режим «Battlers» во второй части) не имеет единого сценария, но более согласован с сюжетом «Блич»: сражения обосновываются событиями оригинальной манги и предваряются короткими высказываниями героев, обосновывающими необходимость схватки.
</p>

<p>Первая часть игровой серии <i>Blade Battlers</i> была анонсирована на официальном сайте компании SCEI <a href="/wiki/24_%D0%B8%D1%8E%D0%BB%D1%8F" title="24 июля">24 июля</a> <a href="/wiki/2006_%D0%B3%D0%BE%D0%B4_%D0%B2_%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D1%8B%D1%85_%D0%B8%D0%B3%D1%80%D0%B0%D1%85" title="2006 год в компьютерных играх">2006 года</a>, сразу после выхода <i><a class="mw-redirect" href="/wiki/Bleach:_Heat_the_Soul_3" title="Bleach: Heat the Soul 3">Bleach: Heat the Soul 3</a></i> от того же издателя. В октябрьском номере японского журнала <i><a href="/wiki/Shonen_Jump" title="Shonen Jump">Shonen Jump</a></i> были напечатаны <a class="mw-redirect" href="/wiki/%D0%A1%D0%BA%D1%80%D0%B8%D0%BD%D1%88%D0%BE%D1%82" title="Скриншот">скриншоты</a> из первой части,<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup> а сама игра вышла <a href="/wiki/12_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="12 октября">12 октября</a>. Также в октябре компанией <a href="/wiki/Shueisha" title="Shueisha">Shueisha</a> было издано руководство по игре <i>Blade Master Book</i> <span style="font-weight: normal;">(<a href="/wiki/%D0%AF%D0%BF%D0%BE%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Японский язык">яп.</a> <span lang="ja" style="font-size: 110%; font-style: normal;">版 ソニー・コンピュータエンタテインメントジャパン公式攻略本 BLEACH ブレイド・バトラーズ BLADE MASTER BOOK</span>)</span>, состоящее из 120 страниц.<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup>
</p><p><i>Blade Battlers 2nd</i>, очень похожая на свою предшественницу, была анонсирована <a href="/wiki/18_%D0%B8%D1%8E%D0%BD%D1%8F" title="18 июня">18 июня</a> <a href="/wiki/2007_%D0%B3%D0%BE%D0%B4_%D0%B2_%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D1%8B%D1%85_%D0%B8%D0%B3%D1%80%D0%B0%D1%85" title="2007 год в компьютерных играх">2007 года</a>.<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup> Тем же летом в июльском номере журнала <i><a href="/wiki/V_Jump" title="V Jump">V Jump</a></i> появились первые скриншоты.<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup> Вышла вторая часть <a href="/wiki/27_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="27 сентября">27 сентября</a><sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup>. В том же месяце было опубликовано руководство <i>Perfect Battlers Bible</i> <span style="font-weight: normal;">(<a href="/wiki/%D0%AF%D0%BF%D0%BE%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Японский язык">яп.</a> <span lang="ja" style="font-size: 110%; font-style: normal;">Bleach ブレイド・バトラーズ2nd パーフェクトバトラーズバイブル</span>)</span>.<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup> Наряду с <i><a class="mw-redirect" href="/wiki/Bleach:_Heat_the_Soul_4" title="Bleach: Heat the Soul 4">Bleach: Heat the Soul 4</a></i>, <i>Blade Battlers 2nd</i> была дополнительно представлена на выставочном стенде компании SCEI на фестивале <span data-interwiki-article="Jump Festa" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=Jump_Festa&amp;action=edit&amp;redlink=1" title="Jump Festa (страница отсутствует)">Jump Festa</a></span><span class="noprint" style="font-style:normal; font-weight:normal;"><sup><a class="extiw" href="https://en.wikipedia.org/wiki/Jump_Festa" title="en:Jump Festa"><span title="Jump Festa — версия статьи «Jump Festa» на английском языке">[en]</span></a></sup></span>, организованном издательством Shueisha в декабре 2007 года.<sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup>
</p><p><i>Blade Battlers</i> с самого начала позиционировалась как игра для фанатов манги и аниме «Блич». Вся серия в первую очередь рассчитана на пользователя, который знаком с первоисточником.<sup class="reference" id="cite_ref-animag_17-1"><a href="#cite_note-animag-17">[17]</a></sup> Разработчики постарались, насколько возможно, перенести в игру атмосферу оригинала и воссоздать анимацию боевых сцен сериала. <a class="mw-redirect" href="/wiki/%D0%A6%D0%B5%D0%BB-%D1%88%D0%B5%D0%B9%D0%B4%D0%B5%D1%80%D1%8B" title="Цел-шейдеры">Цел-шейдерная</a> графика, отличаясь средним техническим уровнем, стилизована под аниме.<sup class="reference" id="cite_ref-juegos_7-1"><a href="#cite_note-juegos-7">[7]</a></sup> Была скопирована практически вся одежда героев, аналогично выполнены дизайны оружия и боевые приёмы персонажей.<sup class="reference" id="cite_ref-progames_11-2"><a href="#cite_note-progames-11">[11]</a></sup> Для рисования иллюстраций были наняты работники <a href="/wiki/Studio_Pierrot" title="Studio Pierrot">Studio Pierrot</a>, выпускающей аниме «Блич».<sup class="reference" id="cite_ref-ascii_29-0"><a href="#cite_note-ascii-29">[29]</a></sup> В начальной заставке обеих игр использованы кадры из «Блич», а в первой части <i>Blade Battlers</i> — музыкальная тема аниме <i>«<a class="mw-redirect" href="/wiki/D-tecnoLife" title="D-tecnoLife">D-tecnoLife</a>»</i> в исполнении группы <a href="/wiki/UVERworld" title="UVERworld">UVERworld</a>.<sup class="reference" id="cite_ref-30"><a href="#cite_note-30">[30]</a></sup>
</p><p><a href="/wiki/%D0%A1%D1%8D%D0%B9%D1%8E" title="Сэйю">Актёры</a>, которые озвучивали аниме, были приглашены для озвучивания обеих игр и сами играли в них перед записью.<sup class="reference" id="cite_ref-31"><a href="#cite_note-31">[31]</a></sup> В первой части <i>Blade Battlers</i> в качестве бонусов после завершения миссий доступны интервью с некоторыми из них.
</p><p>В <i>Blade Battlers 2nd</i> включена аналогичная функция — «Radio Golden», где плюшевый лев Кон приглашает в свою студию гостей и берёт у них интервью.<sup class="reference" id="cite_ref-animag_17-2"><a href="#cite_note-animag-17">[17]</a></sup>

<h2><span id=".D0.9F.D1.80.D0.BE.D0.B4.D0.B0.D0.B6.D0.B8_.D0.B8_.D0.BE.D1.82.D0.B7.D1.8B.D0.B2.D1.8B"></span><span class="mw-headline" id="Продажи_и_отзывы">Продажи и отзывы</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Bleach:_Blade_Battlers&amp;veaction=edit&amp;section=9" title="Редактировать раздел «Продажи и отзывы»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Bleach:_Blade_Battlers&amp;action=edit&amp;section=9" title="Редактировать код раздела «Продажи и отзывы»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Сразу после выхода в октябре 2006 года первая часть серии <i>Blade Battlers</i> попала в списки лидеров продаж в Японии. За первую неделю было продано более 10 тыс. дисков, таким образом, <i>Blade Battlers</i> не только оказалась в числе самых продаваемых игр, но и оказалась одной из двух игр (второй стала <i><a href="/wiki/Final_Fantasy_V#Порт_на_Game_Boy_Advance" title="Final Fantasy V">Final Fantasy V Advance</a></i>), которой удалось преодолеть десятитысячный барьер проданных копий за первую неделю.<sup class="reference" id="cite_ref-32"><a href="#cite_note-32">[32]</a></sup>
</p><p>Рецензент испанского сайта VicioJuegos.com считает, что <a href="/wiki/%D0%9A%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D0%BA%D0%B0" title="Компьютерная графика">графика</a> и звук находятся на среднем уровне, а игровой процесс умеренно захватывает, тем не менее, пишет он, игра выполняет свою задачу — понравиться поклонникам «Блич».<sup class="reference" id="cite_ref-juegos_7-2"><a href="#cite_note-juegos-7">[7]</a></sup> Обозреватель ProGames, положительно отзывался о боевой системе, придуманной разработчиками. «Владельцы PS2 и фанаты аниме обязательно должны опробовать игру, особенно если есть друзья, с которыми можно сыграть в паре», — добавляет он.<sup class="reference" id="cite_ref-progames_11-3"><a href="#cite_note-progames-11">[11]</a></sup> Критике подвергся <a href="/wiki/%D0%A1%D0%B0%D1%83%D0%BD%D0%B4%D1%82%D1%80%D0%B5%D0%BA" title="Саундтрек">саундтрек</a> обеих частей.<sup class="reference" id="cite_ref-33"><a href="#cite_note-33">[33]</a></sup>
</p><p><i>Blade Battlers</i> вошла в сборник лучших игр для платформы PlayStation 2 «PlayStation 2 The Best» (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;"></span>«Лучшее на PlayStation 2»), изданный SCEI в октябре 2007 года.<sup class="reference" id="cite_ref-34"><a href="#cite_note-34">[34]</a></sup> Успех первой части стал одним из решающих факторов, повлиявших на принятие решения о разработке продолжения — <i>Bleach: Blade Battlers 2nd</i>.<sup class="reference" id="cite_ref-pixfans_9-4"><a href="#cite_note-pixfans-9">[9]</a></sup> <i>Blade Battlers 2nd</i> также попала в лидеры продаж в Японии<sup class="reference" id="cite_ref-35"><a href="#cite_note-35">[35]</a></sup> и не была издана в <a class="mw-redirect" href="/wiki/%D0%A1%D0%A8%D0%90" title="США">США</a> и <a href="/wiki/%D0%95%D0%B2%D1%80%D0%BE%D0%BF%D0%B0" title="Европа">странах Европы</a>. Обозреватель «<a class="new" href="/w/index.php?title=%D0%90%D0%BD%D0%B8%D0%9C%D0%B0%D0%B3&amp;action=edit&amp;redlink=1" title="АниМаг (страница отсутствует)">АниМага</a>» положительно отзывается о второй части серии и, упоминая связь игры с первоисточником (аниме и мангой), рекомендует её для широкого круга пользователей, а не только для поклонников «Блич». Рецензент пишет, что люди, не смотревшие аниме, оценят другие плюсы игры: «красочные арены, интересные персонажи, феерические бои».<sup class="reference" id="cite_ref-animag_17-3"><a href="#cite_note-animag-17">[17]</a></sup> «АниМаг» критикует лишь сюжетный режим, сводящийся «к системе „бой-картинка-бой-картинка“».<sup class="reference" id="cite_ref-animag_17-4"><a href="#cite_note-animag-17">[17]</a></sup> По мнению Metodologic, игра идеальна для компании друзей.<sup class="reference" id="cite_ref-metodologic_21-2"><a href="#cite_note-metodologic-21">[21]</a></sup> Обозреватель сайта Pixfans.com сожалеет о том, что так мало файтингов на PlayStation 2 допускают возможность игры четырёх человек, и относит эту особенность к числу несомненных плюсов <i>Blade Battlers</i>, но критикует низкий графический уровень игры и скучный «Режим истории».<sup class="reference" id="cite_ref-pixfans_9-5"><a href="#cite_note-pixfans-9">[9]</a></sup> <a href="/wiki/ASCII_Media_Works" title="ASCII Media Works">ASCII Media Works</a> рекомендует <i>Blade Battlers 2nd</i> для начинающих игроков в файтинги, потому что управлять персонажами можно с помощью простых команд.<sup class="reference" id="cite_ref-ascii_29-1"><a href="#cite_note-ascii-29">[29]</a></sup>
</p>
