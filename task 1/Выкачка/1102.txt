<p><b>Влади́мир Миха́йлович Коре́цкий</b> (<abbr title="6 февраля по юлианскому календарю">6</abbr> (<a href="/wiki/18_%D1%84%D0%B5%D0%B2%D1%80%D0%B0%D0%BB%D1%8F" title="18 февраля"><span title="18 февраля по григорианскому календарю">18</span></a>) февраля <a href="/wiki/1890_%D0%B3%D0%BE%D0%B4" title="1890 год">1890</a>, <a href="/wiki/%D0%94%D0%BD%D0%B5%D0%BF%D1%80_(%D0%B3%D0%BE%D1%80%D0%BE%D0%B4)" title="Днепр (город)">Екатеринослав</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> — <a href="/wiki/25_%D0%B8%D1%8E%D0%BB%D1%8F" title="25 июля">25 июля</a> <a href="/wiki/1984_%D0%B3%D0%BE%D0%B4" title="1984 год">1984</a>, <a href="/wiki/%D0%9A%D0%B8%D0%B5%D0%B2" title="Киев">Киев</a>) — советский юрист-международник, специалист в области международного частного права и всеобщей истории государства и права, академик <a class="mw-redirect" href="/wiki/%D0%90%D0%9D_%D0%A3%D0%A1%D0%A1%D0%A0" title="АН УССР">АН УССР</a> (<a class="mw-redirect" href="/wiki/1948" title="1948">1948</a>).
</p><p>В 1949—1951 годах член <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%BC%D0%B8%D1%81%D1%81%D0%B8%D1%8F_%D0%BC%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D0%BE%D0%B3%D0%BE_%D0%BF%D1%80%D0%B0%D0%B2%D0%B0" title="Комиссия международного права">Комиссии международного права ООН</a>. C <a href="/wiki/1949_%D0%B3%D0%BE%D0%B4" title="1949 год">1949 года</a> возглавлял образованный в <a href="/wiki/%D0%9A%D0%B8%D0%B5%D0%B2" title="Киев">Киеве</a> Сектор государства и права АН УССР, а после преобразования в <a href="/wiki/1969_%D0%B3%D0%BE%D0%B4" title="1969 год">1969 году</a> Сектора в Институт государства и права АН УССР был директором Института до <a href="/wiki/1974_%D0%B3%D0%BE%D0%B4" title="1974 год">1974 года</a>. Входил в состав советских делегаций на сессиях <a href="/wiki/%D0%93%D0%B5%D0%BD%D0%B5%D1%80%D0%B0%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D0%90%D1%81%D1%81%D0%B0%D0%BC%D0%B1%D0%BB%D0%B5%D1%8F_%D0%9E%D0%9E%D0%9D" title="Генеральная Ассамблея ООН">Генеральной Ассамблеи ООН</a> в <a class="mw-redirect" href="/wiki/1946" title="1946">1946</a>, <a class="mw-redirect" href="/wiki/1947" title="1947">1947</a> и <a href="/wiki/1949_%D0%B3%D0%BE%D0%B4" title="1949 год">1949 годах</a> от <a class="mw-redirect" href="/wiki/%D0%A3%D0%A1%D0%A1%D0%A0" title="УССР">УССР</a>; участвовал в работе ряда международных конференций и комиссий ООН. Был членом <a href="/wiki/%D0%9F%D0%BE%D1%81%D1%82%D0%BE%D1%8F%D0%BD%D0%BD%D0%B0%D1%8F_%D0%BF%D0%B0%D0%BB%D0%B0%D1%82%D0%B0_%D1%82%D1%80%D0%B5%D1%82%D0%B5%D0%B9%D1%81%D0%BA%D0%BE%D0%B3%D0%BE_%D1%81%D1%83%D0%B4%D0%B0" title="Постоянная палата третейского суда">Постоянной палаты третейского суда</a> (с <a href="/wiki/1957_%D0%B3%D0%BE%D0%B4" title="1957 год">1957 года</a>), в <a class="mw-redirect" href="/wiki/1961" title="1961">1961</a>—<a href="/wiki/1970_%D0%B3%D0%BE%D0%B4" title="1970 год">1970 годах</a> член (в 1967—1970 годах вице-президент) <a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%81%D1%83%D0%B4_%D0%9E%D0%9E%D0%9D" title="Международный суд ООН">Международного суда ООН</a>. Удостоен звания Героя Социалистического Труда (<a class="mw-redirect" href="/wiki/1980" title="1980">1980</a>). Награждён орденом Ленина и орденом Трудового Красного Знамени.
</p>

<h2><span id=".D0.91.D0.B8.D0.BE.D0.B3.D1.80.D0.B0.D1.84.D0.B8.D1.8F"></span><span class="mw-headline" id="Биография">Биография</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%BE%D1%80%D0%B5%D1%86%D0%BA%D0%B8%D0%B9,_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80_%D0%9C%D0%B8%D1%85%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Биография»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%BE%D1%80%D0%B5%D1%86%D0%BA%D0%B8%D0%B9,_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80_%D0%9C%D0%B8%D1%85%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=1" title="Редактировать код раздела «Биография»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>В. М. Корецкий родился в семье служащего. Уже в детстве он обнаружил способности к гуманитарным наукам. В <a href="/wiki/1910_%D0%B3%D0%BE%D0%B4" title="1910 год">1910 году</a> с золотой медалью окончил Екатеринославскую гимназию<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> и сразу поступил в <a class="mw-redirect" href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D1%83%D0%BD%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D1%82%D0%B5%D1%82" title="Московский университет">Московский университет</a> на юридический факультет<sup class="reference" id="cite_ref-анталогия_3-0"><a href="#cite_note-анталогия-3">[3]</a></sup>. Однако через год, после рождения сына Юрия, переехал в <a href="/wiki/%D0%A5%D0%B0%D1%80%D1%8C%D0%BA%D0%BE%D0%B2" title="Харьков">Харьков</a>, где обучался в <a class="mw-redirect" href="/wiki/%D0%A5%D0%B0%D1%80%D1%8C%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D1%83%D0%BD%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D1%82%D0%B5%D1%82" title="Харьковский университет">Харьковском императорском университете</a>, который окончил в <a href="/wiki/1916_%D0%B3%D0%BE%D0%B4" title="1916 год">1916 году</a> с дипломом первой степени. Корецкого оставили на кафедре гражданского права и судопроизводства для подготовки к получению профессорского звания.
</p>

<h3><span id=".D0.9D.D0.B0.D1.87.D0.B0.D0.BB.D0.BE_.D0.BA.D0.B0.D1.80.D1.8C.D0.B5.D1.80.D1.8B"></span><span class="mw-headline" id="Начало_карьеры">Начало карьеры</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%BE%D1%80%D0%B5%D1%86%D0%BA%D0%B8%D0%B9,_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80_%D0%9C%D0%B8%D1%85%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Начало карьеры»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%BE%D1%80%D0%B5%D1%86%D0%BA%D0%B8%D0%B9,_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80_%D0%9C%D0%B8%D1%85%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=2" title="Редактировать код раздела «Начало карьеры»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Ещё с <a href="/wiki/1907_%D0%B3%D0%BE%D0%B4" title="1907 год">1907 года</a> и во время обучения В. Корецкий зарабатывал на жизнь частными уроками<sup class="reference" id="cite_ref-анталогия_3-1"><a href="#cite_note-анталогия-3">[3]</a></sup>. В <a href="/wiki/1909_%D0%B3%D0%BE%D0%B4" title="1909 год">1909 году</a> он женился на 16-летней Анне Ананьевне Корецкой (девичья фамилия неизвестна). В <a href="/wiki/1916_%D0%B3%D0%BE%D0%B4" title="1916 год">1916 году</a> начал преподавать в начальной торговой школе, а с <a href="/wiki/1919_%D0%B3%D0%BE%D0%B4" title="1919 год">1919 года</a>, после сдачи магистерского экзамена по истории и догме <a href="/wiki/%D0%A0%D0%B8%D0%BC%D1%81%D0%BA%D0%BE%D0%B5_%D0%BF%D1%80%D0%B0%D0%B2%D0%BE" title="Римское право">римского права</a>, — читать лекции по всемирной истории государства и права в Харьковском университете<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>. В это же время В. М. Корецкий участвовал в организации <a href="/wiki/%D0%A5%D0%B0%D1%80%D1%8C%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%B8%D0%BD%D1%81%D1%82%D0%B8%D1%82%D1%83%D1%82_%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D0%BE%D0%B3%D0%BE_%D1%85%D0%BE%D0%B7%D1%8F%D0%B9%D1%81%D1%82%D0%B2%D0%B0" title="Харьковский институт народного хозяйства">Харьковского института народного хозяйства</a> (ХИНХ), где со временем занял должность заместителя декана правового факультета. Здесь же он начал преподавать <a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D0%BE%D0%B5_%D1%87%D0%B0%D1%81%D1%82%D0%BD%D0%BE%D0%B5_%D0%BF%D1%80%D0%B0%D0%B2%D0%BE" title="Международное частное право">международное частное право</a>.
</p>
<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:YuriKoretsky.jpg"><img class="mw-file-element" data-file-height="1012" data-file-width="680" decoding="async" height="298" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/bb/YuriKoretsky.jpg/200px-YuriKoretsky.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/bb/YuriKoretsky.jpg/300px-YuriKoretsky.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/bb/YuriKoretsky.jpg/400px-YuriKoretsky.jpg 2x" width="200"/></a><figcaption>Юрий Владимирович Корецкий</figcaption></figure>
<p>Спустя 10 лет правовой факультет <a href="/wiki/%D0%A5%D0%B0%D1%80%D1%8C%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%B8%D0%BD%D1%81%D1%82%D0%B8%D1%82%D1%83%D1%82_%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D0%BE%D0%B3%D0%BE_%D1%85%D0%BE%D0%B7%D1%8F%D0%B9%D1%81%D1%82%D0%B2%D0%B0" title="Харьковский институт народного хозяйства">ХИНХа</a> был выделен в самостоятельную институцию — Харьковский институт советского строительства и права. До начала <a href="/wiki/%D0%92%D1%82%D0%BE%D1%80%D0%B0%D1%8F_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Вторая мировая война">Второй мировой войны</a> институт ещё два раза сменил своё название: в <a href="/wiki/1933_%D0%B3%D0%BE%D0%B4" title="1933 год">1933 году</a> — на Всеукраинский коммунистический институт советского строительства и права, а в <a class="mw-redirect" href="/wiki/1937" title="1937">1937</a> — на Харьковский юридический институт (сейчас <a class="mw-redirect" href="/wiki/%D0%9D%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%B0%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D1%8E%D1%80%D0%B8%D0%B4%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D0%B0%D0%BA%D0%B0%D0%B4%D0%B5%D0%BC%D0%B8%D1%8F_%D0%B8%D0%BC._%D0%AF%D1%80%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%B0_%D0%9C%D1%83%D0%B4%D1%80%D0%BE%D0%B3%D0%BE" title="Национальная юридическая академия им. Ярослава Мудрого">Национальная юридическая академия им. Ярослава Мудрого</a>)<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>. Все это время Владимир Михайлович занимал должности профессора и с <a href="/wiki/1937_%D0%B3%D0%BE%D0%B4" title="1937 год">1937 года</a> — руководителя кафедры истории государства и права. Кроме того, он параллельно работал заведующим правового подотдела Народного комиссариата иностранных дел УССР (<a class="mw-redirect" href="/wiki/1921" title="1921">1921</a>—<a class="mw-redirect" href="/wiki/1922" title="1922">1922</a>), затем заведующим юридического отдела <a href="/wiki/%D0%A3%D0%BA%D0%BE%D0%BE%D0%BF%D1%81%D0%BE%D1%8E%D0%B7" title="Укоопсоюз">Укоопсоюза</a> с <a class="mw-redirect" href="/wiki/1922" title="1922">1922</a> по <a href="/wiki/1924_%D0%B3%D0%BE%D0%B4" title="1924 год">1924 год</a>; заведующим юридического отдела Украинбанка (<a class="mw-redirect" href="/wiki/1924" title="1924">1924</a>—<a class="mw-redirect" href="/wiki/1930" title="1930">1930</a>), а затем — старшим консультантом Госарбитража при СНК УССР (<a class="mw-redirect" href="/wiki/1930" title="1930">1930</a>—<a class="mw-redirect" href="/wiki/1935" title="1935">1935</a>)<sup class="reference" id="cite_ref-анталогия_3-2"><a href="#cite_note-анталогия-3">[3]</a></sup>.
</p><p>В <a href="/wiki/1939_%D0%B3%D0%BE%D0%B4" title="1939 год">1939 году</a> В. М. Корецкий получил степень доктора юридических наук, защитив диссертацию, посвященную вопросам англо-американской доктрины международного частного права. В самом начале <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B0%D1%8F_%D0%9E%D1%82%D0%B5%D1%87%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Великая Отечественная война">Великой Отечественной войны</a> его эвакуировали в <a href="/wiki/%D0%A3%D0%B7%D0%B1%D0%B5%D0%BA%D0%B8%D1%81%D1%82%D0%B0%D0%BD" title="Узбекистан">Узбекистан</a>, где в <a class="new" href="/w/index.php?title=%D0%A2%D0%B0%D1%88%D0%BA%D0%B5%D0%BD%D1%82%D1%81%D0%BA%D0%B8%D0%B9_%D0%B3%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9_%D1%8E%D1%80%D0%B8%D0%B4%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B8%D0%BD%D1%81%D1%82%D0%B8%D1%82%D1%83%D1%82&amp;action=edit&amp;redlink=1" title="Ташкентский государственный юридический институт (страница отсутствует)">Ташкентском юридическом институте</a> он преподавал всемирную <a href="/wiki/%D0%98%D1%81%D1%82%D0%BE%D1%80%D0%B8%D1%8F_%D0%B3%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B0_%D0%B8_%D0%BF%D1%80%D0%B0%D0%B2%D0%B0" title="История государства и права">историю государства и права</a>. В 1941 году во время <a class="mw-redirect" href="/wiki/%D0%9A%D0%B8%D0%B5%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F_(1941)" title="Киевская операция (1941)">битвы за Киев</a> в бою погиб его единственный сын <a href="/wiki/%D0%9A%D0%BE%D1%80%D0%B5%D1%86%D0%BA%D0%B8%D0%B9,_%D0%AE%D1%80%D0%B8%D0%B9_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Корецкий, Юрий Владимирович">Юрий</a>.
</p><p>По возвращении в Харьков в <a href="/wiki/1944_%D0%B3%D0%BE%D0%B4" title="1944 год">1944 году</a> Корецкий стал куратором курса <a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D0%BE%D0%B5_%D0%BF%D1%80%D0%B0%D0%B2%D0%BE" title="Международное право">международного права</a> Харьковского юридического института, а в <a class="mw-redirect" href="/wiki/1947" title="1947">1947</a> — заведующим кафедры международного права. Спустя год его избрали <a href="/wiki/%D0%90%D0%BA%D0%B0%D0%B4%D0%B5%D0%BC%D0%B8%D0%BA" title="Академик">академиком</a> <a class="mw-redirect" href="/wiki/%D0%90%D0%9D_%D0%A3%D0%A1%D0%A1%D0%A0" title="АН УССР">АН УССР</a>, а в <a href="/wiki/1949_%D0%B3%D0%BE%D0%B4" title="1949 год">1949 году</a> он создал в столице <a href="/wiki/%D0%A3%D0%BA%D1%80%D0%B0%D0%B8%D0%BD%D0%B0" title="Украина">Украины</a> Сектор государства и права АН УССР (с <a href="/wiki/1969_%D0%B3%D0%BE%D0%B4" title="1969 год">1969 года</a> — <a class="mw-redirect" href="/wiki/%D0%98%D0%BD%D1%81%D1%82%D0%B8%D1%82%D1%83%D1%82_%D0%B3%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B0_%D0%B8_%D0%BF%D1%80%D0%B0%D0%B2%D0%B0_%D0%B8%D0%BC._%D0%92.%D0%9C._%D0%9A%D0%BE%D1%80%D0%B5%D1%86%D0%BA%D0%BE%D0%B3%D0%BE_%D0%9D%D0%90%D0%9D_%D0%A3%D0%BA%D1%80%D0%B0%D0%B8%D0%BD%D1%8B" title="Институт государства и права им. В.М. Корецкого НАН Украины">Институт</a>), основной целью которого была разработка научного обоснования международной <a href="/wiki/%D0%9F%D1%80%D0%B0%D0%B2%D0%BE%D1%81%D1%83%D0%B1%D1%8A%D0%B5%D0%BA%D1%82%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Правосубъектность">правосубъектности</a> <a class="mw-redirect" href="/wiki/%D0%A3%D0%A1%D0%A1%D0%A0" title="УССР">УССР</a>. С <a class="mw-redirect" href="/wiki/1949" title="1949">1949</a> по <a href="/wiki/1974_%D0%B3%D0%BE%D0%B4" title="1974 год">1974 год</a> В. М. Корецкий являлся руководителем этого учреждения.
</p>
<h3><span id=".D0.A0.D0.B0.D0.B1.D0.BE.D1.82.D0.B0_.D0.B2_.D0.9E.D0.9E.D0.9D"></span><span class="mw-headline" id="Работа_в_ООН">Работа в ООН</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%BE%D1%80%D0%B5%D1%86%D0%BA%D0%B8%D0%B9,_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80_%D0%9C%D0%B8%D1%85%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Работа в ООН»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%BE%D1%80%D0%B5%D1%86%D0%BA%D0%B8%D0%B9,_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80_%D0%9C%D0%B8%D1%85%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=3" title="Редактировать код раздела «Работа в ООН»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>

<p>В послевоенное время и почти до конца жизни Владимир Михайлович был одним из наиболее активных участников в работе новой международной организации <a class="mw-redirect" href="/wiki/%D0%9E%D0%9E%D0%9D" title="ООН">ООН</a> от Украинский Советской Социалистической Республики. Так, по инициативе министра иностранных дел УССР, академика <a href="/wiki/%D0%9C%D0%B0%D0%BD%D1%83%D0%B8%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9,_%D0%94%D0%BC%D0%B8%D1%82%D1%80%D0%B8%D0%B9_%D0%97%D0%B0%D1%85%D0%B0%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Мануильский, Дмитрий Захарович">Д. М. Мануильского</a>, он входил в состав делегации УССР на I, II и IV сессиях <a href="/wiki/%D0%93%D0%B5%D0%BD%D0%B5%D1%80%D0%B0%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D0%90%D1%81%D1%81%D0%B0%D0%BC%D0%B1%D0%BB%D0%B5%D1%8F_%D0%9E%D0%9E%D0%9D" title="Генеральная Ассамблея ООН">Генеральной Ассамблеи ООН</a> (соответственно, в <a class="mw-redirect" href="/wiki/1946" title="1946">1946</a>, <a class="mw-redirect" href="/wiki/1947" title="1947">1947</a> и <a href="/wiki/1949_%D0%B3%D0%BE%D0%B4" title="1949 год">1949 годах</a>)<sup class="reference" id="cite_ref-Денисов_6-0"><a href="#cite_note-Денисов-6">[6]</a></sup>. Д. З. Мануильский высоко ценил В. М. Корецкого как талантливого ученого и высококвалифицированного специалиста в сфере международного права<sup class="reference" id="cite_ref-istoria_7-0"><a href="#cite_note-istoria-7">[7]</a></sup>. В <a href="/wiki/1946_%D0%B3%D0%BE%D0%B4" title="1946 год">1946 году</a> учёный был советником Представителя СССР в <a href="/wiki/%D0%A1%D0%BE%D0%B2%D0%B5%D1%82_%D0%91%D0%B5%D0%B7%D0%BE%D0%BF%D0%B0%D1%81%D0%BD%D0%BE%D1%81%D1%82%D0%B8_%D0%9E%D0%9E%D0%9D" title="Совет Безопасности ООН">Совете Безопасности ООН</a>. В период с <a href="/wiki/1947_%D0%B3%D0%BE%D0%B4" title="1947 год">1947 года</a> по <a href="/wiki/1949_%D0%B3%D0%BE%D0%B4" title="1949 год">1949 год</a> являлся членом и первым заместителем председателя <a href="/wiki/%D0%9A%D0%BE%D0%BC%D0%B8%D1%81%D1%81%D0%B8%D1%8F_%D0%BF%D0%BE_%D0%BF%D1%80%D0%B0%D0%B2%D0%B0%D0%BC_%D1%87%D0%B5%D0%BB%D0%BE%D0%B2%D0%B5%D0%BA%D0%B0_%D0%9E%D0%9E%D0%9D" title="Комиссия по правам человека ООН">Комиссии по правам человека ООН</a>, на которую в это время была возложена задача разработать проект <a href="/wiki/%D0%92%D1%81%D0%B5%D0%BE%D0%B1%D1%89%D0%B0%D1%8F_%D0%B4%D0%B5%D0%BA%D0%BB%D0%B0%D1%80%D0%B0%D1%86%D0%B8%D1%8F_%D0%BF%D1%80%D0%B0%D0%B2_%D1%87%D0%B5%D0%BB%D0%BE%D0%B2%D0%B5%D0%BA%D0%B0" title="Всеобщая декларация прав человека">Всеобщей декларации прав человека</a>, принятой Генеральной Ассамблеей ООН в <a href="/wiki/1948_%D0%B3%D0%BE%D0%B4" title="1948 год">1948 году</a><sup class="reference" id="cite_ref-Денисов_6-1"><a href="#cite_note-Денисов-6">[6]</a></sup>.
</p><p>Признание международной юридической общественностью таланта В. М. Корецкого выразилось в избрании его членом <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%BC%D0%B8%D1%81%D1%81%D0%B8%D1%8F_%D0%BC%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D0%BE%D0%B3%D0%BE_%D0%BF%D1%80%D0%B0%D0%B2%D0%B0" title="Комиссия международного права">Комитета ООН по прогрессивному развитию международного права</a> <a href="/wiki/31_%D1%8F%D0%BD%D0%B2%D0%B0%D1%80%D1%8F" title="31 января">31 января</a> <a href="/wiki/1947_%D0%B3%D0%BE%D0%B4" title="1947 год">1947 года</a><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>. Таким образом, Владимир Михайлович вошёл в историю как первый член Комиссии среди отечественных юристов-международников<sup class="reference" id="cite_ref-Денисов_6-2"><a href="#cite_note-Денисов-6">[6]</a></sup>. Первая сессия Комиссии прошла с <a href="/wiki/12_%D0%B0%D0%BF%D1%80%D0%B5%D0%BB%D1%8F" title="12 апреля">12 апреля</a> по <a href="/wiki/9_%D0%B8%D1%8E%D0%BD%D1%8F" title="9 июня">9 июня</a> <a href="/wiki/1949_%D0%B3%D0%BE%D0%B4" title="1949 год">1949 года</a>; на ней В. М. Корецкий занимал должность первого вице-председателя<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>.
</p>

В. М. Корецкий второй справа в верхнем ряду (1961).</figcaption></figure>
<p>Однако наибольшим достижением в карьере учёного является его избрание в <a href="/wiki/1960_%D0%B3%D0%BE%D0%B4" title="1960 год">1960 году</a> на XV сессии Генеральной Ассамблеи ООН и заседании Совета Безопасности ООН членом <a class="mw-redirect" href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D0%A1%D1%83%D0%B4_%D0%9E%D0%9E%D0%9D" title="Международный Суд ООН">Международного Суда ООН</a> в <a href="/wiki/%D0%93%D0%B0%D0%B0%D0%B3%D0%B0" title="Гаага">Гааге</a> сроком на 9 лет с <a class="mw-redirect" href="/wiki/1961" title="1961">1961</a> по <a href="/wiki/1970_%D0%B3%D0%BE%D0%B4" title="1970 год">1970 год</a>. В период с <a class="mw-redirect" href="/wiki/1968" title="1968">1968</a> по <a class="mw-redirect" href="/wiki/1970" title="1970">1970</a> он был избран вице-президентом Международного Суда<sup class="reference" id="cite_ref-Денисов_6-3"><a href="#cite_note-Денисов-6">[6]</a></sup>.
</p>
<h3><span id=".D0.A3.D1.87.D0.B0.D1.81.D1.82.D0.B8.D0.B5_.D0.B2_.D0.B7.D0.BD.D0.B0.D1.87.D0.B8.D0.BC.D1.8B.D1.85_.D0.BA.D0.BE.D0.BD.D1.84.D0.B5.D1.80.D0.B5.D0.BD.D1.86.D0.B8.D1.8F.D1.85"></span><span class="mw-headline" id="Участие_в_значимых_конференциях">Участие в значимых конференциях</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%BE%D1%80%D0%B5%D1%86%D0%BA%D0%B8%D0%B9,_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80_%D0%9C%D0%B8%D1%85%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Участие в значимых конференциях»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%BE%D1%80%D0%B5%D1%86%D0%BA%D0%B8%D0%B9,_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80_%D0%9C%D0%B8%D1%85%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=4" title="Редактировать код раздела «Участие в значимых конференциях»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Профессионализм Владимира Корецкого был очень необходим молодой республике УССР, которая после войны получила особый статус международной <a href="/wiki/%D0%9F%D1%80%D0%B0%D0%B2%D0%BE%D1%81%D1%83%D0%B1%D1%8A%D0%B5%D0%BA%D1%82%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Правосубъектность">правосубъектности</a>. Специалистов высокого уровня было недостаточно и, конечно, такая фигура в международно-правовой науке помогала УССР быть достойно представленной на различных международных конференциях и в <a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D0%B0%D1%8F_%D0%BE%D1%80%D0%B3%D0%B0%D0%BD%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F" title="Международная организация">международных организациях</a><sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>. В <a href="/wiki/1953_%D0%B3%D0%BE%D0%B4" title="1953 год">1953 году</a> как советник делегации УССР он принимал участие в Женевской международной конференции по выработке <a class="new" href="/w/index.php?title=%D0%94%D0%BE%D0%BF%D0%BE%D0%BB%D0%BD%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D0%BA%D0%BE%D0%BD%D0%B2%D0%B5%D0%BD%D1%86%D0%B8%D1%8F_%D0%BE%D0%B1_%D1%83%D0%BF%D1%80%D0%B0%D0%B7%D0%B4%D0%BD%D0%B5%D0%BD%D0%B8%D0%B8_%D1%80%D0%B0%D0%B1%D1%81%D1%82%D0%B2%D0%B0_%D0%B8_%D1%80%D0%B0%D0%B1%D0%BE%D1%82%D0%BE%D1%80%D0%B3%D0%BE%D0%B2%D0%BB%D0%B8&amp;action=edit&amp;redlink=1" title="Дополнительная конвенция об упразднении рабства и работорговли (страница отсутствует)">дополнительной конвенции об упразднении рабства и работорговли</a>. В <a href="/wiki/1957_%D0%B3%D0%BE%D0%B4" title="1957 год">1957 году</a> был направлен на конференцию <a href="/wiki/%D0%AE%D0%9D%D0%95%D0%A1%D0%9A%D0%9E" title="ЮНЕСКО">ЮНЕСКО</a> по вопросам мирного сосуществования в <a href="/wiki/%D0%9C%D1%8E%D0%BD%D1%85%D0%B5%D0%BD" title="Мюнхен">Мюнхене</a>, а в <a class="mw-redirect" href="/wiki/1958" title="1958">1958</a> и <a href="/wiki/1960_%D0%B3%D0%BE%D0%B4" title="1960 год">1960 годах</a> возглавлял делегацию УССР на конференциях ООН по <a href="/wiki/%D0%9C%D0%BE%D1%80%D1%81%D0%BA%D0%BE%D0%B5_%D0%BF%D1%80%D0%B0%D0%B2%D0%BE" title="Морское право">морскому праву</a><sup class="reference" id="cite_ref-Денисов_6-4"><a href="#cite_note-Денисов-6">[6]</a></sup>.
</p><p>В. М. Корецкий также принимал участие в работе 48-й сессии <a class="mw-redirect" href="/wiki/%D0%90%D1%81%D1%81%D0%BE%D1%86%D0%B8%D0%B0%D1%86%D0%B8%D1%8F_%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D0%BE%D0%B3%D0%BE_%D0%9F%D1%80%D0%B0%D0%B2%D0%B0" title="Ассоциация Международного Права">Ассоциации международного права</a> в <a href="/wiki/%D0%9D%D1%8C%D1%8E-%D0%99%D0%BE%D1%80%D0%BA" title="Нью-Йорк">Нью-Йорке</a> (<a href="/wiki/1958_%D0%B3%D0%BE%D0%B4" title="1958 год">1958 года</a>) и XIV сессии ассамблеи Всемирной федерации ассоциации содействия ООН (<a class="mw-redirect" href="/wiki/1959" title="1959">1959</a>) в качестве представителя советских общественных организаций. С <a href="/wiki/1957_%D0%B3%D0%BE%D0%B4" title="1957 год">1957 года</a> учёный являлся членом советской группы Постоянной палаты третейского суда в <a href="/wiki/%D0%93%D0%B0%D0%B0%D0%B3%D0%B0" title="Гаага">Гааге</a><sup class="reference" id="cite_ref-Денисов_6-5"><a href="#cite_note-Денисов-6">[6]</a></sup>.
</p><p>Несмотря преклонный возраст, В. М. Корецкий до последних дней сохранял работоспособность. Скончался учёный <a href="/wiki/25_%D0%B8%D1%8E%D0%BB%D1%8F" title="25 июля">25 июля</a> <a href="/wiki/1984_%D0%B3%D0%BE%D0%B4" title="1984 год">1984 года</a>; похоронен на Лукьяновском кладбище в <a href="/wiki/%D0%9A%D0%B8%D0%B5%D0%B2" title="Киев">Киеве</a>.
</p>
<h2><span id=".D0.9D.D0.B0.D1.83.D1.87.D0.BD.D1.8B.D0.B5_.D0.B2.D0.B7.D0.B3.D0.BB.D1.8F.D0.B4.D1.8B_.D0.B8_.D1.82.D1.80.D1.83.D0.B4.D1.8B"></span><span class="mw-headline" id="Научные_взгляды_и_труды">Научные взгляды и труды</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9A%D0%BE%D1%80%D0%B5%D1%86%D0%BA%D0%B8%D0%B9,_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80_%D0%9C%D0%B8%D1%85%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Научные взгляды и труды»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9A%D0%BE%D1%80%D0%B5%D1%86%D0%BA%D0%B8%D0%B9,_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80_%D0%9C%D0%B8%D1%85%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=5" title="Редактировать код раздела «Научные взгляды и труды»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>В своей рецензии на книгу «В. М. Корецкий. Избранные труды» доктор юридических наук, профессор Ю. Я. Баскин называет «ученого одним из виднейших советских юристов»<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup>, развивавшим молодую международно-правовую науку на протяжении практически всей истории советского государства<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>. Наибольшая активность учёного приходится на <a class="mw-redirect" href="/wiki/1920-%D0%B5" title="1920-е">20-е годы</a>. Его научное наследие охватывает исследования в различных областях юридической науки, таких как история государства и права западных стран, <a href="/wiki/%D0%93%D1%80%D0%B0%D0%B6%D0%B4%D0%B0%D0%BD%D1%81%D0%BA%D0%BE%D0%B5_%D0%BF%D1%80%D0%B0%D0%B2%D0%BE" title="Гражданское право">гражданского права</a> <a href="/wiki/%D0%A3%D0%BA%D1%80%D0%B0%D0%B8%D0%BD%D0%B0" title="Украина">Украины</a>, международного частного и публичного права.
</p><p>Весомым вкладом в международно-правовую науку стало его научное обоснование необходимости объединения международного хозяйственного и международного публичного права и создания специальной отрасли <a class="new" href="/w/index.php?title=%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D0%BE%D0%B5_%D1%85%D0%BE%D0%B7%D1%8F%D0%B9%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%BE%D0%B5_%D0%BF%D1%80%D0%B0%D0%B2%D0%BE&amp;action=edit&amp;redlink=1" title="Международное хозяйственное право (страница отсутствует)">международного хозяйственного права</a>, которая бы базировалась на объединении международно-правового и частноправового методов регулирования хозяйственной деятельности мировых хозяйственных процессов. Этот вклад был высоко оценен одним из учеников В. М. Корецкого, Игорем Ивановичем Лукашуком (членом <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%BC%D0%B8%D1%81%D1%81%D0%B8%D1%8F_%D0%BC%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D0%BE%D0%B3%D0%BE_%D0%BF%D1%80%D0%B0%D0%B2%D0%B0" title="Комиссия международного права">Комиссии международного права</a> в период с <a class="mw-redirect" href="/wiki/1995" title="1995">1995</a>—<a href="/wiki/2001_%D0%B3%D0%BE%D0%B4" title="2001 год">2001 год</a><sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup>, который неоднократно писал о том, что с признательностью помнит Корецкого как своего научного руководителя<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup>.
</p>
<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:%D0%9A%D0%BE%D1%80%D0%B5%D1%86%D0%BA%D0%B8%D0%B9_%D0%BA%D0%BD%D0%B8%D0%B3%D0%B0.jpg"><img class="mw-file-element" data-file-height="315" data-file-width="200" decoding="async" height="315" src="//upload.wikimedia.org/wikipedia/commons/7/75/%D0%9A%D0%BE%D1%80%D0%B5%D1%86%D0%BA%D0%B8%D0%B9_%D0%BA%D0%BD%D0%B8%D0%B3%D0%B0.jpg" width="200"/></a><figcaption>Титульная страница монографии В. М. Корецкого «Декларация прав и обязанностей государств»</figcaption></figure>
<p>В своей работе «Очерки международного хозяйственного права», которая была опубликована в <a href="/wiki/1928_%D0%B3%D0%BE%D0%B4" title="1928 год">1928 году</a>, учёный впервые<sup class="reference" id="cite_ref-анталогия_3-3"><a href="#cite_note-анталогия-3">[3]</a></sup> в международно-правовой литературе доказал, что основой закономерностей развития права является его соответствие объективно существующим условиям сущности экономической жизни общества, то есть современной терминологией — соответствие закономерностям развития <a href="/wiki/%D0%93%D1%80%D0%B0%D0%B6%D0%B4%D0%B0%D0%BD%D1%81%D0%BA%D0%BE%D0%B5_%D0%BE%D0%B1%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Гражданское общество">гражданского общества</a><sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup>. Он словно предвидел нарастающую необходимость в подобном исследовании. Его идеи имели новаторский характер, особенно для развития <a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D0%BE%D0%B5_%D1%8D%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%BE%D0%B5_%D0%BF%D1%80%D0%B0%D0%B2%D0%BE" title="Международное экономическое право">международного экономического права</a>. Как отмечает ещё один ученик Корецкого, профессор В. Н. Денисов, время доказало их высокий уровень значимости для развития международного права<sup class="reference" id="cite_ref-анталогия_3-4"><a href="#cite_note-анталогия-3">[3]</a></sup>. Подобные идеи получили своё дальнейшее развитие во второй половине <a href="/wiki/XX_%D0%B2%D0%B5%D0%BA" title="XX век">XX века</a> в работах, посвященных транснациональному праву таких ученых, как <a class="new" href="/w/index.php?title=%D0%94%D0%B6%D0%B5%D1%81%D1%81%D0%B0%D0%BF_%D0%A4%D0%B8%D0%BB%D0%B8%D0%BF%D0%BF&amp;action=edit&amp;redlink=1" title="Джессап Филипп (страница отсутствует)">Ф. Джессап</a><sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup>, <a class="new" href="/w/index.php?title=%D0%9A%D1%83%D0%BD%D1%86_%D0%94%D0%B6%D0%BE%D0%B7%D0%B5%D1%84&amp;action=edit&amp;redlink=1" title="Кунц Джозеф (страница отсутствует)">Дж. Кунц</a><sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup>, <a class="new" href="/w/index.php?title=%D0%A7%D0%B0%D0%BA%D0%BE_%D0%A7%D0%B0%D1%80%D0%BB%D1%8C%D0%B7&amp;action=edit&amp;redlink=1" title="Чако Чарльз (страница отсутствует)">Ч. Чако</a><sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup>, <a class="new" href="/w/index.php?title=%D0%9A%D0%BB%D0%B0%D0%B9%D0%B2_%D0%9F%D1%8D%D1%80%D1%80%D0%B8&amp;action=edit&amp;redlink=1" title="Клайв Пэрри (страница отсутствует)">П. Клайв</a><sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup> и др.
Значимой работой в области международного частного права является монография «Очерки англо-американской доктрины и практики международного частного права»<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup> (1948). В <a href="/wiki/1950_%D0%B3%D0%BE%D0%B4" title="1950 год">1950 году</a> эта работа была издана в <a href="/wiki/%D0%9A%D0%B8%D1%82%D0%B0%D0%B9" title="Китай">Китае</a> на <a href="/wiki/%D0%9A%D0%B8%D1%82%D0%B0%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Китайский язык">китайском языке</a>.
</p><p>Одно из центральных мест в научном творчестве В. М. Корецкого стали его работы «Суверенитет» (<a class="mw-redirect" href="/wiki/1963" title="1963">1963</a>) и «Международно-правовая правосубъектность Украинской ССР»<sup class="reference" id="cite_ref-избранные_21-0"><a href="#cite_note-избранные-21">[21]</a></sup>, (<a class="mw-redirect" href="/wiki/1965" title="1965">1965</a>), в которых он смог привести научные доводы правовых оснований участия УССР в ООН в качестве полноправного члена этой международной организации. Также учёный развивал такие сферы международного публичного права, как <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%BD%D1%82%D0%B8%D0%BD%D0%B5%D0%BD%D1%82%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D1%88%D0%B5%D0%BB%D1%8C%D1%84" title="Континентальный шельф">континентальный шельф</a>, вопросы <a href="/wiki/%D0%9A%D0%BE%D0%B4%D0%B8%D1%84%D0%B8%D0%BA%D0%B0%D1%86%D0%B8%D1%8F" title="Кодификация">кодификации</a> международного права, истории международного права и др. Большинство этих трудов были опубликованы во второй половине <a href="/wiki/XX_%D0%B2%D0%B5%D0%BA" title="XX век">XX века</a>, а после его смерти были собраны в двухтомнике «Избранные труды»<sup class="reference" id="cite_ref-избранные_21-1"><a href="#cite_note-избранные-21">[21]</a></sup>.
</p><p>Учёный также не оставил без внимания фундаментальные вопросы международного права, которые нашли своё отображение в таких трудах, как «Общие принципы права в международном праве» (<a class="mw-redirect" href="/wiki/1957" title="1957">1957</a>)<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup>, «Проблема „основных прав и обязанностей государств“ в международном праве» (1959)<sup class="reference" id="cite_ref-избранные_21-2"><a href="#cite_note-избранные-21">[21]</a></sup>, «Декларация прав и обязанностей государств» (<a class="mw-redirect" href="/wiki/1962" title="1962">1962</a>)<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup>.
</p><p>Значительный вклад В. М. Корецкий внёс в науку <a href="/wiki/%D0%98%D1%81%D1%82%D0%BE%D1%80%D0%B8%D1%8F_%D0%B3%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B0_%D0%B8_%D0%BF%D1%80%D0%B0%D0%B2%D0%B0" title="История государства и права">истории государства и права</a>. Одной из его известных работ является исследование проекта чешского короля <a href="/wiki/%D0%99%D0%B8%D1%80%D0%B6%D0%B8_%D0%B8%D0%B7_%D0%9F%D0%BE%D0%B4%D0%B5%D0%B1%D1%80%D0%B0%D0%B4" title="Йиржи из Подебрад">Иржи Подебрада</a> об организации мира и безопасности <a href="/wiki/1464_%D0%B3%D0%BE%D0%B4" title="1464 год">1464 года</a>, которая получила название «Проект Иржи Подебрада об организации мира и современность» (<a class="mw-redirect" href="/wiki/1966" title="1966">1966</a>). Подробное исследование раннефеодального государства и права <a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%BE%D1%81%D0%B0%D0%BA%D1%81%D1%8B" title="Англосаксы">англосаксов</a> соде́ржится в таких работах учёного, как «Саксонское зерцало» (1985)<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup>, «Суд и судебный процесс во Франкском государстве» (<a class="mw-redirect" href="/wiki/1944" title="1944">1944</a>), «Иммунитет во Франкском государстве» (<a class="mw-redirect" href="/wiki/1944" title="1944">1944</a>), «Суд фемов» (<a class="mw-redirect" href="/wiki/1944" title="1944">1944</a>), «Военный строй Франкского государства» (<a class="mw-redirect" href="/wiki/1947" title="1947">1947</a>), «Англо-Саксонские правды» (<a class="mw-redirect" href="/wiki/1947" title="1947">1947</a>) и др.<sup class="reference" id="cite_ref-избранные_21-3"><a href="#cite_note-избранные-21">[21]</a></sup>
</p>

<p>Имя В. М. Корецкого было и поныне остаётся широко известным юридической научной общественности иностранных государств. Так, он был избран почётным членом Индийского общества международного права (<a href="/wiki/1960_%D0%B3%D0%BE%D0%B4" title="1960 год">1960 год</a>); членом-корреспондентом (<a href="/wiki/1965_%D0%B3%D0%BE%D0%B4" title="1965 год">1965 год</a>), действительным (<a href="/wiki/1977_%D0%B3%D0%BE%D0%B4" title="1977 год">1977 год</a>), а со временем (<a href="/wiki/1983_%D0%B3%D0%BE%D0%B4" title="1983 год">1983 год</a>) почётным членом <a href="/wiki/%D0%98%D0%BD%D1%81%D1%82%D0%B8%D1%82%D1%83%D1%82_%D0%BC%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D0%BE%D0%B3%D0%BE_%D0%BF%D1%80%D0%B0%D0%B2%D0%B0" title="Институт международного права">Института международного права</a>; академиком Мексиканской академии международного права (<a href="/wiki/1968_%D0%B3%D0%BE%D0%B4" title="1968 год">1968 год</a>); членом-корреспондентом Аргентинской <a class="mw-redirect" href="/wiki/%D0%90%D1%81%D1%81%D0%BE%D1%86%D0%B8%D0%B0%D1%86%D0%B8%D1%8F_%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D0%BE%D0%B3%D0%BE_%D0%9F%D1%80%D0%B0%D0%B2%D0%B0" title="Ассоциация Международного Права">ассоциации международного права</a> (<a href="/wiki/1971_%D0%B3%D0%BE%D0%B4" title="1971 год">1971 год</a>)<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup>.
</p>

<p>Известный польский юрист-международник <a class="mw-redirect" href="/wiki/%D0%9C%D0%B0%D0%BD%D1%84%D1%80%D0%B5%D0%B4_%D0%9B%D1%8F%D1%85%D1%81" title="Манфред Ляхс">М. Ляхс</a> в своей книге по истории международного права относит В. М. Корецкого к наиболее известным в мире учителям этой науки<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup>.
</p><p>Владимир Михайлович — единственный учёный-юрист в бывшем СССР, который был удостоен звания <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BE%D0%B9_%D0%A1%D0%BE%D1%86%D0%B8%D0%B0%D0%BB%D0%B8%D1%81%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%BE%D0%B3%D0%BE_%D0%A2%D1%80%D1%83%D0%B4%D0%B0" title="Герой Социалистического Труда">Героя Социалистического Труда</a> (<a href="/wiki/15_%D1%84%D0%B5%D0%B2%D1%80%D0%B0%D0%BB%D1%8F" title="15 февраля">15 февраля</a> <a href="/wiki/1980_%D0%B3%D0%BE%D0%B4" title="1980 год">1980 года</a>)<sup class="reference" id="cite_ref-анталогия_3-5"><a href="#cite_note-анталогия-3">[3]</a></sup>.
</p><p><a href="/wiki/15_%D1%84%D0%B5%D0%B2%D1%80%D0%B0%D0%BB%D1%8F" title="15 февраля">15 февраля</a> <a href="/wiki/1990_%D0%B3%D0%BE%D0%B4" title="1990 год">1990 года</a> <a href="/wiki/%D0%A1%D0%BE%D0%B2%D0%B5%D1%82_%D0%BC%D0%B8%D0%BD%D0%B8%D1%81%D1%82%D1%80%D0%BE%D0%B2" title="Совет министров">Совет министров</a> <a class="mw-redirect" href="/wiki/%D0%A3%D0%A1%D0%A1%D0%A0" title="УССР">УССР</a> своим Постановлением № 36 присвоил <a class="mw-redirect" href="/wiki/%D0%98%D0%BD%D1%81%D1%82%D0%B8%D1%82%D1%83%D1%82_%D0%B3%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B0_%D0%B8_%D0%BF%D1%80%D0%B0%D0%B2%D0%B0_%D0%B8%D0%BC._%D0%92.%D0%9C._%D0%9A%D0%BE%D1%80%D0%B5%D1%86%D0%BA%D0%BE%D0%B3%D0%BE_%D0%9D%D0%90%D0%9D_%D0%A3%D0%BA%D1%80%D0%B0%D0%B8%D0%BD%D1%8B" title="Институт государства и права им. В.М. Корецкого НАН Украины">Институту государства и права АН Украины</a> имя В. М. Корецкого<sup class="reference" id="cite_ref-istoria_7-1"><a href="#cite_note-istoria-7">[7]</a></sup>.
</p><p><a href="/wiki/5_%D0%B4%D0%B5%D0%BA%D0%B0%D0%B1%D1%80%D1%8F" title="5 декабря">5 декабря</a> <a href="/wiki/2007_%D0%B3%D0%BE%D0%B4" title="2007 год">2007 года</a> Координационным советом молодых юристов <a href="/wiki/%D0%A3%D0%BA%D1%80%D0%B0%D0%B8%D0%BD%D0%B0" title="Украина">Украины</a> при <a href="/wiki/%D0%9C%D0%B8%D0%BD%D0%B8%D1%81%D1%82%D0%B5%D1%80%D1%81%D1%82%D0%B2%D0%BE_%D1%8E%D1%81%D1%82%D0%B8%D1%86%D0%B8%D0%B8" title="Министерство юстиции">Министерстве юстиции</a> Украины было принято решение об инициировании Международного юридического конкурса имени В. М. Корецкого по международному праву<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup>.
</p>
