<p><b>Salesforce</b> (<i>salesforce.com</i>) — <a class="mw-redirect" href="/wiki/%D0%A1%D0%A8%D0%90" title="США">американская</a> компания, разработчик одноимённой <a href="/wiki/%D0%A1%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0_%D1%83%D0%BF%D1%80%D0%B0%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F_%D0%B2%D0%B7%D0%B0%D0%B8%D0%BC%D0%BE%D0%BE%D1%82%D0%BD%D0%BE%D1%88%D0%B5%D0%BD%D0%B8%D1%8F%D0%BC%D0%B8_%D1%81_%D0%BA%D0%BB%D0%B8%D0%B5%D0%BD%D1%82%D0%B0%D0%BC%D0%B8" title="Система управления взаимоотношениями с клиентами">CRM-системы</a>, предоставляемой заказчикам исключительно по модели <a class="mw-redirect" href="/wiki/SaaS" title="SaaS">SaaS</a>. Под наименованием Force.com компания предоставляет <a class="mw-redirect" href="/wiki/PaaS" title="PaaS">PaaS</a>-систему для самостоятельной разработки <a href="/wiki/%D0%9F%D1%80%D0%B8%D0%BA%D0%BB%D0%B0%D0%B4%D0%BD%D0%BE%D0%B5_%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%BD%D0%BE%D0%B5_%D0%BE%D0%B1%D0%B5%D1%81%D0%BF%D0%B5%D1%87%D0%B5%D0%BD%D0%B8%D0%B5" title="Прикладное программное обеспечение">приложений</a>, а под брендом Database.com — <a href="/wiki/%D0%9E%D0%B1%D0%BB%D0%B0%D1%87%D0%BD%D1%8B%D0%B5_%D0%B2%D1%8B%D1%87%D0%B8%D1%81%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F" title="Облачные вычисления">облачную</a> <a href="/wiki/%D0%A1%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0_%D1%83%D0%BF%D1%80%D0%B0%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F_%D0%B1%D0%B0%D0%B7%D0%B0%D0%BC%D0%B8_%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85" title="Система управления базами данных">систему управления базами данных</a>. Среди приобретённых в результате поглощений продуктов — платформенный сервис <a href="/wiki/Heroku" title="Heroku">Heroku</a>, сервисная шина <a href="/wiki/MuleESB" title="MuleESB">MuleESB</a>, тиражируемая система визуализации данных <a href="/wiki/Tableau" title="Tableau">Tableau</a>, корпоративный мессенджер <a href="/wiki/Slack" title="Slack">Slack</a>.
</p><p>Основана в марте <a href="/wiki/1999_%D0%B3%D0%BE%D0%B4" title="1999 год">1999 года</a> <a class="mw-redirect" href="/wiki/%D0%9C%D0%B0%D1%80%D0%BA_%D0%91%D0%B5%D0%BD%D0%B8%D0%BE%D1%84%D1%84" title="Марк Бениофф">Марком Бениоффом</a>, Паркером Харрисом (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Parker Harris</span>), Дэйвом Мёленхоффом (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Dave Moellenhoff</span>) и Френком Домингесом (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Frank Dominguez</span>) с изначальной идеей специализации на рынке <a class="mw-redirect" href="/wiki/%D0%A1%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0_%D1%83%D0%BF%D1%80%D0%B0%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F_%D0%B2%D0%B7%D0%B0%D0%B8%D0%BC%D0%BE%D0%B4%D0%B5%D0%B9%D1%81%D1%82%D0%B2%D0%B8%D0%B5%D0%BC_%D1%81_%D0%BA%D0%BB%D0%B8%D0%B5%D0%BD%D1%82%D0%B0%D0%BC%D0%B8" title="Система управления взаимодействием с клиентами">CRM-систем</a> и предоставления доступа к ним исключительно <a class="mw-redirect" href="/wiki/SaaS" title="SaaS">как услуги по подписке</a>, располагая все экземпляры системы в собственных <a class="mw-redirect" href="/wiki/%D0%A6%D0%B5%D0%BD%D1%82%D1%80_%D0%BE%D0%B1%D1%80%D0%B0%D0%B1%D0%BE%D1%82%D0%BA%D0%B8_%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85" title="Центр обработки данных">центрах обработки данных</a>, полностью исключая <a href="/wiki/%D0%A3%D1%81%D1%82%D0%B0%D0%BD%D0%BE%D0%B2%D0%BA%D0%B0_%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%BD%D0%BE%D0%B3%D0%BE_%D0%BE%D0%B1%D0%B5%D1%81%D0%BF%D0%B5%D1%87%D0%B5%D0%BD%D0%B8%D1%8F" title="Установка программного обеспечения">установки</a> систем у заказчиков и обеспечивая доступ конечных пользователей к системам через <a href="/wiki/%D0%92%D1%81%D0%B5%D0%BC%D0%B8%D1%80%D0%BD%D0%B0%D1%8F_%D0%BF%D0%B0%D1%83%D1%82%D0%B8%D0%BD%D0%B0" title="Всемирная паутина">веб</a><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>. Признана одним из пионеров <a class="mw-redirect" href="/wiki/SaaS" title="SaaS">SaaS</a>, <a class="mw-redirect" href="/wiki/PaaS" title="PaaS">PaaS</a>, <a href="/wiki/%D0%9E%D0%B1%D0%BB%D0%B0%D1%87%D0%BD%D1%8B%D0%B5_%D0%B2%D1%8B%D1%87%D0%B8%D1%81%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F" title="Облачные вычисления">облачных вычислений</a><sup class="reference" id="cite_ref-apex-ini_5-0"><a href="#cite_note-apex-ini-5">[5]</a></sup><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>, с 2012 года является мировым лидером рынка CRM-систем<sup class="reference" id="cite_ref-lead-2013_7-0"><a href="#cite_note-lead-2013-7">[7]</a></sup>.
</p><p><a href="/wiki/%D0%A8%D1%82%D0%B0%D0%B1-%D0%BA%D0%B2%D0%B0%D1%80%D1%82%D0%B8%D1%80%D0%B0_%D0%BE%D1%80%D0%B3%D0%B0%D0%BD%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D0%B8" title="Штаб-квартира организации">Штаб-квартира компании</a> расположена в <a href="/wiki/%D0%A1%D0%B0%D0%BD-%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%81%D0%BA%D0%BE" title="Сан-Франциско">Сан-Франциско</a> в небоскрёбе <span data-interwiki-article="Salesforce Tower" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=Salesforce_Tower&amp;action=edit&amp;redlink=1" title="Salesforce Tower (страница отсутствует)">Salesforce Tower</a></span><span class="noprint" style="font-style:normal; font-weight:normal;"><sup><a class="extiw" href="https://en.wikipedia.org/wiki/Salesforce_Tower" title="en:Salesforce Tower"><span title="Salesforce Tower — версия статьи «Salesforce Tower» на английском языке">[en]</span></a></sup></span>, региональные офисы — в 25 странах.

<h3><span id=".D0.9E.D1.81.D0.BD.D0.BE.D0.B2.D0.B0.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Основание">Основание</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Salesforce&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Основание»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Salesforce&amp;action=edit&amp;section=2" title="Редактировать код раздела «Основание»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>До основания компании Бениофф длительное время работал менеджером по продажам в <a href="/wiki/Oracle" title="Oracle">Oracle</a>, и, как получивший в возрасте 25 лет должность вице-президента, отмечается как самый молодой сотрудник, достигший этой должности<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup><sup class="reference" id="cite_ref-_0f2f342b1c2ae1a0_9-0"><a href="#cite_note-_0f2f342b1c2ae1a0-9">[9]</a></sup>. Харрис, Мёлленхофф и Домингес — бывшие разработчики компании Clarify (поглощённой впоследствии <a href="/wiki/Nortel_Networks" title="Nortel Networks">Nortel Networks</a>, которая, в свою очередь, продала права на Clarify CRM и разрабатывающее его подразделение компании <a href="/wiki/Amdocs" title="Amdocs">Amdocs</a> в <a href="/wiki/2001_%D0%B3%D0%BE%D0%B4" title="2001 год">2001 году</a>).
</p><p>Начальными инвесторами проекта были Бениофф, Марк Искаро (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Mark Iscaro</span>), <a class="mw-redirect" href="/wiki/%D0%9B%D0%B0%D1%80%D1%80%D0%B8_%D0%AD%D0%BB%D0%BB%D0%B8%D1%81%D0%BE%D0%BD" title="Ларри Эллисон">Ларри Эллисон</a>, Хелси Майнор (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Halsey Minor</span>), Магдалена Йесиль (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Magdalena Yesil</span>), Игорь Силл (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Igor Sill</span>), а также инвестиционная компания Geneva Venture Partners<sup class="reference" id="cite_ref-_048dbdf2c05d9804_10-0"><a href="#cite_note-_048dbdf2c05d9804-10">[10]</a></sup>. За год до <a class="mw-redirect" href="/wiki/%D0%9A%D1%80%D0%B0%D1%85_%D0%B4%D0%BE%D1%82%D0%BA%D0%BE%D0%BC%D0%BE%D0%B2" title="Крах доткомов">краха доткомов</a>, в <a href="/wiki/%D0%9C%D0%B0%D1%80%D1%82" title="Март">марте</a> <a href="/wiki/1999_%D0%B3%D0%BE%D0%B4" title="1999 год">1999 года</a>, компания <a class="mw-redirect" href="/wiki/%D0%98%D0%BD%D0%BA%D0%BE%D1%80%D0%BF%D0%BE%D1%80%D0%B0%D1%86%D0%B8%D1%8F_(%D0%B1%D0%B8%D0%B7%D0%BD%D0%B5%D1%81)" title="Инкорпорация (бизнес)">зарегистрировалась</a> с названием <i>Salesforce.com</i>. Первым офисом компании была арендованная жилая квартира в <a href="/wiki/%D0%A1%D0%B0%D0%BD-%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%81%D0%BA%D0%BE" title="Сан-Франциско">Сан-Франциско</a><sup class="reference" id="cite_ref-nyse-mag_11-0"><a href="#cite_note-nyse-mag-11">[11]</a></sup><sup class="reference" id="cite_ref-_12f9ee0fb638cc41_12-0"><a href="#cite_note-_12f9ee0fb638cc41-12">[12]</a></sup>.
</p><p>Основными доводами, которыми изначально пользовалась компания для убеждения заказчиков подписаться на услуги Salesforce.com, были: возможность <a href="/wiki/%D0%90%D1%83%D1%82%D1%81%D0%BE%D1%80%D1%81%D0%B8%D0%BD%D0%B3#Аутсорсинг_в_сфере_ИТ" title="Аутсорсинг">аутсорсинга информационных технологий</a>, защита от сбоев и оперативная техническая поддержка (так как все аппаратные ресурсы располагаются в зоне физической доступности поставщика), снижение <a href="/wiki/%D0%A1%D0%BE%D0%B2%D0%BE%D0%BA%D1%83%D0%BF%D0%BD%D0%B0%D1%8F_%D1%81%D1%82%D0%BE%D0%B8%D0%BC%D0%BE%D1%81%D1%82%D1%8C_%D0%B2%D0%BB%D0%B0%D0%B4%D0%B5%D0%BD%D0%B8%D1%8F" title="Совокупная стоимость владения">совокупной стоимости владения</a> информационными технологиями, при том, что сама Salesforce.com обеспечивает себе существенную экономию ресурсов вследствие <a href="/wiki/%D0%9C%D1%83%D0%BB%D1%8C%D1%82%D0%B8%D0%B0%D1%80%D0%B5%D0%BD%D0%B4%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Мультиарендность">мультиарендности</a> — поддержки нескольких экземпляров приложений на одном экземпляре платформенного программного обеспечения<sup class="reference" id="cite_ref-mt_13-0"><a href="#cite_note-mt-13">[13]</a></sup>.
</p><p>В феврале <a href="/wiki/2000_%D0%B3%D0%BE%D0%B4" title="2000 год">2000 года</a> компания впервые обратила на себя внимание <a class="mw-redirect" href="/wiki/%D0%A1%D0%9C%D0%98" title="СМИ">СМИ</a>, разместив рекламный <a href="/wiki/%D0%91%D0%B0%D0%BD%D0%BD%D0%B5%D1%80" title="Баннер">баннер</a> с надписью <i>«The End of Software»</i> (<i>«конец программному обеспечению»</i>) напротив конференц-центра имени Москоне в <a href="/wiki/%D0%A1%D0%B0%D0%BD-%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%81%D0%BA%D0%BE" title="Сан-Франциско">Сан-Франциско</a>, в котором в тот момент проходила ежегодная конференция пользователей <a href="/wiki/Siebel" title="Siebel">Siebel</a><sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup><sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup>, лидера CRM-рынка на тот момент<sup class="reference" id="cite_ref-cnet-siebel-leadership_16-0"><a href="#cite_note-cnet-siebel-leadership-16">[16]</a></sup>.
</p>
<h3><span id=".D0.A0.D0.B0.D0.B7.D0.B2.D0.B8.D1.82.D0.B8.D0.B5"></span><span class="mw-headline" id="Развитие">Развитие</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Salesforce&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Развитие»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Salesforce&amp;action=edit&amp;section=3" title="Редактировать код раздела «Развитие»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Salesforce.com_taxi_in_London_2008.jpg"><img class="mw-file-element" data-file-height="610" data-file-width="981" decoding="async" height="124" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b0/Salesforce.com_taxi_in_London_2008.jpg/200px-Salesforce.com_taxi_in_London_2008.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/b0/Salesforce.com_taxi_in_London_2008.jpg/300px-Salesforce.com_taxi_in_London_2008.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/b0/Salesforce.com_taxi_in_London_2008.jpg/400px-Salesforce.com_taxi_in_London_2008.jpg 2x" width="200"/></a><figcaption>Рекламное такси Salesforce.com в <a href="/wiki/%D0%9B%D0%BE%D0%BD%D0%B4%D0%BE%D0%BD" title="Лондон">Лондоне</a></figcaption></figure>
<p>В <a href="/wiki/2004_%D0%B3%D0%BE%D0%B4" title="2004 год">2004 году</a> компания <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B2%D0%B8%D1%87%D0%BD%D0%BE%D0%B5_%D0%BF%D1%83%D0%B1%D0%BB%D0%B8%D1%87%D0%BD%D0%BE%D0%B5_%D0%BF%D1%80%D0%B5%D0%B4%D0%BB%D0%BE%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5" title="Первичное публичное предложение">стала публичной</a>, её акции начали торговаться на <a href="/wiki/%D0%9D%D1%8C%D1%8E-%D0%99%D0%BE%D1%80%D0%BA%D1%81%D0%BA%D0%B0%D1%8F_%D1%84%D0%BE%D0%BD%D0%B4%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B1%D0%B8%D1%80%D0%B6%D0%B0" title="Нью-Йоркская фондовая биржа">Нью-Йоркской фондовой бирже</a> с <a href="/wiki/%D0%A2%D0%B8%D0%BA%D0%B5%D1%80" title="Тикер">тикером</a> <code>CRM</code>, собрав $110 млн<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup>. В <a href="/wiki/2007_%D0%B3%D0%BE%D0%B4" title="2007 год">2007 году</a> технологическая основа CRM-системы была опубликована как услуга, таким образом, подписчики получили возможность создавать собственные приложения в том же стеке, что и основные системы Salesforce.com. Позднее это <a class="mw-redirect" href="/wiki/PaaS" title="PaaS">PaaS</a>-решение получило доменное имя <i>force.com</i> и соответствующее наименование. В <a href="/wiki/%D0%A1%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8C_2008_%D0%B3%D0%BE%D0%B4%D0%B0" title="Сентябрь 2008 года">сентябре 2008 года</a> котировки Salesforce.com были включены в индекс <a href="/wiki/S%26P_500" title="S&amp;P 500">S&amp;P 500</a><sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup>.
</p><p>В начале <a href="/wiki/2009_%D0%B3%D0%BE%D0%B4" title="2009 год">2009 года</a> компания выделила из CRM-системы в отдельную разработку блок обслуживания клиентов, выпустив его под торговой маркой <i>Service Cloud</i> со специфической ценовой политикой и ориентированным на более специфический сегмент компаний, бизнес которых требует развиты́х процессов клиентского обслуживания<sup class="reference" id="cite_ref-tc-on-service-cloud_19-0"><a href="#cite_note-tc-on-service-cloud-19">[19]</a></sup> (функциональный блок CRM, покрывающий процессы продаж и маркетинга, получил имя <i>Sales Cloud</i>). В <a href="/wiki/2010_%D0%B3%D0%BE%D0%B4" title="2010 год">2010 году</a> компания существенно расширила линейку предлагаемых сервисов благодаря <a class="mw-redirect" href="/wiki/%D0%9F%D0%BE%D0%B3%D0%BB%D0%BE%D1%89%D0%B5%D0%BD%D0%B8%D0%B5_(%D1%8D%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D0%BA%D0%B0)" title="Поглощение (экономика)">поглощениям</a>. Тем не менее, по утверждениям менеджмента, компания сохраняет приоритет на развитие CRM и не планирует продвигать дополнительные решения в среде своих подписчиков, предпочитая развитие партнёрских отношений с <a href="/wiki/%D0%9D%D0%B5%D0%B7%D0%B0%D0%B2%D0%B8%D1%81%D0%B8%D0%BC%D1%8B%D0%B9_%D0%BF%D0%BE%D1%81%D1%82%D0%B0%D0%B2%D1%89%D0%B8%D0%BA_%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%BD%D0%BE%D0%B3%D0%BE_%D0%BE%D0%B1%D0%B5%D1%81%D0%BF%D0%B5%D1%87%D0%B5%D0%BD%D0%B8%D1%8F" title="Независимый поставщик программного обеспечения">независимыми разработчиками</a> на платформе <i>force.com</i> созданию собственных новых <a href="/wiki/%D0%9F%D1%80%D0%B8%D0%BA%D0%BB%D0%B0%D0%B4%D0%BD%D0%BE%D0%B5_%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%BD%D0%BE%D0%B5_%D0%BE%D0%B1%D0%B5%D1%81%D0%BF%D0%B5%D1%87%D0%B5%D0%BD%D0%B8%D0%B5" title="Прикладное программное обеспечение">приложений</a><sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup>. В <a href="/wiki/2011_%D0%B3%D0%BE%D0%B4" title="2011 год">2011 году</a> компания запустила <a href="/wiki/%D0%9E%D0%B1%D0%BB%D0%B0%D1%87%D0%BD%D1%8B%D0%B5_%D0%B2%D1%8B%D1%87%D0%B8%D1%81%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F" title="Облачные вычисления">облачную</a> <a href="/wiki/%D0%A1%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0_%D1%83%D0%BF%D1%80%D0%B0%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F_%D0%B1%D0%B0%D0%B7%D0%B0%D0%BC%D0%B8_%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85" title="Система управления базами данных">систему управления базами данных</a> <i>Database.com</i>, отмечаемую как исторически вторая (после <a href="/wiki/SQL_Azure" title="SQL Azure">SQL Azure</a> корпорации <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a>) значимая облачная СУБД<sup class="reference" id="cite_ref-reuters-on-database_21-0"><a href="#cite_note-reuters-on-database-21">[21]</a></sup>.
</p><p>В <a href="/wiki/2011_%D0%B3%D0%BE%D0%B4" title="2011 год">2011 году</a> компания неоднократно отмечена как один из лучших работодателей (41-я позиция в рейтинге Glassdor<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup>, 52-я — в рейтинге журнала <a href="/wiki/Fortune_(%D0%B6%D1%83%D1%80%D0%BD%D0%B0%D0%BB)" title="Fortune (журнал)">Fortune</a><sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup>, 10-е место в рейтинге лучших мест для работы в сфере информационных технологий по версии журнала <a class="mw-redirect" href="/wiki/Computerworld" title="Computerworld">Computerworld</a><sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup>). Примечательно также первое место в рейтинге журнала <a href="/wiki/Fortune_(%D0%B6%D1%83%D1%80%D0%BD%D0%B0%D0%BB)" title="Fortune (журнал)">Fortune</a> <i>25 top-paying companies</i> (<i>25 лучше всего платящих сотрудникам работодателей</i>), особо указывается средняя годовая зарплата $318 323 для сотрудника, ответственного за работу с выделенным крупным клиентом (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">account executive</span>)<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup>.
</p>
<h3><span id=".D0.9F.D0.BE.D0.B3.D0.BB.D0.BE.D1.89.D0.B5.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="Поглощения">Поглощения</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Salesforce&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Поглощения»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Salesforce&amp;action=edit&amp;section=4" title="Редактировать код раздела «Поглощения»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>

<p>Один из основателей компании, Мёлленхофф, утверждает, что изначальной целью было создание продукта, доступного по цене для средних и малых предприятий, но сопоставимого по качеству с <a class="mw-redirect" href="/wiki/Hi-End" title="Hi-End">Hi-End</a>-продуктами <a class="mw-redirect" href="/wiki/Oracle_Siebel_CRM" title="Oracle Siebel CRM">Siebel CRM</a> и <a class="new" href="/w/index.php?title=Clarify_CRM&amp;action=edit&amp;redlink=1" title="Clarify CRM (страница отсутствует)">Clarify CRM</a><sup class="reference" id="cite_ref-42"><a href="#cite_note-42">[42]</a></sup>. Однако, на момент <a href="/wiki/2011_%D0%B3%D0%BE%D0%B4" title="2011 год">2011 года</a>, по утверждению самой компании, подписаны на CRM-систему от Salesforce.com такие крупные заказчики, как <a href="/wiki/Google_(%D0%BA%D0%BE%D0%BC%D0%BF%D0%B0%D0%BD%D0%B8%D1%8F)" title="Google (компания)">Google</a>, <a href="/wiki/Dell" title="Dell">Dell</a>, <a href="/wiki/Toyota" title="Toyota">Toyota</a>, <a href="/wiki/Cisco" title="Cisco">Cisco</a>, <a href="/wiki/Hitachi" title="Hitachi">Hitachi</a><sup class="reference" id="cite_ref-43"><a href="#cite_note-43">[43]</a></sup>. <a href="/wiki/PC_Magazine" title="PC Magazine">PC Magazine</a> в обзоре CRM указывает на возможность использования решений Salesforce.com как в малом, так и в крупном бизнесе, при этом специально отмечая существенно более высокие цены, чем у конкурентов<sup class="reference" id="cite_ref-pcmag-on-salescloud_44-0"><a href="#cite_note-pcmag-on-salescloud-44">[44]</a></sup>. Всего заявляется о 87 200 заказчиках на начало <a href="/wiki/2010_%D0%B3%D0%BE%D0%B4" title="2010 год">2010 года</a><sup class="reference" id="cite_ref-sec-2010_45-0"><a href="#cite_note-sec-2010-45">[45]</a></sup>.

<p>По оценке <a href="/wiki/Gartner" title="Gartner">Gartner</a> по результатам 2012 года<sup class="reference" id="cite_ref-lead-2013_7-1"><a href="#cite_note-lead-2013-7">[7]</a></sup> компания вышла на первое место на 18-миллиардном мировом рынке <a class="mw-redirect" href="/wiki/CRM" title="CRM">CRM</a>-систем с долей 14 %, опережая <a href="/wiki/SAP" title="SAP">SAP</a> (12,9 %) и <a href="/wiki/Oracle" title="Oracle">Oracle</a> (11,1 %), показывая при этом более быстрый рост, чем рынок в целом и конкуренты (по состоянию на 2008 год компания была третьей на рынке, в два раза отставая от SAP и в полтора раза от Oracle<sup class="reference" id="cite_ref-46"><a href="#cite_note-46">[46]</a></sup><sup class="reference" id="cite_ref-47"><a href="#cite_note-47">[47]</a></sup>). На подсегменте этого рынка — среди CRM-систем, предоставляемых по модели <a class="mw-redirect" href="/wiki/SaaS" title="SaaS">SaaS</a>, — Salesforce.com, по оценкам отдельных специалистов, владеет более чем половиной рынка<sup class="reference" id="cite_ref-48"><a href="#cite_note-48">[48]</a></sup>, среди других заметных игроков в сегменте CRM по подписке наблюдатели выделяют также <a href="/wiki/NetSuite" title="NetSuite">NetSuite</a> и <span data-interwiki-article="RightNow Technologies" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=RightNow&amp;action=edit&amp;redlink=1" title="RightNow (страница отсутствует)">RightNow</a></span><span class="noprint" style="font-style:normal; font-weight:normal;"><sup><a class="extiw" href="https://en.wikipedia.org/wiki/RightNow_Technologies" title="en:RightNow Technologies"><span title="RightNow Technologies — версия статьи «RightNow» на английском языке">[en]</span></a></sup></span><sup class="reference" id="cite_ref-49"><a href="#cite_note-49">[49]</a></sup>, притом первая контролируется основателем Oracle <a href="/wiki/%D0%AD%D0%BB%D0%BB%D0%B8%D1%81%D0%BE%D0%BD,_%D0%9B%D0%B0%D1%80%D1%80%D0%B8" title="Эллисон, Ларри">Ларри Эллисоном</a>, а вторая — поглощена Oracle в 2011 году<sup class="reference" id="cite_ref-rightnow_50-0"><a href="#cite_note-rightnow-50">[50]</a></sup>.
</p><p>Технические эксперты отмечают компанию как «облачный сервис-провайдер № 2» (после <a class="mw-redirect" href="/wiki/Amazon.com" title="Amazon.com">Amazon.com</a> и перед <a class="mw-redirect" href="/wiki/Google,_Inc." title="Google, Inc.">Google</a>)<sup class="reference" id="cite_ref-51"><a href="#cite_note-51">[51]</a></sup><sup class="reference" id="cite_ref-52"><a href="#cite_note-52">[52]</a></sup>. Биржевые наблюдатели относят компанию к «основным игрокам рынка облачных вычислений» в одном ряду с <a href="/wiki/Oracle" title="Oracle">Oracle</a>, <a href="/wiki/SAP" title="SAP">SAP</a>, <a class="mw-redirect" href="/wiki/Google,_Inc." title="Google, Inc.">Google</a>, <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a>, <a class="mw-redirect" href="/wiki/Amazon.com" title="Amazon.com">Amazon.com</a><sup class="reference" id="cite_ref-53"><a href="#cite_note-53">[53]</a></sup>. Запуск в <a href="/wiki/2011_%D0%B3%D0%BE%D0%B4" title="2011 год">2011 году</a> <a href="/wiki/%D0%9E%D0%B1%D0%BB%D0%B0%D1%87%D0%BD%D1%8B%D0%B5_%D0%B2%D1%8B%D1%87%D0%B8%D1%81%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F" title="Облачные вычисления">облачной</a> <a href="/wiki/%D0%A1%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0_%D1%83%D0%BF%D1%80%D0%B0%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F_%D0%B1%D0%B0%D0%B7%D0%B0%D0%BC%D0%B8_%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85" title="Система управления базами данных">системы управления базами данных</a> <i>Database.com</i> был оценён как претензия компании на участие в борьбе за двадцатиодномиллиардный рынок СУБД, на котором доминирует <a href="/wiki/Oracle" title="Oracle">Oracle</a><sup class="reference" id="cite_ref-reuters-on-database_21-1"><a href="#cite_note-reuters-on-database-21">[21]</a></sup>.
</p>

<p>Штаб-квартира компании расположена в центре <a href="/wiki/%D0%A1%D0%B0%D0%BD-%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%81%D0%BA%D0%BE" title="Сан-Франциско">Сан-Франциско</a>. Международные штаб-квартиры, по регионам ответственности:
</p>
<ul><li><a href="/wiki/%D0%A2%D0%BE%D1%80%D0%BE%D0%BD%D1%82%D0%BE" title="Торонто">Торонто</a> — <a href="/wiki/%D0%9A%D0%B0%D0%BD%D0%B0%D0%B4%D0%B0" title="Канада">Канада</a>;</li>
<li><a href="/wiki/%D0%9C%D0%B5%D1%85%D0%B8%D0%BA%D0%BE" title="Мехико">Мехико</a> — <a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%90%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B0" title="Латинская Америка">Латинская Америка</a>;</li>
<li><a href="/wiki/%D0%9C%D0%BE%D1%80%D0%B6_(%D0%B3%D0%BE%D1%80%D0%BE%D0%B4)" title="Морж (город)">Морж (Швейцария)</a> — <a href="/wiki/%D0%95%D0%B2%D1%80%D0%BE%D0%BF%D0%B0" title="Европа">Европа</a>, <a href="/wiki/%D0%91%D0%BB%D0%B8%D0%B6%D0%BD%D0%B8%D0%B9_%D0%92%D0%BE%D1%81%D1%82%D0%BE%D0%BA" title="Ближний Восток">Ближний Восток</a> и <a href="/wiki/%D0%90%D1%84%D1%80%D0%B8%D0%BA%D0%B0" title="Африка">Африка</a>;</li>
<li><a href="/wiki/%D0%A1%D0%B8%D0%BD%D0%B3%D0%B0%D0%BF%D1%83%D1%80" title="Сингапур">Сингапур</a> — <a href="/wiki/%D0%90%D0%B7%D0%B8%D0%B0%D1%82%D1%81%D0%BA%D0%BE-%D0%A2%D0%B8%D1%85%D0%BE%D0%BE%D0%BA%D0%B5%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%80%D0%B5%D0%B3%D0%B8%D0%BE%D0%BD" title="Азиатско-Тихоокеанский регион">Азиатско-Тихоокеанский регион</a>.</li></ul>
<p>Всего, на <a href="/wiki/2011_%D0%B3%D0%BE%D0%B4" title="2011 год">2011 год</a>, офисы компании имеются в 25 странах, крупнейшие офисы расположены в <a href="/wiki/%D0%A1%D0%B0%D0%BD-%D0%9C%D0%B0%D1%82%D0%B5%D0%BE_(%D0%9A%D0%B0%D0%BB%D0%B8%D1%84%D0%BE%D1%80%D0%BD%D0%B8%D1%8F)" title="Сан-Матео (Калифорния)">Сан-Матео</a>, <a href="/wiki/%D0%9B%D0%BE%D0%BD%D0%B4%D0%BE%D0%BD" title="Лондон">Лондоне</a>, <a href="/wiki/%D0%9D%D1%8C%D1%8E-%D0%99%D0%BE%D1%80%D0%BA" title="Нью-Йорк">Нью-Йорке</a>, <a href="/wiki/%D0%A2%D0%BE%D1%80%D0%BE%D0%BD%D1%82%D0%BE" title="Торонто">Торонто</a>, <a href="/wiki/%D0%A1%D0%B8%D0%B4%D0%BD%D0%B5%D0%B9" title="Сидней">Сиднее</a>.
</p><p>На международные продажи на момент <a href="/wiki/2010_%D0%B3%D0%BE%D0%B4" title="2010 год">2010 года</a> приходится около 30 % от общего объёма выручки<sup class="reference" id="cite_ref-sec-2010_45-1"><a href="#cite_note-sec-2010-45">[45]</a></sup>. Решения Salesforce.com переведены на 16 языков<sup class="reference" id="cite_ref-58"><a href="#cite_note-58">[58]</a></sup>, в том числе на русский<sup class="reference" id="cite_ref-59"><a href="#cite_note-59">[59]</a></sup>.

<h3><span class="mw-headline" id="Sales_Cloud">Sales Cloud</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Salesforce&amp;veaction=edit&amp;section=12" title="Редактировать раздел «Sales Cloud»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Salesforce&amp;action=edit&amp;section=12" title="Редактировать код раздела «Sales Cloud»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Функциональная часть CRM-системы для поддержки процессов продаж выделена компанией в продукт под наименованием <i>Sales Cloud</i> (<a href="/wiki/%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Русский язык">рус.</a> <span lang="ru" style="font-style:italic;">облако продаж</span>). Специалисты по продажам компании-подписчика с помощью Sales Cloud ведут данные о потенциальных клиентах, взаимодействуют с ними, выставляют им счета, взаимодействуют с другими специалистами по продажам, проводят маркетинговые кампании, анализируют и <a href="/wiki/%D0%A1%D0%B5%D0%B3%D0%BC%D0%B5%D0%BD%D1%82%D0%B0%D1%86%D0%B8%D1%8F_%D1%80%D1%8B%D0%BD%D0%BA%D0%B0" title="Сегментация рынка">сегментируют</a> клиентскую базу<sup class="reference" id="cite_ref-pcmag-on-salescloud_44-1"><a href="#cite_note-pcmag-on-salescloud-44">[44]</a></sup>. Выделено пять уровней подписки с различной стоимостью: от $5 за пользователя в месяц до $250 за пользователя в месяц. Доступные заказчику функциональные возможности зависят от уровня подписки, так, например, поддержка работы офлайн и потоков утверждений и согласований процессов продаж доступна только при подписке уровня предприятия ($125 за пользователя в месяц)<sup class="reference" id="cite_ref-pcmag-on-salescloud_44-2"><a href="#cite_note-pcmag-on-salescloud-44">[44]</a></sup><sup class="reference" id="cite_ref-price_60-0"><a href="#cite_note-price-60">[60]</a></sup>.
</p>
<h3><span class="mw-headline" id="Service_Cloud">Service Cloud</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Salesforce&amp;veaction=edit&amp;section=13" title="Редактировать раздел «Service Cloud»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Salesforce&amp;action=edit&amp;section=13" title="Редактировать код раздела «Service Cloud»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Service Cloud (<a href="/wiki/%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Русский язык">рус.</a> <span lang="ru" style="font-style:italic;">облако услуг</span>) — функциональная часть CRM для обеспечения процессов обслуживания и поддержки клиентов. Система позволяет компаниям-подписчикам построить сайт самообслуживания и поддержки своих клиентов, специалисты по поддержке получают возможность взаимодействовать с клиентами, вести <a href="/wiki/%D0%91%D0%B0%D0%B7%D0%B0_%D0%B7%D0%BD%D0%B0%D0%BD%D0%B8%D0%B9" title="База знаний">базы знаний</a> (Salesforce.com использует для этой функциональности термин <i>knowledge as a service</i> — <a href="/wiki/%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Русский язык">рус.</a> <span lang="ru" style="font-style:italic;">знание как услуга</span>), управленцы компании-подписчика получают инструменты для анализа эффективности сервисных процессов<sup class="reference" id="cite_ref-tc-on-service-cloud_19-1"><a href="#cite_note-tc-on-service-cloud-19">[19]</a></sup>. Начиная с <a href="/wiki/2011_%D0%B3%D0%BE%D0%B4" title="2011 год">2011 года</a> система оборудована с широким спектром инструментов <a href="/wiki/%D0%A1%D0%BE%D1%86%D0%B8%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_CRM" title="Социальный CRM">социального CRM</a>, таких как средства интеграции с <a class="mw-redirect" href="/wiki/%D0%A1%D0%BE%D1%86%D0%B8%D0%B0%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D1%81%D0%B5%D1%82%D1%8C_(%D0%98%D0%BD%D1%82%D0%B5%D1%80%D0%BD%D0%B5%D1%82)" title="Социальная сеть (Интернет)">социальными сетями</a>, <a class="mw-redirect" href="/wiki/%D0%A4%D0%BE%D1%80%D1%83%D0%BC_(%D0%B8%D0%BD%D1%82%D0%B5%D1%80%D0%BD%D0%B5%D1%82)" title="Форум (интернет)">форумами</a>, <a href="/wiki/%D0%91%D0%BB%D0%BE%D0%B3" title="Блог">блогами</a>, <a href="/wiki/%D0%9C%D0%B8%D0%BA%D1%80%D0%BE%D0%B1%D0%BB%D0%BE%D0%B3%D0%B8%D0%BD%D0%B3" title="Микроблогинг">микроблогами</a>, <a href="/wiki/%D0%97%D0%B0%D0%BA%D0%BB%D0%B0%D0%B4%D0%BA%D0%B0_(%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80)" title="Закладка (компьютер)">интернет-закладками</a> (например, клиенты компании-подписчика могут взаимодействовать со службой поддержки на <a href="/wiki/Facebook" title="Facebook">Facebook</a>, при этом все результаты взаимодействия будут в CRM-системе, и автоматически попадать в нужные каналы в социальной сети прозрачно для клиента)<sup class="reference" id="cite_ref-61"><a href="#cite_note-61">[61]</a></sup>.
</p><p>Выделяется три уровня подписки с различной стоимостью за одного пользователя в месяц: профессиональный ($65), уровня предприятия ($135) и неограниченный ($260). Доступные подписчику функциональные возможности зависят от уровня подписки, например, начиная с уровня предприятия доступны функции интеграции с системами <a href="/wiki/%D0%9A%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D1%82%D0%B5%D0%BB%D0%B5%D1%84%D0%BE%D0%BD%D0%B8%D1%8F" title="Компьютерная телефония">компьютерной телефонии</a><sup class="reference" id="cite_ref-price_60-1"><a href="#cite_note-price-60">[60]</a></sup>.
</p>
<h3><span class="mw-headline" id="Force.com">Force.com</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Salesforce&amp;veaction=edit&amp;section=14" title="Редактировать раздел «Force.com»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Salesforce&amp;action=edit&amp;section=14" title="Редактировать код раздела «Force.com»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p><b>Force.com</b> — программная платформа, на которой разработаны <i>Sales Cloud</i> и <i>Service Cloud</i>, предоставляемая подписчикам для самостоятельной разработки приложений и расширений для CRM-системы Salesforce.com.
</p><p><span id="Apex"></span>
Для разработки пользователями используется собственный <a href="/wiki/Java" title="Java">Java</a>-подобный язык <i>Apex</i> и собственное средство проектирования <i>Visualforce</i> с выходным форматом на основе <a href="/wiki/XML" title="XML">XML</a>, обеспечивающее генерацию пользовательских <a href="/wiki/HTML" title="HTML">HTML</a>/<a href="/wiki/AJAX" title="AJAX">AJAX</a>- и <a class="mw-redirect" href="/wiki/Adobe_Flex" title="Adobe Flex">Flex</a>-интерфейсов. Платформа предоставляется исключительно по подписке, в рамках концепции <a class="mw-redirect" href="/wiki/Platform_as_a_service" title="Platform as a service">PaaS</a>. В зависимости от уровня подписки доступны различные технические возможности. Так, в бесплатной версии подписчики могут создать не более десяти сущностей, а в неограниченной версии с ценой $75 на пользователя в месяц — до 2000<sup class="reference" id="cite_ref-price_60-2"><a href="#cite_note-price-60">[60]</a></sup>. Подписчики могут размещать разработанные приложения на платформе Force.com в специальном каталоге — AppExchange — и предлагать свои разработки другим заказчикам, в том числе на коммерческой основе<sup class="reference" id="cite_ref-62"><a href="#cite_note-62">[62]</a></sup>.
</p><p>В качестве <a href="/wiki/%D0%A1%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0_%D1%83%D0%BF%D1%80%D0%B0%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F_%D0%B1%D0%B0%D0%B7%D0%B0%D0%BC%D0%B8_%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85" title="Система управления базами данных">системы управления базами данных</a> платформа использует три <a href="/wiki/%D0%A0%D0%B5%D0%BF%D0%BB%D0%B8%D0%BA%D0%B0%D1%86%D0%B8%D1%8F_(%D0%B2%D1%8B%D1%87%D0%B8%D1%81%D0%BB%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D1%82%D0%B5%D1%85%D0%BD%D0%B8%D0%BA%D0%B0)" title="Репликация (вычислительная техника)">реплицируемых</a> <a href="/wiki/%D0%9A%D0%BB%D0%B0%D1%81%D1%82%D0%B5%D1%80_(%D0%B3%D1%80%D1%83%D0%BF%D0%BF%D0%B0_%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BE%D0%B2)" title="Кластер (группа компьютеров)">кластера</a> <a href="/wiki/Oracle_RAC" title="Oracle RAC">Oracle RAC</a> из восьми <a href="/wiki/%D0%A1%D0%B5%D1%80%D0%B2%D0%B5%D1%80_(%D0%B0%D0%BF%D0%BF%D0%B0%D1%80%D0%B0%D1%82%D0%BD%D0%BE%D0%B5_%D0%BE%D0%B1%D0%B5%D1%81%D0%BF%D0%B5%D1%87%D0%B5%D0%BD%D0%B8%D0%B5)" title="Сервер (аппаратное обеспечение)">узлов</a> каждый, кластеры расположены в трёх удалённых друг от друга <a href="/wiki/%D0%94%D0%B0%D1%82%D0%B0-%D1%86%D0%B5%D0%BD%D1%82%D1%80" title="Дата-центр">центрах обработки данных</a>. В одной <a href="/wiki/%D0%A1%D1%85%D0%B5%D0%BC%D0%B0_%D0%B1%D0%B0%D0%B7%D1%8B_%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85" title="Схема базы данных">схеме</a> <a href="/wiki/Oracle_Database" title="Oracle Database">Oracle Database</a> обрабатываются данные сразу нескольких компаний-подписчиков, используется стабильная схема данных, предподготовленная к расширению дополнительными объектами таким образом, что в одних и тех же <a class="mw-redirect" href="/wiki/%D0%A2%D0%B0%D0%B1%D0%BB%D0%B8%D1%86%D0%B0_(%D0%B1%D0%B0%D0%B7%D1%8B_%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85)" title="Таблица (базы данных)">таблицах</a> хранятся данные различных подписчиков, независимо от различия в специфических атрибутах объектов для различных подписчиков. Широко используется <a href="/wiki/%D0%A1%D0%B5%D0%BA%D1%86%D0%B8%D0%BE%D0%BD%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5" title="Секционирование">секционирование</a> таблиц базы данных<sup class="reference" id="cite_ref-63"><a href="#cite_note-63">[63]</a></sup>. Утверждается<sup class="reference" id="cite_ref-64"><a href="#cite_note-64">[64]</a></sup>, что на момент <a href="/wiki/2010_%D0%B3%D0%BE%D0%B4" title="2010 год">2010 года</a> платформу и все системы, на ней работавшие, обслуживали всего пять <a href="/wiki/%D0%90%D0%B4%D0%BC%D0%B8%D0%BD%D0%B8%D1%81%D1%82%D1%80%D0%B0%D1%82%D0%BE%D1%80_%D0%B1%D0%B0%D0%B7_%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85" title="Администратор баз данных">администраторов баз данных</a>.
</p>
<h3><span class="mw-headline" id="Database.com">Database.com</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Salesforce&amp;veaction=edit&amp;section=15" title="Редактировать раздел «Database.com»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Salesforce&amp;action=edit&amp;section=15" title="Редактировать код раздела «Database.com»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Под брендом и соответствующим доменным именем <i>Database.com</i> с <a href="/wiki/2011_%D0%B3%D0%BE%D0%B4" title="2011 год">2011 года</a> компания предлагает подписчикам <a href="/wiki/%D0%9E%D0%B1%D0%BB%D0%B0%D1%87%D0%BD%D1%8B%D0%B5_%D0%B2%D1%8B%D1%87%D0%B8%D1%81%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F" title="Облачные вычисления">облачную</a> <a href="/wiki/%D0%A1%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0_%D1%83%D0%BF%D1%80%D0%B0%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F_%D0%B1%D0%B0%D0%B7%D0%B0%D0%BC%D0%B8_%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85" title="Система управления базами данных">систему управления базами данных</a>, являющуюся также слоем обработки данных платформы <i>Force.com</i>. При этом заказчики могут использовать базы данных, развёрнутые на этом сервисе, и как слой данных для приложений на платформе Force.com, так и целиком для сторонних приложений. Salesforce.com заявляет об <i>эластичности</i> обработки данных средствами СУБД в том смысле, что в случае неожиданного роста нагрузки на <a href="/wiki/%D0%91%D0%B0%D0%B7%D0%B0_%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85" title="База данных">базу данных</a>, ей немедленно будут предоставлены необходимые вычислительные ресурсы.
</p><p>Database.com реализована на технологической основе <a href="/wiki/Oracle_Database" title="Oracle Database">Oracle Database</a><sup class="reference" id="cite_ref-reg-on-database_65-0"><a href="#cite_note-reg-on-database-65">[65]</a></sup>. Подписчики с Database.com могут работать точно также, как с большинством обычных <a href="/wiki/%D0%A0%D0%B5%D0%BB%D1%8F%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D0%B0%D1%8F_%D0%A1%D0%A3%D0%91%D0%94" title="Реляционная СУБД">реляционных СУБД</a> с использованием интерфейсов доступа <a href="/wiki/ODBC" title="ODBC">ODBC</a>, <a class="mw-redirect" href="/wiki/JDBC" title="JDBC">JDBC</a> и языка запросов <a href="/wiki/SQL" title="SQL">SQL</a>. Однако Salesforce.com утверждает, что Database.com оптимизирована в первую очередь для работы с <a href="/wiki/%D0%92%D0%B5%D0%B1-%D1%81%D0%BB%D1%83%D0%B6%D0%B1%D0%B0" title="Веб-служба">веб-службами</a> по <a href="/wiki/SOAP" title="SOAP">SOAP</a> и <a href="/wiki/REST" title="REST">REST</a>. Поддерживаются <a href="/wiki/%D0%A2%D1%80%D0%B8%D0%B3%D0%B3%D0%B5%D1%80_(%D0%B1%D0%B0%D0%B7%D1%8B_%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85)" title="Триггер (базы данных)">триггеры</a>, <a href="/wiki/%D0%A5%D1%80%D0%B0%D0%BD%D0%B8%D0%BC%D0%B0%D1%8F_%D0%BF%D1%80%D0%BE%D1%86%D0%B5%D0%B4%D1%83%D1%80%D0%B0" title="Хранимая процедура">хранимые процедуры</a>, в платных версиях подписки — возможна реализация моделей безопасности на уровне записей. Кроме API для веб-служб, предлагаются специализированные выделенные библиотеки для <a href="/wiki/Java" title="Java">Java</a>, <a href="/wiki/PHP" title="PHP">PHP</a>, <a href="/wiki/Ruby" title="Ruby">Ruby</a>, а также инструментальные наборы для <a class="mw-redirect" href="/wiki/Windows_Azure" title="Windows Azure">Microsoft Azure</a>, <a href="/wiki/Android" title="Android">Google Android</a>, <a class="mw-redirect" href="/wiki/Apple_iOS" title="Apple iOS">Apple iOS</a>, <a class="mw-redirect" href="/wiki/Adobe_Flex" title="Adobe Flex">Adobe Flex</a>. Заявляется, что Database.com на конец <a href="/wiki/2010_%D0%B3%D0%BE%D0%B4" title="2010 год">2010 года</a> содержит более 20 млрд записей и обрабатывает 25 млрд транзакций в квартал со средним временем отклика 300 мс<sup class="reference" id="cite_ref-rww-on-database_66-0"><a href="#cite_note-rww-on-database-66">[66]</a></sup>.
</p><p>Варианты подписки на <i>Database.com</i> различаются по стоимости и доступным функциональным возможностям. Так, в бесплатном варианте заказчикам доступно до 100 тыс. записей в базу данных, 50 тыс. транзакций в месяц и заведение трёх пользователей, за $10 на пользователя в месяц предоставляется безопасность на уровне записи (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">row-level security</span>), а дополнительные 150 тыс. транзакций на экземпляр обойдутся подписчику в $10<sup class="reference" id="cite_ref-price_60-3"><a href="#cite_note-price-60">[60]</a></sup><sup class="reference" id="cite_ref-rww-on-database_66-1"><a href="#cite_note-rww-on-database-66">[66]</a></sup>.
</p>
<h3><span class="mw-headline" id="Chatter">Chatter</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Salesforce&amp;veaction=edit&amp;section=16" title="Редактировать раздел «Chatter»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Salesforce&amp;action=edit&amp;section=16" title="Редактировать код раздела «Chatter»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:2011_Salesforce_clouds_CNIT_Paris_Messim_Makhloufi_%26_Dalil_Djidel.jpg"><img class="mw-file-element" data-file-height="2112" data-file-width="2816" decoding="async" height="150" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/2011_Salesforce_clouds_CNIT_Paris_Messim_Makhloufi_%26_Dalil_Djidel.jpg/200px-2011_Salesforce_clouds_CNIT_Paris_Messim_Makhloufi_%26_Dalil_Djidel.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/2011_Salesforce_clouds_CNIT_Paris_Messim_Makhloufi_%26_Dalil_Djidel.jpg/300px-2011_Salesforce_clouds_CNIT_Paris_Messim_Makhloufi_%26_Dalil_Djidel.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/2011_Salesforce_clouds_CNIT_Paris_Messim_Makhloufi_%26_Dalil_Djidel.jpg/400px-2011_Salesforce_clouds_CNIT_Paris_Messim_Makhloufi_%26_Dalil_Djidel.jpg 2x" width="200"/></a><figcaption>Представление Chatter на конференции в <a href="/wiki/%D0%9F%D0%B0%D1%80%D0%B8%D0%B6" title="Париж">Париже</a>, <a href="/wiki/2011_%D0%B3%D0%BE%D0%B4" title="2011 год">2011 год</a></figcaption></figure>
<p>Решение, сформированное на основе сервисов поглощённой компании <i>Activia Live Chat</i>, предоставляется по подписке корпоративным заказчикам для организации внутрикорпоративного <a class="mw-redirect" href="/wiki/Groupware" title="Groupware">средства групповой работы</a>. В бесплатной версии сотрудники могут обмениваться сообщениями, организовывать групповые обсуждения, обмениваться файлами, совместно работать над контентом, поддержан доступ как из стандартных браузеров, так и с мобильных устройств. В версии за $15 в месяц на сотрудника, кроме стандартных возможностей групповой работы, предоставляются ещё элементарные <a class="mw-redirect" href="/wiki/CRM" title="CRM">CRM</a>-функции: ведение, разделение доступа и совместная работа с клиентской базой, а также поддерживается возможность программных расширений и потоков операций<sup class="reference" id="cite_ref-price_60-4"><a href="#cite_note-price-60">[60]</a></sup>.
</p>
<h3><span class="mw-headline" id="Data.com">Data.com</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Salesforce&amp;veaction=edit&amp;section=17" title="Редактировать раздел «Data.com»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Salesforce&amp;action=edit&amp;section=17" title="Редактировать код раздела «Data.com»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p><span id="Jigsaw"></span><b>Data.com</b> — совместно издаваемый бизнес-каталог, содержащий информацию о предприятиях, их продуктах, контактную информацию о сотрудниках. Был приобретён вместе с компанией <b>Jigsaw</b> в <a href="/wiki/2010_%D0%B3%D0%BE%D0%B4" title="2010 год">2010 году</a>, в 2011 году получил торговую марку и доменное имя <i>Data.com</i>. Каталог интегрирован с CRM-системой, и подписчики Sales Cloud или Service Cloud могут использовать Jigsaw как источник основных данных о клиентах.
</p><p>Наблюдатели отмечают эффективность бизнес-каталога в сравнении со многими прочими базами данных о предприятиях, прежде всего, за счёт чёткой ориентации исключительно на структурированные деловые данные: например, в Jigsaw запрещено указывать адреса в доменах сторонних служб электронной почты, и 70 % существующей контактной информации в системе — прямые телефонные номера<sup class="reference" id="cite_ref-lohr-on-jigsaw_67-0"><a href="#cite_note-lohr-on-jigsaw-67">[67]</a></sup>. Каталог, будучи основанным на принципах <a href="/wiki/%D0%9A%D1%80%D0%B0%D1%83%D0%B4%D1%81%D0%BE%D1%80%D1%81%D0%B8%D0%BD%D0%B3" title="Краудсорсинг">краудсорсинга</a> — массовом общественном издании — во многих источниках сравнивается с <a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F" title="Википедия">Википедией</a><sup class="reference" id="cite_ref-nytimes-on-jigsaw_27-1"><a href="#cite_note-nytimes-on-jigsaw-27">[27]</a></sup><sup class="reference" id="cite_ref-lohr-on-jigsaw_67-1"><a href="#cite_note-lohr-on-jigsaw-67">[67]</a></sup><sup class="reference" id="cite_ref-68"><a href="#cite_note-68">[68]</a></sup>. Начиная с 2011 года каталог интегрирован с редакционно-контролируемой базой данных организаций агентства <a href="/wiki/Dun_%26_Bradstreet" title="Dun &amp; Bradstreet">Dun &amp; Bradstreet</a>, таким образом, компания пытается скомбинировать два типа источников данных в одном продукте, при этом, по мнению обозревателей, качество данных из общественной части каталога выше, так как Dun &amp; Bradstreet заинтересован в полных и качественных сведениях только о клиентах, которые ему платят<sup class="reference" id="cite_ref-dnb_69-0"><a href="#cite_note-dnb-69">[69]</a></sup>.
</p><p>Каталог предоставляется с различными уровнями подписки, стоимость подписки зависит от объёма доступных данных и функциональных возможностей по работе с ними. Например, есть вариант подписки с ценой 99<a href="/wiki/%D0%A6%D0%B5%D0%BD%D1%82" title="Цент">¢</a> за одну запись, а при оплате $40 тыс. в месяц предоставляется доступ ко всей базе данных<sup class="reference" id="cite_ref-price_60-5"><a href="#cite_note-price-60">[60]</a></sup>.
</p>
<h2><span id=".D0.9A.D1.80.D0.B8.D1.82.D0.B8.D0.BA.D0.B0"></span><span class="mw-headline" id="Критика">Критика</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Salesforce&amp;veaction=edit&amp;section=18" title="Редактировать раздел «Критика»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Salesforce&amp;action=edit&amp;section=18" title="Редактировать код раздела «Критика»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>В <a href="/wiki/%D0%9D%D0%BE%D1%8F%D0%B1%D1%80%D1%8C_2007_%D0%B3%D0%BE%D0%B4%D0%B0" title="Ноябрь 2007 года">ноябре 2007 года</a> часть клиентов Salesforce.com подверглись <a href="/wiki/%D0%A4%D0%B8%D1%88%D0%B8%D0%BD%D0%B3" title="Фишинг">фишинговой</a> атаке, в результате которой у некоторых из них были украдены пароли и злоумышленники получили доступ к конфиденциальной деловой информации<sup class="reference" id="cite_ref-70"><a href="#cite_note-70">[70]</a></sup><sup class="reference" id="cite_ref-71"><a href="#cite_note-71">[71]</a></sup><sup class="reference" id="cite_ref-72"><a href="#cite_note-72">[72]</a></sup>. Данный случай отмечен как показательный для всей индустрии, демонстрирующий важность глобальной борьбы со <a href="/wiki/%D0%A1%D0%BF%D0%B0%D0%BC" title="Спам">спамом</a><sup class="reference" id="cite_ref-73"><a href="#cite_note-73">[73]</a></sup>.
</p><p>В <a href="/wiki/%D0%AF%D0%BD%D0%B2%D0%B0%D1%80%D1%8C_2009_%D0%B3%D0%BE%D0%B4%D0%B0" title="Январь 2009 года">январе 2009 года</a>, в результате сбоя в <a class="mw-redirect" href="/wiki/%D0%A6%D0%B5%D0%BD%D1%82%D1%80_%D0%BE%D0%B1%D1%80%D0%B0%D0%B1%D0%BE%D1%82%D0%BA%D0%B8_%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85" title="Центр обработки данных">центре обработки данных</a>, сервисы CRM-системы для значительной части заказчиков были недоступны в течение 40 мин., ни один из конечных пользователей нескольких тысяч организаций не мог воспользоваться системой<sup class="reference" id="cite_ref-74"><a href="#cite_note-74">[74]</a></sup>.
</p><p>В <a href="/wiki/2009_%D0%B3%D0%BE%D0%B4" title="2009 год">2009 году</a> <a class="mw-redirect" href="/wiki/%D0%9B%D0%B0%D1%80%D1%80%D0%B8_%D0%AD%D0%BB%D0%BB%D0%B8%D1%81%D0%BE%D0%BD" title="Ларри Эллисон">Ларри Эллисон</a> (основатель и лидер <a href="/wiki/Oracle" title="Oracle">Oracle</a> и изначальный инвестор Salesforce.com, предоставивший Бениоффу $2 млн собственных средств<sup class="reference" id="cite_ref-reg-on-database_65-1"><a href="#cite_note-reg-on-database-65">[65]</a></sup>) иронически отозвался об инновационности решений Salesforce.com и используемой компанией <a href="/wiki/%D0%9E%D0%B1%D0%BB%D0%B0%D1%87%D0%BD%D1%8B%D0%B5_%D0%B2%D1%8B%D1%87%D0%B8%D1%81%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F" title="Облачные вычисления">облачной</a> риторике, отмечая, что их технологический стек полностью построен на вполне традиционных решениях: <a href="/wiki/Oracle_Database" title="Oracle Database">Oracle Database</a> и <a class="mw-redirect" href="/wiki/Oracle_Fusion_Middleware" title="Oracle Fusion Middleware">Oracle Fusion Middleware</a><sup class="reference" id="cite_ref-75"><a href="#cite_note-75">[75]</a></sup>.
</p>
