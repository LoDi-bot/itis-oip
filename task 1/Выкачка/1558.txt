<h3><span id=".D0.A1.D0.B5.D0.BC.D1.8C.D1.8F"></span><span class="mw-headline" id="Семья">Семья</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Семья»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=2" title="Редактировать код раздела «Семья»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Родился в <a href="/wiki/%D0%A0%D1%83%D1%81%D0%B8%D0%BD%D1%8B" title="Русины">русинской</a> семье <a href="/wiki/%D0%A0%D1%83%D1%81%D0%B8%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B3%D1%80%D0%B5%D0%BA%D0%BE%D0%BA%D0%B0%D1%82%D0%BE%D0%BB%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%86%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C" title="Русинская грекокатолическая церковь">грекокатолического</a> <a href="/wiki/%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D0%B8%D0%BA" title="Священник">священника</a> Георгия Товта и его супруги Цецилии близ <a href="/wiki/%D0%9F%D1%80%D0%B5%D1%88%D0%BE%D0%B2" title="Прешов">Прешова</a>, находившегося в то время на территории <a href="/wiki/%D0%98%D1%81%D1%82%D0%BE%D1%80%D0%B8%D1%8F_%D0%92%D0%B5%D0%BD%D0%B3%D1%80%D0%B8%D0%B8" title="История Венгрии">Венгрии</a> в <a href="/wiki/%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B8%D0%B9%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F" title="Австрийская империя">Австрийской империи</a>. Точное место рождения не известно, хотя по некоторым данным им могла быть деревня <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%B1%D1%8B%D0%BB%D0%BD%D0%B8%D1%86%D0%B5_(%D1%80%D0%B0%D0%B9%D0%BE%D0%BD_%D0%A1%D0%B2%D0%B8%D0%B4%D0%BD%D0%B8%D0%BA)" title="Кобылнице (район Свидник)">Кобылнице</a>, <a href="/wiki/%D0%9A%D0%BE%D0%BC%D0%B8%D1%82%D0%B0%D1%82_(%D0%B0%D0%B4%D0%BC%D0%B8%D0%BD%D0%B8%D1%81%D1%82%D1%80%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D0%B0%D1%8F_%D0%B5%D0%B4%D0%B8%D0%BD%D0%B8%D1%86%D0%B0)" title="Комитат (административная единица)">комитат</a> <a href="/wiki/%D0%A8%D0%B0%D1%80%D0%B8%D1%88" title="Шариш">Шарош</a> (ныне <a href="/wiki/%D0%A1%D0%B2%D0%B8%D0%B4%D0%BD%D0%B8%D0%BA_(%D1%80%D0%B0%D0%B9%D0%BE%D0%BD)" title="Свидник (район)">район Свидник</a>, <a href="/wiki/%D0%9F%D1%80%D0%B5%D1%88%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D1%80%D0%B0%D0%B9" title="Прешовский край">Прешовский край</a>, <a href="/wiki/%D0%A1%D0%BB%D0%BE%D0%B2%D0%B0%D0%BA%D0%B8%D1%8F" title="Словакия">Словакия</a>)<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[6]</a></sup>. Крещён в приходе деревни <a class="new" href="/w/index.php?title=%D0%9E%D0%BB%D1%8C%D1%88%D0%B0%D0%B2%D0%B8%D1%86%D0%B0_(%D1%80%D0%B0%D0%B9%D0%BE%D0%BD_%D0%9B%D0%B5%D0%B2%D0%BE%D1%87%D0%B0)&amp;action=edit&amp;redlink=1" title="Ольшавица (район Левоча) (страница отсутствует)">Ольшавица</a>, комитат <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%88" title="Спиш">Сепеш</a> (ныне <a href="/wiki/%D0%9B%D0%B5%D0%B2%D0%BE%D1%87%D0%B0_(%D1%80%D0%B0%D0%B9%D0%BE%D0%BD)" title="Левоча (район)">район Левоча</a>, <a href="/wiki/%D0%9F%D1%80%D0%B5%D1%88%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D1%80%D0%B0%D0%B9" title="Прешовский край">Прешовский край</a>, <a href="/wiki/%D0%A1%D0%BB%D0%BE%D0%B2%D0%B0%D0%BA%D0%B8%D1%8F" title="Словакия">Словакия</a>)<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[7]</a></sup>.
</p><p>Брат Алексея стал грекокатолическим священником, а дядя <span data-interwiki-article="Mikuláš Tóth" data-interwiki-lang="sk"><a class="new" href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B9&amp;action=edit&amp;redlink=1" title="Товт, Николай (страница отсутствует)">Николай</a></span><span class="noprint" style="font-style:normal; font-weight:normal;"><sup><a class="extiw" href="https://sk.wikipedia.org/wiki/Mikul%C3%A1%C5%A1_T%C3%B3th" title="sk:Mikuláš Tóth"><span title="Mikuláš Tóth — версия статьи «Товт, Николай» на словацком языке">[sk]</span></a></sup></span> был <a href="/wiki/%D0%90%D1%80%D1%85%D0%B8%D0%B5%D0%BF%D0%B0%D1%80%D1%85%D0%B8%D1%8F_%D0%9F%D1%80%D0%B5%D1%88%D0%BE%D0%B2%D0%B0" title="Архиепархия Прешова">униатским епископом Прешова</a>.

<p>Начальное образование получил в местной школе и в восьмиклассной гимназии в Прешове, прошёл военную службу, учился в течение года в <a href="/wiki/%D0%9A%D0%B0%D1%82%D0%BE%D0%BB%D0%B8%D1%86%D0%B8%D0%B7%D0%BC" title="Католицизм">римско-католической</a> семинарии в <a href="/wiki/%D0%AD%D1%81%D1%82%D0%B5%D1%80%D0%B3%D0%BE%D0%BC" title="Эстергом">Эстергоме</a>, закончил грекокатолическую семинарию в <a href="/wiki/%D0%A3%D0%B6%D0%B3%D0%BE%D1%80%D0%BE%D0%B4" title="Ужгород">Ужгороде</a> и богословский факультет в <a href="/wiki/%D0%9F%D1%80%D0%B5%D1%88%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D1%83%D0%BD%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D1%82%D0%B5%D1%82" title="Прешовский университет">Прешовском университете</a>. За время учёбы выучил несколько языков (знал <a href="/wiki/%D0%A0%D1%83%D1%81%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Русинский язык">русинский</a>, <a href="/wiki/%D0%92%D0%B5%D0%BD%D0%B3%D0%B5%D1%80%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Венгерский язык">венгерский</a>, <a href="/wiki/%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Русский язык">русский</a>, <a href="/wiki/%D0%9D%D0%B5%D0%BC%D0%B5%D1%86%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Немецкий язык">немецкий</a>, <a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Латинский язык">латинский</a>, отчасти <a href="/wiki/%D0%93%D1%80%D0%B5%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Греческий язык">греческий</a>)<sup class="reference" id="cite_ref-pravenc_9-0"><a href="#cite_note-pravenc-9">[8]</a></sup>.
</p>

<h3><span id=".D0.A1.D0.BB.D1.83.D0.B6.D0.B5.D0.BD.D0.B8.D0.B5_.D0.B2_.D0.93.D1.80.D0.B5.D0.BA.D0.BE.D0.BA.D0.B0.D1.82.D0.BE.D0.BB.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.BE.D0.B9_.D1.86.D0.B5.D1.80.D0.BA.D0.B2.D0.B8"></span><span class="mw-headline" id="Служение_в_Грекокатолической_церкви">Служение в Грекокатолической церкви</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Служение в Грекокатолической церкви»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=4" title="Редактировать код раздела «Служение в Грекокатолической церкви»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>По окончании семинарии женился на дочери священника Розалии Михайлич. <a href="/wiki/18_%D0%B0%D0%BF%D1%80%D0%B5%D0%BB%D1%8F" title="18 апреля">18 апреля</a> <a href="/wiki/1878_%D0%B3%D0%BE%D0%B4" title="1878 год">1878 года</a> был рукоположён своим дядей епископом Прешовским Николаем Товтом в сан священника. Через год овдовел.
</p><p>В мае <a href="/wiki/1879_%D0%B3%D0%BE%D0%B4" title="1879 год">1879 года</a> назначен секретарём Прешовского епископа, заведующим епархиальной канцелярией и приютом для <a href="/wiki/%D0%A1%D0%B8%D1%80%D0%BE%D1%82%D0%B0" title="Сирота">сирот</a>. В <a href="/wiki/1881_%D0%B3%D0%BE%D0%B4" title="1881 год">1881 году</a> стал ректором грекокатолической семинарии в Прешове и профессором кафедры <a href="/wiki/%D0%9A%D0%B0%D0%BD%D0%BE%D0%BD%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%BE%D0%B5_%D0%BF%D1%80%D0%B0%D0%B2%D0%BE" title="Каноническое право">канонического права</a> и церковной истории<sup class="reference" id="cite_ref-pravenc_9-1"><a href="#cite_note-pravenc-9">[8]</a></sup>.
</p><p>В ноябре <a href="/wiki/1889_%D0%B3%D0%BE%D0%B4" title="1889 год">1889 года</a> в ответ на запрос от грекокатолического <a class="external text" href="http://www.stmarysoca.org/about.html" rel="nofollow">прихода святой Марии</a> из американского города <a href="/wiki/%D0%9C%D0%B8%D0%BD%D0%BD%D0%B5%D0%B0%D0%BF%D0%BE%D0%BB%D0%B8%D1%81" title="Миннеаполис">Миннеаполиса</a> (штат <a href="/wiki/%D0%9C%D0%B8%D0%BD%D0%BD%D0%B5%D1%81%D0%BE%D1%82%D0%B0" title="Миннесота">Миннесота</a>) был направлен епископом Прешовским Иоанном Валием в США. Римско-католический архиепископ <a class="extiw" href="https://en.wikipedia.org/wiki/John_Ireland_(archbishop)" title="en:John Ireland (archbishop)"><span title="John Ireland (archbishop) — версия статьи «Айрленд, Джон (епископ)» на английском языке">Джон Айрленд</span></a><span class="noprint" style="font-style:normal; font-weight:normal;"> <span style="display:inline-block; font-size:80%; position:relative;"><span style="font-size:117.6%; margin-left:-0.43em; position:relative; top:0.28em;"> <small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></span><span class="link-ru metadata" style="margin-left:-0.43em; position:absolute; left:0; z-index:1; margin-top:-0.35em;"><span style="visibility:hidden; margin-right:0.099em;"> (</span><span data-interwiki-article="John Ireland (archbishop)" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%90%D0%B9%D1%80%D0%BB%D0%B5%D0%BD%D0%B4,_%D0%94%D0%B6%D0%BE%D0%BD_(%D0%B5%D0%BF%D0%B8%D1%81%D0%BA%D0%BE%D0%BF)&amp;action=edit&amp;redlink=1" title="Айрленд, Джон (епископ) (страница отсутствует)">рус.</a></span></span></span></span>, в епархии которого находился грекокатолический приход, был сторонником американизации и унификации обряда Католической церкви в Америке, а также негативно относился к женатому священству, нелатинскому богослужению и к униатству как таковому<sup class="reference" id="cite_ref-ferencz_3-1"><a href="#cite_note-ferencz-3">[3]</a></sup><sup class="reference" id="cite_ref-connell_10-1"><a href="#cite_note-connell-10">[9]</a></sup><sup class="reference" id="cite_ref-kohanik_11-1"><a href="#cite_note-kohanik-11">[10]</a></sup>. При встрече с отцом Алексием он отказал ему в праве служить на приходе, сославшись на то, что нужды униатского прихода в Миннеаполисе может удовлетворить соседний польский приход.
</p>
<h3><span id=".D0.9F.D0.B5.D1.80.D0.B5.D1.85.D0.BE.D0.B4_.D0.B2_.D0.BF.D1.80.D0.B0.D0.B2.D0.BE.D1.81.D0.BB.D0.B0.D0.B2.D0.B8.D0.B5"></span><span class="mw-headline" id="Переход_в_православие">Переход в православие</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Переход в православие»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=5" title="Редактировать код раздела «Переход в православие»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Louis_Sullivan_-_exterior_-_Holy_Trinity_Russian_%26_Greek_Orthodox_Church,_1121_North_Leavitt_Street,_Chicago,_Cook_County,_IL.jpg"><img class="mw-file-element" data-file-height="4300" data-file-width="3500" decoding="async" height="307" src="//upload.wikimedia.org/wikipedia/commons/thumb/f/fd/Louis_Sullivan_-_exterior_-_Holy_Trinity_Russian_%26_Greek_Orthodox_Church%2C_1121_North_Leavitt_Street%2C_Chicago%2C_Cook_County%2C_IL.jpg/250px-Louis_Sullivan_-_exterior_-_Holy_Trinity_Russian_%26_Greek_Orthodox_Church%2C_1121_North_Leavitt_Street%2C_Chicago%2C_Cook_County%2C_IL.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/f/fd/Louis_Sullivan_-_exterior_-_Holy_Trinity_Russian_%26_Greek_Orthodox_Church%2C_1121_North_Leavitt_Street%2C_Chicago%2C_Cook_County%2C_IL.jpg/375px-Louis_Sullivan_-_exterior_-_Holy_Trinity_Russian_%26_Greek_Orthodox_Church%2C_1121_North_Leavitt_Street%2C_Chicago%2C_Cook_County%2C_IL.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/f/fd/Louis_Sullivan_-_exterior_-_Holy_Trinity_Russian_%26_Greek_Orthodox_Church%2C_1121_North_Leavitt_Street%2C_Chicago%2C_Cook_County%2C_IL.jpg/500px-Louis_Sullivan_-_exterior_-_Holy_Trinity_Russian_%26_Greek_Orthodox_Church%2C_1121_North_Leavitt_Street%2C_Chicago%2C_Cook_County%2C_IL.jpg 2x" width="250"/></a><figcaption><div class="center" style="width:auto; margin-left:auto; margin-right:auto;"><a class="mw-redirect" href="/wiki/%D0%A1%D0%B2%D1%8F%D1%82%D0%BE-%D0%A2%D1%80%D0%BE%D0%B8%D1%86%D0%BA%D0%B0%D1%8F_%D1%86%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C_(%D0%A7%D0%B8%D0%BA%D0%B0%D0%B3%D0%BE)" title="Свято-Троицкая церковь (Чикаго)">Троицкая церковь</a> Владимирского прихода в Чикаго, где служил святой Алексий в 1891—1895 годах. Здание церкви построено в 1902—1903 годах архитектором <a class="mw-redirect" href="/wiki/%D0%A1%D0%B0%D0%BB%D0%BB%D0%B8%D0%B2%D0%B5%D0%BD,_%D0%9B%D1%83%D0%B8%D1%81_%D0%93%D0%B5%D0%BD%D1%80%D0%B8" title="Салливен, Луис Генри">Луисом Салливаном</a> попечением священномученика <a href="/wiki/%D0%9A%D0%BE%D1%87%D1%83%D1%80%D0%BE%D0%B2,_%D0%98%D0%B2%D0%B0%D0%BD_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Кочуров, Иван Александрович">Иоанна Кочурова</a>, настоятеля прихода в 1895—1906 годах. Фото 1965 года</div></figcaption></figure>
<p>Отец Алексий установил связь с другими грекокатолическими священниками, служившими в США и испытывавшими подобную дискриминацию со стороны римско-католического клира<sup class="reference" id="cite_ref-kohanik_11-2"><a href="#cite_note-kohanik-11">[10]</a></sup>. В октябре 1890 года восемь из десяти американских священников грекокатолического обряда собрались в приходе города <a href="/wiki/%D0%A3%D0%B8%D0%BB%D0%BA%D1%81-%D0%91%D0%B0%D1%80%D1%80%D0%B5" title="Уилкс-Барре">Уилкс-Барре</a> (штат <a href="/wiki/%D0%9F%D0%B5%D0%BD%D1%81%D0%B8%D0%BB%D1%8C%D0%B2%D0%B0%D0%BD%D0%B8%D1%8F" title="Пенсильвания">Пенсильвания</a>). Они решили обратиться за поддержкой к грекокатолическим епископам, которые направили их на служение в США. Отец Алексий написал епископу Прешовскому. Однако переписка с ним не помогла пастве униатской общины. Архиепископ Джон Айрленд разместил в соседнем с грекокатолическим польском приходе заявление, что он не признаёт отца Алексия ни католиком, ни священником. Грекокатолики обратились за помощью в <a href="/wiki/%D0%A0%D0%B8%D0%BC%D1%81%D0%BA%D0%B0%D1%8F_%D0%BA%D1%83%D1%80%D0%B8%D1%8F" title="Римская курия">Рим</a>, но получили предписание о возвращении своих священников в <a href="/wiki/%D0%A1%D1%82%D0%B0%D1%80%D1%8B%D0%B9_%D0%A1%D0%B2%D0%B5%D1%82" title="Старый Свет">Старый Свет</a>.
</p><p>Узнав о таком распоряжении из Рима, отец Алексий созвал приходское собрание, где верующие предложили обратиться к <a href="/wiki/%D0%9F%D1%80%D0%B0%D0%B2%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%B8%D0%B5" title="Православие">православному</a> епископу:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
Когда я всё это увидел и услышал, тогда решился на то, что давно уже жило в моём сердце, из-за чего душа болела у меня… быть православным… но как же?… Нужно быть очень осторожным. Та несчастная уния — начало упадка и всего злого — очень прижилась у наших людей, 250 лет прошло, как это бремя на наши шеи возложили!… Я горячо молил Бога, чтобы Он мне дал помощь и силу просветить моих тёмных верных… В том деле помогли мне сами прихожане. Когда я созвал своих прихожан, объяснил им своё печальное положение и заявил, что остаётся только их покинуть, то некоторые из них заявили: — «нет, пойдём к русскому епископу, не вечно ж чужим кланяться!»
</p>
<div class="mw-collapsible mw-collapsed has-collapsible-header ts-Скрытый_блок ts-Скрытый_блок-gray ts-Скрытый_блок-rightHideLink" style=""><div class="ts-Скрытый_блок-title collapsible-header" style="">Оригинальный текст</div><div class="mw-collapsible-content" style="text-align: left;">
<p>Коли я все тое видѣлъ и слышалъ, тогда рѣшился на такое, которое у мене уже давно въ сѣрдце жило, за которымъ у мене душа болѣла… быти православнымъ… но якъ же?… Треба быти дуже осторожнымъ. Тая несчастная унія начало упадка и всего злого, дуже зажилась у нашихъ людей, 250 рокивъ пройшло якъ тое иго на наши шеи положили!… Я горячо молилъ Бога, чтобы Онъ менѣ далъ помощь и силу просвѣтити моихъ темныхъ вѣрниковъ… Въ томъ дѣлѣ помогли менѣ сами прихожане. Коли я скликалъ своихъ прихожанъ, объяснилъ имъ своё печальное положеніе и заявилъ, что остается лишень лишити ихъ, то декоторы изъ нихъ заявили: — «нѣтъ, пойдемъ къ русскому епископу, не вѣчно ж чужимъ кланятися»!
</p>
</div></div>
<div class="ts-Конец_цитаты-source">— <cite>Воспоминания отца А. Товта. По книге протоиерея <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D1%85%D0%B0%D0%BD%D0%B8%D0%BA,_%D0%9F%D0%B5%D1%82%D1%80_%D0%AE%D1%80%D1%8C%D0%B5%D0%B2%D0%B8%D1%87" title="Коханик, Петр Юрьевич">Петра Коханика</a> «Начало истории Американской Руси», с. 488<sup class="reference" id="cite_ref-kohanik_11-3"><a href="#cite_note-kohanik-11">[10]</a></sup></cite></div></blockquote>
<p>В феврале 1891 года в <a href="/wiki/%D0%A1%D0%B0%D0%BD-%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%81%D0%BA%D0%BE" title="Сан-Франциско">Сан-Франциско</a>, где с 1872 года находился центр единственной на то время православной епархии Американского континента (<a href="/wiki/%D0%9D%D1%8C%D1%8E-%D0%99%D0%BE%D1%80%D0%BA%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8_%D0%90%D0%BB%D0%B5%D1%83%D1%82%D1%81%D0%BA%D0%B0%D1%8F_%D0%B5%D0%BF%D0%B0%D1%80%D1%85%D0%B8%D1%8F" title="Нью-Йоркская и Алеутская епархия">Алеутской и Аляскинской епархии</a> в составе <a href="/wiki/%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%B0%D1%8F_%D0%BF%D1%80%D0%B0%D0%B2%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%BD%D0%B0%D1%8F_%D1%86%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C" title="Русская православная церковь">Православной российской церкви</a>), состоялась первая встреча отца Алексия Товта с русским епископом <a href="/wiki/%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80_(%D0%A1%D0%BE%D0%BA%D0%BE%D0%BB%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9-%D0%90%D0%B2%D1%82%D0%BE%D0%BD%D0%BE%D0%BC%D0%BE%D0%B2)" title="Владимир (Соколовский-Автономов)">Владимиром (Соколовским-Автономовым)</a>. Через месяц епископ Владимир прибыл в Миннеаполис и 25 марта 1891 года присоединил отца Алексея со всем его приходом (361<sup class="reference" id="cite_ref-pravenc_9-2"><a href="#cite_note-pravenc-9">[8]</a></sup> или 405 чел.<sup class="reference" id="cite_ref-kohanik_11-4"><a href="#cite_note-kohanik-11">[10]</a></sup>) к Православной церкви. 14 июля 1892 года <a class="mw-redirect" href="/wiki/%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9_%D1%81%D0%B8%D0%BD%D0%BE%D0%B4#В_синодальный_период_(1721—1917)" title="Священный синод">Святейший синод Русской церкви</a> официально подтвердил принятие отца Алексея с прихожанами в состав Алеутской и Аляскинской епархии. Епископ Владимир назначил отца Алексия <a href="/wiki/%D0%91%D0%BB%D0%B0%D0%B3%D0%BE%D1%87%D0%B8%D0%BD%D0%BD%D1%8B%D0%B9" title="Благочинный">благочинным</a> и поручил ему также <a class="mw-redirect" href="/wiki/%D0%A1%D0%B2%D1%8F%D1%82%D0%BE-%D0%A2%D1%80%D0%BE%D0%B8%D1%86%D0%BA%D0%B0%D1%8F_%D1%86%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C_(%D0%A7%D0%B8%D0%BA%D0%B0%D0%B3%D0%BE)" title="Свято-Троицкая церковь (Чикаго)">приход в Чикаго</a>. Приход отца Алексия стал первым православным приходом на всём пространстве между Сан-Франциско и Нью-Йорком<sup class="reference" id="cite_ref-soldatov_12-0"><a href="#cite_note-soldatov-12">[11]</a></sup>.
</p><p>Первое время отцу Алексию и его прихожанам пришлось столкнуться с открытой религиозной и национальной враждой. Его обвиняли в продаже своей веры и русинского первородства «москалям» за деньги. Однако отец Алексей полтора года вообще не получал жалования от Синода из России и, чтобы содержать себя, вынужден был работать в пекарне. Тем не менее, переход общины отца Алексия в православие послужил примером для многих униатских общин в США и Канаде<sup class="reference" id="cite_ref-soldatov_12-1"><a href="#cite_note-soldatov-12">[11]</a></sup>.
</p>
<h3><span id=".D0.9C.D0.B8.D1.81.D1.81.D0.B8.D0.BE.D0.BD.D0.B5.D1.80.D1.81.D0.BA.D0.B8.D0.B5_.D1.82.D1.80.D1.83.D0.B4.D1.8B"></span><span class="mw-headline" id="Миссионерские_труды">Миссионерские труды</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Миссионерские труды»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=6" title="Редактировать код раздела «Миссионерские труды»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure class="mw-default-size mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Alexis_Toth.jpg"><img class="mw-file-element" data-file-height="295" data-file-width="180" decoding="async" height="295" src="//upload.wikimedia.org/wikipedia/commons/5/5e/Alexis_Toth.jpg" width="180"/></a><figcaption><div class="center" style="width:auto; margin-left:auto; margin-right:auto;">Отец Алексий Товт в митре</div></figcaption></figure>
<p>При приходе в Миннеаполисе была открыта школа, преобразованная через несколько лет в миссионерскую. К концу XX века община в Миннеаполисе стала крупнейшим православным приходом США, насчитывающим около тысячи прихожан.
</p><p>В <a class="mw-redirect" href="/wiki/1893" title="1893">1893</a>—<a href="/wiki/1909_%D0%B3%D0%BE%D0%B4" title="1909 год">1909 годах</a> отец Алексий был настоятелем церкви в <a href="/wiki/%D0%A3%D0%B8%D0%BB%D0%BA%D1%81-%D0%91%D0%B0%D1%80%D1%80%D0%B5" title="Уилкс-Барре">Уилкс-Барре</a> (штат Пенсильвания), ставшей центром его миссионерской деятельности среди униатов. Некоторое время занимал должность пенсильванского благочинного. По благословению правящих архиереев епископа <a href="/wiki/%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B9_(%D0%97%D0%B8%D0%BE%D1%80%D0%BE%D0%B2)" title="Николай (Зиоров)">Николая (Зиорова)</a> (1891—1898), архиепископа <a class="mw-redirect" href="/wiki/%D0%A2%D0%B8%D1%85%D0%BE%D0%BD_(%D0%9F%D0%B0%D1%82%D1%80%D0%B8%D0%B0%D1%80%D1%85_%D0%9C%D0%BE%D1%81%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9)" title="Тихон (Патриарх Московский)">Тихона (Беллавина)</a> (1898—1907) и архиепископа <a href="/wiki/%D0%9F%D0%BB%D0%B0%D1%82%D0%BE%D0%BD_(%D0%A0%D0%BE%D0%B6%D0%B4%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D1%81%D0%BA%D0%B8%D0%B9)" title="Платон (Рождественский)">Платона (Рождественского)</a> (1907) проповедовал и вёл миссионерскую работу главным образом среди униатов-русинов. К <a href="/wiki/1898_%D0%B3%D0%BE%D0%B4" title="1898 год">1898 году</a> три униатских священника со своими общинами перешли в Православную церковь, а к <a href="/wiki/1909_%D0%B3%D0%BE%D0%B4" title="1909 год">1909 году</a> — более 17 общин и около 20 тысяч униатов. При деятельном участии отца Алексия печатались различные церковные издания.
</p><p>В <a href="/wiki/1895_%D0%B3%D0%BE%D0%B4" title="1895 год">1895 году</a> для помощи вновь прибывшим в США иммигрантам, а также для консолидации разрозненных православных братств, отец Алексий инициировал создание <a href="/wiki/%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%BE%D0%B5_%D0%BF%D1%80%D0%B0%D0%B2%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%BD%D0%BE%D0%B5_%D0%BA%D0%B0%D1%84%D0%BE%D0%BB%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%BE%D0%B5_%D0%BE%D0%B1%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%BE_%D0%B2%D0%B7%D0%B0%D0%B8%D0%BC%D0%BE%D0%BF%D0%BE%D0%BC%D0%BE%D1%89%D0%B8" title="Русское православное кафолическое общество взаимопомощи">Русского православного кафолического общества взаимопомощи</a> (ROCMAS)<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[12]</a></sup>. Другими видными деятелями этого общества стали будущие <a href="/wiki/%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D0%BE%D0%BC%D1%83%D1%87%D0%B5%D0%BD%D0%B8%D0%BA" title="Священномученик">священномученики</a> <a href="/wiki/%D0%9A%D0%BE%D1%87%D1%83%D1%80%D0%BE%D0%B2,_%D0%98%D0%B2%D0%B0%D0%BD_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Кочуров, Иван Александрович">Иоанн Кочуров</a> и <a href="/wiki/%D0%A5%D0%BE%D1%82%D0%BE%D0%B2%D0%B8%D1%86%D0%BA%D0%B8%D0%B9,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Хотовицкий, Александр Александрович">Александр Хотовицкий</a>.
</p>
<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:St_Alexis_Toth_shrine2.jpg"><img class="mw-file-element" data-file-height="1890" data-file-width="1890" decoding="async" height="250" src="//upload.wikimedia.org/wikipedia/commons/thumb/1/1e/St_Alexis_Toth_shrine2.jpg/250px-St_Alexis_Toth_shrine2.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/1/1e/St_Alexis_Toth_shrine2.jpg/375px-St_Alexis_Toth_shrine2.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/1/1e/St_Alexis_Toth_shrine2.jpg/500px-St_Alexis_Toth_shrine2.jpg 2x" width="250"/></a><figcaption><div class="center" style="width:auto; margin-left:auto; margin-right:auto;">Склеп протоиерея Товта (1916)</div></figcaption></figure>
<p>Миссионерские труды отца Алексия не остались незамеченными в Русской православной Церкви, он был награждён всеми наградами, которые может получить священник, в том числе <a href="/wiki/%D0%9C%D0%B8%D1%82%D1%80%D0%B0_(%D0%B3%D0%BE%D0%BB%D0%BE%D0%B2%D0%BD%D0%BE%D0%B9_%D1%83%D0%B1%D0%BE%D1%80)" title="Митра (головной убор)">митрой</a> от Святейшего Синода, орденами <a href="/wiki/%D0%9E%D1%80%D0%B4%D0%B5%D0%BD_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80%D0%B0" title="Орден Святого Владимира">святого Владимира</a> и <a href="/wiki/%D0%9E%D1%80%D0%B4%D0%B5%D0%BD_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B9_%D0%90%D0%BD%D0%BD%D1%8B" title="Орден Святой Анны">святой Анны</a> от <a class="mw-redirect" href="/wiki/%D0%98%D0%BC%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%BE%D1%80_%D0%B2%D1%81%D0%B5%D1%80%D0%BE%D1%81%D1%81%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9" title="Император всероссийский">императора</a>, а в <a href="/wiki/1907_%D0%B3%D0%BE%D0%B4" title="1907 год">1907 году</a> святитель <a class="mw-redirect" href="/wiki/%D0%A2%D0%B8%D1%85%D0%BE%D0%BD_(%D0%91%D0%B5%D0%BB%D0%BB%D0%B0%D0%B2%D0%B8%D0%BD)" title="Тихон (Беллавин)">Тихон (Беллавин)</a> предложил ему стать епископом <a class="mw-redirect" href="/wiki/%D0%9F%D1%80%D0%B0%D0%B2%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%BD%D0%B0%D1%8F_%D0%A6%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C_%D0%B2_%D0%90%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B5" title="Православная Церковь в Америке">Русской православной греко-кафолической Церкви в Америке</a> для русинов, но он отказался, сославшись на преклонный возраст.
</p><p>В <a href="/wiki/1909_%D0%B3%D0%BE%D0%B4" title="1909 год">1909 году</a>, незадолго до кончины, отец Алексий был возведён в сан <a href="/wiki/%D0%9F%D1%80%D0%BE%D1%82%D0%BE%D0%BF%D1%80%D0%B5%D1%81%D0%B2%D0%B8%D1%82%D0%B5%D1%80" title="Протопресвитер">протопресвитера</a>. К концу 1908 года его здоровье стало ухудшаться, последние два месяца перед смертью он был прикован к постели и <span class="no-wikidata" data-wikidata-property-id="P570">24 апреля (<a href="/wiki/7_%D0%BC%D0%B0%D1%8F" title="7 мая">7 мая</a>) <a href="/wiki/1909_%D0%B3%D0%BE%D0%B4" title="1909 год">1909</a></span> года скончался. Похоронен в <a href="/wiki/%D0%A1%D0%B0%D1%83%D1%82-%D0%9A%D0%B5%D0%B9%D0%BD%D0%B0%D0%BD_(%D0%9F%D0%B5%D0%BD%D1%81%D0%B8%D0%BB%D1%8C%D0%B2%D0%B0%D0%BD%D0%B8%D1%8F)" title="Саут-Кейнан (Пенсильвания)">Саут-Кейнане</a> на кладбище старейшего на Американском континенте православного <a href="/wiki/%D0%A2%D0%B8%D1%85%D0%BE%D0%BD%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%BC%D0%BE%D0%BD%D0%B0%D1%81%D1%82%D1%8B%D1%80%D1%8C_(%D0%A1%D0%B0%D1%83%D1%82-%D0%9A%D0%B5%D0%B9%D0%BD%D0%B0%D0%BD)" title="Тихоновский монастырь (Саут-Кейнан)">монастыря</a>, построенного с его участием<sup class="reference" id="cite_ref-liberovsky_14-0"><a href="#cite_note-liberovsky-14">[13]</a></sup>.
</p>

<h2><span id=".D0.94.D0.B5.D1.8F.D1.82.D0.B5.D0.BB.D1.8C.D0.BD.D0.BE.D1.81.D1.82.D1.8C"></span><span class="mw-headline" id="Деятельность">Деятельность</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Деятельность»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=7" title="Редактировать код раздела «Деятельность»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>В результате миссионерской деятельности протопресвитера Алексия Товта после его кончины, в период до <a href="/wiki/%D0%92%D1%82%D0%BE%D1%80%D0%B0%D1%8F_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Вторая мировая война">Второй мировой войны</a>, в Америке и на родине святого, в Словакии, в православие перешли сотни тысяч униатов (только в Америке около 250 тысяч)<sup class="reference" id="cite_ref-ferencz_3-2"><a href="#cite_note-ferencz-3">[3]</a></sup><sup class="reference" id="cite_ref-pravenc_9-3"><a href="#cite_note-pravenc-9">[8]</a></sup><sup class="reference" id="cite_ref-connell_10-2"><a href="#cite_note-connell-10">[9]</a></sup><sup class="reference" id="cite_ref-soldatov_12-2"><a href="#cite_note-soldatov-12">[11]</a></sup><sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[14]</a></sup>. При этом, в Америке они сформировали около 300 православных приходов<sup class="reference" id="cite_ref-soldatov_12-3"><a href="#cite_note-soldatov-12">[11]</a></sup>. Таким образом, наибольший рост Русской православной греко-кафолической церкви в Америке был связан с обращением грекокатоликов<sup class="reference" id="cite_ref-ferencz_3-3"><a href="#cite_note-ferencz-3">[3]</a></sup>.
</p><p>Поскольку переход святого Алексия в православную веру изначально был вызван конфликтом с католическим архиепископом Джоном Айрлендом, последнего в шутку называют «Отцом Православной Церкви в Америке»<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[15]</a></sup>, а движение по переходу в православие, которое охватило все слои униатов в Америке и длилось по некоторым оценкам с <a class="mw-redirect" href="/wiki/1890" title="1890">1890</a> по <a href="/wiki/1914_%D0%B3%D0%BE%D0%B4" title="1914 год">1914 года</a>, — «движением Товта» или «схизмой Товта»<sup class="reference" id="cite_ref-ferencz_3-4"><a href="#cite_note-ferencz-3">[3]</a></sup>.
</p>
<h3><span id=".D0.A1.D0.BE.D1.87.D0.B8.D0.BD.D0.B5.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="Сочинения">Сочинения</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=8" title="Редактировать раздел «Сочинения»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=8" title="Редактировать код раздела «Сочинения»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>При жизни святого Алексия изданы четыре книги его сочинений, статьи в <a href="/wiki/%D0%90%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D1%80%D0%B0%D0%B2%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%BD%D1%8B%D0%B9_%D0%B2%D0%B5%D1%81%D1%82%D0%BD%D0%B8%D0%BA" title="Американский православный вестник">Американском православном вестнике</a>, газете «Свет» и других изданиях, содержащие объяснения истин православного вероучения, размышления о том, как славяне должны жить в Америке. Статьи и книги святого были написаны простым, понятным для русинов языком, с употреблением поговорок и шуток<sup class="reference" id="cite_ref-soldatov_12-4"><a href="#cite_note-soldatov-12">[11]</a></sup>.
</p><p>Самой известной книгой отца Алексия была «Где искать правду?» (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Where to Seek the Truth?</span>), которая неоднократно переиздавалась на нескольких языках, служа для униатов православным <a href="/wiki/%D0%9A%D0%B0%D1%82%D0%B5%D1%85%D0%B8%D0%B7%D0%B8%D1%81" title="Катехизис">катехизисом</a><sup class="reference" id="cite_ref-ferencz_3-5"><a href="#cite_note-ferencz-3">[3]</a></sup>.
</p>
<h3><span id=".D0.9D.D0.B0.D1.86.D0.B8.D0.BE.D0.BD.D0.B0.D0.BB.D1.8C.D0.BD.D1.8B.D0.B5_.D0.B2.D0.B7.D0.B3.D0.BB.D1.8F.D0.B4.D1.8B"></span><span class="mw-headline" id="Национальные_взгляды">Национальные взгляды</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=9" title="Редактировать раздел «Национальные взгляды»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=9" title="Редактировать код раздела «Национальные взгляды»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Отец Алексей был ярким носителем карпаторусского национального <a href="/wiki/%D0%A1%D0%B0%D0%BC%D0%BE%D1%81%D0%BE%D0%B7%D0%BD%D0%B0%D0%BD%D0%B8%D0%B5" title="Самосознание">самосознания</a><sup class="reference" id="cite_ref-frolov_17-0"><a href="#cite_note-frolov-17">[16]</a></sup>. Как и все русины своего времени, был воспитан в любви к русскому православию<sup class="reference" id="cite_ref-pravenc_9-4"><a href="#cite_note-pravenc-9">[8]</a></sup> и называл свою веру <i>русской</i><sup class="reference" id="cite_ref-ferencz_3-6"><a href="#cite_note-ferencz-3">[3]</a></sup><sup class="reference" id="cite_ref-kohanik_11-5"><a href="#cite_note-kohanik-11">[10]</a></sup>. Считал <a class="mw-redirect" href="/wiki/%D0%91%D1%80%D0%B5%D1%81%D1%82%D1%81%D0%BA%D0%B0%D1%8F_%D1%83%D0%BD%D0%B8%D1%8F_(1596)" title="Брестская уния (1596)">унию</a> насильственной, а её принятие — причиной упадка русинской культуры<sup class="reference" id="cite_ref-kohanik_11-6"><a href="#cite_note-kohanik-11">[10]</a></sup>.
</p>
<h2><span id=".D0.9F.D0.BE.D1.87.D0.B8.D1.82.D0.B0.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Почитание">Почитание</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=10" title="Редактировать раздел «Почитание»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=10" title="Редактировать код раздела «Почитание»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>

<h3><span id=".D0.9A.D0.B0.D0.BD.D0.BE.D0.BD.D0.B8.D0.B7.D0.B0.D1.86.D0.B8.D1.8F"></span><span class="mw-headline" id="Канонизация">Канонизация</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=11" title="Редактировать раздел «Канонизация»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=11" title="Редактировать код раздела «Канонизация»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>В <a href="/wiki/1916_%D0%B3%D0%BE%D0%B4" title="1916 год">1916 году</a>, через семь лет после кончины отца Алексия Товта, когда состоялось перезахоронение его тела в специальный склеп у главного храма <a href="/wiki/%D0%A2%D0%B8%D1%85%D0%BE%D0%BD%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%BC%D0%BE%D0%BD%D0%B0%D1%81%D1%82%D1%8B%D1%80%D1%8C_(%D0%A1%D0%B0%D1%83%D1%82-%D0%9A%D0%B5%D0%B9%D0%BD%D0%B0%D0%BD)" title="Тихоновский монастырь (Саут-Кейнан)">Тихоновского монастыря</a>, оно было обнаружено <a class="mw-redirect" href="/wiki/%D0%9D%D0%B5%D1%82%D0%BB%D0%B5%D0%BD%D0%BD%D1%8B%D0%B5_%D0%BC%D0%BE%D1%89%D0%B8" title="Нетленные мощи">нетленным</a><sup class="reference" id="cite_ref-liberovsky_14-1"><a href="#cite_note-liberovsky-14">[13]</a></sup>. В <a href="/wiki/1994_%D0%B3%D0%BE%D0%B4" title="1994 год">1994 году</a> гроб отца Алексия был открыт ещё раз, и снова его тело нашли нетленным.
</p><p>31 марта 1994 года был канонизирован решением Священного Синода Православной церкви в Америке с установлением дня памяти 7 мая и включением в <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D0%BF%D1%80%D0%B0%D0%B2%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%BD%D1%8B%D1%85_%D1%81%D0%B2%D1%8F%D1%82%D1%8B%D1%85_%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D0%BE%D0%B9_%D0%90%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B8" title="Список православных святых Северной Америки">Собор всех святых Северной Америки</a> (второе воскресение после Пятидесятницы)<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[18]</a></sup>. Официальные торжества по случаю его прославления состоялись 29-<a href="/wiki/30_%D0%BC%D0%B0%D1%8F" title="30 мая">30 мая</a> <a href="/wiki/1994_%D0%B3%D0%BE%D0%B4" title="1994 год">1994 года</a>, к 200-летию православия в Америке. В торжествах принимали участие представители Константинопольского и Московского патриархатов, а также Православной церкви Чешских Земель и Словакии<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[19]</a></sup>.
</p><p>Его мощи перенесены из склепа за алтарной стеной монастырского храма в раку у <a href="/wiki/%D0%98%D0%BA%D0%BE%D0%BD%D0%BE%D1%81%D1%82%D0%B0%D1%81" title="Иконостас">иконостаса</a> внутри храма.
</p><p>В честь святого Алексия были составлены богослужение, тропарь, кондак<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[20]</a></sup> и <a class="external text" href="http://ustavschik.livejournal.com/77952.html" rel="nofollow">акафист</a>.
</p>
<h3><span id=".D0.9F.D0.BE.D1.87.D0.B8.D1.82.D0.B0.D0.BD.D0.B8.D0.B5_.D0.B2_.D0.A1.D0.A8.D0.90"></span><span class="mw-headline" id="Почитание_в_США">Почитание в США</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=12" title="Редактировать раздел «Почитание в США»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=12" title="Редактировать код раздела «Почитание в США»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Ещё при жизни отца Алексия перешедшие из грекокатоличества <a class="mw-redirect" href="/wiki/%D0%9A%D0%B0%D1%80%D0%BF%D0%B0%D1%82%D0%BE%D1%80%D0%BE%D1%81%D1%81%D1%8B" title="Карпатороссы">карпатороссы</a> называли его <i>Батько</i>, чего затем не удостаивался ни один из видных продолжателей его дела<sup class="reference" id="cite_ref-soldatov_12-5"><a href="#cite_note-soldatov-12">[11]</a></sup>.
В <a href="/wiki/2009_%D0%B3%D0%BE%D0%B4" title="2009 год">2009 году</a> в Православной церкви в Америке праздновалось 100-летие со дня преставления святого Алексия Товта<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[21]</a></sup>. Особенно святой Алексий почитается в приходах, история которых связана с его миссионерской деятельностью<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[22]</a></sup><sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[23]</a></sup><sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[24]</a></sup><sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[25]</a></sup><sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[26]</a></sup><sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[27]</a></sup>.
</p><p>В США появляются храмы, посвящённые святому Алексию Товту (в частности, в Константинопольском Патриархате<sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[28]</a></sup>).
</p>
<h3><span id=".D0.9F.D0.BE.D1.87.D0.B8.D1.82.D0.B0.D0.BD.D0.B8.D0.B5_.D0.B7.D0.B0_.D0.BF.D1.80.D0.B5.D0.B4.D0.B5.D0.BB.D0.B0.D0.BC.D0.B8_.D0.A1.D0.A8.D0.90"></span><span class="mw-headline" id="Почитание_за_пределами_США">Почитание за пределами США</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=13" title="Редактировать раздел «Почитание за пределами США»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=13" title="Редактировать код раздела «Почитание за пределами США»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>В <a href="/wiki/2001_%D0%B3%D0%BE%D0%B4" title="2001 год">2001 году</a> святой Алексий Товт был прославлен в <a href="/wiki/%D0%A1%D0%BE%D0%B1%D0%BE%D1%80_%D0%93%D0%B0%D0%BB%D0%B8%D1%86%D0%BA%D0%B8%D1%85_%D1%81%D0%B2%D1%8F%D1%82%D1%8B%D1%85" title="Собор Галицких святых">Соборе Галицких святых</a> в <a class="mw-redirect" href="/wiki/%D0%9B%D1%8C%D0%B2%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8_%D0%93%D0%B0%D0%BB%D0%B8%D1%86%D0%BA%D0%B0%D1%8F_%D0%B5%D0%BF%D0%B0%D1%80%D1%85%D0%B8%D1%8F" title="Львовская и Галицкая епархия">Львовской епархии</a> <a href="/wiki/%D0%A3%D0%BA%D1%80%D0%B0%D0%B8%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BF%D1%80%D0%B0%D0%B2%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%BD%D0%B0%D1%8F_%D1%86%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C_(%D0%9C%D0%BE%D1%81%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%BE%D0%B3%D0%BE_%D0%BF%D0%B0%D1%82%D1%80%D0%B8%D0%B0%D1%80%D1%85%D0%B0%D1%82%D0%B0)" title="Украинская православная церковь (Московского патриархата)">Украинской православной церкви</a> (день памяти — неделя 3-я по Пятидесятнице<sup class="reference" id="cite_ref-30"><a href="#cite_note-30">[29]</a></sup><sup class="reference" id="cite_ref-31"><a href="#cite_note-31">[30]</a></sup>). Почитается в Русской и <a class="mw-redirect" href="/wiki/%D0%9F%D1%80%D0%B0%D0%B2%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%BD%D0%B0%D1%8F_%D0%A6%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C_%D0%A7%D0%B5%D1%88%D1%81%D0%BA%D0%B8%D1%85_%D0%B7%D0%B5%D0%BC%D0%B5%D0%BB%D1%8C_%D0%B8_%D0%A1%D0%BB%D0%BE%D0%B2%D0%B0%D0%BA%D0%B8%D0%B8" title="Православная Церковь Чешских земель и Словакии">Православной церкви Чешских земель и Словакии</a><sup class="reference" id="cite_ref-32"><a href="#cite_note-32">[31]</a></sup>.
</p><p>На <a href="/wiki/%D0%97%D0%B0%D0%BA%D0%B0%D1%80%D0%BF%D0%B0%D1%82%D1%8C%D0%B5" title="Закарпатье">Закарпатье</a> организуются православные молодёжные лагеря в честь святого Алексия<sup class="reference" id="cite_ref-33"><a href="#cite_note-33">[32]</a></sup>.
</p>
<h3><span id=".D0.A0.D0.B0.D1.81.D0.BF.D1.80.D0.BE.D1.81.D1.82.D1.80.D0.B0.D0.BD.D0.B5.D0.BD.D0.B8.D0.B5_.D1.81.D0.B2.D1.8F.D1.82.D1.8B.D1.85_.D0.BC.D0.BE.D1.89.D0.B5.D0.B9"></span><span class="mw-headline" id="Распространение_святых_мощей">Распространение святых мощей</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=14" title="Редактировать раздел «Распространение святых мощей»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D0%BE%D0%B2%D1%82,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=14" title="Редактировать код раздела «Распространение святых мощей»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Частицы <a href="/wiki/%D0%9C%D0%BE%D1%89%D0%B8" title="Мощи">мощей</a> святого Алексия можно встретить не только на территории США, но и в других странах (в России: <a class="mw-redirect" href="/wiki/%D0%A5%D1%80%D0%B0%D0%BC_%D0%B2%D0%B5%D0%BB%D0%B8%D0%BA%D0%BE%D0%BC%D1%83%D1%87%D0%B5%D0%BD%D0%B8%D1%86%D1%8B_%D0%95%D0%BA%D0%B0%D1%82%D0%B5%D1%80%D0%B8%D0%BD%D1%8B_%D0%BD%D0%B0_%D0%92%D1%81%D0%BF%D0%BE%D0%BB%D1%8C%D0%B5" title="Храм великомученицы Екатерины на Всполье">храм великомученицы Екатерины на Всполье</a> (Москва)<sup class="reference" id="cite_ref-34"><a href="#cite_note-34">[33]</a></sup>, на Украине: <a href="/wiki/%D0%A6%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D1%8F_%D0%9F%D0%BE%D0%B1%D0%B5%D0%B4%D0%BE%D0%BD%D0%BE%D1%81%D1%86%D0%B0_(%D0%9B%D1%8C%D0%B2%D0%BE%D0%B2)" title="Церковь Святого Георгия Победоносца (Львов)">церковь Святого Георгия Победоносца (Львов)</a><sup class="reference" id="cite_ref-35"><a href="#cite_note-35">[34]</a></sup>).
</p><p>Митрополит <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD_(%D0%A1%D0%B2%D0%B0%D0%B9%D0%BA%D0%BE)" title="Герман (Свайко)">Герман (Свайко)</a>, предыдущий предстоятель Православной церкви в Америке, во время своего официального визита в <a href="/wiki/%D0%9F%D1%80%D0%B0%D0%B2%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%BD%D0%B0%D1%8F_%D1%86%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C_%D0%A7%D0%B5%D1%88%D1%81%D0%BA%D0%B8%D1%85_%D0%B7%D0%B5%D0%BC%D0%B5%D0%BB%D1%8C_%D0%B8_%D0%A1%D0%BB%D0%BE%D0%B2%D0%B0%D0%BA%D0%B8%D0%B8" title="Православная церковь Чешских земель и Словакии">Православную церковь Чешских земель и Словакии</a> в 2004 году передал частицы мощей святого Алексия нескольким храмам в <a href="/wiki/%D0%A7%D0%B5%D1%85%D0%B8%D1%8F" title="Чехия">Чехии</a> и <a href="/wiki/%D0%A1%D0%BB%D0%BE%D0%B2%D0%B0%D0%BA%D0%B8%D1%8F" title="Словакия">Словакии</a>, преподнёс их также всем архиереям этой <a href="/wiki/%D0%9F%D0%BE%D0%BC%D0%B5%D1%81%D1%82%D0%BD%D0%B0%D1%8F_%D1%86%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C" title="Поместная церковь">Поместной церкви</a><sup class="reference" id="cite_ref-36"><a href="#cite_note-36">[35]</a></sup>.
</p><p>Во время своего первого визита в Россию в качестве предстоятеля Православной церкви в Америке митрополит <a href="/wiki/%D0%98%D0%BE%D0%BD%D0%B0_(%D0%9F%D0%B0%D1%84%D1%84%D1%85%D0%B0%D1%83%D0%B7%D0%B5%D0%BD)" title="Иона (Паффхаузен)">Иона (Паффхаузен)</a> в ознаменование 100-летия преставления святого Алексия <a href="/wiki/26_%D0%B0%D0%BF%D1%80%D0%B5%D0%BB%D1%8F" title="26 апреля">26 апреля</a> <a href="/wiki/2009_%D0%B3%D0%BE%D0%B4" title="2009 год">2009 года</a> преподнёс в подарок патриарху Московскому <a href="/wiki/%D0%9A%D0%B8%D1%80%D0%B8%D0%BB%D0%BB_(%D0%BF%D0%B0%D1%82%D1%80%D0%B8%D0%B0%D1%80%D1%85_%D0%9C%D0%BE%D1%81%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9)" title="Кирилл (патриарх Московский)">Кириллу</a> <a href="/wiki/%D0%98%D0%BA%D0%BE%D0%BD%D0%B0" title="Икона">икону</a> святого с частицей его мощей<sup class="reference" id="cite_ref-37"><a href="#cite_note-37">[36]</a></sup>.
</p>
