<p><b>Opera Unite</b> — расширение <a href="/wiki/%D0%91%D1%80%D0%B0%D1%83%D0%B7%D0%B5%D1%80" title="Браузер">браузера</a> <a href="/wiki/Opera" title="Opera">Opera</a>, построенное по <a href="/wiki/%D0%98%D0%BD%D1%84%D0%BE%D1%80%D0%BC%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D1%8B%D0%B5_%D1%82%D0%B5%D1%85%D0%BD%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D0%B8" title="Информационные технологии">технологии</a> <a href="/wiki/%D0%9E%D0%B4%D0%BD%D0%BE%D1%80%D0%B0%D0%BD%D0%B3%D0%BE%D0%B2%D0%B0%D1%8F_%D1%81%D0%B5%D1%82%D1%8C" title="Одноранговая сеть">одноранговой сети</a>, позволяющее из цепочки <a href="/wiki/%D0%9F%D0%B5%D1%80%D1%81%D0%BE%D0%BD%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80" title="Персональный компьютер">ПК</a> — <a href="/wiki/%D0%A1%D0%B5%D1%80%D0%B2%D0%B5%D1%80_(%D0%B0%D0%BF%D0%BF%D0%B0%D1%80%D0%B0%D1%82%D0%BD%D0%BE%D0%B5_%D0%BE%D0%B1%D0%B5%D1%81%D0%BF%D0%B5%D1%87%D0%B5%D0%BD%D0%B8%D0%B5)" title="Сервер (аппаратное обеспечение)">сервер</a> — ПК убрать сервер и сделать прямое соединение ПК — ПК<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>. С помощью Opera Unite можно, например, слушать музыку и смотреть видео в потоковом режиме, делиться фотографиями и другими файлами прямо со своего компьютера, посредством предоставления доступа к определённой <a href="/wiki/%D0%9A%D0%B0%D1%82%D0%B0%D0%BB%D0%BE%D0%B3_(%D1%84%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2%D0%B0%D1%8F_%D1%81%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0)" title="Каталог (файловая система)">папке</a> (и подпапкам) в <a href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2%D0%B0%D1%8F_%D1%81%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0" title="Файловая система">файловой системе</a>. При создании приложений используются технологии <a href="/wiki/HTML" title="HTML">HTML</a>, <a href="/wiki/CSS" title="CSS">CSS</a>, <a href="/wiki/JavaScript" title="JavaScript">JavaScript</a>, <a href="/wiki/SVG" title="SVG">SVG</a> и <a href="/wiki/AJAX" title="AJAX">AJAX</a><sup class="reference" id="cite_ref-Unite_FAQ_2-0"><a href="#cite_note-Unite_FAQ-2">[2]</a></sup>, а также можно использовать <a href="/wiki/%D0%91%D0%B8%D0%B1%D0%BB%D0%B8%D0%BE%D1%82%D0%B5%D0%BA%D0%B0_(%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5)" title="Библиотека (программирование)">библиотеку</a> шаблонов <a class="external text" href="https://github.com/operasoftware/unite-libraries/tree/master/markuper" rel="nofollow">Markuper API</a>, которая позволяет связать JavaScript и HTML, облегчив тем самым процесс создания приложений. <a href="/wiki/24_%D0%B0%D0%BF%D1%80%D0%B5%D0%BB%D1%8F" title="24 апреля">24 апреля</a> <a href="/wiki/2012_%D0%B3%D0%BE%D0%B4" title="2012 год">2012 года</a> было объявлено о прекращении разработки Opera Unite из-за желания разработчиков сконцентрировать все свои силы над работой с расширениями в браузере и не растрачиваться на поддержку существующих технологий<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>.
</p>

<h2><span id=".D0.98.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.8F"></span><span class="mw-headline" id="История">История</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Opera_Unite&amp;veaction=edit&amp;section=1" title="Редактировать раздел «История»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Opera_Unite&amp;action=edit&amp;section=1" title="Редактировать код раздела «История»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>За четыре дня до выхода разработчиками было объявлено, что некая новая (на тот момент ещё не известная широкой аудитории) технология должна «<i>Заново открыть</i>/<i>Перевернуть интернет</i>» («<i>Reinvents the Web</i>»)<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>. Первая публичная версия являлась лабораторной сборкой Opera 10 <a href="/wiki/16_%D0%B8%D1%8E%D0%BD%D1%8F" title="16 июня">16 июня</a> <a href="/wiki/2009_%D0%B3%D0%BE%D0%B4" title="2009 год">2009 года</a><sup class="reference" id="cite_ref-OperaLabsNews_5-0"><a href="#cite_note-OperaLabsNews-5">[5]</a></sup>.
</p><p><a href="/wiki/3_%D0%B8%D1%8E%D0%BB%D1%8F" title="3 июля">3 июля</a> <a href="/wiki/2009_%D0%B3%D0%BE%D0%B4" title="2009 год">2009 года</a> с выпуском новой <a href="/wiki/%D0%91%D0%B5%D1%82%D0%B0-%D1%82%D0%B5%D1%81%D1%82%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5" title="Бета-тестирование">бета-версии</a> браузера <a href="/wiki/Opera" title="Opera">Opera</a> 10 появилась возможность устанавливать ограничение на ширину канала данных. Также теперь можно настроить <a class="mw-redirect" href="/wiki/%D0%9F%D0%BE%D1%80%D1%82_(TCP/IP)" title="Порт (TCP/IP)">порт</a>, через который будет осуществляться обмен информацией (по умолчанию 8840)<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>.
</p><p>Начиная с Opera 10 beta 2 развитие технологии Unite проходило отдельно от бета-версий браузера<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>. Другими словами, технология Unite была ещё не готова для попадания в бета-версии браузера Opera 10. Поэтому вместе с выходом Opera 10 beta 2 и beta 3 одновременно выходит срез, содержащий в себе Opera Unite, который являлся отдельно развиваемой ветвью лабораторных сборок браузера для тестовых целей. Кроме этого, <a href="/wiki/13_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="13 августа">13 августа</a> <a href="/wiki/2009_%D0%B3%D0%BE%D0%B4" title="2009 год">2009 года</a> вместе с выходом Opera 10 beta 3 разработчики сообщили, что технология Unite не появится в финальной сборке <a class="mw-redirect" href="/wiki/Opera_10" title="Opera 10">Opera 10</a><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>. Технология Unite совершенствовалась параллельно с подготовкой к выходу Opera 10, появление же её было намечено в версии браузера 10.10.
</p><p><a href="/wiki/21_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="21 сентября">21 сентября</a> <a href="/wiki/2009_%D0%B3%D0%BE%D0%B4" title="2009 год">2009 года</a> с выходом очередного тестового билда в предустановленных приложениях Opera Unite был изменён внешний вид (под стиль Opera 10), а «Сервисы» («Services») стали называться «Приложениями» («Applications»), <a href="/wiki/%D0%A0%D0%B0%D1%81%D1%88%D0%B8%D1%80%D0%B5%D0%BD%D0%B8%D0%B5_%D0%B8%D0%BC%D0%B5%D0%BD%D0%B8_%D1%84%D0%B0%D0%B9%D0%BB%D0%B0" title="Расширение имени файла">расширение файлов</a> приложений не .us, а .ua. Также в эту сборку было добавлено приложение «Sonar», которое искусственно нагружало серверы, чтобы предотвратить в дальнейшем сбои в работе технологии<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>.
</p><p><a href="/wiki/14_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="14 октября">14 октября</a> <a href="/wiki/2009_%D0%B3%D0%BE%D0%B4" title="2009 год">2009 года</a> была выпущена Opera 10.10 beta, которая стала первой бета-версией браузера Opera, содержащей Opera Unite<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>. Немногим ранее (<a href="/wiki/9_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="9 октября">9 октября</a> <a href="/wiki/2009_%D0%B3%D0%BE%D0%B4" title="2009 год">2009 года</a>) была закончена кампания по нагрузке прокси-серверов Opera Unite приложением «Sonar»<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup>.
</p><p><a href="/wiki/23_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="23 октября">23 октября</a> <a href="/wiki/2009_%D0%B3%D0%BE%D0%B4" title="2009 год">2009 года</a> был дан старт конкурсу «Battle for the best Opera Unite applications». Суть его состояла в том, что с этого дня каждую неделю определяется лучшее приложение из недавно появившихся на сайте unite.opera.com. Разработчик победившего приложения получает денежный сертификат, равноценный стоимости <a class="mw-redirect" href="/wiki/Nintendo_Wii" title="Nintendo Wii">Nintendo Wii</a><sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>.
</p><p><a href="/wiki/23_%D0%BD%D0%BE%D1%8F%D0%B1%D1%80%D1%8F" title="23 ноября">23 ноября</a> <a href="/wiki/2009_%D0%B3%D0%BE%D0%B4" title="2009 год">2009 года</a> была выпущена финальная версия Opera 10.10 Unite<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup><sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup>.
</p><p><a href="/wiki/26_%D0%B0%D0%BF%D1%80%D0%B5%D0%BB%D1%8F" title="26 апреля">26 апреля</a> <a href="/wiki/2012_%D0%B3%D0%BE%D0%B4" title="2012 год">2012 года</a> вышла первая бета-версия Opera 12, из которой была убрана Opera Unite.<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup>
</p>
<h3><span id=".D0.94.D0.B0.D0.BB.D1.8C.D0.BD.D0.B5.D0.B9.D1.88.D0.B5.D0.B5_.D1.80.D0.B0.D0.B7.D0.B2.D0.B8.D1.82.D0.B8.D0.B5"></span><span class="mw-headline" id="Дальнейшее_развитие">Дальнейшее развитие</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Opera_Unite&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Дальнейшее развитие»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Opera_Unite&amp;action=edit&amp;section=2" title="Редактировать код раздела «Дальнейшее развитие»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Разработчиками предлагалась тесная интеграция технологии с мобильными устройствами<sup class="reference" id="cite_ref-OperaLabsNews_5-1"><a href="#cite_note-OperaLabsNews-5">[5]</a></sup>.
</p>
<h2><span id=".D0.9F.D1.80.D0.B8.D0.BD.D1.86.D0.B8.D0.BF_.D1.80.D0.B0.D0.B1.D0.BE.D1.82.D1.8B"></span><span class="mw-headline" id="Принцип_работы">Принцип работы</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Opera_Unite&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Принцип работы»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Opera_Unite&amp;action=edit&amp;section=3" title="Редактировать код раздела «Принцип работы»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Главным отличием от других сервисов обмена данными является отсутствие необходимости предварительной загрузки файлов пользователя на <a href="/wiki/%D0%9E%D0%B1%D0%BB%D0%B0%D1%87%D0%BD%D1%8B%D0%B5_%D0%B2%D1%8B%D1%87%D0%B8%D1%81%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F" title="Облачные вычисления">облачный сервер</a><sup class="reference" id="cite_ref-user_guide_16-0"><a href="#cite_note-user_guide-16">[16]</a></sup>. Доступ к данным, предоставляемым Opera Unite, осуществляется через серверы компании <a href="/wiki/Opera_Software" title="Opera Software">Opera Software</a>. Сервер используется как <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%BA%D1%81%D0%B8-%D1%81%D0%B5%D1%80%D0%B2%D0%B5%D1%80" title="Прокси-сервер">прокси-сервер</a>, <a href="/wiki/%D0%9C%D0%B5%D0%B6%D1%81%D0%B5%D1%82%D0%B5%D0%B2%D0%BE%D0%B9_%D1%8D%D0%BA%D1%80%D0%B0%D0%BD" title="Межсетевой экран">брандмауэр</a>, система доменных имён (<a href="/wiki/DNS" title="DNS">DNS</a>) и просто для связи компьютеров, которые не находятся в одной локальной сети, а вовсе не как место хранения данных. Если же доступ по локальной сети возможен, Unite осуществит необходимое соединение для более быстрого доступа к материалам (соединение происходит по принципу <a class="mw-redirect" href="/wiki/P2p" title="P2p">p2p</a><sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup>). Для более простой настройки соединения между компьютерами используется технология <a href="/wiki/UPnP" title="UPnP">UPnP</a><sup class="reference" id="cite_ref-user_guide_16-1"><a href="#cite_note-user_guide-16">[16]</a></sup>. Если <a href="/wiki/%D0%9C%D0%B0%D1%80%D1%88%D1%80%D1%83%D1%82%D0%B8%D0%B7%D0%B0%D1%82%D0%BE%D1%80" title="Маршрутизатор">маршрутизатор</a> конечного компьютера поддерживает <a href="/wiki/UPnP" title="UPnP">UPnP</a>, подключение можно осуществить, введя в адресной строке браузера адрес вида: <code>IP-адрес:8840</code><sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup>.
</p>
<h2><span id=".D0.9F.D1.80.D0.B8.D0.BB.D0.BE.D0.B6.D0.B5.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="Приложения">Приложения</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Opera_Unite&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Приложения»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Opera_Unite&amp;action=edit&amp;section=4" title="Редактировать код раздела «Приложения»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Загрузка и установка приложений осуществляется через интерфейс браузера Opera. По умолчанию в боковой панели «Unite» расположены элементы для управления шестью приложениями, которые будут загружаться при первом использовании:
</p>
<ul><li><a class="external text" href="https://web.archive.org/web/20100218125946/http://unite.opera.com/application/132/" rel="nofollow">«File Sharing»</a> — предоставление доступа к файлам на компьютере,</li>
<li><a class="external text" href="https://web.archive.org/web/20100220022734/http://unite.opera.com/application/142/" rel="nofollow">«Fridge»</a> — обмен короткими сообщениями, прикрепляя к виртуальному <a href="/wiki/%D0%A5%D0%BE%D0%BB%D0%BE%D0%B4%D0%B8%D0%BB%D1%8C%D0%BD%D0%B8%D0%BA" title="Холодильник">холодильнику</a> <a href="/wiki/%D0%A1%D1%82%D0%B8%D0%BA%D0%B5%D1%80" title="Стикер">стикеры</a>,</li>
<li><a class="external text" href="https://web.archive.org/web/20100219222954/http://unite.opera.com/application/162/" rel="nofollow">«Media Player»</a> — доступ к музыке на компьютере, возможность прослушивания музыки в потоковом режиме,</li>
<li><a class="external text" href="https://web.archive.org/web/20100208231330/http://unite.opera.com/application/172/" rel="nofollow">«Photo Sharing»</a> — создание <a href="/wiki/%D0%A4%D0%BE%D1%82%D0%BE%D0%B0%D0%BB%D1%8C%D0%B1%D0%BE%D0%BC" title="Фотоальбом">фотоальбома</a>,</li>
<li><a class="external text" href="https://web.archive.org/web/20100219223009/http://unite.opera.com/application/192/" rel="nofollow">«Web Server»</a> — запуск <a class="mw-redirect" href="/wiki/%D0%92%D0%B5%D0%B1-%D1%81%D0%B0%D0%B9%D1%82" title="Веб-сайт">веб-сайта</a> на компьютере,</li>
<li><a class="external text" href="https://web.archive.org/web/20100213123446/http://unite.opera.com/application/412/" rel="nofollow">«Messenger»</a> — общение один на один с друзьями по социальной сети <a href="/wiki/My_Opera" title="My Opera">My Opera</a>.</li></ul>
<p>Аналогами «File Sharing», «Media Player» и «Photo Sharing» являются файлообменники, минусом которых в данном случае является предварительная загрузка файлов. Аналогом «Messenger» является любая <a class="mw-redirect" href="/wiki/%D0%9F%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B0_%D0%BC%D0%B3%D0%BD%D0%BE%D0%B2%D0%B5%D0%BD%D0%BD%D0%BE%D0%B3%D0%BE_%D0%BE%D0%B1%D0%BC%D0%B5%D0%BD%D0%B0_%D1%81%D0%BE%D0%BE%D0%B1%D1%89%D0%B5%D0%BD%D0%B8%D1%8F%D0%BC%D0%B8" title="Программа мгновенного обмена сообщениями">программа мгновенного обмена сообщениями</a>, аналогом «Fridge» является <a class="mw-redirect" href="/wiki/Twitter" title="Twitter">Twitter</a> или любой другой <a href="/wiki/%D0%9C%D0%B8%D0%BA%D1%80%D0%BE%D0%B1%D0%BB%D0%BE%D0%B3%D0%B8%D0%BD%D0%B3" title="Микроблогинг">микроблогинг</a>. Аналогом «Web Server» является настоящий <a href="/wiki/%D0%A1%D0%B5%D1%80%D0%B2%D0%B5%D1%80_(%D0%B0%D0%BF%D0%BF%D0%B0%D1%80%D0%B0%D1%82%D0%BD%D0%BE%D0%B5_%D0%BE%D0%B1%D0%B5%D1%81%D0%BF%D0%B5%D1%87%D0%B5%D0%BD%D0%B8%D0%B5)" title="Сервер (аппаратное обеспечение)">сервер</a>, но в данном случае возможности компьютера обычного пользователя ограничены пропускной способностью канала или мощностью аппаратного составляющего<sup class="reference" id="cite_ref-Unite_FAQ_2-1"><a href="#cite_note-Unite_FAQ-2">[2]</a></sup>. Если сравнивать работу <a href="/wiki/BitTorrent_(%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B0)" title="BitTorrent (программа)">BitTorrent</a>-клиента и «File Sharing», то главным их отличием будет то, что <a class="mw-redirect" href="/wiki/%D0%A1%D0%B8%D0%B4%D0%B5%D1%80" title="Сидер">сидер</a> только один и при одновременной загрузке одного файла между <a class="mw-redirect" href="/wiki/Peer" title="Peer">пирами</a> не происходит обмена фрагментами. Приложения может создать любой желающий. Главным критерием является отсутствие ошибок в работе и приложения не должны иметь вредоносный характер, другими словами, не должны являться <a href="/wiki/%D0%9A%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D1%8B%D0%B9_%D0%B2%D0%B8%D1%80%D1%83%D1%81" title="Компьютерный вирус">вирусами</a><sup class="reference" id="cite_ref-manual_ru_19-0"><a href="#cite_note-manual_ru-19">[19]</a></sup>.
</p><p>Кроме приложений развлекательного характера или тех, которые позволяют обмениваться файлами, есть приложение, которое позволяет включать и выключать установленные в браузере пользовательские <a href="/wiki/JavaScript" title="JavaScript">JavaScript</a>, создавать новые и редактировать существующие. Это UJS Manager<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup>.
</p>
<h2><span id=".D0.98.D1.81.D0.BF.D0.BE.D0.BB.D1.8C.D0.B7.D0.BE.D0.B2.D0.B0.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Использование">Использование</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Opera_Unite&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Использование»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Opera_Unite&amp;action=edit&amp;section=5" title="Редактировать код раздела «Использование»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Чтобы начать использовать Opera Unite, пользователь должен пройти обязательную регистрацию в диалоговом окне программы. После выбора имени данного конкретного компьютера (Home, Mac, Office, Notebook и прочее), приложениями можно будет воспользоваться по адресу <code>&lt;имя компьютера&gt;.&lt;логин учётной записи&gt;.operaunite.com</code><sup class="reference" id="cite_ref-manual_ru_19-1"><a href="#cite_note-manual_ru-19">[19]</a></sup> или через боковую панель браузера. Если пользователь уже имеет аккаунт <a href="/wiki/My_Opera" title="My Opera">My Opera</a>, то он может использовать его при запуске Opera Unite. Чтобы использовать Opera Unite, логин учётной записи в сообществе не должен содержать такие символы, как «/» (<a href="/wiki/%D0%9A%D0%BE%D1%81%D0%B0%D1%8F_%D1%87%D0%B5%D1%80%D1%82%D0%B0" title="Косая черта">косая черта</a>), «.» (<a href="/wiki/%D0%A2%D0%BE%D1%87%D0%BA%D0%B0_(%D0%B7%D0%BD%D0%B0%D0%BA_%D0%BF%D1%80%D0%B5%D0%BF%D0%B8%D0%BD%D0%B0%D0%BD%D0%B8%D1%8F)" title="Точка (знак препинания)">точка</a>), «_» (<a href="/wiki/%D0%9F%D0%BE%D0%B4%D1%87%D1%91%D1%80%D0%BA%D0%B8%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5" title="Подчёркивание">подчёркивание</a>) и <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%B1%D0%B5%D0%BB" title="Пробел">пробел</a>, в противном случае пользователю придётся пройти процедуру переименования на <a class="external text" href="https://web.archive.org/web/20090620011027/http://my.opera.com/community/rename-user/" rel="nofollow">сайте сообщества</a>. Управление приложениями осуществляется через боковую панель браузера. Доступ к любому приложению можно ограничить установкой пароля, чтобы избежать лишней нагрузки или чтобы ограничить круг пользователей. Из настроек возможны установка используемого порта и ограничение скорости доступа к конечному компьютеру<sup class="reference" id="cite_ref-Unite_FAQ_2-2"><a href="#cite_note-Unite_FAQ-2">[2]</a></sup>.
</p><p>Разработчиками сообщалось о возможном нарушении авторских прав пользователем<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup>. Если такой случай будет иметь место, то компания Opera Software попросит пользователя удалить содержимое из открытого доступа, в противном случае, учётная запись пользователя будет заблокирована. Также было объявлено, что Opera Software не будет отслеживать деятельность пользователей Opera Unite<sup class="reference" id="cite_ref-Unite_FAQ_2-3"><a href="#cite_note-Unite_FAQ-2">[2]</a></sup><sup class="reference" id="cite_ref-cnet_22-0"><a href="#cite_note-cnet-22">[22]</a></sup>.
</p><p>
</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
С помощью Opera Unite обмен данными между компьютерами становится таким же простым, как и работа с веб-ресурсами. Конечному пользователю эта технология позволяет лучше контролировать использование его частных данных в открытом доступе. Кроме того, он может удалённо работать с хранящейся на его компьютере информацией с помощью любого другого устройства, оснащённого современным веб-браузером.
</p>

<div dir="auto" lang="en">Opera Unite makes serving data as simple and easy as browsing the Web. For consumers, Opera Unite services give greater control of private data and make it easy to share data with any device equipped with a modern Web browser.

<h2><span id=".D0.9A.D1.80.D0.B8.D1.82.D0.B8.D0.BA.D0.B0"></span><span class="mw-headline" id="Критика">Критика</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Opera_Unite&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Критика»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Opera_Unite&amp;action=edit&amp;section=6" title="Редактировать код раздела «Критика»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Технология неоднократно подверглась критике из-за потенциальных проблем с <a href="/wiki/%D0%98%D0%BD%D1%84%D0%BE%D1%80%D0%BC%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B1%D0%B5%D0%B7%D0%BE%D0%BF%D0%B0%D1%81%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Информационная безопасность">безопасностью</a><sup class="reference" id="cite_ref-cnet_22-1"><a href="#cite_note-cnet-22">[22]</a></sup><sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup><sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup>. Поэтому разработчики рекомендуют использовать только те приложения, которые были загружены с <a class="external text" href="https://web.archive.org/web/20110221142017/http://unite.opera.com/applications/" rel="nofollow">официального сайта</a><sup class="reference" id="cite_ref-Unite_FAQ_2-4"><a href="#cite_note-Unite_FAQ-2">[2]</a></sup>. Некоторые эксперты отмечали, что ограничение свободного доступа путём установки пароля является ни чем иным, как «<i>началом конца</i>» («<i>an avenue to disaster</i>»)<sup class="reference" id="cite_ref-cnet_22-2"><a href="#cite_note-cnet-22">[22]</a></sup>. Кроме того, большим недостатком, вытекающим из принципа работы, является необходимость держать <a href="/wiki/%D0%91%D1%80%D0%B0%D1%83%D0%B7%D0%B5%D1%80" title="Браузер">браузер</a> постоянно запущенным, а компьютер подключённым к <a href="/wiki/%D0%98%D0%BD%D1%82%D0%B5%D1%80%D0%BD%D0%B5%D1%82" title="Интернет">Интернету</a>, для доступа к данным, размещённым на удалённом компьютере, однако этой проблеме было найдено решение<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup>. Большой проблемой может оказаться попытка подключиться к компьютеру, если пропускная способность канала мала или если к этому компьютеру уже подключён один или несколько пользователей<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup>.
</p>
