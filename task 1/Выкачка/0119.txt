<p><b>SciTE</b> (от <a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;"><b>Sci</b>ntilla based <b>T</b>ext <b>E</b>ditor</span>) — <a class="mw-redirect" href="/wiki/%D0%9A%D1%80%D0%BE%D1%81%D1%81%D0%BF%D0%BB%D0%B0%D1%82%D1%84%D0%BE%D1%80%D0%BC%D0%B5%D0%BD%D0%BD%D0%BE%D0%B5_%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%BD%D0%BE%D0%B5_%D0%BE%D0%B1%D0%B5%D1%81%D0%BF%D0%B5%D1%87%D0%B5%D0%BD%D0%B8%D0%B5" title="Кроссплатформенное программное обеспечение">кроссплатформенный</a> <a href="/wiki/%D0%A2%D0%B5%D0%BA%D1%81%D1%82%D0%BE%D0%B2%D1%8B%D0%B9_%D1%80%D0%B5%D0%B4%D0%B0%D0%BA%D1%82%D0%BE%D1%80" title="Текстовый редактор">текстовый редактор</a>, созданный Нилом Ходжсоном (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Neil Hodgson</span>) с использованием разработанного им же компонента редактирования <a href="/wiki/Scintilla" title="Scintilla">Scintilla</a>. Предназначен, в основном, для редактирования исходных текстов <a href="/wiki/%D0%9A%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B0" title="Компьютерная программа">программ</a> и выполняет <a href="/wiki/%D0%9F%D0%BE%D0%B4%D1%81%D0%B2%D0%B5%D1%82%D0%BA%D0%B0_%D1%81%D0%B8%D0%BD%D1%82%D0%B0%D0%BA%D1%81%D0%B8%D1%81%D0%B0" title="Подсветка синтаксиса">подсветку синтаксиса</a> для многих языков программирования (<a href="/wiki/%D0%A1%D0%B8_(%D1%8F%D0%B7%D1%8B%D0%BA_%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D1%8F)" title="Си (язык программирования)">C</a>, <a href="/wiki/C%2B%2B" title="C++">C++</a>, <a href="/wiki/C_Sharp" title="C Sharp">C#</a>, <a href="/wiki/Java" title="Java">Java</a>, <a href="/wiki/Ruby" title="Ruby">Ruby</a>, <a class="mw-redirect" href="/wiki/Pascal" title="Pascal">Pascal</a>, <a href="/wiki/Perl" title="Perl">Perl</a>, <a href="/wiki/Python" title="Python">Python</a> и др.), разметки (<a href="/wiki/HTML" title="HTML">HTML</a>, <a href="/wiki/XML" title="XML">XML</a> и т. п.), описания аппаратуры (<a href="/wiki/VHDL" title="VHDL">VHDL</a>, <a href="/wiki/Verilog" title="Verilog">Verilog</a>, <a href="/wiki/SystemVerilog" title="SystemVerilog">SystemVerilog</a>). SciTE характеризуется<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> как компактный текстовый редактор для <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B8%D1%81%D1%82" title="Программист">программистов</a> с удобно реализованным <a href="/wiki/%D0%A1%D0%B2%D0%BE%D1%80%D0%B0%D1%87%D0%B8%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5_(%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%BD%D0%BE%D0%B5_%D0%BE%D0%B1%D0%B5%D1%81%D0%BF%D0%B5%D1%87%D0%B5%D0%BD%D0%B8%D0%B5)" title="Сворачивание (программное обеспечение)">сворачиванием кода</a>, развитой <a href="/wiki/%D0%9F%D0%BE%D0%B4%D1%81%D0%B2%D0%B5%D1%82%D0%BA%D0%B0_%D1%81%D0%B8%D0%BD%D1%82%D0%B0%D0%BA%D1%81%D0%B8%D1%81%D0%B0" title="Подсветка синтаксиса">подсветкой синтаксиса</a> и другими полезными функциями. SciTe является <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B5%D0%BD%D0%BE%D1%81%D0%B8%D0%BC%D0%BE%D0%B5_%D0%BF%D1%80%D0%B8%D0%BB%D0%BE%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5" title="Переносимое приложение">переносимым приложением</a> и работает при запуске с переносных устройств. Размер <a href="/wiki/%D0%94%D0%B8%D1%81%D1%82%D1%80%D0%B8%D0%B1%D1%83%D1%82%D0%B8%D0%B2" title="Дистрибутив">дистрибутива</a> SciTE не превышает <span class="nowrap">2,5 <a href="/wiki/%D0%9C%D0%B5%D0%B3%D0%B0%D0%B1%D0%B0%D0%B9%D1%82" title="Мегабайт">МБ</a></span>.
</p><p>SciTE был первоначально создан для демонстрации возможностей <a href="/wiki/Scintilla" title="Scintilla">Scintilla</a>, однако по мере развития стал широко используемым текстовым редактором. Количество загрузок программы превысило 4 млн<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>.
</p><p>Кроме Нейла Ходжсона, вклад в работу над SciTE сделали ещё множество других участников<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>.
</p>

<h2><span id=".D0.9F.D1.80.D0.BE.D0.B8.D0.B7.D0.BD.D0.BE.D1.88.D0.B5.D0.BD.D0.B8.D0.B5_.D0.BD.D0.B0.D0.B7.D0.B2.D0.B0.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="Произношение_названия">Произношение названия</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=SciTE&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Произношение названия»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=SciTE&amp;action=edit&amp;section=1" title="Редактировать код раздела «Произношение названия»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Не существует единого мнения о произношении названия «SciTE». (Scintilla Text Editor) <span class="IPA"><a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Международный фонетический алфавит">[saɪntilla]</a></span>.
</p><p>Многие пользователи произносят его как <i>sight</i> или <i>site</i> <span class="IPA"><a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Международный фонетический алфавит">[saɪt]</a></span>, в то время как сам автор придерживается произношения <i>skite</i> <span class="IPA"><a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Международный фонетический алфавит">[skaɪt]</a></span> или <i>shkite</i> <span class="IPA"><a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Международный фонетический алфавит">[ʃkaɪt]</a></span>.<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>
</p>
<h2><span id=".D0.9E.D1.81.D0.BE.D0.B1.D0.B5.D0.BD.D0.BD.D0.BE.D1.81.D1.82.D0.B8_SciTE"></span><span class="mw-headline" id="Особенности_SciTE">Особенности SciTE</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=SciTE&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Особенности SciTE»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=SciTE&amp;action=edit&amp;section=2" title="Редактировать код раздела «Особенности SciTE»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id=".D0.98.D0.BD.D1.82.D0.B5.D1.80.D1.84.D0.B5.D0.B9.D1.81"></span><span class="mw-headline" id="Интерфейс">Интерфейс</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=SciTE&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Интерфейс»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=SciTE&amp;action=edit&amp;section=3" title="Редактировать код раздела «Интерфейс»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/%D0%93%D1%80%D0%B0%D1%84%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B8%D0%BD%D1%82%D0%B5%D1%80%D1%84%D0%B5%D0%B9%D1%81_%D0%BF%D0%BE%D0%BB%D1%8C%D0%B7%D0%BE%D0%B2%D0%B0%D1%82%D0%B5%D0%BB%D1%8F" title="Графический интерфейс пользователя">Интерфейс</a> SciTE однооконный, многодокументный, с использованием <i>вкладок</i> для переключения между открытыми файлами. Присутствуют меню, панель инструментов, панель вкладок и панель статуса. Существуют файлы локализации для нескольких десятков языков, в том числе русского. Помимо естественных языков, существует файл локализации для международного языка <a href="/wiki/%D0%AD%D1%81%D0%BF%D0%B5%D1%80%D0%B0%D0%BD%D1%82%D0%BE" title="Эсперанто">Эсперанто</a>. Новые файлы локализации можно создавать путём перевода файла «<code>locale.properties</code>», входящего в состав дистрибутива<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>.
</p>
<h3><span id=".D0.9D.D0.B0.D1.81.D1.82.D1.80.D0.BE.D0.B9.D0.BA.D0.B0"></span><span class="mw-headline" id="Настройка">Настройка</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=SciTE&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Настройка»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=SciTE&amp;action=edit&amp;section=4" title="Редактировать код раздела «Настройка»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>SciTE обладает широкими возможностями по настройке, которые реализуются путём редактирования файлов конфигурации, быстрый доступ к которым предоставлен в разделе меню «Настройки». Графический интерфейс настройки отсутствует, за исключением настроек отступа<sup class="reference" id="cite_ref-SciTE_Documentation_6-0"><a href="#cite_note-SciTE_Documentation-6">[6]</a></sup>.
</p><p>Также возможности SciTE могут быть расширены с помощью <a class="mw-redirect" href="/wiki/%D0%A1%D0%BA%D1%80%D0%B8%D0%BF%D1%82" title="Скрипт">скриптов</a> на языке Lua, которые выполняются с помощью встроенного <a href="/wiki/%D0%98%D0%BD%D1%82%D0%B5%D1%80%D0%BF%D1%80%D0%B5%D1%82%D0%B0%D1%82%D0%BE%D1%80" title="Интерпретатор">интерпретатора</a> Lua 5.1<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>.
</p>
<h3><span id=".D0.A0.D0.B0.D0.B1.D0.BE.D1.82.D0.B0_.D1.81_.D1.84.D0.B0.D0.B9.D0.BB.D0.B0.D0.BC.D0.B8"></span><span class="mw-headline" id="Работа_с_файлами">Работа с файлами</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=SciTE&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Работа с файлами»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=SciTE&amp;action=edit&amp;section=5" title="Редактировать код раздела «Работа с файлами»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>SciTE позволяет открывать и сохранять файлы как в системной <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%B4%D0%B8%D1%80%D0%BE%D0%B2%D0%BA%D0%B0" title="Кодировка">кодировке</a>, так и в <a class="mw-redirect" href="/wiki/Unicode" title="Unicode">Unicode</a> (<a href="/wiki/UTF-8" title="UTF-8">UTF-8</a> и <a href="/wiki/UTF-16" title="UTF-16">UTF-16</a>).
</p><p>Возможен экспорт файлов в форматах HTML, <a class="mw-redirect" href="/wiki/RTF" title="RTF">RTF</a>, <a class="mw-redirect" href="/wiki/PDF" title="PDF">PDF</a>, <a href="/wiki/LaTeX" title="LaTeX">LaTeX</a>, XML с сохранением подсветки синтаксиса;
</p><p>Существует возможность сохранять и загружать <i>сессию</i> — набор открытых файлов. Это позволяет использовать SciTE для работы с различными проектами, состоящими из большого числа файлов.
</p><p>Существует ограничение на количество одновременно открытых файлов: максимальное значение задаётся в настройках и не может превышать 100.<sup class="reference" id="cite_ref-SciTE_Documentation_6-1"><a href="#cite_note-SciTE_Documentation-6">[6]</a></sup>
</p>
<h3><span id=".D0.A0.D0.B5.D0.B4.D0.B0.D0.BA.D1.82.D0.B8.D1.80.D0.BE.D0.B2.D0.B0.D0.BD.D0.B8.D0.B5_.D1.82.D0.B5.D0.BA.D1.81.D1.82.D0.B0"></span><span class="mw-headline" id="Редактирование_текста">Редактирование текста</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=SciTE&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Редактирование текста»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=SciTE&amp;action=edit&amp;section=6" title="Редактировать код раздела «Редактирование текста»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Особенности реализации отображения текста в SciTE:
</p>
<ul><li>возможность <a class="mw-redirect" href="/wiki/%D0%91%D1%83%D1%84%D0%B5%D1%80%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F" title="Буферизация">буферизации</a> графики, чтобы избежать мерцания при прорисовке символов;</li>
<li>одновременное использование различных, в том числе и <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%BF%D0%BE%D1%80%D1%86%D0%B8%D0%BE%D0%BD%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D1%88%D1%80%D0%B8%D1%84%D1%82" title="Пропорциональный шрифт">пропорциональных</a>, <a href="/wiki/%D0%A8%D1%80%D0%B8%D1%84%D1%82" title="Шрифт">шрифтов</a>;</li>
<li>быстрое изменение размера шрифтов;</li>
<li>отображение номеров строк;</li>
<li>подсветка синтаксиса для большого количества <a href="/wiki/%D0%AF%D0%B7%D1%8B%D0%BA_%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D1%8F" title="Язык программирования">языков программирования</a>;</li>
<li><a href="/wiki/%D0%A1%D0%B2%D0%BE%D1%80%D0%B0%D1%87%D0%B8%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5_(%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%BD%D0%BE%D0%B5_%D0%BE%D0%B1%D0%B5%D1%81%D0%BF%D0%B5%D1%87%D0%B5%D0%BD%D0%B8%D0%B5)" title="Сворачивание (программное обеспечение)">фолдинг</a>, то есть сворачивание и разворачивание структурных блоков текста (<a href="/wiki/%D0%9A%D0%BB%D0%B0%D1%81%D1%81_(%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5)" title="Класс (программирование)">классов</a>, <a href="/wiki/%D0%A4%D1%83%D0%BD%D0%BA%D1%86%D0%B8%D1%8F_(%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5)" title="Функция (программирование)">функций</a>, <a href="/wiki/%D0%A6%D0%B8%D0%BA%D0%BB_(%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5)" title="Цикл (программирование)">циклов</a> и т. п.);</li></ul>
<p>Особенности реализации редактирования:
</p>
<ul><li>автоматическая установка отступов;</li>
<li>подсветка парных или непарных (незакрытых) скобок;</li>
<li>автодополнение (автоматическое завершение) используемых в файле имён типов, функций, переменных;</li>
<li>всплывающие подсказки о параметрах функций;</li>
<li>установка закладок;</li>
<li>быстрый поиск по файлу и фиксируемое цветовое выделение выражения под курсором, клавиатурная навигация по выделенным фрагментам;</li>
<li>подсветка всех видимых копий слова под курсором;</li>
<li>поиск и замена текста с использованием регулярных выражений во всех открытых файлах (вместе с 2 предыдущими пунктами — инструменты упрощения рефакторинга);</li></ul>

<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:%D0%9E%D0%B1%D1%80%D0%B0%D0%B1%D0%BE%D1%82%D0%BA%D0%B0_%D1%81%D0%BE%D0%BE%D0%B1%D1%89%D0%B5%D0%BD%D0%B8%D0%B9_%D0%BA%D0%BE%D0%BC%D0%BF%D0%B8%D0%BB%D1%8F%D1%82%D0%BE%D1%80%D0%B0_%D0%B2_SciTE.png"><img class="mw-file-element" data-file-height="507" data-file-width="648" decoding="async" height="156" src="//upload.wikimedia.org/wikipedia/ru/thumb/4/4f/%D0%9E%D0%B1%D1%80%D0%B0%D0%B1%D0%BE%D1%82%D0%BA%D0%B0_%D1%81%D0%BE%D0%BE%D0%B1%D1%89%D0%B5%D0%BD%D0%B8%D0%B9_%D0%BA%D0%BE%D0%BC%D0%BF%D0%B8%D0%BB%D1%8F%D1%82%D0%BE%D1%80%D0%B0_%D0%B2_SciTE.png/200px-%D0%9E%D0%B1%D1%80%D0%B0%D0%B1%D0%BE%D1%82%D0%BA%D0%B0_%D1%81%D0%BE%D0%BE%D0%B1%D1%89%D0%B5%D0%BD%D0%B8%D0%B9_%D0%BA%D0%BE%D0%BC%D0%BF%D0%B8%D0%BB%D1%8F%D1%82%D0%BE%D1%80%D0%B0_%D0%B2_SciTE.png" srcset="//upload.wikimedia.org/wikipedia/ru/thumb/4/4f/%D0%9E%D0%B1%D1%80%D0%B0%D0%B1%D0%BE%D1%82%D0%BA%D0%B0_%D1%81%D0%BE%D0%BE%D0%B1%D1%89%D0%B5%D0%BD%D0%B8%D0%B9_%D0%BA%D0%BE%D0%BC%D0%BF%D0%B8%D0%BB%D1%8F%D1%82%D0%BE%D1%80%D0%B0_%D0%B2_SciTE.png/300px-%D0%9E%D0%B1%D1%80%D0%B0%D0%B1%D0%BE%D1%82%D0%BA%D0%B0_%D1%81%D0%BE%D0%BE%D0%B1%D1%89%D0%B5%D0%BD%D0%B8%D0%B9_%D0%BA%D0%BE%D0%BC%D0%BF%D0%B8%D0%BB%D1%8F%D1%82%D0%BE%D1%80%D0%B0_%D0%B2_SciTE.png 1.5x, //upload.wikimedia.org/wikipedia/ru/thumb/4/4f/%D0%9E%D0%B1%D1%80%D0%B0%D0%B1%D0%BE%D1%82%D0%BA%D0%B0_%D1%81%D0%BE%D0%BE%D0%B1%D1%89%D0%B5%D0%BD%D0%B8%D0%B9_%D0%BA%D0%BE%D0%BC%D0%BF%D0%B8%D0%BB%D1%8F%D1%82%D0%BE%D1%80%D0%B0_%D0%B2_SciTE.png/400px-%D0%9E%D0%B1%D1%80%D0%B0%D0%B1%D0%BE%D1%82%D0%BA%D0%B0_%D1%81%D0%BE%D0%BE%D0%B1%D1%89%D0%B5%D0%BD%D0%B8%D0%B9_%D0%BA%D0%BE%D0%BC%D0%BF%D0%B8%D0%BB%D1%8F%D1%82%D0%BE%D1%80%D0%B0_%D0%B2_SciTE.png 2x" width="200"/></a><figcaption>Обработка сообщений компилятора в SciTE</figcaption></figure>
<p>SciTE позволяет вызывать внешние программы для обработки открытых файлов (например, <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%BC%D0%BF%D0%B8%D0%BB%D1%8F%D1%82%D0%BE%D1%80%D1%8B" title="Компиляторы">компиляторы</a>) с помощью команд из раздела меню «Tools» (с <span style="font-style:normal"><a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a></span><span class="nowrap"> —</span> «инструменты»). Команды могут быть настроены пользователем и различаются в зависимости от типа обрабатываемого файла.
</p><p>Результаты работы отображаются в <i>окне консоли</i>, которое появляется в нижней (или правой) части окна SciTE. При этом сообщения об ошибках выделяются цветом. С помощью щелчка по сообщению можно перейти к соответствующей строке в редактируемом файле.
</p>

<p>SciTE распространяется как в виде готовых <a href="/wiki/%D0%94%D0%B8%D1%81%D1%82%D1%80%D0%B8%D0%B1%D1%83%D1%82%D0%B8%D0%B2" title="Дистрибутив">дистрибутивов</a> для <a href="/wiki/%D0%9E%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D0%B0%D1%8F_%D1%81%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0" title="Операционная система">ОС</a> <a href="/wiki/Linux" title="Linux">Linux</a> и <a class="mw-redirect" href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a>, так и в виде исходных текстов на C++, которые могут быть <a href="/wiki/%D0%9A%D0%BE%D0%BC%D0%BF%D0%B8%D0%BB%D1%8F%D1%82%D0%BE%D1%80" title="Компилятор">скомпилированы</a>:
</p>

<li>в <a href="/wiki/%D0%9E%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D0%B0%D1%8F_%D1%81%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0" title="Операционная система">ОС</a> <a class="mw-redirect" href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a>, с использованием одного из следующих компиляторов:

<p>В <a href="/wiki/%D0%9E%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D0%B0%D1%8F_%D1%81%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0" title="Операционная система">ОС</a> <a href="/wiki/Linux" title="Linux">Linux</a> <a href="/wiki/%D0%93%D1%80%D0%B0%D1%84%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B8%D0%BD%D1%82%D0%B5%D1%80%D1%84%D0%B5%D0%B9%D1%81_%D0%BF%D0%BE%D0%BB%D1%8C%D0%B7%D0%BE%D0%B2%D0%B0%D1%82%D0%B5%D0%BB%D1%8F" title="Графический интерфейс пользователя">интерфейс</a> SciTE реализован с помощью <a class="mw-redirect" href="/wiki/GTK%2B" title="GTK+">GTK+</a>, в <a href="/wiki/%D0%9E%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D0%B0%D1%8F_%D1%81%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0" title="Операционная система">ОС</a> <a class="mw-redirect" href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a> — на <a href="/wiki/Windows_API" title="Windows API">Windows API</a>.
