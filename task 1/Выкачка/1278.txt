<p><b>Евфроси́ния Колюпа́новская</b> (в миру — <i>Евдоки́я Григо́рьевна Вя́земская</i>; ок. <a class="mw-redirect" href="/wiki/1758" title="1758">1758</a> — 3 июля <a class="mw-redirect" href="/wiki/1855" title="1855">1855</a>, село Колюпаново Алексинского уезда <a href="/wiki/%D0%A2%D1%83%D0%BB%D1%8C%D1%81%D0%BA%D0%B0%D1%8F_%D0%B3%D1%83%D0%B1%D0%B5%D1%80%D0%BD%D0%B8%D1%8F" title="Тульская губерния">Тульской губернии</a>) — <a href="/wiki/%D0%9F%D1%80%D0%B0%D0%B2%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%B8%D0%B5" title="Православие">православная</a> подвижница. Княжна из рода <a href="/wiki/%D0%92%D1%8F%D0%B7%D0%B5%D0%BC%D1%81%D0%BA%D0%B8%D0%B5" title="Вяземские">Вяземских</a>, <a href="/wiki/%D0%A4%D1%80%D0%B5%D0%B9%D0%BB%D0%B8%D0%BD%D0%B0" title="Фрейлина">фрейлина</a> императрицы <a href="/wiki/%D0%95%D0%BA%D0%B0%D1%82%D0%B5%D1%80%D0%B8%D0%BD%D0%B0_II" title="Екатерина II">Екатерины II</a>. Желая посвятить себя <a href="/wiki/%D0%91%D0%BE%D0%B3" title="Бог">Богу</a>, тайно оставила двор, инсценировав свою смерть, и стала <a href="/wiki/%D0%AE%D1%80%D0%BE%D0%B4%D1%81%D1%82%D0%B2%D0%BE" title="Юродство">юродивой</a>.
</p><p><a href="/wiki/%D0%9A%D0%B0%D0%BD%D0%BE%D0%BD%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F" title="Канонизация">Канонизирована</a> в <a href="/wiki/1988_%D0%B3%D0%BE%D0%B4" title="1988 год">1988 году</a> как <a class="mw-redirect" href="/wiki/%D0%9C%D0%B5%D1%81%D1%82%D0%BD%D0%BE%D1%87%D1%82%D0%B8%D0%BC%D1%8B%D0%B9_%D1%81%D0%B2%D1%8F%D1%82%D0%BE%D0%B9" title="Местночтимый святой">местночтимая святая</a> <a href="/wiki/%D0%A2%D1%83%D0%BB%D1%8C%D1%81%D0%BA%D0%B0%D1%8F_%D0%B5%D0%BF%D0%B0%D1%80%D1%85%D0%B8%D1%8F" title="Тульская епархия">Тульской епархии</a> в <a href="/wiki/%D0%9B%D0%B8%D0%BA_%D1%81%D0%B2%D1%8F%D1%82%D0%BE%D1%81%D1%82%D0%B8" title="Лик святости">лике</a> блаженных, память совершается (по <a href="/wiki/%D0%AE%D0%BB%D0%B8%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C" title="Юлианский календарь">юлианскому календарю</a>): 3 июля и 22 сентября (<a href="/wiki/%D0%A1%D0%BE%D0%B1%D0%BE%D1%80_%D0%A2%D1%83%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D1%85_%D1%81%D0%B2%D1%8F%D1%82%D1%8B%D1%85" title="Собор Тульских святых">собор Тульских святых</a>).
</p>

<h3><span id=".D0.9C.D0.B8.D1.80.D1.81.D0.BA.D0.B0.D1.8F_.D0.B6.D0.B8.D0.B7.D0.BD.D1.8C"></span><span class="mw-headline" id="Мирская_жизнь">Мирская жизнь</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%95%D0%B2%D1%84%D1%80%D0%BE%D1%81%D0%B8%D0%BD%D0%B8%D1%8F_%D0%9A%D0%BE%D0%BB%D1%8E%D0%BF%D0%B0%D0%BD%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Мирская жизнь»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%95%D0%B2%D1%84%D1%80%D0%BE%D1%81%D0%B8%D0%BD%D0%B8%D1%8F_%D0%9A%D0%BE%D0%BB%D1%8E%D0%BF%D0%B0%D0%BD%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F&amp;action=edit&amp;section=2" title="Редактировать код раздела «Мирская жизнь»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Сведения о жизни Евфросинии в миру крайне скудны. Родилась она около <a href="/wiki/1758_%D0%B3%D0%BE%D0%B4" title="1758 год">1758 года</a><sup class="reference" id="cite_ref-ПЭ_1-0"><a href="#cite_note-ПЭ-1">[1]</a></sup> в семье князя Григория Ивановича Вяземского (представитель младшей ветви рода князей <a href="/wiki/%D0%92%D1%8F%D0%B7%D0%B5%D0%BC%D1%81%D0%BA%D0%B8%D0%B5" title="Вяземские">Вяземских</a><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>). Княжна Евдокия в <a href="/wiki/1776_%D0%B3%D0%BE%D0%B4" title="1776 год">1776 году</a> стала выпускницей первого выпуска воспитанниц Воспитательного общества благородных девиц при <a href="/wiki/%D0%A1%D0%BC%D0%BE%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BC%D0%BE%D0%BD%D0%B0%D1%81%D1%82%D1%8B%D1%80%D1%8C" title="Смольный монастырь">Смольном монастыре</a> (позже — <a href="/wiki/%D0%A1%D0%BC%D0%BE%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%B8%D0%BD%D1%81%D1%82%D0%B8%D1%82%D1%83%D1%82_%D0%B1%D0%BB%D0%B0%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D1%85_%D0%B4%D0%B5%D0%B2%D0%B8%D1%86" title="Смольный институт благородных девиц">Смольный институт</a>) и была пожалована во <a href="/wiki/%D0%A4%D1%80%D0%B5%D0%B9%D0%BB%D0%B8%D0%BD%D0%B0" title="Фрейлина">фрейлины</a> ко двору императрицы <a href="/wiki/%D0%95%D0%BA%D0%B0%D1%82%D0%B5%D1%80%D0%B8%D0%BD%D0%B0_II" title="Екатерина II">Екатерины II</a><sup class="reference" id="cite_ref-сур_3-0"><a href="#cite_note-сур-3">[3]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:6</sup>.
</p><p>Благодаря своему происхождению Евдокия Вяземская была знакома со многими представителями высшего общества своего времени, включая полководца <a href="/wiki/%D0%A1%D1%83%D0%B2%D0%BE%D1%80%D0%BE%D0%B2,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%92%D0%B0%D1%81%D0%B8%D0%BB%D1%8C%D0%B5%D0%B2%D0%B8%D1%87" title="Суворов, Александр Васильевич">Александра Суворова</a> и князя <a href="/wiki/%D0%94%D0%BE%D0%BB%D0%B3%D0%BE%D1%80%D1%83%D0%BA%D0%BE%D0%B2,_%D0%AE%D1%80%D0%B8%D0%B9_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Долгоруков, Юрий Владимирович">Юрия Долгорукова</a> (будущий московский градоначальник). По её рассказам, она вместе с <a href="/wiki/%D0%9A%D0%B0%D0%BC%D0%B5%D1%80%D0%B3%D0%B5%D1%80" title="Камергер">камергером</a> <a class="mw-redirect" href="/wiki/%D0%9D%D0%B0%D1%80%D1%8B%D1%88%D0%BA%D0%B8%D0%BD,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%9B%D1%8C%D0%B2%D0%BE%D0%B2%D0%B8%D1%87_(1760%E2%80%941826)" title="Нарышкин, Александр Львович (1760—1826)">Александром Нарышкиным</a> (будущий главный директор императорских театров) развлекала скуку императрицы<sup class="reference" id="cite_ref-сур_3-1"><a href="#cite_note-сур-3">[3]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:6</sup>.
</p><p>Сколько времени Евдокия прожила при дворе — неизвестно. Однажды во время пребывания императорского двора в <a class="mw-redirect" href="/wiki/%D0%A6%D0%B0%D1%80%D1%81%D0%BA%D0%BE%D0%B5_%D0%A1%D0%B5%D0%BB%D0%BE" title="Царское Село">Царском Селе</a> она вместе с двумя другими фрейлинами (М. Я. Сониной и девицей Саломией) инсценировала свою смерть. Девушки оставили свои платья на берегу <a href="/wiki/%D0%9F%D1%80%D1%83%D0%B4" title="Пруд">пруда</a>, и, одевшись <a class="mw-redirect" href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82%D1%8C%D1%8F%D0%BD%D0%B5" title="Крестьяне">крестьянками</a>, бежали, начав странствовать<sup class="reference" id="cite_ref-ПЭ_1-1"><a href="#cite_note-ПЭ-1">[1]</a></sup>. Причины и обстоятельства этого поступка остаются неизвестными. По словам <a href="/wiki/%D0%94%D1%83%D1%85%D0%BE%D0%B2%D0%BD%D0%B8%D0%BA" title="Духовник">духовника</a> старицы Евфросинии, причиной её бегства стало желание посвятить себя <a href="/wiki/%D0%91%D0%BE%D0%B3" title="Бог">Богу</a>. Он сообщает следующее предание, связанное с её бегством, в котором отсутствует инсценировка утопления:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
…она просто скрылась из Дворца, почему и было дано приказание по разным дорогам, чтобы задержать её, и когда она, переодевшись в крестьянское платье, пробиралась в Москву, на перевозе через одну реку была узнана исправником, и, согласно предписанию, возвращена в столицу. Императрица приняла её ласково, и испытав о причине ея бегства с другими придворными, притом уверившись в ея твердой решимости посвятить себя на служение Богу, дозволила вступить в монастырь по желанию, и отпуская от себя, подарила иноческое платье из блестящей материи…<sup class="reference" id="cite_ref-dombeseda_4-0"><a href="#cite_note-dombeseda-4">[4]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:18</sup>
</p>
</blockquote>
<h3><span id=".D0.9D.D0.B0.D1.87.D0.B0.D0.BB.D0.BE_.D1.81.D1.82.D1.80.D0.B0.D0.BD.D1.81.D1.82.D0.B2.D0.B8.D0.B9._.D0.92.D0.B2.D0.B5.D0.B4.D0.B5.D0.BD.D1.81.D0.BA.D0.B8.D0.B9_.D0.92.D0.BB.D0.B0.D0.B4.D1.8B.D1.87.D0.BD.D1.8B.D0.B9_.D0.BC.D0.BE.D0.BD.D0.B0.D1.81.D1.82.D1.8B.D1.80.D1.8C"></span><span class="mw-headline" id="Начало_странствий._Введенский_Владычный_монастырь">Начало странствий. Введенский Владычный монастырь</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%95%D0%B2%D1%84%D1%80%D0%BE%D1%81%D0%B8%D0%BD%D0%B8%D1%8F_%D0%9A%D0%BE%D0%BB%D1%8E%D0%BF%D0%B0%D0%BD%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Начало странствий. Введенский Владычный монастырь»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%95%D0%B2%D1%84%D1%80%D0%BE%D1%81%D0%B8%D0%BD%D0%B8%D1%8F_%D0%9A%D0%BE%D0%BB%D1%8E%D0%BF%D0%B0%D0%BD%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F&amp;action=edit&amp;section=3" title="Редактировать код раздела «Начало странствий. Введенский Владычный монастырь»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Evfrosiniya_Kolupanovskaya_02.jpg"><img class="mw-file-element" data-file-height="679" data-file-width="476" decoding="async" height="285" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/8a/Evfrosiniya_Kolupanovskaya_02.jpg/200px-Evfrosiniya_Kolupanovskaya_02.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/8a/Evfrosiniya_Kolupanovskaya_02.jpg/300px-Evfrosiniya_Kolupanovskaya_02.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/8a/Evfrosiniya_Kolupanovskaya_02.jpg/400px-Evfrosiniya_Kolupanovskaya_02.jpg 2x" width="200"/></a><figcaption><center>Старица Евфросиния</center></figcaption></figure>
<p>После ухода из мира Евдокия начала свои скитания по различным монастырям и храмам, которые продолжались более десяти лет. По её рассказам, некоторое время она трудилась на скотном дворе <a href="/wiki/%D0%A1%D0%BF%D0%B0%D1%81%D0%BE-%D0%A1%D1%83%D0%BC%D0%BE%D1%80%D0%B8%D0%BD_%D0%BC%D0%BE%D0%BD%D0%B0%D1%81%D1%82%D1%8B%D1%80%D1%8C" title="Спасо-Суморин монастырь">Спасо-Суморина монастыря</a>, где находились <a href="/wiki/%D0%9C%D0%BE%D1%89%D0%B8" title="Мощи">мощи</a> <a href="/wiki/%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D0%B9_%D0%A2%D0%BE%D1%82%D0%B5%D0%BC%D1%81%D0%BA%D0%B8%D0%B9" title="Феодосий Тотемский">Феодосия Тотемского</a><sup class="reference" id="cite_ref-сур_3-2"><a href="#cite_note-сур-3">[3]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:8</sup>. Когда физические силы её оставили, она направилась в <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%B2%D0%B0" title="Москва">Москву</a>, где в <a href="/wiki/1806_%D0%B3%D0%BE%D0%B4" title="1806 год">1806 году</a> встретилась с митрополитом <a href="/wiki/%D0%9F%D0%BB%D0%B0%D1%82%D0%BE%D0%BD_(%D0%9B%D0%B5%D0%B2%D1%88%D0%B8%D0%BD)" title="Платон (Левшин)">Платоном</a>, который, выслушав её историю, дал ей <a href="/wiki/%D0%91%D0%BB%D0%B0%D0%B3%D0%BE%D1%81%D0%BB%D0%BE%D0%B2%D0%B5%D0%BD%D0%B8%D0%B5" title="Благословение">благословение</a> на подвиг <a href="/wiki/%D0%AE%D1%80%D0%BE%D0%B4%D1%81%D1%82%D0%B2%D0%BE" title="Юродство">юродства</a><sup class="reference" id="cite_ref-ПЭ_1-2"><a href="#cite_note-ПЭ-1">[1]</a></sup>. Со своим письмом митрополит направил княжну во <a href="/wiki/%D0%92%D0%B2%D0%B5%D0%B4%D0%B5%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%92%D0%BB%D0%B0%D0%B4%D1%8B%D1%87%D0%BD%D1%8B%D0%B9_%D0%BC%D0%BE%D0%BD%D0%B0%D1%81%D1%82%D1%8B%D1%80%D1%8C" title="Введенский Владычный монастырь">Серпуховской Владычний Введенский женский монастырь</a> под вымышленным именем «<i>дуры Евфросинии</i>».
</p><p><a class="mw-redirect" href="/wiki/%D0%98%D0%B3%D1%83%D0%BC%D0%B5%D0%BD%D1%8C%D1%8F" title="Игуменья">Игуменья</a> Дионисия милостиво приняла новую насельницу. Вначале она жила в сестринском корпусе, а затем поселилась в небольшой избушке в окрестностях монастыря. По свидетельству Виталии (<a href="/wiki/%D0%9F%D0%BE%D1%81%D0%BB%D1%83%D1%88%D0%BD%D0%B8%D0%BA" title="Послушник">послушница</a>, а затем монахиня Владычного монастыря) митрополит Платон в своём письме известил игуменью о происхождении Евфросинии, о нём знали и некоторые другие монахини<sup class="reference" id="cite_ref-сур_3-3"><a href="#cite_note-сур-3">[3]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:45</sup>.
</p><p>Евфросиния одевалась в рубашку из толстого сукна, поверх её надевала ветхий <a class="new" href="/w/index.php?title=%D0%9A%D0%B0%D0%BF%D0%BE%D1%82_(%D0%BE%D0%B4%D0%B5%D0%B6%D0%B4%D0%B0)&amp;action=edit&amp;redlink=1" title="Капот (одежда) (страница отсутствует)">капот</a>, а в зимнее время — мужской <a href="/wiki/%D0%A2%D1%83%D0%BB%D1%83%D0%BF_(%D0%BE%D0%B4%D0%B5%D0%B6%D0%B4%D0%B0)" title="Тулуп (одежда)">тулуп</a>. В праздничные дни Еврофсиния надевала кафтан из серого домотканого сукна. С её разрешения был сделан её портрет в таком костюме (см. слева)<sup class="reference" id="cite_ref-dombeseda_4-1"><a href="#cite_note-dombeseda-4">[4]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:21</sup>. На теле под одеждой она носила <a href="/wiki/%D0%92%D0%B5%D1%80%D0%B8%D0%B3%D0%B8" title="Вериги">вериги</a> — железную цепь на шее с крупным медным <a href="/wiki/%D0%A0%D0%B0%D1%81%D0%BF%D1%8F%D1%82%D0%B8%D0%B5_(%D0%B4%D0%B5%D0%BA%D0%BE%D1%80%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D0%BE-%D0%BF%D1%80%D0%B8%D0%BA%D0%BB%D0%B0%D0%B4%D0%BD%D0%BE%D0%B5_%D0%B8%D1%81%D0%BA%D1%83%D1%81%D1%81%D1%82%D0%B2%D0%BE)" title="Распятие (декоративно-прикладное искусство)">крестом</a><sup class="reference" id="cite_ref-сур_3-4"><a href="#cite_note-сур-3">[3]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:10</sup>. Веригоношение было тайным. Когда помещица Е. А. Дубровина случайно открыла, что старица носит вериги, то Евфросиния сказала, что это её тайна и запретила её разглашать. Волосы её были короткостриженными, иногда она обматывала голову тряпицей или надевала шапочку. Старица всегда ходила босой, не надевая обувь даже зимой.
</p><p>В своей избушке Евфросиния держала двух <a href="/wiki/%D0%9A%D0%BE%D1%88%D0%BA%D0%B0" title="Кошка">кошек</a>, трёх <a href="/wiki/%D0%A1%D0%BE%D0%B1%D0%B0%D0%BA%D0%B0" title="Собака">собак</a>, <a href="/wiki/%D0%9A%D1%83%D1%80%D0%B8%D1%86%D0%B0" title="Курица">кур</a> и <a href="/wiki/%D0%98%D0%BD%D0%B4%D0%B5%D0%B9%D0%BA%D0%B0" title="Индейка">индеек</a>. При этом спала на полу вместе со своими собаками. На вопрос, зачем она это делает, старица отвечала — «<i>Я хуже собак</i>»<sup class="reference" id="cite_ref-сур_3-5"><a href="#cite_note-сур-3">[3]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:10</sup>. В доме она никогда не убиралась, пол был завален остатками пищи её животных, отчего в помещении стоял <a class="new" href="/w/index.php?title=%D0%A1%D0%BC%D1%80%D0%B0%D0%B4&amp;action=edit&amp;redlink=1" title="Смрад (страница отсутствует)">смрад</a>:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
Однажды игуменья Евгения Озерова сказала «<i>Матушка, зачем вы держите животных? Такой ужасный воздух!</i>». Она засмеялась и отвечала: «<i>Это мне заменяет духи, которых так много я употребляла при дворе</i>»<sup class="reference" id="cite_ref-сур_3-6"><a href="#cite_note-сур-3">[3]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:10</sup>.
</p>
</blockquote>
<p>При этом летом в жару она топила в своей избе <a href="/wiki/%D0%9F%D0%B5%D1%87%D1%8C" title="Печь">печь</a>, а зимой, наоборот, в её доме стоял холод.
</p>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Chapel_in_Vladychny_Monastery.jpg"><img class="mw-file-element" data-file-height="795" data-file-width="581" decoding="async" height="274" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/96/Chapel_in_Vladychny_Monastery.jpg/200px-Chapel_in_Vladychny_Monastery.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/96/Chapel_in_Vladychny_Monastery.jpg/300px-Chapel_in_Vladychny_Monastery.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/9/96/Chapel_in_Vladychny_Monastery.jpg/400px-Chapel_in_Vladychny_Monastery.jpg 2x" width="200"/></a><figcaption><center>Монастырская часовня, которую посещала Евфросиния</center></figcaption></figure>
<p>По ночам старица ходила вокруг монастыря, распевая <a class="mw-redirect" href="/wiki/%D0%9F%D1%81%D0%B0%D0%BB%D0%BC%D1%8B" title="Псалмы">псалмы</a>, а днём собирала в монастырском <a href="/wiki/%D0%91%D0%BE%D1%80_(%D0%BB%D0%B5%D1%81)" title="Бор (лес)">бору</a> <a href="/wiki/%D0%93%D1%80%D0%B8%D0%B1" title="Гриб">грибы</a>, травы и цветы. Травы Евфросиния отдавала приходившим к ней больным. Для молитв Евфросиния посещала <a href="/wiki/%D0%A7%D0%B0%D1%81%D0%BE%D0%B2%D0%BD%D1%8F" title="Часовня">часовню</a>, располагавшуюся рядом с монастырём, а в <a href="/wiki/%D0%9A%D0%B0%D1%84%D0%BE%D0%BB%D0%B8%D0%BA%D0%BE%D0%BD" title="Кафоликон">монастырский собор</a> приходила на <a class="mw-redirect" href="/wiki/%D0%9B%D0%B8%D1%82%D1%83%D1%80%D0%B3%D0%B8%D1%8F" title="Литургия">литургию</a><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>. На <a href="/wiki/%D0%98%D1%81%D0%BF%D0%BE%D0%B2%D0%B5%D0%B4%D1%8C" title="Исповедь">исповедь</a> Евфросиния ходила к монастырскому <a href="/wiki/%D0%94%D1%83%D1%85%D0%BE%D0%B2%D0%BD%D0%B8%D0%BA" title="Духовник">духовнику</a>, причащалась всего один раз в год в <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B8%D0%B9_%D1%87%D0%B5%D1%82%D0%B2%D0%B5%D1%80%D0%B3" title="Великий четверг">Великий четверг</a>. В день празднования <a href="/wiki/%D0%9A%D1%80%D0%B5%D1%89%D0%B5%D0%BD%D0%B8%D0%B5_%D0%93%D0%BE%D1%81%D0%BF%D0%BE%D0%B4%D0%BD%D0%B5" title="Крещение Господне">Крещения Господня</a> вместе с <a href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82%D0%BD%D1%8B%D0%B9_%D1%85%D0%BE%D0%B4" title="Крестный ход">крестным ходом</a> ходила на <a href="/wiki/%D0%A1%D0%B2%D1%8F%D1%82%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B4%D0%B0#Освящение_воды" title="Святая вода">иордань</a> на реке <a href="/wiki/%D0%9D%D0%B0%D1%80%D0%B0_(%D1%80%D0%B5%D0%BA%D0%B0)" title="Нара (река)">Нара</a> и в своей власянице окуналась в воду, призывая других: «<i>Идите, ребята, горячая баня, ступайте, мойтесь!</i>»<sup class="reference" id="cite_ref-ПЭ_1-3"><a href="#cite_note-ПЭ-1">[1]</a></sup>.
</p><p>В период своей жизни во Владычном монастыре Евфросиния неоднократно встречалась с митрополитом Московским и Коломенским <a href="/wiki/%D0%A4%D0%B8%D0%BB%D0%B0%D1%80%D0%B5%D1%82_(%D0%94%D1%80%D0%BE%D0%B7%D0%B4%D0%BE%D0%B2)" title="Филарет (Дроздов)">Филаретом (Дроздовым)</a>, который почитал её как подвижницу:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
Когда митрополит приезжал в монастырь, старица выходила за монастырскую ограду и встречала его, подходила к нему под благословение, целовала ему руку, и он так же целовал у неё руку, а когда он уезжал из монастыря, старица провожала его за монастырские ворота<sup class="reference" id="cite_ref-сур_3-7"><a href="#cite_note-сур-3">[3]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:10</sup>.
</p>
</blockquote>
<p>После прихода в Введенский монастырь новой игуменьи Иларии начались притеснения Евфросинии. В одних жизнеописаниях не указывается характер этих притеснений и сообщается только, что это была <a href="/wiki/%D0%97%D0%B0%D0%B2%D0%B8%D1%81%D1%82%D1%8C" title="Зависть">зависть</a> и злоба<sup class="reference" id="cite_ref-dombeseda_4-2"><a href="#cite_note-dombeseda-4">[4]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:19</sup>. В других причиной отъезда Евфросинии из Владычного монастыря называется убийство её трёх собак, совершённое по указанию новой игуменьи в присутствии старицы<sup class="reference" id="cite_ref-сур_3-8"><a href="#cite_note-сур-3">[3]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:47</sup>. В <a href="/wiki/1845_%D0%B3%D0%BE%D0%B4" title="1845 год">1845 году</a> Евфросиния оставляет монастырь и переселяется в село <a href="/wiki/%D0%9A%D0%BE%D0%BB%D1%8E%D0%BF%D0%B0%D0%BD%D0%BE%D0%B2%D0%BE_(%D1%81%D0%B5%D0%BB%D0%BE,_%D0%A2%D1%83%D0%BB%D1%8C%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C)" title="Колюпаново (село, Тульская область)">Колюпаново</a> <a href="/wiki/%D0%A2%D1%83%D0%BB%D1%8C%D1%81%D0%BA%D0%B0%D1%8F_%D0%B3%D1%83%D0%B1%D0%B5%D1%80%D0%BD%D0%B8%D1%8F" title="Тульская губерния">Тульской губернии</a>, расположенное на правом берегу <a href="/wiki/%D0%9E%D0%BA%D0%B0" title="Ока">Оки</a>.
</p>
<h3><span id=".D0.96.D0.B8.D0.B7.D0.BD.D1.8C_.D0.B2_.D1.81.D0.B5.D0.BB.D0.B5_.D0.9A.D0.BE.D0.BB.D1.8E.D0.BF.D0.B0.D0.BD.D0.BE.D0.B2.D0.BE"></span><span class="mw-headline" id="Жизнь_в_селе_Колюпаново">Жизнь в селе Колюпаново</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%95%D0%B2%D1%84%D1%80%D0%BE%D1%81%D0%B8%D0%BD%D0%B8%D1%8F_%D0%9A%D0%BE%D0%BB%D1%8E%D0%BF%D0%B0%D0%BD%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Жизнь в селе Колюпаново»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%95%D0%B2%D1%84%D1%80%D0%BE%D1%81%D0%B8%D0%BD%D0%B8%D1%8F_%D0%9A%D0%BE%D0%BB%D1%8E%D0%BF%D0%B0%D0%BD%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F&amp;action=edit&amp;section=4" title="Редактировать код раздела «Жизнь в селе Колюпаново»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Evfrosiniya_Kolupanovskaya_05.jpg"><img class="mw-file-element" data-file-height="647" data-file-width="971" decoding="async" height="180" src="//upload.wikimedia.org/wikipedia/commons/thumb/5/58/Evfrosiniya_Kolupanovskaya_05.jpg/270px-Evfrosiniya_Kolupanovskaya_05.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/5/58/Evfrosiniya_Kolupanovskaya_05.jpg/405px-Evfrosiniya_Kolupanovskaya_05.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/5/58/Evfrosiniya_Kolupanovskaya_05.jpg/540px-Evfrosiniya_Kolupanovskaya_05.jpg 2x" width="270"/></a><figcaption><center>Избушка блаженной Евфросинии, построенная ей Алексеем Цемшем</center></figcaption></figure>
<p>В село Колюпаново Евфросиния переселилась по приглашению одной из своих почитательниц — <a href="/wiki/%D0%9F%D0%BE%D0%BC%D0%B5%D1%89%D0%B8%D0%BA" title="Помещик">помещицы</a> Натальи Алексеевны Протопоповой и прожила в нём последние 10 лет своей жизни. Из Владычного монастыря Евфросиния взяла с собой только икону <a href="/wiki/%D0%98%D0%B8%D1%81%D1%83%D1%81_%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%BE%D1%81" title="Иисус Христос">Иисуса Христа</a> с предстоящими <a href="/wiki/%D0%91%D0%BE%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%B8%D1%86%D0%B0" title="Богородица">Богородицей</a> и <a href="/wiki/%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B9_%D0%A7%D1%83%D0%B4%D0%BE%D1%82%D0%B2%D0%BE%D1%80%D0%B5%D1%86" title="Николай Чудотворец">Николаем Чудотворцем</a><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>. Для старицы Наталья Алексеевна построила отдельный домик рядом со своей помещичей <a href="/wiki/%D0%A3%D1%81%D0%B0%D0%B4%D1%8C%D0%B1%D0%B0" title="Усадьба">усадьбой</a>, но юродивая Евфросиния загнала в него корову, а сама поселилась в небольшой комнате в помещениях <a href="/wiki/%D0%9F%D1%80%D0%B8%D1%81%D0%BB%D1%83%D0%B3%D0%B0" title="Прислуга">дворни</a><sup class="reference" id="cite_ref-сур_3-9"><a href="#cite_note-сур-3">[3]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:12</sup>. Как и в монастыре, с собой она поселила животных, которых выгоняла из комнаты только в день принятия ею <a href="/wiki/%D0%95%D0%B2%D1%85%D0%B0%D1%80%D0%B8%D1%81%D1%82%D0%B8%D1%8F" title="Евхаристия">причастия</a>.
</p><p><a href="/wiki/%D0%94%D1%83%D1%85%D0%BE%D0%B2%D0%BD%D0%B8%D0%BA" title="Духовник">Духовником</a> Евфросинии стал священник колюпановской Казанской церкви Павел Просперов<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>. Из села старица уезжала к своим почитателям, у которых гостила иногда длительное время. Путешествовала она чаще всего пешком, реже в <a href="/wiki/%D0%9A%D0%B8%D0%B1%D0%B8%D1%82%D0%BA%D0%B0" title="Кибитка">кибитке</a>, в которую сажала с собой кошку с собакой<sup class="reference" id="cite_ref-dombeseda_4-3"><a href="#cite_note-dombeseda-4">[4]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:20</sup>. Наибольшую привязанность Евфросиния испытывала к управляющему Мышегского чугунолитейного завода Алексею Цемшу, которого называла «сынок». Он построил для старицы в своём саду <a href="/wiki/%D0%9A%D0%B5%D0%BB%D1%8C%D1%8F" title="Келья">келью</a>, в которой она порой проживала по несколько месяцев. В этой келье стоял гроб, в который Евфросиния ложилась отдохнуть, другой мебели в ней не было<sup class="reference" id="cite_ref-сур_3-10"><a href="#cite_note-сур-3">[3]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:56</sup>.
</p>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Evfrosiniya_Kolupanovskaya_04.jpg"><img class="mw-file-element" data-file-height="709" data-file-width="992" decoding="async" height="193" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ee/Evfrosiniya_Kolupanovskaya_04.jpg/270px-Evfrosiniya_Kolupanovskaya_04.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ee/Evfrosiniya_Kolupanovskaya_04.jpg/405px-Evfrosiniya_Kolupanovskaya_04.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ee/Evfrosiniya_Kolupanovskaya_04.jpg/540px-Evfrosiniya_Kolupanovskaya_04.jpg 2x" width="270"/></a><figcaption><center>Часовня и купальня на источнике блаженной Евфросинии</center></figcaption></figure>
<p>Для уединённых молитв Евфросиния выбрала <a href="/wiki/%D0%9E%D0%B2%D1%80%D0%B0%D0%B3" title="Овраг">овраг</a> на берегу <a href="/wiki/%D0%9E%D0%BA%D0%B0" title="Ока">Оки</a> в окрестностях села. Рядом с протекающей по дну оврага речкой она вырыла <a href="/wiki/%D0%9A%D0%BE%D0%BB%D0%BE%D0%B4%D0%B5%D1%86" title="Колодец">колодец</a> и приходившим к ней больным говорила — «<i>Берите воду из моего колодезя — и будете здоровыми</i>»<sup class="reference" id="cite_ref-ПЭ_1-4"><a href="#cite_note-ПЭ-1">[1]</a></sup>.
</p><p>С жизнью Евфросинии в Колюпанове связано появление историй о её даре исцелять больных. Особо разговоры об этом стали распространяться в связи со следующими случаями:
</p>
<ul><li>исцеление помещицы Н. Протопоповой, которая страдала болезнью ног и <a href="/wiki/%D0%9A%D1%80%D0%BE%D0%B2%D0%BE%D1%85%D0%B0%D1%80%D0%BA%D0%B0%D0%BD%D1%8C%D0%B5" title="Кровохарканье">кровохарканием</a> (по указанию старицы её вымыли в <a href="/wiki/%D0%91%D0%B0%D0%BD%D1%8F" title="Баня">бане</a>, в печь которой поставили котёл с лошадиным <a href="/wiki/%D0%9D%D0%B0%D0%B2%D0%BE%D0%B7" title="Навоз">навозом</a>)<sup class="reference" id="cite_ref-сур_3-11"><a href="#cite_note-сур-3">[3]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:26</sup>;</li>
<li>исцеление помещика А. П. Полоскова, племянника Н. Протопоповой, тяжело заболевшего накануне своей свадьбы (для него Евфросиния приготовила ванну с собранными ею травами)<sup class="reference" id="cite_ref-сур_3-12"><a href="#cite_note-сур-3">[3]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:24-25</sup>;</li>
<li>помощь помещице Н. Кореловой, которая трое суток мучилась <a class="mw-redirect" href="/wiki/%D0%A0%D0%BE%D0%B4%D1%8B" title="Роды">родами</a> (Евфросиния растёрла ей бока и спину <a class="mw-redirect" href="/wiki/%D0%94%D0%B5%D1%80%D0%B5%D0%B2%D1%8F%D0%BD%D0%BD%D0%BE%D0%B5_%D0%BC%D0%B0%D1%81%D0%BB%D0%BE" title="Деревянное масло">деревянным маслом</a> и пообещала благополучные роды с первым ударом <a href="/wiki/%D0%9A%D0%BE%D0%BB%D0%BE%D0%BA%D0%BE%D0%BB" title="Колокол">колокола</a>, что исполнилось)<sup class="reference" id="cite_ref-сур_3-13"><a href="#cite_note-сур-3">[3]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:28</sup>.</li></ul>
<p>Также на основе рассказов близких к Евфросинии лиц ей приписывали дар <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%B7%D0%BE%D1%80%D0%BB%D0%B8%D0%B2%D0%BE%D1%81%D1%82%D1%8C" title="Прозорливость">прозорливости</a> (возможность видеть будущее). К наиболее часто упоминаемым историям относятся предсказание <a href="/wiki/%D0%9F%D0%BE%D1%81%D0%BB%D1%83%D1%88%D0%BD%D0%B8%D0%BA" title="Послушник">послушнице</a> Сезёновского монастыря сана игуменьи и назначения на <a href="/wiki/%D0%A2%D1%83%D0%BB%D1%8C%D1%81%D0%BA%D0%B0%D1%8F_%D0%B5%D0%BF%D0%B0%D1%80%D1%85%D0%B8%D1%8F" title="Тульская епархия">Тульскую и Белёвскую кафедру</a> епископа <a href="/wiki/%D0%94%D0%B8%D0%BC%D0%B8%D1%82%D1%80%D0%B8%D0%B9_(%D0%9C%D1%83%D1%80%D0%B5%D1%82%D0%BE%D0%B2)" title="Димитрий (Муретов)">Димитрия (Муретова)</a><sup class="reference" id="cite_ref-ПЭ_1-5"><a href="#cite_note-ПЭ-1">[1]</a></sup>. Почитатели обращались к ней «матушка Евфросиния»<sup class="reference" id="cite_ref-dombeseda_4-4"><a href="#cite_note-dombeseda-4">[4]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:20</sup>, хотя факт и обстоятельства принятия ею <a class="mw-redirect" href="/wiki/%D0%9F%D0%BE%D1%81%D1%82%D1%80%D0%B8%D0%B3" title="Постриг">монашеского пострига</a> неизвестны. Евфросиния получила такую широкую известность, что епископ Димитрий в свою первую поездку по <a href="/wiki/%D0%95%D0%BF%D0%B0%D1%80%D1%85%D0%B8%D1%8F" title="Епархия">епархии</a> заехал в село Колюпаново для встречи с ней.
</p>
<h3><span id=".D0.A1.D0.BC.D0.B5.D1.80.D1.82.D1.8C"></span><span class="mw-headline" id="Смерть">Смерть</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%95%D0%B2%D1%84%D1%80%D0%BE%D1%81%D0%B8%D0%BD%D0%B8%D1%8F_%D0%9A%D0%BE%D0%BB%D1%8E%D0%BF%D0%B0%D0%BD%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Смерть»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%95%D0%B2%D1%84%D1%80%D0%BE%D1%81%D0%B8%D0%BD%D0%B8%D1%8F_%D0%9A%D0%BE%D0%BB%D1%8E%D0%BF%D0%B0%D0%BD%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F&amp;action=edit&amp;section=5" title="Редактировать код раздела «Смерть»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Evfrosiniya_Kolupanovskaya_03.jpg"><img class="mw-file-element" data-file-height="676" data-file-width="933" decoding="async" height="196" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/e2/Evfrosiniya_Kolupanovskaya_03.jpg/270px-Evfrosiniya_Kolupanovskaya_03.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/e2/Evfrosiniya_Kolupanovskaya_03.jpg/405px-Evfrosiniya_Kolupanovskaya_03.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/e2/Evfrosiniya_Kolupanovskaya_03.jpg/540px-Evfrosiniya_Kolupanovskaya_03.jpg 2x" width="270"/></a><figcaption><center>Блаженная Евфросиния на смертном одре</center></figcaption></figure>
<p>За три недели до своей смерти Евфросиния заявила <a href="/wiki/%D0%9D%D1%8F%D0%BD%D1%8F" title="Няня">няне</a>, приставленной к ней Натальей Протопоповой, что «<i>два <a href="/wiki/%D0%90%D0%BD%D0%B3%D0%B5%D0%BB" title="Ангел">ангела</a> в белых одеждах вышли из церкви и зовут меня к себе: „Евфросиньюшка, пора тебе к нам“</i>»<sup class="reference" id="cite_ref-сур_3-14"><a href="#cite_note-сур-3">[3]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:32</sup>. Известие о том, что старица предрекла себе смерть, быстро распространилось по округе, и жители начали приходить к ней проститься и взять <a href="/wiki/%D0%91%D0%BB%D0%B0%D0%B3%D0%BE%D1%81%D0%BB%D0%BE%D0%B2%D0%B5%D0%BD%D0%B8%D0%B5" title="Благословение">благословение</a>. Многим она на память о себе дарила различные предметы — от <a href="/wiki/%D0%A0%D0%B0%D1%81%D0%BF%D1%8F%D1%82%D0%B8%D0%B5_(%D0%B4%D0%B5%D0%BA%D0%BE%D1%80%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D0%BE-%D0%BF%D1%80%D0%B8%D0%BA%D0%BB%D0%B0%D0%B4%D0%BD%D0%BE%D0%B5_%D0%B8%D1%81%D0%BA%D1%83%D1%81%D1%81%D1%82%D0%B2%D0%BE)" title="Распятие (декоративно-прикладное искусство)">крестиков</a> и <a href="/wiki/%D0%98%D0%BA%D0%BE%D0%BD%D0%B0" title="Икона">иконок</a> до травы и одежды. Помещица Протопопова обратилась с письмом к епископу Тульскому и Белёвскому Димитрию с просьбой разрешить похоронить Евфросинию внутри Казанской церкви села Колюпаново. Разрешение на это епископ дал на следующий день после смерти Евфросинии.
</p><p>Скончалась Евфросиния <abbr title="по юлианскому календарю">3</abbr> <a href="/wiki/16_%D0%B8%D1%8E%D0%BB%D1%8F" title="16 июля">(16) июля</a> <a href="/wiki/1855_%D0%B3%D0%BE%D0%B4" title="1855 год">1855</a> года. Смерть её была спокойной — она приняла <a href="/wiki/%D0%95%D0%B2%D1%85%D0%B0%D1%80%D0%B8%D1%81%D1%82%D0%B8%D1%8F" title="Евхаристия">причастие</a> и в три часа дня пополудни, сложив руки на груди, умерла. На её погребение собралось множество народу из соседних сёл. Согласно её последней воле её погребли в полном монашеском облачении. У <a href="/wiki/%D0%93%D1%80%D0%BE%D0%B1" title="Гроб">гроба</a>, края которого были обставлены множеством свечей, все дни служились <a href="/wiki/%D0%9F%D0%B0%D0%BD%D0%B8%D1%85%D0%B8%D0%B4%D0%B0" title="Панихида">панихиды</a>. 7 июля трое священников отслужили заупокойную <a class="mw-redirect" href="/wiki/%D0%9B%D0%B8%D1%82%D1%83%D1%80%D0%B3%D0%B8%D1%8F" title="Литургия">литургию</a> и затем шесть священников совершили <a href="/wiki/%D0%9E%D1%82%D0%BF%D0%B5%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5" title="Отпевание">отпевание</a> Евфросинии. Все заупокойные богослужения возглавлял духовник Евфросинии Павел Просперов. Её тело погребли под полом <a href="/wiki/%D0%A2%D1%80%D0%B0%D0%BF%D0%B5%D0%B7%D0%BD%D0%B0%D1%8F" title="Трапезная">трапезной</a> деревянной Казанской церкви.
</p>
<figure class="mw-default-size mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Evfrosiniya_Kolupanovskaya_06.jpg"><img class="mw-file-element" data-file-height="764" data-file-width="515" decoding="async" height="326" src="//upload.wikimedia.org/wikipedia/commons/thumb/2/2e/Evfrosiniya_Kolupanovskaya_06.jpg/220px-Evfrosiniya_Kolupanovskaya_06.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/2/2e/Evfrosiniya_Kolupanovskaya_06.jpg/330px-Evfrosiniya_Kolupanovskaya_06.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/2/2e/Evfrosiniya_Kolupanovskaya_06.jpg/440px-Evfrosiniya_Kolupanovskaya_06.jpg 2x" width="220"/></a><figcaption><center>Могильная плита</center></figcaption></figure><p>
По инициативе отца Павла на средства Алексея Цемша над могилой Евфросинии была установлена деревянная гробница с чугунной надгробной плитой<sup class="reference" id="cite_ref-ПЭ_1-6"><a href="#cite_note-ПЭ-1">[1]</a></sup><sup class="reference" id="cite_ref-dombeseda_4-5"><a href="#cite_note-dombeseda-4">[4]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:21</sup>. На ней с благословения митрополита Московского <a href="/wiki/%D0%A4%D0%B8%D0%BB%D0%B0%D1%80%D0%B5%D1%82_(%D0%94%D1%80%D0%BE%D0%B7%D0%B4%D0%BE%D0%B2)" title="Филарет (Дроздов)">Филарета (Дроздова)</a><sup class="reference" id="cite_ref-сур_3-15"><a href="#cite_note-сур-3">[3]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:37</sup> была сделана надпись: </p><div class="ts-Цитата"><blockquote><p>Евфросиния неведомая. Буяя мира избра Бог, да премудрыя посрамит.</p></blockquote></div>
<h2><span id=".D0.9F.D0.BE.D1.87.D0.B8.D1.82.D0.B0.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Почитание">Почитание</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%95%D0%B2%D1%84%D1%80%D0%BE%D1%81%D0%B8%D0%BD%D0%B8%D1%8F_%D0%9A%D0%BE%D0%BB%D1%8E%D0%BF%D0%B0%D0%BD%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Почитание»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%95%D0%B2%D1%84%D1%80%D0%BE%D1%81%D0%B8%D0%BD%D0%B8%D1%8F_%D0%9A%D0%BE%D0%BB%D1%8E%D0%BF%D0%B0%D0%BD%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F&amp;action=edit&amp;section=6" title="Редактировать код раздела «Почитание»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Почитание старицы, начавшееся ещё при её жизни, продолжилось после смерти. С <a href="/wiki/1884_%D0%B3%D0%BE%D0%B4" title="1884 год">1884 года</a> в храме перед каждой литургией начали совершать панихиду по Евфросинии. Стали появляться сообщения о чудесных исцелениях и помощи, приписываемых действию <a href="/wiki/%D0%9C%D0%BE%D0%BB%D0%B8%D1%82%D0%B2%D0%B0" title="Молитва">молитв</a> к старице. С <a href="/wiki/1909_%D0%B3%D0%BE%D0%B4" title="1909 год">1909 года</a> по указу Тульской духовной <a href="/wiki/%D0%94%D1%83%D1%85%D0%BE%D0%B2%D0%BD%D0%B0%D1%8F_%D0%BA%D0%BE%D0%BD%D1%81%D0%B8%D1%81%D1%82%D0%BE%D1%80%D0%B8%D1%8F" title="Духовная консистория">консистории</a> в храме завели книгу для записи подобных сообщений<sup class="reference" id="cite_ref-ПЭ_1-7"><a href="#cite_note-ПЭ-1">[1]</a></sup>. В вышедшей в <a href="/wiki/1911_%D0%B3%D0%BE%D0%B4" title="1911 год">1911 году</a> полной биографии Евфросинии собрано 55 подобных свидетельств. Каждый год в день смерти и <a href="/wiki/%D0%98%D0%BC%D0%B5%D0%BD%D0%B8%D0%BD%D1%8B" title="Именины">день ангела</a> Евфросинии в Колюпаново приезжало множество паломников, совершалось соборное служение литургии, а затем панихиды на её гробнице<sup class="reference" id="cite_ref-сур_3-16"><a href="#cite_note-сур-3">[3]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:85</sup>. В <a href="/wiki/1910_%D0%B3%D0%BE%D0%B4" title="1910 год">1910 году</a> паломничество к гробнице Евфросинии совершил епископ Каширский, <a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B0%D1%80%D0%B8%D0%B9" title="Викарий">викарий</a> Тульской епархии <a href="/wiki/%D0%95%D0%B2%D0%B4%D0%BE%D0%BA%D0%B8%D0%BC_(%D0%9C%D0%B5%D1%89%D0%B5%D1%80%D1%81%D0%BA%D0%B8%D0%B9)" title="Евдоким (Мещерский)">Евдоким (Мещерский)</a><sup class="reference" id="cite_ref-сур_3-17"><a href="#cite_note-сур-3">[3]</a></sup><sup class="reference plainlinks" style="white-space:nowrap">:86</sup>. В <a href="/wiki/1914_%D0%B3%D0%BE%D0%B4" title="1914 год">1914 году</a> над её гробницей возвели деревянную позолоченную сень.
</p><p>Почитанием пользовался и выкопанный Евфросинией источник. В <a href="/wiki/1885_%D0%B3%D0%BE%D0%B4" title="1885 год">1885 году</a> над ним установили навес — часовню на столбах, а в <a href="/wiki/1909_%D0%B3%D0%BE%D0%B4" title="1909 год">1909 году</a> — деревянную <a href="/wiki/%D0%A7%D0%B0%D1%81%D0%BE%D0%B2%D0%BD%D1%8F" title="Часовня">часовню</a>, рядом с которой устроили крытую купальню. Каждый год в <a href="/wiki/%D0%94%D0%B5%D0%BD%D1%8C_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B9_%D0%A2%D1%80%D0%BE%D0%B8%D1%86%D1%8B" title="День Святой Троицы">день Святой Троицы</a> из Казанской церкви совершали <a href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82%D0%BD%D1%8B%D0%B9_%D1%85%D0%BE%D0%B4" title="Крестный ход">крестный ход</a> к источнику.
</p><p>В <a href="/wiki/1931_%D0%B3%D0%BE%D0%B4" title="1931 год">1931 году</a> Казанская церковь сгорела, место погребения Евфросинии и её источник пришли в запустение.
</p><p>В <a href="/wiki/1988_%D0%B3%D0%BE%D0%B4" title="1988 год">1988 году</a> состоялась <a href="/wiki/%D0%9C%D0%B5%D1%81%D1%82%D0%BD%D0%BE%D1%87%D1%82%D0%B8%D0%BC%D1%8B%D0%B5_%D1%81%D0%B2%D1%8F%D1%82%D1%8B%D0%B5" title="Местночтимые святые">местная канонизация</a> Евфросинии в составе <a href="/wiki/%D0%A1%D0%BE%D0%B1%D0%BE%D1%80_%D0%A2%D1%83%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D1%85_%D1%81%D0%B2%D1%8F%D1%82%D1%8B%D1%85" title="Собор Тульских святых">Тульских святых</a> в <a href="/wiki/%D0%9B%D0%B8%D0%BA_%D1%81%D0%B2%D1%8F%D1%82%D0%BE%D1%81%D1%82%D0%B8" title="Лик святости">лике</a> <a href="/wiki/%D0%91%D0%BB%D0%B0%D0%B6%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9" title="Блаженный">блаженных</a>, совершённая по инициативе архиепископа Тульского и Белёвского <a href="/wiki/%D0%9C%D0%B0%D0%BA%D1%81%D0%B8%D0%BC_(%D0%9A%D1%80%D0%BE%D1%85%D0%B0)" title="Максим (Кроха)">Максима (Крохи)</a><sup class="reference" id="cite_ref-ПЭ_1-8"><a href="#cite_note-ПЭ-1">[1]</a></sup>. Ей были составлены <a class="mw-redirect" href="/wiki/%D0%A1%D0%BB%D1%83%D0%B6%D0%B1%D0%B0_(%D1%86%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D0%BD%D0%B0%D1%8F)" title="Служба (церковная)">служба</a>, <a href="/wiki/%D0%90%D0%BA%D0%B0%D1%84%D0%B8%D1%81%D1%82" title="Акафист">акафист</a>, <a href="/wiki/%D0%A2%D1%80%D0%BE%D0%BF%D0%B0%D1%80%D1%8C" title="Тропарь">тропарь</a>, <a href="/wiki/%D0%9A%D0%BE%D0%BD%D0%B4%D0%B0%D0%BA" title="Кондак">кондак</a>. В <a href="/wiki/1993_%D0%B3%D0%BE%D0%B4" title="1993 год">1993 году</a> началось строительство каменного храма на месте сгоревшей Казанской церкви. В <a href="/wiki/1995_%D0%B3%D0%BE%D0%B4" title="1995 год">1995 году</a> по благословению митрополита Тульского и Белевского Серапиона, указом Святейшего патриарха <a href="/wiki/%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B8%D0%B9_II" title="Алексий II">Алексия II</a> был учреждён Свято-Казанский женский <a href="/wiki/%D0%9C%D0%BE%D0%BD%D0%B0%D1%81%D1%82%D1%8B%D1%80%D1%8C" title="Монастырь">монастырь</a>, настоятельницей монастыря была назначена игуменья Евфросиния (в миру Мария Васильевна Кушнир). В <a href="/wiki/1996_%D0%B3%D0%BE%D0%B4" title="1996 год">1996 году</a> было завершено строительство нового Казанского храма. В нём над захоронением блаженной Евфросинии устроили <a href="/wiki/%D0%9C%D1%80%D0%B0%D0%BC%D0%BE%D1%80" title="Мрамор">мраморную</a> <a href="/wiki/%D0%A0%D0%B0%D0%BA%D0%B0" title="Рака">раку</a> с резной деревянной сенью<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>. Церковь украшают <a href="/wiki/%D0%A4%D1%80%D0%B5%D1%81%D0%BA%D0%B0" title="Фреска">фрески</a> на сюжеты жития Евфросинии Колюпановской<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>.
</p><p>В <a href="/wiki/2005_%D0%B3%D0%BE%D0%B4" title="2005 год">2005 году</a> в Колюпановском монастыре состоялись торжества, посвящённые 150-летию со дня смерти блаженной Евфросинии. В Казанском храме была совершена литургия, а затем состоялся <a href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82%D0%BD%D1%8B%D0%B9_%D1%85%D0%BE%D0%B4" title="Крестный ход">крестный ход</a> к источнику Евфросинии, который продолжает иметь репутацию целебного и посещается многочисленными паломниками<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>.
</p>
<h2><span id=".D0.98.D1.81.D1.82.D0.BE.D1.87.D0.BD.D0.B8.D0.BA.D0.B8"></span><span class="mw-headline" id="Источники">Источники</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%95%D0%B2%D1%84%D1%80%D0%BE%D1%81%D0%B8%D0%BD%D0%B8%D1%8F_%D0%9A%D0%BE%D0%BB%D1%8E%D0%BF%D0%B0%D0%BD%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Источники»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%95%D0%B2%D1%84%D1%80%D0%BE%D1%81%D0%B8%D0%BD%D0%B8%D1%8F_%D0%9A%D0%BE%D0%BB%D1%8E%D0%BF%D0%B0%D0%BD%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F&amp;action=edit&amp;section=7" title="Редактировать код раздела «Источники»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Первую работу о Евфросинии Колюпановской написал её духовник и настоятель Казанской церкви села Колюпанова Павел Просперов. Она была опубликована в <a href="/wiki/1862_%D0%B3%D0%BE%D0%B4" title="1862 год">1862 году</a> в журнале «Домашняя беседа». В <a href="/wiki/1903_%D0%B3%D0%BE%D0%B4" title="1903 год">1903 году</a> <a class="new" href="/w/index.php?title=%D0%A1%D1%83%D1%80%D0%B8%D0%BA%D0%BE%D0%B2,_%D0%98%D0%BE%D0%B0%D0%BD%D0%BD_%D0%9C%D0%B0%D1%82%D0%B2%D0%B5%D0%B5%D0%B2%D0%B8%D1%87&amp;action=edit&amp;redlink=1" title="Суриков, Иоанн Матвеевич (страница отсутствует)">И. М. Суриков</a> издал подробное жизнеописание старицы, основанное на воспоминаниях знавших её людей — «<i>Жизнеописание подвижницы и прозорливицы блаженной старицы Евфросинии, Христа ради юродивой княжны Вяземской, фрейлины императрицы Екатерины II</i>». В <a href="/wiki/1911_%D0%B3%D0%BE%D0%B4" title="1911 год">1911 году</a> в типографии <a href="/wiki/%D0%A2%D1%80%D0%BE%D0%B8%D1%86%D0%B5-%D0%A1%D0%B5%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B0_%D0%BB%D0%B0%D0%B2%D1%80%D0%B0" title="Троице-Сергиева лавра">Троице-Сергиевой лавры</a> вышло второе, значительно дополненное издание его труда. В <a href="/wiki/2002_%D0%B3%D0%BE%D0%B4" title="2002 год">2002 году</a> эта работа была переиздана с прибавлением дополнительных материалов (история создания Колюпановского монастыря, акафист). В этом  же жизнеописании старицы, современниками Ефросинии указано, что она родилась в 1735 году и прожила 120 лет, но фактического подтверждения этому нет<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup>.
</p><p>Небольшую статью о Евфросинии написала <a href="/wiki/%D0%A6%D0%B2%D0%B5%D1%82%D0%B0%D0%B5%D0%B2%D0%B0,_%D0%90%D0%BD%D0%B0%D1%81%D1%82%D0%B0%D1%81%D0%B8%D1%8F_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%BD%D0%B0" title="Цветаева, Анастасия Ивановна">Анастасия Цветаева</a>, которая в <a href="/wiki/%D0%A1%D0%B8%D0%B1%D0%B8%D1%80%D1%8C" title="Сибирь">сибирской</a> <a href="/wiki/%D0%A1%D1%81%D1%8B%D0%BB%D0%BA%D0%B0_(%D0%BD%D0%B0%D0%BA%D0%B0%D0%B7%D0%B0%D0%BD%D0%B8%D0%B5)" title="Ссылка (наказание)">ссылке</a> прочитала и переписала от руки жизнеописание старицы, составленное И. Суриковым, и перерисовала из него её портрет. Она описала своё выздоровление от болезни горла, которое приписала молитвам к блаженной Евфросинии. В <a href="/wiki/1959_%D0%B3%D0%BE%D0%B4" title="1959 год">1959 году</a> Анастасия посетила источник и могилу старицы и стала бывать в Колюпанове ежегодно<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>.
</p>
