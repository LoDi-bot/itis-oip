<p><b>«Электропрохлади́тельный кисло́тный тест»</b> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">The Electric Kool-Aid Acid Test</span>, 1968, рус. перевод 1996) — книга американского писателя <a href="/wiki/%D0%92%D1%83%D0%BB%D1%84,_%D0%A2%D0%BE%D0%BC_(%D0%B6%D1%83%D1%80%D0%BD%D0%B0%D0%BB%D0%B8%D1%81%D1%82)" title="Вулф, Том (журналист)">Тома Вулфа</a>, описывающая период жизни <a href="/wiki/%D0%9A%D0%B8%D0%B7%D0%B8,_%D0%9A%D0%B5%D0%BD" title="Кизи, Кен">Кена Кизи</a> (автора «<a href="/wiki/%D0%9F%D1%80%D0%BE%D0%BB%D0%B5%D1%82%D0%B0%D1%8F_%D0%BD%D0%B0%D0%B4_%D0%B3%D0%BD%D0%B5%D0%B7%D0%B4%D0%BE%D0%BC_%D0%BA%D1%83%D0%BA%D1%83%D1%88%D0%BA%D0%B8" title="Пролетая над гнездом кукушки">Пролетая над гнездом кукушки</a>») с 1958 по 1966 год и сформировавшуюся вокруг него неформальную <a href="/wiki/%D0%A1%D1%83%D0%B1%D0%BA%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D0%B0" title="Субкультура">субкультурную</a> коммуну — <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8" title="Весёлые проказники">Весёлых проказников</a> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Merry Pranksters</span>), оказавшую <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Популяризация_ЛСД" title="Весёлые проказники">существенное влияние</a> на популяризацию <a href="/wiki/%D0%9B%D0%A1%D0%94" title="ЛСД">ЛСД</a>, <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Психоделическая_революция" title="Весёлые проказники">спровоцировавшую</a> психоделическую (также называемую <i>кислотной</i>) революцию и ставшую <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Связующее_звено_между_битниками_и_хиппи" title="Весёлые проказники">связующим звеном</a> между <a href="/wiki/%D0%91%D0%B8%D1%82%D0%BD%D0%B8%D0%BA%D0%B8" title="Битники">битниками</a> и <a href="/wiki/%D0%A5%D0%B8%D0%BF%D0%BF%D0%B8" title="Хиппи">хиппи</a>.
</p><p>Книга вошла в список<sup class="reference" id="cite_ref-autogenerated1_1-0"><a href="#cite_note-autogenerated1-1">[1]</a></sup> 100 лучших работ по журналистике в <a href="/wiki/%D0%A1%D0%BE%D0%B5%D0%B4%D0%B8%D0%BD%D1%91%D0%BD%D0%BD%D1%8B%D0%B5_%D0%A8%D1%82%D0%B0%D1%82%D1%8B_%D0%90%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B8" title="Соединённые Штаты Америки">США</a> в XX веке по версии Arthur L. Carter Journalism Institute при <a href="/wiki/%D0%9D%D1%8C%D1%8E-%D0%99%D0%BE%D1%80%D0%BA%D1%81%D0%BA%D0%B8%D0%B9_%D1%83%D0%BD%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D1%82%D0%B5%D1%82" title="Нью-Йоркский университет">Нью-Йоркском университете</a>.

<h2><span id=".D0.AD.D1.82.D0.B8.D0.BC.D0.BE.D0.BB.D0.BE.D0.B3.D0.B8.D1.8F_.D0.BD.D0.B0.D0.B7.D0.B2.D0.B0.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="Этимология_названия">Этимология названия</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BF%D1%80%D0%BE%D1%85%D0%BB%D0%B0%D0%B4%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B8%D1%81%D0%BB%D0%BE%D1%82%D0%BD%D1%8B%D0%B9_%D1%82%D0%B5%D1%81%D1%82&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Этимология названия»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BF%D1%80%D0%BE%D1%85%D0%BB%D0%B0%D0%B4%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B8%D1%81%D0%BB%D0%BE%D1%82%D0%BD%D1%8B%D0%B9_%D1%82%D0%B5%D1%81%D1%82&amp;action=edit&amp;section=1" title="Редактировать код раздела «Этимология названия»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Неологизм «электропрохладительный» относится к периоду упадка коммуны и напрямую связан с <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Уоттсовский_тест" title="Весёлые проказники">Уоттсовским тестом</a>, вечеринкой, которую <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8" title="Весёлые проказники">Весёлые проказники</a> провели в пригороде <a href="/wiki/%D0%9B%D0%BE%D1%81-%D0%90%D0%BD%D0%B4%D0%B6%D0%B5%D0%BB%D0%B5%D1%81" title="Лос-Анджелес">Лос-Анджелеса</a>. В оригинале английский термин «<span lang="en" style="font-style:italic;">The Electric Kool-Aid Drink</span>» («электропрохладительный напиток») был предложен<sup class="reference" id="cite_ref-Wolfe_2-0"><a href="#cite_note-Wolfe-2">[2]</a></sup> членом коммуны, <span data-interwiki-article="Wavy Gravy" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%A0%D0%BE%D0%BC%D0%BD%D0%B8,_%D0%A5%D1%8C%D1%8E&amp;action=edit&amp;redlink=1" title="Ромни, Хью (страница отсутствует)">Хью Ромни</a></span> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;"><a class="extiw" href="https://en.wikipedia.org/wiki/Wavy_Gravy" title="en:Wavy Gravy"><span title="Wavy Gravy — версия статьи «Ромни, Хью» на английском языке">Wavy Gravy</span></a></span>), во время вышеупомянутого события. Практика бесплатного угощения прохладительными напитками на вечеринках была распространённым явлением для описываемого в книге времени, но качественно напиток Проказников отличался наличием в нём изрядной доли <a href="/wiki/%D0%9B%D0%A1%D0%94" title="ЛСД">ЛСД</a>. Соответственно, добавленное к слову «Электропрохладительный» выражение «<a href="/wiki/%D0%9A%D0%B8%D1%81%D0%BB%D0%BE%D1%82%D0%BD%D1%8B%D0%B5_%D1%82%D0%B5%D1%81%D1%82%D1%8B" title="Кислотные тесты">кислотный тест</a>» характеризует основную деятельность коммуны Весёлых проказников — проведение вечеринок<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> с угощением <a href="/wiki/%D0%9B%D0%A1%D0%94" title="ЛСД">ЛСД</a> всех желающих.
</p>
<h2><span id=".D0.98.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.8F_.D1.81.D0.BE.D0.B7.D0.B4.D0.B0.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="История_создания">История создания</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BF%D1%80%D0%BE%D1%85%D0%BB%D0%B0%D0%B4%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B8%D1%81%D0%BB%D0%BE%D1%82%D0%BD%D1%8B%D0%B9_%D1%82%D0%B5%D1%81%D1%82&amp;veaction=edit&amp;section=2" title="Редактировать раздел «История создания»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BF%D1%80%D0%BE%D1%85%D0%BB%D0%B0%D0%B4%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B8%D1%81%D0%BB%D0%BE%D1%82%D0%BD%D1%8B%D0%B9_%D1%82%D0%B5%D1%81%D1%82&amp;action=edit&amp;section=2" title="Редактировать код раздела «История создания»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Wolfe,_Garcia,_Scully.jpg"><img class="mw-file-element" data-file-height="480" data-file-width="322" decoding="async" height="253" src="//upload.wikimedia.org/wikipedia/ru/thumb/a/ac/Wolfe%2C_Garcia%2C_Scully.jpg/170px-Wolfe%2C_Garcia%2C_Scully.jpg" srcset="//upload.wikimedia.org/wikipedia/ru/thumb/a/ac/Wolfe%2C_Garcia%2C_Scully.jpg/255px-Wolfe%2C_Garcia%2C_Scully.jpg 1.5x, //upload.wikimedia.org/wikipedia/ru/a/ac/Wolfe%2C_Garcia%2C_Scully.jpg 2x" width="170"/></a><figcaption><a href="/wiki/%D0%93%D0%B0%D1%80%D1%81%D0%B8%D1%8F,_%D0%94%D0%B6%D0%B5%D1%80%D1%80%D0%B8" title="Гарсия, Джерри">Джерри Гарсия</a>, <span data-interwiki-article="Rock Scully" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%A1%D0%BA%D0%B0%D0%BB%D0%BB%D0%B8,_%D0%A0%D0%BE%D0%BA&amp;action=edit&amp;redlink=1" title="Скалли, Рок (страница отсутствует)">Рок Скалли</a></span> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;"><a class="extiw" href="https://en.wikipedia.org/wiki/Rock_Scully" title="en:Rock Scully"><span title="Rock Scully — версия статьи «Скалли, Рок» на английском языке">Rock Scully</span></a></span>) и Том Вулф во время работы над книгой.</figcaption></figure>
<p>Как отмечает<sup class="reference" id="cite_ref-Wolfe_2-1"><a href="#cite_note-Wolfe-2">[2]</a></sup> сам Вулф, в некоторых из описываемых в книге событий он принимал непосредственное участие, а об остальных узнал от свидетелей или участников происходившего. В работе над книгой помогали Кен Кизи, Каролин Адамс, «Черная Мария», Кеннет Баббс, Стюарт Бранд, Пейдж Браунинг, Майк Хейджен, Хью Ромни, Джордж Уокер, Нил Кэссади, Сэнди Леманн-Хаупт, Эд Маккланахэн, Роберт Стоун — то есть непосредственно <a class="mw-selflink-fragment" href="#Персонажи">члены коммуны</a> <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8" title="Весёлые проказники">Весёлых проказников</a>, которых Вулф <a href="/wiki/%D0%98%D0%BD%D1%82%D0%B5%D1%80%D0%B2%D1%8C%D1%8E" title="Интервью">интервьюировал</a> в ходе работы над «Электропрохладительным кислотным тестом».
</p><p>Также в ходе написания книги были использованы:
</p>
<ul><li>Письма Кизи к <a class="mw-redirect" href="/wiki/%D0%9C%D0%B0%D0%BA%D0%BC%D0%B5%D1%80%D1%82%D1%80%D0%B8,_%D0%9B%D0%B0%D1%80%D1%80%D0%B8_%D0%94%D0%B6%D0%B5%D1%84%D1%84" title="Макмертри, Ларри Джефф">Ларри Макмертри</a>. Они относятся к <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Мексика" title="Весёлые проказники">периоду</a>, когда главный Проказник был в бегах в <a href="/wiki/%D0%9C%D0%B5%D0%BA%D1%81%D0%B8%D0%BA%D0%B0" title="Мексика">Мексике</a>.</li>
<li>Архивы коммуны — магнитофонные записи с кинохроникой <a href="/wiki/%D0%94%D0%B0%D0%BB%D1%88%D0%B5" title="Далше">автобусного путешествия</a>, дневники, письма, фотографии.</li>
<li>Магнитные записи, которые <a href="/wiki/%D0%A2%D0%BE%D0%BC%D0%BF%D1%81%D0%BE%D0%BD,_%D0%A5%D0%B0%D0%BD%D1%82%D0%B5%D1%80_%D0%A1%D1%82%D0%BE%D0%BA%D1%82%D0%BE%D0%BD" title="Томпсон, Хантер Стоктон">Хантер Томпсон</a> сделал в период работы над своей книгой «<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%B5%D0%BB%D1%8B_%D0%90%D0%B4%D0%B0_(%D1%80%D0%BE%D0%BC%D0%B0%D0%BD)" title="Ангелы Ада (роман)">Ангелы Ада</a>»</li>
<li>Отчет о <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Уоттсовский_тест" title="Весёлые проказники">Уоттсовском Тесте</a>, написанный репортером <span data-interwiki-article="Los Angeles Free Press" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=Los_Angeles_Free_Press&amp;action=edit&amp;redlink=1" title="Los Angeles Free Press (страница отсутствует)">Free Press</a></span> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;"><a class="extiw" href="https://en.wikipedia.org/wiki/Los_Angeles_Free_Press" title="en:Los Angeles Free Press"><span title="Los Angeles Free Press — версия статьи «Los Angeles Free Press» на английском языке">Los Angeles Free Press</span></a></span>) Клер Браш, участвовавшей в хеппенинге — специально для Вулфа.</li>
<li>Впечатления о Проказниках драматурга Нормана Хартвега (Norman Hartweg), описанные в серии магнитофонных записей.</li>
<li>Множество писем, которые автор книги получал в ходе переписки с людьми<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>, тем или иным образом причастными к жизни коммуны.</li></ul>
<p>По собственному же признанию, <a href="/wiki/%D0%92%D1%83%D0%BB%D1%84,_%D0%A2%D0%BE%D0%BC_(%D0%B6%D1%83%D1%80%D0%BD%D0%B0%D0%BB%D0%B8%D1%81%D1%82)" title="Вулф, Том (журналист)">Том Вулф</a> никогда не употреблял <a href="/wiki/%D0%9B%D0%A1%D0%94" title="ЛСД">ЛСД</a>, только однажды курил <a href="/wiki/%D0%9C%D0%B0%D1%80%D0%B8%D1%85%D1%83%D0%B0%D0%BD%D0%B0" title="Марихуана">марихуану</a><sup class="reference" id="cite_ref-Kesey_5-0"><a href="#cite_note-Kesey-5">[5]</a></sup>.
</p>
<h2><span id=".D0.A1.D0.BE.D0.B4.D0.B5.D1.80.D0.B6.D0.B0.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Содержание">Содержание</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BF%D1%80%D0%BE%D1%85%D0%BB%D0%B0%D0%B4%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B8%D1%81%D0%BB%D0%BE%D1%82%D0%BD%D1%8B%D0%B9_%D1%82%D0%B5%D1%81%D1%82&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Содержание»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BF%D1%80%D0%BE%D1%85%D0%BB%D0%B0%D0%B4%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B8%D1%81%D0%BB%D0%BE%D1%82%D0%BD%D1%8B%D0%B9_%D1%82%D0%B5%D1%81%D1%82&amp;action=edit&amp;section=3" title="Редактировать код раздела «Содержание»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Книга состоит из двадцати семи глав, эпилога и финального «послесловия», где <a href="/wiki/%D0%92%D1%83%D0%BB%D1%84,_%D0%A2%D0%BE%D0%BC_(%D0%B6%D1%83%D1%80%D0%BD%D0%B0%D0%BB%D0%B8%D1%81%D1%82)" title="Вулф, Том (журналист)">Т. Вулф</a> перечисляет всех, кто помогал ему в создании «Электропрохладительного кислотного теста» и выражает слова благодарности. В самом тексте книги содержательно можно выделить три части, соответствующие различным периодам существования коммуны <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8" title="Весёлые проказники">Весёлых проказников</a>, хотя сам автор такового деления не предусматривает.
</p><p>Стоящие особняком<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> от остальных глав, главы I, II и III рассказывают о знакомстве Вулфа с Проказниками и <a href="/wiki/%D0%9A%D0%B8%D0%B7%D0%B8,_%D0%9A%D0%B5%D0%BD" title="Кизи, Кен">Кеном Кизи</a> — данные главы написаны в стилистике, выработанной в «<a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%84%D0%B5%D1%82%D0%BD%D0%BE%D1%80%D0%B0%D1%81%D0%BA%D1%80%D0%B0%D1%88%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B0%D0%BF%D0%B5%D0%BB%D1%8C%D1%81%D0%B8%D0%BD%D0%BD%D0%BE%D0%BB%D0%B5%D0%BF%D0%B5%D1%81%D1%82%D0%BA%D0%BE%D0%B2%D0%B0%D1%8F_%D0%BE%D0%B1%D1%82%D0%B5%D0%BA%D0%B0%D0%B5%D0%BC%D0%B0%D1%8F_%D0%BC%D0%B0%D0%BB%D1%8E%D1%82%D0%BA%D0%B0#Стиль_изложения" title="Конфетнораскрашенная апельсиннолепестковая обтекаемая малютка">Конфетнораскрашенной апельсиннолепестковой обтекаемой малютке</a>», и повествуют о пребывании автора с членами коммуны во время судебного процесса над Кизи и о подготовке к прощальной вечеринке Проказников — «<a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Выпускной_бал._Окончание_Кислотного_Теста" title="Весёлые проказники">Выпускному балу</a>». Описывая свои взаимоотношения с главными действующими лицами книги, Вулф планомерно переносит местами несвязное повествование к хронологическому развитию событий — и «возвращается» назад, в 1958 год, когда всё только начиналось.
</p><p>Главы с IV по VI описывают период <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Предпосылки_к_формированию_коммуны" title="Весёлые проказники">становления</a> коммуны и охватывают период с 1958 по 1964 год. В данных главах рассказывается о переезде <a href="/wiki/%D0%9A%D0%B8%D0%B7%D0%B8,_%D0%9A%D0%B5%D0%BD" title="Кизи, Кен">Кена Кизи</a> в фешенебельный район вблизи <a href="/wiki/%D0%A1%D1%82%D1%8D%D0%BD%D1%84%D0%BE%D1%80%D0%B4%D1%81%D0%BA%D0%B8%D0%B9_%D1%83%D0%BD%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D1%82%D0%B5%D1%82" title="Стэнфордский университет">Стэнфордского университета</a>, Перри-лейн, в связи с поступлением на курсы писательского мастерства. В данной части книги описываются первые опыты с <a class="mw-redirect" href="/wiki/%D0%9F%D1%81%D0%B8%D1%85%D0%BE%D0%B0%D0%BA%D1%82%D0%B8%D0%B2%D0%BD%D1%8B%D0%B5_%D0%B2%D0%B5%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%B0" title="Психоактивные вещества">психотропными веществами</a>, которые вследствие своей новизны и неизученности становятся «лакомым кусочком» для жителей студенческого городка. Растущая популярность некоторых из них, в частности <a href="/wiki/%D0%9B%D0%A1%D0%94" title="ЛСД">ЛСД</a>, вкупе с харизматичностью Кизи, создает вокруг последнего сплоченную компанию единомышленников, приобщившихся к «опыту нового восприятия» — восприятию мира при помощи расширяющего сознание наркотика. Логически данная часть книги заканчивается с вынужденным переездом Кизи с Перри-лейн в <a href="/wiki/%D0%9B%D0%B0-%D0%A5%D0%BE%D0%BD%D0%B4%D0%B0" title="Ла-Хонда">Ла-Хонду</a>, штат <a href="/wiki/%D0%9A%D0%B0%D0%BB%D0%B8%D1%84%D0%BE%D1%80%D0%BD%D0%B8%D1%8F" title="Калифорния">Калифорния</a>, где он покупает дом. Туда же впоследствии переезжает и вся компания, образовавшаяся вокруг молодого писателя.
</p><p><a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Период_расцвета_коммуны" title="Весёлые проказники">Расцвет</a> коммуны, которая получает название «Весёлые проказники», золотой её период, описан с VI по XX главу и охватывает период с 1964 по 1966 год. Данный раздел уместно выделить в связи с примечательным для Проказников событием — автобусным путешествием «<a href="/wiki/%D0%94%D0%B0%D0%BB%D1%88%D0%B5" title="Далше">Далше</a>»<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>, которое отсеивает людей, по тем или иным причинам не принявшим пропагандируемые Кизи <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Мировоззренческие_идеи" title="Весёлые проказники">идеи</a>, и оставляет только преданных своему учителю и духовному лидеру друзей. Лето 1964 года, когда Проказники приехали обратно в Ла-Хонду, окончательно сформировало коммуну. Помимо автобусного путешествия, главы в данном разделе книги повествуют о других, менее значимых, но тем не менее примечательных событиях в жизни коммуны — первые предъявленные её членам <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Полицейский_налет" title="Весёлые проказники">обвинения</a> со стороны полиции, <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Ангелы_Ада" title="Весёлые проказники">знакомство</a> с «<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%B5%D0%BB%D1%8B_%D0%B0%D0%B4%D0%B0" title="Ангелы ада">Ангелами ада</a>», <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Конференция_унитарианской_церкви" title="Весёлые проказники">выступление</a> Кизи на конференции <a class="mw-redirect" href="/wiki/%D0%A3%D0%BD%D0%B8%D1%82%D0%B0%D1%80%D0%B8%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D1%86%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C" title="Унитарианская церковь">Унитарианской церкви</a>, <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Аузли" title="Весёлые проказники">присоединение</a> к коммуне «ЛСДшного гуру» <span data-interwiki-article="Owsley Stanley" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%90%D1%83%D0%B3%D1%83%D1%81%D1%82%D1%83%D1%81,_%D0%A1%D1%82%D1%8D%D0%BD%D0%BB%D0%B8&amp;action=edit&amp;redlink=1" title="Аугустус, Стэнли (страница отсутствует)">Аузли</a></span> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;"><a class="extiw" href="https://en.wikipedia.org/wiki/Owsley_Stanley" title="en:Owsley Stanley"><span title="Owsley Stanley — версия статьи «Аугустус, Стэнли» на английском языке">Owsley Stanley</span></a></span>), <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Антивоенный_митинг_в_Беркли" title="Весёлые проказники">участие</a> Проказников в митинге против <a href="/wiki/%D0%92%D0%BE%D0%B9%D0%BD%D0%B0_%D0%B2%D0%BE_%D0%92%D1%8C%D0%B5%D1%82%D0%BD%D0%B0%D0%BC%D0%B5" title="Война во Вьетнаме">войны во Вьетнаме</a>, проведение <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Кислотные_Тесты" title="Весёлые проказники">Кислотных Тестов</a> и <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Фестиваль_Полетов" title="Весёлые проказники">Фестиваля Полетов</a>. Глава XX закрывает данную часть книги, описывая вынужденный <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Побег_в_Мексику" title="Весёлые проказники">побег</a> опального на тот момент Кизи в <a href="/wiki/%D0%9C%D0%B5%D0%BA%D1%81%D0%B8%D0%BA%D0%B0" title="Мексика">Мексику</a>. С его отъездом золотой период для коммуны завершается.
</p><p><a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Период_упадка_коммуны" title="Весёлые проказники">Распад</a> коммуны описан с XX по XXVII главы и охватывает период с февраля по октябрь 1966 года. Главы данной части включают в себя <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Уоттсовский_тест" title="Весёлые проказники">описание</a> Уоттсовского теста — вечеринки, сыгравшей ключевую роль в вопросе распада коммуны, <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Мексика" title="Весёлые проказники">хроники</a> Проказников во время пребывания их лидера в Мексике, становление <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Выход_за_Пределы_Кислоты" title="Весёлые проказники">идей</a> о прекращении употребления ЛСД, <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Арест" title="Весёлые проказники">арест</a> Кизи по трем предъявленным обвинениям со стороны полиции и ФБР и финальный, <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8#Выпускной_бал._Окончание_Кислотного_Теста" title="Весёлые проказники">прощальный хэппенинг</a>, после которого коммуна прекратила своё существование.
</p>
<h2><span id=".D0.9F.D0.B5.D1.80.D1.81.D0.BE.D0.BD.D0.B0.D0.B6.D0.B8"></span><span class="mw-headline" id="Персонажи">Персонажи</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BF%D1%80%D0%BE%D1%85%D0%BB%D0%B0%D0%B4%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B8%D1%81%D0%BB%D0%BE%D1%82%D0%BD%D1%8B%D0%B9_%D1%82%D0%B5%D1%81%D1%82&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Персонажи»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BF%D1%80%D0%BE%D1%85%D0%BB%D0%B0%D0%B4%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B8%D1%81%D0%BB%D0%BE%D1%82%D0%BD%D1%8B%D0%B9_%D1%82%D0%B5%D1%81%D1%82&amp;action=edit&amp;section=4" title="Редактировать код раздела «Персонажи»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Из-за жанровых <a href="#Жанровые_особенности">особенностей</a> книги, практически каждый упомянутый в тексте человек в определённой мере является её персонажем — из-за низкой информативности и большого объёма полного списка персонажей «Электропрохладительного кислотного теста», ниже указаны только самые примечательные и значимые (с точки зрения сюжета) из них.
</p>
<ul><li><a href="/wiki/%D0%9A%D0%B8%D0%B7%D0%B8,_%D0%9A%D0%B5%D0%BD" title="Кизи, Кен">Кен Кизи</a> — американский писатель (автор романов «<a href="/wiki/%D0%9F%D1%80%D0%BE%D0%BB%D0%B5%D1%82%D0%B0%D1%8F_%D0%BD%D0%B0%D0%B4_%D0%B3%D0%BD%D0%B5%D0%B7%D0%B4%D0%BE%D0%BC_%D0%BA%D1%83%D0%BA%D1%83%D1%88%D0%BA%D0%B8" title="Пролетая над гнездом кукушки">Пролетая над гнездом кукушки</a>», 1962, «<a href="/wiki/%D0%9F%D0%BE%D1%80%D0%BE%D1%8E_%D0%B1%D0%BB%D0%B0%D0%B6%D1%8C_%D0%B2%D0%B5%D0%BB%D0%B8%D0%BA%D0%B0%D1%8F" title="Порою блажь великая">Порою блажь великая</a>», 1964 и «<a href="/wiki/%D0%9F%D0%B5%D1%81%D0%BD%D1%8C_%D0%BC%D0%BE%D1%80%D1%8F%D0%BA%D0%B0" title="Песнь моряка">Песнь моряка</a>», 1992), основатель коммуны, духовный лидер и учитель её членов.</li>
<li><a href="/wiki/%D0%9A%D1%8D%D1%81%D1%81%D0%B8%D0%B4%D0%B8,_%D0%9D%D0%B8%D0%BB" title="Кэссиди, Нил">Нил Кэссади</a> — одна из самых примечательных фигур <a href="/wiki/%D0%91%D0%B8%D1%82-%D0%BF%D0%BE%D0%BA%D0%BE%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5" title="Бит-поколение">бит-поколения</a>, <a href="/wiki/%D0%9F%D1%80%D0%BE%D1%82%D0%BE%D1%82%D0%B8%D0%BF_%D0%BF%D0%B5%D1%80%D1%81%D0%BE%D0%BD%D0%B0%D0%B6%D0%B0" title="Прототип персонажа">прототип</a> Дина Мориарти из романа <a href="/wiki/%D0%9A%D0%B5%D1%80%D1%83%D0%B0%D0%BA,_%D0%94%D0%B6%D0%B5%D0%BA" title="Керуак, Джек">Джека Керуака</a> «<a class="mw-redirect" href="/wiki/%D0%92_%D0%B4%D0%BE%D1%80%D0%BE%D0%B3%D0%B5_(%D1%80%D0%BE%D0%BC%D0%B0%D0%BD)" title="В дороге (роман)">В дороге</a>». Близкий друг Кизи, был водителем автобуса «<a href="/wiki/%D0%94%D0%B0%D0%BB%D1%88%D0%B5" title="Далше">Далше</a>».</li>
<li><span data-interwiki-article="Ken Babbs" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%91%D0%B0%D0%B1%D0%B1%D1%81,_%D0%9A%D0%B5%D0%BD%D0%BD%D0%B5%D1%82&amp;action=edit&amp;redlink=1" title="Баббс, Кеннет (страница отсутствует)">Кеннет Баббс</a></span> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;"><a class="extiw" href="https://en.wikipedia.org/wiki/Ken_Babbs" title="en:Ken Babbs"><span title="Ken Babbs — версия статьи «Баббс, Кеннет» на английском языке">Ken Babbs</span></a></span>) — близкий друг Кизи, писатель, преемник главенства в коммуне в период её распада.</li>
<li><a href="/wiki/%D0%93%D0%B0%D1%80%D1%81%D0%B8%D1%8F,_%D0%94%D0%B6%D0%B5%D1%80%D1%80%D0%B8" title="Гарсия, Джерри">Джерри Гарсия</a> и оригинальный (в период с 1965 по 1967 год) состав группы <a href="/wiki/Grateful_Dead" title="Grateful Dead">Grateful Dead</a>.</li>
<li><span data-interwiki-article="Carolyn Adams" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=%D0%90%D0%B4%D0%B0%D0%BC%D1%81,_%D0%9A%D1%8D%D1%80%D0%BE%D0%BB%D0%B8%D0%BD&amp;action=edit&amp;redlink=1" title="Адамс, Кэролин (страница отсутствует)">Каролин Адамс</a></span> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;"><a class="extiw" href="https://en.wikipedia.org/wiki/Carolyn_Adams" title="en:Carolyn Adams"><span title="Carolyn Adams — версия статьи «Адамс, Кэролин» на английском языке">Carolyn Adams</span></a></span>) — известная среди Проказников как «Горянка». Любовница Кизи, мать<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> его дочери Саншайн (<span lang="en" style="font-style:italic;">Sunshine</span>). Присоединилась к коммуне летом 1964 года по прибытии «<a href="/wiki/%D0%94%D0%B0%D0%BB%D1%88%D0%B5" title="Далше">Далше</a>» в <a href="/wiki/%D0%9B%D0%B0-%D0%A5%D0%BE%D0%BD%D0%B4%D0%B0" title="Ла-Хонда">Ла-Хонду</a>. Впоследствии вышла замуж за Джерри Гарсиа.</li>

</p>
<ul><li><a href="/wiki/%D0%93%D0%B8%D0%BD%D0%B7%D0%B1%D0%B5%D1%80%D0%B3,_%D0%90%D0%BB%D0%BB%D0%B5%D0%BD" title="Гинзберг, Аллен">Аллен Гинзберг</a> — поэт, одна из самых видных фигур <a href="/wiki/%D0%91%D0%B8%D1%82-%D0%BF%D0%BE%D0%BA%D0%BE%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5" title="Бит-поколение">бит-поколения</a>. Был лично знаком с Кизи и некоторыми Проказниками — однако из-за расхождений во взглядах<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> по вопросу употребления и популяризации наркотика, с последними был не в лучших отношениях.</li>
<li><a href="/wiki/%D0%9A%D0%B5%D1%80%D1%83%D0%B0%D0%BA,_%D0%94%D0%B6%D0%B5%D0%BA" title="Керуак, Джек">Джек Керуак</a> — писатель <a href="/wiki/%D0%91%D0%B8%D1%82-%D0%BF%D0%BE%D0%BA%D0%BE%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5" title="Бит-поколение">бит-поколения</a>. Не нашёл общего языка с Кизи и не испытал восторга от знакомства с ним — в хороших отношениях с членами коммуны не был, пропагандируемые ими идеи отвергал.</li>
<li><a href="/wiki/%D0%A2%D0%BE%D0%BC%D0%BF%D1%81%D0%BE%D0%BD,_%D0%A5%D0%B0%D0%BD%D1%82%D0%B5%D1%80_%D0%A1%D1%82%D0%BE%D0%BA%D1%82%D0%BE%D0%BD" title="Томпсон, Хантер Стоктон">Хантер Томпсон</a> — американский писатель и журналист. Познакомил Кизи с Ангелами Ада.</li></ul>
<h2><span id=".D0.96.D0.B0.D0.BD.D1.80.D0.BE.D0.B2.D1.8B.D0.B5_.D0.BE.D1.81.D0.BE.D0.B1.D0.B5.D0.BD.D0.BD.D0.BE.D1.81.D1.82.D0.B8"></span><span class="mw-headline" id="Жанровые_особенности">Жанровые особенности</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BF%D1%80%D0%BE%D1%85%D0%BB%D0%B0%D0%B4%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B8%D1%81%D0%BB%D0%BE%D1%82%D0%BD%D1%8B%D0%B9_%D1%82%D0%B5%D1%81%D1%82&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Жанровые особенности»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BF%D1%80%D0%BE%D1%85%D0%BB%D0%B0%D0%B4%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B8%D1%81%D0%BB%D0%BE%D1%82%D0%BD%D1%8B%D0%B9_%D1%82%D0%B5%D1%81%D1%82&amp;action=edit&amp;section=5" title="Редактировать код раздела «Жанровые особенности»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>В подавляющем большинстве случаев читатели и критики называют «Электропрохладительный кислотный тест» <a href="/wiki/%D0%A0%D0%BE%D0%BC%D0%B0%D0%BD" title="Роман">романом</a>, что является корректным по признаку содержания в книге развернутого повествования о жизни главного героя (<a href="/wiki/%D0%9A%D0%B8%D0%B7%D0%B8,_%D0%9A%D0%B5%D0%BD" title="Кизи, Кен">Кена Кизи</a>) в кризисный период его жизни (увлечение <a class="mw-redirect" href="/wiki/%D0%9F%D1%81%D0%B8%D1%85%D0%BE%D0%B0%D0%BA%D1%82%D0%B8%D0%B2%D0%BD%D1%8B%D0%B5_%D0%B2%D0%B5%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%B0" title="Психоактивные вещества">психоактивными веществами</a>, проблемы с полицией и ФБР), однако по ряду иных (характерных для других литературных жанров) признаков книгу в равной степени можно отнести как к <a href="/wiki/%D0%94%D0%BE%D0%BA%D1%83%D0%BC%D0%B5%D0%BD%D1%82%D0%B0%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D0%BF%D1%80%D0%BE%D0%B7%D0%B0" title="Документальная проза">документальной прозе</a> — по признаку построения сюжетной линии на основе реально происходивших событий, так и к <a href="/wiki/%D0%9D%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B6%D1%83%D1%80%D0%BD%D0%B0%D0%BB%D0%B8%D1%81%D1%82%D0%B8%D0%BA%D0%B0" title="Новая журналистика">новой журналистике</a> на основании наличия в книге подбора фактов, оценок и комментариев участников событий, которые являлись злободневными и значительными в момент издания книги.
</p><p>На официальном сайте Тома Вулфа в краткой аннотации к книге дано<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> определение «журналистская одиссея»<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>.
</p>
<h2><span id=".D0.9E.D1.82.D0.B7.D1.8B.D0.B2.D1.8B_.D0.B8_.D0.BA.D1.80.D0.B8.D1.82.D0.B8.D0.BA.D0.B0"></span><span class="mw-headline" id="Отзывы_и_критика">Отзывы и критика</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BF%D1%80%D0%BE%D1%85%D0%BB%D0%B0%D0%B4%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B8%D1%81%D0%BB%D0%BE%D1%82%D0%BD%D1%8B%D0%B9_%D1%82%D0%B5%D1%81%D1%82&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Отзывы и критика»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BF%D1%80%D0%BE%D1%85%D0%BB%D0%B0%D0%B4%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B8%D1%81%D0%BB%D0%BE%D1%82%D0%BD%D1%8B%D0%B9_%D1%82%D0%B5%D1%81%D1%82&amp;action=edit&amp;section=6" title="Редактировать код раздела «Отзывы и критика»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Пропагандирующий революционные идеи в нескольких литературных жанрах, триумф<sup class="reference" id="cite_ref-Mills_13-0"><a href="#cite_note-Mills-13">[13]</a></sup> <a href="/wiki/%D0%9D%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B6%D1%83%D1%80%D0%BD%D0%B0%D0%BB%D0%B8%D1%81%D1%82%D0%B8%D0%BA%D0%B0" title="Новая журналистика">Новой журналистики</a>, «Электропрохладительный кислотный тест» практически сразу стал<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup> книгой, характеризующей <a href="/wiki/%D0%A1%D0%BE%D0%B5%D0%B4%D0%B8%D0%BD%D1%91%D0%BD%D0%BD%D1%8B%D0%B5_%D0%A8%D1%82%D0%B0%D1%82%D1%8B_%D0%90%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B8" title="Соединённые Штаты Америки">Америку</a> 1960-х годов и подняла<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup> статус <a href="/wiki/%D0%9A%D0%B8%D0%B7%D0%B8,_%D0%9A%D0%B5%D0%BD" title="Кизи, Кен">Кизи</a> до уровня <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%82%D1%80%D0%BA%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D0%B0" title="Контркультура">контркультурного</a> героя и <a href="/wiki/%D0%9B%D0%A1%D0%94" title="ЛСД">ЛСД</a>-гуру.
</p><p>Книга была высоко оценена журналистом <a href="/wiki/The_Washington_Post" title="The Washington Post">The Washington Post</a>, окрестившим её «сияющей, озорной и нереальной»<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup> — талант же самого Вулфа был отмечен<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup> изданием <a href="/wiki/Newsweek" title="Newsweek">Newsweek</a>.
</p><p>Известный американский <a href="/wiki/%D0%9F%D0%B8%D1%81%D0%B0%D1%82%D0%B5%D0%BB%D1%8C" title="Писатель">писатель</a> и <a href="/wiki/%D0%96%D1%83%D1%80%D0%BD%D0%B0%D0%BB%D0%B8%D1%81%D1%82" title="Журналист">радиожурналист</a> <a href="/wiki/%D0%A2%D0%B5%D1%80%D0%BA%D0%B5%D0%BB,_%D0%A1%D1%82%D0%B0%D0%B4%D1%81" title="Теркел, Стадс">Стадс Теркел</a> в своей оценке<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup> книги в ходе передачи <span data-interwiki-article="Book of the Week" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=Book_of_the_Week&amp;action=edit&amp;redlink=1" title="Book of the Week (страница отсутствует)">Book of the Week</a></span> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;"><a class="extiw" href="https://en.wikipedia.org/wiki/Book_of_the_Week" title="en:Book of the Week"><span title="Book of the Week — версия статьи «Book of the Week» на английском языке">Book of the Week</span></a></span>) сравнил Вулфа с <a href="/wiki/%D0%9C%D0%B5%D0%B9%D0%BB%D0%B5%D1%80,_%D0%9D%D0%BE%D1%80%D0%BC%D0%B0%D0%BD" title="Мейлер, Норман">Норманом Мейлером</a> — назвав первого своим кандидатом на роль «величайшего журналиста».
</p><p>В статье «В садах Запада пробил час закрытия»<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup> Елена Дьякова провела параллели между «Электропрохладительным кислотным тестом» и романом «<a href="/wiki/V._(%D1%80%D0%BE%D0%BC%D0%B0%D0%BD)" title="V. (роман)">V.</a>» <a href="/wiki/%D0%9F%D0%B8%D0%BD%D1%87%D0%BE%D0%BD,_%D0%A2%D0%BE%D0%BC%D0%B0%D1%81" title="Пинчон, Томас">Тома Пинчона</a> — отметив красочность работы Вулфа, чья книга «сверкала всеми красками <a href="/wiki/%D0%9A%D0%B0%D0%BB%D0%B8%D1%84%D0%BE%D1%80%D0%BD%D0%B8%D1%8F" title="Калифорния">Калифорнии</a> и неоном ночных хайвеев, звенела бубенчиками и бисером „<a href="/wiki/%D0%A5%D0%B8%D0%BF%D0%BF%D0%B8" title="Хиппи">детей цветов</a>“, отбрасывала в сознание резкие черные тени мексиканских бродяг, голосующих на обочинах».
</p><p>«<a href="/wiki/%D0%90%D0%BF%D0%BE%D0%BA%D1%80%D0%B8%D1%84" title="Апокриф">Апокрифом</a> 60-х» книга названа в статье<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup> В. М. Шпакова «Христос из Спрингфилда» — автор отмечает несостоявшееся становление новой религии (с <a href="/wiki/%D0%9A%D0%B8%D0%B7%D0%B8,_%D0%9A%D0%B5%D0%BD" title="Кизи, Кен">Кеном Кизи</a> в качестве мессии) и проводит аналогии между автобусным путешествием<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup> по <a href="/wiki/%D0%95%D0%B2%D1%80%D0%BE%D0%BF%D0%B0" title="Европа">Европе</a> <a href="/wiki/%D0%A1%D1%83%D0%B1%D0%BA%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D0%B0" title="Субкультура">субкультурной</a> группы художников из <a href="/wiki/%D0%A1%D0%B0%D0%BD%D0%BA%D1%82-%D0%9F%D0%B5%D1%82%D0%B5%D1%80%D0%B1%D1%83%D1%80%D0%B3" title="Санкт-Петербург">Санкт-Петербурга</a>, <a href="/wiki/%D0%9C%D0%B8%D1%82%D1%8C%D0%BA%D0%B8" title="Митьки">Митьков</a>, и путешествием Весёлых проказников «<a href="/wiki/%D0%94%D0%B0%D0%BB%D1%88%D0%B5" title="Далше">Далше</a>», описанным в «Электропрохладительном кислотном тесте» Вулфа.
</p><p>В рецензии<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup> для INOUT.ru, Любовь Атаманова назвала книгу «биографией 60-х» — однако в противовес ей документальность прозы Вулфа ставится под сомнение <a href="/wiki/%D0%9D%D1%83%D0%B3%D0%B0%D1%82%D0%BE%D0%B2,_%D0%92%D0%B0%D0%BB%D0%B5%D1%80%D0%B8%D0%B9_%D0%92%D0%B8%D0%BA%D1%82%D0%BE%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Нугатов, Валерий Викторович">Валерием Нугатым</a> в статье «Сны Орегона»<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup>, где «Электропрохладительный кислотный тест» назван «эпопеей в жанре истерического реализма». Помимо Нугатова, другие исследователи прозы Вулфа также отмечают, что в контексте <a href="/wiki/%D0%96%D1%83%D1%80%D0%BD%D0%B0%D0%BB%D0%B8%D1%81%D1%82%D0%B8%D0%BA%D0%B0" title="Журналистика">журналистики</a> книгу рассматривать не стоит<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup> — учитывая постоянное нахождение главных «героев» под <a href="/wiki/%D0%9B%D0%A1%D0%94" title="ЛСД">ЛСД</a> и относительное участие Вулфа в описываемых событиях.
</p><p>Делая ссылку на процесс написания книги, некоторые авторы подчеркивают<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup>, что Вулф с <a href="/wiki/%D0%92%D0%B5%D1%81%D1%91%D0%BB%D1%8B%D0%B5_%D0%BF%D1%80%D0%BE%D0%BA%D0%B0%D0%B7%D0%BD%D0%B8%D0%BA%D0%B8" title="Весёлые проказники">Проказниками</a> находился сравнительно небольшое количество времени, в то время как основные описанные им события представляют собой только пересказ очевидцев — что в общем смысле ставит под сомнение документальность книги в целом. В ряде работ с критикой «Электропрохладительного кислотного теста» так же отмечается<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup> не просматриваемая позиция самого автора по теме книги.
</p><p>По версии Arthur L. Carter Journalism Institute при <a href="/wiki/%D0%9D%D1%8C%D1%8E-%D0%99%D0%BE%D1%80%D0%BA%D1%81%D0%BA%D0%B8%D0%B9_%D1%83%D0%BD%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D1%82%D0%B5%D1%82" title="Нью-Йоркский университет">Нью-Йоркском университете</a>, книга вошла в список<sup class="reference" id="cite_ref-autogenerated1_1-1"><a href="#cite_note-autogenerated1-1">[1]</a></sup> 100 лучших работ по журналистике в <a href="/wiki/%D0%A1%D0%BE%D0%B5%D0%B4%D0%B8%D0%BD%D1%91%D0%BD%D0%BD%D1%8B%D0%B5_%D0%A8%D1%82%D0%B0%D1%82%D1%8B_%D0%90%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B8" title="Соединённые Штаты Америки">США</a> в XX веке. В <a class="external text" href="https://www.amazon.com/Electric-Kool-Aid-Acid-Test/product-reviews/0553380648/ref=pr_all_summary_cm_cr_acr_img?ie=UTF8&amp;showViewpoints=1" rel="nofollow">пользовательском рейтинге</a> сайта <a class="external text" href="https://www.amazon.com/" rel="nofollow">amazon.com</a> книга имеет <i>примерно</i> 4,2 звезды из 5 возможных.
</p>
<h2><span id=".D0.AD.D0.BA.D1.80.D0.B0.D0.BD.D0.B8.D0.B7.D0.B0.D1.86.D0.B8.D1.8F"></span><span class="mw-headline" id="Экранизация">Экранизация</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BF%D1%80%D0%BE%D1%85%D0%BB%D0%B0%D0%B4%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B8%D1%81%D0%BB%D0%BE%D1%82%D0%BD%D1%8B%D0%B9_%D1%82%D0%B5%D1%81%D1%82&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Экранизация»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BF%D1%80%D0%BE%D1%85%D0%BB%D0%B0%D0%B4%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B8%D1%81%D0%BB%D0%BE%D1%82%D0%BD%D1%8B%D0%B9_%D1%82%D0%B5%D1%81%D1%82&amp;action=edit&amp;section=7" title="Редактировать код раздела «Экранизация»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Премьера экранизации была запланирована<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup> на 2013 год. Режиссёр-постановщик — известный провокационным для России «<a href="/wiki/%D0%A5%D0%B0%D1%80%D0%B2%D0%B8_%D0%9C%D0%B8%D0%BB%D0%BA_(%D1%84%D0%B8%D0%BB%D1%8C%D0%BC)" title="Харви Милк (фильм)">Харви Милком</a>» <a href="/wiki/%D0%92%D0%B0%D0%BD_%D0%A1%D0%B5%D0%BD%D1%82,_%D0%93%D0%B0%D1%81" title="Ван Сент, Гас">Гас Ван Сент</a>, сценарист — <a href="/wiki/%D0%91%D0%BB%D1%8D%D0%BA,_%D0%94%D0%B0%D1%81%D1%82%D0%B8%D0%BD_%D0%9B%D1%8D%D0%BD%D1%81" title="Блэк, Дастин Лэнс">Дастин Лэнс Блэк</a>, также работавший над вышеупомянутым фильмом. На главную роль — <a href="/wiki/%D0%9A%D0%B8%D0%B7%D0%B8,_%D0%9A%D0%B5%D0%BD" title="Кизи, Кен">Кена Кизи</a>, по первоначальному плану<sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup> режиссёра предполагалось пригласить <a href="/wiki/%D0%9B%D0%B5%D0%B4%D0%B6%D0%B5%D1%80,_%D0%A5%D0%B8%D1%82" title="Леджер, Хит">Хита Леджера</a>, но из-за трагической <a href="/wiki/%D0%9B%D0%B5%D0%B4%D0%B6%D0%B5%D1%80,_%D0%A5%D0%B8%D1%82#Смерть" title="Леджер, Хит">смерти</a> актёра вопрос остаётся открытым. По словам самого Ван Сента, роль Кизи, возможно, сыграет <a href="/wiki/%D0%91%D0%BB%D1%8D%D0%BA,_%D0%94%D0%B6%D0%B5%D0%BA" title="Блэк, Джек">Джек Блэк</a> или <a href="/wiki/%D0%A5%D0%B0%D1%80%D1%80%D0%B5%D0%BB%D1%8C%D1%81%D0%BE%D0%BD,_%D0%92%D1%83%D0%B4%D0%B8" title="Харрельсон, Вуди">Вуди Харрельсон</a><sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[29]</a></sup>.
</p>
