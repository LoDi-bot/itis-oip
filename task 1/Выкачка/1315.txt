<p><b>Chrono Resurrection</b>, также известная как <b>Chrono Trigger: Resurrection</b>, — неизданная <a href="/wiki/%D0%9A%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D1%80%D0%BE%D0%BB%D0%B5%D0%B2%D0%B0%D1%8F_%D0%B8%D0%B3%D1%80%D0%B0" title="Компьютерная ролевая игра">компьютерная ролевая игра</a>, созданная поклонниками <a href="/wiki/Chrono_(%D1%81%D0%B5%D1%80%D0%B8%D1%8F_%D0%B8%D0%B3%D1%80)" title="Chrono (серия игр)">серии <i>Chrono</i></a> из <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%90%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B0" title="Северная Америка">североамериканской</a> команды Resurrection Games под началом Натана Лазура, выпуск которой был намечен на <a class="mw-redirect" href="/wiki/%D0%A0%D0%BE%D0%B6%D0%B4%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Рождество">Рождество</a> <a href="/wiki/2004_%D0%B3%D0%BE%D0%B4_%D0%B2_%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D1%8B%D1%85_%D0%B8%D0%B3%D1%80%D0%B0%D1%85" title="2004 год в компьютерных играх">2004 года</a><sup class="reference" id="cite_ref-NWR_2-1"><a href="#cite_note-NWR-2">[2]</a></sup>. Игра основана на популярной<sup class="reference" id="cite_ref-gamespothits_5-0"><a href="#cite_note-gamespothits-5">[5]</a></sup> игре <i><a href="/wiki/Chrono_Trigger" title="Chrono Trigger">Chrono Trigger</a></i>, выпущенной для игровой приставки <a href="/wiki/Super_Nintendo_Entertainment_System" title="Super Nintendo Entertainment System">Super NES</a> японской компанией <a class="mw-redirect" href="/wiki/Square_Co." title="Square Co.">Square Co.</a> в <a href="/wiki/1995_%D0%B3%D0%BE%D0%B4_%D0%B2_%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D1%8B%D1%85_%D0%B8%D0%B3%D1%80%D0%B0%D1%85" title="1995 год в компьютерных играх">1995 году</a>. Изначально проект назывался <i><b>CT64</b></i>, в рамках которого планировалось создать <a href="/wiki/%D0%A0%D0%B5%D0%BC%D0%B5%D0%B9%D0%BA" title="Ремейк">ремейк</a> оригинальной игры для приставки <a href="/wiki/Nintendo_64" title="Nintendo 64">Nintendo 64</a> с возможностью выбрать <a class="mw-redirect" href="/wiki/%D0%94%D0%B2%D1%83%D1%85%D0%BC%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D0%BA%D0%B0" title="Двухмерная графика">2D</a> или <a href="/wiki/%D0%A2%D1%80%D1%91%D1%85%D0%BC%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D0%BA%D0%B0" title="Трёхмерная графика">3D</a> графику.
</p><p>После перерыва в работе, цели проекта были переопределены: разработчики решили создать короткую <a href="/wiki/%D0%94%D0%B5%D0%BC%D0%BE%D0%BD%D1%81%D1%82%D1%80%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D1%8F_%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D0%BE%D0%B9_%D0%B8%D0%B3%D1%80%D1%8B" title="Демонстрационная версия компьютерной игры">демо-версию игры</a> для <a href="/wiki/%D0%9F%D0%B5%D1%80%D1%81%D0%BE%D0%BD%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80" title="Персональный компьютер">персональных компьютеров</a> под управлением операционной системы <a class="mw-redirect" href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a>; планировалось также портировать <i>Resurrection</i> на другие платформы. К команде присоединились новые специалисты, включая профессиональных художников и дизайнеров, которые должны были переделать десять игровых сцен и большинство <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D0%BF%D0%B5%D1%80%D1%81%D0%BE%D0%BD%D0%B0%D0%B6%D0%B5%D0%B9_Chrono_Trigger" title="Список персонажей Chrono Trigger">персонажей</a> из <i>Chrono Trigger</i> для демо. Однако в <a href="/wiki/2004_%D0%B3%D0%BE%D0%B4_%D0%B2_%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D1%8B%D1%85_%D0%B8%D0%B3%D1%80%D0%B0%D1%85" title="2004 год в компьютерных играх">2004 году</a> проект был официально закрыт после того, как компания <a href="/wiki/Square_Enix" title="Square Enix">Square Enix</a> отправила письмо Resurrection Games с требованием прекратить свою деятельность ввиду <a class="mw-redirect" href="/wiki/%D0%9D%D0%B0%D1%80%D1%83%D1%88%D0%B5%D0%BD%D0%B8%D0%B5_%D0%B0%D0%B2%D1%82%D0%BE%D1%80%D1%81%D0%BA%D0%B8%D1%85_%D0%BF%D1%80%D0%B0%D0%B2" title="Нарушение авторских прав">нарушения авторских прав</a><sup class="reference" id="cite_ref-copyright_6-0"><a href="#cite_note-copyright-6">[6]</a></sup>. Несмотря на закрытие, проект получил значительную поддержку от фанатов<sup class="reference" id="cite_ref-official_7-0"><a href="#cite_note-official-7">[7]</a></sup>.
</p>

<h2><span id=".D0.92.D0.B5.D1.80.D1.81.D0.B8.D1.8F_.D0.B4.D0.BB.D1.8F_Nintendo_64"></span><span class="mw-headline" id="Версия_для_Nintendo_64">Версия для Nintendo 64</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Chrono_Resurrection&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Версия для Nintendo 64»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Chrono_Resurrection&amp;action=edit&amp;section=1" title="Редактировать код раздела «Версия для Nintendo 64»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Изначально Натан Лазур планировал создать <a href="/wiki/%D0%A0%D0%B5%D0%BC%D0%B5%D0%B9%D0%BA" title="Ремейк">ремейк</a> популярной компьютерной ролевой игры <i><a href="/wiki/Chrono_Trigger" title="Chrono Trigger">Chrono Trigger</a></i> для приставки <a href="/wiki/Nintendo_64" title="Nintendo 64">Nintendo 64</a><sup class="reference" id="cite_ref-gamespothits_5-1"><a href="#cite_note-gamespothits-5">[5]</a></sup>; <i>Trigger</i> была выпущена в <a href="/wiki/1995_%D0%B3%D0%BE%D0%B4_%D0%B2_%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D1%8B%D1%85_%D0%B8%D0%B3%D1%80%D0%B0%D1%85" title="1995 год в компьютерных играх">1995 году</a> компанией <a class="mw-redirect" href="/wiki/Square_Co." title="Square Co.">Square Co.</a> для <a class="mw-redirect" href="/wiki/Super_NES" title="Super NES">Super NES</a>. Лазур назвал свой проект <i>CT64</i> и приступил к созданию игры с помощью утилит <a href="/wiki/GNU" title="GNU">GNU</a> для создания игр <a href="/wiki/Homebrew_(%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D1%8B%D0%B5_%D0%B8%D0%B3%D1%80%D1%8B)" title="Homebrew (компьютерные игры)">homebrew</a>. Идея изначально пришла к Натану в 1999 году после того, как он поиграл в <i>Chrono Trigger</i> и <i><a href="/wiki/Super_Mario_64" title="Super Mario 64">Super Mario 64</a></i>, а затем задумался, как <i>Trigger</i> выглядел бы в <a href="/wiki/%D0%A2%D1%80%D1%91%D1%85%D0%BC%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D0%BA%D0%B0" title="Трёхмерная графика">3D</a><sup class="reference" id="cite_ref-Retro_8-0"><a href="#cite_note-Retro-8">[8]</a></sup><sup class="reference" id="cite_ref-VGPro_9-0"><a href="#cite_note-VGPro-9">[9]</a></sup>. Этот ремейк, который разрабатывали всего четыре человека, был первой попыткой Натана Лазура создать полноценную игру, продолжающую <a href="/wiki/Chrono_(%D1%81%D0%B5%D1%80%D0%B8%D1%8F_%D0%B8%D0%B3%D1%80)" title="Chrono (серия игр)">серию <i>Chrono</i></a> — ранее он пробовал работать лишь над мини-играми. Планировалось, что в игре будет та же система сражений, как и в <i>Trigger</i>, но появится возможность <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D0%B0%D1%8F_%D0%B8%D0%B3%D1%80%D0%B0_(%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D1%8B%D0%B5_%D0%B8%D0%B3%D1%80%D1%8B)" title="Кооперативная игра (компьютерные игры)">прохождения вдвоём</a><sup class="reference" id="cite_ref-GameSpot_2000_4-1"><a href="#cite_note-GameSpot_2000-4">[4]</a></sup>. В одном из интервью Лазура спросили, как компании <a href="/wiki/Nintendo" title="Nintendo">Nintendo</a> и <a class="mw-redirect" href="/wiki/Square_Co." title="Square Co.">Square Co.</a> относятся к его проекту:
</p><p>
</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
Nintendo не знает о проекте и не оказывает ему никакой поддержки. Но все утилиты, которые я использую в работе, являются свободными (<a href="/wiki/GNU" title="GNU">GNU</a>), поэтому я не нарушаю авторских прав компании. Square тоже ничего не знает об этой игре, но я не думаю, что они обрадуются, узнав, что я использую их товарный знак. Поэтому на логотип было помещено примечание, указывающее, что марка <i>Chrono</i> принадлежит им. Впрочем, я вовсе не думаю, что это имеет какое-то значение, поскольку проект не коммерческий и я не собираюсь зарабатывать деньги за счёт их труда.
</p>
<div class="mw-collapsible mw-collapsed has-collapsible-header ts-Скрытый_блок ts-Скрытый_блок-gray ts-Скрытый_блок-noneHideLink ts-oq has-collapsible-header" style=""><div class="ts-Скрытый_блок-title collapsible-header ts-Скрытый_блок-title-leftTitle ts-oq-header collapsible-header" style="text-align:left;">Оригинальный текст <small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></div><div class="mw-collapsible-content ts-oq-content" style="">
<div dir="auto" lang="en">Nintendo doesn't know about the project, and it isn't supported by them at all. The development system is GNU based, so it doesn't infringe on any of their copyrights. Square Co. Ltd. doesn't know either, but I don't think that they'll appreciate me using their copyrights. Because of this, I have put a disclaimer on the splash screen saying that Chrono Trigger is their copyright, etc. I really don't think that it will matter, though, since this is not a commercial project and that I'm not making any kind of money off of their hard work.</div>
</div></div>
<div class="ts-Конец_цитаты-source">— <cite>Натан Лазур, программист и директор<sup class="reference" id="cite_ref-GameSpot_2000_4-2"><a href="#cite_note-GameSpot_2000-4">[4]</a></sup></cite></div></blockquote>
<p>Игра должна была содержать два режима. Первый подразумевал <a class="mw-redirect" href="/wiki/%D0%94%D0%B2%D1%83%D1%85%D0%BC%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D0%BA%D0%B0" title="Двухмерная графика">двухмерную</a> и предварительно <a href="/wiki/%D0%A0%D0%B5%D0%BD%D0%B4%D0%B5%D1%80%D0%B8%D0%BD%D0%B3" title="Рендеринг">отрендеренную</a> графику с <a href="/wiki/%D0%A2%D1%80%D1%91%D1%85%D0%BC%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D0%BA%D0%B0" title="Трёхмерная графика">трёхмерной</a> анимацией заклинаний и эффектов в битве. <a href="/wiki/%D0%A0%D0%B0%D0%B7%D1%80%D0%B5%D1%88%D0%B5%D0%BD%D0%B8%D0%B5_(%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D0%BA%D0%B0)" title="Разрешение (компьютерная графика)">Разрешение</a> в игре должно было быть стандартным для консоли <a href="/wiki/Nintendo_64" title="Nintendo 64">Nintendo 64</a> — 320 × 240, а <a href="/wiki/%D0%9A%D0%B0%D0%B4%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D1%87%D0%B0%D1%81%D1%82%D0%BE%D1%82%D0%B0" title="Кадровая частота">частота смены кадров</a> — 30 кадров в секунду. Часть анимации, созданной Лазуром, должна была работать с программной частью, а не с аппаратными ресурсами Nintendo 64, так как приставка некорректно выполняла рендеринг<sup class="reference" id="cite_ref-GameSpot_2000_4-3"><a href="#cite_note-GameSpot_2000-4">[4]</a></sup>.
</p><p>Второй режим, по замыслу разработчиков, должен был подразумевать полностью трёхмерную графику с обычным (320 × 240) или высоким (640 × 480) разрешением. Планировалось, что этот режим будет поддерживать различные <a href="/wiki/Level_of_Detail" title="Level of Detail">уровни детализации</a> текстур в зависимости от расстояния до камеры, чтобы максимально повысить производительность и чёткость графики (однако эта функция была включена только для обычного разрешения ввиду больших затрат ресурсов консоли). В обоих режимах вид в игре должен был быть реализован сзади и сверху, хотя во втором режиме камера иногда могла оказываться под другими, более удобными углами, как в <a href="/wiki/The_Legend_of_Zelda:_Ocarina_of_Time" title="The Legend of Zelda: Ocarina of Time">The Legend of Zelda: Ocarina of Time</a>. Частота кадров в <i>Resurrection</i> при таком режиме должна была достигать 25 кадров в секунду при количестве полигонов до 20 000 и с включённым звуком. Лазур отметил в одном из интервью, что создание второго режима игры является наиболее сложным, так как для его реализации требуется выполнить гораздо больше работы, а этим занимаются всего четыре человека<sup class="reference" id="cite_ref-GameSpot_2000_4-4"><a href="#cite_note-GameSpot_2000-4">[4]</a></sup>.
</p><p>Планировалось также добавление галереи, содержащей фанатские рисунки, музыку и <a href="/wiki/%D0%9C%D0%B8%D0%BD%D0%B8-%D0%B8%D0%B3%D1%80%D0%B0" title="Мини-игра">мини-игры</a>, например карточную игру, похожую на Triple Triad из <i><a href="/wiki/Final_Fantasy_VIII" title="Final Fantasy VIII">Final Fantasy VIII</a></i>. Игроки могли бы открывать дополнительные материалы по мере прохождения игры<sup class="reference" id="cite_ref-GameSpot_2000_4-5"><a href="#cite_note-GameSpot_2000-4">[4]</a></sup>. Разработчики хотели сделать игру бесплатной и доступной для загрузки через интернет. Играть в неё нужно было с помощью специального <a href="/wiki/%D0%AD%D0%BC%D1%83%D0%BB%D1%8F%D1%82%D0%BE%D1%80_%D0%B8%D0%B3%D1%80%D0%BE%D0%B2%D0%BE%D0%B9_%D0%BF%D1%80%D0%B8%D1%81%D1%82%D0%B0%D0%B2%D0%BA%D0%B8" title="Эмулятор игровой приставки">эмулятора игровой приставки</a>, так как издание <i>CT64</i> на <a href="/wiki/%D0%98%D0%B3%D1%80%D0%BE%D0%B2%D0%BE%D0%B9_%D0%BA%D0%B0%D1%80%D1%82%D1%80%D0%B8%D0%B4%D0%B6" title="Игровой картридж">картридже</a> было проблематичным по финансовым соображениям<sup class="reference" id="cite_ref-GameSpot_2000_4-6"><a href="#cite_note-GameSpot_2000-4">[4]</a></sup>. В итоге была создана только наполовину работающая версия с трёхмерной графикой, а затем в середине 2000 года проект был заморожен по нескольким причинам, включая случайную потерю данных Лазуром и его желание лучше изучить программирование. Он понял, что ему не хватает опыта, чтобы создать подходящую модель игры, кроме того, он с трудом находил людей для команды<sup class="reference" id="cite_ref-Retro_8-1"><a href="#cite_note-Retro-8">[8]</a></sup><sup class="reference" id="cite_ref-VGPro_9-1"><a href="#cite_note-VGPro-9">[9]</a></sup><sup class="reference" id="cite_ref-compedium_10-0"><a href="#cite_note-compedium-10">[10]</a></sup>.
</p>
<h2><span id=".D0.92.D1.82.D0.BE.D1.80.D0.B0.D1.8F_.D0.B2.D0.B5.D1.80.D1.81.D0.B8.D1.8F"></span><span class="mw-headline" id="Вторая_версия">Вторая версия</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Chrono_Resurrection&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Вторая версия»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Chrono_Resurrection&amp;action=edit&amp;section=2" title="Редактировать код раздела «Вторая версия»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>

<h3><span id=".D0.A0.D0.B0.D0.B7.D1.80.D0.B0.D0.B1.D0.BE.D1.82.D0.BA.D0.B0"></span><span class="mw-headline" id="Разработка">Разработка</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Chrono_Resurrection&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Разработка»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Chrono_Resurrection&amp;action=edit&amp;section=3" title="Редактировать код раздела «Разработка»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Разработка второй версии проекта, изначально названной <i>Chrono Trigger: Brink of Time</i>, а затем <i>Chrono Resurrection</i>, была начата в апреле <a href="/wiki/2003_%D0%B3%D0%BE%D0%B4_%D0%B2_%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D1%8B%D1%85_%D0%B8%D0%B3%D1%80%D0%B0%D1%85" title="2003 год в компьютерных играх">2003 года</a><sup class="reference" id="cite_ref-VGPro_9-2"><a href="#cite_note-VGPro-9">[9]</a></sup>. Лазур объяснил своё желание продолжить работу тем, что, во-первых, он повысил свои навыки, написав программу для карты памяти <a href="/wiki/Dreamcast" title="Dreamcast">Dreamcast</a><sup class="reference" id="cite_ref-Retro_8-2"><a href="#cite_note-Retro-8">[8]</a></sup> и работая в компании DC Studios, разрабатывающей игры<sup class="reference" id="cite_ref-VGPro_9-3"><a href="#cite_note-VGPro-9">[9]</a></sup><sup class="reference" id="cite_ref-compedium_10-1"><a href="#cite_note-compedium-10">[10]</a></sup>, а, во-вторых, он поиграл в <i><a href="/wiki/Chrono_Cross" title="Chrono Cross">Chrono Cross</a></i>, продолжение <i>Trigger</i> для <a href="/wiki/PlayStation" title="PlayStation">PlayStation</a>, и <a href="/wiki/%D0%A1%D0%B8%D0%BA%D0%B2%D0%B5%D0%BB" title="Сиквел">сиквел</a> ему очень понравился<sup class="reference" id="cite_ref-Retro_8-3"><a href="#cite_note-Retro-8">[8]</a></sup>. Программист заметил, что решил создать <i>Resurrection</i> потому, что он большой фанат и, кроме того, «это шанс доказать самому себе, что можешь создать нечто похожее на ту игру, в которую сам любишь играть»<sup class="reference" id="cite_ref-VGPro_9-4"><a href="#cite_note-VGPro-9">[9]</a></sup>. В отличие от <i>CT64</i>, новая игра должна была представлять короткое <a href="/wiki/%D0%94%D0%B5%D0%BC%D0%BE%D0%BD%D1%81%D1%82%D1%80%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D1%8F_%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D0%BE%D0%B9_%D0%B8%D0%B3%D1%80%D1%8B" title="Демонстрационная версия компьютерной игры">демо</a> из 10 эпизодов (сюжет которых основан на событиях <i>Chrono Trigger</i>), разработанное на <a class="mw-redirect" href="/wiki/%D0%9A%D1%80%D0%BE%D1%81%D1%81%D0%BF%D0%BB%D0%B0%D1%82%D1%84%D0%BE%D1%80%D0%BC%D0%B5%D0%BD%D0%BD%D0%BE%D0%B5_%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%BD%D0%BE%D0%B5_%D0%BE%D0%B1%D0%B5%D1%81%D0%BF%D0%B5%D1%87%D0%B5%D0%BD%D0%B8%D0%B5" title="Кроссплатформенное программное обеспечение">кроссплатформенном</a> <a href="/wiki/%D0%98%D0%B3%D1%80%D0%BE%D0%B2%D0%BE%D0%B9_%D0%B4%D0%B2%D0%B8%D0%B6%D0%BE%D0%BA" title="Игровой движок">движке</a> с <a href="/wiki/%D0%A2%D1%80%D1%91%D1%85%D0%BC%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D0%BA%D0%B0" title="Трёхмерная графика">трёхмерной графикой</a>. Натан Лазур объяснил, что для небольшой команды разработчиков создать полноценный <a href="/wiki/%D0%A0%D0%B5%D0%BC%D0%B5%D0%B9%D0%BA" title="Ремейк">ремейк</a> <i>Chrono Trigger</i> будет слишком сложно: это потребует больших финансовых затрат и очень много времени, поэтому разработка небольшой игры будет предпочтительнее. Он также заметил, что очень многие фанаты просили создать полноценную игру, но это было бы возможно только в том случае, если бы компания Square дала своё официальное разрешение, что, по мнению Лазура, «очень маловероятно", кроме того, "они сами сделают это гораздо лучше»<sup class="reference" id="cite_ref-VGPro_9-5"><a href="#cite_note-VGPro-9">[9]</a></sup><sup class="reference" id="cite_ref-Brews_11-0"><a href="#cite_note-Brews-11">[11]</a></sup>. Планировалось, что версия для <a class="mw-redirect" href="/wiki/Microsoft_Windows" title="Microsoft Windows">Windows</a> будет доступна для бесплатной загрузки из интернета, а портированные версии для <a href="/wiki/Nintendo_GameCube" title="Nintendo GameCube">Nintendo GameCube</a> и <a href="/wiki/Xbox" title="Xbox">Xbox</a> вместе с <a href="/wiki/SDK" title="SDK">SDK</a> будут доступны для официальных разработчиков<sup class="reference" id="cite_ref-NWR_2-2"><a href="#cite_note-NWR-2">[2]</a></sup>.
</p><p>Большая часть новой команды разработчиков была нанята через <a class="mw-redirect" href="/wiki/%D0%92%D0%B5%D0%B1-%D1%81%D0%B0%D0%B9%D1%82" title="Веб-сайт">веб-сайт</a> <a class="mw-redirect" href="/wiki/Gamasutra" title="Gamasutra">Gamasutra</a>; создание игры проводилось в небольшом офисе дома у Натана Лазура в <a href="/wiki/%D0%9C%D0%BE%D0%BD%D1%80%D0%B5%D0%B0%D0%BB%D1%8C" title="Монреаль">Монреале</a><sup class="reference" id="cite_ref-compedium_10-2"><a href="#cite_note-compedium-10">[10]</a></sup>. В команду входило девять человек, многие из которых имели около двух с половиной лет опыта работы в области разработки компьютерных игр<sup class="reference" id="cite_ref-Brews_11-1"><a href="#cite_note-Brews-11">[11]</a></sup>. Лазур впоследствии говорил, что начал серьёзно задумываться о качестве демо-игры только после того, как в проект пришёл <a href="/wiki/%D0%90%D1%80%D1%82-%D0%B4%D0%B8%D1%80%D0%B5%D0%BA%D1%82%D0%BE%D1%80" title="Арт-директор">арт-директор</a> Луис Мартинс (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Luis Martins</span>)<sup class="reference" id="cite_ref-Retro_8-4"><a href="#cite_note-Retro-8">[8]</a></sup>. В числе других профессиональных дизайнеров были Моис Брэтон (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Moise Breton</span>), ранее работавший над трёхмерными моделями персонажей для знаменитого фильма <a href="/wiki/%D0%9C%D0%B0%D1%82%D1%80%D0%B8%D1%86%D0%B0:_%D0%9F%D0%B5%D1%80%D0%B5%D0%B7%D0%B0%D0%B3%D1%80%D1%83%D0%B7%D0%BA%D0%B0" title="Матрица: Перезагрузка">Матрица: Перезагрузка</a><sup class="reference" id="cite_ref-VGPro_9-6"><a href="#cite_note-VGPro-9">[9]</a></sup> и Мишель Кадио (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Michel Cadieux</span>), <a class="mw-redirect" href="/wiki/%D0%A5%D1%83%D0%B4%D0%BE%D0%B6%D0%BD%D0%B8%D0%BA-%D0%BC%D1%83%D0%BB%D1%8C%D1%82%D0%B8%D0%BF%D0%BB%D0%B8%D0%BA%D0%B0%D1%82%D0%BE%D1%80" title="Художник-мультипликатор">художник-мультипликатор</a> из компании <a href="/wiki/Microids" title="Microids">Microids</a>. Трудности возникли при создании игрового движка, так как Лазур был единственным программистом и ему приходилось делать всё самостоятельно<sup class="reference" id="cite_ref-VGPro_9-7"><a href="#cite_note-VGPro-9">[9]</a></sup><sup class="reference" id="cite_ref-Brews_11-2"><a href="#cite_note-Brews-11">[11]</a></sup>.
</p><p><a href="/wiki/%D0%A1%D0%B0%D1%83%D0%BD%D0%B4%D1%82%D1%80%D0%B5%D0%BA" title="Саундтрек">Саундтрек</a> для игры был написан Мэтью Валентом (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Mathew Valente</span>), который ранее работал над первым проектом, в рамках которого планировалось создать игру для <a href="/wiki/Nintendo_64" title="Nintendo 64">Nintendo 64</a><sup class="reference" id="cite_ref-Retro_8-5"><a href="#cite_note-Retro-8">[8]</a></sup>. Натан Лазур утверждал, что слушал музыку Валента ещё до того, как познакомился с ним, поэтому был очень рад, когда композитор согласился присоединиться к проекту<sup class="reference" id="cite_ref-VGPro_9-8"><a href="#cite_note-VGPro-9">[9]</a></sup>. Музыкальное сопровождение состояло из <a href="/wiki/%D0%90%D1%80%D0%B0%D0%BD%D0%B6%D0%B8%D1%80%D0%BE%D0%B2%D0%BA%D0%B0" title="Аранжировка">аранжированных версий</a> <a href="/wiki/%D0%9C%D1%83%D0%B7%D1%8B%D0%BA%D0%B0_Chrono_Trigger" title="Музыка Chrono Trigger">официального саундтрека</a> <i>Chrono Trigger</i>, который написали композиторы <a class="mw-redirect" href="/wiki/%D0%AF%D1%81%D1%83%D0%BD%D0%BE%D1%80%D0%B8_%D0%9C%D0%B8%D1%86%D1%83%D0%B4%D0%B0" title="Ясунори Мицуда">Ясунори Мицуда</a>, <a class="mw-redirect" href="/wiki/%D0%9D%D0%BE%D0%B1%D1%83%D0%BE_%D0%A3%D1%8D%D0%BC%D0%B0%D1%86%D1%83" title="Нобуо Уэмацу">Нобуо Уэмацу</a> и <a class="mw-redirect" href="/wiki/%D0%9D%D0%BE%D1%80%D0%B8%D0%BA%D0%BE_%D0%9C%D0%B0%D1%86%D1%83%D1%8D%D0%B4%D0%B0" title="Норико Мацуэда">Норико Мацуэда</a><sup class="reference" id="cite_ref-Rhapsody_12-0"><a href="#cite_note-Rhapsody-12">[12]</a></sup><sup class="reference" id="cite_ref-matsueda_13-0"><a href="#cite_note-matsueda-13">[13]</a></sup><sup class="reference" id="cite_ref-trigger-credits_14-0"><a href="#cite_note-trigger-credits-14">[14]</a></sup>. Валент хотел передать атмосферу, которую создавал оригинальный саундтрек, но при этом усовершенствовать звуковое сопровождение для новой платформы<sup class="reference" id="cite_ref-GameProAU_15-0"><a href="#cite_note-GameProAU-15">[15]</a></sup>. Большинство аранжировок было записано в формате <a href="/wiki/Impulse_Tracker" title="Impulse Tracker">Impulse Tracker</a>, а затем конвертированы в <a href="/wiki/MIDI" title="MIDI">MIDI</a> и улучшены с помощью специальных инструментов<sup class="reference" id="cite_ref-Rhapsody_12-1"><a href="#cite_note-Rhapsody-12">[12]</a></sup>.
</p><p>Среди всех возникающих проблем в процессе разработки Натан Лазур отдельно выделил нехватку времени<sup class="reference" id="cite_ref-VGPro_9-9"><a href="#cite_note-VGPro-9">[9]</a></sup>:
</p><p>
</p>
<blockquote class="ts-Начало_цитаты-quote"><p>
…Всё же самая большая проблема — это нехватка времени. Все члены команды ведут активную жизнь… Кроме того, многие работают в игровой индустрии, и я могу сказать, что такой труд отнимает очень много энергии. Зачастую, когда возвращаешься домой, хочется поспать или просто отдохнуть от компьютера. А мы приходим домой и работаем над этим проектом всё свободное время. В таких условиях очень трудно создать даже небольшое демо, наподобие этого. Скажу честно, мы вкладываем всю душу в эту игру.</p><div class="ts-Конец_цитаты-source">— <cite>Натан Лазур, программист и директор</cite></div></blockquote>
<h3><span id=".D0.A1.D0.BE.D0.B4.D0.B5.D1.80.D0.B6.D0.B0.D0.BD.D0.B8.D0.B5_.D0.B8.D0.B3.D1.80.D1.8B"></span><span class="mw-headline" id="Содержание_игры">Содержание игры</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Chrono_Resurrection&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Содержание игры»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Chrono_Resurrection&amp;action=edit&amp;section=4" title="Редактировать код раздела «Содержание игры»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Chrono_Resurrection_screenshot.jpg"><img alt="Кроно, главный герой игры" class="mw-file-element" data-file-height="312" data-file-width="416" decoding="async" height="188" src="//upload.wikimedia.org/wikipedia/ru/thumb/6/62/Chrono_Resurrection_screenshot.jpg/250px-Chrono_Resurrection_screenshot.jpg" srcset="//upload.wikimedia.org/wikipedia/ru/thumb/6/62/Chrono_Resurrection_screenshot.jpg/375px-Chrono_Resurrection_screenshot.jpg 1.5x, //upload.wikimedia.org/wikipedia/ru/6/62/Chrono_Resurrection_screenshot.jpg 2x" width="250"/></a><figcaption><a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D0%BF%D0%B5%D1%80%D1%81%D0%BE%D0%BD%D0%B0%D0%B6%D0%B5%D0%B9_Chrono_Trigger#crono" title="Список персонажей Chrono Trigger">Хроно</a>, главный герой игры, образ которого взят из <i>Chrono Trigger</i></figcaption></figure>
<p>Команда планировала передать атмосферу оригинальной игры в своём демо, улучшив при этом графику и саундтрек<sup class="reference" id="cite_ref-Brews_11-3"><a href="#cite_note-Brews-11">[11]</a></sup>. В игру должно было входить десять сцен из <i>Chrono Trigger</i><sup class="reference" id="cite_ref-Back_16-0"><a href="#cite_note-Back-16">[16]</a></sup>. Несмотря на просьбы фанатов, разработчики не планировали создать полный ремейк игры, так как у них не было необходимых ресурсов и они считали, что результат был бы не сравним с продуктами компании Square<sup class="reference" id="cite_ref-VGPro_9-10"><a href="#cite_note-VGPro-9">[9]</a></sup>. Сюжет <i>Chrono Resurrection</i> был немного изменён, чтобы запланированные эпизоды сочетались друг с другом<sup class="reference" id="cite_ref-Brews_11-4"><a href="#cite_note-Brews-11">[11]</a></sup>.
</p><p>Предполагалось, что отряд игрока будет состоять из заранее определённого набора персонажей; также можно было бы находить дополнительных (секретных) героев для повышения интереса к повторному прохождению игры. Из-за нехватки времени, разработчики не были уверены, что станут добавлять двух героев — <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D0%BF%D0%B5%D1%80%D1%81%D0%BE%D0%BD%D0%B0%D0%B6%D0%B5%D0%B9_Chrono_Trigger#robo" title="Список персонажей Chrono Trigger">Робо</a> и <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D0%BF%D0%B5%D1%80%D1%81%D0%BE%D0%BD%D0%B0%D0%B6%D0%B5%D0%B9_Chrono_Trigger#ayla" title="Список персонажей Chrono Trigger">Эйлу</a> — из семи, которые фигурировали в оригинальной игре. «Качество, а не количество», — заметил Лазур в одном из интервью<sup class="reference" id="cite_ref-VGPro_9-11"><a href="#cite_note-VGPro-9">[9]</a></sup>. Так как команда планировала улучшить графику по сравнению с предыдущими играми <a href="/wiki/Chrono_(%D1%81%D0%B5%D1%80%D0%B8%D1%8F_%D0%B8%D0%B3%D1%80)" title="Chrono (серия игр)">серии</a>, дизайнеры персонажей столкнулись с трудностями, связанными с созданием героев ввиду разницы между <a href="/wiki/%D0%A1%D0%BF%D1%80%D0%B0%D0%B9%D1%82_(%D0%BA%D0%BE%D0%BC%D0%BF%D1%8C%D1%8E%D1%82%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D0%BA%D0%B0)" title="Спрайт (компьютерная графика)">спрайтами</a>, рисунками и <a class="mw-redirect" href="/wiki/%D0%90%D0%BD%D0%B8%D0%BC%D1%8D" title="Анимэ">анимированной</a> графикой<sup class="reference" id="cite_ref-VGPro_9-12"><a href="#cite_note-VGPro-9">[9]</a></sup>. Тем не менее они отмечали, что смогли справиться с этой проблемой и даже добавили немного своего творчества<sup class="reference" id="cite_ref-Brews_11-5"><a href="#cite_note-Brews-11">[11]</a></sup>.
</p>
<h2><span id=".D0.97.D0.B0.D0.BA.D1.80.D1.8B.D1.82.D0.B8.D0.B5_.D0.BF.D1.80.D0.BE.D0.B5.D0.BA.D1.82.D0.B0_.D0.B8_.D0.BA.D1.80.D0.B8.D1.82.D0.B8.D0.BA.D0.B0"></span><span class="mw-headline" id="Закрытие_проекта_и_критика">Закрытие проекта и критика</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=Chrono_Resurrection&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Закрытие проекта и критика»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Chrono_Resurrection&amp;action=edit&amp;section=5" title="Редактировать код раздела «Закрытие проекта и критика»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Guardia2.jpg"><img alt="Гардия, королевство, фигурировавшее в Chrono Trigger" class="mw-file-element" data-file-height="252" data-file-width="447" decoding="async" height="169" src="//upload.wikimedia.org/wikipedia/ru/thumb/9/95/Guardia2.jpg/300px-Guardia2.jpg" srcset="//upload.wikimedia.org/wikipedia/ru/9/95/Guardia2.jpg 1.5x" width="300"/></a><figcaption>Гардия, королевство, фигурировавшее в <i>Chrono Trigger</i></figcaption></figure>
<p><a class="mw-disambig" href="/wiki/%D0%A0%D0%B5%D0%BB%D0%B8%D0%B7" title="Релиз">Релиз</a> <i>Chrono Resurrection</i> должен был состояться на <a class="mw-redirect" href="/wiki/%D0%A0%D0%BE%D0%B6%D0%B4%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Рождество">Рождество</a> 2004 года<sup class="reference" id="cite_ref-NWR2_3-3"><a href="#cite_note-NWR2-3">[3]</a></sup>. К этому моменту о готовящейся игре узнало множество фанатов и, по словам Натана Лазура, их веб-сайт посетило около миллиона человек, многие оставляли свои комментарии и пожелания<sup class="reference" id="cite_ref-compedium_10-3"><a href="#cite_note-compedium-10">[10]</a></sup>. Однако японская компания Square Enix, создавшая игры серии <i>Chrono</i>, послала команде разработчиков Resurrection Team письмо с требованием закрыть проект и прекратить разработку игры ввиду <a class="mw-redirect" href="/wiki/%D0%9D%D0%B0%D1%80%D1%83%D1%88%D0%B5%D0%BD%D0%B8%D0%B5_%D0%B0%D0%B2%D1%82%D0%BE%D1%80%D1%81%D0%BA%D0%B8%D1%85_%D0%BF%D1%80%D0%B0%D0%B2" title="Нарушение авторских прав">нарушения авторских прав</a><sup class="reference" id="cite_ref-copyright_6-1"><a href="#cite_note-copyright-6">[6]</a></sup>. Сначала Лазур посчитал письмо шуткой, но юрист, прочитав его, ответил: «Это не шутка». «Вместо <a href="/wiki/%D0%A2%D1%80%D0%B5%D0%B9%D0%BB%D0%B5%D1%80_(%D0%BA%D0%B8%D0%BD%D0%B5%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84)" title="Трейлер (кинематограф)">трейлера</a> с финальной надписью „Выход состоится в ближайшее время“ нам пришлось писать „Спасибо за вашу поддержку“», — говорит Натан в одном из интервью<sup class="reference" id="cite_ref-compedium_10-4"><a href="#cite_note-compedium-10">[10]</a></sup>. Таким образом, проект был закрыт 6 сентября 2004 года<sup class="reference" id="cite_ref-1UP_17-0"><a href="#cite_note-1UP-17">[17]</a></sup>. По словам разработчиков, за три месяца до получения ими письма, с <a href="/wiki/IP-%D0%B0%D0%B4%D1%80%D0%B5%D1%81" title="IP-адрес">IP-адресов</a> Square Enix постоянно осуществлялись просмотры <a class="mw-redirect" href="/wiki/%D0%92%D0%B5%D0%B1-%D1%81%D0%B0%D0%B9%D1%82" title="Веб-сайт">веб-сайта</a> <i>Chrono Resurrection</i> и загрузка информации об игре. Они предположили, что сайт посещали скорее рядовые служащие, нежели руководство компании<sup class="reference" id="cite_ref-1UP_17-1"><a href="#cite_note-1UP-17">[17]</a></sup>, и надеялись, что они воспримут демо таким, каким оно было запланировано — данью <i>Chrono Trigger</i>, а не его заменой<sup class="reference" id="cite_ref-Brews_11-6"><a href="#cite_note-Brews-11">[11]</a></sup><sup class="reference" id="cite_ref-Back_16-1"><a href="#cite_note-Back-16">[16]</a></sup>.
</p><p>Веб-сайты, посвящённые компьютерным играм, такие, как <a href="/wiki/1UP.com" title="1UP.com">1UP.com</a> и <a href="/wiki/GameSpot" title="GameSpot">GameSpot</a> назвали проект «амбициозным», хорошо отозвавшись о графике разрабатываемой игры, заметив также, что стиль, в котором нарисованы персонажи, очень похож на стиль <a class="mw-redirect" href="/wiki/%D0%90%D0%BA%D0%B8%D1%80%D0%B0_%D0%A2%D0%BE%D1%80%D0%B8%D1%8F%D0%BC%D0%B0" title="Акира Торияма">Акиры Ториямы</a>, создателя оригинальных персонажей <i>Chrono Trigger</i><sup class="reference" id="cite_ref-1UP_17-2"><a href="#cite_note-1UP-17">[17]</a></sup><sup class="reference" id="cite_ref-GameSpot_18-0"><a href="#cite_note-GameSpot-18">[18]</a></sup>. Рецензент 1UP.com также добавил, что факт закрытия проекта «печален», но Square Enix не могла позволить, чтобы игра, пытающаяся «соперничать» с <i>Trigger</i>, вышла в свет<sup class="reference" id="cite_ref-Rhapsody_12-2"><a href="#cite_note-Rhapsody-12">[12]</a></sup>. Обозреватель GameSpot, помимо своего разочарования, связанного с закрытием проекта, отметил, что никаких новостей, связанных с созданием новой официальной игры серии <i>Chrono</i> (которая должна была выйти под торговой маркой <i><a href="/wiki/Chrono_Break" title="Chrono Break">Chrono Break</a></i><sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup>), так и не поступило. «Фанаты остались ни с чем», — заключает он<sup class="reference" id="cite_ref-GameSpot_18-1"><a href="#cite_note-GameSpot-18">[18]</a></sup>. Обозреватель сайта <span data-interwiki-article="Nintendo World Report" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=Nintendo_World_Report&amp;action=edit&amp;redlink=1" title="Nintendo World Report (страница отсутствует)">Nintendo World Report</a></span><span class="noprint" style="font-style:normal; font-weight:normal;"><sup><a class="extiw" href="https://en.wikipedia.org/wiki/Nintendo_World_Report" title="en:Nintendo World Report"><span title="Nintendo World Report — версия статьи «Nintendo World Report» на английском языке">[en]</span></a></sup></span> высоко оценил графику и музыку <i>Chrono Resurrection</i>, назвав качество работы «профессиональным»<sup class="reference" id="cite_ref-NWR2_3-4"><a href="#cite_note-NWR2-3">[3]</a></sup><sup class="reference" id="cite_ref-NWR_2-3"><a href="#cite_note-NWR-2">[2]</a></sup>. Он также выразил своё сожаление по поводу закрытия проекта:
</p><p>
</p>
<blockquote class="ts-Начало_цитаты-quote"><p>
Очень жаль, что юристы задавили такой многообещающий проект. И, тем не менее, команда не отчаивается и уже планирует создать что-то «новое и свежее». Мы признательны всем, кто работал над этой игрой<sup class="reference" id="cite_ref-NWR2_3-5"><a href="#cite_note-NWR2-3">[3]</a></sup>…</p><div class="ts-Конец_цитаты-source">— <cite>Михаэль Коул, обозреватель сайта <span data-interwiki-article="Nintendo World Report" data-interwiki-lang="en"><a class="new" href="/w/index.php?title=Nintendo_World_Report&amp;action=edit&amp;redlink=1" title="Nintendo World Report (страница отсутствует)">Nintendo World Report</a></span><span class="noprint" style="font-style:normal; font-weight:normal;"><sup><a class="extiw" href="https://en.wikipedia.org/wiki/Nintendo_World_Report" title="en:Nintendo World Report"><span title="Nintendo World Report — версия статьи «Nintendo World Report» на английском языке">[en]</span></a></sup></span></cite></div></blockquote>
<p>Рецензент GamePro Australia назвал проект «вероятно, самым лучшим ремейком, созданным фанатами игры, который был раздавлен гигантским ботинком крупной компании»<sup class="reference" id="cite_ref-GameProAU_15-1"><a href="#cite_note-GameProAU-15">[15]</a></sup>.
</p><p>Фанаты отправили несколько прошений по интернету в компанию Square Enix, надеясь, что проекту всё же будет дан зелёный свет; однако это не оказало никакого эффекта<sup class="reference" id="cite_ref-1UP_17-3"><a href="#cite_note-1UP-17">[17]</a></sup>. Натан Лазур, который, конечно, был расстроен, не стал винить Square, понимая, что она защищала свою интеллектуальную собственность<sup class="reference" id="cite_ref-compedium_10-5"><a href="#cite_note-compedium-10">[10]</a></sup>. Программист сказал, что для него «большой честью является уже то, что его проект заметили»<sup class="reference" id="cite_ref-Retro_8-6"><a href="#cite_note-Retro-8">[8]</a></sup>. Также Лазур отметил, что во избежание таких ситуаций разработчики подобных фанатских игр должны предварительно отправлять демо в компанию-правообладатель<sup class="reference" id="cite_ref-1UP_17-4"><a href="#cite_note-1UP-17">[17]</a></sup>. Впоследствии команда просила, чтобы юрист помог составить им письмо для отправки в Square с просьбой пересмотреть своё решение, но ответа на него не последовало<sup class="reference" id="cite_ref-compedium_10-6"><a href="#cite_note-compedium-10">[10]</a></sup>. Мэтью Валент, композитор Resurrection Team, заметил, что письмо от Square опечалило очень многих фанатов:
</p><p>
</p>
<blockquote class="ts-Начало_цитаты-quote"><p>
Я думаю о <i>Chrono Resurrection</i> и меня всё ещё огорчает то, что случилось. Я знаю очень многих людей, которые действительно любят <i>Chrono Trigger</i> и надеются, что Square Enix наконец-то выпустит продолжение. &lt;…&gt; Я продолжаю надеяться, что в один маловероятный день, Square разрешит нам продолжить свою работу<sup class="reference" id="cite_ref-GameProAU_15-2"><a href="#cite_note-GameProAU-15">[15]</a></sup>…</p><div class="ts-Конец_цитаты-source">— <cite>Мэтью Валент, композитор</cite></div></blockquote>
<p>До закрытия проекта Натан Лазур сообщил, что его команда не планирует создавать ремейки других игр после работы над <i>Resurrection</i> и, скорее, займётся разработкой оригинальной видеоигры, действие которой будет происходить в <a href="/wiki/%D0%98%D1%81%D1%82%D0%BE%D1%80%D0%B8%D1%8F_%D0%AF%D0%BF%D0%BE%D0%BD%D0%B8%D0%B8" title="История Японии">Японии эпохи феодализма</a><sup class="reference" id="cite_ref-VGPro_9-13"><a href="#cite_note-VGPro-9">[9]</a></sup>.
</p>
