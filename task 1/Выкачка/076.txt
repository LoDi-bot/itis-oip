<p><b>Осада Иерусалима</b> — одно из ключевых событий <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B2%D1%8B%D0%B9_%D0%BA%D1%80%D0%B5%D1%81%D1%82%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D0%BE%D1%85%D0%BE%D0%B4" title="Первый крестовый поход">Первого крестового похода</a>. Происходила с <a href="/wiki/7_%D0%B8%D1%8E%D0%BD%D1%8F" title="7 июня">7 июня</a> по <a href="/wiki/15_%D0%B8%D1%8E%D0%BB%D1%8F" title="15 июля">15 июля</a> <a href="/wiki/1099_%D0%B3%D0%BE%D0%B4" title="1099 год">1099 года</a>, в результате город был захвачен войсками крестоносцев. Успешный исход осады привёл к завершению Первого крестового похода и созданию в <a href="/wiki/%D0%9C%D0%B0%D0%BB%D0%B0%D1%8F_%D0%90%D0%B7%D0%B8%D1%8F" title="Малая Азия">Малой Азии</a> <a href="/wiki/%D0%93%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B0_%D0%BA%D1%80%D0%B5%D1%81%D1%82%D0%BE%D0%BD%D0%BE%D1%81%D1%86%D0%B5%D0%B2" title="Государства крестоносцев">государств крестоносцев</a>.
</p>

<h2><span id=".D0.9F.D1.80.D0.B5.D0.B4.D1.88.D0.B5.D1.81.D1.82.D0.B2.D1.83.D1.8E.D1.89.D0.B8.D0.B5_.D1.81.D0.BE.D0.B1.D1.8B.D1.82.D0.B8.D1.8F"></span><span class="mw-headline" id="Предшествующие_события">Предшествующие события</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D0%B0_(1099)&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Предшествующие события»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D0%B0_(1099)&amp;action=edit&amp;section=1" title="Редактировать код раздела «Предшествующие события»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="hatnote navigation-not-searchable">Основные статьи: <b><a href="/wiki/%D0%9A%D0%BB%D0%B5%D1%80%D0%BC%D0%BE%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D0%BE%D0%B1%D0%BE%D1%80" title="Клермонский собор">Клермонский собор</a></b>, <b><a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B2%D1%8B%D0%B9_%D0%BA%D1%80%D0%B5%D1%81%D1%82%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D0%BE%D1%85%D0%BE%D0%B4" title="Первый крестовый поход">Первый крестовый поход</a></b> и <b><a href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82%D1%8C%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D1%80%D0%B5%D1%81%D1%82%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D0%BE%D1%85%D0%BE%D0%B4" title="Крестьянский крестовый поход">Крестьянский крестовый поход</a></b></div>
<p>К концу <a href="/wiki/XI_%D0%B2%D0%B5%D0%BA" title="XI век">XI века</a>, в <a href="/wiki/%D0%97%D0%B0%D0%BF%D0%B0%D0%B4%D0%BD%D0%B0%D1%8F_%D0%95%D0%B2%D1%80%D0%BE%D0%BF%D0%B0" title="Западная Европа">Западной Европе</a> в силу целого ряда причин (прежде всего <a href="/wiki/%D0%AD%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D0%BA%D0%B0" title="Экономика">экономического</a> и <a href="/wiki/%D0%94%D0%B5%D0%BC%D0%BE%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D1%8F" title="Демография">демографического</a> характера) создались предпосылки для крестового похода с целью освобождения <a href="/wiki/%D0%93%D1%80%D0%BE%D0%B1_%D0%93%D0%BE%D1%81%D0%BF%D0%BE%D0%B4%D0%B5%D0%BD%D1%8C" title="Гроб Господень">Гроба Господня</a><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>. <a href="/wiki/26_%D0%BD%D0%BE%D1%8F%D0%B1%D1%80%D1%8F" title="26 ноября">26 ноября</a> <a href="/wiki/1095_%D0%B3%D0%BE%D0%B4" title="1095 год">1095 года</a>, <a class="mw-redirect" href="/wiki/%D0%9F%D0%B0%D0%BF%D0%B0_%D0%A0%D0%B8%D0%BC%D1%81%D0%BA%D0%B8%D0%B9" title="Папа Римский">папа</a> <a href="/wiki/%D0%A3%D1%80%D0%B1%D0%B0%D0%BD_II" title="Урбан II">Урбан II</a>, на <a href="/wiki/%D0%9A%D0%BB%D0%B5%D1%80%D0%BC%D0%BE%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D0%BE%D0%B1%D0%BE%D1%80" title="Клермонский собор">Клермонском соборе</a>, призвал отправиться на Восток и освободить <a href="/wiki/%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC" title="Иерусалим">Иерусалим</a> от владычества <a href="/wiki/%D0%9C%D1%83%D1%81%D1%83%D0%BB%D1%8C%D0%BC%D0%B0%D0%BD%D0%B8%D0%BD" title="Мусульманин">мусульман</a>, обещав участникам <a class="mw-redirect" href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D0%BE%D1%85%D0%BE%D0%B4" title="Крестовый поход">крестового похода</a> отпущение грехов, охрану семей и имущества и освобождение от долгов. Эта идея охватила все христианские государства Западной Европы. Стихийно организованный <a href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82%D1%8C%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D1%80%D0%B5%D1%81%D1%82%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D0%BE%D1%85%D0%BE%D0%B4" title="Крестьянский крестовый поход">поход бедноты</a> окончился полным провалом — практически все его участники были уничтожены <a href="/wiki/%D0%A1%D0%B5%D0%BB%D1%8C%D0%B4%D0%B6%D1%83%D0%BA%D0%B8" title="Сельджуки">сельджуками</a><sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>. Чуть позже выступило <a href="/wiki/%D0%A4%D0%B5%D0%BE%D0%B4%D0%B0%D0%BB%D0%B8%D0%B7%D0%BC" title="Феодализм">феодальное</a> войско <a href="/wiki/%D0%9D%D0%BE%D1%80%D0%BC%D0%B0%D0%BD%D0%BD%D1%8B" title="Норманны">норманнских</a>, <a class="mw-redirect" href="/wiki/%D0%A1%D1%80%D0%B5%D0%B4%D0%BD%D0%B5%D0%B2%D0%B5%D0%BA%D0%BE%D0%B2%D0%B0%D1%8F_%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%8F" title="Средневековая Франция">французских</a> и <a href="/wiki/%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%A0%D0%B8%D0%BC%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F" title="Священная Римская империя">германских</a> <a href="/wiki/%D0%A0%D1%8B%D1%86%D0%B0%D1%80%D1%8C" title="Рыцарь">рыцарей</a>. Армия не являлась единым целым, так как каждый феодал привлекал своих вассалов, а возглавляющие их военачальники оспаривали друг у друга лидерство<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>.
</p><p>Весной <a href="/wiki/1097_%D0%B3%D0%BE%D0%B4" title="1097 год">1097 года</a> объединённое войско переправилось через <a href="/wiki/%D0%91%D0%BE%D1%81%D1%84%D0%BE%D1%80" title="Босфор">Босфор</a> и высадилось в <a href="/wiki/%D0%90%D0%B7%D0%B8%D1%8F" title="Азия">Азии</a>, при этом руководители похода принесли <a href="/wiki/%D0%9A%D0%BE%D0%BC%D0%BC%D0%B5%D0%BD%D0%B4%D0%B0%D1%86%D0%B8%D1%8F" title="Коммендация">вассальную присягу</a> <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D0%B2%D0%B8%D0%B7%D0%B0%D0%BD%D1%82%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D1%85_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%BE%D1%80%D0%BE%D0%B2" title="Список византийских императоров">византийскому императору</a> <a href="/wiki/%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_I_%D0%9A%D0%BE%D0%BC%D0%BD%D0%B8%D0%BD" title="Алексей I Комнин">Алексею I Комнину</a> на те земли, которые могли быть ими завоёваны<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>. В течение следующих двух лет армия медленно продвигалась на юг. При помощи <a class="mw-redirect" href="/wiki/%D0%92%D0%B8%D0%B7%D0%B0%D0%BD%D1%82%D0%B8%D0%B9%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F" title="Византийская империя">византийских</a> отрядов им удалось <a href="/wiki/%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%9D%D0%B8%D0%BA%D0%B5%D0%B8_(1097)" title="Осада Никеи (1097)">захватить Никею</a>, а позднее разгромить армию <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%BD%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D1%83%D0%BB%D1%82%D0%B0%D0%BD%D0%B0%D1%82" title="Конийский султанат">румского султана</a>
<a href="/wiki/%D0%9A%D1%8B%D0%BB%D1%8B%D1%87-%D0%90%D1%80%D1%81%D0%BB%D0%B0%D0%BD_I" title="Кылыч-Арслан I">Кылыч-Арслана I</a> в <a href="/wiki/%D0%91%D0%B8%D1%82%D0%B2%D0%B0_%D0%BF%D1%80%D0%B8_%D0%94%D0%BE%D1%80%D0%B8%D0%BB%D0%B5%D0%B5_(1097)" title="Битва при Дорилее (1097)">битве при Дорилее</a>. Дойдя до <a href="/wiki/%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D1%8F" title="Антиохия">Антиохии</a>, густонаселённого и хорошо укреплённого города, крестоносцы <a href="/wiki/%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B8" title="Осада Антиохии">осадили его</a>, столкнувшись с большими трудностями<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>. Поход к <a href="/wiki/%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC" title="Иерусалим">Иерусалиму</a> был продолжен лишь спустя полгода после окончательного падения Антиохии <a href="/wiki/28_%D0%B8%D1%8E%D0%BD%D1%8F" title="28 июня">28 июня</a> <a href="/wiki/1098_%D0%B3%D0%BE%D0%B4" title="1098 год">1098 года</a>. За эти несколько месяцев армия понесла серьёзные потери — в августе в лагере христиан началась <a href="/wiki/%D0%AD%D0%BF%D0%B8%D0%B4%D0%B5%D0%BC%D0%B8%D1%8F" title="Эпидемия">эпидемия</a> <a href="/wiki/%D0%AD%D0%BF%D0%B8%D0%B4%D0%B5%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D1%8B%D0%BF%D0%BD%D0%BE%D0%B9_%D1%82%D0%B8%D1%84" title="Эпидемический сыпной тиф">тифа</a>, которая унесла жизни нескольких тысяч человек, в том числе <a href="/wiki/%D0%9F%D0%B0%D0%BF%D1%81%D0%BA%D0%B8%D0%B9_%D0%BB%D0%B5%D0%B3%D0%B0%D1%82" title="Папский легат">папского легата</a> и духовного лидера крестоносцев <a href="/wiki/%D0%95%D0%BF%D0%B8%D1%81%D0%BA%D0%BE%D0%BF" title="Епископ">епископа</a> <a href="/wiki/%D0%90%D0%B4%D0%B5%D0%BC%D0%B0%D1%80_%D0%9C%D0%BE%D0%BD%D1%82%D0%B5%D0%B9%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9" title="Адемар Монтейльский">Адемара Монтейльского</a><sup class="reference" id="cite_ref-Рамбо1_8-0"><a href="#cite_note-Рамбо1-8">[8]</a></sup>.
</p>
<h3><span id=".D0.9F.D1.80.D0.BE.D0.B4.D0.BE.D0.BB.D0.B6.D0.B5.D0.BD.D0.B8.D0.B5_.D0.BF.D0.BE.D1.85.D0.BE.D0.B4.D0.B0"></span><span class="mw-headline" id="Продолжение_похода">Продолжение похода</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D0%B0_(1099)&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Продолжение похода»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D0%B0_(1099)&amp;action=edit&amp;section=2" title="Редактировать код раздела «Продолжение похода»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Armenian_Colophon_of_1099.jpg"><img class="mw-file-element" data-file-height="1333" data-file-width="1911" decoding="async" height="174" src="//upload.wikimedia.org/wikipedia/commons/thumb/7/7d/Armenian_Colophon_of_1099.jpg/250px-Armenian_Colophon_of_1099.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/7/7d/Armenian_Colophon_of_1099.jpg/375px-Armenian_Colophon_of_1099.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/7/7d/Armenian_Colophon_of_1099.jpg/500px-Armenian_Colophon_of_1099.jpg 2x" width="250"/></a><figcaption>Армянский <a href="/wiki/%D0%9A%D0%BE%D0%BB%D0%BE%D1%84%D0%BE%D0%BD_(%D1%87%D0%B0%D1%81%D1%82%D1%8C_%D0%BA%D0%BD%D0%B8%D0%B3%D0%B8)" title="Колофон (часть книги)">колофон</a> 1099 года о взятии Иерусалима<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup></figcaption></figure>
<p>В стане военачальников крестоносцев продолжались раздоры. Провансальский маркграф <a class="mw-redirect" href="/wiki/%D0%A0%D0%B0%D0%B9%D0%BC%D1%83%D0%BD%D0%B4_IV_%D0%A2%D1%83%D0%BB%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9" title="Раймунд IV Тулузский">Раймунд Тулузский</a>, верный союзу заключённому с императором Византии Алексеем Комнином, долго отказывался признать справедливыми притязяния <a class="mw-redirect" href="/wiki/%D0%91%D0%BE%D1%8D%D0%BC%D1%83%D0%BD%D0%B4_I,_%D0%BA%D0%BD%D1%8F%D0%B7%D1%8C_%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D0%B8" title="Боэмунд I, князь Антиохии">Боэмунда Тарентского</a> на Антиохию, но в конце концов под давлением соратников ему пришлось уступить. Боэмунд был провозглашен князем Антиохии, после чего решил остаться править в новообразованном государстве и не продолжать поход<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>. <a class="mw-redirect" href="/wiki/%D0%91%D0%B0%D0%BB%D0%B4%D1%83%D0%B8%D0%BD_I_(%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D1%8C_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D0%B0)" title="Балдуин I (король Иерусалима)">Балдуин Булонский</a>, который правил в основанном им в <a href="/wiki/1098_%D0%B3%D0%BE%D0%B4" title="1098 год">1098 году</a> <a class="mw-redirect" href="/wiki/%D0%93%D1%80%D0%B0%D1%84%D1%81%D1%82%D0%B2%D0%BE_%D0%AD%D0%B4%D0%B5%D1%81%D1%81%D0%BA%D0%BE%D0%B5" title="Графство Эдесское">графстве Эдесском</a>, тоже не собирался присоединяться к походу на <a href="/wiki/%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC" title="Иерусалим">Иерусалим</a>. На этом этапе поход окончили брат французского короля <a href="/wiki/%D0%93%D1%83%D0%B3%D0%BE_%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B8%D0%B9_(%D0%B3%D1%80%D0%B0%D1%84_%D0%92%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%B4%D1%83%D0%B0)" title="Гуго Великий (граф Вермандуа)">Гуго де Вермандуа</a> и <a href="/wiki/%D0%AD%D1%82%D1%8C%D0%B5%D0%BD_II_%D0%B4%D0%B5_%D0%91%D0%BB%D1%83%D0%B0" title="Этьен II де Блуа">Этьен, граф Блуаский</a>, которые со своими войсками вернулись на родину<sup class="reference" id="cite_ref-Успенский_11-0"><a href="#cite_note-Успенский-11">[11]</a></sup>.
</p><p>В ноябре <a href="/wiki/1098_%D0%B3%D0%BE%D0%B4" title="1098 год">1098 года</a> граф Раймунд Тулузский и подчиненные ему рыцари <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D1%81" title="Прованс">Прованса</a> выдвинулись из <a href="/wiki/%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D1%8F" title="Антиохия">Антиохии</a> на юго-восток к <a class="mw-redirect" href="/wiki/%D0%9C%D0%B0%D0%B0%D1%80%D1%80%D0%B5%D1%82-%D1%8D%D0%BD-%D0%9D%D1%83%D1%83%D0%BC%D0%B0%D0%BD" title="Мааррет-эн-Нууман">Маарре</a> и <a href="/wiki/23_%D0%BD%D0%BE%D1%8F%D0%B1%D1%80%D1%8F" title="23 ноября">23 ноября</a> <a href="/wiki/%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%9C%D0%B0%D0%B0%D1%80%D1%80%D1%8B" title="Осада Маарры">осадили город</a>. По приказу Раймунда из вырубленного в окрестностях леса были построены осадные орудия, и в том числе башня в четыре яруса, с верхней площадки которой крестоносцы метали в осажденных камни. Гарнизон Маарры упорно сопротивлялся, в свою очередь забрасывая лагерь христиан камнями, стрелами и <a href="/wiki/%D0%93%D1%80%D0%B5%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BE%D0%B3%D0%BE%D0%BD%D1%8C" title="Греческий огонь">греческим огнём</a>. Вскоре к осаде присоединилась норманнская армия Боэмунда Тарентского — будучи давним соперником Раймунда, он не хотел, чтобы тот взял Маарру единолично, — и <a href="/wiki/11_%D0%B4%D0%B5%D0%BA%D0%B0%D0%B1%D1%80%D1%8F" title="11 декабря">11 декабря</a> <a href="/wiki/1098_%D0%B3%D0%BE%D0%B4" title="1098 год">1098 года</a> Маарра, атакованная сразу с двух сторон, пала, после чего крестоносцы разграбили город и почти поголовно истребили его население. После захвата города между норманнами и провансальцами начались раздоры, доходившие до вооружённых стычек<sup class="reference" id="cite_ref-Рамбо1_8-1"><a href="#cite_note-Рамбо1-8">[8]</a></sup>.
</p><p>К концу года простые воины начали проявлять недовольство тем, что поход затягивается, и грозили отправиться к своей главной цели — Иерусалиму — самостоятельно, не дожидаясь, пока примут решение военачальники. Возмущение было настолько велико, что лидеры крестоносцев были вынуждены прислушаться к голосу воинов, и <a href="/wiki/13_%D1%8F%D0%BD%D0%B2%D0%B0%D1%80%D1%8F" title="13 января">13 января</a> <a href="/wiki/1099_%D0%B3%D0%BE%D0%B4" title="1099 год">1099 года</a> продолжить поход на юг, в сторону Иерусалима<sup class="reference" id="cite_ref-Заборов_1-3"><a href="#cite_note-Заборов-1">[1]</a></sup>. Продвигаясь по побережью <a href="/wiki/%D0%A1%D1%80%D0%B5%D0%B4%D0%B8%D0%B7%D0%B5%D0%BC%D0%BD%D0%BE%D0%B5_%D0%BC%D0%BE%D1%80%D0%B5" title="Средиземное море">Средиземного моря</a>, армия христиан практически не встречала сопротивления (снабжение провиантом осуществлял <a href="/wiki/%D0%9F%D0%B8%D0%B7%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D1%80%D0%B5%D1%81%D0%BF%D1%83%D0%B1%D0%BB%D0%B8%D0%BA%D0%B0" title="Пизанская республика">пизанский</a> флот). Очередная задержка произошла у <a href="/wiki/%D0%A2%D1%80%D0%B8%D0%BF%D0%BE%D0%BB%D0%B8_(%D0%9B%D0%B8%D0%B2%D0%B0%D0%BD)" title="Триполи (Ливан)">Триполи</a>, которой был осаждён Раймундом Тулузским. Осада продолжалась более чем полгода и была прекращена по требованию основной массы войска<sup class="reference" id="cite_ref-Успенский_11-1"><a href="#cite_note-Успенский-11">[11]</a></sup>. Далее, чтобы не терять времени, крестоносцы обходили стороной <a class="mw-redirect" href="/wiki/%D0%A2%D0%B8%D1%80_(%D0%B3%D0%BE%D1%80%D0%BE%D0%B4)" title="Тир (город)">Тир</a>, <a class="mw-redirect" href="/wiki/%D0%90%D0%BA%D1%80%D0%B0" title="Акра">Акру</a>, <a href="/wiki/%D0%9A%D0%B5%D1%81%D0%B0%D1%80%D0%B8%D1%8F_%D0%9F%D0%B0%D0%BB%D0%B5%D1%81%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B0%D1%8F" title="Кесария Палестинская">Кесарею</a> и другие хорошо укрепленные города. Достигнув <a href="/wiki/%D0%A0%D0%B0%D0%BC%D0%BB%D0%B0" title="Рамла">Рамлы</a>, военачальники похода опять разошлись во мнениях относительно того, как поступить дальше — напасть на <a href="/wiki/%D0%94%D0%B0%D0%BC%D0%B0%D1%81%D0%BA" title="Дамаск">Дамаск</a> или поразить Фатимидов в <a href="/wiki/%D0%9A%D0%B0%D0%B8%D1%80" title="Каир">Каире</a>. Однако было принято решение не отступать от намеченной цели и продолжить наступление на Иерусалим<sup class="reference" id="cite_ref-Харпер_12-0"><a href="#cite_note-Харпер-12">[12]</a></sup>.
</p>
<h2><span id=".D0.9E.D1.81.D0.B0.D0.B4.D0.B0"></span><span class="mw-headline" id="Осада">Осада</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D0%B0_(1099)&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Осада»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D0%B0_(1099)&amp;action=edit&amp;section=3" title="Редактировать код раздела «Осада»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Taking_of_Jerusalem_by_the_Crusaders,_15th_July_1099.jpg"><img class="mw-file-element" data-file-height="1312" data-file-width="2287" decoding="async" height="172" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/d2/Taking_of_Jerusalem_by_the_Crusaders%2C_15th_July_1099.jpg/300px-Taking_of_Jerusalem_by_the_Crusaders%2C_15th_July_1099.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/d/d2/Taking_of_Jerusalem_by_the_Crusaders%2C_15th_July_1099.jpg/450px-Taking_of_Jerusalem_by_the_Crusaders%2C_15th_July_1099.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/d/d2/Taking_of_Jerusalem_by_the_Crusaders%2C_15th_July_1099.jpg/600px-Taking_of_Jerusalem_by_the_Crusaders%2C_15th_July_1099.jpg 2x" width="300"/></a><figcaption>Завоевание Иерусалима крестоносцами, 15 июля 1099. Художник — <a href="/wiki/%D0%A1%D0%B8%D0%BD%D1%8C%D0%BE%D0%BB%D1%8C,_%D0%AD%D0%BC%D0%B8%D0%BB%D1%8C" title="Синьоль, Эмиль">Эмиль Синьоль</a>, 1847 год.</figcaption></figure>
<p>Ранним утром, во вторник <a href="/wiki/7_%D0%B8%D1%8E%D0%BD%D1%8F" title="7 июня">7 июня</a> <a href="/wiki/1099_%D0%B3%D0%BE%D0%B4" title="1099 год">1099 года</a> крестоносцы добрались до Иерусалима. Всего к Иерусалиму подошла армия из 40 000 человек (по другим данным не превышала 20 000<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup>), половину из которых составляли пешие воины, и полторы тысячи рыцарей<sup class="reference" id="cite_ref-Жан_Ришар_14-0"><a href="#cite_note-Жан_Ришар-14">[14]</a></sup>. Многие воины в религиозном порыве опускались на колени, плакали и молились, увидев издалека в рассветных лучах солнца столь желанные стены Священного Города, ради освобождения которого вышли в поход три года назад и прошли многие тысячи километров.
Иерусалим находился под властью <a href="/wiki/%D0%A4%D0%B0%D1%82%D0%B8%D0%BC%D0%B8%D0%B4%D1%8B" title="Фатимиды">фатимидского халифа</a>, который отнял его у <a href="/wiki/%D0%A1%D0%B5%D0%BB%D1%8C%D0%B4%D0%B6%D1%83%D0%BA" title="Сельджук">сельджуков</a>. <a href="/wiki/%D0%AD%D0%BC%D0%B8%D1%80" title="Эмир">Эмир</a> Иерусалима <a class="new" href="/w/index.php?title=%D0%98%D1%84%D1%82%D0%B8%D0%BA%D0%B0%D1%80_%D0%B0%D0%B4-%D0%94%D0%B0%D1%83%D0%BB%D0%B0&amp;action=edit&amp;redlink=1" title="Ифтикар ад-Даула (страница отсутствует)">Ифтикар ад-Даула</a> направил к крестоносцам посольство, предложив им беспрепятственно совершать <a href="/wiki/%D0%9F%D0%B0%D0%BB%D0%BE%D0%BC%D0%BD%D0%B8%D1%87%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Паломничество">паломничество</a> к <a href="/wiki/%D0%A1%D0%B2%D1%8F%D1%82%D0%B0%D1%8F_%D0%B7%D0%B5%D0%BC%D0%BB%D1%8F" title="Святая земля">святым местам</a>, небольшими и к тому же невооружёнными группами. Однако, вожди похода ответили на это предложение категорическим отказом, не помышляя оставлять величайшие христианские святыни под властью неверных <a href="/wiki/%D0%98%D1%81%D0%BB%D0%B0%D0%BC" title="Ислам">мусульман</a><sup class="reference" id="cite_ref-Жан_Ришар_14-1"><a href="#cite_note-Жан_Ришар-14">[14]</a></sup><sup class="reference" id="cite_ref-Рамбо2_15-0"><a href="#cite_note-Рамбо2-15">[15]</a></sup>.
</p><p>Роберт Нормандский разбил лагерь с северной стороны около церкви Св. Стефана. Далее расположилась армия
<a href="/wiki/%D0%A0%D0%BE%D0%B1%D0%B5%D1%80%D1%82_II_(%D0%B3%D1%80%D0%B0%D1%84_%D0%A4%D0%BB%D0%B0%D0%BD%D0%B4%D1%80%D0%B8%D0%B8)" title="Роберт II (граф Фландрии)">Роберта Фландрского</a>. Отряды <a href="/wiki/%D0%93%D0%BE%D1%82%D1%84%D1%80%D0%B8%D0%B4_%D0%91%D1%83%D0%BB%D1%8C%D0%BE%D0%BD%D1%81%D0%BA%D0%B8%D0%B9" title="Готфрид Бульонский">Готфрида Бульонского</a> и <a href="/wiki/%D0%A2%D0%B0%D0%BD%D0%BA%D1%80%D0%B5%D0%B4_%D0%A2%D0%B0%D1%80%D0%B5%D0%BD%D1%82%D1%81%D0%BA%D0%B8%D0%B9" title="Танкред Тарентский">Танкреда Тарентского</a> стояли на западе напротив <a href="/wiki/%D0%91%D0%B0%D1%88%D0%BD%D1%8F_%D0%94%D0%B0%D0%B2%D0%B8%D0%B4%D0%B0" title="Башня Давида">башни Давида</a> и <a href="/wiki/%D0%AF%D1%84%D1%84%D1%81%D0%BA%D0%B8%D0%B5_%D0%B2%D0%BE%D1%80%D0%BE%D1%82%D0%B0" title="Яффские ворота">Яффских ворот</a>, через которые обычно проходили прибывшие из Европы паломники. На юге занял позиции Раймунд Тулузский, укрепившись на горе <a href="/wiki/%D0%A1%D0%B8%D0%BE%D0%BD" title="Сион">Сион</a> близ церкви Св. Марии.
Армия крестоносцев, согласно хронике Раймунда Ажильского, состояла из 1200—1300 рыцарей и 12 000 пехотинцев (в начале похода их было соответственно около 7000 и 20 000 человек)<sup class="reference" id="cite_ref-Раймунд_Ажильский_2-1"><a href="#cite_note-Раймунд_Ажильский-2">[2]</a></sup>. В дополнение к тому могло быть ещё несколько тысяч воинов из <a href="/wiki/%D0%9C%D0%B0%D1%80%D0%BE%D0%BD%D0%B8%D1%82%D1%8B" title="Марониты">маронитов</a>, немногочисленных местных христиан и остатков христианского <a href="/wiki/%D0%9E%D0%BF%D0%BE%D0%BB%D1%87%D0%B5%D0%BD%D0%B8%D0%B5" title="Ополчение">ополчения</a> прибывшего сюда ранее и присоединившегося к Крестоносному войску шедшему на освобождение Святого Города. Ввиду этого общая численность христиан вполне могла составлять 30 — 35 тысяч человек, что было намного меньше гарнизона и жителей города. Но христианское войско было воодушевлено близостью заветной цели и пребывало в хорошем моральном состоянии.
</p>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Siege_of_Jerusalem_1099_(2).jpg"><img class="mw-file-element" data-file-height="310" data-file-width="550" decoding="async" height="169" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/6f/Siege_of_Jerusalem_1099_%282%29.jpg/300px-Siege_of_Jerusalem_1099_%282%29.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/6f/Siege_of_Jerusalem_1099_%282%29.jpg/450px-Siege_of_Jerusalem_1099_%282%29.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/6/6f/Siege_of_Jerusalem_1099_%282%29.jpg 2x" width="300"/></a><figcaption>Осада Иерусалима в 1099 году<br/>(<a href="/wiki/%D0%9C%D0%B8%D0%BD%D0%B8%D0%B0%D1%82%D1%8E%D1%80%D0%B0" title="Миниатюра">миниатюра</a> <a href="/wiki/XV_%D0%B2%D0%B5%D0%BA" title="XV век">XV века</a>).</figcaption></figure>
<p>Осада Иерусалима во многом повторяла историю осады Антиохии. Перед прибытием врага эмир Фатимидов изгнал из Иерусалима местных христиан и укрепил крепостные стены. Крестоносцы же — как и полгода назад — из-за нехватки продовольствия и воды претерпевали большие мучения, нежели осажденные<sup class="reference" id="cite_ref-Рамбо2_15-1"><a href="#cite_note-Рамбо2-15">[15]</a></sup>. Мусульмане отравили и загрязнили все окрестные колодцы, поэтому крестоносцам приходилось привозить воду из источника в 10 километрах от города в бурдюках, наскоро сшитых из бычьих шкур<sup class="reference" id="cite_ref-fr_16-0"><a href="#cite_note-fr-16">[16]</a></sup>.
</p><p>Понимая, что время дорого и с каждым днём положение будет только усугубляться, <a href="/wiki/13_%D0%B8%D1%8E%D0%BD%D1%8F" title="13 июня">13 июня</a> крестоносцы пошли на штурм крепостных стен. Поднявшись по приставным лестницам, они вступили в жестокую схватку с гарнизоном, однако ввиду высоты и мощи стен, осаждённым удалось отбить атаку. В это время пришло известие, что основные силы <a href="/wiki/%D0%93%D0%B5%D0%BD%D1%83%D1%8D%D0%B7%D1%81%D0%BA%D0%B0%D1%8F_%D1%80%D0%B5%D1%81%D0%BF%D1%83%D0%B1%D0%BB%D0%B8%D0%BA%D0%B0" title="Генуэзская республика">генуэзской</a> эскадры, направленной на помощь крестоносцам, были разбиты <a class="mw-redirect" href="/wiki/%D0%95%D0%B3%D0%B8%D0%BF%D0%B5%D1%82_%D0%BE%D1%82_%D1%80%D0%B0%D1%81%D0%BF%D0%B0%D0%B4%D0%B0_%D0%A5%D0%B0%D0%BB%D0%B8%D1%84%D0%B0%D1%82%D0%B0_%D0%B4%D0%BE_%D0%9E%D1%81%D0%BC%D0%B0%D0%BD%D1%81%D0%BA%D0%BE%D0%B9_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D0%B8" title="Египет от распада Халифата до Османской империи">египетским</a> флотом. Однако, <a href="/wiki/17_%D0%B8%D1%8E%D0%BD%D1%8F" title="17 июня">17 июня</a> в <a href="/wiki/%D0%AF%D1%84%D1%84%D0%B0" title="Яффа">Яффу</a> прибыло шесть уцелевших кораблей с продовольствием — благодаря чему угроза голода на время отступила — и также различными инструментами для сооружения военных машин<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup>. Понимая какую важность составляет доставленный груз, Раймунд Тулузский направил в гавань отряд из сотни рыцарей для охраны судов, но те столкнулись с засадой мусульман, и в завязавшемся бою обе стороны понесли потери. В конце июня тревожные слухи стали подтверждаться и до рыцарского войска дошли вести о том, что из <a href="/wiki/%D0%95%D0%B3%D0%B8%D0%BF%D0%B5%D1%82" title="Египет">Египта</a> на помощь Иерусалиму выдвинулась фатимидская армия<sup class="reference" id="cite_ref-Успенский_11-2"><a href="#cite_note-Успенский-11">[11]</a></sup>.
</p>
<h3><span id=".D0.9A.D1.80.D0.B5.D1.81.D1.82.D0.BD.D0.BE.D0.B5_.D1.88.D0.B5.D1.81.D1.82.D0.B2.D0.B8.D0.B5_.D0.B8_.D1.80.D0.B5.D1.88.D0.B0.D1.8E.D1.89.D0.B8.D0.B9_.D1.88.D1.82.D1.83.D1.80.D0.BC_.D0.98.D0.B5.D1.80.D1.83.D1.81.D0.B0.D0.BB.D0.B8.D0.BC.D0.B0"></span><span class="mw-headline" id="Крестное_шествие_и_решающий_штурм_Иерусалима">Крестное шествие и решающий штурм Иерусалима</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D0%B0_(1099)&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Крестное шествие и решающий штурм Иерусалима»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D0%B0_(1099)&amp;action=edit&amp;section=4" title="Редактировать код раздела «Крестное шествие и решающий штурм Иерусалима»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>В начале июля одному из монахов явилось видение <a href="/wiki/%D0%95%D0%BF%D0%B8%D1%81%D0%BA%D0%BE%D0%BF" title="Епископ">епископа</a> Адемара Монтейльского, год назад скончавшегося в <a href="/wiki/%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D1%8F" title="Антиохия">Антиохии</a>, который предписал воинам «устроить Бога ради крестное шествие вокруг укреплений Иерусалима, усердно молиться, творить милостыню и соблюдать пост» и тогда на девятый день Иерусалим падёт<sup class="reference" id="cite_ref-fr_16-1"><a href="#cite_note-fr-16">[16]</a></sup>.
<a href="/wiki/6_%D0%B8%D1%8E%D0%BB%D1%8F" title="6 июля">6 июля</a> военачальники и епископы провели совет, на котором решили исполнить предписание Адемара, и в пятницу <a href="/wiki/8_%D0%B8%D1%8E%D0%BB%D1%8F" title="8 июля">8 июля</a> босые крестоносцы в сопровождении своих духовных наставников — <a href="/wiki/%D0%9F%D1%91%D1%82%D1%80_%D0%90%D0%BC%D1%8C%D0%B5%D0%BD%D1%81%D0%BA%D0%B8%D0%B9" title="Пётр Амьенский">Петра Пустынника</a>, Раймунда Ажильского и Арнульфа Шокесского — провели крестную процессию вокруг стен Иерусалима и, распевая <a class="mw-redirect" href="/wiki/%D0%9F%D1%81%D0%B0%D0%BB%D0%BC" title="Псалм">псалмы</a>, дошли до <a class="mw-redirect" href="/wiki/%D0%9C%D0%B0%D1%81%D0%BB%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D0%B3%D0%BE%D1%80%D0%B0" title="Масличная гора">Масличной горы</a>, чем вызвали недоумение, страх и ярость мусульман, которые во время чтения епископами молитв, выкрикивали оскорбления в адрес христиан и священных таинств. Последнее и вызвало гнев крестоносцев как в процессе штурма так и в ходе захвата города.
</p>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:1099_Siege_of_Jerusalem.jpg"><img class="mw-file-element" data-file-height="900" data-file-width="661" decoding="async" height="408" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/33/1099_Siege_of_Jerusalem.jpg/300px-1099_Siege_of_Jerusalem.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/33/1099_Siege_of_Jerusalem.jpg/450px-1099_Siege_of_Jerusalem.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/3/33/1099_Siege_of_Jerusalem.jpg/600px-1099_Siege_of_Jerusalem.jpg 2x" width="300"/></a><figcaption>Крестоносцы штурмуют город с помощью осадных башен (средневековая <a href="/wiki/%D0%9C%D0%B8%D0%BD%D0%B8%D0%B0%D1%82%D1%8E%D1%80%D0%B0" title="Миниатюра">миниатюра</a>)</figcaption></figure>
<p>Понимая, что пассивная осада может затянуться, крестоносцы углубились в окрестные земли <a href="/wiki/%D0%A1%D0%B0%D0%BC%D0%B0%D1%80%D0%B8%D1%8F" title="Самария">Самарии</a>, чтобы нарубить деревьев для осадных машин, после чего плотники соорудили две осадные башни, метательные машины и другие военные приспособления. Затем состоялся совет, на котором был отдан приказ готовиться к схватке<sup class="reference" id="cite_ref-Харпер_12-1"><a href="#cite_note-Харпер-12">[12]</a></sup>.
</p>
<div class="ts-Цитата"><blockquote><p>Пусть каждый человек приготовится к бою на 14-е. Пока же пусть все пребывают на страже, молятся и творят милостыню. Повозки вместе с мастерами пусть будут впереди с тем, чтобы мастеровые снесли стволы, колья и жерди, а девицы пускай плетут фашины из прутьев. Повелевается, чтобы каждые два рыцаря изготовили один плетеный щит, либо лестницу. Выкиньте прочь всякие сомнения насчет того, чтобы сразиться за Бога, ибо в ближайшие же дни он завершит ваши ратные труды<sup class="reference" id="cite_ref-Раймунд_Ажильский_2-2"><a href="#cite_note-Раймунд_Ажильский-2">[2]</a></sup>.</p></blockquote></div>
<p>Все это было тщательно выполнено. Затем постановили, кто из князей, действуя со своими людьми, какую часть города будет атаковать и в каких местах будут пущены в ход осадные орудия, а также какие именно. Штурм Иерусалима начался на рассвете <a href="/wiki/14_%D0%B8%D1%8E%D0%BB%D1%8F" title="14 июля">14 июля</a>. Крестоносцы забрасывали город камнями из метательных машин, а оборонявшиеся осыпали их градом <a href="/wiki/%D0%A1%D1%82%D1%80%D0%B5%D0%BB%D0%B0" title="Стрела">стрел</a> и бросали со стен камни, лили кипяток, сбрасывали утыканные гвоздями «просмоленные &lt;…&gt; деревяшки, обертывая их в горящие тряпки»<sup class="reference" id="cite_ref-fr_16-2"><a href="#cite_note-fr-16">[16]</a></sup>. Обстрел камнями, однако, не причинил городу особого вреда, так как жители города защитили стены мешками, набитыми хлопком, которые смягчали удар. Под непрекращающимся обстрелом — как пишет <a href="/wiki/%D0%93%D0%B8%D0%B9%D0%BE%D0%BC_%D0%A2%D0%B8%D1%80%D1%81%D0%BA%D0%B8%D0%B9" title="Гийом Тирский">Гийом Тирский</a>, «стрелы и дротики сыпались на людей с обеих сторон, подобно граду» — крестоносцы начали придвигать к стенам Иерусалима осадные башни, однако им помешал опоясывающий город глубокий ров, который начали засыпать ещё <a href="/wiki/12_%D0%B8%D1%8E%D0%BB%D1%8F" title="12 июля">12 июля</a><sup class="reference" id="cite_ref-tir_18-0"><a href="#cite_note-tir-18">[18]</a></sup>.
</p><p>Сражение продолжалось весь день, но город держался. Когда наступила ночь, обе стороны продолжали бодрствовать — жители города боялись, что последует новая атака, а католические <a href="/wiki/%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%B8%D0%B0%D0%BD%D1%81%D1%82%D0%B2%D0%BE" title="Христианство">христиане</a> опасались, что осажденным удастся каким-то образом поджечь осадные орудия.
Утро <a href="/wiki/15_%D0%B8%D1%8E%D0%BB%D1%8F" title="15 июля">15 июля</a>, началось с общей молитвы и песнопений, христиане громко пели священные псалмы и подняв сотни знамён, стальными клиньями устремились к стенам. Европейские арбалетчики метко стреляя, пробивали мусульман стрелами насквозь, чем вызывали страх перед этим оружием. И когда ров был засыпан, крестоносцы смогли наконец беспрепятственно приблизить башни к крепостным стенам, лучники подожгли защищающие их мешки<sup class="reference" id="cite_ref-Гибб_19-0"><a href="#cite_note-Гибб-19">[19]</a></sup>. и смели со стен защитников. Толпы воинов и рыцарей в священном рвении и экстазе бросились на стены, разя длинными обоюдоострыми мечами и тяжёлыми топорами, переламывая арабские кривые сабли, разбивая кожаные шлемы и чалмы, сметая всё на своём пути. Такого невероятного давления оборонявшиеся не выдержали, защитники дрогнули, и уже ничто не могло больше удержать крестоносцев от вступления в город.
</p><p>Это стало переломным моментом в атаке — крестоносцы под непрекращающийся рёв и воинственные крики перекинули на стены деревянные мостки и раздавив защитников — толпами устремились за стены. Первым согласно преданию прорвался рыцарь Летольд, за ним последовали Готфрид Бульонский, Танкред Тарентский и храбрейшие рыцари и воины Европы.
Раймунд Тулузский, армия которого штурмовала город с другой стороны, узнал о прорыве и тоже устремился в Иерусалим через южные ворота. Увидев, что город уже пал, эмир гарнизона башни Давида был сломлен происходящим и открыл Яффские ворота<sup class="reference" id="cite_ref-Егер_20-0"><a href="#cite_note-Егер-20">[20]</a></sup>.
</p>
<h3><span id=".D0.A0.D0.B5.D0.B7.D0.BD.D1.8F_.D0.B8_.D1.80.D0.B0.D0.B7.D0.B3.D1.80.D0.B0.D0.B1.D0.BB.D0.B5.D0.BD.D0.B8.D0.B5_.D0.B3.D0.BE.D1.80.D0.BE.D0.B4.D0.B0"></span><span class="mw-headline" id="Резня_и_разграбление_города">Резня и разграбление города</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D0%B0_(1099)&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Резня и разграбление города»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D0%B0_(1099)&amp;action=edit&amp;section=5" title="Редактировать код раздела «Резня и разграбление города»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:EroberungJerusalems2.jpg"><img class="mw-file-element" data-file-height="979" data-file-width="1034" decoding="async" height="284" src="//upload.wikimedia.org/wikipedia/commons/thumb/6/6b/EroberungJerusalems2.jpg/300px-EroberungJerusalems2.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/6/6b/EroberungJerusalems2.jpg/450px-EroberungJerusalems2.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/6/6b/EroberungJerusalems2.jpg/600px-EroberungJerusalems2.jpg 2x" width="300"/></a><figcaption>Резня в Иерусалиме и кровавая река, вытекающая из городских ворот (<a href="/wiki/%D0%9C%D0%B8%D0%BD%D0%B8%D0%B0%D1%82%D1%8E%D1%80%D0%B0" title="Миниатюра">миниатюра</a>, <a href="/wiki/XIII_%D0%B2%D0%B5%D0%BA" title="XIII век">XIII век</a>)</figcaption></figure>
<p>После того, как крестоносцы ворвались в город, началась резня. Нападавшие убивали всех подряд — <a href="/wiki/%D0%98%D1%81%D0%BB%D0%B0%D0%BC" title="Ислам">мусульман</a>, <a href="/wiki/%D0%95%D0%B2%D1%80%D0%B5%D0%B8" title="Евреи">евреев</a>, православных христиан<sup class="ts-fix-template noprint">[<i><a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%A1%D1%81%D1%8B%D0%BB%D0%BA%D0%B8_%D0%BD%D0%B0_%D0%B8%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8" title="Википедия:Ссылки на источники"><span title="нет источника (2 декабря 2022)">источник не указан 443 дня</span></a></i>]</sup> и превратили Иерусалим в настоящую бойню<sup class="reference" id="cite_ref-Рамбо2_15-2"><a href="#cite_note-Рамбо2-15">[15]</a></sup>. Некоторые из горожан попытались укрыться на крыше храма (хроника утверждает, что это произошло в <a href="/wiki/%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D1%81%D0%BA%D0%B8%D0%B9_%D1%85%D1%80%D0%B0%D0%BC" title="Иерусалимский храм">храме Соломона</a>, но, так как он был разрушен в <a href="/wiki/I_%D0%B2%D0%B5%D0%BA" title="I век">I веке</a> при взятии Иерусалима полководцем <a class="mw-redirect" href="/wiki/%D0%A2%D0%B8%D1%82_%D0%A4%D0%BB%D0%B0%D0%B2%D0%B8%D0%B9_%D0%92%D0%B5%D1%81%D0%BF%D0%B0%D1%81%D0%B8%D0%B0%D0%BD" title="Тит Флавий Веспасиан">Титом</a>, вероятно имеется в виду <a href="/wiki/%D0%9C%D0%B5%D1%87%D0%B5%D1%82%D1%8C" title="Мечеть">мечеть</a> <a href="/wiki/%D0%90%D0%BB%D1%8C-%D0%90%D0%BA%D1%81%D0%B0" title="Аль-Акса">Аль-Акса</a> на <a href="/wiki/%D0%A5%D1%80%D0%B0%D0%BC%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B3%D0%BE%D1%80%D0%B0" title="Храмовая гора">Храмовой горе</a>, которая, по мнению крестоносцев, располагалась на месте Храма). Сначала Танкред Тарентский и <a href="/wiki/%D0%93%D0%B0%D1%81%D1%82%D0%BE%D0%BD_IV_(%D0%B2%D0%B8%D0%BA%D0%BE%D0%BD%D1%82_%D0%91%D0%B5%D0%B0%D1%80%D0%BD%D0%B0)" title="Гастон IV (виконт Беарна)">Гастон Беарнский</a> взяли их под свою защиту, передав в качестве охранного символа свои знамёна, однако к утру крестоносцы перебили всех уцелевших. Была сожжена <a href="/wiki/%D0%A1%D0%B8%D0%BD%D0%B0%D0%B3%D0%BE%D0%B3%D0%B0" title="Синагога">синагога</a> вместе с находившимися там людьми<sup class="reference" id="cite_ref-Гибб_19-1"><a href="#cite_note-Гибб-19">[19]</a></sup>. Так, к утру <a href="/wiki/16_%D0%B8%D1%8E%D0%BB%D1%8F" title="16 июля">16 июля</a> практически всё население Иерусалима было перебито. По свидетельству западных хронистов было уничтожено около 10 тысяч жителей, арабские источники приводят значения в несколько раз больше<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup>.
</p><p>О суровой жестокости христиан, сообщается во многих хрониках и документах того времени. И если арабские и византийские источники довольно сухо сообщают о большом числе жертв<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup><sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup>, то западные изобилуют подробностями:
</p><p>Раймунд Ажильский, наблюдавший падение Иерусалима своими глазами, сообщает, что на городских улицах можно было увидеть кучи человеческих тел<sup class="reference" id="cite_ref-Раймунд_Ажильский_2-3"><a href="#cite_note-Раймунд_Ажильский-2">[2]</a></sup>.
</p><p>Раймунду вторит Гийом Тирский:
</p>
<div class="ts-Цитата"><blockquote><p>Невозможно было смотреть без ужаса, как валялись всюду тела убитых и разбросанные части тела и как вся земля была залита кровью. И не только обезображенные трупы и отрубленные головы представляли страшное зрелище, но ещё более приводило в содрогание то, что сами победители с головы до пят были в крови и наводили ужас на всякого встречного. В черте храма, говорят, погибло около 10 тысяч врагов, не считая тех, что были убиты повсюду в городе и устилали улицы и площади; число их, говорят, было не меньше. Остальные части войска разошлись по городу и, выволакивая, как скот, из узких и отдаленных переулков несчастных, которые хотели укрыться там от смерти, убивали их секирами.
Другие, разделившись на отряды, врывались в дома и хватали отцов семейств с женами, детьми и всеми домочадцами и закалывали их мечами или сбрасывали с каких-либо возвышенных мест на землю, так что они погибали, разбившись. При этом каждый ворвавшись в дом, обращал его в свою собственность со всем, что находилось в нём, ибо ещё до взятия города было согласовано между крестоносцами, что по завоевании каждый сможет владеть на вечные времена по праву собственности, всем, что ему удастся захватить. Потому они особенно тщательно осматривали город и  убивали сопротивляющихся. Они проникали в самые уединенные и тайные убежища, вламывались в дома жителей, и каждый христианский рыцарь, вешал на дверях дома щит или какое-либо другое оружие, в знак для приближающегося — не останавливаться здесь, а проходить мимо, ибо место это уже занято другими<sup class="reference" id="cite_ref-tir_18-1"><a href="#cite_note-tir-18">[18]</a></sup>.</p></blockquote></div>
<p>В то же время, некоторые христианские и даже арабские хроники утверждают, что отряды Раймонда Тулузского, не только не участвовали в избиении жителей, но напротив отпустили сдавшийся им гарнизон башни Давида<sup class="reference" id="cite_ref-Жан_Ришар_14-2"><a href="#cite_note-Жан_Ришар-14">[14]</a></sup>. Помимо уничтожения жителей, крестоносцы полностью разграбили город. Они врывались в дома и храмы, присваивая себе все ценности, какие только могли найти. <a href="/wiki/%D0%A2%D0%B0%D0%BD%D0%BA%D1%80%D0%B5%D0%B4_%D0%A2%D0%B0%D1%80%D0%B5%D0%BD%D1%82%D1%81%D0%BA%D0%B8%D0%B9" title="Танкред Тарентский">Танкред Тарентский</a> вынес из храма Соломона «несметное количество золота, серебра и драгоценных камней». Правда, по утверждению Гийома Тирского, через некоторое время он вернул награбленное<sup class="reference" id="cite_ref-tir_18-2"><a href="#cite_note-tir-18">[18]</a></sup>.
</p><p>По мнению французского исследователя Ж. Ришара, причиной для бойни послужили оскорбления крестоносцев, которые жители города выкрикивали во время Крестного шествия вокруг <a href="/wiki/%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC" title="Иерусалим">Иерусалима</a><sup class="reference" id="cite_ref-Жан_Ришар_14-3"><a href="#cite_note-Жан_Ришар-14">[14]</a></sup>. Советский историк М. А. Заборов, в дополнение к этому считает причиной религиозный фанатизм крестоносцев, тяжелые потери соратников в пути и боях и сам фактор ожесточения в ходе осады и всего похода<sup class="reference" id="cite_ref-Заборов_1-4"><a href="#cite_note-Заборов-1">[1]</a></sup>.
</p>
<h2><span id=".D0.9F.D0.BE.D1.81.D0.BB.D0.B5_.D0.BF.D0.BE.D0.B1.D0.B5.D0.B4.D1.8B"></span><span class="mw-headline" id="После_победы">После победы</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D0%B0_(1099)&amp;veaction=edit&amp;section=6" title="Редактировать раздел «После победы»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D0%B0_(1099)&amp;action=edit&amp;section=6" title="Редактировать код раздела «После победы»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Gustave_dore_crusades_the_discovery_of_the_true_cross.jpg"><img class="mw-file-element" data-file-height="1210" data-file-width="952" decoding="async" height="381" src="//upload.wikimedia.org/wikipedia/commons/thumb/2/25/Gustave_dore_crusades_the_discovery_of_the_true_cross.jpg/300px-Gustave_dore_crusades_the_discovery_of_the_true_cross.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/2/25/Gustave_dore_crusades_the_discovery_of_the_true_cross.jpg/450px-Gustave_dore_crusades_the_discovery_of_the_true_cross.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/2/25/Gustave_dore_crusades_the_discovery_of_the_true_cross.jpg/600px-Gustave_dore_crusades_the_discovery_of_the_true_cross.jpg 2x" width="300"/></a><figcaption>Крестоносцы у найденной священной реликвии — креста, на котором распяли Иисуса</figcaption></figure>
<p>Взятие Иерусалима 15 июля 1099 года отмечено во всех исторических источниках начала <a href="/wiki/XII_%D0%B2%D0%B5%D0%BA" title="XII век">XII века</a>, в том числе и в русской <a href="/wiki/%D0%9F%D0%BE%D0%B2%D0%B5%D1%81%D1%82%D1%8C_%D0%B2%D1%80%D0%B5%D0%BC%D0%B5%D0%BD%D0%BD%D1%8B%D1%85_%D0%BB%D0%B5%D1%82" title="Повесть временных лет">«Повести временных лет»</a><sup class="reference" id="cite_ref-Заборов_1-5"><a href="#cite_note-Заборов-1">[1]</a></sup>. После падения города правителем новообразованного <a href="/wiki/%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D1%81%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D0%B5%D0%B2%D1%81%D1%82%D0%B2%D0%BE" title="Иерусалимское королевство">Иерусалимского королевства</a> стал Готфрид Бульонский. Готфрид не захотел называться королём в городе, где <a href="/wiki/%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%BE%D1%81" title="Христос">Христос</a> был коронован терновым венцом, поэтому <a href="/wiki/22_%D0%B8%D1%8E%D0%BB%D1%8F" title="22 июля">22 июля</a> <a href="/wiki/1099_%D0%B3%D0%BE%D0%B4" title="1099 год">1099 года</a> принял титул <b>Защитника Гроба Господня</b> (<a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Латинский язык">лат.</a> <span lang="la" style="font-style:italic;">Advocatus Sancti Sepulchri</span>). <a href="/wiki/1_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="1 августа">1 августа</a> был избран первый <a class="mw-redirect" href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B0%D1%82%D1%80%D0%B8%D0%B0%D1%80%D1%85_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D0%B0" title="Латинский патриарх Иерусалима">латинский патриарх Иерусалима</a>. Им стал <a href="/wiki/%D0%90%D1%80%D0%BD%D1%83%D0%BB%D1%8C%D1%84_%D0%B4%D0%B5_%D0%A0%D0%BE%D0%BE%D0%BB" title="Арнульф де Роол">Арнульф Шокесский</a>, <a href="/wiki/%D0%9A%D0%B0%D0%BF%D0%B5%D0%BB%D0%BB%D0%B0%D0%BD" title="Капеллан">капеллан</a> Роберта Нормандского<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup>. <a href="/wiki/5_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="5 августа">5 августа</a>, допросив нескольких чудом уцелевших горожан, Арнульф узнал местонахождение священной реликвии — <a class="mw-redirect" href="/wiki/%D0%96%D0%B8%D0%B2%D0%BE%D1%82%D0%B2%D0%BE%D1%80%D1%8F%D1%89%D0%B5%D0%B5_%D0%94%D1%80%D0%B5%D0%B2%D0%BE" title="Животворящее Древо">Животворящего Креста</a>, на котором распяли Иисуса, что вызвало новый религиозный подъём<sup class="reference" id="cite_ref-Егер_20-1"><a href="#cite_note-Егер-20">[20]</a></sup>.
</p><p>В начале августа Готфрид возглавил поход против приближающейся египетской армии <a class="new" href="/w/index.php?title=%D0%90%D0%BB-%D0%90%D1%84%D0%B4%D0%B0%D0%BB&amp;action=edit&amp;redlink=1" title="Ал-Афдал (страница отсутствует)">ал-Афдала</a> и <a href="/wiki/12_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="12 августа">12 августа</a> разбил мусульман <a href="/wiki/%D0%91%D0%B8%D1%82%D0%B2%D0%B0_%D0%BF%D1%80%D0%B8_%D0%90%D1%81%D0%BA%D0%B0%D0%BB%D0%BE%D0%BD%D0%B5" title="Битва при Аскалоне">при Аскалоне</a>. После этой победы угроза Иерусалиму была ликвидирована и воины Христовы сочли свой долг выполненным, большинство из них вернулось на родину<sup class="reference" id="cite_ref-Харпер_12-2"><a href="#cite_note-Харпер-12">[12]</a></sup>. Первый крестовый поход увенчался успехом, в результате чего на Востоке образовались несколько <a href="/wiki/%D0%93%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B0_%D0%BA%D1%80%D0%B5%D1%81%D1%82%D0%BE%D0%BD%D0%BE%D1%81%D1%86%D0%B5%D0%B2" title="Государства крестоносцев">государств крестоносцев</a>. Эти государства представляли собой плацдарм «Западного мира» во враждебном окружении и требовали постоянной помощи извне, что делало неизбежным последующие крестовые походы<sup class="reference" id="cite_ref-Жан_Ришар_14-4"><a href="#cite_note-Жан_Ришар-14">[14]</a></sup>. Опустошение целых территорий <a href="/wiki/%D0%91%D0%BB%D0%B8%D0%B6%D0%BD%D0%B8%D0%B9_%D0%92%D0%BE%D1%81%D1%82%D0%BE%D0%BA" title="Ближний Восток">Ближнего Востока</a> в ходе крестового похода, а в особенности резня устроенная после захвата Иерусалима, обусловила многовековую конфронтацию католического Запада с мусульманским Востоком<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup>.
</p>
<h2><span id=".D0.9F.D1.80.D0.B8.D0.BC.D0.B5.D1.87.D0.B0.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="Примечания">Примечания</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D0%B0_(1099)&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Примечания»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC%D0%B0_(1099)&amp;action=edit&amp;section=7" title="Редактировать код раздела «Примечания»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
