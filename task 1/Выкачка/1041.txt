<p><b>Лактоферрин</b> — полифункциональный <a href="/wiki/%D0%91%D0%B5%D0%BB%D0%BA%D0%B8" title="Белки">белок</a> из семейства <a class="mw-redirect" href="/wiki/%D0%A2%D1%80%D0%B0%D0%BD%D1%81%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD%D1%8B" title="Трансферрины">трансферринов</a>. Лактоферрин является <a href="/wiki/%D0%93%D0%BB%D0%BE%D0%B1%D1%83%D0%BB%D1%8F%D1%80%D0%BD%D1%8B%D0%B5_%D0%B1%D0%B5%D0%BB%D0%BA%D0%B8" title="Глобулярные белки">глобулярным</a> <a class="mw-redirect" href="/wiki/%D0%93%D0%BB%D0%B8%D0%BA%D0%BE%D0%BF%D1%80%D0%BE%D1%82%D0%B5%D0%B8%D0%BD" title="Гликопротеин">гликопротеином</a> с молекулярной массой около 80 кДа и широко представлен в различных секреторных жидкостях, таких как <a href="/wiki/%D0%9C%D0%BE%D0%BB%D0%BE%D0%BA%D0%BE" title="Молоко">молоко</a>, <a href="/wiki/%D0%A1%D0%BB%D1%8E%D0%BD%D0%B0" title="Слюна">слюна</a>, <a href="/wiki/%D0%A1%D0%BB%D0%B5%D0%B7%D0%B0" title="Слеза">слёзы</a>, секреты носовых желез.
</p><p>Лактоферрин является одним из компонентов <a href="/wiki/%D0%98%D0%BC%D0%BC%D1%83%D0%BD%D0%BD%D0%B0%D1%8F_%D1%81%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0" title="Иммунная система">иммунной системы</a> организма, принимает участие в системе неспецифического гуморального <a href="/wiki/%D0%98%D0%BC%D0%BC%D1%83%D0%BD%D0%B8%D1%82%D0%B5%D1%82" title="Иммунитет">иммунитета</a>, регулирует функции иммунокомпетентных клеток и является белком острой фазы <a href="/wiki/%D0%92%D0%BE%D1%81%D0%BF%D0%B0%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5" title="Воспаление">воспаления</a>.
</p><p>Белок взаимодействует с <a class="mw-redirect" href="/wiki/%D0%94%D0%9D%D0%9A" title="ДНК">ДНК</a> и <a class="mw-redirect" href="/wiki/%D0%A0%D0%9D%D0%9A" title="РНК">РНК</a>, <a href="/wiki/%D0%9F%D0%BE%D0%BB%D0%B8%D1%81%D0%B0%D1%85%D0%B0%D1%80%D0%B8%D0%B4%D1%8B" title="Полисахариды">полисахаридами</a>, <a href="/wiki/%D0%93%D0%B5%D0%BF%D0%B0%D1%80%D0%B8%D0%BD" title="Гепарин">гепарином</a>, причем некоторые из своих биологических функций лактоферрин проявляет в виде комплексов с этими <a href="/wiki/%D0%9B%D0%B8%D0%B3%D0%B0%D0%BD%D0%B4_(%D0%B1%D0%B8%D0%BE%D1%85%D0%B8%D0%BC%D0%B8%D1%8F)" title="Лиганд (биохимия)">лигандами</a>.
</p>

<h2><span id=".D0.A1.D1.82.D1.80.D0.BE.D0.B5.D0.BD.D0.B8.D0.B5_.D0.B8_.D1.84.D0.B8.D0.B7.D0.B8.D0.BA.D0.BE-.D1.85.D0.B8.D0.BC.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.B8.D0.B5_.D1.81.D0.B2.D0.BE.D0.B9.D1.81.D1.82.D0.B2.D0.B0"></span><span class="mw-headline" id="Строение_и_физико-химические_свойства">Строение и физико-химические свойства</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Строение и физико-химические свойства»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;action=edit&amp;section=1" title="Редактировать код раздела «Строение и физико-химические свойства»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id=".D0.A1.D1.82.D1.80.D0.BE.D0.B5.D0.BD.D0.B8.D0.B5_.D0.BC.D0.BE.D0.BB.D0.B5.D0.BA.D1.83.D0.BB.D1.8B_.D0.BB.D0.B0.D0.BA.D1.82.D0.BE.D1.84.D0.B5.D1.80.D1.80.D0.B8.D0.BD.D0.B0"></span><span class="mw-headline" id="Строение_молекулы_лактоферрина">Строение молекулы лактоферрина</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Строение молекулы лактоферрина»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;action=edit&amp;section=2" title="Редактировать код раздела «Строение молекулы лактоферрина»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Лактоферрин принадлежит к семейству белков-трансферринов, осуществляющих перенос <a href="/wiki/%D0%96%D0%B5%D0%BB%D0%B5%D0%B7%D0%BE" title="Железо">железа</a> в клетки и контролирующих уровень свободного железа в крови и во внешних секретах. Лактоферрин представлен в <a href="/wiki/%D0%96%D0%B5%D0%BD%D1%81%D0%BA%D0%BE%D0%B5_%D0%BC%D0%BE%D0%BB%D0%BE%D0%BA%D0%BE" title="Женское молоко">молоке человека</a> и других млекопитающих.<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>
</p><p>Было показано, что помимо молока лактоферрин содержится в <a href="/wiki/%D0%9F%D0%BB%D0%B0%D0%B7%D0%BC%D0%B0_%D0%BA%D1%80%D0%BE%D0%B2%D0%B8" title="Плазма крови">плазме</a> крови, <a class="mw-redirect" href="/wiki/%D0%9D%D0%B5%D0%B9%D1%82%D1%80%D0%BE%D1%84%D0%B8%D0%BB" title="Нейтрофил">нейтрофилах</a> и является одним из основных белков практически всех экзокринных секретов млекопитающих, таких как <a href="/wiki/%D0%A1%D0%BB%D1%8E%D0%BD%D0%B0" title="Слюна">слюна</a>, <a class="mw-redirect" href="/wiki/%D0%96%D0%B5%D0%BB%D1%87%D1%8C" title="Желчь">желчь</a>, <a href="/wiki/%D0%A1%D0%BB%D0%B5%D0%B7%D0%B0" title="Слеза">слёзы</a>, секрет <a href="/wiki/%D0%9F%D0%BE%D0%B4%D0%B6%D0%B5%D0%BB%D1%83%D0%B4%D0%BE%D1%87%D0%BD%D0%B0%D1%8F_%D0%B6%D0%B5%D0%BB%D0%B5%D0%B7%D0%B0" title="Поджелудочная железа">поджелудочной железы</a><sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>.
</p><p>Концентрация лактоферрина в молоке изменяется от 7 мг/мл в <a href="/wiki/%D0%9C%D0%BE%D0%BB%D0%BE%D0%B7%D0%B8%D0%B2%D0%BE" title="Молозиво">молозиве</a> до 1 мг/мл в зрелом молоке.
</p><p>Согласно данным <a href="/wiki/%D0%A0%D0%B5%D0%BD%D1%82%D0%B3%D0%B5%D0%BD%D0%BE%D1%81%D1%82%D1%80%D1%83%D0%BA%D1%82%D1%83%D1%80%D0%BD%D1%8B%D0%B9_%D0%B0%D0%BD%D0%B0%D0%BB%D0%B8%D0%B7" title="Рентгеноструктурный анализ">рентгеноструктурного анализа</a> белок образован одной <a class="mw-redirect" href="/wiki/%D0%9F%D0%BE%D0%BB%D0%B8%D0%BF%D0%B5%D0%BF%D1%82%D0%B8%D0%B4" title="Полипептид">полипептидной</a> цепью, которая содержит 692 аминокислотных остатка и образует два гомологичных глобулярных <a href="/wiki/%D0%94%D0%BE%D0%BC%D0%B5%D0%BD_%D0%B1%D0%B5%D0%BB%D0%BA%D0%B0" title="Домен белка">домена</a>, называемых N- и С-долями (N-доля, остатки 1–333; C-доля, остатки 345–692), концы которых соединены короткой α-спиралью<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup><sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>.
</p><p>Каждая доля состоит из двух доменов N1, N2 и C1, C2 и содержит один железосвязывающий сайт и один сайт <a href="/wiki/%D0%93%D0%BB%D0%B8%D0%BA%D0%BE%D0%B7%D0%B8%D0%BB%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5" title="Гликозилирование">гликозилирования</a>. Степень гликозилирования белка может быть различной, именно поэтому молекулярная масса белка по разным данным составляет от 76 до 80 кДа. Было показано, что устойчивость лактоферрина к деградации <a href="/wiki/%D0%9F%D1%80%D0%BE%D1%82%D0%B5%D0%B0%D0%B7%D0%B0" title="Протеаза">протеазами</a> или при низких значениях <a class="mw-redirect" href="/wiki/%D0%A0%D0%9D" title="РН">рН</a> обусловлена высокой степенью гликозилирования белка<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>.
</p><p>Лактоферрин относят к щелочным белкам, его <a href="/wiki/%D0%98%D0%B7%D0%BE%D1%8D%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%82%D0%BE%D1%87%D0%BA%D0%B0" title="Изоэлектрическая точка">изоэлектрическая точка</a> составляет 8.7. Белок существует в двух формах – железонасыщенной (холо-ЛФ) и железоненасыщенной (апо-ЛФ), их третичные структуры различны: для апо-ЛФ характерна  “открытая” конформация N-доли и “закрытая” конформация С-доли, а для холо-ЛФ характерна закрытая конформация обеих долей<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>.
</p><p>Каждая молекула белка может обратимо связывать два иона трехвалентного <a href="/wiki/%D0%96%D0%B5%D0%BB%D0%B5%D0%B7%D0%BE" title="Железо">железа</a> или ионы <a href="/wiki/%D0%A6%D0%B8%D0%BD%D0%BA" title="Цинк">цинка</a>, <a href="/wiki/%D0%9C%D0%B5%D0%B4%D1%8C" title="Медь">меди</a> и других металлов<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup>.
</p><p>Центры связывания локализованы в каждой из двух белковых глобул, составляющих молекулу лактоферрина. В каждой доле атом железа скоординирован с шестью лигандами, четыре из которых предоставлены цепью полипептида (два остатка <a href="/wiki/%D0%A2%D0%B8%D1%80%D0%BE%D0%B7%D0%B8%D0%BD" title="Тирозин">тирозина</a>, один <a class="mw-redirect" href="/wiki/%D0%9E%D1%81%D1%82%D0%B0%D1%82%D0%BE%D0%BA_(%D1%85%D0%B8%D0%BC%D0%B8%D1%8F)" title="Остаток (химия)">остаток</a> <a href="/wiki/%D0%93%D0%B8%D1%81%D1%82%D0%B8%D0%B4%D0%B8%D0%BD" title="Гистидин">гистидина</a> и один остаток <a href="/wiki/%D0%90%D1%81%D0%BF%D0%B0%D1%80%D0%B0%D0%B3%D0%B8%D0%BD%D0%BE%D0%B2%D0%B0%D1%8F_%D0%BA%D0%B8%D1%81%D0%BB%D0%BE%D1%82%D0%B0" title="Аспарагиновая кислота">аспарагиновой кислоты</a>), а оставшиеся две связи железо образует с ионом <a class="mw-redirect" href="/wiki/%D0%9A%D0%B0%D1%80%D0%B1%D0%BE%D0%BD%D0%B0%D1%82" title="Карбонат">карбоната</a> или <a class="new" href="/w/index.php?title=%D0%91%D0%B8%D0%BA%D0%B0%D1%80%D0%B1%D0%BE%D0%BD%D0%B0%D1%82&amp;action=edit&amp;redlink=1" title="Бикарбонат (страница отсутствует)">бикарбоната</a>.
</p><p>Лактоферрин образует с железом комплекс красноватого цвета, его сродство к железу в 300 раз выше, чем сродство <a href="/wiki/%D0%A2%D1%80%D0%B0%D0%BD%D1%81%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD" title="Трансферрин">трансферрина</a><sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>.
</p><p>Кроме того, показано, что в слабокислой среде сродство белка к железу повышается, что облегчает переход металла с трансферрина на лактоферрин при <a href="/wiki/%D0%92%D0%BE%D1%81%D0%BF%D0%B0%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5" title="Воспаление">воспалении</a>, когда рН тканей снижается за счет <a href="/wiki/%D0%9C%D0%BE%D0%BB%D0%BE%D1%87%D0%BD%D0%B0%D1%8F_%D0%BA%D0%B8%D1%81%D0%BB%D0%BE%D1%82%D0%B0" title="Молочная кислота">молочной</a> и других кислот<sup class="reference" id="cite_ref-Sousa_13-0"><a href="#cite_note-Sousa-13">[13]</a></sup>.
</p><p>Степень насыщения железом лактоферрина <a href="/wiki/%D0%96%D0%B5%D0%BD%D1%81%D0%BA%D0%BE%D0%B5_%D0%BC%D0%BE%D0%BB%D0%BE%D0%BA%D0%BE" title="Женское молоко">женского молока</a> составляет по оценкам разных авторов от 10 до 30 %. Показано, что белок участвует не только в транспорте ионов железа, цинка и меди, но и в регуляции их всасывания<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup>.
</p><p>Наличие непрочно связанных ионов цинка и меди не влияет на железосвязывающую функцию лактоферрина, и фактически даже усиливает её.
</p>
<h3><span id=".D0.9F.D0.BE.D0.BB.D0.B8.D0.BC.D0.B5.D1.80.D0.BD.D1.8B.D0.B5_.D1.84.D0.BE.D1.80.D0.BC.D1.8B"></span><span class="mw-headline" id="Полимерные_формы">Полимерные формы</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Полимерные формы»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;action=edit&amp;section=3" title="Редактировать код раздела «Полимерные формы»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Как в плазме крови, так и в секреторных жидкостях лактоферрин может существовать в виде различных <a class="mw-redirect" href="/wiki/%D0%9F%D0%BE%D0%BB%D0%B8%D0%BC%D0%B5%D1%80" title="Полимер">полимерных</a> форм от <a href="/wiki/%D0%9C%D0%BE%D0%BD%D0%BE%D0%BC%D0%B5%D1%80" title="Мономер">мономера</a> до <a href="/wiki/%D0%A2%D0%B5%D1%82%D1%80%D0%B0%D0%BC%D0%B5%D1%80" title="Тетрамер">тетрамера</a>. Показано, что белок обнаруживает резко выраженную тенденцию к полимеризации <i>in vitro</i> и <i>in vivo</i>, и при высоких концентрациях преобладают полимерные формы лактоферрина.<sup class="reference" id="cite_ref-Sousa_13-1"><a href="#cite_note-Sousa-13">[13]</a></sup>
</p><p>Кроме того, рядом авторов было обнаружено, что доминирующей формой лактоферрина в <a href="/wiki/%D0%A4%D0%B8%D0%B7%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B5_%D1%83%D1%81%D0%BB%D0%BE%D0%B2%D0%B8%D1%8F" title="Физиологические условия">физиологических условиях</a> является тетрамер; соотношение мономер : тетрамер при концентрации белка 10<sup>−5</sup> М составляет 1 : 4.<sup class="reference" id="cite_ref-Bennett_15-0"><a href="#cite_note-Bennett-15">[15]</a></sup><sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup><sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup>
</p><p>Существует предположение, что <a href="/wiki/%D0%9E%D0%BB%D0%B8%D0%B3%D0%BE%D0%BC%D0%B5%D1%80" title="Олигомер">олигомерное</a> состояние лактоферрина определяется концентрацией данного белка в среде, кроме того, <a href="/wiki/%D0%9F%D0%BE%D0%BB%D0%B8%D0%BC%D0%B5%D1%80%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F" title="Полимеризация">полимеризация</a> лактоферрина строго зависит от присутствия ионов Ca<sup>2+</sup>. В присутствии ионов <a href="/wiki/%D0%9A%D0%B0%D0%BB%D1%8C%D1%86%D0%B8%D0%B9" title="Кальций">кальция</a> и при концентрации белка менее 10<sup>−10</sup>-10<sup>−11</sup> M наблюдали преобладание мономерной формы белка. При концентрациях лактоферрина более 10<sup>−9</sup>-10<sup>−10</sup> M происходил переход в тетрамерную форму<sup class="reference" id="cite_ref-Furmanski_18-0"><a href="#cite_note-Furmanski-18">[18]</a></sup><sup class="reference" id="cite_ref-Bennett_15-1"><a href="#cite_note-Bennett-15">[15]</a></sup>.
</p><p><a class="mw-redirect" href="/wiki/%D0%A2%D0%B8%D1%82%D1%80_%D1%80%D0%B0%D1%81%D1%82%D0%B2%D0%BE%D1%80%D0%B0" title="Титр раствора">Титр</a> лактоферрина в крови соответствует величине именно этой “переходной концентрации” и, таким образом, лактоферрин в крови должен быть представлен как в виде мономера, так и тетрамера.
</p><p>Также было показано, что ряд функциональных свойств лактоферрина определяется олигомерным состоянием молекулы белка. Так, лактоферрин в виде мономера способен к прочному связыванию с ДНК и регуляции процессов <a href="/wiki/%D0%93%D1%80%D0%B0%D0%BD%D1%83%D0%BB%D0%BE%D0%BF%D0%BE%D1%8D%D0%B7" title="Гранулопоэз">гранулопоэза</a>, а тетрамерная форма не связывает ДНК. Есть все основания полагать, что в организации олигомерного состояния белка участвуют слабые взаимодействия — преимущественно <a href="/wiki/%D0%93%D0%B8%D0%B4%D1%80%D0%BE%D1%84%D0%BE%D0%B1%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Гидрофобность">гидрофобные</a> и электростатические контакты боковых групп остатков аминокислот молекулы лактоферрина и, возможно, <a class="mw-redirect" href="/wiki/%D0%93%D0%BB%D0%B8%D0%BA%D0%BE%D0%B7%D0%B8%D0%B4" title="Гликозид">гликозидных</a> остатков белка.
</p>
<h2><span id=".D0.91.D0.B8.D0.BE.D0.BB.D0.BE.D0.B3.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.B8.D0.B5_.D1.84.D1.83.D0.BD.D0.BA.D1.86.D0.B8.D0.B8"></span><span class="mw-headline" id="Биологические_функции">Биологические функции</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Биологические функции»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;action=edit&amp;section=4" title="Редактировать код раздела «Биологические функции»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Лактоферрин относят к системе врожденного <a href="/wiki/%D0%98%D0%BC%D0%BC%D1%83%D0%BD%D0%B8%D1%82%D0%B5%D1%82" title="Иммунитет">иммунитета</a>, существуют данные о том, что лактоферрин опосредованно вовлечен в процессы клеточного иммунитета. Главные биологические функции белка — это связывание и транспорт ионов железа, но кроме этого лактоферрин обладает антибактериальной, антивирусной, антипаразитарной, различными каталитическими активностями, а также противораковым, антиаллергическим, иммуномодулирующим действиями и радиопротективными свойствами<sup class="ts-fix-template noprint">[<i><a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%A1%D1%81%D1%8B%D0%BB%D0%BA%D0%B8_%D0%BD%D0%B0_%D0%B8%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8" title="Википедия:Ссылки на источники"><span title="нет источника (28 марта 2019)">источник не указан 1798 дней</span></a></i>]</sup>.
</p>
<h3><span id=".D0.90.D0.BD.D1.82.D0.B8.D0.B1.D0.B0.D0.BA.D1.82.D0.B5.D1.80.D0.B8.D0.B0.D0.BB.D1.8C.D0.BD.D0.B0.D1.8F_.D0.B0.D0.BA.D1.82.D0.B8.D0.B2.D0.BD.D0.BE.D1.81.D1.82.D1.8C"></span><span class="mw-headline" id="Антибактериальная_активность">Антибактериальная активность</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Антибактериальная активность»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;action=edit&amp;section=5" title="Редактировать код раздела «Антибактериальная активность»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Наиболее изученным является механизм антибактериальной активности лактоферрина. Антибактериальные свойства белка обусловлены способностью лактоферрина связывать железо и тем самым лишать бактериальную микрофлору необходимого для её роста и жизнедеятельности микроэлемента<sup class="reference" id="cite_ref-Farnaud_19-0"><a href="#cite_note-Farnaud-19">[19]</a></sup>.
Бактерицидные свойства белка также обусловлены наличием специфических лактоферриновых <a href="/wiki/%D0%A0%D0%B5%D1%86%D0%B5%D0%BF%D1%82%D0%BE%D1%80" title="Рецептор">рецепторов</a> на клеточной поверхности микроорганизмов. Показано, что лактоферрин связывает липополисахариды (ЛПС) бактериальных стенок, а входящая в состав белка окисленная форма железа инициирует их <a class="new" href="/w/index.php?title=%D0%9F%D0%B5%D1%80%D0%B5%D0%BA%D0%B8%D1%81%D0%BD%D0%BE%D0%B5_%D0%BE%D0%BA%D0%B8%D1%81%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5&amp;action=edit&amp;redlink=1" title="Перекисное окисление (страница отсутствует)">перекисное окисление</a>. Это приводит к изменению мембранной проницаемости и последующему <a href="/wiki/%D0%9B%D0%B8%D0%B7%D0%B8%D1%81" title="Лизис">лизису</a> клеток<sup class="reference" id="cite_ref-Farnaud_19-1"><a href="#cite_note-Farnaud-19">[19]</a></sup>.
</p><p>Все вышеперечисленные механизмы антибактериального действия лактоферрина напрямую зависят от железосвязывающих свойств белка. Вместе с тем в основе антиинфекционной активности лактоферрина могут лежать и другие механизмы, не зависящие от способности белка связывать ионы железа, например стимулирующее действие лактоферрина на фагоцитоз и влияние на активность <a href="/wiki/%D0%A1%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0_%D0%BA%D0%BE%D0%BC%D0%BF%D0%BB%D0%B5%D0%BC%D0%B5%D0%BD%D1%82%D0%B0" title="Система комплемента">комплемента</a><sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup>.
</p><p>Однако наиболее изученным механизмом антибактериального действия белка, не зависящим от его железосвязывающей способности, является специфическое взаимодействие лактоферрина с наружной бактериальной мембраной, которое приводит к гибели клеток бактерий<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup>.
</p><p>Было показано, что белок разрушает бактериальную мембрану и даже проникает внутрь клетки. За специфическое связывание с бактериальной стенкой ответственен лактоферрицин — <a class="mw-redirect" href="/wiki/%D0%9F%D0%B5%D0%BF%D1%82%D0%B8%D0%B4" title="Пептид">пептид</a>, расположенный на N-конце молекулы лактоферрина и получаемый <i>in vitro</i> протеолитическим расщеплением белка <a href="/wiki/%D0%A2%D1%80%D0%B8%D0%BF%D1%81%D0%B8%D0%BD" title="Трипсин">трипсином</a><sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup><sup class="reference" id="cite_ref-Sojar_23-0"><a href="#cite_note-Sojar-23">[23]</a></sup>.
</p>
<h3><span id=".D0.90.D0.BD.D1.82.D0.B8.D0.B2.D0.B8.D1.80.D1.83.D1.81.D0.BD.D0.B0.D1.8F_.D0.B0.D0.BA.D1.82.D0.B8.D0.B2.D0.BD.D0.BE.D1.81.D1.82.D1.8C"></span><span class="mw-headline" id="Антивирусная_активность">Антивирусная активность</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Антивирусная активность»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;action=edit&amp;section=6" title="Редактировать код раздела «Антивирусная активность»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Показано, что лактоферрин обладает антивирусной активностью против широкого спектра <a class="mw-redirect" href="/wiki/%D0%92%D0%B8%D1%80%D1%83%D1%81" title="Вирус">вирусов</a> человека и животных с ДНК и РНК <a href="/wiki/%D0%93%D0%B5%D0%BD%D0%BE%D0%BC" title="Геном">геномами</a><sup class="reference" id="cite_ref-vander_24-0"><a href="#cite_note-vander-24">[24]</a></sup>.
</p><p>Лактоферрин связывает различные антигены вирусной природы преимущественно в условиях <i>in vitro</i>. На данный момент показано действие белка против <a href="/wiki/%D0%92%D0%B8%D1%80%D1%83%D1%81_%D0%BF%D1%80%D0%BE%D1%81%D1%82%D0%BE%D0%B3%D0%BE_%D0%B3%D0%B5%D1%80%D0%BF%D0%B5%D1%81%D0%B0_%D0%B2%D1%82%D0%BE%D1%80%D0%BE%D0%B3%D0%BE_%D1%82%D0%B8%D0%BF%D0%B0" href="#cite_note-Sojar-23">[23]</a></sup>, COVID-19.<sup class="reference" id="cite_ref-32"><a href="#cite_note-32">[32]</a></sup>
</p><p>Наиболее изученным механизмом антивирусной активности лактоферрина является предотвращение попадания вирусных частиц в клетки-мишени. Многие вирусы имеют тенденцию связывать гепаринсульфат-гликозоаминогликаны и <a href="/wiki/%D0%9B%D0%B8%D0%BF%D0%BE%D0%BF%D1%80%D0%BE%D1%82%D0%B5%D0%B8%D0%BD%D1%8B" title="Липопротеины">липопротеины</a> мембран эукариотических клеток. После первоначального контакта с клеткой <a class="mw-redirect" href="/wiki/%D0%92%D0%B8%D1%80%D1%83%D1%81" title="Вирус">вирусная</a> частица взаимодействует со специфическими вирусными рецепторами и проникает в клетку.<sup class="reference" id="cite_ref-Nozaki_30-1"><a href="#cite_note-Nozaki-30">[30]</a></sup>
</p><p>Лактоферрин связывает гепарансульфат-гликозоаминогликаны и липопротеины на поверхности клеток, препятствуя тем самым связыванию с ними вирусных частиц и дальнейшему проникновению вируса в клетку. Интересно, что лактоферрицин, который обеспечивает основные антимикробные свойства лактоферрина, практически не проявляет антивирусной активности, кроме того, апо-ЛФ в большинстве случаев показывает гораздо большее антивирусное действие, чем металлонасыщенный лактоферрин.<sup class="reference" id="cite_ref-vander_24-1"><a href="#cite_note-vander-24">[24]</a></sup>
</p><p>Помимо взаимодействия с клеточными рецепторами, лактоферрин напрямую связывает вирусные частицы и препятствует их проникновению в клетки. Это подтверждается наличием антивирусного действия белка против ротавирусов,<sup class="reference" id="cite_ref-Sojar_23-3"><a href="#cite_note-Sojar-23">[23]</a></sup> для которых клеточными рецепторами являются углеводные остатки, отличающиеся по составу от гликозаминогликанов, было показано взаимодействие лактоферрина с белками оболочки вируса <a href="/wiki/%D0%92%D0%B8%D1%80%D1%83%D1%81%D0%BD%D1%8B%D0%B9_%D0%B3%D0%B5%D0%BF%D0%B0%D1%82%D0%B8%D1%82" title="Вирусный гепатит">гепатита</a>.<sup class="reference" id="cite_ref-Nozaki_30-2"><a href="#cite_note-Nozaki-30">[30]</a></sup> Последним моментом в развитии антивирусной активности белка является ингибирование вирусной репликации уже после попадания вируса в клетку.<sup class="reference" id="cite_ref-Sojar_23-4"><a href="#cite_note-Sojar-23">[23]</a></sup><sup class="reference" id="cite_ref-Puddi_28-1"><a href="#cite_note-Puddi-28">[28]</a></sup> Такое непрямое противовирусное влияние осуществляется за счет регуляции синтеза <a href="/wiki/%D0%95%D1%81%D1%82%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B%D0%B5_%D0%BA%D0%B8%D0%BB%D0%BB%D0%B5%D1%80%D1%8B" title="Естественные киллеры">натуральных киллеров</a>, <a href="/wiki/%D0%93%D1%80%D0%B0%D0%BD%D1%83%D0%BB%D0%BE%D1%86%D0%B8%D1%82%D1%8B" title="Гранулоциты">гранулоцитов</a> и <a href="/wiki/%D0%9C%D0%B0%D0%BA%D1%80%D0%BE%D1%84%D0%B0%D0%B3%D0%B8" title="Макрофаги">макрофагов</a> -– клеток, которые играют решающую роль на ранних стадиях развития <a href="/wiki/%D0%92%D0%B8%D1%80%D1%83%D1%81%D1%8B" title="Вирусы">вирусной</a> инфекции. Методами <a class="mw-redirect" href="/wiki/%D0%9F%D0%A6%D0%A0_%D0%B2_%D1%80%D0%B5%D0%B0%D0%BB%D1%8C%D0%BD%D0%BE%D0%BC_%D0%B2%D1%80%D0%B5%D0%BC%D0%B5%D0%BD%D0%B8" title="ПЦР в реальном времени">ПЦР в реальном времени</a> и при помощи <a href="/wiki/%D0%94%D0%9D%D0%9A-%D0%BC%D0%B8%D0%BA%D1%80%D0%BE%D1%87%D0%B8%D0%BF" title="ДНК-микрочип">ДНК-микрочипов</a> показано повышение <a href="/wiki/%D0%AD%D0%BA%D1%81%D0%BF%D1%80%D0%B5%D1%81%D1%81%D0%B8%D1%8F_%D0%B3%D0%B5%D0%BD%D0%BE%D0%B2" title="Экспрессия генов">экспрессии гена</a> лактоферрина при <a href="/wiki/%D0%A2%D1%8F%D0%B6%D1%91%D0%BB%D1%8B%D0%B9_%D0%BE%D1%81%D1%82%D1%80%D1%8B%D0%B9_%D1%80%D0%B5%D1%81%D0%BF%D0%B8%D1%80%D0%B0%D1%82%D0%BE%D1%80%D0%BD%D1%8B%D0%B9_%D1%81%D0%B8%D0%BD%D0%B4%D1%80%D0%BE%D0%BC" title="Тяжёлый острый респираторный синдром">тяжёлом остром респираторном синдроме</a> (SARS).<sup class="reference" id="cite_ref-33"><a href="#cite_note-33">[33]</a></sup>
</p>
<h3><span id=".D0.90.D0.BD.D1.82.D0.B8.D0.B3.D1.80.D0.B8.D0.B1.D0.BA.D0.BE.D0.B2.D0.B0.D1.8F_.D0.B0.D0.BA.D1.82.D0.B8.D0.B2.D0.BD.D0.BE.D1.81.D1.82.D1.8C"></span><span class="mw-headline" id="Антигрибковая_активность">Антигрибковая активность</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Антигрибковая активность»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;action=edit&amp;section=7" title="Редактировать код раздела «Антигрибковая активность»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Показано, что лактоферрин и лактоферрицин ингибируют <i>in vitro</i> рост <i>Trichophyton mentagrophytes</i>, который вызывает ряд кожных заболеваний, например, <a class="mw-redirect" href="/wiki/%D0%A1%D1%82%D1%80%D0%B8%D0%B3%D1%83%D1%89%D0%B8%D0%B9_%D0%BB%D0%B8%D1%88%D0%B0%D0%B9" title="Стригущий лишай">стригущий лишай</a>.<sup class="reference" id="cite_ref-34"><a href="#cite_note-34">[34]</a></sup> Лактоферрин обладает активностью против <i><a class="mw-redirect" href="/wiki/Candida" title="Candida">Candida</a> albicans</i>—симбионтов, образующих колонии в <a class="mw-redirect" href="/wiki/%D0%A1%D0%BB%D0%B8%D0%B7%D0%B8%D1%81%D1%82%D1%8B%D0%B5_%D0%BE%D0%B1%D0%BE%D0%BB%D0%BE%D1%87%D0%BA%D0%B8" title="Слизистые оболочки">слизистой оболочке</a> <a href="/wiki/%D0%A0%D0%BE%D1%82%D0%BE%D0%B2%D0%B0%D1%8F_%D0%BF%D0%BE%D0%BB%D0%BE%D1%81%D1%82%D1%8C" title="Ротовая полость">ротовой полости</a> здоровых людей.<sup class="reference" id="cite_ref-Lupetti_35-0"><a href="#cite_note-Lupetti-35">[35]</a></sup><sup class="reference" id="cite_ref-Viejo_36-0"><a href="#cite_note-Viejo-36">[36]</a></sup> Тем не менее, этот вид является основным грибковым патогеном у людей с оппортунистическими инфекциями ротовой полости и организма в целом, а также у <a href="/wiki/%D0%98%D0%BC%D0%BC%D1%83%D0%BD%D0%BE%D0%B4%D0%B5%D1%84%D0%B8%D1%86%D0%B8%D1%82" title="Иммунодефицит">иммунодефицитных</a> больных. Долгое время в качестве основного антигрибкового препарата против <i>Candida albicans</i>, использовали <a class="mw-redirect" href="/wiki/%D0%A4%D0%BB%D1%8E%D0%BA%D0%BE%D0%BD%D0%B0%D0%B7%D0%BE%D0%BB" title="Флюконазол">флюконазол</a>, что привело к появлению <a href="/wiki/%D0%A8%D1%82%D0%B0%D0%BC%D0%BC" title="Штамм">штаммов</a>, устойчивых к воздействию этого лекарства. Было показано, что лактоферрин в комплексе с флюконазолом обладает антигрибковой активностью против флюконазол-устойчивых штаммов <i>Candida albicans</i>, также других видов <a class="mw-redirect" href="/wiki/Candida" title="Candida">Candida</a>: <i>C. glabrata</i>,<i> C. krusei</i>, <i>C. parapsilosis</i> и <i>C. tropicalis</i>.<sup class="reference" id="cite_ref-Lupetti_35-1"><a href="#cite_note-Lupetti-35">[35]</a></sup> Антигрибковую активность наблюдают только в случае последовательной инкубации клеток <i>Candida</i> с лактоферрином, а потом с флюконазолом, но не наоборот. Лактоферрицин обладает антигрибковой активностью, превосходящей активность самого лактоферрина, в составе лактоферрицина изучали два пептида, содержащие 1-11 и 17-26 аминокислотные остатки лактоферрина. Синтетически синтезированный пептид 1-11 обладает гораздо большей активностью против <i>Candida albicans</i>, чем нативный ЛФ.<sup class="reference" id="cite_ref-Lupetti_35-2"><a href="#cite_note-Lupetti-35">[35]</a></sup> Кроме того, показано, что пептид, содержащий 4-11 аминокислотные остатки, не вызывает гибели клеток <i>Candida</i>, что указывает на важность 1-4 аминокислот белка в проявления антигрибковой активности.<sup class="reference" id="cite_ref-37"><a href="#cite_note-37">[37]</a></sup> Исследования синтетического пептида 17-26 показали, что он стимулирует образование гидроксильных радикалов мышиными нейтрофилами и в комбинации с антигрибковым препаратом амфотерицином В защищает животных от летальных для них инфекций <i>Candida albicans</i> и <i>Aspergillus fumigatus</i> в гораздо большей степени, чем <a class="mw-redirect" href="/wiki/%D0%90%D0%BC%D1%84%D0%BE%D1%82%D0%B5%D1%80%D0%B8%D1%86%D0%B8%D0%BD_%D0%92" title="Амфотерицин В">амфотерицин В</a> один, причем <i>in vitro</i> антигрибковая активность <a class="mw-redirect" href="/wiki/%D0%9F%D0%B5%D0%BF%D1%82%D0%B8%D0%B4" title="Пептид">пептида</a> была в 10 раз ниже, чем активность амфотерицина В.<sup class="reference" id="cite_ref-38"><a href="#cite_note-38">[38]</a></sup>
</p><p>Оральное введение лактоферрина через питьевую воду мышам с ослабленным иммунитетом и симптомами <a href="/wiki/%D0%A1%D1%82%D0%BE%D0%BC%D0%B0%D1%82%D0%B8%D1%82" title="Стоматит">стоматита</a> приводило к значительному уменьшению числа <i>Candida albicans</i> в ротовой полости и размеров повреждения языка.<sup class="reference" id="cite_ref-39"><a href="#cite_note-39">[39]</a></sup> Показано, что оральное введение лактоферрина уменьшает количество патогенных организмов в тканях, близких к <a href="/wiki/%D0%96%D0%B5%D0%BB%D1%83%D0%B4%D0%BE%D1%87%D0%BD%D0%BE-%D0%BA%D0%B8%D1%88%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D1%82%D1%80%D0%B0%D0%BA%D1%82" title="Желудочно-кишечный тракт">желудочно-кишечному тракту</a> у нескольких моделей инфицированных животных. Более того, <a href="/wiki/%D0%92%D0%98%D0%A7-%D0%B8%D0%BD%D1%84%D0%B5%D0%BA%D1%86%D0%B8%D1%8F" title="ВИЧ-инфекция">ВИЧ-инфицированные</a> больные с <i>Candida albicans</i>, устойчивые к антигрибковым препаратам, полностью избавлялись от грибковой инфекции после введения смеси, содержащей лактоферрин, <a href="/wiki/%D0%9B%D0%B8%D0%B7%D0%BE%D1%86%D0%B8%D0%BC" title="Лизоцим">лизоцим</a> и <a class="new" href="/w/index.php?title=%D0%98%D0%BD%D1%82%D1%80%D0%BE%D0%B0%D0%BA%D0%BE%D0%BD%D0%B0%D0%B7%D0%BE%D0%BB&amp;action=edit&amp;redlink=1" title="Интроаконазол (страница отсутствует)">интроаконазол</a>.<sup class="reference" id="cite_ref-40"><a href="#cite_note-40">[40]</a></sup>
</p><p>В отличие от антивирусного и антибактериального действия лактоферрина, очень мало известно о противогрибковом механизме действия белка. Показано, что антигрибковая активность лактоферрина обеспечивается разрушением <a href="/wiki/%D0%9A%D0%BB%D0%B5%D1%82%D0%BE%D1%87%D0%BD%D0%B0%D1%8F_%D1%81%D1%82%D0%B5%D0%BD%D0%BA%D0%B0" title="Клеточная стенка">клеточной стенки</a> и связыванием белка с <a class="mw-redirect" href="/wiki/%D0%9F%D0%BB%D0%B0%D0%B7%D0%BC%D0%B0%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D0%BC%D0%B5%D0%BC%D0%B1%D1%80%D0%B0%D0%BD%D0%B0" title="Плазматическая мембрана">плазматической мембраной</a> <i>C. albicans</i>. Действие лактоферрина на <i>C. albicans</i> <i>in vitro</i> приводит к изменению мембранного потенциала и закислению цитоплазмы клеток <i>Candida</i>, что говорит о прямом или косвенном взаимодействии лактоферрина с плазматической мембраной.<sup class="reference" id="cite_ref-Viejo_36-1"><a href="#cite_note-Viejo-36">[36]</a></sup>
</p><p>Поскольку лактоферрин позволяет минимизировать концентрации лекарств, при которых происходит эффективное лечение грибковых инфекций, возможно его использование совместно с противогрибковыми препаратами в терапии заболеваний, вызываемых штаммами с лекарственной устойчивостью. Синергическое действие лактоферрина с <a class="mw-redirect" href="/wiki/%D0%90%D0%BD%D1%82%D0%B8%D0%B1%D0%B8%D0%BE%D1%82%D0%B8%D0%BA" title="Антибиотик">антибиотиками</a>, антигрибковыми и антибактериальными препаратами против патогенных микроорганизмов может быть очень эффективным. Кроме того, использование лактоферрина, который является одним из факторов неспецифической защиты в слюне, позволяет уменьшить распространение флюконазол-устойчивых видов <i>Candida</i> среди людей с ослабленной иммунной системой, в частности среди больных СПИДом.<sup class="reference" id="cite_ref-41"><a href="#cite_note-41">[41]</a></sup>
</p>
<h3><span id=".D0.92.D0.B7.D0.B0.D0.B8.D0.BC.D0.BE.D0.B4.D0.B5.D0.B9.D1.81.D1.82.D0.B2.D0.B8.D0.B5_.D1.81_.D0.BD.D1.83.D0.BA.D0.BB.D0.B5.D0.B8.D0.BD.D0.BE.D0.B2.D1.8B.D0.BC.D0.B8_.D0.BA.D0.B8.D1.81.D0.BB.D0.BE.D1.82.D0.B0.D0.BC.D0.B8"></span><span class="mw-headline" id="Взаимодействие_с_нуклеиновыми_кислотами">Взаимодействие с нуклеиновыми кислотами</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;veaction=edit&amp;section=8" title="Редактировать раздел «Взаимодействие с нуклеиновыми кислотами»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;action=edit&amp;section=8" title="Редактировать код раздела «Взаимодействие с нуклеиновыми кислотами»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Взаимодействие белков c ДНК является объектом многочисленных исследований, направленных на рассмотрение механизмов, контролирующих <a href="/wiki/%D0%AD%D0%BA%D1%81%D0%BF%D1%80%D0%B5%D1%81%D1%81%D0%B8%D1%8F_%D0%B3%D0%B5%D0%BD%D0%BE%D0%B2" title="Экспрессия генов">экспрессию генов</a>.<sup class="reference" id="cite_ref-Bennett_15-2"><a href="#cite_note-Bennett-15">[15]</a></sup><sup class="reference" id="cite_ref-42"><a href="#cite_note-42">[42]</a></sup> Одним из важных свойств лактоферрина считают его способность связывать нуклеиновые кислоты. Обнаружено, что фракция белка, выделенного  из  молока, содержит 3.3 % РНК,<sup class="reference" id="cite_ref-Bennett_15-3"><a href="#cite_note-Bennett-15">[15]</a></sup> кроме того, при взаимодействии с одноцепочечной и двуцепочечной ДНК белок предпочтительнее связывает двуцепочечную ДНК. Взаимодействие лактоферрина с <a class="mw-redirect" href="/wiki/%D0%94%D0%9D%D0%9A" title="ДНК">ДНК</a> частично ингибировалось добавлением <a class="mw-redirect" href="/wiki/%D0%98%D0%BC%D0%BC%D1%83%D0%BD%D0%BE%D0%B3%D0%BB%D0%BE%D0%B1%D1%83%D0%BB%D0%B8%D0%BD%D1%8B" title="Иммуноглобулины">антител</a> к ДНК, полученных из плазмы крови больных <a href="/wiki/%D0%A1%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%BD%D0%B0%D1%8F_%D0%BA%D1%80%D0%B0%D1%81%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%BB%D1%87%D0%B0%D0%BD%D0%BA%D0%B0" title="Системная красная волчанка">системной красной волчанкой</a>.
</p><p>Способность лактоферрина связывать ДНК активно используют для выделения и очистки белка с помощью <a href="/wiki/%D0%90%D1%84%D1%84%D0%B8%D0%BD%D0%BD%D0%B0%D1%8F_%D1%85%D1%80%D0%BE%D0%BC%D0%B0%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D1%8F" title="Аффинная хроматография">аффинной хроматографии</a> на колонках с иммобилизованными ДНК-содержащими <a class="mw-redirect" href="/wiki/%D0%A1%D0%BE%D1%80%D0%B1%D0%B5%D0%BD%D1%82" title="Сорбент">сорбентами</a>. Была предложена методика выделения лактоферрина из молочной плазмы на <a href="/wiki/%D0%90%D0%B3%D0%B0%D1%80%D0%BE%D0%B7%D0%B0" title="Агароза">агарозе</a> с иммобилизованной одноцепочечной ДНК.<sup class="reference" id="cite_ref-43"><a href="#cite_note-43">[43]</a></sup> При выделении белка из мочи новорожденных на этом сорбенте было обнаружено, что при прохождении через желудочно-кишечный тракт кроме неповрежденного лактоферрина (78 кДа) образуются два фрагмента 51 и 39 кДа, которые также связывают ДНК.<sup class="reference" id="cite_ref-Furmanski_18-1"><a href="#cite_note-Furmanski-18">[18]</a></sup>
</p>
<h3><span id=".D0.A4.D0.B5.D1.80.D0.BC.D0.B5.D0.BD.D1.82.D0.B0.D1.82.D0.B8.D0.B2.D0.BD.D1.8B.D0.B5_.D0.B0.D0.BA.D1.82.D0.B8.D0.B2.D0.BD.D0.BE.D1.81.D1.82.D0.B8_.D0.BB.D0.B0.D0.BA.D1.82.D0.BE.D1.84.D0.B5.D1.80.D1.80.D0.B8.D0.BD.D0.B0"></span><span class="mw-headline" id="Ферментативные_активности_лактоферрина">Ферментативные активности лактоферрина</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;veaction=edit&amp;section=9" title="Редактировать раздел «Ферментативные активности лактоферрина»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;action=edit&amp;section=9" title="Редактировать код раздела «Ферментативные активности лактоферрина»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Лактоферрин гидролизует <a class="mw-redirect" href="/wiki/%D0%A0%D0%9D%D0%9A" title="РНК">РНК</a> и проявляет свойства <a href="/wiki/%D0%9F%D0%B8%D1%80%D0%B8%D0%BC%D0%B8%D0%B4%D0%B8%D0%BD" title="Пиримидин">пиримидин</a>-специфических секреторных <a href="/wiki/%D0%A0%D0%B8%D0%B1%D0%BE%D0%BD%D1%83%D0%BA%D0%BB%D0%B5%D0%B0%D0%B7%D1%8B" title="Рибонуклеазы">рибонуклеаз</a>.
</p><p>Сравнительный анализ данных <a href="/wiki/%D0%A0%D0%B5%D0%BD%D1%82%D0%B3%D0%B5%D0%BD%D0%BE%D1%81%D1%82%D1%80%D1%83%D0%BA%D1%82%D1%83%D1%80%D0%BD%D1%8B%D0%B9_%D0%B0%D0%BD%D0%B0%D0%BB%D0%B8%D0%B7" title="Рентгеноструктурный анализ">рентгеноструктурного анализа</a> лактоферрина и комплекса <a class="mw-redirect" href="/wiki/%D0%A0%D0%9D%D0%9A%D0%B0%D0%B7%D0%B0_%D0%90" title="РНКаза А">РНКазы А</a> с аналогом субстрата 2’-5’ CpG в совокупности с анализом гомологии первичной последовательности этих двух белков показал наличие в молекуле лакоферрина структурных мотивов, подобных активному центру белков суперсемейства РНКазы А. Используя метод молекулярного моделирования, был проведен анализ топографии потенциального активного центра лактоферрина по аналогии с таковым для РНКазы А.<sup class="reference" id="cite_ref-44"><a href="#cite_note-44">[44]</a></sup> Кандидатами на роль ключевых аминокислотных остатков РНК-гидролизующего центра лактоферрина являются остатки аминокислот His-91, His-246 и Lys-237, которые находятся в оптимальной для <a href="/wiki/%D0%9A%D0%B0%D1%82%D0%B0%D0%BB%D0%B8%D0%B7" title="Катализ">катализа</a> ориентации к другим ближайшим остаткам в возможном активном центре. Субстрат-связывающий участок образуют Asp-244, Lys-241 и Thr-90. Таким образом, потенциальный РНКазный центр лактоферрина расположен в междоменной области белка.
</p><p>Способность лактоферрина гидролизовать РНК открывает ещё один аспект функций белка в организме, так как показано, что РНКазы молока путём разрушения РНК-генома ингибируют обратную транскрипцию <a class="mw-redirect" href="/wiki/%D0%A0%D0%B5%D1%82%D1%80%D0%BE%D0%B2%D0%B8%D1%80%D1%83%D1%81" title="Ретровирус">ретровирусов</a>, вызывающих <a href="/wiki/%D0%A0%D0%B0%D0%BA_%D0%BC%D0%BE%D0%BB%D0%BE%D1%87%D0%BD%D0%BE%D0%B9_%D0%B6%D0%B5%D0%BB%D0%B5%D0%B7%D1%8B" title="Рак молочной железы">рак молочной железы</a> у мышей.<sup class="reference" id="cite_ref-45"><a href="#cite_note-45">[45]</a></sup> Было показано, что у женщин группы Парси в Западной <a href="/wiki/%D0%98%D0%BD%D0%B4%D0%B8%D1%8F" title="Индия">Индии</a> уровень РНКаз молока заметно ниже такового в остальных группах, а частота заболеваемости <a href="/wiki/%D0%A0%D0%B0%D0%BA_%D0%BC%D0%BE%D0%BB%D0%BE%D1%87%D0%BD%D0%BE%D0%B9_%D0%B6%D0%B5%D0%BB%D0%B5%D0%B7%D1%8B" title="Рак молочной железы">раком молочной железы</a> превышает в три раза среднестатистическую заболеваемость.<sup class="reference" id="cite_ref-46"><a href="#cite_note-46">[46]</a></sup> Таким образом, можно предположить, что <a class="mw-redirect" href="/wiki/%D0%A0%D0%B8%D0%B1%D0%BE%D0%BD%D1%83%D0%BA%D0%BB%D0%B5%D0%B0%D0%B7%D0%B0" title="Рибонуклеаза">РНКазы</a> молока и лактоферрин в частности играют важную роль в <a href="/wiki/%D0%9F%D0%B0%D1%82%D0%BE%D0%B3%D0%B5%D0%BD%D0%B5%D0%B7" title="Патогенез">патогенезе</a> заболеваний, вызываемых различными <a class="mw-redirect" href="/wiki/%D0%A0%D0%B5%D1%82%D1%80%D0%BE%D0%B2%D0%B8%D1%80%D1%83%D1%81" title="Ретровирус">ретровирусами</a>.
</p><p>Также было показано, что изоформы лактоферрина обладают <a href="/wiki/%D0%9D%D1%83%D0%BA%D0%BB%D0%B5%D0%B0%D0%B7%D0%B0" title="Нуклеаза">нуклеазными</a> и <a href="/wiki/%D0%A4%D0%BE%D1%81%D1%84%D0%B0%D1%82%D0%B0%D0%B7%D0%B0" title="Фосфатаза">фосфатазными</a> активностями.<sup class="reference" id="cite_ref-Babina_47-0"><a href="#cite_note-Babina-47">[47]</a></sup> Гидролиз олигодезоксирибо<a class="mw-redirect" href="/wiki/%D0%9D%D1%83%D0%BA%D0%BB%D0%B5%D0%BE%D1%82%D0%B8%D0%B4" title="Нуклеотид">нуклеотидов</a> лактоферрином протекал значительно медленнее, чем высокомолекулярных ДНК-<a class="mw-disambig" href="/wiki/%D0%A1%D1%83%D0%B1%D1%81%D1%82%D1%80%D0%B0%D1%82" title="Субстрат">субстратов</a>. Исследуемая фракция белка не содержала ионов железа. Оптимальное значение pH реакционной смеси при гидролизе ДНК оценено близким 7.0-7.5. Эта величина значительно превышает значение рН оптимума других известных <a class="new" href="/w/index.php?title=%D0%94%D0%9D%D0%9A%D0%B0%D0%B7%D0%B0&amp;action=edit&amp;redlink=1" title="ДНКаза (страница отсутствует)">ДНКаз</a> (5.0-5.5 у ДНКазы II крови человека). Кроме того по таким свойствам, как активация каталитической функции низкомолекулярными эффекторами и ионами металлов, лактоферрин существенно отличается от других ферментов с ДНКазной активностью.<sup class="reference" id="cite_ref-48"><a href="#cite_note-48">[48]</a></sup>
</p><p>Была показана нуклеотид-гидролизующая активность препаратов лактоферрина.<sup class="reference" id="cite_ref-Babina_47-1"><a href="#cite_note-Babina-47">[47]</a></sup> Лактоферрину, помимо АТРазной активности, принадлежит активность, отщепляющая фосфатную группу от любых рибо- и дезоксирибонуклеозид- моно-, ди- и трифосфатов. Лактоферрин обладает свойствами неспецифической нуклеозид-5’-моно-, ди- и трифосфатфосфатазы, которую назвали нуклеотидфосфатазой.<sup class="reference" id="cite_ref-Babina_47-2"><a href="#cite_note-Babina-47">[47]</a></sup> При этом скорости дефосфорилирования <a class="mw-redirect" href="/wiki/%D0%9D%D1%83%D0%BA%D0%BB%D0%B5%D0%BE%D1%82%D0%B8%D0%B4" title="Нуклеотид">нуклеотидов</a> ниже, чем для классических АТРаз и нуклеозидаз, но сравнимы с таковыми или даже выше, чем для других широко распространенных ферментов, типа <a href="/wiki/%D0%AD%D0%BD%D0%B4%D0%BE%D0%BD%D1%83%D0%BA%D0%BB%D0%B5%D0%B0%D0%B7%D1%8B_%D1%80%D0%B5%D1%81%D1%82%D1%80%D0%B8%D0%BA%D1%86%D0%B8%D0%B8" title="Эндонуклеазы рестрикции">эндонуклеаз рестрикции</a>.<sup class="reference" id="cite_ref-49"><a href="#cite_note-49">[49]</a></sup><sup class="reference" id="cite_ref-50"><a href="#cite_note-50">[50]</a></sup> Показано, что АТР-гидролизующая активность является железонезависимым свойством белка, а конформационные перестройки, вызванные координацией ионов железа, не влияют на АТРазный центр молекулы.<sup class="reference" id="cite_ref-Babina_47-3"><a href="#cite_note-Babina-47">[47]</a></sup>
</p><p>Показано, что не разделенные на изоформы препараты лактоферрина разных доноров гидролизовали только 4,6-этилиден(G7)-п-нитрофенил-(G1)-a,D-мальтогептаозид, но не другие <a href="/wiki/%D0%9E%D0%BB%D0%B8%D0%B3%D0%BE%D1%81%D0%B0%D1%85%D0%B0%D1%80%D0%B8%D0%B4%D1%8B" title="Олигосахариды">олигосахариды</a> с иными связями между моносахаридными остатками (<a href="/wiki/%D0%A6%D0%B5%D0%BB%D0%BB%D0%BE%D0%B1%D0%B8%D0%BE%D0%B7%D0%B0" title="Целлобиоза">целлобиоза</a>, <a href="/wiki/%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D0%B7%D0%B0" title="Лактоза">лактоза</a>, <a href="/wiki/%D0%A1%D0%B0%D1%85%D0%B0%D1%80%D0%BE%D0%B7%D0%B0" title="Сахароза">сахароза</a>).<sup class="reference" id="cite_ref-Babina_47-4"><a href="#cite_note-Babina-47">[47]</a></sup> Препараты лактоферрина катализировали гидролиз мальтогептаозида с <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B0_%D0%9C%D0%B8%D1%85%D0%B0%D1%8D%D0%BB%D0%B8%D1%81%D0%B0" title="Константа Михаэлиса">Km</a> = (2.0±0.9) мМ. В целом эта величина Km для мальтогептаозида сопоставима или даже выше, чем таковые для α-амилазы (0.2-5 мМ) и <a href="/wiki/%D0%90%D0%B1%D0%B7%D0%B8%D0%BC%D1%8B" title="Абзимы">каталитически активных антител</a> молока и крови больных <a href="/wiki/%D0%90%D1%83%D1%82%D0%BE%D0%B8%D0%BC%D0%BC%D1%83%D0%BD%D0%BD%D1%8B%D0%B5_%D0%B7%D0%B0%D0%B1%D0%BE%D0%BB%D0%B5%D0%B2%D0%B0%D0%BD%D0%B8%D1%8F" title="Аутоиммунные заболевания">аутоиммунными заболеваниями</a> (~ 10<sup>−4</sup> М).<sup class="reference" id="cite_ref-51"><a href="#cite_note-51">[51]</a></sup><sup class="reference" id="cite_ref-52"><a href="#cite_note-52">[52]</a></sup>
</p>
<h2><span id=".D0.93.D0.B5.D0.BD.D1.8B_.D0.BB.D0.B0.D0.BA.D1.82.D0.BE.D1.84.D0.B5.D1.80.D1.80.D0.B8.D0.BD.D0.B0"></span><span class="mw-headline" id="Гены_лактоферрина">Гены лактоферрина</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;veaction=edit&amp;section=10" title="Редактировать раздел «Гены лактоферрина»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;action=edit&amp;section=10" title="Редактировать код раздела «Гены лактоферрина»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Изучено 60 последовательностей генов лактоферрина 11 видов млекопитающих.<sup class="reference" id="cite_ref-Jing_53-0"><a href="#cite_note-Jing-53">[53]</a></sup>
</p><p>У большинства видов, <a href="/wiki/%D0%A1%D1%82%D0%BE%D0%BF-%D0%BA%D0%BE%D0%B4%D0%BE%D0%BD" title="Стоп-кодон">стоп-кодоном</a> является <code>TAA</code>, и <code>TGA</code> у <i><a class="mw-redirect" href="/wiki/Mus_musculus" title="Mus musculus">Mus musculus</a></i>. Кодирующая часть из-за <a href="/wiki/%D0%94%D0%B5%D0%BB%D0%B5%D1%86%D0%B8%D1%8F" title="Делеция">делеций</a>, <a href="/wiki/%D0%98%D0%BD%D1%81%D0%B5%D1%80%D1%86%D0%B8%D1%8F" title="Инсерция">инсерций</a>, а также мутаций стоп-кодонов значительно отличается и имеет длину от 2,055 до 2,190 пар <a class="mw-redirect" href="/wiki/%D0%9D%D1%83%D0%BA%D0%BB%D0%B5%D0%BE%D1%82%D0%B8%D0%B4" title="Нуклеотид">нуклеотидов</a>. <a class="mw-redirect" href="/wiki/SNP" title="SNP">Полиморфизм генов</a> между видами существенно превышает внутривидовой полиморфизм лактоферрина. Обнаружены отличия в аминокислотных последовательностях: 8 у <i><a class="mw-redirect" href="/wiki/Homo_sapiens" title="Homo sapiens">Homo sapiens</a></i>, 6 у <i><a class="mw-redirect" href="/wiki/Mus_musculus" title="Mus musculus">Mus musculus</a></i>, 6 у <i><a class="mw-redirect" href="/wiki/Capra_hircus" title="Capra hircus">Capra hircus</a></i>, 10 у <i><a href="/wiki/Bos_taurus" title="Bos taurus">Bos taurus</a></i>, и 20 в случае <i><a class="mw-redirect" href="/wiki/Sus_scrofa" title="Sus scrofa">Sus scrofa</a></i>. Такой разброс может свидетельствовать о функциональных отличиях лактоферринов разных видов.<sup class="reference" id="cite_ref-Jing_53-1"><a href="#cite_note-Jing-53">[53]</a></sup>
</p><p>У человека <a href="/wiki/%D0%93%D0%B5%D0%BD" title="Ген">ген</a> лактоферрина <i>LTF</i> располагается на третьей <a href="/wiki/%D0%A5%D1%80%D0%BE%D0%BC%D0%BE%D1%81%D0%BE%D0%BC%D0%B0" title="Хромосома">хромосоме</a>, в <a href="/wiki/%D0%9B%D0%BE%D0%BA%D1%83%D1%81" title="Локус">локусе</a> 3q21-q23.
</p><p>У <a href="/wiki/%D0%9D%D0%B0%D1%81%D1%82%D0%BE%D1%8F%D1%89%D0%B8%D0%B5_%D0%B1%D1%8B%D0%BA%D0%B8" title="Настоящие быки">быка</a> кодирующая последовательность состоит из 17 <a href="/wiki/%D0%AD%D0%BA%D0%B7%D0%BE%D0%BD" title="Экзон">экзонов</a> и имеет длину около 34,5 тысяч пар <a class="mw-redirect" href="/wiki/%D0%9D%D1%83%D0%BA%D0%BB%D0%B5%D0%BE%D1%82%D0%B8%D0%B4" title="Нуклеотид">нуклеотидов</a>. Экзоны гена лактоферрина быка имеют сходный размер с экзонами других генов семейства <a href="/wiki/%D0%A2%D1%80%D0%B0%D0%BD%D1%81%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD" title="Трансферрин">трансферринов</a>, в то время как размеры интронов внутри семейства отличаются. Эволюционное сходство размеров экзонов и их распределения в доменах белковой молекулы указывает на то, что ген лактоферрина произошёл путём <a href="/wiki/%D0%94%D1%83%D0%BF%D0%BB%D0%B8%D0%BA%D0%B0%D1%86%D0%B8%D1%8F" title="Дупликация">дупликации</a>. Последовательность <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%BC%D0%BE%D1%82%D0%BE%D1%80" title="Промотор">промотороного</a> участка гена лактоферрина быка не имеет некоторых участков связывания <a href="/wiki/%D0%AD%D0%BD%D1%85%D0%B0%D0%BD%D1%81%D0%B5%D1%80" title="Энхансер">энхансеров</a> транскрипции по сравнению с соответствующии последовательностями гена лактоферрина <a href="/wiki/%D0%A7%D0%B5%D0%BB%D0%BE%D0%B2%D0%B5%D0%BA" title="Человек">человека</a> и <a href="/wiki/%D0%94%D0%BE%D0%BC%D0%BE%D0%B2%D0%B0%D1%8F_%D0%BC%D1%8B%D1%88%D1%8C" title="Домовая мышь">мыши</a>, что объясняет относительно низкую <a href="/wiki/%D0%AD%D0%BA%D1%81%D0%BF%D1%80%D0%B5%D1%81%D1%81%D0%B8%D1%8F_%D0%B3%D0%B5%D0%BD%D0%BE%D0%B2" title="Экспрессия генов">экспрессию гена</a> лактоферрина у быка.<sup class="reference" id="cite_ref-54"><a href="#cite_note-54">[54]</a></sup>
</p><p>Изучение <a class="mw-redirect" href="/wiki/SNP" title="SNP">полиморфизма генов</a>, кодирующих лактоферрин, может способствовать выведению пород сельскохозяйственных животных, устойчивых к <a href="/wiki/%D0%9C%D0%B0%D1%81%D1%82%D0%B8%D1%82" title="Мастит">маститу</a><sup class="reference" id="cite_ref-Biochimie_2009_55-0"><a href="#cite_note-Biochimie_2009-55">[55]</a></sup> а также для создания лекарственных препаратов на основе белка. <a href="/wiki/%D0%98%D0%BD%D1%81%D1%82%D0%B8%D1%82%D1%83%D1%82_%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D0%B8_%D0%B3%D0%B5%D0%BD%D0%B0_%D0%A0%D0%90%D0%9D" title="Институт биологии гена РАН">Институтом биологии гена РАН</a> в 2007 году был создан <a href="/wiki/%D0%A0%D0%B5%D0%BA%D0%BE%D0%BC%D0%B1%D0%B8%D0%BD%D0%B0%D0%BD%D1%82%D0%BD%D0%B0%D1%8F_%D1%81%D1%82%D1%80%D1%83%D0%BA%D1%82%D1%83%D1%80%D0%B0" title="Рекомбинантная структура">рекомбинантный</a> лактоферрин <a class="new" href="/w/index.php?title=%D0%9D%D0%B5%D0%BE%D0%BB%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;action=edit&amp;redlink=1" title="Неолактоферрин (страница отсутствует)">неолактоферрин</a>.
</p>
<h2><span id=".D0.A0.D0.B5.D1.86.D0.B5.D0.BF.D1.82.D0.BE.D1.80_.D0.BB.D0.B0.D0.BA.D1.82.D0.BE.D1.84.D0.B5.D1.80.D1.80.D0.B8.D0.BD.D0.B0"></span><span class="mw-headline" id="Рецептор_лактоферрина">Рецептор лактоферрина</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;veaction=edit&amp;section=11" title="Редактировать раздел «Рецептор лактоферрина»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9B%D0%B0%D0%BA%D1%82%D0%BE%D1%84%D0%B5%D1%80%D1%80%D0%B8%D0%BD&amp;action=edit&amp;section=11" title="Редактировать код раздела «Рецептор лактоферрина»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Рецептор лактоферрина играет важную роль в процессе <a href="/wiki/%D0%98%D0%BD%D1%82%D0%B5%D1%80%D0%BD%D0%B0%D0%BB%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F" title="Интернализация">интернализации</a> лактоферрина и облегчает абсорбцию ионов <a href="/wiki/%D0%96%D0%B5%D0%BB%D0%B5%D0%B7%D0%BE" title="Железо">железа</a>, связанных с лактоферрином. Методом <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%BB%D0%B8%D1%87%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%9F%D0%A6%D0%A0" title="Количественная ПЦР">количественной ПЦР</a> было показано повышение <a href="/wiki/%D0%AD%D0%BA%D1%81%D0%BF%D1%80%D0%B5%D1%81%D1%81%D0%B8%D1%8F_%D0%B3%D0%B5%D0%BD%D0%BE%D0%B2" title="Экспрессия генов">экспрессии</a> <a href="/wiki/%D0%A0%D0%B5%D1%86%D0%B5%D0%BF%D1%82%D0%BE%D1%80" title="Рецептор">рецептора</a> лактоферрина с возрастом в <a class="mw-redirect" href="/wiki/%D0%94%D0%B2%D0%B5%D0%BD%D0%B0%D0%B4%D1%86%D0%B0%D1%82%D0%B8%D0%BF%D0%B5%D1%80%D1%81%D1%82%D0%BD%D0%B0%D1%8F_%D0%BA%D0%B8%D1%88%D0%BA%D0%B0" title="Двенадцатиперстная кишка">двенадцатиперстной кишке</a> и снижение экспрессии в <a class="mw-redirect" href="/wiki/%D0%A2%D0%BE%D1%89%D0%B0%D1%8F_%D0%BA%D0%B8%D1%88%D0%BA%D0%B0" title="Тощая кишка">тощей кишке</a>.<sup class="reference" id="cite_ref-56"><a href="#cite_note-56">[56]</a></sup>
</p>
