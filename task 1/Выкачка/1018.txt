<p><b>Амьенская операция</b> (<b>Битва при Амьене (1918)</b>, <b>Амьенское сражение;</b> <a href="/wiki/8_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="8 августа">8 августа</a> — <a href="/wiki/13_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="13 августа">13 августа</a> <a href="/wiki/1918_%D0%B3%D0%BE%D0%B4" title="1918 год">1918 года</a>) — широкомасштабное наступление <a href="/wiki/%D0%90%D0%BD%D1%82%D0%B0%D0%BD%D1%82%D0%B0" title="Антанта">союзных</a> войск против <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F" title="Германская империя">германской</a> армии во время <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B2%D0%B0%D1%8F_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Первая мировая война">Первой мировой войны</a> у французского города <a href="/wiki/%D0%90%D0%BC%D1%8C%D0%B5%D0%BD" title="Амьен">Амьен</a>; завершилось прорывом германского фронта и победой войск Антанты.
</p><p>В результате боевых действий на <a href="/wiki/%D0%97%D0%B0%D0%BF%D0%B0%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D1%80%D0%BE%D0%BD%D1%82_%D0%9F%D0%B5%D1%80%D0%B2%D0%BE%D0%B9_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%BE%D0%B9_%D0%B2%D0%BE%D0%B9%D0%BD%D1%8B" title="Западный фронт Первой мировой войны">Западном фронте</a> <a href="/wiki/%D0%92%D0%B5%D1%81%D0%B5%D0%BD%D0%BD%D0%B5%D0%B5_%D0%BD%D0%B0%D1%81%D1%82%D1%83%D0%BF%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5_(1918)" title="Весеннее наступление (1918)">весной</a> <a href="/wiki/1918_%D0%B3%D0%BE%D0%B4" title="1918 год">1918 года</a>, произошли значительные изменения в линии фронта. Образовались два так называемых «выступа», глубоко вклинившихся в оборону войск <a href="/wiki/%D0%90%D0%BD%D1%82%D0%B0%D0%BD%D1%82%D0%B0" title="Антанта">Антанты</a>. Они получили название Амьенский и Сен-Миельский выступы. В ходе кампании <a href="/wiki/1918_%D0%B3%D0%BE%D0%B4" title="1918 год">1918 года</a> союзным войскам удалось ликвидировать обе эти стратегические неровности фронта (Амьенская и <a href="/wiki/%D0%A1%D0%B5%D0%BD-%D0%9C%D0%B8%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F" title="Сен-Миельская операция">Сен-Миельская операция</a>).
</p><p>Амьенская битва является первой операцией союзных армий в рамках <a href="/wiki/%D0%A1%D1%82%D0%BE%D0%B4%D0%BD%D0%B5%D0%B2%D0%BD%D0%BE%D0%B5_%D0%BD%D0%B0%D1%81%D1%82%D1%83%D0%BF%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5" title="Стодневное наступление">Стодневного наступления</a>. Начавшаяся <a href="/wiki/8_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="8 августа">8 августа</a> операция положила начало всеобщему наступлению союзных войск на <a href="/wiki/%D0%97%D0%B0%D0%BF%D0%B0%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D1%80%D0%BE%D0%BD%D1%82_%D0%9F%D0%B5%D1%80%D0%B2%D0%BE%D0%B9_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%BE%D0%B9_%D0%B2%D0%BE%D0%B9%D0%BD%D1%8B" title="Западный фронт Первой мировой войны">Западном фронте</a>. Амьенское сражение было переломным моментом кампании <a href="/wiki/1918_%D0%B3%D0%BE%D0%B4" title="1918 год">1918 года</a>, войскам Антанты удалось полностью выполнить задачу прорыва германского фронта, генерал <a class="mw-redirect" href="/wiki/%D0%9B%D1%8E%D0%B4%D0%B5%D0%BD%D0%B4%D0%BE%D1%80%D1%84,_%D0%AD%D1%80%D0%B8%D1%85_%D0%A4%D1%80%D0%B8%D0%B4%D1%80%D0%B8%D1%85_%D0%92%D0%B8%D0%BB%D1%8C%D0%B3%D0%B5%D0%BB%D1%8C%D0%BC" title="Людендорф, Эрих Фридрих Вильгельм">Людендорф</a> позже назвал 8 августа «чёрным днём» для германской армии. Амьенская операция завершила позиционную войну на Западном фронте. Война перешла в стадию манёвренной, до самого окончания боевых действий <a href="/wiki/11_%D0%BD%D0%BE%D1%8F%D0%B1%D1%80%D1%8F" title="11 ноября">11 ноября</a> <a href="/wiki/1918_%D0%B3%D0%BE%D0%B4" title="1918 год">1918 года</a>.
</p>

<h2><span id=".D0.9F.D0.B5.D1.80.D0.B5.D0.B4_.D0.BD.D0.B0.D1.81.D1.82.D1.83.D0.BF.D0.BB.D0.B5.D0.BD.D0.B8.D0.B5.D0.BC"></span><span class="mw-headline" id="Перед_наступлением">Перед наступлением</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%90%D0%BC%D1%8C%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Перед наступлением»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D0%BC%D1%8C%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F&amp;action=edit&amp;section=1" title="Редактировать код раздела «Перед наступлением»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id=".D0.91.D0.BE.D0.B5.D0.B2.D1.8B.D0.B5_.D0.B4.D0.B5.D0.B9.D1.81.D1.82.D0.B2.D0.B8.D1.8F_.D0.B2_.D0.BD.D0.B0.D1.87.D0.B0.D0.BB.D0.B5-.D1.81.D0.B5.D1.80.D0.B5.D0.B4.D0.B8.D0.BD.D0.B5_1918_.D0.B3.D0.BE.D0.B4.D0.B0"></span><span class="mw-headline" id="Боевые_действия_в_начале-середине_1918_года">Боевые действия в начале-середине 1918 года</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%90%D0%BC%D1%8C%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Боевые действия в начале-середине 1918 года»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D0%BC%D1%8C%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F&amp;action=edit&amp;section=2" title="Редактировать код раздела «Боевые действия в начале-середине 1918 года»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a href="/wiki/21_%D0%BC%D0%B0%D1%80%D1%82%D0%B0" title="21 марта">21 марта</a> <a href="/wiki/1918_%D0%B3%D0%BE%D0%B4" title="1918 год">1918 года</a> германское командование приступило к реализации операции «Михаэль» (<a href="/wiki/%D0%92%D0%B5%D1%81%D0%B5%D0%BD%D0%BD%D0%B5%D0%B5_%D0%BD%D0%B0%D1%81%D1%82%D1%83%D0%BF%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5_(1918)" title="Весеннее наступление (1918)">Весеннее наступление (1918)</a>). Это наступление должно было переломить ход войны в пользу <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F" title="Германская империя">Германии</a>. Германская армия надеялась захватить наступательную инициативу, расчленить союзные армии, британцев скинуть в море, а затем приступить к уничтожению французской армии и продвижению к <a href="/wiki/%D0%9F%D0%B0%D1%80%D0%B8%D0%B6" title="Париж">Парижу</a>.
</p><p>Заключив <a href="/wiki/%D0%91%D1%80%D0%B5%D1%81%D1%82%D1%81%D0%BA%D0%B8%D0%B9_%D0%BC%D0%B8%D1%80" title="Брестский мир">Брестский мир</a> с <a class="mw-redirect" href="/wiki/%D0%A0%D0%A1%D0%A4%D0%A1%D0%A0" title="РСФСР">Советской Россией</a>, <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%B8%D1%8F" title="Германия">Германия</a> сумела перебросить 44 дивизии с переставшего существовать <a href="/wiki/%D0%92%D0%BE%D1%81%D1%82%D0%BE%D1%87%D0%BD%D1%8B%D0%B9_%D1%84%D1%80%D0%BE%D0%BD%D1%82_%D0%9F%D0%B5%D1%80%D0%B2%D0%BE%D0%B9_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%BE%D0%B9_%D0%B2%D0%BE%D0%B9%D0%BD%D1%8B" title="Восточный фронт Первой мировой войны">Восточного фронта</a> на <a href="/wiki/%D0%97%D0%B0%D0%BF%D0%B0%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D1%80%D0%BE%D0%BD%D1%82_%D0%9F%D0%B5%D1%80%D0%B2%D0%BE%D0%B9_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%BE%D0%B9_%D0%B2%D0%BE%D0%B9%D0%BD%D1%8B" title="Западный фронт Первой мировой войны">Западный</a>, создав численный перевес над союзными армиями. Первоначально германской армии сопутствовал успех, наступавшие сумели продавить оборону союзников и продвинуться на несколько десятков километров. После завершения <a href="/wiki/%D0%92%D0%B5%D1%81%D0%B5%D0%BD%D0%BD%D0%B5%D0%B5_%D0%BD%D0%B0%D1%81%D1%82%D1%83%D0%BF%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5_(1918)" title="Весеннее наступление (1918)">операции «Мишель»</a> германское наступление продолжилось. Одержав <a href="/wiki/%D0%91%D0%B8%D1%82%D0%B2%D0%B0_%D0%BD%D0%B0_%D0%9B%D0%B8%D1%81%D0%B5" title="Битва на Лисе">победы у Лиса</a> и в <a href="/wiki/%D0%A2%D1%80%D0%B5%D1%82%D1%8C%D1%8F_%D0%B1%D0%B8%D1%82%D0%B2%D0%B0_%D0%BD%D0%B0_%D0%AD%D0%BD%D0%B5" title="Третья битва на Эне">битве при Айсине</a>, германцы приблизились к <a href="/wiki/%D0%9F%D0%B0%D1%80%D0%B8%D0%B6" title="Париж">Парижу</a> на 56 километров.
</p><p>К июлю германские войска вышли к <a href="/wiki/%D0%9C%D0%B0%D1%80%D0%BD%D0%B0_(%D1%80%D0%B5%D0%BA%D0%B0)" title="Марна (река)">Марне</a>, однако войска были истощены, резервы исчерпаны. В этих условиях командующий союзными войсками генерал <a class="mw-redirect" href="/wiki/%D0%A4%D0%BE%D1%88" title="Фош">Фош</a> отдал приказ о начале контрнаступления.<sup class="reference" id="cite_ref-марна2_3-0"><a href="#cite_note-марна2-3">[3]</a></sup>
В июле-августе состоялась <a href="/wiki/%D0%91%D0%B8%D1%82%D0%B2%D0%B0_%D0%BD%D0%B0_%D0%9C%D0%B0%D1%80%D0%BD%D0%B5_(1918)" title="Битва на Марне (1918)">вторая битва на Марне</a>, в которой войска Антанты сумели одержать победу над немецкими войсками, которые были вынуждены отступить.
</p><p>После сражения на Марне стратегическая инициатива окончательно перешла в руки союзных войск.
</p>
<h3><span id=".D0.9F.D0.BB.D0.B0.D0.BD_.D1.81.D0.BE.D1.8E.D0.B7.D0.BD.D0.BE.D0.B3.D0.BE_.D0.BA.D0.BE.D0.BC.D0.B0.D0.BD.D0.B4.D0.BE.D0.B2.D0.B0.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="План_союзного_командования">План союзного командования</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%90%D0%BC%D1%8C%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F&amp;veaction=edit&amp;section=3" title="Редактировать раздел «План союзного командования»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D0%BC%D1%8C%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F&amp;action=edit&amp;section=3" title="Редактировать код раздела «План союзного командования»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>После того как немецкое наступление было остановлено, командование Антанты приняло решение о подготовке серии наступательных операций, которые должны были привести к окончательному разгрому германской армии и окончанию войны.
</p><p><a href="/wiki/24_%D0%B8%D1%8E%D0%BB%D1%8F" title="24 июля">24 июля</a> <a href="/wiki/1918_%D0%B3%D0%BE%D0%B4" title="1918 год">1918 года</a> в Бомбоне<sup class="reference" id="cite_ref-Kearsey2_4-0"><a href="#cite_note-Kearsey2-4">[4]</a></sup> состоялось совещание командующих союзными армиями <a class="mw-redirect" href="/wiki/%D0%9F%D0%B5%D1%82%D0%B5%D0%BD,_%D0%90%D0%BD%D1%80%D0%B8" title="Петен, Анри">Петэна</a>, <a href="/wiki/%D0%A5%D0%B5%D0%B9%D0%B3,_%D0%94%D1%83%D0%B3%D0%BB%D0%B0%D1%81" title="Хейг, Дуглас">Хейга</a> и <a href="/wiki/%D0%9F%D0%B5%D1%80%D1%88%D0%B8%D0%BD%D0%B3,_%D0%94%D0%B6%D0%BE%D0%BD" title="Першинг, Джон">Першинга</a>, на котором главнокомандующий союзными войсками <a class="mw-redirect" href="/wiki/%D0%A4%D0%BE%D1%88" title="Фош">Фош</a> изложил план дальнейших действий союзных войск. Основа этого плана состояла в том, что необходимо было отказаться от оборонительного образа действий и перейти в наступление. <a href="/wiki/24_%D0%B8%D1%8E%D0%BB%D1%8F" title="24 июля">24 июля</a> союзное командование издало директиву о проведении ряда наступательных операций с целью ликвидировать выступы в линии фронта, образовавшиеся в результате германского весеннего наступления (Амьенский и <a href="/wiki/%D0%A1%D0%B5%D0%BD-%D0%9C%D0%B8%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F" title="Сен-Миельская операция">Сен-Миельский выступы</a>). Далее планировалось перейти в общее наступление по всему фронту и не позволить войскам противника отойти на ранее подготовленные оборонительные рубежи, как это было в <a href="/wiki/1917_%D0%B3%D0%BE%D0%B4" title="1917 год">1917 году</a>. Переоценив оборонительные возможности немцев, <a class="mw-redirect" href="/wiki/%D0%A4%D0%BE%D1%88" title="Фош">Фош</a> планировал закончить войну лишь в следующем <a href="/wiki/1919_%D0%B3%D0%BE%D0%B4" title="1919 год">1919 году</a>.
</p>
<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Amiens_the_key_of_the_west.jpg"><img class="mw-file-element" data-file-height="600" data-file-width="930" decoding="async" height="161" src="//upload.wikimedia.org/wikipedia/commons/thumb/f/f2/Amiens_the_key_of_the_west.jpg/250px-Amiens_the_key_of_the_west.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/f/f2/Amiens_the_key_of_the_west.jpg/375px-Amiens_the_key_of_the_west.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/f/f2/Amiens_the_key_of_the_west.jpg/500px-Amiens_the_key_of_the_west.jpg 2x" width="250"/></a><figcaption><i><a href="/wiki/%D0%A1%D1%82%D1%80%D0%B8%D1%82%D0%BE%D0%BD,_%D0%90%D1%80%D1%82%D1%83%D1%80" title="Стритон, Артур">Артур Стритон</a></i>. <b>Амьен — ключ к западу.</b></figcaption></figure>
<p>Германское командование не представляло всей серьёзности обстановки. Командующий германскими войсками генерал <a class="mw-redirect" href="/wiki/%D0%9B%D1%8E%D0%B4%D0%B5%D0%BD%D0%B4%D0%BE%D1%80%D1%84,_%D0%AD%D1%80%D0%B8%D1%85_%D0%A4%D1%80%D0%B8%D0%B4%D1%80%D0%B8%D1%85_%D0%92%D0%B8%D0%BB%D1%8C%D0%B3%D0%B5%D0%BB%D1%8C%D0%BC" title="Людендорф, Эрих Фридрих Вильгельм">Эрих Людендорф</a> отказался отвести войска с захваченных летом позиций по политическим причинам. Однако, <a href="/wiki/2_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="2 августа">2 августа</a> <a href="/wiki/1918_%D0%B3%D0%BE%D0%B4" title="1918 год">1918 года</a> Людендорф издал директиву, в которой указывалось: «Обстановка требует, чтобы мы, с одной стороны, перешли к обороне, а с другой, — как только представится возможность, вновь предприняли бы наступление». <a href="/wiki/%D0%9D%D0%B5%D0%BC%D1%86%D1%8B" title="Немцы">Немцами</a> планировалось провести ещё ряд наступательных операций для улучшения тактического положения во <a href="/wiki/%D0%A4%D0%BB%D0%B0%D0%BD%D0%B4%D1%80%D0%B8%D1%8F_(%D0%B8%D1%81%D1%82%D0%BE%D1%80%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C)" title="Фландрия (историческая область)">Фландрии</a>. <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F" title="Германская империя">Германское</a> командование, в отличие от союзного командования, наоборот, переоценило свои силы, считая, что союзные армии в ходе весеннего наступления понесли большие потери и не планируют активных боевых действий.<sup class="reference" id="cite_ref-Kearsey3_5-0"><a href="#cite_note-Kearsey3-5">[5]</a></sup>
</p>
<h3><span id=".D0.9F.D0.BE.D0.B4.D0.B3.D0.BE.D1.82.D0.BE.D0.B2.D0.BA.D0.B0_.D0.BA_.D0.90.D0.BC.D1.8C.D0.B5.D0.BD.D1.81.D0.BA.D0.BE.D0.B9_.D0.BE.D0.BF.D0.B5.D1.80.D0.B0.D1.86.D0.B8.D0.B8"></span><span class="mw-headline" id="Подготовка_к_Амьенской_операции">Подготовка к Амьенской операции</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%90%D0%BC%D1%8C%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Подготовка к Амьенской операции»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D0%BC%D1%8C%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F&amp;action=edit&amp;section=4" title="Редактировать код раздела «Подготовка к Амьенской операции»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Первой из намеченных союзниками операций по ликвидации выступов в линии фронта была Амьенская операция. Планируя её, командование <a href="/wiki/%D0%90%D0%BD%D1%82%D0%B0%D0%BD%D1%82%D0%B0" title="Антанта">Антанты</a> рассчитывало очистить от германских войск амьенский выступ, «ликвидировать угрозу Амьену и железной дороге <a href="/wiki/%D0%9F%D0%B0%D1%80%D0%B8%D0%B6" title="Париж">Париж</a> — <a href="/wiki/%D0%90%D0%BC%D1%8C%D0%B5%D0%BD" title="Амьен">Амьен</a>, а также разбить и отбросить противника между реками Сомма и Авр».
К проведению операции привлекались 4-я британская, 1-я и 3-я французские армии под общим командованием фельдмаршала <a href="/wiki/%D0%A5%D0%B5%D0%B9%D0%B3,_%D0%94%D1%83%D0%B3%D0%BB%D0%B0%D1%81" title="Хейг, Дуглас">Хейга</a>. В состав наступательной группировки были включены 17 пехотных и 3 кавалерийские дивизии, 2684 артиллерийских орудия, 532 <a href="/wiki/%D0%A2%D0%B0%D0%BD%D0%BA" title="Танк">танка</a> (тяжёлые танки марки «<a class="mw-redirect" href="/wiki/Mark_V_(%D1%82%D0%B0%D0%BD%D0%BA)" title="Mark V (танк)">MV</a>» и «<a class="mw-redirect" href="/wiki/Mark_V**_(%D1%82%D0%B0%D0%BD%D0%BA)" title="Mark V** (танк)">MV звезда</a>» и <a class="mw-redirect" href="/wiki/Mark_A_(%D1%82%D0%B0%D0%BD%D0%BA)" title="Mark A (танк)">средние танки марки «Уиппет»</a>), 16 бронеавтомобилей и около 1900 самолётов. В наступлении планировалось активно использовать танки, которых у союзников было достаточно.
Оборону на данном участке фронта занимала 2-я германская армия, имевшая в своём составе 7 пехотных дивизий, 840 орудий и 106 самолётов.<sup class="reference" id="cite_ref-Kearsey4_6-0"><a href="#cite_note-Kearsey4-6">[6]</a></sup>
</p><p>В первый день наступления <a href="/wiki/8_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="8 августа">8 августа</a> планировалось наступление на фронте в 25 км силами 4-й английской армии и левофлангового (31-го) корпуса 1-й французской армии. Затем должно было начаться наступление 3-й армии и основных сил 1-й армии. По настоянию командующих канадских и австралийских дивизий (которым предстояло впервые сражаться плечом к плечу),<sup class="reference" id="cite_ref-Kearsey2_4-1"><a href="#cite_note-Kearsey2-4">[4]</a></sup> для усиления эффекта неожиданности в полосе наступления была отменена артиллерийская подготовка, планировалось задействовать большое количество танков. Однако в полосе наступления 1-й французской армии без артиллерийской подготовки было не обойтись. Поэтому, чтобы не терять элемент неожиданности, было принято решение о том, что основные французские силы начнут наступать на 45 минут позже 4-й британской армии.
</p><p>Характерной особенностью подготовки операции было то, что австралийские части британской армии уже с конца апреля начали вести мелкие бои за улучшение своего тактического положения. В результате этого 2-я немецкая армия потеряла полосу боевого охранения и оборонялась на недостаточно развитых в глубину позициях.<sup class="reference" id="cite_ref-Blaxland_7-0"><a href="#cite_note-Blaxland-7">[7]</a></sup>
</p><p>Эти бои, а также на редкость успешная <a href="/wiki/%D0%90%D1%8D%D1%80%D0%BE%D1%84%D0%BE%D1%82%D0%BE%D1%81%D1%8A%D1%91%D0%BC%D0%BA%D0%B0" title="Аэрофотосъёмка">аэрофотосъёмка</a> позволили английскому командованию составить полную картину системы германской обороны. Начало наступления было намечено на 4 часа 20 минут.
1/3 орудий должны были создать <a href="/wiki/%D0%9E%D0%B3%D0%BD%D0%B5%D0%B2%D0%BE%D0%B9_%D0%B2%D0%B0%D0%BB_(%D1%82%D0%B0%D0%BA%D1%82%D0%B8%D0%BA%D0%B0)" title="Огневой вал (тактика)">огневой вал</a>, а остальные 2/3 — вести огонь по пехотным и артиллерийским позициям, командным пунктам, путям подхода резервов. Огневой вал в течение трёх минут должен был держаться на передовых германских позициях. За это время танки и атакующая <a href="/wiki/%D0%9F%D0%B5%D1%85%D0%BE%D1%82%D0%B0" title="Пехота">пехота</a> должны были вплотную подойти к огневому валу и непосредственно следовать за ним. Огневой вал надлежало вести скачками сначала через 2 минуты, затем через 3 <a href="/wiki/%D0%9C%D0%B8%D0%BD%D1%83%D1%82%D0%B0" title="Минута">минуты</a>, а позднее через 4 минуты.
Весьма детально был спланирован порядок наступления в глубине германской обороны. Через 2 часа после начала атаки, в 6 часов 20 минут, пехота и танки должны были достигнуть первого рубежа атаки. Затем продвижение останавливалось, подтягивалась артиллерия, атака возобновлялась в 8 часов 20 минут и продолжалась до второго рубежа, а затем без перерыва до третьего рубежа, который находился в 9-12 км от исходных позиций наступавших. Затем кавалерийский корпус 4-й армии должен был обогнать пехоту, выйти на третий рубеж обороны и удерживать его до подхода основных сил, а после этого развивать успех далее.<sup class="reference" id="cite_ref-Blaxland1_8-0"><a href="#cite_note-Blaxland1-8">[8]</a></sup>
</p><p>Отличительной чертой подготовки операции была также абсолютная секретность. Весь район сосредоточения прикрывался <a href="/wiki/%D0%90%D0%B2%D0%B8%D0%B0%D1%86%D0%B8%D1%8F" title="Авиация">авиацией</a>, благодаря хорошему состоянию железнодорожных путей в район наступления прибыло 230 воинских эшелонов и более 60 поездов с боеприпасами. <a href="/wiki/%D0%90%D1%80%D1%82%D0%B8%D0%BB%D0%BB%D0%B5%D1%80%D0%B8%D1%8F" title="Артиллерия">Артиллерия</a> заняла свои позиции в последние 2-3 дня перед наступлением, а танки в ночь на <a href="/wiki/8_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="8 августа">8 августа</a>. Для введения противника в заблуждение в районе <a href="/wiki/%D0%98%D0%BF%D1%80" title="Ипр">Ипра</a> по распоряжению английского командования велись широкие демонстративные действия.<sup class="reference" id="cite_ref-Blaxland3_9-0"><a href="#cite_note-Blaxland3-9">[9]</a></sup>
</p><p>В последние дни перед наступлением с передовых немецких позиций стали поступать донесения о подозрительных шумах в тылу противника, а воздушная разведка доложила о передвижении колонны танков. На все эти сообщения германское командование не обратило большого внимания.<sup class="reference" id="cite_ref-Blaxland2_10-0"><a href="#cite_note-Blaxland2-10">[10]</a></sup>

<h2><span id=".D0.91.D0.BE.D0.B5.D0.B2.D1.8B.D0.B5_.D0.B4.D0.B5.D0.B9.D1.81.D1.82.D0.B2.D0.B8.D1.8F"></span><span class="mw-headline" id="Боевые_действия">Боевые действия</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%90%D0%BC%D1%8C%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F&amp;veaction=edit&amp;section=8" title="Редактировать раздел «Боевые действия»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D0%BC%D1%8C%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F&amp;action=edit&amp;section=8" title="Редактировать код раздела «Боевые действия»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id=".D0.9D.D0.B0.D1.87.D0.B0.D0.BB.D0.BE_.D0.BD.D0.B0.D1.81.D1.82.D1.83.D0.BF.D0.BB.D0.B5.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="Начало_наступления">Начало наступления</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%90%D0%BC%D1%8C%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F&amp;veaction=edit&amp;section=9" title="Редактировать раздел «Начало наступления»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D0%BC%D1%8C%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F&amp;action=edit&amp;section=9" title="Редактировать код раздела «Начало наступления»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Battle_of_Amiens_Hundred_Days_Offensive.jpg"><img class="mw-file-element" data-file-height="925" data-file-width="597" decoding="async" height="465" src="//upload.wikimedia.org/wikipedia/commons/thumb/0/04/Battle_of_Amiens_Hundred_Days_Offensive.jpg/300px-Battle_of_Amiens_Hundred_Days_Offensive.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/0/04/Battle_of_Amiens_Hundred_Days_Offensive.jpg/450px-Battle_of_Amiens_Hundred_Days_Offensive.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/0/04/Battle_of_Amiens_Hundred_Days_Offensive.jpg 2x" width="300"/></a><figcaption>Битва при Амьене.</figcaption></figure>
<p><a href="/wiki/8_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="8 августа">8 августа</a> <a href="/wiki/1918_%D0%B3%D0%BE%D0%B4" title="1918 год">1918 года</a> в 4 часа 20 минут <a href="/wiki/%D0%90%D0%BD%D1%82%D0%B0%D0%BD%D1%82%D0%B0" title="Антанта">союзная</a> <a href="/wiki/%D0%90%D1%80%D1%82%D0%B8%D0%BB%D0%BB%D0%B5%D1%80%D0%B8%D1%8F" title="Артиллерия">артиллерия</a> открыла мощный огонь по позициям, командным и наблюдательным пунктам, узлам связи и тыловым объектам 2-й <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F" title="Германская империя">германской</a> армии.<sup class="reference" id="cite_ref-Kearsey3_5-1"><a href="#cite_note-Kearsey3-5">[5]</a></sup> Треть артиллерии организовала огневой вал, под прикрытием которого дивизии 4-й британской армии и 415 танков двинулись в атаку. Атака стала полной неожиданностью для немцев. <a href="/wiki/%D0%A2%D1%83%D0%BC%D0%B0%D0%BD" title="Туман">Туман</a> и разрывы химических и дымовых снарядов ограничили видимость до 10-15 метров. На позиции германских войск обрушились английские танки. Танки обстреливали из пулемётов немецкую пехоту, уничтожали телефонные и телеграфные столбы.
</p><p>В 5 часов 05 минут, после артиллерийской подготовки, в сражение вступил 31-й корпус французской армии, прорыв германской обороны развивался в строгом соответствии с ранее намеченным планом. К 6 часам 20 минутам 7 британских дивизий, в основном, достигли первого рубежа обороны. Затем войска приостановились на 2 часа, чтобы подтянулась артиллерия.<sup class="reference" id="cite_ref-Kearsey5_12-0"><a href="#cite_note-Kearsey5-12">[12]</a></sup> Затем наступление возобновилось, и к 13 часам 30 минутам наступавшие достигли третьего оборонительного рубежа. Однако дальнейшие попытки английских и французских войск продвинуться вглубь германской обороны были безуспешны. Германское командование спешно перебрасывало в район <a href="/wiki/%D0%90%D0%BC%D1%8C%D0%B5%D0%BD" title="Амьен">Амьена</a> дивизии с других участков фронта.<sup class="reference" id="cite_ref-autogenerated2_13-0"><a href="#cite_note-autogenerated2-13">[13]</a></sup>
</p><p>Итоги первого дня боёв были для германской армии плачевными. Немецкие войска потеряли 27 000 человек убитыми и пленными, около 400 орудий. Союзным войскам удалось захватить большое число других военных трофеев.<sup class="reference" id="cite_ref-amien_14-0"><a href="#cite_note-amien-14">[14]</a></sup> Также союзникам удалось сбить 62 немецких самолёта.<sup class="reference" id="cite_ref-Der_Weltkrieg_15-0"><a href="#cite_note-Der_Weltkrieg-15">[15]</a></sup> Германские войска были деморализованы, начались массовые сдачи в плен. Британские и французские войска за первый день боёв потеряли 8800 человек. Внезапность атаки прервала связь между германскими дивизиями. Большое число немецких офицеров попало в плен.
</p>
<h3><span id=".D0.94.D0.B0.D0.BB.D1.8C.D0.BD.D0.B5.D0.B9.D1.88.D0.B8.D0.B5_.D0.B1.D0.BE.D0.B8"></span><span class="mw-headline" id="Дальнейшие_бои">Дальнейшие бои</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%90%D0%BC%D1%8C%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F&amp;veaction=edit&amp;section=10" title="Редактировать раздел «Дальнейшие бои»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D0%BC%D1%8C%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F&amp;action=edit&amp;section=10" title="Редактировать код раздела «Дальнейшие бои»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Harry_Miner_Headstone.JPG"><img class="mw-file-element" data-file-height="3008" data-file-width="2000" decoding="async" height="226" src="//upload.wikimedia.org/wikipedia/commons/thumb/5/58/Harry_Miner_Headstone.JPG/150px-Harry_Miner_Headstone.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/5/58/Harry_Miner_Headstone.JPG/225px-Harry_Miner_Headstone.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/5/58/Harry_Miner_Headstone.JPG/300px-Harry_Miner_Headstone.JPG 2x" width="150"/></a><figcaption>Надгробие канадского солдата, погибшего в битве при Амьене.</figcaption></figure>
<p><a href="/wiki/9_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="9 августа">9 августа</a> наступление союзных войск продолжилось, в сражение вступили 1-я и 3-я французские армии. Германская артиллерия была перегруппирована для борьбы с танками, в результате чего англичане и французы начали нести большие потери в танках. Например, за первый день боёв из 415 танков союзники потеряли около 100 машин. <a href="/wiki/9_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="9 августа">9 августа</a> из 145 танков союзники потеряли 39. Это оказало значительное влияние на снижение темпов наступления. Вскоре танки были выведены в тыл и в дальнейшем в сражении не участвовали.
</p><p>Наибольших успехов добились канадские и австралийские дивизии, которые двигались вглубь вражеской обороны с внушительной быстротой. Южнее <a href="/wiki/%D0%A1%D0%BE%D0%BC%D0%BC%D0%B0_(%D1%80%D0%B5%D0%BA%D0%B0)" title="Сомма (река)">Соммы</a> в германской обороне образовался разрыв в 24 км. Севернее Соммы результаты для союзников были не такими впечатляющими.
</p><p><a href="/wiki/12_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="12 августа">12 августа</a> бои шли лишь на некоторых участках фронта, к исходу дня германские войска были вытеснены на линию Альбер, Брэ, Шон, западнее Руа. С <a href="/wiki/13_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="13 августа">13 августа</a> продвижение союзных войск прекратилось. В среднем союзные войска продвинулись на 11 км.<sup class="reference" id="cite_ref-Kearsey2_4-2"><a href="#cite_note-Kearsey2-4">[4]</a></sup> Канадские дивизии на 13 км, австралийские дивизии на 11 км,<sup class="reference" id="cite_ref-австралийцы_16-0"><a href="#cite_note-австралийцы-16">[16]</a></sup> англичане на 3 км, французские войска на 8 км.
</p>
<h2><span id=".D0.9F.D0.BE.D1.81.D0.BB.D0.B5.D0.B4.D1.81.D1.82.D0.B2.D0.B8.D1.8F_.D0.BE.D0.BF.D0.B5.D1.80.D0.B0.D1.86.D0.B8.D0.B8"></span><span class="mw-headline" id="Последствия_операции">Последствия операции</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%90%D0%BC%D1%8C%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F&amp;veaction=edit&amp;section=11" title="Редактировать раздел «Последствия операции»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D0%BC%D1%8C%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F&amp;action=edit&amp;section=11" title="Редактировать код раздела «Последствия операции»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id=".D0.92.D0.BE.D0.B5.D0.BD.D0.BD.D1.8B.D0.B5_.D0.B8.D1.82.D0.BE.D0.B3.D0.B8_.D0.BD.D0.B0.D1.81.D1.82.D1.83.D0.BF.D0.BB.D0.B5.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="Военные_итоги_наступления">Военные итоги наступления</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%90%D0%BC%D1%8C%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F&amp;veaction=edit&amp;section=12" title="Редактировать раздел «Военные итоги наступления»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D0%BC%D1%8C%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F&amp;action=edit&amp;section=12" title="Редактировать код раздела «Военные итоги наступления»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>В результате наступления союзные войска выполнили стоявшую перед ними задачу прорыва германского фронта. Они продвинулись от 10 до 18 км на фронте в 75 км, ликвидировав угрозу Амьену и железной дороге <a href="/wiki/%D0%9F%D0%B0%D1%80%D0%B8%D0%B6" title="Париж">Париж</a> − <a href="/wiki/%D0%90%D0%BC%D1%8C%D0%B5%D0%BD" title="Амьен">Амьен</a>. Потери германской армии составили 74 000 человек из них 33 000 пленными<sup class="reference" id="cite_ref-Рипли_2-2"><a href="#cite_note-Рипли-2">[2]</a></sup>. Союзники потеряли около 46 000 человек (в том числе британцы и американцы — 7100 человек, канадцы — 9100, австралийцы — 6000, французы — 24 000)<sup class="reference" id="cite_ref-Рипли_2-3"><a href="#cite_note-Рипли-2">[2]</a></sup>.
</p>
<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Capturedgermanartillery.JPG"><img class="mw-file-element" data-file-height="1728" data-file-width="2304" decoding="async" height="150" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Capturedgermanartillery.JPG/200px-Capturedgermanartillery.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Capturedgermanartillery.JPG/300px-Capturedgermanartillery.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ec/Capturedgermanartillery.JPG/400px-Capturedgermanartillery.JPG 2x" width="200"/></a><figcaption>Германское орудие <i><a href="/wiki/7.7_cm_FK_96_n.A." title="7.7 cm FK 96 n.A.">7.7 cm FK 96 n.A.</a></i>, захваченное в битве при Амьене.</figcaption></figure>
<p>Успех операции был обусловлен удачным выбором участка прорыва, где германская оборона имела наименьшую глубину. Успех был также обусловлен огромным превосходством союзников в силе, тщательной подготовкой, внезапностью удара и массированным применением танков. Большое значение для результатов операции имел результат взаимодействия мобильных сил британцев - танков и конницы<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup>. В ходе Амьенской операции впервые в истории танковых войск было осуществлено нечто похожее на оперативный прорыв - оторвавшись от пехоты и сопровождаемые конницей, танки провели рейд по тылам германских войск<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup>.
</p><p>Однако, итог операции исчерпывался практически результатом первых двух-трёх дней наступления. Продвижение с интервалами по времени, дало возможность германскому командованию подтянуть резервы, вследствие чего союзники понесли ощутимые потери. Союзникам прежде всего из-за отсутствия взаимодействия родов войск (прежде всего кавалерии и танков) не удалось превратить тактический прорыв фронта в оперативный.
</p><p>После Амьенской операции союзные войска приступили к расширению фронта наступления на флангах продвинувшихся вперёд 4-й английской, 1-й и 3-й французских армий и вытеснению противника на позицию <a href="/wiki/%D0%9B%D0%B8%D0%BD%D0%B8%D1%8F_%D0%93%D0%B8%D0%BD%D0%B4%D0%B5%D0%BD%D0%B1%D1%83%D1%80%D0%B3%D0%B0" title="Линия Гинденбурга">Зигфрида</a>. К северу от Соммы предполагалось осуществить наступление 3-й английской армии, южнее Соммы, в наступление переходила 10-я французская армия.<sup class="reference" id="cite_ref-autogenerated1_19-0"><a href="#cite_note-autogenerated1-19">[19]</a></sup>
</p><p>Германское командование приняло сугубо оборонительный план войны. «Ни пяди земли не оставлять без ожесточённой борьбы» — таков был приказ германским войскам.<sup class="reference" id="cite_ref-Амьен_20-0"><a href="#cite_note-Амьен-20">[20]</a></sup>
</p>
<h3><span id=".D0.9F.D0.BE.D0.BB.D0.B8.D1.82.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.B8.D0.B5_.D0.B8.D1.82.D0.BE.D0.B3.D0.B8_.D0.BD.D0.B0.D1.81.D1.82.D1.83.D0.BF.D0.BB.D0.B5.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="Политические_итоги_наступления">Политические итоги наступления</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%90%D0%BC%D1%8C%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F&amp;veaction=edit&amp;section=13" title="Редактировать раздел «Политические итоги наступления»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D0%BC%D1%8C%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F&amp;action=edit&amp;section=13" title="Редактировать код раздела «Политические итоги наступления»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Победа под Амьеном окончательно закрепила стратегическую инициативу за <a href="/wiki/%D0%90%D0%BD%D1%82%D0%B0%D0%BD%D1%82%D0%B0" title="Антанта">Антантой</a>. После Амьенской операции <a href="/wiki/%D0%9B%D1%8E%D0%B4%D0%B5%D0%BD%D0%B4%D0%BE%D1%80%D1%84,_%D0%AD%D1%80%D0%B8%D1%85" title="Людендорф, Эрих">Эрих Людендорф</a> писал:
</p>
<div class="ts-Цитата"><blockquote><p><a href="/wiki/8_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="8 августа">8 августа</a> <a href="/wiki/1918_%D0%B3%D0%BE%D0%B4" title="1918 год">1918 года</a> представляет самый чёрный день германской армии в истории мировой войны.<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup></p></blockquote></div><p> После наступления союзников падение боевого духа и желание закончить войну в немецких войсках ускорилось. В частях, предназначенных для переброски под Амьен, началось брожение, имелись случаи массового дезертирства.<sup class="reference" id="cite_ref-Амьен_20-1"><a href="#cite_note-Амьен-20">[20]</a></sup>
</p><p>В этих условиях германскому руководству стало ясно, что надежда на победу рухнула и дальнейшее продолжение войны безнадёжно.<sup class="reference" id="cite_ref-Hindenburg_22-0"><a href="#cite_note-Hindenburg-22">[22]</a></sup> Нужно было принимать чрезвычайные меры. <a href="/wiki/13_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="13 августа">13 августа</a> <a href="/wiki/1918_%D0%B3%D0%BE%D0%B4" title="1918 год">1918 года</a> в ставке верховного командования в <a href="/wiki/%D0%A1%D0%BF%D0%B0" title="Спа">Спа</a> состоялось совещание германского командования, <a href="/wiki/%D0%93%D0%B5%D1%80%D1%82%D0%BB%D0%B8%D0%BD%D0%B3,_%D0%93%D0%B5%D0%BE%D1%80%D0%B3_%D1%84%D0%BE%D0%BD" title="Гертлинг, Георг фон">канцлера Гертлинга</a> и статс-секретаря ведомства иностранных дел <a class="new" href="/w/index.php?title=%D0%93%D0%B8%D0%BD%D1%86%D0%B5,_%D0%9F%D0%B0%D1%83%D0%BB%D1%8C_%D1%84%D0%BE%D0%BD&amp;action=edit&amp;redlink=1" title="Гинце, Пауль фон (страница отсутствует)">Гинце</a>. Людендорф заявил о том, что германская армия уже не в состоянии сломить противника наступлением; добиться мира оборонительными действиями, несмотря на подводную войну, также невозможно, а поэтому для окончания войны следует перейти к мирным переговорам. В <a href="/wiki/%D0%A1%D0%BF%D0%B0" title="Спа">Спа</a> также прибыли представители союзной Австро-Венгрии: император <a class="mw-redirect" href="/wiki/%D0%9A%D0%B0%D1%80%D0%BB_I_(%D0%B0%D0%B2%D1%81%D1%82%D1%80%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D0%BE%D1%80)" title="Карл I (австрийский император)">Карл I</a>, министр иностранных дел граф фон Буриан и начальник австрийского Генерального штаба <a class="mw-redirect" href="/wiki/%D0%A8%D1%82%D1%80%D0%B0%D1%83%D1%81%D1%81%D0%B5%D0%BD%D0%B1%D1%83%D1%80%D0%B3,_%D0%90%D1%80%D1%82%D1%83%D1%80_%D0%90%D1%80%D1%86_%D1%84%D0%BE%D0%BD" title="Штрауссенбург, Артур Арц фон">Арц фон Штрауссенбург</a>.<sup class="reference" id="cite_ref-Амьен_20-2"><a href="#cite_note-Амьен-20">[20]</a></sup>
</p><p>Однако переговоры с представителями <a href="/wiki/%D0%90%D0%BD%D1%82%D0%B0%D0%BD%D1%82%D0%B0" title="Антанта">Антанты</a> не были начаты. <a href="/wiki/%D0%93%D0%B8%D0%BD%D0%B4%D0%B5%D0%BD%D0%B1%D1%83%D1%80%D0%B3,_%D0%9F%D0%B0%D1%83%D0%BB%D1%8C_%D1%84%D0%BE%D0%BD" title="Гинденбург, Пауль фон">Гинденбург</a> ещё надеялся, что германские войска не будут вытеснены с территории <a class="mw-redirect" href="/wiki/%D0%A2%D1%80%D0%B5%D1%82%D1%8C%D1%8F_%D0%A0%D0%B5%D1%81%D0%BF%D1%83%D0%B1%D0%BB%D0%B8%D0%BA%D0%B0" title="Третья Республика">Франции</a> и <a href="/wiki/%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D1%8F" title="Бельгия">Бельгии</a><sup class="reference" id="cite_ref-Hindenburg_22-1"><a href="#cite_note-Hindenburg-22">[22]</a></sup>, благодаря чему удастся заключить выгодное перемирие, однако эти надежды не оправдались.
</p>
