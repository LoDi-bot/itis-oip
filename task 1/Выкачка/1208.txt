<p><b><a class="mw-redirect" href="/wiki/%D0%A8%D1%82%D1%83%D1%80%D0%BC_(%D0%B0%D1%82%D0%B0%D0%BA%D0%B0)" title="Штурм (атака)">Шту́рм</a> <a href="/wiki/%D0%9B%D1%8C%D0%B5%D0%B6" title="Льеж">Лье́жа</a></b> (<a href="/wiki/5_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="5 августа">5 августа</a> <a class="mw-redirect" href="/wiki/1914" title="1914">1914</a> — <a href="/wiki/16_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="16 августа">16 августа</a> <a class="mw-redirect" href="/wiki/1914" title="1914">1914</a>) — военная операция во время <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B2%D0%B0%D1%8F_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Первая мировая война">Первой мировой войны</a>, в ходе которой немецкая армия штурмом взяла укреплённую бельгийскую <a class="mw-redirect" href="/wiki/%D0%9A%D1%80%D0%B5%D0%BF%D0%BE%D1%81%D1%82%D1%8C_%D0%9B%D1%8C%D0%B5%D0%B6" title="Крепость Льеж">крепость Льеж</a>.
</p><p>После начала <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B2%D0%B0%D1%8F_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Первая мировая война">Первой мировой войны</a> <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D1%81%D0%BA%D0%B0%D1%8F_%D0%B0%D1%80%D0%BC%D0%B8%D1%8F" title="Германская имперская армия">германская армия</a> вторглась на территорию Бельгии и начала штурм крепости Льеж, которая прикрывала важнейшие переправы через <a class="mw-redirect" href="/wiki/%D0%9C%D0%B0%D0%B0%D1%81_(%D1%80%D0%B5%D0%BA%D0%B0)" title="Маас (река)">реку Маас</a>. Первые атаки не принесли результата, однако после подвоза тяжёлой артиллерии и отхода основных бельгийских сил из города немецкие войска сумели захватить крепость.
</p>

<p>Согласно <a href="/wiki/%D0%9F%D0%BB%D0%B0%D0%BD_%D0%A8%D0%BB%D0%B8%D1%84%D1%84%D0%B5%D0%BD%D0%B0" title="План Шлиффена">германскому плану войны</a>, разработанному начальником Генерального Штаба германской армии <a href="/wiki/%D0%A8%D0%BB%D0%B8%D1%84%D1%84%D0%B5%D0%BD,_%D0%90%D0%BB%D1%8C%D1%84%D1%80%D0%B5%D0%B4_%D1%84%D0%BE%D0%BD" title="Шлиффен, Альфред фон">Шлиффеном</a> ещё в <a href="/wiki/1905_%D0%B3%D0%BE%D0%B4" title="1905 год">1905 году</a>, <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D1%81%D0%BA%D0%B0%D1%8F_%D0%B0%D1%80%D0%BC%D0%B8%D1%8F" title="Германская имперская армия">германская армия</a> в <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B2%D0%B0%D1%8F_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Первая мировая война">предстоящей войне</a> должна провести <a href="/wiki/%D0%91%D0%BB%D0%B8%D1%86%D0%BA%D1%80%D0%B8%D0%B3" title="Блицкриг">блицкриг</a>, используя территорию Бельгии для обхода основных оборонительных рубежей французов и охвата французской армии с фланга<sup class="reference" id="cite_ref-Колен_4-0"><a href="#cite_note-Колен-4">[4]</a></sup>. В итоге германское командование планировало нанести сокрушительный удар по <a class="mw-redirect" href="/wiki/%D0%A2%D1%80%D0%B5%D1%82%D1%8C%D1%8F_%D0%A0%D0%B5%D1%81%D0%BF%D1%83%D0%B1%D0%BB%D0%B8%D0%BA%D0%B0" title="Третья Республика">Франции</a> через территорию <a href="/wiki/%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D1%8F" title="Бельгия">Бельгии</a>. Таким образом Бельгия оказывалась втянутой в предстоящую войну.
</p><p>После начала Первой мировой войны <a href="/wiki/%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D1%8F" title="Бельгия">Бельгия</a> заявила о своём <a href="/wiki/%D0%9D%D0%B5%D0%B9%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D1%82%D0%B5%D1%82" title="Нейтралитет">нейтралитете</a>. Однако <a href="/wiki/2_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="2 августа">2 августа</a> <a href="/wiki/1914_%D0%B3%D0%BE%D0%B4" title="1914 год">1914 года</a> <a href="/wiki/%D0%91%D0%B5%D1%80%D0%BB%D0%B8%D0%BD" title="Берлин">Берлин</a> предъявил <a href="/wiki/%D0%A3%D0%BB%D1%8C%D1%82%D0%B8%D0%BC%D0%B0%D1%82%D1%83%D0%BC" title="Ультиматум">ультиматум</a> бельгийскому правительству с требованием пропустить через территорию Бельгии <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D1%81%D0%BA%D0%B0%D1%8F_%D0%B0%D1%80%D0%BC%D0%B8%D1%8F" title="Германская имперская армия">германские войска</a>, на ответ отводилось 12 часов. На следующий день бельгийское руководство ответило решительным отказом на требование немцев, после чего <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%B8%D1%8F" title="Германия">Германия</a> <a href="/wiki/%D0%9E%D0%B1%D1%8A%D1%8F%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5_%D0%B2%D0%BE%D0%B9%D0%BD%D1%8B" title="Объявление войны">объявила</a> Бельгии <a href="/wiki/%D0%92%D0%BE%D0%B9%D0%BD%D0%B0" title="Война">войну</a>. Таким образом, Бельгия была вынуждена вступить в Первую мировую войну, несмотря на провозглашённый <a href="/wiki/%D0%9D%D0%B5%D0%B9%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D1%82%D0%B5%D1%82" title="Нейтралитет">нейтралитет</a>.
</p><p>Являясь гарантом бельгийского нейтралитета, <a href="/wiki/4_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="4 августа">4 августа</a> войну Германии объявила <a href="/wiki/%D0%91%D1%80%D0%B8%D1%82%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F" title="Британская империя">Британская империя</a>. В Бельгии была объявлена <a href="/wiki/%D0%9C%D0%BE%D0%B1%D0%B8%D0%BB%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F" title="Мобилизация">мобилизация</a>. Армия страны готовилась к обороне против наступающих германских войск, которые уже <a href="/wiki/2_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="2 августа">2 августа</a> <a href="/wiki/%D0%9E%D0%BA%D0%BA%D1%83%D0%BF%D0%B0%D1%86%D0%B8%D1%8F" title="Оккупация">оккупировали</a> <a href="/wiki/%D0%9B%D1%8E%D0%BA%D1%81%D0%B5%D0%BC%D0%B1%D1%83%D1%80%D0%B3" title="Люксембург">Люксембург</a>.
</p><p><a href="/wiki/4_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="4 августа">4 августа</a> германские войска уже пересекли германо-бельгийскую границу и вторглись в Бельгию. Завязались первые бои между немецкой кавалерией и <a class="new" href="/w/index.php?title=12-%D1%8F_%D0%BF%D0%B5%D1%85%D0%BE%D1%82%D0%BD%D0%B0%D1%8F_%D0%B1%D1%80%D0%B8%D0%B3%D0%B0%D0%B4%D0%B0&amp;action=edit&amp;redlink=1" title="12-я пехотная бригада (страница отсутствует)">12-й пехотной бригадой</a> <a href="/wiki/%D0%92%D0%BE%D0%BE%D1%80%D1%83%D0%B6%D1%91%D0%BD%D0%BD%D1%8B%D0%B5_%D1%81%D0%B8%D0%BB%D1%8B_%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D0%B8" title="Вооружённые силы Бельгии">бельгийской армии</a>. Немецкие войска двинулись в направлении <a href="/wiki/%D0%9B%D1%8C%D0%B5%D0%B6" title="Льеж">Льежа</a>, который находился в 30 километрах от границы и прикрывал переправы через <a class="mw-redirect" href="/wiki/%D0%9C%D0%B0%D0%B0%D1%81_(%D1%80%D0%B5%D0%BA%D0%B0)" title="Маас (река)">реку Маас</a>. Бельгийцами были взорваны все <a href="/wiki/%D0%9C%D0%BE%D1%81%D1%82" title="Мост">мосты</a> через Маас. <a href="/wiki/%D0%9D%D0%B5%D0%BC%D1%86%D1%8B" title="Немцы">Немцы</a>, выйдя к взорванным мостам, принялись наводить <a href="/wiki/%D0%9F%D0%BE%D0%BD%D1%82%D0%BE%D0%BD%D0%BD%D1%8B%D0%B9_%D0%BC%D0%BE%D1%81%D1%82" title="Понтонный мост">понтонные мосты</a>, однако попали под обстрел бельгийцев. После этой неудачи к германскому командованию приходит осознание того, что Льеж им обязательно придётся штурмовать. Таким образом, для того, чтобы продолжить наступление, германским войскам было необходимо как можно скорее захватить Льеж и форсировать Маас<sup class="reference" id="cite_ref-Ребольд_5-0"><a href="#cite_note-Ребольд-5">[5]</a></sup>.
</p>
<h2><span id=".D0.A3.D0.BA.D1.80.D0.B5.D0.BF.D0.BB.D0.B5.D0.BD.D0.B8.D1.8F_.D0.BA.D1.80.D0.B5.D0.BF.D0.BE.D1.81.D1.82.D0.B8"></span><span class="mw-headline" id="Укрепления_крепости">Укрепления крепости</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A8%D1%82%D1%83%D1%80%D0%BC_%D0%9B%D1%8C%D0%B5%D0%B6%D0%B0&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Укрепления крепости»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A8%D1%82%D1%83%D1%80%D0%BC_%D0%9B%D1%8C%D0%B5%D0%B6%D0%B0&amp;action=edit&amp;section=2" title="Редактировать код раздела «Укрепления крепости»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="hatnote navigation-not-searchable">Основная статья: <b><a class="mw-redirect" href="/wiki/%D0%9A%D1%80%D0%B5%D0%BF%D0%BE%D1%81%D1%82%D1%8C_%D0%9B%D1%8C%D0%B5%D0%B6" title="Крепость Льеж">Крепость Льеж</a></b></div>

<p>Укрепления Льежа были построены в <a class="mw-redirect" href="/wiki/1880-%D0%B5" title="1880-е">80-х годах</a> <a href="/wiki/XIX_%D0%B2%D0%B5%D0%BA" title="XIX век">XIX века</a> под руководством <a href="/wiki/%D0%98%D0%BD%D0%B6%D0%B5%D0%BD%D0%B5%D1%80" title="Инженер">инженера</a> <a href="/wiki/%D0%91%D1%80%D0%B8%D0%B0%D0%BB%D1%8C%D0%BC%D0%BE%D0%BD,_%D0%90%D0%BD%D1%80%D0%B8_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B8%D1%81" title="Бриальмон, Анри Алексис">Анри Бриальмона</a>. Оборонительные сооружения Льежа составляли 6 больших фортов и 6 малых<sup class="reference" id="cite_ref-_60eb2502c6210981_8-0"><a href="#cite_note-_60eb2502c6210981-8">[6]</a></sup>. Форты были треугольные или (в зависимости от ландшафта) четырёхугольные. Всё же, чаще всего форты строились треугольными, хотя такой тип форта подвергался критике ведущих инженеров с самого начала<sup class="reference" id="cite_ref-_807210ebc9295f23_9-0"><a href="#cite_note-_807210ebc9295f23-9">[7]</a></sup>. Форты были рассчитаны на удары самой тяжёлой на тот момент (1888 год) 210-мм артиллерии. Бетон заливался обычной массой без <a class="mw-redirect" href="/wiki/%D0%90%D1%80%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5" title="Армирование">армирования</a>. Бетонирование велось лишь днём, что вызывало слабую сцепку между слоями, уложенными в разные дни. Каждый форт был оборудован паровым <a class="mw-redirect" href="/wiki/%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%B3%D0%B5%D0%BD%D0%B5%D1%80%D0%B0%D1%82%D0%BE%D1%80" title="Электрогенератор">электрогенератором</a> для освещения, <a href="/wiki/%D0%9D%D0%B0%D1%81%D0%BE%D1%81" title="Насос">насосов</a> и <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%B6%D0%B5%D0%BA%D1%82%D0%BE%D1%80" title="Прожектор">прожекторов</a><sup class="reference" id="cite_ref-_bc4b10575250af86_10-0"><a href="#cite_note-_bc4b10575250af86-10">[8]</a></sup>. <a href="/wiki/%D0%A4%D0%BE%D1%80%D1%82" title="Форт">Форты</a> находились на обоих берегах Мааса на расстоянии 6—9 км от города. Основная часть форта находилась под землёй, на поверхности виднелась лишь насыпь с броневыми башнями, откуда вёлся артиллерийский огонь. Также имелись башни меньшего размера для скорострельных пушек и <a href="/wiki/%D0%9F%D1%83%D0%BB%D0%B5%D0%BC%D1%91%D1%82" title="Пулемёт">пулемётов</a>, глубокие <a href="/wiki/%D0%A0%D0%BE%D0%B2" title="Ров">рвы</a>, которые обстреливались из бетонных казематов. Шесть крупных фортов, а также шесть более мелких имели в общей сложности 400 <a href="/wiki/%D0%90%D1%80%D1%82%D0%B8%D0%BB%D0%BB%D0%B5%D1%80%D0%B8%D0%B9%D1%81%D0%BA%D0%BE%D0%B5_%D0%BE%D1%80%D1%83%D0%B4%D0%B8%D0%B5" title="Артиллерийское орудие">орудий</a>. Тяжёлыми орудиями были 210-мм <a href="/wiki/%D0%93%D0%B0%D1%83%D0%B1%D0%B8%D1%86%D0%B0" title="Гаубица">гаубицы</a>. Вооружение форта составляли до восьми орудий калибром 120—200 мм, три или четыре 57-мм противоштурмовых орудия. Гарнизон крупного форта состоял из 400 человек, малые форты обороняли 80-100 человек<sup class="reference" id="cite_ref-ВИЗ_1-1"><a href="#cite_note-ВИЗ-1">[1]</a></sup>.
</p>

<p>Бельгийские форты имели малые запасы <a class="mw-redirect" href="/wiki/%D0%9F%D1%80%D0%BE%D0%B2%D0%B8%D0%B7%D0%B8%D1%8F" title="Провизия">провизии</a>, только для повседневных нужд гарнизона, уборные, душевые, кухни и морг, расположенные в <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%82%D1%80%D1%8D%D1%81%D0%BA%D0%B0%D1%80%D0%BF" title="Контрэскарп">контрэскарпе</a>, что было совершенно ненадёжно в военное время. Это негативно повлияло на способность фортов выдерживать длительный штурм. Служебные помещения располагались прямо за <a href="/wiki/%D0%9A%D0%B0%D0%B7%D0%B0%D1%80%D0%BC%D0%B0" title="Казарма">казармой</a>, открыто к тыльному рву форта, с более слабой защитой, чем две «основные» стороны<sup class="reference" id="cite_ref-_bc4b0e575250ac79_11-0"><a href="#cite_note-_bc4b0e575250ac79-11">[9]</a></sup>. Тыловые стороны фортов были намеренно ослаблены — с расчётом на то, чтобы в случае взятия форта противником бельгийской армии было легче вернуть его атакой сзади. Кроме того, такая конструкция позволяла устроить естественную вентиляцию жилых и служебных помещений (а механическая вентиляция в те времена только зарождалась). Но этот теоретический расчёт на практике оказался губительным. Обстрел тяжёлой артиллерией делал тыльный ров ненадёжным, и немецкие подразделения, овладев межфортовыми промежутками, могли атаковать форты сзади<sup class="reference" id="cite_ref-_bc4b0e575250ac7d_12-0"><a href="#cite_note-_bc4b0e575250ac7d-12">[10]</a></sup>. Промежутки между фортами должны были удерживаться <a href="/wiki/%D0%9F%D0%B5%D1%85%D0%BE%D1%82%D0%B0" title="Пехота">пехотными подразделениями</a>. Однако бельгийцы не хотели давать Германии повод обвинить Бельгию в подготовке к войне, и начали работы по созданию окопов между фортами только в первый день войны — 2 августа 1914 года. В результате бельгийские солдаты оборонялись в наскоро сделанных <a href="/wiki/%D0%9E%D0%BA%D0%BE%D0%BF" title="Окоп">окопах</a>, прикрытых <a href="/wiki/%D0%9A%D0%BE%D0%BB%D1%8E%D1%87%D0%B0%D1%8F_%D0%BF%D1%80%D0%BE%D0%B2%D0%BE%D0%BB%D0%BE%D0%BA%D0%B0" title="Колючая проволока">проволочными заграждениями</a>. Работы по снесению домов (для лучшего обзора при стрельбе) и вырубке леса начались также только 2 августа.
</p><p>Основными конструктивными недостатками крепости были:
</p>
<ol><li>Отсутствие центральной ограды, что давало противнику возможность прорваться в город и оттуда угрожать всем фортам с тыла.</li>
<li>Отсутствие заблаговременной подготовки промежутков между фортами, что привело к их быстрому прорыву.</li></ol>
<p>Кроме технических, большое значение имели организационные недостатки. Эти форты:
</p>
<ol><li>Были плохо применены к местности;</li>
<li>Не могли поддерживать друг друга, поэтому легко окружались неприятелем и захватывались поодиночке;</li>
<li>Имели малую площадь, на которой весьма тесно и скученно были расположены оборонительные элементы. Это приводило к быстрому разрушению всех этих элементов и деморализовывало гарнизон, даже когда по нему начинали стрелять орудия 210-мм калибра, на который и были рассчитаны форты<sup class="reference" id="cite_ref-_02a3f6ea390d0b92_13-0"><a href="#cite_note-_02a3f6ea390d0b92-13">[11]</a></sup>.</li></ol>
<p>Помимо этих серьёзных недостатков укрепления не модернизировались и к 1914 году уже серьёзно устарели. Старые форты не могли противостоять новым образцам тяжёлых орудий. Крепостные орудия имели меньший радиус обстрела, чем осадная артиллерия, и последняя могла безнаказанно обстреливать крепость. Поэтому, несмотря на все усилия генерала Лемана по укреплению крепости, форты начали сдаваться один за другим, как только немцы начинали обстрел тяжёлыми орудиями<sup class="reference" id="cite_ref-_bc4b10575250af86_10-1"><a href="#cite_note-_bc4b10575250af86-10">[8]</a></sup>.
</p>
<h2><span id=".D0.A1.D0.B8.D0.BB.D1.8B_.D1.81.D1.82.D0.BE.D1.80.D0.BE.D0.BD"></span><span class="mw-headline" id="Силы_сторон">Силы сторон</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A8%D1%82%D1%83%D1%80%D0%BC_%D0%9B%D1%8C%D0%B5%D0%B6%D0%B0&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Силы сторон»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A8%D1%82%D1%83%D1%80%D0%BC_%D0%9B%D1%8C%D0%B5%D0%B6%D0%B0&amp;action=edit&amp;section=3" title="Редактировать код раздела «Силы сторон»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id=".D0.91.D0.B5.D0.BB.D1.8C.D0.B3.D0.B8.D1.8F"></span><span class="mw-headline" id="Бельгия">Бельгия</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A8%D1%82%D1%83%D1%80%D0%BC_%D0%9B%D1%8C%D0%B5%D0%B6%D0%B0&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Бельгия»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A8%D1%82%D1%83%D1%80%D0%BC_%D0%9B%D1%8C%D0%B5%D0%B6%D0%B0&amp;action=edit&amp;section=4" title="Редактировать код раздела «Бельгия»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>В Льеже находилась <a class="new" href="/w/index.php?title=3-%D1%8F_%D0%BF%D0%B5%D1%85%D0%BE%D1%82%D0%BD%D0%B0%D1%8F_%D0%B4%D0%B8%D0%B2%D0%B8%D0%B7%D0%B8%D1%8F_(%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D1%8F)&amp;action=edit&amp;redlink=1" title="3-я пехотная дивизия (Бельгия) (страница отсутствует)">3-я пехотная дивизия</a>, а также пехотная бригада из состава <a class="new" href="/w/index.php?title=4-%D1%8F_%D0%BF%D0%B5%D1%85%D0%BE%D1%82%D0%BD%D0%B0%D1%8F_%D0%B4%D0%B8%D0%B2%D0%B8%D0%B7%D0%B8%D1%8F_(%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D1%8F)&amp;action=edit&amp;redlink=1" title="4-я пехотная дивизия (Бельгия) (страница отсутствует)">4-й пехотной дивизии</a>. Помимо этого в Льеже находился <a href="/wiki/%D0%93%D0%B0%D1%80%D0%BD%D0%B8%D0%B7%D0%BE%D0%BD" title="Гарнизон">гарнизон</a> и другие войска, всего около 36 000 солдат и офицеров. Однако гарнизон крепости практически полностью состоял из <a class="mw-redirect" href="/wiki/%D0%92%D0%BE%D0%B5%D0%BD%D0%BD%D1%8B%D0%B5_%D1%80%D0%B5%D0%B7%D0%B5%D1%80%D0%B2%D1%8B" title="Военные резервы">резервистов</a> старших возрастов. Также в каждой роте бельгийских войск был всего лишь один <a href="/wiki/%D0%9E%D1%84%D0%B8%D1%86%D0%B5%D1%80" title="Офицер">офицер</a>. Командующим 3-й пехотной дивизии и комендантом крепости был <a href="/wiki/%D0%93%D0%B5%D0%BD%D0%B5%D1%80%D0%B0%D0%BB-%D0%BB%D0%B5%D0%B9%D1%82%D0%B5%D0%BD%D0%B0%D0%BD%D1%82" title="Генерал-лейтенант">генерал-лейтенант</a> <a href="/wiki/%D0%9B%D0%B5%D0%BC%D0%B0%D0%BD,_%D0%96%D0%B5%D1%80%D0%B0%D1%80" title="Леман, Жерар">Жерар Леман</a><sup class="reference" id="cite_ref-ВИЗ_1-2"><a href="#cite_note-ВИЗ-1">[1]</a></sup>.
</p>

<h3><span id=".D0.93.D0.B5.D1.80.D0.BC.D0.B0.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="Германия">Германия</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A8%D1%82%D1%83%D1%80%D0%BC_%D0%9B%D1%8C%D0%B5%D0%B6%D0%B0&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Германия»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A8%D1%82%D1%83%D1%80%D0%BC_%D0%9B%D1%8C%D0%B5%D0%B6%D0%B0&amp;action=edit&amp;section=5" title="Редактировать код раздела «Германия»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Германское командование для захвата Льежа создало группировку войск, которая получила название <a class="new" href="/w/index.php?title=%D0%9C%D0%B0%D0%B0%D1%81%D1%81%D0%BA%D0%B0%D1%8F_%D0%B0%D1%80%D0%BC%D0%B8%D1%8F_(%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F)&amp;action=edit&amp;redlink=1" title="Маасская армия (Германская империя) (страница отсутствует)">Маасской армии</a>, под командованием генерала <a href="/wiki/%D0%AD%D0%BC%D0%BC%D0%B8%D1%85,_%D0%9E%D1%82%D1%82%D0%BE_%D1%84%D0%BE%D0%BD" title="Эммих, Отто фон">Эммиха</a>. В состав войск, которые должны были взять крепость, вошли шесть усиленных пехотных бригад из <a href="/wiki/2-%D1%8F_%D0%B0%D1%80%D0%BC%D0%B8%D1%8F_(%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F)" title="2-я армия (Германская империя)">2-й армии</a>, <a class="new" href="/w/index.php?title=2-%D0%B9_%D0%BA%D0%B0%D0%B2%D0%B0%D0%BB%D0%B5%D1%80%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%BE%D1%80%D0%BF%D1%83%D1%81_(%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F)&amp;action=edit&amp;redlink=1" title="2-й кавалерийский корпус (Германская империя) (страница отсутствует)">2-й кавалерийский корпус</a> (три дивизии) под командованием генерала <a class="mw-redirect" href="/wiki/%D0%9C%D0%B0%D1%80%D0%B2%D0%B8%D1%86,_%D0%93%D0%B5%D0%BE%D1%80%D0%B3_%D0%B4%D0%B5%D1%80_%D1%84%D0%BE%D0%BD" title="Марвиц, Георг дер фон">Марвица</a>. Первоначально силы немцев составляли 59 800 солдат и офицеров при 124 орудиях, среди которых были 4 тяжелые <a href="/wiki/%D0%9C%D0%BE%D1%80%D1%82%D0%B8%D1%80%D0%B0" title="Мортира">мортиры</a> калибром 210 мм<sup class="reference" id="cite_ref-Der_Weltkrieg1_2-1"><a href="#cite_note-Der_Weltkrieg1-2">[2]</a></sup>. Также в осуществлении операции участвовал представитель генерального штаба <a class="mw-redirect" href="/wiki/%D0%9B%D1%8E%D0%B4%D0%B5%D0%BD%D0%B4%D0%BE%D1%80%D1%84,_%D0%AD%D1%80%D0%B8%D1%85_%D0%A4%D1%80%D0%B8%D0%B4%D1%80%D0%B8%D1%85_%D0%92%D0%B8%D0%BB%D1%8C%D0%B3%D0%B5%D0%BB%D1%8C%D0%BC" title="Людендорф, Эрих Фридрих Вильгельм">Эрих Людендорф</a>.
</p>

<h2><span id=".D0.A8.D1.82.D1.83.D1.80.D0.BC"></span><span class="mw-headline" id="Штурм">Штурм</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A8%D1%82%D1%83%D1%80%D0%BC_%D0%9B%D1%8C%D0%B5%D0%B6%D0%B0&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Штурм»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A8%D1%82%D1%83%D1%80%D0%BC_%D0%9B%D1%8C%D0%B5%D0%B6%D0%B0&amp;action=edit&amp;section=6" title="Редактировать код раздела «Штурм»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>К концу дня <a href="/wiki/5_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="5 августа">5 августа</a> германские войска подошли к линии фортов Льежа. Немецкое командование приняло решение захватить крепость внезапной атакой. 5 августа после <a href="/wiki/%D0%90%D1%80%D1%82%D0%BF%D0%BE%D0%B4%D0%B3%D0%BE%D1%82%D0%BE%D0%B2%D0%BA%D0%B0" title="Артподготовка">артиллерийской подготовки</a>, немецкие полки пошли в наступление на промежутки между фортами. Однако <a href="/wiki/%D0%90%D1%80%D1%82%D0%B8%D0%BB%D0%BB%D0%B5%D1%80%D0%B8%D1%8F" title="Артиллерия">артиллерия</a> немцев не причинила никакого вреда бельгийским фортам. <a class="mw-redirect" href="/wiki/%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D0%B9%D1%86%D1%8B" title="Бельгийцы">Бельгийцы</a> же ответным огнём нанесли атаковавшим тяжелейшие потери. В некоторых местах немцам удалось пробиться на близкое расстояние к фортам, однако огонь пулемётов уничтожил прорвавшихся<sup class="reference" id="cite_ref-ПМВ_14-0"><a href="#cite_note-ПМВ-14">[12]</a></sup>. У форта Баршон завязался <a href="/wiki/%D0%A8%D1%82%D1%8B%D0%BA#Штыковой_бой" title="Штык">штыковой бой</a>, в котором бельгийские подразделения сумели отбросить германцев. Расчет на внезапную мощную атаку немцев не оправдался, германские войска понесли тяжелые потери. Бельгийский офицер позже вспоминал подробности атаки германских войск:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p class="mw-empty-elt">
</p><p>
Они даже не старались рассредоточиться, они падали друг на друга, образуя страшную баррикаду из убитых и раненых. Мы даже стали опасаться, что она закроет нам обзор и мы не сможем вести прицельный огонь. Гора трупов уже стала огромной, и мы думали, стрелять ли прямо в неё или выходить и самим растаскивать трупы. Поверите или нет, эта настоящая стена из мёртвых и умирающих позволила немцам подползти ближе и броситься на передние скаты фортов, но им не удалось пробежать и половины пути — наши пулеметы и винтовки разом смели их прочь. Разумеется, мы тоже несли потери, но они были незначительными по сравнению с той бойней, которую мы учинили противнику.</p><div class="ts-Конец_цитаты-source">— <cite><sup class="reference" id="cite_ref-Офицер_15-0"><a href="#cite_note-Офицер-15">[13]</a></sup></cite></div></blockquote>
<figure class="mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Siege_of_liege.jpg"><img class="mw-file-element" data-file-height="640" data-file-width="874" decoding="async" height="183" src="//upload.wikimedia.org/wikipedia/commons/thumb/5/5d/Siege_of_liege.jpg/250px-Siege_of_liege.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/5/5d/Siege_of_liege.jpg/375px-Siege_of_liege.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/5/5d/Siege_of_liege.jpg/500px-Siege_of_liege.jpg 2x" width="250"/></a><figcaption>Штурм Льежа.</figcaption></figure>
<p>В <a href="/wiki/%D0%9D%D0%BE%D1%87%D1%8C" title="Ночь">ночь</a> на <a href="/wiki/6_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="6 августа">6 августа</a> немцы приняли решение провести новую атаку. Генерал Людендорф возглавил 14-ю бригаду вместо убитого командира бригады <a href="/wiki/%D0%93%D0%B5%D0%BD%D0%B5%D1%80%D0%B0%D0%BB-%D0%BC%D0%B0%D0%B9%D0%BE%D1%80" title="Генерал-майор">генерал-майора</a> фон Вюссова<sup class="reference" id="cite_ref-Штурм1_16-0"><a href="#cite_note-Штурм1-16">[14]</a></sup>. В новой атаке немцам удалось прорваться в промежутке между фортами Флерон и Эвенье. К середине дня 6 августа части 14-й германской бригады прорвались через кольцо фортов и закрепились на правом берегу Мааса. Также 6 августа впервые в <a href="/wiki/%D0%92%D0%BE%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B8%D1%81%D1%82%D0%BE%D1%80%D0%B8%D1%8F" title="Военная история">истории войн</a> немцы применили воздушный налёт. <a href="/wiki/%D0%A6%D0%B5%D0%BF%D0%BF%D0%B5%D0%BB%D0%B8%D0%BD_(%D0%B4%D0%B8%D1%80%D0%B8%D0%B6%D0%B0%D0%B1%D0%BB%D1%8C)" title="Цеппелин (дирижабль)">Цеппелин</a>, вылетевший из <a href="/wiki/%D0%9A%D1%91%D0%BB%D1%8C%D0%BD" title="Кёльн">Кёльна</a>, сбросил 13 <a href="/wiki/%D0%91%D0%BE%D0%BC%D0%B1%D0%B0" title="Бомба">бомб</a> на Льеж, убив 9 мирных жителей. Кроме того, артиллерия 14-й бригады с господствующих высот принялась обстреливать город. Эти меры были приняты для устрашения защитников Льежа<sup class="reference" id="cite_ref-Штурм2_17-0"><a href="#cite_note-Штурм2-17">[15]</a></sup>.
</p><p>Помимо этого, немцы прибегли к военной хитрости, направив в город отряд, переодетый в форму, похожую на английскую. Немцы, прибыв в Льеж на <a href="/wiki/%D0%90%D0%B2%D1%82%D0%BE%D0%BC%D0%BE%D0%B1%D0%B8%D0%BB%D1%8C" title="Автомобиль">автомобиле</a> к штабу коменданта Лемана, потребовали встречи с генералом. Однако их план провалился и они были разоблачены. После этого Леман покинул город и отправился в форт Лонсен<sup class="reference" id="cite_ref-Штурм1_16-1"><a href="#cite_note-Штурм1-16">[14]</a></sup>.
</p>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Destroyed_fortification_at_Liege.JPG"><img class="mw-file-element" data-file-height="725" data-file-width="608" decoding="async" height="238" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ed/Destroyed_fortification_at_Liege.JPG/200px-Destroyed_fortification_at_Liege.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ed/Destroyed_fortification_at_Liege.JPG/300px-Destroyed_fortification_at_Liege.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ed/Destroyed_fortification_at_Liege.JPG/400px-Destroyed_fortification_at_Liege.JPG 2x" width="200"/></a><figcaption>Фотография разрушенного форта Лонсен в Льеже.</figcaption></figure>
<p>После того как немецкой бригаде удалось прорваться через кольцо фортов, Леману стало ясно, что необходимо сохранить 3-ю пехотную дивизию. Опасаясь, что она окажется отрезанной от основных сил, генерал Леман отдал приказ об отходе 3-й дивизии на западный берег Мааса<sup class="reference" id="cite_ref-Штурм2_17-1"><a href="#cite_note-Штурм2-17">[15]</a></sup>.
</p><p>В ходе отступления бельгийских войск Людендорф вместе с адъютантом, опережая свои войска на автомобиле, сумел первым ворваться в пустеющий Льеж, после чего он проехал через весь город к цитадели, гарнизон которой, поддавшись панике, сдался ему в плен<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[16]</a></sup>.
</p><p>После того как 3-я дивизия покинула город, подразделения 14-й немецкой бригады вошли в <a href="/wiki/%D0%9B%D1%8C%D0%B5%D0%B6" title="Льеж">Льеж</a>. Таким образом, <a href="/wiki/7_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="7 августа">7 августа</a> немцам удалось захватить Льеж и несколько переправ через Маас. Однако это не принесло немцам никакой пользы, поскольку основные переправы находились под огнём фортов, которые продолжали сопротивляться<sup class="reference" id="cite_ref-Штурм3_19-0"><a href="#cite_note-Штурм3-19">[17]</a></sup>.
</p><p>После этого ещё три германские пехотные бригады прорвались в город. Для того чтобы захватить форты Льежа и начать форсирование Мааса, германское командование подтянуло ещё три армейских корпуса, доведя число своих войск у Льежа до 100 000 человек<sup class="reference" id="cite_ref-Der_Weltkrieg2_20-0"><a href="#cite_note-Der_Weltkrieg2-20">[18]</a></sup>. <a href="/wiki/10_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="10 августа">10 августа</a> <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D1%81%D0%BA%D0%B0%D1%8F_%D0%B0%D1%80%D0%BC%D0%B8%D1%8F" title="Германская имперская армия">немецкая пехота</a> захватила первый форт Баршон. 12 августа к Льежу прибыли осадные орудия <a href="/wiki/Skoda_305_mm_Model_1911" title="Skoda 305 mm Model 1911">305-мм</a> и <a href="/wiki/%D0%91%D0%BE%D0%BB%D1%8C%D1%88%D0%B0%D1%8F_%D0%91%D0%B5%D1%80%D1%82%D0%B0_(%D0%BE%D1%80%D1%83%D0%B4%D0%B8%D0%B5)" title="Большая Берта (орудие)">420-мм калибра</a><sup class="reference" id="cite_ref-_60eb2502c6210982_21-0"><a href="#cite_note-_60eb2502c6210982-21">[19]</a></sup>. Эти мощные орудия легко разрушали устаревшие укрепления фортов<sup class="reference" id="cite_ref-Der_Weltkrieg3_22-0"><a href="#cite_note-Der_Weltkrieg3-22">[20]</a></sup>. Первым пал форт Понтисс, а затем пали и все остальные форты Льежа кроме последнего — Лонсена. <a href="/wiki/16_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="16 августа">16 августа</a>, во время бомбардировки, в форте Лонсен взорвался склад боеприпасов и большая часть защитников форта погибла или была ранена. <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D1%81%D0%BA%D0%B0%D1%8F_%D0%B0%D1%80%D0%BC%D0%B8%D1%8F" title="Германская имперская армия">Германские солдаты</a>, ворвавшись внутрь, обнаружили генерала <a href="/wiki/%D0%9B%D0%B5%D0%BC%D0%B0%D0%BD,_%D0%96%D0%B5%D1%80%D0%B0%D1%80" title="Леман, Жерар">Лемана</a> в бессознательном состоянии и взяли его в <a href="/wiki/%D0%92%D0%BE%D0%B5%D0%BD%D0%BD%D0%BE%D0%BF%D0%BB%D0%B5%D0%BD%D0%BD%D1%8B%D0%B5_%D0%9F%D0%B5%D1%80%D0%B2%D0%BE%D0%B9_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%BE%D0%B9_%D0%B2%D0%BE%D0%B9%D0%BD%D1%8B" title="Военнопленные Первой мировой войны">плен</a><sup class="reference" id="cite_ref-Штурм4_23-0"><a href="#cite_note-Штурм4-23">[21]</a></sup>.
</p><p>За взятие Льежа генералы Людендорф и Эммих были удостоены <a class="mw-redirect" href="/wiki/%D0%9E%D1%80%D0%B4%D0%B5%D0%BD_%C2%ABPour_le_M%C3%A9rite%C2%BB_(%D0%9F%D1%80%D1%83%D1%81%D1%81%D0%B8%D1%8F)" title="Орден «Pour le Mérite» (Пруссия)">ордена «Pour le Mérite» (За заслуги)</a><sup class="reference" id="cite_ref-Штурм4_23-1"><a href="#cite_note-Штурм4-23">[21]</a></sup>.

<p>После окончательного падения Льежа <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D1%81%D0%BA%D0%B0%D1%8F_%D0%B0%D1%80%D0%BC%D0%B8%D1%8F" title="Германская имперская армия">германские войска</a> <a href="/wiki/17_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="17 августа">17 августа</a> перешли в решительное наступление к франко-бельгийской границе. Началось движение 1-й, 2-й и 3-й германских армий через Бельгию. <a href="/wiki/1-%D1%8F_%D0%B0%D1%80%D0%BC%D0%B8%D1%8F_(%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F)" title="1-я армия (Германская империя)">1-я армия</a> <a href="/wiki/%D0%9A%D0%BB%D1%8E%D0%BA,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D1%84%D0%BE%D0%BD" title="Клюк, Александр фон">фон Клюка</a> двинулась на <a href="/wiki/%D0%91%D1%80%D1%8E%D1%81%D1%81%D0%B5%D0%BB%D1%8C" title="Брюссель">Брюссель</a>. Слева от неё пошла к французской границе <a href="/wiki/2-%D1%8F_%D0%B0%D1%80%D0%BC%D0%B8%D1%8F_(%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F)" title="2-я армия (Германская империя)">2-я армия</a> <a href="/wiki/%D0%91%D1%8E%D0%BB%D0%BE%D0%B2,_%D0%9A%D0%B0%D1%80%D0%BB_%D1%84%D0%BE%D0%BD" title="Бюлов, Карл фон">фон Бюлова</a>, ещё левее продолжила наступление <a href="/wiki/3-%D1%8F_%D0%B0%D1%80%D0%BC%D0%B8%D1%8F_(%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F)" title="3-я армия (Германская империя)">3-я армия</a> <a href="/wiki/%D0%A5%D0%B0%D1%83%D0%B7%D0%B5%D0%BD,_%D0%9C%D0%B0%D0%BA%D1%81_%D1%84%D0%BE%D0%BD" title="Хаузен, Макс фон">фон Хаузена</a>. Германское командование планировало начать общее наступление <a href="/wiki/15_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="15 августа">15 августа</a>, поэтому сопротивление Льежа задержало немцев лишь на 2 дня, а не 2 недели, вопреки распространенному мнению<sup class="reference" id="cite_ref-Штурм4_23-2"><a href="#cite_note-Штурм4-23">[21]</a></sup>.
</p><p>После падения Льежа, части 1-й и 2-й армии начали форсирование Мааса. После этого германское командование приступило к реализации плана наступления на Францию через территорию Бельгии. Бельгийская армия прикрывала <a href="/wiki/%D0%91%D1%80%D1%8E%D1%81%D1%81%D0%B5%D0%BB%D1%8C" title="Брюссель">Брюссель</a>, часть сил находились на <a href="/wiki/%D0%94%D0%B8%D0%BB%D1%8C_(%D1%80%D0%B5%D0%BA%D0%B0)" title="Диль (река)">реке Диль</a>, одна пехотная дивизия оставалась для защиты <a href="/wiki/%D0%9A%D1%80%D0%B5%D0%BF%D0%BE%D1%81%D1%82%D1%8C_%D0%9D%D0%B0%D0%BC%D1%8E%D1%80" title="Крепость Намюр">Намюра</a>. К <a href="/wiki/18_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="18 августа">18 августа</a> между наступающими германскими армиями и бельгийскими войсками завязались бои, в результате которых бельгийские части отошли за реку Диль. <a href="/wiki/20_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="20 августа">20 августа</a> отступающая бельгийская армия подошла к <a href="/wiki/%D0%90%D0%BD%D1%82%D0%B2%D0%B5%D1%80%D0%BF%D0%B5%D0%BD" title="Антверпен">Антверпену</a>. Германское командование выделило часть сил из состава 1-й армии для <a href="/wiki/%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%90%D0%BD%D1%82%D0%B2%D0%B5%D1%80%D0%BF%D0%B5%D0%BD%D0%B0_(1914)" title="Осада Антверпена (1914)">осады Антверпена</a>. В это время основные силы германских войск захватили столицу Бельгии — Брюссель. Таким образом 1-я, 2-я и 3-я немецкие армии преодолели значительную часть территории Бельгии и вышли к на линию Брюссель — Намюр — <a href="/wiki/%D0%94%D0%B8%D0%BD%D0%B0%D0%BD_(%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D1%8F)" title="Динан (Бельгия)">Динан</a>, готовясь вступить в <a class="mw-redirect" href="/wiki/%D0%9F%D0%BE%D0%B3%D1%80%D0%B0%D0%BD%D0%B8%D1%87%D0%BD%D0%BE%D0%B5_%D1%81%D1%80%D0%B0%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5_(1914)" title="Пограничное сражение (1914)">сражение</a> с основными силами <a href="/wiki/%D0%92%D0%BE%D0%BE%D1%80%D1%83%D0%B6%D1%91%D0%BD%D0%BD%D1%8B%D0%B5_%D1%81%D0%B8%D0%BB%D1%8B_%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D0%B8" title="Вооружённые силы Франции">французской армии</a><sup class="reference" id="cite_ref-Новицкий_24-0"><a href="#cite_note-Новицкий-24">[22]</a></sup>.
</p>
<h2><span id=".D0.98.D1.82.D0.BE.D0.B3.D0.B8"></span><span class="mw-headline" id="Итоги">Итоги</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A8%D1%82%D1%83%D1%80%D0%BC_%D0%9B%D1%8C%D0%B5%D0%B6%D0%B0&amp;veaction=edit&amp;section=8" title="Редактировать раздел «Итоги»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A8%D1%82%D1%83%D1%80%D0%BC_%D0%9B%D1%8C%D0%B5%D0%B6%D0%B0&amp;action=edit&amp;section=8" title="Редактировать код раздела «Итоги»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Во время осады укрепления выполнили свою роль, задержав немецкую армию на время, достаточное для мобилизации армий Франции и Бельгии. Осада вскрыла недостатки устройства фортов и бельгийской стратегии вообще. Сами форты страдали от плохого понимания бетонной технологии строителями, а также от абсолютно недостаточной защищённости гарнизона и складов боеприпасов от огня сверхтяжёлой артиллерии. Воздух, непригодный для дыхания от бомбардировок, дыма собственных орудий и человеческих отходов был причиной сдачи большинства позиций<sup class="reference" id="cite_ref-_958f9fadcd92f14c_25-0"><a href="#cite_note-_958f9fadcd92f14c-25">[23]</a></sup>. И всё же, задержка возле крепости позволила Бельгии и, что более важно, Франции провести мобилизацию сил. Если бы немцам удалось захватить Льеж так быстро, как они рассчитывали, то немецкая армия оказалась бы под <a href="/wiki/%D0%9F%D0%B0%D1%80%D0%B8%D0%B6" title="Париж">Парижем</a> прежде, чем французы смогли бы организовать его защиту в <a class="mw-redirect" href="/wiki/%D0%9F%D0%B5%D1%80%D0%B2%D0%B0%D1%8F_%D0%B1%D0%B8%D1%82%D0%B2%D0%B0_%D0%BD%D0%B0_%D0%9C%D0%B0%D1%80%D0%BD%D0%B5" title="Первая битва на Марне">Первой битве на Марне</a><sup class="reference" id="cite_ref-_f5659296742232de_26-0"><a href="#cite_note-_f5659296742232de-26">[24]</a></sup>.
</p>
