<figure class="mw-default-size mw-halign-right" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Carl_Jacobi.jpg"><img class="mw-file-element" data-file-height="1008" data-file-width="879" decoding="async" height="252" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/90/Carl_Jacobi.jpg/220px-Carl_Jacobi.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/90/Carl_Jacobi.jpg/330px-Carl_Jacobi.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/9/90/Carl_Jacobi.jpg/440px-Carl_Jacobi.jpg 2x" width="220"/></a><figcaption>Карл Густав Якоб Якоби (1804—1851).</figcaption></figure>
<p><b>Си́мвол Я́коби</b> — <a href="/wiki/%D0%A2%D0%B5%D0%BE%D1%80%D0%B8%D1%8F_%D1%87%D0%B8%D1%81%D0%B5%D0%BB" title="Теория чисел">теоретико-числовая</a> функция двух <a class="mw-redirect" href="/wiki/%D0%90%D1%80%D0%B3%D1%83%D0%BC%D0%B5%D0%BD%D1%82_%D1%84%D1%83%D0%BD%D0%BA%D1%86%D0%B8%D0%B8" title="Аргумент функции">аргументов</a>, введённая <a href="/wiki/%D0%AF%D0%BA%D0%BE%D0%B1%D0%B8,_%D0%9A%D0%B0%D1%80%D0%BB_%D0%93%D1%83%D1%81%D1%82%D0%B0%D0%B2_%D0%AF%D0%BA%D0%BE%D0%B1" title="Якоби, Карл Густав Якоб">К. Якоби</a> в <a href="/wiki/1837_%D0%B3%D0%BE%D0%B4" title="1837 год">1837 году</a>. Является квадратичным <a href="/wiki/%D0%A5%D0%B0%D1%80%D0%B0%D0%BA%D1%82%D0%B5%D1%80_(%D1%82%D0%B5%D0%BE%D1%80%D0%B8%D1%8F_%D1%87%D0%B8%D1%81%D0%B5%D0%BB)" title="Характер (теория чисел)">характером</a> в <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%BB%D1%8C%D1%86%D0%BE_%D0%B2%D1%8B%D1%87%D0%B5%D1%82%D0%BE%D0%B2" title="Кольцо вычетов">кольце вычетов</a>.
</p><p>Символ Якоби обобщает <a href="/wiki/%D0%A1%D0%B8%D0%BC%D0%B2%D0%BE%D0%BB_%D0%9B%D0%B5%D0%B6%D0%B0%D0%BD%D0%B4%D1%80%D0%B0" title="Символ Лежандра">символ Лежандра</a> на все <a class="mw-redirect" href="/wiki/%D0%9D%D0%B5%D1%87%D1%91%D1%82%D0%BD%D1%8B%D0%B5_%D1%87%D0%B8%D1%81%D0%BB%D0%B0" title="Нечётные числа">нечётные числа</a>, большие единицы. <a href="/wiki/%D0%A1%D0%B8%D0%BC%D0%B2%D0%BE%D0%BB_%D0%9A%D1%80%D0%BE%D0%BD%D0%B5%D0%BA%D0%B5%D1%80%D0%B0_%E2%80%94_%D0%AF%D0%BA%D0%BE%D0%B1%D0%B8" title="Символ Кронекера — Якоби">Символ Кронекера — Якоби</a>, в свою очередь, обобщает символ Якоби на все <a class="mw-redirect" href="/wiki/%D0%A6%D0%B5%D0%BB%D1%8B%D0%B5_%D1%87%D0%B8%D1%81%D0%BB%D0%B0" title="Целые числа">целые числа</a>, но в практических задачах символ Якоби играет гораздо более важную роль, чем символ Кронекера — Якоби.
</p>

<h3><span id=".D0.9E_.D0.B2.D1.8B.D1.87.D0.B8.D1.81.D0.BB.D0.B5.D0.BD.D0.B8.D0.B8"></span><span class="mw-headline" id="О_вычислении">О вычислении</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B8%D0%BC%D0%B2%D0%BE%D0%BB_%D0%AF%D0%BA%D0%BE%D0%B1%D0%B8&amp;veaction=edit&amp;section=4" title="Редактировать раздел «О вычислении»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B8%D0%BC%D0%B2%D0%BE%D0%BB_%D0%AF%D0%BA%D0%BE%D0%B1%D0%B8&amp;action=edit&amp;section=4" title="Редактировать код раздела «О вычислении»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Символ Якоби практически никогда не вычисляют по определению. Чаще всего для вычисления используют свойства символа Якоби, главным образом — квадратичный закон взаимности.
</p><p>Более того, несмотря на то, что символ Якоби является обобщением <a href="/wiki/%D0%A1%D0%B8%D0%BC%D0%B2%D0%BE%D0%BB_%D0%9B%D0%B5%D0%B6%D0%B0%D0%BD%D0%B4%D1%80%D0%B0" title="Символ Лежандра">символа Лежандра</a> и определяется через него, чаще именно символ Лежандра вычисляют с помощью символа Якоби, а не наоборот. См. <a class="mw-selflink-fragment" href="#Пример_вычисления">Пример</a>
</p>

<h3><span id=".D0.9E.D1.81.D0.BD.D0.BE.D0.B2.D0.BD.D0.B0.D1.8F_.D0.B8.D0.B4.D0.B5.D1.8F"></span><span class="mw-headline" id="Основная_идея">Основная идея</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B8%D0%BC%D0%B2%D0%BE%D0%BB_%D0%AF%D0%BA%D0%BE%D0%B1%D0%B8&amp;veaction=edit&amp;section=10" title="Редактировать раздел «Основная идея»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B8%D0%BC%D0%B2%D0%BE%D0%BB_%D0%AF%D0%BA%D0%BE%D0%B1%D0%B8&amp;action=edit&amp;section=10" title="Редактировать код раздела «Основная идея»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Ключевое используемое при вычислении свойство символа Якоби — квадратичный закон взаимности. Благодаря ему алгоритм похож на <a href="/wiki/%D0%90%D0%BB%D0%B3%D0%BE%D1%80%D0%B8%D1%82%D0%BC_%D0%95%D0%B2%D0%BA%D0%BB%D0%B8%D0%B4%D0%B0" title="Алгоритм Евклида">алгоритм Евклида</a> нахождения <a href="/wiki/%D0%9D%D0%B0%D0%B8%D0%B1%D0%BE%D0%BB%D1%8C%D1%88%D0%B8%D0%B9_%D0%BE%D0%B1%D1%89%D0%B8%D0%B9_%D0%B4%D0%B5%D0%BB%D0%B8%D1%82%D0%B5%D0%BB%D1%8C" title="Наибольший общий делитель">наибольшего общего делителя</a> (НОД) двух чисел, в котором тоже аргументы на каждом шаге меняются местами. Аналогично <a href="/wiki/%D0%90%D0%BB%D0%B3%D0%BE%D1%80%D0%B8%D1%82%D0%BC_%D0%95%D0%B2%D0%BA%D0%BB%D0%B8%D0%B4%D0%B0" title="Алгоритм Евклида">алгоритму Евклида</a>, при перестановке аргументов больший заменяется на <a href="/wiki/%D0%94%D0%B5%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5_%D1%81_%D0%BE%D1%81%D1%82%D0%B0%D1%82%D0%BA%D0%BE%D0%BC" title="Деление с остатком">остаток от деления</a> на меньший. Это возможно благодаря <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B8%D0%BE%D0%B4%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%84%D1%83%D0%BD%D0%BA%D1%86%D0%B8%D1%8F" title="Периодическая функция">периодичности</a> символа Якоби. Однако, поскольку символ Якоби определён только при условии нечётности второго аргумента, то до перестановки выделяется чётная часть первого аргумента.
</p>
<h3><span id=".D0.A4.D0.BE.D1.80.D0.BC.D0.B0.D0.BB.D1.8C.D0.BD.D0.BE.D0.B5_.D0.BE.D0.BF.D0.B8.D1.81.D0.B0.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Формальное_описание">Формальное описание</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A1%D0%B8%D0%BC%D0%B2%D0%BE%D0%BB_%D0%AF%D0%BA%D0%BE%D0%B1%D0%B8&amp;veaction=edit&amp;section=11" title="Редактировать раздел «Формальное описание»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A1%D0%B8%D0%BC%D0%B2%D0%BE%D0%BB_%D0%AF%D0%BA%D0%BE%D0%B1%D0%B8&amp;action=edit&amp;section=11" title="Редактировать код раздела «Формальное описание»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
