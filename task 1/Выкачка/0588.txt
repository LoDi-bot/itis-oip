<h2><span id=".D0.9F.D1.80.D0.BE.D0.B8.D1.81.D1.85.D0.BE.D0.B6.D0.B4.D0.B5.D0.BD.D0.B8.D0.B5_.D0.BD.D0.B0.D0.B7.D0.B2.D0.B0.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="Происхождение_названия">Происхождение названия</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%BA&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Происхождение названия»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%BA&amp;action=edit&amp;section=1" title="Редактировать код раздела «Происхождение названия»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Как свидетельствуют <a href="/wiki/%D0%9B%D0%B5%D1%82%D0%BE%D0%BF%D0%B8%D1%81%D1%8C" title="Летопись">летописи</a> и другие исторические документы, в <a class="mw-redirect" href="/wiki/XIV" title="XIV">XIV</a>—<a href="/wiki/XV_%D0%B2%D0%B5%D0%BA" title="XV век">XV веках</a> существовал именно «Тиверский городок». Название «Тиверск» начинает встречаться в источниках лишь с <a href="/wiki/XIX_%D0%B2%D0%B5%D0%BA" title="XIX век">XIX века</a>.
</p><p>Изначальное карельское название городища — Тиури, Тиурин-линна (от <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Карельский язык">карел.</a> <span lang="krl" style="font-style:italic;">tiuri, tiveri</span> — водопад, перекат и <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Карельский язык">карел.</a> <span lang="krl" style="font-style:italic;">linna</span> — город).
</p>
<h2><span id=".D0.98.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.8F"></span><span class="mw-headline" id="История">История</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%BA&amp;veaction=edit&amp;section=2" title="Редактировать раздел «История»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%BA&amp;action=edit&amp;section=2" title="Редактировать код раздела «История»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>По мнению ряда исследователей, изначально <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B5%D0%BB%D1%8B" title="Карелы">карельское</a> городище Тиурин-линна возникло на острове Тиури посреди <a href="/wiki/%D0%92%D1%83%D0%BE%D0%BA%D1%81%D0%B0_(%D1%80%D0%B5%D0%BA%D0%B0)" title="Вуокса (река)">Вуоксы</a> вблизи Тиверских порогов не позднее <a href="/wiki/IX_%D0%B2%D0%B5%D0%BA" title="IX век">IX века</a>. Поселение являлось столицей карельской родовой группировки <a class="new" href="/w/index.php?title=%D0%A2%D0%B8%D0%B2%D1%80%D1%83%D0%BB%D1%8C%D1%86%D1%8B&amp;action=edit&amp;redlink=1" title="Тиврульцы (страница отсутствует)">тиврульцев</a> — одного из так называемых «<a href="/wiki/%D0%9A%D0%BE%D1%80%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B5_%D0%B4%D0%B5%D1%82%D0%B8" title="Корельские дети">пяти родов карельских детей</a>»<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>, затем столицей удельного <a class="new" href="/w/index.php?title=%D0%A2%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BD%D1%8F%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%BE&amp;action=edit&amp;redlink=1" title="Тиверское княжество (страница отсутствует)">Тиверского княжества</a>. На месте городища найдены вещи, относящиеся к доновгородскому периоду в истории острова: западноевропейская <a href="/wiki/%D0%9C%D0%BE%D0%BD%D0%B5%D1%82%D0%B0" title="Монета">монета</a> <a href="/wiki/XI_%D0%B2%D0%B5%D0%BA" title="XI век">XI века</a>, поясные крючки, обломки <a href="/wiki/%D0%91%D1%80%D0%B0%D1%81%D0%BB%D0%B5%D1%82" title="Браслет">браслетов</a>, бубенчик, бусины, <a href="/wiki/%D0%A2%D0%BE%D0%BF%D0%BE%D1%80" title="Топор">топор</a>, датирующиеся <a href="/wiki/X_%D0%B2%D0%B5%D0%BA" title="X век">X</a> — началом <a href="/wiki/XII_%D0%B2%D0%B5%D0%BA" title="XII век">XII веков</a><sup class="reference" id="cite_ref-kochk._2-0"><a href="#cite_note-kochk.-2">[2]</a></sup>.
</p><p>Существует противоположное мнение, согласно которому данных о существовании Тиверского городка ранее <a href="/wiki/XIV_%D0%B2%D0%B5%D0%BA" title="XIV век">XIV века</a> нет, и его основание относится к самому началу XIV века либо к концу <a href="/wiki/XIII_%D0%B2%D0%B5%D0%BA" title="XIII век">XIII века</a><sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>. Первое упоминание Тиверского городка встречается в «<a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D1%80%D1%83%D1%81%D1%81%D0%BA%D0%B8%D1%85_%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%BE%D0%B2_%D0%B4%D0%B0%D0%BB%D1%8C%D0%BD%D0%B8%D1%85_%D0%B8_%D0%B1%D0%BB%D0%B8%D0%B6%D0%BD%D0%B8%D1%85" title="Список русских городов дальних и ближних">Списке русских городов дальних и ближних</a>», составленном, по мнению историков, в конце XIV века:<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup> <i>«А се Залескии: ..Ладога камен. Орешек. Корельскыи. Тиверьскии..»</i> Не исключено, что обжитый на рубеже X–XI веков остров был впоследствии покинут, а в XIV веке, в обстановке шведской военной угрозы, заселён вновь.
</p><p>В XIII–XIV веках Тиверский городок представлял собою одну из лучших крепостей в регионе, стратегическая цель которой заключалась в том, чтобы задержать врага на подступах к крепости <a href="/wiki/%D0%9A%D0%BE%D1%80%D0%B5%D0%BB%D0%B0_(%D0%BA%D1%80%D0%B5%D0%BF%D0%BE%D1%81%D1%82%D1%8C)" title="Корела (крепость)">Корела</a>. Фактически крепость контролировала водный путь между <a href="/wiki/%D0%92%D1%8B%D0%B1%D0%BE%D1%80%D0%B3" title="Выборг">Выборгом</a> и Корелой. Шведы в первый раз уничтожили его, вероятно, в <a href="/wiki/1295_%D0%B3%D0%BE%D0%B4" title="1295 год">1295 году</a>, когда шли походом на Корелу (Кякисалми).
</p><p>В <a href="/wiki/1323_%D0%B3%D0%BE%D0%B4" title="1323 год">1323 году</a> вблизи Тиверского городка прошла установленная <a class="mw-redirect" href="/wiki/%D0%9E%D1%80%D0%B5%D1%85%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%BC%D0%B8%D1%80%D0%BD%D1%8B%D0%B9_%D0%B4%D0%BE%D0%B3%D0%BE%D0%B2%D0%BE%D1%80" title="Ореховский мирный договор">Ореховским мирным договором</a> граница между <a class="mw-redirect" href="/wiki/%D0%9D%D0%BE%D0%B2%D0%B3%D0%BE%D1%80%D0%BE%D0%B4" title="Новгород">Новгородом</a> и <a href="/wiki/%D0%A8%D0%B2%D0%B5%D1%86%D0%B8%D1%8F" title="Швеция">Швецией</a>. Один из пунктов договора запрещал шведам и новгородцам строительство крепостей по обе стороны границы. Данный факт свидетельствует в пользу того, что Тиверский городок как укреплённая крепость возник до 1323 года. С другой стороны, сооружение вблизи границы крепости как дополнительной преграды на пути к племенному центру Кореле могло быть ответной акцией на возросшую активность шведов, выразившуюся, в частности, в строительстве в <a href="/wiki/1293_%D0%B3%D0%BE%D0%B4" title="1293 год">1293 году</a> Выборга. Исходя из этих дат можно предположить, что сооружение Тиверского городка произошло между 1293 и 1323 годами<sup class="reference" id="cite_ref-kochk._2-1"><a href="#cite_note-kochk.-2">[2]</a></sup>.
</p><p>Приблизительно в <a class="new" href="/w/index.php?title=1330-%D0%B5&amp;action=edit&amp;redlink=1" title="1330-е (страница отсутствует)">1330-х годах</a>, во времена новгородского воеводы <a class="new" href="/w/index.php?title=%D0%92%D0%B0%D0%BB%D0%B8%D1%82%D0%B0_%D0%9A%D0%BE%D1%80%D0%B5%D0%BB%D1%8F%D0%BD%D0%B8%D0%BD&amp;action=edit&amp;redlink=1" title="Валита Корелянин (страница отсутствует)">Валиты Корелянина</a>, (возможно, имелся в виду просто карельский <a class="mw-disambig" href="/wiki/%D0%92%D0%B0%D0%BB%D0%B8%D1%82" title="Валит">валит</a> того времени)городище было окружено <a href="/wiki/%D0%97%D0%B5%D0%BC%D0%BB%D1%8F%D0%BD%D0%BE%D0%B9_%D0%B2%D0%B0%D0%BB" title="Земляной вал">валом</a> и в большей части — каменной <a href="/wiki/%D0%A1%D1%82%D0%B5%D0%BD%D0%B0" title="Стена">стеной</a> двухметровой высоты, сложенной насухо из больших <a href="/wiki/%D0%92%D0%B0%D0%BB%D1%83%D0%BD" title="Валун">валунов</a>. По верху стен в древности шёл деревянный, высотой также около двух метров <a href="/wiki/%D0%91%D1%80%D1%83%D1%81%D1%82%D0%B2%D0%B5%D1%80" title="Бруствер">бруствер</a> — боевое укрытие для защиты воинов от неприятельского огня и удобства стрельбы.
</p><p>Тиверский городок пережил эпоху расцвета в XIV веке. Город, по предположению учёных, являлся оплотом <a href="/wiki/%D0%AF%D0%B7%D1%8B%D1%87%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Язычество">языческой</a> оппозиции, это был опорный пункт карельской племенной верхушки, недовольной тем, что власть уходила у неё из рук. Здесь наверняка находились и <a href="/wiki/%D0%9F%D1%80%D0%B0%D0%B2%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%B8%D0%B5" title="Православие">православный</a> храм (что официально требовалось для торгово-ремесленного города Корельской земли), и заповедные жертвенные камни.
</p><p>Согласно <a href="/wiki/%D0%9D%D0%B8%D0%BA%D0%BE%D0%BD%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%BB%D0%B5%D1%82%D0%BE%D0%BF%D0%B8%D1%81%D1%8C" title="Никоновская летопись">Никоновской летописи</a>, в <a href="/wiki/1404_%D0%B3%D0%BE%D0%B4" title="1404 год">1404 году</a> Тиверский городок вместе с 13 другими селениями был отдан в кормление <a href="/wiki/%D0%A1%D0%BC%D0%BE%D0%BB%D0%B5%D0%BD%D1%81%D0%BA" title="Смоленск">смоленскому</a> князю <a href="/wiki/%D0%AE%D1%80%D0%B8%D0%B9_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%B8%D1%87" title="Юрий Святославич">Юрию Святославичу</a> после того, как Смоленск был захвачен литовским князем <a href="/wiki/%D0%92%D0%B8%D1%82%D0%BE%D0%B2%D1%82" title="Витовт">Витовтом</a>.
</p><p>В <a href="/wiki/1411_%D0%B3%D0%BE%D0%B4" title="1411 год">1411 году</a> <a href="/wiki/%D0%A8%D0%B2%D0%B5%D1%86%D0%B8%D1%8F" title="Швеция">шведы</a> при очередном набеге на <a href="/wiki/%D0%9D%D0%BE%D0%B2%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D0%B7%D0%B5%D0%BC%D0%BB%D1%8F" title="Новгородская земля">Новгородскую землю</a> разрушили Тиверский городок, <a href="/wiki/%D0%9D%D0%BE%D0%B2%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D1%81%D0%BA%D0%B8%D0%B5_%D0%BA%D0%BD%D1%8F%D0%B7%D1%8C%D1%8F" title="Новгородские князья">новгородский князь</a> <a href="/wiki/%D0%9B%D1%83%D0%B3%D0%B2%D0%B5%D0%BD%D0%B8%D0%B9" title="Лугвений">Лугвень</a> (в крещении Симеон, из династии <a href="/wiki/%D0%93%D0%B5%D0%B4%D0%B8%D0%BC%D0%B8%D0%BD%D0%BE%D0%B2%D0%B8%D1%87%D0%B8" title="Гедиминовичи">Гедиминовичей</a>) в том же году отбил город обратно, после чего новгородские власти решили его не восстанавливать. Его укрепления уже не могли бы противостоять сильному противнику, вооружённому <a href="/wiki/%D0%90%D1%80%D0%B1%D0%B0%D0%BB%D0%B5%D1%82" title="Арбалет">арбалетами</a> и <a href="/wiki/%D0%9E%D0%B3%D0%BD%D0%B5%D1%81%D1%82%D1%80%D0%B5%D0%BB%D1%8C%D0%BD%D0%BE%D0%B5_%D0%BE%D1%80%D1%83%D0%B6%D0%B8%D0%B5" title="Огнестрельное оружие">огнестрельным оружием</a>.
</p><p>Намного позже, вследствие сильного падения уровня воды в старом русле реки Вуоксы (после <a href="/wiki/%D0%92%D1%83%D0%BE%D0%BA%D1%81%D0%B0_(%D1%80%D0%B5%D0%BA%D0%B0)#История" title="Вуокса (река)">проведенных в 1857 году взрывных работ на Кивиниемском перешейке</a>), остров Тиури соединился с материком, а на месте восточного рукава реки образовалось сухое каменистое ложе.
</p><p>Поперёк городища была проложена почтовая дорога, а в стенах сделан пролом. Через западный рукав были сооружены два моста, опирающиеся на выступавший из воды гранитный гребень, и поставлена <a href="/wiki/%D0%9C%D0%B5%D0%BB%D1%8C%D0%BD%D0%B8%D1%86%D0%B0" title="Мельница">мельница</a>, от которой сейчас остался только <a href="/wiki/%D0%A4%D1%83%D0%BD%D0%B4%D0%B0%D0%BC%D0%B5%D0%BD%D1%82" title="Фундамент">фундамент</a>. Мельница прекратила работу после нового обмеления русла <a href="/wiki/%D0%92%D1%83%D0%BE%D0%BA%D1%81%D0%B0_(%D1%80%D0%B5%D0%BA%D0%B0)" title="Вуокса (река)">Вуоксы</a> в <a href="/wiki/1857_%D0%B3%D0%BE%D0%B4" title="1857 год">1857 году</a> в результате взрывных работ в <a class="new" href="/w/index.php?title=%D0%9B%D0%BE%D1%81%D0%B5%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%BF%D1%80%D0%BE%D1%82%D0%BE%D0%BA%D0%B0&amp;action=edit&amp;redlink=1" title="Лосевская протока (страница отсутствует)">Лосевской протоке</a>. До сих пор на некоторых камнях в протоке заметны две белые полоски — показатели уровня воды в 1818 и 1857 годах.
</p><p>Впервые Тиверское городище описал известный ученый-филолог <a href="/wiki/%D0%93%D1%80%D0%BE%D1%82,_%D0%AF%D0%BA%D0%BE%D0%B2_%D0%9A%D0%B0%D1%80%D0%BB%D0%BE%D0%B2%D0%B8%D1%87" title="Грот, Яков Карлович">Я. К. Грот</a> в <a href="/wiki/1847_%D0%B3%D0%BE%D0%B4" title="1847 год">1847 году</a>. В <a href="/wiki/1890_%D0%B3%D0%BE%D0%B4" title="1890 год">1890 году</a> были произведены <a href="/wiki/%D0%90%D1%80%D1%85%D0%B5%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F" title="Археология">археологические</a> раскопки и составлен план и описание городища. На территории городища был найден клад, в котором среди различных предметов оказались две арабские монеты <a href="/wiki/X_%D0%B2%D0%B5%D0%BA" title="X век">X века</a>.
</p>
<h2><span id=".D0.96.D0.B5.D1.80.D1.82.D0.B2.D0.B5.D0.BD.D0.BD.D1.8B.D0.B5_.D0.BA.D0.B0.D0.BC.D0.BD.D0.B8"></span><span class="mw-headline" id="Жертвенные_камни">Жертвенные камни</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%BA&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Жертвенные камни»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%BA&amp;action=edit&amp;section=3" title="Редактировать код раздела «Жертвенные камни»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Tiversk_Jertv_kamni.JPG"><img class="mw-file-element" data-file-height="729" data-file-width="972" decoding="async" height="188" src="//upload.wikimedia.org/wikipedia/commons/thumb/7/71/Tiversk_Jertv_kamni.JPG/250px-Tiversk_Jertv_kamni.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/7/71/Tiversk_Jertv_kamni.JPG/375px-Tiversk_Jertv_kamni.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/7/71/Tiversk_Jertv_kamni.JPG/500px-Tiversk_Jertv_kamni.JPG 2x" width="250"/></a><figcaption>Жертвенные камни</figcaption></figure>
<p>В северо-восточной части Тиверского городища располагаются несколько крупных валунов высотой от полутора до двух с половиной метров. На западной стороне одного из камней выбит небольшой равносторонний крест, заключённый в окружность диаметром около 10 см.
</p><p>По поводу значения валунов для прежних жителей и появления креста существует несколько предположений.
</p><p>Наиболее общепринятой считается версия о мистическом значении камней для проживавших здесь <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B5%D0%BB%D1%8B" title="Карелы">карел</a>. В результате археологических исследований на <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B5%D1%80%D0%B5%D1%88%D0%B5%D0%B5%D0%BA" title="Карельский перешеек">Карельском перешейке</a> учёными было обнаружено большое количество <a href="/wiki/%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D1%8B%D0%B5_%D0%BA%D0%B0%D0%BC%D0%BD%D0%B8" title="Священные камни">священных (жертвенных) камней</a> вблизи бывших карельских поселений. Тиверский городок в <a class="mw-redirect" href="/wiki/IX" title="IX">IX</a>–<a href="/wiki/XIII_%D0%B2%D0%B5%D0%BA" title="XIII век">XIII веках</a> являлся центральным поселением карельского племени <a class="new" href="/w/index.php?title=%D0%A2%D0%B8%D0%B2%D1%80%D1%83%D0%BB%D1%8C%D1%86%D1%8B&amp;action=edit&amp;redlink=1" title="Тиврульцы (страница отсутствует)">тиврульцев</a>, поэтому можно предположить, что у жителей Тиверского городка имелось своё священное место, непременным атрибутом которых у карел являлись жертвенные камни.
</p><p>По поводу происхождения отметки в форме креста также нет единого мнения. Подобными знаками помечали выдающиеся из-под земли гранитные глыбы и большие камни в <a class="mw-redirect" href="/wiki/XIV" title="XIV">XIV</a>–<a href="/wiki/XVI_%D0%B2%D0%B5%D0%BA" title="XVI век">XVI веках</a> в двух случаях. Во-первых, при маркировке (обозначении) <a href="/wiki/%D0%93%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B3%D1%80%D0%B0%D0%BD%D0%B8%D1%86%D0%B0" title="Государственная граница">государственных границ</a>. Во-вторых, при освящении бывших <a href="/wiki/%D0%AF%D0%B7%D1%8B%D1%87%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Язычество">языческих</a> <a href="/wiki/%D0%9A%D0%B0%D0%BF%D0%B8%D1%89%D0%B5" title="Капище">капищ</a>. Однако следует заметить, что граница со <a href="/wiki/%D0%A8%D0%B2%D0%B5%D1%86%D0%B8%D1%8F" title="Швеция">Швецией</a>, установленная <a class="mw-redirect" href="/wiki/%D0%9E%D1%80%D0%B5%D1%85%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%BC%D0%B8%D1%80%D0%BD%D1%8B%D0%B9_%D0%B4%D0%BE%D0%B3%D0%BE%D0%B2%D0%BE%D1%80" title="Ореховский мирный договор">Ореховским мирным договором</a>, проходила не в этом месте, а несколько западнее.
</p><p>В пользу версии об освящении капища свидетельствует факт, что известный иеромонах Илия из Новгородского Архиерейского дома дважды в <a class="mw-redirect" href="/wiki/1530-%D0%B5" title="1530-е">1530-х годах</a> приезжал в <a href="/wiki/%D0%9A%D0%BE%D1%80%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D1%83%D0%B5%D0%B7%D0%B4" title="Корельский уезд">Корельский уезд</a>, чтобы «разорять и истреблять скверные мольбища». Ему в помощь архиепископ Новгородский Макарий давал отряд служилых людей. Не исключено, что тиверский камень был освящён по православному обряду весной <a href="/wiki/1534_%D0%B3%D0%BE%D0%B4" title="1534 год">1534 года</a>, когда отец Илия прибыл на <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B5%D1%80%D0%B5%D1%88%D0%B5%D0%B5%D0%BA" title="Карельский перешеек">Карельский перешеек</a> с миссионерско-ревизионной целью.
</p><p>Существует версия, предполагающая, что крест на камне был высечен ранее, однако поскольку ещё в XIV веке Тиверский городок являлся одним из значимых населённых пунктов для карел-язычников, то следует предположить, что освящение камней произошло уже после разрушения городка, то есть после <a href="/wiki/1411_%D0%B3%D0%BE%D0%B4" title="1411 год">1411 года</a>.
</p><p>Вероятно, окрестные карелы продолжали приходить на святые для них места поклоняться прежним языческим <a href="/wiki/%D0%98%D0%B4%D0%BE%D0%BB" title="Идол">идолам</a>, что и вызвало недовольство церкви и последующее освящение камней.
</p>
<h2><span id=".D0.92_.D0.BD.D0.B0.D1.88.D0.B8_.D0.B4.D0.BD.D0.B8"></span><span class="mw-headline" id="В_наши_дни">В наши дни</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%BA&amp;veaction=edit&amp;section=4" title="Редактировать раздел «В наши дни»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%BA&amp;action=edit&amp;section=4" title="Редактировать код раздела «В наши дни»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Tiversk-znak.JPG"><img class="mw-file-element" data-file-height="729" data-file-width="972" decoding="async" height="188" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b9/Tiversk-znak.JPG/250px-Tiversk-znak.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/b9/Tiversk-znak.JPG/375px-Tiversk-znak.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/b9/Tiversk-znak.JPG/500px-Tiversk-znak.JPG 2x" width="250"/></a><figcaption>Памятный знак</figcaption></figure>
<p>В настоящее время топографические условия изменились: восточная протока <a href="/wiki/%D0%92%D1%83%D0%BE%D0%BA%D1%81%D0%B0_(%D1%80%D0%B5%D0%BA%D0%B0)" title="Вуокса (река)">Вуоксы</a> высохла, обнажив каменную гряду, через западную перекинут мост. Дорога <a href="/wiki/%D0%9C%D0%B5%D0%BB%D1%8C%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2%D0%BE_(%D0%9B%D0%B5%D0%BD%D0%B8%D0%BD%D0%B3%D1%80%D0%B0%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C)" title="Мельниково (Ленинградская область)">Мельниково</a> — <a href="/wiki/%D0%93%D1%80%D0%BE%D0%BC%D0%BE%D0%B2%D0%BE_(%D0%9B%D0%B5%D0%BD%D0%B8%D0%BD%D0%B3%D1%80%D0%B0%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C)" title="Громово (Ленинградская область)">Громово</a>, проходящая через городище, соединяет посёлок Мельниково с основной магистралью <a href="/wiki/%D0%9F%D1%80%D0%B8%D0%BE%D0%B7%D0%B5%D1%80%D1%81%D0%BA" title="Приозерск">Приозерск</a> — <a href="/wiki/%D0%A1%D0%B0%D0%BD%D0%BA%D1%82-%D0%9F%D0%B5%D1%82%D0%B5%D1%80%D0%B1%D1%83%D1%80%D0%B3" title="Санкт-Петербург">Санкт-Петербург</a>.
</p><p>В межень Вуокса в этом месте пересыхает, и отсюда начинаются так называемые Тиверские волоки: первый — от моста по левому (по ходу) берегу 130 метров, второй и третий — по пересохшему руслу 50—80 метров и четвёртый — 100 метров по тропе. Все волоки хорошо видны, так как по ним проходят многочисленные байдарочные группы.
</p><p>Длина городища внутри валов с севера на юг составляет 225 м, ширина — от 45 до 60 м. Вал у своего основания имеет ширину от 4 до 5 м; его высота с внутренней стороны каменной кладки достигает 0,8 м, с внешней стороны — 1,7 м. Лучше всего сохранился вал в южной части, где он достигал максимальной высоты и ширины. Площадь городища примерно равняется 1 га.
</p><p>В связи со строительством домов и погреба в <a href="/wiki/XIX_%D0%B2%D0%B5%D0%BA" title="XIX век">XIX веке</a> целостность стен в юго-западной части памятника нарушена, о них напоминают лишь беспорядочные развалы камней. Значительная часть археологического памятника была уничтожена во второй половине <a href="/wiki/XX_%D0%B2%D0%B5%D0%BA" title="XX век">XX века</a> во время строительства в этой местности автодороги Громово — Мельниково.
</p><p>В <a href="/wiki/1988_%D0%B3%D0%BE%D0%B4" title="1988 год">1988 году</a> по инициативе председателя Мельниковского сельского совета Свинарева Ю.П. на территории Тиверского городища, вблизи автотрассы, был установлен памятный знак. Надпись на нём гласит: «Нашим предкам, русским, обильно полившим эту землю потом и кровью»<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>.
</p>

</p><p><span class="ts-fix-text" style="-webkit-box-decoration-break: clone">Мегалиты городища выделяет нехарактерный для местности <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B5%D1%80%D0%B5%D1%88%D0%B5%D0%B5%D0%BA" title="Карельский перешеек">Природа Карельского перешейка</a> большой размер и расположение.Вместе они образуют подобие ворот бывшего судоходного протока, возвышаясь над ним на 10 метров.  На фото камень под мегалитом со следами сверления буром.</span><sup class="ts-fix-template noprint">[<i><a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%A1%D1%81%D1%8B%D0%BB%D0%BA%D0%B8_%D0%BD%D0%B0_%D0%B8%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8" title="Википедия:Ссылки на источники"><span title="не указан источник на утверждение (26 декабря 2017)">источник не указан 2255 дней</span></a></i>]</sup>
</p><p><span class="ts-fix-text" style="-webkit-box-decoration-break: clone">Учитывая стратегическое расположение Тиверского городища на единственном сухопутном пересечении Вуоксы, - логично ожидать, что поверх мегалитов ранее имелось оборонительное сооружение. Однако археологические финские экспедиции не описали территорию западнее крепостных стен,- примыкающего к дороге в сторону Швеции-  указав о наличии посада (севернее).</span><sup class="ts-fix-template noprint">[<i><a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%A1%D1%81%D1%8B%D0%BB%D0%BA%D0%B8_%D0%BD%D0%B0_%D0%B8%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8" title="Википедия:Ссылки на источники"><span title="не указан источник на утверждение (26 декабря 2017)">источник не указан 2255 дней</span></a></i>]</sup> Сейчас там на удалении 5-10м от берега в валуны вделаны металлические кольца для лодок. Местность непосредственно вокруг переправы через Вуоксу изменена при реконструкции шоссе в 1970х. Сохранились оборонительные сооружения времен WW2 вдоль шоссе.
</p><p>Непосредственно в воде расположен только один <span class="ts-fix-text" style="-webkit-box-decoration-break: clone">мегалит сопоставимого размера и формы</span><sup class="ts-fix-template noprint">[<i><a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%A1%D1%81%D1%8B%D0%BB%D0%BA%D0%B8_%D0%BD%D0%B0_%D0%B8%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8" title="Википедия:Ссылки на источники"><span title="не указан источник на утверждение (26 декабря 2017)">источник не указан 2255 дней</span></a></i>]</sup>, имеющий отметки уровня воды в 1800-х годах (2 белые полоски выше нынешнего уровня на 1,5 метра). В верхней части углубления для крепления настила причала. Камень находится в бухте перед сужением фарватера и уменьшением глубины. Непосредственно за ним на берегу ровное поле со следами построек (2016- отдано под частную застройку).
</p><p>Новое звучание получает народное предание о постройке Кякисалми , по которому основной город планировался у порогов Тиури (<a class="mw-redirect" href="/wiki/%D0%A2%D0%B8%D1%83%D1%80%D0%B8%D0%BD-%D0%BB%D0%B8%D0%BD%D0%BD%D0%B0" title="Тиурин-линна">Тиуринлинна</a> — крепости на месте Тиверского городища) — но построенный за один день и разрушенный за одну ночь духами был  перенесен на место теперешнего Приозерска (<a href="/wiki/%D0%9A%D0%BE%D1%80%D0%B5%D0%BB%D0%B0_(%D0%BA%D1%80%D0%B5%D0%BF%D0%BE%D1%81%D1%82%D1%8C)" title="Корела (крепость)">крепости Корелы</a>).
</p>

<h2><span id=".D0.A0.D0.B5.D0.B7.D1.83.D0.BB.D1.8C.D1.82.D0.B0.D1.82.D1.8B_.D0.B0.D1.80.D1.85.D0.B5.D0.BE.D0.BB.D0.BE.D0.B3.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.B8.D1.85_.D0.B8.D1.81.D1.81.D0.BB.D0.B5.D0.B4.D0.BE.D0.B2.D0.B0.D0.BD.D0.B8.D0.B9"></span><span class="mw-headline" id="Результаты_археологических_исследований">Результаты археологических исследований</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%A2%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%BA&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Результаты археологических исследований»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%A2%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%BA&amp;action=edit&amp;section=5" title="Редактировать код раздела «Результаты археологических исследований»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:South_rov.JPG"><img class="mw-file-element" data-file-height="729" data-file-width="972" decoding="async" height="188" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/ca/South_rov.JPG/250px-South_rov.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/c/ca/South_rov.JPG/375px-South_rov.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/c/ca/South_rov.JPG/500px-South_rov.JPG 2x" width="250"/></a><figcaption>Южный вал снаружи</figcaption></figure>
<p>Археологические раскопки выявили жилые и хозяйственные комплексы, оборонительные сооружения и <a href="/wiki/%D0%9C%D0%BE%D0%B3%D0%B8%D0%BB%D0%B0" title="Могила">захоронения</a> погибших защитников крепости. <a href="/wiki/%D0%A4%D1%83%D0%BD%D0%B4%D0%B0%D0%BC%D0%B5%D0%BD%D1%82" title="Фундамент">Фундаменты</a> жилищ сложены из мелких, вплотную пригнанных камней, обмазанных <a href="/wiki/%D0%93%D0%BB%D0%B8%D0%BD%D0%B0" title="Глина">глиной</a>. Верхние части домов были деревянными. Дно и стенки <a href="/wiki/%D0%9E%D1%87%D0%B0%D0%B3" title="Очаг">очагов</a> тоже делались из мелких камней и находились обычно в северо-западном углу, ближе к входу. Площадь жилищ колебалась от 18–20 до 54 м². Постройки составляли двухрядную цепочку соответственно конфигурации острова, располагаясь в шахматном порядке, и представляли собой оборонительную линию.
</p><p>Южная часть городища была защищена мощным <a href="/wiki/%D0%97%D0%B5%D0%BC%D0%BB%D1%8F%D0%BD%D0%BE%D0%B9_%D0%B2%D0%B0%D0%BB" title="Земляной вал">валом</a> с каменным основанием, засыпанным землёй. Обороне этого участка придавали серьёзное значение: неприятель мог появиться только перед южными укреплениями. Именно в данной части городища находилось почти всё собранное при раскопках <a href="/wiki/%D0%9E%D1%80%D1%83%D0%B6%D0%B8%D0%B5" title="Оружие">оружие</a>. В северной части Тиверского городка, как в более безопасной, была сосредоточена хозяйственная жизнь. Каменная стена, сложенная из валунов, во многих местах сохранила здесь почти первоначальную высоту. В восточной части с внутренней стороны оборонительной стены пристроены три прямоугольные камеры тоже из крупных валунов.
</p>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Tiurinlinna_(R%C3%A4is%C3%A4l%C3%A4)_hoard_objects_-_National_Museum_of_Finland_-_DSC04186.JPG"><img class="mw-file-element" data-file-height="3240" data-file-width="4320" decoding="async" height="188" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/88/Tiurinlinna_%28R%C3%A4is%C3%A4l%C3%A4%29_hoard_objects_-_National_Museum_of_Finland_-_DSC04186.JPG/250px-Tiurinlinna_%28R%C3%A4is%C3%A4l%C3%A4%29_hoard_objects_-_National_Museum_of_Finland_-_DSC04186.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/88/Tiurinlinna_%28R%C3%A4is%C3%A4l%C3%A4%29_hoard_objects_-_National_Museum_of_Finland_-_DSC04186.JPG/375px-Tiurinlinna_%28R%C3%A4is%C3%A4l%C3%A4%29_hoard_objects_-_National_Museum_of_Finland_-_DSC04186.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/88/Tiurinlinna_%28R%C3%A4is%C3%A4l%C3%A4%29_hoard_objects_-_National_Museum_of_Finland_-_DSC04186.JPG/500px-Tiurinlinna_%28R%C3%A4is%C3%A4l%C3%A4%29_hoard_objects_-_National_Museum_of_Finland_-_DSC04186.JPG 2x" width="250"/></a><figcaption>Предметы из клада из Тиверского городища. Национальный музей Финляндии</figcaption></figure><p> В 400 м к северо-западу от городища, в лесу у берега реки, находились квадратные в плане фундаменты из мелких <a href="/wiki/%D0%92%D0%B0%D0%BB%D1%83%D0%BD" title="Валун">валунов</a>, высотой до 1 м. Скорее всего здесь находился Тиверский <a href="/wiki/%D0%9F%D0%BE%D1%81%D0%B0%D0%B4" title="Посад">посад</a>.
</p><p>В <a href="/wiki/%D0%9D%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BC%D1%83%D0%B7%D0%B5%D0%B9_%D0%A4%D0%B8%D0%BD%D0%BB%D1%8F%D0%BD%D0%B4%D0%B8%D0%B8" title="Национальный музей Финляндии">Национальном музее Финляндии</a> хранятся находки из клада, найденного в Тиуринлинна — Тиверском городище.
</p><p>В Тиверске керамика группы 2 соответствует посуде типа III из приладожских курганов, а лепные горшки группы 3 тождественны керамике типа II. Лепная керамика <a class="new" href="/w/index.php?title=%D0%9F%D1%80%D0%B8%D0%BB%D0%B0%D0%B4%D0%BE%D0%B6%D1%81%D0%BA%D0%B0%D1%8F_%D0%BA%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D0%B0&amp;action=edit&amp;redlink=1" title="Приладожская культура (страница отсутствует)">приладожской культуры</a> бытует в X — начале XI века. Аналогии трём группам лепных сосудов из раскопок Тиверского городища обнару­живаются в керамике курганов Юго-Восточного Приладожья, в посуде Новгородской, Ленинградской, Псковской и Вологодской областей. Орнаментированный горшок с подправкой на круге группы 3 похож на раннегончарный сосуд из Шугозеро X—XI веков. Керамика первых двух групп соответствует посуде типа 1 в Старой Ладоге, бытовавшей период со второй половины IX до первой половины XI века. Горшкам групп 1, 2 и 3 имеются аналогии в горизонтах Рюрикового Горо­дища, датирующихся второй половиной IX — началом X века. Горшки группы 2 схожи с керамикой типа Ш-2 и сосудами  вида 3-А типа III из 27 яруса <a class="mw-redirect" href="/wiki/%D0%9D%D0%B5%D1%80%D0%B5%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D1%80%D0%B0%D1%81%D0%BA%D0%BE%D0%BF" title="Неревский раскоп">Неревского раскопа</a> (972—989 годы) и нижележащих слоёв. Горшки группы 3 тождественны новгородским сосудам типа VI-А из 26 яруса (989—106 годы) Неревского раскопа и ранее, раннегончарный горшок (группа 3) повторяет форму сосудов типа VI-В из 28 яруса (953—972 годы). Несколько обломкам лепной по­суды из Тиверска аналогичны фрагментам керамики с орнаментом из <a href="/wiki/%D0%93%D0%BE%D1%80%D0%BE%D0%B4%D0%BE%D0%BA_%D0%BD%D0%B0_%D0%9C%D0%B0%D1%8F%D1%82%D0%B5" title="Городок на Маяте">Городка на Маяте</a><sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>.
</p>
