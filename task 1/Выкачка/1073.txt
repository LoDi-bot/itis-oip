<p><b>Мавзолей Галлы Плацидии</b> (<a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D1%8C%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Итальянский язык">итал.</a> <span lang="it" style="font-style:italic;">Mausoleo di Galla Placidia</span>) — крестообразная постройка, расположенная рядом с базиликой <a href="/wiki/%D0%A1%D0%B0%D0%BD-%D0%92%D0%B8%D1%82%D0%B0%D0%BB%D0%B5" title="Сан-Витале">Сан-Витале</a> в <a href="/wiki/%D0%A0%D0%B0%D0%B2%D0%B5%D0%BD%D0%BD%D0%B0" title="Равенна">Равенне</a>.
</p><p>Мавзолей датируется приблизительно второй четвертью <a href="/wiki/V_%D0%B2%D0%B5%D0%BA" title="V век">V века</a> и украшен самыми ранними из сохранившихся равеннских мозаик.<sup class="reference" id="cite_ref-lazarev_1-0"><a href="#cite_note-lazarev-1">[1]</a></sup> Хотя постройка приписывается <a href="/wiki/%D0%93%D0%B0%D0%BB%D0%BB%D0%B0_%D0%9F%D0%BB%D0%B0%D1%86%D0%B8%D0%B4%D0%B8%D1%8F" title="Галла Плацидия">Галле Плацидии</a>, дочери императора <a href="/wiki/%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D0%B9_I_%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B8%D0%B9" title="Феодосий I Великий">Феодосия Великого</a>, <a href="/wiki/%D0%9C%D0%B0%D0%B2%D0%B7%D0%BE%D0%BB%D0%B5%D0%B9" title="Мавзолей">мавзолей</a> не стал местом её погребения. В <a href="/wiki/1996_%D0%B3%D0%BE%D0%B4" title="1996 год">1996 году</a> мавзолей в числе других раннехристианских памятников Равенны был включён в число объектов <a href="/wiki/%D0%92%D1%81%D0%B5%D0%BC%D0%B8%D1%80%D0%BD%D0%BE%D0%B5_%D0%BD%D0%B0%D1%81%D0%BB%D0%B5%D0%B4%D0%B8%D0%B5" title="Всемирное наследие">Всемирного наследия</a> под № 788.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>
</p>

<h2><span id=".D0.98.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.8F_.D0.BC.D0.B0.D0.B2.D0.B7.D0.BE.D0.BB.D0.B5.D1.8F"></span><span class="mw-headline" id="История_мавзолея">История мавзолея</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D0%B2%D0%B7%D0%BE%D0%BB%D0%B5%D0%B9_%D0%93%D0%B0%D0%BB%D0%BB%D1%8B_%D0%9F%D0%BB%D0%B0%D1%86%D0%B8%D0%B4%D0%B8%D0%B8&amp;veaction=edit&amp;section=1" title="Редактировать раздел «История мавзолея»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D0%B2%D0%B7%D0%BE%D0%BB%D0%B5%D0%B9_%D0%93%D0%B0%D0%BB%D0%BB%D1%8B_%D0%9F%D0%BB%D0%B0%D1%86%D0%B8%D0%B4%D0%B8%D0%B8&amp;action=edit&amp;section=1" title="Редактировать код раздела «История мавзолея»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Vasiliy_smirnov_002.jpg"><img class="mw-file-element" data-file-height="1153" data-file-width="1200" decoding="async" height="240" src="//upload.wikimedia.org/wikipedia/commons/thumb/0/01/Vasiliy_smirnov_002.jpg/250px-Vasiliy_smirnov_002.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/0/01/Vasiliy_smirnov_002.jpg/375px-Vasiliy_smirnov_002.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/0/01/Vasiliy_smirnov_002.jpg/500px-Vasiliy_smirnov_002.jpg 2x" width="250"/></a><figcaption><center>Утренний выход византийской царицы к гробницам своих предков (<a href="/wiki/%D0%A1%D0%BC%D0%B8%D1%80%D0%BD%D0%BE%D0%B2,_%D0%92%D0%B0%D1%81%D0%B8%D0%BB%D0%B8%D0%B9_%D0%A1%D0%B5%D1%80%D0%B3%D0%B5%D0%B5%D0%B2%D0%B8%D1%87_(%D1%85%D1%83%D0%B4%D0%BE%D0%B6%D0%BD%D0%B8%D0%BA)" title="Смирнов, Василий Сергеевич (художник)">В. С. Смирнов</a>, <a class="mw-redirect" href="/wiki/1880-%D0%B5" title="1880-е">1880-е</a>). Действие разворачивается в мавзолее Галлы Плацидии.</center></figcaption></figure>
<p>Мавзолей долгое время служил <a href="/wiki/%D0%9E%D1%80%D0%B0%D1%82%D0%BE%D1%80%D0%B8%D0%B9" title="Ораторий">ораторием</a> при несохранившейся дворцовой <a href="/wiki/%D0%91%D0%B0%D0%B7%D0%B8%D0%BB%D0%B8%D0%BA%D0%B0" title="Базилика">базилике</a> Санта-Кроче. Вероятно, он был капеллой-молельней, посвящённой особо почитавшемуся в роду Галлы Плацидии великомученику <a class="mw-redirect" href="/wiki/%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B9_%D0%9B%D0%B0%D0%B2%D1%80%D0%B5%D0%BD%D1%82%D0%B8%D0%B9" title="Святой Лаврентий">Лаврентию</a>, чьё изображение помещено на самом видном месте — в <a href="/wiki/%D0%9B%D1%8E%D0%BD%D0%B5%D1%82_(%D0%B0%D1%80%D1%85%D0%B8%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D0%B0)" title="Люнет (архитектура)">люнете</a> напротив входа<sup class="reference" id="cite_ref-РИИ-13_3-0"><a href="#cite_note-РИИ-13-3">[3]</a></sup>.
</p><p>Постройка, несмотря на мозаики, связанные с заупокойным культом, не стала местом погребения Галлы Плацидии — она умерла <a href="/wiki/27_%D0%BD%D0%BE%D1%8F%D0%B1%D1%80%D1%8F" title="27 ноября">27 ноября</a> <a href="/wiki/450_%D0%B3%D0%BE%D0%B4" title="450 год">450 года</a> в <a href="/wiki/%D0%A0%D0%B8%D0%BC" title="Рим">Риме</a><sup class="reference" id="cite_ref-РИИ-12_4-0"><a href="#cite_note-РИИ-12-4">[4]</a></sup> и, более чем вероятно, была погребена в родовой усыпальнице императора <a href="/wiki/%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D0%B9_I_%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B8%D0%B9" title="Феодосий I Великий">Феодосия</a> Сан-Петронила недалеко от <a href="/wiki/%D0%A1%D0%BE%D0%B1%D0%BE%D1%80_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%9F%D0%B5%D1%82%D1%80%D0%B0" title="Собор Святого Петра">собора святого Петра</a><sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>. Ни один античный историк не упоминал о погребении императрицы в Равенне, молчит об этом и <a class="new" href="/w/index.php?title=%D0%90%D0%BD%D0%B4%D1%80%D0%B5%D0%BE_%D0%90%D0%B3%D0%BD%D0%B5%D0%BB%D0%BB%D0%BE&amp;action=edit&amp;redlink=1" title="Андрео Агнелло (страница отсутствует)">Андрео Агнелло</a> в своей хронике «<i>Liber Pontificalis Ecclesiae Ravennatis</i>». Имеющиеся в мавзолее <a href="/wiki/%D0%A1%D0%B0%D1%80%D0%BA%D0%BE%D1%84%D0%B0%D0%B3" title="Саркофаг">саркофаги</a>, приписываемые самой Галле и её ближайшим родственникам, по мнению ряда исследователей, изначально не находились в нём<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>, первый раз о них упоминает в <a href="/wiki/XIV_%D0%B2%D0%B5%D0%BA" title="XIV век">XIV веке</a> епископ Ринальдо да Конкореджио<sup class="reference" id="cite_ref-РИИ-13_3-1"><a href="#cite_note-РИИ-13-3">[3]</a></sup>. После XIV века многочисленные источники уже уверенно называют здание мавзолеем Галлы Плацидии. Среди возможных причин этого можно назвать как определённое внешнее сходство мозаик мавзолея с мозаиками римской церкви <a class="mw-redirect" href="/wiki/%D0%A1%D0%B0%D0%BD%D1%82%D0%B0-%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%86%D0%B0" title="Санта-Констанца">Санта-Констанца</a> (местом погребения одной из дочерей <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD_%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B8%D0%B9" title="Константин Великий">Константина Великого</a>), так и необычная форма погребения тела в одном из саркофагов (тело было усажено на <a href="/wiki/%D0%9A%D0%B8%D0%BF%D0%B0%D1%80%D0%B8%D1%81" title="Кипарис">кипарисовый</a> <a href="/wiki/%D0%A2%D1%80%D0%BE%D0%BD" title="Трон">трон</a>)<sup class="reference" id="cite_ref-РИИ-13_3-2"><a href="#cite_note-РИИ-13-3">[3]</a></sup>.
</p>
<h2><span id=".D0.90.D1.80.D1.85.D0.B8.D1.82.D0.B5.D0.BA.D1.82.D1.83.D1.80.D0.BD.D1.8B.D0.B5_.D0.BE.D1.81.D0.BE.D0.B1.D0.B5.D0.BD.D0.BD.D0.BE.D1.81.D1.82.D0.B8"></span><span class="mw-headline" id="Архитектурные_особенности">Архитектурные особенности</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D0%B2%D0%B7%D0%BE%D0%BB%D0%B5%D0%B9_%D0%93%D0%B0%D0%BB%D0%BB%D1%8B_%D0%9F%D0%BB%D0%B0%D1%86%D0%B8%D0%B4%D0%B8%D0%B8&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Архитектурные особенности»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D0%B2%D0%B7%D0%BE%D0%BB%D0%B5%D0%B9_%D0%93%D0%B0%D0%BB%D0%BB%D1%8B_%D0%9F%D0%BB%D0%B0%D1%86%D0%B8%D0%B4%D0%B8%D0%B8&amp;action=edit&amp;section=2" title="Редактировать код раздела «Архитектурные особенности»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-default-size" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Mausoleum_of_Galla_Placidia._Plan.jpg"><img class="mw-file-element" data-file-height="513" data-file-width="444" decoding="async" height="254" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/de/Mausoleum_of_Galla_Placidia._Plan.jpg/220px-Mausoleum_of_Galla_Placidia._Plan.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/d/de/Mausoleum_of_Galla_Placidia._Plan.jpg/330px-Mausoleum_of_Galla_Placidia._Plan.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/d/de/Mausoleum_of_Galla_Placidia._Plan.jpg/440px-Mausoleum_of_Galla_Placidia._Plan.jpg 2x" width="220"/></a><figcaption><center>План мавзолея</center></figcaption></figure>
<p>Здание построено в технике <a href="/wiki/%D0%9B%D0%BE%D0%BC%D0%B1%D0%B0%D1%80%D0%B4%D0%B8%D1%8F" title="Ломбардия">ломбардской</a> <a href="/wiki/%D0%9A%D0%B8%D1%80%D0%BF%D0%B8%D1%87" title="Кирпич">кирпичной</a> кладки. Внешний вид напоминает крепостное строение, что подчёркивается толстыми стенами и узкими окнами. Пропорции здания несколько исказились из-за того, что за прошедшие века <a href="/wiki/%D0%9A%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D0%BD%D1%8B%D0%B9_%D1%81%D0%BB%D0%BE%D0%B9" title="Культурный слой">культурный слой</a> поднялся на 1,5 метра<sup class="reference" id="cite_ref-RCA-86_7-0"><a href="#cite_note-RCA-86-7">[7]</a></sup>. Мавзолей в плане представляет собой <a href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82#Латинский_крест" title="Крест">латинский крест</a> (длинная <a class="mw-disambig" href="/wiki/%D0%9E%D1%81%D1%8C" title="Ось">ось</a> — 12,75 м, поперечная — 10,25 м)<sup class="reference" id="cite_ref-РИИ-13_3-3"><a href="#cite_note-РИИ-13-3">[3]</a></sup>. <a href="/wiki/%D0%A1%D1%80%D0%B5%D0%B4%D0%BE%D0%BA%D1%80%D0%B5%D1%81%D1%82%D0%B8%D0%B5" title="Средокрестие">Средокрестие</a> увенчано <a href="/wiki/%D0%9A%D1%83%D0%B1" title="Куб">кубической</a> башней, в которую изнутри вписан <a href="/wiki/%D0%9A%D1%83%D0%BF%D0%BE%D0%BB" title="Купол">купол</a>, невидимый снаружи здания.
</p><p>Длинная <a class="mw-disambig" href="/wiki/%D0%9E%D1%81%D1%8C" title="Ось">ось</a> здания направлена не по линии <a href="/wiki/%D0%97%D0%B0%D0%BF%D0%B0%D0%B4" title="Запад">запад</a>-<a href="/wiki/%D0%92%D0%BE%D1%81%D1%82%D0%BE%D0%BA" title="Восток">восток</a>, как обычно принято в христианских храмах, а по линии <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80" title="Север">север</a>-<a href="/wiki/%D0%AE%D0%B3" title="Юг">юг</a>. Исследователи объясняют эту особенность тем, что при строительстве мавзолей соединялся с <a class="mw-redirect" href="/wiki/%D0%9D%D0%B0%D1%80%D1%82%D0%B5%D0%BA%D1%81" title="Нартекс">нартексом</a> <span data-interwiki-article="Chiesa di Santa Croce (Ravenna)" data-interwiki-lang="it"><a class="new" href="/w/index.php?title=%D0%A6%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C_%D0%A1%D0%B0%D0%BD%D1%82%D0%B0-%D0%9A%D1%80%D0%BE%D1%87%D0%B5&amp;action=edit&amp;redlink=1" title="Церковь Санта-Кроче (страница отсутствует)">церкви Санта-Кроче</a></span><span class="noprint" style="font-style:normal; font-weight:normal;"><sup><a class="extiw" href="https://it.wikipedia.org/wiki/Chiesa_di_Santa_Croce_(Ravenna)" title="it:Chiesa di Santa Croce (Ravenna)"><span title="Chiesa di Santa Croce (Ravenna) — версия статьи «Церковь Санта-Кроче» на итальянском языке">[it]</span></a></sup></span> через свою наиболее вытянутую часть. С северной стороны нартекса находилась симметричная постройка.<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup> Вынужденную архитектурную ошибку дважды исправляли мозаичисты, ориентируя мозаики внутри мавзолея на восток<sup class="reference" id="cite_ref-РИИ-16_9-0"><a href="#cite_note-РИИ-16-9">[9]</a></sup>.
</p><p>Наружный <a class="mw-redirect" href="/wiki/%D0%90%D1%81%D0%BA%D0%B5%D1%82" title="Аскет">аскетический</a> вид мавзолея украшают только плоские вертикальные выступы — лопатки, соединенные с плоскими <a href="/wiki/%D0%90%D1%80%D0%BA%D0%B0" title="Арка">арками</a> («<i>глухая аркада</i>»). Последние своим ритмичным чередованием оживляют внешний вид трёх сторон мавзолея. <a href="/wiki/%D0%9F%D1%80%D0%B8%D1%82%D0%BE%D0%BB%D0%BE%D0%BA%D0%B0" title="Притолока">Притолока</a> над входом является древнеримским карнизом I века с вакхическим <a href="/wiki/%D0%A4%D1%80%D0%B8%D0%B7_(%D0%B0%D1%80%D1%85%D0%B8%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D0%B0)" title="Фриз (архитектура)">фризом</a><sup class="reference" id="cite_ref-RLA-76_10-0"><a href="#cite_note-RLA-76-10">[10]</a></sup> с двумя <a href="/wiki/%D0%9F%D0%B0%D0%BD%D1%82%D0%B5%D1%80%D1%8B" title="Пантеры">пантерами</a> и виноградными лозами<sup class="reference" id="cite_ref-RCA-86_7-1"><a href="#cite_note-RCA-86-7">[7]</a></sup>.
</p><p>Четырнадцать окон мавзолея имеют форму <a href="/wiki/%D0%91%D0%BE%D0%B9%D0%BD%D0%B8%D1%86%D0%B0" title="Бойница">бойниц</a>, расширяющихся внутрь здания. В <a href="/wiki/1908_%D0%B3%D0%BE%D0%B4" title="1908 год">1908 году</a> окна были зашиты <a href="/wiki/%D0%90%D0%BB%D0%B5%D0%B1%D0%B0%D1%81%D1%82%D1%80" title="Алебастр">алебастровыми</a> плитами — даром <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D1%80%D0%BE%D0%BB%D0%B5%D0%B2%D1%81%D1%82%D0%B2%D0%BE_%D0%98%D1%82%D0%B0%D0%BB%D0%B8%D1%8F_(1861-1946)" title="Королевство Италия (1861-1946)">итальянского короля</a> <a href="/wiki/%D0%92%D0%B8%D0%BA%D1%82%D0%BE%D1%80_%D0%AD%D0%BC%D0%BC%D0%B0%D0%BD%D1%83%D0%B8%D0%BB_III" title="Виктор Эммануил III">Виктора Эммануила III</a><sup class="reference" id="cite_ref-РИИ-13_3-4"><a href="#cite_note-РИИ-13-3">[3]</a></sup>.
</p>
<h2><span id=".D0.9E.D1.81.D0.BE.D0.B1.D0.B5.D0.BD.D0.BD.D0.BE.D1.81.D1.82.D0.B8_.D0.B8.D0.BD.D1.82.D0.B5.D1.80.D1.8C.D0.B5.D1.80.D0.B0"></span><span class="mw-headline" id="Особенности_интерьера">Особенности интерьера</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D0%B2%D0%B7%D0%BE%D0%BB%D0%B5%D0%B9_%D0%93%D0%B0%D0%BB%D0%BB%D1%8B_%D0%9F%D0%BB%D0%B0%D1%86%D0%B8%D0%B4%D0%B8%D0%B8&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Особенности интерьера»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D0%B2%D0%B7%D0%BE%D0%BB%D0%B5%D0%B9_%D0%93%D0%B0%D0%BB%D0%BB%D1%8B_%D0%9F%D0%BB%D0%B0%D1%86%D0%B8%D0%B4%D0%B8%D0%B8&amp;action=edit&amp;section=3" title="Редактировать код раздела «Особенности интерьера»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Ravenna_1978_079.jpg"><img class="mw-file-element" data-file-height="790" data-file-width="525" decoding="async" height="271" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/36/Ravenna_1978_079.jpg/180px-Ravenna_1978_079.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/36/Ravenna_1978_079.jpg/270px-Ravenna_1978_079.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/3/36/Ravenna_1978_079.jpg/360px-Ravenna_1978_079.jpg 2x" width="180"/></a><figcaption><center>Внутренний вид мавзолея</center></figcaption></figure>
<p>Ветви «<a href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82" title="Крест">креста</a>» (воспринимается внутри как равноконечный) перекрыты цилиндрическими <a href="/wiki/%D0%A1%D0%B2%D0%BE%D0%B4" title="Свод">сводами</a>, а центральное пространство увенчано тяжёлым куполом на «<a href="/wiki/%D0%9F%D0%B0%D1%80%D1%83%D1%81_(%D0%B0%D1%80%D1%85%D0%B8%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D0%B0)" title="Парус (архитектура)">парусах</a>» без барабана. Все поверхности купола, арок и люнет покрыты <a href="/wiki/%D0%9C%D0%BE%D0%B7%D0%B0%D0%B8%D0%BA%D0%B0" title="Мозаика">мозаиками</a>. Реставрационные работы, проведённые на рубеже <a href="/wiki/XIX_%D0%B2%D0%B5%D0%BA" title="XIX век">XIX</a>-<a href="/wiki/XX_%D0%B2%D0%B5%D0%BA" title="XX век">XX веков</a>, ещё более усилили мягкое и сдержанное настроение, создаваемое мозаиками. Теперь свет, проникающий в мавзолей лишь сквозь небольшие узкие окна, приобретает золотистый оттенок из-за того, что проходит через <a href="/wiki/%D0%90%D0%BB%D0%B5%D0%B1%D0%B0%D1%81%D1%82%D1%80" title="Алебастр">алебастровые</a> плиты, которыми зашили окна в <a href="/wiki/1908_%D0%B3%D0%BE%D0%B4" title="1908 год">1908 году</a>. Впечатление от золотистого света ещё более усиливают желтоватые <a href="/wiki/%D0%9C%D1%80%D0%B0%D0%BC%D0%BE%D1%80" title="Мрамор">мраморные</a> плиты, которыми нижняя часть стен была облицована в конце XIX века<sup class="reference" id="cite_ref-РИИ-13_3-5"><a href="#cite_note-РИИ-13-3">[3]</a></sup>.
</p>
<h2><span id=".D0.9C.D0.BE.D0.B7.D0.B0.D0.B8.D0.BA.D0.B8"></span><span class="mw-headline" id="Мозаики">Мозаики</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D0%B2%D0%B7%D0%BE%D0%BB%D0%B5%D0%B9_%D0%93%D0%B0%D0%BB%D0%BB%D1%8B_%D0%9F%D0%BB%D0%B0%D1%86%D0%B8%D0%B4%D0%B8%D0%B8&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Мозаики»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D0%B2%D0%B7%D0%BE%D0%BB%D0%B5%D0%B9_%D0%93%D0%B0%D0%BB%D0%BB%D1%8B_%D0%9F%D0%BB%D0%B0%D1%86%D0%B8%D0%B4%D0%B8%D0%B8&amp;action=edit&amp;section=4" title="Редактировать код раздела «Мозаики»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%D0%9C%D0%BE%D0%B7%D0%B0%D0%B8%D0%BA%D0%B0" title="Мозаика">Мозаичное</a> убранство мавзолея отличается редкой пышностью и ставит его в ряд важнейших памятников <a href="/wiki/%D0%A0%D0%B0%D0%BD%D0%BD%D0%B5%D1%85%D1%80%D0%B8%D1%81%D1%82%D0%B8%D0%B0%D0%BD%D1%81%D0%BA%D0%BE%D0%B5_%D0%B8%D1%81%D0%BA%D1%83%D1%81%D1%81%D1%82%D0%B2%D0%BE" title="Раннехристианское искусство">раннехристианского искусства</a>. Мозаики, хоть и посвящённые различным сюжетам, образуют органически единое целое. Все они следуют античной <a href="/wiki/%D0%94%D1%80%D0%B5%D0%B2%D0%BD%D0%B8%D0%B9_%D0%A0%D0%B8%D0%BC" title="Древний Рим">римско</a>-<a href="/wiki/%D0%AD%D0%BB%D0%BB%D0%B8%D0%BD%D0%B8%D0%B7%D0%BC" title="Эллинизм">эллинистической</a> мозаичной традиции, хотя происхождение самих мастеров остаётся спорным (в числе возможных версий называются <a href="/wiki/%D0%A0%D0%B8%D0%BC" title="Рим">Рим</a>, <a href="/wiki/%D0%9C%D0%B8%D0%BB%D0%B0%D0%BD" title="Милан">Медиолан</a>, <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD%D0%BE%D0%BF%D0%BE%D0%BB%D1%8C" title="Константинополь">Константинополь</a>, <a href="/wiki/%D0%A1%D0%B8%D1%80%D0%B8%D1%8F" title="Сирия">Сирия</a>, <a class="mw-redirect" href="/wiki/%D0%90%D1%84%D1%80%D0%B8%D0%BA%D0%B0_(%D0%BF%D1%80%D0%BE%D0%B2%D0%B8%D0%BD%D1%86%D0%B8%D1%8F)" title="Африка (провинция)">Африка</a>)<sup class="reference" id="cite_ref-РИИ-19_11-0"><a href="#cite_note-РИИ-19-11">[11]</a></sup>. Здесь римское художественное наследие с его тщательно выполненными деталями и пластичными фигурами слилось с христианским стремлением к абсолютности и трансцендентности, что создало неповторимый мозаичный ансамбль<sup class="reference" id="cite_ref-RCA-86_7-2"><a href="#cite_note-RCA-86-7">[7]</a></sup>.
</p><p>Русский искусствовед <a class="mw-redirect" href="/wiki/%D0%9F%D0%B0%D0%B2%D0%B5%D0%BB_%D0%9C%D1%83%D1%80%D0%B0%D1%82%D0%BE%D0%B2" title="Павел Муратов">Павел Муратов</a> писал о мавзолее Галлы Плацидии в начале XX века:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
<i>Итальянские мозаичисты любили немногие, густые и глубокие цвета — <a href="/wiki/%D0%A1%D0%B8%D0%BD%D0%B8%D0%B9_%D1%86%D0%B2%D0%B5%D1%82" title="Синий цвет">синий</a>, <a href="/wiki/%D0%97%D0%B5%D0%BB%D1%91%D0%BD%D1%8B%D0%B9_%D1%86%D0%B2%D0%B5%D1%82" title="Зелёный цвет">зелёный</a> и винно-<a href="/wiki/%D0%9A%D1%80%D0%B0%D1%81%D0%BD%D1%8B%D0%B9_%D1%86%D0%B2%D0%B5%D1%82" title="Красный цвет">красный</a>. Необычайно и как-то непостижимо глубок очень тёмный синий цвет на потолке мавзолея Галлы Плацидии. В зависимости от игры света, проникающего сюда сквозь маленькие оконца, он изумительно и неожиданно прекрасно переливает то зеленоватыми, то лиловыми, то багряными оттенками. На этот фон положено знаменитое изображение юного Доброго пастыря, сидящего среди белоснежных овец. Полукруги у окон украшает крупный орнамент с оленями, пьющими из источника. Гирлянды листьев и плодов вьются по низеньким аркам. При виде их великолепия невольно думается, что человечество никогда не создавало лучшего художественного средства для убранства церковных стен. И здесь благодаря крохотным размерам надгробной часовни мозаика не кажется делом суетной и холодной пышности. Сияющий синим огнём воздух, которым окружен саркофаг, некогда содержавший набальзамированное тело императрицы, достоин быть мечтой пламенно-религиозного воображения. Не к этому ли стремились, только другим путём, художники цветных стекол в готических соборах?</i><sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>
</p>
</blockquote>

<p>В центре <a href="/wiki/%D0%A1%D1%84%D0%B5%D1%80%D0%B0" title="Сфера">полусферического</a> <a href="/wiki/%D0%9A%D1%83%D0%BF%D0%BE%D0%BB" title="Купол">купола</a> помещён <a href="/wiki/%D0%97%D0%BE%D0%BB%D0%BE%D1%82%D0%BE" title="Золото">золотой</a> латинский <a href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82" title="Крест">крест</a>, окружённый восьмьюстами золотыми <a href="/wiki/%D0%97%D0%B2%D0%B5%D0%B7%D0%B4%D0%B0" title="Звезда">звёздами</a>, объединёнными в концентрические <a href="/wiki/%D0%9E%D0%BA%D1%80%D1%83%D0%B6%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Окружность">окружности</a>. Крест и звёзды помещены на <a href="/wiki/%D0%98%D0%BD%D0%B4%D0%B8%D0%B3%D0%BE" title="Индиго">индиго</a>-синем фоне, реалистически изображающем <a href="/wiki/%D0%9D%D0%BE%D1%87%D1%8C" title="Ночь">ночное</a> небо. Эта мозаика демонстрирует торжество <a href="/wiki/%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%BE%D1%81" title="Христос">Христа</a> над смертью, Его абсолютную власть над тварным миром. Христос символически показан как <a href="/wiki/%D0%A1%D0%BE%D0%BB%D0%BD%D1%86%D0%B5" title="Солнце">Солнце</a> правды (<a href="/wiki/%D0%9A%D0%BD%D0%B8%D0%B3%D0%B0_%D0%BF%D1%80%D0%BE%D1%80%D0%BE%D0%BA%D0%B0_%D0%9C%D0%B0%D0%BB%D0%B0%D1%85%D0%B8%D0%B8" title="Книга пророка Малахии">Мал.</a> <a class="extiw" href="https://ru.wikisource.org/wiki/%D0%9A%D0%BD%D0%B8%D0%B3%D0%B0_%D0%BF%D1%80%D0%BE%D1%80%D0%BE%D0%BA%D0%B0_%D0%9C%D0%B0%D0%BB%D0%B0%D1%85%D0%B8%D0%B8#4" title="s:Книга пророка Малахии">4:2</a>), окружённое Его верными, просвещёнными им и сияющими в мире Его светом (<a href="/wiki/%D0%95%D0%B2%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D0%B8%D0%B5_%D0%BE%D1%82_%D0%9C%D0%B0%D1%82%D1%84%D0%B5%D1%8F" title="Евангелие от Матфея">Мф.</a> <a class="extiw" href="https://ru.wikisource.org/wiki/%D0%9E%D1%82_%D0%9C%D0%B0%D1%82%D1%84%D0%B5%D1%8F_%D1%81%D0%B2%D1%8F%D1%82%D0%BE%D0%B5_%D0%B1%D0%BB%D0%B0%D0%B3%D0%BE%D0%B2%D0%B5%D1%81%D1%82%D0%B2%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5#5" title="s:От Матфея святое благовествование">5:16</a>). Мозаика указывает на Христа как на невечерний свет (<a href="/wiki/%D0%9A%D0%BD%D0%B8%D0%B3%D0%B0_%D0%BF%D1%80%D0%BE%D1%80%D0%BE%D0%BA%D0%B0_%D0%97%D0%B0%D1%85%D0%B0%D1%80%D0%B8%D0%B8" title="Книга пророка Захарии">Зах.</a> <a class="extiw" href="https://ru.wikisource.org/wiki/%D0%9A%D0%BD%D0%B8%D0%B3%D0%B0_%D0%BF%D1%80%D0%BE%D1%80%D0%BE%D0%BA%D0%B0_%D0%97%D0%B0%D1%85%D0%B0%D1%80%D0%B8%D0%B8#14" title="s:Книга пророка Захарии">14:6-7</a>), который не может объять тьма (<a href="/wiki/%D0%95%D0%B2%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D0%B8%D0%B5_%D0%BE%D1%82_%D0%98%D0%BE%D0%B0%D0%BD%D0%BD%D0%B0" title="Евангелие от Иоанна">Ин.</a> <a class="extiw" href="https://ru.wikisource.org/wiki/%D0%9E%D1%82_%D0%98%D0%BE%D0%B0%D0%BD%D0%BD%D0%B0_%D1%81%D0%B2%D1%8F%D1%82%D0%BE%D0%B5_%D0%B1%D0%BB%D0%B0%D0%B3%D0%BE%D0%B2%D0%B5%D1%81%D1%82%D0%B2%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5#1" title="s:От Иоанна святое благовествование">1:4-5</a>). Христос по мысли мозаичиста является смыслом и центром существования вселенной, готовым и желающим объединить вокруг себя всё человечество. Подчёркивая высокий смысл мозаики, художник направил длинный конец креста не вдоль длинной оси мавзолея (то есть по линии <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80" title="Север">север</a>-<a href="/wiki/%D0%AE%D0%B3" title="Юг">юг</a>), а на <a href="/wiki/%D0%92%D0%BE%D1%81%D1%82%D0%BE%D0%BA" title="Восток">восток</a>, в сторону которого традиционно ориентировались <a href="/wiki/%D0%90%D0%BB%D1%82%D0%B0%D1%80%D1%8C" title="Алтарь">алтари</a> в христианских храмах<sup class="reference" id="cite_ref-РИИ-14-16_13-0"><a href="#cite_note-РИИ-14-16-13">[13]</a></sup>.
</p><p>В <a href="/wiki/%D0%A5%D0%BE%D1%80%D0%BE%D0%B2%D0%BE%D0%B4" title="Хоровод">хоровод</a> звёзд, окружающих крест, включаются находящиеся в углах свода <a class="mw-redirect" href="/wiki/%D0%95%D0%B2%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D0%B8%D1%81%D1%82" title="Евангелист">евангелисты</a>, представленные их золотыми <a class="mw-redirect" href="/wiki/%D0%A1%D0%B8%D0%BC%D0%B2%D0%BE%D0%BB%D1%8B_%D0%B5%D0%B2%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D0%B8%D1%81%D1%82%D0%BE%D0%B2" title="Символы евангелистов">символическими изображениями</a>: <a href="/wiki/%D0%9B%D0%B5%D0%B2" title="Лев">львом</a> (<a class="mw-redirect" href="/wiki/%D0%9C%D0%B0%D1%80%D0%BA_(%D0%B0%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB)" title="Марк (апостол)">Марк</a>), <a href="/wiki/%D0%A2%D0%B5%D0%BB%D1%91%D0%BD%D0%BE%D0%BA" title="Телёнок">тельцом</a> (<a class="mw-redirect" href="/wiki/%D0%9B%D1%83%D0%BA%D0%B0_(%D0%B0%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB)" title="Лука (апостол)">Лука</a>), <a href="/wiki/%D0%9E%D1%80%D0%BB%D1%8B" title="Орлы">орлом</a> (<a href="/wiki/%D0%98%D0%BE%D0%B0%D0%BD%D0%BD_%D0%91%D0%BE%D0%B3%D0%BE%D1%81%D0%BB%D0%BE%D0%B2" title="Иоанн Богослов">Иоанн Богослов</a>) и <a href="/wiki/%D0%90%D0%BD%D0%B3%D0%B5%D0%BB" title="Ангел">ангелом</a> (<a href="/wiki/%D0%9B%D0%B5%D0%B2%D0%B8%D0%B9_%D0%9C%D0%B0%D1%82%D1%84%D0%B5%D0%B9" title="Левий Матфей">Матфей</a>)<sup class="reference" id="cite_ref-РИИ-16_9-1"><a href="#cite_note-РИИ-16-9">[9]</a></sup>.
</p><p>Потолок в «ветвях» мавзолея покрыт сложным мозаичным орнаментом, символизирующим <a href="/wiki/%D0%AD%D0%B4%D0%B5%D0%BC" title="Эдем">райский сад</a>. На тёмно-синем фоне разбросаны <a href="/wiki/%D0%9E%D0%BA%D1%80%D1%83%D0%B6%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Окружность">окружности</a>, звёзды и <a href="/wiki/%D0%A6%D0%B2%D0%B5%D1%82%D0%BE%D0%BA" title="Цветок">цветы</a>. Такой орнамент является необычным для западной части <a href="/wiki/%D0%A0%D0%B8%D0%BC%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F" title="Римская империя">Римской империи</a>, отдалённое подобие можно найти только в римской церкви <a class="mw-redirect" href="/wiki/%D0%A1%D0%B0%D0%BD%D1%82%D0%B0-%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%86%D0%B0" title="Санта-Констанца">Санта-Констанца</a><sup class="reference" id="cite_ref-РИИ-14_14-0"><a href="#cite_note-РИИ-14-14">[14]</a></sup>.
</p>
<h3><span id=".D0.90.D0.BF.D0.BE.D1.81.D1.82.D0.BE.D0.BB.D1.8B"></span><span class="mw-headline" id="Апостолы">Апостолы</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D0%B2%D0%B7%D0%BE%D0%BB%D0%B5%D0%B9_%D0%93%D0%B0%D0%BB%D0%BB%D1%8B_%D0%9F%D0%BB%D0%B0%D1%86%D0%B8%D0%B4%D0%B8%D0%B8&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Апостолы»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D0%B2%D0%B7%D0%BE%D0%BB%D0%B5%D0%B9_%D0%93%D0%B0%D0%BB%D0%BB%D1%8B_%D0%9F%D0%BB%D0%B0%D1%86%D0%B8%D0%B4%D0%B8%D0%B8&amp;action=edit&amp;section=6" title="Редактировать код раздела «Апостолы»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Apostles_in_lunette.jpg"><img class="mw-file-element" data-file-height="1224" data-file-width="1632" decoding="async" height="188" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/bc/Apostles_in_lunette.jpg/250px-Apostles_in_lunette.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/bc/Apostles_in_lunette.jpg/375px-Apostles_in_lunette.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/bc/Apostles_in_lunette.jpg/500px-Apostles_in_lunette.jpg 2x" width="250"/></a><figcaption><center>Изображения апостолов в люнете</center></figcaption></figure>
<p>В верхних <a href="/wiki/%D0%9B%D1%8E%D0%BD%D0%B5%D1%82" title="Люнет">люнетах</a> по сторонам от окон помещены парные изображения восьми из <a href="/wiki/%D0%94%D0%B2%D0%B5%D0%BD%D0%B0%D0%B4%D1%86%D0%B0%D1%82%D1%8C_%D0%B0%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB%D0%BE%D0%B2" title="Двенадцать апостолов">двенадцати апостолов</a>. Так как в середине каждого из четырёх люнетов находится окно, мозаичист был поставлен перед выбором: изобразить всех 12 апостолов и нарушить <a href="/wiki/%D0%A1%D0%B8%D0%BC%D0%BC%D0%B5%D1%82%D1%80%D0%B8%D1%8F" title="Симметрия">симметрию</a> (в этом случае в каждом люнете два апостола оказывались бы с одной стороны окна и только один — по левую), либо сохранить симметрию, пожертвовав четырьмя апостолами. Стремление к симметрии победило: «лишние» четыре апостола вытеснены в «ветви» мавзолея<sup class="reference" id="cite_ref-РИИ-16-17_15-0"><a href="#cite_note-РИИ-16-17-15">[15]</a></sup>.
</p><p>Ученики Христа изображены в полный рост с руками, воздетыми к изображённому на потолке <a href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82" title="Крест">кресту</a>, выражая <a href="/wiki/%D0%95%D0%B2%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D0%B8%D0%B5" title="Евангелие">евангельский</a> призыв Иисуса: «<i>возьми крест свой, и следуй за Мною</i>» (<a href="/wiki/%D0%95%D0%B2%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D0%B8%D0%B5_%D0%BE%D1%82_%D0%9C%D0%B0%D1%82%D1%84%D0%B5%D1%8F" title="Евангелие от Матфея">Мф.</a> <a class="extiw" href="https://ru.wikisource.org/wiki/%D0%9E%D1%82_%D0%9C%D0%B0%D1%82%D1%84%D0%B5%D1%8F_%D1%81%D0%B2%D1%8F%D1%82%D0%BE%D0%B5_%D0%B1%D0%BB%D0%B0%D0%B3%D0%BE%D0%B2%D0%B5%D1%81%D1%82%D0%B2%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5#16" title="s:От Матфея святое благовествование">16:24</a>). Апостолы изображены в <a href="/wiki/%D0%A1%D0%B5%D0%BD%D0%B0%D1%82_(%D0%94%D1%80%D0%B5%D0%B2%D0%BD%D0%B8%D0%B9_%D0%A0%D0%B8%D0%BC)" title="Сенат (Древний Рим)">сенаторских</a> <a href="/wiki/%D0%A2%D0%BE%D0%B3%D0%B0" title="Тога">тогах</a>, их рука поднята вверх в традиционном сенаторском жесте приветствия<sup class="reference" id="cite_ref-RCA-90_16-0"><a href="#cite_note-RCA-90-16">[16]</a></sup>. Все апостолы наделены специфическим портретными чертами, хотя ввиду того, что <a href="/wiki/%D0%98%D0%BA%D0%BE%D0%BD%D0%BE%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D1%8F" title="Иконография">иконографический</a> канон в <a href="/wiki/V_%D0%B2%D0%B5%D0%BA" title="V век">V веке</a> ещё не сформировался, опознать изображённых персонажей невозможно. Исключениями являются апостолы <a class="mw-redirect" href="/wiki/%D0%9F%D1%91%D1%82%D1%80_(%D0%B0%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB)" title="Пётр (апостол)">Пётр</a> (изображён с <a href="/wiki/%D0%97%D0%B0%D0%BC%D0%BA%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BA%D0%BB%D1%8E%D1%87" title="Замковый ключ">ключами</a>) и <a class="mw-redirect" href="/wiki/%D0%9F%D0%B0%D0%B2%D0%B5%D0%BB_(%D0%B0%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB)" title="Павел (апостол)">Павел</a> (высокий лоб, типичные <a class="mw-redirect" href="/wiki/%D0%95%D0%B2%D1%80%D0%B5%D0%B9" title="Еврей">еврейские</a> черты)<sup class="reference" id="cite_ref-РИИ-17_17-0"><a href="#cite_note-РИИ-17-17">[17]</a></sup>.
</p>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Mausoleums_der_Galla_Placidia_in_Ravenna_008.jpg"><img class="mw-file-element" data-file-height="573" data-file-width="800" decoding="async" height="179" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/8c/Mausoleums_der_Galla_Placidia_in_Ravenna_008.jpg/250px-Mausoleums_der_Galla_Placidia_in_Ravenna_008.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/8c/Mausoleums_der_Galla_Placidia_in_Ravenna_008.jpg/375px-Mausoleums_der_Galla_Placidia_in_Ravenna_008.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/8c/Mausoleums_der_Galla_Placidia_in_Ravenna_008.jpg/500px-Mausoleums_der_Galla_Placidia_in_Ravenna_008.jpg 2x" width="250"/></a><figcaption><center>Голуби, пьющие из источника воды живой</center></figcaption></figure>
<p>Апостолы изображены так, что продолжают поворотами своих фигур начатое в мозаике купола <a href="/wiki/%D0%9A%D1%80%D1%83%D0%B3" title="Круг">круговое</a> движение. Оно завершается в восточном люнете, где изображены апостолы <a class="mw-redirect" href="/wiki/%D0%9F%D1%91%D1%82%D1%80_(%D0%B0%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB)" title="Пётр (апостол)">Пётр</a> и <a class="mw-redirect" href="/wiki/%D0%9F%D0%B0%D0%B2%D0%B5%D0%BB_(%D0%B0%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB)" title="Павел (апостол)">Павел</a>. Здесь, как и в мозаике <a href="/wiki/%D0%9A%D1%83%D0%BF%D0%BE%D0%BB" title="Купол">купола</a>, ради направления на <a href="/wiki/%D0%92%D0%BE%D1%81%D1%82%D0%BE%D0%BA" title="Восток">Восток</a> художник нарушил логику, предписываемую <a href="/wiki/%D0%90%D1%80%D1%85%D0%B8%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D0%B0" title="Архитектура">архитектурой</a> здания (длинная <a class="mw-disambig" href="/wiki/%D0%9E%D1%81%D1%8C" title="Ось">ось</a> которого ориентирована по линии север-юг)<sup class="reference" id="cite_ref-РИИ-17_17-1"><a href="#cite_note-РИИ-17-17">[17]</a></sup>.
</p><p>Под окном в каждом люнете, то есть между фигурами апостолов, помещены мозаичные изображения <a href="/wiki/%D0%A7%D0%B0%D1%88%D0%B0" title="Чаша">чаш</a> или <a href="/wiki/%D0%A4%D0%BE%D0%BD%D1%82%D0%B0%D0%BD" title="Фонтан">фонтанов</a>, из которых пьёт (или рядом с которыми сидит) пара <a class="mw-redirect" href="/wiki/%D0%93%D0%BE%D0%BB%D1%83%D0%B1%D1%8C" title="Голубь">голубей</a>. Это — раннехристианское (часто встречающееся в <a href="/wiki/%D0%9A%D0%B0%D1%82%D0%B0%D0%BA%D0%BE%D0%BC%D0%B1%D1%8B" title="Катакомбы">катакомбах</a>) символическое изображение душ, пьющих из источника воды живой в <a href="/wiki/%D0%A0%D0%B0%D0%B9" title="Рай">раю</a><sup class="reference" id="cite_ref-RCA-90_16-1"><a href="#cite_note-RCA-90-16">[16]</a></sup>.
</p>
<h3><span id=".D0.94.D0.BE.D0.B1.D1.80.D1.8B.D0.B9_.D0.9F.D0.B0.D1.81.D1.82.D1.8B.D1.80.D1.8C_.E2.80.94_.D0.BC.D0.BE.D0.B7.D0.B0.D0.B8.D0.BA.D0.B0_.D1.81.D0.B5.D0.B2.D0.B5.D1.80.D0.BD.D0.BE.D0.B3.D0.BE_.D0.BB.D1.8E.D0.BD.D0.B5.D1.82.D0.B0"></span><span class="mw-headline" id="Добрый_Пастырь_—_мозаика_северного_люнета">Добрый Пастырь — мозаика северного люнета</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D0%B2%D0%B7%D0%BE%D0%BB%D0%B5%D0%B9_%D0%93%D0%B0%D0%BB%D0%BB%D1%8B_%D0%9F%D0%BB%D0%B0%D1%86%D0%B8%D0%B4%D0%B8%D0%B8&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Добрый Пастырь — мозаика северного люнета»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D0%B2%D0%B7%D0%BE%D0%BB%D0%B5%D0%B9_%D0%93%D0%B0%D0%BB%D0%BB%D1%8B_%D0%9F%D0%BB%D0%B0%D1%86%D0%B8%D0%B4%D0%B8%D0%B8&amp;action=edit&amp;section=7" title="Редактировать код раздела «Добрый Пастырь — мозаика северного люнета»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:%22The_good_Shepherd%22_mosaic_-_Mausoleum_of_Galla_Placidia.jpg"><img class="mw-file-element" data-file-height="2645" data-file-width="4179" decoding="async" height="165" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/db/%22The_good_Shepherd%22_mosaic_-_Mausoleum_of_Galla_Placidia.jpg/260px-%22The_good_Shepherd%22_mosaic_-_Mausoleum_of_Galla_Placidia.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/d/db/%22The_good_Shepherd%22_mosaic_-_Mausoleum_of_Galla_Placidia.jpg/390px-%22The_good_Shepherd%22_mosaic_-_Mausoleum_of_Galla_Placidia.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/d/db/%22The_good_Shepherd%22_mosaic_-_Mausoleum_of_Galla_Placidia.jpg/520px-%22The_good_Shepherd%22_mosaic_-_Mausoleum_of_Galla_Placidia.jpg 2x" width="260"/></a><figcaption><center>Мозаика «Христос — Добрый Пастырь»</center></figcaption></figure>
<p>Мозаика северного нижнего <a href="/wiki/%D0%9B%D1%8E%D0%BD%D0%B5%D1%82" title="Люнет">люнета</a> (стена над входом). Исследователи отмечают, что сюжет мозаики часто встречается в христианских <a href="/wiki/%D0%9A%D0%B0%D1%82%D0%B0%D0%BA%D0%BE%D0%BC%D0%B1%D1%8B" title="Катакомбы">катакомбах</a> и относится к категории заупокойных.<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup>
</p><p><a href="/wiki/%D0%98%D0%B8%D1%81%D1%83%D1%81_%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%BE%D1%81" title="Иисус Христос">Иисус</a> изображён в образе молодого безбородого юноши-пастуха, вокруг которого по зелёной траве гуляют овцы, и <a href="/wiki/%D0%9C%D0%B5%D1%81%D1%81%D0%B8%D1%8F" title="Мессия">Мессия</a> ласково прикасается к одной из них. В отличие от катакомбной живописи, где <a href="/wiki/%D0%94%D0%BE%D0%B1%D1%80%D1%8B%D0%B9_%D0%9F%D0%B0%D1%81%D1%82%D1%8B%D1%80%D1%8C" title="Добрый Пастырь">Пастырь</a> был обычным деревенским пастухом, здесь Иисус одет в золотой <a href="/wiki/%D0%A5%D0%B8%D1%82%D0%BE%D0%BD" title="Хитон">хитон</a>, а на коленях лежит <a href="/wiki/%D0%9F%D1%83%D1%80%D0%BF%D1%83%D1%80" title="Пурпур">пурпурный</a> <a href="/wiki/%D0%9F%D0%BB%D0%B0%D1%89" title="Плащ">плащ</a>. Он сидит на пригорке (образ <a href="/wiki/%D0%A2%D1%80%D0%BE%D0%BD" title="Трон">трона</a>), держа в руке <a href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82" title="Крест">крест</a>, выступающий в роли императорского <a href="/wiki/%D0%96%D0%B5%D0%B7%D0%BB" title="Жезл">жезла</a>. Академик <a href="/wiki/%D0%9B%D0%B0%D0%B7%D0%B0%D1%80%D0%B5%D0%B2,_%D0%92%D0%B8%D0%BA%D1%82%D0%BE%D1%80_%D0%9D%D0%B8%D0%BA%D0%B8%D1%82%D0%B8%D1%87" title="Лазарев, Виктор Никитич">В. Н. Лазарев</a> отмечает величественную позу Христа: его ноги скрещены, правая рука касается головы овечки, но взгляд при этом обращён в другую сторону. Благодаря такой позе, Пастырь становится смысловым центром мозаики: он видит всех своих овец, и все овцы смотрят на него<sup class="reference" id="cite_ref-РИИ-17-19_19-0"><a href="#cite_note-РИИ-17-19-19">[19]</a></sup>.
</p><p>Для мозаики характерны тонкие переходы цвета, использование полутонов для изображения кожи. Всё это наряду с обрамляющей мозаику пышной рамой свидетельствует о влиянии античного искусства.
</p>
<h3><span id=".D0.9C.D0.BE.D0.B7.D0.B0.D0.B8.D0.BA.D0.B0_.D1.8E.D0.B6.D0.BD.D0.BE.D0.B3.D0.BE_.D0.BB.D1.8E.D0.BD.D0.B5.D1.82.D0.B0"></span><span class="mw-headline" id="Мозаика_южного_люнета">Мозаика южного люнета</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D0%B2%D0%B7%D0%BE%D0%BB%D0%B5%D0%B9_%D0%93%D0%B0%D0%BB%D0%BB%D1%8B_%D0%9F%D0%BB%D0%B0%D1%86%D0%B8%D0%B4%D0%B8%D0%B8&amp;veaction=edit&amp;section=8" title="Редактировать раздел «Мозаика южного люнета»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D0%B2%D0%B7%D0%BE%D0%BB%D0%B5%D0%B9_%D0%93%D0%B0%D0%BB%D0%BB%D1%8B_%D0%9F%D0%BB%D0%B0%D1%86%D0%B8%D0%B4%D0%B8%D0%B8&amp;action=edit&amp;section=8" title="Редактировать код раздела «Мозаика южного люнета»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:LawrenceRavenna.jpg"><img class="mw-file-element" data-file-height="500" data-file-width="787" decoding="async" height="165" src="//upload.wikimedia.org/wikipedia/commons/thumb/2/23/LawrenceRavenna.jpg/260px-LawrenceRavenna.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/2/23/LawrenceRavenna.jpg/390px-LawrenceRavenna.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/2/23/LawrenceRavenna.jpg/520px-LawrenceRavenna.jpg 2x" width="260"/></a><figcaption><center>Святой Лаврентий, идущий на костёр (?)</center></figcaption></figure>
<p>В южном нижнем <a href="/wiki/%D0%9B%D1%8E%D0%BD%D0%B5%D1%82" title="Люнет">люнете</a> помещена мозаика следующего содержания. Справа человек в развевающихся белых одеждах, с <a href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82" title="Крест">крестом</a> и книгой в руках, идет к центру мозаики. Слева в раскрытом <a href="/wiki/%D0%A8%D0%BA%D0%B0%D1%84" title="Шкаф">шкафу</a> на полках лежат четыре книги, подписанные <a href="/wiki/%D0%9B%D0%B5%D0%B2%D0%B8%D0%B9_%D0%9C%D0%B0%D1%82%D1%84%D0%B5%D0%B9" title="Левий Матфей">Матфей</a>, <a class="mw-redirect" href="/wiki/%D0%9C%D0%B0%D1%80%D0%BA_(%D0%B0%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB)" title="Марк (апостол)">Марк</a>, <a class="mw-redirect" href="/wiki/%D0%9B%D1%83%D0%BA%D0%B0_(%D0%B0%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB)" title="Лука (апостол)">Лука</a> и <a href="/wiki/%D0%98%D0%BE%D0%B0%D0%BD%D0%BD_%D0%91%D0%BE%D0%B3%D0%BE%D1%81%D0%BB%D0%BE%D0%B2" title="Иоанн Богослов">Иоанн</a>, то есть, очевидно, четыре <a href="/wiki/%D0%95%D0%B2%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D0%B8%D0%B5" title="Евангелие">Евангелия</a>. В центре мозаики под решёткой пылает огонь. Поскольку главный герой мозаики не подписан, высказывается несколько версий, объясняющих значение мозаики.
</p>
<ul><li>Человек в белых одеждах — <a href="/wiki/%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%BE%D1%81" title="Христос">Христос</a> в Своём <a class="mw-redirect" href="/wiki/%D0%92%D1%82%D0%BE%D1%80%D0%BE%D0%B5_%D0%BF%D1%80%D0%B8%D1%88%D0%B5%D1%81%D1%82%D0%B2%D0%B8%D0%B5" title="Второе пришествие">втором пришествии</a>. В этом случае книга, которую Он держит в руках, истолковывается как одна из книг, сообразно с которыми будут судимы живые и мёртвые (<a href="/wiki/%D0%9E%D1%82%D0%BA%D1%80%D0%BE%D0%B2%D0%B5%D0%BD%D0%B8%D0%B5_%D0%98%D0%BE%D0%B0%D0%BD%D0%BD%D0%B0_%D0%91%D0%BE%D0%B3%D0%BE%D1%81%D0%BB%D0%BE%D0%B2%D0%B0" title="Откровение Иоанна Богослова">Отк.</a> <a class="extiw" href="https://ru.wikisource.org/wiki/%D0%9E%D1%82%D0%BA%D1%80%D0%BE%D0%B2%D0%B5%D0%BD%D0%B8%D0%B5_%D1%81%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%98%D0%BE%D0%B0%D0%BD%D0%BD%D0%B0_%D0%91%D0%BE%D0%B3%D0%BE%D1%81%D0%BB%D0%BE%D0%B2%D0%B0#20" title="s:Откровение святого Иоанна Богослова">20:12</a>). Огонь в центре в этом случае становится знаком <a href="/wiki/%D0%93%D0%B5%D0%B5%D0%BD%D0%BD%D0%B0" title="Геенна">геенны огненной</a>.</li>
<li>В белых одеждах изображён <a href="/wiki/%D0%90%D0%BD%D0%B3%D0%B5%D0%BB" title="Ангел">Ангел</a> с «книжкой раскрытой»(<a href="/wiki/%D0%9E%D1%82%D0%BA%D1%80%D0%BE%D0%B2%D0%B5%D0%BD%D0%B8%D0%B5_%D0%98%D0%BE%D0%B0%D0%BD%D0%BD%D0%B0_%D0%91%D0%BE%D0%B3%D0%BE%D1%81%D0%BB%D0%BE%D0%B2%D0%B0" title="Откровение Иоанна Богослова">Отк.</a> <a class="extiw" href="https://ru.wikisource.org/wiki/%D0%9E%D1%82%D0%BA%D1%80%D0%BE%D0%B2%D0%B5%D0%BD%D0%B8%D0%B5_%D1%81%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%98%D0%BE%D0%B0%D0%BD%D0%BD%D0%B0_%D0%91%D0%BE%D0%B3%D0%BE%D1%81%D0%BB%D0%BE%D0%B2%D0%B0#10" title="s:Откровение святого Иоанна Богослова">10:1</a>), объявляющий день <a href="/wiki/%D0%A1%D1%82%D1%80%D0%B0%D1%88%D0%BD%D1%8B%D0%B9_%D1%81%D1%83%D0%B4" title="Страшный суд">Страшного суда</a>.</li>
<li>Человек в белых одеждах — один из <a href="/wiki/%D0%9E%D1%82%D1%86%D1%8B_%D0%A6%D0%B5%D1%80%D0%BA%D0%B2%D0%B8" title="Отцы Церкви">Отцов Церкви</a>, готовый бросить в огонь <a href="/wiki/%D0%95%D1%80%D0%B5%D1%81%D1%8C" title="Ересь">еретические писания</a><sup class="reference" id="cite_ref-РИИ-13-14_20-0"><a href="#cite_note-РИИ-13-14-20">[20]</a></sup>.</li></ul>
<ul><li>Наиболее распространённая версия — на мозаике изображён <a class="mw-redirect" href="/wiki/%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B9_%D0%9B%D0%B0%D0%B2%D1%80%D0%B5%D0%BD%D1%82%D0%B8%D0%B9" title="Святой Лаврентий">святой Лаврентий</a>, идущий принять смерть на горящем в центре композиции костре. Его развевающиеся одежды демонстрируют желание мученика принять смерть за Христа, а решётка в этом случае легко истолковывается как орудие его казни. Крест в руках мученика объясняется как <a href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82%D0%BD%D1%8B%D0%B9_%D1%85%D0%BE%D0%B4" title="Крестный ход">процессионный</a> крест, а книга — <a class="mw-redirect" href="/wiki/%D0%9F%D1%81%D0%B0%D0%BB%D1%82%D1%8B%D1%80%D1%8C" title="Псалтырь">Псалтырь</a>, оба предмета указывают на <a class="mw-redirect" href="/wiki/%D0%94%D1%8C%D1%8F%D0%BA%D0%BE%D0%BD" title="Дьякон">дьяконский</a> сан Лаврентия. В комбинации с Евангелиями, лежащими в шкафу, крест и книга в руках Лаврентия символизируют, что он принял мученичество подражая Христу и усвоив Его учение.<sup class="reference" id="cite_ref-lazarev_1-1"><a href="#cite_note-lazarev-1">[1]</a></sup></li></ul>

<h3><span id=".D0.9C.D0.BE.D0.B7.D0.B0.D0.B8.D0.BA.D0.B8_.D0.B2_.D0.B7.D0.B0.D0.BF.D0.B0.D0.B4.D0.BD.D0.BE.D0.B9_.D0.B8_.D0.B2.D0.BE.D1.81.D1.82.D0.BE.D1.87.D0.BD.D0.BE.D0.B9_.C2.AB.D0.B2.D0.B5.D1.82.D0.B2.D1.8F.D1.85.C2.BB"></span><span class="mw-headline" id="Мозаики_в_западной_и_восточной_«ветвях»">Мозаики в западной и восточной «ветвях»</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D0%B2%D0%B7%D0%BE%D0%BB%D0%B5%D0%B9_%D0%93%D0%B0%D0%BB%D0%BB%D1%8B_%D0%9F%D0%BB%D0%B0%D1%86%D0%B8%D0%B4%D0%B8%D0%B8&amp;veaction=edit&amp;section=9" title="Редактировать раздел «Мозаики в западной и восточной «ветвях»»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D0%B2%D0%B7%D0%BE%D0%BB%D0%B5%D0%B9_%D0%93%D0%B0%D0%BB%D0%BB%D1%8B_%D0%9F%D0%BB%D0%B0%D1%86%D0%B8%D0%B4%D0%B8%D0%B8&amp;action=edit&amp;section=9" title="Редактировать код раздела «Мозаики в западной и восточной «ветвях»»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>В западной и восточной «ветвях» мавзолея, за <a href="/wiki/%D0%A1%D0%B0%D1%80%D0%BA%D0%BE%D1%84%D0%B0%D0%B3" title="Саркофаг">саркофагами</a>, можно видеть ещё две символические раннехристианские мозаики. На них пара <a class="mw-redirect" href="/wiki/%D0%9E%D0%BB%D0%B5%D0%BD%D1%8C" title="Олень">оленей</a> жадно пьет из источника. Сюжет мозаики навеян стихами <a class="mw-redirect" href="/wiki/%D0%9F%D1%81%D0%B0%D0%BB%D0%BE%D0%BC_41" title="Псалом 41">41 псалма</a>: «<i>Как лань желает к потокам воды, так желает душа моя к Тебе, Боже</i>!» (<a href="/wiki/%D0%9F%D1%81%D0%B0%D0%BB%D1%82%D0%B8%D1%80%D1%8C" title="Псалтирь">Пс.</a> <a class="extiw" href="https://ru.wikisource.org/wiki/%D0%9F%D1%81%D0%B0%D0%BB%D1%82%D0%B8%D1%80%D1%8C#41" title="s:Псалтирь">41:2</a>). Традиционно пара оленей истолковывается как символическое изображение христиан, обратившихся из <a class="mw-redirect" href="/wiki/%D0%98%D1%83%D0%B4%D0%B5%D0%B9%D1%81%D1%82%D0%B2%D0%BE" title="Иудейство">иудеев</a> и <a class="mw-redirect" href="/wiki/%D0%AF%D0%B7%D1%8B%D1%87%D0%BD%D0%B8%D0%BA" title="Язычник">язычников</a><sup class="reference" id="cite_ref-РИИ-17_17-2"><a href="#cite_note-РИИ-17-17">[17]</a></sup>. Этот сюжет встречается в <a href="/wiki/%D0%9A%D0%B0%D1%82%D0%B0%D0%BA%D0%BE%D0%BC%D0%B1%D1%8B" title="Катакомбы">катакомбах</a> и активно использовался впоследствии (например, в алтарной мозаике в <a href="/wiki/%D0%A0%D0%B8%D0%BC" title="Рим">римской</a> <a href="/wiki/%D0%91%D0%B0%D0%B7%D0%B8%D0%BB%D0%B8%D0%BA%D0%B0_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%9A%D0%BB%D0%B8%D0%BC%D0%B5%D0%BD%D1%82%D0%B0" title="Базилика Святого Климента">базилике Сан-Клементе</a>)
</p>
<h2><span id=".D0.A1.D0.B0.D1.80.D0.BA.D0.BE.D1.84.D0.B0.D0.B3.D0.B8"></span><span class="mw-headline" id="Саркофаги">Саркофаги</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9C%D0%B0%D0%B2%D0%B7%D0%BE%D0%BB%D0%B5%D0%B9_%D0%93%D0%B0%D0%BB%D0%BB%D1%8B_%D0%9F%D0%BB%D0%B0%D1%86%D0%B8%D0%B4%D0%B8%D0%B8&amp;veaction=edit&amp;section=10" title="Редактировать раздел «Саркофаги»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D0%B0%D0%B2%D0%B7%D0%BE%D0%BB%D0%B5%D0%B9_%D0%93%D0%B0%D0%BB%D0%BB%D1%8B_%D0%9F%D0%BB%D0%B0%D1%86%D0%B8%D0%B4%D0%B8%D0%B8&amp;action=edit&amp;section=10" title="Редактировать код раздела «Саркофаги»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>В интерьере — три <a href="/wiki/%D0%A1%D0%B0%D1%80%D0%BA%D0%BE%D1%84%D0%B0%D0%B3" title="Саркофаг">саркофага</a> из греческого мрамора, приписываемые самой Галле Плацидии, её сыну <a href="/wiki/%D0%92%D0%B0%D0%BB%D0%B5%D0%BD%D1%82%D0%B8%D0%BD%D0%B8%D0%B0%D0%BD_III" title="Валентиниан III">Валентиниану</a> и второму супругу <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%86%D0%B8%D0%B9_III" title="Констанций III">Констанцию III</a>.
</p>

<td style="text-align: left;"><center><i><a class="mw-redirect" href="/wiki/%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%91%D0%BB%D0%BE%D0%BA" title="Александр Блок">Александр Блок</a>, «Равенна»</i>

<ul><li><i>Саркофаг Галлы Плацидии</i> — занимает центральное место, лишён какого-либо декора и вероятно является незаконченным. С учётом необычно больших размеров саркофага и отсутствия на нём какой-либо христианской символики, монумент приписывается богатому и знатному язычнику. Возможность погребения в нём Галлы Плацидии отвергается современными историками. Тем не менее, источники <a href="/wiki/XIV_%D0%B2%D0%B5%D0%BA" title="XIV век">XIV</a>-<a href="/wiki/XVI_%D0%B2%D0%B5%D0%BA" title="XVI век">XVI веков</a> (в том числе, архиепископ Равенны Ринальдо да Конкореджио) утверждают, что через большое окно в задней части саркофага (сейчас замуровано) можно было видеть погребённое там тело, усаженное на <a href="/wiki/%D0%9A%D0%B8%D0%BF%D0%B0%D1%80%D0%B8%D1%81" title="Кипарис">кипарисовый</a> <a href="/wiki/%D0%A2%D1%80%D0%BE%D0%BD" title="Трон">трон</a>. Предположительно, речь идёт о теле, погребённом таким необычным образом не ранее <a href="/wiki/XIII_%D0%B2%D0%B5%D0%BA" title="XIII век">XIII</a>-XIV века с возможным намерением <a href="/wiki/%D0%A4%D0%B0%D0%BB%D1%8C%D1%81%D0%B8%D1%84%D0%B8%D0%BA%D0%B0%D1%86%D0%B8%D1%8F" title="Фальсификация">имитировать</a> останки <a class="mw-redirect" href="/wiki/%D0%90%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0_(%D1%82%D0%B8%D1%82%D1%83%D0%BB)" title="Августа (титул)">Августы</a>. В <a href="/wiki/1577_%D0%B3%D0%BE%D0%B4" title="1577 год">1577 году</a> местные мальчишки из любопытства просунули через указанное окно саркофага горящие <a href="/wiki/%D0%A1%D0%B2%D0%B5%D1%87%D0%B0" title="Свеча">свечи</a>, которые, едва прикоснувшись к кипарисовому трону, вызвали <a href="/wiki/%D0%9F%D0%BE%D0%B6%D0%B0%D1%80" title="Пожар">пожар</a>. Вследствие пожара, в саркофаге сохранились только несколько костей, <a href="/wiki/%D0%A7%D0%B5%D1%80%D0%B5%D0%BF" title="Череп">череп</a> и обугленные куски дерева, обнаруженные там при вскрытиях саркофага в 1577 и <a href="/wiki/1898_%D0%B3%D0%BE%D0%B4" title="1898 год">1898 годах</a><sup class="reference" id="cite_ref-РИИ-19_11-1"><a href="#cite_note-РИИ-19-11">[11]</a></sup>.</li>
<li><i>Саркофаг Констанция</i> — создание датируется <a href="/wiki/V_%D0%B2%D0%B5%D0%BA" title="V век">V веком</a>, установлен в левой ветви «креста». На его передней стенке изображён <a href="/wiki/%D0%98%D0%B8%D1%81%D1%83%D1%81_%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%BE%D1%81" title="Иисус Христос">Христос</a> в образе <a href="/wiki/%D0%90%D0%B3%D0%BD%D0%B5%D1%86_%D0%91%D0%BE%D0%B6%D0%B8%D0%B9" title="Агнец Божий">агнца</a>, голова его окружена <a href="/wiki/%D0%9D%D0%B8%D0%BC%D0%B1" title="Нимб">нимбом</a>, содержащим <a href="/wiki/%D0%9C%D0%BE%D0%BD%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B0" title="Монограмма">монограмму</a> <a href="/wiki/%D0%98%D0%B8%D1%81%D1%83%D1%81_%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%BE%D1%81" title="Иисус Христос">Христа</a> — переплетённые <a href="/wiki/%D0%93%D1%80%D0%B5%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Греческий язык">греческие</a> буквы Χ и Ρ. Агнец стоит на скале, из которой вытекают четыре потока, изображающие четыре реки <a href="/wiki/%D0%AD%D0%B4%D0%B5%D0%BC" title="Эдем">Эдема</a>. Справа и слева от скалы находятся два агнца уже без нимбов, символизирующие <a href="/wiki/%D0%90%D0%BF%D0%BE%D1%81%D1%82%D0%BE%D0%BB%D1%8B" title="Апостолы">апостолов</a>. Эти образы обрамлены двумя <a href="/wiki/%D0%9F%D0%B0%D0%BB%D1%8C%D0%BC%D0%BE%D0%B2%D1%8B%D0%B5" title="Пальмовые">пальмами</a>, символизирующими жизнь праведников. В <a href="/wiki/1738_%D0%B3%D0%BE%D0%B4" title="1738 год">1738 году</a> саркофаг был вскрыт, и исследователи обнаружили в нём два хорошо сохранившиеся <a href="/wiki/%D0%A7%D0%B5%D1%80%D0%B5%D0%BF" title="Череп">черепа</a> с зубами<sup class="reference" id="cite_ref-РИИ-21_21-0"><a href="#cite_note-РИИ-21-21">[21]</a></sup>.</li>
<li><i>Саркофаг Валентиниана</i> — датируется <a href="/wiki/VI_%D0%B2%D0%B5%D0%BA" title="VI век">VI веком</a>, установлен в правой ветви «креста». Имеет полуцилиндрическую крышку с <a href="/wiki/%D0%A7%D0%B5%D1%88%D1%83%D1%8F" title="Чешуя">чешуйчатым</a> <a href="/wiki/%D0%9E%D1%80%D0%BD%D0%B0%D0%BC%D0%B5%D0%BD%D1%82" title="Орнамент">орнаментом</a>. На передней стенке изображен агнец-Христос, стоящий у подножия холма, из которого вытекают четыре райские реки, холм увенчан крестом, на перекладине которого сидят два <a class="mw-redirect" href="/wiki/%D0%93%D0%BE%D0%BB%D1%83%D0%B1%D1%8C" title="Голубь">голубя</a>. На обеих боковых стенках изображен крест с <a href="/wiki/%D0%A0%D0%B0%D0%BA%D0%BE%D0%B2%D0%B8%D0%BD%D0%B0" title="Раковина">раковиной</a> (часто использовавшийся в <a href="/wiki/%D0%9A%D0%B0%D1%82%D0%B0%D0%BA%D0%BE%D0%BC%D0%B1%D1%8B" title="Катакомбы">катакомбной</a> живописи символ смерти, из которой возрождается жизнь). В 1738 году этот саркофаг также был вскрыт, в нём были обнаружены костные останки мужчины и женщины<sup class="reference" id="cite_ref-РИИ-21_21-1"><a href="#cite_note-РИИ-21-21">[21]</a></sup>.</li></ul>
