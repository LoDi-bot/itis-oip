<p><b>Влахе́рнская ико́на Бо́жией Ма́тери</b> (<a href="/wiki/%D0%93%D1%80%D0%B5%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Греческий язык">греч.</a> <span lang="el"><span style="font-family: palatino linotype, new athena unicode, athena, gentium, code2000, serif; font-size: 105%;">Θεοτόκος των Βλαχερνών</span></span>), или <b>Влахернетисса</b> (<a href="/wiki/%D0%93%D1%80%D0%B5%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Греческий язык">греч.</a> <span lang="el"><span style="font-family: palatino linotype, new athena unicode, athena, gentium, code2000, serif; font-size: 105%;">Βλαχερνίτισσα</span></span>) — почитаемая в <a href="/wiki/%D0%9F%D1%80%D0%B0%D0%B2%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%B8%D0%B5" title="Православие">Православной церкви</a> <a href="/wiki/%D0%98%D0%BA%D0%BE%D0%BD%D0%B0" title="Икона">икона</a> <a href="/wiki/%D0%91%D0%BE%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%B8%D1%86%D0%B0" title="Богородица">Богородицы</a>. Привезена в Россию из <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD%D0%BE%D0%BF%D0%BE%D0%BB%D1%8C" title="Константинополь">Константинополя</a> в <a href="/wiki/1653_%D0%B3%D0%BE%D0%B4" title="1653 год">1653 году</a> в качестве дара царю <a href="/wiki/%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%9C%D0%B8%D1%85%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2%D0%B8%D1%87" title="Алексей Михайлович">Алексею Михайловичу</a>. Выполнена в редкой технике <a href="/wiki/%D0%92%D0%BE%D1%81%D0%BA%D0%BE%D0%BC%D0%B0%D1%81%D1%82%D0%B8%D0%BA%D0%B0" title="Воскомастика">воскомастики</a> и относится к иконописному типу <a href="/wiki/%D0%9E%D0%B4%D0%B8%D0%B3%D0%B8%D1%82%D1%80%D0%B8%D1%8F" title="Одигитрия">Одигитрия</a>.
</p><p>Икона почитается у верующих <a href="/wiki/%D0%A7%D1%83%D0%B4%D0%BE%D1%82%D0%B2%D0%BE%D1%80%D0%BD%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%B0" title="Чудотворная икона">чудотворной</a>, празднование в её честь совершается: <abbr title="по юлианскому календарю">2</abbr> <a href="/wiki/15_%D0%B8%D1%8E%D0%BB%D1%8F" title="15 июля">(15) июля</a> (в праздник <a class="mw-redirect" href="/wiki/%D0%A0%D0%B8%D0%B7%D0%BE%D0%BF%D0%BE%D0%BB%D0%BE%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5" title="Ризоположение">Ризоположения</a>), <abbr title="по юлианскому календарю">7</abbr> <a href="/wiki/20_%D0%B8%D1%8E%D0%BB%D1%8F" title="20 июля">(20) июля</a> и в <a href="/wiki/%D0%A1%D1%83%D0%B1%D0%B1%D0%BE%D1%82%D0%B0_%D0%90%D0%BA%D0%B0%D1%84%D0%B8%D1%81%D1%82%D0%B0" title="Суббота Акафиста">субботу Акафиста</a> (в субботу пятой <a class="mw-redirect" href="/wiki/%D0%A1%D0%B5%D0%B4%D0%BC%D0%B8%D1%86%D0%B0" title="Седмица">седмицы</a> <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B8%D0%B9_%D0%BF%D0%BE%D1%81%D1%82" title="Великий пост">Великого поста</a>).
</p>

<h2><span id=".D0.98.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.8F_.D0.B8.D0.BA.D0.BE.D0.BD.D1.8B"></span><span class="mw-headline" id="История_иконы">История иконы</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%B0&amp;veaction=edit&amp;section=1" title="Редактировать раздел «История иконы»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%B0&amp;action=edit&amp;section=1" title="Редактировать код раздела «История иконы»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id=".D0.92.D0.B8.D0.B7.D0.B0.D0.BD.D1.82.D0.B8.D0.B9.D1.81.D0.BA.D0.B8.D0.B9_.D0.BF.D0.B5.D1.80.D0.B8.D0.BE.D0.B4"></span><span class="mw-headline" id="Византийский_период">Византийский период</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%B0&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Византийский период»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%B0&amp;action=edit&amp;section=2" title="Редактировать код раздела «Византийский период»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Влахернская икона входит в число тех старинных икон, создание которых предание приписывает евангелисту <a href="/wiki/%D0%9B%D1%83%D0%BA%D0%B0_(%D0%B5%D0%B2%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D0%B8%D1%81%D1%82)" title="Лука (евангелист)">Луке</a><sup class="reference" id="cite_ref-Vlax1_1-0"><a href="#cite_note-Vlax1-1">[1]</a></sup>. Икона изначально хранилась на родине евангелиста в <a href="/wiki/%D0%90%D0%BD%D1%82%D0%B8%D0%BE%D1%85%D0%B8%D1%8F" title="Антиохия">Антиохии</a>, а затем была перенесена в <a href="/wiki/%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC" title="Иерусалим">Иерусалим</a>. Императрица <a href="/wiki/%D0%95%D0%B2%D0%B4%D0%BE%D0%BA%D0%B8%D1%8F_(%D1%81%D1%83%D0%BF%D1%80%D1%83%D0%B3%D0%B0_%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D1%8F_II)" title="Евдокия (супруга Феодосия II)">Евдокия</a> привезла её в <a href="/wiki/439_%D0%B3%D0%BE%D0%B4" title="439 год">439 году</a> из Иерусалима в <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD%D0%BE%D0%BF%D0%BE%D0%BB%D1%8C" title="Константинополь">Константинополь</a> вместе с другими христианскими <a href="/wiki/%D0%A0%D0%B5%D0%BB%D0%B8%D0%BA%D0%B2%D0%B8%D1%8F" title="Реликвия">реликвиями</a>. Переданная в дар <a href="/wiki/%D0%9F%D1%83%D0%BB%D1%8C%D1%85%D0%B5%D1%80%D0%B8%D1%8F" title="Пульхерия">Пульхерии</a>, сестре императора <a href="/wiki/%D0%A4%D0%B5%D0%BE%D0%B4%D0%BE%D1%81%D0%B8%D0%B9_II" title="Феодосий II">Феодосия II</a>, икона была помещена в <a href="/wiki/%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D1%86%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C" title="Влахернская церковь">церкви Богородицы во Влахернах</a>. От имени храма икона получила своё настоящее название.
</p><p>Влахернская икона считалась покровительницей города Константинополя и византийских императоров. В <a href="/wiki/626_%D0%B3%D0%BE%D0%B4" title="626 год">626 году</a> во время <a href="/wiki/%D0%9E%D1%81%D0%B0%D0%B4%D0%B0_%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD%D0%BE%D0%BF%D0%BE%D0%BB%D1%8F_(626)" title="Осада Константинополя (626)">осады города</a> <a href="/wiki/%D0%A1%D0%B0%D1%81%D0%B0%D0%BD%D0%B8%D0%B4%D1%8B" title="Сасаниды">персами</a>, <a href="/wiki/%D0%90%D0%B2%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%B0%D0%B3%D0%B0%D0%BD%D0%B0%D1%82" title="Аварский каганат">аварами</a> и <a href="/wiki/%D0%94%D1%80%D0%B5%D0%B2%D0%BD%D0%B8%D0%B5_%D1%81%D0%BB%D0%B0%D0%B2%D1%8F%D0%BD%D0%B5" title="Древние славяне">славянами</a> патриарх <a class="mw-redirect" href="/wiki/%D0%A1%D0%B5%D1%80%D0%B3%D0%B8%D0%B9_I_(%D0%9F%D0%B0%D1%82%D1%80%D0%B8%D0%B0%D1%80%D1%85_%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD%D0%BE%D0%BF%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9)" title="Сергий I (Патриарх Константинопольский)">Сергий</a> обошёл с ней <a href="/wiki/%D0%A1%D1%82%D0%B5%D0%BD%D1%8B_%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD%D0%BE%D0%BF%D0%BE%D0%BB%D1%8F" title="Стены Константинополя">стены Константинополя</a>, и, по сообщению <a href="/wiki/%D0%9F%D0%B0%D1%81%D1%85%D0%B0%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D1%85%D1%80%D0%BE%D0%BD%D0%B8%D0%BA%D0%B0" title="Пасхальная хроника">Пасхальной хроники</a>, враги отступили, заявив, что видели на стене Богородицу. С этого момента возникла традиция ежегодно в Великий пост переносить эту икону из <a class="mw-redirect" href="/wiki/%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D0%B0" title="Влахерна">Влахерн</a> в <a href="/wiki/%D0%91%D0%BE%D0%BB%D1%8C%D1%88%D0%BE%D0%B9_%D0%B4%D0%B2%D0%BE%D1%80%D0%B5%D1%86_(%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD%D0%BE%D0%BF%D0%BE%D0%BB%D1%8C)" title="Большой дворец (Константинополь)">императорский дворец</a>, где она находилась до понедельника <a href="/wiki/%D0%A1%D0%B2%D0%B5%D1%82%D0%BB%D0%B0%D1%8F_%D1%81%D0%B5%D0%B4%D0%BC%D0%B8%D1%86%D0%B0" title="Светлая седмица">Светлой седмицы</a>.
</p><p>В период <a href="/wiki/%D0%98%D0%BA%D0%BE%D0%BD%D0%BE%D0%B1%D0%BE%D1%80%D1%87%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Иконоборчество">иконоборчества</a> икону прятали в <a href="/wiki/%D0%9C%D0%BE%D0%BD%D0%B0%D1%81%D1%82%D1%8B%D1%80%D1%8C_%D0%9F%D0%B0%D0%BD%D1%82%D0%BE%D0%BA%D1%80%D0%B0%D1%82%D0%BE%D1%80%D0%B0" title="Монастырь Пантократора">монастыре Пантократора</a>. Там же её укрывали в период захвата Константинополя <a class="mw-redirect" href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82%D0%BE%D0%BD%D0%BE%D1%81%D1%86%D1%8B" title="Крестоносцы">крестоносцами</a>. Икона была известна русским паломникам, о торжественных процессиях с ней пишет <a href="/wiki/%D0%A1%D1%82%D0%B5%D1%84%D0%B0%D0%BD_%D0%9D%D0%BE%D0%B2%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%B5%D1%86" title="Стефан Новгородец">Стефан Новгородец</a>, посетивший Константинополь в <a href="/wiki/XIV_%D0%B2%D0%B5%D0%BA" title="XIV век">XIV веке</a>:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
Одному человеку поставят икону на плечи стоймя, а он руки распрострёт, словно его распяли, и глаза у него закатятся, так что смотреть страшно, и по площади бросает его туда и сюда, и вертит его в разные стороны, а он даже не понимает, куда его икона носит. Потом другой подхватит её, и с тем бывает так же, а затем и третий, и четвёртый подхватывают, и они поют с дьяконами пение великое, а народ с плачем взывает: <i>«Господи, помилуй!»</i>. Два дьякона держат <a href="/wiki/%D0%A0%D0%B8%D0%BF%D0%B8%D0%B4%D0%B0" title="Рипида">рипиды</a>, а остальные <a href="/wiki/%D0%9A%D0%B8%D0%BE%D1%82" title="Киот">киот</a> перед иконой. Дивное зрелище: семь человек или восемь поставят икону на плечи одному человеку, а он, изволением Божиим, ходит, будто ничем не нагруженный<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>.
</p>
<div class="ts-Конец_цитаты-source">— <cite>«Хождение Стефана Новгородца»</cite></div></blockquote>
<p>О судьбе данной иконы вплоть до турецкого завоевания в <a href="/wiki/1453_%D0%B3%D0%BE%D0%B4" title="1453 год">1453 году</a> достоверной информации нет. <a href="/wiki/%D0%9F%D0%B0%D0%B2%D0%B5%D0%BB_%D0%90%D0%BB%D0%B5%D0%BF%D0%BF%D1%81%D0%BA%D0%B8%D0%B9" title="Павел Алеппский">Павел Алеппский</a> пишет о той иконе, которая была прислана в Москву, следующее:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
Когда мы там были<i> [в Константинополе]</i>, явилась в <a href="/wiki/%D0%9F%D0%BE%D0%B4%D0%B2%D0%BE%D1%80%D1%8C%D0%B5" title="Подворье">подворье</a> Воскресения одна вдова и подарила настоятелю икону Владычицы, известную под именем Влахернской. Это та самая икона, которую некогда патриарх обносил кругом стен города, при чём она обратила в позорное бегство напавших на него врагов. Женщина рассказала, что нашла её в стене своего дома с горящей перед нею лампадой… Наш владыка патриарх употреблял все старания, предлагая много червонцев, чтобы получить её от упомянутого настоятеля, но напрасно. Впоследствии игумен послал её с одним купцом царю московскому, который принял её наилучшим образом… Царь взамен её послал тому <a href="/wiki/%D0%98%D0%B3%D1%83%D0%BC%D0%B5%D0%BD" title="Игумен">игумену</a> 800 динаров, кроме того, что дал человеку, который её доставил<sup class="reference" id="cite_ref-Алеп11_3-0"><a href="#cite_note-Алеп11-3">[3]</a></sup>.
</p>
</blockquote>
<p>Эта история совпадает с преданием периода иконоборчества, когда Влахернскую икону вместе с горящей <a href="/wiki/%D0%9B%D0%B0%D0%BC%D0%BF%D0%B0%D0%B4%D0%B0" title="Лампада">лампадой</a> замуровали в стене монастыря Пантократора, где она находилась около ста лет<sup class="reference" id="cite_ref-Vlax1_1-1"><a href="#cite_note-Vlax1-1">[1]</a></sup>.
</p>
<h3><span id=".D0.9F.D0.B5.D1.80.D0.B5.D0.BD.D0.B5.D1.81.D0.B5.D0.BD.D0.B8.D0.B5_.D0.B2_.D0.9C.D0.BE.D1.81.D0.BA.D0.B2.D1.83"></span><span class="mw-headline" id="Перенесение_в_Москву">Перенесение в Москву</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%B0&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Перенесение в Москву»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%B0&amp;action=edit&amp;section=3" title="Редактировать код раздела «Перенесение в Москву»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Dormition_(Kremlin).JPG"><img class="mw-file-element" data-file-height="2048" data-file-width="1536" decoding="async" height="320" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/9d/Dormition_%28Kremlin%29.JPG/240px-Dormition_%28Kremlin%29.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/9d/Dormition_%28Kremlin%29.JPG/360px-Dormition_%28Kremlin%29.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/9/9d/Dormition_%28Kremlin%29.JPG/480px-Dormition_%28Kremlin%29.JPG 2x" width="240"/></a><figcaption><center>Успенский собор <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%9A%D1%80%D0%B5%D0%BC%D0%BB%D1%8C" title="Московский Кремль">Московского Кремля</a></center></figcaption></figure><p>
В <a href="/wiki/1653_%D0%B3%D0%BE%D0%B4" title="1653 год">1653 году</a> икона, которую считали Влахернской, была прислана в дар русскому царю <a href="/wiki/%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%9C%D0%B8%D1%85%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2%D0%B8%D1%87" title="Алексей Михайлович">Алексею Михайловичу</a> от <a class="mw-redirect" href="/wiki/%D0%9F%D1%80%D0%BE%D1%82%D0%BE%D1%81%D0%B8%D0%BD%D0%B3%D0%B5%D0%BB" title="Протосингел">протосингела</a> иерусалимского престола Гавриила с торговым греком Дмитрием Евстафьевым (Остафьевым), по другим сведениям — Дмитрием Костинари<sup class="reference" id="cite_ref-Vlax2_4-0"><a href="#cite_note-Vlax2-4">[4]</a></sup>. В записях XVII века о царских выходах под 7162 годом (<a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD%D0%BE%D0%BF%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B0%D1%8F_%D1%8D%D1%80%D0%B0" title="Константинопольская эра">константинопольской эры</a>) записано<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>: </p><div class="ts-Цитата"><blockquote><p>Октября в 16 день встречал Государь образ пречистыя Богородицы Одигитрия, что принесен из Грек, из Лахернской церкви.</p></blockquote></div>
<p>На следующий день в честь иконы, помещённой в <a class="mw-redirect" href="/wiki/%D0%A3%D1%81%D0%BF%D0%B5%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D0%BE%D0%B1%D0%BE%D1%80_(%D0%9C%D0%BE%D1%81%D0%BA%D0%B2%D0%B0)" title="Успенский собор (Москва)">Успенский собор Московского кремля</a>, была совершена торжественная служба. Икона, бывшая покровительницей византийских императоров, сразу стала одной из важнейших святынь русского государства<sup class="reference" id="cite_ref-Vlax2_4-1"><a href="#cite_note-Vlax2-4">[4]</a></sup>. Она была включена в церемониальный царский обиход и литургическую практику Успенского собора, её поместили на <a href="/wiki/%D0%96%D0%B5%D1%80%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%B8%D0%BA_(%D0%B2_%D0%BF%D1%80%D0%B0%D0%B2%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%BD%D0%BE%D0%BC_%D1%85%D1%80%D0%B0%D0%BC%D0%B5)" title="Жертвенник (в православном храме)">жертвеннике</a> собора. Для неё был изготовлен <a class="mw-redirect" href="/wiki/%D0%A0%D0%B8%D0%B7%D0%B0_(%D0%BE%D0%BA%D0%BB%D0%B0%D0%B4)" title="Риза (оклад)">оклад</a> из чеканного золота, украшенный большим количеством <a href="/wiki/%D0%94%D1%80%D0%B0%D0%B3%D0%BE%D1%86%D0%B5%D0%BD%D0%BD%D1%8B%D0%B5_%D0%BA%D0%B0%D0%BC%D0%BD%D0%B8" title="Драгоценные камни">драгоценных камней</a> и <a href="/wiki/%D0%96%D0%B5%D0%BC%D1%87%D1%83%D0%B3" title="Жемчуг">жемчуга</a> и золотыми пластинками с текстом, излагавшим легенду об иконе. О драгоценном окладе пишет <a href="/wiki/%D0%9F%D0%B0%D0%B2%D0%B5%D0%BB_%D0%90%D0%BB%D0%B5%D0%BF%D0%BF%D1%81%D0%BA%D0%B8%D0%B9" title="Павел Алеппский">Павел Алеппский</a>: «<i>Кроме лика и рук Владычицы и Господа ничего из неё не видно, всё остальное покрыто золотом</i>»<sup class="reference" id="cite_ref-Алеп11_3-1"><a href="#cite_note-Алеп11-3">[3]</a></sup>.
</p><p>Алексею Михайловичу была направлена свидетельствованная грамота об иконе, написанная константинопольским Патриархом <a class="mw-redirect" href="/wiki/%D0%9F%D0%B0%D0%B8%D1%81%D0%B8%D0%B9_I_(%D0%9F%D0%B0%D1%82%D1%80%D0%B8%D0%B0%D1%80%D1%85_%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD%D0%BE%D0%BF%D0%BE%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9)" title="Паисий I (Патриарх Константинопольский)">Паисием</a>, её оригинальный греческий текст был обнаружен в <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D0%B3%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9_%D0%B0%D1%80%D1%85%D0%B8%D0%B2_%D0%B4%D1%80%D0%B5%D0%B2%D0%BD%D0%B8%D1%85_%D0%B0%D0%BA%D1%82%D0%BE%D0%B2" title="Российский государственный архив древних актов">Российском государственном архиве древних актов</a>. В ней данная икона впервые названа Влахернской<sup class="reference" id="cite_ref-ПЭ_6-0"><a href="#cite_note-ПЭ-6">[6]</a></sup>, а название на сам образ было надписано только при его реставрации в <a href="/wiki/1813_%D0%B3%D0%BE%D0%B4" title="1813 год">1813 году</a> (см. <a href="#Реставрации_иконы">Реставрации иконы</a>).
</p><p>При патриархе <a class="mw-redirect" href="/wiki/%D0%9D%D0%B8%D0%BA%D0%BE%D0%BD_(%D0%9F%D0%B0%D1%82%D1%80%D0%B8%D0%B0%D1%80%D1%85_%D0%9C%D0%BE%D1%81%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9)" title="Никон (Патриарх Московский)">Никоне</a> возникает обычай ежегодно в <a class="mw-redirect" href="/wiki/%D0%90%D0%BA%D0%B0%D1%84%D0%B8%D1%81%D1%82%D0%BD%D0%B0%D1%8F_%D0%A1%D1%83%D0%B1%D0%B1%D0%BE%D1%82%D0%B0" title="Акафистная Суббота">пятую субботу</a> <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B8%D0%B9_%D0%BF%D0%BE%D1%81%D1%82" title="Великий пост">Великого поста</a> ставить икону между <a href="/wiki/%D0%90%D0%BC%D0%B2%D0%BE%D0%BD" title="Амвон">амвоном</a> и <a href="/wiki/%D0%A6%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B5_%D0%B2%D1%80%D0%B0%D1%82%D0%B0" title="Царские врата">царскими вратами</a> и читать перед ней <a href="/wiki/%D0%90%D0%BA%D0%B0%D1%84%D0%B8%D1%81%D1%82" title="Акафист">акафист</a> в память спасения Константинополя в 626 году<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>. Такая традиция сохраняется до второй половины <a href="/wiki/XVIII_%D0%B2%D0%B5%D0%BA" title="XVIII век">XVIII века</a>. В «<i>Книге записи ежедневных служб, совершавшихся в Успенском соборе и об особых службах и крестных ходах 1666—1743 гг.</i>» Влахернская икона уступает по числу упоминаний о её участии в богослужении только <a class="mw-redirect" href="/wiki/%D0%9F%D0%B5%D1%82%D1%80%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%B0" title="Петровская икона">Петровской иконе</a> (предание приписывает её авторство митрополиту <a href="/wiki/%D0%9F%D1%91%D1%82%D1%80_(%D0%BC%D0%B8%D1%82%D1%80%D0%BE%D0%BF%D0%BE%D0%BB%D0%B8%D1%82_%D0%9A%D0%B8%D0%B5%D0%B2%D1%81%D0%BA%D0%B8%D0%B9)" title="Пётр (митрополит Киевский)">Петру</a>)<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>:
</p>
<figure typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Wjatscheslaw_Grigorjewitsch_Schwarz_002.jpg"><img class="mw-file-element" data-file-height="1278" data-file-width="2648" decoding="async" height="145" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/c5/Wjatscheslaw_Grigorjewitsch_Schwarz_002.jpg/300px-Wjatscheslaw_Grigorjewitsch_Schwarz_002.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/c/c5/Wjatscheslaw_Grigorjewitsch_Schwarz_002.jpg/450px-Wjatscheslaw_Grigorjewitsch_Schwarz_002.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/c/c5/Wjatscheslaw_Grigorjewitsch_Schwarz_002.jpg/600px-Wjatscheslaw_Grigorjewitsch_Schwarz_002.jpg 2x" width="300"/></a><figcaption><center>Шествие на осляти времён Алексея Михайловича <br/>(<a href="/wiki/%D0%A8%D0%B2%D0%B0%D1%80%D1%86,_%D0%92%D1%8F%D1%87%D0%B5%D1%81%D0%BB%D0%B0%D0%B2_%D0%93%D1%80%D0%B8%D0%B3%D0%BE%D1%80%D1%8C%D0%B5%D0%B2%D0%B8%D1%87" title="Шварц, Вячеслав Григорьевич">В. Г. Шварц</a>, 1865 год)</center></figcaption></figure>
<ul><li>икона участвовала в <a href="/wiki/%D0%9A%D1%80%D0%B5%D1%81%D1%82%D0%BD%D1%8B%D0%B9_%D1%85%D0%BE%D0%B4" title="Крестный ход">крестном ходе</a> на <a href="/wiki/%D0%9F%D0%BE%D0%BA%D1%80%D0%BE%D0%B2_%D0%9F%D1%80%D0%B5%D1%81%D0%B2%D1%8F%D1%82%D0%BE%D0%B9_%D0%91%D0%BE%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%B8%D1%86%D1%8B" title="Покров Пресвятой Богородицы">Покров</a> к <a href="/wiki/%D0%A5%D1%80%D0%B0%D0%BC_%D0%92%D0%B0%D1%81%D0%B8%D0%BB%D0%B8%D1%8F_%D0%91%D0%BB%D0%B0%D0%B6%D0%B5%D0%BD%D0%BD%D0%BE%D0%B3%D0%BE" title="Храм Василия Блаженного">Покровскому собору на Рву</a>;</li>
<li>на <a href="/wiki/%D0%91%D0%BE%D0%B3%D0%BE%D1%8F%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5" title="Богоявление">Богоявление</a> во главе шествия на <a href="/wiki/%D0%A1%D0%B2%D1%8F%D1%82%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B4%D0%B0#Освящение_воды" title="Святая вода">иордань</a> несли Влахернскую икону, а во время <a href="/wiki/%D0%A1%D0%B2%D1%8F%D1%82%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B4%D0%B0" title="Святая вода">водосвятия</a> возлагали на <a href="/wiki/%D0%90%D0%BD%D0%B0%D0%BB%D0%BE%D0%B9" title="Аналой">аналое</a> вместе с иконой Московских святителей;</li>
<li>на праздник <a href="/wiki/%D0%A1%D0%BE%D0%B1%D0%BE%D1%80_%D1%82%D1%80%D1%91%D1%85_%D1%81%D0%B2%D1%8F%D1%82%D0%B8%D1%82%D0%B5%D0%BB%D0%B5%D0%B9" title="Собор трёх святителей">Собора трёх святителей</a> икону помещали рядом с <a href="/wiki/%D0%9C%D0%BE%D1%89%D0%B8" title="Мощи">мощами</a> <a href="/wiki/%D0%98%D0%BE%D0%B0%D0%BD%D0%BD_%D0%97%D0%BB%D0%B0%D1%82%D0%BE%D1%83%D1%81%D1%82" title="Иоанн Златоуст">Иоанна Златоуста</a>, <a href="/wiki/%D0%92%D0%B0%D1%81%D0%B8%D0%BB%D0%B8%D0%B9_%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B8%D0%B9" title="Василий Великий">Василия Великого</a> и <a href="/wiki/%D0%93%D1%80%D0%B8%D0%B3%D0%BE%D1%80%D0%B8%D0%B9_%D0%91%D0%BE%D0%B3%D0%BE%D1%81%D0%BB%D0%BE%D0%B2" title="Григорий Богослов">Григория Богослова</a>;</li>
<li>икону выносили на водоосвящении 1 августа в день <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%B8%D1%81%D1%85%D0%BE%D0%B6%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5_%D1%87%D0%B5%D1%81%D1%82%D0%BD%D1%8B%D1%85_%D0%B4%D1%80%D0%B5%D0%B2_%D0%96%D0%B8%D0%B2%D0%BE%D1%82%D0%B2%D0%BE%D1%80%D1%8F%D1%89%D0%B5%D0%B3%D0%BE_%D0%9A%D1%80%D0%B5%D1%81%D1%82%D0%B0" title="Происхождение честных древ Животворящего Креста">Происхождения честных древ Животворящего Креста</a>;</li>
<li>в день празднования новолетия (1 сентября) икона за богослужением ставилась на аналое вместе с <a class="mw-redirect" href="/wiki/%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%B0" title="Владимирская икона">Владимирской иконой</a>;</li>
<li>икона участвовала в «<a href="/wiki/%D0%A8%D0%B5%D1%81%D1%82%D0%B2%D0%B8%D0%B5_%D0%BD%D0%B0_%D0%BE%D1%81%D0%BB%D1%8F%D1%82%D0%B8" title="Шествие на осляти">шествии на осляти</a>» — крестном ходе в <a href="/wiki/%D0%9F%D1%80%D0%B0%D0%B7%D0%B4%D0%BD%D0%B8%D0%BA" title="Праздник">праздник</a> <a href="/wiki/%D0%92%D1%85%D0%BE%D0%B4_%D0%93%D0%BE%D1%81%D0%BF%D0%BE%D0%B4%D0%B5%D0%BD%D1%8C_%D0%B2_%D0%98%D0%B5%D1%80%D1%83%D1%81%D0%B0%D0%BB%D0%B8%D0%BC" title="Вход Господень в Иерусалим">Входа Господня в Иерусалим</a>, устраивавшемся в Москве в XVI—XVII веках.</li></ul>
<p>Влахернская икона стала почитаться как заступница против вражеских нашествий: сразу после её перенесения в Москву Алексей Михайлович взял её с собой на войну, а по возвращении, по свидетельству Павла Алепского, «<i>вёз перед собою</i>». Во время смоленского похода её ставили напротив царя в санях.
</p><p>К началу <a href="/wiki/XIX_%D0%B2%D0%B5%D0%BA" title="XIX век">XIX века</a> популярность иконы ослабевает, в <a href="/wiki/1812_%D0%B3%D0%BE%D0%B4" title="1812 год">1812 году</a>, когда <a href="/wiki/%D0%9D%D0%B0%D0%BF%D0%BE%D0%BB%D0%B5%D0%BE%D0%BD_I" title="Наполеон I">наполеоновские</a> войска вошли в Москву, то Влахернская икона не была вывезена из Успенского собора и после изгнания войск была обнаружена повреждённой. Образ врезали в новую кипарисную доску и украсили серебряной ризой. В XIX — начале XX веков икона находилась в Петропавловском приделе Успенского собора, где её поместили над <a href="/wiki/%D0%A0%D0%B0%D0%BA%D0%B0" title="Рака">ракой</a> святителя <a href="/wiki/%D0%A4%D0%B5%D0%BE%D0%B3%D0%BD%D0%BE%D1%81%D1%82_(%D0%BC%D0%B8%D1%82%D1%80%D0%BE%D0%BF%D0%BE%D0%BB%D0%B8%D1%82_%D0%9A%D0%B8%D0%B5%D0%B2%D1%81%D0%BA%D0%B8%D0%B9)" title="Феогност (митрополит Киевский)">Феогноста</a>, однако продолжали выносить на крестных ходах<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>. В <a href="/wiki/1918_%D0%B3%D0%BE%D0%B4" title="1918 год">1918 году</a>, после <a href="/wiki/%D0%9E%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8C%D1%81%D0%BA%D0%B0%D1%8F_%D1%80%D0%B5%D0%B2%D0%BE%D0%BB%D1%8E%D1%86%D0%B8%D1%8F" title="Октябрьская революция">Октябрьской революции</a>, её переместили в Крестовоздвиженскую церковь на Воздвиженке. В <a href="/wiki/1931_%D0%B3%D0%BE%D0%B4" title="1931 год">1931 году</a> Влахернская икона вошла в собрание музеев Московского кремля.
</p>
<h3><span id=".D0.A0.D0.B5.D1.81.D1.82.D0.B0.D0.B2.D1.80.D0.B0.D1.86.D0.B8.D0.B8_.D0.B8.D0.BA.D0.BE.D0.BD.D1.8B"></span><span class="mw-headline" id="Реставрации_иконы">Реставрации иконы</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%B0&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Реставрации иконы»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%B0&amp;action=edit&amp;section=4" title="Редактировать код раздела «Реставрации иконы»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>В 1674 году икона была отреставрирована по указанию Алексея Михайловича иконописцами <a href="/wiki/%D0%9E%D1%80%D1%83%D0%B6%D0%B5%D0%B9%D0%BD%D0%B0%D1%8F_%D0%BF%D0%B0%D0%BB%D0%B0%D1%82%D0%B0" title="Оружейная палата">Оружейной палаты</a> <a href="/wiki/%D0%A3%D1%88%D0%B0%D0%BA%D0%BE%D0%B2,_%D0%A1%D0%B8%D0%BC%D0%BE%D0%BD_%D0%A4%D1%91%D0%B4%D0%BE%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Ушаков, Симон Фёдорович">Симоном Ушаковым</a> и Никитой Павловцем. Было сделано укрепление ветхой доски подведением новой, кипарисной.
</p><p>В 1813 году при реставрации на ней были сделаны новые греческие надписи: «Госпожа обители Влахернской» и «Многоценное сокровище», а также надпись на русском языке: «0AWГІ (1813) года августа А (1) дня обновлен сей образ».
</p><p>В 1983 году реставрацию Влахернской иконы произвёл в Реставрационном центре имени академика Грабаря реставратор Д. А. Дунаев. При данной реставрации на иконе было раскрыто оригинальное греческое надписание — <span lang="el"><span style="font-family: palatino linotype, new athena unicode, athena, gentium, code2000, serif; font-size: 105%;">Η ΘΕΟΣΚΕΠΑ[Σ]ΤΟΣ</span></span> («Богохранимая» или «Богом Хранимая»). Это редкий эпитет Богородицы, известный лишь по двум примерам (печать <a class="new" href="/w/index.php?title=%D0%A2%D1%80%D0%B0%D0%BF%D0%B5%D0%B7%D1%83%D0%BD%D0%B4%D1%81%D0%BA%D0%B8%D0%B9_%D0%BC%D0%BE%D0%BD%D0%B0%D1%81%D1%82%D1%8B%D1%80%D1%8C&amp;action=edit&amp;redlink=1" title="Трапезундский монастырь (страница отсутствует)">Трапезундского монастыря</a> с изображением <a href="/wiki/%D0%9E%D1%80%D0%B0%D0%BD%D1%82%D0%B0" title="Оранта">Оранты</a> (XIII век) и название чтимой <a href="/wiki/%D0%9A%D0%B8%D0%BF%D1%80_(%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2)" title="Кипр (остров)">кипрской</a> иконы из церкви Феоскепасту (XIV век))<sup class="reference" id="cite_ref-ПЭ_6-1"><a href="#cite_note-ПЭ-6">[6]</a></sup>.
</p>
<h2><span id=".D0.A5.D0.B0.D1.80.D0.B0.D0.BA.D1.82.D0.B5.D1.80.D0.B8.D1.81.D1.82.D0.B8.D0.BA.D0.B0_.D0.BE.D0.B1.D1.80.D0.B0.D0.B7.D0.B0"></span><span class="mw-headline" id="Характеристика_образа">Характеристика образа</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%B0&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Характеристика образа»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%B0&amp;action=edit&amp;section=5" title="Редактировать код раздела «Характеристика образа»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Размеры иконы 46 × 37,5 × 4 см. Хранится в <a class="mw-redirect" href="/wiki/%D0%A3%D1%81%D0%BF%D0%B5%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D0%BE%D0%B1%D0%BE%D1%80_%D0%9C%D0%BE%D1%81%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%BE%D0%B3%D0%BE_%D0%9A%D1%80%D0%B5%D0%BC%D0%BB%D1%8F" title="Успенский собор Московского Кремля">Успенском соборе Московского Кремля</a>. Влахернская икона относится к <a href="/wiki/%D0%9F%D1%80%D0%B0%D0%B2%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%BD%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%BE%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D1%8F_%D0%91%D0%BE%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%B8%D1%86%D1%8B" title="Православная иконография Богородицы">иконописному типу</a> <a href="/wiki/%D0%9E%D0%B4%D0%B8%D0%B3%D0%B8%D1%82%D1%80%D0%B8%D1%8F" title="Одигитрия">Одигитрия</a> и выполнена в технике <a href="/wiki/%D0%92%D0%BE%D1%81%D0%BA%D0%BE%D0%BC%D0%B0%D1%81%D1%82%D0%B8%D0%BA%D0%B0" title="Воскомастика">воскомастики</a>, то есть является <a href="/wiki/%D0%A0%D0%B5%D0%BB%D1%8C%D0%B5%D1%84_(%D1%81%D0%BA%D1%83%D0%BB%D1%8C%D0%BF%D1%82%D1%83%D1%80%D0%B0)" title="Рельеф (скульптура)">рельефным</a> изображением. <a href="/wiki/%D0%9F%D0%B0%D0%B2%D0%B5%D0%BB_%D0%90%D0%BB%D0%B5%D0%BF%D0%BF%D1%81%D0%BA%D0%B8%D0%B9" title="Павел Алеппский">Павел Алеппский</a>, посетивший Москву в <a href="/wiki/1655_%D0%B3%D0%BE%D0%B4" title="1655 год">1655 году</a>, писал о ней: «не нарисована красками, но как будто телесная или изображённая мастикой, ибо части её тела сильно выступают с поверхности доски, к большому удивлению смотрящего, внушая благоговейный страх… Она как будто воплощённая». Сама воскомастика сделана «смешением от святых <a href="/wiki/%D0%9C%D0%BE%D1%89%D0%B8" title="Мощи">мощей</a> и от иного многого благоуханного состава», то есть икона представляет собой своеобразный <a href="/wiki/%D0%9C%D0%BE%D1%89%D0%B5%D0%B2%D0%B8%D0%BA" title="Мощевик">мощевик</a><sup class="reference" id="cite_ref-Vlax1_1-2"><a href="#cite_note-Vlax1-1">[1]</a></sup>. Источники XIX века сообщают, что среди мощей, использованных для написания иконы, был пепел <a href="/wiki/%D0%9D%D0%B8%D0%BA%D0%BE%D0%BC%D0%B8%D0%B4%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B5_%D0%BC%D1%83%D1%87%D0%B5%D0%BD%D0%B8%D0%BA%D0%B8" title="Никомидийские мученики">мучеников Никомидийских</a> (Виктора, Викторина, Клавдиана, Вассы — погибли в IV веке), <a class="mw-redirect" href="/wiki/%D0%9F%D1%80%D0%B5%D0%BF%D0%BE%D0%B4%D0%BE%D0%B1%D0%BD%D1%8B%D0%B5_%D0%BE%D1%82%D1%86%D1%8B,_%D0%B2_%D0%A1%D0%B8%D0%BD%D0%B0%D0%B5_%D0%B8_%D0%A0%D0%B0%D0%B8%D1%84%D0%B5_%D0%B8%D0%B7%D0%B1%D0%B8%D0%B5%D0%BD%D0%BD%D1%8B%D0%B5" title="Преподобные отцы, в Синае и Раифе избиенные">Синайских и Раифских</a><sup class="reference" id="cite_ref-ПЭ_6-2"><a href="#cite_note-ПЭ-6">[6]</a></sup>.
</p><p>Иконография Влахернской иконы наиболее подробно исследована <a href="/wiki/%D0%9A%D0%BE%D0%BD%D0%B4%D0%B0%D0%BA%D0%BE%D0%B2,_%D0%9D%D0%B8%D0%BA%D0%BE%D0%B4%D0%B8%D0%BC_%D0%9F%D0%B0%D0%B2%D0%BB%D0%BE%D0%B2%D0%B8%D1%87" title="Кондаков, Никодим Павлович">Никодимом Кондаковым</a> в сочинении «Иконография Богоматери»<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>. Он относит Влахернскую икону к числу трёх икон Одигитрии, которые наиболее точно передают данный иконописный тип (две другие — <a class="new" href="/w/index.php?title=%D0%A6%D0%B8%D0%BB%D0%BA%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%B0_%D0%91%D0%BE%D0%B6%D0%B8%D0%B5%D0%B9_%D0%9C%D0%B0%D1%82%D0%B5%D1%80%D0%B8&amp;action=edit&amp;redlink=1" title="Цилканская икона Божией Матери (страница отсутствует)">Цилканская икона Божией Матери</a> IX—XIII веков и <a href="/wiki/%D0%A2%D0%B0%D0%BB%D1%8C%D0%BA%D0%BE%D1%85%D0%BB%D0%BE%D1%80%D0%B8%D1%82" title="Талькохлорит">стеатитовый</a> образ из <a href="/wiki/%D0%92%D0%B0%D1%82%D0%BE%D0%BF%D0%B5%D0%B4" title="Ватопед">Ватопедского монастыря</a> XII—XIV веков). Влахернетисса относится к классическому типу Одигитрии, сложившемуся в константинопольском искусстве в XIII—XIV века. Особенностями иконографии являются: массивность фигуры Богородицы и её почти погрудный обрез, заходящие на поля крупные <a href="/wiki/%D0%9D%D0%B8%D0%BC%D0%B1" title="Нимб">нимбы</a>, высоко поднятые ноги Богомладенца, которые касаются полей иконы.
</p>
<h3><span id=".D0.94.D0.B0.D1.82.D0.B8.D1.80.D0.BE.D0.B2.D0.BA.D0.B0_.D0.B8.D0.BA.D0.BE.D0.BD.D1.8B"></span><span class="mw-headline" id="Датировка_иконы">Датировка иконы</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%B0&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Датировка иконы»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%B0&amp;action=edit&amp;section=6" title="Редактировать код раздела «Датировка иконы»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="mw-redirect" href="/wiki/%D0%9B%D0%B5%D0%B2%D0%BE%D0%BD_%D0%9D%D0%B5%D1%80%D1%81%D0%B5%D1%81%D1%8F%D0%BD" title="Левон Нерсесян">Левон Нерсесян</a> сообщает, что в средневековых источниках<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> Влахернская икона выступает как «<i>некая <a href="/wiki/%D0%9C%D1%80%D0%B0%D0%BC%D0%BE%D1%80" title="Мрамор">мраморная</a> плита с изображением Богоматери в рост, которая находилась в часовне при святом источнике, и вода святая изливалась из дырочек на пол, к праху Богоматери</i>»<sup class="reference" id="cite_ref-12"><a href="#cite_note-12">[12]</a></sup>. Само наименование привезённой в Москву иконы «Влахернская» появляется только в середине XVII века в грамоте патриарха Паисия. Этот эпитет в Византии обычно применяли к иконописным изображениям <a href="/wiki/%D0%9E%D1%80%D0%B0%D0%BD%D1%82%D0%B0" title="Оранта">Оранты</a>, <a href="/wiki/%D0%9C%D0%BE%D0%B7%D0%B0%D0%B8%D0%BA%D0%B0" title="Мозаика">мозаика</a> которой, по мнению Н. П. Кондакова, украшала <a class="mw-redirect" href="/wiki/%D0%90%D0%BF%D1%81%D0%B8%D0%B4%D0%B0_(%D0%B0%D1%80%D1%85%D0%B8%D1%82%D0%B5%D0%BA%D1%82%D1%83%D1%80%D0%B0)" title="Апсида (архитектура)">апсиду</a> <a href="/wiki/%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D1%86%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C" title="Влахернская церковь">Влахернского храма</a><sup class="reference" id="cite_ref-ПЭ_6-3"><a href="#cite_note-ПЭ-6">[6]</a></sup>. По причинам этих противоречий вопрос о времени создания иконы, присланной Алексею Михайловичу, является спорным.
</p><p>Иконографические и эпиграфические аналоги иконы датируются XIII—XIV веками<sup class="reference" id="cite_ref-ПЭ_6-4"><a href="#cite_note-ПЭ-6">[6]</a></sup>. Серьёзная реставрация Влахернской иконы в 1674 году свидетельствует о древнем происхождении образа. С другой стороны, по мнению исследователей:

</p>
<blockquote class="ts-Начало_цитаты-quote">
<p>
В легенде об иконе конкретный рассказ о ней начинается лишь после событий 1453 года. Можно предположить, что именно тогда, во второй половине XV века, возникла потребность воссоздать какую-то святыню, связанную с Влахернским храмом. Не исключено также, что на старой основе был реконструирован сильно повреждённый в перипетиях турецкого завоевания древний памятник<sup class="reference" id="cite_ref-Vlax2_4-2"><a href="#cite_note-Vlax2-4">[4]</a></sup>.
</p>
</blockquote>
<h2><span id=".D0.A1.D0.BF.D0.B8.D1.81.D0.BA.D0.B8"></span><span class="mw-headline" id="Списки">Списки</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%B0&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Списки»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%B0&amp;action=edit&amp;section=7" title="Редактировать код раздела «Списки»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<figure class="mw-default-size" typeof="mw:File/Thumb"><a class="mw-file-description" href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Vlahernskaya.jpg"><img class="mw-file-element" data-file-height="450" data-file-width="351" decoding="async" height="282" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/31/Vlahernskaya.jpg/220px-Vlahernskaya.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/31/Vlahernskaya.jpg/330px-Vlahernskaya.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/3/31/Vlahernskaya.jpg 2x" width="220"/></a><figcaption><center>Строгановский список иконы</center></figcaption></figure>
<p>Известно три списка иконы, сделанных в различное время:
</p>
<ol><li>принадлежавший роду <a href="/wiki/%D0%A1%D1%82%D1%80%D0%BE%D0%B3%D0%B0%D0%BD%D0%BE%D0%B2%D1%8B" title="Строгановы">Строгановых</a>, а с середины XVIII века — роду <a href="/wiki/%D0%93%D0%BE%D0%BB%D0%B8%D1%86%D1%8B%D0%BD%D1%8B" title="Голицыны">Голицыных</a>. Икона хранилась в имении селе Влахернском (<a href="/wiki/%D0%9A%D1%83%D0%B7%D1%8C%D0%BC%D0%B8%D0%BD%D0%BA%D0%B8_(%D1%83%D1%81%D0%B0%D0%B4%D1%8C%D0%B1%D0%B0)" title="Кузьминки (усадьба)">Кузьминках</a>) под Москвой, где была построена церковь, освящённая в честь Влахернской иконы<sup class="reference" id="cite_ref-13"><a href="#cite_note-13">[13]</a></sup>. В настоящее время этот список хранится в <a href="/wiki/%D0%A2%D1%80%D0%B5%D1%82%D1%8C%D1%8F%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%B3%D0%B0%D0%BB%D0%B5%D1%80%D0%B5%D1%8F" title="Третьяковская галерея">Третьяковской галерее</a>. Первое упоминание об иконе относится к <a href="/wiki/1716_%D0%B3%D0%BE%D0%B4" title="1716 год">1716 году</a>, но по датировке Г. В. Сидоренко образ относится к <a href="/wiki/VII_%D0%B2%D0%B5%D0%BA" title="VII век">VII веку</a><sup class="reference" id="cite_ref-ПЭ_6-5"><a href="#cite_note-ПЭ-6">[6]</a></sup>. Древнее происхождение иконы связано с семейным преданием Строгановых и Голицыных о том, что царю Алексею Михайловичу из Константинополя привезли не одну, а две иконы. О возможности изготовления списка иконы в технике, аналогичной первообразу, свидетельствует письменное указание о составе мастики и отливке — «<i>О составлении иконы указ, како образ устроен Влахернской Пресвятой Богородицы</i>» из сборника XVII века<sup class="reference" id="cite_ref-ПЭ_6-6"><a href="#cite_note-ПЭ-6">[6]</a></sup>;</li>
<li>из храма Сергия Радонежского в московском <a href="/wiki/%D0%92%D1%8B%D1%81%D0%BE%D0%BA%D0%BE-%D0%9F%D0%B5%D1%82%D1%80%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%BC%D0%BE%D0%BD%D0%B0%D1%81%D1%82%D1%8B%D1%80%D1%8C" title="Высоко-Петровский монастырь">Высоко-Петровском монастыре</a> (создан в <a href="/wiki/1701_%D0%B3%D0%BE%D0%B4" title="1701 год">1701 году</a>). После революции список хранился в запасниках Исторического музея, потом был передан в <a class="mw-redirect" href="/wiki/%D0%A6%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C_%D0%A0%D0%BE%D0%B6%D0%B4%D0%B5%D1%81%D1%82%D0%B2%D0%B0_%D0%9F%D1%80%D0%B5%D1%81%D0%B2%D1%8F%D1%82%D0%BE%D0%B9_%D0%91%D0%BE%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%B8%D1%86%D1%8B_%D0%B2_%D0%A1%D1%82%D0%B0%D1%80%D0%BE%D0%BC_%D0%A1%D0%B8%D0%BC%D0%BE%D0%BD%D0%BE%D0%B2%D0%B5" title="Церковь Рождества Пресвятой Богородицы в Старом Симонове">храм Рождества Пресвятой Богородицы в Старом Симонове</a>. 2 марта 2017 г. икона была возвращена в Высоко-Петровский монастырь и находится в Сергиевском храме <sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup>;</li>
<li>из <a href="/wiki/%D0%A1%D0%BF%D0%B0%D1%81%D0%BE-%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BC%D0%BE%D0%BD%D0%B0%D1%81%D1%82%D1%8B%D1%80%D1%8C" title="Спасо-Влахернский монастырь">Спасо-Влахернского монастыря</a> под г. <a class="mw-redirect" href="/wiki/%D0%94%D0%BC%D0%B8%D1%82%D1%80%D0%BE%D0%B2_(%D0%B3%D0%BE%D1%80%D0%BE%D0%B4)" title="Дмитров (город)">Дмитровом</a> (одна из святынь рода <a href="/wiki/%D0%93%D0%BE%D0%BB%D0%BE%D0%B2%D0%B8%D0%BD%D1%8B" title="Головины">Головиных</a>, создана в <a href="/wiki/1705_%D0%B3%D0%BE%D0%B4" title="1705 год">1705 году</a>, в настоящее время находится в <a class="mw-redirect" href="/wiki/%D0%A6%D0%B5%D0%BD%D1%82%D1%80%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BC%D1%83%D0%B7%D0%B5%D0%B9_%D0%B4%D1%80%D0%B5%D0%B2%D0%BD%D0%B5%D1%80%D1%83%D1%81%D1%81%D0%BA%D0%BE%D0%B9_%D0%BA%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D1%8B_%D0%B8_%D0%B8%D1%81%D0%BA%D1%83%D1%81%D1%81%D1%82%D0%B2%D0%B0_%D0%B8%D0%BC%D0%B5%D0%BD%D0%B8_%D0%90%D0%BD%D0%B4%D1%80%D0%B5%D1%8F_%D0%A0%D1%83%D0%B1%D0%BB%D1%91%D0%B2%D0%B0" title="Центральный музей древнерусской культуры и искусства имени Андрея Рублёва">музее Андрея Рублёва</a>)<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup>.</li></ol>
<p>Второй и третий списки являются точными копиям кремлёвской иконы, имели аналогичные драгоценные оклады с надписью о спасении заступничеством иконы Царьграда от врагов. В подражание оригиналу эти два списка изготовлены как иконы-<a href="/wiki/%D0%9C%D0%BE%D1%89%D0%B5%D0%B2%D0%B8%D0%BA" title="Мощевик">мощевики</a> — венцы «<i>исполнены множеством частиц святых мощей разных угодников Божиих</i>». В описаниях эти иконы называются воскомастичными, но на самом деле это рельефные деревянные изображения, покрытые <a href="/wiki/%D0%9B%D0%B5%D0%B2%D0%BA%D0%B0%D1%81" title="Левкас">левкасом</a> и расписанные <a href="/wiki/%D0%A2%D0%B5%D0%BC%D0%BF%D0%B5%D1%80%D0%B0" title="Темпера">темперой</a>, то есть они отличаются от византийского первообраза.
</p>
<h2><span id=".D0.A5.D1.80.D0.B0.D0.BC.D1.8B_.D0.B2_.D1.87.D0.B5.D1.81.D1.82.D1.8C_.D0.92.D0.BB.D0.B0.D1.85.D0.B5.D1.80.D0.BD.D1.81.D0.BA.D0.BE.D0.B9_.D0.B8.D0.BA.D0.BE.D0.BD.D1.8B"></span><span class="mw-headline" id="Храмы_в_честь_Влахернской_иконы">Храмы в честь Влахернской иконы</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%B0&amp;veaction=edit&amp;section=8" title="Редактировать раздел «Храмы в честь Влахернской иконы»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BA%D0%BE%D0%BD%D0%B0&amp;action=edit&amp;section=8" title="Редактировать код раздела «Храмы в честь Влахернской иконы»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>В конце X века <a href="/wiki/%D0%90%D0%B1%D1%85%D0%B0%D0%B7%D1%81%D0%BA%D0%BE%D0%B5_%D1%86%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%BE" title="Абхазское царство">абхазским царём</a> Багратом II (позже ставшим <a class="mw-redirect" href="/wiki/%D0%A6%D0%B0%D1%80%D0%B8_%D0%93%D1%80%D1%83%D0%B7%D0%B8%D0%B8" title="Цари Грузии">царём Грузии</a> под именем <a href="/wiki/%D0%91%D0%B0%D0%B3%D1%80%D0%B0%D1%82_III" title="Баграт III">Баграт III</a>) в честь Влахернской иконы Божией Матери был построен <a href="/wiki/%D0%91%D0%B5%D0%B4%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D0%BE%D0%B1%D0%BE%D1%80" title="Бедийский собор">Бедийский собор</a>, в котором сам царь позже был похоронен<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup>.
</p><p><a class="mw-redirect" href="/wiki/%D0%A6%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C_%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%BE%D0%B9_%D0%91%D0%BE%D0%B3%D0%BE%D0%BC%D0%B0%D1%82%D0%B5%D1%80%D0%B8_%D0%B2_%D0%9A%D1%83%D0%B7%D1%8C%D0%BC%D0%B8%D0%BD%D0%BA%D0%B0%D1%85" title="Церковь Влахернской Богоматери в Кузьминках">Храм XVIII века Влахернской иконы Божией Матери в Кузьминках</a> (Москва) — вотчинный храм в усадьбе Кузьминки, давший ей второе (официальное) название — Влахернское.
</p><p>В 1825—1830 годах в <a href="/wiki/%D0%93%D1%80%D1%83%D0%B7%D0%B8%D1%8F" title="Грузия">грузинском</a> городе <a href="/wiki/%D0%97%D1%83%D0%B3%D0%B4%D0%B8%D0%B4%D0%B8" title="Зугдиди">Зугдиди</a> была построена <a href="/wiki/%D0%A1%D0%BE%D0%B1%D0%BE%D1%80_%D0%92%D0%BB%D0%B0%D1%85%D0%B5%D1%80%D0%BD%D1%81%D0%BA%D0%BE%D0%B9_%D0%B8%D0%BA%D0%BE%D0%BD%D1%8B_%D0%91%D0%BE%D0%B6%D0%B8%D0%B5%D0%B9_%D0%9C%D0%B0%D1%82%D0%B5%D1%80%D0%B8" title="Собор Влахернской иконы Божией Матери">Влахернская церковь</a>, в которую поместили чтимый в Грузии список иконы. Средства на строительство выделил император <a href="/wiki/%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_I" title="Александр I">Александр I</a>, которому этот список был прислан в дар мегрельской княгиней <a href="/wiki/%D0%94%D0%B0%D0%B4%D0%B8%D0%B0%D0%BD%D0%B8,_%D0%9D%D0%B8%D0%BD%D0%BE_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%BD%D0%B0" title="Дадиани, Нино Георгиевна">Нино</a>.
</p>
