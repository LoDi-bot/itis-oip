<div class="vector-body" id="bodyContent">
<div class="noprint" id="siteSub">Материал из Википедии — свободной энциклопедии</div>

<p><b><a href="/wiki/%D0%9E%D0%BF%D1%82%D0%B8%D0%BA%D0%B0" title="Оптика">Опти́ческий</a> <a href="/wiki/%D0%9F%D0%B8%D0%BD%D1%86%D0%B5%D1%82" title="Пинцет">пинце́т</a></b> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">optical tweezers</span>), иногда «лазерный пинцет» или «оптическая <a href="/wiki/%D0%9B%D0%BE%D0%B2%D1%83%D1%88%D0%BA%D0%B0" title="Ловушка">ловушка</a>» — оптический инструмент, который позволяет <a href="/wiki/%D0%9C%D0%B0%D0%BD%D0%B8%D0%BF%D1%83%D0%BB%D1%8F%D1%86%D0%B8%D1%8F_%D0%B0%D1%82%D0%BE%D0%BC%D0%B0%D0%BC%D0%B8" title="Манипуляция атомами">манипулировать</a> <a class="new" href="/w/index.php?title=%D0%9C%D0%B8%D0%BA%D1%80%D0%BE%D1%81%D0%BA%D0%BE%D0%BF%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BE%D0%B1%D1%8A%D0%B5%D0%BA%D1%82&amp;action=edit&amp;redlink=1" title="Микроскопический объект (страница отсутствует)">микроскопическими объектами</a> с помощью <a href="/wiki/%D0%9B%D0%B0%D0%B7%D0%B5%D1%80" title="Лазер">лазерного</a> света (обычно испускаемого <a href="/wiki/%D0%9B%D0%B0%D0%B7%D0%B5%D1%80%D0%BD%D1%8B%D0%B9_%D0%B4%D0%B8%D0%BE%D0%B4" title="Лазерный диод">лазерным диодом</a>). Он позволяет прикладывать к <a class="mw-redirect" href="/wiki/%D0%94%D0%B8%D1%8D%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%B8%D0%BA%D0%B8" title="Диэлектрики">диэлектрическим</a> объектам силы от <a href="/wiki/%D0%9D%D1%8C%D1%8E%D1%82%D0%BE%D0%BD_(%D0%B5%D0%B4%D0%B8%D0%BD%D0%B8%D1%86%D0%B0_%D0%B8%D0%B7%D0%BC%D0%B5%D1%80%D0%B5%D0%BD%D0%B8%D1%8F)" title="Ньютон (единица измерения)">фемтоньютонов</a> до <a href="/wiki/%D0%9D%D1%8C%D1%8E%D1%82%D0%BE%D0%BD_(%D0%B5%D0%B4%D0%B8%D0%BD%D0%B8%D1%86%D0%B0_%D0%B8%D0%B7%D0%BC%D0%B5%D1%80%D0%B5%D0%BD%D0%B8%D1%8F)" title="Ньютон (единица измерения)">наноньютонов</a> и измерять <a href="/wiki/%D0%A0%D0%B0%D1%81%D1%81%D1%82%D0%BE%D1%8F%D0%BD%D0%B8%D0%B5" title="Расстояние">расстояния</a> от нескольких <a href="/wiki/%D0%9D%D0%B0%D0%BD%D0%BE%D0%BC%D0%B5%D1%82%D1%80" title="Нанометр">нанометров</a> до <a class="mw-redirect mw-disambig" href="/wiki/%D0%9C%D0%B8%D0%BA%D1%80%D0%BE%D0%BD" title="Микрон">микронов</a>. В последние годы оптические пинцеты начали использовать в <a href="/wiki/%D0%91%D0%B8%D0%BE%D1%84%D0%B8%D0%B7%D0%B8%D0%BA%D0%B0" title="Биофизика">биофизике</a> для изучения структуры и принципа работы <a class="mw-redirect" href="/wiki/%D0%91%D0%B5%D0%BB%D0%BE%D0%BA" title="Белок">белков</a><sup class="reference" id="cite_ref-NKJ201812_1-0"><a href="#cite_note-NKJ201812-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>.
</p><p>В 2018 году нобелевская премия по физике «за изобретение оптического пинцета и его применение в биологических системах» была присуждена создателю оптического пинцета <a href="/wiki/%D0%AD%D1%88%D0%BA%D0%B8%D0%BD,_%D0%90%D1%80%D1%82%D1%83%D1%80" title="Эшкин, Артур">Артуру Эшкину</a><sup class="reference" id="cite_ref-Physics2018_3-0"><a href="#cite_note-Physics2018-3">[3]</a></sup>.
</p>

<h2><span id=".D0.98.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.8F"></span><span class="mw-headline" id="История">История</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9E%D0%BF%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B8%D0%BD%D1%86%D0%B5%D1%82&amp;veaction=edit&amp;section=1" title="Редактировать раздел «История»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9E%D0%BF%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B8%D0%BD%D1%86%D0%B5%D1%82&amp;action=edit&amp;section=1" title="Редактировать код раздела «История»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Ещё в <a href="/wiki/XVII_%D0%B2%D0%B5%D0%BA" title="XVII век">XVII веке</a> <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%B8%D1%8F" title="Германия">немецкий</a> <a class="mw-redirect" href="/wiki/%D0%90%D1%81%D1%82%D1%80%D0%BE%D0%BD%D0%BE%D0%BC%D1%8B" title="Астрономы">астроном</a> <a href="/wiki/%D0%9A%D0%B5%D0%BF%D0%BB%D0%B5%D1%80,_%D0%98%D0%BE%D0%B3%D0%B0%D0%BD%D0%BD" title="Кеплер, Иоганн">Иоганн Кеплер</a>, исходя из наблюдений за хвостами <a href="/wiki/%D0%9A%D0%BE%D0%BC%D0%B5%D1%82%D0%B0" title="Комета">комет</a> при приближении к <a href="/wiki/%D0%A1%D0%BE%D0%BB%D0%BD%D1%86%D0%B5" title="Солнце">Солнцу</a>, предположил, что свет может оказывать давление на вещество. Хотя позже оказалось, что это не единственный механизм этого отклонения, идея Кеплера оказалась плодотворной для развития астрономии. Например, было показано, что световое (<a class="mw-redirect" href="/wiki/%D0%94%D0%B0%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5_%D1%81%D0%B2%D0%B5%D1%82%D0%B0" title="Давление света">радиационное давление</a>) — один из самых главных механизмов, которые отвечают за динамику частиц в межзвёздном пространстве.
</p><p>Через два столетия после исследований Кеплера <a href="/wiki/%D0%9C%D0%B0%D0%BA%D1%81%D0%B2%D0%B5%D0%BB%D0%BB,_%D0%94%D0%B6%D0%B5%D0%B9%D0%BC%D1%81_%D0%9A%D0%BB%D0%B5%D1%80%D0%BA" title="Максвелл, Джеймс Клерк">Джеймс Максвелл</a> рассчитал значение светового давления с помощью своей <a href="/wiki/%D0%A3%D1%80%D0%B0%D0%B2%D0%BD%D0%B5%D0%BD%D0%B8%D1%8F_%D0%9C%D0%B0%D0%BA%D1%81%D0%B2%D0%B5%D0%BB%D0%BB%D0%B0" title="Уравнения Максвелла">теории электромагнитных явлений</a>. Этот эффект был экспериментально измерен в 1910 году российским физиком <a href="/wiki/%D0%9B%D0%B5%D0%B1%D0%B5%D0%B4%D0%B5%D0%B2,_%D0%9F%D1%91%D1%82%D1%80_%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B5%D0%B2%D0%B8%D1%87" title="Лебедев, Пётр Николаевич">Петром Лебедевым</a>, который продемонстрировал, что свет оказывает давление на тела.
</p><p>В 1970 году описание действия сил, связанных с рассеиванием и градиентами интенсивности света на частицы микронных размеров, было опубликовано в научной литературе <a class="mw-redirect" href="/wiki/%D0%90%D1%88%D0%BA%D0%B8%D0%BD,_%D0%90%D1%80%D1%82%D1%83%D1%80" title="Ашкин, Артур">Артуром Ашкином</a> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Arthur Ashkin</span>), сотрудником <a class="mw-redirect" href="/wiki/Bell_Labs" title="Bell Labs">Bell Labs</a><sup class="reference" id="cite_ref-ashkin_PRB_1970_4-0"><a href="#cite_note-ashkin_PRB_1970-4">[4]</a></sup>.
</p><p>Много позже Ашкин и коллеги сообщили о первом наблюдении того, что в настоящий момент называется оптической ловушкой, то есть сфокусированного пучка света, способного к удержанию микроскопических частиц (10 <a class="mw-redirect" href="/wiki/%D0%9D%D0%BC" title="Нм">нм</a> — 10 <a class="mw-redirect" href="/wiki/%D0%9C%D0%BA%D0%BC" title="Мкм">мкм</a>) неподвижно в трёх измерениях<sup class="reference" id="cite_ref-5"><a href="#cite_note-5">[5]</a></sup>.
</p><p>Подобный принцип используется и для <a href="/wiki/%D0%9B%D0%B0%D0%B7%D0%B5%D1%80%D0%BD%D0%BE%D0%B5_%D0%BE%D1%85%D0%BB%D0%B0%D0%B6%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5" title="Лазерное охлаждение">лазерного охлаждения</a> — метода, который позволил довести <a href="/wiki/%D0%A2%D0%B5%D0%BC%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D1%83%D1%80%D0%B0" title="Температура">температуру</a> атомов в оптической ловушке до наименьших значений, недосягаемых другими средствами. Метод был предложен советским физиком <a href="/wiki/%D0%9B%D0%B5%D1%82%D0%BE%D1%85%D0%BE%D0%B2,_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BB%D0%B5%D0%BD_%D0%A1%D1%82%D0%B5%D0%BF%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87" title="Летохов, Владилен Степанович">Летоховым</a> в 1968 году<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup> и реализован той же группой Ашкина в 1978 году<sup class="reference" id="cite_ref-ashkin_PRL_1978_7-0"><a href="#cite_note-ashkin_PRL_1978-7">[7]</a></sup>. Исследовательскую работу продолжил <a class="mw-redirect" href="/wiki/%D0%A1%D1%82%D0%B8%D0%B2%D0%B5%D0%BD_%D0%A7%D1%83" title="Стивен Чу">Стивен Чу</a> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Steven Chu</span>) (в прошлом сотрудник Ашкина), который за эту работу получил <a href="/wiki/%D0%9D%D0%BE%D0%B1%D0%B5%D0%BB%D0%B5%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%BF%D1%80%D0%B5%D0%BC%D0%B8%D1%8F" title="Нобелевская премия">Нобелевскую премию</a> в 1997 году.
</p><p>В 1980-х годах <a class="new" href="/w/index.php?title=%D0%A1%D1%82%D0%B8%D0%B2%D0%B5%D0%BD_%D0%91%D0%BB%D0%BE%D0%BA&amp;action=edit&amp;redlink=1" title="Стивен Блок (страница отсутствует)">Стивен Блок</a> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Steven Block</span>) и <a class="new" href="/w/index.php?title=%D0%93%D0%BE%D0%B2%D0%B0%D1%80%D0%B4_%D0%91%D0%B5%D1%80%D0%B3&amp;action=edit&amp;redlink=1" title="Говард Берг (страница отсутствует)">Говард Берг</a> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Howard Berg</span>) впервые применили технологию оптического пинцета в <a href="/wiki/%D0%91%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F" title="Биология">биологии</a>, используя её, чтобы удержать <a class="mw-redirect" href="/wiki/%D0%91%D0%B0%D0%BA%D1%82%D0%B5%D1%80%D0%B8%D1%8F" title="Бактерия">бактерию</a> с целью изучения бактериальных <a href="/wiki/%D0%96%D0%B3%D1%83%D1%82%D0%B8%D0%BA" title="Жгутик">жгутиков</a>. Уже в 1990-х годах исследователи, такие как <a class="new" href="/w/index.php?title=%D0%9A%D0%B0%D1%80%D0%BB%D0%BE%D1%81_%D0%91%D1%83%D1%81%D1%82%D0%B0%D0%BC%D0%B0%D0%BD%D1%82%D0%B5&amp;action=edit&amp;redlink=1" title="Карлос Бустаманте (страница отсутствует)">Карлос Бустаманте</a> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Carlos Bustamante</span>), <a class="new" href="/w/index.php?title=%D0%94%D0%B6%D0%B5%D0%B9%D0%BC%D1%81_%D0%A1%D0%BF%D1%83%D0%B4%D0%B8%D1%87&amp;action=edit&amp;redlink=1" title="Джеймс Спудич (страница отсутствует)">Джеймс Спудич</a> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">James Spudich</span>) и Стивен Блок применили принцип оптической <a href="/wiki/%D0%A1%D0%B8%D0%BB%D0%BE%D0%B2%D0%B0%D1%8F_%D1%81%D0%BF%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D1%81%D0%BA%D0%BE%D0%BF%D0%B8%D1%8F" title="Силовая спектроскопия">силовой спектроскопии</a>, чтобы характеризовать <a class="new" href="/w/index.php?title=%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D1%8F%D1%80%D0%BD%D1%8B%D0%B9_%D0%BC%D0%BE%D1%82%D0%BE%D1%80&amp;action=edit&amp;redlink=1" title="Молекулярный мотор (страница отсутствует)">биологические двигатели молекулярного масштаба</a>. Эти молекулярные моторы встречаются повсеместно в биологии, и отвечают за <a class="new" href="/w/index.php?title=%D0%9F%D0%B5%D1%80%D0%B5%D0%B4%D0%B2%D0%B8%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5_%D0%BA%D0%BB%D0%B5%D1%82%D0%BE%D0%BA&amp;action=edit&amp;redlink=1" title="Передвижение клеток (страница отсутствует)">передвижение клеток</a>, изменение их формы и за <a class="new" href="/w/index.php?title=%D0%9A%D0%BB%D0%B5%D1%82%D0%BE%D1%87%D0%BD%D1%8B%D0%B9_%D1%82%D1%80%D0%B0%D0%BD%D1%81%D0%BF%D0%BE%D1%80%D1%82&amp;action=edit&amp;redlink=1" title="Клеточный транспорт (страница отсутствует)">транспорт</a> в пределах <a href="/wiki/%D0%9A%D0%BB%D0%B5%D1%82%D0%BA%D0%B0" title="Клетка">клетки</a>. Оптические ловушки позволили этим<sup class="ts-fix-template">[<i><a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%9F%D1%80%D0%BE%D0%B2%D0%B5%D1%80%D1%8F%D0%B5%D0%BC%D0%BE%D1%81%D1%82%D1%8C#Бремя_доказательства" title="Википедия:Проверяемость"><span title="информация нуждается в подтверждении авторитетными источниками">уточнить</span></a></i>]</sup> <a href="/wiki/%D0%91%D0%B8%D0%BE%D1%84%D0%B8%D0%B7%D0%B8%D0%BA%D0%B0" title="Биофизика">биофизикам</a> наблюдать силы и динамику молекулярных моторов на примере одной молекулы. Оптическая силовая спектроскопия позволила лучше понять стохастическую (случайную) природу этих преобразующих энергию молекул.<sup class="ts-fix-template">[<i><a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%9F%D1%80%D0%BE%D0%B2%D0%B5%D1%80%D1%8F%D0%B5%D0%BC%D0%BE%D1%81%D1%82%D1%8C#Бремя_доказательства" title="Википедия:Проверяемость"><span title="информация нуждается в подтверждении авторитетными источниками">уточнить</span></a></i>]</sup>
</p><p>Оптический пинцет оказался полезными также и в других областях <a href="/wiki/%D0%91%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F" title="Биология">биологии</a>. Например, в 2003 году метод оптического удержания был использован для <a class="new" href="/w/index.php?title=%D0%A1%D0%BE%D1%80%D1%82%D0%B8%D1%80%D0%BE%D0%B2%D0%BA%D0%B0_%D0%BA%D0%BB%D0%B5%D1%82%D0%BE%D0%BA&amp;action=edit&amp;redlink=1" title="Сортировка клеток (страница отсутствует)">сортировки клеток</a>. Создавая большую оптическую интенсивность над образцом, клетки можно сортировать по их собственным оптическим характеристикам<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup><sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>. Оптический пинцет также используется для исследования белков, формирующих <a href="/wiki/%D0%A6%D0%B8%D1%82%D0%BE%D1%81%D0%BA%D0%B5%D0%BB%D0%B5%D1%82" title="Цитоскелет">цитоскелет</a><sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup>, измерения вязкости и упругости <a href="/wiki/%D0%91%D0%B8%D0%BE%D0%BF%D0%BE%D0%BB%D0%B8%D0%BC%D0%B5%D1%80%D1%8B" title="Биополимеры">биополимеров</a> и изучения движения клеток.
</p>
<h2><span id=".D0.A4.D0.B8.D0.B7.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.B8.D0.B5_.D0.BF.D1.80.D0.B8.D0.BD.D1.86.D0.B8.D0.BF.D1.8B"></span><span class="mw-headline" id="Физические_принципы">Физические принципы</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9E%D0%BF%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B8%D0%BD%D1%86%D0%B5%D1%82&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Физические принципы»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9E%D0%BF%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B8%D0%BD%D1%86%D0%B5%D1%82&amp;action=edit&amp;section=2" title="Редактировать код раздела «Физические принципы»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Объекты, представляемые в виде маленьких <a class="mw-redirect" href="/wiki/%D0%94%D0%B8%D1%8D%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%B8%D0%BA%D0%B8" title="Диэлектрики">диэлектрических</a> сфер взаимодействуют с <a href="/wiki/%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%BE%D0%B5_%D0%BF%D0%BE%D0%BB%D0%B5" title="Электрическое поле">электрическим полем</a>, созданным <a href="/wiki/%D0%A1%D0%B2%D0%B5%D1%82" title="Свет">световой</a> волной, за счёт индуцированного на сфере <a class="mw-redirect" href="/wiki/%D0%94%D0%B8%D0%BF%D0%BE%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BC%D0%BE%D0%BC%D0%B5%D0%BD%D1%82" title="Дипольный момент">дипольного момента</a>. В результате взаимодействия этого диполя с электрическим полем электромагнитной волны объект перемещается вдоль <a href="/wiki/%D0%93%D1%80%D0%B0%D0%B4%D0%B8%D0%B5%D0%BD%D1%82" title="Градиент">градиента</a> электрического поля. Кроме <i>градиентной силы</i>, на объект также действует сила, вызванная давлением (<a href="/wiki/%D0%9E%D1%82%D1%80%D0%B0%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5_(%D1%84%D0%B8%D0%B7%D0%B8%D0%BA%D0%B0)" title="Отражение (физика)">отражением</a>) света от его поверхности. Эта сила толкает сферу по направлению пучка света. Однако, если луч света сильно сфокусирован, величина градиента интенсивности может быть больше величины давления света<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup>.
</p><p>Более детальный анализ основан на двух механизмах, предложенных Ашкином, в зависимости от размера частицы. Из <a href="/wiki/%D0%A0%D0%B0%D1%81%D1%81%D0%B5%D1%8F%D0%BD%D0%B8%D0%B5_%D1%81%D0%B2%D0%B5%D1%82%D0%B0" title="Рассеяние света">теории рассеяния света</a> известно, что механизм рассеяния света частицей зависит от соотношения размеров частицы и длины световой волны. Если размер рассеивающих частиц намного меньше, чем <a href="/wiki/%D0%94%D0%BB%D0%B8%D0%BD%D0%B0_%D0%B2%D0%BE%D0%BB%D0%BD%D1%8B" title="Длина волны">длина волны</a> света, то имеет место <a href="/wiki/%D0%A0%D1%8D%D0%BB%D0%B5%D0%B5%D0%B2%D1%81%D0%BA%D0%BE%D0%B5_%D1%80%D0%B0%D1%81%D1%81%D0%B5%D1%8F%D0%BD%D0%B8%D0%B5" title="Рэлеевское рассеяние">рэлеевское рассеяние</a>. Когда свет рассеивается на частицах (пыль, дым, водные капельки), которые имеют размер больше, чем длина волны, это <a href="/wiki/%D0%A0%D0%B0%D1%81%D1%81%D0%B5%D1%8F%D0%BD%D0%B8%D0%B5_%D0%9C%D0%B8" title="Рассеяние Ми">рассеяние Ми</a> (по имени <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%B8%D1%8F" title="Германия">немецкого</a> физика <a class="mw-redirect" href="/wiki/%D0%93%D1%83%D1%81%D1%82%D0%B0%D0%B2_%D0%9C%D0%B8" title="Густав Ми">Густава Ми</a>). Рассеяние Ми отвечает за белый и серый цвет <a class="mw-redirect" href="/wiki/%D0%9E%D0%B1%D0%BB%D0%B0%D0%BA%D0%BE" title="Облако">облаков</a>.
</p><p>Придерживаясь той же идеи, Ашкин предложил, что для математического анализа оптического микроманипулирования можно использовать два разных метода, а именно: подходом <a class="mw-redirect" href="/wiki/%D0%A4%D0%B8%D0%B7%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D1%82%D0%B8%D0%BA%D0%B0" title="Физическая оптика">физической оптики</a> для миевских частиц (когда диаметр частицы больше <a href="/wiki/%D0%94%D0%BB%D0%B8%D0%BD%D0%B0_%D0%B2%D0%BE%D0%BB%D0%BD%D1%8B" title="Длина волны">длины волны</a> света <i>d</i> &gt; λ) и в приближении <a class="mw-redirect" href="/wiki/%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B4%D0%B8%D0%BF%D0%BE%D0%BB%D1%8C" title="Электрический диполь">электрического диполя</a> для релеевских частиц (<i>d</i> &lt; λ).
</p>
<h3><span id=".D0.A4.D0.B8.D0.B7.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.B0.D1.8F_.D0.BE.D0.BF.D1.82.D0.B8.D0.BA.D0.B0"></span><span class="mw-headline" id="Физическая_оптика">Физическая оптика</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9E%D0%BF%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B8%D0%BD%D1%86%D0%B5%D1%82&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Физическая оптика»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9E%D0%BF%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B8%D0%BD%D1%86%D0%B5%D1%82&amp;action=edit&amp;section=3" title="Редактировать код раздела «Физическая оптика»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>

<p>При анализе с использованием физической оптики рассмотрение процессов преломления и отражения света от микросферы достаточно, чтобы проанализировать втягивание в оптическую ловушку (см. рисунок справа).
</p><p>Самый простой расчёт действующих сил в пределах подхода физической оптики основывается на <a href="/wiki/%D0%93%D0%B5%D0%BE%D0%BC%D0%B5%D1%82%D1%80%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%BF%D1%82%D0%B8%D0%BA%D0%B0" title="Геометрическая оптика">геометрической оптике</a>. Рассмотрение луча указывает на изменение <a href="/wiki/%D0%98%D0%BC%D0%BF%D1%83%D0%BB%D1%8C%D1%81" title="Импульс">импульса</a> света при отражении и преломлении. Таким образом, это изменение импульса (<a href="/wiki/%D0%A4%D0%BE%D1%82%D0%BE%D0%BD" title="Фотон">фотона</a> как частицы), согласно второму <a href="/wiki/%D0%97%D0%B0%D0%BA%D0%BE%D0%BD%D1%8B_%D0%9D%D1%8C%D1%8E%D1%82%D0%BE%D0%BD%D0%B0" title="Законы Ньютона">закону Ньютона</a>, будет приводить к возникновению силы.
</p><p>Используя простую диаграмму лучей и вектора силы, можно показать, что на микросферу действуют две разные оптические силы благодаря <a href="/wiki/%D0%98%D0%BD%D0%B5%D1%80%D1%86%D0%B8%D1%8F" title="Инерция">инерции</a> падающего и <a href="/wiki/%D0%9F%D1%80%D0%B5%D0%BB%D0%BE%D0%BC%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5" title="Преломление">преломлённого</a> света. Как это видно из диаграммы, результирующая сила толкает сферу в направлении области наивысшей интенсивности луча. Такая сила называется <i>градиентной силой</i>.
</p><p>Ашкин в своём первом эксперименте<sup class="reference" id="cite_ref-ashkin_PRB_1970_4-1"><a href="#cite_note-ashkin_PRB_1970-4">[4]</a></sup> использовал милливаттный <a href="/wiki/%D0%93%D0%B0%D1%83%D1%81%D1%81%D0%BE%D0%B2_%D0%BF%D1%83%D1%87%D0%BE%D0%BA" title="Гауссов пучок">гауссов пучок</a> одномодового (TEM<sub>00</sub>) <a href="/wiki/%D0%90%D1%80%D0%B3%D0%BE%D0%BD%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BB%D0%B0%D0%B7%D0%B5%D1%80" title="Аргоновый лазер">аргонового лазера</a> с длиной волны 514,5 нм, сфокусированного в пятно диаметром <i>w</i><sub>0</sub> = 6,2 мкм. Он двигал с помощью этого пучка <a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B5%D0%BA%D1%81" title="Латекс">латексные</a> сферы диаметром 0,51; 1,31 и 2,68 мкм в воде и воздухе. Это можно сделать за счёт двух встречных пучков света, которые толкают сферу в противоположных направлениях, или с помощью сильно сфокусированного <a href="/wiki/%D0%93%D0%B0%D1%83%D1%81%D1%81%D0%BE%D0%B2_%D0%BF%D1%83%D1%87%D0%BE%D0%BA" title="Гауссов пучок">гауссового пучка</a> (с высокой <a class="mw-redirect" href="/wiki/%D0%A7%D0%B8%D1%81%D0%BB%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B0%D0%BF%D0%B5%D1%80%D1%82%D1%83%D1%80%D0%B0" title="Числовая апертура">числовой апертурой</a>, NA &gt; 1,0), чтобы компенсировать давление света высокой <i>градиентной силой</i>.
</p><p>С другой стороны, в релеевском режиме частицы не ограничены по форме. Вообще, для наименьших частиц нужна наименьшая сила притягивания. В большинстве случаев для объяснения рабочего механизма лазерного пинцета для любой формы частиц используют модель приведённого диполя. Электромагнитное излучение будет индуцировать дипольный момент, или <a class="mw-redirect" href="/wiki/%D0%9F%D0%BE%D0%BB%D1%8F%D1%80%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F_%D0%B4%D0%B8%D1%8D%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%B8%D0%BA%D0%B0" title="Поляризация диэлектрика">поляризацию</a>, в диэлектрической частице. Сила взаимодействия этого диполя со светом приводит к <i>градиентной силе</i> притягивания.

<p>Квадрат величины электрического поля равен интенсивности луча как функция координат. Поэтому результат указывает, что сила, действующая на диэлектрическую частицу, в приближении точечного диполя является пропорциональной градиенту интенсивности пучка. Другими словами, описанная здесь сила приводит к притяжению частицы в область с самой высокой интенсивностью. В действительности сила, возникающая при рассеянии света зависит линейно от интенсивности луча, поперечного сечения частицы и <a class="mw-redirect" href="/wiki/%D0%9A%D0%BE%D1%8D%D1%84%D1%84%D0%B8%D1%86%D0%B8%D0%B5%D0%BD%D1%82_%D0%BF%D1%80%D0%B5%D0%BB%D0%BE%D0%BC%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F" title="Коэффициент преломления">показателя преломления</a> среды, в которой находится ловушка (например, вода), работает против <i>градиентной силы</i> в осевом направлении ловушки, приводя к тому, что равновесное положение смещается немного вниз от положения максимума интенсивности.
</p>
<h2><span id=".D0.9B.D0.B0.D0.B7.D0.B5.D1.80.D0.BD.D1.8B.D0.B9_.D0.BF.D0.B8.D0.BD.D1.86.D0.B5.D1.82.2C_.D0.BE.D1.81.D0.BD.D0.BE.D0.B2.D0.B0.D0.BD.D0.BD.D1.8B.D0.B9_.D0.BD.D0.B0_.D0.B0.D0.BB.D1.8C.D1.82.D0.B5.D1.80.D0.BD.D0.B0.D1.82.D0.B8.D0.B2.D0.BD.D1.8B.D1.85_.D1.80.D0.B5.D0.B6.D0.B8.D0.BC.D0.B0.D1.85_.D1.80.D0.B0.D0.B1.D0.BE.D1.82.D1.8B_.D0.BB.D0.B0.D0.B7.D0.B5.D1.80.D0.B0"></span><span class="mw-headline" id="Лазерный_пинцет,_основанный_на_альтернативных_режимах_работы_лазера">Лазерный пинцет, основанный на альтернативных режимах работы лазера</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9E%D0%BF%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B8%D0%BD%D1%86%D0%B5%D1%82&amp;veaction=edit&amp;section=5" title="Редактировать раздел «Лазерный пинцет, основанный на альтернативных режимах работы лазера»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9E%D0%BF%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B8%D0%BD%D1%86%D0%B5%D1%82&amp;action=edit&amp;section=5" title="Редактировать код раздела «Лазерный пинцет, основанный на альтернативных режимах работы лазера»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Со времени изобретения первого лазерного пинцета, основанного на одном гауссовом пучке (фундаментальная лазерная мода TEM<sub>00</sub>) А. Ашкином в 1986 году<sup class="reference" id="cite_ref-14"><a href="#cite_note-14">[14]</a></sup> концепция одномодовых лазерных пучков развилась за счёт использования <a class="new" href="/w/index.php?title=%D0%9B%D0%B0%D0%B7%D0%B5%D1%80%D0%BD%D1%8B%D0%B5_%D0%BC%D0%BE%D0%B4%D1%8B&amp;action=edit&amp;redlink=1" title="Лазерные моды (страница отсутствует)">лазерных мод</a> высокого порядка, то есть эрмит-гауссовых пучков (TEM<sub>nm</sub>), лагерр-гауссовых пучков (LG, TEM<sub>pl</sub>) и <a href="/wiki/%D0%9B%D1%83%D1%87_%D0%91%D0%B5%D1%81%D1%81%D0%B5%D0%BB%D1%8F" title="Луч Бесселя">бесселевых пучков</a> (<i>J<sub>n</sub></i>).
</p><p>Лагерр-гауссовы пучки обладают уникальным свойством втягивать в оптическую ловушку оптически отражающие и поглощающие частицы. Пучки с круговой <a href="/wiki/%D0%9F%D0%BE%D0%BB%D1%8F%D1%80%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F_%D0%B2%D0%BE%D0%BB%D0%BD" title="Поляризация волн">поляризацией</a> имеют <a href="/wiki/%D0%A1%D0%BF%D0%B8%D0%BD" title="Спин">спиновый орбитальный момент</a> и могут вращать частицы. У лагерр-гауссовых пучков также есть собственный <a class="mw-redirect" href="/wiki/%D0%A3%D0%B3%D0%BB%D0%BE%D0%B2%D0%BE%D0%B9_%D0%BC%D0%BE%D0%BC%D0%B5%D0%BD%D1%82" title="Угловой момент">угловой момент</a>, который может вращать частицы вокруг центра пучка<sup class="reference" id="cite_ref-15"><a href="#cite_note-15">[15]</a></sup><sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup>. Этот эффект наблюдается без внешней механической или электрической регуляции луча.
</p><p>Кроме лагерр-гауссовых пучков, <a href="/wiki/%D0%9B%D1%83%D1%87_%D0%91%D0%B5%D1%81%D1%81%D0%B5%D0%BB%D1%8F" title="Луч Бесселя">бесселевы пучки</a> как нулевого, так и высших порядков имеют орбитальный момент, а также уникальное свойство одновременно удерживать много частиц на некотором расстоянии<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup>.
</p>
<h2><span id=".D0.9C.D1.83.D0.BB.D1.8C.D1.82.D0.B8.D0.BF.D0.BB.D0.B5.D0.BA.D1.81.D0.BD.D1.8B.D0.B5_.D0.BB.D0.B0.D0.B7.D0.B5.D1.80.D0.BD.D1.8B.D0.B5_.D0.BF.D0.B8.D0.BD.D1.86.D0.B5.D1.82.D1.8B"></span><span class="mw-headline" id="Мультиплексные_лазерные_пинцеты">Мультиплексные лазерные пинцеты</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9E%D0%BF%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B8%D0%BD%D1%86%D0%B5%D1%82&amp;veaction=edit&amp;section=6" title="Редактировать раздел «Мультиплексные лазерные пинцеты»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9E%D0%BF%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B8%D0%BD%D1%86%D0%B5%D1%82&amp;action=edit&amp;section=6" title="Редактировать код раздела «Мультиплексные лазерные пинцеты»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Типичная установка имеет только один или два лазерных луча. Более сложные эксперименты требуют много ловушек, работающих одновременно. Этого можно достичь, используя единственный лазер, свет которого проходит через <a class="new" href="/w/index.php?title=%D0%90%D0%BA%D1%83%D1%81%D1%82%D0%BE-%D0%BE%D0%BF%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BC%D0%BE%D0%B4%D1%83%D0%BB%D1%8F%D1%82%D0%BE%D1%80&amp;action=edit&amp;redlink=1" title="Акусто-оптический модулятор (страница отсутствует)">акусто-оптический модулятор</a> или через управляемые электроникой зеркала. С помощью этих устройств лазерное излучение можно разделить во времени на несколько лучей, а с помощью <a href="/wiki/%D0%94%D0%B8%D1%84%D1%80%D0%B0%D0%BA%D1%86%D0%B8%D1%8F" title="Дифракция">дифракционных</a> оптических элементов — разбить на несколько лучей в пространстве<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup><sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup><sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup><sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup>.
</p>
<h2><span id=".D0.9B.D0.B0.D0.B7.D0.B5.D1.80.D0.BD.D1.8B.D0.B5_.D0.BF.D0.B8.D0.BD.D1.86.D0.B5.D1.82.D1.8B.2C_.D0.BE.D1.81.D0.BD.D0.BE.D0.B2.D0.B0.D0.BD.D0.BD.D1.8B.D0.B5_.D0.BD.D0.B0_.D0.BE.D0.BF.D1.82.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.B8.D1.85_.D0.B2.D0.BE.D0.BB.D0.BE.D0.BA.D0.BD.D0.B0.D1.85"></span><span class="mw-headline" id="Лазерные_пинцеты,_основанные_на_оптических_волокнах">Лазерные пинцеты, основанные на оптических волокнах</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9E%D0%BF%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B8%D0%BD%D1%86%D0%B5%D1%82&amp;veaction=edit&amp;section=7" title="Редактировать раздел «Лазерные пинцеты, основанные на оптических волокнах»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9E%D0%BF%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B8%D0%BD%D1%86%D0%B5%D1%82&amp;action=edit&amp;section=7" title="Редактировать код раздела «Лазерные пинцеты, основанные на оптических волокнах»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>В этом типе устройств лазерное излучение подаётся через <a href="/wiki/%D0%9E%D0%BF%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%BE%D0%B5_%D0%B2%D0%BE%D0%BB%D0%BE%D0%BA%D0%BD%D0%BE" title="Оптическое волокно">оптическое волокно</a>. Если один конец оптического волокна формирует поверхность, похожую по свойствам на линзу, это позволит сфокусировать свет в оптической ловушке с большой числовой апертурой<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup>.
</p><p>Если же концы волокна не выпуклы, лазерный свет будет отклоняться, и потому стабильная оптическая ловушка может быть создана только с помощью двух концов волокон по обе стороны от оптической ловушки, и балансирующих <i>градиентных сил</i> и давлений света. <i>Градиентные силы</i> удерживают частицы в поперечном направлении, тогда как осевая оптическая сила возникает от давления двух встречных пучков света, которые выходят и распространяются из двух оптических волокон. Равновесное положение сферы вдоль оси <i>z</i> в такой ловушке — положение, где давления света равняются друг другу. Такие лазерные пинцеты были впервые созданы А. Констеблем<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup> и Дж. Гюком<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup>, которые использовали эту методику для растяжения микрочастиц. Манипулируя входной мощностью из обоих концов оптоволокна, возможно регулировать растягивающую силу. Такую систему можно использовать, чтобы измерять вязкость и эластичность клеток с чувствительностью, достаточной, чтобы различить разные <a href="/wiki/%D0%A6%D0%B8%D1%82%D0%BE%D1%81%D0%BA%D0%B5%D0%BB%D0%B5%D1%82" title="Цитоскелет">цитоскелеты</a>, например, <a class="mw-redirect" href="/wiki/%D0%AD%D1%80%D0%B8%D1%82%D1%80%D0%BE%D1%86%D0%B8%D1%82" title="Эритроцит">эритроцитов</a> человека и <a href="/wiki/%D0%A4%D0%B8%D0%B1%D1%80%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82" title="Фибробласт">фибробластов</a> мышей. Недавние исследования достигли большого успеха в дифференциации <a class="mw-redirect" href="/wiki/%D0%A0%D0%B0%D0%BA_(%D0%B1%D0%BE%D0%BB%D0%B5%D0%B7%D0%BD%D1%8C)" title="Рак (болезнь)">раковых</a> клеток от нормальных<sup class="reference" id="cite_ref-25"><a href="#cite_note-25">[25]</a></sup>.
</p>

<p>Одна из наиболее распространённых систем сортировки <a href="/wiki/%D0%9A%D0%BB%D0%B5%D1%82%D0%BA%D0%B0" title="Клетка">клеток</a> использует метод <a href="/wiki/%D0%A4%D0%BB%D1%83%D0%BE%D1%80%D0%B5%D1%81%D1%86%D0%B5%D0%BD%D1%86%D0%B8%D1%8F" title="Флуоресценция">флуоресцентной</a> <a href="/wiki/%D0%9F%D1%80%D0%BE%D1%82%D0%BE%D1%87%D0%BD%D0%B0%D1%8F_%D1%86%D0%B8%D1%82%D0%BE%D0%BC%D0%B5%D1%82%D1%80%D0%B8%D1%8F" title="Проточная цитометрия">проточной цитометрии</a>. В этом методе <a href="/wiki/%D0%A1%D1%83%D1%81%D0%BF%D0%B5%D0%BD%D0%B7%D0%B8%D1%8F" title="Суспензия">суспензия</a> биологических клеток сортируется в несколько контейнеров согласно флюоресцентным характеристиках каждой клетки в потоке. Процесс сортировки контролируется электростатической системой отклонения, которая направляет клетку к определённому контейнеру изменением напряжения приложенного электрического поля.
</p><p>В оптически управляемой системе сортировки клетки пропускают через двух- или трёхмерные оптические решётки. Без индуцируемого электрического напряжения клетки сортируются по их свойствам преломления света. Группа исследователей под руководством Кишана Долакиа разработала методику использования дифракционной оптики и других оптических элементов для создания таких оптических решёток<sup class="reference" id="cite_ref-26"><a href="#cite_note-26">[26]</a></sup>. С другой стороны, группа учёных из <a class="mw-redirect" href="/wiki/%D0%A2%D0%BE%D1%80%D0%BE%D0%BD%D1%82%D1%81%D0%BA%D0%B8%D0%B9_%D1%83%D0%BD%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D1%82%D0%B5%D1%82" title="Торонтский университет">университета города Торонто</a> построила автоматическую сортировальную систему, используя пространственный модулятор света<sup class="reference" id="cite_ref-27"><a href="#cite_note-27">[27]</a></sup>.
</p><p>Главный механизм сортировки — расположение узлов оптической решётки. Когда поток клеток проходит через оптические решётки, силы трения частиц непосредственно конкурируют с оптической <i>градиентной силой</i> от соседнего узла оптической решётки. Изменяя расположение узлов, возможно создать оптическую дорожку, по который будут двигаться клетки. Но такая дорожка будет эффективной только для клеток с определённым коэффициентом преломления, которые и будут эффективно отклоняться. Регулируя скорость потока клеток и мощность света, возможно получить хорошую оптическую сортировку клеток.
</p><p>Соревнование сил в системе сортировки нуждается в точной юстировке, чтобы достичь высокой эффективности оптической сортировки. В настоящий момент в <a href="/wiki/%D0%A1%D0%B5%D0%BD%D1%82-%D0%AD%D0%BD%D0%B4%D1%80%D1%8E%D1%81%D1%81%D0%BA%D0%B8%D0%B9_%D1%83%D0%BD%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D1%82%D0%B5%D1%82" title="Сент-Эндрюсский университет">Сент-Эндрюсском университете</a> создана большая исследовательская группа для работы над этой проблемой. В случае успеха эта технология сможет заменить традиционную флюоресцентную сортировку клеток<sup class="reference" id="cite_ref-28"><a href="#cite_note-28">[28]</a></sup>.
</p>
<h3><span id=".D0.9B.D0.B0.D0.B7.D0.B5.D1.80.D0.BD.D1.8B.D0.B5_.D0.BF.D0.B8.D0.BD.D1.86.D0.B5.D1.82.D1.8B.2C_.D0.BE.D1.81.D0.BD.D0.BE.D0.B2.D0.B0.D0.BD.D0.BD.D1.8B.D0.B5_.D0.BD.D0.B0_.D0.B7.D0.B0.D1.82.D1.83.D1.85.D0.B0.D1.8E.D1.89.D0.B8.D1.85_.D0.BF.D0.BE.D0.BB.D1.8F.D1.85"></span><span class="mw-headline" id="Лазерные_пинцеты,_основанные_на_затухающих_полях">Лазерные пинцеты, основанные на затухающих полях</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9E%D0%BF%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B8%D0%BD%D1%86%D0%B5%D1%82&amp;veaction=edit&amp;section=9" title="Редактировать раздел «Лазерные пинцеты, основанные на затухающих полях»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9E%D0%BF%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B8%D0%BD%D1%86%D0%B5%D1%82&amp;action=edit&amp;section=9" title="Редактировать код раздела «Лазерные пинцеты, основанные на затухающих полях»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p><a class="new" href="/w/index.php?title=%D0%97%D0%B0%D1%82%D1%83%D1%85%D0%B0%D1%8E%D1%89%D0%B5%D0%B5_%D0%BF%D0%BE%D0%BB%D0%B5&amp;action=edit&amp;redlink=1" title="Затухающее поле (страница отсутствует)">Затухающее поле</a> — <a href="/wiki/%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BC%D0%B0%D0%B3%D0%BD%D0%B8%D1%82%D0%BD%D0%BE%D0%B5_%D0%BF%D0%BE%D0%BB%D0%B5" title="Электромагнитное поле">электромагнитное поле</a>, которое проникает вглубь вещества, например при <a class="mw-redirect" href="/wiki/%D0%9F%D0%BE%D0%BB%D0%BD%D0%BE%D0%B5_%D0%B2%D0%BD%D1%83%D1%82%D1%80%D0%B5%D0%BD%D0%BD%D0%B5%D0%B5_%D0%BE%D1%82%D1%80%D0%B0%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5" title="Полное внутреннее отражение">полном внутреннем отражении</a><sup class="reference" id="cite_ref-29"><a href="#cite_note-29">[29]</a></sup><sup class="reference" id="cite_ref-30"><a href="#cite_note-30">[30]</a></sup>. Электрическое поле в световой волне затухает по <a href="/wiki/%D0%AD%D0%BA%D1%81%D0%BF%D0%BE%D0%BD%D0%B5%D0%BD%D1%82%D0%B0" title="Экспонента">экспоненте</a>. Затухающее поле нашло целый ряд применений в оптической микроскопии нанометровых объектов, оптическая микроманипуляция (лазерные пинцеты) становятся ещё одним его приложением.
</p><p>В лазерных пинцетах непрерывное затухающее поле может быть создано, когда свет распространяется через оптический <a href="/wiki/%D0%92%D0%BE%D0%BB%D0%BD%D0%BE%D0%B2%D0%BE%D0%B4" title="Волновод">волновод</a> (многократное полное внутреннее отражение). Результирующее затухающее поле имеет направленный импульс и может двигать микрочастицы вдоль направления своего распространения. Этот эффект был открыт учёными С. Каватой и Т. Сугиурой в 1992 году<sup class="reference" id="cite_ref-31"><a href="#cite_note-31">[31]</a></sup><sup class="reference" id="cite_ref-32"><a href="#cite_note-32">[32]</a></sup>. Они показали, что поле может связывать частицы, находящиеся на расстоянии около <span class="nowrap">100 нм</span>. Это прямое связывание поля рассматривается как <a href="/wiki/%D0%A2%D1%83%D0%BD%D0%BD%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D1%8D%D1%84%D1%84%D0%B5%D0%BA%D1%82" title="Туннельный эффект">туннелирование</a> фотонов через промежуток между <a href="/wiki/%D0%9F%D1%80%D0%B8%D0%B7%D0%BC%D0%B0_(%D0%BE%D0%BF%D1%82%D0%B8%D0%BA%D0%B0)" title="Призма (оптика)">призмой</a> и микрочастицами. В результате возникает направленная оптическая сила.
</p><p>Недавняя версия лазерных пинцетов основанных на затухающем поле использует большую поверхность с оптической решёткой, что позволяет одновременно связывать много частиц и направлять их в желаемом направлении без использования волновода. Эта методика названа «безлинзовая оптическая ловушка» (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">lensless optical trapping</span>, LOT)<sup class="reference" id="cite_ref-33"><a href="#cite_note-33">[33]</a></sup>. Точно направленному движению частиц помогает <a class="new" href="/w/index.php?title=%D0%A0%D0%B5%D1%88%D1%91%D1%82%D0%BA%D0%B0_%D0%A0%D0%BE%D0%BD%D1%87%D0%B8&amp;action=edit&amp;redlink=1" title="Решётка Рончи (страница отсутствует)">решётка Рончи</a> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Ronchi ruling</span>) или создание чётких оптических <a href="/wiki/%D0%9F%D0%BE%D1%82%D0%B5%D0%BD%D1%86%D0%B8%D0%B0%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D1%8F%D0%BC%D0%B0" title="Потенциальная яма">потенциальных ям</a> в стеклянной пластинке. В настоящий момент учёные также работают над фокусировкой затухающих полей.
</p>
<h2><span id=".D0.9D.D0.B5.D0.BF.D1.80.D1.8F.D0.BC.D0.BE.D0.B9_.D0.BF.D0.BE.D0.B4.D1.85.D0.BE.D0.B4_.D0.BA_.D0.BE.D0.BF.D1.82.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.B8.D0.BC_.D0.BF.D0.B8.D0.BD.D1.86.D0.B5.D1.82.D0.B0.D0.BC"></span><span class="mw-headline" id="Непрямой_подход_к_оптическим_пинцетам">Непрямой подход к оптическим пинцетам</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9E%D0%BF%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B8%D0%BD%D1%86%D0%B5%D1%82&amp;veaction=edit&amp;section=10" title="Редактировать раздел «Непрямой подход к оптическим пинцетам»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9E%D0%BF%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B8%D0%BD%D1%86%D0%B5%D1%82&amp;action=edit&amp;section=10" title="Редактировать код раздела «Непрямой подход к оптическим пинцетам»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Ещё один вариант манипулирования микрочастицами с помощью света был разработан Минг Ву (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a> <span lang="en" style="font-style:italic;">Ming Wu</span>), профессором факультета Радиотехники и информатики в <a href="/wiki/%D0%9A%D0%B0%D0%BB%D0%B8%D1%84%D0%BE%D1%80%D0%BD%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%83%D0%BD%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D1%82%D0%B5%D1%82_%D0%B2_%D0%91%D0%B5%D1%80%D0%BA%D0%BB%D0%B8" title="Калифорнийский университет в Беркли">Калифорнийском университете</a>. Его система не использует световой импульс непосредственно. В отличие от этого, в построенной им системе, частицы, которыми нужно манипулировать, расположены неподалёку от стеклянной пластинки, покрытой фотопроводящим веществом. На эту пластинку подается небольшое напряжение с целью создания электростатического заряда на частицах. Фотопроводящая пластинка освещается светодиодами, мощность которых может модулироваться, проецируя на поверхность любое динамическое изображение. Под действием света фотопроводящая поверхность заряжается, начиная притягивать или отталкивать частицы. Процесс манипуляции выполняется посредством изменения электрического поля и включается с помощью спроектированного изображения<sup class="reference" id="cite_ref-34"><a href="#cite_note-34">[34]</a></sup>.
</p><p>Одно из применений этого метода — сортировка живых и мёртвых клеток. Сортировка основывается на том, что живые клетки наполнены <a href="/wiki/%D0%AD%D0%BB%D0%B5%D0%BA%D1%82%D1%80%D0%BE%D0%BB%D0%B8%D1%82" title="Электролит">электролитом</a>, а мёртвые — нет, и их можно легко разделить. Эта система позволяет манипулировать 10000 клеток или частиц одновременно<sup class="reference" id="cite_ref-35"><a href="#cite_note-35">[35]</a></sup>.
</p>
<h3><span id=".D0.9E.D0.BF.D1.82.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.BE.D0.B5_.D1.81.D0.B2.D1.8F.D0.B7.D1.8B.D0.B2.D0.B0.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Оптическое_связывание">Оптическое связывание</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9E%D0%BF%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B8%D0%BD%D1%86%D0%B5%D1%82&amp;veaction=edit&amp;section=11" title="Редактировать раздел «Оптическое связывание»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9E%D0%BF%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B8%D0%BD%D1%86%D0%B5%D1%82&amp;action=edit&amp;section=11" title="Редактировать код раздела «Оптическое связывание»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Когда множество микрочастиц поддерживается монохроматическим лазерным пучком, расположение микрочастиц в пределах оптической ловушки зависит от перераспределения оптических сил между частицами. Можно сказать, что кластер микрочастиц <i>связывается</i> вместе светом. Первые эксперименты по оптическому связыванию были поставлены в лаборатории <a href="/wiki/%D0%93%D0%BE%D0%BB%D0%BE%D0%B2%D1%87%D0%B5%D0%BD%D0%BA%D0%BE,_%D0%94%D0%B6%D0%B8%D0%BD_%D0%AD%D0%BD%D0%B4%D1%80%D1%8E" title="Головченко, Джин Эндрю">Евгения Головченко</a> в <a href="/wiki/%D0%93%D0%B0%D1%80%D0%B2%D0%B0%D1%80%D0%B4%D1%81%D0%BA%D0%B8%D0%B9_%D1%83%D0%BD%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D1%82%D0%B5%D1%82" title="Гарвардский университет">Гарвардском университете</a><sup class="reference" id="cite_ref-36"><a href="#cite_note-36">[36]</a></sup>.
</p>
<h2><span id=".D0.98.D0.B7.D0.BC.D0.B5.D1.80.D0.B5.D0.BD.D0.B8.D0.B5_.D0.BE.D0.BF.D1.82.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.B8.D1.85_.D1.81.D0.B8.D0.BB"></span><span class="mw-headline" id="Измерение_оптических_сил">Измерение оптических сил</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a class="mw-editsection-visualeditor" href="/w/index.php?title=%D0%9E%D0%BF%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B8%D0%BD%D1%86%D0%B5%D1%82&amp;veaction=edit&amp;section=12" title="Редактировать раздел «Измерение оптических сил»"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9E%D0%BF%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B8%D0%BD%D1%86%D0%B5%D1%82&amp;action=edit&amp;section=12" title="Редактировать код раздела «Измерение оптических сил»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>В настоящее время сила притягивания может быть измерена как на одно-, так и на двухпучковых лазерных пинцетах (<a class="new" href="/w/index.php?title=%D0%A4%D0%BE%D1%82%D0%BE%D0%BD%D0%BD%D1%8B%D0%B9_%D1%81%D0%B8%D0%BB%D0%BE%D0%B2%D0%BE%D0%B9_%D0%BC%D0%B8%D0%BA%D1%80%D0%BE%D1%81%D0%BA%D0%BE%D0%BF&amp;action=edit&amp;redlink=1" title="Фотонный силовой микроскоп (страница отсутствует)">фотонный силовой микроскоп</a>)<sup class="reference" id="cite_ref-37"><a href="#cite_note-37">[37]</a></sup><sup class="reference" id="cite_ref-38"><a href="#cite_note-38">[38]</a></sup>. Недавно начались работы по измерению оптических сил в <a href="/wiki/%D0%93%D0%BE%D0%BB%D0%BE%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D1%8F" title="Голография">голографических</a> лазерных пинцетах, чтобы достичь высокой точности позиционирования ловушек для отдельных атомов<sup class="reference" id="cite_ref-39"><a href="#cite_note-39">[39]</a></sup><sup class="reference" id="cite_ref-40"><a href="#cite_note-40">[40]</a></sup><sup class="reference" id="cite_ref-41"><a href="#cite_note-41">[41]</a></sup>.
</p><p>Основной принцип измерения оптической силы лазерных пинцетов — передача <a href="/wiki/%D0%98%D0%BC%D0%BF%D1%83%D0%BB%D1%8C%D1%81" title="Импульс">импульса</a> света, связанная с <a href="/wiki/%D0%9F%D1%80%D0%B5%D0%BB%D0%BE%D0%BC%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5" title="Преломление">преломлением света</a> на частицах. Изменение направления распространения света как в поперечном, так и в продольному направлении обеспечивает силу, которая действует на объект. Поэтому наименьшая поперечная сила может быть измерена по отклонению пучка, который прошёл сквозь частицу. Такое отклонение может быть легко измерено с помощью детектора осевого положения, самый простой из которых — <a class="new" href="/w/index.php?title=%D0%9A%D0%B2%D0%B0%D0%B4%D1%80%D0%B0%D0%BD%D1%82%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D1%82%D0%BE%D0%B4%D0%B8%D0%BE%D0%B4&amp;action=edit&amp;redlink=1" title="Квадрантный фотодиод (страница отсутствует)">квадрантный фотодиод</a>: пластинка, разделённая на четыре сектора, с пучком света сфокусированным в её центре. При частице в центре на секторы падает свет равной мощности, но если на частицу действует сила, мощности уже не будут равны, и их разница пропорциональна этой силе.
</p><p>Такой принцип может применяться с любыми лазерными пинцетами. Наибольшей проблемой при таких измерениях будет <a href="/wiki/%D0%91%D1%80%D0%BE%D1%83%D0%BD%D0%BE%D0%B2%D1%81%D0%BA%D0%BE%D0%B5_%D0%B4%D0%B2%D0%B8%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5" title="Броуновское движение">броуновское движение</a> (шум). Тем не менее силы порядка пиконьютона и сдвиг порядка нанометров обычно можно измерить<sup class="reference" id="cite_ref-42"><a href="#cite_note-42">[42]</a></sup>.
</p>
